# Vajrayana/Tantra (金刚乘/密宗核心)

### The Four Classes of Tantra (密宗四续部详表)
| 续部名称 | 梵文 (Sanskrit) | 核心特质与修行重点 | 心理/精神隐喻 | 专业视角 (Professional Perspective) |
| :--- | :--- | :--- | :--- | :--- |
| **事续** | Kriya Tantra | 强调外部清净、仪式和洗涤。 | 通过行为的规范引导内心秩序。 | 旨在建立基础的敬畏感与纪律性。 |
| **行续** | Charya Tantra | 内外平衡；将行为与禅定结合。 | 将神圣性融入日常互动。 | 是从外在转向内在观想的过渡阶段。 |
| **瑜伽续** | Yoga Tantra | 强调内在的禅定与空性观想。 | 深度的心理整合与原型共振。 | 核心是通过“五相成身”实现身口意的本尊化。 |
| **无上瑜伽续** | Anuttarayoga Tantra | 处理最细微的意识与能量（气脉明点）。 | 彻底的本初清净与大乐合一。 | 包含父续（明性）、母续（空性）及不二续。 |

### The Vajra Body: Nadi, Prana, Bindu (金刚身：脉、气、明点系统表)
| 组件名称 | 梵文/藏文 (Skt/Tib) | 定义与功能 | 心理/生理对应 | 专业视角 (Professional Perspective) |
| :--- | :--- | :--- | :--- | :--- |
| **脉** | Nadi / rtsa | 能量流通的通道。重点：中脉、左右脉。 | 系统结构的蓝图；类似神经系统。 | 脉的结使直接限制了意识的扩展。 |
| **气** | Prana / rlung | 能量流；意识的“马”。 | 维持生命活动与思维波动的驱动力。 | 气乱则心乱，气定则心定。 |
| **明点** | Bindu / thig le | 能量的精华；认知的载体。 | 意识最细微的种子；遗传与智慧的汇聚。 | 红色与白色明点的平衡决定了深层生理的和谐。 |
| **成就位** | Chakras | 脉的交汇中心。 | 能量与认知的交叉节点；特定的能量泵。 | 不同脉轮的开启对应不同的心理功能提升。 |

| 修持阶段 | 过程描述 (Process) | 心理学效应 | 目的 (Goal) | 备注 |
| :--- | :--- | :--- | :--- | :--- |

| **圆满次第** | Completion Stage | 融摄观想进入气脉，现证大乐空性。 | 触发大脑最深层的生化平衡：内啡肽、多巴胺的极致融合。 | 实现意识的自性解脱。 |
| **自生本尊** | Self-Generation | 观想自己即是本尊。 | 构建一种神圣、完整且无畏的自我认同。 | 对抗由自卑、创伤引起的自我否定。 |
| **对生本尊** | Front-Generation | 观想本尊在前方虚空降临加持。 | 建立超越性的依恋与外部导向的虔诚心。 | 适合建立基础信心与安全底座。 |
