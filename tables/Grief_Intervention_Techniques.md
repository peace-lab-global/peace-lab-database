# Grief Intervention Techniques (哀伤干预技术详表)

> **哀伤干预技术 (Grief Intervention Techniques / Technicae Interventionis Luctus)**
> 
> *"陪伴悲伤不是修复，而是见证、承载与促进转化的艺术。"*
> *"Accompanying grief is not about fixing—it is the art of witnessing, holding, and facilitating transformation."*
> 
> 本文档系统梳理哀伤干预的专业技术，涵盖循证心理治疗、叙事方法、仪式干预、团体支持及整合性方法，为临床工作者提供技术参考。

---

## 1. 核心术语与词源表 (Core Terminology & Etymology)

| 术语 (中文) | 英文名 (English Term) | 原始语言/词源 (Original Language) | 词源含义 | 精确定义 | 专业视角 (Professional Perspective) |
|---|---|---|---|---|---|
| **悲伤辅导** | Grief Counseling | 拉丁语: *Consilium* (建议) | 给予建议 | 帮助正常悲伤过程的支持性干预 | 适用于大多数丧亲者 |
| **悲伤治疗** | Grief Therapy | 希腊语: *Therapeia* (治疗) | 治愈的服务 | 针对复杂性/延长哀伤的专业心理治疗 | 适用于病理性悲伤 |
| **复杂性悲伤治疗** | Complicated Grief Treatment (CGT) | - | Katherine Shear开发的疗法 | 针对延长哀伤障碍的结构化循证治疗 | A级证据支持 |
| **叙事暴露** | Narrative Exposure | 拉丁语: *Narrare* (讲述) | 讲述故事的暴露 | 通过反复讲述丧失故事来整合记忆 | 创伤与悲伤整合 |
| **想象对话** | Imaginal Dialogue / Empty Chair | - | 格式塔技术应用 | 与逝者进行想象中的对话 | 处理未尽事宜 |
| **持续联结** | Continuing Bonds | - | 与逝者的持续内在关系 | 丧失后维持与逝者转化后的心理联系 | 现代悲伤理论核心 |
| **意义重建** | Meaning Reconstruction | 拉丁语: *Re-* (重新) + *Construere* (建造) | 重新建构意义 | 在丧失后重新建构生命意义和自我叙事 | Neimeyer建构主义 |
| **仪式化** | Ritualization | 拉丁语: *Ritualis* (仪式的) | 赋予仪式性 | 通过仪式行为标记和处理丧失 | 跨文化通用 |
| **悼念信** | Grief Letter / Memorial Letter | - | 写给逝者的信件 | 以书写形式表达对逝者的思念和未说的话 | 表达性写作技术 |
| **双过程振荡** | Dual Process Oscillation | 拉丁语: *Oscillare* (摇摆) | 两个过程间的摇摆 | 丧失导向与恢复导向之间的健康往返 | Stroebe & Schut理论 |
| **同伴支持** | Peer Support | 拉丁语: *Par* (同等的) | 同等者的支持 | 有相似经历的人提供的相互支持 | 丧亲支持小组基础 |

## 2. 悲伤干预的层级框架 (Hierarchical Framework of Grief Intervention)

### 表2.1 悲伤干预层级 (Levels of Grief Intervention)

| 层级 | 目标人群 | 干预目标 | 干预者 | 设置 |
|---|---|---|---|---|
| **一级：普遍性支持** | 所有丧亲者 | 正常化；信息；基本支持 | 社区、家人、非专业人员 | 自然环境 |
| **二级：支持性辅导** | 需要额外支持的丧亲者 | 促进健康悲伤过程 | 受训志愿者、悲伤辅导员 | 社区机构、支持小组 |
| **三级：悲伤咨询** | 高风险或困难悲伤 | 预防复杂化；处理困难 | 心理咨询师、社工 | 专业咨询机构 |
| **四级：悲伤治疗** | 复杂性/延长哀伤障碍 | 治疗病理性悲伤 | 临床心理师、精神科医师 | 心理治疗机构 |

### 表2.2 干预指征评估 (Assessment for Intervention Level)

| 评估维度 | 低强度干预指征 | 高强度干预指征 |
|---|---|---|
| **症状严重度** | 正常范围悲伤反应 | 符合PGD/复杂性悲伤诊断 |
| **功能损害** | 轻微、暂时 | 显著、持续 |
| **时间进程** | 逐渐改善 | 无改善或恶化 |
| **风险因素** | 较少 | 多重风险因素 |
| **社会支持** | 充足 | 缺乏 |
| **既往史** | 无精神障碍史 | 有精神障碍或创伤史 |
| **丧失类型** | 自然、预期 | 创伤性、突然、自杀 |
| **自杀风险** | 无 | 有任何自杀意念 |

## 3. 复杂性悲伤治疗 (Complicated Grief Treatment, CGT)

### 表3.1 CGT概述 (CGT Overview)

| 项目 | 内容 |
|---|---|
| **开发者** | Katherine Shear, MD (Columbia University) |
| **理论基础** | 依恋理论 + 双过程模型 + 创伤理论 |
| **目标人群** | 延长哀伤障碍(PGD) |
| **疗程** | 16次，每次60-90分钟 |
| **证据等级** | A级（多项RCT支持） |
| **主要成分** | 丧失聚焦 + 恢复聚焦技术 |

### 表3.2 CGT治疗阶段与技术 (CGT Phases and Techniques)

| 阶段 | 会谈数 | 目标 | 核心技术 |
|---|---|---|---|
| **导入阶段** | 1-3 | 建立联盟；心理教育；设定目标 | 悲伤监测日记；个人目标设定；心理教育 |
| **核心阶段** | 4-9 | 处理丧失和恢复的阻碍 | 想象重述；想象对话；情境暴露；活动计划 |
| **巩固阶段** | 10-16 | 巩固收获；预防复发 | 回顾进步；未来规划；毕业仪式 |

### 表3.3 CGT核心技术详解 (CGT Core Techniques)

| 技术 | 目的 | 操作要点 | 频率 |
|---|---|---|---|
| **悲伤监测** | 追踪悲伤强度和触发因素 | 每日0-10评分；记录最高和最低时刻 | 每日(整个治疗) |
| **想象重述** | 整合丧失记忆；减少回避 | 闭眼详细叙述死亡相关经历；录音并在会谈间倾听 | 会谈4-9 |
| **想象对话** | 表达未说的话；维持联结 | 闭眼想象与逝者对话；说出想说的话 | 会谈5-7 |
| **情境暴露** | 减少回避行为 | 列出回避的情境/活动；逐步面对 | 会谈5-9 |
| **个人目标** | 重新投入生活 | 设定3个有意义的个人目标；制定行动计划 | 会谈1起 |
| **积极记忆** | 平衡痛苦与积极记忆 | 有意回忆与逝者的美好时光 | 穿插进行 |
| **图片技术** | 促进持续联结 | 携带逝者照片；向治疗师介绍逝者 | 早期会谈 |

### 表3.4 CGT想象重述协议 (Imaginal Revisiting Protocol)

| 步骤 | 操作 | 注意事项 |
|---|---|---|
| **1. 准备** | 解释原理；建立安全感 | 确保来访者准备好 |
| **2. 闭眼** | 请来访者闭上眼睛 | 创造内在聚焦 |
| **3. 引导** | "回到那个时候...告诉我发生了什么" | 使用现在时 |
| **4. 叙述** | 来访者详细叙述死亡相关经历 | 鼓励细节；不打断情感表达 |
| **5. 处理** | 处理叙述中出现的情感 | 验证；正常化 |
| **6. 录音** | 录制整个过程 | 用于家庭作业 |
| **7. 回听** | 要求每天听录音至少一次 | 通过重复减少痛苦 |
| **8. 讨论** | 下次会谈讨论回听体验 | 追踪变化 |

## 4. 意义重建方法 (Meaning Reconstruction Approaches)

### 表4.1 Neimeyer意义重建框架 (Neimeyer's Meaning Reconstruction)

| 概念 | 定义 | 临床意义 |
|---|---|---|
| **意义的中断** | 丧失打破了自我叙事的连续性 | 需要重新建构 |
| **感知利益** | 在丧失中发现积极改变 | 创伤后成长 |
| **身份重构** | 在丧失后重新定义自我 | 核心治疗任务 |
| **持续联结** | 与逝者维持转化后的关系 | 健康悲伤成分 |
| **叙事整合** | 将丧失整合进人生故事 | 治疗目标 |

### 表4.2 意义重建叙事技术 (Narrative Techniques for Meaning Reconstruction)

| 技术 | 操作 | 目的 | 使用时机 |
|---|---|---|---|
| **关系传记** | 写下与逝者关系的完整故事 | 整合关系；处理矛盾 | 早中期 |
| **章节隐喻** | 将生命视为有不同章节的书 | 看到丧失是一个章节而非结局 | 中后期 |
| **写信给逝者** | 写信表达想法和情感 | 表达未说的话；维持联结 | 任何阶段 |
| **逝者回信** | 从逝者视角写一封回信 | 获得新视角；自我安慰 | 中后期 |
| **人生回顾** | 回顾整个人生叙事 | 看到丧失在更大叙事中的位置 | 后期 |
| **遗产问题** | "逝者希望你如何继续生活？" | 从逝者获得许可 | 中后期 |
| **双重叙事** | 同时讲述丧失故事和生存故事 | 整合丧失与恢复 | 中后期 |
| **仪式创造** | 设计有意义的纪念仪式 | 标记；表达；转化 | 任何阶段 |

### 表4.3 意义寻求的治疗性问题 (Therapeutic Questions for Meaning-Making)

| 问题类型 | 示例问题 | 目的 |
|---|---|---|
| **关系意义** | "这段关系教会了你什么？" | 提取关系遗产 |
| **改变意义** | "这个丧失如何改变了你？" | 探索成长 |
| **价值意义** | "这让你更清楚什么对你重要吗？" | 澄清价值 |
| **继续意义** | "你认为逝者希望你如何生活？" | 获得内在许可 |
| **遗产意义** | "你想如何纪念他/她？" | 建立持续联结 |
| **世界观意义** | "这改变了你对生命的看法吗？" | 世界观重建 |
| **身份意义** | "现在的你和以前有什么不同？" | 身份整合 |

## 5. 持续联结工作 (Continuing Bonds Work)

### 表5.1 持续联结的维度 (Dimensions of Continuing Bonds)

| 维度 | 内容 | 适应性表现 | 不适应性表现 |
|---|---|---|---|
| **内在联结** | 内心中与逝者的关系 | 温暖的内在对话 | 强迫性反刍 |
| **外在联结** | 外在的纪念行为 | 有意义的纪念物 | 过度保留/回避遗物 |
| **社会联结** | 通过他人维持联结 | 分享逝者故事 | 不允许他人提及 |
| **灵性联结** | 灵性或超自然层面的联结 | 感到逝者的精神存在 | 强迫性寻求"迹象" |
| **遗产联结** | 通过继承遗产维持联结 | 继续逝者的事业或价值 | 被迫承担逝者角色 |

### 表5.2 促进健康持续联结的技术 (Techniques for Healthy Continuing Bonds)

| 技术 | 操作 | 目的 |
|---|---|---|
| **介绍逝者** | 带照片向治疗师介绍逝者 | 建立联结；表达爱 |
| **逝者访谈** | 详细询问逝者生平、性格、关系 | 丰富逝者形象 |
| **最好记忆** | 邀请分享最美好的记忆 | 平衡痛苦 |
| **礼物问题** | "如果逝者可以给你一个礼物，会是什么？" | 内化逝者智慧 |
| **信息获取** | "如果逝者在这里，他/她会说什么？" | 获得内在指导 |
| **遗产整合** | 探索如何将逝者的价值或特质融入自己生活 | 建立遗产联结 |
| **纪念仪式** | 设计个人化的纪念方式 | 外化联结 |
| **重新定位** | 帮助理解"不是放下，是重新定位关系" | 认知重构 |

### 表5.3 持续联结vs.病理性联结 (Healthy vs. Pathological Bonds)

| 维度 | 健康持续联结 | 病理性联结 |
|---|---|---|
| **功能** | 带来安慰；支持功能 | 干扰功能；阻碍生活 |
| **灵活性** | 可以选择何时联结 | 强迫性、不可控 |
| **现实接受** | 接受死亡现实 | 否认死亡现实 |
| **情感** | 悲伤中带有温暖 | 只有痛苦或渴望 |
| **生活投入** | 可以投入新生活 | 无法投入任何新事物 |
| **关系** | 可以有新关系 | 无法与他人建立关系 |
| **时间** | 随时间逐渐转化 | 持续僵化 |

## 6. 仪式干预 (Ritual Interventions)

### 表6.1 仪式在悲伤中的功能 (Functions of Ritual in Grief)

| 功能 | 内容 | 心理机制 |
|---|---|---|
| **标记** | 标记丧失和过渡 | 给予丧失具体形式 |
| **表达** | 提供表达渠道 | 合法化的情感出口 |
| **共同体** | 创造共同经历 | 社会支持；见证 |
| **意义** | 赋予丧失意义 | 文化和灵性框架 |
| **过渡** | 帮助过渡到新状态 | 身份转换 |
| **持续联结** | 维持与逝者联系 | 仪式化的联结 |
| **控制感** | 在无控感中提供一些掌控 | 主动性行为 |

### 表6.2 治疗性仪式类型 (Types of Therapeutic Rituals)

| 仪式类型 | 目的 | 示例 | 适用情况 |
|---|---|---|---|
| **告别仪式** | 正式道别 | 写信烧掉；墓前对话 | 无法正式告别时 |
| **纪念仪式** | 纪念逝者 | 点蜡烛；周年祭 | 持续联结 |
| **释放仪式** | 释放负面情感 | 写下内疚并仪式化释放 | 被困的情感 |
| **转化仪式** | 标记过渡 | 脱下婚戒；改变房间 | 身份转换 |
| **遗物仪式** | 处理遗物 | 有意义地整理和分配 | 遗物处理困难 |
| **命名仪式** | 为丧失命名 | 为流产婴儿命名 | 未被承认的丧失 |
| **和解仪式** | 处理关系中的问题 | 写原谅信 | 矛盾关系 |

### 表6.3 设计治疗性仪式的原则 (Principles for Designing Therapeutic Rituals)

| 原则 | 操作要点 | 注意事项 |
|---|---|---|
| **个人意义** | 仪式应对来访者有个人意义 | 不强加治疗师的想法 |
| **文化契合** | 与来访者文化和信仰契合 | 了解文化背景 |
| **具体行动** | 包含具体可见的行动 | 不只是想想 |
| **感官参与** | 尽可能涉及多种感官 | 视觉、听觉、触觉等 |
| **时间选择** | 选择有意义的时间 | 纪念日或特殊时刻 |
| **空间选择** | 选择有意义的地点 | 或创造特殊空间 |
| **见证** | 考虑是否需要见证者 | 治疗师或重要他人 |
| **重复可能** | 可以重复或定期进行 | 或是一次性 |

### 表6.4 周年纪念日干预 (Anniversary Intervention)

| 阶段 | 干预内容 | 目的 |
|---|---|---|
| **预期** | 讨论即将到来的纪念日 | 准备；正常化 |
| **计划** | 计划如何度过这一天 | 增加控制感 |
| **选择** | 讨论不同选择(纪念/正常/混合) | 尊重自主 |
| **执行** | 按计划进行 | 主动应对 |
| **反思** | 事后讨论体验 | 整合经验 |

## 7. 团体支持与同伴支持 (Group and Peer Support)

### 表7.1 悲伤支持小组类型 (Types of Grief Support Groups)

| 类型 | 特点 | 适用人群 | 带领者 |
|---|---|---|---|
| **同质性支持小组** | 相同类型丧失 | 如丧子父母、丧偶者 | 志愿者或专业人员 |
| **异质性支持小组** | 不同类型丧失 | 一般丧亲者 | 志愿者或专业人员 |
| **教育性小组** | 心理教育为主 | 需要信息的丧亲者 | 专业人员 |
| **治疗性小组** | 治疗为目标 | 复杂性悲伤 | 专业人员 |
| **在线支持小组** | 网络进行 | 无法参加面对面的人 | 多种 |
| **特定人群小组** | 针对特定群体 | 如自杀丧亲者、围产期丧失 | 专业人员或受训志愿者 |

### 表7.2 悲伤支持小组的治疗因子 (Therapeutic Factors in Grief Groups)

| 治疗因子 | Yalom概念 | 在悲伤小组的体现 |
|---|---|---|
| **普遍性** | Universality | "原来不只有我这样" |
| **希望灌注** | Instillation of Hope | 看到更久的丧亲者好转 |
| **信息传递** | Imparting Information | 了解悲伤知识 |
| **利他主义** | Altruism | 帮助他人悲伤时感到有价值 |
| **社交学习** | Development of Socializing Techniques | 学习如何谈论丧失 |
| **行为模仿** | Imitative Behavior | 看到他人如何应对 |
| **人际学习** | Interpersonal Learning | 获得反馈和新视角 |
| **团体凝聚** | Group Cohesiveness | 归属感；被接纳 |
| **宣泄** | Catharsis | 在安全环境表达情感 |
| **存在因素** | Existential Factors | 共同面对生命终极问题 |

### 表7.3 悲伤小组带领技术 (Group Facilitation Techniques)

| 技术 | 操作 | 目的 |
|---|---|---|
| **开场轮** | 每人简短分享当前状态 | 连接；了解情况 |
| **命名逝者** | 鼓励说出逝者名字 | 承认；持续联结 |
| **蜡烛仪式** | 点蜡烛纪念逝者 | 仪式化；共同悼念 |
| **主题讨论** | 围绕特定主题讨论 | 结构化；深入探索 |
| **故事分享** | 分享与逝者的故事 | 维持记忆；被见证 |
| **资源分享** | 分享有用的资源和策略 | 相互帮助；实用信息 |
| **结束仪式** | 以仪式结束每次聚会 | 容器化；过渡 |

### 表7.4 同伴支持志愿者培训要点 (Peer Support Volunteer Training)

| 培训模块 | 内容 | 时长(建议) |
|---|---|---|
| **悲伤基础** | 悲伤理论、正常反应、任务模型 | 4小时 |
| **倾听技能** | 积极倾听、共情回应、不给建议 | 4小时 |
| **边界** | 角色边界、自我照顾、转介时机 | 2小时 |
| **特殊议题** | 自杀丧亲、创伤性丧失 | 2小时 |
| **实践演练** | 角色扮演、案例讨论 | 4小时 |
| **持续督导** | 定期督导会议 | 持续 |

## 8. 儿童悲伤干预技术 (Child Grief Intervention Techniques)

### 表8.1 儿童悲伤治疗方法 (Child Grief Treatment Approaches)

| 方法 | 描述 | 适用年龄 | 证据基础 |
|---|---|---|---|
| **游戏治疗** | 通过游戏表达和处理悲伤 | 3-12岁 | 强 |
| **艺术治疗** | 通过绘画等艺术表达悲伤 | 4岁+ | 中等 |
| **沙盘治疗** | 使用沙盘和小物件表达 | 3-12岁 | 中等 |
| **故事治疗** | 使用治疗性故事 | 4-12岁 | 中等 |
| **认知行为** | 适应儿童的CBT技术 | 7岁+ | 强 |
| **家庭治疗** | 整个家庭参与 | 所有年龄 | 强 |

### 表8.2 游戏治疗在悲伤中的应用 (Play Therapy in Grief)

| 游戏类型 | 功能 | 示例活动 |
|---|---|---|
| **角色扮演** | 表达和处理情感 | 玩偶演出；医生游戏 |
| **象征性游戏** | 间接表达 | 用动物代表家庭成员 |
| **创造性游戏** | 表达和控制感 | 建造和破坏；黏土 |
| **故事游戏** | 叙事和意义 | 创作关于丧失的故事 |
| **记忆游戏** | 维持联结 | 制作记忆盒、相册 |
| **仪式游戏** | 标记和处理 | 模拟葬礼 |

### 表8.3 儿童悲伤常用活动 (Common Activities for Grieving Children)

| 活动 | 目的 | 年龄范围 | 材料 |
|---|---|---|---|
| **记忆盒** | 保存与逝者相关的物品 | 4岁+ | 盒子、装饰材料 |
| **记忆书** | 记录关于逝者的记忆 | 5岁+ | 本子、绘画材料 |
| **画出感觉** | 表达复杂情感 | 4岁+ | 绘画材料 |
| **身体地图** | 定位身体中的悲伤感觉 | 5岁+ | 身体轮廓、彩笔 |
| **担忧盒** | 存放担忧 | 5岁+ | 盒子、纸条 |
| **写信给逝者** | 表达未说的话 | 7岁+ | 纸、笔 |
| **种植仪式** | 象征生命和纪念 | 3岁+ | 种子、土壤、盆 |
| **天堂想象** | 探索对死后的想象 | 5岁+ | 绘画材料 |

### 表8.4 与丧亲儿童工作的指导原则 (Guidelines for Working with Bereaved Children)

| 原则 | 具体做法 | 避免事项 |
|---|---|---|
| **跟随儿童** | 让儿童引导节奏和主题 | 强迫谈论或活动 |
| **使用儿童语言** | 用简单、具体的词汇 | 抽象或成人化表达 |
| **允许游戏** | 理解游戏是儿童的语言 | 强迫语言表达 |
| **正常化** | 正常化各种反应 | 评判或纠正感受 |
| **创造安全** | 建立安全、可预测的环境 | 不稳定或不可预测 |
| **包括家长** | 与家长合作 | 忽视家庭系统 |
| **灵活性** | 适应儿童发展水平 | 一刀切方法 |
| **耐心** | 允许重复和时间 | 急于"解决" |

## 9. 正念与身体为本的悲伤干预 (Mindfulness and Body-Based Grief Interventions)

### 表9.1 正念在悲伤干预中的应用 (Mindfulness in Grief Intervention)

| 应用方式 | 目的 | 具体技术 |
|---|---|---|
| **临在** | 帮助处在当下而非反刍 | 呼吸觉察；身体扫描 |
| **接纳** | 允许悲伤情感存在 | 不评判的观察 |
| **情绪调节** | 在强烈情绪中稳定 | RAIN技术；呼吸锚定 |
| **自我慈悲** | 减少自责和自我批评 | 自我慈悲练习 |
| **认知去融合** | 与痛苦想法保持距离 | 思维观察 |
| **持续联结** | 正念地联结逝者 | 慈悲冥想中包括逝者 |

### 表9.2 悲伤中的RAIN技术 (RAIN for Grief)

| 步骤 | 操作 | 悲伤中的应用 |
|---|---|---|
| **R - 认知** | Recognize what is happening | "我注意到我正在感到深深的悲伤" |
| **A - 允许** | Allow the experience to be there | "让这个悲伤存在是可以的" |
| **I - 探究** | Investigate with interest and care | "这个悲伤在身体哪里？感觉像什么？" |
| **N - 不认同** | Non-identification / Nurture | "这是悲伤，但我不等于悲伤"；给予自己温柔 |

### 表9.3 躯体聚焦的悲伤干预 (Somatic Approaches to Grief)

| 方法 | 原理 | 技术示例 |
|---|---|---|
| **躯体体验** | 悲伤存储在身体 | 追踪身体感觉；允许释放 |
| **呼吸工作** | 通过呼吸调节神经系统 | 深呼吸；叹气释放 |
| **渐进放松** | 释放身体紧张 | 系统放松肌肉群 |
| **运动** | 释放；调节神经系统 | 行走；瑜伽 |
| **触碰** | 安抚神经系统 | 自我拥抱；触碰心口 |
| **姿势工作** | 身体姿势影响情绪 | 打开姿势；站立 |

### 表9.4 创伤知情的悲伤干预原则 (Trauma-Informed Grief Intervention)

| 原则 | 操作 | 理由 |
|---|---|---|
| **安全第一** | 建立安全感优先于探索 | 创伤性丧失损害安全感 |
| **选择权** | 给予来访者选择 | 恢复控制感 |
| **节奏控制** | 来访者控制节奏 | 避免再创伤 |
| **资源建立** | 先建立内外部资源 | 为处理做准备 |
| **窗口工作** | 在容纳窗口内工作 | 避免过度激活或关闭 |
| **接地** | 教导和使用接地技术 | 调节激活状态 |
| **滴定** | 小剂量处理 | 可控的暴露 |
| **整合** | 创伤记忆与悲伤分开和整合处理 | 区分不同类型的痛苦 |

## 10. 药物在悲伤中的作用 (Role of Medication in Grief)

### 表10.1 药物干预的考量 (Considerations for Pharmacological Intervention)

| 情况 | 药物考量 | 注意事项 |
|---|---|---|
| **正常悲伤** | 一般不建议药物 | 药物可能干扰自然悲伤过程 |
| **严重睡眠障碍** | 短期睡眠辅助可考虑 | 非苯二氮卓类；短期使用 |
| **严重焦虑** | 谨慎考虑 | 苯二氮卓类有依赖风险 |
| **共病抑郁** | 抗抑郁药可能有帮助 | 鉴别悲伤vs.抑郁 |
| **延长哀伤障碍** | 研究中 | CGT是一线治疗 |
| **共病PTSD** | 按PTSD指南 | 整合治疗 |

### 表10.2 悲伤与抑郁的药物治疗鉴别 (Medication Considerations: Grief vs. Depression)

| 维度 | 悲伤为主 | 抑郁为主 | 药物决策 |
|---|---|---|---|
| **情绪特点** | 波动；与逝者相关 | 弥漫性低落 | 抑郁特征→考虑抗抑郁药 |
| **自尊** | 保持 | 低自尊 | 低自尊→考虑抗抑郁药 |
| **功能** | 逐渐恢复 | 持续损害 | 持续损害→考虑抗抑郁药 |
| **自杀** | 被动想法("想与TA团聚") | 主动自杀意念 | 主动意念→可能需要药物 |
| **睡眠** | 暂时困难 | 严重持续失眠 | 严重失眠→可考虑短期辅助 |
