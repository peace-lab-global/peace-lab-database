# 自生训练 (Autogenic Training, AT)

> **模块定位**：自生训练的系统化专业知识，涵盖Schultz六标准公式、Luthe高级技术、神经生理机制、临床应用协议及循证基础。
> **核心关键词**：自生训练 (Autogenes Training)、被动专注 (Passive Concentration)、自律神经调节 (Autonomic Self-Regulation)、自我暗示 (Autosuggestion)

---

## 一、自生训练核心术语表 (Core Terminology)

| 中文术语 | 英文术语 | 德语/原始语言 | 专业定义 |
|---------|---------|--------------|---------|
| 自生训练 | Autogenic Training (AT) | *Autogenes Training* | 希腊语 *autos* (自我) + *genos* (产生)；Johannes Schultz于1932年创立的自我调节放松技术 |
| 被动专注 | Passive Concentration | *Passive Konzentration* | AT的核心注意力态度：轻柔地关注身体感觉，不强迫、不努力达成特定结果 |
| 标准公式 | Standard Formulas | *Standardübungen* | AT基础训练的六个自我暗示语句，系统引导生理变化 |
| 高级公式 | Advanced Formulas | *Oberstufe* | Luthe发展的深层自我探索技术，涉及意象与潜意识工作 |
| 自律神经训练 | Autonomic Training | *Autonomes Training* | 通过自我暗示影响自主神经系统功能的训练 |
| 重感公式 | Heaviness Formula | *Schwereübung* | "我的手臂很重"—第一标准公式，诱导肌肉放松 |
| 温感公式 | Warmth Formula | *Wärmeübung* | "我的手臂很温暖"—第二标准公式，诱导血管舒张 |
| 心脏公式 | Heart Formula | *Herzübung* | "我的心跳平静有力"—第三标准公式，调节心脏功能 |
| 呼吸公式 | Breathing Formula | *Atemübung* | "我的呼吸平静自如"—第四标准公式，调节呼吸 |
| 腹部公式 | Abdominal Formula | *Bauchübung* / *Sonnengeflechtsübung* | "我的太阳神经丛温暖"—第五标准公式，调节内脏功能 |
| 前额公式 | Forehead Formula | *Stirnübung* | "我的前额清凉"—第六标准公式，调节头部血流 |
| 自生解除 | Autogenic Discharge | *Autogene Entladung* | 训练中可能出现的情绪或躯体释放现象 |
| 取消动作 | Cancellation | *Zurücknehmen* | 结束AT练习的标准程序：握拳、弯曲手臂、深呼吸、睁眼 |
| 器官特异公式 | Organ-Specific Formulas | *Organspezifische Formeln* | 针对特定器官或症状的个性化自我暗示语句 |

## 二、自生训练历史与理论发展 (Historical Development)

### 2.1 发展里程碑 (Key Milestones)

| 年份 | 事件 | 代表人物 | 核心贡献 |
|-----|------|---------|---------|
| **1890s** | 催眠研究基础 | Oskar Vogt | 研究"预防性休息-催眠"，发现受试者可自我诱导催眠状态 |
| **1905** | 柏林神经研究所 | Vogt | 提出"自我催眠"概念，Schultz为其学生 |
| **1920s** | 系统化发展 | Johannes H. Schultz | 整合催眠、瑜伽、生理学，发展标准化训练 |
| **1932** | 《自生训练》首版出版 | Schultz | *Das Autogene Training* 德文原著出版 |
| **1959** | 国际推广 | Wolfgang Luthe | 英文版 *Autogenic Training* 出版，引入北美 |
| **1969-1973** | 六卷本系统著作 | Luthe & Schultz | *Autogenic Therapy* 六卷本，涵盖理论、方法、临床应用 |
| **1970s** | 高级技术发展 | Luthe | *Autogenic Modification* 深化潜意识工作技术 |
| **1980s-至今** | 循证研究积累 | 多研究团队 | RCT证实对焦虑、高血压、偏头痛等的疗效 |

### 2.2 Schultz原始理论框架 (Schultz's Original Theory)

| 理论要点 | 描述 |
|---------|------|
| **催眠-瑜伽整合** | AT融合西方催眠与东方瑜伽/冥想传统 |
| **自我调节原则** | 身体具有自我调节能力，通过特定练习可增强这种能力 |
| **被动专注核心** | 与积极努力相反，放松通过"允许"而非"强迫"发生 |
| **生理-心理互动** | 心理暗示可引起真实的生理变化（如血管舒张、肌肉放松） |
| **系统渐进训练** | 按特定顺序学习六个标准公式，逐步深化 |
| **个性化适应** | 基于标准公式，可发展针对特定症状的个性化公式 |

## 三、六标准公式详解 (Six Standard Formulas)

### 3.1 标准公式总览 (Standard Formulas Overview)

| 序号 | 公式名称 | 德语原文 | 英文 | 中文标准语句 | 目标生理效应 |
|-----|---------|---------|------|-------------|-------------|
| 1 | 重感公式 | *Schwereübung* | Heaviness | "我的（右）手臂很重" | 肌肉放松 |
| 2 | 温感公式 | *Wärmeübung* | Warmth | "我的（右）手臂很温暖" | 血管舒张 |
| 3 | 心脏公式 | *Herzübung* | Heart | "我的心跳平静有力" | 心脏调节 |
| 4 | 呼吸公式 | *Atemübung* | Breathing | "我的呼吸平静自如" | 呼吸调节 |
| 5 | 腹部公式 | *Sonnengeflechtsübung* | Solar Plexus | "我的太阳神经丛温暖" | 内脏调节 |
| 6 | 前额公式 | *Stirnübung* | Forehead | "我的前额清凉舒适" | 头部血流调节 |
| — | 平静公式 | *Ruheübung* | Calm | "我非常平静" | 整体平静状态 |

### 3.2 各公式详细说明 (Detailed Formula Descriptions)

#### 公式1：重感公式 (Heaviness Formula)

| 维度 | 描述 |
|-----|------|
| **标准语句** | "我的右手臂很重"（*Mein rechter Arm ist schwer*） |
| **进阶语句** | "我的双臂很重" → "我的腿很重" → "我的四肢都很重" |
| **生理机制** | 肌肉张力降低→本体感觉改变→产生"重"的感觉 |
| **神经通路** | 减少肌梭传入→脊髓/脑干网状结构活动↓ |
| **训练时长** | 1-2周掌握单侧手臂，4-6周扩展到全身 |
| **常见体验** | 肢体沉重感、无力感、与床/椅子融为一体感 |

#### 公式2：温感公式 (Warmth Formula)

| 维度 | 描述 |
|-----|------|
| **标准语句** | "我的右手臂很温暖"（*Mein rechter Arm ist warm*） |
| **进阶语句** | "我的双臂很温暖" → "我的腿很温暖" → "我的四肢都很温暖" |
| **生理机制** | 外周血管舒张→血流增加→皮肤温度上升 |
| **神经通路** | 交感神经血管收缩活动↓→α肾上腺素能受体抑制 |
| **训练时长** | 2-3周掌握，在重感稳定后开始 |
| **常见体验** | 温暖感、脉动感、轻微刺痛感（血管舒张） |
| **客观测量** | 指尖温度可上升1-4°C |

#### 公式3：心脏公式 (Heart Formula)

| 维度 | 描述 |
|-----|------|
| **标准语句** | "我的心跳平静有力"（*Mein Herz schlägt ruhig und kräftig*） |
| **替代语句** | "我的心跳平静规律"（对心脏敏感者） |
| **生理机制** | 副交感神经张力增加→心率减缓、心率变异性改善 |
| **注意事项** | 心脏病史者需医师评估后方可练习 |
| **训练时长** | 2-3周，在温感稳定后开始 |
| **禁忌调整** | 对心脏过度关注/焦虑者可跳过此公式 |

#### 公式4：呼吸公式 (Breathing Formula)

| 维度 | 描述 |
|-----|------|
| **标准语句** | "我的呼吸平静自如"（*Es atmet mich* / *Mein Atem ist ruhig*） |
| **德语原意** | *Es atmet mich* 字面意为"它在呼吸我"，强调被动性 |
| **生理机制** | 膈肌放松、呼吸频率降低、潮气量增加 |
| **训练要点** | 不主动控制呼吸，而是观察并允许自然呼吸 |
| **训练时长** | 2周，在心脏公式后 |
| **常见体验** | 呼吸自然变深变慢、腹式呼吸自发出现 |

#### 公式5：腹部/太阳神经丛公式 (Solar Plexus Formula)

| 维度 | 描述 |
|-----|------|
| **标准语句** | "我的太阳神经丛温暖"（*Mein Sonnengeflecht ist strömend warm*） |
| **替代语句** | "我的腹部温暖" |
| **解剖定位** | 太阳神经丛 (腹腔神经丛) 位于上腹部剑突下方 |
| **生理机制** | 腹腔血管舒张→内脏血流增加→消化功能调节 |
| **训练时长** | 2-3周，在呼吸公式后 |
| **适用情境** | 功能性消化问题、肠易激综合征、慢性腹部紧张 |
| **注意事项** | 腹部手术史、腹部疾病者需谨慎 |

#### 公式6：前额公式 (Forehead Formula)

| 维度 | 描述 |
|-----|------|
| **标准语句** | "我的前额清凉舒适"（*Meine Stirn ist angenehm kühl*） |
| **生理机制** | 头部血管收缩→头部充血减少→清凉感 |
| **与其他公式对比** | 唯一追求"凉"而非"暖"的公式（防止头部充血） |
| **训练时长** | 2周，作为最后一个标准公式 |
| **适用情境** | 偏头痛预防、紧张性头痛、头部发热感 |
| **注意事项** | 低血压者可能感到头晕，需减少练习强度 |

### 3.3 标准训练进度表 (Standard Training Schedule)

| 周次 | 学习公式 | 每日练习 | 单次时长 | 目标 |
|-----|---------|---------|---------|------|
| **第1-2周** | 重感（单侧手臂） | 3次/日 | 3-5分钟 | 稳定感受手臂沉重 |
| **第3-4周** | 重感（扩展） + 温感（手臂） | 3次/日 | 5-8分钟 | 重感泛化，开始温感 |
| **第5-6周** | 重感+温感（四肢） | 2-3次/日 | 8-10分钟 | 四肢重+温稳定 |
| **第7-8周** | +心脏公式 | 2-3次/日 | 10-12分钟 | 心脏调节 |
| **第9-10周** | +呼吸公式 | 2次/日 | 12-15分钟 | 呼吸调节 |
| **第11-12周** | +腹部公式 | 2次/日 | 15-18分钟 | 内脏调节 |
| **第13-14周** | +前额公式 | 2次/日 | 18-20分钟 | 完整六公式 |
| **第15周+** | 完整练习 + 简化版 | 1-2次/日 | 15-20分钟 | 技能巩固与迁移 |

## 五、自生训练神经生理机制 (Neurophysiological Mechanisms)

### 5.1 自主神经系统调节 (Autonomic Regulation)

| 公式 | 主要神经通路 | 生理效应 | 测量指标 |
|-----|-------------|---------|---------|
| **重感** | 肌梭传入↓→脊髓→脑干 | 骨骼肌张力↓ | EMG活动↓ |
| **温感** | 交感血管收缩神经↓ | 外周血管舒张 | 指尖温度↑ |
| **心脏** | 迷走神经张力↑ | 心率↓, HRV↑ | 心率、RMSSD |
| **呼吸** | 呼吸中枢调节 | 呼吸频率↓, 潮气量↑ | 呼吸率 |
| **腹部** | 内脏副交感↑ | 内脏血流↑, 蠕动调节 | 肠鸣音 |
| **前额** | 头部血管收缩 | 头部血流↓ | 颞动脉脉动↓ |

### 5.2 脑电与脑区变化 (EEG & Brain Changes)

| 指标 | AT后变化 | 功能意义 |
|-----|----------|---------|
| **α波功率** | ↑ (8-12 Hz) | 放松警觉状态 |
| **θ波功率** | ↑ (4-8 Hz) | 深度放松、内省状态 |
| **β波功率** | ↓ (>13 Hz) | 焦虑/过度警觉减少 |
| **α波同步性** | ↑ | 大脑整合状态 |
| **前额不对称** | 左侧相对↑ | 趋近动机、积极情绪 |

### 5.3 下丘脑-垂体-肾上腺轴 (HPA Axis)

| 指标 | AT后变化 | 临床意义 |
|-----|---------|---------|
| **皮质醇** | ↓ (长期练习) | 慢性应激缓解 |
| **ACTH** | ↓ | HPA轴下调 |
| **皮质醇觉醒反应** | 正常化 | 应激反应性改善 |

## 六、自生训练临床应用循证 (Clinical Applications Evidence)

### 6.1 按障碍类型的证据等级 (Evidence by Disorder)

| 障碍/状况 | 证据等级 | Meta分析效应量 | 核心研究 | 临床建议 |
|----------|---------|---------------|---------|---------|
| **高血压** | A级 | SBP↓ 8-10 mmHg | Linden, 1994 | AHA辅助推荐，长期练习 |
| **偏头痛** | A级 | 频率↓ 30-50% | Kaushik, 2005 | 预防性效果优于急性期 |
| **紧张性头痛** | A级 | d=0.55 | Stetter & Kupper, 2002 | 与PMR疗效相当 |
| **雷诺病** | A级 | 症状改善60-80% | Freedman, 1991 | 温感公式重点 |
| **广泛性焦虑** | A级 | d=0.50 | Stetter & Kupper, 2002 | 与其他放松技术相当 |
| **失眠** | B级 | d=0.45 | Morin, 2006 | 睡前练习有效 |
| **肠易激综合征** | B级 | 症状改善40-60% | Shinozaki, 2010 | 腹部公式重点 |
| **哮喘** | B级 | 改善生活质量 | Huntley, 2002 | 辅助常规治疗 |
| **冠心病辅助** | B级 | 改善生活质量 | van Dixhoorn, 2005 | 心脏康复整合 |
| **癌症支持** | B级 | 减少焦虑/疼痛 | Luebbert, 2001 | 化疗/围手术期支持 |
| **特应性皮炎** | B级 | 瘙痒减少 | Ehlers, 1995 | 配合皮肤护理 |

*证据等级：A级=多项高质量RCT/Meta分析支持；B级=有限RCT支持*

### 6.2 与药物治疗对比 (Comparison with Pharmacotherapy)

| 适应症 | AT效应 | 药物效应 | 优势对比 |
|-------|-------|---------|---------|
| **高血压** | SBP↓ 8-10 mmHg | 降压药更显著 | AT无副作用，可辅助减药 |
| **偏头痛预防** | 与β-blocker相当 | 托吡酯等药物 | AT无药物副作用 |
| **焦虑** | d=0.50 | 苯二氮卓类更快速 | AT无依赖风险 |
| **失眠** | 与安眠药短期相当 | 安眠药更快速 | AT效果更持久 |

### 6.3 特殊人群应用 (Special Populations)

| 人群 | 调整建议 | 注意事项 |
|-----|---------|---------|
| **儿童 (8-12岁)** | 简化语句、缩短时间、增加趣味性 | 需要父母参与、可视化引导 |
| **青少年** | 强调自主性、可配合音频 | 尊重隐私、避免强制 |
| **老年人** | 减慢进度、简化公式数量 | 关注听力/认知能力、避免久坐 |
| **孕妇** | 避免腹部公式过度关注、侧卧位 | 分娩准备整合 |
| **心脏病患者** | 心脏公式需谨慎或跳过 | 医师评估后进行 |
| **精神疾病患者** | 避免高级阶段、需专业督导 | 急性期禁用 |

## 七、自生训练禁忌症与风险 (Contraindications & Risks)

### 7.1 禁忌症表 (Contraindications)

| 类别 | 具体情况 | 涉及公式 | 风险说明 |
|-----|---------|---------|---------|
| **绝对禁忌** | 急性精神病发作 | 所有 | 可能加重症状 |
| | 严重解离状态 | 高级阶段 | 可能诱发解离 |
| | 急性心肌梗死期 | 心脏公式 | 需医疗稳定后 |
| **相对禁忌** | 未控制的糖尿病 | 温感公式 | 血糖波动风险 |
| | 严重低血压 | 所有深度放松 | 晕厥风险 |
| | 未控制的高血压 | 腹部/头部公式 | 需先药物控制 |
| | 癫痫 | 高级阶段 | 需医师指导 |
| | 解离性障碍 | 高级阶段 | 需专业督导 |
| | 严重抑郁 | 单独使用 | 需综合治疗 |

### 7.2 自生解除现象 (Autogenic Discharges)

| 类型 | 表现 | 处理方法 |
|-----|------|---------|
| **运动性解除** | 肌肉抽动、不自主运动 | 正常化、继续练习、自然消退 |
| **感觉性解除** | 麻木、刺痛、疼痛 | 正常化、调整公式强度 |
| **情绪性解除** | 突然哭泣、愤怒、焦虑 | 允许表达、必要时暂停、后续处理 |
| **意象性解除** | 自发意象、记忆浮现 | 观察不评判、记录、专业讨论 |
| **前庭性解除** | 头晕、旋转感 | 减少前额公式强度、渐进适应 |

### 7.3 不良反应监测 (Adverse Event Monitoring)

| 不良反应 | 可能原因 | 处理策略 |
|---------|---------|---------|
| **头痛/头晕** | 前额公式过强、低血压 | 减少强度、起身缓慢 |
| **心悸** | 心脏公式过度关注 | 暂停心脏公式、正常化 |
| **恶心** | 腹部公式不适 | 减少腹部关注、分散注意 |
| **焦虑增加** | 放松诱发焦虑 | 缩短时间、睁眼练习 |
| **睡眠障碍** | 傍晚练习过度放松 | 调整练习时间 |
| **情绪波动** | 情绪解除 | 专业支持、处理潜在议题 |

## 八、自生训练实施协议 (Implementation Protocol)

### 8.1 标准会谈结构 (Standard Session Structure)

| 阶段 | 时长 | 内容 |
|-----|------|------|
| **准备阶段** | 2-3分钟 | 调整姿势、闭眼、几次深呼吸、"我非常平静" |
| **主体练习** | 15-20分钟 | 按序进行各公式，每个公式重复4-6次 |
| **过渡** | 各公式间几秒 | "我继续平静…" |
| **取消动作** | 1-2分钟 | 握拳→弯曲手臂→深呼吸→"我清醒警觉"→睁眼 |
| **整合** | 几分钟 | 记录体验、讨论（如有治疗师） |

### 8.2 姿势选择 (Posture Options)

| 姿势 | 描述 | 适用情境 |
|-----|------|---------|
| **仰卧位** | 平躺，头下垫小枕，双腿略分开，手臂置于体侧 | 最放松，初学者/睡前 |
| **躺椅位** | 半躺于支撑躺椅，头颈腰有支撑 | 治疗室标准 |
| **坐姿（高背椅）** | 坐于高背椅，头靠椅背，双臂放扶手 | 办公室环境 |
| **马车夫姿势** | 坐于无靠背椅，前倾，前臂放大腿，头低垂 | 任何椅子可用 |

### 8.3 标准指导语示例 (Sample Script)

```
【引入】
请调整到舒适的姿势…闭上眼睛…
做几次深呼吸，让呼吸自然变慢变深…

【平静公式】
现在，在心中默念：
"我非常平静…"

【重感公式】
将注意力轻轻放在你的右手臂上…
在心中默念：
"我的右手臂很重…"
感受那种沉重的感觉…
"我的右手臂很重…"
不需要努力，只是允许这种感觉发生…

（重复4-6次，然后扩展到左臂、双腿）

【温感公式】
现在，继续关注你的右手臂…
在心中默念：
"我的右手臂很温暖…"
感受温暖的感觉在手臂中蔓延…
"我的右手臂很温暖…"

（继续其他公式…）

【取消动作】
现在，准备结束练习…
握紧你的双拳…
弯曲你的手臂…
深深吸一口气…
在心中说"我清醒警觉"…
慢慢睁开眼睛…
```

## 九、AT与其他技术对比 (Comparison with Other Techniques)

### 9.1 AT vs PMR

| 维度 | 自生训练 (AT) | 渐进性肌肉放松 (PMR) |
|-----|--------------|---------------------|
| **创始人** | Schultz (1932) | Jacobson (1929) |
| **作用机制** | 被动自我暗示→生理变化 | 主动肌肉张力→释放 |
| **核心操作** | 默念公式，被动专注 | 收紧-释放肌肉 |
| **学习难度** | 中-高 | 低-中 |
| **掌握时间** | 3-6个月 | 8-10周 |
| **适合人群** | 认知能力较好、愿意长期投入 | 广泛适用、初学者友好 |
| **器官特异性** | 高（可针对特定器官） | 低（主要针对骨骼肌） |
| **文化背景** | 德国/欧洲为主 | 美国为主 |

### 9.2 AT vs 催眠

| 维度 | 自生训练 | 催眠 |
|-----|---------|------|
| **自主性** | 自我引导为主 | 治疗师引导为主 |
| **依赖性** | 独立自我练习 | 需治疗师在场（传统） |
| **标准化** | 高度标准化公式 | 灵活个性化 |
| **意识状态** | 保持清醒觉察 | 可能涉及更深催眠状态 |
| **可及性** | 自学可行 | 通常需专业指导 |

### 9.3 AT vs 冥想

| 维度 | 自生训练 | 正念冥想 |
|-----|---------|---------|
| **目标** | 生理放松反应 | 觉察与接纳 |
| **操作** | 自我暗示公式 | 观察当下体验 |
| **评判态度** | 追求特定感觉（如重、温） | 非评判性观察 |
| **理论框架** | 医学/生理学 | 佛教心理学 |
| **适用范围** | 心身症状 | 广泛心理健康 |

## 十、自助实践指南 (Self-Practice Guide)

### 10.1 初学者建议 (Beginner Tips)

| 建议 | 说明 |
|-----|------|
| **从重感开始** | 不要一开始就学全部公式，花2周掌握重感 |
| **规律练习** | 每日2-3次，每次5-15分钟，比偶尔长练习更有效 |
| **不要强迫** | 如果某种感觉没出现，不要焦虑，继续练习 |
| **选择安静时段** | 避免干扰，创造仪式感 |
| **记录体验** | 用日记记录每次练习的感受，追踪进步 |
| **耐心** | AT需要数月掌握，不要期望立即效果 |

### 10.2 常见问题解答 (FAQ)

| 问题 | 回答 |
|-----|------|
| **"我感觉不到重/温怎么办？"** | 正常。继续练习，感觉会逐渐出现。不要强迫。 |
| **"练习时睡着了"** | 如果不是睡前练习，换成坐姿或调整时间。 |
| **"我总是走神"** | 正常。轻柔地将注意力带回公式，不要自责。 |
| **"我感到不舒服的情绪"** | 可能是自生解除。允许它存在，如果太强烈可暂停。 |
| **"需要多久才能掌握？"** | 通常3-6个月可掌握基本六公式。这是正常进度。 |
| **"可以自学还是需要老师？"** | 基础训练可自学。高级阶段或有复杂症状建议专业指导。 |

### 10.3 数字资源 (Digital Resources)

| 资源类型 | 推荐 | 特点 |
|---------|------|------|
| **音频引导** | Insight Timer (AT tracks) | 免费、多语言 |
| **APP** | Autogenic Training Pro | 系统化课程 |
