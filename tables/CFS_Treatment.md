# 慢性疲劳综合征治疗方法 (CFS Treatment Methods)

> ME/CFS目前尚无根治方法，治疗目标是症状管理、功能优化和预防恶化。活动管理(Pacing)是治疗基础，辅以针对性症状治疗、心理支持和生活方式调整。本文档基于最新NICE 2021指南和循证证据，提供全面的治疗框架。

---

### 2.2 NICE 2021核心建议

| 原则 | 具体内容 |
|------|---------|
| **以患者为中心** | 尊重患者体验和偏好，共同决策 |
| **个体化** | 根据严重度、症状特点制定方案 |
| **不造成伤害** | GET可能有害，应避免 |
| **整体管理** | 关注所有症状和生活质量 |
| **支持性** | 提供长期支持和随访 |
| **灵活调整** | 根据反应调整计划 |

### 2.3 GET争议与NICE立场

| 方面 | 旧立场 | NICE 2021新立场 |
|------|--------|----------------|
| GET推荐 | 推荐作为核心治疗 | **不再推荐**，可能有害 |
| 证据基础 | PACE试验支持 | 重新分析显示效果有限，存在方法学问题 |
| 患者反馈 | 被忽视 | 大量患者报告GET后恶化 |
| 替代方案 | — | 推荐活动管理/Pacing |

> **"任何声称可通过固定增量运动或活动来治疗或治愈ME/CFS的项目，或基于去适应化和避免活动导致ME/CFS的假设的项目，都不应该提供给ME/CFS患者。"**
> — NICE 2021

## 3. 活动管理 (Activity Management / Pacing)

### 3.1 活动管理核心概念

#### 3.1.1 能量包络理论

| 概念 | 描述 |
|------|------|
| **能量包络** | 患者可用能量的边界范围 |
| **留有余量** | 活动保持在包络内，预留缓冲 |
| **累积负荷** | 注意多种活动的总和效应 |
| **波动预期** | 能量包络会波动，需灵活调整 |

#### 3.1.2 活动管理原则

```
活动管理核心原则

1. 识别能量包络边界
   │
   ▼
2. 设定活动基线(在当前能力内舒适完成的量)
   │
   ▼
3. 将活动分散到全天
   │
   ▼
4. 在活动之间安排休息
   │
   ▼
5. 避免"繁荣-萧条"循环
   │
   ▼
6. 根据反应调整
```

### 3.2 心率监测法 (Heart Rate Monitoring)

#### 3.2.1 基本原理

| 概念 | 描述 |
|------|------|
| **无氧阈(AT)** | 从有氧代谢转向无氧代谢的心率点 |
| **目标心率** | 保持在AT以下，预防PEM |
| **计算方法** | 220 - 年龄 - 10-20% (保守估计) |
| **替代方法** | AT约为最大心率的55-60% |

#### 3.2.2 实施步骤

| 步骤 | 内容 | 注意事项 |
|------|------|---------|
| 1 | 确定目标心率上限 | 可从保守值开始 |
| 2 | 佩戴心率监测设备 | 手表或胸带 |
| 3 | 活动时监测心率 | 包括日常活动 |
| 4 | 心率接近上限时休息 | 不等到超过 |
| 5 | 记录活动与心率关系 | 了解个人模式 |
| 6 | 调整目标心率 | 根据PEM反应调整 |

#### 3.2.3 心率上限参考

| 年龄 | 保守目标心率上限(bpm) |
|------|---------------------|
| 20岁 | 100-110 |
| 30岁 | 95-105 |
| 40岁 | 90-100 |
| 50岁 | 85-95 |
| 60岁 | 80-90 |

**注意：** 这是一般参考，个体差异大，需根据反应调整

### 3.3 避免"繁荣-萧条"循环

```
繁荣-萧条循环 vs 活动管理

繁荣-萧条循环(有害):
好日子 → 过度活动 → PEM/崩溃 → 被迫休息 → 恢复 → 好日子...
   ↑________________________________↓

活动管理(目标):
       ┌─────────────────────────────────────────┐
       │                                         │
日常   │  ────────────────────────────────────   │ 能量包络上限
活动   │                                         │
       │  - - - - - - - - - - - - - - - - - - -  │ 活动基线
       │                                         │
       └─────────────────────────────────────────┘
              稳定、可持续的活动水平
```

### 3.4 休息技巧

#### 3.4.1 休息类型

| 类型 | 描述 | 适用情况 |
|------|------|---------|
| **预防性休息** | 活动前后计划性休息 | 所有患者 |
| **周期性休息** | 活动间短暂休息 | 日常管理 |
| **深度休息** | 躺下、减少刺激 | 疲劳明显时 |
| **睡眠休息** | 白天小睡(如需要) | 注意不影响夜间睡眠 |
| **认知休息** | 减少认知负荷 | 脑雾明显时 |
| **感觉休息** | 减少光、声等刺激 | 感觉过敏时 |

#### 3.4.2 有效休息要点

- 休息不仅是"不活动"，是主动恢复
- 环境安静、舒适
- 可以躺下(如直立不耐受)
- 减少感觉刺激
- 不要等到累了才休息

### 3.5 活动日志

| 记录内容 | 目的 |
|---------|------|
| 活动类型 | 识别触发因素 |
| 活动时长 | 量化活动负荷 |
| 强度/心率 | 监控强度 |
| 症状反应 | 评估活动影响 |
| 休息时长 | 确保充分恢复 |
| 睡眠质量 | 监控睡眠 |
| 整体状态(0-10) | 追踪趋势 |

## 4. 症状管理药物治疗 (Pharmacological Symptom Management)

### 4.1 用药原则

| 原则 | 说明 |
|------|------|
| **从低剂量开始** | ME/CFS患者常对药物敏感 |
| **缓慢增量** | 避免副作用 |
| **目标是症状控制** | 非治愈疾病 |
| **注意相互作用** | 多药物使用常见 |
| **定期评估** | 评估疗效和副作用 |
| **个体化** | 反应个体差异大 |

### 4.2 睡眠障碍药物

#### 4.2.1 常用睡眠药物

| 药物 | 类型 | 用法 | 注意事项 |
|------|------|------|---------|
| **褪黑素** | 激素 | 0.5-5mg睡前 | 调节节律，副作用少 |
| **曲唑酮** | 抗抑郁药(低剂量) | 25-100mg睡前 | 镇静作用，非成瘾性 |
| **阿米替林** | 三环类抗抑郁药(低剂量) | 10-50mg睡前 | 改善睡眠和疼痛 |
| **唑吡坦/佐匹克隆** | 非苯二氮卓类 | 短期使用 | 依赖风险，短期使用 |
| **苏沃雷生** | 食欲素受体拮抗剂 | 10-20mg | 新型，可能适合部分患者 |
| **加巴喷丁** | 抗惊厥药 | 100-300mg睡前 | 也可帮助疼痛 |

#### 4.2.2 睡眠药物选择考虑

| 患者特点 | 优先考虑 |
|---------|---------|
| 伴疼痛 | 阿米替林、加巴喷丁 |
| 节律紊乱 | 褪黑素 |
| 入睡困难 | 唑吡坦(短期) |
| 焦虑明显 | 曲唑酮 |
| 药物敏感 | 从最低剂量开始 |

### 4.3 疼痛管理药物

#### 4.3.1 疼痛药物分类

| 类别 | 药物 | 用途 | 注意事项 |
|------|------|------|---------|
| **神经调节剂** | | | |
| | 阿米替林 | 神经性疼痛、纤维肌痛 | 低剂量开始(10mg) |
| | 度洛西汀 | 纤维肌痛、广泛疼痛 | FDA批准用于纤维肌痛 |
| | 普瑞巴林 | 纤维肌痛、神经痛 | FDA批准用于纤维肌痛 |
| | 加巴喷丁 | 神经痛、纤维肌痛 | 低剂量开始 |
| **其他** | | | |
| | 对乙酰氨基酚 | 轻度疼痛 | 注意肝脏安全 |
| | NSAIDs | 炎症性疼痛、头痛 | 短期使用，注意胃肠 |
| | 低剂量纳曲酮(LDN) | 广泛疼痛、免疫调节 | 0.5-4.5mg，研究阶段 |
| | 肌肉松弛剂 | 肌肉痉挛 | 按需使用 |

#### 4.3.2 低剂量纳曲酮(LDN)

| 方面 | 描述 |
|------|------|
| **机制** | 可能通过调节小胶质细胞、内啡肽系统发挥作用 |
| **剂量** | 0.5mg开始，逐渐增至1.5-4.5mg |
| **证据** | 小规模研究显示部分患者获益，大型RCT缺乏 |
| **适应情况** | 疼痛、疲劳、其他治疗效果不佳 |
| **副作用** | 初期可能睡眠障碍、头痛(通常短暂) |
| **获取** | 需要处方，可能需要药房配制 |

### 4.4 直立不耐受/POTS药物

#### 4.4.1 非药物治疗优先

| 措施 | 描述 |
|------|------|
| 增加水分摄入 | 每日2-3升 |
| 增加盐摄入 | 每日3-10g(无高血压) |
| 压力袜 | 腰高或至少膝高 |
| 腹部绑带 | 减少血液淤积 |
| 避免快速站立 | 缓慢改变体位 |
| 避免长时间站立 | 坐或躺 |
| 避免热环境 | 热可加重症状 |

#### 4.4.2 POTS药物治疗

| 药物 | 类型 | 机制 | 注意事项 |
|------|------|------|---------|
| **氟氢可的松** | 盐皮质激素 | 增加血容量 | 0.1-0.2mg/日，监测电解质 |
| **米多君** | α1激动剂 | 血管收缩 | 2.5-10mg，每日3次，注意仰卧位高血压 |
| **普萘洛尔** | β阻滞剂 | 控制心率 | 10-20mg，每日2-3次，低剂量开始 |
| **伊伐布雷定** | If通道阻滞剂 | 控制心率 | 2.5-7.5mg，每日2次，新选择 |
| **吡斯的明** | 胆碱酯酶抑制剂 | 增强自主神经 | 30-60mg，每日3次 |
| **屈昔多巴** | 去甲肾上腺素前体 | 增加血压 | 研究阶段 |

#### 4.4.3 POTS药物选择策略

| 主要问题 | 优先药物 |
|---------|---------|
| 心动过速为主 | β阻滞剂、伊伐布雷定 |
| 低血压为主 | 氟氢可的松、米多君 |
| 混合型 | 联合用药 |
| 疲劳明显 | 注意β阻滞剂可能加重疲劳 |

### 4.5 认知症状药物

| 药物 | 证据 | 注意事项 |
|------|------|---------|
| 莫达非尼/阿莫达非尼 | 有限证据，部分患者获益 | 可能增加焦虑，谨慎使用 |
| 哌醋甲酯 | 个案报告 | 刺激剂，需谨慎 |
| 低剂量抗抑郁药 | 间接证据 | 如改善睡眠可能改善认知 |

**注意：** 认知症状的药物治疗证据有限，活动管理和睡眠改善可能更重要

### 4.6 其他症状药物

| 症状 | 药物选择 | 备注 |
|------|---------|------|
| **头痛** | 预防性：阿米替林、β阻滞剂；急性：对乙酰氨基酚、曲坦类 | 根据头痛类型选择 |
| **胃肠症状** | PPI、促动力药、解痉药 | 针对具体症状 |
| **过敏/MCAS** | 抗组胺药(H1+H2)、肥大细胞稳定剂 | 如共病MCAS |
| **抑郁/焦虑** | SSRIs/SNRIs | 注意与疲劳的关系 |

## 5. 心理支持与治疗 (Psychological Support & Treatment)

### 5.1 心理支持原则

| 原则 | 描述 |
|------|------|
| **验证体验** | 承认疾病是真实的、不是想象的 |
| **支持性** | 帮助应对而非改变疾病信念 |
| **非指责性** | 疾病不是患者的错 |
| **知情** | 治疗师需了解ME/CFS |
| **灵活** | 适应患者能量限制 |
| **创伤知情** | 许多患者经历医疗创伤 |

### 5.2 认知行为疗法(CBT)在ME/CFS中的角色

#### 5.2.1 NICE 2021立场

| 方面 | 说明 |
|------|------|
| **用途** | 支持应对疾病，管理症状影响，而非治愈疾病 |
| **不应** | 试图改变"不良疾病信念"，声称可治愈ME/CFS |
| **目标** | 改善生活质量、心理健康、应对能力 |
| **要求** | 治疗师需接受ME/CFS专门培训 |

#### 5.2.2 适当的CBT内容

| 可以包括 | 不应包括 |
|---------|---------|
| 活动管理技能 | 挑战"不良"疾病信念 |
| 睡眠卫生 | 分级运动推动 |
| 压力管理 | 认为症状是心理因素所致 |
| 应对策略 | 指责患者"避免活动" |
| 情绪调节 | 暗示患者可通过认知改变而康复 |
| 接纳与调适 | — |

### 5.3 接纳与承诺疗法(ACT)

| 方面 | 描述 |
|------|------|
| **核心** | 接纳困难，承诺价值导向行动 |
| **适用性** | 可能比传统CBT更适合ME/CFS |
| **内容** | 心理灵活性、正念、价值澄清、承诺行动 |
| **证据** | 有初步研究支持 |

### 5.4 其他心理支持

| 方法 | 描述 | 适用情况 |
|------|------|---------|
| **支持性心理治疗** | 倾听、支持、验证 | 所有患者 |
| **正念减压(MBSR)** | 正念技术适应ME/CFS | 适度改编后适用 |
| **哀伤辅导** | 处理功能丧失的哀伤 | 功能丧失明显 |
| **夫妻/家庭治疗** | 改善关系动态 | 家庭问题突出 |
| **同伴支持** | 患者互助团体 | 社交支持需要 |

### 5.5 重度患者的心理支持考虑

| 考虑 | 说明 |
|------|------|
| 能量限制 | 可能无法参加常规疗程 |
| 形式调整 | 电话/视频、短时间、低频率 |
| 感觉敏感 | 需要安静环境 |
| 内容调整 | 更多支持性，更少技术性 |
| 家属支持 | 支持照护者 |

## 6. 非药物干预 (Non-Pharmacological Interventions)

### 6.1 睡眠卫生

| 建议 | 说明 |
|------|------|
| 规律作息 | 固定起床时间(即使没睡好) |
| 睡眠环境 | 黑暗、安静、凉爽 |
| 限制咖啡因 | 尤其下午后避免 |
| 减少屏幕 | 睡前1-2小时避免蓝光 |
| 放松活动 | 睡前轻柔放松 |
| 避免卧床非睡眠活动 | 仅用于睡眠(ME/CFS需灵活) |

**ME/CFS特殊考虑：** 标准睡眠限制疗法对ME/CFS可能不适合，因患者需要更多休息

### 6.2 营养与饮食

#### 6.2.1 基本营养原则

| 原则 | 说明 |
|------|------|
| 均衡饮食 | 多样化、营养丰富 |
| 规律进食 | 避免血糖波动 |
| 充足水分 | 尤其POTS患者(2-3L/日) |
| 适量盐分 | POTS患者可能需增加 |
| 避免酒精 | 可能加重症状 |
| 减少加工食品 | 可能影响炎症 |

#### 6.2.2 可能有益的饮食模式

| 饮食模式 | 理论基础 | 证据 |
|---------|---------|------|
| 抗炎饮食 | 减少慢性炎症 | 一般健康证据，ME/CFS特异证据有限 |
| 低FODMAP | 改善IBS症状 | 对共病IBS可能有效 |
| 无麸质/无乳制品 | 减少敏感反应 | 部分患者报告获益，需个体评估 |
| 地中海饮食 | 综合健康益处 | 一般健康证据 |

### 6.3 补充剂

#### 6.3.1 常用补充剂

| 补充剂 | 可能机制 | 证据 | 注意事项 |
|--------|---------|------|---------|
| **辅酶Q10 (CoQ10)** | 线粒体支持 | 有限但支持性 | 100-300mg/日 |
| **D-核糖** | ATP合成 | 小规模研究支持 | 5g/日，多次 |
| **NADH** | 能量代谢 | 有限证据 | 5-10mg/日 |
| **镁** | 肌肉功能、睡眠 | 缺乏时有益 | 检测后补充 |
| **维生素D** | 免疫调节 | 缺乏时应补充 | 检测后补充 |
| **维生素B12** | 神经功能 | 缺乏时有益 | 检测后补充 |
| **Omega-3** | 抗炎 | 一般健康证据 | 1-3g/日 |
| **益生菌** | 肠道健康 | 有限证据 | 选择优质产品 |

#### 6.3.2 补充剂使用原则

| 原则 | 说明 |
|------|------|
| 检测后补充 | 优先纠正明确的缺乏 |
| 一次一种 | 便于评估效果 |
| 充分时间 | 给予4-8周评估 |
| 记录反应 | 记录症状变化 |
| 质量产品 | 选择可信品牌 |
| 告知医生 | 避免相互作用 |

### 6.4 物理治疗考虑

| 治疗 | 考虑 | 注意事项 |
|------|------|---------|
| 手法治疗 | 可能帮助疼痛 | 避免过度，评估PEM |
| 水疗 | 温和、支撑性 | 注意温度和时长 |
| 针灸 | 部分患者报告获益 | 个体反应不同 |
| 按摩 | 可能帮助放松、疼痛 | 温和，注意PEM |

**关键：** 任何物理治疗都需在能量包络内，监测PEM

### 6.5 环境与生活方式调整

| 领域 | 调整建议 |
|------|---------|
| **工作/学习** | 减少时间、弹性安排、远程工作、调整职责 |
| **家务** | 分散任务、寻求帮助、简化流程、使用辅助工具 |
| **社交** | 选择性参与、短时间、低强度、视频替代 |
| **环境** | 减少噪音、控制光线、温度舒适、休息空间 |
| **出行** | 轮椅/助行器(如需要)、停车位、避免拥挤 |
| **辅助设备** | 淋浴凳、扶手、电动车、语音助手 |

## 7. 特定症状管理方案 (Specific Symptom Management)

### 7.1 PEM管理

| 策略 | 描述 |
|------|------|
| **预防** | 活动管理是关键，保持在能量包络内 |
| **识别** | 学习识别个人PEM触发因素和早期信号 |
| **应对** | PEM发生时充分休息，减少活动和刺激 |
| **恢复** | 不急于恢复活动，等待充分恢复 |
| **记录** | 记录触发因素和恢复时间，优化管理 |

### 7.2 认知症状("脑雾")管理

| 策略 | 描述 |
|------|------|
| **认知pacing** | 认知活动也需要pacing |
| **最佳时间** | 在认知最佳时段处理重要任务 |
| **环境** | 减少干扰，安静环境 |
| **辅助工具** | 笔记、提醒、清单、语音助手 |
| **分解任务** | 将复杂任务分解为小步骤 |
| **休息** | 认知活动间休息 |
| **睡眠** | 良好睡眠可改善认知 |

### 7.3 睡眠障碍管理

| 策略 | 描述 |
|------|------|
| **睡眠卫生** | 基本原则(见上) |
| **睡眠环境** | 黑暗、安静、凉爽、舒适 |
| **睡前routine** | 放松活动、减少刺激 |
| **药物** | 如需要，从低剂量开始 |
| **不强制限制** | 避免严格睡眠限制(可能不适合ME/CFS) |
| **白天小睡** | 如需要可以，注意不影响夜间 |

### 7.4 疼痛管理

| 层次 | 方法 |
|------|------|
| **第一线** | 活动管理、姿势、热敷/冷敷 |
| **第二线** | 对乙酰氨基酚、局部用药 |
| **第三线** | 低剂量神经调节剂(阿米替林、加巴喷丁) |
| **第四线** | 疼痛专科评估、LDN |
| **辅助** | 物理治疗(温和)、放松、正念 |

### 7.5 直立不耐受管理

| 层次 | 方法 |
|------|------|
| **第一线** | 增加水分和盐、压力袜、缓慢站立 |
| **第二线** | 药物治疗(氟氢可的松、米多君等) |
| **行为** | 避免长时间站立、避免热环境 |
| **辅助** | 倾斜位睡眠、反搏动作 |

## 9. 研究性治疗 (Investigational Treatments)

### 9.1 免疫调节治疗

| 治疗 | 状态 | 注意事项 |
|------|------|---------|
| **利妥昔单抗** | RCT未确认早期阳性结果 | 不推荐常规使用 |
| **免疫球蛋白(IVIG)** | 有限证据，部分亚组可能获益 | 费用高，需更多研究 |
| **免疫吸附** | 小规模研究中 | 研究阶段 |
| **Rintatolimod(Ampligen)** | 有限国家批准 | 获取困难 |

### 9.2 代谢干预

| 治疗 | 理论基础 | 状态 |
|------|---------|------|
| **Suramin** | 逆转细胞危险反应 | 早期研究，安全性问题 |
| **代谢鸡尾酒** | 支持线粒体 | 研究中 |
| **Oxaloacetate** | 代谢支持 | 小规模研究 |

### 9.3 抗病毒治疗

| 治疗 | 适应情况 | 证据 |
|------|---------|------|
| **伐昔洛韦** | EBV/HHV-6再激活证据 | 有限，亚组可能获益 |
| **其他抗病毒药** | 研究阶段 | 需要更多证据 |

**注意：** 研究性治疗应在知情同意下、最好在临床试验框架内进行

## 10. 治疗计划与随访 (Treatment Planning & Follow-up)

### 10.1 个体化治疗计划

```
ME/CFS个体化治疗计划框架

1. 评估
   ├── 严重度分级
   ├── 主要症状识别
   ├── 共病评估
   └── 功能水平评估
          ↓
2. 目标设定
   ├── 与患者共同设定
   ├── 现实可行
   └── 阶段性目标
          ↓
3. 干预选择
   ├── 活动管理(所有患者)
   ├── 症状针对性药物
   ├── 心理支持
   └── 生活调整
          ↓
4. 实施与监测
   ├── 逐步实施
   ├── 记录反应
   └── 定期评估
          ↓
5. 调整优化
   ├── 根据反应调整
   ├── 处理新问题
   └── 优化方案
```

### 10.2 随访计划

| 阶段 | 频率 | 内容 |
|------|------|------|

| **稳定期** | 1-3月 | 常规随访 |
| **长期** | 3-6月 | 维持随访 |
| **恶化时** | 按需 | 评估原因、调整方案 |

## 11. 专家治疗观点 (Expert Treatment Perspectives)

### 11.1 治疗关键点

> **"活动管理是ME/CFS治疗的基石。没有它，其他治疗都难以发挥最佳效果。"**
> — Lucinda Bateman, MD

> **"我们不能治愈ME/CFS，但我们可以帮助患者最大化生活质量。这需要耐心、创造力和与患者的真正合作。"**
> — Charles Lapp, MD

> **"不要做的事情有时比做什么更重要。不推动、不强迫、不否认——这是帮助ME/CFS患者的起点。"**
> — NICE 2021

### 11.2 常见治疗误区

| 误区 | 正确理解 |
|------|---------|
| "多运动会好" | GET可能有害，需要活动管理 |
| "心理问题导致" | 是生物医学疾病，心理支持是辅助 |
| "没有治疗方法所以不能帮助" | 可以显著改善生活质量 |
| "推动患者走出舒适区" | 推动可能导致持续恶化 |
| "检查正常就没问题" | 缺乏生物标志物不等于没有疾病 |

### 11.3 患者赋权

| 方面 | 内容 |
|------|------|
| **知识赋权** | 提供充分的疾病信息 |
| **自我管理** | 教授活动管理技能 |
| **共同决策** | 治疗决定与患者共同做出 |
| **倡导** | 支持患者倡导自己的需求 |
| **资源连接** | 连接患者组织和支持资源 |
