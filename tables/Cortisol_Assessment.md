# 皮质醇检测与评估 (Cortisol Assessment & Testing)

## 核心术语与词源 (Core Terminology & Etymology)

| 中文术语 | 英文术语 | 拉丁/希腊词源 | 定义与说明 |
|---------|---------|--------------|-----------|
| 免疫测定法 | Immunoassay | *Immuno-* (拉丁语 *immunis* = 免疫) + *assay* | 利用抗原-抗体反应检测分析物的方法 |
| 液相色谱-质谱联用 | LC-MS/MS | *Liquid Chromatography-Mass Spectrometry* | 分离和检测皮质醇的金标准方法 |
| 游离皮质醇 | Free Cortisol | — | 未与蛋白结合的活性皮质醇，约占总量3-5% |
| 皮质醇结合球蛋白 | CBG (Transcortin) | — | 血浆中结合80-90%皮质醇的主要载体蛋白 |
| 地塞米松抑制试验 | Dexamethasone Suppression Test (DST) | — | 评估HPA轴负反馈功能的动态试验 |
| ACTH刺激试验 | ACTH Stimulation Test | — | 评估肾上腺皮质储备功能的动态试验 |
| 皮质醇觉醒反应 | Cortisol Awakening Response (CAR) | — | 觉醒后皮质醇急剧上升的现象及其测量 |
| 曲线下面积 | Area Under the Curve (AUC) | — | 评估一段时间内皮质醇总暴露量的指标 |
| 昼夜节律斜率 | Diurnal Slope | — | 皮质醇从晨到晚下降的速率 |
| 唾液皮质醇 | Salivary Cortisol | — | 反映游离皮质醇的无创样本 |
| 尿游离皮质醇 | Urinary Free Cortisol (UFC) | — | 24小时尿液中皮质醇总量 |
| 毛发皮质醇 | Hair Cortisol | — | 反映数周至数月平均皮质醇水平 |
| 交叉反应 | Cross-Reactivity | — | 免疫测定中与类似结构物质的非特异性反应 |
| 敏感性 | Sensitivity | 拉丁语 *sensus* (感觉) | 检测方法识别真阳性的能力 |
| 特异性 | Specificity | 拉丁语 *species* (种类) | 检测方法排除假阳性的能力 |

## 皮质醇检测样本类型比较 (Sample Types Comparison)

| 样本类型 | 反映内容 | 优点 | 缺点 | 主要应用 |
|---------|---------|-----|-----|---------|
| **血清/血浆** | 总皮质醇 (游离+结合) | 临床标准、广泛可及 | 有创、受CBG影响、单点采样 | 临床诊断、动态试验 |
| **唾液** | 游离皮质醇 | 无创、多点采样方便、反映活性形式 | 标准化变异、口腔污染风险 | 研究、昼夜节律、CAR |
| **24h尿液** | 日总游离皮质醇 | 避免昼夜节律影响、整合指标 | 收集依从性、肾功能影响 | 库欣综合征筛查 |
| **毛发** | 数周-数月平均皮质醇 | 长期暴露评估、回顾性 | 方法学变异、外部污染 | 慢性应激研究 |
| **指尖采血 (DBS)** | 总皮质醇 | 微量、便捷、可邮寄 | 转化因子需要、标准化发展中 | 研究、远程监测 |

## 血清/血浆皮质醇检测 (Serum/Plasma Cortisol)

### 检测方法比较

| 方法 | 原理 | 优点 | 缺点 | 临床应用 |
|-----|-----|-----|-----|---------|
| **免疫测定法 (Immunoassay)** | | | | |
| 化学发光免疫分析 (CLIA) | 抗体-抗原反应+化学发光 | 自动化、高通量、广泛可及 | 交叉反应、方法间差异 | 常规临床 |
| 电化学发光 (ECLIA) | 电化学发光检测 | 灵敏度高、自动化 | 交叉反应 | 常规临床 |
| 放射免疫测定 (RIA) | 放射性标记 | 历史金标准 | 放射性、逐渐淘汰 | 减少使用 |
| **色谱-质谱法** | | | | |
| LC-MS/MS | 液相色谱分离+质谱检测 | 高特异性、无交叉反应、可同时测多种类固醇 | 设备昂贵、技术要求高 | 金标准、复杂情况 |
| GC-MS | 气相色谱+质谱 | 高特异性 | 需衍生化、通量低 | 特殊情况 |

### 参考范围

| 时间点 | 成人参考范围 (μg/dL) | 参考范围 (nmol/L) | 说明 |
|-------|---------------------|------------------|-----|
| **晨起 (08:00 ± 1h)** | 6-23 | 166-635 | 个体和实验室差异 |
| **下午 (16:00)** | 3-15 | 83-414 | 约为晨值的50-65% |
| **午夜** | <1.8 | <50 | 库欣筛查切点 |
| **峰值 (觉醒后30-45分钟)** | 10-25 | 275-690 | CAR峰值 |

### 影响血清皮质醇的因素

| 因素类别 | 具体因素 | 影响方向 | 机制/说明 |
|---------|---------|---------|----------|
| **生理因素** | | | |
| 昼夜节律 | 晨高晚低 | 变化50-60% | 自然节律 |
| 应激 | 急性应激 | ↑ | HPA轴激活 |
| 进食 | 餐后 | ↑ | 血糖波动 |
| 运动 | 中高强度 | ↑ | 应激反应 |
| 月经周期 | 黄体期 | ↑ | 雌激素影响CBG |
| 妊娠 | — | ↑↑ (2-3倍) | CBG增加、胎盘皮质醇 |
| **药物影响** | | | |
| 雌激素/口服避孕药 | — | ↑ (总皮质醇) | CBG升高 |
| 糖皮质激素 | 外源性 | ↓ (内源性) | 负反馈抑制 |
| 苯妥英/利福平 | 肝酶诱导 | ↓ | 加速代谢 |
| 阿片类 | — | ↓ | 抑制HPA轴 |
| **病理因素** | | | |
| 肝病 | CBG合成减少 | ↓ (总)，游离不变 | 需测游离皮质醇 |
| 肾病综合征 | CBG丢失 | ↓ (总) | 需测游离皮质醇 |

| 甲状腺功能亢进 | CBG变化 | ↓ | 代谢增加 |

### 采样注意事项

| 要素 | 建议 | 理由 |
|-----|-----|-----|
| **时间** | 记录精确采样时间 | 昼夜节律影响大 |
| **姿势** | 静坐15-30分钟后采样 | 姿势影响结果 |
| **应激** | 避免采样前应激事件 | 应激升高皮质醇 |
| **禁食** | 晨起禁食状态采样 | 进食影响结果 |
| **运动** | 采样前避免剧烈运动 | 运动升高皮质醇 |
| **药物** | 记录相关药物 | 药物干扰解读 |
| **女性** | 记录月经周期、是否避孕药 | 雌激素影响CBG |

## 唾液皮质醇检测 (Salivary Cortisol)

### 唾液皮质醇的特点

| 特点 | 说明 | 意义 |
|-----|-----|-----|
| **反映游离皮质醇** | 唾液中皮质醇为未结合形式 | 反映生物活性部分 |
| **与血清游离相关** | r = 0.8-0.9 | 良好替代指标 |
| **无创采样** | 不需静脉穿刺 | 适合多点采样、应激研究 |
| **家庭自采** | 患者可自行采集 | 提高昼夜节律评估可行性 |
| **浓度较低** | 约为血清的1/10-1/20 | 需灵敏方法 |

### 参考范围

| 时间点 | 参考范围 (nmol/L) | 参考范围 (μg/dL) | 说明 |
|-------|------------------|------------------|-----|
| **觉醒时** | 5-25 | 0.18-0.91 | 起床即刻 |
| **觉醒后30分钟 (CAR峰值)** | 10-40 | 0.36-1.45 | 正常CAR |
| **08:00-09:00** | 5-25 | 0.18-0.91 | 晨间参考 |
| **下午 (15:00-17:00)** | 2-10 | 0.07-0.36 | 下午参考 |
| **睡前 (22:00-24:00)** | <2-4 | <0.07-0.15 | 库欣筛查切点 |

### 采样方法

| 方法 | 操作 | 优点 | 缺点 |
|-----|-----|-----|-----|
| **被动流口水** | 自然流入管中 | 最少污染风险 | 某些人产生困难 |
| **Salivette棉签** | 咀嚼棉签2-3分钟 | 方便、标准化 | 可能刺激分泌 |
| **合成棉签** | 聚酯/人造丝棉签 | 减少皮质醇吸附 | 较新方法 |
| **口腔拭子** | 拭子刮擦口腔 | 快速 | 可能污染 |

### 采样协议

| 要素 | 建议 | 理由 |
|-----|-----|-----|
| **采样前** | 30分钟内避免进食、饮水(除白水)、刷牙、吸烟 | 减少污染、pH影响 |
| **血液污染** | 检查口腔有无出血 | 血液污染显著升高结果 |
| **时间记录** | 精确记录各样本时间 | 昼夜节律解读 |
| **储存** | 采样后冷藏或冷冻 | 保持稳定性 |
| **运输** | 冰袋或冷藏运输 | 减少降解 |
| **标记** | 清晰标记被试信息、时间点 | 避免混淆 |

## 24小时尿游离皮质醇 (24-Hour Urinary Free Cortisol, UFC)

### 检测特点

| 特点 | 说明 | 临床意义 |
|-----|-----|---------|
| **整合指标** | 反映24小时皮质醇分泌总量 | 避免单点波动 |
| **游离皮质醇** | 未结合皮质醇滤过排泄 | 不受CBG影响 |
| **库欣筛查** | 敏感性高的初筛方法 | 指南推荐 |
| **定量负荷** | 皮质醇过量的整体评估 | 库欣诊断支持 |

### 参考范围

| 方法 | 参考范围 | 库欣筛查切点 | 说明 |
|-----|---------|-------------|-----|
| **免疫测定法** | 20-90 μg/24h (55-248 nmol/24h) | >3倍上限高度提示库欣 | 方法间差异大 |
| **LC-MS/MS** | 4-50 μg/24h (11-138 nmol/24h) | 更精确 | 金标准方法 |

### 采集注意事项

| 要素 | 建议 | 理由 |
|-----|-----|-----|
| **完整收集** | 所有24小时尿液均收集 | 不完整导致假阴性 |
| **首次尿弃去** | 开始时排空膀胱、弃去 | 避免前一天残留 |
| **冷藏保存** | 收集期间冷藏 | 防止降解 |
| **测尿量** | 记录总尿量 | 评估收集完整性 |
| **测肌酐** | 同时测24h尿肌酐 | 评估收集完整性 (肌酐应恒定) |
| **避免应激** | 收集日避免重大应激 | 代表基础分泌 |
| **液体摄入** | 正常液体摄入 | 避免过度稀释/浓缩 |

### 假阳性/假阴性原因

| 结果偏差 | 原因 | 处理建议 |
|---------|-----|---------|
| **假阳性升高** | 假性库欣状态 (抑郁、酒精、肥胖) | 结合临床判断 |
| | 高液体摄入 (>5L/日) | 复查限制液体后 |
| | 急性疾病/应激 | 恢复后复查 |
| | 交叉反应 (免疫法) | 使用LC-MS/MS |
| **假阴性降低** | 收集不完整 | 检查尿肌酐 |
| | 周期性库欣 | 多次收集 |
| | 肾功能减退 (GFR<60) | 改用其他方法 |
| | 轻度库欣 | 敏感性限制 |

## 毛发皮质醇检测 (Hair Cortisol)

### 检测特点

| 特点 | 说明 | 优势/局限 |
|-----|-----|----------|
| **反映长期暴露** | 1cm头发≈1个月皮质醇 | 慢性应激评估 |
| **回顾性评估** | 可回溯数月应激水平 | 研究价值 |
| **无创采样** | 剪取少量头发 | 依从性好 |
| **方法学发展中** | 标准化程度低于血/唾液 | 解读需谨慎 |

### 采样方法

| 步骤 | 操作 | 要点 |
|-----|-----|-----|
| **采样位置** | 后枕部 (后顶部头皮) | 生长速度最一致 |
| **采样量** | 约10-50mg (约5-15根发束) | 足够检测量 |
| **采样长度** | 贴近头皮剪取 | 确定时间窗口 |
| **标记根端** | 标记靠近头皮的一端 | 确定时间方向 |
| **储存** | 室温、避光、干燥保存 | 长期稳定 |
| **分析段** | 通常取头皮端1-3cm | 反映近1-3个月 |

### 影响因素

| 因素 | 影响 | 处理建议 |
|-----|-----|---------|
| **染发/烫发** | 可能降低皮质醇 | 记录、排除或校正 |
| **洗发频率** | 频繁洗头可能降低 | 标准化询问 |
| **头发颜色** | 深色可能略高 | 纳入分析 |
| **发质** | 粗/细、直/卷可能影响 | 纳入分析 |
| **外部污染** | 汗液、环境皮质醇 | 清洗协议 |
| **季节** | 可能有季节性变化 | 纳入分析 |

### 参考范围

| 人群 | 参考范围 (pg/mg) | 说明 |
|-----|-----------------|-----|
| **健康成人** | 5-25 | 方法依赖性大 |
| **慢性应激升高** | >25-30 | 提示长期高应激 |
| **库欣综合征** | 显著升高 (数倍) | 筛查价值 |

## 皮质醇觉醒反应 (Cortisol Awakening Response, CAR)

### CAR定义与意义

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **定义** | 觉醒后30-45分钟内皮质醇浓度急剧上升50-75%的现象 | 独立于昼夜节律的HPA轴反应 |
| **功能意义** | 预期应激的准备、能量动员、认知警醒 | 适应性功能 |
| **神经调控** | 下丘脑视交叉上核、海马参与 | 不同于应激反应的调控 |
| **临床意义** | 反映HPA轴功能完整性 | 多种障碍中异常 |

### CAR采样协议

| 时间点 | 采样说明 | 注意事项 |
|-------|---------|---------|
| **S1: 觉醒即刻** | 睁眼后立即采样 (0分钟) | 不要下床 |
| **S2: 觉醒后15分钟** | 精确计时 | 可正常活动 |
| **S3: 觉醒后30分钟** | 预期峰值点 | 精确计时 |
| **S4: 觉醒后45分钟** | 可选 | 确认峰值 |
| **S5: 觉醒后60分钟** | 可选 | 恢复趋势 |

### CAR分析指标

| 指标 | 计算方法 | 意义 |
|-----|---------|-----|
| **CARi (Increase)** | Peak - Awakening | 上升幅度 (nmol/L) |
| **CARi%** | (Peak - Awakening) / Awakening × 100% | 相对上升百分比 |
| **AUCg (Ground)** | 曲线下总面积 | 总体皮质醇暴露 |
| **AUCi (Increase)** | 超过基线的面积 | 觉醒反应特异性 |
| **Mean Increase** | 多个post-awakening样本均值 - 觉醒值 | 平均上升 |

### CAR异常模式

| 异常类型 | 特征 | 相关状态 | 解读 |
|---------|-----|---------|-----|
| **CAR缺失/减弱** | 上升<25%或缺如 | 慢性疲劳、倦怠、PTSD、抑郁亚型 | HPA轴钝化 |
| **CAR增强** | 上升>100% | 工作应激、预期焦虑、高神经质 | HPA轴高反应 |
| **CAR延迟** | 峰值后移至45-60分钟 | 睡眠问题、某些抑郁 | 节律异常 |
| **CAR不稳定** | 日间变异大 | 不稳定HPA轴状态 | 需多日采样 |

### CAR研究注意事项

| 要素 | 建议 | 理由 |
|-----|-----|-----|
| **觉醒时间确认** | 使用运动检测器或电子监测 | 自报告不可靠 |
| **多日采样** | 至少2-3个工作日 | 日间变异大 |
| **依从性检查** | 检查采样时间记录、设置闹钟提醒 | 依从性影响结果 |
| **排除因素** | 记录睡眠质量、闹钟唤醒vs自然醒 | 影响CAR解读 |
| **周末效应** | 分开分析工作日/周末 | 应激预期不同 |

## 动态功能试验 (Dynamic Function Tests)

### 地塞米松抑制试验 (DST)

#### 低剂量过夜DST (1mg)

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **适应症** | 库欣综合征筛查 | 一线筛查 |
| **方法** | 23:00-24:00口服地塞米松1mg，次日08:00测血清皮质醇 | 简便 |
| **正常反应** | 皮质醇<1.8 μg/dL (50 nmol/L) | 抑制良好 |
| **库欣提示** | 皮质醇>1.8 μg/dL | 需进一步检查 |
| **敏感性** | ~95% | 高敏感筛查 |
| **特异性** | ~80% | 假阳性常见 |

#### 假阳性原因

| 原因类别 | 具体原因 | 处理建议 |
|---------|---------|---------|
| **假性库欣** | 抑郁、酒精、肥胖 | 临床评估、治疗原发病后复查 |
| **药物** | 苯妥英、巴比妥、利福平 (加速代谢) | 了解用药史 |
| **药物** | 雌激素 (升高CBG) | 测游离皮质醇或停药后复查 |
| **应激** | 急性疾病、住院 | 恢复后复查 |
| **不依从** | 未服药或服药时间错误 | 确认依从性 |

#### 2日低剂量DST (2mg/日×2天)

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **适应症** | 1mgDST假阳性后确认 | 特异性更高 |
| **方法** | 地塞米松0.5mg q6h×2天，末次服药后6小时测皮质醇 | 更长抑制 |
| **正常反应** | 皮质醇<1.8 μg/dL | 抑制良好 |
| **特异性** | ~95% | 更好排除假阳性 |

#### 高剂量DST (8mg)

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **适应症** | 库欣病因鉴别 | 垂体vs异位ACTH |
| **方法** | 过夜8mg或2天高剂量 | 抑制垂体源 |
| **库欣病 (垂体)** | 皮质醇抑制>50% | 保留部分负反馈 |
| **异位ACTH** | 无抑制 | 不受负反馈调控 |
| **注意** | 现更多使用CRH试验、岩下窦静脉采样 | 金标准方法更准确 |

### ACTH刺激试验 (Synacthen Test)

#### 标准剂量 (250μg)

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **适应症** | 肾上腺功能不全筛查/诊断 | 评估肾上腺储备 |
| **方法** | IV/IM注射合成ACTH 250μg，测0, 30, 60分钟皮质醇 | 标准方法 |
| **正常反应** | 峰值皮质醇>18-20 μg/dL (500-550 nmol/L) | 肾上腺储备充足 |
| **异常** | 峰值<18 μg/dL | 提示肾上腺功能不全 |
| **局限** | 不能检出轻度或近期继发性 | 需结合临床 |

#### 低剂量 (1μg)

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **适应症** | 轻度/早期/继发性肾上腺功能不全 | 更敏感 |
| **方法** | IV注射1μg ACTH，测0, 20, 30分钟皮质醇 | 敏感方法 |
| **正常反应** | 峰值>18 μg/dL | 较敏感指标 |
| **优势** | 检出轻度继发性功能不全 | 不刺激正常最大反应 |

### CRH刺激试验

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **适应症** | 库欣病因鉴别、继发性肾上腺功能不全 | 特殊情况 |
| **方法** | IV注射人CRH 100μg或羊CRH 1μg/kg，测0, 15, 30, 60分钟皮质醇和ACTH | 评估垂体反应 |
| **库欣病** | 皮质醇和ACTH过度反应 | 垂体腺瘤 |
| **异位ACTH** | 无反应 | 不受CRH调控 |
| **继发性肾上腺不全** | ACTH反应延迟或过度 | 区分垂体vs下丘脑 |

### 胰岛素耐量试验 (ITT)

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **适应症** | HPA轴完整性金标准评估 | 垂体功能评估 |
| **方法** | IV注射胰岛素诱导低血糖 (<40 mg/dL)，测皮质醇反应 | 生理应激刺激 |
| **正常反应** | 皮质醇峰值>18-20 μg/dL | HPA轴完整 |
| **禁忌** | 癫痫、心脏病、老年人 | 低血糖风险 |
| **需要** | 医疗监督、抢救准备 | 安全保障 |

## 昼夜节律评估 (Circadian Rhythm Assessment)

### 昼夜节律采样方案

| 方案类型 | 采样时间点 | 适用场景 | 分析指标 |
|---------|-----------|---------|---------|
| **简化方案** | 08:00, 16:00, 24:00 | 临床筛查 | 晨/晚比值 |
| **标准方案** | 08:00, 12:00, 16:00, 20:00, 24:00 | 研究 | 曲线形态、AUC |
| **详细方案** | 每2-4小时，24小时内 | 深入研究 | 完整曲线 |
| **CAR+日间** | 觉醒系列 + 日间数点 | 综合评估 | CAR + 日内节律 |

### 昼夜节律分析指标

| 指标 | 计算方法 | 意义 | 正常参考 |
|-----|---------|-----|---------|
| **晨峰值** | 08:00或觉醒+30min皮质醇 | HPA轴功能 | 10-20 μg/dL |
| **夜间谷值** | 24:00或夜间皮质醇 | 库欣筛查 | <1.8 μg/dL (唾液<0.15 μg/dL) |
| **晨/晚比值** | AM/PM cortisol | 节律保存 | >2-3 |
| **日内斜率** | 皮质醇下降速率 | 节律强度 | 负斜率 (陡峭) |
| **AUCg** | 曲线下总面积 | 总暴露量 | 个体化 |
| **曲线扁平度** | 峰值-谷值差异 | 节律丧失 | 大差异为正常 |

### 节律紊乱模式识别

| 模式 | 特征 | 相关状态 | 图示特点 |
|-----|-----|---------|---------|
| **正常节律** | 晨高晚低、明显昼夜变化 | 健康状态 | 陡峭下降曲线 |
| **曲线扁平化** | 晨低晚高、差异小 | 抑郁、PTSD、慢性应激 | 平坦曲线 |
| **持续升高** | 全天高水平 | 库欣、急性应激 | 高位平坦 |
| **持续降低** | 全天低水平 | 肾上腺功能不全 | 低位平坦 |
| **相位延迟** | 峰值后移 | 昼夜节律障碍、晚睡 | 曲线右移 |
| **相位提前** | 峰值前移 | 老年人、早醒 | 曲线左移 |

## 应激反应性评估 (Stress Reactivity Assessment)

### 心理社会应激范式

#### 特里尔社会应激测试 (TSST)

| 方面 | 内容 | 说明 |
|-----|-----|-----|
| **组成** | 模拟面试 (5分钟) + 心算任务 (5分钟) | 社会评价 + 认知挑战 |
| **评委** | 2-3名表情冷淡的评委 | 社会评价压力 |
| **录像** | 告知被录像 | 增加社会评价感 |
| **皮质醇峰值** | 应激后15-30分钟 | TSST后典型峰值时间 |
| **典型反应** | 皮质醇升高2-4倍 | 正常反应者 |
| **低反应者** | 升高<50%或无反应 | HPA轴钝化 |
| **高反应者** | 升高>4倍 | HPA轴高敏感 |

#### TSST采样方案

| 时间点 | 相对TSST时间 | 说明 |
|-------|-------------|-----|
| **基线1** | -30分钟 | 适应期 |
| **基线2** | -10分钟 | 预应激基线 |
| **应激后1** | +0分钟 | 应激结束即刻 |
| **应激后2** | +10分钟 | 上升期 |
| **应激后3** | +20分钟 | 接近峰值 |
| **应激后4** | +30分钟 | 峰值 |
| **应激后5** | +45分钟 | 恢复期 |
| **应激后6** | +60分钟 | 恢复期 |
| **应激后7** | +90分钟 | 可选，完全恢复 |

#### 其他应激范式

| 范式 | 应激类型 | 皮质醇反应 | 适用研究 |
|-----|---------|-----------|---------|
| **冷加压试验 (CPT)** | 躯体疼痛 | 中等反应 | 疼痛研究 |
| **蒙特利尔成像应激任务 (MIST)** | 心算+社会评价 | 类似TSST | fMRI兼容 |
| **马斯特里赫特急性应激任务 (MAST)** | CPT + 心算 | 复合反应 | 结合躯体+心理 |
| **公开演讲** | 社会评价 | 中-强反应 | 社交焦虑研究 |
| **Stroop任务** | 认知冲突 | 弱反应 | 轻度认知应激 |

### 应激反应分析指标

| 指标 | 计算方法 | 意义 |
|-----|---------|-----|
| **反应性** | 峰值 - 基线 | 应激诱发的皮质醇上升量 |
| **相对反应** | (峰值 - 基线) / 基线 × 100% | 相对于基线的上升比例 |
| **达峰时间** | 峰值出现的时间点 | 反应速度 |
| **恢复时间** | 返回基线所需时间 | 恢复效率 |
| **AUCi (反应)** | 超过基线的面积 | 总应激反应量 |
| **AUCg (总量)** | 曲线下总面积 | 总皮质醇暴露 |

## 临床应用场景 (Clinical Application Scenarios)

### 库欣综合征评估流程

| 步骤 | 检查内容 | 判断标准 | 下一步 |
|-----|---------|---------|-------|
| **1. 临床筛查** | 体征评估 | 典型库欣体征 | 进行实验室筛查 |
| **2. 初筛 (任选2项)** | 24h UFC、1mg DST、午夜唾液皮质醇 | 任一异常 | 需确认 |
| **3. 确认** | 重复异常项目 | 2次以上异常 | 定位检查 |
| **4. 病因定位** | ACTH、影像、CRH试验、IPSS | ACTH依赖vs非依赖 | 治疗决策 |

### 肾上腺功能不全评估

| 步骤 | 检查内容 | 判断标准 | 说明 |
|-----|---------|---------|-----|
| **1. 临床怀疑** | 症状体征 | 疲劳、低血压、色素沉着等 | 启动评估 |
| **2. 晨起皮质醇** | 08:00血清皮质醇 | <3 μg/dL确诊，3-15需刺激试验 | 初步判断 |
| **3. ACTH刺激试验** | 250μg或1μg ACTH | 峰值<18 μg/dL确诊 | 确认 |
| **4. 病因鉴别** | ACTH水平、影像 | 原发vs继发 | 治疗指导 |

### 精神科应用考虑

| 应用场景 | 推荐检测 | 临床价值 | 注意事项 |
|---------|---------|---------|---------|
| **治疗抵抗抑郁** | 晨起皮质醇、DST | 中等 | 排除器质性病因 |
| **非典型表现** | 全面内分泌评估 | 中-高 | 排除库欣/Addison |
| **研究目的** | CAR、昼夜节律、应激反应 | 高 | 标准化方案 |
| **常规评估** | 不推荐 | 低 | 特异性/敏感性不足 |

## 实验室质量控制 (Laboratory Quality Control)

### 分析前因素

| 因素 | 控制措施 | 重要性 |
|-----|---------|-------|
| **采样时间** | 标准化、记录精确时间 | 高 |
| **样本处理** | 及时离心、冷冻 | 高 |
| **溶血/脂血** | 避免、检查、记录 | 中 |
| **患者准备** | 禁食、避免应激、姿势 | 高 |
| **药物干扰** | 详细用药史 | 高 |

### 分析中因素

| 因素 | 控制措施 | 重要性 |
|-----|---------|-------|
| **方法学** | 使用经验证的方法 | 高 |
| **校准** | 定期校准、溯源标准 | 高 |
| **质控品** | 每批次运行QC样本 | 高 |
| **交叉反应** | 了解方法特异性限制 | 中-高 |
| **批间变异** | 同一患者同批次测量 | 中 |

### 结果解读

| 要素 | 考虑内容 | 建议 |
|-----|---------|-----|
| **参考范围** | 使用本实验室建立的范围 | 不同方法/人群差异 |
| **临床情境** | 结合症状体征 | 不单独依赖实验室值 |
| **动态变化** | 纵向比较优于单点 | 建立个体基线 |
| **多指标综合** | 不依赖单一指标 | 综合评估 |

## 专家建议 (Expert Recommendations)

### 临床评估建议

| 场景 | 推荐方法 | 理由 |
|-----|---------|-----|
| **库欣筛查** | 24h UFC、1mg DST、午夜唾液 (≥2项) | 指南推荐 |
| **肾上腺功能不全筛查** | 晨起皮质醇、ACTH刺激试验 | 标准流程 |
| **昼夜节律评估** | 唾液皮质醇多点采样 | 无创、方便 |
| **CAR研究** | 觉醒后0、15、30、45分钟唾液 | 标准化方案 |
| **应激反应研究** | TSST + 多时间点采样 | 金标准范式 |
| **长期应激评估** | 毛发皮质醇 | 回顾性评估 |

### 研究设计建议

| 建议 | 理由 |
|-----|-----|
| **多时间点采样** | 单点变异大 |
| **多日采样** | 日间变异 |
| **控制混杂因素** | 时间、姿势、药物、月经周期 |
| **使用标准化范式** | 可比性 |
| **报告详细方法** | 可重复性 |
| **考虑个体差异** | 基因、早年经历、性别 |
