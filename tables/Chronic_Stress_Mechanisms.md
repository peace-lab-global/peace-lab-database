## 1. HPA轴：核心调节系统 (HPA Axis: Central Regulatory System)

### 1.1 HPA轴术语表

| 中文术语 | 英文术语 | 拉丁语/希腊语词源 | 定义 |
|:---------|:---------|:-----------------|:-----|
| 下丘脑 | Hypothalamus | 希腊语 ὑπό (hypó, 下方) + θάλαμος (thálamos, 内室) | 控制内分泌和自主功能的脑区 |
| 室旁核 | Paraventricular Nucleus (PVN) | 拉丁语 *para* (旁边) + *ventriculus* (小腔) | 下丘脑中合成CRH的核团 |
| 垂体 | Pituitary | 拉丁语 *pituita* (黏液) | "主腺"，分泌多种激素 |
| 肾上腺 | Adrenal Gland | 拉丁语 *ad* (在旁) + *ren* (肾) | 位于肾上方的内分泌腺 |
| 肾上腺皮质 | Adrenal Cortex | 拉丁语 *cortex* (树皮、外层) | 合成皮质醇的区域 |
| 促肾上腺皮质激素释放激素 | Corticotropin-Releasing Hormone (CRH) | — | 由PVN分泌，刺激ACTH释放 |
| 促肾上腺皮质激素 | Adrenocorticotropic Hormone (ACTH) | — | 由垂体前叶分泌，刺激皮质醇合成 |
| 皮质醇 | Cortisol | 拉丁语 *cortex* (皮层) + *-ol* | 主要糖皮质激素，"压力激素" |
| 糖皮质激素 | Glucocorticoid | 希腊语 γλυκύς (glykýs, 甜) + *cortex* | 调节代谢和应激反应的激素类别 |
| 负反馈 | Negative Feedback | — | 产物抑制上游活动的调节机制 |
| 糖皮质激素受体 | Glucocorticoid Receptor (GR) | — | 皮质醇的主要受体，调节负反馈 |
| 盐皮质激素受体 | Mineralocorticoid Receptor (MR) | — | 对皮质醇高亲和力受体，调节基础节律 |

### 1.2 HPA轴激活级联

```
应激源感知
      ↓
杏仁核、海马、前额叶皮质整合
      ↓
下丘脑室旁核(PVN)
      ↓
CRH + AVP 释放
      ↓
垂体前叶促皮质激素细胞
      ↓
ACTH 释放入血
      ↓
肾上腺皮质束状带
      ↓
皮质醇合成与分泌
      ↓
[负反馈] → 海马、下丘脑、垂体
```

### 1.3 皮质醇功能与作用

| 功能系统 | 急性效应 | 慢性过度效应 |
|:---------|:---------|:-------------|
| **代谢** | 糖原分解、糖异生、蛋白质分解、脂肪分解 | 内脏脂肪堆积、胰岛素抵抗、高血糖 |
| **免疫** | 急性免疫增强后抑制 | 慢性免疫抑制、感染风险增加 |
| **心血管** | 增加心输出量、血管张力 | 高血压、内皮功能障碍 |
| **中枢神经** | 警觉性增强、情绪记忆巩固 | 海马损伤、抑郁、认知障碍 |
| **骨骼肌肉** | 能量动员 | 骨质疏松、肌肉萎缩 |
| **生殖** | 短期抑制 | 性腺功能低下、不孕 |

### 1.4 慢性压力导致的HPA轴改变

| 改变类型 | 表现 | 机制 | 临床意义 |
|:---------|:-----|:-----|:---------|
| **皮质醇节律紊乱** | 晨峰降低、傍晚升高、夜间皮质醇升高 | 中枢调控失调 | 睡眠障碍、疲劳 |
| **负反馈受损** | 地塞米松抑制试验阳性率降低 | GR下调、海马损伤 | HPA持续激活 |
| **基础皮质醇改变** | 初期升高，后期可能低下("倦怠") | 适应性改变或耗竭 | 抑郁vs疲劳综合征 |
| **皮质醇觉醒反应(CAR)改变** | 增强或钝化 | 取决于压力类型和阶段 | 评估工具 |
| **CRH过度表达** | 中枢CRH持续升高 | 慢性应激驱动 | 焦虑、抑郁 |

### 1.5 皮质醇节律指标

| 指标 | 英文 | 测量方法 | 临床意义 |
|:-----|:-----|:---------|:---------|
| **皮质醇觉醒反应(CAR)** | Cortisol Awakening Response | 醒后30-45分钟皮质醇变化 | HPA反应性指标 |
| **昼夜斜率** | Diurnal Slope | (晨峰-睡前)/时间 | 斜率变平与不良健康相关 |
| **曲线下面积(AUC)** | Area Under Curve | 一天多点采样计算 | 总皮质醇暴露量 |
| **平均皮质醇** | Mean Cortisol | 多点平均 | 整体水平 |
| **发际皮质醇** | Hair Cortisol | 头发皮质醇浓度 | 过去1-3个月的累积皮质醇 |

## 2. SAM系统：急性应激通路 (SAM System: Acute Stress Pathway)

### 2.1 SAM系统术语表

| 中文术语 | 英文术语 | 词源 | 定义 |
|:---------|:---------|:-----|:-----|
| 交感神经系统 | Sympathetic Nervous System | 希腊语 συμπάθεια (sympátheia, 共感) | 激活性自主神经分支 |
| 肾上腺髓质 | Adrenal Medulla | 拉丁语 *medulla* (髓) | 分泌儿茶酚胺的肾上腺内部 |
| 肾上腺素 | Epinephrine/Adrenaline | 希腊语 ἐπί (epí, 在上) + νεφρός (nephrós, 肾) | 主要的急性应激激素 |
| 去甲肾上腺素 | Norepinephrine/Noradrenaline | *nor-* (正常) + epinephrine | 交感神经递质和激素 |
| 蓝斑核 | Locus Coeruleus | 拉丁语 *locus* (地点) + *caeruleus* (蓝色) | 脑干去甲肾上腺素能核团 |
| 儿茶酚胺 | Catecholamines | — | 肾上腺素、去甲肾上腺素、多巴胺的总称 |
| 战斗-逃跑反应 | Fight-or-Flight Response | — | 急性威胁时的SAM激活模式 |

### 2.2 SAM vs HPA时间特征对比

| 特征 | SAM系统 | HPA轴 |
|:-----|:--------|:------|
| **激活速度** | 秒级(数秒) | 分钟级(数分钟) |
| **峰值时间** | 数秒至1-2分钟 | 15-30分钟 |
| **持续时间** | 数分钟 | 数小时 |
| **主要效应器** | 心血管、呼吸、肌肉 | 代谢、免疫、认知 |
| **主要激素** | 肾上腺素、去甲肾上腺素 | 皮质醇 |
| **神经通路** | 直接神经支配+激素 | 神经内分泌级联 |

### 2.3 慢性压力对SAM系统的影响

| 改变 | 表现 | 后果 |
|:-----|:-----|:-----|
| **基础交感张力升高** | 静息心率增加、血压升高、HRV降低 | 心血管负荷增加 |
| **β受体下调** | 对儿茶酚胺反应减弱 | 适应性降低 |
| **α受体敏化** | 血管反应性改变 | 高血压风险 |
| **蓝斑核功能改变** | 警觉过度或麻木 | 注意力和情绪问题 |

## 4. 端粒与细胞老化 (Telomeres & Cellular Aging)

### 4.1 端粒术语表

| 中文术语 | 英文术语 | 词源/定义 |
|:---------|:---------|:----------|
| 端粒 | Telomere | 希腊语 τέλος (télos, 末端) + μέρος (méros, 部分) - 染色体末端保护结构 |
| 端粒酶 | Telomerase | 延长端粒的逆转录酶 |
| 端粒缩短 | Telomere Attrition | 每次细胞分裂端粒丢失约50-200bp |
| 复制性衰老 | Replicative Senescence | 端粒缩短导致的细胞分裂停止 |
| 端粒酶活性 | Telomerase Activity | 端粒酶延长端粒的能力 |

### 4.2 压力与端粒的关系

| 发现 | 研究者 | 关键结果 |
|:-----|:-------|:---------|
| **照护者端粒缩短** | Epel等, 2004 | 照护压力大的母亲端粒显著更短，相当于9-17年额外老化 |
| **童年逆境与端粒** | 多项研究 | 早期逆境与成年后较短端粒相关 |
| **抑郁与端粒** | 元分析 | 重性抑郁障碍患者端粒更短 |
| **社会地位与端粒** | 多项研究 | 低社会经济地位与端粒缩短相关 |
| **压力认知与端粒** | Epel等 | 对压力的负面评价预测端粒缩短 |

### 4.3 压力影响端粒的机制

| 机制 | 描述 |
|:-----|:-----|
| **氧化应激** | 皮质醇和炎症增加氧化损伤，端粒对氧化敏感 |
| **炎症** | 炎性细胞因子促进细胞分裂消耗端粒 |
| **端粒酶抑制** | 皮质醇可能抑制端粒酶活性 |
| **表观遗传** | 端粒区域组蛋白修饰改变 |

### 4.4 干预与端粒

| 干预 | 对端粒的影响 | 证据质量 |
|:-----|:-------------|:---------|
| **综合生活方式改变** | 端粒酶活性增加 | Ornish等, 随机对照试验 |
| **冥想** | 端粒酶活性增加 | 多项研究，效应量中等 |
| **运动** | 端粒维护改善 | 流行病学和干预证据 |
| **营养** | 地中海饮食与端粒长度正相关 | 观察性研究 |

## 5. 表观遗传学机制 (Epigenetic Mechanisms)

### 5.1 表观遗传术语表

| 中文术语 | 英文术语 | 定义 |
|:---------|:---------|:-----|
| 表观遗传学 | Epigenetics | 希腊语 ἐπί (epí, 在上) - 不改变DNA序列的可遗传基因表达变化 |
| DNA甲基化 | DNA Methylation | 向DNA添加甲基，通常抑制基因表达 |
| 组蛋白修饰 | Histone Modification | 改变染色质结构和基因可及性 |
| 组蛋白乙酰化 | Histone Acetylation | 通常增加基因表达 |
| 组蛋白甲基化 | Histone Methylation | 可增加或抑制基因表达 |
| 染色质重塑 | Chromatin Remodeling | 改变DNA-组蛋白相互作用 |
| 微RNA | MicroRNA (miRNA) | 调控基因表达的小RNA |
| 表观遗传时钟 | Epigenetic Clock | 基于DNA甲基化的生物年龄预测 |

### 5.2 压力相关表观遗传改变

| 靶基因 | 表观遗传改变 | 功能后果 | 研究背景 |
|:-------|:-------------|:---------|:---------|

| **FKBP5** | 甲基化降低，表达增加 | GR功能改变，应激反应敏化 | 创伤暴露、PTSD |
| **BDNF** | 启动子甲基化增加 | 神经营养因子减少，可塑性降低 | 慢性应激、抑郁 |
| **SLC6A4 (5-HTT)** | 甲基化改变 | 5-HT功能改变 | 早期逆境 |
| **OXTR** | 启动子甲基化 | 催产素功能改变，社会功能影响 | 早期逆境 |
| **促炎基因** | 乙酰化增加 | 炎症反应增强 | 慢性应激 |

### 5.3 跨代表观遗传传递

|:---------|:-----|:-----|

| **父系传递** | 精子表观遗传改变 | 动物研究，早期人类证据 |
| **跨代传递** | 影响延续至孙代 | 动物模型，人类证据有限 |

### 5.4 表观遗传年龄加速

| 概念 | 描述 | 临床意义 |
|:-----|:-----|:---------|
| **Horvath时钟** | 基于353个CpG位点的年龄预测 | 金标准表观遗传时钟 |
| **GrimAge** | 预测死亡风险的表观遗传时钟 | 更强的健康预测 |
| **表观遗传年龄加速** | 表观遗传年龄 > 实际年龄 | 与慢性压力、健康不良相关 |

## 6. 神经可塑性改变 (Neuroplasticity Changes)

### 6.1 压力敏感脑区结构改变

| 脑区 | 慢性压力改变 | 分子机制 | 功能后果 |
|:-----|:-------------|:---------|:---------|
| **海马** | 体积缩小5-10%，CA3树突萎缩，齿状回神经发生减少 | 皮质醇毒性、BDNF降低、谷氨酸兴奋毒性 | 记忆损害、HPA失调 |
| **杏仁核** | 体积增大，基底外侧核树突增生 | 去甲肾上腺素增强、BDNF局部增加 | 恐惧敏化、焦虑增强 |
| **前额叶皮质** | 体积减少、树突简化、突触减少 | 皮质醇、多巴胺失调 | 执行功能、情绪调节受损 |
| **纹状体** | 可塑性改变 | 多巴胺失调 | 习惯形成、奖赏处理改变 |
| **前扣带回** | 结构和功能改变 | — | 情绪整合、疼痛处理改变 |

### 6.2 神经发生与压力

| 因素 | 对成年神经发生的影响 |
|:-----|:--------------------|
| **急性压力** | 短暂抑制 |
| **慢性压力** | 持续抑制 |
| **皮质醇** | 抑制祖细胞增殖 |
| **BDNF** | 减少→神经发生减少 |
| **炎症** | 抑制神经发生 |
| **运动** | 对抗压力抑制作用 |
| **抗抑郁药** | 促进神经发生 |

### 6.3 突触可塑性改变

| 可塑性类型 | 急性压力 | 慢性压力 |
|:-----------|:---------|:---------|
| **LTP (长时程增强)** | 海马CA1增强 | 受损 |
| **LTD (长时程抑制)** | 变化不一 | 增强 |
| **突触修剪** | — | 前额叶过度修剪 |
| **树突棘密度** | — | 海马、前额叶降低；杏仁核增加 |

### 6.4 可逆性与干预窗口

| 阶段 | 可逆性 | 干预时机 |
|:-----|:-------|:---------|
| **早期慢性** | 高度可逆 | 最佳干预窗口 |
| **中期慢性** | 部分可逆 | 需要更强干预 |
| **长期慢性** | 有限可逆 | 可能需要长期干预 |
| **发育关键期** | 影响持久 | 早期干预至关重要 |

## 7. 自主神经失调 (Autonomic Dysregulation)

| 中文术语 | 英文术语 | 定义 |

| 迷走张力 | Vagal Tone | 迷走神经对心脏的调节作用 |

### 7.2 多迷走神经理论与压力反应

| 系统 | 英文 | 进化层次 | 功能 | 慢性压力影响 |
|:-----|:-----|:---------|:-----|:-------------|
| **腹侧迷走复合体** | Ventral Vagal Complex | 最新(哺乳动物) | 社会参与、平静、安全 | 功能降低 |
| **交感神经系统** | Sympathetic NS | 中间(脊椎动物) | 动员、战斗-逃跑 | 过度激活 |
| **背侧迷走复合体** | Dorsal Vagal Complex | 最古老 | 不动化、关闭、冻结 | 可能被激活 |

### 7.3 HRV指标与压力

| 指标 | 域 | 反映 | 慢性压力变化 |
|:-----|:---|:-----|:-------------|
| **SDNN** | 时域 | 总体HRV | 降低 |
| **RMSSD** | 时域 | 副交感活动 | 降低 |
| **pNN50** | 时域 | 副交感活动 | 降低 |
| **HF** | 频域 | 迷走活动 | 降低 |
| **LF** | 频域 | 混合(可能交感为主) | 相对增加 |
| **LF/HF** | 频域 | 交感-副交感平衡 | 升高 |

### 7.4 迷走神经功能下降的后果

| 系统 | 后果 |
|:-----|:-----|
| **心血管** | 心率加快、血压升高、心血管疾病风险 |
| **免疫** | 炎症反射减弱、慢性炎症 |
| **消化** | 蠕动减慢、消化问题 |
| **情绪** | 情绪调节能力下降 |
| **社交** | 社会参与困难 |
| **恢复** | 应激后恢复变慢 |

## 8. 代谢改变 (Metabolic Changes)

### 8.1 压力代谢综合征机制

| 机制 | 描述 | 后果 |
|:-----|:-----|:-----|
| **皮质醇代谢效应** | 促进糖异生、蛋白质分解、脂肪重分布 | 高血糖、肌肉萎缩、中心性肥胖 |
| **胰岛素抵抗** | 皮质醇对抗胰岛素作用 | 糖代谢紊乱、2型糖尿病风险 |
| **内脏脂肪堆积** | 皮质醇促进腹部脂肪沉积 | 代谢综合征、炎症 |
| **瘦素抵抗** | 对饱感信号不敏感 | 过度进食、肥胖 |
| **压力性进食** | 高皮质醇促进对高热量食物偏好 | 体重增加 |
| **AMPK功能改变** | 能量感应通路受损 | 代谢调节障碍 |

### 8.2 肠脑轴失调

| 组成部分 | 压力影响 | 后果 |
|:---------|:---------|:-----|
| **肠道微生物组** | 多样性降低、有害菌增加 | 炎症、神经递质改变 |
| **肠道屏障** | 通透性增加("肠漏") | 内毒素入血、全身炎症 |
| **迷走神经信号** | 传导减弱 | 肠脑沟通障碍 |
| **肠道内分泌** | 激素分泌改变 | 食欲和情绪调节异常 |
| **肠道神经系统** | 功能改变 | IBS等功能性胃肠病 |

## 9. 昼夜节律紊乱 (Circadian Disruption)

### 9.1 压力与昼夜节律相互作用

| 方面 | 描述 |
|:-----|:-----|
| **HPA-昼夜节律整合** | 皮质醇本身有明显昼夜节律，慢性压力破坏此节律 |
| **时钟基因与压力基因** | 相互调节，压力影响时钟基因表达 |
| **睡眠-觉醒周期** | 慢性压力导致睡眠紊乱，进一步恶化节律 |
| **光照敏感性** | 压力可能改变对光照的节律重置反应 |
| **分子时钟** | 外周组织时钟与中枢时钟解耦联 |

### 9.2 昼夜节律紊乱的后果

| 系统 | 影响 |
|:-----|:-----|
| **代谢** | 葡萄糖耐量下降、肥胖风险 |
| **免疫** | 免疫功能紊乱 |
| **认知** | 注意力、记忆力下降 |
| **情绪** | 抑郁、焦虑风险增加 |
| **心血管** | 血压调节异常 |
| **癌症** | 某些癌症风险可能增加 |

## 10. 氧化应激与线粒体功能 (Oxidative Stress & Mitochondrial Function)

### 10.1 压力诱导的氧化应激

| 来源 | 机制 |
|:-----|:-----|
| **线粒体呼吸增加** | 代谢需求增加导致ROS产生增加 |
| **炎症** | 炎症细胞产生ROS |
| **儿茶酚胺代谢** | 代谢产物具有促氧化作用 |
| **抗氧化系统耗竭** | 持续应激消耗抗氧化储备 |
| **铁代谢改变** | 游离铁增加Fenton反应 |

### 10.2 线粒体功能改变

| 改变 | 描述 | 后果 |
|:-----|:-----|:-----|
| **线粒体DNA损伤** | 氧化损伤累积 | 功能下降 |
| **线粒体生物发生减少** | PGC-1α表达降低 | 线粒体数量减少 |
| **膜电位改变** | 电子传递链功能下降 | ATP产生减少 |
| **线粒体动力学改变** | 分裂/融合失衡 | 质量控制受损 |
| **线粒体自噬异常** | 清除受损线粒体能力下降 | 功能障碍线粒体累积 |

## 11. 整合模型：异稳态负荷 (Integrated Model: Allostatic Load)

### 11.1 异稳态负荷的四种模式

| 模式 | 描述 | 示例 |
|:-----|:-----|:-----|
| **重复激活** | 对反复出现的应激源持续反应 | 反复的工作截止期限 |
| **缺乏适应** | 对重复应激源无法习惯化 | 每次乘飞机都高度焦虑 |
| **反应延长** | 应激结束后恢复延迟 | 反刍思维导致皮质醇持续升高 |
| **反应不足** | 一个系统功能不足导致其他系统代偿性过度激活 | HPA钝化导致炎症增加 |

### 11.2 异稳态负荷指标体系(McEwen)

| 类别 | 生物标志物 | 意义 |
|:-----|:-----------|:-----|
| **HPA轴** | 皮质醇(12小时尿)、DHEA-S | 神经内分泌调节 |
| **交感神经** | 肾上腺素、去甲肾上腺素(12小时尿) | 交感激活水平 |
| **心血管** | 收缩压、舒张压 | 心血管负荷 |
| **代谢** | 糖化血红蛋白、HDL、总胆固醇/HDL比、腰臀比 | 代谢状态 |
| **炎症** | CRP、纤维蛋白原、IL-6 | 炎症水平 |

## 12. 专业视角 (Professional Perspectives)

### 神经内分泌学视角

> **Bruce McEwen (1938-2020):** "异稳态的概念强调，我们的身体通过持续变化来维持表面的稳定。但这种适应是有代价的——我称之为异稳态负荷。理解这一机制对于预防慢性病至关重要。"

### 心理神经免疫学视角

> **Robert Ader (1932-2011):** "心理神经免疫学揭示了心理状态如何通过神经和内分泌途径影响免疫功能。慢性压力的免疫后果是理解压力-疾病关系的关键。"

### 细胞老化视角

> **Elizabeth Blackburn (诺贝尔奖得主):** "端粒就像鞋带末端的塑料套，保护染色体不受损伤。慢性压力加速端粒缩短，这是压力如何'进入细胞'导致老化的一个重要机制。"

### 表观遗传学视角

> **Michael Meaney:** "母亲的养育行为可以通过表观遗传机制改变后代的应激反应系统。这表明环境经历可以在分子水平上留下持久印记，但这种印记也可能通过干预而逆转。"
