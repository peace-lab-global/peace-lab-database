# 推手体系 | Push Hands System

> "由招熟而渐悟懂劲，由懂劲而阶及神明"——推手是太极拳从"形"走向"劲"的桥梁。

---

## 一、推手的本质与目的

### 1.1 推手的定义

太极推手（Tui Shou / Push Hands）是两人按照太极原理进行的接触性对抗训练，通过肢体接触来感知、化解、引导对方力量。

### 1.2 推手的四大功能

| 功能 | 说明 |
|------|------|
| 听劲 | 通过接触感知对方力量的方向、大小、意图 |
| 化劲 | 化解对方的来力，使之落空 |
| 发劲 | 在合适时机将力量发出对方 |
| 粘黏 | 保持接触不脱离，始终控制对方 |

### 1.3 推手的核心原则

- **不丢不顶**: 不脱离接触（丢）、不硬抗对抗（顶）
- **舍己从人**: 不主动强加力量，顺应对方的力
- **引进落空**: 引导对方力量使其失去目标
- **借力打力**: 利用对方的力量反击

## 二、四正手与四隅手

### 2.1 四正手（Four Primary Hands）

四正手是推手的核心技术框架：

| 劲法 | 方向 | 特征 | 五行对应 |
|------|------|------|---------|
| 掤（Peng） | 向前上方 | 弹性撑托 | 金 |
| 捋（Lu） | 向后侧方 | 牵引化力 | 木 |
| 挤（Ji） | 向前 | 挤压推进 | 水 |
| 按（An） | 向前下方 | 下按沉降 | 火 |

**四正手循环:**
```
甲掤 → 乙捋 → 甲挤 → 乙按 → 甲掤 → 循环...
```

### 2.2 四隅手（Four Corner Hands）

| 劲法 | 特征 | 应用 |
|------|------|------|
| 采（Cai） | 向下采拿 | 抓拿对方手腕/肘向下拽 |
| 挒（Lie） | 旋转撕裂 | 双手反方向旋转 |
| 肘（Zhou） | 肘击 | 近距离以肘攻击 |
| 靠（Kao） | 肩靠 | 极近距离以肩/身冲撞 |

## 三、听劲训练体系

### 3.1 什么是听劲

听劲（Ting Jin）是通过肢体接触感知对方力量信息的能力——如同用皮肤"听"到对方的意图。

### 3.2 听劲的三个层次

| 层次 | 名称 | 能力 |
|------|------|------|
| 初级 | 皮肤听劲 | 感知力量的方向与大小 |
| 中级 | 骨节听劲 | 感知对方的重心与结构 |
| 高级 | 意念听劲 | 在接触前就感知对方意图 |

### 3.3 听劲训练方法

**闭眼推手:**
- 双方闭眼进行推手
- 完全依赖触觉感知
- 训练细腻的触觉灵敏度

**慢速推手:**
- 极慢速进行推手循环
- 关注每一瞬间的力量变化
- 如同放慢电影观看每一帧

**单手推手:**
- 仅用一只手接触
- 要求更高的感知精度
- 训练一手化发的能力

## 四、推手形式进阶

### 4.1 定步推手

- 双方脚位固定、不移动
- 专注于上肢化发技术
- 入门必修

### 4.2 活步推手

- 允许步法移动
- 进退顾盼定五步配合
- 中级训练

### 4.3 大履推手

- 以四个斜角方向大范围移动
- 八门五步全部应用
- 最接近实战的推手形式

### 4.4 乱环推手（自由推手）

- 无固定模式、自由发挥
- 考验综合实力
- 高级训练/竞赛形式

## 五、推手竞赛规则

### 5.1 国内竞赛体系

**评分标准:**
| 得分方式 | 分值 | 说明 |
|---------|------|------|
| 出圈 | 1分 | 对方脚踏出比赛圈 |
| 倒地 | 2分 | 对方倒地（非故意摔） |
| 移动 | 1分 | 对方脚步移动超规定 |
| 警告 | 对手1分 | 犯规动作 |

**禁止动作:**
- 抓衣服、抱摔、踢打
- 故意使用关节技
- 超出推手范畴的暴力行为

### 5.2 国际竞赛

- 国际武术联合会（IWUF）推手竞赛规则
- 按体重分级
- 圆形比赛场地（直径4-6米）

## 六、推手修炼的常见误区

| 误区 | 正确理解 |
|------|---------|
| "推手=比力气" | 推手比的是感知与技术 |
| "一定要赢" | 推手是学习过程、非争胜负 |
| "只用蛮力" | 松柔是推手的第一要求 |
| "推手可以替代拳架" | 拳架是根基、推手是应用 |
| "急于发劲" | 先学会化劲、发劲自然来 |
