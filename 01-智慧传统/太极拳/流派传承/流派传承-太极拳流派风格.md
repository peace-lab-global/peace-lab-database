## 陈式太极拳

### 历史传承

- **创始人**: 陈王廷（约1600-1680），河南温县陈家沟
- **理论来源**: 融合戚继光《拳经》三十二势、导引吐纳术、经络学说、阴阳五行
- **传承谱系**: 陈王廷 → 陈长兴(大架) / 陈有本(小架) → 陈发科(北京传拳) → 陈照丕/陈照奎 → 陈正雷/陈小旺/王西安/朱天才("四大金刚")

### 技术特征

| 特征 | 描述 |
|------|------|
| 缠丝劲 | 螺旋运动贯穿全身，顺缠/逆缠交替 |
| 快慢相间 | 柔缓与爆发交替，有蓄有发 |
| 松活弹抖 | 发力如弹簧弹射，整体协调 |
| 刚柔相济 | 阴阳互含，一路柔多刚少，二路刚多柔少 |
| 低架运动 | 重心较低，腿部负荷大 |

### 代表套路

| 套路 | 式数 | 特点 | 难度 |
|------|------|------|------|
| 陈式一路（老架） | 74/75式 | 柔多刚少、缓慢舒展 | ★★★☆☆ |
| 陈式二路（炮捶） | 43式 | 刚多柔少、发力跳跃多 | ★★★★★ |
| 陈式小架 | 64式 | 紧凑细腻、内劲为主 | ★★★★☆ |
| 陈式56式 | 56式 | 竞赛套路、规范化 | ★★★★☆ |
| 陈式太极剑 | 49式 | 剑法与缠丝劲结合 | ★★★★☆ |

### 缠丝劲详解

缠丝劲（Silk-Reeling Energy）是陈式太极拳的核心特色：

- **基本分类**: 顺缠（拇指内旋向小指方向）、逆缠（小指外旋向拇指方向）
- **五对缠丝**: 出入缠、进退缠、上下缠、左右缠、顺逆缠
- **单式训练**: 正面缠丝 → 侧面缠丝 → 双手缠丝 → 配合步法缠丝
- **生理机制**: 螺旋运动增强筋膜弹性、关节灵活性和本体感觉

## 杨式太极拳

### 历史传承

- **创始人**: 杨露禅（1799-1872），河北永年人，师从陈长兴
- **发展**: 杨露禅 → 杨班侯(小架)/杨健侯(中架) → 杨澄甫(大架定型)
- **里程碑**: 杨露禅入京授拳，打遍北京无敌手，人称"杨无敌"
- **杨澄甫贡献**: 定型85式大架，编著《太极拳体用全书》，奠定杨式现代传播基础

### 技术特征

| 特征 | 描述 |
|------|------|
| 舒展大方 | 动作伸展、姿态优美、大开大合 |
| 柔和均匀 | 速度均匀、动作连绵、一气呵成 |
| 绵里藏针 | 外柔内刚、暗含劲力 |
| 虚实分明 | 重心分配清晰、单腿承重 |
| 中正安舒 | 身体中正、不偏不倚 |

### 杨澄甫"太极拳十要"

| 序号 | 要领 | 核心 |
|------|------|------|
| 1 | 虚灵顶劲 | 头部轻灵上领 |
| 2 | 含胸拔背 | 胸微含、背微拔 |
| 3 | 松腰 | 腰为主宰 |
| 4 | 分虚实 | 阴阳分明 |
| 5 | 沉肩坠肘 | 肩松肘沉 |
| 6 | 用意不用力 | 意到气到 |
| 7 | 上下相随 | 一动无有不动 |
| 8 | 内外相合 | 神形合一 |
| 9 | 相连不断 | 绵绵不绝 |
| 10 | 动中求静 | 以静制动 |

### 代表套路

| 套路 | 式数 | 特点 |
|------|------|------|
| 杨式85式（传统大架） | 85式 | 传统完整版、约25-30分钟 |
| 24式简化太极拳 | 24式 | 1956年国家编定、全球最普及 |
| 42式综合太极拳 | 42式 | 竞赛套路、融合多派特色 |
| 48式太极拳 | 48式 | 中级健身套路 |
| 杨式太极剑 | 32/42式 | 剑法优美、适合表演 |

## 武式太极拳

### 历史传承

- **创始人**: 武禹襄（1812-1880），河北永年望族
- **特色**: 武禹襄为读书人，重视理论研究，留下大量拳论
- **传承**: 武禹襄 → 李亦畬 → 郝为真 → 郝月如 → 郝少如
- **重要文献**: 武禹襄发现王宗岳《太极拳论》，并著《十三势说略》等

### 技术特征

| 特征 | 描述 |
|------|------|
| 紧凑严密 | 步小架高、动作紧凑 |
| 开合有序 | 一开一合、开中有合、合中有开 |
| 尾闾中正 | 严格要求身法端正 |
| 内外兼修 | 重视内劲修炼、理法并重 |
| 起承转合 | 每势清晰分段、层次分明 |

### 核心拳论

- 《十三势说略》: "以心行气，务令沉着"
- 《太极拳解》: "身虽动，心贵静"
- 《十三势行功心解》: "以意导气，以气运身"
- 《四字秘诀》: 敷、盖、对、吞

## 吴式太极拳

### 历史传承

- **创始人**: 全佑（1834-1902），满族旗人，师从杨班侯
- **定型**: 吴鉴泉（1870-1942），全佑之子，定型吴式
- **传播中心**: 上海（鉴泉太极拳社，1935年成立）
- **当代传承**: 吴公藻/吴公仪/吴英华 → 马岳梁 等

### 技术特征

| 特征 | 描述 |
|------|------|
| 柔化细腻 | 最柔软的太极流派、化劲精妙 |
| 斜中寓正 | 身体微倾但重心稳定 |
| 轻灵圆活 | 转换灵活、无棱角 |
| 川字步型 | 两脚平行、间距较窄 |
| 揉搓劲 | 独特的化劲发劲方式 |

## 孙式太极拳

### 历史传承

- **创始人**: 孙禄堂（1860-1933），"虎头少保"
- **融合三家**: 形意拳（师郭云深）+ 八卦掌（师程廷华）+ 太极拳（师郝为真）
- **独创体系**: 以形意之理为体、八卦之法为用、太极之功为基
- **著作**: 《太极拳学》《拳意述真》等五部武学经典

### 技术特征

| 特征 | 描述 |
|------|------|
| 活步太极 | 进退相随、步活身灵 |
| 开合鼓荡 | 每式以开合为节奏 |
| 高架轻灵 | 步高架活、适合老弱 |
| 三拳合一 | 融合形意/八卦/太极精华 |
| 转换灵活 | 以八卦步法带动方向变化 |

### 代表特色——进步必跟、退步必撤

孙式太极拳步法独特，前进时后脚必跟半步，后退时前脚必撤半步，使重心始终在两脚之间，灵活稳定。

## 流派选择指南

### 按需求推荐

| 需求 | 推荐流派 | 理由 |
|------|---------|------|
| 初学入门 | 杨式/24式 | 最普及、师资最多、动作舒展易学 |
| 追求技击 | 陈式 | 保留最多实战技法、发力训练完整 |
| 年长/体弱 | 孙式 | 高架活步、身体负荷小 |
| 内功修炼 | 武式 | 理论深厚、内劲修炼精细 |
| 柔化艺术 | 吴式 | 柔软细腻、化劲精妙 |
| 全面武术 | 孙式 | 三拳合一、视野最广 |

### 选择决策要素

1. **身体条件**: 膝关节状况、柔韧性、体力
2. **学习目标**: 健身/技击/修身/竞赛
3. **师资条件**: 当地可获得的优质老师
4. **时间投入**: 传统套路需更多学习时间
5. **个人偏好**: 刚柔偏好、审美倾向
