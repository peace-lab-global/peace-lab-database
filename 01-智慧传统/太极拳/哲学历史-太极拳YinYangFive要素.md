# 阴阳五行与太极 | Yin-Yang, Five Elements & Tai Chi

> 阴阳五行不仅是中医的理论基石，也是太极拳身法、劲法、步法的内在逻辑框架。

---

## 一、阴阳理论在太极拳中的系统体现

### 1.1 身体层面的阴阳划分

| 阴阳维度 | 阳 | 阴 |
|---------|---|---|
| 身体部位 | 上体、背部、外侧 | 下体、腹部、内侧 |
| 运动状态 | 动、开、升、伸 | 静、合、降、屈 |
| 力量表现 | 刚、快、显 | 柔、慢、隐 |
| 重心分布 | 实腿（阳） | 虚腿（阴） |
| 呼吸 | 呼气（阳） | 吸气（阴） |
| 意念 | 外放、专注 | 内收、散漫 |

### 1.2 拳架中的阴阳转换规律

- **虚实转换**: 每个步法都伴随虚实（阴阳）的交替，重心从一腿移至另一腿
- **开合相随**: 手臂展开（阳）时胸廓打开（阳），但下肢稳固扎根（阴）
- **蓄发循环**: 蓄劲为阴（收、吸、屈）→发劲为阳（放、呼、伸）
- **快慢相间**: 杨式匀速缓慢（阴中含阳），陈式快慢交替（阴阳分明）

### 1.3 阴阳平衡的评估标准

1. **身法中正**: 不偏不倚，头顶百会与会阴成一线
2. **虚实分明**: 重心清楚，非双重（非两腿均实）
3. **上下相随**: 手动腰随、腰动手随，内外合一
4. **动静有常**: 动中有静意、静中有动势

## 二、五行理论与太极拳的对应

### 2.1 五行与五拳（五种劲法）对应

| 五行 | 对应劲法 | 特性 | 拳式举例 |
|------|---------|------|---------|
| 金 | 掤劲（Peng） | 如金属弹性、向外撑 | 掤手上势 |
| 木 | 捋劲（Lu） | 如树枝柔韧、引化 | 捋势 |
| 水 | 挤劲（Ji） | 如水渗透、向前挤 | 挤势 |
| 火 | 按劲（An） | 如火蔓延、向下按 | 按势 |
| 土 | 中定（Zhongding） | 如大地承载、稳定 | 无极桩/中定 |

### 2.2 五行生克与推手策略

**五行相生（劲法递进）:**
```
掤（金）生挤（水）→ 挤（水）生捋（木）→ 捋（木）生按（火）→ 按（火）生中定（土）→ 中定（土）生掤（金）
```

**五行相克（劲法克制）:**
```
掤（金）克捋（木）→ 捋（木）克中定（土）→ 中定（土）克挤（水）→ 挤（水）克按（火）→ 按（火）克掤（金）
```

**推手应用**: 当对方用掤劲时，以按劲克之；对方用挤劲时，以掤劲克之——循环无端，变化无穷。

### 2.3 五行与五方步法

| 五行 | 方位 | 步法 |
|------|------|------|
| 金 | 西 | 右顾 |
| 木 | 东 | 左盼 |
| 水 | 北 | 后退 |
| 火 | 南 | 前进 |
| 土 | 中 | 中定 |

合称"八门五步"：掤捋挤按采挒肘靠（八门）+ 进退顾盼定（五步）= 太极十三势。

## 三、脏腑经络与太极修炼

### 3.1 五行-脏腑-拳法对应表

| 五行 | 脏 | 腑 | 情志 | 拳法功效 |
|------|---|---|------|---------|
| 木 | 肝 | 胆 | 怒 | 疏肝理气、舒展筋脉 |
| 火 | 心 | 小肠 | 喜 | 宁心安神、调和血脉 |
| 土 | 脾 | 胃 | 思 | 健脾和胃、稳固中焦 |
| 金 | 肺 | 大肠 | 忧 | 宣肺理气、增强呼吸 |
| 水 | 肾 | 膀胱 | 恐 | 固肾壮腰、培补元气 |

### 3.2 太极套路对经络的调理

- **手臂三阴三阳经**: 云手、揽雀尾等手臂旋转动作疏通手三阴/三阳经
- **足三阴三阳经**: 弓步、虚步转换刺激足部经络
- **督脉**: 虚灵顶劲、含胸拔背激活督脉阳气
- **任脉**: 气沉丹田、松腰落胯疏通任脉
- **带脉**: 腰部旋转（如云手）环绕刺激带脉

### 3.3 练拳时辰与经络流注

| 时辰 | 经络 | 练拳建议 |
|------|------|---------|
| 卯时（5-7） | 大肠经 | 晨练最佳，宣发肺气 |
| 辰时（7-9） | 胃经 | 饭后30分钟不宜练 |
| 巳时（9-11） | 脾经 | 上午练习精力充沛 |
| 酉时（17-19） | 肾经 | 傍晚练习补肾养气 |
| 亥时（21-23） | 三焦经 | 睡前宜练静功/站桩 |

## 四、太极十三势的阴阳五行整合

### 4.1 十三势总览

```
太极十三势
├── 八门（手法 / 空间方位）
│   ├── 四正：掤（金）、捋（木）、挤（水）、按（火）
│   └── 四隅：采、挒、肘、靠
└── 五步（步法 / 运动方向）
    ├── 进（火）
    ├── 退（水）
    ├── 顾（金）
    ├── 盼（木）
    └── 定（土）
```

### 4.2 十三势的动态平衡模型

- 每一势都包含阴阳两面：有形（阳）的动作 + 无形（阴）的意念
- 八门五步的组合形成 8×5 = 40 种基本变化
- 实际应用中变化无穷，如同五行生克的排列组合

|---------|-------------|------------|
| 高血压（肝阳上亢） | 阳亢、木旺 | 多练降按势、气沉丹田 |

| 消化不良（脾虚） | 土弱 | 加强中定练习、腰部旋转 |
| 慢性疲劳（肾气虚） | 水不足 | 强化下盘、腰部动作、站桩 |
| 焦虑（心肺气虚） | 金弱火扰 | 深呼吸配合舒缓套路 |
