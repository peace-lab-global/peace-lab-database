## 气功总览

### 定义

气功（Qigong）是中国传统的身心锻炼方法，通过调身（姿势）、调息（呼吸）、调心（意念）三调合一，达到增强体质、防治疾病、延年益寿的目的。

### 气功分类

| 分类维度 | 类型 | 代表功法 |
|---------|------|---------|
| 按运动形式 | 静功 | 站桩、坐功、卧功 |
| | 动功 | 八段锦、五禽戏、六字诀 |
| | 动静结合 | 太极气功、导引术 |
| 按传承体系 | 道家气功 | 内丹术、周天功 |
| | 佛家气功 | 禅定、观想 |
| | 医家气功 | 华佗五禽戏、导引养生功 |
| | 儒家气功 | 静坐修养 |
| | 武术气功 | 硬气功、内功 |

### "气"的多层含义

| 层次 | 含义 | 现代解读 |
|------|------|---------|
| 呼吸之气 | 通过鼻口呼吸的空气 | 氧气交换、呼吸生理 |
| 水谷之气 | 饮食转化的营养精微 | 代谢能量、ATP |
| 元气 | 先天禀赋的生命能量 | 遗传素质、线粒体功能 |
| 经络之气 | 循行于经络的能量 | 生物电、筋膜传导 |
| 内气 | 练功中体内气感 | 本体感觉、血液循环变化 |

## 太极与气功的关系

### 共同点

| 维度 | 太极拳 | 气功 |
|------|--------|------|
| 三调合一 | ✓ 调身调息调心 | ✓ 调身调息调心 |
| 哲学基础 | 道家阴阳 | 道家/佛家/医家 |
| 呼吸训练 | 丹田呼吸 | 多种呼吸法 |
| 意念运用 | 以意导气 | 意守/观想 |
| 松静要求 | 身松心静 | 身松心静 |

### 差异点

| 维度 | 太极拳 | 气功 |
|------|--------|------|
| 主要功能 | 武术+养生+修身 | 养生+治疗+修炼 |
| 运动形式 | 连贯套路为主 | 独立功法/静功为主 |
| 技击含义 | 每个动作含技击 | 一般不含技击 |
| 学习路径 | 套路→推手→散手 | 功法→内功→高级修炼 |
| 难度 | 动作较复杂 | 多数较简单易学 |

### 整合原理

```
气功基础功 → 太极拳套路 → 太极内功
 (培元固本)     (动中练功)     (高级整合)
```

- **气功为体**: 桩功和吐纳培养内气基础
- **太极为用**: 在套路运动中运用内气
- **内功为归**: 太极内功是二者高级整合

## 动功体系

### 八段锦

中国最普及的健身气功之一，八个动作系统锻炼全身：

| 式序 | 名称 | 功效 | 对应脏腑 |
|------|------|------|---------|
| 1 | 两手托天理三焦 | 拉伸全身、调理三焦 | 三焦 |
| 2 | 左右开弓似射雕 | 扩展胸肺、强壮手臂 | 肺 |
| 3 | 调理脾胃须单举 | 健脾和胃、拉伸侧腰 | 脾胃 |
| 4 | 五劳七伤往后瞧 | 缓解疲劳、活化颈椎 | 心/全身 |
| 5 | 摇头摆尾去心火 | 降心火、活腰胯 | 心 |
| 6 | 两手攀足固肾腰 | 壮腰固肾、拉伸后链 | 肾 |
| 7 | 攒拳怒目增气力 | 增加力量、疏肝理气 | 肝 |
| 8 | 背后七颠百病消 | 震动脊柱、调和气血 | 全身 |

- **练习时间**: 每式6-8次，全套约15-20分钟
- **适用人群**: 全年龄段，特别适合办公族和老年人
- **科学证据**: 多项RCT证实改善血脂、血压、腰痛、情绪

## 吐纳呼吸法

### 基础呼吸法

| 方法 | 步骤 | 功效 | 适合阶段 |
|------|------|------|---------|
| 自然呼吸 | 观察呼吸，不干预 | 放松入静 | 入门 |
| 腹式呼吸 | 吸气鼓腹、呼气收腹 | 增加肺活量 | 初级 |
| 逆腹式呼吸 | 吸气收腹、呼气鼓腹 | 增强内压、按摩脏腑 | 中级 |
| 提肛呼吸 | 吸气提肛、呼气放松 | 强化盆底肌 | 中级 |

### 进阶呼吸法

| 方法 | 要领 | 注意事项 |
|------|------|---------|
| 数息法 | 呼吸时默数1-10，循环 | 有助入静、分散杂念 |
| 听息法 | 用心"听"呼吸声 | 庄子所述、极细微 |
| 调息法 | 控制吸呼比为1:2 | 延长呼气、促进放松 |
| 闭息法 | 吸气后短暂屏息 | 需指导、不宜过长 |

## 内功心法

### 意守丹田

| 丹田 | 位置 | 功能 | 意守要领 |
|------|------|------|---------|
| 上丹田 | 印堂/眉间 | 神明之府 | 轻守、不宜长时 |
| 中丹田 | 膻中/两乳间 | 气汇之所 | 配合呼吸、调气 |
| 下丹田 | 脐下1.5寸 | 精气之根 | 最常用、安全 |

### 小周天修炼（概述）

小周天是道家内功核心修炼法，使内气循行于任脉和督脉：

```
百会 (督脉顶)
  ↑
泥丸 → 玉枕 → 夹脊 → 命门 → 尾闾 (督脉/后升)
  ↓
印堂 → 人中 → 膻中 → 中脘 → 丹田 (任脉/前降)
```

- **注意**: 小周天修炼需要有经验的老师指导
- **常见偏差**: 急于求成导致"走火入魔"（气功偏差）
- **安全原则**: 自然为上、不强引导、有不适即停

## 安全注意事项

### 气功偏差（"走火入魔"）

| 类型 | 表现 | 处理 |
|------|------|------|
| 头部不适 | 头痛、头涨、头晕 | 停练、意守涌泉、自然散步 |
| 胸闷气短 | 呼吸不畅、胸部紧迫 | 停练、自然呼吸、拍打膻中 |
| 气冲感 | 体内气感窜动 | 停练、意守丹田、缓慢活动 |
| 幻觉 | 视觉/听觉/体感异常 | 立即停练、睁眼、接地练习 |
| 情绪波动 | 不明原因哭泣/恐惧 | 停练、社交接触、必要时就医 |

### 预防原则

1. **师承指导**: 跟随有经验的老师学习，尤其是内功修炼
2. **循序渐进**: 不急于追求气感或特殊体验
3. **自然为上**: 不强行导引、不刻意追求
4. **知止即止**: 出现不适立即停止
5. **生活规律**: 保持正常作息和社交
