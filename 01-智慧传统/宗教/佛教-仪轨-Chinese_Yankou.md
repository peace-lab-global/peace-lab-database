# 汉传佛教瑜伽焰口施食仪轨 (Chinese Yogacara Flaming Mouth Ritual)

> **瑜伽焰口 (Yogacara Flaming Mouth / Yankou)**：汉传佛教最普及的度亡施食仪轨，源于唐代不空三藏译《佛说救拔焰口饿鬼陀罗尼经》。通过观想、持咒、施食，救度饿鬼道中受苦众生。
>
> 若以一抟之食，起大悲心，称陀罗尼名号，施一饿鬼，能令俱胝饿鬼饱满。 —— 《救拔焰口经》

## 3. 核心真言 (Core Mantras)

### 3.1 变食真言

**梵音：**
```
南无萨嚩怛他誐哆 嚩嚧枳帝
唵 参婆罗参婆罗 吽
```

**罗马转写：**
```
Namo sarva tathāgatāvalokite
Oṃ saṃbhāra saṃbhāra hūṃ
```

**功能：** 将少量食物变为无量供品。

### 3.2 甘露水真言

**梵音：**
```
南无苏噜婆耶 怛他誐哆耶 怛侄他
唵 苏噜苏噜 钵罗苏噜 钵罗苏噜 娑婆诃
```

**功能：** 将清水化为甘露，解除饿鬼饥渴。

### 3.3 开咽喉真言

**梵音：**
```
唵 步步底哩 伽哆哩 怛哆耶哆耶 娑婆诃
```

**功能：** 开启饿鬼咽喉，使其能受用食物。

### 3.4 一字水轮真言

**梵音：**
```
唵 钵罗摩 邻陀宁 娑婆诃
```

### 3.5 四如来名

```
南无多宝如来（能破饿鬼悭吝业）
南无宝胜如来（能破饿鬼贪欲业）
南无妙色身如来（能破饿鬼丑陋形）
南无广博身如来（能破饿鬼 skinny 形）
南无离怖畏如来（能除饿鬼怖畏苦）
南无甘露王如来（能除饿鬼饥渴苦）
```
