# Dzongsar Jamyang Khyentse Rinpoche | 宗萨蒋扬钦哲仁波切

> 宗萨蒋扬钦哲仁波切（Dzongsar Jamyang Khyentse Chökyi Lodrö, 1961—），藏传佛教利美（Rimé）传承的核心人物，当代最具影响力的佛教导师之一。同时也是电影导演，代表作《旅行者与巫师》《嘿玛嘿玛》。

## 作者简介

| 条目 | 内容 |
|------|------|
| 藏文 | འཇམ་དབྱངས་མཁྱེན་བརྩེའི་དབང་པོ |
| 英文 | Dzongsar Jamyang Khyentse Rinpoche |
| 出生 | 1961年，不丹 |
| 传承 | 宁玛派/萨迦派/噶举派（利美传承） |
| 认定 | 被认定为蒋扬钦哲旺波（1820-1892）的转世 |
| 核心事业 | 佛教教育、经典翻译（84000: Reading Room）、电影创作 |
| 电影作品 | 《旅行者与巫师》(2003)、《嘿玛嘿玛》(2016) |
| 佛教学院 | 悉达多本愿会 (Siddhartha's Intent) |

```
宗萨蒋扬钦哲仁波切核心教导
├── 见地 (View) — 最被强调的部分

│   ├── 空性 / 无自性

├── 现代态度
│   ├── 质疑一切（包括佛陀）

│   └── 佛教不是逃避现实
└── 独特贡献

    ├── 打破佛教的"宗教"外壳
    └── 金刚乘智慧的现代诠释
```
