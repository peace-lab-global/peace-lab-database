# 比克拉姆·乔杜里（Bikram Choudhury / बिक्रम चौधरी）

> 热瑜伽帝国的缔造者——从瑜伽冠军到性侵被告的陨落

---

## 一、概述（Overview）

| 项目 | 信息 |
|------|------|
| 全名 | Bikram Choudhury |
| 生卒 | 1944- |
| 出生地 | 加尔各答，印度 |

---

## 二、Bikram Yoga 体系（The Bikram System）

### 2.1 26 式固定序列

- 2 组呼吸法 + 26 个体式，严格固定顺序
- 在加热的房间中进行（约 105°F / 40°C）
- 90 分钟课程，使用标准化的教学对白
- 高峰期全球约 1,500+ 家授权工作室

### 2.2 商业帝国

- 特许经营模式——每家工作室需支付授权费
- 9 周教师培训项目（TTC），费用高达 $12,000+
- 试图为瑜伽体式序列申请版权（最终被法院驳回）
- 2012 年联邦法院裁定：瑜伽体式序列不受版权保护

## 三、性侵指控与法律后果（Sexual Assault Allegations）

### 3.1 指控时间线

| 时间 | 事件 |
|------|------|
| 2013-2016 | 6 名女性提起性侵/骚扰民事诉讼 |
| 2016 | Minakshi Jafa-Bodden（前法律顾问）胜诉，获赔 $6.5M |
| 2017 | 针对他的逮捕令签发 |
| 2017- | 逃离美国，据报在墨西哥和印度之间活动 |
| 2020+ | 继续在海外进行培训（引发行业谴责） |

### 3.2 Netflix 纪录片

2019 年 Netflix 纪录片《Bikram: Yogi, Guru, Predator》详细记录了：
- 多名受害者的证词
- 培训中的权力滥用
- 邪教式的控制手段
- 对异议者的报复

## 四、行业影响与教训（Industry Impact & Lessons）

| 影响 | 说明 |
|------|------|
| 品牌去 Bikram 化 | 大量工作室更名为"热瑜伽"或"26&2"，去除 Bikram 品牌 |
| 行业伦理讨论 | 推动了对灵性领袖权力监督的广泛反思 |
| 版权判例 | 确立了瑜伽体式不可版权保护的法律先例 |
| 热瑜伽存续 | 热瑜伽作为练习形式继续存在，但与 Bikram 个人脱钩 |
| 教师培训改革 | 行业呼吁更透明的培训环境和学员保护机制 |
