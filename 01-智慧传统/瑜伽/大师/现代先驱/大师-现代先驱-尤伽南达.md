# 帕拉宏撒·尤迦南达（Paramahansa Yogananda / परमहंस योगानन्द）

> 《一个瑜伽行者的自传》作者——将克利亚瑜伽带向西方的灵性大使

---

## 一、概述（Overview）

| 项目 | 信息 |
|------|------|
| 全名 | Paramahansa Yogananda |
| 俗名 | Mukunda Lal Ghosh |
| 生卒 | 1893-1952 |
| 出生地 | 戈勒克布尔，印度 |
| 传统 | Kriya Yoga（克利亚瑜伽） |
| 师父 | Swami Sri Yukteswar Giri |
| 传承 | Lahiri Mahasaya → Sri Yukteswar → Yogananda |
| 创立组织 | Self-Realization Fellowship (SRF, 1920) |
| 代表作 | 《Autobiography of a Yogi》（一个瑜伽行者的自传，1946） |

---

## 二、核心贡献（Core Contributions）

### 2.1 《一个瑜伽行者的自传》

- 1946 年出版，被翻译成 50+ 种语言
- 被誉为 20 世纪最重要的灵性书籍之一
- 乔布斯（Steve Jobs）一生中唯一一本每年重读的书
- 猫王（Elvis Presley）在去世时身边放着这本书
- 将印度灵性传统以生动故事形式介绍给数百万西方读者

### 2.2 Kriya Yoga 传播

Kriya Yoga 的核心：
- 系统性的呼吸和冥想技术
- 据说可以加速灵性进化
- 需要通过正式的入门仪式（Diksha）接受
- SRF 提供系统性的家庭学习课程

### 2.3 Self-Realization Fellowship

- 1920 年在美国创立
- 全球 500+ 冥想中心和小组
- 提供系统性的 Kriya Yoga 函授/在线课程
- 总部在加利福尼亚州洛杉矶
- 在印度的姊妹组织：Yogoda Satsanga Society (YSS)

---

## 三、教导特色（Teaching Characteristics）

| 特色 | 说明 |
|------|------|
| 东西融合 | 将印度瑜伽与基督教神秘主义融合 |
| 科学化表述 | 用科学语言解释灵性概念 |
| 个人魅力 | 极具感染力的演讲和著述风格 |
| 系统化教学 | 建立了可复制的课程体系 |
| 强调冥想 | 以冥想为核心而非体式 |

---

## 四、影响与遗产（Legacy）

- 是继 Vivekananda 之后最成功地将印度精神传统带向西方的人
- 影响了整个 20 世纪 60-70 年代的西方灵性运动
- SRF 至今仍是全球最大的 Kriya Yoga 组织
- 1952 年在洛杉矶去世，据说遗体多日未腐坏（引起广泛讨论）
