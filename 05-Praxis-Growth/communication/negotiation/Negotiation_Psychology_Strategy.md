## 一、核心概念

### 1.1 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| BATNA | Best Alternative To Negotiated Agreement | 谈判破裂时的最佳替代方案 |
| ZOPA | Zone of Possible Agreement | 双方可接受范围的重叠区域 |
| 锚定 | Anchoring | 先出价格设定谈判参考点 |
| 保留价格 | Reservation Price | 可接受的最低/最高价格 |
| 分配型谈判 | Distributive | 零和博弈，固定饼的分割 |
| 整合型谈判 | Integrative | 扩大饼，创造共赢 |
| 框架效应 | Framing | 同一提议的不同表述影响决策 |

### 1.2 谈判类型

| 类型 | 特征 | 策略重点 | 示例 |
|------|------|---------|------|
| 分配型 | 单一议题、零和 | 锚定、让步策略 | 价格谈判 |
| 整合型 | 多议题、可创造价值 | 利益探索、打包交易 | 商业合作 |
| 多方谈判 | 三方以上 | 联盟建设、议程设置 | 国际谈判 |
| 代理谈判 | 代表他人谈判 | 授权范围、委托人管理 | 律师/经纪人 |

## 二、经典谈判框架

### 2.1 哈佛原则谈判法

> Fisher & Ury《Getting to Yes》(1981) — 谈判学奠基之作

| 原则 | 内容 | 应用 |
|------|------|------|
| 人事分开 | 将人际关系与实质问题分离 | "我们是在解决问题，不是在对抗" |
| 关注利益 | 聚焦潜在利益而非表面立场 | "你为什么想要这个？" |
| 创造选项 | 在决定之前生成多种可能方案 | 头脑风暴、打包提案 |
| 客观标准 | 基于客观标准而非意志力 | 市场价、行业惯例、先例 |

### 2.2 FBI谈判术

> Chris Voss《Never Split the Difference》(2016)

| 技术 | 说明 | 示例 |
|------|------|------|
| 战术同理心 | 识别和表达对方情感 | "看起来你对这个很担忧" |
| 标注 | 用语言标注对方情绪 | "听起来你觉得不公平" |
| 镜像 | 重复对方最后几个关键词 | 对方说"预算有限"，你说"预算有限？" |
| 校准问题 | 用"如何/什么"开头的开放问题 | "我们怎么才能让这个方案更好？" |
| 沉默 | 提出方案后保持沉默 | 让对方先回应 |
| 指控审计 | 先列出对方可能的负面看法 | "你可能觉得这个价格太高了..." |

### 2.3 BATNA分析

| 步骤 | 操作 |
|------|------|
| 1. 列出替代方案 | 如果谈判失败，你有哪些其他选择？ |
| 2. 评估每个替代方案 | 每个方案的价值和可行性 |
| 3. 选择最佳替代方案 | 这就是你的BATNA |
| 4. 改善BATNA | 在谈判前尽可能改善你的替代方案 |
| 5. 设定保留价格 | 基于BATNA设定你的底线 |

## 三、谈判心理学

### 3.1 认知偏差与谈判

| 偏差 | 影响 | 应对 |
|------|------|------|
| 锚定效应 | 首个数字影响后续判断 | 做好准备，设定自己的锚点 |
| 框架效应 | "省200元"vs"多花300元" | 意识到框架，重新定义 |
| 禀赋效应 | 高估自己已有的东西 | 从对方角度评估价值 |
| 确认偏差 | 只关注支持自己立场的信息 | 刻意寻找反面证据 |
| 固定饼偏差 | 假设谈判是零和的 | 主动探索扩大饼的机会 |
| 反应性贬值 | 因为是对方提出的就低估方案 | 基于客观标准评估 |
| 过度自信 | 高估自己的谈判实力 | 客观评估BATNA |

### 3.2 情绪在谈判中的作用

| 情绪 | 影响 | 策略性应用 |
|------|------|-----------|
| 愤怒 | 可能获得更多让步（但损害关系） | 谨慎使用，区分真怒与策略怒 |
| 焦虑 | 更快妥协、更低期望 | 充分准备降低焦虑 |
| 快乐 | 更合作、更有创造力 | 创造积极氛围 |
| 失望 | 可能引发对方让步 | 适度表达 |
| 惊讶 | 打破锚定 | 用事实让对方重新评估 |

## 四、实战技巧

### 4.1 开局技巧

| 技巧 | 说明 |
|------|------|
| 高锚定 | 先出一个有利的锚点（有依据的高/低价） |
| 渐进让步 | 每次让步幅度递减（示意接近底线） |
| 议程设置 | 先讨论容易达成共识的议题 |
| 关系建立 | 开始前花时间建立融洽关系 |

### 4.2 中场技巧

| 技巧 | 说明 |
|------|------|
| 打包提案 | 多议题同时谈，而非逐一谈 |
| 有条件让步 | "如果你能...那我可以..." |
| 虚拟选项 | 提出多个方案让对方选择 |
| 暂停 | 需要时间思考就暂停 |
| 改变维度 | 从价格转向条款/时间/数量 |

### 4.3 收尾技巧

| 技巧 | 说明 |
|------|------|
| 总结确认 | 逐条确认达成的协议 |
| 书面化 | 立即书面化协议内容 |
| 留面子 | 让对方感觉获得了好交易 |
| 建立后续 | 为未来合作留下空间 |

## 五、日常谈判场景

### 5.1 薪资谈判

| 阶段 | 策略 |
|------|------|
| 准备 | 研究市场薪资(Glassdoor/猎聘/LinkedIn)、明确BATNA |
| 时机 | 在offer阶段或绩效评估后 |
| 锚定 | 给出薪资范围的高端（有数据支持） |
| 扩展 | 不只谈薪资——考虑奖金/股权/假期/远程 |
| 框架 | 强调你带来的价值，而非你的需求 |

### 5.2 日常消费谈判

| 场景 | 策略 |
|------|------|
| 租房 | 研究周边价格、长期租约换优惠、指出缺陷 |
| 汽车 | 了解成本价、多家比价、月底/季末谈 |
| 服务费 | 忠诚客户折扣、竞品报价对比 |
