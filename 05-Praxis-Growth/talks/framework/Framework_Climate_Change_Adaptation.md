# 气候变化适应 | Climate Change Adaptation

> **气候变化适应（Climate Change Adaptation）** 是指人类和自然系统对实际或预期的气候变化及其影响做出的调整过程，旨在减轻危害或利用有利机会。与减缓（Mitigation）不同，适应关注的是如何应对已经发生或不可避免的气候变化后果。本文档系统梳理气候变化适应的核心概念、理论框架、适应策略和实践案例，为气候政策研究者、城市规划者和社区工作者提供全面的知识参考。

## 一、核心概念体系

### 1.1 气候变化适应基本概念

| 中文术语 | 英文 | 定义说明 |
|---------|------|----------|
| 气候变化适应 | Climate Change Adaptation | 对实际或预期气候变化影响的调整行为 |
| 气候减缓 | Climate Mitigation | 减少温室气体排放或增加碳汇的干预措施 |
| 气候韧性 | Climate Resilience | 系统在气候冲击下维持功能和恢复的能力 |
| 气候脆弱性 | Climate Vulnerability | 系统对气候变化不利影响的敏感程度 |
| 气候暴露度 | Climate Exposure | 系统面临的气候变化压力的性质和程度 |
| 适应能力 | Adaptive Capacity | 系统、机构或个体调整以应对气候变化的能力 |
| 气候风险 | Climate Risk | 气候危害与脆弱性和暴露度交互作用产生的不良后果 |
| 渐进适应 | Incremental Adaptation | 在现有系统框架内逐步调整的适应行为 |
| 转型适应 | Transformational Adaptation | 改变系统根本属性以适应新气候条件的适应行为 |
| 适应赤字 | Adaptation Deficit | 当前气候条件下已存在的适应能力不足 |
| 气候正义 | Climate Justice | 在气候行动中保障公平和权利的原则 |
| 基于自然的解决方案 | Nature-based Solutions (NbS) | 利用自然过程应对气候变化和社会挑战的方法 |
| 损失与损害 | Loss and Damage | 气候变化造成的不可逆转的负面影响 |
| 气候移民 | Climate Migration | 因气候变化因素导致的人口迁移 |

### 2.2 气候风险评估模型

#### IPCC风险框架（AR6）

```
气候风险 = 危害(Hazard) × 暴露度(Exposure) × 脆弱性(Vulnerability)

                    ┌──────────────┐
                    │   气候风险    │
                    └──────┬───────┘
              ┌──────────┼──────────┐
              ▼          ▼          ▼
        ┌─────────┐ ┌─────────┐ ┌──────────┐
        │  危害    │ │ 暴露度  │ │ 脆弱性   │
        │ Hazard  │ │Exposure │ │Vulnerab. │
        └────┬────┘ └────┬────┘ └─────┬────┘
             │           │            │
      气候相关   社会经济      礏感性 +
      物理事件   系统位置      适应能力

危害类型：
• 极端高温
• 极端降水/洪水
• 干旱
• 海平面上升
• 风暴/飓风
• 野火
```

#### 适应评估矩阵

| 评估维度 | 评估内容 | 评估方法 | 数据来源 |
|---------|---------|----------|----------|
| 气候风险 | 当前和未来气候危害 | 气候模型、历史数据分析 | 气象数据、CMIP6 |
| 脆弱性 | 社会经济敏感性和适应能力 | 指标体系、专家评估 | 统计数据、调查 |
| 暴露度 | 人口和资产在危险区的分布 | GIS分析、遥感 | 人口、资产数据库 |
| 适应能力 | 现有制度、技术和资金能力 | 能力评估框架 | 政策文件、访谈 |
| 适应成本 | 适应措施的投资需求 | 成本效益分析 | 工程估算、经济模型 |
| 残余风险 | 适应后仍存在的风险 | 情景分析 | 综合模型 |

### 2.3 适应策略理论模型

| 策略类型 | 英文 | 具体措施 | 适用条件 |
|---------|------|----------|----------|
| 保护型 | Protection | 防洪堤、海堤、防风林 | 中等程度风险 |
| 适应型 | Accommodation | 抬高建筑、耐涝品种、预警系统 | 风险可管理 |

### 3.2 典型案例

#### 案例一：荷兰"还地于河"计划（Room for the River）

**背景：** 荷兰面临海平面上升和极端降水增加的双重挑战。

**核心策略：**
- 从"抵抗洪水"转向"与水共存"的适应哲学
- 拓宽河道、降低河岸、开辟洪水溢流区
- 结合城市规划和景观设计创造多功能的滨水空间

**成效：**
- 显著降低了莱茵河和马斯河沿岸的洪水风险
- 创造了新的休闲和生态空间
- 成为全球河流流域适应的标杆

**启示：** 适应策略需要突破传统工程思维，采用系统性和创造性的解决方案。

#### 案例二：孟加拉国社区适应项目

**背景：** 孟加拉国是全球最易受气候变化影响的国家之一。

**核心策略：**
- 社区层面的灾害预警系统建设
- 浮动农业和耐盐水稻品种推广
- 气候韧性住房设计
- 生计多样化支持

**成效：**
- 社区灾害应对能力显著提升
- 农业生产在极端天气下的韧性增强
- 社区自主适应能力得到培育

**启示：** 低成本、社区主导的适应措施在资源有限的情况下同样有效。

#### 案例三：中国海绵城市建设

**背景：** 城市内涝问题日益严重，传统排水系统难以应对。

**核心策略：**
- "渗、滞、蓄、净、用、排"六位一体
- 透水铺装、绿色屋顶、下沉式绿地
- 雨水花园和生态滞留设施
- 城市水系综合整治

**成效：**
- 试点城市内涝频率显著降低
- 城市水环境质量改善
- 城市热岛效应缓解

**启示：** 基于自然的适应方案可以同时解决多个城市问题。

| 评估标准 | 具体内容 | 评估方法 | 权重建议 |

### 5.2 关键挑战与展望

**适应差距扩大：** 尽管适应意识增强，但全球适应行动的规模和速度远低于需求。发展中国家面临的适应资金缺口尤其严重，需要大幅增加适应融资。

**复合与级联风险：** 气候变化的影响不再是单一维度的，而是多种风险交织叠加的复合效应。适应策略需要从单一灾害应对转向系统性风险管理。

**损失与损害机制的落实：** 损失与损害基金的建立是全球气候正义的重要进展，但其资金规模、分配机制和支持范围仍需进一步明确和完善。

**适应与减缓的协同：** 最好的气候行动是能够同时实现减缓和适应目标的行动。基于自然的解决方案、可再生能源、可持续农业等领域都存在显著的协同效应。

**从脆弱性到转型：** 未来的适应需要超越减少脆弱性的被动模式，转向积极创造韧性和推动社会转型的主动适应。这需要从根本上重新思考发展路径、制度安排和治理体系。
