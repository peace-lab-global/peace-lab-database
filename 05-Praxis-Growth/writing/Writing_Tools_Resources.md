# 写作工具与资源大全 | Writing Tools and Resources Compendium

> **写作工具与资源大全**为写作者提供全面的创作支持体系，涵盖硬件设备、软件工具、学习资源、出版渠道等各个方面，帮助创作者构建高效的写作生态系统，提升创作效率和作品质量。

## 一、硬件设备配置

### 1.1 基础写作设备

#### 计算机设备选择
```

二合一：触屏可旋转，支持手写输入

• 机械键盘：提升打字手感和效率
```
