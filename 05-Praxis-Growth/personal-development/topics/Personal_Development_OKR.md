## 核心内容与观点 (Core Content & Perspectives)

### OKR定义

| 组成部分 | 定义 | 特点 | 示例 |
| :--- | :--- | :--- | :--- |
| **O (Objective)** | 目标：你想达成什么 | 定性、鼓舞人心、有时间限制 | "成为市场领导者" |
| **KR (Key Result)** | 关键结果：如何衡量进展 | 定量、可衡量、3-5个 | "市场份额达到25%" |

### OKR vs KPI

| 维度 | OKR | KPI |
| :--- | :--- | :--- |
| **目的** | 推动突破性进步 | 监控日常运营 |
| **性质** | 挑战性、可能失败 | 必须达成 |
| **聚焦** | 最重要的少数事项 | 全面覆盖 |
| **频率** | 季度 | 年度或持续 |
| **关联薪酬** | 不直接关联 | 通常关联 |

### OKR的四大超能力

#### 超能力1：聚焦 (Focus)

| 原则 | 描述 | 操作 |
| :--- | :--- | :--- |
| **少即是多** | 每个周期3-5个目标 | 勇于说"不" |
| **优先级** | 最重要的事情优先 | 问"这是最重要的吗？" |
| **清晰度** | 目标必须明确 | 每个人都能理解 |

#### 超能力2：协同 (Align)

| 方向 | 描述 | 效果 |
| :--- | :--- | :--- |
| **自上而下** | 公司目标分解到团队 | 战略一致 |
| **自下而上** | 个人目标支持团队目标 | 员工参与 |
| **横向协同** | 部门间目标协调 | 打破孤岛 |

#### 超能力3：追踪 (Track)

| 实践 | 描述 | 频率 |
| :--- | :--- | :--- |
| **持续追踪** | 定期检查进度 | 每周 |
| **评分** | 0-1.0评分关键结果 | 季度末 |
| **复盘** | 分析成功和失败原因 | 季度末 |

#### 超能力4：延展 (Stretch)

| 目标类型 | 描述 | 预期完成率 |
| :--- | :--- | :--- |
| **承诺型目标** | 必须达成的目标 | 100% |
| **延展型目标** | 有挑战性的目标 | 70%（完全完成说明不够有挑战性） |

### OKR评分

| 分数 | 含义 | 解读 |
| :--- | :--- | :--- |
| **0.0-0.3** | 失败 | 需要分析原因 |
| **0.4-0.6** | 有进展 | 部分完成 |
| **0.7-1.0** | 成功 | 如果总是1.0，目标可能太容易 |

### CFR：OKR的搭档

| 组成部分 | 英文 | 描述 | 频率 |
| :--- | :--- | :--- | :--- |
| **C** | Conversations | 一对一对话，讨论进展 | 每周或双周 |
| **F** | Feedback | 双向反馈 | 持续 |
| **R** | Recognition | 认可和庆祝 | 即时 |

### OKR周期

| 阶段 | 时间 | 活动 |
| :--- | :--- | :--- |
| **设定** | 季度开始前 | 设定目标和关键结果 |
| **追踪** | 季度期间 | 每周检查进度 |
| **评分** | 季度末 | 为每个KR评分 |
| **复盘** | 季度末 | 分析学习 |
| **下一周期** | 新季度开始 | 设定新OKR |

### OKR示例

**个人OKR示例**

| 目标 (O) | 关键结果 (KR) |
| :--- | :--- |
| **O1: 提升个人生产力** | KR1: 每天完成3小时深度工作 |
| | KR2: 每周完成GTD回顾 |
| | KR3: 每月读完2本书 |
| **O2: 改善身体健康** | KR1: 每周锻炼4次，每次30分钟 |
| | KR2: 体重减少5公斤 |
| | KR3: 每晚睡眠7小时以上 |

## 实践建议 (Practical Recommendations)

### OKR设定检查清单

| 检查项 | 问题 | 标准 |
| :--- | :--- | :--- |
| **目标** | 是否鼓舞人心？ | 读起来令人振奋 |
| **目标** | 是否足够聚焦？ | 3-5个 |
| **目标** | 是否有时间限制？ | 季度或年度 |
| **关键结果** | 是否可衡量？ | 有具体数字 |
| **关键结果** | 是否有挑战性？ | 70%完成率为最佳 |
| **关键结果** | 是否3-5个？ | 不要太多 |

### OKR模板

```
目标 (O): [用一句话描述你想达成的目标]
├── 关键结果1: [可衡量的指标] - 从X到Y
├── 关键结果2: [可衡量的指标] - 从X到Y
└── 关键结果3: [可衡量的指标] - 从X到Y
```

### 与其他实践的整合

| 整合领域 | 结合方式 | 增强效果 |
| :--- | :--- | :--- |
| **GTD** | OKR定方向，GTD管执行 | 战略和战术结合 |
| **深度工作** | 为深度工作设定OKR | 保护和追踪深度时间 |
| **习惯养成** | 用KR追踪习惯 | 将习惯与目标关联 |
