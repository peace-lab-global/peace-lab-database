## 核心内容与观点 (Core Content & Perspectives)

### NVC四要素模型

| 要素 | 英文 | 定义 | 目的 |
| :--- | :--- | :--- | :--- |
| **观察** | Observation | 不带评判地描述发生的事实 | 建立共同现实基础 |
| **感受** | Feeling | 识别和表达内心的感受 | 连接内在体验 |
| **需要** | Need | 识别感受背后的需要 | 找到行为的根源 |
| **请求** | Request | 明确、具体、可行的请求 | 促进合作行动 |

### NVC表达公式

```
当我看到/听到 [观察]...
我感到 [感受]...
因为我需要 [需要]...
你是否愿意 [请求]...？
```

### 观察 vs 评论

| 评论 (避免) | 观察 (使用) |
| :--- | :--- |
| "你总是迟到" | "这周你三次在9点之后到达" |
| "你很懒" | "我注意到昨天的报告没有完成" |
| "你不尊重我" | "当我说话时，你在看手机" |
| "这太糟糕了" | "这个结果与我的预期不同" |

### 感受 vs 想法

| 想法 (避免) | 感受 (使用) |
| :--- | :--- |
| "我觉得被忽视了" | "我感到失望/孤独" |
| "我觉得你不在乎" | "我感到担心/难过" |
| "我觉得这不公平" | "我感到沮丧/愤怒" |
| "我觉得被利用了" | "我感到疲惫/失望" |

### 需要的类别

| 类别 | 需要示例 |
| :--- | :--- |
| **自主** | 选择、自由、独立、空间 |
| **庆祝** | 认可、哀悼、纪念 |
| **正直** | 真实、诚信、意义、自我尊重 |
| **相互依存** | 接纳、欣赏、亲密、社区、信任、支持 |
| **游戏** | 乐趣、欢笑、放松 |
| **灵性** | 美、和谐、秩序、和平 |
| **身体** | 食物、水、空气、休息、安全、触碰、性表达 |

### 请求 vs 要求

| 要求 (避免) | 请求 (使用) |
| :--- | :--- |
| 对方必须答应 | 对方可以说"不" |
| 不答应会有后果 | 不答应时探讨其他方式 |
| 关注服从 | 关注连接和满足双方需要 |
| "你必须..." | "你是否愿意...?" |

### 同理心倾听

| 步骤 | 描述 | 示例 |
| :--- | :--- | :--- |
| **临在** | 全身心在场 | 放下自己的议程 |
| **反映观察** | 反映你听到的事实 | "你是说...发生了？" |
| **反映感受** | 猜测对方的感受 | "你是否感到...?" |
| **反映需要** | 猜测感受背后的需要 | "因为你需要...?" |
| **请求** | 确认对方的请求 | "你是否希望...?" |

### 自我同理

| 步骤 | 问题 | 目的 |
| :--- | :--- | :--- |
| **观察** | 发生了什么？ | 明确事实 |
| **感受** | 我现在感觉如何？ | 连接内在体验 |
| **需要** | 我的什么需要没有得到满足？ | 找到根源 |
| **请求** | 我可以怎么做？ | 找到行动方向 |

### 愤怒的转化

| 步骤 | 描述 | 操作 |
| :--- | :--- | :--- |
| **停止** | 暂停反应 | 深呼吸 |
| **识别想法** | 识别引发愤怒的想法 | "我在想什么？" |
| **连接感受和需要** | 转向感受和未满足的需要 | 用NVC四要素 |
| **表达** | 用NVC表达 | 观察-感受-需要-请求 |

## 实践建议 (Practical Recommendations)

### NVC日常练习

| 情境 | 传统反应 | NVC反应 |
| :--- | :--- | :--- |
| 同事迟到 | "你怎么又迟到了？" | "我注意到你比约定时间晚到了20分钟，我感到担心，因为我需要确定性。你是否愿意告诉我发生了什么？" |
| 孩子不做作业 | "你太懒了！" | "我看到作业还没完成，我感到担忧，因为我关心你的学业。你是否愿意和我谈谈发生了什么？" |
| 伴侣忘记重要日期 | "你根本不在乎我！" | "今天是我们的纪念日，你没有提起，我感到难过，因为我需要感到被重视。你是否愿意和我一起庆祝？" |

### 感受词汇表

| 需要满足时的感受 | 需要未满足时的感受 |
| :--- | :--- |
| 高兴、满足、兴奋、感激、平静、轻松、希望、自信、安全 | 伤心、沮丧、焦虑、恐惧、愤怒、疲惫、孤独、困惑、不安 |
