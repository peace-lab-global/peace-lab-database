## 核心内容与观点 (Core Content & Perspectives)

### 关键对话的定义

| 特征 | 描述 | 示例 |
| :--- | :--- | :--- |
| **高风险** | 结果对双方都很重要 | 绩效评估、薪资谈判 |
| **不同意见** | 双方观点存在分歧 | 战略决策、资源分配 |
| **强烈情绪** | 对话涉及强烈的情感 | 信任破裂、冲突对峙 |

### 对话困境的三种反应

| 反应 | 描述 | 结果 |
| :--- | :--- | :--- |
| **沉默** | 回避、退缩、不说话 | 问题未解决、积怨 |
| **暴力** | 攻击、控制、贴标签 | 关系损害、对抗升级 |
| **对话** | 开放、诚实、尊重地交流 | 问题解决、关系增强 |

### 共享意义库 (Pool of Shared Meaning)

| 概念 | 描述 | 重要性 |
| :--- | :--- | :--- |
| **定义** | 对话中所有参与者分享的信息、想法、理论和感受的集合 | 信息越丰富，决策越好 |
| **目标** | 让所有人的想法自由流入池中 | 集体智慧胜过个人 |
| **障碍** | 沉默和暴力都会减少信息流入 | 需要创造安全环境 |

### 核心技能框架

#### 1. 从心开始 (Start with Heart)

| 原则 | 问题 | 操作 |
| :--- | :--- | :--- |
| **明确目标** | 我真正想要什么？ | 关注真正的目标而非情绪反应 |
| **拒绝"傻瓜式选择"** | 只有沉默或暴力两个选择吗？ | 寻找第三种可能 |
| **双赢思维** | 如何实现我想要的并且对方也满意？ | AND而非OR |

#### 2. 学会观察 (Learn to Look)

| 观察点 | 信号 | 意义 |
| :--- | :--- | :--- |
| **内容** | 谈话的主题 | 表面问题 |
| **氛围** | 人们是否感到安全 | 对话条件 |
| **自己的行为** | 你是否在沉默或暴力 | 自我觉察 |

#### 3. 营造安全 (Make It Safe)

| 安全条件 | 定义 | 修复方法 |
| :--- | :--- | :--- |
| **共同目标** | 对方相信你关心他们的利益 | 明确共同目标 |
| **互相尊重** | 对方感到被尊重 | 对比说明（我不是要...我是要...） |

#### 4. 控制你的故事 (Master My Stories)

| 步骤 | 描述 | 操作 |
| :--- | :--- | :--- |
| **觉察行为** | 注意自己的行为 | "我正在做什么？" |
| **识别感受** | 识别背后的情绪 | "我感觉如何？" |
| **分析故事** | 找出引发情绪的故事 | "我告诉自己什么故事？" |
| **回到事实** | 将故事与事实分开 | "我的证据是什么？" |

#### 5. STATE方法 - 陈述你的观点

| 字母 | 含义 | 操作 |
| :--- | :--- | :--- |
| **S** | Share facts | 分享事实 |
| **T** | Tell your story | 说出你的想法 |
| **A** | Ask for others' paths | 询问对方的观点 |
| **T** | Talk tentatively | 用试探性语气说话 |
| **E** | Encourage testing | 鼓励对方挑战你的观点 |

#### 6. 探询他人的观点 (Explore Others' Paths)

| 技术 | 描述 | 示例 |
| :--- | :--- | :--- |
| **询问** | 表达真诚的兴趣 | "我想了解你的想法" |
| **映射** | 反映你听到的 | "所以你是说..." |
| **释义** | 用自己的话复述 | "如果我理解正确..." |
| **启发** | 当对方沉默时鼓励分享 | "你还有什么想说的吗？" |

#### 7. 转向行动 (Move to Action)

| 步骤 | 描述 | 操作 |
| :--- | :--- | :--- |
| **确定行动** | 明确谁做什么、何时完成 | 具体、可衡量 |
| **明确责任** | 每个行动有唯一负责人 | 避免模糊 |
| **设定跟进** | 安排检查进度的时间 | 保持问责 |
