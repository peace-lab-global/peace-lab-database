# 职场语言霸凌与PUA：案例库与行为模式分析 (Case Library & Behavioral Pattern Analysis)

> **定位**：将典型职场语言霸凌与PUA事件拆解为可比对、可复用、可检索的案例条目，帮助识别行为模式、证据重点与应对路径
> **使用方式**：先看案例标签，再看行为模式，最后看证据与处置建议

## 二、典型案例条目

## 2.1 案例A：周会公开羞辱建立权威
- **场景类型**：会议现场
- **权力关系**：上对下
- **主要行为模式**：公开贬低、比较羞辱、群体规训
- **典型话术**：
  - “你这点能力也配带项目？”
  - “大家都看看，这就是不做准备的下场。”
- **心理机制**：羞耻诱导、创伤性警觉
- **关键证据**：会议纪要、在场证人、会后确认邮件
- **主要风险**：团队沉默、受害者不敢继续表达风险
- **推荐路径**：现场降温 + 会后书面确认 + 观察重复性 + 必要时升级
- **交叉引用**：
  - [心理学机制文档](Workplace_Verbal_Bullying_PUA_Psychology.md)
  - [高压会议现场应对手册](Workplace_High_Pressure_Meeting_Handbook.md)

## 2.2 案例B：私下长期否定导致自我怀疑
- **场景类型**：一对一谈话
- **权力关系**：上对下
- **主要行为模式**：人格化评价、模糊否定、情感抽离
- **典型话术**：
  - “你最大的问题是人不行。”
  - “我已经懒得教你了。”
- **心理机制**：煤气灯效应、习得性无助
- **关键证据**：聊天记录、录音、任务反馈对照
- **主要风险**：受害者把问题全部内化，逐步丧失判断力
- **推荐路径**：要求具体标准 + 建立任务对照表 + 引入第三方

## 2.3 案例C：忽冷忽热式认可控制
- **场景类型**：日常管理
- **权力关系**：上对下
- **主要行为模式**：间歇性强化、情绪操纵、奖惩不稳定
- **典型表现**：
  - 一天内先夸“你最可靠”，后又说“你什么都做不好”
  - 在资源和认可上反复拉扯
- **心理机制**：关系成瘾、外部评价依赖
- **关键证据**：连续时间线、消息截图、任务分配变化
- **主要风险**：受害者持续追逐认可，越来越难退出
- **推荐路径**：看长期模式，不看单次夸奖；减少情绪性解释依赖

## 2.4 案例D：申诉后遭遇报复性低绩效
- **场景类型**：绩效管理
- **权力关系**：上对下 + 组织中介失灵
- **主要行为模式**：报复、标签化、程序不透明
- **典型表现**：
  - 提交投诉后突然被打低绩效
  - 绩效标准与过往版本明显不一致
- **心理机制**：恐惧强化、沉默驯化
- **关键证据**：申诉时间点、历史绩效记录、面谈纪要、周报
- **主要风险**：从管理问题升级为劳动争议
- **推荐路径**：固定申诉前后变化、申请绩效复核、准备维权材料
- **交叉引用**：
  - [PIP应对指南](Workplace_PIP_Response_Guide.md)
  - [职场维权申诉邮件与仲裁请求模板](Workplace_Complaint_Email_and_Arbitration_Request_Templates.md)

## 2.5 案例E：孕产节点下的语言压迫与边缘化
- **场景类型**：返岗、晋升、项目分配
- **权力关系**：上对下 + 组织偏见
- **主要行为模式**：婚育偏见、机会剥夺、柔性打压
- **典型话术**：
  - “你现在还有精力带核心项目吗？”
  - “团队也要考虑稳定性。”
- **心理机制**：身份污名化、角色去能力化
- **关键证据**：返岗前后职责变化、晋升口径变化、对比样本
- **主要风险**：歧视被包装成“业务安排”
- **推荐路径**：做差别对待对照、固定返岗前后变化、必要时同步法规路径

## 2.6 案例F：违法调岗被包装成“给你新机会”
- **场景类型**：组织调整
- **权力关系**：上对下
- **主要行为模式**：角色剥离、模糊协商、柔性施压
- **典型话术**：
  - “这次调岗是公司对你的培养。”
  - “你不接受就是不顾大局。”
- **心理机制**：道德绑架、顺从压力
- **关键证据**：劳动合同、岗位说明、调岗通知、薪酬变化
- **主要风险**：被迫接受实质性不利变更
- **推荐路径**：区分合理调配与实质变更，不轻易口头同意

## 2.7 案例G：劝退协商中的软性恐吓
- **场景类型**：离职协商
- **权力关系**：管理者 + HR
- **主要行为模式**：暗示性威胁、未来损害预告、逼签
- **典型话术**：
  - “主动走对你简历好看。”
  - “如果走到正式流程，对你不会有好处。”
- **心理机制**：预期恐惧、声誉控制
- **关键证据**：会议录音、离职协议版本、催签记录
- **主要风险**：员工在高压下签下不利文件
- **推荐路径**：申请冷静期、书面确认、法务或外部咨询后再决定

## 2.8 案例H：群体式冷暴力与信息隔离
- **场景类型**：团队协作
- **权力关系**：同级围猎 / 群体排斥
- **主要行为模式**：拉群排除、故意漏发信息、甩锅
- **心理机制**：社会排斥、归属威胁
- **关键证据**：任务系统记录、群消息、邮件抄送缺失、时间线
- **主要风险**：问题被伪装成“沟通不顺”或“配合能力差”
- **推荐路径**：拉齐任务边界、书面留痕、引入更高层协调

## 三、行为模式总表

| 模式 | 典型语言信号 | 深层控制目标 | 高风险后果 |
| --- | --- | --- | --- |
| 贬低型 | “你不行”“你不配” | 打击自尊，建立从属 | 长期自我怀疑 |
| 煤气灯型 | “我没说过”“是你理解错” | 扭曲现实感 | 证据意识下降 |
| 比较型 | “别人都可以就你不行” | 制造羞耻与服从 | 竞争性孤立 |
| 冷暴力型 | 不回应、不说明、临时否定 | 制造不确定感 | 认知过载 |
| 间歇强化型 | 时夸时踩 | 让人沉迷认可 | 难以脱离 |
| 报复型 | 投诉后低绩效/边缘化 | 压制反抗 | 劳动争议升级 |
| 劝退型 | “主动走更好看” | 推进离职而规避责任 | 高压签约 |
