## 职业转型概述

### 转型类型

| 类型 | 说明 | 挑战 |
|------|------|------|
| 行业内跨函数 | 同行业换岗位(销售→市场) | 技能迁移/重新定位 |
| 跨行业同函数 | 换行业保留职能(金融销售→医疗销售) | 行业知识/网络重建 |
| 完全转型 | 行业+职能都变(银行家→瑜伽教练) | 身份重构/收入过渡 |
| 创业转型 | 从雇员到创业者 | 风险/不确定/多角色 |
| 组合式转型 | 多种兼职/自由职业组合 | 时间管理/收入波动 |
| 安可职业 | 退休后有意义的工作 | 重新定义价值/适应 |

### 转型动机光谱

| 推力(离开原因) | 拉力(前往原因) |
|--------------|--------------|
| 倦怠/无意义感 | 热情/使命感 |
| 组织变革/裁员 | 成长机会 |
| 健康问题 | 生活方式改善 |
| 关系/家庭需求 | 自主/灵活 |
| 薪资/发展天花板 | 财务/影响力 |
| 价值观冲突 | 价值观一致 |

## 转型心理过程

### Bridges转变三阶段

| 阶段 | 特征 | 任务 |
|------|------|------|
| 结束(Ending) | 失去/哀伤/恐惧/解脱 | 承认失去/告别旧身份 |
| 中间地带(Neutral Zone) | 混乱/迷茫/创造力 | 耐心等待/探索/实验 |
| 新开始(New Beginning) | 能量/方向/重新承诺 | 行动/整合/新身份 |

### Herminia Ibarra工作身份模型

| 原则 | 说明 |
|------|------|
| 行动先于认知 | 先做再想(而非想清楚再做) |
| 试验性自我 | 尝试"可能的自我"(小实验) |
| 弱联系网络 | 新关系比旧关系更有用 |
| 有意义的叙事 | 构建连贯的职业故事 |
| 非线性过程 | 接受混乱/反复/不确定 |

### 转型中的情绪管理

| 情绪 | 来源 | 应对 |
|------|------|------|
| 恐惧 | 未知/经济不安全 | 最坏情境规划/应急基金 |
| 哀伤 | 失去身份/关系/地位 | 承认并允许哀伤 |
| 兴奋 | 新可能性/自由 | 享受但保持现实 |
| 怀疑 | 冒名顶替综合征/自我怀疑 | 外部验证/小成功积累 |
| 孤独 | 脱离原有社交圈 | 加入转型社群 |

## 转型策略

### 安全转型路径

| 策略 | 说明 | 风险等级 |
|------|------|---------|
| 内部转岗 | 在现有组织内换部门/角色 | 低 |
| 兼职试水 | 保留主业+副业探索 | 低-中 |
| 技能桥梁 | 学习新技能→过渡认证 | 中 |
| 信息访谈 | 与目标行业人士深度对话 | 低 |
| 影子跟岗 | 一周体验新工作 | 低 |
| 间隔年/休假 | 系统探索+重新充电 | 中 |
| 一步到位 | 辞职→全力投入新方向 | 高 |

### 技能迁移矩阵

| 可迁移技能 | 从→到 示例 |
|-----------|-----------|
| 项目管理 | 任何管理/协调角色 |
| 沟通/演讲 | 培训/销售/咨询 |
| 数据分析 | 市场研究/金融/运营 |
| 领导力 | 创业/管理/社区领袖 |
| 写作/内容 | 内容营销/出版/教育 |
| 问题解决 | 咨询/战略/创新 |

### 转型财务准备

| 维度 | 建议 |
|------|------|
| 应急基金 | 6-12个月生活费用 |
| 收入过渡 | 副业先行/渐进式转型 |
| 投资教育 | 预算新技能培训费用 |
| 减少固定支出 | 降低生活成本/还清高息债务 |
| 收入多元化 | 避免单一收入来源 |

## 特殊转型情境

### 中年职业转型

| 挑战 | 应对 |
|------|------|
| 年龄歧视 | 强调经验价值/选择看重成熟度的行业 |
| 收入降低恐惧 | 财务规划/渐进过渡 |
| 家庭责任 | 与家人沟通/共同规划 |
| 身份认同 | "我不只是我的头衔" |
| 优势 | 经验/网络/情商/自我觉察 |

### 被动转型(裁员/行业消失)

| 阶段 | 建议 |
|------|------|
| 即时 | 情绪处理/法律权利/社保/应急 |
| 1-4周 | 反思/自我评估/职业盘点 |
| 1-3月 | 积极求职/网络/技能提升 |
| 3-6月 | 扩大选择/考虑转型/灵活就业 |
| 6月+ | 重新评估方向/接受帮助 |

### 创业转型

| 阶段 | 关键 |
|------|------|
| 验证 | MVP/客户访谈/市场测试 |
| 过渡 | 兼职→全职的渐进路径 |
| 心态 | 从"完美"→"迭代"/从"员工"→"创始人" |
| 支持 | 创业社群/导师/共创空间 |
