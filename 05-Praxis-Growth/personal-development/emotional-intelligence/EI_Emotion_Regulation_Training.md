# 情绪调节训练 | Emotion Regulation Training

> **学科定位**: 临床心理学 x 自我管理 x 行为训练
> **核心主张**: 情绪调节是可训练的核心生活技能，不同策略适用于不同情境

---

## 一、情绪调节理论

### 1.1 Gross过程模型

> James Gross (1998) — 情绪调节领域最有影响力的模型

| 策略 | 时间点 | 方法 | 效果 |
|------|--------|------|------|
| 情境选择 | 情绪发生前 | 主动选择或避免特定情境 | 预防性 |
| 情境修改 | 情绪发生前 | 改变情境以减少情绪影响 | 预防性 |
| 注意部署 | 情绪早期 | 转移注意力、正念关注 | 即时 |
| 认知重评 | 情绪中期 | 改变对情境的解释 | 长效、健康 |
| 反应调节 | 情绪晚期 | 改变情绪的表达方式 | 有代价 |

**核心发现**: 认知重评(前因聚焦)通常比表达抑制(反应聚焦)更健康、更有效。

### 1.2 情绪调节策略效果对比

| 策略 | 短期效果 | 长期效果 | 认知代价 | 社交影响 |
|------|---------|---------|---------|---------|
| 认知重评 | 降低负面情绪 | 更好的幸福感 | 低 | 正面 |
| 表达抑制 | 减少表达 | 增加压力 | 高(耗损认知资源) | 负面(减少亲密) |
| 接纳 | 降低挣扎感 | 增加心理灵活性 | 低 | 正面 |
| 正念 | 降低反应性 | 改善情绪基线 | 低 | 正面 |
| 反刍 | 暂时理解感 | 加剧抑郁焦虑 | 高 | 负面 |
| 回避 | 短期缓解 | 维持/加重问题 | 中 | 负面 |

## 二、核心技能训练

### 2.1 情绪觉察训练

#### RAIN技术

| 步骤 | 英文 | 操作 |
|------|------|------|
| R | Recognize | 识别："我现在感到焦虑" |
| A | Allow | 允许："焦虑在这里，这没关系" |
| I | Investigate | 探索："焦虑在身体哪里？它想告诉我什么？" |
| N | Non-identification | 不认同："我有焦虑，但我不是焦虑" |

#### 情绪日志

| 字段 | 内容 |
|------|------|
| 时间 | 何时发生 |
| 触发 | 什么引发了情绪 |
| 情绪名称 | 具体情绪（用精确词汇） |
| 强度 | 1-10分 |
| 身体感受 | 身体哪里有感觉 |
| 想法 | 当时在想什么 |
| 行为 | 做了什么 |
| 后续 | 情绪如何消退 |

### 2.2 认知重评训练

| 类型 | 方法 | 示例 |
|------|------|------|
| 重新定义 | 改变事件的意义 | "失败"→"学习机会" |
| 放大视角 | 从更大时间/空间看 | "5年后这件事重要吗？" |
| 他人视角 | 从他人角度看 | "如果是朋友遇到这事，我会怎么说？" |
| 银色衬里 | 找积极面 | "虽然失去了工作，但有时间思考方向" |
| 去灾难化 | 质疑最坏预期 | "最坏情况发生的概率有多大？" |
| 正常化 | 将体验正常化 | "这种感受是正常的人类反应" |

### 2.3 呼吸调节

| 技术 | 方法 | 适用 |
|------|------|------|
| 盒式呼吸 | 吸4-屏4-呼4-屏4 | 快速平静 |
| 4-7-8呼吸 | 吸4-屏7-呼8 | 深度放松 |
| 生理叹息 | 双吸一呼(Huberman) | 即时平静(30秒) |
| 延长呼气 | 呼气时间是吸气2倍 | 激活副交感 |
| 腹式呼吸 | 膈肌呼吸，腹部起伏 | 基础放松 |

## 三、高级情绪技能

### 3.1 同理心训练

| 层次 | 类型 | 训练 |
|------|------|------|
| 1 | 认知同理心 | 观点采择练习、角色扮演 |
| 2 | 情感同理心 | 镜像情绪、想象对方感受 |
| 3 | 同理关怀 | 将感受转化为帮助行动 |

### 3.2 情绪颗粒度

> Lisa Feldman Barrett — 情绪建构理论

| 概念 | 说明 |
|------|------|
| 情绪颗粒度 | 能精确区分和命名情绪的能力 |
| 高颗粒度 | "我感到被忽视而不是愤怒" → 更精确的应对 |
| 低颗粒度 | "我心情不好" → 模糊导致不精确的应对 |
| 训练方法 | 扩展情绪词汇、细化情绪标注 |
| 效果 | 更好的情绪调节、更少的冲动行为 |

### 3.3 窗口耐受度

> Daniel Siegel — 适度激活窗口

| 状态 | 特征 | 干预 |
|------|------|------|
| 过度激活 | 焦虑、愤怒、恐慌 | 呼吸、接地、下调 |
| 耐受窗口内 | 可以思考和感受 | 正常功能 |
| 低度激活 | 麻木、抑郁、解离 | 运动、社交、上调 |

**目标**: 逐步扩大窗口耐受度，能在更广范围的情绪中保持功能。

## 四、日常练习方案

### 4.1 每日情商练习

| 时间 | 练习 | 时长 |
|------|------|------|
| 晨起 | 情绪检入(Check-in): "今天我感觉怎样？" | 2分钟 |
| 通勤 | 正念聆听或身体扫描 | 10分钟 |
| 午间 | 情绪标注: 给当前情绪精确命名 | 1分钟 |
| 互动后 | 迷你反思: "刚才的交流中情绪如何？" | 2分钟 |
| 睡前 | 情绪日志: 记录今天的情绪事件 | 5分钟 |

### 4.2 每周高级练习

| 练习 | 方法 | 频率 |
|------|------|------|
| 同理心练习 | 与一个人进行深度倾听 | 每周1次 |
| 认知重评 | 回顾一个困难情境，尝试3种重评 | 每周2次 |
| 情绪暴露 | 有意识地接触轻度不适情绪 | 每周1次 |
| 感恩练习 | 写3件感恩的事（含对他人） | 每周3次 |
