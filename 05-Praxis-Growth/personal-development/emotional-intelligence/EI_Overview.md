# 情商总览 | Emotional Intelligence Overview

> **学科定位**: 心理学 x 组织行为学 x 个人发展
> **核心主张**: 情商是识别、理解、管理和利用情绪的能力，对人生成功的影响可能超过智商

---

## 一、情商的定义与模型

### 1.1 核心定义

情商（Emotional Intelligence, EI/EQ）：准确感知、评估和表达情绪的能力；接触和产生有助于思维的情绪的能力；理解情绪和情绪知识的能力；以及调节情绪以促进情绪和智力发展的能力。(Mayer & Salovey, 1997)

### 1.2 三大模型对比

| 维度 | Salovey & Mayer (1990) | Goleman (1995) | Bar-On (1997) |
|------|----------------------|----------------|---------------|
| 类型 | 能力模型 | 混合模型 | 混合模型 |
| 核心 | 情绪作为信息加工 | 情绪能力与人格 | 情绪-社交能力 |
| 维度 | 4个分支 | 5个领域(后修为4) | 5个复合量表 |
| 测量 | MSCEIT(能力测验) | ECI(360度评估) | EQ-i(自评) |
| 学术地位 | 最严谨 | 最流行 | 临床应用广 |

### 1.3 Mayer-Salovey四分支模型

| 分支 | 内容 | 复杂度 |
|------|------|--------|
| 1. 情绪感知 | 识别自己和他人面部、声音、图像中的情绪 | 基础 |
| 2. 情绪促进思维 | 利用情绪辅助判断、创造力和问题解决 | 中级 |
| 3. 情绪理解 | 理解情绪语言、情绪间关系、情绪变化规律 | 高级 |
| 4. 情绪管理 | 调节自己和他人的情绪以达成目标 | 最高 |

### 1.4 Goleman模型（修订版）

| 领域 | 能力 | 说明 |
|------|------|------|
| **自我觉察** | 情绪自觉 | 识别自己的情绪及其影响 |
| | 精确自我评估 | 了解自身优缺点 |
| | 自信 | 对自我价值的确信 |
| **自我管理** | 情绪控制 | 管理破坏性情绪 |
| | 透明度 | 诚实正直 |
| | 适应力 | 灵活应对变化 |
| | 成就导向 | 追求卓越 |
| | 主动性 | 把握机会 |
| | 乐观 | 积极看待事物 |
| **社交觉察** | 同理心 | 感知他人情绪和关切 |
| | 组织觉察 | 阅读群体情绪和权力关系 |
| | 服务导向 | 预见和满足他人需求 |
| **关系管理** | 影响力 | 有效说服他人 |
| | 培养他人 | 指导和辅导 |
| | 冲突管理 | 解决分歧 |
| | 团队合作 | 建设性协作 |

## 二、情商的科学基础

### 2.1 神经科学

| 脑区 | 情商功能 |
|------|---------|
| 杏仁核 | 情绪检测与反应（恐惧/威胁识别） |
| 前额叶皮质 | 情绪调节、决策、冲动控制 |
| 岛叶 | 内感受觉知（身体情绪信号） |
| 前扣带回 | 情绪冲突监控 |
| 镜像神经元系统 | 共情基础（观察=体验） |
| 梭状回 | 面部情绪识别 |

### 2.2 情商与表现

| 领域 | 研究发现 |
|------|---------|
| 工作表现 | EI解释工作表现的方差约为1-6%(能力模型)到15-20%(混合模型) |
| 领导力 | 高EI领导者的团队绩效更高(Rosete & Ciarrochi, 2005) |
| 心理健康 | EI与较低的抑郁、焦虑正相关(r=0.30-0.50) |
| 人际关系 | EI与关系满意度正相关(r=0.25-0.40) |
| 学业 | EI与学业成绩弱正相关(r=0.10-0.20) |
| 身体健康 | 高EI与更好的健康行为相关 |

## 三、核心情绪技能

### 3.1 情绪词汇表

| 基本情绪 | 强度梯度(低→高) |
|---------|----------------|
| 快乐 | 满足→愉悦→快乐→兴奋→狂喜 |
| 悲伤 | 失落→沮丧→悲伤→悲痛→绝望 |
| 愤怒 | 不满→恼怒→愤怒→暴怒→狂怒 |
| 恐惧 | 不安→担忧→害怕→恐惧→恐慌 |
| 惊讶 | 好奇→惊讶→震惊→目瞪口呆 |
| 厌恶 | 不适→反感→厌恶→憎恶 |

### 3.2 情绪识别训练

| 通道 | 训练方法 |
|------|---------|
| 面部表情 | 学习Ekman的微表情系统(FACS) |
| 声音语调 | 听声辨情绪练习 |
| 身体语言 | 观察姿势、手势、距离 |
| 内感受 | 身体扫描觉察情绪的身体信号 |
| 情境线索 | 理解情境对情绪的预测 |

## 四、情商发展

### 4.1 情商可以提高吗？

| 证据 | 发现 |
|------|------|
| 元分析 | Mattingly & Kraiger (2019): 情商训练有效(d=0.46) |
| 年龄效应 | EI随年龄温和增长 |
| 训练效果 | 短期培训效果有限，持续练习效果更好 |
| 关键因素 | 动机、反馈、练习机会是提升的关键 |

### 4.2 提升路径

| 能力 | 练习方法 |
|------|---------|
| 情绪觉察 | 正念冥想、情绪日志、身体扫描 |
| 情绪理解 | 扩展情绪词汇、学习情绪规律 |
| 情绪调节 | 认知重评、呼吸调节、自我对话 |
| 同理心 | 积极倾听、观点采择、共情练习 |
| 社交技能 | 非暴力沟通、反馈练习、冲突解决 |
