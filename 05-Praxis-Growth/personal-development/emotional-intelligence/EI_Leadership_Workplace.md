# 情商与领导力/职场 | EI Leadership & Workplace

> **学科定位**: 组织心理学 x 领导力发展 x 人力资源
> **核心主张**: 情商是区分优秀领导者和普通领导者的关键因素，在职场的每个层面发挥作用

---

## 一、情商领导力

### 1.1 Goleman的领导力研究

> 《What Makes a Leader》(HBR, 1998) — 情商领导力奠基之作

| 发现 | 数据 |
|------|------|
| 核心主张 | 情商是领导力最重要的因素 |
| 能力分析 | 区分明星绩效者的能力中，67%是情商相关 |
| 层级效应 | 领导层级越高，情商能力越重要 |
| 技术智商 | 技术和智商是"入门门槛"，情商是"区分因素" |

### 1.2 六种领导风格

> Goleman《Leadership That Gets Results》(HBR, 2000)

| 风格 | 情商基础 | 适用情境 | 对氛围影响 |
|------|---------|---------|-----------|
| 愿景型 | 自信、同理心、催化变革 | 需要新方向时 | 最积极 |
| 教练型 | 同理心、自我觉察、培养他人 | 发展人才时 | 积极 |
| 亲和型 | 同理心、关系建设、沟通 | 修复团队裂痕时 | 积极 |
| 民主型 | 合作、团队领导、沟通 | 需要共识时 | 积极 |
| 标杆型 | 成就导向、主动性 | 团队优秀且自主时 | 常为消极 |
| 命令型 | 成就导向、主动性、自控 | 危机或转型时 | 最消极 |

**最有效**: 灵活运用多种风格（"情商高的领导者至少擅长4种"）

## 二、职场情商应用

### 2.1 情商在不同职场场景

| 场景 | 需要的情商能力 | 具体应用 |
|------|-------------|---------|
| 面试 | 情绪感知、自我呈现 | 读懂面试官情绪、展现自信 |
| 向上管理 | 同理心、组织觉察 | 理解上司需求、适时汇报 |
| 团队协作 | 情绪管理、社交技能 | 化解冲突、激励同事 |
| 客户服务 | 同理心、情绪控制 | 处理投诉、建立信任 |
| 绩效反馈 | 情绪觉察、坦诚 | 给出建设性反馈 |
| 裁员/坏消息 | 同理心、情绪管理 | 以尊重和关怀的方式传达 |

### 2.2 情绪劳动

| 概念 | 说明 |
|------|------|
| 表层扮演 | 改变外在表达而不改变内在感受 | 
| 深层扮演 | 努力真正感受工作要求的情绪 |
| 情绪劳动代价 | 持续表层扮演导致情绪耗竭 |
| 应对策略 | 发展真实的情绪能力减少扮演需求 |

### 2.3 职场心理安全

> Amy Edmondson — 心理安全的团队更高绩效

| 领导行为 | 情商基础 |
|---------|---------|
| 承认自己的不确定和错误 | 自我觉察、谦逊 |
| 邀请成员发言和挑战 | 开放性、好奇心 |
| 对失败给予建设性回应 | 情绪管理、成长导向 |
| 积极倾听不同意见 | 同理心、社交觉察 |
| 表达感谢和认可 | 关系管理 |

## 三、团队情商

### 3.1 团队情商规范

| 层面 | 规范 | 具体行为 |
|------|------|---------|
| 个人层面 | 相互理解 | 了解每个成员的个人风格 |
| | 正面对待 | 假定善意、积极诠释 |
| 团体层面 | 团队自觉 | 定期检查团队情绪温度 |
| | 情绪表达 | 创造安全表达感受的空间 |
| 跨团队 | 组织理解 | 了解其他团队的需求和约束 |

### 3.2 情绪传染

| 概念 | 说明 |
|------|------|
| 情绪传染 | 团队成员的情绪会在群体中扩散 |
| 领导者效应 | 领导者的情绪对团队影响最大 |
| 积极螺旋 | 积极情绪传播→团队绩效提升 |
| 消极螺旋 | 消极情绪传播→合作破裂 |
| 管理策略 | 领导者要主动管理自己的情绪状态 |

## 四、建设性反馈

### 4.1 SBI反馈模型

| 步骤 | 英文 | 内容 | 示例 |
|------|------|------|------|
| S | Situation | 描述具体情境 | "在昨天的项目会议上" |
| B | Behavior | 描述具体行为 | "你打断了小李的发言三次" |
| I | Impact | 描述影响 | "这让小李不再愿意分享想法" |

### 4.2 接收反馈

| 步骤 | 说明 |
|------|------|
| 1. 暂停 | 管理第一反应（防御/愤怒） |
| 2. 倾听 | 真正理解反馈内容 |
| 3. 澄清 | 提问以确保理解准确 |
| 4. 感谢 | 感谢对方的勇气和关心 |
| 5. 反思 | 稍后理性评估反馈的价值 |
| 6. 行动 | 选择性采纳并改进 |
