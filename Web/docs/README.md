# Peace Lab Database 文档阅读器

欢迎使用 Peace Lab Database 知识库文档系统！

## 快速开始

### 使用侧边栏导航

在左侧侧边栏，您可以浏览项目的五大知识支柱：

- **🏛️ 智慧传承** — 哲学、宗教、瑜伽、中医养生等智慧传统
- **🧠 心智心理** — 冥想、心理学、人际关系、心理治疗方法
- **🧬 生物科学** — 生物学、死亡学、食物营养、性学研究
- **🎨 人文艺术** — 艺术、文学、电影、音乐、阅读心理
- **🌱 实践成长** — 沟通、个人发展、演讲资源、写作指南

点击各分类展开子目录，找到您感兴趣的主题。

### 使用搜索功能

侧边栏顶部有搜索框，您可以：

1. 输入关键词搜索全部文档内容
2. 使用中文或英文进行搜索
3. 点击搜索结果直接跳转到相关文档

### 阅读文档

- 文档支持完整的 Markdown 格式
- 代码块支持语法高亮和一键复制
- 页面底部有上一篇/下一篇导航
