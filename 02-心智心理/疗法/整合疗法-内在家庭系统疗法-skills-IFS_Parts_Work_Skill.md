## 6F 步骤详解

| 步骤 | 提问 | 目的 |
|:-----|:-----|:-----|
| **Find（找到）** | "你现在注意到身体/心里的哪个反应？" | 识别当下活跃的部分 |
| **Focus（聚焦）** | "把注意力放到那个感觉上" | 集中注意 |
| **Flesh out（具象化）** | "它在身体哪里？什么感觉？多大？" | 获取特征 |
| **Feel toward（感受）** | "你对这个部分有什么感受？" | **关键**：若有评判，先请评判部分退后 |
| **Befriend（结交）** | "它想为你做什么？它担心什么？" | 理解保护意图 |
| **Fear（恐惧）** | "如果它不这么做，最怕发生什么？" | 连接流放者 |

**核心原则**：所有部分都有正面意图。永远不与保护者对抗，先获得它们的许可。

## 适用场景示例

### 场景 1：自我批评内耗

> **用户**: "我总在心里骂自己没用，一边骂一边更不想做事"

**IFS 流程**:
1. Find → 识别"内在批评者"（管理者）
2. Feel toward → 用户对批评者感到愤怒 → 愤怒是另一个部分 → 请它退后
3. Befriend → 批评者说"我骂你是为了让你努力"
4. Fear → "如果我不骂你，你就不努力了，就会被淘汰"
5. 许可 → 批评者愿意让我们看看"被淘汰"的恐惧
6. 接触流放者 → 童年"不够好"的受伤小孩 → 卸下羞耻负担

### 场景 2：戒不掉的暴食

> **用户**: "我发誓不吃甜食，但压力一大就控制不住暴食"

**IFS 流程**:
1. 识别"暴食者"（消防员）
2. 理解它的功能："暴食让压力感暂时消失"
3. 它在保护什么 → 流放者"被压垮的孤独感"
4. 在 Self 状态下陪伴流放者 → 卸下孤独负担
5. 消防员卸下暴食角色 → 回到健康功能

## 转介指征（红旗）

| 信号 | 行动 |
|:-----|:-----|
| 出现解离/人格切换 | 暂停 IFS，转介 DID 专科 |
| 创伤闪回加剧 | 切换到稳定化/SE 躯体接地 |
| 自伤/自杀意念升级 | 启动危机协议 |
| 用户无法进入 Self 状态 | 先做 Self 接触练习，不强行进入流放者 |

## 注意事项

1. **永远先与保护者结盟**：跳过管理者/消防员直接接触流放者会导致系统崩溃
2. **Feel toward 是诊断核心**：用户对部分的感受告诉你是否有其他部分干扰
3. **Self 不是部分**：不要把慈悲/好奇当成"另一个部分"
4. **治疗师自身的部分**：若治疗师被触发，需先自我调节（co-regulation）
5. **节奏由保护者决定**：快速推进会导致消防员反应（恶化）
