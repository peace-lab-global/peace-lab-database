## 核心技术详解

### 1. 唤起水平评估

| 状态 | 神经系统 | 信号 | 干预 |
|:-----|:---------|:-----|:-----|
| **过度唤起** | 交感激活 | 心跳快、肌肉紧、呼吸浅、焦虑愤怒 | 先降唤起（接地/呼气延长） |
| **过低唤起** | 背侧迷走 | 麻木、瘫软、空白、解离 | 先升唤起（活动/定向） |
| **容纳窗口内** | 腹侧迷走 | 可对话、有感受、能连接 | 可进入滴定工作 |

### 2. 资源化（Resourcing）

引导用户找到身体的"安全岛"：
- "你身体哪个部位感觉最放松/安全？"
- "有没有一段让你感到平静的记忆？"
- "谁/什么让你感到安全？"

资源是钟摆的"安全端"，确保随时可退。

### 3. 滴定（Titration）

> "我们不去挖整座山，每次只取一小勺土。"

极小剂量地接触创伤性身体感受（如关注紧绷感的边缘 5 秒），然后立即回到资源。**宁可太少，不可太多**。

### 4. 钟摆（Pendulation）

```
   安全/平静 ←————→ 紧绷/不适
        ↑ 钟摆 ↑
```

每次摆动到不适边缘 → 回到安全 → 再推进一点。逐渐扩展耐受窗，让冻结能量在安全语境中缓慢释放。

### 5. 允许自发释放

当冻结能量开始释放时，身体可能出现：
- **颤抖/发抖**（最常见）
- **深呼吸/叹气**
- **打哈欠**
- **发热/出汗**
- **流泪**

**治疗师的角色**：识别、正常化、允许——"这是你的身体在释放，是安全的"。

### 6. 完成性动作

冻结时身体准备了战斗/逃跑动作但被抑制。引导这些**未完成的防御动作**缓慢、受控地完成（如手自然握紧再松开、腿做出轻微蹬踏）。

## 适用场景示例

### 场景 1：事故后冻结感

> **用户**: "车祸后我总觉得身体是僵的，动不动就冻住"

**SE 流程**:
1. 评估 → 过度/过低唤起交替
2. 资源化 → 找到双脚踩地的安全感
3. 滴定 → 关注"僵"的边缘感受 3 秒
4. 钟摆 → 回到双脚踩地 → 再触碰僵感
5. 允许 → 用户出现轻微颤抖 → 正常化
6. 完成 → 引导腿部做出轻微蹬踏动作（未完成的逃跑反应）

### 场景 2：解离倾向

> **用户**: "压力大我就整个人空白了，断片"

**SE 流程**:
1. 评估 → 过低唤起（解离）
2. 先升唤起 → 定向练习（环顾房间，命名 5 个看到的物体）
3. 接地 → 感觉椅子的支撑、双脚的位置
4. 建立最低身体觉察后再进入滴定
5. 不急于处理创伤内容，先建立调节能力

## 转介指征（红旗）

| 信号 | 行动 |
|:-----|:-----|
| 解离加重 | 停止 SE，回到纯接地稳定化 |
| 闪回加剧 | 可能需要 EMDR 专科评估 |
| 自伤/自杀升级 | 启动危机协议 |
| 出现精神病性症状 | 转介精神科 |
| 用户无法耐受任何身体觉察 | 转介先做能力建设（CBT/正念基础） |

## 注意事项

1. **慢即是快**：SE 的极慢节奏（滴定）恰恰是安全处理创伤的关键
2. **身体比故事重要**：不追问创伤细节，聚焦当下躯体感受
3. **颤抖是好事**：用户颤抖时常恐惧，治疗师要正常化并支持
4. **永远有退路**：每次接触不适前，先确认"安全端"可随时回到
5. **不挖故事**：过度追问细节可能再创伤
6. **治疗师自身调节**：SE 需要治疗师保持自身稳定（co-regulation）
