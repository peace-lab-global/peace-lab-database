## 关于本目录

`冥想整合/` 目录收录冥想与不同人类活动领域交叉整合的深度文档。这些文档旨在搭建冥想实践与具体生活/专业领域之间的桥梁，提供理论框架、实践技术、科学证据与案例研究。

## 文档清单

### 1. [Meditation_Creativity_Flow.md](应用-冥想CreativityFlow.md) — 冥想与创造力/艺术

**核心内容：**
- 创造力的心理学模型（Wallas四阶段、Csikszentmihalyi的Flow、Default Mode Network）
- 冥想促进创造力的神经机制（前额叶活动降低、DMN-ECN动态平衡）
- 分艺术门类的冥想技术：音乐即兴、视觉艺术、写作、舞蹈/运动即兴
- 科学证据：发散思维测验、艺术家群体研究、创意产业正念项目
- 案例研究：David Lynch、史蒂夫·乔布斯、Ray Dalio
- **图表：** 2+ mermaid 图示、多个对比表格

**关键词：** 创造力、Flow、默认模式网络、发散思维、心流、艺术治疗

### 2. [Meditation_Intimacy_Relationships.md](应用-冥想亲密关系.md) — 冥想与亲密关系

- 关系结束的正念支持（离婚/分手冥想、共同育儿、怨恨放下）
- 科学证据：HRV耦合、关系教育RCT、亲子正念效果
- **图表：** 2+ mermaid 图示、多个对比表格

**关键词：** 亲密关系、正念沟通、非暴力沟通、伴侣治疗、家庭正念、依恋理论

## 整合阅读路径

### 路径 A：个人应用导向

```
确定你的主要领域 → 阅读对应文档 → 选择1-2个入门练习 → 持续2周 → 加入第二个领域
```

### 路径 B：理论整合导向

```
阅读所有文档的"心理学模型/正念基础"章节 → 比较三个领域的 Flow 应用 → 提炼个人整合模型
```

### 路径 C：教学/教练导向

```
阅读对应领域的"科学证据" → 了解标准化方案（如MSPE） → 设计适配你对象的微型干预
```
