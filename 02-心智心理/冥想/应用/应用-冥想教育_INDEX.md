# meditation-education | 冥想与教育 Index

## 概述

本专题探讨**冥想作为教育内容**的完整图景：如何在学校、家庭与专业培训场景中教授冥想，如何设计适龄课程，以及如何评估教学效果。内容覆盖从儿童青少年正念课程到成人 MBSR 八周课程的教学设计，也涉及冥想教师的培训、督导与伦理规范。核心立场是：冥想教学不是技术的搬运，而是学习生态的营造——安全感、节奏与师生关系先于任何技法。

## 核心要点

- **分龄设计**：儿童课程以游戏化、短时、多感官为原则；青少年课程需回应学业压力与身份探索；成人课程可进入完整的八周结构化训练。
- **课程范式**：以 MBSR 为原型结构（每周一课 + 一日静修 + 每日家庭作业），并按场景裁剪为十日线、企业工作坊等变体。
- **教学安全**：冥想教育必须包含不良反应识别与转介机制，教师须明确自身角色边界（不诊断、不替代治疗）。
