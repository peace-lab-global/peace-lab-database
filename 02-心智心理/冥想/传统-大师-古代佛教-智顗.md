# 智顗（Zhiyi / 智顗大師）

> 天台宗的实际创立者——中国佛教止观冥想体系的集大成者

---

| 项目 | 信息 |
|------|------|

| 生卒 | 538-597 |
| 朝代 | 南北朝-隋朝 |
| 出生地 | 荆州（今湖北） |

## 二、核心教导（Core Teachings）

### 2.1 止观双运（Shamatha-Vipassana / 止观）

智顗建立了中国佛教中最完整的止观冥想体系：

| 概念 | 梵文 | 含义 |
|------|------|------|
| **止** | Shamatha | 安住心、止息妄念、定 |
| **观** | Vipassana | 如实观照、智慧洞察、慧 |

核心理念：止和观必须同时修习（止观双运），不可偏废。

### 2.2 《小止观》—— 最实用的冥想入门手册

《小止观》（又名《修习止观坐禅法要》）是中国佛教最经典的冥想入门文本：

| 章节 | 内容 |
|------|------|
| 具缘 | 修行的前提条件（持戒、衣食、闲静处、善知识） |
| 诃欲 | 远离五欲（色声香味触） |
| 弃盖 | 去除五盖（贪、嗔、睡眠、掉举、疑） |
| 调和 | 调身、调息、调心 |
| 正修 | 止观的具体修法 |
| 善根 | 修行中出现的善根征兆 |
| 觉魔 | 识别修行中的障碍 |
| 治病 | 通过禅修治疗疾病 |
| 证果 | 修行的成就 |

### 2.3 一念三千

天台宗独特的哲学核心：
- 一个念头中包含三千种世界/状态
- 心、佛、众生三无差别
- 一切现象互相包含（圆融）

## 三、对冥想传统的影响

| 影响 | 说明 |
|------|------|
| 日本天台宗 | 最澄大师将天台传统带到日本 |
| 日本禅宗 | 日本禅宗的止观基础来自天台 |
| 净土宗 | 天台的止观方法影响了念佛冥想 |
| 现代正念 | 止观双运的理念与现代正念的"注意力训练+洞察"结构相通 |
| 中国禅修 | 《小止观》至今仍是中国寺院最常用的冥想教材之一 |
