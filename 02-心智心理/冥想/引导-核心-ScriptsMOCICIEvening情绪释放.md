# MOCICI 晚间情绪释放冥想引导词 | Evening Emotional Release Script

> **所属课程**：MOCICI 心境重建训练营
> **类型**：晚间情绪释放 · 睡前过渡冥想
> **时长**：约 12–15 分钟
> **适用场景**：一天结束后的情绪清理、睡前放松过渡、自我疗愈
> **姿势**：仰卧（首选）或半靠坐姿
> **上次更新**：2026-06

---

## 实施说明

| 项目 | 要求 |
|------|------|
| 语速 | 90–100 字/分钟，温柔缓慢 |
| 语调 | 全程温暖、低沉、具有包裹感 |
| 背景 | 无音乐 或 极低音量自然白噪音 |
| 交付形式 | 录音产品 / 一对一私教 / 睡前团体 |
| 安全提示 | 如练习者出现强烈情绪反应，提示可随时睁眼、接地 |

## 台词本正文

### 第一段：开场安顿 `[00:00 – 02:40]`

`[语调：温暖、低沉、缓慢]`

你好，欢迎来到 MOCICI 心境重建训练营。欢迎来到这个属于一个人的冥想时刻。

`[停顿 3s]`

请你找到一个最舒适的姿势。可以平躺下来。

`[停顿 5s]`

也可以在椅子上微微向后依靠。让双手自然的、温柔的放在身体两侧，或者轻轻交叠在小腹上。

`[停顿 3s]`

请轻轻的闭上眼睛。

`[停顿 5s]`

现在告诉自己：接下来的几分钟，我不需要成为任何人，不需要做任何事，我只是在这里。安全的、完整的，与自己待在一起。

`[停顿 5s]`

请将你的注意力从外在的世界，慢慢的收回到你的呼吸上。不需要刻意改变呼吸的节奏。只是去观察它。

`[停顿 3s]`

去感受空气，经由鼻腔微微清凉的进入身体，感受你的胸腔和腹部，随着气息微微的、自然的起伏。

`[停顿 5s]`

在吸气时，在心里默念：**我觉察到我在吸入。**

`[停顿 3s]`

呼气时在心里默念：**我允许一切情绪随着呼气慢慢离开。**

`[停顿 10s]`

---

### 第二段：疗愈呼吸与躯体释放 `[02:41 – 06:28]`

`[语调：温暖、深情]`

现在让我们做三次深情的、疗愈的呼吸。深吸一口气，吸进宇宙间温柔的善意。

`[停顿 3s]`

缓缓的呼气。呼出今天所有的紧绷。

`[停顿 5s]`

`[语调：温柔、引导性]`

现在请把意识带到你的眉头，这里是否还在紧锁着？是否还堆积着白天的思虑？

`[停顿 3s]`

现在在呼气的时候，想象一阵温柔的风吹过这里。让眉心的皮肤变得平滑柔软。向下沉，向两侧舒展开。

`[停顿 5s]`

将意识带到你的肩膀，肩膀是我们习惯背负压力与责任的地方。

`[停顿 3s]`

现在，请深深地吸气。

`[停顿 2s]`

在呼气时，缓缓的呼气。让肩膀上所有沉重的、不属于你的重担，顺着你的手臂从指尖滑落。

`[停顿 5s]`

让肩膀像融化的黄油一样向下坠落。贴近床面或椅面。

`[停顿 5s]`

`[语调：更轻柔]`

将意识带到你的胸腔和胃部。这里是我们情绪的储存室。如果今天有什么让你感到委屈、愤怒或者焦虑，请不要评判他们。只是看着他们像一团灰色的雾气凝聚在那里。

`[停顿 5s]`

现在，吸气，吸入一缕光。

`[停顿 8s]`

---

### 第三段：自我慈悲 `[06:29 – 09:03]`

`[语调：温暖、深情、像对一个受伤的朋友说话]`

也许今天你经历了一些不愉快。或者你感到有些疲惫和无力。

`[停顿 3s]`

你可以轻轻的对自己说：

**我看见了你的辛苦。**

`[停顿 3s]`

**我允许这种感受存在。**

`[停顿 3s]`

**因为我不必对抗我的感受。**

`[静默 30s]`

---

### 第四段：叶子与河流 `[09:04 – 11:10]`

`[语调：平静、缓慢、像描述一幅画面]`

心里的那些杂念，那些烦躁，那些自我怀疑，就像一片片落叶落在了河面上。你不用去捞起它们，也不用去驱赶它们，你只是静静地看着。

`[停顿 5s]`

看这片叶子，随着河水的流动，慢慢的、慢慢的飘远，直到它消失在你的视线里。

`[停顿 10s]`

在心里对自己说：

**我愿意释放那些不再服务于我的情绪，今晚我只留温柔给自己。**

`[静默 30s]`

---

### 第五段：肯定语与温柔结束 `[11:11 – 结束]`

`[语调：温暖、稳定]`

现在请将意识带回你的呼吸。带回你此刻坐着的或躺着的这片空间。感受床或垫子温柔的承托着你，就像大地承托着万物一样。

`[停顿 5s]`

在今晚入睡前，请对自己说出这三句肯定语：

`[停顿 3s]`

**今天所有的发生，都是为了我的最佳利益。**

`[停顿 5s]`

**我原谅今天所有的发生。我放下今天所有的故事。**

`[停顿 5s]`

**此刻我是安全的。我是完整的，我被深深地爱着。**

`[停顿 8s]`

`[语调：极轻、极慢]`

现在，请慢慢的动动手指。动动脚趾。像一个刚刚睡醒的婴儿一样，轻轻地舒展一下身体，不需要急着睁开眼睛。

`[停顿 5s]`

如果你准备入睡，请继续保持这个放松的姿态。让意识沉入甜美的梦乡。

`[停顿 5s]`

如果你还要起身，请像从深海慢慢浮出水面一样。缓缓的睁开双眼。带着这份宁静回到当下的空间。

`[停顿 5s]`

`[语调：温暖收束]`

愿你的夜晚充满安宁与修复，晚安。愿你被世界温柔以待。

`[此后保持安静，让练习者自然入睡或缓慢起身]`

## 执行标记总览

| 标记类型 | 本脚本使用 |
|----------|----------|
| 姿势适应 | 仰卧（首选）/ 半靠坐姿 |
| 语调指引 | 温暖、低沉、深情、极轻极慢（末段） |
| 核心技法 | 呼吸觉察、躯体放松、自我慈悲、认知解离（叶子意象）、肯定语 |
| 创伤知情 | 非侵入性、提供选择权、不强制情绪表达 |
| 结束方式 | 双轨结束（入睡 / 起身） |

## 教学要点

| 常见情况 | 执行师回应 |
|----------|----------|
| 练习者中途哭泣 | 正常的情绪释放。温柔地说“允许泪水流动”，不打断引导节奏 |
| 练习者无法放松肩膀 | 可先做“耸肩—释放”动作预备（耸肩5s后突然释放），再进入引导 |
| 练习者报告“叶子飘不走” | 不评判。提示“也可以让那片叶子就停在那里，我们只是看着它” |
| 练习者不认同肯定语 | 可改为“我愿意探索这种可能性”的弱化版本 |
