# MBSR 正念减压 RCT 证据摘要

> **状态**: 实证 (A级证据)
> **最后更新**: 2026-05-18
> **证据等级**: A — 强证据（多项高质量RCT及多项元分析支持）

---

## 代表性 RCT 证据汇总

### 1. Carlson et al. (2007) — 乳腺癌生存者RCT

| 字段 | 内容 |
|------|------|
| **第一作者** | Carlson, LE |
| **发表年份** | 2007 |
| **期刊** | Psychosomatic Medicine, 69(7), 587-596 |
| **研究设计** | RCT + 等待名单对照 |
| **样本量** | N = 227（MBCT组 n=113，对照组 n=114） |
| **研究周期** | 8周干预 + 6个月随访 |
| **研究地区** | 加拿大 |

**研究人群**
- 诊断: I-III期乳腺癌生存者
- 时间: 诊断后平均 22 个月

**干预方案**
- MBSR标准8周课程（每周2.5小时）
- 包含身体扫描、正念瑜伽、冥想

**主要结局**

| 指标 | 测量工具 | MBSR组 | 对照组 | 结果 |
|------|---------|--------|-------|------|
| 压力感 | PSS | 显著降低 | — | d=0.42 |
| 焦虑 | STAI | 显著降低 | — | d=0.38 |
| 抑郁 | CES-D | 显著降低 | — | d=0.33 |
| 生活质量 | FACT-G | 显著改善 | — | d=0.35 |

**效应量**: d = 0.33-0.42
**证据等级**: A

---

### 2. Keng et al. (2021) — 压力管理元分析

| 字段 | 内容 |
|------|------|
| **研究类型** | 元分析 |
| **样本量** | 29项RCT，N=3,532 |
| **结论** | MBSR显著降低感知压力（SMD=0.37） |

**效应量**: SMD = 0.37
**证据等级**: A

---

### 3. Speca et al. (2000) — 癌症患者RCT

| 字段 | 内容 |
|------|------|
| **第一作者** | Speca, M |
| **发表年份** | 2000 |
| **期刊** | Cancer, 88(4), 858-867 |
| **研究设计**** | RCT + 等待名单对照 |
| **样本量** | N = 89（MBSR组 n=47，对照组 n=42） |

**主要结局**
- 情绪困扰显著降低（d=0.50）
- 睡眠质量改善（d=0.40）
- 效果维持至6个月

**效应量**: d = 0.40-0.50
**证据等级**: A

---

### 4. Grossman et al. (2004) — 慢性疼痛元分析

| 字段 | 内容 |
|------|------|
| **研究类型** | 元分析 |
| **样本量** | 20项研究，N=1,534 |
| **结论** | MBSR对慢性疼痛有效，疼痛强度降低 0.48 SD |

**效应量**: SMD = 0.48
**证据等级**: A
