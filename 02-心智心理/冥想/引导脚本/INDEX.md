# guided-scripts | 专业冥想引导词脚本库 Index

> **专业定位**：冥想引导词可复用脚本模板库与音频制作资产包

## 专业执行师使用指南

**实施与交付框架** (所有脚本均遵循)：
- **[执行标记系统]**：`[停顿 Xs]`、`[静默 X分钟]`、`[语调：温暖/稳定/坚定]`、`[创伤知情]`、`[禁忌：X]`、`[适应：坐姿/躺姿/行走]`、`[交付：团体/私教/录音/企业]`
- **教师要求**：每日自我练习30min+、每月督导、熟悉创伤知情原则、不越界诊断
- **交付矩阵**：现场团体 (1:12师生比)、一对一 (个性化调整)、专业录音 (90-110字/分钟)、企业工作坊 (ROI评估)
- **安全协议**：始终提供选择权、提前告知内容、识别转介信号 (强烈解离、持续低落、自伤意念)、准备本地心理热线资源
