# mandala-meditation | Index

## 概述

坛城（Mandala，梵语"圆轮具足"）冥想是藏传佛教金刚乘的核心观想实践之一：以高度结构化的圆形宇宙图式为所缘，观想自己与本尊居于坛城中央，借由"全景式"的观想统摄散乱之心。在藏传佛教传统中，坛城既是观修工具，也是"心性本具圆满秩序"的表征——外坛城映照内坛城，观修的最终指向是认识自心本有的完整与庄严。

## 核心要点

- **结构象征**：坛城通常外圆内方——外圈火焰与智慧护法象征防护与转化，中央宫殿与主尊象征觉醒的核心维度，四方位各有配色与寓意。
- **观修次第**：从粗大的外相观想入手，逐渐细化到种子字、光相与空性体验，遵循"生起次第 → 圆满次第"的传统路径。
- **心理维度**：荣格将曼陀罗视为"自性"（Self）整合的象征，推动了曼陀罗绘画在心理治疗与表达性艺术治疗中的应用；现代研究也关注坛城观想在注意训练与情绪整合上的作用。
- **跨学科视角**：本专题同时覆盖神经科学（观想训练的脑机制）、临床应用（绘画曼陀罗的疗愈使用）与传统仪轨三个层面。
