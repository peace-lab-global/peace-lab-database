# 道元禅师（Dōgen Zenji / 道元禅師）

> 曹洞宗的创立者——"只管打坐"的日本禅修大师

---

## 一、概述（Overview）

| 项目 | 信息 |
|------|------|
| 全名 | 道元希玄 / Dōgen Kigen |
| 生卒 | 1200-1253 |
| 出生地 | 日本京都 |
| 传统 | 曹洞宗（Sōtō Zen） |
| 核心贡献 | 将曹洞禅从中国带回日本并系统化 |
| 创立寺院 | 永平寺（Eiheiji，1244，福井县） |
| 核心著作 | 《正法眼藏》（Shōbōgenzō）—— 日本佛教哲学的巅峰之作 |
| 核心教导 | 只管打坐（Shikantaza） |

---

## 二、核心教导（Core Teachings）

### 2.1 只管打坐（Shikantaza）

- "只管"= 仅仅、专一地
- "打坐"= 坐禅
- 不设定任何冥想对象（无呼吸观、无公案）
- 仅仅是坐着，保持清醒觉知
- 坐禅本身就是觉悟的表达，而非达到觉悟的手段

### 2.2 修证一等

道元最革命性的教导：
- 修行（Practice）和证悟（Enlightenment）不是因果关系
- 修行本身就是证悟
- "修证一等"—— 修行和觉悟是同一件事
- 不是"为了"觉悟而打坐，打坐本身即觉悟的显现

### 2.3 《正法眼藏》核心概念

| 概念 | 日文 | 含义 |
|------|------|------|
| 现成公案 | Genjōkōan | 当下即是觉悟的完整表达 |
| 有时 | Uji | 存在即时间，时间即存在 |
| 身心脱落 | Shinjin Datsuraku | 超越身心二元对立的体验 |
| 佛性 | Busshō | 一切存在皆是佛性 |

## 三、对现代冥想的影响

| 影响 | 说明 |
|------|------|
| Shikantaza | 直接影响了 SFZC（铃木俊隆）等西方禅修中心 |
| 过程导向 | "旅途就是目的地"的理念与正念运动共鸣 |
| 日常即修行 | 洗碗、扫地都是修行——影响了正念的日常化 |
| 非目标导向 | 不追求特殊体验——与现代正念的"非评判"态度一致 |
