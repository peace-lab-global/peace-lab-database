# 佩玛·丘卓 (Pema Chödrön)

> "Nothing ever goes away until it has taught us what we need to know."
> --- Pema Chödrön

## 概述 (Overview)

佩玛·丘卓（Pema Chödrön，1936-）是藏传佛教宁玛派和噶举派最知名的西方女性导师之一。她以将藏传佛教中"无常"、"痛苦"和"困难情境"转化为觉醒契机的智慧，深深影响了西方世界的冥想实践。她长期担任加拿大新斯科舍省甘波修道院（Gampo Abbey）的首席驻院导师，是将藏传佛教核心教义以现代、通俗、且充满慈悲的方式传递给西方受众的关键桥梁人物。

## 生平与求道历程 (Biography & Spiritual Journey)

### 早年生活 (Early Life)

佩玛·丘卓出生于纽约市的一个中上阶层家庭，原名德尔德丽·布洛姆菲尔德-布朗（Deirdre Blomfield-Brown）。

- **教育背景**：在康涅狄格州的一家私立女子学校接受教育，后在加州大学伯克利分校获得文学学士学位
- **职业早期**：曾担任学校教师，与丈夫结婚并育有两子一女
- **人生转折**：35岁左右，她的婚姻突然破裂——丈夫告诉她，他有了外遇——这一事件成为她精神觉醒的起点

### 遇见佛法 (Encountering the Dharma)

婚姻的破裂将她推向了深度的心灵探索。她开始阅读各种灵性书籍，最终在加州的一次 retreat 中遇见了藏传佛教导师 ** Lama Chime Rinpoche **，并首次接触到藏传佛教的深邃智慧。

1972年，她在法国多尔多涅的 **Plum Village**（梅村）参加了一行禅师的禅修营，这进一步坚定了她走上 Buddhist path 的决心。

### 与创巴仁波切的因缘

1972年，她遇见了对她影响最深的上师——**秋吉·创巴仁波切（Chögyam Trungpa Rinpoche）**。创巴仁波切是藏传佛教噶举派和宁玛派的重要传承人，也是将藏传佛教引入西方的先驱之一。

在创巴仁波切的指导下：
- 1974年，她剃度出家，成为一名藏传佛教比丘尼
- 获得法名"佩玛"（Pema，意为"莲花"）和"丘卓"（Chödrön，意为"灯之光"）
- 接受了完整的藏传佛教修行训练，包括前行（ngöndro）、本尊修持（yidam practice）和大手印（Mahamudra）

### 甘波修道院时期

1984年，创巴仁波切在加拿大新斯科舍省建立了 **甘波修道院（Gampo Abbey）**，这是北美第一座藏传佛教西式寺院。佩玛·丘卓被任命为首任首席驻院导师（Abbess），并在此居住和教学超过40年，直到2025年圆寂。

## 核心教学理念 (Core Teachings)

### 1. 与困难共处的智慧

佩玛·丘卓最核心的教导是：**不要逃避痛苦，而要将其作为觉醒的入口**。

| 常见反应 | 佩玛·丘卓的教导 |
|---------|--------------|
| 逃避痛苦（distraction, addiction） | "走向痛苦，而不是远离它" |
| 对抗困难（resistance, anger） | "困难是唤醒我们的闹钟" |
| 自我批判（self-blame） | "对自己温柔，如同对待一个受伤的朋友" |
| 寻求速效解决方案 | "痛苦不会消失，直到它教会我们需要的功课" |

### 2. 无常（Impermanence）

受创巴仁波切"香巴拉路径"的影响，佩玛·丘卓深刻阐述了无常的积极面向：

> "无常意味着事情总在变化——这既包括痛苦会过去，也包括美好会过去。理解无常让我们不再那么执着于抓住什么，也不再那么恐惧失去什么。"

**核心实践**：将每一次失落、每一次意外的变化，都视为练习"放下"（letting go）的机会。

### 3. 自他交换（Tonglen）

佩玛·丘卓是在西方推广 **自他交换法（Tonglen）** 的最重要人物之一。这是一种藏传佛教的慈悲冥想：

- **吸气**：想象吸入他人的痛苦（以黑烟的形式）
- **呼气**：想象呼出自己的轻松、温暖和光明（以白光的形式）

**关键洞见**：Tonglen 不是自我牺牲，而是认识到"自我与他人并非分离"。在痛苦中，我们建立与一切众生的联结。

### 4. 中道与"不抓取、不拒绝"

佩玛·丘卓常用藏传佛教的术语 **Shenpa（缠绕/上钩）** 来描述人类痛苦的根源：

```
触发事件
    ↓
Shenpa —— 一种内在的"上钩"反应（难以抗拒的冲动）
    ↓
自动化反应：抓取（attraction）或拒绝（aversion）
    ↓
痛苦循环维持
```

**解脱之道**：在 Shenpa 升起的刹那，只是觉察它——"我注意到我正在上钩"——而不跟随自动化反应。这就是自由的时刻。

## 教学风格与方法 (Teaching Style)

### 特点

| 维度 | 描述 |
|------|------|
| **语言** | 通俗、口语化、充满幽默感；避免过多术语 |
| **基调** | 温暖、包容、无评判；常常自嘲 |
| **形式** | 讲座、闭关、著作、音频课程 |
| **受众** | 从完全的初学者到长期修行者 |
| **核心** | 将深刻的佛法转化为日常生活中的实用智慧 |

### 经典教学场景

佩玛·丘卓的 retreat 通常包括：
- **静坐冥想**（Shamatha / 止禅）
- **行走冥想**
- **Tonglen（自他交换）练习**
- **开示与问答**
- **阅读与讨论她的著作**

## 影响与遗产 (Influence & Legacy)

### 对西方佛教的影响

| 领域 | 贡献 |
|------|------|
| **女性修行** | 作为藏传佛教西方女性比丘尼的典范，激励了无数女性走上修行之路 |
| **困难转化** | 将"痛苦作为觉醒路径"的理念普及化，深刻影响了西方正念和心理健康领域 |
| **Tonglen 推广** | 使自他交换法从藏传佛教密法成为西方冥想者广泛练习的技术 |
| **香巴拉传承** | 延续并传播了创巴仁波切的香巴拉法教 |

### 与当代运动的联结

- **正念运动**：她的教导影响了大量正念教师对"困难体验"的处理方式
- **创伤知情正念**："与痛苦共处"的理念与创伤治疗的"耐受窗口"概念有深层呼应
- **自我慈悲运动**：Kristin Neff 的自我慈悲研究与佩玛的教导高度一致

### 争议

| 议题 | 说明 |
|------|------|
| **创巴仁波切的遗产** | 她长期是创巴仁波切的主要传承者之一。创巴仁波切晚年因酗酒等问题受到争议，佩玛对其持保护态度 |
| **圆寂** | 于2025年7月16日（其79岁生日后两天）在加拿大甘波修道院圆寂，留下深远的修行遗产 |
