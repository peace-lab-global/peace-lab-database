## 1. 八阶段引导流程详解

### 阶段一：入息准备（Internalization）— 5%

**目标**：帮助练习者从外在世界转向内在，建立安全的容器。

**引导要点**：
- 姿势调整：Savasana（摊尸式），身体完全放松
- 环境设定：温度、光线、安静
- 初步放松：三次深呼吸；觉察身体与地面的接触

**示例引导语**：
> "请找到一个舒适的位置躺下...让双腿自然分开，双臂放在身体两侧，掌心向上...
> 
> 轻轻闭上眼睛...开始将注意力从外在世界转向内在...
> 
> 感受身体与地面的接触...感受重力将你温柔地拉向大地...
> 
> 做三次深呼吸...吸气时感受身体的扩展...呼气时感受身体的放松..."

---

### 阶段二：Sankalpa（决心/誓言）— 5%

**目标**：在开放意识状态下植入一个简短、积极的意图。

**Sankalpa 的制定原则**：

| 原则 | 说明 | 示例 |
|------|------|------|
| **简短** | 一句话，最好 5-8 个词 | ✓ "我是平静和力量的" |
| **积极肯定** | 用现在时态；避免否定词 | ✗ "我不再焦虑" → ✓ "我是平静和安宁的" |
| **个人相关** | 针对当前生命阶段的核心需求 | 健康、勇气、接纳、爱 |
| **重复不变** | 每次练习使用相同的 sankalpa | 持续数周或数月 |

**示例引导语**：
> "现在，在心中 gently 升起你的 sankalpa——那个简短而有力的意图...
> 
> 用现在时态，清晰地陈述它...
> 
> 感受这个意图已经在你的生命中成真...
> 
> [暂停 10-15 秒]
> 
> 现在，放下 sankalpa，让它潜入意识的深层..."

---

### 阶段三：身体旋转觉知（Rotation of Consciousness）— 30%

**目标**：系统性地将意识移过身体各部位，诱发放松和躯体脱敏。

**标准顺序（Satyananda 体系）**：

```
1. 右脸侧面 → 右手大拇指 → 食指 → 中指 → 无名指 → 小指
   → 右手掌 → 手背 → 手腕 → 前臂 → 肘部 → 上臂 → 肩膀
   
2. 左脸侧面 → 左手大拇指 → 食指 → 中指 → 无名指 → 小指
   → 左手掌 → 手背 → 手腕 → 前臂 → 肘部 → 上臂 → 肩膀
   
3. 背部右侧 → 背部左侧
   
4. 右臀部 → 右大腿 → 右膝 → 右小腿 → 右脚踝 → 右脚背 → 右脚趾
   
5. 左臀部 → 左大腿 → 左膝 → 左小腿 → 左脚踝 → 左脚背 → 左脚趾
   
6. 会阴 → 生殖器官 → 腹部 → 胃部 → 胸部 → 喉咙
   
7. 下巴 → 嘴唇 → 舌头 → 鼻子 → 右眼 → 左眼 → 右耳 → 左耳
   → 头顶 → 后脑勺 → 整个头部
```

**技术要点**：
- **速度**：快速但清晰；每个部位 2-3 秒
- **不寻找感受**：只是"经过"每个部位，不强求感受
- **声音变化**：引导语的声音保持平稳、单调（促进放松）
- **重复**：可重复 2-3 轮

**示例引导语**：
> "将注意力带到右手的拇指...只是觉察它...现在移到食指...中指...无名指...小指...
> 
> 觉察整个右手掌...手背...手腕...前臂...肘部...上臂...右肩...
> 
> 现在移到左侧...左手拇指...食指...中指...无名指...小指..."

---

### 阶段四：呼吸觉察（Breath Awareness）— 10%

**目标**：通过呼吸深化放松，增强内感受觉察。

**常用技术**：

| 技术 | 描述 | 适用 |
|------|------|------|
| **自然呼吸觉察** | 只是觉察呼吸的自然流动 | 通用 |
| **数息** | 吸气数1，呼气数2...到10后重数 | 注意力分散时 |
| **定位呼吸** | 感受呼吸在特定身体部位 | 需要 grounding 时 |
| **等长呼吸** | 吸气和呼气等长（如4-4） | 需要平衡时 |

**示例引导语**：
> "现在，将注意力带到呼吸上...
> 
> 不需要改变呼吸...只是觉察它自然的流动...
> 
> 感受空气进入鼻孔...温暖地离开...
> 
> [可选] 在心中默默数息：吸气——1...呼气——2..."

---

### 阶段五：对立感受（Opposite Sensations）— 10%

**目标**：训练神经系统的灵活性和耐受性；促进左右脑整合。

**常用对立配对**：

| 配对 | 引导方式 |
|------|---------|
| **重 / 轻** | "感受身体变得沉重...像沉入大地...现在感受身体变得轻盈...像飘浮在空中..." |
| **热 / 冷** | "感受温暖在体内扩散...现在感受清凉..." |
| **痛 / 乐** | "[谨慎使用] 感受任何不适...现在感受愉悦..." |
| **扩张 / 收缩** | "感受身体向外扩展...现在向内收缩..." |

**技术要点**：
- 每种感受持续 30-60 秒
- 过渡要缓慢
- 创伤敏感：避免可能引发创伤的对立（如强烈的痛/乐）

---

### 阶段六：意象引导（Visualization）— 25%

**目标**：进入潜意识层面；促进情绪处理和洞察。

**常见意象序列**：

| 类型 | 示例 | 目的 |
|------|------|------|
| **自然场景** | 森林、海滩、山脉、花园 | 放松；自然联结 |
| **符号意象** | 光、火焰、莲花、河流 | 灵性象征 |
| **身体意象** | 光在体内流动；脉轮发光 | 能量工作 |
| **叙事意象** | 一个故事或旅程 | 潜意识探索 |
| **iRest 替代** | 感受、情绪、信念的觉察 | 去宗教化；心理化 |

**示例引导语（自然场景）**：
> "现在，想象你站在一片宁静的森林中...
> 
> 阳光透过树叶，在地上投下斑驳的光影...
> 
> 你闻到松树的清香...听到远处溪水的声音...
> 
> 感受微风轻抚你的皮肤...
> 
> [暂停 30-60 秒]
> 
> 在你面前，有一条小路...你沿着小路走去..."

---

### 阶段七：Sankalpa 重复 — 5%

**目标**：在深层状态下强化 sankalpa。

**技术要点**：
- 使用与阶段二**完全相同**的 sankalpa
- 在深层放松后重复，此时 sankalpa 更容易被潜意识接收
- 感受 sankalpa 已经在生命中成真

**示例引导语**：
> "现在，再次在心中升起你的 sankalpa...
> 
> [重复 sankalpa]
> 
> 感受这个意图已经在你的生命中成真...
> 
> 让 sankalpa 像种子一样，深深植入你意识的土壤中..."

---

### 阶段八：返回（Externalization）— 10%

**目标**：温和地回到清醒状态；整合体验。

**步骤**：
1. 重新觉察身体和地面的接触
2. 觉察环境中的声音
3. 重新觉察呼吸
4. 轻轻活动手指和脚趾
5. 缓慢地转向一侧，然后坐起

**示例引导语**：
> "现在，开始将注意力带回身体和这个空间...
> 
> 重新觉察你与地面的接触...
> 
> 听到周围的声音...
> 
> 做三次深呼吸...
> 
> 轻轻活动你的手指...和脚趾...
> 
> 当你准备好时，慢慢地转向右侧...
> 
> 用左手支撑，缓慢地坐起来...
> 
> 慢慢睁开眼睛..."

## 2. 核心技术要素

### 2.1 Sankalpa 的制定与使用

**制定流程**：

```
1. 识别当前生命阶段的核心需求
    └── 健康？勇气？接纳？爱？清明？

2. 用第一人称、现在时态、积极肯定表述
    └── ✓ "我是健康的"
    └── ✗ "我希望我能健康"（未来时）
    └── ✗ "我不再生病"（否定式）

3. 保持简洁（5-10个词）

4. 在 sankalpa 中感受情感
    └── 不是机械重复，而是感受它已在成真

5. 持续使用同一 sankalpa
    └── 至少 4-6 周；直到感觉"完成"
```

### 2.2 引导者的声音技术

| 要素 | 说明 |
|------|------|
| **语速** | 缓慢、平稳；身体旋转阶段稍快 |
| **音量** | 轻柔但清晰；在放松阶段更低 |
| **音调** | 平稳、单调（避免过多情感表达） |
| **节奏** | 与呼吸同步；在呼气时引导放松 |
| **暂停** | 频繁使用；给练习者内在处理的时间 |

### 2.3 创伤知情调整

| 标准元素 | 创伤知情调整 | 理由 |
|---------|------------|------|
| 闭眼 | 提供睁眼的选项 | 闭眼可能增加脆弱感 |
| 仰卧 | 提供侧卧或坐姿选项 | 仰卧对某些创伤幸存者不安全 |
| 身体扫描 | 快速经过；允许跳过区域 | 某些身体部位可能储存创伤记忆 |
| 深层意象 | 使用安全、中性的意象 | 避免可能触发创伤的象征 |
| 团体环境 | 确保 exits 可见；允许中途离开 | 安全感优先 |

## 3. 不同时长的练习设计

### 3.1 微型练习（5-10分钟）

| 阶段 | 内容 | 时长 |
|------|------|------|
| 入息准备 | 快速放松；三次深呼吸 | 1分钟 |
| 身体旋转 | 简化版（大区域而非逐部位） | 3-4分钟 |
| 呼吸觉察 | 自然呼吸 | 1-2分钟 |
| 返回 | 缓慢恢复 | 1分钟 |

**适用场景**：工作间隙；课间；快速恢复

### 3.2 短版练习（15-20分钟）

| 阶段 | 内容 | 时长 |
|------|------|------|
| 入息准备 | 标准放松 | 2分钟 |
| Sankalpa | 植入意图 | 1分钟 |
| 身体旋转 | 标准版 | 6-8分钟 |
| 呼吸觉察 | 数息或自然呼吸 | 3分钟 |
| [跳过对立感受和深层意象] | | |
| Sankalpa 重复 | 强化意图 | 1分钟 |
| 返回 | 标准恢复 | 2分钟 |

**适用场景**：午休；入睡前；日常维护

### 3.3 标准版练习（30-45分钟）

包含所有八个阶段，每个阶段充分展开。

**适用场景**：深度练习；retreat；治疗性应用

### 3.4 长版练习（60+分钟）

- 扩展的意象引导
- 多次身体旋转
- 更深的 sankalpa 工作
- 可能包含声音（颂钵、唱诵）

**适用场景**：专业 retreat；深度治疗；灵性探索

## 4. 各流派技术比较

| 维度 | Satyananda | iRest | Amrit Method |
|------|-----------|-------|-------------|
| **阶段数** | 8 | 10 | 7 |
| **sankalpa** | 核心 | 转为"意图/心愿" | 核心 |
| **意象** | 传统象征 | 去象征化；感受聚焦 | 较少意象 |
| **宗教元素** | 保留 | 去除 | 保留部分 |
| **身体旋转** | 快速 | 缓慢 | 能量感知 |
| **临床认证** | 无 | 有 | 无 |
| **适用** | 传统瑜伽 | 临床/世俗 | 能量工作者 |

### iRest 十阶段协议

| 阶段 | 内容 |
|------|------|
| 1 | 入息准备 |
| 2 | 意图（Intention） |
| 3 | 内心资源（Inner Resource）— 安全基地 |
| 4 | 身体扫描 |
| 5 | 呼吸和能量觉察 |
| 6 | 感受觉察 |
| 7 | 情绪觉察 |
| 8 | 信念和认知觉察 |
| 9 | 喜悦/幸福感 |
| 10 | 返回 |

**iRest 的独特元素**：
- **Inner Resource**：在开始前建立一个个人的"安全基地"意象
- **感受/情绪/信念的分层觉察**：更系统化的心理学化
- **去宗教化**：完全世俗化，适合临床环境

## 5. 引导词撰写原则

### 5.1 语言原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **现在时态** | 创造即时性 | "你现在正躺在..." |
| **第二人称** | 直接对话 | "你感受你的右手..." |
| **邀请式** | 尊重自主 | "你可以将注意力带到..." |
| **开放式** | 不预设体验 | "注意那里有什么感受..."（而非"感受温暖"） |
| **避免否定** | 潜意识不处理否定 | "保持清醒"（而非"不要睡着"） |

### 5.2 结构原则

- **由外到内**：从环境 → 身体表面 → 身体内部 → 心灵
- **由粗到细**：从大块身体区域 → 精细部位
- **由已知到未知**：从安全的体验 → 更深的探索
- **对称性**：左右交替；上下平衡

## 6. 常见挑战与应对

| 挑战 | 原因 | 应对 |
|------|------|------|
| **睡着** | 过于疲劳；过于放松 | 缩短时长；睁眼练习；坐直姿势 |
| **无法放松** | 焦虑；环境不适 | 调整环境；从呼吸开始；缩短练习 |
| **出现不舒服的情绪** | 潜意识材料浮现 | 这是正常的；温和地面对；必要时寻求支持 |
| **感到无聊** | 期望过高；习惯了刺激 | 接纳无聊本身；这是练习的一部分 |
| **无法跟随引导** | 注意力分散；语言障碍 | 使用母语录音；允许走神后再回来 |
| **身体不适（疼痛）** | 姿势不当；既有身体问题 | 使用支撑物（枕头、毯子）；调整姿势 |
| **解离感** | 深层放松的正常反应 | grounding 技术；缓慢返回 |
