# 胜利冥想 · 录制日清单

> **用途**：录音棚/工作室录制当天的操作手册
> **配套**：[脚本.md](脚本.md)（引导词主文件）
> **适用人员**：录制执行师、录音工程师、制作人

---

## 一、录制前准备（提前1天）

- [ ] 确认钢琴曲demo到位（C大调、70-90 BPM、明亮律动）
- [ ] 打印一份 脚本.md 纸质版（方便标记、翻页）
- [ ] 打印 发音.md 鼻音速查页，放在录音棚谱架上
- [ ] 确认录音环境：安静、温度适宜、手机飞行模式
- [ ] 设备测试：麦克风、耳机、录音软件、增益设置
- [ ] 确认采样率 ≥ 48kHz / 24bit

## 二、录制当天流程

### 2.1 热身阶段（10分钟）

| 时间 | 动作 | 要点 |
|------|------|------|
| 0-3min | 鼻音对比朗读 | 朗读 发音.md「最危险5组对比」×3遍 |
| 3-5min | 全文通读一遍 | 用铅笔标记个人易卡点（绕口处） |
| 5-8min | 节奏校准 | 跟钢琴demo走开场段，确认语速80-95字/分钟 |
| 8-10min | 喝水、调整姿势 | 找到胸腔共鸣的发声状态 |

### 2.2 录制阶段

**建议录制顺序**：

| 顺序 | 段落 | 预估时长 | 重点难点 |
|------|------|----------|----------|
| 1 | 开场·落地 | 2min | 语气定调——自信明朗、温暖有力 |
| 2 | 第一乐章·回顾 | 5min | 邀请式引导，不施压；躯体锚点自然过渡 |
| 3 | 第三乐章·宣告 | 5min | 高潮段需与钢琴四分音符同步（"一次。又一次。又一次。"） |
| 4 | 第二乐章·直面 | 5min | 不回避的清醒感；安全退出通道自然流畅 |
| 5 | 收束·带走 | 3min | 语速略缓；"存入"动作留出想象空间 |

> **为什么这个顺序？** 先录能量最高的第三乐章，趁状态饱满时锁定高潮；第二乐章直面段放最后，让情绪稳定收束。

### 2.3 每段录制要点

**开场·落地**
- [ ] 第一个字"欢迎来到"——声音要像门打开一样自然
- [ ] 脊柱描述时语速放慢，让听众跟上身体动作
- [ ] "身体重量……稳稳落在下方"——声音要有"沉下去"的质感

**第一乐章·回顾**
- [ ] 邀请式语气，不可变成命令
- [ ] "你走过来了"——轻轻说，如确认一个事实
- [ ] "不是第一次赢"——嘴角带笑，声音里有"我知道你行"的感觉

**第二乐章·直面**
- [ ] "不躲。不逃。"——三字句干脆利落
- [ ] 安全退出段（"如果感受太强烈……"）——声音自然转柔，不打断氛围
- [ ] "本身就是力量"——笃定、简短、不需要解释

**第三乐章·宣告（关键段）**
- [ ] "挺直脊柱"——声音要有物理感，像一根绳子拉紧
- [ ] "一次。又一次。又一次。"——**与钢琴四分音符严格同步**
- [ ] 音量三档：①→②→③ 自然递增，不可突兀跳变
- [ ] "Vijaya"——念出这个词时，声音里有庄重和喜悦
- [ ] 三重递进收束："更安静→更清醒→更笃定"——每层声音微微递进

**收束·带走**
- [ ] "存入脊柱……存入胸膛……存入双脚"——每处停顿2-3秒，给想象留空间
- [ ] 落地过渡（"先动动手指……"）——声音回到日常温度
- [ ] 最后一句——温暖收尾，像跟朋友说再见

## 三、质量控制检查

### 3.1 现场快速检查（每段录完后）

| # | 检查项 | 标准 | 通过 |
|---|--------|------|------|
| 1 | 有无明显口误/吞字 | 零容忍，有则重录 | ☐ |
| 2 | 语速是否在 80-95 字/分钟范围 | 与钢琴节奏匹配 | ☐ |
| 3 | 鼻音字清晰度 | 对照 发音.md 核查 | ☐ |
| 4 | 语气与标注是否一致 | 对照 脚本.md 语气方向标注 | ☐ |
| 5 | 停顿是否给够 | 对照 脚本.md 停顿标注 | ☐ |
| 6 | 音量三档切换是否自然 | 无突兀跳变 | ☐ |
| 7 | 呼吸声是否自然 | 标注处有意识捕捉 | ☐ |

### 3.2 后期混音检查

- [ ] 人声与钢琴比例：人声占主导，钢琴为人声的 30-40%
- [ ] 人声低频增强：+1.5dB @ 100-250Hz（增加胸腔共鸣感）
- [ ] Room Reverb：轻微添加，营造空间感但不模糊
- [ ] 停顿处钢琴自然填充，不完全静默
- [ ] 呼吸声保留，增加真实感和亲密感

## 四、交付验收标准

| # | 验收项 | 标准 |
|---|--------|------|
| 1 | 总时长 | 14:30-15:30 |
| 2 | 无底噪/杂音 | 安静段底噪 ≤ -60dB |
| 3 | 全程无口误 | 或已剪辑修复 |
| 4 | 人声清晰度 | 高频不刺耳、低频不浑浊 |
| 5 | 音乐融合度 | 钢琴不抢人声、停顿过渡自然 |
| 6 | 情绪完整性 | 从头到尾有清晰的情绪弧线 |

## 五、备注

- 如某段需要多次录制，保留所有take，后期选择最佳片段
- 第三乐章是整曲核心，建议预留充足时间，至少录2-3遍
- 录音师全程注意人声峰值，避免削波（clip）
- 录制过程中如发现鼻音问题，立即暂停参考 发音.md 修正
