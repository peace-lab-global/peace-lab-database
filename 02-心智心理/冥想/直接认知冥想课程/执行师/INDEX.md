# MOCICI 直接认知冥想 · 冥想执行师课程

> **Course 1: Meditator** — 港中文 & MOCICI 联合认证冥想执行师完整课程体系
>
> 核心理念：**看见 → 了解 → 接纳 → 转化** — 从间接认知走向直接认知
>
> 方法论：**觉察 → 设感 → 专注 → 观 → 直接认知**（前三为「止」，后二为「观」）
