# 流派对比表

> 《风暴中的宁静》与 MBSR、MBCT、传统 CBT、ACT 的详细对比。
> 用于读书会答疑、进阶学习、以及理解本书在正念谱系中的位置。

---

## 一、总览表

| 维度 | MBSR | MBCT | 传统 CBT | ACT | 本书《风暴中的宁静》 |
|---|---|---|---|---|---|
| **创始人** | Jon Kabat-Zinn | Segal / Williams / Teasdale | Aaron Beck | Steven Hayes | 冯晓东 |
| **诞生时间** | 1979 | 1990s | 1960s | 1980s | 2025 |
| **原始目标** | 压力管理、慢性病疼痛 | 预防抑郁复发 | 治疗焦虑/抑郁/多种障碍 | 提升心理灵活性 | 转化焦虑、抑郁、失眠 |
| **理论基础** | 正念 + 瑜伽 + 医学 | 正念 + CBT | 认知模型 + 行为技术 | 接纳承诺疗法 / RFT | 正念 + CBT + 五蕴 |
| **疗程结构** | 8 周，每周 2.5 小时 | 8 周，每周 2 小时 | 通常 12–20 次 | 灵活 | 自助书籍 + 8 音频 |
| **对念头的策略** | 觉察，不跟随 | 觉察，改变与念头的关系 | 识别、质疑、重构 | 解离，不纠结内容 | 觉察 + CBT 双框架 |
| **核心练习** | 身体扫描、正念瑜伽、静坐 | 身体扫描、认知解离、行走 | 认知重构、行为实验 | 接纳、解离、价值行动 | 数息、身体扫描、阴式呼吸 |
| **文化语境** | 美国医疗体系 | 英国 NHS 医疗体系 | 西方心理治疗 | 西方心理治疗 | 中国都市职场 |
| **宗教性** | 去宗教化 | 去宗教化 | 无 | 无 | 去宗教化，保留五蕴 |

## 二、核心理念对比

### 1. 对"念头"的态度

```
传统 CBT：识别 → 质疑 → 重构（改变内容）
MBCT：    觉察 → 标注 → 不跟随（改变关系）
ACT：     解离 → 接纳 → 转向价值行动（改变功能）
MBSR：    觉察 → 安住当下（培养觉察本身）
本书：    觉察 + 理解循环 + 逐环破解（整合觉察与 CBT）
```

### 2. 对"情绪"的态度

| 流派 | 核心态度 |
|---|---|
| 传统 CBT | 情绪由认知引发，改变认知以调节情绪 |
| MBCT | 允许情绪存在，不被情绪驱动 |
| ACT | 接纳情绪，同时按价值行动 |
| MBSR | 与情绪同在，觉察身体的情绪信号 |
| 本书 | 觉察 → 允许 → 转化；减少"第二支箭" |

### 3. 对"行为"的态度

| 流派 | 核心策略 |
|---|---|
| 传统 CBT | 行为实验、暴露、行为激活 |
| MBCT | 把觉察带入日常行为 |
| ACT | 承诺行动，按价值生活 |
| MBSR | 把正念带入日常活动 |
| 本书 | 从回避模式转向接纳行动 |

## 五、本书在流派谱系中的位置

```
直接性/深度 弱 ←──────────────────────────→ 强

传统 CBT    MBCT    MBSR    本书    ACT    佛教内观
(改变想法)  (觉察   (压力    (整合   (价值   (证知
            想法)   管理)    CBT+   行动)   实相)
                            五蕴)
```

**本书的独特位置**：
- 比传统 CBT 更强调觉察和接纳
- 比 MBCT/MBSR 更聚焦焦虑/抑郁/失眠三症状
- 比 ACT 更结构化、更少术语
- 比佛教内观更可操作、更去宗教化

## 六、给读书会主持人的提示

1. **被问"这本书和 MBCT 有什么区别"时**：
   > "MBCT 是 8 周团体课程，主要预防抑郁复发；本书是自助书，聚焦焦虑、抑郁、失眠三症状，并且用五蕴模型做了解释。"

2. **被问"是不是读这本书就够了"时**：
   > "如果症状较轻、想自助，这本书是很好的起点。如果症状严重或反复发作，建议在专业治疗同时进行。"

3. **被问"应该先学哪个"时**：
   > "想系统学正念 → MBSR；有抑郁复发史 → MBCT；想快速处理焦虑念头 → CBT；感到人生无意义、想澄清价值 → ACT；想一本书打通焦虑/抑郁/失眠 → 本书。"
