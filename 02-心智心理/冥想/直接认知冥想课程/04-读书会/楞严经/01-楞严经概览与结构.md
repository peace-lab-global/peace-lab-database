# 01 · 《楞严经》概览与结构

> 《楞严经》（全称《大佛顶如来密因修证了义诸菩萨万行首楞严经》）是汉传禅宗极受推重的经典，素有"开悟的楞严，成佛的法华"之说。它以一场"心在何处"的追问开端，逐步深入真心与妄心之辨，是禅修者辨认自心的地图。

## 概述

据传本经由般剌密谛（Paramiti）于唐神龙元年（705 年）译出，房融笔受，共十卷。近代以来，梁启超、吕澂等学者对其梵本来源提出质疑，学界多倾向认为它是汉地编集或高度汉化的大乘经典——但这并不影响它千年来在禅修传统中的实际地位：历代禅门大德（从永明延寿到虚云、宣化上人）多以本经印证见地。

## 十卷结构速览

| 段落 | 内容 | 关键词 |
|------|------|--------|
| 卷一 | 阿难示现堕淫室，佛说七处征心 | 心不在内、外、中间……何处？ |
| 卷二–三 | 十番显见，辨析见性与五蕴、六入、十二处、十八界皆如来藏妙真如性 | 见性不灭 |
| 卷四 | 富楼那问：清净本然，云何忽生山河大地 | 一念无明的世界起源论 |
| 卷五–六 | 二十五位圣者各述圆通法门；观音耳根圆通居末，文殊选为第一 | 反闻闻自性 |
| 卷七 | 楞严咒与建立坛场的轨则 | 护持行者 |
| 卷八 | 修证次第：干慧地、十信、十住、十行、十回向、四加行、十地、等觉 | 五十五位真菩提路 |
| 卷九–十 | 五十种阴魔——禅那中五阴区宇可能出现的境界偏差 | 不作圣心，名善境界 |

## 核心主题

- **七处征心**：佛问阿难"心在何处"，逐一破斥七种回答——不是为了给出第八个答案，而是让学人亲见"能找的那个"从来就不是真心。
- **二种根本**：无始生死根本（攀缘妄心）与菩提涅槃元清净体（常住真心）——修行全在辨清这两者，用错心，修如煮沙成饭。
- **耳根圆通**：观音法门"入流亡所"——不随声尘流转，返观能闻之性，是汉传佛教最受推崇的入手处。
- **五十阴魔**：禅坐中一切殊胜境界，"若作圣解，即受群邪"——这是禅门对修行境界最系统的风险提示，与当代"灵性绕道"（spiritual bypassing）的警示遥相呼应。

## 读书会提示

- 本经文字繁复，建议抓主线：卷一征心 → 卷二显见 → 卷六耳根圆通 → 卷十阴魔，其余可略读。
- 研读姿势宜"以经印心"而非"以解代修"：每读一段，停下来问——这在说我的哪一种经验？
