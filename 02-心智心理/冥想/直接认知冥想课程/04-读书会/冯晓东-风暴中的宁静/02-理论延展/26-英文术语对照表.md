# 英文术语对照表

> 《风暴中的宁静》及正念、CBT 领域常见英文术语的中英文对照与简要解释。

---

## 一、正念相关术语

| 英文 | 中文 | 解释 |
|---|---|---|
| Mindfulness | 正念 | 有意识地、不评判地专注于当下而升起的觉知 |
| Awareness | 觉察 | 对当下身心状态的注意与感知 |
| Attention | 注意力 | 心理资源指向特定对象的能力 |
| Presence | 临在 | 全然处于当下的状态 |
| Non-judgment | 不评判 | 不对经验贴好坏标签的态度 |
| Acceptance | 接纳 | 允许经验如其所是地存在 |
| Equanimity | 平等心 | 对各种经验保持平衡、不反应过激 |
| Beginner's Mind | 初心 | 以新鲜、开放的态度看待事物 |
| Letting Go | 放下 | 不执著于念头、情绪或经验 |
| Being Mode | 存在模式 | 当下导向、过程导向的体验模式 |
| Doing Mode | 行动模式 | 目标导向、结果导向的问题解决模式 |
