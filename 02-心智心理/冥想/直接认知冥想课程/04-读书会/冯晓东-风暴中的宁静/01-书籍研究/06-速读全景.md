## 命题一：恶性循环——焦虑/抑郁/失眠的共同根源

> "正念：离苦之道"——本书副标题即方法论。
> "前言标题"

**逻辑**：焦虑、抑郁、失眠看似是三个不同的问题，但它们共享同一个深层结构——**「负面思维—情绪—行为」的恶性循环**。

**循环的结构**：

```
        负面思维（自动化念头）
           ↗            ↘
   恶性循环               负面情绪（焦虑/低落）
           ↖            ↙
        回避/压抑行为
```

**关键论证——为什么循环会自我加强**：

| 环节 | 机制 | 例子 |
|------|------|------|
| 负面思维 → 负面情绪 | 自动化念头直接触发情绪反应 | "他没回我消息" → 焦虑 |
| 负面情绪 → 回避行为 | 不适感驱动逃避 | 焦虑 → 不敢发消息/回避社交 |
| 回避行为 → 负面思维 | 逃避强化了"果然有问题"的信念 | 关系疏远 → "果然他讨厌我" |
| 负面思维 → （更强的）负面情绪 | 循环升级 | 更焦虑 → 更回避 → 更疏远 |

**三个症状的共同根源**：

| 症状 | 表面特征 | 深层循环 |
|------|---------|---------|
| 焦虑 | 过度担忧未来 | 灾难化思维 → 焦虑 → 逃避 → 信心下降 → 更多焦虑 |
| 抑郁 | 反复反刍过去 | 自我贬低思维 → 低落 → 退缩 → 功能下降 → 更多贬低 |
| 失眠 | 无法停止思维 | 担心睡不着 → 警觉升高 → 更难入睡 → 更担心 → 循环 |

**核心洞见**：药物可以调节神经递质（治标），但如果**认知—情绪—行为的循环不打破**，停药后循环依然会重启。正念的切入点，是在循环的**每一环**插入一个"觉察"的间隙——**先看见，才能松开**。

**与叔本华"痛苦与无聊的钟摆"的结构平行**：

| 叔本华 | 冯晓东 |
|--------|--------|
| 意志的本质是无尽欲求 | 负面思维的本质是自动化循环 |
| 欲求受阻 = 痛苦 | 循环加强 = 症状加重 |
| 欲求满足 = 无聊 | 暂时缓解 = 回避行为的短效 |
| 钟摆永不停止 | 循环自我加强 |
| 解脱 = 意志的自我否定 | 转化 = 循环的觉察中断 |

## 命题二：觉察（Awareness）——看见循环的那一刻，循环就开始松了

> 觉察是后续一切转化的前提——先看见，才能松开。

**逻辑**：恶性循环之所以"恶性"，关键在于它是**自动化的**——我们没意识到自己在循环里。觉察的作用，是在刺激（事件）和反应（情绪/行为）之间插入一个**有意识的间隙**。

**觉察的三个层次**：

| 层次 | 含义 | 例子 |
|------|------|------|
| 身体觉察 | 感知身体的信号——紧张、呼吸、心跳 | "我注意到肩膀紧了" |
| 情绪觉察 | 识别当下的情绪，给它命名 | "这是焦虑，不是愤怒" |
| 思维觉察 | 观察自动化念头，不与之认同 | "我注意到我有个念头：'他讨厌我'" |

**关键区分——觉察 vs 思考**：

| 维度 | 思考（日常模式） | 觉察（正念模式） |
|------|-----------------|-----------------|
| 与念头的关系 | 被念头卷走（在念头里） | 观察念头（在念头外） |
| 时间指向 | 过去（反刍）或未来（担忧） | 当下（此时此刻） |
| 评判 | 自动评判（好/坏/对/错） | 不评判（如实看见） |
| 结果 | 强化循环 | 中断循环 |

**练习：正念呼吸**（第 2 章）

呼吸是觉察训练最经典的"锚"——它永远在当下，不需要任何工具。把注意力放在呼吸上，走神了就带回来。这个"带回来"的动作，就是在练觉察力的肌肉。

**与直接认知冥想的关联**：觉察训练的深层目标，是从"被概念裹挟"转向"直接看见"——这与直接认知冥想的"如实知见"在结构上高度一致。详见 [04-与直接认知冥想课程的关联](../02-理论延展/04-与直接认知冥想课程的关联.md)。

## 命题三：感知（Perception）——如实观察，卸下过去的滤镜

> 我们如何感知这个世界——过去影响我们的感知——正念感知，放下过去。

**逻辑**：我们的感知从来不是"客观"的。**过去经验（记忆、创伤、信念）给当下加了一层滤镜**。正念感知 = 卸下滤镜，如实观察。

**感知的滤镜机制**：

| 滤镜类型 | 来源 | 对感知的影响 | 例子 |
|---------|------|-------------|------|
| 记忆滤镜 | 过去的类似经历 | 把当下"读"成过去 | "上次被拒绝，这次也会被拒绝" |
| 创伤滤镜 | 未处理的创伤 | 对特定刺激过度反应 | 声音/表情/场景触发闪回 |
| 信念滤镜 | 核心信念（我不好/世界不安全） | 选择性注意 confirming evidence | 只注意到被忽视，忽略被善待 |
| 文化滤镜 | 社会文化编程 | 用群体规范评判个体行为 | "男人不该哭" |

**正念感知的"卸滤镜"过程**：

```
日常模式：  事件 → [滤镜：记忆/创伤/信念] → 扭曲的感知 → 情绪反应 → 行为
                                                    ↑
                                               循环的起点

正念模式：  事件 → 觉察到"滤镜正在工作" → 暂停评判 → 如实观察 → 选择回应
                                       ↑
                                  循环的中断点
```

**"如实观察"的深度——与直接认知冥想的深层对话**：

本书第 3 章的"如实观察"概念，与直接认知冥想课程主线中的"直接认知 / 如实知见"存在深层结构平行：

| 直接认知冥想（课程主线） | 本书对应表达 |
|------------------------|-------------|
| 现量（pratyakṣa）——离分别、不错乱的直接感知 | "如实观察"——卸下滤镜的纯净注意 |
| 所缘（ālambana）——注意力的对象 | 呼吸、身体感受、声音等感知对象 |
| 遍计所执性——概念化造成的扭曲 | "过去的滤镜"——记忆/创伤/信念的干扰 |
| 依他起性——缘起的显现 | 如实感知事物本来的样子 |
| 止观（śamatha-vipaśyanā） | 正念呼吸（止）+ 如实观察（观） |

**关键洞察**：本书用 **CBT 的语言**（"认知扭曲""自动化思维"）把直接认知冥想中"如实知见"这个**有点抽象的核心**翻译成了现代人听得懂、练得动的入口。**先看见扭曲，才能放下扭曲。**

**练习：正念感知练习**（第 3 章）——对感官所缘的纯净注意，不加概念、不加评判。

## 命题四：认知（Cognition）——识别自动化思维与认知扭曲

> ⚠️ 第 4 章完整副标题待 APP 内核实。以下基于全书链条推断 + CBT 理论框架展开。

**逻辑**：在"感知"之上，处理**自动化思维**——即 CBT 中的「认知扭曲」。我们的念头不是事实，但如果没有觉察，我们会把念头当作事实来反应。

**CBT 十大认知扭曲**（本书第 4 章的理论基础，与正念认知高度关联）：

| 认知扭曲 | 定义 | 例子 | 正念解药 |
|---------|------|------|---------|
| 灾难化 | 把小事无限放大成灾难 | "他没回消息 → 他一定出事了/讨厌我了" | "这只是一个念头，不是事实" |
| 非黑即白 | 只有完美或失败两种 | "做不到100分就是0分" | "看见灰色地带" |
| 过度概括 | 一个事件推广到所有 | "这次失败 → 我什么都做不好" | "这是这一次，不是每一次" |
| 读心术 | 假定知道别人在想什么 | "他一定看不起我" | "我无法知道他在想什么" |
| 预言 | 预测未来且信以为真 | "我一定会搞砸" | "未来不可知" |
| 情绪推理 | 用感受来定义事实 | "我感觉很糟 → 事情一定很糟" | "感受是感受，事实是事实" |
| 应该陈述 | 用"应该"折磨自己 | "我应该更努力/更聪明/更好" | "放下'应该'，回到'是什么'" |
| 贴标签 | 给自己/他人贴固定标签 | "我是个失败者" | "我正在经历失败，但我不是失败" |
| 个人化 | 把不相关的事归因于自己 | "他心情不好 → 一定是我做错了" | "很多因素与我无关" |
| 过滤 | 只看负面，忽略正面 | 一天10件好事1件坏事，只记住坏事 | "也注意到好的部分" |

**正念对认知扭曲的作用机制**：

```
传统CBT路径：  识别扭曲 → 用理性反驳 → 认知重构
                                                 ↑
                                        需要"用力"——有时反而强化了思维

正念路径：    觉察到念头升起 → 标注"这是念头" → 不跟随/不压抑 → 念头自行消退
                                                    ↑
                                        不用力——只是"看见"并"放手"
```

**MBCT（正念认知疗法）的关键洞见**：认知行为疗法（CBT）教人"改变想法"，正念认知疗法（MBCT）教人**"改变与想法的关系"**——不是消灭念头，而是不再被念头卷走。这一区分是本书第 4 章与 MBCT 的深层连接点。

**与唯识学"想蕴"的对照**：

| CBT | 唯识学 | 本书的整合表达 |
|-----|--------|--------------|
| 自动化思维 | 想蕴（saññā）——概念化、标记、归类 | "念头"——对感知的概念化加工 |
| 认知扭曲 | 遍计所执性——虚妄分别 | "扭曲的认知"——对实相的错误建构 |
| 认知重构 | 转识成智——如实知见 | "如实认知"——从扭曲到如实 |

## 命题五：情绪（Emotion）——不是压住，而是觉察、允许、转化

> 正念从不是端坐蒲团上的刻板修行，也不是强行压制情绪的"心灵硬壳"——它可以藏在一场坦诚的哭泣里。
> ——豆瓣书评

**逻辑**：情绪不是要被"压住"的——压抑只会让情绪在别处爆发。正念的路径是：**觉察 → 允许 → 转化**。

**情绪的正念处理三步法**：

| 步骤 | 做什么 | 不做什么 |
|------|--------|---------|
| 1. 觉察 | "我注意到有一种情绪升起了" | 不忽视、不否认 |
| 2. 允许 | "这个情绪可以在这里" | 不压抑、不逃避、不评判"不该有" |
| 3. 转化 | "我选择如何回应这个情绪" | 不被情绪驱动做出自动化行为 |

**情绪的"两个箭"模型**（佛教与正念心理学共通）：

```
第一箭：不可控的事件本身带来的痛苦（身体的/情境的）
     ↓
第二箭：我们对痛苦的评判和反应带来的额外痛苦（"不该这样""我受不了"）
```

- **第一箭不可避免**——生活中必然有不如意
- **第二箭可以选择**——正念训练的核心就是减少第二箭
- **痛苦（pain）vs 苦（suffering）**：痛苦是第一箭，苦是第一箭+第二箭的总和

**与叔本华"痛苦是积极的"的对照**：

| 叔本华 | 正念心理学 |
|--------|----------|
| 痛苦是积极的、直接的感受 | 第一箭是不可避免的 |
| 快乐是消极的——只是痛苦的暂时消除 | 真正的"快乐"是第二箭的减少 |
| 意志的无尽欲求 → 必然痛苦 | 评判和执取 → 苦的叠加 |
| 审美沉思 = 暂时从意志中解脱 | 正念觉察 = 暂时从自动化反应中解脱 |
| 意志否定 = 永久解脱 | 觉察习惯化 = 长期转化 |

**情绪章的篇幅意义**：全书篇幅最重的章节之一（约 1.4 万字），说明作者认为情绪处理是正念落地的**关键节点**——从觉察到认知都是"看见"，到了情绪才真正需要"转化"。

## 命题六：行为（Behavior）——从回避模式转向接纳行动

> ⚠️ 行为章的确切章号待核实。以下基于全书链条推断 + CBT 行为激活理论展开。

**逻辑**：恶性循环的"行为"环节，核心特征是**回避**——逃避引发焦虑的情境，短期内减少不适，长期却强化了"那个情境确实危险"的信念。

**回避的代价**：

| 回避类型 | 短期效果 | 长期代价 | 例子 |
|---------|---------|---------|------|
| 情境回避 | 焦虑暂时下降 | 功能领域缩小，信心下降 | 不敢开会发言 → 能力被质疑 |
| 情绪回避 | 不适暂时缓解 | 情绪累积，突然爆发 | 压抑愤怒 → 某天爆发伤人/伤己 |
| 认知回避 | 暂时不用面对 | 问题恶化 | 拖延 → 越拖越焦虑 → 彻夜失眠 |
| 社交回避 | 暂时安全 | 孤立加重抑郁 | 不见朋友 → 更孤独 → 更低落 |

**正念行为转变的路径**：

```
回避模式：  不适 → 回避 → 短期缓解 → 信念强化 → 功能缩小 → 更多不适
                                                        ↑
                                                    循环升级

接纳模式：  不适 → 觉察到不适 → 允许不适存在 → 带着不适行动 → 发现"没有想象的糟" → 信心增长 → 循环减弱
```

**"接纳"不是"忍受"**：

| 接纳 | 忍受 |
|------|------|
| 清醒地看见不适，选择不逃避 | 咬牙切齿地忍耐 |
| 带着不适继续有价值的行为 | 被动承受，等待结束 |
| 减少第二箭（评判） | 强化第二箭（"不该这样"） |
| 长期 → 信心增长，功能扩大 | 长期 → 耗竭，功能缩小 |

**与佛教"正精进"的对照**：佛教八正道中的"正精进"不是"更用力"，而是**正确的努力方向**——不是与不适对抗，而是朝着有价值的方向行动。正念行为转变与此高度一致。

## 命题七：助眠（Sleep）——把前面的觉察整合为一套睡眠方案

> 正念助眠——将觉察、呼吸、身体放松整合为针对失眠的方案。

**逻辑**：失眠本质上是**唤醒系统过度激活**——身体和思维无法从"白天模式"切换到"休息模式"。正念助眠的核心，是用呼吸和身体觉察来**主动激活副交感神经系统**，让身体自然入睡。

**失眠的恶性循环**：

```
白天压力 → 神经系统过度唤醒
     ↓
夜间思维无法停止（反刍/担忧）
     ↓
入睡困难 → 焦虑"我睡不着"
     ↓
焦虑进一步激活唤醒系统
     ↓
更难入睡 → 更焦虑 → 循环
```

**正念助眠的破解路径**：

| 步骤 | 做什么 | 对应全书前序章节 |
|------|--------|----------------|
| 觉察到"思维在转" | 不对抗，只是注意到 | 第 2 章 · 觉察 |
| 回到身体感觉 | 把注意力从思维转向身体 | 第 3 章 · 感知 |
| 呼吸调节（阴式呼吸） | 用特定呼吸模式激活副交感神经 | 第 2 章 · 呼吸练习 |
| 身体扫描放松 | 逐部位释放身体紧张 | 练习清单 · 身体扫描 |
| 允许"睡不着" | 放下"必须睡着"的执念 | 第 5 章 · 情绪：允许 |

**与 CBT-I（失眠认知行为疗法）的关联**：

| CBT-I 要素 | 正念助眠对应 |
|-----------|-------------|
| 刺激控制（床=睡觉） | 正念感知——觉察在床上的身体状态 |
| 睡眠限制 | （本书未明确涉及） |
| 认知重构 | 正念认知——"睡不着不是灾难" |
| 放松训练 | 阴式呼吸、身体扫描 |
| 睡眠卫生教育 | （本书未明确涉及） |

**关键差异**：CBT-I 侧重**行为规则**（几点上床、几点起床），正念助眠侧重**内在状态转化**——不是控制睡眠，而是放下对睡眠的控制。

## 命题八：跳出循环（Loop）——解脱痛苦

> 第 8 章把全书方法论重新收束到"循环"这个核心隐喻,并回答一个关键问题:**谁才是循环的主人?**

**逻辑**:恶性循环之所以让人痛苦,不仅在于它自动运转,更在于我们误以为自己是它的奴隶。觉察的深入带来一个根本转变——我们开始看见循环本身,而不再只看见循环里的内容。

**循环的三个维度**:

| 维度 | 内容 | 正念介入点 |
|---|---|---|
| 因与果 | 每一个反应都会成为下一个反应的因 | 在"因"上断——选择不自动化反应 |
| 动力 | 逃避短期不适、追求短期舒适维持循环 | 看见动力,才能不被动力驱动 |
| 主人 | 真正的主人是觉察本身,不是被念头/情绪卷走的"我" | 从"我在焦虑"到"我注意到焦虑正在发生" |

**关键论点**:**我们不是循环的奴隶,可以成为循环的主人。** 这个"主人"不是意志力,而是稳定的觉察。

## 命题九：正念生活（Life）——享受当下

> 第 9 章把正念从"练习垫"带入日常生活,核心对照是**行动模式（Doing）vs 存在模式（Being）**。

**现代人的困境**:我们过度活在行动模式——永远在解决问题、达成目标、追求效率。行动模式本身没有错,但当我们只能活在其中,就会失去"正在活着"的体验。

| 行动模式（Doing） | 存在模式（Being） |
|---|---|
| 目标导向 | 当下导向 |
| "我要完成这件事" | "我正在经历这件事" |
| 关注结果 | 关注过程 |
| 容易焦虑/紧绷 | 容易放松/清晰 |
| 与正念的关系:正念不是取消行动,而是让我们在行动中有觉察 | 与正念的关系:存在模式是正念训练的直接产物 |

**生活中的正念**:一行禅师的"正念吃橘子"、本书的"正念行走",都是把日常动作变成正念入口——不是专门找时间修行,而是让修行发生在生活的每一刻。

## 命题十：由定生慧（Wisdom）——正念的升华

> 第 10 章是全书升华,核心论点:**由定生慧**——通过稳定的正念练习（定）,发展出清晰的洞察力（慧）。

**逻辑**:

```
定（稳定的觉察/专注力）
     ↓
观（如实观察/看见循环）
     ↓
慧（洞察力/对实相的直接认知）
```

**与直接认知冥想课程的深层关联**:

| 本书表达 | 课程主线术语 |
|---|---|
| 由定生慧 | 止观（śamatha-vipaśyanā） |
| 稳定的觉察 | 止（śamatha） |
| 如实观察 | 观（vipaśyanā） |
| 洞察力 | 现量（pratyakṣa）/直接认知 |

**关键洞察**:本书作为大众自助书,把"定→观→慧"翻译成了"练习→觉察→洞察";而直接认知冥想课程则进一步指向"见实相/解脱"。两者是同一座山的不同高度。

## 双框架分析：五蕴模型 × 认知行为模型

本书第 2 章的理论核心，是用**佛教「五蕴模型」+ 现代心理学「认知行为模型（CBT）」**双框架解释苦因。这是作者"海文学院 + 心理咨询师"双重背景的体现，也是本书区别于同类正念书籍的理论特色。

### 五蕴模型（佛教框架）

| 五蕴 | 梵文 | 含义 | 在感知链条中的角色 |
|------|------|------|-------------------|
| 色 | rūpa | 物质形态/身体 | 感知的对象（所缘） |
| 受 | vedanā | 感受（苦/乐/不苦不乐） | 对所缘的初始感受 |
| 想 | saññā | 概念化/标记/归类 | 对感受的概念化加工 |
| 行 | saṃskāra | 意志/行动/习惯模式 | 概念化驱动的行为反应 |
| 识 | vijñāna | 意识/觉知 | 贯穿整个过程的觉知 |

### 认知行为模型（CBT 框架）

```
事件 → 认知（自动化思维） → 情绪 → 行为
 ↑                                    ↓
 └──────── 结果成为新事件 ←──────────┘
```

### 双框架对照

| 五蕴 | CBT 对应 | 整合后的含义 |
|------|---------|-------------|
| 色（物质/身体） | 事件（触发情境） | 苦的"所缘"——外部的或身体的触发 |
| 受（感受） | 生理/情绪反应的初始信号 | 对事件的最初身体反应 |
| 想（概念化） | 认知（自动化思维） | 对感受的解读、评判、归因 |
| 行（意志/行动） | 行为 | 由认知驱动的行为反应 |
| 识（意识） | 觉察力 | 贯穿全过程的觉知——正念训练的核心 |

**双框架的互补关系**：

| 维度 | 五蕴模型擅长 | CBT 模型擅长 |
|------|-------------|-------------|
| 分析深度 | 本体论——"自我"如何在感知中构造 | 功能性——症状如何在认知-行为循环中维持 |
| 语言风格 | 抽象/形而上 | 具体/可操作 |
| 目标 | 解脱——从"我执"中解放 | 症状转化——缓解焦虑/抑郁/失眠 |
| 与正念的关系 | 正念是核心修行方法 | 正念作为CBT的"增强模块"（→MBCT） |
| 适用人群 | 有佛教/冥想背景的学员 | 都市上班族、临床来访者 |

**关键洞察**：双框架不是简单叠加，而是**互补深化**——五蕴模型提供了"苦为什么会自我构造"的本体论深度，CBT 模型提供了"苦如何在日常生活中维持"的可操作诊断。两者结合，既有深度（为什么），又有路径（怎么做）。

## 六个常见误读——读本书前必须知道的事

### 误读一："正念就是放松"

**真相**：正念的核心不是放松，而是**觉察**。放松可能是正念练习的**副产品**，但不是目标。如果把正念当作放松工具，就会在"放松不下来"时更加焦虑——这恰恰强化了恶性循环。正念是**觉察当前状态**，包括紧张、不安、走神——如实看见，而非改变成另一种状态。

### 误读二："正念是佛教修行，需要信仰"

**真相**：虽然正念源自佛教传统，但本书的正念心理学是**去宗教化**的——它用现代心理学（CBT）的语言重新表达了佛教的洞见。不需要成为佛教徒，不需要相信轮回或业力。正念是一种**注意力训练方法**，其有效性有神经科学研究支持，与任何宗教信仰无关。

### 误读三："正念可以替代药物/心理治疗"

**真相**：本书定位是**自助/辅助**，不是替代医疗。对于中度以上的焦虑、抑郁或慢性失眠，**专业医疗（药物治疗+心理治疗）是第一选择**。正念可以作为治疗的补充——增强自我调节能力、减少复发——但不应替代医生的治疗方案。书中推荐的评估工具（GAD-7/PHQ-9/ISI）正是为了帮助读者判断是否需要专业帮助。

### 误读四："正念就是不想、不评判"

**真相**：正念不是"不想"，而是**觉察到自己在想**。念头不是敌人，觉察到念头后不与之认同、不被卷走，才是正念的核心。同样，"不评判"不是"没有评判"——而是**觉察到评判升起，但不被评判驱动**。把"不评判"变成"应该不评判"，恰恰是一种评判。

### 误读五："心灵健身房 = 更努力地训练"

**真相**：本书最著名的比喻"心灵健身房"容易引发一个误读——把正念变成另一种"用力"。正念的"训练"不是"更用力地集中注意力"，而是**更温柔地、更持续地回到当下**。走神不是"训练失败"，而是"肌肉在生长"的过程——发现自己走神并带回来，就是训练本身。

### 误读六："正念只适合有心理问题的人"

**真相**：正念不仅是"治苦"工具，更是**生命品质的提升**。即使没有明显的焦虑/抑郁/失眠，正念练习也能增强注意力稳定性、情绪调节能力、人际敏感度和整体幸福感。本书的"心灵健身房"比喻恰恰暗示——正念是日常锻炼，不是等生病才去医院。

| 五蕴 | 五蕴模型 | 完全一致的框架 | 本书与CBT并用，课程在唯识框架下深化 |
| 离苦 | "离苦之道" | 同一目标 | 本书偏"症状转化"，课程偏"根本解脱" |
| 现量（pratyakṣa） | 如实观察 | 都指向"离分别的直接感知" | 课程的现量含瑜伽现量（修行者深定中的直接认知），本书未涉及 |
| 遍计所执性 | "过去的滤镜" | 都指向"概念化造成的扭曲" | 课程更精细（四分说），本书更可操作 |
| 止观（śamatha-vipaśyanā） | 正念呼吸+如实观察 | "止"=呼吸定力，"观"=如实观察 | 结构高度一致 |

### 三个关键差异

| 维度 | 本书 | 直接认知冥想课程 |
|------|------|----------------|
| 目标层级 | 症状转化（焦虑/抑郁/失眠） | 根本认知转变（见实相/解脱） |
| 理论框架 | CBT + 五蕴（双框架） | 唯识学（四分说、三性等） |
| 直接认知的深度 | 识别并放下认知扭曲 | 证知万法唯识、能所双亡 |

### 这不是冲突——而是"同一棵树的不同高度"

```
树冠 ← 本书：《风暴中的宁静》
  │     接住最多的人（从症状入口）
  │     语言：CBT + 五蕴
  │
树干 ← 课程教案：疗愈师/正念转化焦虑
  │     口语化、可逐日打卡
  │     语言：日常正念 + 焦虑转化
  │
树根 ← 课程主线：直接认知冥想
        向根本处去（从实相入口）
        语言：唯识、止观、现量
```

**读书会可抛给学员**：你想从"树冠"（先解决睡不着、焦虑）爬起，还是从"树根"（直接看实相）修起？三者能否并行？
