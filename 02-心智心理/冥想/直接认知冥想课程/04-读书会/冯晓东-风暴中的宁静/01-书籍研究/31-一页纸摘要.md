# 一页纸摘要 ·《风暴中的宁静》

> **冯晓东用「正念心理学」把焦虑、抑郁、失眠的深层根源归结为「负面思维—情绪—行为」的恶性循环，再用佛教五蕴模型与现代认知行为疗法（CBT）双框架逐环破解——在身心的风暴中心，借正念获得宁静。**

---

## 书眼金句

> 中道的智慧，是一种风暴中心的宁静——正念可以让我们拥有风眼位置上的宁静与稳定感，静观这场风暴的生起和消退。（第10章）

---

## 全书骨架（十章递进链条）

```
第1章  正念 = 心灵健身房（总论：正念是可训练的心理能力）
  │
  ├─ 第2章  觉察  ── 找出苦因（CBT + 五蕴双框架）
  ├─ 第3章  感知  ── 如实观察（卸下过去的滤镜）
  ├─ 第4章  认知  ── 减少内耗（情绪ABC + 一念之转）
  ├─ 第5章  情绪  ── 离苦得乐（焦虑/抑郁/情绪三步法）
  ├─ 第6章  行为  ── 切断苦因（回避→接纳，行为转化）
  ├─ 第7章  健康  ── 正视疾病（身心关系 + 正念助眠）
  ├─ 第8章  循环  ── 解脱痛苦（因与果 + 循环的主人）
  ├─ 第9章  生活  ── 享受当下（行动模式 vs 存在模式）
  └─ 第10章 智慧  ── 由定生慧（定→观→慧）
```

---

## 五个核心概念

| 概念 | 一句话 | 章节 |
|------|--------|------|
| 恶性循环 | 焦虑/抑郁/失眠共享同一根源：负面思维→情绪→行为→强化 | 前言+第2章 |
| 心灵健身房 | 正念不是玄学，是可重复、可增量、可训练的心理肌肉 | 第1章 |
| 觉察→允许→转化 | 情绪不是被压住的，而是被觉察、允许、转化的 | 第5章 |
| 两支箭 | 第一箭不可避免（事件），第二箭可以选择（评判） | 第5章 |
| 由定生慧 | 定（稳定觉察）→观（如实观察）→慧（洞察力） | 第10章 |

## 六个常见误读

| 误读 | 真相 |
|------|------|
| "正念就是放松" | 核心是觉察，不是放松。放松是副产品 |
| "正念是佛教修行" | 去宗教化的注意力训练，不需信仰 |
| "可以替代药物" | 定位是自助/辅助，中度以上需专业医疗 |
| "正念就是不想" | 不是不想，而是觉察到自己在想 |
| "心灵健身房=更用力" | 不是更用力，而是更温柔地回到当下 |
| "只适合有心理问题的人" | 是日常锻炼，不是等生病才用 |
