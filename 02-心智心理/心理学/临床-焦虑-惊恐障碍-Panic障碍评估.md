# Panic Disorder Assessment (惊恐障碍评估)

> **惊恐障碍评估 (Panic Disorder Assessment / Aestimatio Morbi Panici)**
> 
> *"没有准确的评估，就没有有效的治疗。"*
> *"Without accurate assessment, there can be no effective treatment."*
> 
> 本文档系统呈现惊恐障碍的评估工具、诊断访谈技术及鉴别诊断框架，为精准诊断和治疗监测提供专业指导。

---

## 1. 评估术语表 (Assessment Terminology)

| 术语 (中文) | 英文名 | 精确定义 | 临床应用 |
|---|---|---|---|
| **惊恐发作频率** | Panic Attack Frequency | 单位时间内惊恐发作的次数 | 严重度指标；疗效指标 |
| **焦虑敏感性** | Anxiety Sensitivity | 对焦虑症状本身有害性的信念 | 认知脆弱性评估 |
| **内感受准确性** | Interoceptive Accuracy | 对身体内部信号的感知精度 | 内感受暴露基线 |
| **广场恐惧回避** | Agoraphobic Avoidance | 因恐惧而回避的情境范围 | 功能损害评估 |
| **身体警觉** | Body Vigilance | 对身体感觉的监控程度 | 维持因素评估 |

## 2. 惊恐障碍专用评估量表 (PD-Specific Scales)

### 表2.1 惊恐障碍严重度量表 (PDSS)

| 维度 | 内容 |
|---|---|
| **全称** | Panic Disorder Severity Scale (惊恐障碍严重度量表) |
| **开发者** | Shear et al. (1997) |
| **条目数** | 7项 |
| **评估维度** | (1)发作频率 (2)发作期痛苦 (3)预期焦虑严重度 (4)广场恐惧回避 (5)内感受回避 (6)工作功能损害 (7)社交功能损害 |
| **评分** | 每项0-4分；总分0-28分 |
| **临床切分** | 0-1=正常；2-5=边缘；6-9=轻度；10-13=中度；14-28=重度 |
| **施测方式** | 临床医生评定(PDSS)或自评(PDSS-SR) |
| **信度(α)** | 0.65(PDSS)；0.92(PDSS-SR) |
| **用途** | 疗效监测的金标准工具 |
| **优势** | 覆盖惊恐障碍各维度；对治疗变化敏感 |
| **局限** | 需培训(医生版)；PDSS-SR可替代 |

### 表2.2 焦虑敏感性指数-3 (ASI-3)

| 维度 | 内容 |
|---|---|
| **全称** | Anxiety Sensitivity Index-3 (焦虑敏感性指数第3版) |
| **开发者** | Taylor et al. (2007) |
| **条目数** | 18项 |
| **三个分量表** | (1)躯体关注(Physical Concerns, 6项) (2)认知关注(Cognitive Concerns, 6项) (3)社交关注(Social Concerns, 6项) |
| **评分** | 0-4分(非常少-非常多)；总分0-72分 |
| **惊恐障碍相关性** | 躯体关注分量表与惊恐障碍最密切相关 |
| **信度(α)** | 总量表0.93；各分量表0.80-0.90 |
| **用途** | 认知脆弱性评估；治疗靶点确定；预后预测 |

### 表2.3 广场恐惧认知问卷 (ACQ)

| 维度 | 内容 |
|---|---|
| **全称** | Agoraphobic Cognitions Questionnaire (广场恐惧认知问卷) |
| **开发者** | Chambless et al. (1984) |
| **条目数** | 14项 |
| **两个分量表** | (1)失控关注(Loss of Control, 7项) (2)躯体关注(Physical Concerns, 7项) |
| **评分** | 1-5分(从不-总是) |
| **临床意义** | 评估惊恐发作时的灾难化认知 |
| **信度(α)** | 0.80-0.86 |

### 表2.4 身体感觉问卷 (BSQ)

| 维度 | 内容 |
|---|---|
| **全称** | Body Sensations Questionnaire (身体感觉问卷) |
| **开发者** | Chambless et al. (1984) |
| **条目数** | 17项 |
| **评估内容** | 对17种身体感觉的恐惧程度(如心跳加速、头晕、出汗) |
| **评分** | 1-5分(一点也不害怕-极度害怕) |
| **与ACQ配对使用** | ACQ评估认知，BSQ评估躯体恐惧 |
| **信度(α)** | 0.87 |

### 表2.5 活动回避量表 (MI)

| 维度 | 内容 |
|---|---|
| **全称** | Mobility Inventory for Agoraphobia (广场恐惧活动清单) |
| **开发者** | Chambless et al. (1985) |
| **条目数** | 27项情境 |
| **评估模式** | 每项评2次：(1)独自时的回避程度 (2)有人陪伴时的回避程度 |
| **评分** | 1-5分(从不回避-总是回避) |
| **特色** | 区分"独自"vs"有陪伴"的功能差异 |
| **信度(α)** | >0.90 |

### 表2.6 量表选择决策指南 (Scale Selection Guide)

| 评估目的 | 推荐量表 | 施测时间 | 适用场景 |
|---|---|---|---|
| **筛查** | PHQ-PD(15项) | 3分钟 | 初级保健筛查 |
| **严重度评估** | PDSS-SR | 5分钟 | 专科；疗效监测 |
| **认知脆弱性** | ASI-3 | 5分钟 | CBT治疗计划 |
| **灾难化认知** | ACQ | 3分钟 | 认知重构靶点 |
| **躯体恐惧** | BSQ | 3分钟 | 内感受暴露计划 |
| **广场恐惧** | MI | 10分钟 | 暴露层级设计 |
| **综合评估** | PDSS + ACQ + BSQ + MI | 20分钟 | 初始全面评估 |

## 3. 惊恐发作日记 (Panic Attack Diary)

### 表3.1 惊恐发作记录模板 (Panic Attack Record Template)

| 记录项目 | 记录内容 | 目的 |
|---|---|---|
| **日期/时间** | 发作的具体日期和时间 | 识别时间模式 |
| **情境** | 发作时在哪里？在做什么？ | 识别触发因素 |
| **预期/意外** | 发作是预期的还是意外的？ | 诊断确认 |
| **首个注意到的感觉** | 最先出现的躯体感觉是什么？ | 识别触发感觉 |
| **症状清单** | 勾选出现的13项症状 | 症状概况 |
| **最高焦虑水平** | 0-10分(0=无；10=最严重) | 严重度监测 |
| **灾难化想法** | "我以为会发生什么？" | 认知重构靶点 |
| **持续时间** | 从开始到症状缓解约多久？ | 发作特征 |
| **安全行为** | 做了什么来应对？ | 安全行为识别 |
| **结果** | 实际发生了什么？ | 证伪灾难预测 |

## 4. 内感受敏感性评估 (Interoceptive Sensitivity Assessment)

### 表4.1 内感受暴露测试清单 (Interoceptive Exposure Test Battery)

| 练习 | 英文 | 操作方法 | 诱发感觉 | 持续时间 | 惊恐相关性 |
|---|---|---|---|---|---|
| **过度换气** | Hyperventilation | 快速深呼吸 | 头晕、麻木、不真实感 | 60秒 | 高(呼吸症状) |
| **转椅旋转** | Spinning | 坐椅上旋转 | 头晕、眩晕 | 60秒 | 高(前庭症状) |
| **通过吸管呼吸** | Straw Breathing | 用吸管呼吸(鼻夹住) | 气短、窒息感 | 120秒 | 高(呼吸症状) |
| **屏气** | Breath Holding | 深吸气后屏住 | 气短、心跳感 | 30秒或耐受极限 | 中(憋闷感) |
| **原地跑步** | Running in Place | 原地快跑 | 心跳加速、出汗 | 60秒 | 高(心血管症状) |
| **身体紧绷** | Body Tension | 全身肌肉同时紧绷 | 颤抖、肌肉紧张 | 60秒 | 中(紧张感) |
| **头部快速左右摇** | Head Shaking | 快速左右摇头 | 头晕、视觉模糊 | 30秒 | 中(头晕) |
| **热刺激** | Heat Stimulation | 手放暖气上/穿厚衣 | 发热、出汗 | 120秒 | 中(潮热) |
| **视觉凝视** | Visual Fixation | 盯住某点不眨眼 | 视觉模糊、不真实感 | 90秒 | 中(解体感) |

### 表4.2 内感受评估记录表 (Interoceptive Assessment Record)

| 记录维度 | 评分范围 | 说明 |
|---|---|---|
| **诱发感觉强度** | 0-8分 | 练习诱发了多强的身体感觉 |
| **焦虑水平** | 0-8分 | 感觉引起了多大焦虑 |
| **与惊恐发作相似度** | 0-8分 | 与实际惊恐发作的感觉有多相似 |
| **最相关感觉** | 文字描述 | 哪种感觉最让你害怕 |

## 5. 结构化诊断访谈 (Structured Diagnostic Interview)

### 表5.1 惊恐障碍临床访谈框架 (Clinical Interview Framework)

| 访谈领域 | 关键问题 | 评估目的 |
|---|---|---|
| **发作描述** | "请描述您最近一次惊恐发作的完整经历" | 确认是否符合惊恐发作标准 |
| **发作频率** | "过去一个月您有几次惊恐发作？" | 严重度评估 |
| **触发因素** | "发作前是否有明确的触发因素？" | 区分意外vs预期发作 |
| **首次发作** | "您还记得第一次发作是什么情况吗？" | 了解起病背景 |
| **预期焦虑** | "在发作之间，您是否担心下次发作？" | DSM-5标准B |
| **行为改变** | "因为惊恐发作，您的生活有什么改变？" | 功能损害评估 |
| **安全行为** | "您做了哪些事情来预防或应对发作？" | 安全行为识别 |
| **回避** | "有哪些地方或活动您因为害怕发作而回避？" | 广场恐惧评估 |
| **灾难化** | "发作时您最害怕什么？您以为会发生什么？" | 认知靶点确定 |
| **共病筛查** | 抑郁、其他焦虑、物质使用相关问题 | 共病评估 |
| **医学排除** | "您是否做过心电图/甲功检查？" | 排除躯体原因 |
| **治疗史** | "您之前接受过什么治疗？效果如何？" | 治疗计划参考 |

## 6. 鉴别诊断决策树 (Differential Diagnosis Decision Tree)

```
患者报告"惊恐发作"
    │
    ├─ 是否满足≥4项惊恐发作症状？
    │   ├─ 否 → 症状限制性发作(LSA)，监测观察
    │   └─ 是 → 继续评估
    │       │
    │       ├─ 是否有明确的躯体病因？
    │       │   ├─ 是 → 躯体疾病所致焦虑(需排除甲亢/心律失常/嗜铬细胞瘤等)
    │       │   └─ 否 → 继续评估
    │       │       │
    │       │       ├─ 是否与物质使用/戒断相关？
    │       │       │   ├─ 是 → 物质/药物诱发的焦虑
    │       │       │   └─ 否 → 继续评估
    │       │       │       │
    │       │       │       ├─ 发作是否"意外"(无明确触发)？
    │       │       │       │   ├─ 总是有明确触发 →
    │       │       │       │   │   ├─ 社交情境触发 → 社交焦虑障碍
    │       │       │       │   │   ├─ 特定对象触发 → 特定恐惧症
    │       │       │       │   │   ├─ 分离触发 → 分离焦虑障碍
    │       │       │       │   │   └─ 创伤提醒触发 → PTSD
    │       │       │       │   └─ 至少部分为意外 → 继续评估
    │       │       │       │       │
    │       │       │       │       ├─ 是否有持续担忧(≥1月)和/或行为改变？
    │       │       │       │       │   ├─ 否 → 单纯惊恐发作(非惊恐障碍)
    │       │       │       │       │   └─ 是 → 惊恐障碍(300.01/F41.0)
    │       │       │       │       │       │
    │       │       │       │       │       ├─ 评估广场恐惧共病
    │       │       │       │       │       ├─ 评估抑郁共病
    │       │       │       │       │       └─ 评估物质使用共病
```

## 7. 评估伦理与注意事项 (Assessment Ethics)

### 表7.1 惊恐障碍评估伦理要点 (Ethical Considerations)

| 伦理原则 | 具体要求 | 实践要点 |
|---|---|---|
| **医学排除优先** | 首次评估必须排除躯体疾病 | 推荐ECG、甲功检查(首次) |
| **正常化** | 评估过程中正常化惊恐发作(常见，非危险) | 减少患者病耻感 |
| **知情同意** | 内感受暴露测试前充分说明 | 尊重患者拒绝权 |
| **安全保障** | 评估中如引发严重焦虑需有应对预案 | 提供安全环境 |
| **文化敏感** | 考虑文化对躯体症状解释的影响 | 避免文化偏差 |
| **共病警觉** | 常规筛查自杀风险(尤其伴抑郁时) | PHQ-9 Item 9 |
