## 日常策略

### 每日练习清单

**早晨（准备日间）**：
- [ ] 回顾今天的价值承诺行动
- [ ] 准备面对可能的OCD触发
- [ ] 提醒自己"焦虑会来也会走"

**日间（应对OCD）**：
- [ ] 识别OCD思维（"这是OCD"）
- [ ] 使用解离技术
- [ ] 实践反应预防
- [ ] 记录成功和挑战

**晚间（反思）**：
- [ ] 记录今天的OCD事件
- [ ] 评估反应预防成功率
- [ ] 庆祝今天的进步
- [ ] 计划明天

### 快速应对技术

**当OCD发作时（5步）**：
1. **停**：停止当前行为
2. **吸**：深呼吸
3. **标签**："这是OCD"
4. **等**：允许焦虑自然消退
5. **行**：做价值导向的事

## 支持资源

### 自助资源

**书籍**：
- "Brain Lock" by Jeffrey Schwartz
- "The Mindfulness Workbook for OCD" by Jon Hershfield
- "Freedom from Obsessive Compulsive Disorder" by Jonathan Grayson

**应用程序**：
- nOCD（暴露追踪和症状监测）
- Joyable（ACT和CBT工具）
- Woebot（认知重构支持）

### 寻求专业帮助

**何时寻求专业帮助**：
- OCD严重影响了日常生活
- 自我帮助策略不够
- 出现抑郁或自杀念头
- 无法控制强迫行为

**专业治疗师类型**：
- 精神科医生（可开药）
- 临床心理学家（专门做ERP）
- 持证心理咨询师（需确认有OCD经验）

## 危机处理

### 识别危机信号

需要立即帮助的情况：
- 有自伤或自杀想法
- 无法控制冲动
- 严重的抑郁症状
- 感到无法应对

### 危机应对步骤

1. **安全第一**：如果担心自伤，确保身边有人
2. **联系支持**：打电话给亲友或专业人士
3. **热线帮助**：
   - 心理危机干预热线
   - 当地急诊
4. **制定安全计划**：记录可以做的安全行为

### 如果有伤害想法

**重要澄清**：
- 侵入性的伤害想法≠你想要伤害
- 这是OCD的典型症状
- 这种情况需要专业帮助
- 寻求帮助是正确的做法

## 维持进步

### 防止复发

**识别复发信号**：
- 强迫行为增加
- 回避行为增加
- 功能下降
- 自知力减弱

**复发预防计划**：
1. 识别个人OCD模式
2. 准备应对策略
3. 建立支持系统
4. 持续练习技能
5. 早期干预

### 长期维持策略

| 策略 | 说明 |
|------|------|
| 持续练习ERP | 即使好了也要偶尔练习 |
| 价值导向生活 | 关注生活而非症状 |
| 管理压力 | 压力会诱发OCD |
| 监测 | 继续使用OCD日志 |
| 社区支持 | 与理解你的人联系 |

### 成功指标

**你的进步证据**：
- 反应预防成功率提高
- 强迫行为减少
- 焦虑耐受提高
- 功能改善
- 价值行动增加

---

*OCD的康复是可能的，但通常需要时间、耐心和坚持。如果自助策略不够，请寻求专业帮助。*
