# 抑郁状态综合评估技能

## 技能信息

**技能ID**: S_010
**技能名称**: 抑郁状态综合评估
**功能**: 系统评估用户抑郁状态，筛查抑郁症状，判断严重程度，提供分级干预建议
**适用人群**: 主诉情绪低落、兴趣丧失、生活动力下降的用户
**理论基础**: DSM-5 重性抑郁障碍诊断标准；PHQ-9 患者健康问卷
**输出**: 结构化评估报告 + 严重程度分级 + 分层干预建议

---

## 评估协议

### 第一步：收集基本信息

通过开放式提问收集以下核心信息：

| 信息项 | 问题示例 | 重要性 |
|:-------|:---------|:------:|
| 情绪状态 | "最近心情怎么样？能描述一下吗？" | 必需 |
| 兴趣与愉悦感 | "以前喜欢做的事情现在还有兴趣吗？" | 必需 |
| 睡眠状况 | "睡眠怎么样？有没有入睡困难或早醒？" | 必需 |
| 食欲变化 | "最近胃口怎么样？有没有明显变化？" | 必需 |
| 精力水平 | "平时精力怎么样？容易累吗？" | 必需 |
| 注意力 | "能集中注意力做事吗？" | 必需 |
| 自我评价 | "你对自己现在的状态怎么看？" | 必需 |
| 自杀意念 | "有没有想过不想活了或者活着没意思？" | 必需 |
| 症状持续时间 | "这种状态持续多久了？" | 必需 |
| 功能影响 | "对你的工作/学习/人际关系影响大吗？" | 重要 |
| 既往史 | "以前有过类似经历吗？接受过治疗吗？" | 参考 |
| 家族史 | "家族中有人有类似的情绪问题吗？" | 参考 |

> **注意**: 询问自杀意念时需保持温和、非评判的态度，使用直接但关怀的方式提问。

### 第二步：PHQ-9 筛查

使用 **患者健康问卷-9（PHQ-9）** 进行标准化筛查：

**指导语**: "在过去两周里，以下问题困扰您的频率是？"

| 题号 | 问题 | 评分标准 |
|:----:|:-----|:---------|
| 1 | 做事时提不起劲或没有兴趣 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 2 | 感到心情低落、沮丧或绝望 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 3 | 入睡困难、睡不安稳或睡眠过多 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 4 | 感觉疲倦或没有活力 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 5 | 食欲不振或吃太多 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 6 | 觉得自己很糟糕——觉得自己是个失败者或让自己和家人失望了 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 7 | 对事物专注有困难（如阅读报纸或看电视） | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 8 | 动作或说话速度慢到别人可能注意到？或相反——因烦躁不安而比平时活动多得多 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |
| 9 | 有不如死掉或用某种方式伤害自己的念头 | 0=完全没有 1=好几天 2=一半以上天数 3=几乎每天 |

**评分说明**:
- 总分范围: 0-27 分
- 反向计分: 无（所有题目正向计分）
- 第9题（自杀意念）需单独标记，任何得分 ≥1 即触发安全评估

### 第三步：严重程度分类

| 总分 | 严重程度 | 临床意义 | 对应干预层级 |
|:----:|:--------:|:---------|:------------:|
| 0-4 | 极轻微 (Minimal) | 无明显抑郁症状 | Level 0 |
| 5-9 | 轻度 (Mild) | 可能存在轻度抑郁 | Level 1 |
| 10-14 | 中度 (Moderate) | 存在中度抑郁症状 | Level 2 |
| 15-19 | 中重度 (Moderately Severe) | 明显抑郁症状 | Level 2-3 |
| 20-27 | 重度 (Severe) | 严重抑郁症状 | Level 3 |

**附加判定规则**:
- 若 PHQ-9 第9题 ≥1，无论总分如何，均需进行自杀风险评估
- 若功能损害严重（无法工作/学习/自理），上调一个等级
- 若症状持续 ≥2年，考虑持续性抑郁障碍（Persistent Depressive Disorder）

### 第四步：鉴别诊断要点

| 鉴别方向 | 关键区分点 | 处理建议 |
|:---------|:----------|:---------|
| 正常悲伤 (Grief) vs 抑郁症 | 悲伤中仍可体验积极情绪，自我评价通常完整；抑郁症则持续兴趣丧失、自我价值感全面下降 | 评估丧失事件的时间线和情绪波动模式 |
| 双相障碍筛查 (Bipolar Screening) | 是否有过持续 ≥4天的情绪高涨、精力充沛、睡眠需求减少的时期 | 任何疑似躁狂发作 → 转介精神科评估 |
| 物质诱发抑郁 (Substance-Induced) | 症状是否与物质使用（酒精、药物）的时间线相关 | 评估物质使用史，建议戒断后重新评估 |
| 躯体疾病相关 (Medical Conditions) | 甲状腺功能减退、贫血、慢性疼痛、神经系统疾病等 | 建议基础体检（甲功、血常规、维生素D等） |
| 适应障碍 (Adjustment Disorder) | 症状是否与明确的应激源相关，且在应激源消除后6个月内缓解 | 评估应激源，区分情境性反应与内源性抑郁 |

## 干预建议层级

### Level 0 — 极轻微（0-4分）

- 无需临床干预
- 提供心理健康教育
- 建议维持健康生活方式

### Level 1 — 轻度（5-9分）— 自助策略

优先采用非药物干预：

1. **行为激活 (Behavioral Activation)**
   - 制定每日活动计划，增加愉悦性和掌控性活动
   - 从简单、可完成的任务开始，逐步增加难度
   - 记录活动-情绪日记

2. **运动处方 (Exercise Prescription)**
   - 每周 ≥150分钟中等强度有氧运动（如快走、游泳）
   - 规律运动已被证实具有抗抑郁效果（证据等级 A）

3. **社交连接 (Social Connection)**
   - 鼓励与信任的人保持联系
   - 减少社会隔离，参与轻度社交活动

4. **睡眠卫生 (Sleep Hygiene)**
   - 规律作息，固定起床时间
   - 减少睡前屏幕使用

5. **正念与放松 (Mindfulness & Relaxation)**
   - 每日10-15分钟正念冥想
   - 呼吸练习（参见 Bio_Breathwork_Pranayama.md）

### Level 2 — 中度（10-19分）— 建议专业评估

在 Level 1 策略基础上，强烈建议：

1. **专业心理咨询**
   - 认知行为治疗 (CBT): 针对抑郁的认知扭曲和行为模式
   - 人际关系治疗 (IPT): 针对人际关系问题引发的抑郁
   - 接纳承诺治疗 (ACT): 针对回避行为和价值澄清

2. **药物咨询 (Pharmacological Consultation)**
   - 建议精神科或心理科就诊
   - 常见一线药物：SSRIs（如舍曲林、艾司西酞普兰）
   - 药物治疗需在专业医师指导下进行

3. **定期复查**
   - 每2-4周重复 PHQ-9 评估
   - 监测症状变化和治疗反应

### Level 3 — 重度（20-27分）— 立即转介

1. **紧急精神科评估**
   - 强烈建议尽快就诊精神科或心理卫生中心
   - 评估是否需要住院治疗

2. **药物治疗 (Pharmacotherapy)**
   - 重度抑郁通常需要药物治疗
   - 可能需要联合用药方案
   - 考虑电休克治疗 (ECT) 或经颅磁刺激 (TMS) 作为辅助选择

3. **多学科团队协作**
   - 精神科医师 + 心理治疗师 + 社工
   - 制定个体化治疗计划

## 转介红线

以下情况需 **立即转介**，不适用本技能的常规评估流程：

| 红旗症状 | 严重程度 | 处置 |
|:---------|:--------:|:-----|
| **自杀意念或计划** | 🔴 紧急 | 立即启动危机干预，协助拨打急救电话或心理援助热线 |
| **精神病性症状**（幻觉、妄想） | 🔴 紧急 | 建议紧急精神科就诊 |
| **严重功能损害**（无法自理、无法进食） | 🟠 高危 | 建议24小时内就诊 |

| **妊娠期/产后抑郁** | 🟡 重点关注 | 建议围产期精神科门诊 |
| **儿童青少年抑郁** | 🟡 重点关注 | 建议专科评估，通知监护人 |

## 使用示例

**场景**: 用户主诉持续情绪低落，对生活失去兴趣

**用户**: "我最近几个月一直开心不起来，对什么都提不起兴趣，觉得自己很没用，活着没意思。"

**评估流程**:

```
1. 识别入口关键词: "开心不起来"、"没意思"、"没用"、"活着没意思"
2. 安全筛查: "活着没意思" → 需评估自杀意念
3. 启动技能 S_010

[安全确认]
Agent: "我听到你说活着没意思，我很在意你的感受。我想先确认一下，
       你有没有想过伤害自己或者不想活了的念头？"

用户: "有时候会想，但没有具体的计划。"

Agent: "谢谢你信任我告诉我。我接下来想做一些了解，帮助你更好地
       理解目前的状态，可以吗？"

[PHQ-9 筛查]
Agent: "接下来我想问你9个问题，了解你最近两周的状态..."
（逐一完成 PHQ-9 评估）

假设结果: PHQ-9 = 16分（中重度），第9题 = 1分（有自杀意念）
```

**输出报告**:
```
【抑郁评估报告】

元数据
- 技能ID: S_010
- 技能名称: 抑郁状态综合评估
- 调用时间: 2026-05-19 14:30
- 会话阶段: assessment

一、PHQ-9 总分: 16（中重度抑郁）
二、严重程度: 中重度 (Moderately Severe)
三、核心症状:
   - 持续情绪低落（数月）
   - 兴趣丧失（对所有活动）
   - 自我评价降低（"很没用"）
   - 存在被动自杀意念（无具体计划）
四、鉴别诊断提示:
   - 需排除双相障碍（询问躁狂史）
   - 需排除躯体疾病（建议甲功检查）
   - 需评估物质使用情况
五、建议行动:
   1. 【优先】建议尽快就诊精神科或心理科（Level 3 转介）
   2. 建立安全计划：识别危机时刻的应对策略和支持资源
   3. 告知心理援助热线
   4. 同步开始行为激活和社交连接
六、安全检查:
   - 红旗症状: 有（被动自杀意念）
   - 转介需求: 是（建议48小时内就诊）
   - 安全计划: 已建立
七、技能链状态更新:
   - 已完成技能: [S_010]
   - 待完成技能: [Crisis_Intervention（持续监测）]
   - 用户画像: {phq9: 16, severity: "moderately_severe", suicidal_ideation: "passive", duration: "数月"}
```
