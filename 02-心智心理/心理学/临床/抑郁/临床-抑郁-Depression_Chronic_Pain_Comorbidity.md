# Depression & Chronic Pain Comorbidity (抑郁症与慢性疼痛交叉专题)

> **抑郁症与慢性疼痛共病 (Depression-Chronic Pain Comorbidity)**
>
> *"疼痛和抑郁共享同一根神经——它们不是两个独立的问题，而是同一系统的两面。"*
> *"Pain and depression share the same neural roots—they are not two separate problems, but two faces of the same system."*
>
> 慢性疼痛与抑郁症是最常见的共病组合之一。约 30-50% 的慢性疼痛患者伴有抑郁，50-65% 的抑郁症患者报告慢性疼痛。两者相互加重、相互维持，形成「疼痛-抑郁恶性循环」。本文档整合其共享机制与整合治疗策略。

## 1. 流行病学与双向关系 (Epidemiology & Bidirectional Relations)

### 表1.1 常见慢性疼痛类型与抑郁共病率

| 疼痛类型 | 人群患病率 | 伴抑郁率 | 抑郁相对风险 | 主要特征 |
|---|---|---|---|---|
| **纤维肌痛综合征 (FMS)** | 2-4% | 20-80% | RR 3-5 | 广泛性疼痛 + 疲劳 + 睡眠障碍 + 认知困难 |
| **慢性腰背痛** | 10-20% | 30-50% | RR 2-3 | 最常见慢性疼痛；功能受限显著 |
| **紧张型头痛/偏头痛** | 10-15% | 40-50% | RR 2-3 | 5-HT系统功能障碍；发作-缓解模式 |
| **颞下颌关节紊乱 (TMD)** | 5-10% | 30-50% | RR 2-3 | 面部疼痛；与焦虑/抑郁高度共病 |
| **肠易激综合征 (IBS)** | 10-15% | 30-50% | RR 2-3 | 脑-肠轴功能障碍；功能性胃肠病 |
| **慢性盆腔痛** | 5-10%（女性） | 30-60% | RR 2-3 | 妇科/泌尿科常见；诊断困难 |
| **神经病理性疼痛** | 6-10% | 25-40% | RR 2 | 糖尿病神经病变、带状疱疹后神经痛 |
| **骨关节炎** | 10-15%（老年） | 20-30% | RR 1.5-2 | 退行性关节病；活动受限 |
| **类风湿关节炎** | 0.5-1% | 20-40% | RR 2-3 | 自身免疫；炎症驱动疼痛和抑郁 |
| **癌症相关慢性疼痛** | 30-50%（幸存者） | 30-40% | RR 2-3 | 治疗后遗症；生存恐惧 |

### 表1.2 疼痛-抑郁恶性循环模型

```
    慢性疼痛/组织损伤/炎症
           │
           ▼
    ┌──────────────┐
    │  疼痛信号上传  │ → 感觉皮层：疼痛感知
    │  (脊髓→丘脑→皮层) │ → 边缘系统：情绪反应
    └──────────────┘
           │
           ▼
    情绪反应：恐惧、焦虑、无助
           │
           ▼
    ┌──────────────┐
    │  中枢敏化     │ → 疼痛阈值降低
    │  下行抑制受损  │ → 内源性镇痛减弱
    │  炎症信号增强  │ → 促炎细胞因子↑
    └──────────────┘
           │
           ▼
    抑郁发作：情绪低落、兴趣丧失、无助感
           │
           ▼
    ┌──────────────┐
    │  抑郁加重疼痛  │ → 疼痛灾难化
    │  行为退缩      │ → 去conditioning、肌肉萎缩
    │  睡眠障碍      │ → 疼痛敏感化
    │  炎症状态      │ → 低度慢性炎症
    │  社交隔离      │ → 支持丧失、反刍
    └──────────────┘
           │
           ▼
    疼痛加重、扩散、慢性化
           │
           └──────→ 循环往复
```

## 2. 共享神经生物学机制 (Shared Neurobiology)

### 表2.1 疼痛-抑郁共享的神经解剖与递质系统

| 系统/结构 | 疼痛功能 | 抑郁功能 | 共病中的改变 | 治疗靶点 |
|---|---|---|---|---|
| **前扣带回皮层 (ACC)** | 疼痛的情绪/厌恶成分 | 情绪调节；错误监测 | 活动异常；灰质体积变化 | rTMS；药物 |
| **前额叶皮层 (PFC)** | 下行疼痛抑制；疼痛认知评估 | 执行功能；情绪调节 | 功能连接异常；体积减小 | CBT；rTMS；药物 |
| **岛叶** | 疼痛的内感受觉察 | 情绪意识；内感受 | 活动增强；与ACC连接增强 | 正念；药物 |
| **杏仁核** | 疼痛的恐惧/威胁评估 | 情绪反应；负性偏向 | 过度激活；对疼痛/情绪均过度反应 | 药物；暴露治疗 |
| **海马** | 疼痛记忆；情境学习 | 情绪记忆；HPA轴调节 | 体积减小；神经发生减少 | 运动；药物 |
| **导水管周围灰质 (PAG)** | 内源性镇痛中枢 | 情绪调节 | 功能异常；与PFC连接减弱 | 运动；药物 |
| **中缝核 (5-HT)** | 下行疼痛抑制；情绪调节 | 情绪稳定；冲动控制 | 功能降低 | SSRIs/SNRIs |
| **蓝斑核 (NE)** | 疼痛警觉；应激反应 | 警觉；能量；注意力 | 功能异常 | SNRIs；TCAs |
| **多巴胺系统** | 疼痛的奖赏/动机维度 | 快感；动机 | 功能降低 | 安非他酮；多巴胺能药物 |
| **内源性阿片系统** | 天然镇痛 | 情绪调节；社交纽带 | 功能降低 | 运动；社交；药物间接调节 |
| **谷氨酸系统** | 疼痛传递（NMDA）| 神经可塑性；认知 | 过度活跃 | 氯胺酮；美金刚 |
| **HPA轴** | 应激-疼痛敏化 | 应激反应；情绪调节 | 过度激活；负反馈受损 | 运动；正念；药物 |
| **炎症系统** | 外周/中枢炎症致痛 | 炎症致抑郁（细胞因子假说）| IL-6、TNF-α、CRP升高 | 抗炎药物；ω-3；运动 |

### 表2.2 炎症：疼痛与抑郁的共同土壤

| 炎症介质 | 疼痛作用 | 抑郁作用 | 来源 |
|---|---|---|---|
| **IL-6** | 中枢敏化；痛觉过敏 | HPA轴激活；5-HT合成抑制；快感缺失 | 巨噬细胞、脂肪细胞、免疫细胞 |
| **TNF-α** | 神经病理性疼痛；痛觉过敏 | 神经发生抑制；抑郁样行为 | 巨噬细胞、小胶质细胞 |
| **IL-1β** | 热痛觉过敏；炎症痛 | 行为绝望；食欲改变；睡眠障碍 | 小胶质细胞、星形胶质细胞 |
| **CRP** | 全身炎症标志 | 抑郁风险预测；治疗反应预测 | 肝脏（受IL-6刺激） |
| **前列腺素E2** | 外周/中枢痛觉敏化 | — | COX-2催化 |
| **神经生长因子 (NGF)** | 炎症性疼痛；神经可塑性 | — | 免疫细胞、角质细胞 |

> **临床要点**：高炎症标志物（hs-CRP > 3 mg/L）的患者对常规抗抑郁药反应较差，但对抗炎策略（如英夫利昔单抗、ω-3）和特定抗抑郁药（如安非他酮、伏硫西汀）可能反应更好。

## 3. 临床评估 (Clinical Assessment)

### 表3.1 疼痛-抑郁整合评估框架

| 评估维度 | 内容 | 工具 | 目的 |
|---|---|---|---|
| **疼痛特征** | 部位、性质、强度、时间模式、加重/缓解因素 | VAS/NRS、McGill疼痛问卷、疼痛日记 | 疼痛特征化 |
| **疼痛影响** | 功能、睡眠、情绪、工作、关系 | BPI（Brief Pain Inventory） | 疼痛多维影响 |
| **抑郁严重度** | 情绪、认知、躯体、自杀 | PHQ-9、HAM-D | 抑郁分层治疗 |
| **灾难化** | 疼痛相关灾难化思维 | PCS（Pain Catastrophizing Scale） | CBT核心靶点 |
| **恐惧-回避** | 对活动和疼痛的恐惧 | TSK（Tampa Scale of Kinesiophobia）、FABQ | 行为激活靶点 |
| **自我效能** | 疼痛自我管理能力 | PSEQ（Pain Self-Efficacy Questionnaire） | 预测功能恢复 |
| **疼痛接受** | 对疼痛的心理接纳 | CPAQ（Chronic Pain Acceptance Questionnaire） | ACT核心指标 |
| **功能状态** | 日常活动、工作能力 | ODI（Oswestry）、NDI、FRI | 康复目标 |
| **睡眠** | 疼痛对睡眠的影响 | PSQI | 睡眠干预靶点 |
| **阿片使用** | 阿片依赖/滥用风险 | ORT（Opioid Risk Tool） | 安全管理 |
| **神经病理性成分** | 烧灼感、刺痛、感觉异常 | DN4、PainDETECT | 指导药物选择 |

### 表3.2 疼痛相关灾难化思维（CBT核心靶点）

| 灾难化维度 | 典型思维 | 替代思维 |
|---|---|---|
| **放大 (Rumination)** | "我无法停止想这个痛" "痛占据了我全部注意力" | "我可以将注意力转移到其他事物上" "痛只是体验的一部分" |
| **无助 (Helplessness)** | "我什么都做不了" "没有药能帮我" | "我可以做很多事情来管理疼痛" "虽然痛，但我仍能行动" |
| **夸大 (Magnification)** | "这个痛会越来越严重" "我会残废" | "疼痛会波动；有起伏是正常的" "我在采取措施控制它" |

## 4. 整合治疗策略 (Integrated Treatment Strategies)

### 4.1 药物治疗

#### 表4.1 疼痛-抑郁共病的药物选择

| 药物类别 | 代表药物 | 抗抑郁效果 | 镇痛效果 | 主要适应症 | 注意 |
|---|---|---|---|---|---|
| **SNRIs** | 度洛西汀、文拉法辛、米那普仑 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | **首选**；纤维肌痛、糖尿病神经痛、慢性肌肉骨骼痛 | 高血压（文拉法辛）；恶心 |
| **TCAs** | 阿米替林、去甲丙咪嗪、丙咪嗪 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 神经病理性疼痛、纤维肌痛、头痛 | 抗胆碱能；心脏毒性；镇静 |
| **SSRIs** | 舍曲林、艾司西酞普兰 | ⭐⭐⭐⭐⭐ | ⭐⭐ | 抑郁为主，疼痛较轻 | 疼痛效果有限 |
| **NaSSA** | 米氮平 | ⭐⭐⭐⭐ | ⭐⭐ | 伴失眠、食欲差、恶心 | 体重增加；镇静 |
| **NDRI** | 安非他酮 | ⭐⭐⭐⭐ | ⭐⭐ | 疲劳突出；阿片戒断；认知症状 | 癫痫风险；失眠 |
| **抗惊厥药** | 普瑞巴林、加巴喷丁 | ⭐⭐ | ⭐⭐⭐⭐ | 神经病理性疼痛、纤维肌痛 | 头晕；嗜睡；体重增加 |
| **NMDA拮抗剂** | 氯胺酮（静脉/鼻喷） | ⭐⭐⭐⭐⭐（快速） | ⭐⭐⭐⭐⭐ | 难治性疼痛+抑郁 | 解离；滥用风险 |
| **局部用药** | 利多卡因贴剂、辣椒素 | — | ⭐⭐⭐ | 局部神经病理性疼痛 | 局部反应 |
| **阿片类** | — | ⭐ | ⭐⭐⭐⭐ | 急性/癌痛；**慢性非癌痛不推荐** | 依赖；滥用；抑郁恶化；痛觉过敏 |
| **抗炎药** | NSAIDs、COX-2抑制剂 | — | ⭐⭐⭐ | 炎症性疼痛 | GI、心血管风险；不改善抑郁 |
| **大麻素** |  nabiximols（部分国家） | 研究中 | ⭐⭐⭐ | 多发性硬化痉挛痛；神经病理性疼痛 | 精神活性；依赖；认知影响 |

> **核心原则**：SNRIs 和 TCAs 是疼痛-抑郁共病的核心药物，因为它们同时作用于5-HT和NE系统——这两个系统既是情绪调节的关键，也是下行疼痛抑制通路的核心。

### 4.2 心理治疗

| 方法 | 证据 | 核心机制 | 适用疼痛类型 | 疗程 |
|---|---|---|---|---|
| **认知行为治疗 (CBT-Pain)** | **A级** | 改变疼痛灾难化；行为激活；放松； pacing | 所有慢性疼痛 | 8-16周 |
| **接纳承诺治疗 (ACT-Pain)** | **A级** | 疼痛接纳；价值导向行为；认知解离 | 所有慢性疼痛；尤其高灾难化者 | 8-12周 |
| **正念减压 (MBSR)** | **A级** | 疼痛觉察而非反应；去中心化 | 所有慢性疼痛 | 8周 |
| **正念认知疗法 (MBCT)** | **B级** | 预防复发；反刍打断 | 疼痛伴抑郁复发 | 8周 |
| **行为激活 (BA)** | **B级** | 打破回避-退缩循环；逐步活动 | 活动回避突出者 | 8-12周 |
| **生物反馈** | **B级** | 学习控制生理反应（肌电、皮温、HRV） | 紧张型头痛、TMD、肌肉疼痛 | 8-12周 |
| **催眠治疗** | **B级** | 改变疼痛感知；放松； dissociation | 急性疼痛；癌痛；功能性疼痛 | 4-10次 |
| **团体治疗** | **B级** | 同伴支持；正常化；技能学习 | 所有类型 | 持续 |
| **基于暴露的疼痛治疗** | **B级** | 减少恐惧-回避；恢复活动 | 恐惧-回避突出者 | 6-12周 |

### 4.3 物理与康复干预

| 干预 | 证据 | 机制 | 适用类型 |
|---|---|---|---|
| **运动/物理治疗** | **A级** | 内啡肽；BDNF；抗炎；肌肉强化；功能恢复 | 所有可运动者；腰背痛、纤维肌痛 |
| **针灸** | **A级** | 内源性阿片释放；闸门控制；神经调节 | 广泛；尤其肌肉骨骼痛 |
| **经皮神经电刺激 (TENS)** | **B级** | 闸门控制理论 | 局部肌肉骨骼痛 |
| **热/冷疗** | **B级** | 血流；肌肉放松；炎症控制 | 急性肌肉痛；慢性僵硬 |
| **按摩/手法治疗** | **B级** | 肌肉放松；血流；内啡肽 | 肌肉紧张；纤维肌痛 |
| **瑜伽/太极** | **B级** | 身心整合；柔韧；力量；正念；呼吸 | 纤维肌痛；腰背痛；抑郁共病 |
| **水疗** | **B级** | 浮力减轻负荷；温热放松 | 关节炎；纤维肌痛 |

### 4.4 整合疼痛-抑郁管理模式

| 层级 | 干预 | 适用 |
|---|---|---|
| **第一层：基础** | 心理教育；运动处方；睡眠卫生； pacing | 所有患者 |
| **第二层：核心** | CBT/ACT/MBSR + SNRI/TCA | 轻中度 |
| **第三层：强化** | 上述 + 物理治疗 + 生物反馈 + 团体 | 中重度 |
| **第四层：专科** | 疼痛专科；介入治疗（神经阻滞）；氯胺酮；多学科疼痛中心 | 难治性 |

## 5. 特定疼痛综合征的抑郁管理 (Specific Pain Syndromes)

### 表5.1 纤维肌痛综合征 (FMS)

| 维度 | 内容 |
|---|---|
| **核心特征** | 广泛性疼痛（双侧、上下躯干） + 压痛点 + 疲劳 + 睡眠障碍 + 认知困难（"纤维 fog"） |
| **与抑郁关系** | 最高共病率之一（60-80%）；共享中枢敏化机制 |
| **一线药物** | 度洛西汀、米那普仑、普瑞巴林（FDA批准） |
| **一线非药物** | 运动（尤其水中运动）；CBT；正念；睡眠优化 |
| **关键原则** | 患者教育（非进行性/致命疾病）；设定现实期望；逐步功能恢复 |
| **避免** | 长期卧床；阿片类；苯二氮卓；过度检查 |

### 表5.2 慢性腰背痛

| 维度 | 内容 |
|---|---|
| **核心特征** | 持续 > 3月的腰痛；影像学常无明确对应 |
| **与抑郁关系** | 30-50%共病；互为因果；功能丧失是连接点 |
| **一线治疗** | 保持活动（避免卧床）；CBT；运动康复；SNRI |
| **影像学警示** | 早期MRI常无帮助，且可能强化疾病信念；除非红旗信号 |
| **红旗信号** | 癌症史、发热、体重下降、神经缺损、大小便障碍 |

### 表5.3 头痛（紧张型/偏头痛）

| 维度 | 内容 |
|---|---|
| **核心特征** | 5-HT功能障碍是共同基础；发作性与慢性 |
| **与抑郁关系** | 双向；慢性偏头痛转化与抑郁进展同步 |
| **预防药物** | TCAs（阿米替林）；SNRIs（文拉法辛）；β受体阻滞剂；抗惊厥药 |
| **急性治疗** | 曲普坦类；NSAIDs；避免过度使用（致反跳性头痛） |
| **非药物** | 生物反馈；放松；CBT；规律作息；触发因素管理 |
