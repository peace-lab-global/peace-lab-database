# 压力日记分析技能

## 技能信息

**技能ID**: S_008
**技能名称**: 压力日记分析
**功能**: 分析用户提供的压力日记，识别模式和触发因素
**输入**: 用户填写的压力日记数据
**输出**: 分析报告 + 个性化建议

---

## 压力日记模板

### 每日记录（建议连续2-4周）

| 时间 | 情境 | 压力感知(1-10) | 身体感觉 | 情绪 | 应对策略 | 效果(1-10) |
|:----:|:----:|:--------------:|:--------:|:----:|:--------:|:-----------:|
| | | | | | | |

### 简化版每日记录

| 项目 | 填写内容 |
|:-----|:---------|
| 日期 | |
| 睡眠时长/质量 | / (1-10分) |
| 晨起精力 | /10 |
| 主要压力事件 | |
| 压力水平(1-10) | |
| 应对措施 | |
| 压力是否缓解 | 是/否/部分 |
| 晚间精力 | /10 |

## 分析决策流程

```
分析流程

STEP 1: 验证数据质量
│
├─ 数据是否连续记录 ≥5天？
│   └─ 否 → 建议继续记录
│
├─ 是否有每日多次记录？
│   └─ 否 → 建议增加记录频率
│
└─ 数据是否完整？
    └─ 否 → 指出缺失项

STEP 2: 基础统计分析
│
├─ 平均压力水平
├─ 最高/最低压力日
├─ 睡眠-压力相关性
└─ 能量-压力相关性

STEP 3: 模式识别
│
├─ 时间模式
│   ├─ 星期几最高？
│   ├─ 一天中什么时候最高？
│   └─ 周末vs工作日
│
├─ 触发因素模式
│   ├─ 哪些情境反复出现？
│   ├─ 哪些人反复触发？
│   └─ 哪些话题反复引发？
│
├─ 应对效果分析
│   ├─ 哪种策略最有效？
│   └─ 哪种策略效果差？
│
└─ 睡眠-压力关系
    ├─ 前一天睡眠差是否预测高压力？
    └─ 压力大是否预测睡眠差？

STEP 4: 生成报告
```

## 分析维度详解

### 1. 时间模式分析

| 分析点 | 问题 | 评估 |
|:-------|:-----|:-----|
| 周间模式 | 压力最高的是哪天？ | 工作压力集中日 |
| 日内模式 | 什么时候压力最高？ | 高压时段识别 |
| 周末效应 | 周末压力是否更低？ | 工作相关程度 |
| 趋势 | 整体压力趋势如何？ | 恶化/改善/稳定 |

### 2. 触发因素分析

| 分类 | 常见触发 | 识别关键词 |
|:-----|:---------|:-----------|
| 工作相关 | 截止日期，会议、沟通 | "老板"、"同事"、"工作" |
| 人际关系 | 冲突、被评价、社交 | "和XX吵架"、"聚会" |
| 健康相关 | 身体症状、疾病担忧 | "不舒服"、"担心健康" |
| 财务相关 | 账单、经济担忧 | "钱"、"财务" |
| 家庭相关 | 家庭责任、冲突 | "家人"、"孩子" |
| 睡眠相关 | 睡不好、失眠 | "睡眠"、"疲惫" |

### 3. 应对策略效果分析

| 应对类型 | 效果评估标准 | 典型有效策略 |
|:---------|:-------------|:-------------|
| 问题聚焦 | 直接解决问题根源 | 问题解决、时间管理 |
| 情绪聚焦 | 短期缓解情绪 | 倾诉、转移注意力 |
| 回避 | 短期有效，长期有害 | 刷手机、躺平 |
| 身心技术 | 调节生理和情绪 | 呼吸、冥想、运动 |
| 社会支持 | 获取资源和建议 | 与人倾诉 |

| 应对策略效果 | 判断标准 |
|:-------------|:---------|
| 高效 | 效果评分≥7，持续有效 |
| 中效 | 效果评分4-6，间歇有效 |
| 低效 | 效果评分≤3，效果差或反效果 |

### 4. 睡眠-压力相关性

| 关系类型 | 识别标准 | 推断 |
|:---------|:---------|:-----|
| 睡眠→压力 | 前晚睡眠差→次日压力大 | 睡眠管理优先 |
| 压力→睡眠 | 前日压力大→当晚睡眠差 | 压力管理优先 |
| 双向 | 两者相互影响 | 综合干预 |
| 无关 | 相关性低 | 独立分析 |

| 日期 | 睡眠/10 | 晨起精力/10 | 压力事件 | 压力/10 | 应对 | 效果/10 |

| 周一 | 5 | 4 | 项目截止 | 8 | 加班 | 3 |
