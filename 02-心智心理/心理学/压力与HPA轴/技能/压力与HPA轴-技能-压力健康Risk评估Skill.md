# 压力相关健康风险评估技能

## 技能信息

**技能ID**: S_007
**技能名称**: 压力相关健康风险评估
**功能**: 评估长期压力对多系统健康的影响，识别风险等级
**输入**: 压力状态和身体症状
**输出**: 风险评估报告 + 干预优先级

---

## 多系统健康影响概览

### 压力影响的主要系统

| 系统 | 主要影响 | 症状表现 |
|:-----|:--------|:---------|
| 心血管 | 血压升高、心率变异降低 | 高血压、心悸，胸闷 |
| 代谢 | 血糖升高、腹部脂肪堆积 | 体重增加、血糖升高 |
| 免疫 | 炎症增加、免疫力下降 | 频繁感染、自身免疫 |
| 神经 | HPA轴紊乱、神经递质失衡 | 焦虑、抑郁、睡眠障碍 |
| 消化 | 肠道功能紊乱 | 胃痛、腹泻/便秘 |
| 肌肉骨骼 | 肌肉紧张、疼痛 | 头痛、肩颈痛、背痛 |
| 生殖 | 激素紊乱 | 月经不调、性功能障碍 |

---

## 系统性评估问卷

### 心血管系统

| 问题 | 评估点 |
|:-----|:-------|
| "你有高血压病史吗？" | 已知风险 |
| "压力大时是否感觉心跳加快或不规律？" | 心悸、心律失常 |
| "是否经常感到胸闷或胸痛？" | 需排除心脏疾病 |
| "有没有在放松时仍然感觉紧张？" | 持续警觉 |
| "是否经常感到呼吸急促？" | 过度换气、交感激活 |

**红旗症状**（需立即医学评估）:
- 持续胸痛
- 呼吸困难伴胸痛
- 严重心律失常
- 晕厥

### 代谢系统

| 问题 | 评估点 |
|:-----|:-------|
| "你的体重在过去一年有明显变化吗？" | 体重变化方向 |
| "腹部（腹部）是否明显增大？" | 向心性肥胖 |
| "有没有被诊断过血糖升高或糖尿病？" | 代谢风险 |
| "压力大时食欲如何？" | 食欲变化（增加/减少）|
| "是否经常感到渴或尿频？" | 血糖问题信号 |

**代谢风险指标**:

| 指标 | 关注值 |
|:-----|:-------|
| 腰围 | 男>90cm，女>80cm |
| 体重变化 | 一年增加>5kg |
| 血压 | 收缩压>130mmHg |
| 空腹血糖 | >100mg/dL |

### 神经精神系统

| 问题 | 评估点 |
|:-----|:-------|
| "是否经常感到焦虑、担心或紧张？" | 焦虑症状 |
| "是否经常感到悲伤、绝望或兴趣减退？" | 抑郁症状 |
| "是否有睡眠问题？" | 失眠类型 |
| "是否难以集中注意力或记忆力下降？" | 认知功能 |

**使用PHQ-4进行快速筛查**:

| 问题 | 评分(0-3) |
|:-----|:--------:|
| 做事时提不起劲或没有兴趣 | |
| 感到心情低落、抑郁或绝望 | |
| 感觉睡眠不安稳或睡太多 | |
| 感觉疲劳或没有活力 | |
| **总分** | |

| 总分 | 风险等级 |
|:----:|:--------:|
| 0-2 | 低风险 |
| 3-5 | 低-中风险 |
| 6-8 | 中-高风险 |
| >8 | 高风险，建议专业评估 |

---

## 综合风险评估矩阵

### 风险等级判定

| 系统 | 低风险(0) | 中风险(1) | 高风险(2) |
|:-----|:---------|:---------|:---------|
| 心血管 | 无症状 | 偶有心悸、轻度高血压 | 高血压、心律失常 |
| 代谢 | 无变化 | 体重增加10%、腰围超标 | 糖尿病前期/确诊 |
| 免疫 | 正常 | 频繁轻感冒 | 严重/反复感染 |
| 神经精神 | 偶有压力 | 失眠、焦虑 | 抑郁、严重失眠 |
| 消化 | 正常 | 偶有消化不适 | 诊断的消化疾病 |
| 肌肉骨骼 | 正常 | 偶有头痛/肩痛 | 慢性疼痛 |

**综合风险评分** = 各系统分数之和

| 总分 | 风险等级 | 建议 |
|:----:|:--------:|:-----|
| 0-3 | 低风险 | 维持性干预 |
| 4-6 | 中风险 | 主动干预 |
| 7-9 | 高风险 | 积极干预+监测 |
| 10-12 | 极高风险 | 医疗评估优先 |

## 使用示例

**用户输入**: "我已经连续一年多工作压力很大，经常加班，晚上也睡不好，白天感觉累，最近体重增加了不少，特别是肚子这块，血压也有点高。"

**评估过程**:
1. 心血管：工作压力+血压高 → 中-高风险
2. 代谢：腹部肥胖+体重增加 → 高风险
3. 神经精神：长期压力+失眠+疲劳 → 高风险
