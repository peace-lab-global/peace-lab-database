# Bipolar Depression: Differential Diagnosis & Management (双相抑郁鉴别与管理)

> **双相抑郁 (Bipolar Depression)**
>
> *"双相抑郁最大的危险不是躁狂，而是被误诊为单相抑郁后，抗抑郁药 alone 引发的转相风暴。"*
> *"The greatest danger of bipolar depression is not mania itself, but the switch into mania triggered by antidepressants alone after misdiagnosis as unipolar depression."*
>
> 约 20% 被诊断为抑郁症的患者实际为双相谱系障碍，而首次就诊为抑郁发作的双相患者平均误诊延迟达 7-10 年。错误的诊断导致错误的治疗——单用抗抑郁药可诱发转相、快速循环和自杀风险增加。本文档系统整合双相抑郁的识别、鉴别与管理策略。

## 1. 核心概念与诊断标准 (Core Concepts & Diagnostic Criteria)

### 表1.1 双相障碍分类（DSM-5）

| 类型 | 核心特征 | 抑郁发作 | 躁狂/轻躁狂 | 临床意义 |
|---|---|---|---|---|
| **双相 I 型** | 至少1次完全躁狂发作 | 通常存在（非必需） | 完全躁狂（≥7天或需住院） | 最严重；精神病性特征常见；功能损害大 |
| **双相 II 型** | 至少1次轻躁狂 + 至少1次重性抑郁 | 必须存在 | 轻躁狂（≥4天，无精神病性，功能无严重损害）| 抑郁更严重、更慢性；自杀风险高 |
| **环性心境障碍** | 2年以上多次轻躁狂和轻抑郁交替 | 未达重性抑郁标准 | 未达轻躁狂标准 | 慢性波动；可进展为双相I/II |
| **其他特定双相障碍** | 症状模式不符合上述 | 变化 | 变化 | 需要个体化评估 |
| **未特定双相障碍** | 信息不足 | — | — | 临时诊断 |

### 表1.2 双相抑郁 vs 单相抑郁：鉴别要点

| 鉴别维度 | 双相抑郁 | 单相抑郁 |
|---|---|---|
| **发病年龄** | 较早（通常 < 25岁）；首次发作多为抑郁 | 较广；任何年龄 |
| **发作突然性** | 更急骤（数小时至数天） | 较渐进（数周） |
| **发作频率** | 发作次数更多；更短（平均 3-6月） | 较少；更长（平均 6-12月） |
| **精神病性特征** | 更常见 | 较少见（重度时可见） |
| **非典型特征** | 更常见（嗜睡、食欲增加、体重增加、铅样麻痹） | 较少 |
| **混合特征** | 常见（抑郁 + 躁狂症状共存） | 罕见 |
| **睡眠需求** | 增加（非典型）或严重失眠（伴混合特征） | 通常减少/失眠 |
| **运动性迟滞** | 更突出（精神运动性抑制） | 可变 |
| **情绪反应性** | 可能存在（情绪可被提升） | 通常无 |
| **既往躁狂/轻躁狂史** | **确诊关键** | 无 |
| **家族史** | 双相/躁狂/精神病史阳性率高 | 抑郁家族史 |
| **药物反应** | 抗抑郁药单用易转相 | 抗抑郁药通常有效 |
| **季节性** | 可能冬季抑郁、夏季躁狂 | 冬季多见 |
| **产后触发** | 产后躁狂/精神病风险高 | 产后抑郁 |

## 2. 筛查工具 (Screening Instruments)

### 表2.1 双相谱系筛查工具

| 工具 | 条目 | 切点 | 灵敏度 | 特异度 | 用途 | 注意 |
|---|---|---|---|---|---|---|
| **MDQ (Mood Disorder Questionnaire)** | 13项症状 + 2个功能问题 | ≥7项 + 功能损害 | 0.73 | 0.90 | 广泛筛查 | 敏感性中等；假阴性常见于双相II |
| **HCL-32 (Hypomania Checklist)** | 32项轻躁狂症状 | ≥14项 | 0.80 | 0.76 | 筛查轻躁狂 | 对双相II更敏感 |
| **BSDS (Bipolar Spectrum Diagnostic Scale)** | 描述性叙事 + 问题 | ≥14分 | 0.76 | 0.81 | 叙事式筛查 | 患者易懂 |
| **CMPS (Composite International Diagnostic Interview)** | 结构化访谈 | — | 金标准 | 金标准 | 研究/确诊 | 需培训 |
| **SCID-5 (Structured Clinical Interview)** | 模块化访谈 | — | 金标准 | 金标准 | 临床确诊 | 需培训 |

### 表2.2 轻躁狂识别要点（患者常不自知）

| 领域 | 轻躁狂表现 | 患者常见解读 |
|---|---|---|
| **情绪** | 异常高涨、兴奋、易激惹 | "我状态很好" "那段时间我很高效" |
| **精力** | 精力增加；睡眠需求减少（不感疲劳） | "我不需要那么多睡眠" |
| **自尊** | 自信心膨胀；夸大计划 | "我有好多好主意" |
| **思维** | 思维奔逸；想法跳跃；话多 | "我思维很敏捷" |
| **活动** | 活动量增加；社交增多；冲动行为 | "我很活跃" "我做了很多决定" |
| **注意力** | 容易分心 | "我兴趣广泛" |
| **风险行为** | 过度消费；轻率投资；轻率性行为；鲁莽驾驶 | 事后常后悔但否认与情绪相关 |

> **关键临床操作**：
> 1. **不要直接问"你有没有过躁狂？"** → 改为"有没有过一段时间，你感觉特别好、精力充沛、效率特别高？"
> 2. **询问重要他人**：患者常对轻躁狂期有积极回忆，不认为是问题。家属/朋友往往更客观。
> 3. **查看生活事件时间线**：重大决策（突然辞职、大额消费、婚姻/离婚）是否与情绪高涨期重合？
> 4. **抗抑郁药反应史**：服用抗抑郁药后出现激越、失眠、冲动、情绪高涨？

## 3. 双相抑郁的治疗原则 (Treatment Principles)

### 表3.1 双相抑郁治疗的核心原则

| 原则 | 内容 | 理由 |
|---|---|---|
| **心境稳定剂为基础** | 所有双相抑郁治疗必须以心境稳定剂或第二代抗精神病药为基础 | 预防转相；稳定情绪基线 |
| **抗抑郁药需谨慎** | 避免单用抗抑郁药；如需使用，必须与心境稳定剂联用且短期 | 诱发转躁、快速循环、自杀 |
| **识别混合特征** | 抑郁伴躁狂症状时需高度警惕 | 混合状态自杀风险最高；治疗不同 |
| **监测转相信号** | 治疗中密切观察转躁/轻躁狂迹象 | 早期识别可及时调整 |
| **长期维持优先** | 双相是慢性复发性疾病；维持治疗是核心 | 预防复发比急性治疗更重要 |
| **功能恢复目标** | 不仅消除症状，更要恢复社会/职业功能 | 功能损害是双相的主要负担 |
| **患者教育关键** | 疾病知识、药物依从性、早期信号识别、生活方式 | 依从性是复发的最大可变因素 |
| **家庭参与** | 家庭教育；早期信号监测；危机应对 | 家庭是维持治疗的重要盟友 |

### 表3.2 双相抑郁的药物治疗

| 药物/类别 | 证据等级 | 作用 | 优势 | 风险/注意 |
|---|---|---|---|---|
| **锂盐 (Lithium)** | **A级** | 心境稳定；抗抑郁（尤其经典双相）；抗自杀 | 抗自杀效应独特；神经保护；长期数据最多 | 肾毒性；甲状腺；震颤；中毒；需血药监测 |
| **喹硫平 (Quetiapine)** | **A级** | 抗抑郁 + 抗躁狂 | 单药即可用于双相抑郁；镇静助眠 | 代谢综合征；镇静；体重增加 |
| **鲁拉西酮 (Lurasidone)** | **A级** | 抗抑郁（需联合锂/丙戊酸） | 代谢影响小 | 静坐不能；恶心；需与食物同服 |
| **拉莫三嗪 (Lamotrigine)** | **B级** | 预防抑郁复发；轻度抗抑郁 | 耐受性好；无代谢副作用 | 严重皮疹（SJS）；缓慢滴定 |
| **丙戊酸钠 (Valproate)** | **B级** | 心境稳定；抗躁狂 > 抗抑郁 | 快速起效 | 肝毒性；多囊卵巢；致畸；体重增加 |
| **卡马西平 (Carbamazepine)** | **C级** | 心境稳定；难治性病例 | 对混合状态有效 | 骨髓抑制；皮疹；药物相互作用多 |
| **奥氮平+氟西汀 (OFC)** | **A级** | 抗抑郁 + 抗精神病 | FDA批准；快速 | 显著代谢副作用 |
| **阿立哌唑 (Aripiprazole)** | **B级** | 维持治疗；防躁狂 | 代谢影响小 | 静坐不能；失眠；激活 |
| **抗抑郁药联用** | **C级/争议** | 抗抑郁效应 | 对严重抑郁可能必要 | **必须**与心境稳定剂联用；监测转相；短期使用 |
| — 安非他酮 | 较安全 | NDRI | 转相风险相对较低 | 激活；失眠 |
| — SSRI（如舍曲林）| 可用 | 5-HT↑ | 相对安全 | 转相风险；混合状态恶化 |
| — 文拉法辛 | 风险较高 | SNRI | — | 转相风险更高 |

#### 表3.3 抗抑郁药在双相抑郁中的使用规则

| 情境 | 建议 |
|---|---|
| **可以使用** | 严重抑郁；心境稳定剂/抗精神病药 alone 无效；必须联用心境稳定剂 |
| **优先选择** | 安非他酮或SSRI（转相风险相对较低） |
| **避免使用** | TCAs（高转相风险）；SNRIs（文拉法辛转相风险高） |
| **监测要求** | 用药后密切监测激越、失眠、冲动、情绪高涨；出现即停用 |
| **疗程限制** | 短期使用（通常6-12周）；缓解后逐步减停 |
| **禁忌** | 混合状态；快速循环；既往抗抑郁药诱发躁狂 |

## 4. 心理治疗 (Psychotherapy)

| 方法 | 证据 | 核心内容 | 适用阶段 |
|---|---|---|---|
| **心理教育** | **A级** | 疾病知识；药物重要性；早期信号；触发因素管理 | 全程；尤其诊断初期 |
| **认知行为治疗 (CBT)** | **B级** | 认知重构；行为激活；复发预防；药物依从性 | 缓解期；维持期 |
| **人际与社会节奏治疗 (IPSRT)** | **A级** | 稳定日常节律；管理人际问题；睡眠-觉醒规律 | 尤其重要；急性+维持 |
| **家庭聚焦治疗 (FFT)** | **A级** | 家庭教育；沟通训练；问题解决；早期信号监测 | 急性+维持 |
| **辩证行为治疗 (DBT)** | **C级** | 情绪调节；痛苦耐受；冲动控制 | 共病边缘型特质；自伤 |
| **团体心理教育** | **B级** | 同伴支持；知识分享；正常化 | 维持期 |

### 4.1 人际与社会节奏治疗 (IPSRT) 详解

IPSRT 是双相障碍最特异的心理治疗方法。

| 核心模块 | 内容 | 操作 |
|---|---|---|
| **社会节奏度量 (SRM)** | 监测日常活动的时间规律性 | 记录15项日常活动的时间；评估规律性 |
| **节律稳定** | 建立稳定的睡眠-觉醒、用餐、社交、工作时间 | 固定起床时间；规律三餐；限制晚间刺激 |
| **人际问题处理** | 角色转换、角色冲突、哀伤、社交隔离 | IPT技术；聚焦当前人际问题 |
| **药物依从性** | 理解药物重要性；处理副作用担忧 | 动机访谈；解决障碍 |

> **研究证据**：Frank et al. (2005) 显示 IPSRT 可延长缓解期、降低复发率。节律稳定是双相治疗中被低估的核心。

## 5. 特殊临床情境 (Special Clinical Situations)

### 表5.1 混合特征 (Mixed Features)

| 特征 | 描述 | 治疗 |
|---|---|---|
| **定义** | 抑郁发作期间同时存在3个以上躁狂/轻躁狂症状 | 心境稳定剂 + 抗精神病药；**避免抗抑郁药** |
| **常见症状** | 精神运动性激越、 racing thoughts、话多、注意力分散、睡眠需求减少、增加目标导向活动、过度参与高风险活动 | — |
| **风险** | **自杀风险最高**的抑郁状态（高于纯抑郁或纯躁狂） | 密切监测；住院评估 |
| **药物选择** | 锂盐；丙戊酸；第二代抗精神病药（喹硫平、奥氮平、阿立哌唑） | — |

### 表5.2 快速循环 (Rapid Cycling)

| 特征 | 描述 | 治疗 |
|---|---|---|
| **定义** | 1年内 ≥ 4次情感发作（抑郁、躁狂、轻躁狂、混合） | 复杂；需专家管理 |
| **诱因** | 抗抑郁药；甲状腺问题；物质使用；生活事件；不规律作息 | 去除诱因；优化心境稳定剂 |
| **治疗** | 停用抗抑郁药；优化锂/丙戊酸/拉莫三嗪；考虑甲状腺素；严格节律管理 | 往往需要联合用药 |
| **预后** | 治疗反应较慢；维持治疗更关键 | 长期管理 |

### 表5.3 产后双相风险

| 特征 | 描述 | 管理 |
|---|---|---|
| **风险** | 双相障碍患者产后躁狂/精神病风险极高（30-50%） | 预防性用药；密切监测 |
| **预防** | 孕期继续心境稳定剂（锂/拉莫三嗪相对安全）；产后立即加强监测 | 产科-精神科协作 |
| **治疗** | 产后躁狂/精神病需紧急精神科干预；可能需ECT | 安全第一 |

## 6. 自杀风险管理 (Suicide Risk Management)

### 表6.1 双相障碍自杀风险特征

| 特征 | 说明 | 管理 |
|---|---|---|
| **终生自杀企图率** | 25-50% | 终身监测 |
| **自杀死亡率** | 15-20倍于一般人群 | 积极风险管理 |
| **最高风险期** | 混合状态；抑郁发作早期；从抑郁向躁狂转化期 | 密切监测 |
| **预测因素** | 既往未遂；自杀意念；混合特征；物质使用；焦虑；绝望感；冲动性；近期丧失 | 综合评估 |
| **保护因素** | 子女；宗教信仰；治疗联盟；社会支持；对未来有希望 | 强化保护 |
| **锂盐的特殊作用** | 锂盐是唯一被证实可降低双相自杀死亡率的药物 | 强烈建议使用（如无禁忌） |

## 7. 长期维持与复发预防 (Maintenance & Relapse Prevention)

### 表7.1 维持治疗建议

| 患者特征 | 推荐方案 | 理由 |
|---|---|---|
| **首次发作** | 心境稳定剂维持 ≥ 1-2年 | 高复发风险 |
| **多次发作** | 长期/终身维持 | 每次发作增加未来复发风险 |
| **严重发作** | 长期维持 | 功能损害大 |
| **自杀未遂史** | 长期维持；优先锂盐 | 风险管理 |
| **快速循环** | 长期维持；多药联合 | 慢性化 |
| ** psychotic 特征** | 长期维持 | 严重度标志 |

### 表7.2 双相复发预防计划

| 模块 | 内容 | 患者行动 |
|---|---|---|
| **我的信号** | 识别个人躁狂/抑郁早期预警信号 | 列出情绪、睡眠、行为、认知的预警信号 |
| **触发因素** | 识别个人触发因素 | 睡眠剥夺、压力、季节变化、物质、人际冲突 |
| **日常维护** | 维持稳定的生活节奏 | 固定睡眠、规律用餐、适度活动、限制酒精 |
| **药物依从** | 坚持服药，不自行调整 | 设置提醒；药盒；与医生讨论副作用 |
| **早期应对** | 信号出现时的策略 | 联系医生/治疗师；增加睡眠；减少刺激；启动支持 |
| **危机预案** | 严重恶化时的计划 | 紧急联系人；医院信息；手段限制 |
| **支持网络** | 关键联系人 | 家人、朋友、医生、治疗师、支持团体 |
