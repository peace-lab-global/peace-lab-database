# Schizoaffective Disorder Self-Assessment & Screening (分裂情感性障碍评估与筛查指南)

> **分裂情感性障碍评估与筛查 (Schizoaffective Disorder Assessment & Screening)**
> 
> *"分裂情感性障碍的识别关键在于捕捉精神病与心境两条线索的交织模式。"*
> *"The key to identifying schizoaffective disorder lies in capturing the interweaving pattern of psychotic and mood threads."*
> 
> 本文档旨在帮助个体和照护者理解分裂情感性障碍的特征，了解与相近疾病的鉴别要点。**自评工具不能替代专业诊断，仅作为筛查参考。** 详见 [严重精神障碍总览](临床-障碍-Severe_Mental_Illness_Overview.md)。

---

## 1. 识别的核心挑战 (The Core Challenge of Identification)

### 表1.1 为什么分裂情感性障碍难以自我识别

| 挑战 | 原因 | 应对建议 |
|---|---|---|
| **症状交叠** | 精神病性和心境症状常同时出现，难以区分 | 记录症状日记，分别追踪两类症状 |
| **诊断不稳定** | 36-54%的患者在随访中被重新诊断 | 接受诊断可能随病程调整的事实 |
| **缺乏自知力** | 精神病性症状期间可能不认为自己有病 | 照护者观察尤为重要 |
| **心境影响判断** | 躁狂期可能感觉"很好"、抑郁期可能归因于"正常悲伤" | 询问知情者的观察 |
| **社会偏见** | 病耻感阻碍就医 | 正常化精神卫生就医行为 |

## 2. 心境发作时间线制图 (Mood Episode Timeline Mapping)

### 表2.1 时间线制图方法 (Timeline Mapping Method)

> **核心目的**：分裂情感性障碍的诊断依赖于精神病性症状与心境症状的时间关系。绘制时间线是最有效的评估工具。

| 步骤 | 具体操作 | 记录要点 |
|---|---|---|
| **1. 标记心境发作** | 在时间轴上标出每次躁狂/抑郁发作的起止时间 | 起始日期、结束日期、发作类型（躁狂/抑郁/混合） |
| **2. 标记精神病性症状** | 在时间轴上标出精神病性症状（幻觉/妄想）出现的时间 | 起始日期、结束日期、症状类型 |
| **3. 标记用药变化** | 记录药物调整时间 | 药物名称、剂量、开始/停止日期 |
| **4. 识别"独立精神病期"** | 检查是否有≥2周精神病性症状存在而无心境症状的时段 | 这是区分分裂情感性与双相伴精神病性的**关键** |
| **5. 计算心境发作占比** | 心境发作占整个疾病活动期的比例 | 若占大部分时间→支持分裂情感性；若仅短暂→倾向精分 |

### 表2.2 诊断决策树 (Diagnostic Decision Tree)

| 问题 | 是 → | 否 → |
|---|---|---|
| **是否有精神病性症状（幻觉/妄想）？** | 继续下一步 | 不符合分裂情感性/精分标准 |
| **是否有显著的心境发作（躁狂或重性抑郁）？** | 继续下一步 | 可能为精神分裂症 |
| **精神病性症状是否曾在无心境症状时持续≥2周？** | 可能为分裂情感性障碍 | 可能为双相/抑郁伴精神病性 |
| **心境症状是否占疾病活动期的大部分时间？** | 支持分裂情感性障碍诊断 | 若心境症状仅短暂→倾向精神分裂症 |
| **是否有过躁狂发作？** | 分裂情感性-双相型 | 分裂情感性-抑郁型 |

## 3. 照护者观察清单 (Caregiver Observation Checklist)

### 表3.1 双维度观察清单 (Dual-Dimension Observation)

> **指导语**：请分别观察以下两类症状。如果两类症状均存在且持续，建议寻求精神科评估。

**A. 精神病性症状观察**

| 观察项目 | ☐ | 描述 |
|---|---|---|
| 自言自语/对空气讲话 | ☐有 ☐无 | 可能在回应幻听 |
| 表达不可能的信念 | ☐有 ☐无 | 如被监控、有特殊使命 |
| 言语内容难以理解 | ☐有 ☐无 | 思维紊乱表现 |
| 行为怪异/不可预测 | ☐有 ☐无 | 紊乱行为 |
| 多疑/警觉性异常增高 | ☐有 ☐无 | 偏执症状 |

**B. 心境症状观察**

| 观察项目 | ☐ | 提示方向 |
|---|---|---|
| 情绪异常高涨/易怒、精力极度充沛 | ☐有 ☐无 | 躁狂特征 |
| 睡眠显著减少但不感到疲倦 | ☐有 ☐无 | 躁狂特征 |
| 话特别多/语速极快 | ☐有 ☐无 | 躁狂特征 |
| 持续悲伤/空虚/绝望 | ☐有 ☐无 | 抑郁特征 |
| 对一切失去兴趣 | ☐有 ☐无 | 抑郁特征 |
| 提及死亡/自杀 | ☐有 ☐无 | **紧急**——立即就医 |

### 表3.2 观察结果建议

| 情况 | 建议 |
|---|---|
| **仅A类症状** | 建议精神科评估（可能为精神分裂症或其他精神病） |
| **仅B类症状** | 建议精神科评估（可能为双相障碍或抑郁症） |
| **A+B类症状同时或交替出现** | 强烈建议精神科评估——可能为分裂情感性障碍 |
| **任何自杀/攻击信号** | 立即急诊就医（120/110） |

## 4. 专业评估简介 (Professional Assessment Overview)

### 表4.1 常用评估工具 (Assessment Tools)

| 工具 | 英文/缩写 | 评估内容 | 在分裂情感性诊断中的作用 |
|---|---|---|---|
| **结构化临床访谈** | SCID-5 | DSM-5全面诊断访谈 | 诊断金标准 |
| **PANSS** | Positive and Negative Syndrome Scale | 精神病性症状评估 | 监测精神病性维度 |
| **YMRS** | Young Mania Rating Scale | 躁狂症状评估 | 监测双相型的躁狂维度 |
| **MADRS** | Montgomery-Åsberg Depression Rating Scale | 抑郁症状评估 | 监测抑郁维度 |
| **PSP** | Personal and Social Performance Scale | 社会功能评估 | 功能评估 |
| **生命周期图** | Life Chart Method | 纵向病程绘制 | **核心诊断工具**——梳理症状时间线 |

## 5. 自我管理建议 (Self-Management Recommendations)

### 表5.1 日常自我管理策略 (Daily Self-Management)

| 策略 | 方法 | 目的 |
|---|---|---|
| **症状日记** | 每日记录：心情(1-10)、睡眠时长、精神病性症状有/无、药物服用 | 识别波动模式；就诊提供信息 |
| **服药依从** | 固定时间服药；药盒管理；家属提醒 | 预防复发 |
| **睡眠卫生** | 固定作息；躁狂倾向时尤其保护睡眠 | 睡眠减少常是躁狂先兆 |
| **压力管理** | 识别个人压力触发因素；练习放松技巧 | 减少应激诱发 |
| **避免物质** | 避免酒精、大麻和其他物质 | 防止症状加重 |
| **维持社交** | 保持有规律的社交联系 | 防止孤立 |
