# 慢性压力干预决策技能

## 技能信息

**技能ID**: S_002
**技能名称**: 慢性压力干预决策
**功能**: 根据压力状态推荐合适的干预方法
**输入**: 压力评估结果或用户求助
**输出**: 分层干预方案

---

## 干预框架概览

### 多层次干预模型

| 层次 | 目标 | 干预类型 | 实施者 |
|:-----|:-----|:---------|:------:|
| 个体层面 | 增强应对能力 | 心理治疗、放松训练、生活方式 | 个人+治疗师 |
| 人际层面 | 改善社会支持 | 人际治疗、家庭治疗、支持小组 | 治疗师+小组 |
| 组织层面 | 减少职业压力 | 组织干预、EAP、工作重新设计 | 雇主+HR |
| 社会层面 | 减少结构性压力 | 政策干预、社区项目 | 政策制定者 |

### 干预时机选择

| 时机 | 目标 | 策略 | 适用情况 |
|:-----|:-----|:-----|:---------|
| 初级预防 | 预防发生 | 健康教育、压力管理培训 | 低风险人群 |
| 次级预防 | 早期识别干预 | 筛查、高危人群项目 | 中等压力 |
| 三级预防 | 减少已有影响 | 综合治疗、康复 | 严重/慢性压力 |

## 核心干预技术详解

### 1. 认知行为疗法 (CBT)

**适用情况**: 中高-高压力，伴有适应不良的思维模式

**核心模块**:

| 模块 | 内容 | 周期 |
|:-----|:-----|:----:|
| 心理教育 | 压力机制、认知-情绪-行为联系 | 第1-2周 |
| 认知重构 | 识别自动思维、认知扭曲、平衡思维 | 第2-5周 |
| 问题解决 | 问题定义、方案生成、决策、执行、评估 | 第4-7周 |
| 行为技能 | 时间管理、沟通技能、自我主张 | 第5-8周 |
| 放松整合 | 放松技术与CBT框架整合 | 持续 |
| 预防复发 | 高危情境识别、应对计划 | 第7-8周 |

**证据等级**: ★★★（强，d≈0.5-0.7）
**疗程**: 8-16周，每周1次

### 2. 正念减压 (MBSR)

**适用情况**: 各程度慢性压力，特别适用于情绪调节困难

**8周结构**:

| 周次 | 主题 | 核心技术 | 家庭作业 |
|:----:|:-----|:---------|:--------:|
| 1 | 自动导航 | 葡萄干练习 | 45分钟/天身体扫描 |
| 2 | 感知觉察 | 身体扫描、正念呼吸 | 45分钟/天 |
| 3 | 愉悦体验 | 正念瑜伽 | 45分钟/天 |
| 4 | 压力反应 | STOP技术 | 30分钟/天 |
| 5 | 反应vs回应 | 坐禅 | 45分钟/天 |
| 6 | 困难情绪 | 正念沟通 | 45分钟/天 |
| 全天 | 静默 | 整体练习 | — |
| 7 | 整合 | 所有练习整合 | 45分钟/天 |
| 8 | 维持 | 个人化计划 | 维持练习 |

**证据等级**: ★★★（强，d≈0.3-0.5）
**适用**: 慢性压力、焦虑、抑郁、慢性疼痛

### 3. 接纳承诺疗法 (ACT)

**适用情况**: 高压力，伴有回避行为或完美主义

**六原则应用**:

| 原则 | 在压力管理中的应用 |
|:-----|:------------------|
| 接纳 | 接纳不适情绪，而非消除 |
| 认知解离 | 将自我与压力思维分离 |
| 当下时刻 | 专注当下，减少担忧 |
| 观察者自我 | 发展稳定的自我感 |
| 价值澄清 | 明确什么真正重要 |
| 承诺行动 | 基于价值行动，即使不适 |

**证据等级**: ★★☆（中等-强）

## 分层干预方案模板

### 轻度压力（PSS 0-13）

```
【初级预防方案】

目标: 维持健康，预防压力累积

建议内容:
1. 生活方式基础
   - 规律运动（每周150分钟中等强度）
   - 充足睡眠（7-9小时）
   - 均衡饮食
   - 限制酒精/咖啡因

2. 压力预防知识
   - 识别早期压力信号
   - 建立健康边界
   - 维护社会支持

3. 建议资源
   - 推荐阅读：压力管理科普
   - 每日5-10分钟基础放松（可选）
```

### 中轻度压力（PSS 14-19）

```
【自我调节方案】

目标: 提升应对能力，防止恶化

建议内容:
1. 每日放松实践
   - 深度呼吸练习（5分钟/天）
   - 渐进性肌肉放松（10分钟/天）
   或
   - 正念冥想（10分钟/天）

2. 睡眠优化
   - 睡眠卫生教育
   - 固定作息时间
   - 睡前放松程序

3. 运动处方
   - 每周3-5次有氧运动
   - 每次30分钟中等强度

4. 自我监测
   - 压力日记（2周）
   - 识别触发因素

复评: 4周后评估效果
```

### 中高度压力（PSS 20-26）

```
【主动干预方案】

目标: 显著降低压力，改善功能

建议内容:
1. 心理治疗
   - 推荐: CBT或ACT
   - 频率: 每周1次
   - 周期: 8-12周

2. 技能训练
   - 认知重构技术
   - 问题解决训练
   - 放松技术进阶

3. 生活方式调整
   - 工作压力管理
   - 人际边界设置
   - 睡眠强化干预

4. 社会支持
   - 支持小组（可选）
   - 亲密关系维护

复评: 4周后初步评估，8周后整体评估
```

### 高压力（PSS > 26）

```
【强化干预方案】

目标: 稳定状态，防止并发症

建议内容:
1. 专业评估
   - 建议心理咨询/精神科评估
   - 排除抑郁/焦虑障碍共病
   - 评估是否需要药物治疗

2. 密集心理治疗
   - 推荐: 个体CBT（首选）或MBSR
   - 频率: 每周1-2次
   - 周期: 12-16周

3. 强化生活方式干预
   - 结构化每日放松程序
   - 睡眠治疗（CBT-I）
   - 运动方案（需个体化）

4. 危机预案
   - 识别预警信号
   - 建立支持联系
   - 紧急情况应对计划

转介指征:
- 自伤/自杀念头 → 立即转介急救
- 严重抑郁 → 评估药物治疗
- 疑似CFS → CFS_Recognition_Skill
```
