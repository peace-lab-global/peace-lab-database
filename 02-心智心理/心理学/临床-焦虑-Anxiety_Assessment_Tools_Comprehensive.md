# Anxiety Assessment Tools Comprehensive (焦虑症综合评估工具)

> **焦虑症评估工具总览 (Anxiety Assessment Instruments / Instrumenta Aestimationis Anxietatis)**
> 
> *"测量是科学的基础，评估是临床的根本。"*
> *"Measurement is the foundation of science; assessment is the bedrock of clinical practice."*
> 
> 本文档系统整理焦虑症跨诊断评估量表、功能评估工具、生物标志物辅助指标、结构化诊断访谈及标准化评估流程。

---

## 1. 跨诊断自评量表 (Transdiagnostic Self-Report Scales)

### 表1.1 贝克焦虑量表 (BAI)

| 维度 | 内容 |
|---|---|
| **全称** | Beck Anxiety Inventory (贝克焦虑量表) |
| **开发者** | Beck et al. (1988) |
| **条目数** | 21项 |
| **评估焦点** | 焦虑的躯体和认知症状(偏向躯体) |
| **评分** | 0-3分(完全没有-严重)；总分0-63 |
| **临床切分** | 0-7=最低；8-15=轻度；16-25=中度；26-63=重度 |
| **信度(α)** | 0.92 |
| **优势** | 广泛使用；与抑郁区分(偏躯体) |
| **局限** | 对惊恐/GAD敏感性高，对社交焦虑敏感性偏低 |
| **推荐场景** | 焦虑严重度监测；CBT疗效评估 |

### 表1.2 抑郁焦虑压力量表-21 (DASS-21)

| 维度 | 内容 |
|---|---|
| **全称** | Depression Anxiety Stress Scales-21 (抑郁焦虑压力量表21项版) |
| **开发者** | Lovibond & Lovibond (1995) |
| **条目数** | 21项(每个分量表7项) |
| **三个分量表** | (1)抑郁(D) (2)焦虑(A) (3)压力(S) |
| **评分** | 0-3分×2(乘以2计算总分)；每分量表0-42 |
| **焦虑分量表切分** | 0-7=正常；8-9=轻度；10-14=中度；15-19=重度；≥20=极重度 |
| **信度(α)** | 焦虑分量表0.81；总量表0.93 |
| **特色** | 同时评估焦虑、抑郁和压力；免费 |
| **推荐场景** | 跨诊断评估；共病筛查 |

### 表1.3 状态-特质焦虑问卷 (STAI)

| 维度 | 内容 |
|---|---|
| **全称** | State-Trait Anxiety Inventory (状态-特质焦虑问卷) |
| **开发者** | Spielberger (1983) |
| **条目数** | 40项(S-AI 20项 + T-AI 20项) |
| **两个分量表** | (1)状态焦虑(S-AI)：当前焦虑水平 (2)特质焦虑(T-AI)：焦虑倾向 |
| **评分** | 1-4分(每量表20-80分) |
| **信度(α)** | S-AI: 0.86-0.95；T-AI: 0.89-0.91 |
| **特色** | 区分状态(可变)和特质(稳定)焦虑 |
| **推荐场景** | 研究用途；治疗前后对比 |

### 表1.4 Hamilton焦虑量表 (HAM-A)

| 维度 | 内容 |
|---|---|
| **全称** | Hamilton Anxiety Rating Scale (Hamilton焦虑评定量表) |
| **开发者** | Hamilton (1959) |
| **条目数** | 14项 |
| **评估维度** | 精神性焦虑(7项) + 躯体性焦虑(7项) |
| **评分** | 0-4分(无-极重)；总分0-56 |
| **临床切分** | <17=轻度；18-24=中度；≥25=重度 |
| **施测方式** | 临床医生评定(需培训) |
| **信度** | 评分者间信度0.74-0.96 |
| **推荐场景** | 临床试验金标准；药物疗效评估 |

### 表1.5 PHQ-4超短筛查 (Ultra-Brief Screening)

| 维度 | 内容 |
|---|---|
| **全称** | Patient Health Questionnaire-4 (患者健康问卷4项版) |
| **条目数** | 4项(焦虑2项GAD-2 + 抑郁2项PHQ-2) |
| **施测时间** | <1分钟 |
| **临床切分** | 焦虑分量表≥3提示需进一步评估 |
| **推荐场景** | 初级保健快速筛查；大规模人群筛查 |

### 表1.6 量表对比总览 (Scale Comparison Overview)

| 量表 | 条目 | 施测时间 | 评估焦点 | 费用 | 敏感度 | 推荐场景 |
|---|---|---|---|---|---|---|
| **BAI** | 21 | 5分钟 | 躯体+认知焦虑 | 付费 | 惊恐/GAD高 | 严重度监测 |
| **DASS-21** | 21(焦虑7) | 5分钟 | 焦虑+抑郁+压力 | 免费 | 中等 | 跨诊断筛查 |
| **STAI** | 40 | 10分钟 | 状态+特质焦虑 | 付费 | 广泛 | 研究 |
| **HAM-A** | 14 | 15分钟 | 精神+躯体焦虑 | 免费 | 高 | 临床试验 |
| **GAD-7** | 7 | 2分钟 | GAD核心症状 | 免费 | GAD高 | 筛查+监测 |
| **PHQ-4** | 4 | <1分钟 | 焦虑+抑郁超短 | 免费 | 筛查 | 初级保健 |

## 2. 功能损害评估 (Functional Impairment Assessment)

### 表2.1 Sheehan失能量表 (SDS)

| 维度 | 内容 |
|---|---|
| **全称** | Sheehan Disability Scale |
| **条目数** | 3项(工作/社交/家庭) + 2项(功能缺失天数) |
| **评分** | 每项0-10分(VAS)；总分0-30 |
| **信度(α)** | 0.89 |
| **推荐** | 简短高效；焦虑治疗临床试验常用 |

### 表2.2 WHO残疾评定量表2.0 (WHODAS 2.0)

| 维度 | 内容 |
|---|---|
| **全称** | WHO Disability Assessment Schedule 2.0 |
| **条目数** | 12项(简版)或36项(完整版) |
| **评估领域** | 认知、活动、自理、人际、社会参与、日常活动 |
| **特色** | WHO开发；跨文化适用；DSM-5推荐 |
| **推荐** | 全面功能评估；跨文化研究 |

## 3. 生物标志物辅助评估 (Biomarker-Assisted Assessment)

### 表3.1 焦虑症相关生物标志物 (Anxiety-Related Biomarkers)

| 生物标志物 | 评估方法 | 焦虑中的变化 | 临床实用性 | 局限性 |
|---|---|---|---|---|
| **唾液皮质醇** | 唾液样本(多时间点) | CAR异常；日间曲线平坦化 | 中等 | 个体变异大；需标准化采集 |
| **头发皮质醇** | 头发样本(3cm) | 慢性应激指标 | 中等 | 反映3月平均；不适于急性 |
| **心率变异性(HRV)** | 可穿戴设备/ECG | RMSSD/HF降低(副交感抑制) | 较高 | 受体位/运动/药物影响 |
| **皮肤电导(GSR)** | 生物反馈设备 | 基线升高；反应增强 | 中等 | 需专业设备 |
| **CRP/IL-6** | 血液检测 | 慢性焦虑可能轻度升高 | 低 | 非特异性 |
| **BDNF** | 血液检测 | 可能降低 | 低(研究中) | 非特异性 |

### 表3.2 HRV评估在焦虑中的应用 (HRV in Anxiety Assessment)

| 维度 | 内容 |
|---|---|
| **原理** | HRV反映自主神经灵活性；焦虑患者副交感活动降低→HRV降低 |
| **关键指标** | RMSSD(副交感)、HF(高频功率)、LF/HF比值 |
| **临床应用** | 治疗监测(HRV提升提示改善)；生物反馈训练目标 |
| **设备** | 胸带(Polar)、可穿戴手表(Garmin/Apple Watch)、专业设备 |
| **注意** | 短期HRV(5分钟)vs长期HRV(24小时)解读不同 |

## 4. 数字化评估工具 (Digital Assessment Tools)

### 表4.1 生态瞬时评估 (EMA)

| 维度 | 内容 |
|---|---|
| **全称** | Ecological Momentary Assessment (生态瞬时评估) |
| **原理** | 在日常生活中实时多次评估焦虑水平和触发因素 |
| **实施** | 手机APP推送(每日3-6次随机)；即时回答2-5个问题 |
| **优势** | 减少回忆偏差；捕捉实时变化；识别触发模式 |
| **局限** | 患者依从性；可能增加焦虑觉察 |
| **代表工具** | MetricWire、PIEL Survey、Qualtrics Mobile |

### 表4.2 可穿戴设备在焦虑评估中的应用 (Wearable Devices)

| 设备类型 | 评估指标 | 应用场景 | 限制 |
|---|---|---|---|
| **智能手表** | HRV、心率、睡眠质量、活动量 | 长期监测；治疗反应 | 精度有限 |
| **胸带心率监测** | 精确HRV | 生物反馈训练 | 佩戴不便 |
| **GSR手环** | 皮肤电导(交感活动) | 焦虑触发实时检测 | 受温度/汗液干扰 |
| **睡眠追踪器** | 睡眠时间/质量/结构 | 焦虑对睡眠的影响 | 与PSG有差异 |

## 5. 结构化诊断访谈 (Structured Diagnostic Interviews)

### 表5.1 主要结构化访谈工具 (Major Interview Tools)

| 工具 | 英文 | 时间 | 覆盖范围 | 特色 | 适用 |
|---|---|---|---|---|---|
| **SCID-5** | Structured Clinical Interview for DSM-5 | 60-120分钟 | 所有DSM-5障碍 | 金标准 | 研究；复杂病例 |
| **MINI** | Mini International Neuropsychiatric Interview | 15-30分钟 | 主要精神障碍 | 高效；适合门诊 | 临床筛查 |
| **ADIS-5** | Anxiety and Related Disorders Interview Schedule | 90-120分钟 | 焦虑及相关障碍 | 焦虑专项最全面 | 焦虑专科评估 |
| **CIDI** | Composite International Diagnostic Interview | 90分钟 | WHO标准化访谈 | 跨文化标准化 | 流行病学研究 |

### 表5.2 ADIS-5焦虑障碍访谈要点 (ADIS-5 Key Features)

| 维度 | 内容 |
|---|---|
| **覆盖障碍** | 全部焦虑障碍 + OCD + PTSD + MDD + 物质使用 |
| **严重度评定** | 每个障碍0-8分CSR(Clinical Severity Rating) |
| **鉴别诊断** | 内建跳转逻辑帮助鉴别 |
| **时间线** | 评估当前和终生 |
| **优势** | 焦虑评估最全面；有儿童版(ADIS-C/P) |

## 6. 标准化评估流程 (Standardized Assessment Protocol)

### 表6.1 焦虑症评估六步流程 (Six-Step Assessment Protocol)

| 步骤 | 目标 | 工具/方法 | 时间 |
|---|---|---|---|
| **1. 筛查** | 快速识别可能病例 | PHQ-4 / GAD-7 | 2-5分钟 |
| **2. 诊断确认** | 确定具体诊断 | MINI / ADIS-5 / 临床访谈 | 30-90分钟 |
| **3. 严重度量化** | 量化症状严重程度 | 障碍特异性量表(PDSS/LSAS/GAD-7/BAI) | 5-10分钟 |
| **4. 功能评估** | 评估功能损害程度 | SDS / WHODAS 2.0 | 5分钟 |
| **5. 共病评估** | 识别共病(抑郁/物质/人格) | PHQ-9 / AUDIT / 临床访谈 | 10-20分钟 |
| **6. 风险评估** | 评估自伤/自杀风险 | C-SSRS / PHQ-9 Item 9 | 5分钟 |

### 表6.2 评估报告标准模板 (Assessment Report Template)

| 报告部分 | 内容 |
|---|---|
| **基本信息** | 姓名、年龄、性别、转介来源 |
| **主诉** | 患者自述主要问题 |
| **现病史** | 症状起始、发展、触发因素 |
| **评估结果** | 量表分数及解读 |
| **诊断印象** | 主诊断+共病(含CSR) |
| **功能评估** | 各领域功能损害 |
| **个案公式化** | 基于认知行为模型 |
| **治疗建议** | 推荐治疗方案 |
| **风险评估** | 自伤/自杀风险等级 |

## 7. 跨文化评估注意事项 (Cross-Cultural Assessment)

### 表7.1 文化因素对焦虑评估的影响 (Cultural Factors in Assessment)

| 文化因素 | 影响 | 应对策略 |
|---|---|---|
| **躯体化倾向** | 亚洲/拉美文化中焦虑常以躯体主诉就诊 | 探查躯体症状背后的心理成分 |
| **病耻感** | 某些文化对精神障碍病耻感强 | 正常化；强调可治疗性 |
| **量表翻译** | 直译可能改变含义 | 使用经验证的本地化版本 |
| **表达规范** | 情绪表达规范因文化而异 | 避免以西方标准判断 |
| **求助模式** | 可能优先求助传统医学/宗教 | 尊重并整合 |
