# Depression CBT Manual (抑郁症认知行为治疗操作手册)

> **抑郁症认知行为治疗操作手册 (Cognitive Behavioral Therapy for Depression: Clinical Manual)**
>
> *"改变的不是现实，而是我们与现实的关系。CBT教你成为自己思维的观察者，而非囚徒。"*
> *"CBT does not change reality, but our relationship with it. It teaches you to become an observer of your own thoughts, not their prisoner."*
>
> 认知行为治疗（CBT）是抑郁症循证最强的心理治疗方法之一。本手册面向接受过CBT培训的治疗师，提供结构化、可操作的临床操作指南。非专业人员可在自助指南中获取简化版本。

## 1. CBT抑郁模型 (CBT Model of Depression)

### 1.1 贝克抑郁认知模型

```
                    早期经历
                      │
                      ▼
              ┌──────────────┐
              │  核心信念     │
              │ (关于自我/世界/未来的根本性信念)
              └──────────────┘
                      │
                      ▼
        应激事件 ──→ ┌──────────────┐
        (如丧失、    │  中间信念     │
         失败、      │ (假设、规则、态度)
         冲突)       └──────────────┘
                      │
                      ▼
              ┌──────────────┐
              │  自动思维     │ ←──────── 情境触发
              │ (对事件的即时解释)
              └──────────────┘
                      │
        ┌─────────────┼─────────────┐
        ▼             ▼             ▼
    ┌───────┐    ┌───────┐    ┌───────┐
    │ 情绪  │    │ 行为  │    │ 躯体  │
    │(抑郁/ │    │(退缩/ │    │(疲劳/ │
    │ 焦虑) │    │ 回避) │    │ 疼痛) │
    └───────┘    └───────┘    └───────┘
        │             │             │
        └─────────────┴─────────────┘
                      │
                      ▼
              反馈循环：情绪/行为/躯体
              强化核心信念和自动思维
```

### 1.2 抑郁认知三联征 (Cognitive Triad)

| 维度 | 负性核心信念示例 | 中间信念示例 | 自动思维示例 |
|---|---|---|---|
| **对自我** | "我不可爱" "我无能" "我是负担" | "如果我表现不完美，就说明我是失败者" | "我又搞砸了" "我不值得被帮助" |
| **对世界/他人** | "世界是危险的" "他人不可信" "别人总是批评我" | "如果别人真正了解我，就会拒绝我" | "大家都在评判我" "没人理解我" |
| **对未来** | "未来是绝望的" "事情永远不会变好" | "如果一件事可能出错，它就一定会出错" | "努力也没用" "我不会好起来的" |

### 1.3 行为维持机制

| 机制 | 描述 | 示例 | CBT干预 |
|---|---|---|---|
| **回避-退缩循环** | 抑郁导致活动减少，活动减少导致正性体验减少，从而维持抑郁 | 因情绪低落取消社交→更加孤独→情绪更低落 | 行为激活；活动计划 |
| **快感缺失-动机下降** | 奖赏系统功能降低→活动减少→进一步降低奖赏体验 | 不再从事爱好→失去快乐来源→更无动力 | 行为实验；渐进愉悦活动 |
| **反刍-情绪恶化** | 反复思考负面内容→情绪加重→更多反刍 | 反复想「我为什么这么失败」→更加抑郁 | 正念；注意力转移； worry time |
| **社交技能下降** | 社交回避→技能生疏→更多回避 | 长期不社交→不知如何开始对话→继续回避 | 社交技能训练；渐进暴露 |
| **睡眠紊乱** | 抑郁影响睡眠→睡眠不足加重情绪症状 | 深夜反刍→失眠→次日疲劳烦躁 | 睡眠卫生；CBT-I |

## 2. 治疗结构与流程 (Treatment Structure)

### 表2.1 标准CBT疗程结构

| 阶段 | 会谈次数 | 核心任务 | 目标 |
|---|---|---|---|
| **评估期** | 1-2次 | 全面评估；建立关系；心理教育；设定目标；引入CBT模型 | 建立治疗联盟；理解问题 |
| **急性期** | 6-12次 | 识别自动思维；认知重构；行为激活；问题解决 | 症状减轻50%+ |
| **巩固期** | 4-6次 | 识别中间信念；核心信念工作；预防复发技能 | 功能恢复；技能内化 |
| **维持/结束** | 2-4次（间隔） | 复发预防计划；结束治疗； booster 会谈 | 长期维持 |
| **总计** | **16-20次** | — | 缓解 + 复发预防 |

> **频率**：急性期通常每周1次；巩固期每2周1次；维持期每月1次。

### 表2.2 单次会谈结构（50分钟）

| 时间 | 内容 | 操作要点 |
|---|---|---|
| **0-5分钟** | 心境检查 | PHQ-2或简式情绪评分；本周整体感受 |
| **5-10分钟** | 议程设置 | 与患者共同确定本次重点（通常1-3个议题） |
| **10-35分钟** | 核心工作 | 回顾作业；新技能学习/练习；处理当前问题 |
| **35-45分钟** | 新作业布置 | 具体、可完成、与本次内容相关；预测障碍 |
| **45-50分钟** | 总结与反馈 | 患者总结本次收获；治疗师确认；预约下次 |

### 表2.3 治疗师核心能力

| 能力 | 描述 | 示例 |
|---|---|---|
| **苏格拉底式提问** | 通过提问引导患者自己发现替代视角 | "支持这个想法的证据是什么？反对的证据呢？" |
| **协作 empiricism** | 将信念视为假说，共同设计行为实验验证 | "我们能否设计一个实验来检验这个想法？" |
| **引导发现** | 不直接纠正，而是引导患者到达新认知 | "如果你最好的朋友有这个想法，你会对他说什么？" |
| **概念化能力** | 整合信息形成个案概念化 | 识别患者的核心信念、触发因素、维持模式 |
| **结构化** | 保持会谈聚焦、高效、目标导向 | 设置议程；时间管理；适时引导回主题 |
| **共情与联盟** | 建立温暖、信任的治疗关系 | 验证感受；表达理解；非评判 |
| **文化敏感性** | 理解患者的文化背景如何影响认知和行为 | 探索文化对情绪表达、求助、家庭角色的影响 |

## 3. 认知技术 (Cognitive Techniques)

### 3.1 自动思维识别与评估

**识别技术**：

| 技术 | 操作 | 提问示例 |
|---|---|---|
| **情绪追踪** | 记录强烈情绪出现时的情境 | "当时发生了什么？你脑子里闪过了什么？" |
| **意象技术** | 关注情绪伴随的心理图像 | "当你感到抑郁时，脑海中出现了什么画面？" |
| **箭头向下技术** | 逐层追问想法背后的含义 | "如果那是真的，意味着什么？那对你又意味着什么？" |
| **角色扮演** | 再现情境，捕捉即时思维 | "让我们重现那个场景，你在想什么？" |
| **梦的分析** | 分析梦境中的主题和情绪 | "这个梦让你感受到了什么？它反映了什么担忧？" |

**评估技术（认知重构）**：

| 技术 | 适用偏差 | 操作 |
|---|---|---|
| **证据检验** | 任意推断、情绪化推理 | 列出支持和反对自动思维的证据 |
| **替代解释** | 个人化、选择性概括 |  brainstorm 其他可能的解释 |
| **去灾难化** | 灾难化 | "最坏会发生什么？我能承受吗？最好的呢？最可能的呢？" |
| **比例尺技术** | 全或无思维 | "0-100%，你有多大程度相信这个想法？" |
| **旁观者视角** | 任意偏差 | "如果你的朋友遇到同样的事，你会怎么看？" |
| **功能性分析** | 固执于想法 | "相信这个想法对你有什么帮助？有什么代价？" |
| **去中心化** | 想法-事实融合 | "我注意到我有一个想法……"（将想法视为心理事件） |

### 3.2 思维记录表（核心工具）

```
┌─────────────────────────────────────────────────────────┐
│                   思维记录表                             │
├──────────┬──────────┬──────────┬──────────┬──────────┤
│  情境    │  自动思维  │  情绪    │  替代思维  │  再评情绪  │
│          │          │ (0-100%) │          │ (0-100%) │
├──────────┼──────────┼──────────┼──────────┼──────────┤
│ 发生了什么？│ 当时脑中闪 │ 悲伤___% │ 更平衡、现实的│ 悲伤___% │
│ 何时？何地？│ 过什么想法？│ 焦虑___% │ 想法是什么？ │ 焦虑___% │
│          │          │ 愤怒___% │          │ 愤怒___% │
│          │          │ 其他___% │          │ 其他___% │
├──────────┴──────────┴──────────┴──────────┴──────────┤
│ 证据支持自动思维：        证据反对自动思维：              │
│ _____________________    _____________________          │
│                                                          │
│ 如果朋友有同样情况，我会告诉他/她：_____________________ │
│                                                          │
│ 行动计划：_____________________________________________ │
└─────────────────────────────────────────────────────────┘
```

### 3.3 核心信念与中间信念工作

**核心信念识别**：

| 方法 | 操作 |
|---|---|
| **主题分析** | 分析多个自动思维的共同主题 |
| **箭头向下** | 反复追问「那意味着什么？」直到触及核心 |
| **早期回忆** | 探索与核心信念相关的童年/青少年记忆 |
| **核心信念量表** | 使用标准化量表（如SCBQ） |

**核心信念修正技术**：

| 技术 | 操作 | 示例 |
|---|---|---|
| **认知连续体** | 将绝对信念置于连续体上 | 「无能」→ 从0到100，什么程度算无能？ |
| **优点清单** | 系统收集个人优点和成就 | 每日记录3件做得好的事，持续2周 |
| **饼图技术** | 将事件原因多方归因 | 项目失败 → 因素分配：准备不足30%、资源限制25%、运气20%、他人因素15%、自身10% |
| **认知重构早期记忆** | 从成人视角重新解读早期经历 | "现在我理解了，父母的批评反映的是他们的压力，不是我的价值" |
| **体验式技术** | 通过情感唤起改变深层信念 | 空椅技术；意象重构；角色扮演 |
| **行为实验** | 设计行动检验核心信念 | "我计划主动提出一个建议，测试『我的意见没人重视』是否成立" |

**中间信念工作**：

| 类型 | 示例 | 修正策略 |
|---|---|---|
| **条件假设** | "如果我表现不好，别人就会拒绝我" | 检验条件关系；寻找反例 |
| **强制性规则** | "我必须 always 让别人满意" | 检验规则来源；评估代价；建立弹性规则 |
| **态度** | "失败是不可接受的" | 重新定义失败；探讨失败的价值 |

## 4. 行为技术 (Behavioral Techniques)

### 4.1 行为激活 (Behavioral Activation)

| 步骤 | 操作 | 技术细节 |
|---|---|---|
| **1. 活动监测** | 患者记录1-2周的活动和情绪 | 每小时记录；标注愉悦感(P)和掌控感(M) 0-10分 |
| **2. 识别模式** | 分析活动-情绪关联 | 找出「抑郁陷阱」（减少情绪的活动）和「提升活动」 |
| **3. 目标设定** | 与患者共同确定活动目标 | 价值导向（基于患者重视的生活领域） |
| **4. 活动计划** | 制定具体、可测量的活动计划 | 使用日程表；从低门槛开始；考虑时机和情境 |
| **5. 实施与反馈** | 执行计划；记录实际结果 | 关注「做了」而非「做好」 |
| **6. 调整与维持** | 根据反馈调整；建立长期习惯 | 将有效活动固化为日常 |

**活动计划原则**：

| 原则 | 说明 |
|---|---|
| **循序渐进** | 从极小的步骤开始（如「穿上运动鞋」而非「跑步5公里」） |
| **预约具体化** | 将活动安排在具体时间，而非「有空时做" |
| **多样平衡** | 愉悦活动 + 掌控/成就活动 + 社交活动 + 健康活动 |
| **预测与比较** | 行动前预测结果，行动后比较实际 vs 预测 |
| **问题解决障碍** | 提前识别可能障碍，制定应对方案 |

### 4.2 暴露与分级任务

| 技术 | 适用 | 操作 |
|---|---|---|
| **分级任务分配** | 回避行为 | 将困难任务分解为10个难度递增的子任务，逐级完成 |
| **角色扮演** | 社交回避 | 在治疗中练习困难对话 |
| **现场暴露** | 情境回避 | 制定恐惧层级；从最低级开始逐步暴露 |
| **行为实验** | 负性预期 | 将信念转化为可检验的预测；设计实验验证 |

### 4.3 放松与躯体技术

| 技术 | 操作 | 适用 |
|---|---|---|
| **渐进肌肉放松 (PMR)** | 系统紧张和放松16组肌肉 | 躯体紧张；焦虑；入睡困难 |
| **腹式呼吸** | 慢速深呼吸（吸气4秒-屏息2秒-呼气6秒） | 急性焦虑；恐慌；急性应激 |
| **正念呼吸** | 觉察呼吸，不控制 | 反刍；慢性焦虑；情绪调节 |
| **想象放松** | 引导想象安全/平静场景 | 焦虑；睡眠准备；急性应激 |

## 5. 会话内容指南 (Session-by-Session Guide)

### 表5.1 典型16次CBT会谈内容

| 会谈 | 主题 | 核心内容 | 家庭作业 |
|---|---|---|---|
| **1** | 评估与心理教育 | 建立关系；评估症状/功能；介绍CBT模型；设定目标 | 情绪/活动监测 |
| **2** | 引入自动思维 | 识别自动思维；情绪-思维链接；介绍思维记录表 | 完成2-3份思维记录表 |
| **3** | 认知重构基础 | 证据检验；替代思维；认知偏差介绍 | 继续思维记录表 |
| **4** | 行为激活引入 | 活动监测回顾；识别回避；引入活动计划 | 计划并执行2-3项活动 |
| **5** | 深化认知重构 | 去灾难化；比例尺技术；功能性分析 | 思维记录表 + 活动计划 |
| **6** | 行为实验 | 设计行为实验检验负性预测 | 执行1-2个行为实验 |
| **7** | 问题解决 | 系统化问题解决步骤 | 用问题解决处理一个实际问题 |
| **8** | 中期评估 | 评估进展；调整治疗计划；处理障碍 | 继续已有作业 |
| **9** | 中间信念 | 识别条件假设和规则；箭头向下技术 | 识别个人中间信念 |
| **10** | 核心信念初探 | 主题分析；早期回忆；核心信念识别 | 核心信念工作表 |
| **11** | 核心信念修正 | 认知连续体；优点清单；饼图技术 | 核心信念修正练习 |
| **12** | 复发预防技能 | 识别早期预警信号；应对策略预演 | 制定个人复发预防计划 |
| **13** | 社交/人际技能 | 社交技能训练；沟通练习； assertiveness | 社交行为实验 |
| **14** | 结束准备 | 回顾收获；巩固技能；处理结束情绪 | 总结个人CBT工具箱 |
| **15** | 复发预防计划 | 完整复发预防计划； booster 安排 | 维护计划执行 |
| **16** | 结束会谈 | 最终评估；庆祝进步；告别 | 独立维持 |

> **灵活性**：以上为理想模板。实际治疗需根据患者反应、严重度、共病和文化背景调整。重度抑郁可能需要更多行为激活，反刍型患者可能需要更早引入正念元素。

## 6. 困难情境处理 (Managing Difficulties)

### 表6.1 常见治疗障碍与应对

| 障碍 | 表现 | 应对策略 |
|---|---|---|
| **治疗不依从** | 不完成作业；频繁缺席 | 探索障碍（技能不足？动机不足？信念？）；简化作业；增加会谈频率；动机访谈 |
| **认知僵硬** | 拒绝接受替代思维；"但我的情况真的不同" | 缩小焦点；从更中性的自动思维开始；使用行为实验而非纯认知 |
| **情绪表达困难** | 无法识别或命名情绪 | 情绪教育；身体感受锚定；使用情绪轮盘；意象技术 |
| **过度理智化** | 用智力防御情感 | 引入体验式技术（意象、角色扮演、空椅）；关注身体感受 |
| **自杀风险** | 自杀意念出现或加重 | 安全计划；增加会谈频率；考虑住院；家属参与；药物评估 |
| **共病人格障碍** | 边缘型、回避型、强迫型特征 | 调整治疗框架（DBT技能、图式技术）；延长治疗；处理 alliance 破裂 |
| **文化不匹配** | CBT概念与患者文化框架冲突 | 文化适应（如集体主义文化中的自我概念调整）；整合本土资源 |
| ** alliance 破裂** | 患者对治疗师愤怒、失望、不信任 | 直接讨论关系； meta-communication；道歉（如适用）；调整方法 |
| **反刍严重** | 无法停止负面思维循环 | 引入正念/ACT元素； worry time；注意力转移训练；行为激活优先 |
