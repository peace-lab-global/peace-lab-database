# 强迫症综合评估 | OCD Assessment Skill

> **技能ID**: OCD_001
> **版本**: 1.0
> **角色**: 入口技能 (Entry)
> **最后更新**: 2026-05-18

---

## 一、触发条件

### 入口关键词
- "强迫"、"洁癖"、"检查"
- "重复"、"控制不住"、"清洗"
- "总是不放心"、"停不下来"

### 触发场景
- 用户描述重复行为或思维（如反复洗手、检查、计数）
- 用户询问自己是否得了强迫症
- 用户表达对特定想法或行为的无法控制感

### 前置要求
- 无（入口技能，无需前置）

---

## 二、评估流程

### 第一步：确认主诉

**问题序列**：

1. "你有什么反复出现的想法或行为吗？"
2. "这些想法或行为有多频繁？"
3. "你能控制它们吗？还是觉得必须做？"

**记录内容**：
- 主诉类型：强迫思维 / 强迫行为 / 混合
- 持续时间：<2周 / 2周-6月 / >6月
- 意识程度：知道不合理 / 觉得可能合理 / 完全相信

---

### 第二步：强迫分型

**快速分型问题**：

| 问题 | 目的 |
|------|------|
| "你的强迫主要是什么？（怕脏/怕出错/完美主义/其他）" | 识别强迫类型 |
| "你是反复做某些事，还是反复想某些事？" | 区分思维vs行为 |
| "做这些事的时候，心里是什么感觉？" | 评估焦虑驱动 |

**分型决策树**：

```
主诉分析
   │
   ├─ 怕脏、污染、接触脏东西
   │   └─ → 污染型强迫
   │
   ├─ 反复检查门/炉子/锁等
   │   └─ → 检查型强迫
   │
   ├─ 追求完美、对称、秩序
   │   └─ → 对称型强迫
   │
   ├─ 突然出现的侵入性想法（暴力/性/宗教）
   │   └─ → 侵入型强迫
   │
   └─ 多种混合
       └─ → 混合型强迫
```

---

### 第三步：严重程度评估

**使用 Y-BOCS 简化版评估**：

请用户对以下维度按 0-4 评分（0=无，1=轻，2=中，3=重，4=极重）：

**强迫思维维度**：
| 序号 | 内容 | 评分 |
|:----:|:-----|:----:|
| 1 | 强迫思维出现频率 | 0-4 |
| 2 | 强迫思维造成困扰程度 | 0-4 |
| 3 | 抵抗强迫思维的困难程度 | 0-4 |
| 4 | 强迫思维对功能的干扰程度 | 0-4 |

**强迫行为维度**：
| 序号 | 内容 | 评分 |
|:----:|:-----|:----:|
| 5 | 强迫行为出现频率 | 0-4 |
| 6 | 强迫行为造成困扰程度 | 0-4 |
| 7 | 抵抗强迫行为的困难程度 | 0-4 |
| 8 | 强迫行为对功能的干扰程度 | 0-4 |

**总分计算**：将所有分数相加（0-32）

| 总分 | 严重程度 | 说明 |
|:----:|:--------:|:-----|
| 0-7 | 轻度 | 轻度强迫，可通过自助技术管理 |
| 8-15 | 中度 | 中度强迫，建议结合技能训练和专业支持 |
| 16-23 | 重度 | 重度强迫，建议寻求专业帮助 |
| 24-32 | 极重 | 极重度强迫，强烈建议专业治疗 |

---

### 第四步：功能影响评估

**问题**：
- "这些强迫思维或行为每天花你多少时间？"
- "它们影响你的工作/学习/社交了吗？"
- "有没有因为强迫而错过重要的事情？"

**记录**：
- 每日耗时：<1小时 / 1-3小时 / 3-8小时 / >8小时
- 功能影响等级：无 / 轻度 / 中度 / 重度

---

### 第五步：安全检查

**红旗症状筛查**：

| 红旗症状 | 问题 |
|:---------|:-----|
| 自伤/自杀风险 | "你有没有因为强迫的痛苦想过伤害自己？" |
| 严重功能损害 | "你能正常工作和生活吗？" |
| 共病严重抑郁 | "你现在感觉有多绝望？" |

**如有任一红旗症状**：
- 立即输出「紧急转介报告」
- 不继续评估流程
- 建议寻求专业帮助

## 五、禁忌与注意事项

### 禁忌症
- 用户有自杀念头 → 立即转介危机干预
- 严重功能损害 → 建议专业治疗为主
- 共病严重抑郁 → 需要同时处理抑郁

### 注意事项
- 评估应保持非评判性态度
- 避免将正常的" perfectionism"病理化
- 承认强迫体验的真实痛苦
- 强调强迫症是可治疗的
