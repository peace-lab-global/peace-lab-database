# 焦虑状态综合评估 | Anxiety Assessment Skill

> **技能ID**: ANX_001
> **版本**: 1.0
> **角色**: 入口技能 (Entry)
> **最后更新**: 2026-05-18

---

## 一、触发条件

### 入口关键词
- "焦虑"、"担心"、"紧张"、"害怕"、"恐惧"
- "恐慌"、"心慌"、"坐立不安"
- "总是想太多"、"脑子停不下来"

### 触发场景
- 用户主动描述焦虑经历或症状
- 用户询问如何评估自己的焦虑状态
- 用户表达对特定情境的恐惧或回避

### 前置要求
- 无（入口技能，无需前置）

---

## 二、评估流程

### 第一步：确认主诉

**问题序列**：

1. "你今天因为什么感到困扰？"
2. "你能描述一下目前感受到的焦虑是什么样的吗？"
3. "这种感觉持续多久了？"

**记录内容**：
- 主诉类型：担心 / 恐惧 / 紧张 / 躯体症状 / 混合
- 持续时间：<2周 / 2周-6月 / >6月
- 诱因：明确 / 不明确 / 无诱因

---

### 第二步：焦虑分型

**快速分型问题**：

| 问题 | 目的 |
|------|------|
| "你最担心的事情是什么？" | 识别 GAD |
| "在哪些场合会让你特别紧张？" | 识别社交焦虑 |
| "有没有突然发作的强烈恐惧感？" | 识别惊恐 |
| "有什么特定的东西或情境让你非常害怕？" | 识别特定恐惧 |
| "在面对这些事情时，身体有什么反应？" | 评估躯体症状 |

**分型决策树**：

```
主诉分析
   │
   ├─ 担心内容广泛、难以控制
   │   └─ → GAD 类型
   │
   ├─ 担心被他人评价、回避社交场合
   │   └─ → 社交焦虑类型
   │
   ├─ 突然发作、心悸、呼吸困难、濒死感
   │   └─ → 惊恐类型
   │
   ├─ 对特定事物强烈恐惧
   │   └─ → 特定恐惧类型
   │
   └─ 多种焦虑混合
       └─ → 混合类型
```

---

| 36-63 | 重度 | 重度焦虑，建议寻求专业帮助 |

---

### 第四步：功能影响评估

**问题**：
- "焦虑对你的工作/学习影响大吗？"
- "你的日常生活受影响了吗？（社交/出行/睡眠）"
- "有没有因为焦虑而回避某些活动或场合？"

**记录**：
- 功能影响等级：无 / 轻度 / 中度 / 重度
- 主要受影响的领域：工作 / 社交 / 睡眠 / 家庭

---

### 第五步：安全检查

**红旗症状筛查**：

| 红旗症状 | 问题 |
|:---------|:-----|
| 自杀风险 | "你有没有过不想活的想法或尝试？" |
| 惊恐发作 | "有没有突然发作、感觉快要死了？" |
| 严重功能损害 | "你目前能正常上班/出门吗？" |

**如有任一红旗症状**：
- 立即输出「紧急转介报告」
- 不继续评估流程
- 建议寻求专业帮助

## 五、禁忌与注意事项

### 禁忌症
- 用户正在经历急性惊恐发作 → 先稳定情绪，不立即评估
- 用户有自杀念头 → 立即转介危机干预
- 严重创伤后应激 → 避免某些暴露技术

### 注意事项
- 评估应保持非评判性态度
- 避免给用户贴标签
- 承认焦虑的真实性，但不夸大
- 强调焦虑是可管理的
