## 认知重构原理

### 什么是认知重构

认知重构（Cognitive Restructuring）是认知行为疗法（CBT）的核心技术，指识别和改变不适应性自动思维，建立更平衡、现实的认知模式。其原理基于ABC模型：

- **A**dversity（逆境）：触发事件
- **B**eliefs（信念）：对事件的解读
- **C**onsequences（后果）：情绪和行为反应

关键发现：不是事件本身，而是对事件的解读（信念）导致了情绪后果。改变信念即可改变情绪反应。

### 认知层次模型

```
表层：自动化思维
   ↓ 自动涌现、无意识
中层：条件性假设
   ↓ "如果...就..."的思维模式
深层：核心信念
   ↓ 关于自我、他人、世界的根本假设
```

**玻璃心的典型深层信念**：
- "我必须被所有人喜欢"
- "我必须永远不犯错"
- "如果被批评，说明我不够好"
- "我的价值取决于我的成就"

## 认知扭曲识别

### 十种常见认知扭曲

| 扭曲类型 | 定义 | 玻璃心示例 |
|----------|------|------------|
| 全或无思维 | 用极端类别看待事物 | "要么完美，要么就是失败" |
| 过度泛化 | 基于单一事件得出普遍结论 | "这次被批评，我永远都会失败" |
| 心理过滤 | 只关注负面细节 | 只记住批评，忽略9次表扬 |
| 否定正面 | 贬低正面经历 | "那只是客气话，不是真的" |
| 读心术 | 确信知道他人想法 | "他们一定在心里嘲笑我" |
| 灾难化 | 预测最坏结果 | "如果被开除，我就完了" |
| 情绪推理 | 认为情绪反映事实 | "我感到羞耻，所以我一定可耻" |
| 应该陈述 | 用"应该"强加规则 | "我应该永远不犯错" |
| 标签化 | 用标签替代评价 | "我是失败者" |
| 个人化归因 | 过度承担不属于自己的责任 | "他生气一定是我的错" |

### 思维记录技术

**结构化思维日志**：

| 项目 | 内容 |
|------|------|
| 日期/时间 | 2024-XX-XX 14:30 |
| 情境 | 在部门会议上，我的提议被否定了 |
| 情绪（强度1-100） | 羞耻（75）、尴尬（60）、失落（50） |
| 自动思维 | "他们觉得我不行"、"这是个失败"、"我让大家失望了" |
| 认知扭曲 | 全或无、过度泛化、读心术 |
| 替代解释 | "我的提议有优点也有缺点"、"反对是正常的讨论"、"还有人支持我的想法" |
| 结果 | 重评后情绪：羞耻（30）、尴尬（20） |

## 重构技术详解

### 苏格拉底提问法

通过系列提问引导发现思维的局限性：

**质疑证据类**：
- 支持这个想法的证据是什么？
- 反对这个想法的证据是什么？
- 我是否忽略了什么重要信息？

**质疑逻辑类**：
- 这个想法在逻辑上合理吗？
- A一定导致B吗？
- 有没有其他解释？

**质疑假设类**：
- 我的假设是什么？
- 这个假设一定成立吗？
- 如果假设不成立，会怎样？

**扩展视角类**：
- 我的朋友遇到同样情况，我会对他说什么？
- 五年后，这件事还重要吗？
- 最坏的结果真的会发生吗？发生了又怎样？

### 行为实验

**定义**：通过实际行动验证思维的现实性。

**实施步骤**：
1. 确定要检验的思维
2. 设计实验（最坏/最好/最可能发生的情况）
3. 预测每种结果的可能性
4. 执行实验
5. 比较实际结果与预测
6. 基于结果更新思维

**玻璃心行为实验示例**：
- 假设："如果我提出不同意见，别人会讨厌我"
- 实验：在会议中提出一个小建议
- 预测：60%可能性被否定，30%可能性被忽视，10%可能性被接受
- 实际结果：建议被部分采纳，没有任何人表现讨厌
- 学习：提出不同意见并不会导致被讨厌

## 实践训练方案

### 入门练习：思维卡片

**材料**：索引卡或手机笔记

**步骤**：
1. 记录一个触发情境
2. 写下自动化思维
3. 识别认知扭曲类型
4. 写下3个替代解释
5. 标注每个替代的证据支持程度

### 中级练习：认知日记

**每日记录**（至少5条）：
- 情境（S）
- 情绪强度（1-10）
- 自动化思维（T）
- 认知扭曲类型（D）
- 替代思维（R）
- 新情绪强度（1-10）

### 高级练习：核心信念检验

**工作表**：

| 项目 | 内容 |
|------|------|
| 核心信念 | "我必须被所有人喜欢" |
| 支持证据 | 童年时被不喜欢会导致被欺负 |
| 反对证据 | 有人不喜欢我，但我依然生活得好 |
| 平衡观点 | 被人喜欢是愉快的，但不是必须；我可以选择被谁喜欢 |
| 新信念 | "我希望被喜欢，但即使有人不喜欢我，我的价值依然不变" |

## 高级重构策略

### 重新标签技术

将负面标签转化为中性或成长性描述：

| 旧标签 | 新标签 |
|--------|--------|
| "失败者" | "正在学习者" |
| "玻璃心" | "高敏感性人士" |
| "脆弱" | "感受丰富" |
| "软弱" | "有同理心" |

### 观察者技术

当陷入负面自我评价时，想象自己是第三方观察者：

1. 想象一台摄像机记录整个场景
2. 从摄像机角度描述"那个人"的表现
3. 观察整个情境，而非只关注负面
4. 作为观察者，你会对他说什么？

### 自我疏离化技术

当自动化思维激活时，尝试：
- 用第三人称称呼自己（"张三现在感到很尴尬"而非"我感到很尴尬"）
- 用幽默的方式描述（"哦，又来了，又是那个'被批评恐慌'程序"）
- 想象把情绪"放在架子上"，稍后处理

### 接纳承诺重构

1. **接纳**：承认情绪存在，不抗拒
2. **观察**：将自我与思维解离（"我有一个想法..."）
3. **澄清价值**：问自己"我真正在乎的是什么？"
4. **行动承诺**：基于价值采取行动，即使不舒服
