# 玻璃心与反玻璃心 (Fragile Ego & Resilience) | Index

> 玻璃心指情绪过度敏感、易受批评伤害的心理状态；反玻璃心（心理韧性）指在挫折、批评、拒绝中快速恢复并成长的能力。本模块提供双向干预：既理解玻璃心的成因机制，也系统构建心理韧性。
