# 焦虑障碍神经生物学机制 | Neurobiological Mechanisms of Anxiety Disorders

> **目标**：从精神科神经科学视角，系统阐述焦虑障碍的神经环路、神经递质、HPA轴、遗传与表观遗传学证据。
> **标准来源**：APA Practice Guidelines（2019）、Bandelow et al. WFSBP（2015）、Nature Reviews Neuroscience。

---

## 一、核心神经环路

### 1.1 杏仁核-前额叶通路

**杏仁核（Amygdala）**：
- 恐惧条件化的核心结构
- 中央核（CeA）：输出恐惧反应（自主神经、内分泌、行为）
- 基底外侧核（BLA）：接收感觉输入，进行威胁评估
- 焦虑障碍中杏仁核过度激活，对威胁刺激反应增强

**前额叶皮层（PFC）**：
- 腹内侧前额叶（vmPFC）：恐惧消退学习、情绪调节
- 背外侧前额叶（dlPFC）：认知重评、执行控制
- 前扣带回（ACC）：冲突监测、情绪-认知整合

**焦虑障碍中的环路失衡**：

```
正常状态：
  威胁信号 → 杏仁核适度激活 → PFC下行调控 → 适应性反应

焦虑状态：
  模糊/安全信号 → 杏仁核过度激活 → PFC调控不足 → 过度恐惧反应
```

**关键证据**：
- fMRI显示GAD患者杏仁核对模糊面孔反应增强
- 惊恐障碍患者杏仁核对惊恐相关词汇反应增强
- 社交焦虑患者杏仁核对负性评价预期反应增强
- 前额叶-杏仁核功能连接减弱预测焦虑严重度

### 1.2 海马与情境加工

| 功能 | 焦虑障碍中的表现 |
|------|-----------------|
| **情境辨别** | 海马功能受损 → 泛化恐惧（安全情境也触发焦虑） |
| **恐惧消退** | 海马-vmPFC连接减弱 → 消退记忆巩固障碍 |
| **时间编码** | 时间维度焦虑（对未来威胁的过度预期） |

### 1.3 岛叶与内感受

| 功能 | 焦虑障碍中的表现 |
|------|-----------------|
| **内感受加工** | 岛叶过度激活 → 对躯体感觉的灾难化解释（惊恐障碍核心） |
| **预测编码** | 对预期躯体感觉的预测误差放大 |
| **自我觉察** | 自我相关威胁加工增强（社交焦虑） |

### 1.4 默认模式网络（DMN）

| 特征 | 表现 |
|------|------|
| **DMN过度活跃** | 反刍思维、担忧、自我聚焦 |
| **DMN-CEN切换异常** | 难以从担忧中脱离，转向任务 |
| **后扣带回（PCC）** | 自我相关加工增强（社交焦虑中的被审视感） |

## 二、神经递质系统

### 2.1 5-羟色胺（5-HT）

| 通路 | 功能 | 焦虑障碍中的改变 |
|------|------|-----------------|
| **中缝背核-杏仁核通路** | 恐惧调节、情绪稳定 | 功能低下 → 杏仁核去抑制 |
| **中缝背核-PFC通路** | 认知控制、情绪调节 | 功能低下 → PFC调控减弱 |
| **中缝背核-海马通路** | 情境加工、消退学习 | 功能低下 → 泛化恐惧 |

**SSRIs的作用机制**：
- 急性期：增加突触间隙5-HT → 初期可能加重焦虑（5-HT1A/2C受体激活）
- 慢性期（2-4周后）：5-HT1A自受体脱敏 → 神经可塑性改变 → 杏仁核反应性降低

### 2.2 去甲肾上腺素（NE）

| 通路 | 功能 | 焦虑障碍中的改变 |
|------|------|-----------------|
| **蓝斑核-杏仁核通路** | 警觉、唤醒 | 过度激活 → 高警觉状态 |
| **蓝斑核-PFC通路** | 认知功能、注意 | 过度激活 → 注意力分散 |
| **蓝斑核-自主神经** | 交感激活 | 过度激活 → 心悸、出汗、震颤 |

**NE与惊恐障碍**：
- 蓝斑核电刺激可诱发惊恐样反应
- α2受体功能低下 → NE释放失控
- 育亨宾（α2拮抗剂）可诱发惊恐发作（实验室模型）

### 2.3 γ-氨基丁酸（GABA）

| 功能 | 焦虑障碍中的改变 |
|------|-----------------|
| **主要抑制性神经递质** | GABA-A受体功能低下 → 神经环路过度兴奋 |
| **BZD的作用靶点** | GABA-A受体正变构调节 → 氯离子内流增加 → 神经元超极化 |
| **神经甾体** | 别孕烯醇酮（allopregnanolone）→ GABA-A受体正变构调节 → 抗焦虑 |

### 2.4 谷氨酸

| 功能 | 焦虑障碍中的改变 |
|------|-----------------|
| **主要兴奋性神经递质** | 过度兴奋 → 神经毒性 |
| **杏仁核-BLA** | 谷氨酸能传递增强 → 恐惧记忆巩固增强 |
| **NMDA受体** | 恐惧学习的关键受体 |

### 2.5 多巴胺

| 通路 | 与焦虑的关系 |
|------|-------------|
| **中脑边缘通路** | 奖赏预期 → 焦虑与奖赏不确定性相关 |
| **中脑皮层通路** | 认知功能 → 焦虑伴注意力障碍 |

## 三、HPA轴与应激系统

### 3.1 下丘脑-垂体-肾上腺轴

```
应激/威胁
    ↓
下丘脑（PVN）释放CRH
    ↓
垂体释放ACTH
    ↓
肾上腺释放皮质醇
    ↓
负反馈：皮质醇→海马/垂体/下丘脑（GR/MR受体）
```

### 3.2 焦虑障碍中的HPA轴改变

| 改变 | 表现 | 临床意义 |
|------|------|---------|
| **基础皮质醇** | 多数焦虑障碍正常或轻度升高 | 与抑郁不同（抑郁常升高） |
| **应激反应** | 反应性增强或正常 | 对应激过度敏感 |
| **负反馈** | 部分研究显示负反馈减弱 | 难以从应激中恢复 |
| **CRH** | CSF中CRH升高（部分研究） | CRH受体拮抗剂为潜在靶点 |
| **DHEA-S** | 相对较低 | DHEA补充为潜在干预 |

### 3.3 自主神经系统

| 系统 | 焦虑障碍中的表现 |
|------|-----------------|
| **交感神经过度激活** | 心率加快、血压升高、出汗、瞳孔扩大 |
| **副交感功能不足** | 心率变异性（HRV）降低 → 情绪调节能力下降 |
| **迷走神经** | 迷走神经张力降低 → 抗炎作用减弱 |

## 四、遗传与表观遗传学

### 4.1 遗传度

| 焦虑障碍 | 遗传度（heritability） |
|---------|---------------------|
| **GAD** | 30-40% |
| **惊恐障碍** | 40-50% |
| **社交焦虑** | 30-50% |
| **特定恐惧** | 30-60% |

### 4.2 候选基因

| 基因 | 功能 | 与焦虑的关联 |
|------|------|------------|
| **5-HTTLPR（SLC6A4）** | 5-HT转运体 | 短等位基因（S）→ 焦虑/抑郁易感性增加（尤其在应激下） |
| **COMT** | 儿茶酚-O-甲基转移酶 | Val158Met多态性 → Met等位基因 → 焦虑易感性 |
| **BDNF** | 脑源性神经营养因子 | Val66Met → Met等位基因 → 海马体积减小、焦虑易感性 |
| **CRHR1/CRHR2** | CRH受体 | 变异 → HPA轴反应性改变 |
| **FKBP5** | HPA轴调节 | 变异 → 负反馈减弱 → PTSD/焦虑易感性 |
| **GAD1/GAD2** | GABA合成酶 | 变异 → GABA功能改变 |
| **NPY** | 神经肽Y | 低表达 → 焦虑增加 |
| **CYP450酶系** | 药物代谢 | 影响药物血药浓度 → 治疗反应差异 |

### 4.3 表观遗传学

| 机制 | 与焦虑的关系 |
|------|------------|
| **NR3C1甲基化** | 糖皮质激素受体基因 → 负反馈减弱 |
| **FKBP5甲基化** | HPA轴调节基因 |
| **童年应激** | 早期逆境 → 表观遗传标记 → 长期焦虑易感性 |
| **跨代传递** | 动物模型显示应激的表观遗传跨代效应 |

## 五、神经可塑性与焦虑

### 5.1 海马体积

| 发现 | 意义 |
|------|------|
| **GAD患者海马体积正常** | 与抑郁不同（抑郁常减小） |
| **PTSD患者海马体积减小** | 创伤特异性改变 |
| **慢性应激** | 可导致海马树突萎缩 |

### 5.2 杏仁核体积

| 发现 | 意义 |
|------|------|
| **社交焦虑患者杏仁核体积增大** | 恐惧加工增强的结构基础 |
| **治疗反应** | 有效治疗后杏仁核体积可减小 |

### 5.3 前额叶皮层厚度

| 发现 | 意义 |
|------|------|
| **vmPFC厚度** | 与恐惧消退能力正相关 |
| **治疗反应** | CBT可增加vmPFC灰质密度 |

## 六、整合神经生物学模型

```
遗传易感性（5-HTTLPR S、COMT Met、BDNF Met等）
    + 童年应激/逆境（表观遗传改变）
    + 近期应激/生活事件
                ↓
    ┌─────────────────────────────┐
    │    HPA轴反应性增强           │
    │    CRH释放增加               │
    │    皮质醇负反馈减弱          │
    └─────────────────────────────┘
                ↓
    ┌─────────────────────────────┐
    │    杏仁核过度激活            │
    │    对威胁/模糊信号过度反应   │
    │    恐惧条件化增强            │
    └─────────────────────────────┘
                ↓
    ┌─────────────────────────────┐
    │    PFC调控不足               │
    │    vmPFC消退学习减弱         │
    │    dlPFC认知重评减弱         │
    └─────────────────────────────┘
                ↓
    ┌─────────────────────────────┐
    │    神经递质失衡              │
    │    5-HT功能低下              │
    │    NE过度激活                │
    │    GABA功能低下              │
    └─────────────────────────────┘
                ↓
    ┌─────────────────────────────┐
    │    临床表现                  │
    │    过度担忧/恐惧             │
    │    回避行为                  │
    │    躯体症状                  │
    │    功能损害                  │
    └─────────────────────────────┘
```

## 七、生物标志物展望

| 领域 | 潜在标志物 | 应用 |
|------|-----------|------|
| **神经影像** | 杏仁核反应性、PFC-杏仁核功能连接 | 预测治疗反应、鉴别亚型 |
| **HPA轴** | 皮质醇觉醒反应（CAR）、DEX抑制试验 | 预测复发、指导治疗 |
| **炎症** | IL-6、CRP、TNF-α | 预测抗抑郁药/抗炎治疗反应 |
| **遗传** | 药物代谢基因（CYP450） | 指导药物选择 |
| **HRV** | 心率变异性 | 预测治疗效果、监测进展 |
