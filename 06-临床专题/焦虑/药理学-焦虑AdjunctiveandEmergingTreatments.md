# 焦虑障碍辅助与新兴治疗 | Adjunctive and Emerging Treatments for Anxiety Disorders

> **目标**：系统阐述焦虑障碍的增效策略、新型药物、物理治疗及前沿干预手段。
> **标准来源**：APA Practice Guidelines（2019）、Nature Reviews Drug Discovery、新兴临床试验证据。

---

## 一、非药物增效策略

### 1.1 运动处方

| 参数 | 推荐 |
|------|------|
| **类型** | 有氧运动（快走、慢跑、游泳、骑车） |
| **强度** | 中等（最大心率60-75%） |
| **频率** | 每周3-5次 |
| **时长** | 每次30-45分钟 |
| **疗程** | 至少10-12周见效 |
| **证据** | A级（GAD、广泛性焦虑症状） |
| **机制** | 增加5-HT、NE、BDNF；降低皮质醇；改善HRV |

### 1.2 光照治疗

| 参数 | 推荐 |
|------|------|
| **强度** | 10000 lux |
| **时长** | 30分钟/日 |
| **时间** | 晨间 |
| **证据** | B级（GAD、季节性焦虑） |

### 1.3 营养补充

| 补充剂 | 剂量 | 证据 | 注意 |
|--------|------|------|------|
| **Omega-3（EPA）** | 1-2g EPA/d | B级 | 抗炎；心血管益处 |
| **镁剂** | 200-400mg/d | C级 | 肌肉放松；睡眠 |
| **L-茶氨酸** | 200-400mg/d | C级 | GABA能增强 |
| **肌醇** | 12-18g/d | C级 | 惊恐障碍；胃肠道副作用 |
| **5-HTP** | 100-300mg/d | C级 | 血清素前体；不与SSRI同用 |
| **圣约翰草** | 300-600mg/d | C级 | 光敏；药物相互作用 |
| **藏红花提取物** | 30mg/d | C级 | 新兴证据 |
| **益生菌** | 多种菌株 | C级 | 肠-脑轴 |

### 1.4 其他辅助

| 干预 | 证据 | 说明 |
|------|------|------|
| **音乐治疗** | C级 | 放松；降低生理唤起 |
| **芳香疗法** | C级 | 薰衣草、柑橘 |
| **针灸** | C级 | GAD |
| **瑜伽** | B级 | GAD、PTSD |
| **太极** | B级 | GAD、老年焦虑 |

## 二、新型药物

### 2.1 氯胺酮（Ketamine）

| 特征 | 说明 |
|------|------|
| **机制** | NMDA受体拮抗剂；快速增加谷氨酸 → BDNF释放 → 突触可塑性 |
| **给药** | 静脉注射 0.5mg/kg，40分钟；或鼻内（艾司氯胺酮） |
| **起效** | 数小时 |
| **证据** | B级（GAD、社交焦虑、PTSD）；A级（抑郁） |
| **限制** | 需医疗机构给药；dissociation、血压升高等副作用；滥用风险 |
| **当前** | FDA批准用于抑郁（spravato）；焦虑为off-label |

### 2.2 致幻剂辅助心理治疗（Psychedelic-Assisted Therapy）

| 物质 | 研究阶段 | 适应症 | 注意 |
|------|---------|--------|------|
| **MDMA（摇头丸）** | III期临床试验 | PTSD（FDA突破性疗法认定） | 辅助心理治疗；心内膜病风险 |
| **裸盖菇素（Psilocybin）** | II期临床试验 | GAD、社交焦虑、临终焦虑 | 辅助心理治疗；持续效果 |
| **DMT/死藤水（Ayahuasca）** | 早期研究 | GAD、抑郁 | 合法性限制 |
| **LSD** | 早期研究 | 临终焦虑 | 微剂量研究 |

**共同特征**：
- 单次或少数几次给药
- 需专业心理支持（set and setting）
- 可能通过促进神经可塑性、增强情绪加工起效
- 目前多为研究阶段；FDA未批准用于焦虑

### 2.3 CRH受体拮抗剂

| 药物 | 靶点 | 研究阶段 | 前景 |
|------|------|---------|------|
| **Pexacerfont** | CRH1 | 已完成II期 | GAD未达主要终点 |
| **Verucerfont** | CRH1 | 早期 | 潜力有限 |

### 2.4 NK1受体拮抗剂

| 药物 | 研究 | 结果 |
|------|------|------|
| **Aprepitant** | 社交焦虑 | 阴性结果 |
| **其他NK1拮抗剂** | GAD/抑郁 | 总体阴性 |

### 2.5 神经甾体调节剂

| 药物 | 机制 | 研究 |
|------|------|------|
| **别孕烯醇酮类似物（如SAGE-217/Zuranolone）** | GABA-A受体正变构调节 | 产后抑郁（FDA批准）；焦虑研究中 |

### 2.6 食欲素（Orexin）受体拮抗剂

| 药物 | 机制 | 应用 |
|------|------|------|
| **Suvorexant** | Orexin受体拮抗 | 失眠（FDA批准）；焦虑辅助 |

## 三、物理治疗

### 3.1 重复经颅磁刺激（rTMS）

| 参数 | 说明 |
|------|------|
| **靶点** | 右DLPFC（低频1Hz，抑制性） |
| | 左DLPFC（高频10Hz，对抑郁有效，焦虑证据较弱） |
| | 内侧前额叶（mPFC） |
| **证据** | B级（GAD）；B级（PTSD）；证据有限但积极 |
| **参数** | 110-120% MT；3000脉冲/次；每周5次，4-6周 |
| **优势** | 无创；门诊；副作用少 |

### 3.2 经颅直流电刺激（tDCS）

| 参数 | 说明 |
|------|------|
| **靶点** | 右DLPFC（阴极，降低兴奋性） |
| **证据** | C级；研究阶段 |
| **优势** | 极便宜；可家用 |

### 3.3 迷走神经刺激（VNS）

| 应用 | 证据 |
|------|------|
| **GAD/PTSD** | C级；研究阶段 |
| **方法** | 植入式（手术）或经皮式 |

### 3.4 生物反馈

| 类型 | 应用 | 证据 |
|------|------|------|
| **心率变异性（HRV）** | 自主神经调节 | C级 |
| **皮肤电反应（GSR）** | 生理唤起 | C级 |
| **肌电图（EMG）** | 肌肉放松 | C级 |
| **脑电（EEG）** | 神经反馈 | C级 |

### 3.5 虚拟现实暴露治疗（VRET）

| 适应症 | 证据 |
|--------|------|
| **飞行恐惧** | A级 |
| **高处恐惧** | A级 |
| **蜘蛛恐惧** | A级 |
| **社交焦虑** | A级 |
| **广场恐惧** | B级 |
| **PTSD** | A级 |
