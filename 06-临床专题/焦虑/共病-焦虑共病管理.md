# 焦虑障碍共病管理 | Anxiety Comorbidity Management

> **目标**：系统阐述焦虑障碍与抑郁、双相、PTSD、OCD、物质使用、躯体疾病等常见共病的整合管理策略。

---

## 一、焦虑-抑郁共病

### 1.1 流行病学

- GAD与MDD共病率：50-70%
- 社交焦虑与MDD共病率：50%
- 惊恐障碍与MDD共病率：30-50%

### 1.2 治疗策略

| 策略 | 说明 |
|------|------|
| **首选药物** | SSRI/SNRI（同时覆盖焦虑和抑郁） |
| **心理治疗** | CBT整合（焦虑暴露 + 抑郁行为激活/认知重构） |
| **注意** | 单纯治疗焦虑可能使抑郁恶化；反之亦然；需同时关注两者 |
| **预后** | 共病患者治疗反应较慢、复发率较高 |

---

## 二、焦虑-双相共病

### 2.1 风险

- 焦虑是双相障碍最常见的共病（>50%）
- 抗抑郁药可能诱发躁狂/快速循环

### 2.2 治疗策略

| 策略 | 说明 |
|------|------|
| **基础** | 心境稳定剂（锂盐、丙戊酸盐、拉莫三嗪）或第二代抗精神病药 |
| **焦虑症状** | 优先考虑：喹硫平、奥氮平、拉莫三嗪（均有抗焦虑证据） |
| **慎用** | 抗抑郁药仅在联合心境稳定剂下谨慎使用 |
| **首选心理治疗** | CBT、正念、DBT技能 |

---

## 三、焦虑-PTSD共病

### 3.1 重叠与鉴别

| 特征 | PTSD | 惊恐/GAD |
|------|------|---------|
| **触发** | 创伤相关线索 | 非特异性/内在躯体感觉 |
| **核心** | 再体验、回避、高警觉 | 担忧/恐惧 |
| **惊恐发作** | 可有 | 核心/常见 |

### 3.2 治疗策略

| 策略 | 说明 |
|------|------|
| **药物** | 舍曲林、帕罗西汀、文拉法辛XR（均对PTSD和焦虑有效） |
| **心理治疗** | 稳定化→创伤聚焦治疗（CPT/PE/EMDR）→焦虑管理 |
| **注意** | 暴露治疗需在稳定化后进行；解离严重者需特别谨慎 |

---

## 四、焦虑-OCD共病

### 4.1 治疗策略

| 策略 | 说明 |
|------|------|
| **药物** | 高剂量SSRI（氟西汀/舍曲林/氟伏沙明60-80mg等效） |
| **心理治疗** | ERP（暴露与反应预防）为一线 |
| **整合** | CBT-I（针对失眠）+ ERP + 焦虑管理 |
| **增效** | 低剂量抗精神病药（如阿立哌唑、利培酮）用于难治性OCD |

---

## 五、焦虑-物质使用共病

### 5.1 常见模式

| 物质 | 使用模式 | 治疗 |
|------|---------|------|
| **酒精** | 自我治疗焦虑→依赖→戒断焦虑加重 | 先戒酒；SSRI治疗焦虑；注意肝功 |
| **苯二氮卓类** | 处方/非处方使用→依赖 | 缓慢 taper；替代为SSRI/CBT |
| **大麻** | 自我治疗→依赖→戒断焦虑加重 | 戒断；SSRI；CBT |
| **尼古丁** | 吸烟缓解焦虑 | 安非他酮（戒烟+抗抑郁） |

---

## 六、焦虑-躯体疾病共病

### 6.1 心血管疾病

| 问题 | 管理 |
|------|------|
| **焦虑导致的心悸/胸痛** | 排除心脏疾病后，呼吸训练 + SSRI |
| **心律失常患者** | 避免三环类；TCA过量可致命；SSRI相对安全 |
| **高血压** | 避免文拉法辛（剂量依赖性升压）；普瑞巴林可用 |

### 6.2 呼吸系统疾病

| 问题 | 管理 |
|------|------|
| **哮喘/COPD** | 焦虑加重气短感；需呼吸科协作；呼吸训练；SSRI |
| **过度换气综合征** | 呼吸再训练；认知行为治疗 |

### 6.3 胃肠道疾病

| 问题 | 管理 |
|------|------|
| **IBS** | 焦虑与IBS高度共病；SSRIs（尤其帕罗西汀）；TCA（低剂量）；CBT |
