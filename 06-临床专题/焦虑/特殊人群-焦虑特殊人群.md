# 焦虑障碍特殊人群临床方案 | Anxiety Disorders in Special Populations

> **目标**：建立儿童青少年、老年、围产期及文化适应的焦虑障碍临床方案。

---

## 一、儿童青少年焦虑障碍

### 1.1 流行病学

- 儿童焦虑障碍患病率：5-15%
- 社交焦虑、分离焦虑、GAD最常见
- 早期干预可预防成年后焦虑/抑郁

### 1.2 发展性表现

| 年龄 | 正常发育性恐惧 | 病理性焦虑 |
|------|--------------|-----------|
| **0-2岁** | 陌生人焦虑、分离焦虑 | 过度/持续；功能损害 |
| **3-6岁** | 黑暗、怪物、动物 | 回避日常活动；睡眠严重障碍 |
| **7-12岁** | 学业、同伴、健康 | 拒绝上学；社交回避 |
| **13-18岁** | 社交、未来、认同 | 社交隔离；物质使用 |

### 1.3 治疗

| 类型 | 首选 | 证据 |
|------|------|------|
| **心理治疗** | CBT（儿童友好版本） | A级 |
| **家长参与** | 家长管理训练 | A级 |
| **学校干预** | 社交技能训练；暴露支持 | B级 |
| **药物** | SSRI（氟西汀/舍曲林/艾司西酞普兰） | A级（≥7岁） |
| | | 黑框警告：监测自杀意念 |

## 二、老年焦虑障碍

### 2.1 特点

- 患病率：约10-15%
- 常表现为躯体症状（心悸、头晕、胃肠不适）
- 与抑郁、认知下降、躯体疾病共病常见
- 常被误诊为躯体疾病

### 2.2 治疗

| 方面 | 建议 |
|------|------|
| **药物** | 首选SSRI（舍曲林/艾司西酞普兰）；避免BZD（跌倒/认知损害） |
| **心理治疗** | CBT（简化版）；放松训练；问题解决 |
| **注意** | 药物相互作用；多重用药；起始低剂量 |

---

## 三、围产期焦虑

### 3.1 特点

- 孕期/产后焦虑患病率：约10-20%
- 产后焦虑常被忽视（更多关注产后抑郁）
- 可影响母婴依恋

### 3.2 治疗

| 方面 | 建议 |
|------|------|
| **药物** | SSRIs（舍曲林/氟西汀数据最多）；避免帕罗西汀 |
| **心理治疗** | CBT；正念；团体支持 |
| **哺乳** | 舍曲林/帕罗西汀优先 |

## 四、跨文化适应

### 4.1 文化相关综合征

| 综合征 | 文化背景 | 与焦虑的关系 |
|--------|---------|------------|
| **缩阳症（Koro）** | 东南亚/中国 | 对生殖器缩入体内的恐惧；急性焦虑 |
| **恐缩症** | 中国 | 类似koro |
| **taijin kyofusho** | 日本 | 社交焦虑的变异；害怕冒犯/使他人不适 |
| **ataque de nervios** | 拉丁美洲 | 类似惊恐发作；文化表达 |
| **神经衰ruò** | 中国 | 躯体化表达的焦虑/抑郁 |

### 4.2 文化适应原则

- 了解患者的文化背景对症状的理解
- 避免将文化正常的恐惧病理化
- 家庭参与的重要性（集体主义文化）
- 语言的细微差别（翻译量表需谨慎）
