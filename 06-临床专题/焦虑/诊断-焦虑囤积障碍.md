# 囤积障碍诊断与治疗 | Hoarding Disorder: Diagnosis and Treatment

> **目标**：建立囤积障碍（Hoarding Disorder）的精神科标准诊断与治疗框架。
> **标准来源**：DSM-5-TR（2022）、ICD-11（2022）、APA Practice Guidelines、Steketee & Frost (2014)。

---

## 一、DSM-5-TR 诊断标准

### 1.1 诊断要件

**A.** 长期难以丢弃物品，无论其实际价值如何。

**B.** 这种困难源于对丢弃物品感到痛苦以及感到需要保存物品。

**C.** 物品堆积导致生活区域拥挤和凌乱，严重到其预定用途受到影响。如果生活区域不凌乱，仅因第三方的干预（如家庭成员、清洁人员、当局）。

**D.** 囤积引起临床上显著的痛苦或社交、职业或其他重要领域的功能损害（包括为自己和他人保持安全环境的能力）。

**E.** 囤积不能归因于其他躯体疾病（如脑损伤、脑血管病、Prader-Willi综合征）。

**F.**  hoarding 不能用重性抑郁障碍、强迫症、创伤后应激障碍、精神病性障碍、重度神经认知障碍或孤独症谱系障碍的症状更好解释。

### 1.2 诊断特征

| 特征 | 说明 |
|------|------|
| **核心信念** | "以后可能用到""这有价值""扔掉是浪费""丢了就再也找不到了" |
| **情感依恋** | 对物品产生强烈的情感联结；物品成为自我的延伸 |
| **信息加工缺陷** | 分类困难、决策困难、记忆不信任（"如果我不看到它，我就会忘记"） |
| **回避** | 回避做出丢弃决定；拖延整理 |
| **获取行为** | 过度获取免费物品、廉价物品或不需要的物品 |
| **洞察力** | 多数缺乏洞察力（认为"没有问题"或"只是有点乱"） |

### 1.3 亚型

| 亚型 | 特征 |
|------|------|
| **良好洞察力** | 承认囤积有问题 |
| **不良洞察力** | 认为囤积不严重（但不完全否认） |
| **缺乏洞察力/妄想性信念** | 完全否认有问题 |

---

## 二、鉴别诊断

| 障碍 | 鉴别要点 |
|------|---------|
| **OCD** | OCD的囤积是对强迫思维的强迫反应；无情感依恋；常伴随清洁仪式 |
| **重性抑郁** | 抑郁导致的能量缺乏/动机下降；抑郁改善后 clutter 可改善 |
| **精神病性障碍** | 妄想性信念（如"政府派人来偷我的文件"） |
| **重度神经认知障碍** | 认知衰退导致无法组织/丢弃；伴随其他认知症状 |
| **孤独症谱系** | 刻板性收藏；对物品的兴趣而非情感依恋 |
| **Prader-Willi** | 食物囤积为特征；基因诊断 |
| **正常收藏** | 有组织、有选择；不造成功能损害 |

## 三、评估

### 3.1 标准化量表

| 量表 | 项目 | 用途 | 临界值 |
|------|------|------|--------|
| **SI-R（Saving Inventory-Revised）** | 23项 | 最常用；严重程度评估 | ≥41分：临床显著 |
| **HRS（Hoarding Rating Scale）** | 5项 | 临床访谈；严重程度 | 总分≥14分 |
| **HOMES（Home Environment Scale）** | — | 客观评估 clutter 程度 | 访视频率 |
| **UHSS（Uniform Hoarding Severity Scale）** | — | 标准化 severity 评估 | — |
| ** clutter Image Rating** | 9张图片 | 患者/评估者选择最像的图片 | — |

### 3.2 功能评估

| 领域 | 评估内容 |
|------|---------|
| **居住安全** | 火灾隐患、跌倒风险、卫生条件 |
| **社交功能** | 是否邀请他人到家；家庭冲突 |
| **职业/学业** | 工作空间 clutter；效率 |
| **健康** | 清洁困难；害虫；霉菌 |

## 四、治疗

### 4.1 认知行为治疗（CBT）— 一线

| 成分 | 方法 | 说明 |
|------|------|------|
| **动机访谈** | 解决矛盾心理 | 多数患者缺乏洞察力；强制开始 |
| **认知重构** | 挑战囤积信念 | "以后可能用到"→"过去3年没用到，概率低" |
| **分类训练** | 学习分类系统 | 将相似物品归类；使用标签 |
| **决策训练** | 快速决策规则 | "如果1年内没用到→丢弃/捐赠" |
| **非获取练习** | 阻止新获取 | 避免免费物品；购物清单 |
| **丢弃暴露** | 逐步丢弃物品 | 从最容易的开始；记录情绪变化 |
| **组织技能** | 建立维持系统 | "进一出一"规则；定期整理时间 |

### 4.2 CBT治疗结构（20-26次）

| 阶段 | 次数 | 内容 |
|------|------|------|
| **评估与动机** | 1-3 | 评估；心理教育；动机访谈 |
| **技能训练** | 4-8 | 分类、决策、认知重构 |
| **家中治疗** | 9-20 | 治疗师到家协助整理和丢弃 |
| **维持** | 21-26 | 预防复发；长期维持计划 |

### 4.3 药物治疗

| 药物 | 证据 | 说明 |
|------|------|------|
| **SSRI（如帕罗西汀）** | 有限 | 对共病焦虑/抑郁有效；对 hoarding 核心症状效果有限 |
| **文拉法辛** | 个案 | 可能对部分患者有效 |
| **注意** | — | 药物不是 hoarding 的一线治疗 |

### 4.4 其他干预

| 干预 | 证据 | 说明 |
|------|------|------|
| **支持性整理** | 有限 | 专业组织者协助；但若无 CBT 易复发 |
| **团体 CBT** | B级 | 同伴支持；成本效益 |
| **家庭干预** | C级 | 教育家庭成员；减少冲突 |
| **强制清理** | 不推荐 | 无心理治疗的情况下强制清理通常无效且造成创伤 |

## 五、特殊人群

### 5.1 老年人

| 注意 | 说明 |
|------|------|
| **起病** | 老年 hoarding 可能是神经认知障碍的早期表现 |
| **评估** | 需认知评估排除痴呆 |
| **安全** | 跌倒、火灾风险更高 |
| **治疗** | 可能需要更多次、更短的治疗会谈 |

### 5.2 儿童/青少年

| 注意 | 说明 |
|------|------|
| **表现** | 物品收藏；难以整理房间 |
| **家庭干预** | 父母参与为核心 |
| **学校** | 书包/储物柜 clutter |
