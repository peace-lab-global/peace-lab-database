# 身体聚焦重复行为：拔毛癖与抓痕障碍 | BFRB: Trichotillomania and Excoriation Disorder

> **目标**：建立拔毛癖（TTM）与抓痕障碍（ExD）的精神科标准诊断与治疗框架。
> **标准来源**：DSM-5-TR（2022）、APA Practice Guidelines、Franklin & Tolin (2007)。

---

## 一、拔毛癖（Trichotillomania, TTM）

### 1.1 DSM-5-TR 诊断标准

**A.** 反复拔自己的毛发，导致毛发脱落。

**B.** 反复尝试减少或停止拔毛发。

**C.** 拔毛发引起临床上显著的痛苦或社交、职业或其他重要领域的功能损害。

**D.** 拔毛发或毛发脱落不能归因于其他躯体疾病（如皮肤病）。

**E.** 拔毛发不能用其他精神障碍的症状更好地解释（如拔毛发以改善妄想或强迫思维中出现的外貌缺陷）。

### 1.2 诊断特征

| 特征 | 说明 |
|------|------|
| **部位** | 头皮（最常见）、眉毛、睫毛、 pubic/ body hair |
| **模式** | 可能为焦点型（有意识）或自动型（无意识） |
| **前兆感觉** | 拔毛前常有紧张感或瘙痒感；拔毛后缓解 |
| **功能** | 情绪调节（缓解焦虑、无聊、紧张） |
| **后果** | 脱发斑块、毛囊炎、社交回避、羞耻 |

---

## 二、抓痕障碍（Excoriation Disorder, ExD / Skin-Picking）

### 2.1 DSM-5-TR 诊断标准

**A.** 反复搔抓自己的皮肤，导致皮肤病变。

**B.** 反复尝试减少或停止搔抓皮肤。

**C.** 搔抓皮肤引起临床上显著的痛苦或社交、职业或其他重要领域的功能损害。

**D.** 搔抓皮肤不能归因于其他躯体疾病（如 scabies）或精神障碍（如精神病性障碍的妄想、物质使用的幻觉）。

**E.** 搔抓皮肤不能用其他精神障碍的症状更好地解释（如拔毛癖、躯体变形障碍的搔抓）。

### 2.2 诊断特征

| 特征 | 说明 |
|------|------|
| **部位** | 面部、手臂、手、躯干 |
| **模式** | 焦点型（有意识）或自动型（无意识） |
| **前兆** | 皮肤瑕疵、焦虑、无聊；搔抓后缓解 |
| **后果** | 皮肤损伤、感染、疤痕、社交回避 |

---

## 三、鉴别诊断

| 障碍 | TTM 鉴别 | ExD 鉴别 |
|------|---------|---------|
| **OCD** | OCD 的拔毛是对强迫思维的仪式反应 | OCD 的搔抓是对污染的清洁 |
| **BDD** | BDD 可能拔毛以改善感知缺陷 | BDD 可能搔抓以改善皮肤 |
| **精神病性** | 妄想驱动（如认为毛发内有虫） | 妄想/幻觉驱动 |
| **物质所致** | 兴奋剂使用（如冰毒） | 兴奋剂/阿片戒断 |
| **躯体疾病** | 斑秃、甲减、真菌感染 | 疥疮、湿疹、疱疹 |
| **正常行为** | 偶尔拔分叉发梢；不造成功能损害 | 偶尔挤痘痘；不造成功能损害 |

## 四、评估

### 4.1 标准化量表

| 量表 | 用途 | 适用 |
|------|------|------|
| **MGH-HPS（Massachusetts General Hospital Hairpulling Scale）** | TTM 严重程度 | TTM |
| **SPIS（Skin Picking Impact Scale）** | ExD 严重程度 | ExD |
| **NEQ（Nail Excoriation Questionnaire）** | 咬指甲严重程度 | 咬指甲 |
| **BFRB 综合评估** | 多种 BFRB 评估 | 所有 BFRB |

### 4.2 功能分析

| 问题 | 目的 |
|------|------|
| "什么时候最常拔毛/搔抓？" | 识别触发情境 |
| "拔毛/搔抓前你在做什么？感觉如何？" | 识别前兆情绪 |
| "拔毛/搔抓后感觉如何？" | 识别强化后果 |
| "一天花多少时间？" | 评估频率和耗时 |

## 五、治疗

### 5.1 习惯逆转训练（Habit Reversal Training, HRT）— 一线

HRT 是 TTM 和 ExD 的金标准心理治疗。

| 成分 | 方法 |
|------|------|
| **意识训练** | 识别拔毛/搔抓的前兆信号（情境、情绪、躯体感觉） |
| **竞争反应训练** | 用不兼容的行为替代（如握拳、玩减压球、涂抹护手霜） |
| **刺激控制** | 减少触发物（如镜子覆盖、戴手套、修剪指甲） |
| **动机维持** | 记录进展；奖励目标达成 |

### 5.2 HRT 协议（8-12次）

| 阶段 | 内容 |
|------|------|
| **1-2次** | 评估；心理教育；功能分析 |
| **3-5次** | 意识训练；识别前兆 |
| **6-8次** | 竞争反应训练；练习替代行为 |
| **9-10次** | 刺激控制；环境改造 |
| **11-12次** | 维持计划；复发预防 |

### 5.3 竞争反应示例

| BFRB | 竞争反应 |
|------|---------|
| **拔头发** | 握拳（手指无法抓握）；玩减压球；戴帽子 |
| **搔抓皮肤** | 涂抹护手霜（滑腻感阻止搔抓）；戴手套；握拳 |
| **咬指甲** | 涂抹苦味指甲油；玩小物件；戴指套 |
| **咬嘴唇/颊** | 嚼无糖口香糖；含硬糖 |

### 5.4 药物治疗

| 药物 | 证据 | 说明 |
|------|------|------|
| **NAC（N-乙酰半胱氨酸）** | B级 | 1200-2400mg/日；谷氨酸调节；TTM 和 ExD 均有效 |
| **SSRI（如氟西汀）** | C级 | 对共病焦虑/抑郁有效；对 BFRB 核心症状效果有限 |
| **拉莫三嗪** | C级 | 个案报告 |
| **奥氮平** | C级 | 小剂量增效 |

**注意：** NAC 是目前 BFRB 最有前景的药物；非 FDA 批准适应症。

### 5.5 其他治疗

| 治疗 | 证据 | 说明 |
|------|------|------|
| **接纳与承诺治疗（ACT）** | B级 | 接纳冲动；价值观导向行为 |
| **辩证行为治疗（DBT）** | C级 | 情绪调节；冲动容忍 |
| **正念训练** | C级 | 增加对前兆的觉察 |

## 六、共病

| 共病 | 患病率 | 管理 |
|------|--------|------|
| **焦虑** | 40-60% | HRT + 焦虑管理 |
| **抑郁** | 30-50% | 治疗抑郁；SSRI |
| **OCD** | 10-20% | 标准 OCD 治疗 |
| **BDD** | 5-15% | BDD 优先 |
| **ADHD** | 10-20% | ADHD 治疗可改善自动型 BFRB |
| **其他 BFRB** | 常见 | 咬指甲、咬嘴唇等 |

## 七、特殊人群

### 7.1 儿童/青少年

| 注意 | 说明 |
|------|------|
| **常见** | TTM 在青少年中常见 |
| **自动型为主** | 多为无意识行为（看电视、做作业时） |
| **家庭干预** | 父母参与；不批评；帮助识别触发 |
