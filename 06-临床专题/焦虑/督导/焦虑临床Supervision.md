# 焦虑障碍临床督导与质量控制 | Clinical Supervision and Quality Control for Anxiety Disorders

> **目标**：建立精神科标准的焦虑障碍临床督导框架与质量控制体系。

---

## 一、督导核心议题

| 议题 | 内容 |
|------|------|
| **暴露治疗质量** | 暴露层级是否合理；患者是否真正体验到焦虑上升与下降 |
| **安全行为识别** | 患者是否在暴露中隐藏安全行为（携带水、靠近出口、紧握手机等） |
| **依从性** | 家庭作业完成率；暴露练习的频率和时长 |
| **共病识别** | 是否遗漏抑郁、物质使用、人格障碍 |
| **药物整合** | 心理治疗师与精神科医生的协作 |
| **反移情** | 治疗师对回避行为的挫败感；对暴露的恐惧（担心伤害患者） |

## 二、质量控制指标

| KPI | 目标 |
|-----|------|
| 暴露治疗使用率 | ≥80%（特定恐惧/广场恐惧应100%） |
| 家庭作业完成率 | ≥70% |
| 急性期治愈率 | ≥40%（12周） |
| 有效率 | ≥60%（12周） |
| 脱落率 | ≤20% |
| 患者满意度 | ≥4.0/5.0 |

## 三、文档标准

- 暴露层级表
- 每次暴露记录（SUDs评分、安全行为、结果）
- 量表追踪
- 安全计划（中高风险）
