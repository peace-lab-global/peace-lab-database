# CBT-E 认知行为治疗 (Cognitive Behaviour Therapy — Enhanced)

> **核心定位**：Christopher Fairburn 开发的跨诊断进食障碍治疗方案，适用于 AN/BN/BED/OSFED 所有诊断。20次/20周标准方案是当前证据最强的进食障碍心理治疗。

---

## 1. 理论模型

### 1.1 跨诊断核心维持机制

```
┌─────────────────────────────────────────────┐
│           CBT-E 跨诊断维持模型               │
│                                             │
│  临床完美主义 ←──→ 自我评价过度依赖体型/体重  │
│       ↕                    ↕               │
│  核心低自尊    ←──→  人际困难               │
│       ↕                                    │
│  情绪事件触发 → 限制进食 ←→ 暴食 ←→ 清除    │
│                                             │
│  饮食规则 → 违反规则 → 失控感 → 暴食        │
│     ↑                                        │
│     └────────────────────────────────────────┘
└─────────────────────────────────────────────┘
```

### 1.2 四大维持因素

| 维持因素 | 描述 | 干预靶点 |
|:---------|:-----|:---------|
| **饮食限制** | 刻意控制进食量/食物类型/时间 | 打破规则、规律进食 |
| **暴食事件** | 限制→饥饿→暴食恶性循环 | 规律进食、情绪管理 |
| **补偿行为** | 清除/过度运动/禁食 | 减少→消除 |
| **体型过度评价** | 自我价值过度依赖外表 | 拓宽自尊来源 |

### 1.3 可选附加模块

| 模块 | 适用条件 | 干预内容 |
|:-----|:---------|:---------|
| **临床完美主义** | 完美主义维持ED | 灵活标准/实验法 |
| **核心低自尊** | 低自尊维持ED | 自我接纳/成就重评 |
| **人际困难** | 人际问题维持ED | 人际问题解决 |
| **情绪不耐受** | 情绪调节困难维持暴食 | 情绪觉察/痛苦耐受 |

## 2. 治疗方案结构

### 2.1 标准方案参数

| 参数 | 标准版 | 重度版 (AN) |
|:-----|:-------|:-----------|
| 总次数 | 20 次 | 40 次 |
| 总时长 | 20 周 | 40 周 |
| 单次时长 | 50 分钟 | 50 分钟 |
| 频率 | 每周1次（前8次可2次/周） | 每周1-2次 |
| 适用诊断 | BN/BED/OSFED | AN/严重 BN |

### 2.2 四阶段结构

| 阶段 | 次数 | 目标 | 核心技术 |
|:-----|:----:|:-----|:---------|
| **Stage 1: 启动** | 1-8次 | 建立联盟/稳定进食/心理教育 | 周称体重/自我监测/规律进食/心理教育/个案概念化 |
| **Stage 2: 回顾** | 9-10次 | 阶段性回顾/调整计划 | 评估进展/识别障碍/修改策略 |
| **Stage 3: 核心治疗** | 11-17次 | 处理维持因素 | 饮食规则/暴食触发/体型回避/事件处理 |
| **Stage 4: 结束** | 18-20次 | 维持与复发预防 | 回顾进步/复发预防计划/未来规划 |

### 2.3 Stage 1 核心技术详解

| 技术 | 说明 | 实施方式 |
|:-----|:-----|:---------|
| **协作称重** | 每次就诊固定时间称重 | 治疗师与患者一起看数字（非盲称） |
| **实时自我监测** | 记录所有进食+想法+情绪 | 实时记录（非事后回忆），使用日记本/APP |
| **规律进食** | 3餐+2-3次点心，间隔≤4小时 | 打破限制→暴食循环的基础 |
| **心理教育** | ED 模型/限制→暴食机制/体重设定点 | 帮助理解为什么治疗有效 |
| **个案概念化** | 个体化的维持机制图 | 治疗师与患者共同绘制 |

### 2.4 Stage 3 核心技术详解

| 技术 | 靶点 | 方法 |
|:-----|:-----|:-----|
| **饮食规则消除** | 全或无规则 | 识别规则→渐进打破→暴露"禁忌"食物 |
| **事件分析** | 暴食/清除事件 | 事件链分析：触发→思维→行为→后果→替代策略 |
| **体像暴露** | 体型回避/检查 | 减少回避（镜子/泳装/照片）+ 减少检查（频繁称重/捏肉） |
| **问题解决** | 日常困难 | 5步问题解决法（识别→生成→评估→选择→回顾） |
| **认知重构** | 自动化思维 | Socratic 提问/行为实验/证据审查 |
| **拓宽自尊** | 体型过度评价 | "生活饼图"→发展非外貌自我价值来源 |

## 3. 适应症与选择

### 3.1 适用范围

| 诊断 | 证据 | NICE推荐 |
|:-----|:-----|:---------|
| BN | 强（多项RCT） | ✅ 一线 |
| BED | 强（多项RCT） | ✅ 一线 |
| AN（成人） | 中-强 | ✅ 一线 |
| AN（青少年） | 中（FBT优先） | 二线（FBT后） |
| OSFED | 中 | ✅ 推荐 |

### 3.2 CBT-E vs 其他治疗选择

| 比较 | CBT-E 优势 | 其他方案优势 |
|:-----|:-----------|:------------|
| vs FBT（青少年AN） | 跨诊断/成人 | FBT 青少年AN 效果更好 |
| vs IPT | 更快见效/暴食更优 | IPT 长期人际改善 |
| vs DBT | 全谱系ED | DBT 情绪调节/BPD共病 |
| vs SSCM | 结构化更强 | SSCM AN 体重恢复相当 |
| vs MANTRA | 证据更充分 | MANTRA AN 动机提升 |

## 4. 特殊考量

### 4.1 AN 中的 CBT-E 调整

| 调整 | 说明 |
|:-----|:-----|
| 延长至40次 | 重度/低体重患者 |
| 体重恢复优先 | Stage 1 更强调营养恢复 |
| 动机工作 | 更多 Motivational Interviewing 整合 |
| 家庭参与 | 可选的 "重要他人" 参与环节 |
| 多专业协作 | 与营养师/医生密切配合 |
| 盲称 vs 明称 | CBT-E 用明称（协作性） |

### 4.2 治疗脱落预防

| 策略 | 说明 |
|:-----|:-----|
| 早期联盟建设 | Stage 1 投入时间建立信任 |
| 动机评估 | 定期评估改变意愿 |
| 灵活调整 | 根据个体需要增减模块 |
| 预期管理 | 体重恢复期的不适是正常的 |
| 危机处理 | 有自杀/严重共病时优先处理 |

## 5. 疗效证据

| 研究/综述 | 诊断 | 效果 |
|:---------|:-----|:-----|
| Fairburn et al., 2009 (多中心RCT) | BN/BED/OSFED | 暴食停止率 ~45-50% |
| Fairburn et al., 2015 (AN RCT) | AN 成人 | 体重恢复+症状改善与 SSCM 相当 |
| Atkinson & Wade, 2023 (Meta) | 所有 ED | CBT-E 优于等待列表和 TAU |
| NICE NG69, 2017 | BN/BED | 推荐为一线 |

### 5.1 疗效预测因子

| 良好预测 | 不良预测 |
|:---------|:---------|
| 早期体重增加（AN） | 严重完美主义 |
| 早期减少暴食/清除 | 高度饮食限制 |
| 治疗联盟好 | 严重共病人格障碍 |
| 高治疗动机 | 低动机/治疗不愿 |
| 完成全部阶段 | 早期脱落 |
