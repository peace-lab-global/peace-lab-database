# 营养康复与体重恢复 (Nutrition Rehabilitation)

> **核心定位**：营养康复是进食障碍治疗的基石（"starved brain can't do therapy"）。注册营养师在多学科团队中负责膳食评估、再喂养、营养教育与长期维持。

---

## 1. 营养师在 MDT 中的角色

| 职责 | 说明 |
|:-----|:-----|
| 营养评估 | 当前摄入/营养素缺乏/饮食史 |
| 再喂养方案 | 安全启动体重恢复 |
| 膳食计划 | 个体化膳食结构 |
| 营养教育 | 食物知识/均衡饮食/直觉进食 |
| 餐时支持 | 用餐陪伴/挑战恐惧食物 |
| 长期维持 | 体重维持/预防复发 |

---

## 2. 再喂养方案（详细）

### 2.1 启动原则

| 参数 | 门诊 | 住院 |
|:-----|:-----|:-----|
| 起始热量 | 1,200-1,500 kcal/d | 800-1,200 kcal/d |
| 递增速度 | 每2天 +200 kcal | 每1-2天 +100-200 kcal |
| 目标热量 | 2,500-4,000 kcal/d | 3,000-5,000 kcal/d |
| 体重增长目标 | 0.25-0.5 kg/周 | 0.5-1.0 kg/周 |
| 磷监测 | 基线+第3/5/7天 | 每日（前7天） |

### 2.2 再喂养综合征预防

| 措施 | 说明 |
|:-----|:-----|
| 风险分层 | BMI/体重下降速度/摄入量 |
| 预防性补磷 | 高风险患者口服磷 |
| 缓慢启动 | 高风险患者更低起始热量 |
| 密切监测 | 磷/钾/镁/液体平衡 |
| 液体限制 | 水肿/低钠时限制液体 |

---

## 3. 膳食计划

### 3.1 AN 膳食计划原则

| 原则 | 说明 |
|:-----|:-----|
| 规律进食 | 3餐 + 2-3点心（间隔≤4小时） |
| 均衡宏量 | 碳水 50-55% / 蛋白 15-20% / 脂肪 25-30% |
| 渐进扩展 | 从安全食物→恐惧食物 |
| 热量密度 | 体重恢复期需高热量密度 |
| 营养补充 | 维生素D/钙/铁/锌/B族 |

### 3.2 BN/BED 膳食计划原则

| 原则 | 说明 |
|:-----|:-----|
| 规律进食 | 消除限制→暴食循环的关键 |
| 中性化食物 | "禁止食物"→"中性食物" |
| 暴露进食 | 渐进面对触发暴食物 |
| 直觉进食 | 长期目标（恢复后） |

---

## 4. 直觉进食 (Intuitive Eating)

| 原则 | 说明 | 时机 |
|:-----|:-----|:-----|
| 拒绝节食心态 | 放弃"好/坏"食物二分法 | 体重恢复后 |
| 尊重饥饿 | 响应身体饥饿信号 | 恢复中后期 |
| 和平共处食物 | 允许所有食物 | 恢复后期 |
| 挑战食物警察 | 停止内心"食物道德化" | 全程 |
| 尊重饱腹 | 识别饱腹信号 | 恢复后 |
| 尊重身体 | 接纳自然体型 | 维持期 |

> ⚠️ **重要**：直觉进食适用于**体重恢复后**。急性期需要结构化膳食计划（不能依赖直觉——饥饿/饱腹信号已被扰乱）。
