# 回避性/限制性摄食障碍 (ARFID)

> **核心定位**：DSM-5 新增诊断，以非体像驱动的进食回避/限制为特征，儿童多见但成人亦有发生。与 AN 的核心区别在于**无恐胖/无体像关注**。

---

## 1. 诊断标准

### 1.1 DSM-5-TR 诊断标准 (307.59 / F50.82)

| 标准 | 内容 | 操作化 |
|:-----|:-----|:-------|
| **A. 进食/喂养障碍** | 明显无法满足营养/能量需求的进食回避或限制 | 可表现为食物种类极少/总量极少/两者兼具 |
| **B. ≥1项后果** | ①显著体重下降/不增长 ②显著营养缺乏 ③依赖肠内营养/口服补充 ④显著心理社会功能损害 | 至少1项 |
| **C. 排除** | 不能用食物不可得或文化习俗解释 | 宗教斋戒/贫困/文化禁忌 |
| **D. 非体像驱动** | 进食障碍**不是**由于对体型/体重的关注 | 与 AN/BN 的核心区别 |
| **E. 排除** | 不能用其他医学状况或精神障碍完全解释 | 如共病 AN/BN/神经性厌食则不诊断 ARFID |

### 1.2 ARFID 三种亚型

| 亚型 | 核心机制 | 常见人群 | 典型表现 |
|:-----|:---------|:---------|:---------|
| **感觉敏感型** | 对食物的颜色/质地/气味/温度/味道过度敏感 | 儿童/ASD/SPD | 只吃特定品牌/颜色/质地的食物 |
| **恐惧型（厌恶经历）** | 因呛噎/呕吐/过敏等经历形成条件性恐惧 | 任何年龄 | 经历呛噎后回避固体食物 |
| **低兴趣型** | 对进食缺乏兴趣/低食欲/容易分心 | 婴幼儿/老年人 | "忘记吃饭"/进食如完成任务 |

---

## 2. 流行病学

| 指标 | 数值 |
|:-----|:-----|
| 儿童患病率 | 1.5-5.0%（儿科门诊更高） |
| 成人患病率 | 0.3-1.0%（识别不足） |
| 性别比 | 接近 1:1（无明显女性优势） |
| 起病年龄 | 多为儿童期，但可任何年龄 |
| ASD 共病率 | 12-33%（ASD 患者中 ARFID 比例高） |
| 焦虑共病率 | 20-40% |
| 自然缓解率 | 部分儿童随年龄改善，但可转为 AN |

## 3. 评估

### 3.1 评估工具

| 工具 | 类型 | 适用人群 | 特点 |
|:-----|:-----|:---------|:-----|
| Nine Item ARFID Screen (NIAS) | 自填问卷 | 成人/青少年 | 9项快速筛查 |
| Eating Pathology Symptoms Inventory (EPSI) | 自填问卷 | 成人 | 含 ARFID 分量表 |
| ChEAT (Children's EAT) | 自填问卷 | 儿童 | 家长/儿童填写 |
| ARFID 模块（C-YFAS/临床访谈） | 结构化访谈 | 儿童/成人 | 最准确 |
| 营养评估 | 临床评估 | 所有 | 24h 膳食回顾/营养素分析 |
| 生长曲线 | 体格评估 | 儿童 | 身高/体重百分位追踪 |

### 3.2 评估重点

| 维度 | 评估内容 |
|:-----|:---------|
| 食物品种 | 可接受食物的完整清单（种类/品牌/制备方式） |
| 营养状态 | 宏量/微量营养素摄入 vs 需求 |
| 生长/体重 | 生长曲线偏离/体重下降幅度 |
| 功能损害 | 社交回避（不能在外吃饭）/学业影响 / 家庭冲突 |
| 感觉特征 | 触觉/嗅觉/味觉敏感度（Sensory Profile） |
| 恐惧经历 | 呛噎/呕吐/过敏事件史 |
| 共病筛查 | ASD / ADHD / 焦虑 / OCD |

## 4. 鉴别诊断

| 鉴别 | ARFID vs | 关键区分 |
|:-----|:---------|:---------|
| AN | AN | **有无体像关注/恐胖** |
| 正常挑食 | 正常发展性挑食 | ARFID 有营养/功能损害 |
| GI 疾病 | IBD/乳糜泻/胃排空障碍 | 排除器质性病因 |
| 食物过敏 | IgE/非IgE 过敏 | 过敏原回避 vs 泛化回避 |
| ASD 进食问题 | ASD 进食困难 | ARFID 有独立临床显著性 |
| OCD 相关 | OCD 食物强迫 | 强迫思维驱动 vs 感觉/恐惧 |

## 5. 治疗

### 5.1 治疗原则

| 原则 | 说明 |
|:-----|:-----|
| **无通用一线方案** | ARFID 治疗研究远少于 AN/BN |
| **个体化** | 根据亚型选择策略 |
| **多学科** | 心理 + 营养 + OT（感觉型）+ 儿科 |
| **家庭参与** | 家长培训是核心 |
| **渐进暴露** | 所有亚型都需要系统脱敏 |

### 5.2 按亚型的治疗策略

| 亚型 | 核心治疗 | 具体技术 |
|:-----|:---------|:---------|
| **感觉敏感型** | 感觉整合 + 食物暴露 | 食物链法(food chaining)/感觉游戏/渐进暴露 |
| **恐惧型** | 暴露疗法 + CBT | 系统脱敏/内脏暴露/认知重构 |
| **低兴趣型** | 行为激活 + 结构化进食 | 定时进食/强化物/食欲刺激 |

### 5.3 营养干预

| 策略 | 说明 |
|:-----|:-----|
| 口服营养补充 | 确保营养充足（如 Pediasure/Ensure） |
| 食物链法 | 从已接受食物→相似食物的渐进扩展 |
| 管饲（严重） | 营养补充/体重恢复 |
| 维生素/矿物质补充 | 纠正缺乏（铁/锌/维生素D等） |

### 5.4 药物（辅助/有限证据）

| 药物 | 用途 | 证据 |
|:-----|:-----|:-----|
| 赛庚啶(cyproheptadine) | 食欲刺激 | 弱（儿科病例报告） |
| 米氮平(mirtazapine) | 食欲刺激/焦虑 | 弱 |
| 奥氮平 | 严重限制型 | 初步证据 |

---

## 6. 预后

| 结果 | 说明 |
|:-----|:-----|
| 儿童自然改善 | 部分随年龄扩展食物品种 |
| 持续至成年 | ~15-30% 可能持续到成年 |
| 转为 AN 风险 | ~10-15%（需监测体像关注的发展） |
| 营养后遗症 | 长期限制可能导致生长迟缓/骨质疏松 |
