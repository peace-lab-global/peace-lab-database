# DBT 辩证行为疗法在进食障碍中的应用

> **核心定位**：针对暴食行为（BED/BN）的情绪调节取向治疗。核心假设：暴食是**功能失调的情绪调节策略**。DBT-BED 是 BED 的有效替代方案（CBT 后二线）。

---

## 1. DBT-ED 理论模型

```
生物易感性（高情绪敏感）
        +
无效化环境（情绪不被认可）
        ↓
情绪调节缺陷
        ↓
暴食 = 快速但有害的情绪调节策略
        ↓
短期缓解 → 长期恶化 → 更多负面情绪 → 更多暴食
```

---

## 2. 四模块在 ED 中的应用

### 2.1 模块概览

| 模块 | 核心技能 | ED 应用 |
|:-----|:---------|:--------|
| **正念 (Mindfulness)** | 觉察当下/不评判 | 觉察饥饿/饱腹信号/情绪触发 |
| **痛苦耐受 (Distress Tolerance)** | 在不暴食的情况下承受痛苦 | 暴食冲动的替代行为 |
| **情绪调节 (Emotion Regulation)** | 识别/标记/改变情绪 | 减少情绪性暴食 |
| **人际效能 (Interpersonal Effectiveness)** | 需求表达/边界设定 | 减少人际触发的暴食 |

### 2.2 关键技能在 ED 中的具体应用

| 技能 | 暴食触发场景 | 替代应用 |
|:-----|:------------|:---------|
| **STOP** | 暴食冲动来袭 | 停下来→观察→拉开距离→继续行动 |
| **TIPP** | 强烈情绪+暴食冲动 | 温度/剧烈运动/节律呼吸/配对放松 |
| **自我安抚（五感）** | 孤独/焦虑触发暴食 | 热茶/音乐/香薰/柔软织物替代食物 |
| **相反行动** | 因恐惧回避食物 | 主动面对恐惧食物（暴露） |
| **PLEASE** | 生理状态差→暴食 | 照顾身体/规律进食/睡眠/运动/药物 |
| **冲浪冲动 (Urge Surfing)** | 暴食冲动 | 观察冲动起伏而非行动 |
| **DEAR MAN** | 人际冲突→暴食 | 有效表达需求/设定边界 |

## 3. 治疗结构

### 3.1 DBT-BED 标准方案（Safer, Telch & Wilfley）

| 组件 | 频率 | 时长 |
|:-----|:-----|:-----|
| 个体治疗 | 每周1次 | 50分钟 |
| 技能团体 | 每周1次 | 90-120分钟 |
| 电话教练 | 按需 | 10-15分钟 |
| 咨询团队会 | 每周1次 | 60分钟 |
| 总疗程 | — | 20周 |

### 3.2 简化版 DBT-BED（20次个体方案）

| 阶段 | 次数 | 内容 |
|:-----|:----:|:-----|
| 初始/承诺 | 1-2 | 行为链分析/治疗承诺 |
| 正念 | 3-5 | 正念进食/身体扫描 |
| 情绪调节 | 6-12 | 情绪识别/PLEASE/相反行动 |
| 痛苦耐受 | 13-17 | TIPP/自我安抚/冲动冲浪 |
| 结束 | 18-20 | 回顾/复发预防 |

## 4. 疗效证据

| 研究 | 诊断 | 效果 |
|:-----|:-----|:-----|
| Safer et al., 2001 (RCT) | BN | DBT 暴食减少率 50%+ |
| Telch et al., 2009 (RCT) | BED | DBT 暴食停止率 56% |
| Safer et al., 2010 (RCT) | BED | DBT vs 等待列表：显著优于 |
| Hill et al., 2011 (Meta) | BED/BN | DBT 效果中等偏大 |

### 4.1 DBT-ED 最佳适应场景

| 场景 | 说明 |
|:-----|:-----|
| CBT 无效/不耐受 | CBT 后仍暴食 |
| 情绪触发明显 | 暴食主要由负面情绪驱动 |
| 共病 BPD | 同时处理 BPD+ED |
| 自伤/NSSI | 多靶点行为 |
| 物质滥用共病 | 共享情绪调节缺陷 |
