# 其他特定进食障碍 (OSFED) 与非特定进食障碍 (UFED)

> **核心定位**：OSFED 是临床上**最常见的进食障碍诊断**（占 ED 门诊 30-50%），涵盖有临床显著进食障碍症状但不完全满足 AN/BN/BED/ARFID 标准的情况。

---

## 1. 诊断框架

### 1.1 OSFED vs UFED

| 诊断 | DSM-5-TR 代码 | 定义 | 使用场景 |
|:-----|:-------------|:-----|:---------|
| **OSFED** | 307.50 (F50.89) | 有进食障碍症状→临床显著痛苦/功能损害，但不完全满足 AN/BN/BED/ARFID 全标准 | 临床医生可指明具体原因 |
| **UFED** | 307.50 (F50.9) | 有进食障碍症状→临床显著痛苦，但信息不足以做特定诊断 | 急诊/信息不全/不愿透露 |

### 1.2 OSFED 五种亚型（DSM-5-TR 列出的示例）

| 亚型 | 与标准诊断的差异 | 临床特征 |
|:-----|:----------------|:---------|
| **1. 非典型神经性厌食** | 满足 AN 全部标准，但体重在正常范围 | 同样严重的医学/心理后果 |
| **2. 低频/限时 BN** | 满足 BN 全部标准，但暴食/清除 <1次/周 或 <3月 | 同样痛苦的暴食-清除循环 |
| **3. 低频/限时 BED** | 满足 BED 全部标准，但暴食 <1次/周 或 <3月 | 同样痛苦的暴食 |
| **4. 清除障碍** | 规律性清除行为（呕吐/泻药），但**无暴食** | 独立于暴食的清除行为 |
| **5. 夜间进食综合征** | 夜间觉醒后进食 或 晚餐后过量进食 | 不是梦游症/非药物引起 |

---

## 2. 各亚型详解

### 2.1 非典型神经性厌食 (Atypical AN)

| 维度 | 说明 |
|:-----|:-----|
| 定义 | 满足 AN 所有标准（恐胖/体像扭曲/限制行为），但 BMI 在正常范围（≥18.5） |
| 患病率 | 可能是典型 AN 的 2-3 倍（~3-5%） |
| 医学风险 | **同样严重**：心动过缓/低血压/骨密度降低/内分泌紊乱 |
| 临床盲点 | 常被忽视（"体重正常"→"不严重"的错误假设） |
| 治疗 | 与 AN 相同：CBT-E / FBT + 营养咨询 |
| 体重恢复目标 | 回到病前体重（而非基于群体BMI） |

> ⚠️ **关键信息**：体重从 90kg 降至 55kg 的人可能有与 BMI 14 的 AN 患者同样严重的医学并发症——**体重下降幅度和速度比绝对 BMI 更重要**。

### 2.2 低频/限时 BN 与 BED

| 维度 | 说明 |
|:-----|:-----|
| 与标准 BN/BED 差异 | 频率更低或持续时间更短 |
| 痛苦程度 | **同样痛苦** |
| 治疗 | 与标准 BN/BED 相同方案 |
| 临床意义 | 避免"不够严重"的误判 |

### 2.3 清除障碍 (Purging Disorder)

| 维度 | 说明 |
|:-----|:-----|
| 定义 | 规律性清除（呕吐/泻药/利尿剂/灌肠）但**无暴食** |
| 动机 | 控制体重/体型（区别于 ARFID 的恐惧型呕吐） |
| 体像关注 | 有（区别于其他原因呕吐） |
| 医学风险 | 与 BN 清除型相似 |
| 治疗 | CBT-E（适应无暴食的情况） |

### 2.4 夜间进食综合征 (Night Eating Syndrome, NES)

| 维度 | 说明 |
|:-----|:-----|
| 核心特征 | ①夜间觉醒后必须进食才能再入睡 和/或 ②晚餐后摄入全天热量 ≥25% |
| 意识状态 | **清醒且能回忆**（区别于睡眠相关进食障碍 SRED） |
| 共病 | 肥胖/抑郁/睡眠障碍 |
| 机制 | 昼夜节律延迟 + 夜间褪黑素/皮质醇异常 |
| 治疗 | 光照疗法/褪黑素/CBT-I/SSRI(舍曲林) |

---

## 3. 流行病学

| 亚型 | 估计患病率 | 性别比 |
|:-----|:----------|:------:|
| OSFED 总体 | 3-5% | 2-3:1 |
| 非典型 AN | 2-4% | ~3:1 |
| 低频 BN | 1-2% | ~3:1 |
| 低频 BED | 1-2% | ~1.5:1 |
| 清除障碍 | 0.5-1% | ~5:1 |
| 夜间进食综合征 | 1-2% | ~1:1 |

> **临床事实**：OSFED 患者占进食障碍专科门诊的 30-50%，是最常见的 ED 诊断。

---

## 4. 评估与转归

### 4.1 OSFED 的转归数据

| 转归 | 比率 | 说明 |
|:-----|:-----|:-----|
| 恢复 | ~50-60% | 5年内 |
| 持续 OSFED | ~20-25% | 慢性 |
| 转为 AN/BN/BED | ~15-25% | 症状"升级" |
| 自然缓解 | ~10-15% | 无需治疗 |

### 4.2 评估原则

| 原则 | 说明 |
|:-----|:-----|
| 同等重视 | OSFED 严重程度不低于典型 AN/BN |
| 完整评估 | 与典型 ED 相同的评估流程 |
| 指明亚型 | 尽可能具体说明属于哪种 OSFED |
| 监测转归 | 定期评估是否发展为典型 ED |

---

## 5. 治疗原则

| 原则 | 说明 |
|:-----|:-----|
| **参照对应典型诊断方案** | 非典型AN→AN方案；低频BN→BN方案 |
| **不降格治疗** | OSFED≠"轻度"→需同等强度干预 |
| **监测体重变化** | 特别关注非典型AN的体重恢复 |
| **灵活应用** | 根据个体情况调整治疗方案 |
