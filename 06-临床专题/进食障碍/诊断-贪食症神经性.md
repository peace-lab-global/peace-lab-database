# 神经性贪食 (Bulimia Nervosa)

> **核心定位**：以反复暴食-补偿行为循环为核心特征，患者体重常在正常范围，因此具有高度隐蔽性。CBT-BN/CBT-E 为一线循证治疗。

---

## 1. 诊断标准

### 1.1 DSM-5-TR 诊断标准 (307.51 / F50.2)

| 标准 | 内容 | 操作化要点 |
|:-----|:-----|:-----------|
| **A. 反复暴食** | 在限定时间内（如2小时）进食量明显大于多数人 + **失控感** | 关注主观失控感，食物量不必极大 |
| **B. 反复不当补偿行为** | 自我诱吐、泻药/利尿剂/灌肠滥用、禁食、过度运动 | 至少1种规律性补偿行为 |
| **C. 频率/持续** | 暴食与补偿行为均 **≥ 1次/周 × 3月** | DSM-5 降低了频率阈值 |
| **D. 自我评价** | 过度受体型和体重影响 | 与 AN 共享此特征 |
| **E. 排除** | 非仅在 AN 发作期间出现 | 若 AN+BN 同时存在，诊断 AN 暴食清除型 |

### 1.2 严重程度分级（按频率）

| 等级 | 补偿行为频率/周 | 临床意义 |
|:-----|:---------------|:---------|
| 轻度 | 1-3 次/周 | 可能自助 CBT |
| 中度 | 4-7 次/周 | 引导自助或门诊 CBT |
| 重度 | 8-13 次/周 | 专科门诊 CBT |
| 极重度 | ≥ 14 次/周 | 专科强化治疗/住院考虑 |

### 1.3 BN 亚型

| 亚型 | 特征 | 占比 | 医学风险 |
|:-----|:-----|:----:|:---------|
| **清除型 (Purging)** | 呕吐/泻药/利尿剂/灌肠 | ~80-90% | 电解质紊乱/食管损伤/牙齿侵蚀 |
| **非清除型 (Non-purging)** | 禁食/过度运动（无呕吐/泻药） | ~10-20% | 运动损伤/能量耗竭 |

## 3. 临床特征

### 3.1 暴食行为模式

| 特征 | 描述 |
|:-----|:-----|
| 触发因素 | 负面情绪（孤独/焦虑/愤怒）、限制饮食后反弹、人际冲突 |
| 食物选择 | 高热量/禁忌食物（甜食/碳水/脂肪）、"禁止食物"清单 |
| 进食方式 | 快速、秘密性、机械化、味觉钝化 |
| 持续时间 | 通常 30-120 分钟 |
| 热量范围 | 1,000-10,000+ kcal/次 |
| 后遗情绪 | 罪恶感/厌恶/恐惧增重/抑郁→驱动补偿 |

### 3.2 补偿行为模式

| 行为 | 频率 | 医学后果 |
|:-----|:-----|:---------|
| 自我诱吐 | 最常见（~80%） | 电解质紊乱/食管炎/Mallory-Weiss撕裂/Russell征 |
| 泻药滥用 | ~30-40% | 泻药结肠/电解质紊乱/依赖性便秘 |
| 利尿剂 | ~10-15% | 低钾/反跳性水肿/肾损伤 |
| 过度运动 | ~40-60% | 运动损伤/骨应力性骨折/心脏过载 |
| 禁食/极度限制 | ~50% | 驱动下一轮暴食→恶性循环 |

### 3.3 躯体体征

| 体征 | 表现 | 发生率 |
|:-----|:-----|:------:|
| Russell 征 | 指关节皮肤增厚/疤痕（反复刺激咽部） | ~30% |
| 腮腺肿大 | 无痛性双侧腮腺肿大 | ~25-50% |
| 牙釉质侵蚀 | 舌侧牙面侵蚀（呕吐酸蚀） | ~35-50% |
| 眼结膜出血 | 呕吐时压力导致 | ~10% |
| 手颤 | 电解质紊乱/低血糖 | 可变 |
| 体重波动 | 5-10kg 范围波动 | 常见 |

## 4. 评估

### 4.1 结构化评估流程

| 步骤 | 工具 | 目的 |
|:-----|:-----|:-----|
| 1. 筛查 | SCOFF（5项） | 快速识别进食障碍风险 |
| 2. 诊断 | EDE 17.0D 访谈 或 EDE-Q 6.0 | 暴食/清除频率与严重程度 |
| 3. 共病 | PHQ-9 / GAD-7 / AUDIT | 抑郁/焦虑/物质使用 |
| 4. 自杀风险 | C-SSRS | 自杀意念/行为评估 |
| 5. 躯体 | 电解质/K+/Cl-/HCO3-/Mg2+/ECG | 医学并发症筛查 |
| 6. 牙科 | 牙科检查 | 牙釉质侵蚀评估 |
| 7. 功能损害 | CIA (Clinical Impairment Assessment) | 心理社会功能影响 |

### 4.2 实验室检查重点

| 检查 | 异常表现 | 机制 |
|:-----|:---------|:-----|
| 血钾 | ↓ 低钾血症 | 呕吐/泻药/利尿剂 |
| 血氯 | ↓ 低氯血症 | 呕吐丢失 HCl |
| 碳酸氢盐 | ↑ 代谢性碱中毒（呕吐型）/ ↓ 酸中毒（泻药型） | 酸碱平衡紊乱 |
| 淀粉酶 | ↑ 唾液腺来源 | 腮腺肿大/反复呕吐 |
| 血常规 | 正常或轻度贫血 | 营养不足 |
| 肾功能 | BUN/Cr 可能异常 | 脱水/泻药肾毒性 |
| ECG | QT延长/U波/T波平坦 | 低钾相关 |

## 5. 治疗方案

### 5.1 NICE 指南推荐

| 治疗线 | 方案 | 疗程 | 证据 |
|:-------|:-----|:-----|:-----|
| **一线** | CBT-BN（或 CBT-E） | 20 次/20 周 | 强（多项 RCT） |
| **一线替代** | 引导自助 CBT（guided self-help） | 8-12 周 | 中 |
| **二线** | IPT（人际关系疗法） | 16-20 次 | 中（效果较慢但持久） |
| **三线/药物** | SSRI（氟西汀 60mg/d） | 持续 | 中（独立于心理治疗） |
| **不推荐** | 精神动力疗法（证据不足）/ 安非拉酮 | — | — |

### 5.2 CBT-BN 核心结构

| 阶段 | 时间 | 目标 | 核心技术 |
|:-----|:-----|:-----|:---------|
| **Phase 1** | 第1-8次 | 稳定进食模式 | 心理教育/自我监测/规律进食/体重管理 |
| **Phase 2** | 第9-14次 | 处理饮食规则与回避 | 暴露进食（禁忌食物）/认知重构/问题解决 |
| **Phase 3** | 第15-20次 | 体像与自尊/复发预防 | 体像暴露/自尊增强/复发预防计划 |

### 5.3 药物治疗

| 药物 | 剂量 | 效果 | 说明 |
|:-----|:-----|:-----|:-----|
| **氟西汀** | 60mg/d（高于抑郁剂量） | 减少暴食/清除频率 ~50% | FDA 唯一批准 BN 药物 |
| 其他 SSRI | 标准-高剂量 | 相似效果 | 超说明书使用 |
| 托吡酯 | 50-100mg/d | 减少暴食 | 认知副作用（注意/记忆） |
| 昂丹司琼 | 4-8mg/d | 减少呕吐冲动 | 超说明书 |
| **禁忌：安非拉酮** | — | 降低癫痫阈值 | BN 患者癫痫风险已增加 |

### 5.4 疗效指标

| 指标 | 时间点 | 良好反应标准 |
|:-----|:-------|:------------|
| 暴食频率 | 治疗结束 | 减少 ≥ 50% 或完全停止 |
| 清除频率 | 治疗结束 | 减少 ≥ 50% 或完全停止 |
| EDE-Q 总分 | 治疗结束 | 降至社区常模范围（< 1.5-2.0） |
| CIA 总分 | 治疗结束 | 功能损害显著降低 |
| 抑郁/焦虑 | 治疗结束 | PHQ-9/GAD-7 显著改善 |
| 电解质 | 4-8周 | 恢复正常 |

## 6. 鉴别诊断

| 鉴别 | BN vs | 关键区分点 |
|:-----|:------|:-----------|
| AN 暴食清除型 | AN | BMI：AN<18.5 / BN 正常或超重 |
| BED | BED | 有无规律补偿行为 |
| 边缘型人格障碍 | BPD | 暴食是否为核心症状 vs 情绪不稳定的一般表现 |
| 躯体疾病致呕吐 | GI 疾病 | 呕吐前有无暴食/有无体像关注 |
| 泻药滥用 | 原发性便秘/IBS | 泻药使用动机（减重 vs 通便） |

## 7. 预后

| 时间窗 | 完全恢复 | 部分恢复 | 慢性 | 死亡 |
|:-------|:--------|:--------|:-----|:-----|
| 5 年 | 40-50% | 30-35% | 15-20% | 1-2% |
| 10 年 | 50-70% | 20-25% | 10-15% | 2-3% |

**良好预后因子**：早期干预 / 无严重共病 / 完成 CBT / 家庭支持 / 无物质滥用
**不良预后因子**：长病程 / 高频清除 / 共病BPD / 低动机 / 治疗脱落
