# 身体意象正念干预 (Body Image Mindfulness Intervention)

> **核心定位**：整合正念/接纳/身体功能欣赏，处理进食障碍中核心的体型过度评价。跨 02-心理学（自我调节）与 06-临床专题的桥梁。

---

## 1. 身体意象障碍模型

| 层面 | 内容 | ED 中的表现 |
|:-----|:-----|:-----------|
| **知觉层面** | 对自己体型的感知 | 感知扭曲/过度检查 |
| **认知层面** | 对体型的评价/信念 | "我的价值=我的体型" |
| **情感层面** | 对体型的情绪反应 | 厌恶/焦虑/羞耻 |
| **行为层面** | 回避/检查/比较 | 回避镜子/频繁称重 |

---

## 2. 正念干预技术

### 2.1 身体扫描（ED 改良版）

| 标准身体扫描 | ED 改良 |
|:------------|:--------|
| 逐部位觉察感觉 | 强调**功能**而非外形 |
| 中性观察 | 加入**感恩/欣赏**元素 |
| 20-45分钟 | 缩短至 10-15分钟（减少不适） |
| 躺姿 | 可选坐姿（减少身体意识焦虑） |

### 2.2 身体功能欣赏 (Functionality Appreciation)

| 练习 | 说明 |
|:-----|:-----|
| "我的身体能做什么" | 列出身体功能清单（走路/拥抱/呼吸/思考） |
| 身体感恩日记 | 每日记录1件身体为你做的事 |
| 能力 vs 外貌 | 将注意力从"看起来如何"转向"能做什么" |

### 2.3 媒体素养训练

| 练习 | 说明 |
|:-----|:-----|
| 图片解构 | 分析广告/社交媒体中的修图/摆拍 |
| "真实 vs 编辑" | 对比原图与修图后效果 |
| 算法觉察 | 理解推荐算法如何强化体型焦虑 |
| 主动选择 | 关注身体积极/中性账号 |

## 3. 接纳取向策略

| 策略 | 说明 |
|:-----|:-----|
| **身体中性** | "我的身体不需要完美，只需要工作" |
| **自我关怀** | Neff 自我关怀三要素：正念/共同人性/自我善意 |
| **价值观导向** | 将精力从外貌转向人生价值 |
| **减少比较** | 识别并减少外貌比较行为 |
