# 进食障碍评估 Agent Skill

> **功能定位**：当用户表达进食相关困扰时，通过 SCOFF 快速筛查 + EDE-Q 深度评估 + 风险分层 + 转介决策，提供结构化评估与下一步建议。

---

## 1. 触发条件

| 触发类型 | 示例 |
|:---------|:-----|
| **直接表达** | "我可能有进食障碍"/"我总是暴食"/"我吃完会吐" |
| **间接表达** | "我害怕吃东西"/"我对体重很焦虑"/"我控制不住吃" |
| **身体描述** | "我越来越瘦但停不下来"/"我吃了停不下来" |
| **情绪+食物** | "我一难过就吃"/"吃东西让我内疚" |

---

## 2. 评估流程

### 2.1 Step 1: 共情回应 + 安全筛查

```
→ 首先表达理解和非评判态度
→ 筛查紧急风险：
  - 是否有自杀/自伤想法？
  - 是否晕倒/心悸/严重虚弱？
  - 是否在服用泻药/利尿剂？
→ 若有紧急风险 → 立即建议寻求紧急帮助
```

### 2.2 Step 2: SCOFF 快速筛查

向用户提出 5 个问题（口语化版本）：

| # | 口语化问题 | 评分 |
|:--|:-----------|:-----|
| S | 你吃太多之后会不会让自己吐？ | 是=1 |
| C | 你觉得自己在吃东西方面失控了吗？ | 是=1 |
| O | 你最近3个月是不是瘦了超过6公斤？ | 是=1 |
| F | 你是不是觉得自己胖，但别人说你太瘦？ | 是=1 |
| F | 你觉得食物占据了你生活的很多注意力吗？ | 是=1 |

**解读**：≥2/5 = 进食障碍风险，建议专业评估

### 2.3 Step 3: 补充评估（根据 SCOFF 结果）

| SCOFF 结果 | 下一步 |
|:-----------|:-------|
| 0-1 | 提供预防性信息，鼓励健康饮食态度 |
| 2-3 | 深入询问进食行为模式/情绪/体像 |
| 4-5 | 强烈建议专业评估，提供资源 |

**深入评估问题**（如需要）：
1. 进食行为模式：暴食/限制/清除/过度运动
2. 频率与持续时间
3. 体重变化
4. 情绪触发
5. 共病信号（焦虑/抑郁/自伤）

## 3. 风险分层与转介决策树

```
┌──────────────────────────────────────────────┐
│           进食障碍转介决策树                    │
├──────────────────────────────────────────────┤
│                                              │
│ 🔴 紧急转介（立即就医）                       │
│   → 自杀意念/计划                             │
│   → 晕厥/心悸/严重虚弱                       │
│   → 严重低体重 + 拒绝进食                    │
│                                              │
│ 🟠 尽快转介（1-2周内就医）                    │
│   → SCOFF ≥ 3 + 体重快速下降                │
│   → 规律清除行为（呕吐/泻药）                │
│   → 共病严重抑郁/焦虑                        │
│                                              │
│ 🟡 建议评估（预约就诊）                       │
│   → SCOFF ≥ 2 + 进食困扰                    │
│   → 暴食但无清除                             │
│   → 身体意象困扰                             │
│                                              │
│ 🟢 预防性指导                                 │
│   → SCOFF 0-1 + 轻度担忧                    │
│   → 提供健康饮食信息/资源                     │
│                                              │
└──────────────────────────────────────────────┘
```

## 5. 安全边界

| 原则 | 说明 |
|:-----|:-----|
| **不做诊断** | AI 不能做出临床诊断 |
| **紧急优先** | 紧急情况下优先安全而非评估 |
| **转介为主** | 始终鼓励寻求专业帮助 |
| **非评判** | 避免评判体重/体型/进食行为 |
| **去污名** | 进食障碍≠"想变美"/"不吃饭" |
| **男性包容** | 不以女性为默认 |
