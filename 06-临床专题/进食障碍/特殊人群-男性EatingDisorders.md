# 男性进食障碍 (Male Eating Disorders)

> **核心定位**：男性进食障碍严重低估（实际可能占 25%+），表现与女性不同（肌肉取向 vs 瘦取向），评估工具需性别适配。

---

## 1. 流行病学与低估问题

| 指标 | 数值 | 说明 |
|:-----|:-----|:-----|
| 官方统计占比 | 10-15% | AN/BN 中 |
| 实际可能占比 | 25-33% | 含 OSFED/BED |
| 求助延迟 | 比女性长 2-3 年 | 性别污名 |
| 诊断延迟 | 常被漏诊 | 工具/临床偏见 |

### 1.1 低估原因

| 原因 | 说明 |
|:-----|:-----|
| **性别偏见** | "进食障碍=女性疾病"的刻板印象 |
| **表现不同** | 男性更关注肌肉不足而非"胖" |
| **量表偏差** | 标准工具聚焦瘦/恐胖 |
| **求助障碍** | 男性更少求助心理问题 |
| **转介路径** | 男性更可能通过运动医学/肠胃科而非精神科进入 |

---

## 2. 男性 ED 特殊表现

### 2.1 体像关注差异

| 维度 | 女性典型 | 男性典型 |
|:-----|:---------|:---------|
| 理想体型 | 瘦 | 肌肉发达/倒三角 |
| 核心恐惧 | "太胖" | "不够壮/肌肉不够" |
| 行为焦点 | 减重/限制 | 增肌/高蛋白饮食/过度运动 |
| 身体检查 | 称体重/照镜子 | 测臂围/胸围/比较他人 |

### 2.2 肌肉上瘾症 (Muscle Dysmorphia / "Bigorexia")

| 特征 | 说明 |
|:-----|:-----|
| DSM 定位 | 身体变形障碍(BDD)亚型 |
| 核心关注 | 认为自己"太小/不够肌肉"（尽管客观上很壮） |
| 行为表现 | 过度举重/严格高蛋白饮食/回避社交/使用增肌药物 |
| 功能损害 | 职业/社交/健康受损 |
| 与 ED 交叉 | 限制性饮食/暴食蛋白质/过度运动 |

---

## 3. 评估调整

| 标准工具问题 | 男性适配方案 |
|:------------|:------------|
| EDE-Q "怕胖" | 加入"肌肉不足"相关项目 |
| 缺乏肌肉取向 | 补充 Drive for Muscularity Scale (DMS) |
| 进食关注偏差 | 评估蛋白质强迫/补充剂使用 |
| 运动评估 | 评估过度举重/运动成瘾 |
| 体像评估 | 肌肉不满意量表 |

---

## 4. 治疗考量

| 考虑 | 说明 |
|:-----|:-----|
| 去污名化 | 明确 ED 不是"女性疾病" |
| 动机工作 | 男性动机模式不同 |
| 体像工作 | 处理肌肉取向而非瘦取向 |
| 运动行为 | 过度举重 vs 过度有氧 的区别 |
| 团体治疗 | 全男性团体可能更合适 |
| 共病 | 男性更高物质滥用/反社会特质 |
