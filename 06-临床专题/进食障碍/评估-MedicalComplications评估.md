# 进食障碍医学并发症评估 (Medical Complications Assessment)

> **核心定位**：进食障碍是致死率最高的精神障碍类别之一，医学并发症的早期识别与系统评估直接关系到治疗安全。本文覆盖各系统并发症评估方案与紧急处理原则。

---

## 1. 并发症概览

### 1.1 按严重程度分层

| 紧急程度 | 系统 | 关键并发症 | 致死风险 |
|:---------|:-----|:-----------|:---------|
| **🔴 立即致命** | 心血管 | 心律失常/QT延长/心肌病 | 高 |
| | 电解质 | 低钾/低磷（再喂养） | 高 |
| **🟠 高风险** | 营养 | 再喂养综合征 | 中高 |
| | 消化 | 急性胃扩张/胃穿孔 | 中 |
| | 肾脏 | 急性肾损伤 | 中 |
| **🟡 中长期** | 骨骼 | 骨质疏松/骨折 | 低-中 |
| | 内分泌 | 性腺轴抑制/甲功异常 | 低 |
| | 牙齿 | 牙釉质侵蚀 | 低 |
| | 神经 | 脑萎缩（可逆） | 低 |

---

## 2. 电解质与代谢

### 2.1 电解质紊乱

| 异常 | 机制 | ED 类型 | 临床后果 | 紧急阈值 |
|:-----|:-----|:--------|:---------|:---------|
| **低钾** | 呕吐/泻药/利尿剂 | BN/AN-BP | 心律失常/QT延长/肌无力 | < 3.0 mmol/L |
| **低磷** | 再喂养/营养不良 | AN/所有ED | 再喂养综合征/溶血/呼吸衰竭 | < 0.5 mmol/L |
| **低钠** | 水中毒/ADH异常 | AN | 癫痫/脑水肿 | < 125 mmol/L |
| **低氯** | 呕吐丢失HCl | BN-清除型 | 代谢性碱中毒 | < 90 mmol/L |
| **代谢性碱中毒** | 呕吐 | BN-清除型 | 呼吸代偿 | HCO3- > 30 |
| **代谢性酸中毒** | 泻药/饥饿酮症 | BN-泻药型/AN | 多器官功能受损 | pH < 7.35 |
| **低镁** | 营养不良/泻药 | AN/BN | 心律失常/肌痉挛 | < 0.7 mmol/L |

### 2.2 实验室检查方案

| 检查 | AN | BN | BED | 频率 |
|:-----|:--:|:--:|:---:|:-----|
| 全血细胞计数(CBC) | ✅ | ✅ | — | 初始+每3月 |
| 电解质(K/Na/Cl/Mg/PO4) | ✅ | ✅ | — | 初始+按需（再喂养期每日） |
| 肝肾功能(LFT/RFT) | ✅ | ✅ | — | 初始+每3月 |
| 甲状腺功能(TSH/fT4) | ✅ | — | — | 初始+每6月 |
| 空腹血糖/HbA1c | — | — | ✅ | 初始+每6月 |
| 血脂全套 | — | — | ✅ | 初始+每年 |
| 维生素D/铁蛋白/B12 | ✅ | ✅ | — | 初始+每6月 |
| 淀粉酶/脂肪酶 | — | ✅ | — | 初始+按症状 |
| 凝血功能 | ✅ | — | — | BMI<15 或按症状 |

## 3. 心血管系统

### 3.1 AN 心脏改变

| 改变 | 发生率 | 机制 | 临床意义 |
|:-----|:------:|:-----|:---------|
| **窦性心动过缓** | 60-95% | 迷走神经张力增加/代谢减慢 | HR<40bpm 需住院 |
| **QT 延长** | 15-30% | 电解质紊乱/心肌营养不良 | QTc>450ms 致命风险 |
| **低血压** | 30-50% | 低血容量/心输出量降低 | <80/50 mmHg |
| **二尖瓣脱垂** | 20-30% | 左室缩小 | 通常良性，体重恢复后改善 |
| **心肌萎缩** | 重度AN | 蛋白质分解/心肌纤维化 | 体重恢复后可部分逆转 |
| **心包积液** | 15-25% | 低蛋白血症 | 少量通常无需处理 |

### 3.2 BN 心脏风险

| 风险 | 机制 |
|:-----|:-----|
| 低钾→心律失常 | 清除行为导致钾丢失 |
| 吐根糖浆中毒 | 滥用吐根催吐→心肌病（历史问题） |
| QT延长 | 低钾+药物（如抗抑郁药）叠加 |

### 3.3 心脏监测方案

| 检查 | 指征 | 频率 |
|:-----|:-----|:-----|
| 12导联ECG | 所有初诊ED患者 | 初始+每次住院 |
| 24h Holter | 心律失常症状/QTc>440ms | 按需 |
| 超声心动图 | 心脏杂音/心衰症状/BMI<14 | 初始+按症状 |
| 连续心电监测 | 再喂养期/BMI<13 | 住院期持续 |

## 4. 骨骼系统

### 4.1 AN 骨病

| 改变 | 发生率 | 机制 |
|:-----|:------:|:-----|
| **骨减少(Osteopenia)** | 50-90% | 低雌激素/高皮质醇/营养不良 |
| **骨质疏松(Osteoporosis)** | 13-50% | 骨形成减少+骨吸收增加 |
| **骨折** | 3-7x | 骨密度降低+跌倒风险 |
| **生长迟缓** | 青少年 | 骨骼未成熟期营养不足 |

### 4.2 骨密度评估

| 工具 | 适应证 | 要点 |
|:-----|:-------|:-----|
| DXA(双能X线) | AN 病程>6月/闭经/骨折史 | Z值<-2.0（年轻人）/T值<-1.0 |
| 重复DXA | 每12-24月 | 监测骨密度变化 |
| 骨代谢标志物 | 可选 | P1NP(形成)/CTX(吸收) |

> ⚠️ **体重恢复是骨密度改善的最有效策略**。单独补充钙/维生素D/雌激素效果有限。

---

## 5. 消化系统

| 并发症 | ED类型 | 机制 | 评估 |
|:-------|:-------|:-----|:-----|
| **胃排空延迟** | AN | 胃平滑肌萎缩 | 症状：早饱/腹胀/恶心 |
| **便秘** | AN | 肠蠕动减慢/脱水 | 纤维/水分不足 |
| **食管炎** | BN | 胃酸反复反流 | 上腹痛/吞咽困难 |
| **Mallory-Weiss 撕裂** | BN | 剧烈呕吐 | 呕血→紧急处理 |
| **食管破裂(Boerhaave)** | BN | 罕见但致命 | 胸痛/纵隔气肿→紧急手术 |
| **急性胃扩张** | AN-BP/BED | 暴食后胃扩张 | 腹痛/休克→紧急处理 |
| **泻药结肠** | BN | 长期泻药→结肠无力 | 停用泻药+渐进恢复 |
| **胰腺炎** | BN/BED | 暴食/高甘油三酯 | 腹痛/淀粉酶↑ |
| **肝酶升高** | AN | 自噬性肝损伤 | AST/ALT 轻-中度升高 |

## 6. 内分泌系统

| 轴/系统 | AN 改变 | BN 改变 | 恢复 |
|:---------|:--------|:--------|:-----|
| **下丘脑-垂体-性腺轴** | GnRH↓→LH/FSH↓→雌激素/睾酮↓→闭经 | 通常正常 | 体重恢复后恢复（AN 6-12月） |
| **甲状腺轴** | T3↓(euthyroid sick) | 通常正常 | 体重恢复后恢复 |
| **HPA 轴** | 皮质醇↑ | 可变 | 体重恢复后正常化 |
| **生长激素/IGF-1** | GH↑/IGF-1↓(GH抵抗) | — | 营养恢复后改善 |
| **饥饿激素** | Ghrelin↑/Leptin↓ | 可变 | 体重恢复后正常化 |
| **糖代谢** | 低血糖风险 | 通常正常 | 规律进食改善 |

---

## 7. 牙齿与口腔

| 改变 | ED类型 | 表现 | 处理 |
|:-----|:-------|:-----|:-----|
| 牙釉质侵蚀 | BN(呕吐) | 舌侧牙面侵蚀(Perimylolysis) | 呕吐后漱口(不刷牙)/氟化物 |
| 腮腺肿大 | BN/AN-BP | 无痛性双侧肿大 | 热敷/体重恢复后消退 |
| 口腔干燥 | AN | 唾液减少 | 人工唾液/充分饮水 |
| 龋齿增加 | BN/BED | 酸蚀+糖摄入 | 定期牙科检查 |

---

## 8. 再喂养综合征

### 8.1 风险评估

| 风险因素 | 高风险标志 |
|:---------|:-----------|
| BMI | < 16（成人） |
| 体重下降 | > 15% 在 3-6 月内 |
| 进食量 | < 500 kcal/d 连续 5 天以上 |
| 电解质 | 入院前已有低磷/低钾/低镁 |
| 合并症 | 酒精滥用/糖尿病/胰岛素使用 |

### 8.2 再喂养方案

| 阶段 | 热量 | 磷补充 | 监测频率 |
|:-----|:-----|:-------|:---------|
| Day 1-3 | 10-20 kcal/kg/d 或 1200 kcal | 预防性口服磷 | 电解质每日2次 |
| Day 4-7 | 每2天 +200 kcal | 按血磷调整 | 电解质每日1次 |
| Week 2+ | 目标热量(2500-4000 kcal) | 按需 | 隔日 |

### 8.3 紧急处理

| 状况 | 处理 |
|:-----|:-----|
| 血磷 < 0.5 mmol/L | **立即静脉补磷** + 暂停/减少热量 |
| 血钾 < 3.0 mmol/L | 口服/静脉补钾 + ECG 监测 |
| 液体潴留/水肿 | 限盐/抬高下肢/监测出入量 |
| 心律失常 | 紧急纠正电解质 + 心电监护 |
| 低血糖 | 口服葡萄糖/静脉葡萄糖 |

---

## 9. 住院指征汇总

| 指征 | 阈值 |
|:-----|:-----|
| BMI | < 13（成人）/ < 第2百分位（儿童） |
| 心率 | < 40 bpm |
| 血压 | < 80/50 mmHg |
| 体温 | < 35°C |
| 血钾 | < 3.0 mmol/L |
| 血磷 | < 0.5 mmol/L |
| QTc | > 450 ms |
| 口服摄入 | < 500 kcal/d 持续 |
| 自杀风险 | 有明确计划/意图 |
| 门诊治疗失败 | 持续体重下降/无改善 |
