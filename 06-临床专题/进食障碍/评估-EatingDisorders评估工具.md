# 进食障碍评估工具系统 (Assessment Tools)

> **核心定位**：覆盖从快速筛查到深度临床访谈的完整评估工具链，含评分标准、解释规则、文化适应性与使用注意事项。

---

## 1. 评估工具金字塔

```
                    ┌──────────┐
                    │ EDE 访谈  │  ← 金标准（60-90分钟）
                    ├──────────┤
                  │ EDE-Q 6.0 │  ← 深度自评（10-15分钟）
                  ├───────────┤
                │ EAT-26      │  ← 风险筛查（5-10分钟）
                ├─────────────┤
              │ SCOFF (5项)   │  ← 快速筛查（1-2分钟）
              └───────────────┘
```

---

## 2. 快速筛查工具

### 2.1 SCOFF 问卷

| 项目 | 内容 | 英文 |
|:----:|:-----|:-----|
| S | 你是否因吃得太饱而感到**不适(Sick)**？ | Do you make yourself **Sick** because you feel uncomfortably full? |
| C | 你是否担心自己**失去了对进食的控制(Control)**？ | Do you worry you have lost **Control** over how much you eat? |
| O | 你是否在最近3个月内体重下降了超过**14磅(One stone/6.35kg)**？ | Have you recently lost more than **One stone** (14 pounds) in a 3-month period? |
| F | 你是否认为自己**胖(Fat)**而别人说你太瘦？ | Do you believe yourself to be **Fat** when others say you are too thin? |
| F | 你是否觉得**食物(Food)**主宰了你的生活？ | Would you say **Food** dominates your life? |

| 参数 | 数值 |
|:-----|:-----|
| 评分 | ≥ 2/5 阳性 |
| 敏感度 | AN 79-84%, BN 92-100% |
| 特异度 | 87-94% |
| 使用场景 | 全科/急诊/学校/初级筛查 |
| 局限 | 对 BED/ARFID/男性敏感性较低 |

### 2.2 BEDS-7（暴食障碍专用筛查）

| 参数 | 数值 |
|:-----|:-----|
| 项目数 | 7 项 |
| 时间 | 2-3 分钟 |
| 敏感度 | 83.6% |
| 特异度 | 84.5% |
| 核心评估 | 失控感 + 暴食频率 + 痛苦 |

### 2.3 ESP（早期识别进食障碍）

| 参数 | 说明 |
|:-----|:-----|
| 项目数 | 20+ 项 |
| 目标人群 | 青少年/学校 |
| 特点 | 覆盖 AN/BN/BED/运动强迫 |

## 3. 风险筛查量表

### 3.1 EAT-26 (Eating Attitudes Test)

| 参数 | 说明 |
|:-----|:-----|
| 项目数 | 26 项 |
| 评分 | 6级Likert→3个分量表 |
| 时间 | 5-10 分钟 |
| 切分 | ≥ 20 分 |

**三个分量表**：

| 分量表 | 项目数 | 评估内容 | 示例项目 |
|:-------|:------:|:---------|:---------|
| **节食 (Dieting)** | 13 | 限制性进食/体重关注 | "我极度害怕超重" |
| **暴食与食物关注 (Bulimia)** | 6 | 暴食思维/食物强迫 | "我发现自己一直在想食物" |
| **口腔控制 (Oral Control)** | 7 | 进食自控/周围压力 | "我觉得周围人希望我吃更多" |

**附加行为问题**（EAT-26 后6项）：
- 暴食次数/月
- 泻药/利尿剂使用
- 自我诱吐
- 过度运动（>60分钟/天用于控制体重）
- 体重变化（过去6个月）

### 3.2 其他筛查工具

| 工具 | 项目数 | 特点 |
|:-----|:------:|:-----|
| EAT-40 | 40 | EAT-26 前身，更详细 |
| BITE (Bulimic Investigatory Test) | 30+3 | 专门评估 BN 严重程度 |
| DOS (Diagnostic Survey for ED) | 44 | 结构化诊断 |
| QEWP-R (Questionnaire on Eating and Weight Patterns) | 28 | BED 专用 |

## 4. 深度评估工具

### 4.1 EDE-Q 6.0 (Eating Disorder Examination Questionnaire)

| 参数 | 说明 |
|:-----|:-----|
| 项目数 | 28 项（行为计数）+ 4 个分量表 |
| 评分时间窗 | 过去 28 天 |
| 时间 | 10-15 分钟 |
| 适用人群 | 14+ 岁 |

**四个分量表**：

| 分量表 | 项目数 | 评估内容 | 分值范围 |
|:-------|:------:|:---------|:---------|
| **限制 (Restraint)** | 5 | 刻意限制进食/空腹规则 | 0-6 |
| **进食关注 (Eating Concern)** | 5 | 对进食的焦虑/恐惧/罪恶 | 0-6 |
| **体型关注 (Shape Concern)** | 8 | 对体型的过度评价/回避/比较 | 0-6 |
| **体重关注 (Weight Concern)** | 5 | 对体重的过度评价/恐惧增重 | 0-6 |

**行为计数**（过去28天）：
- 客观暴食次数
- 主观暴食次数
- 自我诱吐次数
- 泻药/利尿剂使用次数
- 过度运动天数

**解释标准**：

| EDE-Q 总分 | 解释 |
|:----------|:-----|
| < 1.5 | 正常范围 |
| 1.5-2.5 | 亚临床/边缘 |
| 2.5-3.5 | 临床显著 |
| > 3.5 | 严重 |

**社区常模**（Mond et al., 2006）：
- 女性平均总分：1.52 (SD=1.54)
- 男性平均总分：0.55 (SD=0.86)

### 4.2 EDE 17.0D 访谈 (Eating Disorder Examination)

| 参数 | 说明 |
|:-----|:-----|
| 类型 | 半结构化临床访谈 |
| 时间 | 60-90 分钟 |
| 评估者 | 受过培训的专业人员 |
| 地位 | 进食障碍评估**金标准** |
| 覆盖 | 所有 DSM-5 ED 诊断 |

**访谈结构**：

| 部分 | 内容 | 时间 |
|:-----|:-----|:----:|
| 进食行为 | 暴食/清除/限制/过度运动 | 20分钟 |
| 限制与饮食规则 | 刻意节食/食物规则/空腹感 | 15分钟 |
| 体型关注 | 体型不满/回避/检查行为 | 15分钟 |
| 体重关注 | 体重不满/恐增重/体重检查 | 10分钟 |
| 其他心理 | 自尊/完美主义/情绪 | 10分钟 |

**优势**：比自评问卷更准确区分暴食 vs 主观暴食 vs 过量进食

## 5. 功能损害评估

### 5.1 CIA (Clinical Impairment Assessment)

| 参数 | 说明 |
|:-----|:-----|
| 项目数 | 16 项 |
| 时间窗 | 过去 28 天 |
| 评分 | 0-3 三级 Likert |
| 时间 | 3-5 分钟 |
| 总分范围 | 0-48 |

**三个领域**：

| 领域 | 项目数 | 评估内容 |
|:-----|:------:|:---------|
| **个人认知** | 5 | 注意力/自控/自我评价受 ED 影响程度 |
| **社交功能** | 6 | 人际关系/社交活动/避免聚餐 |
| **职业功能** | 5 | 工作/学习/日常任务受影响程度 |

**切分标准**：
- ≥ 15 分：临床显著功能损害
- ≥ 20 分：中度功能损害
- ≥ 30 分：严重功能损害

## 6. 特殊人群评估适配

### 6.1 儿童与青少年

| 工具 | 适配版本 | 年龄范围 | 注意事项 |
|:-----|:---------|:---------|:---------|
| ChEAT | 儿童 EAT | 8-13岁 | 家长版+儿童版 |
| EDE-Q | 儿童版 | 8-14岁 | 语言简化 |
| EDE | 儿童版 | 8-14岁 | 更短/更结构化 |
| NIAS | ARFID 筛查 | 儿童/成人 | 9项快速 |

### 6.2 男性

| 注意事项 | 说明 |
|:---------|:-----|
| 肌肉取向 | 标准量表偏重"恐胖"，男性可能表现为"肌肉不足"关注 |
| 补充工具 | Drive for Muscularity Scale (DMS) |
| 语言调整 | "fat"→可能不认同；关注行为而非感受 |
| 低估风险 | 男性更少自报/更晚求助 |

### 6.3 文化适应

| 文化因素 | 评估影响 | 对策 |
|:---------|:---------|:-----|
| 体重理想不同 | 不同文化对体型有不同评价 | 关注功能而非文化体型观 |
| 食物文化意义 | 食物在文化中的角色不同 | 理解文化背景下的"正常"进食 |
| 求助模式 | 某些文化不鼓励自报心理痛苦 | 行为观察 vs 自我报告 |
| 语言翻译 | 量表翻译可能改变项目含义 | 使用已验证的中文版本 |

## 7. 评估流程整合

### 7.1 标准评估路径

```
┌─────────────────────────────────────────────┐
│ Step 1: 快速筛查 (SCOFF/BEDS-7)            │
│   ↓ 阳性                                     │
│ Step 2: 风险评估 (EAT-26/EDE-Q 6.0)        │
│   ↓ 高于切分                                 │
│ Step 3: 诊断评估 (EDE 17.0D 访谈)           │
│   ↓ 确定诊断                                 │
│ Step 4: 功能评估 (CIA)                      │
│   ↓ 评估损害程度                             │
│ Step 5: 共病评估 (PHQ-9/GAD-7/C-SSRS)      │
│   ↓ 确定治疗优先级                           │
│ Step 6: 医学评估 (体格+实验室+ECG)          │
└─────────────────────────────────────────────┘
```

### 7.2 治疗中监测

| 工具 | 频率 | 监测内容 |
|:-----|:-----|:---------|
| EDE-Q | 每 4 周 | 症状变化 |
| CIA | 每 4 周 | 功能改善 |
| 体重 | 每次就诊 | 体重变化趋势 |
| 暴食/清除日记 | 每周 | 行为频率 |
| PHQ-9/GAD-7 | 每 4-8 周 | 共病改善 |
