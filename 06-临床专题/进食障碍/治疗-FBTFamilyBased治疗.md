# FBT 家庭治疗 (Family-Based Treatment)

> **核心定位**：又称 Maudsley 方法，是青少年神经性厌食的**一线循证治疗**。核心理念是将体重恢复的责任暂时交给家长，而非责备家长。

---

## 1. 理论基础

### 1.1 五大核心原则

| 原则 | 说明 |
|:-----|:-----|
| **1. 不可知论立场 (Agnostic view)** | 不追究病因/谁的责任，聚焦"现在怎么办" |
| **2. 非指责性 (Non-authoritarian stance)** | 治疗师支持家庭，不批判家长 |
| **3. 家长赋权 (Parental empowerment)** | 家长是体重恢复的主要执行者 |
| **4. 外化疾病 (Externalization)** | "厌食症"是入侵者，不是孩子本人的选择 |
| **5. 年龄适宜自主 (Developmentally appropriate)** | 随年龄/恢复程度逐步交还自主权 |

### 1.2 外化技术

```
                    "厌食症的声音"
                         ↓
    "这不是你的女儿想这样做——
     是厌食症在控制她的大脑"
                         ↓
         家长与孩子 共同对抗 厌食症
```

## 2. 三阶段治疗结构

### 2.1 总体框架

| 阶段 | 时间 | 目标 | 频率 |
|:-----|:-----|:-----|:-----|
| **Phase 1: 体重恢复** | 第1-10次（~3-6月） | 体重恢复/消除ED行为 | 每周1-2次 |
| **Phase 2: 交还控制权** | 第11-15次（~3-6月） | 青少年逐步自主进食 | 每2周1次 |
| **Phase 3: 青少年议题** | 第16-20次（~3-6月） | 发展议题/自尊/独立 | 每月1次 |
| **总计** | ~12-18月 | — | 20次 |

### 2.2 Phase 1: 体重恢复（详细）

| 内容 | 说明 |
|:-----|:-----|
| **治疗师角色** | 教练/顾问——指导家长如何帮助孩子进食 |
| **家庭用餐** | 前几次治疗包含家庭现场用餐观察 |
| **家长职责** | 决定吃什么/多少/何时吃（替代被AN劫持的决策） |
| **孩子角色** | 执行进食（即使不情愿） |
| **治疗师任务** | 支持家长/处理家庭冲突/监测体重/处理抵抗 |

**家庭用餐会议 (Family Meal Session)**：
1. 治疗师观察家庭互动
2. 评估家长执行能力
3. 现场指导/支持
4. 识别阻碍因素

### 2.3 Phase 2: 交还控制权

| 标志 | 何时进入 Phase 2 |
|:-----|:-----------------|
| 体重 | 达到目标体重的 ~87% |
| 行为 | 暴食/清除行为停止 |
| 家庭 | 家长对管理进食有信心 |
| 孩子 | 开始主动表达进食意愿 |

**渐进交还步骤**：
1. 孩子选择部分食物（如一种蔬菜）
2. 孩子自主准备一份点心
3. 孩子在学校午餐
4. 孩子独立决定一餐
5. 完全自主进食

### 2.4 Phase 3: 青少年议题

| 议题 | 处理方式 |
|:-----|:---------|
| 独立与自主 | 正常的青少年发展任务 |
| 同伴关系 | 恢复社交/同龄互动 |
| 自尊与身份 | 超越"厌食症患者"身份 |
| 家庭关系 | 从"围绕ED"恢复到正常家庭功能 |
| 复发预防 | 预警信号/应对计划 |

## 3. 适应症与选择

### 3.1 最佳适应症

| 因素 | 说明 |
|:-----|:-----|
| **年龄** | 12-18 岁（证据最强） |
| **诊断** | AN（限制型/暴食清除型） |
| **病程** | < 3 年（早期效果最佳） |
| **家庭** | 家长愿意且能够参与 |
| **严重程度** | 轻-中度（重度住院后出院适用） |

### 3.2 FBT vs CBT-E 选择

| 维度 | FBT 首选 | CBT-E 首选 |
|:-----|:---------|:-----------|
| 年龄 | 青少年（<18岁） | 成人（>18岁） |
| 诊断 | AN | BN/BED/OSFED |
| 家庭 | 家长积极参与 | 独立/家长不参与 |
| 病程 | 早期（<3年） | 慢性/复杂 |
| 共病 | 无严重BPD | BPD共病 |

## 4. 疗效证据

| 研究 | 结论 |
|:-----|:-----|
| Eisler et al., 2000 (RCT) | FBT 优于个体治疗（青少年AN） |
| Lock et al., 2010 (多中心RCT) | FBT 体重恢复率 ~60-70% |
| Le Grange et al., 2014 (RCT) | FBT vs AFT: FBT 12月体重恢复更优 |
| NICE NG69, 2017 | 推荐为青少年 AN 一线 |
| Madden et al., 2014 (RCT) | FBT vs PFT: 短期体重恢复相似，FBT 长期更优 |

### 4.1 疗效预测因子

| 良好预测 | 不良预测 |
|:---------|:---------|
| 早期体重增加（前4周） | 严重家庭冲突 |
| 家长高EE↓ | 家长拒绝参与 |
| 短病程 | 长病程（>5年） |
| 家庭凝聚力 | 严重共病BPD/物质滥用 |

## 5. 特殊调整

### 5.1 FBT-BN (青少年贪食症)

- 调整为聚焦减少暴食/清除（而非单纯增重）
- 家长角色类似但更聚焦行为监督

### 5.2 FBT 在单亲/非传统家庭中的应用

| 家庭类型 | 调整 |
|:---------|:-----|
| 单亲 | 支持家长资源不足/引入支持网络 |
| 离异/共同监护 | 协调两个家庭的进食方案 |
| 祖父母/寄养 | 赋权照护者 |
| 文化差异 | 尊重文化饮食习俗/调整外化隐喻 |
