# 进食障碍 DSM-5-TR 分类总览 (Eating Disorders Classification)

> **核心定位**：提供 DSM-5-TR / ICD-11 进食障碍全谱系分类框架，建立流行病学认知与多因素病因模型理解。

---

## 1. 分类体系概览

### 1.1 DSM-5-TR 进食障碍谱系

| 障碍名称 | DSM-5-TR 代码 | ICD-11 代码 | 核心特征 | 终生患病率 |
|:---------|:-------------|:-----------|:---------|:----------|
| 神经性厌食 (AN) | 307.1 (F50.01/F50.02) | 6B80 | 限制能量摄入→低体重 + 强烈恐胖 + 体型认知扭曲 | 0.3-1.0% |
| 神经性贪食 (BN) | 307.51 (F50.2) | 6B81 | 反复暴食 + 补偿行为 + 自我评价过度受体型影响 | 1.0-1.5% |
| 暴食障碍 (BED) | 307.51 (F50.8) | 6B82 | 反复暴食（无补偿行为）+ 失控感 + 痛苦 | 1.9-3.0% |
| 回避性/限制性摄食 (ARFID) | 307.59 (F50.82) | 6B83 | 回避/限制食物（非体像关注驱动） | 0.3-3.2% |
| 异食癖 (Pica) | 307.52 (F50.89/F98.3) | 6B84 | 持续食用非营养物质 ≥1月 | 罕见（特定人群） |
| 反刍障碍 (Rumination) | 307.53 (F98.21) | 6B85 | 反复反流再咀嚼 ≥1月（非GI疾病） | 罕见 |
| 其他特定进食障碍 (OSFED) | 307.50 (F50.89) | 6B8Y | 不符合上述任何障碍全标准但有临床显著痛苦 | ~3-5% |
| 非特定进食障碍 (UFED) | 307.50 (F50.9) | 6B8Z | 症状存在但信息不足以分类 | — |

### 1.2 DSM-5-TR 关键更新（vs DSM-IV）

| 变更 | DSM-IV | DSM-5-TR | 临床意义 |
|:-----|:-------|:---------|:---------|
| AN 闭经标准 | 必须闭经 ≥3周期 | **删除** | 男性/仍有月经的女性不再被排除 |
| BED 独立诊断 | 研究附录 | **正式诊断** | 暴食≠BN，无需补偿行为 |
| ARFID 新增 | 不存在 | **全新诊断** | 非体像驱动的摄食障碍获得认可 |
| BN 暴食频率 | ≥2次/周 × 6月 | **≥1次/周 × 3月** | 降低阈值，提高检出率 |
| AN 严重程度 | 未分级 | BMI 分级：轻/中/重/极重 | 标准化严重程度评估 |
| 净化行为范围 | 仅限呕吐/泻药 | **含运动/禁食** | 更全面捕捉补偿行为 |

### 1.3 ICD-11 vs DSM-5-TR 主要差异

| 维度 | DSM-5-TR | ICD-11 |
|:-----|:---------|:-------|
| 分类结构 | 8个独立诊断 | 简化为6个主类别 |
| AN 亚型 | 限制型 vs 暴食清除型 | 不区分亚型，标注严重程度 |
| BN 严重程度 | 按频率分级 | 不按频率分级 |
| BED | 独立诊断 | 独立诊断（编码 6B82） |
| ARFID | 独立诊断 | 独立诊断（编码 6B83） |
| 跨诊断视角 | 分类优先 | 更强调谱系连续性 |

## 2. 流行病学

### 2.1 全球患病率数据

| 障碍 | 女性终生患病率 | 男性终生患病率 | 女:男比 | 平均起病年龄 | 全球趋势 |
|:-----|:--------------|:--------------|:--------|:------------|:---------|
| AN | 0.9-1.4% | 0.2-0.3% | 3-10:1 | 15-19 岁 | 稳定/略升 |
| BN | 1.5-2.0% | 0.3-0.5% | 3-5:1 | 18-25 岁 | 稳定 |
| BED | 3.0-3.5% | 1.5-2.0% | 1.5-2:1 | 20-40 岁 | 上升 |
| ARFID | 1.5-5.0%（儿童） | 相近 | ~1:1 | 儿童/青少年 | 识别率上升 |
| OSFED | 3-5% | 1-2% | 2-3:1 | 任何年龄 | — |

### 2.2 中国流行病学（有限数据）

| 研究 | 样本 | AN 患病率 | BN 患病率 | BED 患病率 | 备注 |
|:-----|:-----|:---------|:---------|:----------|:-----|
| Tong et al., 2014 | 大学生 N=2,352 | 0.3% | 1.1% | 2.5% | SCOFF 筛查 |
| He et al., 2020 | 中学生 N=5,813 | — | — | 1.8% | EAT-26 高风险 11.4% |
| 系统综述 2022 | 24项中国研究 | 0.1-0.7% | 0.5-1.9% | 1.0-4.6% | 方法异质性大 |

**关键观察**：
- 中国进食障碍检出率低于西方，但呈上升趋势
- 男性、农村居民、中老年群体数据严重不足
- "减肥文化"与社交媒体驱动的新发病例增加

### 2.3 死亡率与疾病负担

| 指标 | AN | BN | BED |
|:-----|:---|:---|:---|
| 标准化死亡比 (SMR) | **5.1-10.0** | 1.9-2.3 | 1.5-2.0 |
| 自杀率 | 15-25%（死因构成） | 10-15% | 5-10% |
| DALY（伤残调整寿命年） | 中高 | 中等 | 中高（因肥胖共病） |
| 平均病程 | 5-7 年 | 5-8 年 | 慢性/反复 |
| 完全恢复率 | 46-51% | 50-70% | 50-60% |

> ⚠️ **AN 是精神障碍中致死率最高的疾病之一**，SMR 可达 10 倍于一般人群。

## 3. 病因学模型

### 3.1 多因素风险模型

| 风险层级 | 因素类型 | 具体因素 | 效应方向 |
|:---------|:---------|:---------|:---------|
| **遗传/生物** | 遗传 | AN 遗传率 48-74%，BN 55-83% | 强 |
| | 神经递质 | 5-HT 功能异常、多巴胺奖赏通路 | 中 |
| | 脑结构 | 岛叶/前额叶/纹状体功能改变 | 中 |
| | 内分泌 | 饥饿激素(ghrelin)/瘦素/皮质醇紊乱 | 中 |
| **心理** | 人格特质 | 完美主义/冲动性/强迫特质/神经质 | 中-强 |
| | 认知风格 | 全或无思维/灾难化/过度评价体型 | 中 |
| | 情绪调节 | 情绪不耐受/alexithymia/回避应对 | 中 |
| | 创伤史 | 性虐待/童年创伤/霸凌 | 中 |
| **社会文化** | 文化压力 | 瘦理想内化/外貌比较/媒体暴露 | 中-强 |
| | 家庭因素 | 家庭进食态度/高冲突/过度保护 | 中 |
| | 同伴影响 | 体重相关嘲笑/节食同伴/运动队压力 | 中 |
| | 职业风险 | 芭蕾/体操/模特/长跑/摔跤 | 强 |

### 3.2 跨诊断模型 (Transdiagnostic Model — Fairburn)

```
┌─────────────────────────────────────────────────────┐
│               跨诊断核心维持机制                       │
│                                                     │
│  临床完美主义 ←→ 自我评价过度依赖体型/体重            │
│        ↕                    ↕                       │
│  情绪不耐受   ←→  人际困难                          │
│        ↕                    ↕                       │
│  认知僵化（全或无饮食规则）                           │
│                                                     │
│  → 维持：限制进食 ↔ 暴食 ↔ 清除 ↔ 过度运动          │
└─────────────────────────────────────────────────────┘
```

**核心观点**：AN、BN、BED 共享相同的维持机制，区别在于表现形式。
- **AN 限制型**：临床完美主义 + 极度限制
- **AN 暴食清除型 / BN**：限制→暴食→清除循环
- **BED**：情绪不耐受 + 暴食（无清除）

### 3.3 发展轨迹模型

| 阶段 | 年龄窗口 | 关键事件 | 风险升级因素 |
|:-----|:---------|:---------|:------------|
| **易感期** | 儿童期 | 气质特质（完美/冲动/焦虑）、早期喂养问题 | 家族史、创伤 |
| **触发期** | 青春期 | 青春期体型变化、同伴压力、节食启动 | 体重嘲笑、社交媒体 |
| **维持期** | 青壮年 | 进食障碍行为成为应对策略 | 共病（焦虑/抑郁）、环境强化 |
| **慢性化/恢复期** | 成年期 | 治疗干预 vs 自然恢复 vs 慢性化 | 治疗可及性、社会支持 |

## 4. 鉴别诊断要点

### 4.1 核心鉴别矩阵

| 鉴别维度 | AN | BN | BED | ARFID | 医学状况 |
|:---------|:---|:---|:----|:------|:---------|
| BMI | <18.5 | 正常或超重 | 正常或超重 | 可变 | 可变 |
| 体像关注 | **强烈** | **强烈** | 可变 | **无** | 无 |
| 暴食 | 否/是 | **是** | **是** | 否 | 否 |
| 清除 | 否/是 | **是** | **否** | 否 | 否 |
| 恐胖 | **核心** | **核心** | 可变 | **无** | 无 |
| 食物回避原因 | 恐胖 | 恐胖 | 无 | 感觉/恐惧/低兴趣 | 器质性 |

### 4.2 常见鉴别陷阱

| 陷阱 | 描述 | 正确做法 |
|:-----|:-----|:---------|
| AN vs 医学消瘦 | 甲亢/糖尿病/IBD/恶性肿瘤 | 排除器质性病因（实验室+影像） |
| BN vs BED | 是否规律性出现补偿行为 | 详细行为史访谈 |
| ARFID vs 挑食 | ARFID 有营养/心理/社会功能损害 | 评估功能损害程度 |
| BED vs 肥胖 | BED 有暴食+失控感+痛苦 | 评估暴食行为模式 |
| AN vs 回避饮食文化 | 宗教斋戒/素食/文化饮食禁忌 | 评估体像关注与恐胖 |

## 5. 共病图谱

| 共病类别 | AN 共病率 | BN 共病率 | BED 共病率 | 临床影响 |
|:---------|:---------|:---------|:----------|:---------|
| **焦虑障碍** | 48-65% | 50-63% | 37-45% | OCD/SAD/GAD 最常见 |
| **抑郁障碍** | 42-56% | 50-70% | 46-57% | 双相关系 |
| **物质使用** | 12-24% | 22-35% | 15-25% | BN 酒精/兴奋剂多见 |
| **人格障碍** | 20-40% | 30-50% | 25-35% | AN↔C簇 / BN↔B簇 |
| **PTSD/创伤** | 10-20% | 25-40% | 20-30% | 创伤→暴食作为应对 |
| **自伤/NSSI** | 15-25% | 25-40% | 15-25% | 疼痛-惩罚功能 |
| **自杀意念/行为** | 15-25% | 20-35% | 10-20% | AN 致死率最高 |
| **肥胖（BMI≥30）** | 罕见 | 10-20% | 40-60% | BED 需联合减重 |

## 6. 治疗原则概览

### 6.1 NICE 指南分层推荐

| 严重程度 | AN 推荐 | BN 推荐 | BED 推荐 |
|:---------|:--------|:--------|:---------|
| **轻度** | 门诊 CBT-ED/FBT | 自助 CBT-BN | 自助 CBT-BED |
| **中度** | 门诊/日间 CBT-ED + 营养 | 引导自助→CBT-BN | 引导自助→CBT-BED |
| **重度** | 专科门诊/住院 + FBT | 专科 CBT-BN / IPT | 专科 CBT-BED / IPT |
| **极重/危机** | 住院/强制治疗 | 住院（躯体不稳） | 住院（共病危机） |

### 6.2 多学科团队 (MDT) 构成

| 角色 | 职责 | 必要性 |
|:-----|:-----|:------:|
| 精神科医生 | 诊断、药物、躯体监测 | 核心 |
| 临床心理师 | CBT-E/FBT/IPT/DBT | 核心 |
| 注册营养师 | 营养评估、膳食计划、再喂养 | 核心 |
| 内科/儿科医生 | 躯体并发症管理 | 按需 |
| 家庭治疗师 | FBT/家庭干预 | 青少年核心 |
| 社会工作者 | 社会功能、学校/职业支持 | 按需 |
| 护士 | 体重监测、进食观察、危机管理 | 住院/日间 |

## 7. 预后因素

| 预后因子 | 良好预后 | 不良预后 |
|:---------|:---------|:---------|
| **起病年龄** | 青少年期起病 | 成年晚期起病 |
| **病程** | 早期干预（<3年） | 慢性病程（>10年） |
| **亚型** | BN/BED | AN 暴食清除型 |
| **BMI 恢复** | 早期体重恢复 | 极低BMI/反复波动 |
| **共病** | 无严重共病 | 共病人格障碍/物质滥用 |
| **家庭** | 家庭参与治疗 | 家庭高冲突/高EE |
| **治疗** | 早期完成CBT-E/FBT | 反复住院/脱落 |
| **心理** | 动机强/内省力好 | 完美主义僵化/否认严重性 |
