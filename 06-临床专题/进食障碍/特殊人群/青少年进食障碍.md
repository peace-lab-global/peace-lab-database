# 青少年进食障碍 (Adolescent Eating Disorders)

> **核心定位**：青少年是进食障碍的高峰起病期，早期干预预后最佳。FBT 是青少年 AN 一线治疗，学校与家庭需协同参与。

---

## 1. 青少年 ED 特殊性

### 1.1 与成人的关键差异

| 维度 | 青少年特点 | 临床意义 |
|:-----|:-----------|:---------|
| **体重评估** | 使用 BMI 百分位（非绝对值） | < 第5百分位 = 显著低体重 |
| **闭经** | 初潮前起病者不适用闭经标准 | 可能影响骨骼发育 |
| **体像关注** | 可能不如成人明确 | 行为观察更重要 |
| **生长影响** | 可能影响最终身高 | 生长曲线监测 |
| **认知能力** | 抽象思维尚在发展 | 简化认知技术 |
| **家庭依赖** | 经济/情感依赖家庭 | 家庭参与治疗是必须的 |
| **同伴影响** | 同伴压力高峰 | 学校/社交干预 |
| **治疗动机** | 更低（自我协调） | 动机工作更关键 |

### 1.2 青少年 ED 类型分布

| 诊断 | 青少年占比 | 说明 |
|:-----|:----------:|:-----|
| AN | ~40% | 青少年 ED 最常见类型 |
| BN | ~20% | 多在青春期后期/早期成年 |
| BED | ~15% | 随肥胖率上升而增加 |
| ARFID | ~10% | 多为儿童期起病持续 |
| OSFED | ~15% | 非典型 AN 在青少年中特别常见 |

---

## 2. 评估要点

| 评估内容 | 青少年特殊性 |
|:---------|:------------|
| 生长曲线 | 使用 CDC/WHO 生长图表 |
| 青春期评估 | Tanner 分期（影响营养需求） |
| 家长访谈 | **必须**（青少年常低报症状） |
| 同伴关系 | 体重嘲笑/社交回避 |
| 学业影响 | 注意力/缺课/体育免修 |
| 社交媒体 | 饮食内容/比较行为/身体检查 |
| 自伤/自杀 | 青少年自伤率高 |

---

## 3. 治疗方案

### 3.1 青少年 ED 治疗选择

| 诊断 | 一线 | 二线 | 说明 |
|:-----|:-----|:-----|:-----|
| **AN** | FBT | CBT-E-Adapted | FBT 效果优于个体治疗 |
| **BN** | FBT-BN 或 CBT-BN | — | 家庭参与版本 |
| **BED** | CBT-BED（青少年版） | FBT 调整 | — |
| **ARFID** | 多学科+暴露 | — | 无标准方案 |

### 3.2 学校干预

| 策略 | 说明 |
|:-----|:-----|
| 午餐监督 | 家长/学校协调午餐计划 |
| 体育课调整 | 限制运动（AN）/适当活动 |
| 缺课处理 | 灵活出勤/家庭教师 |
| 同伴教育 | 反体重嘲笑/身体多样性 |
| 隐私保护 | 限制学校知情范围 |

---

## 4. 预后

| 因素 | 青少年优势 | 青少年劣势 |
|:-----|:----------|:----------|
| 早期干预 | 病程短/效果好 | 常延迟就诊 |
| 脑可塑性 | 恢复潜力大 | 营养不良影响脑发育 |
| 家庭支持 | FBT 可动员 | 家庭冲突时阻碍 |
| 长期 | 完全恢复率高于成人 | 慢性化可延续至成年 |
