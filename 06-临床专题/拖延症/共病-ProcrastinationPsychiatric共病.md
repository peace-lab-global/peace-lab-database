# 拖延的精神科共病 | Psychiatric Comorbidity in Procrastination

> **目标**：系统阐述拖延与常见精神障碍的共病关系、鉴别要点与整合治疗策略。
> **标准来源**：Steel (2007)、DSM-5-TR、ADHD临床指南。

---

## 一、共病率总览

| 共病障碍 | 拖延者中共病率 | 说明 |
|---------|--------------|------|
| **抑郁障碍** | 25-50% | 最常见共病 |
| **焦虑障碍（尤其GAD、社交焦虑）** | 20-40% | 焦虑驱动回避 |
| **ADHD** | 15-30%（成人） | 执行功能缺陷 |
| **物质使用障碍** | 10-20% | 自我药疗/逃避 |
| **OCD/强迫相关** | 5-15% | 完美主义、检查 |
| **人格障碍（尤其回避型、强迫型）** | 10-20% | 人格模式 |
| **游戏障碍/网络成瘾** | 10-25% | 拖延的数字化形式 |

---

## 二、拖延与抑郁

### 2.1 双向关系

| 方向 | 机制 |
|------|------|
| **抑郁→拖延** | 精力缺乏；快感缺失；无望感；注意力困难 |
| **拖延→抑郁** | 自责；功能下降；慢性压力；失败体验 |
| **恶性循环** | 拖延→失败/自责→抑郁加重→更多拖延 |

### 2.2 鉴别

| 特征 | 抑郁为主 | 拖延为主 |
|------|---------|---------|
| **情绪** | 广泛低落；持续整天 | 特定于任务；做愉快事时情绪正常 |
| **兴趣** | 广泛丧失 | 仅对特定困难任务 |
| **精力** | 普遍低下 | 做喜欢的事时精力充沛 |
| **睡眠/食欲** | 常有改变 | 通常正常 |
| **自杀意念** | 可能有 | 无 |

### 2.3 治疗策略

| 优先级 | 干预 |
|--------|------|
| **重度抑郁** | 优先治疗抑郁（药物+CBT）；拖延为次要目标 |
| **轻中度抑郁+拖延** | 可整合治疗；行为激活同时改善两者 |
| **残留抑郁** | 拖延可能是残留症状；加强行为激活 |

---

## 三、拖延与焦虑

### 3.1 GAD与拖延

| 关联 | 说明 |
|------|------|
| **担忧→拖延** | 对任务结果的过度担忧导致回避 |
| **拖延→更多担忧** | 未完成任务成为新的担忧来源 |
| **完美主义中介** | GAD常伴随"如果我不完美，坏事情会发生" |

### 3.2 社交焦虑与拖延

| 关联 | 说明 |
|------|------|
| **人际任务回避** | 电话、邮件、会议准备 |
| **表现焦虑** | 害怕他人评价导致任务启动困难 |
| **安全行为** | 过度准备以避免负面评价 |

### 3.3 治疗策略

| 策略 | 说明 |
|------|------|
| **暴露** | 逐步面对拖延任务；验证焦虑预测 |
| **认知重构** | 挑战"做不好会被评价"的信念 |
| **CBT-I（如共病失眠）** | 焦虑常伴随睡眠问题 |
| **药物** | SSRI/SNRI（中重度焦虑） |

---

## 四、拖延与ADHD

### 4.1 鉴别要点

| 维度 | ADHD | 纯拖延 |
|------|------|--------|
| **起病** | 童年期（12岁前） | 青少年/成年 |
| **领域** | 广泛（包括感兴趣任务） | 主要厌恶任务 |
| **注意力** | 持续问题 | 情境性 |
| **组织** | 系统性 | 可选择性组织 |
| **冲动** | 广泛 | 任务相关 |
| **家族史** | 常见 | 不一定 |

### 4.2 共病时的治疗

| 策略 | 说明 |
|------|------|
| **药物** | 兴奋剂为ADHD一线；可能间接改善拖延 |
| **执行功能训练** | 外部化策略（日历、提醒、清单） |
| **CBT+ADHD调整** | 更结构化；更频繁检查；更短的会谈 |
| **环境改造** | 比纯拖延者更关键 |

---

## 五、拖延与OCD/强迫相关

### 5.1 关联

| 特征 | 说明 |
|------|------|
| **检查型OCD** | 反复检查导致任务无法完成 |
| **对称/精确型** | 对秩序的高标准导致任务拖延 |
| **囤积障碍** | 决策困难导致物品/任务堆积 |
| **身体变形障碍** | 反复照镜子/寻求 reassurance 消耗时间 |

### 5.2 治疗

| 策略 | 说明 |
|------|------|
| **ERP** | 暴露与反应预防；核心OCD治疗 |
| **行为实验** | 故意不完美/不检查 |
| **SSRI** | 中重度OCD |

---

## 六、拖延与物质使用

### 6.1 关联

| 方向 | 说明 |
|------|------|
| **拖延→物质使用** | 物质用于逃避拖延带来的焦虑 |
| **物质使用→拖延** | 物质使用本身消耗时间；戒断影响功能 |
| **共同机制** | 冲动性；即时满足偏好 |

### 6.2 治疗

| 优先级 | 说明 |
|--------|------|
| **先稳定物质使用** | 急性期后处理拖延 |
| **共同机制** | 冲动控制训练；延迟满足练习 |

---

## 七、拖延与人格障碍

### 7.1 回避型人格障碍

| 特征 | 说明 |
|------|------|
| **核心** | 对拒绝/批评的极度敏感 |
| **拖延** | 避免任何可能被评价的情境 |
| **治疗** | 图式治疗；暴露；社交技能训练 |

### 7.2 强迫型人格障碍（OCPD）

| 特征 | 说明 |
|------|------|
| **核心** | 完美主义、控制、僵化 |
| **拖延** | "还没准备好"；对细节的过度关注 |
| **鉴别** | OCPD的完美主义是自我和谐的（"这是正确的方式"） |
| **治疗** | 认知治疗（灵活性、容忍不确定性） |

---

## 八、整合治疗原则

| 原则 | 说明 |
|------|------|
| **先处理安全/危机** | 自杀风险；严重物质使用 |
| **优先严重共病** | 重度抑郁/焦虑先治疗 |
| **区分原发继发** | 抑郁导致的拖延 vs 拖延导致的抑郁 |
| **共同机制** | 识别并针对共享机制（如情绪调节、冲动性） |
| **灵活调整** | 根据共病调整标准拖延治疗方案 |
