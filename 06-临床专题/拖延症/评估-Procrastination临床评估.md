# 拖延临床评估 | Clinical Assessment of Procrastination

> **目标**：建立拖延的系统化临床评估工具包，涵盖标准化量表、结构化访谈与功能分析。
> **标准来源**：Steel (2007)、APA、临床行为分析框架。

## 二、标准化评估量表

### 2.1 纯拖延量表

| 量表 | 项目 | 用途 | 特点 |
|------|------|------|------|
| **PURE Procrastination Scale** | 12项 | 一般拖延倾向 | 简短；广泛用于研究 |
| **Aitken Procrastination Inventory (API)** | 19项 | 大学生学业拖延 | 经典量表 |
| **Tuckman Procrastination Scale** | 16项 | 一般拖延 | 评估拖延习惯 |
| **Irrational Procrastination Scale (IPS)** | 9项 | 非理性拖延 | 简短；临床可用 |
| **Decisional Procrastination Scale** | 5项 | 决策拖延 | 评估选择困难 |

### 2.2 拖延与相关构念

| 量表 | 评估内容 | 用途 |
|------|---------|------|
| **General Procrastination Scale (Lay, 1986)** | 一般拖延倾向 | 最广泛使用的量表之一 |
| **Adult Inventory of Procrastination (AIP)** | 成人拖延 | 评估习惯性拖延 |
| **Procrastination Assessment Scale-Students (PASS)** | 学生拖延（认知/行为/决策） | 多维评估 |
| **Susceptibility to Temptation Scale** | 诱惑易感性 | 评估冲动/诱惑 |
| **Consideration of Future Consequences (CFC)** | 未来后果考量 | 时间取向 |

### 2.3 相关心理构念量表

| 量表 | 用途 |
|------|------|
| **Frost Multidimensional Perfectionism Scale (FMPS)** | 完美主义多维评估 |
| **Self-Regulation Scale** | 自我调节能力 |
| **Trait Self-Control Scale** | 特质自我控制 |
| **UPPS-P Impulsive Behavior Scale** | 冲动性五因素 |
| **Fear of Failure Scale** | 失败恐惧 |
| **Self-Efficacy Scale** | 自我效能 |
| **Depression Anxiety Stress Scales (DASS-21)** | 共病情绪症状 |
| **Adult ADHD Self-Report Scale (ASRS-v1.1)** | ADHD筛查 |

### 2.4 行为测量

| 方法 | 说明 |
|------|------|
| **行为日志** | 记录计划vs实际执行；时间追踪 |
| **客观任务完成** | 在实验室设置中测量任务启动延迟 |
| **手机/电脑使用数据** | 屏幕时间作为拖延行为的客观指标 |
| **生态瞬时评估（EMA）** | 随机时间抽样记录当下情绪和行为 |

## 三、功能分析（Functional Analysis）

### 3.1 ABC分析

| 成分 | 问题 |
|------|------|
| **A（前因 Antecedent）** | 什么情境/想法/情绪在拖延前出现？ |
| **B（行为 Behavior）** | 具体的拖延行为是什么？ |
| **C（后果 Consequence）** | 短期后果（情绪缓解）？长期后果（负面结果）？ |

### 3.2 功能分析示例

| 情境 | A（前因） | B（拖延） | C-短期 | C-长期 |
|------|----------|----------|--------|--------|
| 写论文 | "我不知道怎么开始"（焦虑↑） | 刷手机2小时 | 焦虑↓ | 时间紧迫；质量下降；自责 |
| 回邮件 | 害怕对方反应（社交焦虑） | 延迟回复 | 暂时回避焦虑 | 关系紧张；工作延误 |
| 整理房间 | 任务庞大、无聊 | 做其他"重要"小事 | 感觉"在做有用的事" | 环境持续混乱 |

### 3.3 识别拖延的功能

| 功能 | 说明 |
|------|------|
| **负强化** | 逃避/减少负面情绪（最常见） |
| **正强化** | 获得即时愉悦（手机、游戏） |
| **自我障碍** | "如果我拖延，失败就不是因为我能力不足" |
| **自主性维护** | 反抗外部控制/要求 |
| **情绪调节** | 通过拖延调节当下情绪状态 |

## 四、鉴别诊断评估

### 4.1 ADHD vs 拖延

| 维度 | ADHD | 纯拖延 |
|------|------|--------|
| **起病** | 童年期（12岁前） | 青少年/成年期 |
| **领域** | 广泛（包括感兴趣的任务） | 主要限于厌恶/困难任务 |
| **注意力** | 持续涣散 | 可专注（在有趣任务上） |
| **组织** | 系统性困难 | 特定任务困难 |
| **冲动** | 广泛（言语/行为） | 主要限于任务回避 |
| **家族史** | 常有 | 不一定 |

### 4.2 抑郁 vs 拖延

| 维度 | 抑郁 | 纯拖延 |
|------|------|--------|
| **情绪** | 广泛低落；持续 | 特定于任务；短暂 |
| **兴趣** | 广泛丧失 | 仅对特定任务 |
| **精力** | 普遍降低 | 仅在拖延任务时 |
| **睡眠/食欲** | 常有改变 | 通常正常 |
| **自杀意念** | 可能有 | 无 |
