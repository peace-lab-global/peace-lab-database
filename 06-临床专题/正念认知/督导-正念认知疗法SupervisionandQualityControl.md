# MBCT 督导与质量控制 | Supervision and Quality Control for MBCT

> **目标**：建立精神科标准的 MBCT 督导框架、教师胜任力体系、伦理边界、文档记录标准与跨学科团队协作规范。
> **原则**：MBCT 的教学质量直接影响疗效和安全性；精神科督导是保障临床标准的关键环节。

---

## 一、MBCT 教师胜任力体系

### 1.1 入门要求（Good Practice Guidelines, UK Network）

| 领域 | 要求 | 证明方式 |
|------|------|---------|
| **个人正念修习** | 持续每日练习 ≥ 1年 | 自我报告 + 督导确认 |
| **密集静修** | 至少1次 ≥ 5天静默静修 | 培训证书 |
| **作为参与者** | 完整参加8周MBCT/MBSR课程 | 参与证明 |
| **专业培训** | 完成MBCT教师培训（≥4天） | 培训证书 |
| **专业背景** | 精神卫生专业资格（精神医学、临床心理学、心理治疗、社工） | 执业证书 |
| **CBT知识** | 理解认知行为理论和抑郁复发模型 | 教育背景或培训证明 |
| **教学经验** | 团体教学能力 | 实践经验 |

### 1.2 督导实践路径

| 阶段 | 内容 | 时长 | 产出 |
|------|------|------|------|
| **观察** | 观察资深教师教授完整MBCT课程 | 1-2个课程 | 观察报告 |
| **协同教学** | 与督导教师共同教授 | 1-2个课程 | 协同教学反馈 |
| **督导下独立教学** | 在督导下独立教授 | 2-3个课程 | MBI-TAC评估 |
| **独立教学** | 独立教授 | 持续 | 年度督导维持 |

### 1.3 MBI-TAC 评估框架（Mindfulness-Based Interventions Teaching Assessment Criteria）

| 领域 | 具体能力 | 合格标准 |
|------|---------|---------|
| **覆盖（Coverage）** | 涵盖课程核心主题和内容 | 完整覆盖8周核心主题 |
| **节奏与组织** | 时间管理；流畅过渡 | 准时；自然流动 |
| **关系技能** | 温暖、真诚、接纳 | 建立安全团体环境 |
| **具身化（Embodiment）** | 教师自身体现正念态度 | 当下、真实、正念典范 |
| **引导正念练习** | 清晰、节奏适当的引导 | 支持性、灵活、回应性 |
| **探询（Inquiry）** | 探索参与者体验的技能 | 好奇、具体、支持去中心化 |
| **传达课程主题** | 清晰传达理论和原理 | 整合体验和理论 |
| **持有团体** | 促进安全、互动的团体 | 平衡个人和团体 |

**评分**：每个领域1-6分；总分决定胜任力等级。

## 二、精神科督导框架

### 2.1 督导模式

| 督导类型 | 频率 | 内容 | 参与者 |
|---------|------|------|--------|
| **个人督导** | 每2周1次 | 个案概念化；困难参与者；反移情；伦理困境 | MBCT教师 + 资深MBCT督导 |
| **团体督导** | 每月1次 | 团体动力；课程进度；共性困难；标准化 | 所有MBCT教师 + 督导 |
| **精神科案例督导** | 每4周1次或按需 | 诊断确认；药物管理；危机决策；风险分层 | MBCT教师 + 精神科医生 |
| **同伴督导** | 每2周1次 | 经验分享；情感支持；实践反思 | MBCT教师同伴 |

### 2.2 督导核心议题

| 议题 | 精神科关注点 |
|------|-------------|
| **入组评估** | 诊断准确性；排除标准执行；风险评估充分性 |
| **危机处理** | 自杀风险评估质量；安全计划 adequacy；转介及时性 |
| **药物议题** | 药物调整建议的边界；停药症状 vs 复发鉴别 |
| **共病管理** | 人格障碍对团体的影响；创伤知情实践 |
| **疗效监测** | 量表使用规范性；无应答者识别；升级决策 |
| **脱落管理** | 脱落原因分析；系统性改进 |
| **教师福祉** | 替代性创伤；耗竭；个人练习维持 |

## 三、伦理边界与执业范围

### 3.1 MBCT教师的执业边界

| 可独立执行 | 需协作/转介 | 禁止执行 |
|-----------|------------|---------|
| 教授MBCT课程 | 精神科诊断确认 | 开具精神药物处方 |
| 引导正念练习 | 药物管理建议 | 改变患者药物方案 |
| 探询和团体带领 | 危机干预决策 | 独立诊断精神障碍 |
| 症状监测和量表施测 | 住院/强制治疗决策 | 提供非MBCT的心理治疗 |
| 练习日志反馈 | 复杂的个案概念化 | 超出训练范围的治疗 |
| 维持团体带领 | 法律/法医评估 | 承诺治愈或特定疗效 |

### 3.2 知情同意中的伦理要素

| 要素 | 内容 |
|------|------|
| **治疗性质** | MBCT是心理教育团体，不是个体心理治疗 |
| **教师资质** | 教师的培训和督导状态 |
| **保密边界** | 团体保密；危险情况突破保密 |
| **退出权利** | 随时退出，无需理由 |
| **费用** | 课程费用；评估费用；督导费用 |
| **替代方案** | 其他可用治疗（药物、CBT、IPT等） |
| **风险告知** | 情绪恶化、解离等潜在不良反应 |

### 3.3 双重关系管理

| 情况 | 风险 | 管理 |
|------|------|------|
| **教师同时是患者的朋友/家人** | 边界模糊；影响团体动力 | 转介给其他教师 |
| **教师与患者有商业关系** | 利益冲突 | 转介或终止商业关系 |
| **教师与患者有浪漫关系** | 严重伦理违规 | 绝对禁止；转介；督导处理 |
| **患者在教师所在机构工作** | 权力不对等 | 评估风险；可能需要转介 |

## 四、文档记录标准

### 4.1 必需文档清单

| 文档 | 内容要求 | 保存期限 |
|------|---------|---------|
| **入组评估表** | 诊断、量表分数、风险分层、排除项核查、知情同意 | ≥7年 |
| **安全计划** | 六步安全计划（SPI）；危机卡片；紧急联系人 | 随个案记录 |
| **课程记录** | 每次课程出席、内容覆盖、关键事件、家庭作业布置 | ≥7年 |
| **过程监测** | PHQ-9等量表结果；临床观察；练习日志反馈 | ≥7年 |
| **督导记录** | 督导日期、议题、建议、教师反思 | ≥7年 |
| **不良事件报告** | 事件描述、严重程度、处理、转归 | ≥7年 |
| **终点评估** | 量表变化、CGI-I、复发事件、维持计划 | ≥7年 |
| **随访记录** | 3月、6月、12月追踪结果 | ≥7年 |

### 4.2 电子病历规范

| 要求 | 标准 |
|------|------|
| **加密** | 所有电子记录必须加密存储 |
| **访问控制** | 基于角色的访问权限 |
| **审计追踪** | 记录所有访问和修改 |
| **备份** | 定期异地备份 |
| **匿名化** | 研究使用数据需去标识化 |

## 五、跨学科团队协作

### 5.1 团队角色与沟通

| 角色 | 核心职责 | 与MBCT教师沟通 |
|------|---------|---------------|
| **精神科医生** | 诊断；药物；危机；风险最终评估 | 入组前确认；每4-8周更新；危机即时 |
| **临床心理师** | 入组评估；个案概念化；个体支持 | 持续；共享评估报告 |
| **MBCT教师** | 课程实施；过程监测；症状变化报告 | 向精神科报告重大变化24小时内 |
| **护士/助理** | 量表施测；预约协调；危机联络 | 日常协调 |
| **社工** | 社会支持评估；资源链接；家庭工作 | 需要时介入 |

## 六、质量改进循环

### 6.1 关键质量指标（KPIs）

| 指标 | 目标值 | 监测频率 |
|------|--------|---------|
| **完成率** | ≥ 75%（完成≥6/8周） | 每课程 |
| **有效率** | ≥ 60%（PHQ-9下降≥50%） | 每课程 |
| **复发率（12月）** | ≤ 30%（≥3次发作者） | 每季度 |
| **不良事件率** | 记录并分析所有事件 | 持续 |
| **患者满意度** | ≥ 4.0/5.0 | 每课程 |
| **教师督导完成率** | 100% | 持续 |

### 6.2 持续质量改进（CQI）流程

```
数据收集 → 指标分析 → 识别差距 → 根因分析 → 改进措施 → 实施 → 再评估
    ↑                                                              │
    └──────────────────────────────────────────────────────────────┘
```

| 差距类型 | 可能根因 | 改进措施 |
|---------|---------|---------|
| 完成率低 | 入组标准过宽；交通障碍；团体不适 | 加强筛查；提供在线选项；改善团体动力 |
| 有效率低 | 教师培训不足；练习依从性低 | 加强教师督导；增强动机访谈 |
| 复发率高 | 维持期支持不足 | 建立毕业生团体；加强复训 |
| 不良事件多 | 筛查不足；安全协议缺失 | 加强入组评估；完善危机协议 |
