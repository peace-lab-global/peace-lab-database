# 哀伤的维持治疗与随访 | Maintenance and Follow-Up for Grief

> **目标**：建立哀伤干预后的长期随访框架，预防复发，支持持续康复。
> **标准来源**：APA PGD临床手册、丧亲随访研究。

---

## 一、随访频率

| 阶段 | 时间 | 频率 | 重点 |
|------|------|------|------|
| **急性期后** | 治疗结束1-3月 | 每月 | 功能维持；应对新挑战 |
| **中期** | 3-12月 | 每2-3月 | 周年纪念准备；持续评估 |
| **长期** | >1年 | 每6月或按需 | 重大纪念日；生活变化 |

---

## 二、关键随访时间点

### 2.1 周年纪念

| 时间 | 说明 |
|------|------|
| **第一年** | 最困难；可能症状加重 |
| **准备** | 提前讨论应对策略 |
| **仪式** | 计划 memorial 活动 |
| **注意** | 不是"应该已经好了"的标志 |

### 2.2 其他重要日期

| 日期 | 说明 |
|------|------|
| **逝者的生日** | 可能触发强烈思念 |
| **节假日** | 家庭聚会时的空缺感 |
| **个人里程碑** | 毕业、结婚、生子（"逝者不在"） |

---

## 三、复发预防

### 3.1 维持策略

| 策略 | 说明 |
|------|------|
| **持续联结** | 保持与逝者的适应性联结 |
| **纪念仪式** | 定期的个人/家庭 memorial |
| **意义维持** | 继续 legacy 活动 |
| **社会支持** | 维持支持网络 |
| **自我照顾** | 健康生活方式 |

### 3.2 预警信号

| 信号 | 说明 |
|------|------|
| **症状加重** | 悲伤、失眠、功能下降 |
| **回避增加** | 重新开始回避提醒物 |
| **社会退缩** | 孤立增加 |
| **物质使用增加** | 应对方式退化 |
| **自杀意念** | 任何自杀想法 |

### 3.3 复发应对

| 步骤 | 行动 |
|------|------|
| **1. 识别** | 觉察症状变化 |
| **2. 激活资源** | 联系治疗师/支持系统 |
| **3. 使用技能** | 重新使用治疗中学到的技能 |
| **4. 评估需要** | 是否需要额外会谈 |
| **5. 加强支持** | 增加随访频率 |

---

## 四、长期康复标志

| 标志 | 说明 |
|------|------|
| **哀伤融入身份** | "我有这段丧失的经历"而非"我是破碎的" |
| **功能恢复** | 工作、社交、自我照顾 |
| **新意义** | 找到丧失后的个人意义 |
| **持续联结** | 与逝者的适应性内在关系 |
| **新角色** | 接受并适应新身份 |
| **希望** | 对未来有期待 |

---

## 五、终身视角

| 认识 | 说明 |
|------|------|
| **哀伤不会"结束"** | 哀伤融入生命；强度随时间变化 |
| **触发持续存在** | 重大纪念日、生活里程碑可能触发 |
| **成长可能** | 创伤后成长；从哀伤中发现意义 |
| **持续支持** | 知道需要帮助时如何获取 |
