# 创伤性哀伤干预 | Interventions for Traumatic Grief

> **目标**：系统阐述创伤性哀伤的临床特征、评估与整合治疗策略。
> **标准来源**：APA PGD临床手册、Nielsen et al. (2019)、Boelen & Smid (2017)。

---

## 一、创伤性哀伤的定义

### 1.1 核心概念

**创伤性哀伤（Traumatic Grief）** 指丧亲同时伴随创伤性应激反应。死亡情境为暴力、意外、可怖或与创伤性事件相关。

### 1.2 与PGD的区别

| 特征 | PGD | 创伤性哀伤 |
|------|-----|-----------|
| **核心** | 分离痛苦、渴望 | 创伤性再体验 + 哀伤 |
| **侵入** | 可能轻度 | 强烈（死亡场景闪回） |
| **回避** | 回避提醒物 | 广泛回避 |
| **警觉** | 不明显 | 高度警觉 |
| **死亡类型** | 任何 | 通常暴力/意外/可怖 |
| ** guilt** | 可能有 | 常强烈（幸存者 guilt ） |

---

## 二、临床表现

### 2.1 哀伤症状

| 症状 | 说明 |
|------|------|
| **分离痛苦** | 强烈的渴望/思念 |
| **认同瓦解** | "我的一部分已死去" |
| **无意义感** | 生活失去意义 |

### 2.2 创伤症状

| 症状 | 说明 |
|------|------|
| **侵入** | 死亡场景的闪回、噩梦 |
| **回避** | 回避与死亡相关的任何提醒 |
| **负性认知** | "世界完全危险""我随时会死" |
| **高度警觉** | 惊跳、睡眠障碍、注意力警觉 |
| **幸存者 guilt** | "为什么是我活下来" |

---

## 三、治疗原则

### 3.1 整合模型

创伤性哀伤需要**同时处理创伤和哀伤**，而非先后。

| 传统顺序 | 整合模型 |
|---------|---------|
| 先稳定化 → 再PTSD → 再哀伤 | 同时进行；根据患者状态灵活调整 |

### 3.2 治疗框架

| 阶段 | 目标 | 技术 |
|------|------|------|
| **稳定化** | 安全；情绪调节；建立联盟 |  grounding；DBT技能；安全计划 |
| **创伤处理** | 加工创伤记忆 | EMDR；PE；叙事暴露 |
| **哀伤处理** | 处理丧失 | CGT技术；意义重建 |
| **整合** | 重建生活 | 行为激活；身份重建 |

---

## 四、具体干预技术

### 4.1 EMDR（眼动脱敏再加工）

| 应用 | 说明 |
|------|------|
| **创伤记忆** | 死亡场景的闪回和噩梦 |
| **幸存者 guilt** | 加工"为什么不是我"的信念 |
| **优势** | 相对快速；不需要详细叙述 |
| **注意** | 需EMDR培训；确保稳定化 |

### 4.2 延长暴露（PE）

| 应用 | 说明 |
|------|------|
| **创伤叙述** | 详细写出/讲述死亡经历 |
| **体内暴露** | 面对回避的情境 |
| **哀伤调整** | 可在暴露中加入对逝者的积极回忆 |

### 4.3 CGT调整（针对创伤性哀伤）

| 调整 | 说明 |
|------|------|
| **更长的稳定化** | 创伤性哀伤者需要更长时间建立安全感 |
| **创伤处理整合** | 在CGT中加入创伤加工 |
| ** survivor guilt** | 专门的认知重构 |
| **注意 flashback** | 在治疗中可能触发 flashback → 需 grounding 技能 |

### 4.4 幸存者 guilt 干预

| 技术 | 说明 |
|------|------|
| **认知重构** | "活下来不是我的错""死亡原因是X，不是我" |
| **责任现实检验** | 患者实际有多大的控制/责任？ |
| **意义重建** | "我如何纪念逝者？""我的生命如何有意义？" |
| **幸存者使命** | 将幸存者身份转化为积极使命 |

## 五、特殊情境

### 5.1 大规模创伤（灾难、疫情）

| 特征 | 说明 |
|------|------|
| **集体创伤** | 社区共享创伤；可能缺乏个体化支持 |
| **资源不足** | 大量受影响者；服务 overwhelmed |
| **干预** | 群体干预；社区支持；筛查转诊 |

### 5.2 他杀（Homicide）

| 特征 | 司法程序延长哀伤；愤怒；对司法系统的不信任 |
|------|------------------------------------------|
| **特殊需求** | 处理愤怒（对凶手、对系统）；司法过程支持 |

### 5.3 战争/恐怖主义

| 特征 | 政治因素；集体创伤；多重丧失 |
|------|---------------------------|
| **干预** | 可能需要政治/社会层面的干预 |
