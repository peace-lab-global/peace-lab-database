# 延长哀伤障碍的临床管理 | Clinical Management of Prolonged Grief Disorder

> **目标**：系统阐述延长哀伤障碍（PGD）的循证心理治疗、药物治疗与复杂病例管理。
> **标准来源**：APA PGD临床手册（2022）、Shear et al. (2005) 复杂性哀伤治疗（CGT）、WHO指南。

---

## 一、治疗原则

### 1.1 总体原则

| 原则 | 说明 |
|------|------|
| **尊重哀伤** | 哀伤是正常的；治疗目标是帮助"适应性哀伤"而非消除悲伤 |
| **个体化** | 无统一时间表；尊重个人节奏 |
| **安全优先** | 评估自杀风险；危机干预优先 |
| **整合方法** | 通常需要心理社会干预为主；药物辅助 |

### 1.2 治疗目标

| 短期（4-8周） | 中期（8-16周） | 长期（>16周） |
|--------------|---------------|--------------|
| 建立治疗联盟 | 处理丧失相关情绪 | 重建身份和意义 |
| 安全评估 | 减少回避 | 恢复功能 |
| 心理教育 | 改善功能 | 预防复发 |
| 稳定化 | 重建社会连接 | 整合哀伤 |

## 二、复杂性哀伤治疗（Complicated Grief Treatment, CGT）

### 2.1 核心特征

| 特征 | 说明 |
|------|------|
| **证据等级** | A级（RCT显示优于标准IPT和抗抑郁药） |
| **疗程** | 16次个体会谈（每周） |
| **理论基础** | 依恋理论 + 双过程模型 |
| **核心** | 通过"重温（revisiting）"故事促进适应性哀伤 |

### 2.2 治疗结构

| 阶段 | 次数 | 内容 |
|------|------|------|
| **第一阶段：建立关系与心理教育** | 1-3 | 建立联盟；介绍双过程模型；设定目标 |
| **第二阶段：处理丧失** | 4-9 | 重温故事（imaginal revisiting）；情绪处理 |
| **第三阶段：恢复与重建** | 10-13 | 行为激活；重建活动；目标设定 |
| **第四阶段：意义与终结** | 14-16 | 意义重建；巩固收益； relapse 预防 |

### 2.3 核心技术

#### 重温故事（Imaginal Revisiting）

| 步骤 | 说明 |
|------|------|
| **准备** | 解释目的；确保安全感 |
| **叙述** | 详细讲述死亡的故事（包括当时的情境、感受） |
| **情感表达** | 鼓励表达被压抑的情绪 |
| **认知加工** | 挑战灾难化认知（"如果我做了X..."） |
| **重复** | 多次重复叙述直到情绪强度降低 |

#### 情境性 revisiting（Situational Revisiting）

| 步骤 | 说明 |
|------|------|
| **识别回避情境** | 患者回避的与丧失相关的地点/活动 |
| **层级暴露** | 建立回避情境的层级 |
| **逐步暴露** | 从最容易的开始；治疗师陪同 |
| **情感处理** | 在情境中体验和处理情绪 |

#### 纪念与持续联结

| 技术 | 说明 |
|------|------|
| **记忆访谈** | 详细回忆逝者的生平、关系、美好回忆 |
| ** Legacy 活动** | 以有意义的方式纪念逝者 |
| **持续联结仪式** | 创建个人化的纪念仪式 |

### 2.4 行为激活（针对恢复导向）

| 技术 | 说明 |
|------|------|
| **活动日程** | 计划愉快的和有成就感的活动 |
| **逐步恢复** | 从简单活动开始；逐步增加 |
| **社交重建** | 重新联系朋友；参加社交活动 |
| **新角色探索** | 探索丧失后的新身份和角色 |

## 三、其他心理治疗方法

### 3.1 人际治疗（IPT）

| 调整 | 说明 |
|------|------|
| **角色转换** | 丧亲作为"角色转换"问题 |
| **哀悼过程** | 促进哀悼任务的完成 |
| **支持系统** | 激活社会支持 |

### 3.2 认知行为治疗（CBT）

| 技术 | 应用 |
|------|------|
| **认知重构** | "我的生活永远无意义"→"我正在寻找新意义" |
| **行为实验** | 测试"参加活动不会减少对逝者的爱" |
| **暴露** | 对丧失记忆的逐步暴露 |

### 3.3 意义中心心理治疗（Meaning-Centered Psychotherapy）

| 来源 | 基于Frankl的意义疗法；针对癌症患者家属调整 |
|------|------------------------------------------|
| **核心** | 帮助患者发现丧亲后仍存在的意义 |
| **技术** | 意义日记、 legacy 工作、价值观澄清 |

### 3.4 家庭/伴侣治疗

| 适用 | 说明 |
|------|------|
| **共同丧失** | 家庭成员共同丧亲（如失去孩子） |
| **支持丧失者** | 帮助配偶支持丧亲的一方 |
| **注意** | 家庭成员可能哀伤节奏不同 → 冲突 |

## 四、药物治疗

### 4.1 总原则

| 原则 | 说明 |
|------|------|
| **心理社会干预优先** | 药物为辅助 |
| **共病导向** | 主要针对共病抑郁/焦虑/失眠 |
| **PGD特异性** | 目前无FDA批准专用于PGD的药物 |

### 4.2 共病抑郁

| 药物 | 说明 |
|------|------|
| **SSRI** | 一线（舍曲林、艾司西酞普兰） |
| **SNRI** | 文拉法辛、度洛西汀 |
| **注意** | 药物可能减轻症状但不一定解决哀伤核心；需配合心理治疗 |

### 4.3 共病焦虑/PTSD

| 药物 | 说明 |
|------|------|
| **SSRI** | 一线 |
| **哌唑嗪** | PTSD噩梦 |

### 4.4 失眠

| 药物 | 说明 |
|------|------|
| **曲唑酮** | 低剂量；共病抑郁时 |
| **多塞平** | 低剂量 |
| **褪黑素** | 轻度 |
| **避免** | BZD（依赖风险） |

### 4.5 研究中的药物

| 药物 | 机制 | 研究阶段 |
|------|------|---------|
| **Naltrexone** | 阿片拮抗剂（可能减弱分离痛苦） | 早期研究 |
| **MDMA辅助心理治疗** | 创伤性哀伤 | 临床试验 |

## 五、复杂病例管理

### 5.1 创伤性哀伤

| 特征 | 死亡为暴力/意外/可怖；侵入性记忆；回避 |
|------|--------------------------------------|
| **治疗** | CGT + EMDR/PE（针对PTSD症状） |
| **顺序** | 通常先稳定化；然后哀伤处理；必要时先PTSD治疗 |

### 5.2 自杀死亡

| 特征 | 羞耻、自责、"为什么"、被遗弃感、社会 stigma |
|------|------------------------------------------|
| **特殊需求** | 处理自责和"如果"；应对外部 stigma |
| **支持** | 自杀丧失支持团体尤为重要 |

### 5.3 儿童死亡

| 特征 | 最严重的哀伤之一；父母身份危机 |
|------|---------------------------|
| **特殊需求** | 婚姻压力（父母哀伤不同步）；对兄弟姐妹的关注 |
| **治疗** | 家庭治疗；夫妻治疗 |

### 5.4 分层治疗模型

| 层级 | 人群 | 干预 |
|------|------|------|
| **0：普遍预防** | 所有丧亲者 | 信息、资源、社区支持 |
| **1：选择性预防** | 高危丧亲者 | 筛查；简短干预；支持团体 |
| **2：针对性干预** | 亚临床显著 | 简短心理治疗；支持性咨询 |
| **3：强化治疗** | PGD诊断 | CGT；IPT；CBT |
| **4：专科/住院** | 严重/复杂 | 多学科团队；危机干预 |
