# 哀伤的循证干预总览 | Evidence-Based Interventions for Grief

> **目标**：系统阐述哀伤干预的循证基础，从普遍预防到强化治疗的分层模型。
> **标准来源**：APA PGD临床手册、Cochrane Reviews、Wittouck et al. (2011) 元分析。

---

## 一、干预效果总览

### 1.1 元分析发现

| 干预类型 | 效应量（d） | 证据等级 | 适用 |
|---------|-----------|---------|------|
| **复杂性哀伤治疗（CGT）** | 1.0-1.4 | A级 | PGD |
| **人际治疗（IPT）** | 0.6-0.8 | A级 | 共病抑郁的哀伤 |
| **认知行为治疗（CBT）** | 0.5-0.9 | B级 | 共病焦虑/PTSD |
| **EMDR** | 0.7-1.0 | B级 | 创伤性哀伤 |
| **意义中心治疗** | 0.5-0.7 | B级 | 意义丧失 |
| **支持团体** | 0.3-0.5 | C级 | 一般支持 |
| **普遍性预防** | 小或无 | — | 所有丧亲者 |
| **药物（抗抑郁药）** | 0.4-0.6 | C级 | 共病抑郁 |

**关键发现：**
- 对**普通丧亲者**的普遍性干预效果有限（哀伤是正常过程）
- 对**高危/临床人群**的针对性干预效果显著
- **CGT**是目前证据最强的PGD治疗

---

## 二、分层干预模型

### 2.1 公共卫生模型

| 层级 | 目标人群 | 干预 | 提供者 |
|------|---------|------|--------|
| **0：普遍预防** | 所有丧亲者 | 信息、资源、仪式、社区支持 | 社区、宗教 |
| **1：选择性预防** | 高危丧亲者 | 筛查、简短支持、支持团体 | 社工、护士 |
| **2：针对性治疗** | 亚临床显著 | 简短心理治疗、咨询 | 心理咨询师 |
| **3：强化治疗** | PGD诊断 | CGT、IPT、CBT | 临床心理师/精神科医生 |
| **4：专科/住院** | 严重/复杂/危机 | 多学科团队、危机干预 | 专科中心 |

### 2.2 筛查与分层

| 筛查工具 | 用途 | 转诊 |
|---------|------|------|
| **PG-13** | PGD筛查 | ≥30分→层级3 |
| **ICG** | 复杂性哀伤 | ≥25分→层级2-3 |
| **PHQ-9** | 抑郁筛查 | ≥15分→层级2-3 |
| **PCL-5** | PTSD筛查 | ≥33分→层级2-3 |
| **C-SSRS** | 自杀风险 | 阳性→紧急评估 |

## 三、普遍性支持（层级0-1）

### 3.1 普遍原则

| 原则 | 说明 |
|------|------|
| **陪伴** | 不试图"修复"；倾听和见证 |
| **正常化** | "你的反应是正常的" |
| **提供信息** | 哀伤的正常过程；资源 |
| **保持联系** | 丧亲后持续支持（不止葬礼时） |

### 3.2 支持团体

| 类型 | 适用 |
|------|------|
| **丧偶支持团体** | 丧偶者 |
| **父母丧失团体** | 失去孩子的父母 |
| **自杀丧失团体** | 自杀死亡幸存者 |
| **在线支持团体** | 无法参加面授者 |

### 3.3 仪式与文化

| 仪式 | 作用 |
|------|------|
| **葬礼** | 社会认可丧失；开始哀悼 |
| **周年纪念** | 持续的纪念；意义建构 |
| **个人仪式** | 私人的纪念方式 |

---

## 四、简短干预（层级2）

### 4.1 丧亲后简短咨询

| 形式 | 1-6次；结构化 |
|------|-------------|
| **内容** | 评估；心理教育；应对策略；资源链接 |
| **适用** | 功能损害轻度-中度 |

### 4.2 危机干预（Crisis Intervention）

| 原则 | 说明 |
|------|------|
| **安全优先** | 自杀风险评估 |
| **稳定化** | 减少急性痛苦 |
| **问题解决** | 处理即刻实际问题 |
| **资源链接** | 转介到适当服务 |

---

## 五、强化心理治疗（层级3）

### 5.1 复杂性哀伤治疗（CGT）

| 特征 | 说明 |
|------|------|
| **证据** | A级（ strongest evidence） |
| **疗程** | 16次 |
| **核心** | imaginal revisiting；行为激活 |
| **详情** | 见 `complicated-grief/Prolonged_Grief_Disorder_Management.md` |

### 5.2 人际治疗（IPT）

| 调整 | 丧亲作为"角色转换" |
|------|-------------------|
| **重点** | 哀悼过程；社会支持；新角色 |
| **疗程** | 12-16次 |

### 5.3 CBT调整

| 技术 | 应用 |
|------|------|
| **认知重构** | "无意义感""自责" |
| **行为激活** | 恢复活动 |
| **暴露** | 对丧失记忆的暴露 |

### 5.4 EMDR

| 适用 | 创伤性哀伤；侵入性记忆 |
|------|---------------------|
| **疗程** | 6-12次 |

---

## 六、药物治疗

### 6.1 原则

| 原则 | 说明 |
|------|------|
| **辅助地位** | 心理治疗为主 |
| **共病导向** | 针对共病抑郁/焦虑/失眠 |
| **无PGD专药** | 无FDA批准专用于PGD的药物 |

### 6.2 具体应用

| 共病 | 药物 |
|------|------|
| **中度-重度抑郁** | SSRI（舍曲林、艾司西酞普兰） |
| **焦虑** | SSRI；短期BZD（谨慎） |
| **PTSD** | SSRI；哌唑嗪（噩梦） |
| **失眠** | 曲唑酮；多塞平；褪黑素 |
| **严重激越** | 低剂量非典型抗精神病药（短期） |

---

## 七、数字干预与远程治疗

| 形式 | 说明 |
|------|------|
| **在线CGT** | 初步研究显示有效 |
| **哀伤APP** | 自助工具；日记；资源 |
| **视频治疗** | 远程CGT/IPT；尤其偏远地区 |
| **在线支持团体** | 可及性高；匿名 |
