# 哀伤理论与模型 | Theories and Models of Grief

> **目标**：系统阐述哀伤的主要理论模型，为临床干预提供理论基础。
> **标准来源**：Stroebe & Schut (2010)、Bowlby (1969)、Klass et al. (1996)、Neimeyer (2001)。

---

## 一、经典理论

### 1.1 Freud的哀悼工作（Mourning Work, 1917）

| 核心概念 | 说明 |
|---------|------|
| **哀悼工作** | 逐渐从逝者身上撤回情感能量（libido） |
| **过程** | 现实检验 → 接受逝者不在 → 释放情感能量 → 投入新关系 |
| **目标** | "放下"逝者；情感脱离 |
| **现代评价** | 过于线性；忽视了持续联结的价值 |

### 1.2 Bowlby的依恋理论（1969-1980）

| 阶段 | 特征 | 正常反应 |
|------|------|---------|
| **震惊与麻木** | 难以置信；情感麻木 | 数小时到数周 |
| **渴望与寻找** | 强烈思念；寻找逝者；愤怒/内疚 | 数周到数月 |
| **混乱与绝望** | 接受现实；抑郁；退缩 | 数月 |
| **重组** | 重建生活；新身份 | 数月到数年 |

**核心：** 哀伤是依恋系统的正常反应；与依恋类型相关（焦虑型→更强烈哀伤；回避型→情感压抑）。

### 1.3 Kübler-Ross的五个阶段（1969）

| 阶段 | 特征 | 注意 |
|------|------|------|
| **否认** | "这不可能发生" | 保护性缓冲 |
| **愤怒** | "为什么是我？" | 可指向医疗系统、逝者、自己 |
| **讨价还价** | "如果我当时..." | 试图改变结果 |
| **抑郁** | 绝望、退缩 | 需与临床抑郁鉴别 |
| **接受** | 整合丧失 | 非所有人经历所有阶段 |

**现代评价：** 描述性而非规定性；非线性；个体差异大；不应作为"必须完成"的清单。

## 二、现代双过程模型（Stroebe & Schut, 1995, 2010）

### 2.1 核心概念

哀伤者需要在两个方向之间**摆动（oscillation）**：

| 过程 | 关注 | 行为 |
|------|------|------|
| **丧失导向（Loss-Oriented）** | 逝者、死亡、情感 | 哀悼、思念、哭泣、回忆 |
| **恢复导向（Restoration-Oriented）** | 生活调整、新角色 | 工作、社交、新活动、身份重建 |

### 2.2 临床意义

| 适应不良模式 | 表现 | 干预 |
|------------|------|------|
| **过度丧失导向** | 沉浸在悲伤中；无法恢复功能 | 鼓励恢复活动；逐步增加功能 |
| **过度恢复导向** | 回避悲伤；过早"恢复正常" | 创造哀悼空间；允许情感表达 |
| **适应性摆动** | 在两者之间自然摆动 | 支持自然过程 |

### 2.3 摆动模式

```
丧失导向 ←——摆动——→ 恢复导向
   ↑                       ↑
悲伤、回忆、仪式          工作、社交、新身份
   ↑_______________________↓
            适应性哀伤
```

## 三、意义重建模型（Neimeyer, 2001）

### 3.1 核心概念

哀伤的核心任务是**重建意义系统**：

| 层面 | 丧失前 | 丧失后 |
|------|--------|--------|
| **意义** | 世界是有意义的、可预测的 | 世界变得无意义、不可预测 |
| **身份** | "我是某人的配偶/父母/孩子" | 身份空缺 |
| **信念** | 正义世界、控制感 | 信念动摇 |

### 3.2 意义重建过程

| 过程 | 说明 |
|------|------|
| ** sense making** | "为什么会发生？"——寻找解释 |
| ** benefit finding** | "从中我学到了什么？"——寻找意义 |
| ** identity change** | "我是谁 now？"——重建身份 |

### 3.3 临床技术

| 技术 | 方法 |
|------|------|
| **叙事重建** | 重写生命故事，纳入丧失 |
| **意义日记** | 记录从哀伤中发现的任何意义 |
| **价值观澄清** | 识别什么仍然重要 |
| ** Legacy work** | 创建 legacy 项目以纪念逝者 |

## 四、持续联结模型（Klass et al., 1996）

### 4.1 核心概念

**健康的哀伤不一定是"放手"，而是与逝者建立新的、持续的内在关系。**

| 形式 | 说明 |
|------|------|
| **内在对话** | 继续与逝者"对话" |
| **回忆整合** | 逝者的记忆融入日常生活 |
| **价值观传承** | 继承和践行逝者的价值观 |
| **纪念活动** | 定期纪念逝者 |
| **象征性联结** | 物品、地点作为联结载体 |

### 4.2 适应性的持续联结 vs 病理性联结

| 适应性 | 病理性 |
--------|--------|
| 联结带来安慰 | 联结导致痛苦 |
| 允许新关系 | 阻碍新关系 |
| 灵活调整 | 僵化、强迫性 |
| 与现实连接 | 脱离现实 |

## 五、依恋理论与哀伤（Stroebe et al.）

| 依恋类型 | 哀伤特征 | 干预要点 |
|---------|---------|---------|
| **安全型** | 自然哀伤过程；寻求支持；最终整合 | 一般支持即可 |
| **焦虑型** | 强烈、持久的悲伤；分离焦虑； preoccupation | 情绪调节；安全感建立 |
| **回避型** | 情感压抑；表面"坚强"；躯体化 | 温和鼓励情感表达 |
| **混乱型** | 矛盾、不可预测的哀伤反应 | 稳定的治疗关系；创伤处理 |

## 六、文化因素

| 维度 | 差异 |
|------|------|
| **个人主义文化** | 强调"放手"、个人独立、情感表达 |
| **集体主义文化** | 强调持续联结、家庭责任、仪式 |
| **宗教/灵性** | 影响死亡解释、来世信念、仪式 |
| **性别角色** | 文化对男性/女性哀悼方式的期望 |
