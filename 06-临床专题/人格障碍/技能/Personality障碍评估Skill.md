# 人格障碍评估 Agent Skill

> **核心定位**：结构化决策树驱动的人格障碍评估技能。从初步筛查到分类诊断、严重度评估和转介建议，提供完整的临床决策支持。

---

## 临床免责声明

> ⚠️ 本技能为**教育和决策支持工具**，不替代临床诊断。人格障碍诊断需要训练有素的专业人员进行结构化访谈。所有高风险情况（自杀/自伤/他人伤害风险）应立即转介至精神科急诊。

---

## 1. 评估决策树

```
START: 用户描述人格/关系/情绪困扰
│
├─ Step 1: 红旗筛查 ──────────────────────────┐
│   ├─ 当前自杀/自伤意念？ → 立即转介危机服务    │
│   ├─ 对他人暴力意念？ → 立即转介              │
│   ├─ 精神病性症状？ → 精神科急诊              │
│   └─ 无红旗 → 继续 Step 2                     │
│                                                │
├─ Step 2: 人格障碍筛查 ───────────────────────┐
│   ├─ IPDE (8项筛查)                           │
│   │   ├─ ≥4 阳性 → 可能 PD → Step 3         │
│   │   └─ <4 阳性 → 人格特质/非PD → 一般建议   │
│   └─ 临床判断（长期模式/多领域受损）            │
│                                                │
├─ Step 3: 分类与聚类 ─────────────────────────┐
│   ├─ A簇（古怪/偏执）→ 偏执/分裂样/分裂型     │
│   ├─ B簇（戏剧/不稳定）→ BPD/NPD/ASPD/HPD    │
│   ├─ C簇（焦虑/恐惧）→ AvPD/DPD/OCPD         │
│   └─ 跨簇特征 → 考虑 AMPD 维度评估            │
│                                                │
├─ Step 4: 严重度评估 ─────────────────────────┐
│   ├─ LPFS 人格功能评估                        │
│   │   ├─ Level 0-1: 无/轻度 → 人格特质       │
│   │   ├─ Level 2: 中度 → 人格障碍            │
│   │   ├─ Level 3: 重度 → 严重 PD             │
│   │   └─ Level 4: 极重度 → 极严重            │
│   └─ 功能领域：自我/人际/职业/社会             │
│                                                │
└─ Step 5: 转介建议 ──────────────────────────┐
    ├─ 轻度 → 心理教育+自我帮助+随访            │
    ├─ 中度 → 专科评估+结构化治疗（DBT/Schema） │
    ├─ 重度 → 专科+药物+多学科团队              │
    └─ 极重度 → 住院/日间项目+密集治疗          │
```

## 2. 筛查工具库

| 工具 | 项目数 | 时长 | 用途 |
|:-----|:------:|:----:|:-----|
| IPDE | 8 题 | 2 min | 快速 PD 筛查 |
| SAPAS | 8 题 | 2 min | 初级保健 PD 筛查 |
| SCID-5-PD | 106 题 | 60 min | 结构化诊断访谈 |
| PID-5 (220) | 220 题 | 30 min | 特质评估 |
| PID-5-SF (25) | 25 题 | 5 min | 快速特质筛查 |
| LPFS-BF | 12-80 题 | 5-15 min | 功能水平评估 |

## 3. 关键鉴别问题

| 鉴别 | 关键问题 |
|:-----|:---------|
| PD vs 轴 I 障碍 | 模式是否**持久且跨情境**？（≥2 领域） |
| BPD vs 双相 | 情绪波动是**小时级**（BPD）还是**天-周级**（双相）？ |
| BPD vs CPTSD | 有无**关系不稳定/认同紊乱/冲动**？（BPD 更显著） |
| NPD vs 双相躁狂 | 夸大是**持续特质**（NPD）还是**发作性**（躁狂）？ |
| AvPD vs SAD | 恐惧是**被拒绝**（AvPD）还是**尴尬**（SAD）？ |
| ASPD vs 物质滥用 | 反社会行为是否在**物质使用之前**就存在？ |

---

## 4. 转介分级标准

| 级别 | 标准 | 建议 |
|:----:|:-----|:-----|
| 1（自助） | 人格特质，功能完好 | 心理教育/自助书籍 |
| 2（初级） | 轻度 PD，功能轻度受损 | 一般心理咨询/CBT |
| 3（专科） | 中-重度 PD，功能明显受损 | 结构化 PD 治疗（DBT/Schema/MBT/TFP） |
| 4（多学科） | 严重 PD，共病/高风险 | 精神科+心理治疗+社工+药物 |
| 5（紧急） | 急性自杀/自伤/暴力风险 | 急诊/危机干预/住院 |
