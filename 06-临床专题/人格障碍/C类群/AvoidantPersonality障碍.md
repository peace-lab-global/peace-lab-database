# 回避型人格障碍 (Avoidant Personality Disorder, AvPD)

> **核心定位**：以社交抑制、不适当感和对负面评价的过度敏感为特征。患病率 2.4%，与社会焦虑障碍高度共病但概念不同。

---

## 1. DSM-5 诊断标准（≥4/7）

| # | 标准 |
|:--|:-----|
| 1 | 因害怕批评/不赞成/拒绝而回避涉及人际接触的职业活动 |
| 2 | 不愿与人打交道，除非确定会被喜欢 |
| 3 | 因害羞/尴尬/被嘲笑而在亲密关系中有所保留 |
| 4 | 在社交场合全神贯注于被批评或被拒绝 |
| 5 | 因不适当感而在新的人际关系中受到抑制 |
| 6 | 认为自己社交无能/个人魅力不足/不如他人 |
| 7 | 因可能令人尴尬而特别不愿冒个人风险或从事新活动 |

---

## 2. AvPD vs 社交焦虑障碍 (SAD)

| 维度 | AvPD | SAD |
|:-----|:-----|:----|
| 核心恐惧 | 被拒绝/不被喜欢 | 尴尬/丢脸 |
| 自我概念 | 内在缺陷感/自卑 | 对特定情境的恐惧 |
| 回避范围 | 广泛（所有亲密关系） | 特定（演讲/聚会） |
| 起始 | 早年起病 | 青春期 |
| 治疗难度 | 更难（人格结构） | 中等（CBT 效果好） |
| 共病率 | ~43% 共病 SAD | ~14% 共病 AvPD |

---

## 3. 治疗

| 方法 | 说明 |
|:-----|:-----|
| CBT | 认知重建+渐进暴露+社交技能训练 |
| 图式疗法 | 处理"缺陷/羞耻"和"社会隔离"图式 |
| 心理动力学 | 探索早期拒绝经历的内在表征 |
| 团体治疗 | 安全环境中的社交练习（需筛选） |
| 药物 | SSRIs/SNRIs（靶症状：焦虑/抑郁） |
