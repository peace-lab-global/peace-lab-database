# 人格障碍 DSM-5 分类总览

> **核心定位**：人格障碍是持久性、弥漫性的内在体验和行为模式偏离文化期望，导致功能损害或主观痛苦。DSM-5 列出了 10 种特定类型，分为 A/B/C 三簇。

---

## 1. DSM-5 十型分类

### 1.1 A 簇：奇异/古怪型 (Odd/Eccentric)

| 障碍 | 核心特征 | 患病率 | 关键鉴别 |
|:-----|:---------|:------:|:---------|
| 偏执型 (PPD) | 普遍不信任/多疑 | 2.3-4.4% | vs 精神分裂症/妄想障碍 |
| 分裂样 (SzPD) | 社交脱离/情感平淡 | 3.1-4.9% | vs 自闭谱系/抑郁 |
| 分裂型 (StPD) | 认知/感知扭曲+古怪行为 | 3.9-4.6% | vs 精神分裂症（轻度） |

### 1.2 B 簇：戏剧化/情绪化/不稳定型

| 障碍 | 核心特征 | 患病率 | 关键鉴别 |
|:-----|:---------|:------:|:---------|
| **边缘型 (BPD)** | 不稳定关系/自我形象/情绪 + 冲动 | 1.6-5.9% | vs 双相/CPTSD/ADHD |
| **自恋型 (NPD)** | 夸大/需要赞美/缺乏共情 | 0.5-6.2% | vs 双相躁狂/ASPD |
| **反社会型 (ASPD)** | 无视/侵犯他人权利 | 0.6-3.6% | vs 物质滥用/犯罪（非障碍） |
| 表演型 (HPD) | 过度情绪化/寻求关注 | 1.8-2.8% | vs BPD/NPD |

### 1.3 C 簇：焦虑/恐惧型

| 障碍 | 核心特征 | 患病率 | 关键鉴别 |
|:-----|:---------|:------:|:---------|
| **回避型 (AvPD)** | 社交抑制/不足感/对拒绝过敏 | 2.5-5.2% | vs 社交焦虑障碍 |
| **依赖型 (DPD)** | 过度需要被照顾/顺从/分离恐惧 | 0.6-1.5% | vs BPD/抑郁 |
| 强迫型 (OCPD) | 秩序/完美主义/控制（非OCD） | 2.1-7.9% | vs OCD |

## 2. 流行病学

| 指标 | 数值 |
|:-----|:-----|
| 一般人群任何PD患病率 | 9-13% |
| 精神科门诊 | 30-50% |
| 精神科住院 | 50-70% |
| 法医人群 | 60-80% |
| 共病率（PD+轴I障碍） | 50-70% |
| 自杀风险（BPD） | 8-10%（终生） |

---

## 3. 病因学模型

### 3.1 生物-心理-社会模型

| 因素 | 说明 | 证据 |
|:-----|:-----|:-----|
| **遗传** | 遗传率 40-60%（因类型不同） | 双生子研究 |
| **气质** | 高神经质/低宜人性/冲动性 | 纵向追踪 |
| **童年创伤** | 虐待/忽视/丧失（BPD 尤其强） | 大量证据 |
| **依恋** | 不安全/紊乱依恋 | 中-强 |
| **家庭环境** | 无效化/高冲突/过度保护 | 中 |
| **社会文化** | 社会变迁/城市化/文化适应 | 弱-中 |

### 3.2 Linehan 生物社会理论（BPD 专用）

```
生物易感性（高情绪敏感 + 慢恢复）
        +
无效化环境（情绪不被认可/惩罚）
        ↓
情绪调节障碍
        ↓
BPD 症状群
```

## 4. 评估框架

### 4.1 人格评估工具

| 工具 | 类型 | 覆盖 | 时间 |
|:-----|:-----|:-----|:----:|
| SCID-5-PD | 结构化访谈 | DSM-5 全部10型 | 60-90分钟 |
| DAPP-BQ | 自评问卷 | 18维度人格特质 | 30分钟 |
| PID-5 | 自评问卷 | AMPD 25特质 | 30分钟 |
| MSI-BPD | BPD筛查 | BPD | 5分钟 |
| PCL-R | 评估 | 精神病态/ASPD | 90-120分钟 |
| IPDE | 结构化访谈 | ICD/DSM | 60分钟 |

### 4.2 人格障碍 vs 人格特质

| 维度 | 人格特质 | 人格障碍 |
|:-----|:---------|:---------|
| 情境灵活性 | 高（不同情境不同表现） | 低（跨情境一致） |
| 适应功能 | 无损害 | 功能损害 |
| 主观痛苦 | 无 | 有（或他人痛苦） |
| 持续性 | 可变 | 长期稳定（≥成年早期） |
| 文化偏差 | 在文化范围内 | 偏离文化期望 |

## 5. 治疗原则

| 原则 | 说明 |
|:-----|:-----|
| **BPD 可治疗** | 过去认为不可治→现有多项循证治疗 |
| **长程治疗** | 通常需 1-3 年 |
| **治疗联盟优先** | 联盟质量是疗效最强预测因子 |
| **共病管理** | 轴I共病需同时处理 |
| **危机管理** | BPD 自杀/自伤需明确方案 |
| **反移情管理** | 治疗师需持续督导 |
