# MBT 心智化疗法 (Mentalization-Based Treatment)

> **核心定位**：Anthony Bateman 和 Peter Fonagy 开发，聚焦于**心智化能力**（理解自己和他人心理状态的能力）的培养。BPD 一线循证治疗之一。

---

## 1. 核心理论

### 1.1 心智化 (Mentalization) 定义

> "在心理状态的语境中理解自己和他人行为的能力"——即 **thinking about thinking**。

| 维度 | 说明 |
|:-----|:-----|
| **对象** | 自我心智化 / 他人心智化 |
| **模式** | 自动（直觉）/ 控制（深思） |
| **焦点** | 内在（想法/感受）/ 外在（行为/表情） |
| **时间** | 过去 / 现在 / 未来 |

### 1.2 心智化失败的前心智化模式

| 模式 | 描述 | BPD 中的表现 |
|:-----|:-----|:------------|
| **等价模式 (Psychic Equivalence)** | 内心=现实（"我感觉如此，所以一定如此"） | 偏执/确信被抛弃 |
| **假装模式 (Pretend Mode)** | 内心与现实完全分离 | 解离/虚假叙述 |
| **目的论模式 (Teleological)** | 只有可观察的行为才算 | "如果你真的在乎我，你就应该..." |

### 1.3 心智化与依恋

```
安全依恋 → 被理解的经验 → 发展心智化能力
不安全依恋 → 不被理解/创伤 → 心智化受损
        ↓
BPD：依恋系统激活时（分离/冲突）→ 心智化崩溃 → 前心智化模式
```

---

## 2. 治疗结构

| 组件 | 频率 | 时长 | 功能 |
|:-----|:-----|:-----|:-----|
| **个体治疗** | 每周1次 | 50分钟 | 探索心智化困难/依恋模式 |
| **团体治疗** | 每周1次 | 75分钟 | 在人际互动中练习心智化 |
| 总疗程 | — | 12-18月 | — |

### 2.1 MBT 治疗师姿态

| 原则 | 说明 |
|:-----|:-----|
| **不知道立场 (Not-knowing)** | 好奇/探索/不假设知道患者想法 |
| **心智化姿态** | 关注"发生了什么"而非"该做什么" |
| **情感聚焦** | 优先处理情感而非内容 |
| **此时此地** | 聚焦当下体验/治疗关系中的心智化 |
| **暂停与倒带** | 在心智化失败处暂停/回到之前 |

---

## 3. 适应症与证据

| 研究 | 发现 |
|:-----|:-----|
| Bateman & Fonagy, 1999/2001 | MBT vs 常规：减少自杀/住院/症状 |
| Bateman & Fonagy, 2009 (8.5年随访) | MBT 长期效果持续 |
| Jørgensen et al., 2013 | MBT vs 支持性：MBT 更优 |
