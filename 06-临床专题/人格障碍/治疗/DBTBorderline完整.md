# DBT 辩证行为疗法完整体系 (Dialectical Behavior Therapy)

> **核心定位**：Marsha Linehan 开发的多组件治疗，BPD 的证据最强心理治疗（>30 RCT）。标准 DBT 包含四个必要组件：个体治疗 + 技能团体 + 电话教练 + 咨询团队会。

---

## 1. 生物社会理论

```
生物易感性（高情绪敏感 + 高情绪强度 + 慢恢复基线）
        +
无效化环境（情绪不被认可/惩罚/忽视）
        ↓
情绪调节缺陷 → 无法耐受/调节强烈情绪
        ↓
┌─────────────────────────────────────────┐
│ BPD 症状群：                              │
│ · 情绪不稳定 → 关系不稳定                 │
│ · 冲动控制差 → 自伤/物质/暴食            │
│ · 自我形象不稳 → 身份紊乱                 │
│ · 认知紊乱 → 解离/偏执（压力下）          │
│ · 人际混乱 → 害怕被抛弃                   │
└─────────────────────────────────────────┘
```

---

## 2. 标准 DBT 四组件

| 组件 | 频率 | 时长 | 功能 |
|:-----|:-----|:-----|:-----|
| **个体治疗** | 每周1次 | 50分钟 | 动机增强/行为链分析/危机处理 |
| **技能团体** | 每周1次 | 2-2.5小时 | 四模块技能教学 |
| **电话教练** | 按需 | 10-20分钟 | 危机时实时技能应用指导 |
| **咨询团队会** | 每周1次 | 1-1.5小时 | 治疗师支持/个案讨论/ adherence |

---

## 3. 四模块技能

### 3.1 模块概览

| 模块 | 核心目标 | 关键技能 | 课时 |
|:-----|:---------|:---------|:----:|
| **正念 (Mindfulness)** | 觉察当下/不评判 | "What"技能(观察/描述/参与)+"How"技能(不评判/聚焦/有效) | 2-3周 |
| **人际效能 (Interpersonal)** | 有效沟通/边界 | DEAR MAN/GIVE/FAST/关系有效性 | 6-8周 |
| **情绪调节 (Emotion Regulation)** | 识别/改变情绪 | 情绪功能/ABC PLEASE/相反行动/问题解决 | 6-8周 |
| **痛苦耐受 (Distress Tolerance)** | 不恶化地承受 | TIPP/STOP/自我安抚/利弊分析/RADICAL接受 | 6-8周 |

### 3.2 关键技能详解

| 技能 | 缩写 | 内容 | 应用场景 |
|:-----|:-----|:-----|:---------|
| **STOP** | S-T-O-P | Stop→Take a step back→Observe→Proceed mindfully | 冲动来临时 |
| **TIPP** | T-I-P-P | Temperature(Ice)/Intense exercise/Paced breathing/Paired muscle relaxation | 极度情绪痛苦 |
| **DEAR MAN** | 7步 | Describe/Express/Assert/Reinforce + Mindful/Appear confident/Negotiate | 需求表达 |
| **GIVE** | 4步 | Gentle/Interested/Validate/Easy manner | 维护关系 |
| **FAST** | 4步 | Fair/No Apologies/Stick to values/Truthful | 维护自尊 |
| **PLEASE** | 6项 | Physical illness/treat/balanced Eating/avoid mood-Altering/Sleep/Exercise | 生理基础 |
| **相反行动** | — | 识别情绪→识别行动冲动→做相反的事 | 不合理情绪 |
| **冲浪冲动** | — | 观察冲动的升起、高峰、消退（不行动） | 自伤/暴食冲动 |
| **激进接受** | — | 完全接受现实（非放弃） | 无法改变的痛苦 |

## 4. 个体治疗结构

### 4.1 治疗目标层级

| 层级 | 目标 | 优先级 |
|:-----|:-----|:------:|
| **Stage 0** | 治疗承诺/减少脱落 | 前提 |
| **Stage 1** | 行为控制（自杀→自伤→治疗干扰→生活质量） | 最高 |
| **Stage 2** | 情绪体验（创伤处理/情绪调节深化） | 高 |
| **Stage 3** | 建立值得过的生活（自尊/关系/目标） | 中 |
| **Stage 4** | 完整性/超越性（灵性/连接/意义） | 可选 |

### 4.2 行为链分析 (Chain Analysis)

```
触发事件 → 脆弱因素 → 链式反应（想法→情绪→身体→行为）
                                                   ↓
                                              问题行为
                                                   ↓
                                          后果（短期/长期）
```

每次问题行为后详细分析链条→识别干预点→制定替代策略

## 5. 疗效证据

| 研究 | 发现 |
|:-----|:-----|
| Linehan et al., 1991 | DBT vs TAU: 减少自杀企图/住院/愤怒 |
| Linehan et al., 2006 | DBT vs CTBE: 自杀企图减半 |
| McMain et al., 2009 | DBT vs GPM: 等效 |
| Cristea et al., 2017 (Meta) | DBT 优于 TAU（自伤/自杀/住院） |
| Stoffers et al., 2012 (Cochrane) | DBT 有显著效果 |
