# 自恋型人格障碍 (Narcissistic Personality Disorder, NPD)

> **核心定位**：以夸大的自我重要感、需要赞美、缺乏共情为特征。临床就诊率低（自我协调性高），但对他人的影响可能显著。治疗极具挑战性。

---

## 1. 诊断标准（DSM-5，≥5/9）

| # | 标准 |
|:--|:-----|
| 1 | 夸大的自我重要感 |
| 2 | 沉湎于无限成功/权力/才华/美貌的幻想 |
| 3 | 相信自己是"特殊的"，只能被特殊人/机构理解 |
| 4 | 需要过度赞美 |
| 5 | 权利感（不合理地期望特殊待遇） |
| 6 | 人际关系中的剥削性 |
| 7 | 缺乏共情 |
| 8 | 常常嫉妒他人或认为他人嫉妒自己 |
| 9 | 傲慢/高高在上的态度或行为 |

---

## 2. 亚型区分

| 维度 | 夸大自恋 (Grandiose) | 脆弱自恋 (Vulnerable) |
|:-----|:---------------------|:----------------------|
| 外表 | 自信/外向/支配 | 内向/敏感/防御 |
| 自尊 | 膨胀但不稳定 | 低/脆弱/波动 |
| 对批评反应 | 愤怒/蔑视/反击 | 退缩/羞耻/抑郁 |
| 人际风格 | 支配/剥削/魅力 | 回避/被动攻击/怨恨 |
| 共情 | 缺乏（不在意他人） | 缺乏（过度自我关注） |
| 就诊原因 | 关系/职业危机 | 抑郁/焦虑 |
| 治疗困难 | 不认为有问题 | 过度脆弱/不信任 |

---

## 3. 流行病学与共病

| 指标 | 数值 |
|:-----|:-----|
| 患病率 | 0.5-6.2% |
| 男:女比 | 男性略多（但差异不大） |
| 共病抑郁 | 40-60% |
| 共病物质使用 | 25-40% |
| 共病 BPD | 15-25% |
| 共病双相 | 5-15% |

---

## 4. 治疗挑战

| 挑战 | 说明 |
|:-----|:-----|
| **低求助率** | 通常因共病（抑郁/关系问题）而非 NPD 本身求助 |
| **治疗联盟** | 极难建立（理想化→贬低治疗师） |
| **动机** | 低（NPD 高度自我协调） |
| **脱落率** | 高 |
| **自杀风险** | 自恋受伤(narcissistic injury)后可急性升高 |

### 4.1 治疗取向

| 取向 | 代表 | 方法 |
|:-----|:-----|:-----|
| **移情聚焦 (TFP)** | Kernberg | 分析移情中的自恋结构 |
| **自体心理学** | Kohut | 共情沉浸/镜映/理想化 |
| **图式疗法** | Young | 处理"权利/夸大"图式+早期剥夺 |
| **MBT** | Bateman | 心智化他人/理解影响 |
| **CBT** | Beck | 挑战全或无思维/共情训练 |

---

## 5. 鉴别诊断

| 鉴别 | NPD vs | 关键区分 |
|:-----|:-------|:---------|
| BPD | BPD | NPD 无自伤/无关系不稳定 |
| 双相躁狂 | 躁狂 | NPD 是特质（持续）非发作性 |
| ASPD | ASPD | NPD 不一定有违法行为 |
| 正常自信 | 高自尊 | NPD 需要持续外部验证 |
