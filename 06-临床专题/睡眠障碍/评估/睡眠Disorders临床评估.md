# 睡眠障碍临床评估 | Clinical Assessment of Sleep-Wake Disorders

> **目标**：建立精神科标准的睡眠障碍评估体系，涵盖病史采集、量表工具、客观监测与鉴别诊断。
> **标准来源**：AASM Clinical Practice Guidelines、APA Practice Guidelines、ISHLT共识。

---

## 一、睡眠病史采集框架（BEDSIDE评估）

| 字母 | 内容 | 关键问题 |
|------|------|---------|
| **B** | Bedtime habits | 就寝时间、入睡潜伏期、睡前活动 |
| **E** | Environment | 卧室环境（光、噪音、温度、床用途） |
| **D** | Daytime functioning | 日间嗜睡、疲劳、注意力、情绪 |
| **S** | Sleep schedule | 工作日vs周末差异、轮班工作、时区变化 |
| **I** | Interruptions | 觉醒次数、原因（排尿、疼痛、呼吸暂停） |
| **D** | Duration & Depth | 总睡眠时间、主观睡眠质量 |
| **E** | Extra information | 打鼾、腿动、梦话、睡眠行为、药物/物质 |

## 二、标准化评估量表

### 2.1 失眠评估

| 量表 | 项目 | 用途 | 临界值 |
|------|------|------|--------|
| **ISI（失眠严重指数量表）** | 7项 | 失眠严重程度、治疗反应 | ≥8分：亚临床；≥15分：中重度 |
| **PSQI（匹兹堡睡眠质量指数）** | 19项/7成分 | 主观睡眠质量综合评估 | >5分：睡眠质量差 |
| **AIS（雅典失眠量表）** | 8项 | 失眠诊断筛查 | ≥6分：疑似失眠 |
| **DBAS（睡眠不良信念与态度量表）** | 30项 | 识别认知扭曲 | 高分提示需CBT-I |
| **FIRST（ Ford失眠应激反应测试）** | 9项 | 预测失眠易感性 | — |

### 2.2 日间嗜睡评估

| 量表 | 项目 | 用途 | 临界值 |
|------|------|------|--------|
| **ESS（Epworth嗜睡量表）** | 8项 | 日间嗜睡程度 | ≥10分：异常嗜睡；≥16分：严重嗜睡 |
| **SSS（Stanford嗜睡量表）** | 1项 | 即刻警觉状态 | — |
| **KSS（Karolinska嗜睡量表）** | 1项 | 即刻困倦程度 | — |

### 2.3 不宁腿综合征

| 量表 | 项目 | 用途 | 临界值 |
|------|------|------|--------|
| **IRLS（国际RLS严重程度量表）** | 10项 | RLS严重程度 | 1-10轻度；11-20中度；21-30重度；31-40极重度 |
| **RLS-QOL** | 12项 | RLS生活质量影响 | — |

### 2.4 睡眠呼吸障碍筛查

| 工具 | 用途 | 说明 |
|------|------|------|
| **STOP-BANG问卷** | OSA筛查 | ≥3分：高危 |
| **Berlin问卷** | OSA筛查 | 高风险：2个类别阳性 |
| **NoSAS评分** | OSA筛查 | 颈围、BMI、性别、年龄、打鼾 |

### 2.5 其他专用量表

| 量表 | 用途 |
|------|------|
| **PSG/Actigraphy** | 客观睡眠监测 |
| **MSLT** | 发作性睡病诊断 |
| **MWT** | 维持觉醒能力（职业评估） |
| **PDSS（儿童日间嗜睡量表）** | 儿童嗜睡 |
| **CSHQ（儿童睡眠习惯问卷）** | 儿童睡眠问题 |

## 三、客观睡眠监测

### 3.1 多导睡眠监测（PSG）

| 参数 | 正常范围 | 临床意义 |
|------|---------|---------|
| **总睡眠时间（TST）** | 成人6-9h | <6h：睡眠不足 |
| **睡眠效率** | >85% | <85%：睡眠碎片化 |
| **入睡潜伏期（SOL）** | <30min | >30min：入睡困难 |
| **觉醒后入睡潜伏期（WASO）** | <20min | >20min：睡眠维持困难 |
| **N1占比** | 5-10% | >10%：睡眠浅 |
| **N3占比** | 15-25% | <10%：深睡不足 |
| **REM占比** | 20-25% | 异常提示REM问题 |
| **REM潜伏期** | 70-110min | <50min：REM前置（抑郁/发作性睡病） |
| **AHI** | <5/h | ≥5/h：OSA |
| **PLM指数** | <5/h | ≥5/h：PLMD |
| **最低SpO₂** | >90% | <90%：低氧血症 |

### 3.2 体动记录仪（Actigraphy）

| 用途 | 优势 |
|------|------|
| 昼夜节律评估 | 客观记录睡眠-觉醒模式（1-2周） |
| 失眠辅助诊断 | 区分睡眠不足与床上清醒 |
| 治疗效果监测 | 客观记录治疗前后变化 |
| 特殊人群 | 儿童、痴呆患者（无法配合PSG） |

### 3.3 家庭睡眠呼吸监测（HSAT）

| 适用 | 不适用 |
|------|--------|
| 中重度OSA疑似 | 合并心肺疾病 |
| 无复杂共病 | 神经肌肉疾病 |
| 无法到睡眠实验室 | 中枢性睡眠呼吸暂停疑似 |

## 四、鉴别诊断流程

### 4.1 失眠鉴别诊断树

```
主诉失眠
│
├─ 是否有充足睡眠机会？
│   └─ 否 → 睡眠不足综合征（行为性）
│
├─ 是否物质/药物相关？
│   └─ 是 → 物质所致睡眠障碍
│
├─ 是否躯体疾病相关？
│   ├─ 疼痛 → 疼痛相关失眠
│   ├─ 呼吸问题 → OSA/哮喘/COPD
│   ├─ 内分泌 → 甲减/甲亢/更年期
│   ├─ 神经 → 不宁腿/帕金森
│   └─ 是 → 躯体疾病所致失眠
│
├─ 是否精神障碍相关？
│   ├─ 抑郁 → 早醒、入睡困难
│   ├─ 焦虑 → 入睡困难、夜间觉醒
│   ├─ 躁狂 → 睡眠需求减少
│   ├─ PTSD → 噩梦、警觉性增高
│   ├─ 精神病 → 睡眠-觉醒紊乱
│   └─ 是 → 精神障碍所致失眠（仍需独立诊断失眠障碍）
│
├─ 是否昼夜节律问题？
│   ├─ 延迟睡眠相位 → 晚睡晚起
│   ├─ 提前睡眠相位 → 早睡早醒
│   ├─ 轮班工作 → 轮班相关
│   └─ 是 → 昼夜节律睡眠-觉醒障碍
│
└─ 无上述原因 → 原发性/慢性失眠障碍
```

## 五、实验室检查

| 检查 | 适应症 |
|------|--------|
| **甲状腺功能（TSH/T3/T4）** | 所有慢性失眠；RLS；嗜睡 |
| **铁蛋白** | RLS（<75 ng/mL提示铁缺乏） |
| **血糖/HbA1c** | 肥胖+打鼾（代谢综合征筛查） |
| **肾功能** | RLS；不宁腿可能与肾衰竭相关 |
| **肝功能** | 肝功能不全影响药物代谢 |
| **药物浓度** | 使用影响睡眠的药物时 |
| **CSF下丘脑分泌素** | 发作性睡病1型确诊 |
| **HLA-DQB1*06:02** | 发作性睡病支持性检查 |
