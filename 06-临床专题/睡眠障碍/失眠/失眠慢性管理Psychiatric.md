# 慢性失眠障碍的精神科管理 | Chronic Insomnia Disorder: Psychiatric Management

> **目标**：建立慢性失眠障碍的综合精神科管理方案，整合CBT-I、药物治疗与共病管理。
> **标准来源**：AASM Clinical Practice Guidelines（2017）、ACP Guidelines（2016）、APA Practice Guidelines。

---

## 一、治疗原则

### 1.1 阶梯治疗模型

| 阶梯 | 干预 | 证据等级 |
|------|------|---------|
| **第一阶梯** | CBT-I（认知行为治疗失眠） | A级（一线） |
| **第二阶梯** | 短效非苯二氮卓类催眠药（Z-drugs） | B级（短期/辅助） |
| **第三阶梯** | 褪黑素受体激动剂（雷美替胺） | B级（特定人群） |
| **第四阶梯** | 低剂量镇静抗抑郁药（曲唑酮、多塞平） | C级（共病抑郁/焦虑） |
| **第五阶梯** | 食欲素受体拮抗剂（苏沃雷生等） | A级（2019后FDA批准） |

### 1.2 治疗目标

| 短期（4-8周） | 中期（3-6月） | 长期（>6月） |
|--------------|--------------|--------------|
| 缩短入睡潜伏期 | 维持睡眠改善 | 预防复发 |
| 减少夜间觉醒 | 减少药物依赖 | 维持CBT-I技能 |
| 改善睡眠质量主观感受 | 恢复日间功能 | 持续改善生活质量 |
| 建立规律作息 | 处理残留症状 | 最小化药物使用 |

## 二、CBT-I核心成分

### 2.1 五成分模型

| 成分 | 目标 | 方法 |
|------|------|------|
| **刺激控制** | 重建床与睡眠的条件反射 | 仅困倦时上床；20分钟未入睡离开床；固定起床时间 |
| **睡眠限制** | 提高睡眠效率 | 限制卧床时间=实际睡眠时间；逐步增加 |
| **认知重构** | 改变不良睡眠信念 | 识别和挑战灾难化思维、过度担忧 |
| **放松训练** | 降低睡前生理唤起 | 渐进肌肉放松、腹式呼吸、正念 |
| **睡眠卫生** | 优化睡眠环境 | 光、温度、噪音、咖啡因、酒精、运动时机 |

### 2.2 睡眠限制疗法步骤

**步骤1：记录睡眠日记1-2周**
- 计算平均总睡眠时间（TST）
- 确定固定起床时间

**步骤2：设定初始睡眠窗口**
- 卧床时间 = TST（最少5小时）
- 例如：平均睡眠5.5小时 → 就寝12:30，起床6:00

**步骤3：每周调整**
- 睡眠效率 >85%：增加卧床时间15-30分钟
- 睡眠效率 80-85%：维持当前窗口
- 睡眠效率 <80%：减少卧床时间15-30分钟

**步骤4：逐步达到理想睡眠时长**
- 目标：7-8小时卧床时间

## 三、药物治疗

### 3.1 一线药物：食欲素受体拮抗剂（DORA）

| 药物 | 剂量 | 半衰期 | 特点 |
|------|------|--------|------|
| **苏沃雷生（Suvorexant）** | 10-20mg | 12h | 首个DORA；可次日残留镇静 |
| **莱博雷生（Lemborexant）** | 5-10mg | 17h | 可次日残留；食欲素1/2双拮抗 |
| **达利雷生（Daridorexant）** | 25-50mg | 8h | 半衰期较短；较少次日效应 |

**优势：**
- 非GABA-A机制（不抑制呼吸）
- 无耐受/依赖/反跳证据
- 改善睡眠维持
- 可用于OSA合并失眠

**注意：**
- 次日嗜睡（尤其高剂量）
- 抑郁患者慎用（食欲素系统与动机/奖赏相关）
- 驾驶/操作机械警告

### 3.2 苯二氮卓受体激动剂（BZRAs）

| 分类 | 药物 | 剂量 | 半衰期 | 适用 |
|------|------|------|--------|------|
| **非苯二氮卓类（Z-drugs）** | 唑吡坦 | 5-10mg | 2.5h | 入睡困难 |
| | 唑吡坦CR | 6.25-12.5mg | 2.8h | 入睡+维持 |
| | 右佐匹克隆 | 1-3mg | 6h | 入睡+维持 |
| | 扎来普隆 | 5-10mg | 1h | 入睡困难；可半夜服用 |
| **苯二氮卓类** | 艾司唑仑 | 1-2mg | 10-24h | 维持困难 |
| | 替马西泮 | 7.5-30mg | 8-15h | 维持困难 |
| | 三唑仑 | 0.125-0.25mg | 1.5-5h | 仅限短期（反跳风险） |

**限制：**
- 仅短期使用（一般≤4周）
- 耐受、依赖、戒断风险
- 呼吸抑制（OSA禁用）
- 老年人跌倒/认知损害风险

### 3.3 褪黑素与褪黑素受体激动剂

| 药物 | 剂量 | 适应症 | 证据 |
|------|------|--------|------|
| **褪黑素（OTC）** | 0.5-5mg | 昼夜节律问题；老年人 | C级 |
| **雷美替胺（Ramelteon）** | 8mg | 入睡困难；无依赖风险 | B级 |
| **他司美琼（Tasimelteon）** | 20mg | 非24小时睡眠-觉醒障碍 | A级 |

### 3.4 低剂量镇静抗抑郁药

| 药物 | 剂量 | 机制 | 适应症 | 注意 |
|------|------|------|--------|------|
| **曲唑酮** | 25-100mg | 5-HT2拮抗 | 共病抑郁/焦虑；老年人 | 体位性低血压；阴茎异常勃起 |
| **多塞平** | 3-6mg | H1拮抗 | 睡眠维持困难 | 抗胆碱能副作用（高剂量） |
| **米氮平** | 7.5-15mg | H1/5-HT2/α2 | 共病抑郁+食欲差 | 体重增加；次日镇静 |
| **阿米替林** | 10-50mg | 多机制 | 慢性疼痛+失眠 | 抗胆碱能；心脏毒性 |

### 3.5 其他

| 药物 | 剂量 | 适应症 | 证据 |
|------|------|--------|------|
| **加巴喷丁** | 300-900mg | 焦虑+失眠；RLS | C级 |
| **普瑞巴林** | 75-300mg | 焦虑+失眠；RLS | C级 |

## 四、特殊人群管理

### 4.1 老年人（≥65岁）

| 原则 | 推荐 |
|------|------|
| **首选** | CBT-I；非药物优先 |
| **药物** | 低剂量多塞平；褪黑素；雷美替胺 |
| **避免** | BZD；Z-drugs（跌倒风险） |
| **注意** | 多重用药；认知损害；日间镇静 |

### 4.2 妊娠期

| 原则 | 推荐 |
|------|------|
| **首选** | CBT-I；睡眠卫生 |
| **相对安全** | 低剂量曲唑酮；多塞平（必要时） |
| **避免** | BZD（致畸风险）；Z-drugs（数据不足） |
| **注意** | 妊娠晚期尤其谨慎 |

### 4.3 儿童青少年

| 原则 | 推荐 |
|------|------|
| **首选** | 行为治疗；父母教育 |
| **药物** | 仅严重病例；褪黑素（ADHD/ASD） |
| **避免** | BZD；Z-drugs |

## 五、难治性失眠策略

| 策略 | 说明 |
|------|------|
| **强化CBT-I** | 延长治疗；团体→个体；电话/网络辅助 |
| **联合治疗** | CBT-I + 短期药物 |
| **共病优化** | 确保抑郁/焦虑/疼痛充分治疗 |
| **排除其他** | OSA；RLS；昼夜节律问题 |
| **转诊睡眠专科** | 复杂病例；疑似发作性睡病/RBD |
