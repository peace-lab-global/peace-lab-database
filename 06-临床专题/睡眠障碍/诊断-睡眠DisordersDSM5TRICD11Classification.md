# 睡眠障碍DSM-5-TR与ICD-11分类 | Sleep-Wake Disorders Classification

> **目标**：建立精神科标准的睡眠-觉醒障碍诊断分类体系，涵盖DSM-5-TR与ICD-11的完整编码与诊断标准。
> **标准来源**：DSM-5-TR（2022）、ICD-11（2022）、AASM ICSD-3（2014）。

---

## 一、DSM-5-TR 睡眠-觉醒障碍总览

### 1.1 核心分类

| 类别 | DSM-5-TR编码 | 主要特征 |
|------|-------------|---------|
| **失眠障碍** | F51.01（急性）、F51.02（持续性） | 入睡困难、维持困难、早醒；日间功能损害 |
| **嗜睡障碍** | F51.11 | 日间过度嗜睡；非恢复性睡眠 |
| **发作性睡病** | G47.41（伴猝倒）、G47.42（不伴猝倒） | 日间不可抗拒的睡眠发作；猝倒；睡眠瘫痪；入睡前幻觉 |
| **阻塞性睡眠呼吸暂停低通气** | G47.33 | 反复上气道阻塞；打鼾；血氧下降 |
| **中枢性睡眠呼吸暂停** | G47.31 | 通气驱动缺失；无呼吸努力 |
| **睡眠相关通气不足** | G47.34 | CO₂潴留；常与神经肌肉疾病相关 |
| **昼夜节律睡眠-觉醒障碍** | G47.2x | 睡眠-觉醒时序异常；与社会期望冲突 |
| **非快速眼动睡眠唤醒障碍** | F51.3（睡行症）、F51.4（夜惊症） | NREM期部分觉醒；复杂行为；事后遗忘 |
| **梦魇障碍** | F51.5 | REM期反复惊醒；梦境记忆清晰；快速恢复定向 |
| **快速眼动睡眠行为障碍** | G47.52 | REM期肌张力失弛缓；梦境演绎行为 |
| **不宁腿综合征** | G25.81 | 静息时下肢不适感；活动后缓解；夜间加重 |
| **物质/药物所致睡眠障碍** | 依物质编码 | 物质使用/戒断直接引起 |
| **其他特定/未特定睡眠-觉醒障碍** | F51.8/F51.9 | 不完全符合上述标准 |

## 三、失眠障碍诊断标准（DSM-5-TR）

### 3.1 诊断要件

**A.** 主诉对睡眠数量或质量的不满意，表现为以下一种或多种：
1. 入睡困难（儿童：在无照顾者干预下入睡超过20分钟）
2. 睡眠维持困难（频繁觉醒或难以再次入睡）
3. 比期望时间早醒，且无法再入睡
4. 儿童：在无照顾者干预下难以再次入睡

**B.** 睡眠困难尽管有充足的睡眠机会和环境，仍每周至少出现3次，持续至少3个月。

**C.** 日间功能损害至少一项：
- 疲劳/萎靡
- 注意力、专注力或记忆力下降
- 社交、家庭、职业或学业功能损害
- 情绪易激惹
- 日间嗜睡
- 行为问题（如多动、冲动、攻击性）
- 动力/能量/主动性降低
- 易犯错或事故
- 对睡眠的持续担忧

**D.** 睡眠-觉醒困难不能更好地用另一种睡眠-觉醒障碍解释。

**E.** 非物质/躯体疾病直接所致（如排除甲状腺功能亢进、疼痛）。

**F.** 共存的精神/躯体疾病不能充分解释主诉失眠的严重程度。

### 3.2 亚型

| 亚型 | 特征 |
|------|------|
| **睡眠起始型** | 入睡困难 |
| **睡眠维持型** | 频繁觉醒或难以再次入睡 |
| **早醒型** | 比期望时间早醒 |
| **混合型** | 上述组合 |

### 3.3 严重程度

| 程度 | 标准 |
|------|------|
| **轻度** | 日间功能轻微损害 |
| **中度** | 日间功能明显但可容忍的损害 |
| **重度** | 日间功能严重损害 |

## 四、发作性睡病诊断标准（DSM-5-TR）

### 4.1 诊断要件

**A.** 反复日间不可抗拒的睡眠发作、躺倒入睡或延长午睡（≥9小时/日），至少3个月。

**B.** 以下至少一项：
1. **猝倒**（Cataplexy）：清醒状态下突然发作的肌张力丧失，由情绪（通常是大笑或愤怒）触发
2. **下丘脑分泌素缺乏**（CSF hypocretin-1 ≤ 110 pg/mL或<正常均值1/3）
3. **夜间多导睡眠图（PSG）**：入睡潜伏期 ≤ 15分钟；REM睡眠潜伏期 ≤ 15分钟
4. **多次睡眠潜伏期测试（MSLT）**：平均睡眠潜伏期 ≤ 8分钟；≥2次睡眠起始REM期（SOREMP）

### 4.2 亚型

| 亚型 | 特征 | 编码 |
|------|------|------|
| **1型（伴猝倒/下丘脑分泌素缺乏）** | 猝倒 + 下丘脑分泌素缺乏 | G47.411 |
| **2型（不伴猝倒）** | 无猝倒；MSLT阳性 | G47.419 |

## 五、阻塞性睡眠呼吸暂停低通气（OSA）

### 5.1 成人诊断标准（AASM）

| 严重程度 | AHI（呼吸暂停低通气指数） | 最低SaO₂ |
|---------|-------------------------|---------|
| **正常** | <5 | >90% |
| **轻度** | 5-15 | 85-90% |
| **中度** | 15-30 | 80-85% |
| **重度** | >30 | <80% |

**诊断要件：**
- PSG或家庭睡眠监测：AHI ≥ 5次/小时 + 相关症状（打鼾、 witnessed呼吸暂停、日间嗜睡、高血压）
- 或AHI ≥ 15次/小时（无论症状）

## 六、不宁腿综合征（RLS）诊断标准（IRLSSG）

### 6.1 四个必要标准

1. **活动欲望**：有移动腿部的欲望，通常伴随腿部不适感（感觉异常）
2. **静息加重**：静息或不活动（坐位或躺卧）时症状开始或加重
3. **活动缓解**：活动（行走或伸展）后症状部分或完全缓解
4. **昼夜节律**：症状傍晚或夜间加重（仅严重病例可能白天也有）

### 6.2 支持特征

- 家族史阳性（一级亲属50-90%）
- 对多巴胺能药物反应良好
- 周期性肢体运动（PLM）≥5次/小时

## 七、梦魇障碍诊断标准

**A.** 反复从主要睡眠期或午睡中觉醒，对延长、极其不安的梦境有详细回忆，通常涉及生存、安全或躯体完整性威胁。

**B.** 从梦魇中觉醒时迅速恢复定向和警觉。

**C.** 梦境体验或睡眠紊乱导致显著痛苦或日间功能损害。

**D.**  nightmares非物质/躯体疾病直接所致。

**E.** 共病精神障碍不能充分解释症状。

---

## 八、快速眼动睡眠行为障碍（RBD）

**A.** 反复睡眠中发声和/或复杂运动行为。

**B.** PSG显示REM睡眠肌张力失弛缓（RWA）。

**C.** 行为经PSG证实在REM期出现，或基于临床病史高度怀疑REM期。

**D.** 不能用其他睡眠障碍、物质或躯体疾病更好解释。
