# 睡眠障碍维持治疗与复发预防 | Maintenance and Relapse Prevention for Sleep Disorders

> **目标**：建立慢性睡眠障碍的长期管理框架，预防复发，维持治疗效果。
> **标准来源**：AASM Clinical Practice Guidelines、CBT-I实践标准。

---

## 一、慢性失眠的复发模式

### 1.1 复发率

| 数据 | 说明 |
|------|------|
| **CBT-I后** | 约20-30%在1-2年内出现不同程度复发 |
| **药物停药后** | 50-80%在停药后数周至数月复发 |
| **预测因素** | 残留症状、生活应激、缺乏维持技能、共病未处理 |

### 1.2 复发类型

| 类型 | 定义 | 示例 |
|------|------|------|
| **复发（Relapse）** | 缓解后症状再现 | CBT-I后6月，因工作压力再次出现失眠 |
| **反跳（Rebound）** | 停药后短期症状加重 | 停用唑吡坦后2-3天严重失眠 |
| **波动（Fluctuation）** | 轻度、短暂的症状变化 | 偶尔入睡困难，但很快恢复 |

## 二、CBT-I维持策略

### 2.1 急性期后维持

| 阶段 | 时间 | 频率 | 重点 |
|------|------|------|------|
| **急性期** | 0-8周 | 每周 | 核心技能学习 |
| **巩固期** | 2-3月 | 每2周 | 技能强化；问题解决 |
| **维持期** | 3-6月 | 每月 | 监测；微调 |
| **自主维持** | >6月 | 按需/每3月 | 预防复发 |

### 2.2 核心维持技能

| 技能 | 维持方法 |
|------|---------|
| **刺激控制** | 持续遵守：仅困倦时上床；20分钟未入睡离开床 |
| **规律作息** | 固定起床时间（即使在周末，变化≤1小时） |
| **睡眠卫生** | 持续优化睡眠环境；管理咖啡因/酒精/屏幕 |
| **认知技能** | 持续识别和挑战睡眠灾难化思维 |
| **放松技能** | 日常练习；尤其压力期 |

### 2.3 "预防性"CBT-I

| 情境 | 策略 |
|------|------|
| **预期应激** | 提前强化应对技能（如重要事件前1-2周） |
| **偶尔失眠夜** | 不恐慌；第二天坚持固定起床时间；不补觉 |
| **旅行/时差** | 提前调整作息；目的地时间立即适应 |
| **疾病/疼痛** | 临时调整；疾病恢复后尽快恢复正常作息 |

## 三、药物维持与减停

### 3.1 减停原则

| 原则 | 说明 |
|------|------|
| **CBT-I为减停基础** | 药物减停应在CBT-I建立后进行 |
| **极缓慢减量** | 每1-2周减原剂量的25-50% |
| **间歇给药** | 过渡到按需/隔日服用 |
| **监测** | 每周评估睡眠日记；警惕反跳 |
| **必要时暂停减量** | 症状加重时暂停或回到前一级剂量 |

### 3.2 各药物减停策略

| 药物 | 减量速度 | 反跳风险 | 注意 |
|------|---------|---------|------|
| **唑吡坦** | 每1-2周减50% | 中等 | 可能短暂反跳；通常1-2周消退 |
| **右佐匹克隆** | 每1-2周减50% | 中等 | 同上 |
| **艾司唑仑** | 每2-4周减25% | 高 | 半衰期长；需更慢减量 |
| **曲唑酮** | 每1-2周减25-50mg | 低 | 相对容易减停 |
| **多塞平** | 每1-2周减1.5-3mg | 低 | 低剂量时直接停用 |

### 3.3 长期药物维持的适应证

| 情况 | 策略 |
|------|------|
| **严重慢性失眠** | 最低有效剂量长期维持 |
| **反复发作** | 按需给药（每周2-3次） |
| **共病严重精神障碍** | 精神科稳定优先；睡眠药物辅助 |
| **DORA药物** | 可能更适合长期（耐受/依赖证据少） |

## 四、OSA的维持管理

### 4.1 CPAP长期管理

| 方面 | 策略 |
|------|------|
| **依从性监测** | 远程监测；每3-6月回顾下载数据 |
| **面罩更换** | 每3-6月更换；确保适配 |
| **压力调整** | 体重变化>10%或症状变化时重新滴定 |
| **并发症监测** | 血压、血糖、心血管状况 |
| **再评估** | 每年PSG（尤其症状变化时） |

### 4.2 减重维持

| 策略 | 目标 |
|------|------|
| **体重维持** | 减重后体重增加<3% |
| **持续运动** | 独立于减重改善OSA |
| **定期随访** | 营养师/减重门诊 |

## 五、发作性睡病的长期管理

| 方面 | 策略 |
|------|------|
| **药物调整** | 随时间调整剂量；应对耐受 |
| **新药评估** | 新疗法出现时评估转换 |
| **神经退行性疾病监测** | RBD患者监测帕金森症状 |
| **功能评估** | 驾驶能力；职业适应 |
| **心理支持** | 长期疾病适应；支持团体 |

## 六、复发预警与应对

### 6.1 失眠复发预警

| 预警信号 | 应对 |
|---------|------|
| **入睡潜伏期>30分钟（连续3晚）** | 重新实施刺激控制；检查诱因 |
| **夜间觉醒>20分钟** | 强化睡眠限制；避免看时间 |
| **开始使用安全行为** | 识别并停止（如依赖酒精、长时间卧床） |
| **对睡眠的担忧增加** | 认知重构；减少睡眠监控 |
| **生活应激事件** | 提前启动应对策略；考虑增加治疗频率 |

## 七、长期随访计划

### 7.1 随访频率

| 阶段 | 频率 | 内容 |
|------|------|------|
| **急性治疗后1-3月** | 每2-4周 | 症状；技能执行；日记回顾 |
| **3-6月** | 每月 | 维持；微调；应激管理 |
| **6-12月** | 每2-3月 | 巩固；复发预防 |
| **>1年** | 每3-6月或按需 | 维持；新发问题 |

### 7.2 随访内容

| 评估 | 工具 |
|------|------|
| **失眠严重程度** | ISI |
| **日间功能** | ESS；疲劳评分 |
| **睡眠质量** | PSQI |
| **药物使用** | 类型、剂量、频率 |
| **CBT-I技能执行** | 自评问卷 |
| **共病变化** | 抑郁/焦虑症状 |
| **生活应激** | 近期事件 |
