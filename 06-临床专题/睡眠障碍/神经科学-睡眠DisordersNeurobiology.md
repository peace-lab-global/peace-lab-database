# 睡眠障碍神经生物学 | Neurobiology of Sleep-Wake Disorders

> **目标**：系统阐述睡眠-觉醒的神经解剖、神经化学基础及主要睡眠障碍的神经机制。
> **标准来源**：Nature Reviews Neuroscience、Sleep Medicine Reviews、AASM。

---

## 一、睡眠-觉醒调节的神经解剖

### 1.1 觉醒促进系统

| 核团/系统 | 神经递质 | 投射 | 功能 |
|----------|---------|------|------|
| **蓝斑（LC）** | 去甲肾上腺素（NE） | 广泛皮层、丘脑 | 警觉、注意力、应激 |
| **中缝核** | 5-羟色胺（5-HT） | 广泛皮层 | 情绪调节；促进NREM |
| **结节乳头体核（TMN）** | 组胺（HA） | 广泛皮层 | 觉醒；认知 |
| **腹侧被盖区（VTA）** | 多巴胺（DA） | 前额叶、边缘系统 | 奖赏、动机、觉醒 |
| **基底前脑** | 乙酰胆碱（ACh） | 皮层、海马 | 皮层激活；REM生成 |
| **外侧下丘脑（LH）** | 食欲素（Orexin/Hypocretin） | 广泛觉醒核团 | 觉醒稳定；食欲；奖赏 |

### 1.2 睡眠促进系统

| 核团/系统 | 神经递质 | 功能 |
|----------|---------|------|
| **腹外侧视前区（VLPO）** | GABA、甘丙肽 | NREM睡眠启动；抑制觉醒核团 |
| **旁正中丘脑** | GABA | NREM维持 |
| **下丘脑分泌素神经元** | 下丘脑分泌素（夜间不活跃） | 夜间允许睡眠 |

### 1.3 REM睡眠生成系统

| 核团 | 神经递质 | 功能 |
|------|---------|------|
| **脑桥背外侧被盖（LDT/PPT）** | 乙酰胆碱 | REM启动和维持 |
| **蓝斑/中缝核（REM-off）** | NE/5-HT | REM期抑制；停药产生REM反跳 |
| **延髓腹侧（sublaterodorsal nucleus）** | GABA/甘氨酸 | 肌张力抑制（REM弛缓） |
| **外侧下丘脑（抑制区）** | 食欲素 | 稳定REM-NREM转换 |

## 二、睡眠-觉醒调节模型

### 2.1 双过程模型（Borbély, 1982）

| 过程 | 机制 | 特征 |
|------|------|------|
| **过程S（睡眠稳态）** | 腺苷积累（觉醒期）→ 清除（睡眠期） | 睡眠压力随觉醒时间增加；深度睡眠（N3）反映过程S |
| **过程C（昼夜节律）** | 视交叉上核（SCN）驱动的内源性节律 | 独立于睡眠；约24.2小时周期；光照同步 |

**相互作用：**
- 过程S和过程C共同决定睡眠-觉醒时相
- 时差：过程C未同步，过程S正常 → 白天困倦、夜间清醒
- 失眠：过程S和过程C失调

### 2.2 三因素模型（ Spielman ）

| 因素 | 说明 | 示例 |
|------|------|------|
| **易感因素** | 增加失眠易感性的稳定特征 | 高觉醒特质；完美主义；遗传 |
| **诱发因素** | 触发失眠的事件 | 应激；疾病；生活变化 |
| **维持因素** | 使失眠持续的认知和行为 | 延长卧床时间；白天补觉；睡眠担忧 |

## 三、失眠的神经生物学

### 3.1 过度觉醒理论（Hyperarousal）

| 层面 | 表现 |
|------|------|
| **认知** | 睡前反刍；对睡眠的灾难化思维 |
| **情绪** | 焦虑、担忧 |
| **生理** | 心率增快、HRV降低、皮质醇升高、体温下降延迟 |
| **皮层** | fMRI显示失眠者睡眠期和觉醒期均皮层过度激活 |
| **代谢** | 24小时代谢率增加 |

### 3.2 神经影像学发现

| 发现 | 意义 |
|------|------|
| **前额叶过度激活** | 执行功能/情绪调节区在睡眠期未适当关闭 |
| **杏仁核-前额叶连接** | 情绪调节失调 |
| **默认模式网络（DMN）** | 自我参照加工过度活跃 |
| **NREM慢波减少** | 睡眠深度不足 |

## 四、发作性睡病的神经生物学

### 4.1 下丘脑分泌素系统

| 特征 | 说明 |
|------|------|
| **神经元位置** | 下丘脑外侧区（约70000个神经元） |
| **分泌产物** | 下丘脑分泌素-1（Hcrt-1/Orexin-A）；下丘脑分泌素-2 |
| **受体** | Hcrt-1受体（OX1R）；Hcrt-2受体（OX2R） |
| **功能** | 稳定觉醒；抑制REM；调节食欲/奖赏/自主神经 |

### 4.2 1型发作性睡病的病理

| 发现 | 说明 |
|------|------|
| **神经元丧失** | 下丘脑分泌素神经元丧失>90% |
| **机制** | 自身免疫（H1N1感染/疫苗后） |
| **遗传** | HLA-DQB1*06:02；T细胞受体多态性 |
| **CSF** | 下丘脑分泌素-1 <110 pg/mL |

### 4.3 食欲素受体拮抗剂的作用机制

| 药物 | 机制 | 结果 |
|------|------|------|
| **苏沃雷生等DORA** | 阻断OX1R/OX2R | 降低觉醒信号；促进睡眠 |

## 五、OSA的神经生物学

### 5.1 间歇性低氧的影响

| 系统 | 影响 |
|------|------|
| **氧化应激** | 活性氧（ROS）增加；炎症因子升高 |
| **交感神经过度激活** | 反复觉醒→交感激活→高血压 |
| **内皮功能障碍** | 一氧化氮减少；血管收缩 |
| **代谢紊乱** | 胰岛素抵抗；脂肪因子改变 |
| **认知** | 海马损伤；灰质减少（前额叶、边缘系统） |

### 5.2 睡眠碎片化

| 影响 | 机制 |
|------|------|
| **N3减少** | 慢波睡眠不足 → 记忆巩固受损 |
| **认知功能** | 注意力、执行功能下降 |
| **情绪调节** | 前额叶-杏仁核连接受损 |

## 六、昼夜节律的神经生物学

### 6.1 视交叉上核（SCN）

| 特征 | 说明 |
|------|------|
| **位置** | 下丘脑视交叉上方 |
| **功能** | 主生物钟；约24.2小时内源性周期 |
| **光输入** | 视网膜-下丘脑束（ipRGCs → SCN） |
| **输出** | 通过自主神经、激素（褪黑素）、体温调节全身节律 |

### 6.2 褪黑素

| 特征 | 说明 |
|------|------|
| **合成** | 松果体；黑暗时合成增加 |
| **受体** | MT1（睡眠启动）；MT2（昼夜节律相位） |
| **功能** | "黑暗信号"；促进睡眠；相位调整 |
| **分泌模式** | 夜间峰值（约2-4AM）；白天极低 |

### 6.3 分子生物钟

| 组分 | 功能 |
|------|------|
| **CLOCK/BMAL1** | 正向转录因子 |
| **PER/CRY** | 负向反馈抑制CLOCK/BMAL1 |
| **周期** | 约24小时 |
| **调控** | 光→SCN→褪黑素→外周时钟同步 |

## 七、睡眠与记忆的神经生物学

| 睡眠阶段 | 记忆类型 | 机制 |
|---------|---------|------|
| **NREM N2（睡眠纺锤波）** | 陈述性记忆巩固 | 丘脑-皮层回路；信息重放 |
| **NREM N3（慢波睡眠）** | 陈述性记忆；程序性记忆 | 系统巩固；海马-新皮层转移 |
| **REM睡眠** | 情绪记忆加工；程序性记忆 | 杏仁核激活；去甲肾上腺素低；情绪记忆去敏感化 |

**睡眠剥夺影响：**
- N3剥夺 → 陈述性记忆损害
- REM剥夺 → 情绪记忆加工障碍；情绪调节受损

## 八、睡眠与神经退行性疾病

### 8.1 双向关系

| 方向 | 机制 |
|------|------|
| **睡眠障碍→神经退行** | 睡眠障碍加速蛋白病理性聚集 |
| **神经退行→睡眠障碍** | 睡眠调控核团受损 |

### 8.2 蛋白质清除与睡眠

| 系统 | 功能 |
|------|------|
| **类淋巴系统（Glymphatic）** | 睡眠期间脑脊液流动增加；清除代谢废物 |
| **β-淀粉样蛋白** | NREM慢波睡眠减少→清除减少→沉积增加 |
| **Tau蛋白** | 睡眠剥夺→Tau增加 |
| **α-突触核蛋白** | RBD→α-突触核蛋白病风险 |

### 8.3 睡眠障碍作为神经退行性疾病的早期标志

| 睡眠障碍 | 预测疾病 |
|---------|---------|
| **RBD** | 帕金森病、路易体痴呆、多系统萎缩（10年内>50%） |
| **失眠** | 阿尔茨海默病风险增加 |
| **OSA** | 认知下降、痴呆风险增加 |
