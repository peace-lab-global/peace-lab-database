# CBT-I 标准化治疗协议模板 | CBT-I Protocol Template

> **目标**：提供可直接临床使用的CBT-I标准化协议模板。
> **标准来源**：AASM CBT-I实践标准、ESRS指南、Mannual for CBT-I（Perlis et al.）。

---

## 一、治疗结构

| 参数 | 设置 |
|------|------|
| **疗程** | 4-8次个体或团体会谈 |
| **频率** | 每周1次 |
| **时长** | 30-60分钟/次 |
| **格式** | 个体（首选）或团体（6-10人） |
| **辅助** | 睡眠日记（每日）、 workbook |

---

## 二、逐次会谈内容

### 第1次：评估与教育（60分钟）

| 内容 | 具体 |
|------|------|
| **评估** | 睡眠病史（BEDSIDE）；PSQI；ISI；ESS；DBAS |
| **睡眠日记指导** | 发放日记；详细讲解填写方法 |
| **心理教育** | 睡眠生理学；失眠维持模型；治疗概述 |
| **设定目标** | 患者个人治疗目标 |
| **家庭作业** | 每日睡眠日记；记录一周 |

### 第2次：刺激控制与睡眠卫生（45-60分钟）

| 内容 | 具体 |
|------|------|
| **回顾日记** | 分析睡眠模式；计算睡眠效率 |
| **刺激控制** | 5条核心规则；解释原理 |
| **睡眠卫生** | 逐一讲解；患者自我评估 |
| **设定起床时间** | 固定时间（即使周末） |
| **家庭作业** | 严格执行刺激控制；继续日记 |

**刺激控制5规则：**
1. 仅当困倦时才上床
2. 床只用于睡眠（和性活动）
3. 20分钟未入睡 → 离开床，做安静活动
4. 再次困倦时回床
5. 无论睡多久，固定时间起床

### 第3次：睡眠限制疗法（45-60分钟）

| 内容 | 具体 |
|------|------|
| **回顾** | 睡眠日记；刺激控制执行情况 |
| **计算睡眠窗口** | TST（平均总睡眠时间）；设定卧床时间=minimum(TST, 5-6小时) |
| **解释原理** | 提高睡眠效率；重建睡眠驱动力 |
| **预期反应** | 初期可能更困倦；这是正常的 |
| **安全** | 白天不可驾驶/操作机械若极度困倦 |
| **家庭作业** | 执行睡眠限制；继续日记 |

### 第4次：认知重构（45-60分钟）

| 内容 | 具体 |
|------|------|
| **回顾** | 睡眠限制效果；调整睡眠窗口 |
| **识别认知扭曲** | 常见睡眠相关不良信念 |
| **认知重构技术** | 证据检验；替代思维；行为实验 |
| **DBAS评估** | 识别具体认知靶点 |
| **家庭作业** | 记录睡眠相关思维；练习认知重构 |

**常见睡眠认知扭曲：**
- "我必须睡够8小时"
- "如果今晚睡不好，明天就完了"
- "我根本无法控制自己的睡眠"
- "失眠会严重损害我的健康"
- "我需要躺在床上试图入睡"

### 第5次：放松训练（45分钟）

| 内容 | 具体 |
|------|------|
| **回顾** | 睡眠日记；认知重构练习 |
| **渐进肌肉放松** | 教授16组肌肉放松 |
| **腹式呼吸** | 教授4-7-8或标准腹式呼吸 |
| **正念呼吸** | 介绍正念作为放松替代 |
| **选择** | 患者选择最适合的放松方法 |
| **家庭作业** | 每日练习放松（但不在床上） |

### 第6次：整合与微调（45分钟）

| 内容 | 具体 |
|------|------|
| **回顾** | 全面评估进展；睡眠效率趋势 |
| **调整睡眠窗口** | 如SE>85%：增加15-30分钟；如SE<80%：减少15分钟 |
| **解决问题** | 具体困难（如配偶干扰、轮班） |
| **强化认知技能** | 继续挑战不良信念 |
| **家庭作业** | 维持所有技能；继续日记 |

### 第7次：复发预防（45分钟）

| 内容 | 具体 |
|------|------|
| **回顾** | 整体进步；目标达成情况 |
| **复发预防** | 识别预警信号；建立应对计划 |
| **维持计划** | 哪些技能需要长期坚持 |
| **应对未来应激** | 如何使用CBT-I技能应对 |
| **家庭作业** | 制定个人维持计划 |

### 第8次：结束与随访（30-45分钟）

| 内容 | 具体 |
|------|------|
| **最终评估** | ISI；PSQI；睡眠日记回顾 |
| **总结** | 治疗收获；技能总结 |
| **维持计划确认** | 书面的个人维持计划 |
| **随访安排** | 1月、3月、6月随访 |
| **应急计划** | 复发时如何求助 |

## 四、睡眠窗口调整规则

| 睡眠效率（周平均） | 下周调整 |
|------------------|---------|
| **>90%** | 增加卧床时间15-30分钟 |
| **85-90%** | 维持当前卧床时间 |
| **<85%** | 减少卧床时间15-30分钟（最低5小时） |

## 五、常见问题处理

| 问题 | 处理 |
|------|------|
| **患者不愿离开床** | 解释原理；强调20分钟是最大限；可从10分钟开始 |
| **极度日间困倦** | 确保睡眠窗口不过低（最低5h）；周末不补觉；必要时调整 |
| **配偶干扰** | 分床短期（2-4周）；沟通治疗计划 |
| **轮班工作** | 固定轮班；以主睡眠期计算；光疗辅助 |
| **共病抑郁/焦虑** | 优先处理安全问题；CBT-I与心理治疗协调 |
| **对睡眠限制恐惧** | 解释原理； reassurance；小步开始 |
| **复发** | 重新实施核心技能；必要时"强化"会谈 |

## 六、疗效评估

| 时间 | 评估内容 | 工具 |
|------|---------|------|
| **基线** | 失眠严重度；睡眠质量；日间功能；不良信念 | ISI；PSQI；ESS；DBAS |
| **第4次** | 中期评估 | ISI；睡眠日记 |
| **第8次** | 结束评估 | ISI；PSQI；ESS；DBAS |
| **1月随访** | 维持情况 | ISI；睡眠日记（3天） |
| **3月随访** | 长期效果 | ISI；PSQI |
| **6月随访** | 复发筛查 | ISI；维持计划执行 |

**疗效标准：**
- ISI评分降低≥8分；或
- ISI <8分（亚临床水平）；或
- 睡眠效率>85%；或
- 主观睡眠质量显著改善
