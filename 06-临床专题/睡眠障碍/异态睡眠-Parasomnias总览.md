# 异态睡眠概述 | Parasomnias Overview

> **目标**：系统阐述NREM相关、REM相关及其他异态睡眠的诊断、鉴别与治疗。
> **标准来源**：DSM-5-TR、AASM ICSD-3。

---

## 一、概述

异态睡眠是指入睡、睡眠期间或觉醒转换时出现的非主观的异常行为、体验或生理事件。根据睡眠期分为NREM相关、REM相关及其他异态睡眠。

---

## 二、NREM相关异态睡眠

### 2.1 睡行症（Sleepwalking）

| 参数 | 说明 |
|------|------|
| **患病率** | 儿童15%；成人1-4% |
| **发作期** | N3期（慢波睡眠） |
| **表现** | 床上坐起→下床行走；目光呆滞；对刺激反应差；事后遗忘 |
| **持续时间** | 数分钟至30分钟 |
| **危险** | 跌倒、自伤、出走、暴力行为（尤其被试图唤醒时） |

**诊断标准：**
- 反复从睡眠中起身行走
- 发作时对交流反应迟钝；被唤醒时意识混浊
- 事后遗忘或部分记忆
- 发作后迅速恢复定向
- 日间功能损害或伤害风险

**诱发因素：**
- 睡眠剥夺
- 发热（儿童）
- 酒精
- 镇静药物
- 睡眠呼吸障碍
- 周期性肢体运动
- 应激

### 2.2 夜惊症（Sleep Terrors）

| 参数 | 说明 |
|------|------|
| **患病率** | 儿童1-6%；成人<1% |
| **发作期** | N3期 |
| **表现** | 突然坐起、尖叫、极度恐惧；自主神经激活（心动过速、出汗、瞳孔散大）；事后遗忘 |
| **与噩梦区别** | 夜惊：N3期、遗忘、无法安抚、定向差；噩梦：REM期、记忆清晰、可安抚 |

### 2.3 意识模糊性觉醒（Confusional Arousals）

| 参数 | 说明 |
|------|------|
| **特征** | 从NREM睡眠觉醒时意识模糊、定向差、行为不当 |
| **常见表现** | 言语不清、反应迟钝、攻击性行为（尤其被唤醒时） |
| **"睡眠惯性"** | 早晨觉醒困难；暴力反应 |

### 2.4 NREM异态睡眠的治疗

| 策略 | 说明 |
|------|------|
| **安全预防** | 卧室安全化；门窗报警；移走危险物品；一楼卧室 |
| **规律作息** | 充足睡眠；固定就寝时间 |
| **避免诱因** | 睡眠剥夺；酒精；镇静药 |
| **治疗共病** | OSA；PLMD；焦虑 |
| **药物治疗** | 严重/危险病例：氯硝西泮0.25-2mg睡前；SSRIs |
| **预期觉醒** | 在典型发作时间前15-20分钟唤醒 |
| ** hypnosis** | 儿童有效 |

## 三、REM相关异态睡眠

### 3.1 梦魇障碍（Nightmare Disorder）

| 参数 | 说明 |
|------|------|
| **患病率** | 儿童50%偶尔；成人2-8%频繁 |
| **发作期** | REM睡眠（后半夜多见） |
| **表现** | 从噩梦中惊醒；详细梦境记忆（威胁生存/安全）；快速恢复定向；再入睡困难 |
| **影响** | 睡眠恐惧、回避睡眠、日间焦虑、功能损害 |

**治疗：**

| 治疗 | 证据 | 方法 |
|------|------|------|
| **意象排演治疗（IRT）** | A级 | 清醒时重写噩梦剧本；心理排练新结局 |
| **暴露治疗** | B级 | 逐步暴露噩梦内容 |
| **CBT-I** | B级 | 改善整体睡眠；减少睡眠恐惧 |
| **药物治疗** | C级 | 哌唑嗪（PTSD相关）；曲唑酮；可乐定 |
| **褪黑素** | C级 | 减少REM密度 |

### 3.2 REM睡眠行为障碍（RBD）

| 参数 | 说明 |
|------|------|
| **患病率** | 普通人群0.5%；老年男性更高 |
| **机制** | REM期肌张力失弛缓（正常应为肌无力） |
| **表现** | 梦境演绎行为：拳击、踢打、跳跃、喊叫；可伤及床伴 |
| **梦境内容** | 多为被攻击/追逐（防御性/攻击性反应） |
| **神经退行性疾病标志** | 50-80%最终发展为帕金森病、路易体痴呆或多系统萎缩 |

**诊断：**
- PSG：REM期肌电图显示肌张力增高（RWA）
- 病史：梦境演绎行为

**治疗：**

| 药物 | 剂量 | 效果 |
|------|------|------|
| **氯硝西泮** | 0.25-2mg睡前 | 一线；减少行为强度 |
| **褪黑素** | 3-12mg睡前 | 有效；尤其老年人 |
| **普拉克索** | 0.25-1mg | 多巴胺激动剂 |
| **左乙拉西坦** | 500-1500mg | 替代选择 |

**安全措施：**
- 移走卧室危险物品
- 床周软垫
- 分床睡（保护床伴）
- 窗户安全锁

### 3.3 复发性孤立性睡眠瘫痪

| 参数 | 说明 |
|------|------|
| **特征** | 入睡或觉醒时短暂无法活动或说话；意识清醒；可伴幻觉 |
| **持续时间** | 数秒至数分钟 |
| **人群** | 发作性睡病常见；孤立性：约7-8%人群一生中至少一次 |

## 四、其他异态睡眠

### 4.1 爆炸头综合征（Exploding Head Syndrome）

| 参数 | 说明 |
|------|------|
| **特征** | 入睡或觉醒时感知到头部巨大爆炸声/闪光 |
| **性质** | 良性；无痛；无器质性病变 |
| **治疗** | 通常无需；钙通道阻滞剂（如有困扰） |

### 4.2 睡眠相关幻觉

| 类型 | 特征 |
|------|------|
| **入睡前幻觉** | 入睡时生动的视/听/触觉幻觉 |
| **觉醒前幻觉** | 觉醒时幻觉 |
| **关联** | 发作性睡病常见；孤立性可见 |

### 4.3 睡眠相关进食障碍（SRED）

| 参数 | 说明 |
|------|------|
| **特征** | 夜间反复无意识进食；高碳水/高热量食物；事后遗忘或模糊记忆 |
| **关联** | 常与唑吡坦等Z-drugs使用相关；也可能自发 |
| **鉴别** | 夜食综合征（NES）：意识清醒、记忆完整 |
| **治疗** | 停用诱发药物；托吡酯；SSRI |

## 五、异态睡眠的鉴别诊断

| 特征 | 夜惊 | 梦魇 | RBD | 癫痫 |
|------|------|------|-----|------|
| **睡眠期** | N3 | REM | REM | 任何 |
| **时间** | 前半夜 | 后半夜 | 后半夜 | 任何 |
| **记忆** | 无/片段 | 清晰 | 清晰梦境 | 无/片段 |
| **定向** | 差 | 好 | 好 | 差 |
| **自主神经激活** | 显著 | 轻度 | 中度 | 可变 |
| **行为** | 简单、刻板 | 无行为 | 复杂、梦境相关 | 刻板 |
| **PSG** | N3觉醒 | REM觉醒 | REM肌张力增高 | 癫痫样放电 |
