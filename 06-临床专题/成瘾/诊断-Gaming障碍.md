# 游戏障碍 (Gaming Disorder, ICD-11 6C51)

> **核心定位**：ICD-11 正式承认的行为成瘾（DSM-5 Section III 列为研究诊断"网络游戏障碍"IGD）。特征是对游戏行为的**控制力受损**，**优先级超过其他活动**，**尽管负面后果仍持续**。

---

## 1. ICD-11 诊断标准（3 核心 + 功能损害，12 个月）

| # | 核心标准 |
|:--|:---------|
| 1 | 对游戏的控制受损（频率/强度/时长/开始/结束） |
| 2 | 游戏优先于其他生活兴趣和日常活动 |
| 3 | 尽管负面后果仍继续或增加游戏 |
| + | 导致个人/家庭/社会/教育/职业功能**显著损害** |

> DSM-5 IGD 标准：9 项中≥5（更细化，含"使用游戏逃避"项）

---

## 2. 流行病学

| 数据 | 值 |
|:-----|:---|
| 全球患病率 | 1.96%（Stevens et al. 2021, meta） |
| 亚洲 | 2.5-5%（更高） |
| 性别 | 男性 > 女性（2-3:1） |
| 年龄 | 青少年/青年最高风险 |
| 游戏类型 | MMORPG/MOBA/大逃杀风险最高 |

---

## 3. 风险因素

| 类别 | 因素 |
|:-----|:-----|
| **个体** | 冲动性/高感觉寻求/社交技能缺陷/低自尊 |
| **心理** | 抑郁/焦虑/ADHD/孤独感 |
| **游戏设计** | 可变比率奖励/社交义务/FOMO/付费战利品箱 |
| **家庭** | 亲子冲突/低监控/家庭功能障碍 |
| **社会** | 孤独/社交孤立/学业压力/文化规范 |

---

| 工具 | 说明 |

---

## 5. 治疗

| 方法 | 说明 | 证据 |
|:-----|:-----|:-----|
| **CBT** | 认知重建+行为替代+时间管理 | **一线** |
| **MI** | 增强减少动机（低动机青少年） | 一线联合 |
| **家庭干预** | 亲子沟通/设定边界/替代活动 | 青少年首选联合 |
| **药物治疗** | 安非他酮/SSRIs（共病时） | 辅助 |
| **数字断联** | 结构化减少使用时间 | 辅助 |
| **替代活动** | 运动/社交/创造性活动 | 维持 |

---

## 6. 争议

| 争议点 | 说明 |
|:-------|:-----|
| 过度诊断 | 将热情游戏病理化 vs 真正成瘾 |
| 道德恐慌 | 媒体驱动 vs 真实患病率 |
| 因果方向 | 游戏导致问题 vs 问题驱动游戏 |
| 文化偏见 | 亚洲定义 vs 西方定义 |
| 产业影响 | 游戏设计与公共卫生的张力 |
