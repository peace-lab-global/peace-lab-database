# 减少伤害策略 (Harm Reduction Approach)

> **核心定位**：**实用主义**而非道德判断。目标不是"完全戒断"，而是**减少物质使用相关的伤害**（健康/社会/法律）。尊重个人自主权，在"人在哪里就从哪里开始"。

---

## 1. 核心原则

| 原则 | 说明 |
|:-----|:-----|
| **人本主义** | 尊重使用物质者的尊严和权利 |
| **实用主义** | 聚焦实际伤害减少（非意识形态） |
| **个人主义** | 个体化策略（非一刀切） |
| **自主权** | 使用者参与决策 |
| **渐进性** | 任何减少都是进步 |
| **问责** | 平衡权利与责任 |

---

## 2. 减少伤害策略谱

```
完全戒断 ←───────────────────────────→ 无干预
         ↑                ↑              ↑
     戒断导向        减少伤害         自然使用
                     策略区间

层级：
├─ Level 5: 减少使用频率/量
├─ Level 4: 更安全的使用方式（非注射→口服）
├─ Level 3: 减少使用的有害后果（Naloxone/清洁针具）
├─ Level 2: 替代治疗（美沙酮/丁丙诺啡）
└─ Level 1: 完全戒断（如个人选择）
```

---

## 3. 核心干预

| 干预 | 说明 | 证据 |
|:-----|:-----|:-----|
| **针具交换** | 减少 HIV/HCV 传播 | **强** |
| **Naloxone 分发** | 阿片过量逆转 | **强** |
| **监督消费室** | 减少公开使用/过量死亡 | 强 |
| **阿片替代治疗** | 美沙酮/丁丙诺啡 | **强** |
| **药物检测** | 芬太尼/苯二氮卓检测条 | 中-强 |
| **安全供应** | 处方级阿片/兴奋剂 | 初步 |
| **减少饮酒** | 非戒断目标/适度饮酒 | 中 |
| **减少赌博** | 非完全戒赌/设定限额 | 初步 |

---

## 4. Harm Reduction vs 戒断模式

| 维度 | 减少伤害 | 戒断模式 |
|:-----|:---------|:---------|
| 目标 | 减少伤害 | 完全停止使用 |
| 哲学 | 实用/人本 | 道德/疾病 |
| 成功标准 | 任何改善 | 完全戒断 |
| 失误 | 预期内/学习机会 | 失败/重新开始 |
| 门槛 | 低（任何人可参与） | 高（需承诺戒断） |
| 证据 | 减少死亡率/HIV | 部分有效 |
| 伦理 | 尊重自主 | 保护性 |

---

## 5. 批评与回应

| 批评 | 回应 |
|:-----|:-----|
| "鼓励使用" | 研究：不增加使用频率/起始率 |
| "阻碍戒断" | 研究：HR 使用者更可能进入治疗 |
| "社区安全" | 研究：减少公开注射/针具丢弃 |
| "道德问题" | 实用主义：减少死亡和疾病 |
