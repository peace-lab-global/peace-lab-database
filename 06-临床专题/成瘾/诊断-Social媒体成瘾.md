# 社交媒体成瘾 (Social Media Addiction)

> **核心定位**：虽未被 DSM-5/ICD-11 正式列为独立诊断，但大量研究证实社交媒体过度使用具有成瘾特征。青少年和青年为高风险群体，与焦虑/抑郁/睡眠问题/自杀意念显著相关。

---

## 1. 成瘾特征

| 特征 | 社交媒体表现 |
|:-----|:-----------|
| 突显性 | 社交媒体成为生活的核心活动 |
| 情绪调节 | 使用社交媒体缓解负面情绪 |
| 耐受 | 需要更多时间/更多平台 |
| 戒断 | 不使用时焦虑/不安 |
| 冲突 | 与学业/工作/关系的冲突 |
| 复发 | 尝试减少后反复回到高水平使用 |

---

## 2. 神经机制

| 机制 | 说明 |
|:-----|:-----|
| 可变比率奖赏 | 随机点赞/评论/通知→最强行为维持 |
| 社交奖赏 | 人类对社会认可/排斥高度敏感 |
| 多巴胺循环 | 期待→检查→奖赏/失望→更多检查 |
| 下行调节 | 持续高刺激→基线多巴胺下调 |
| 前额叶 | 执行控制受损/冲动性增加 |

---

## 3. FoMO (Fear of Missing Out)

| 维度 | 说明 |
|:-----|:-----|
| 定义 | 对错过他人正在经历的奖赏体验的持续担忧 |
| 功能 | 驱动持续在线/频繁检查 |
| 关联 | 与焦虑/孤独/社交媒体成瘾强相关 |
| 干预 | 正念/JOMO(Joy of Missing Out)/价值导向 |

---

## 4. 平台设计策略

| 策略 | 说明 |
|:-----|:-----|
| 无限滚动 | 消除停止信号 |
| 推送通知 | 外部触发/线索反应 |
| 红点/数字 | 制造紧迫感 |
| 故事/限时内容 | 过期焦虑 |
| 社交证明 | 点赞数/关注数/可见性 |
| 算法推荐 | 个性化成瘾内容 |
