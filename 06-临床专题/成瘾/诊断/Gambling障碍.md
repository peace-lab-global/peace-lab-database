# 赌博障碍 (Gambling Disorder)

> **核心定位**：DSM-5 唯一的正式行为成瘾诊断（从"冲动控制障碍"重新分类到"成瘾及相关障碍"）。ICD-11 编码 6C50。患病率 0.2-2%，男性显著高于女性。

---

## 1. DSM-5 诊断标准（≥4/9，12 个月内）

| # | 标准 |
|:--|:-----|
| 1 | 需要增加赌注以获得期望的兴奋 |
| 2 | 减少或停止赌博时感到不安/易怒 |
| 3 | 反复 unsuccessfully 控制/减少/停止赌博 |
| 4 | 全神贯注于赌博（回忆/计划/获取赌资） |
| 5 | 在痛苦时赌博（缓解不良情绪） |
| 6 | 输钱后回头追损失（chasing losses） |
| 7 | 对赌博程度说谎 |
| 8 | 因赌博危及/失去重要关系/工作/机会 |
| 9 | 依赖他人提供金钱（缓解财务绝望） |

---

## 2. 分类亚型

| 亚型 | 说明 |
|:-----|:-----|
| **发作性** | 发作间有缓解期 |
| **持续性** | 持续症状，无缓解期 |
| **轻度** | 4-5 标准 |
| **中度** | 6-7 标准 |
| **重度** | 8-9 标准 |

---

## 3. 神经生物学

| 机制 | 说明 |
|:-----|:-----|
| 腹侧纹状体 | 赌博时 DA 释放增加（类似可卡因） |
| 前额叶/ACC | 抑制控制↓/决策缺陷 |
| 几乎赢效应 | "差点赢"激活奖赏回路→强化赌博 |
| 认知偏差 | 赌徒谬误/控制错觉/近因偏差 |
| 遗传 | 遗传度 ~50%（与物质成瘾共遗传因子） |

---

## 4. 评估

| 工具 | 说明 |
|:-----|:-----|
| SOGS | 20 项，≥5 可能病理性赌博 |
| PGSI | 9 项，0-27 分，四级 |
| NODS | DSM-IV 标准访谈 |
| Lie/Bet | 2 项超短筛查 |
| 财务评估 | 债务/资产/收入全面评估 |

---

## 5. 循证治疗

| 方法 | 说明 | 证据 |
|:-----|:-----|:-----|
| **CBT** | 认知重建（挑战偏差）+行为替代+复发预防 | **一线** |
| **MI** | 增强改变动机（低动机患者） | 一线/联合 |
| **纳曲酮** | 阿片受体拮抗剂/减少渴求 | FDA off-label |
| **SSRI** | 共病抑郁/焦虑时 | 辅助 |
| **GA (12 步骤)** | Gamblers Anonymous | 社区支持 |
| **排除/自我禁止** | 赌场/网站自我排除 | 环境控制 |
| **财务咨询** | 债务管理/财务监护 | 辅助 |

---

## 6. 共病
