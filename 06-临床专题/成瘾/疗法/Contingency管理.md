# 应急管理 (Contingency Management, CM)

> **核心定位**：基于操作性条件反射的循证行为干预。通过**即时强化**（代金券/奖品/特权）奖励目标行为（如尿液阴性/出勤/药物依从），是**物质使用障碍最有效干预之一**。

---

## 1. 理论基础

| 概念 | 说明 |
|:-----|:-----|
| 操作性条件反射 | Skinner：行为受其后果控制 |
| 正强化 | 目标行为后给予奖赏→行为增加 |
| 即时性 | 强化越及时越有效（vs 自然延迟奖赏） |
| 替代强化 | 用替代奖赏替代药物产生的奖赏 |
| 渐隐 | 逐步减少强化频率→自然维持 |

---

## 2. 核心要素

| 要素 | 说明 |
|:-----|:-----|
| **目标行为** | 清晰可观察（尿液阴性/出勤/服药） |
| **监测** | 客观可验证（尿检/呼吸分析/出勤记录） |
| **强化物** | 代金券/奖品抽签/现金/特权 |
| **强化计划** | 连续→间歇（VR/VI） |
| **重置** | 目标行为未达成→重置到起始 |
| **渐隐** | 治疗后期逐步降低强化 |

---

## 3. 常见 CM 方案

| 方案 | 说明 |
|:-----|:-----|
| **代金券 (Voucher)** | 每次阴性尿检获得代金券（递增+重置） |
| **奖品抽奖 (Prize)** | 每次阴性获得1次抽奖机会（大奖/中/小/谢谢） |
| **Fishbowl** | 抽奖碗中放不同价值纸条 |
| **现金 CM** | 直接现金奖励 |
| **就业 CM** | 出勤/工作表现奖励 |

---

## 4. 证据

| 数据 | 发现 |
|:-----|:-----|
| 效应量 | d = 0.62（vs 对照，最高行为治疗之一） |
| 兴奋剂 | **一线**（无可替代药物时的首选心理干预） |
| 阿片 | 与药物联合有效 |
| 酒精 | 中效 |
| 大麻 | 中效 |
| 戒烟 | 有效 |
| 维持 | 治疗期间效果最佳，停止后衰减 |

---

## 5. 实施挑战

| 挑战 | 说明 | 解决 |
|:-----|:-----|:-----|
| 成本 | 奖品/代金券费用 | 低成本奖品/社区捐赠 |
| 伦理争议 | "贿赂"患者 | 证据支持/渐进减少 |
| 维持 | 停止CM后效果衰减 | 结合CBT/自然强化 |
| 可及性 | 需要频繁监测 | 远程/即时检测 |
