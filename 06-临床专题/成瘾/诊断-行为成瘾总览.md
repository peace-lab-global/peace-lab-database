# 行为成瘾总览 (Behavioral Addiction Overview)

> **核心定位**：行为成瘾（过程成瘾）与物质成瘾共享相似的神经生物学机制（奖赏回路/多巴胺/耐受/戒断），但**不涉及外源性物质**。DSM-5 仅正式承认**赌博障碍**，ICD-11 增加了**游戏障碍**。

---

## 1. 行为成瘾 vs 物质成瘾

| 维度 | 物质成瘾 | 行为成瘾 |
|:-----|:---------|:---------|
| 外源物质 | ✓ | ✗ |
| 多巴胺奖赏 | ✓ | ✓ |
| 耐受性 | ✓ | ✓ |
| 戒断症状 | 躯体+心理 | **主要心理** |
| 失控 | ✓ | ✓ |
| 功能损害 | ✓ | ✓ |
| 否认/最小化 | ✓ | ✓ |
| 治疗相似性 | CBT/MI/12步骤 | CBT/MI/12步骤（调整） |

---

## 2. DSM-5 / ICD-11 分类地位

| 行为 | DSM-5 | ICD-11 | 证据状态 |
|:-----|:------|:-------|:---------|
| **赌博** | 正式诊断（NAD） | 正式诊断 | 强 |
| **游戏** | Section III（IGD） | 正式诊断 (6C51) | 中-强 |
| **社交媒体** | 未列入 | 未列入 | 弱-中 |
| **购物** | 未列入 | 冲动控制障碍 | 弱 |
| **性** | 未列入 | 强迫性行为障碍 | 弱-中 |
| **运动** | 未列入 | 未列入 | 弱 |
| **食物** | 未列入 | 未列入 | 争议 |

---

## 3. 神经生物学共性

| 机制 | 说明 |
|:-----|:-----|
| 中脑边缘多巴胺 | 伏隔核 DA 释放（赌博/游戏时类似可卡因） |
| 前额叶功能 | 执行控制↓/冲动性↑ |
| 奖赏预测误差 | 间歇强化（可变比率）→最强维持 |
| 线索反应 | 环境线索触发渴求（conditioned cue reactivity） |
| 耐受 | 需要更强刺激获得同等快感 |
| 戒断 | 停止后烦躁/焦虑/易怒/渴求 |

---

## 4. 各行为成瘾概览

| 成瘾类型 | 患病率 | 核心特征 | 一线治疗 |
|:---------|:-------|:---------|:---------|
| 赌博障碍 | 0.2-2% | 追逐损失/借钱赌博/隐瞒 | CBT/MI/纳曲酮 |
| 游戏障碍 | 1-3%（亚洲更高） | 优先级/失控/功能损害 | CBT/家庭干预 |
| 社交媒体 | 5-15%（青少年） | 错失恐惧(FoMO)/过度使用 | CBT/数字断联 |
| 强迫购物 | 5-8% | 情绪驱动/无法控制/财务后果 | CBT/MI |
| 强迫性行为 | 3-6% | 失控/关系损害/道德困扰 | CBT/MI/SSRI |

| 工具 | 评估 |
