# 抑郁症临床督导与质量控制 | Clinical Supervision and Quality Control for Depression

> **目标**：建立精神科标准的抑郁症临床督导框架、治疗师胜任力体系、伦理边界、文档标准与多学科协作规范。

---

## 一、治疗师胜任力体系

### 1.1 核心胜任力领域

| 领域 | 具体能力 | 评估方式 |
|------|---------|---------|
| **知识** | 抑郁障碍诊断标准、亚型、鉴别诊断；治疗指南；药物知识 | 考试、案例讨论 |
| **评估技能** | 结构化访谈；量表使用；自杀风险评估；功能评估 | 角色扮演、录像评估 |
| **治疗技能** | CBT/IPT/ACT/MBCT技术；个案概念化；治疗计划制定 | 录像督导、现场观察 |
| **危机管理** | 自杀风险评估；安全计划；危机干预；转介决策 | 模拟演练、案例回顾 |
| **伦理与法律** | 知情同意；保密边界；强制报告；执业范围 | 案例讨论、伦理委员会 |
| **文化胜任力** | 跨文化评估；文化适应干预；减少偏见 | 案例讨论、自我反思 |
| **自我关怀** | 替代性创伤识别；耗竭预防；个人治疗/督导 | 自我报告、同行反馈 |

### 1.2 督导等级

| 等级 | 描述 | 督导频率 |
|------|------|---------|
| **受训者** | 精神科住院医师/心理学研究生 | 每周1-2次 |
| **初级治疗师** | 刚毕业/新手治疗师 | 每周1次 |
| **中级治疗师** | 2-5年经验 | 每2周1次 |
| **高级治疗师** | >5年经验 | 每月1次或按需 |

## 二、督导模式

### 2.1 督导类型

| 类型 | 内容 | 参与者 |
|------|------|--------|
| **个人督导** | 个案概念化；治疗困难；反移情；伦理困境 | 治疗师 + 督导 |
| **团体督导** | 共性议题；案例讨论；同伴学习 | 治疗师团队 + 督导 |
| **现场督导** | 督导实时观察治疗过程（单向镜/录像） | 治疗师 + 督导 |
| **同侪督导** | 平等同伴间的案例讨论和情感支持 | 治疗师同伴 |
| **跨学科督导** | 精神科、心理、社工联合讨论复杂案例 | 多学科团队 |

### 2.2 督导核心议题

| 议题 | 精神科关注点 |
|------|-------------|
| **诊断准确性** | 是否遗漏双相、精神病性特征、人格障碍？ |
| **治疗计划** | 是否符合循证指南？药物+心理整合？ |
| **疗效监测** | 量表使用是否规范？无应答者是否及时识别？ |
| **自杀风险** | 每次督导必须回顾高风险患者的当前状态 |
| **药物管理** | 非精神科治疗师的药物知识边界 |
| **共病处理** | 人格障碍、物质使用对治疗联盟的影响 |
| **伦理困境** | 双重关系；保密突破；强制住院决策 |
| **反移情** | 治疗师对抑郁患者的过度认同/拯救欲/无力感 |
| **耗竭** | 长期接触抑郁/自杀患者的情感负担 |

## 三、伦理边界与执业范围

### 3.1 各专业执业边界

| 专业 | 可独立执行 | 需协作/转介 | 禁止 |
|------|-----------|------------|------|
| **精神科医生** | 诊断；处方；ECT；rTMS；住院决策；危机干预 | 心理治疗（可但非专长） | 超出执业范围的治疗 |
| **临床心理师** | 心理评估；心理治疗（CBT/IPT/ACT/MBCT）；个案概念化 | 精神科诊断确认；药物处方；ECT | 诊断；处方；ECT |
| **注册心理治疗师/咨询师** | 心理治疗；支持性咨询；心理教育 | 精神科评估；药物；危机处理 | 诊断；处方；ECT |
| **精神科护士** | 护理评估；药物管理；患者教育；危机联络 | 诊断；独立处方（部分国家可） | 超出培训范围的治疗 |
| **社工** | 社会评估；资源链接；家庭工作；案例管理 | 诊断；心理治疗（有限制） | 诊断；处方 |

### 3.2 知情同意标准

| 要素 | 内容 |
|------|------|
| **诊断信息** | 诊断名称、严重程度、预后 |
| **治疗选择** | 所有可用治疗选项（药物、心理、物理） |
| **推荐方案** | 医生建议的首选方案及理由 |
| **预期效果** | 可能的疗效；起效时间；治愈率 |
| **潜在风险** | 药物不良反应；心理治疗中的情绪恶化；物理治疗风险 |
| **替代方案** | 其他可用选项及其利弊 |
| **不接受治疗的后果** | 自然病程；可能恶化 |
| **费用** | 治疗相关费用 |
| **保密限制** | 何时需要突破保密（自杀风险、虐待等） |
| **退出权利** | 患者有权随时退出治疗 |

### 3.3 强制治疗与保护性强制住院

| 情况 | 法律依据 | 程序 |
|------|---------|------|
| **自杀风险极高** | 精神卫生法 | 精神科评估 → 家属同意/强制住院程序 |
| **自伤风险极高** | 精神卫生法 | 同上 |
| **严重拒食危及生命** | 精神卫生法 | 同上；可能需要鼻饲 |
| **伤害他人风险** | 精神卫生法/刑法 | 报警 + 精神科急诊 + 强制住院 |
| **无行为能力** | 民法/监护法 | 监护人决策 |

## 四、文档记录标准

### 4.1 必需文档

| 文档 | 内容 | 保存期限 |
|------|------|---------|
| **初诊评估报告** | 主诉、病史、精神状态检查、诊断、风险分层、治疗计划 | ≥15年 |
| **治疗记录** | 每次会谈内容、症状变化、治疗调整、家庭作业 | ≥15年 |
| **量表结果** | 所有标准化评估的原始分数和解释 | ≥15年 |
| **安全计划** | 六步安全计划、危机卡片、 lethal means 评估 | 随个案 |
| **知情同意** | 签名版知情同意书 | ≥15年 |
| **危机事件记录** | 时间、事件描述、处理措施、后续安排 | ≥15年 |
| **转介记录** | 转介原因、接收方信息、转介内容 | ≥15年 |
| **终止/出院总结** | 治疗总结、当前状态、后续建议、复发预防计划 | ≥15年 |

### 5.1 团队角色

| 角色 | 核心职责 |

## 六、质量改进指标

### 6.1 关键绩效指标（KPIs）

| 指标 | 目标值 | 监测频率 |
|------|--------|---------|
| **初诊至治疗开始时间** | ≤2周 | 每月 |
| **急性期治愈率（Remission）** | ≥30%（12周） | 每季度 |
| **急性期有效率（Response）** | ≥50%（12周） | 每季度 |
| **自杀风险评估完成率** | 100% | 持续 |
| **安全计划制定率（中高风险）** | 100% | 持续 |
| **量表使用率** | ≥80% | 每月 |
| **随访完成率** | ≥75% | 每月 |
| **患者满意度** | ≥4.0/5.0 | 每季度 |
| **治疗脱落率** | ≤25% | 每季度 |
| **出院后7天内随访率** | ≥90% | 每月 |

### 6.2 质量审查流程

| 类型 | 内容 | 频率 |
|------|------|------|
| **病例回顾** | 随机抽取病例审查文档完整性、诊断准确性、治疗规范性 | 每月 |
| **不良事件分析** | 所有自杀attempt、住院、治疗相关严重事件 | 实时 |
| **患者反馈** | 满意度调查、投诉分析 | 每季度 |
| ** outcome 分析** | 治愈率、复发率、功能恢复率 | 每年 |
| **同行评审** | 复杂病例的多学科讨论 | 按需 |
