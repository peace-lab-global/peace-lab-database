# 抑郁症精神科标准评估组合 | Psychiatric Assessment Battery for Depression

> **目标**：建立符合精神科临床标准的抑郁症评估体系，涵盖结构化访谈、症状量表、功能评估、认知筛查、自杀风险与生物标志物。
> **适用对象**：精神科医生、临床心理师、精神科规培生。

---

## 一、结构化临床访谈

### 1.1 入组/初诊评估清单

| 模块 | 工具/内容 | 目的 | 执行者 |
|------|----------|------|--------|
| **轴I障碍筛查** | SCID-5-CV 或 MINI 7.0 | 确认MDD诊断；排除双相、精神病性障碍、物质使用 | 精神科医生/临床心理师 |
| **自杀风险评估** | C-SSRS + 临床深度访谈 | 量化自杀意念、计划、企图、意图 | 精神科医生 |
| **精神病性症状筛查** | PANSS阳性子量表 或 临床访谈 | 排除活跃期精神病性症状 | 精神科医生 |
| **轻躁狂/躁狂史** | MDQ + HCL-32 + 临床访谈 | 排除未识别双相谱系 | 临床心理师 |
| **焦虑共病筛查** | GAD-7 + 临床访谈 | 评估焦虑严重度及类型 | 临床心理师 |
| **创伤史** | PCL-5 + LEC-5 | 评估PTSD/C-PTSD共病 | 临床心理师 |
| **人格评估** | SCID-5-PD 或 PID-5 简版 | 识别人格病理对治疗的影响 | 临床心理师 |
| **躯体疾病筛查** | 病史采集 + 体格检查 + 实验室检查 | 排除躯体疾病所致抑郁 | 精神科医生 |
| **物质使用筛查** | AUDIT + DAST-10 + 尿液毒理 | 排除物质所致抑郁 | 精神科医生 |
| **认知功能筛查** | MMSE 或 MoCA | 排除认知障碍；老年患者必做 | 临床心理师 |
| **功能评估** | SDS + WHODAS 2.0 | 工作、社交、家庭功能 | 临床心理师 |

### 1.2 必需实验室检查

| 检查项目 | 目的 | 所有患者 | 初诊老年/躯体症状突出 |
|---------|------|---------|---------------------|
| **血常规** | 排除贫血、感染 | ✅ | ✅ |
| **甲状腺功能（TSH、FT3、FT4）** | 排除甲减/甲亢 | ✅ | ✅ |
| **肝肾功能** | 用药前基线；排除代谢异常 | ✅ | ✅ |
| **空腹血糖/HbA1c** | 排除糖尿病；糖尿病共病筛查 | ✅ | ✅ |
| **电解质** | 排除低钠等；用药安全 | ✅ | ✅ |
| **维生素B12、叶酸** | 排除营养缺乏 | ✅ | ✅ |
| **25-OH维生素D** | 低维生素D与抑郁相关 | 建议 | ✅ |
| **CRP/ESR** | 排除炎症/自身免疫 | 建议 | ✅ |
| **心电图** | 三环类抗抑郁药等心脏安全性 | 用药前 | ✅ |
| **药物浓度（如适用）** | 监测治疗药物浓度 | 必要时 | 必要时 |
| **尿液毒理筛查** | 排除物质使用 | 怀疑时 | 怀疑时 |
| **脑MRI/CT** | 排除神经系统疾病 | 神经系统体征时 | ✅ |
| **性激素** | 排除内分泌相关抑郁 | 围产期/更年期 | 必要时 |

## 二、症状严重程度量表

### 2.1 自评量表

#### PHQ-9（患者健康问卷-9项）

| 项目 | 内容 |
|------|------|
| **用途** | 抑郁症状筛查、严重程度评估、治疗反应追踪 |
| **题数** | 9（对应DSM-5的9项症状） |
| **评分** | 0-3四级评分，总分0-27 |
| **临界值** | 5-9轻度；10-14中度；15-19中重度；20-27重度 |
| **敏感项目** | 第9题（自杀意念）：≥1分需进一步临床评估 |
| **最小临床重要差异（MCID）** | 下降≥5分 |

**使用建议**：
- 初诊：建立基线
- 治疗期：每2-4周施测
- 维持期：每月施测，直至稳定后每3月

#### BDI-II（贝克抑郁量表-II）

| 项目 | 内容 |
|------|------|
| **用途** | 抑郁严重程度深度评估；对认知症状敏感 |
| **题数** | 21 |
| **评分** | 0-3四级评分，总分0-63 |
| **临界值** | 0-13极轻；14-19轻度；20-28中度；29-63重度 |
| **优势** | 对反刍、消极自我概念、认知扭曲敏感 |
| **MCID** | 下降≥8分 |

#### SDS（Zung抑郁自评量表）

| 项目 | 内容 |
|------|------|
| **用途** | 快速筛查；患者自我监测 |
| **题数** | 20 |
| **评分** | 1-4四级评分，原始分×1.25得标准分 |
| **临界值** | <50无抑郁；50-59轻度；60-69中度；≥70重度 |

### 2.2 他评量表

#### HAMD-17（汉密尔顿抑郁量表-17项）

| 项目 | 内容 |
|------|------|
| **用途** | 精神科临床标准；研究金标准 |
| **题数** | 17（另有HAMD-21、HAMD-24版本） |
| **评分** | 0-2或0-4级评分 |
| **临界值** | 0-7正常；8-16轻度；17-23中度；≥24重度 |
| **特点** | 对焦虑躯体化、失眠、认知障碍、迟滞评估详细 |
| **MCID** | 下降≥50%为有效；≤7分为临床治愈 |

#### MADRS（蒙哥马利-艾斯伯格抑郁评定量表）

| 项目 | 内容 |
|------|------|
| **用途** | 研究首选；对变化敏感 |
| **题数** | 10 |
| **评分** | 0-6六级评分，总分0-60 |
| **临界值** | 0-6正常；7-19轻度；20-34中度；35-60重度 |
| **特点** | 核心症状聚焦；不含焦虑/躯体症状；对药物试验最敏感 |
| **MCID** | 下降≥50%为有效；≤10分为临床治愈 |

#### CGI-S / CGI-I（临床总体印象量表）

| 量表 | 用途 |
|------|------|
| **CGI-S** | 1-7级严重程度评定（1=正常，7=极重度） |
| **CGI-I** | 1-7级改善程度评定（1=明显改善，4=无变化，7=明显恶化） |
| **CGI-E** | 疗效指数（疗效/副作用权衡） |

### 2.3 专项量表

| 量表 | 评估维度 | 适用场景 |
|------|---------|---------|
| **QIDS-SR-16** | 快速抑郁症状自评 | 网络评估；临床快速筛查 |
| **IDS-SR / IDS-C** | 30项抑郁症状清单（自评/临床版） | 全面症状评估；研究 |
| **HAMD-6** | HAMD核心6项 | 超快速疗效监测 |
| **MADRS-S** | MADRS自评版 | 患者自我追踪 |
| **SHAPS** | 快感缺失量表 | 评估anhedonia核心症状 |
| **PDI** | 妄想观念量表 | 评估精神病性特征 |

## 三、功能评估

### 3.1 工作/社会功能

| 量表 | 内容 | 用途 |
|------|------|------|
| **SDS（Sheehan残疾量表）** | 工作、社交、家庭/生活功能，每项0-10 | 快速功能评估；MCID=下降≥1分/项 |
| **WHO-DAS 2.0** | 36项全面功能评估 | 残疾评定；研究 |
| **FAST（功能评估简式测验）** | 功能领域快速评估 | 双相障碍功能评估也可用 |
| **SAS-SR（社会适应量表-自评）** | 多领域社会功能 | 长期追踪 |

### 3.2 生活质量

| 量表 | 内容 |
|------|------|
| **QOLIE** | 生活质量综合评估 |
| **SF-36** | 健康相关生活质量 |
| **EuroQOL-5D** | 卫生经济学研究常用 |

## 四、自杀风险评估

### 4.1 C-SSRS（哥伦比亚自杀严重程度评定量表）

| 评估域 | 内容 | 行动阈值 |
|--------|------|---------|
| **自杀意念** | wish to be dead, non-specific active suicidal thoughts | 任何肯定回答需进一步评估 |
| **有方法无计划** | 有具体方法但无实施计划 | 中高风险 |
| **有具体计划** | 有明确计划和意图 | 高风险；立即干预 |
| **自杀行为** | 实际未遂行为 | 极高风险 |
| **自伤（非自杀意图）** | NSSI | 评估功能；安全计划 |

### 4.2 其他自杀风险评估工具

| 工具 | 特点 | 适用 |
|------|------|------|
| **SSI（自杀意念量表）** | 19项，评估意念强度和特征 | 研究；临床深度评估 |
| **BSS（贝克自杀意念量表）** | 19项，评估意念严重度 | 研究；住院患者 |
| **RASS（重复自杀_attempt评估）** | 预测再次自杀_attempt风险 | 急诊；出院前 |
| **NGASR（护士用自杀风险评估量表）** | 15项临床指标 | 住院患者护理评估 |

## 五、认知功能评估

| 工具 | 用途 | 临界值 |
|------|------|--------|
| **MMSE** | 认知功能快速筛查 | <24分异常 |
| **MoCA** | 轻度认知障碍筛查（比MMSE敏感） | <22分提示MCI |
| **HVLT-R** | 言语记忆 | 抑郁假性痴呆评估 |
| **WCST** | 执行功能 | 前额叶功能 |
| **TMT-A/B** | 处理速度/执行功能 | B-A时间差反映执行功能 |
| **DSF/DSB** | 工作记忆 | 数字顺背/倒背 |
| **CPT** | 持续注意 | 注意缺陷评估 |

**抑郁假性痴呆 vs 真性痴呆鉴别**：
- 抑郁假性痴呆：起病快、患者主诉认知下降突出、回答"不知道"多、晨重夜轻、治疗后可逆
- 真性痴呆：起病隐匿、家属发现为主、回答似是而非、进行性地恶化

## 六、整合评估时间表

| 时间点 | 评估内容 | 目的 |
|--------|---------|------|
| **初诊（T0）** | SCID/MINI + PHQ-9 + BDI-II + GAD-7 + C-SSRS + SDS + MMSE/MoCA + 实验室检查 | 全面基线；诊断确认；风险分层 |
| **治疗2周** | PHQ-9 + C-SSRS（简短） + 药物不良反应 | 早期疗效；安全性 |
| **治疗4周** | PHQ-9 + HAMD-17/MADRS + SDS + 不良反应 | 疗效评估；剂量调整 |
| **治疗8-12周** | 全部量表重测 + CGI-I | 急性期终点评估 |
| **维持期（每月）** | PHQ-9 + SDS + C-SSRS | 维持监测；复发预警 |
| **维持期（每3月）** | BDI-II + HAMD-17/MADRS + 功能评估 | 深度评估 |
| **停药前** | 全面评估 + 复发风险评估 | 停药决策 |
| **停药后3/6/12月** | PHQ-9 + C-SSRS + SDS | 复发监测 |
