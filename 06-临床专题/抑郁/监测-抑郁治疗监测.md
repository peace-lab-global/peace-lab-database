# 抑郁症治疗监测与预后 | Depression Treatment Monitoring and Prognosis

> **目标**：建立精神科标准的抑郁症治疗监测体系，涵盖疗效评估、换药时机、维持治疗决策、复发监测与长期预后因素。

---

## 一、治疗反应定义

### 1.1 疗效标准

| 标准 | 定义 | 测量工具 |
|------|------|---------|
| **临床治愈（Remission）** | 症状完全或几乎完全消失；功能恢复 | HAMD-17 ≤ 7；MADRS ≤ 10；PHQ-9 < 5；SDS ≤ 6 |
| **有效（Response）** | 症状显著改善但未完全消失 | 量表分数下降 ≥ 50%（如HAMD-17）；CGI-I ≤ 2 |
| **部分应答（Partial Response）** | 症状有改善但不充分 | 量表分数下降 25-49% |
| **无应答（Non-Response）** | 症状无显著改善 | 量表分数下降 < 25% |
| **恶化** | 症状加重 | 量表分数增加 ≥ 25% |

### 1.2 功能恢复标准

| 领域 | 标准 |
|------|------|
| **工作/学业** | 恢复至病前水平或接近；SDS工作项 ≤ 2 |
| **社交** | 恢复社交活动；SDS社交项 ≤ 2 |
| **家庭/生活** | 履行家庭责任；SDS家庭项 ≤ 2 |
| **自我照顾** | 日常生活自理 |

---

## 二、急性期监测流程

### 2.1 监测时间表

| 时间点 | 评估内容 | 决策 |
|--------|---------|------|
| **基线（T0）** | HAMD-17/MADRS + PHQ-9 + SDS + C-SSRS + 功能评估 | 建立基线；制定治疗计划 |
| **第2周** | PHQ-9 + 不良反应 + C-SSRS | 早期疗效信号；安全性 |
| **第4周** | HAMD-17/MADRS + PHQ-9 + SDS + 不良反应 | 疗效评估；考虑剂量调整 |
| **第6-8周** | HAMD-17/MADRS + PHQ-9 + SDS + CGI-I + 不良反应 | **关键决策点**：是否换药/增效 |
| **第12周** | 全面评估 | 急性期终点 |

### 2.2 早期疗效预测

| 时间点 | 预测价值 | 临床意义 |
|--------|---------|---------|
| **第2周PHQ-9下降 ≥ 20%** | 预测第8周有效概率 > 80% | 继续当前方案 |
| **第2周PHQ-9无变化** | 预测第8周有效概率 < 30% | 需调整方案（换药或增效） |
| **第4周HAMD下降 ≥ 30%** | 预测第8周治愈概率 > 60% | 继续当前方案 |
| **第4周无改善** | 强烈预测最终无应答 | 考虑换药 |

**早期无改善的应对**：
- 确认诊断正确
- 确认足量足疗程
- 确认依从性
- 评估共病
- 考虑药物浓度监测（如怀疑代谢问题）
- 考虑换药或增效

---

## 三、换药与增效决策

### 3.1 换药（Switching）策略

| 情况 | 策略 |
|------|------|
| **SSRI无效** | 换另一种SSRI（50%对第二种有效）或换SNRI/米氮平/安非他酮 |
| **SSRI不耐受** | 换不同类别（如SSRI→米氮平/安非他酮） |
| **两种抗抑郁药无效** | 考虑TCA、MAOI、ECT、rTMS |

**STAR*D研究启示**：
- 第1步（SSRI）：约30-40%治愈
- 第2步（换药或增效）：额外20-30%治愈
- 第3步（再次换药）：额外10-20%治愈
- 第4步（MAOI/TCA）：额外10%治愈

### 3.2 增效（Augmentation）策略

| 增效时机 | 策略 |
|---------|------|
| **部分应答（下降25-49%）** | 增效优于换药 |
| **无效（下降<25%）** | 换药优于增效 |

| 增效方案 | 证据 | 首选场景 |
|---------|------|---------|
| **阿立哌唑** | A级 | 标准增效首选 |
| **喹硫平XR** | A级 | 伴失眠/焦虑 |
| **锂盐** | A级 | 双相谱系；难治性 |
| **甲状腺素（T3）** | B级 | 伴亚临床甲减 |
| **安非他酮** | B级 | 伴性功能障碍/疲乏 |
| **米氮平** | B级 | 伴失眠/食欲差 |
| **非典型抗精神病药** | A级 | 伴精神病性特征 |

## 四、维持期与复发预防

### 4.1 维持治疗决策

| 复发史 | 建议维持时间 | 剂量 |
|--------|------------|------|
| **首次发作，完全缓解** | 6-12月 | 急性期有效剂量 |
| **2次发作** | 2-3年 | 急性期有效剂量 |
| **≥3次发作** | 长期（可能终身） | 急性期有效剂量 |
| **慢性抑郁（≥2年）** | 长期 | 急性期有效剂量 |
| **伴精神病性特征** | 长期 | 抗抑郁药+抗精神病药 |
| **伴严重自杀attempt史** | 长期 | 急性期有效剂量；考虑锂盐 |

### 4.2 复发预警信号

| 类型 | 信号 | 应对 |
|------|------|------|
| **早期信号（数天-1周）** | 睡眠变差、精力下降、兴趣减退、反刍增加、社交退缩 | 增加随访频率；强化自我监测；调整生活方式 |
| **中期信号（1-2周）** | PHQ-9上升3-5分、工作效率下降、情绪波动增加 | 加强心理治疗；考虑药物剂量调整；安全评估 |
| **晚期信号（>2周）** | PHQ-9上升>5分、功能明显受损、自杀意念出现 | 精神科紧急评估；考虑恢复/加强治疗；住院评估 |

### 4.3 复发应对

```
复发信号出现
    │
    ├── 轻度波动（PHQ-9上升<5分，<1周）
    │       → 加强监测；增加心理社会支持；生活方式调整
    │
    ├── 中度恶化（PHQ-9上升3-5分，1-2周）
    │       → 增加随访（每周）；强化心理治疗；考虑药物剂量调整
    │
    └── 严重复发（PHQ-9上升>5分，>2周，功能受损）
            → 精神科评估；恢复急性期治疗强度；考虑住院；安全评估
```

## 五、难治性抑郁（TRD）

### 5.1 定义

| 定义 | 标准 |
|------|------|
| **传统定义** | 至少两种足量足疗程（≥4-6周）抗抑郁药无效 |
| **更严格定义** | 两种不同类别药物无效 |
| **最严格定义** | 药物+心理治疗+ECT均无效 |

### 5.2 TRD评估清单

| 评估项 | 内容 |
|--------|------|
| **诊断准确性** | 是否遗漏双相、精神病性特征、人格障碍？ |
| **治疗依从性** | 患者是否真正按处方服药？ |
| **剂量充分性** | 是否达到最高推荐剂量？ |
| **疗程充分性** | 是否至少6-8周？ |
| **药物浓度** | 血药浓度是否在治疗范围？（尤其代谢快者） |
| **共病** | 物质使用、甲状腺问题、睡眠呼吸暂停、慢性疼痛 |
| **心理治疗** | 是否接受过足量CBT/IPT/ACT？ |
| **社会因素** | 持续应激、虐待、贫困、孤立 |
| **生物学因素** | 炎症标志物、神经内分泌、基因多态性 |

### 5.3 TRD治疗阶梯

| 阶梯 | 治疗 | 证据 |
|------|------|------|
| **1** | 换用另一种SSRI或SNRI | A级 |
| **2** | 增效（阿立哌唑、喹硫平、锂盐、甲状腺素） | A级 |
| **3** | 联合两种抗抑郁药（SSRI+米氮平/安非他酮） | A级 |
| **4** | TCA（氯咪帕明、阿米替林） | A级 |
| **5** | MAOI（苯乙肼、反苯环丙胺） | A级 |
| **6** | ECT | A级 |
| **7** | rTMS | A级 |
| **8** | 艾司氯胺酮 | A级（FDA批准TRD） |
| **9** | VNS / DBS | B级（试验性） |
| **整合** | 运动 + 光照 + 睡眠治疗 + 营养 + 社会节律 | B级 |

## 六、预后因素

### 6.1 良好预后因素

| 因素 | 说明 |
|------|------|
| **首次发作** | 复发风险低 |
| **急性起病** | 对治疗反应好 |
| **明显应激诱因** | 问题解决后预后好 |
| **无共病** | 治疗反应单纯 |
| **良好社会支持** | 缓冲应激 |
| **良好治疗依从性** | 疗效维持 |
| **完全缓解** | 功能恢复好 |
| **认知功能保留** | 心理治疗效果好 |

### 6.2 不良预后因素

| 因素 | 说明 |
|------|------|
| **早发（<20岁）** | 可能提示双相谱系；病程更长 |
| **多次复发（≥3次）** | 复发风险递增 |
| **慢性抑郁（≥2年）** | 治疗反应差 |
| **共病焦虑/物质使用/人格障碍** | 复杂化治疗 |
| **残留症状** | 预示早期复发 |
| **失眠残留** | 强烈预示复发 |
| **认知损害** | 功能恢复差 |
| **严重功能损害** | 恢复慢 |
| **缺乏社会支持** | 缓冲差 |
| **持续应激** | 维持症状 |
