# 创伤知情照护 (Trauma-Informed Care, TIC)

> **核心定位**：不是特定疗法，而是**组织和系统层面的范式转变**。从"你有什么问题？"转变为"你经历了什么？"。SAMHSA 四原则框架指导所有服务接触。

---

## 1. SAMHSA 四 R 原则

| 原则 | 说明 |
|:-----|:-----|
| **Realize** 认知 | 理解创伤的普遍性和影响 |
| **Recognize** 识别 | 识别服务使用者/员工的创伤迹象 |
| **Respond** 回应 | 将创伤知识融入政策/程序/实践 |
| **Resist** 预防 | 主动预防再创伤（retraumatization） |

---

## 2. 六项关键原则

| # | 原则 | 实践 |
|:--|:-----|:-----|
| 1 | **安全** | 物理和情感安全 |
| 2 | **信任与透明** | 决策透明/一致性 |
| 3 | **同伴支持** | 有创伤经历者参与 |
| 4 | **协作与选择** | 共享决策/尊重自主 |
| 5 | **赋权/声音/选择** | 强化优势/能力建设 |
| 6 | **文化/历史/性别议题** | 承认历史创伤/文化敏感 |

---

## 3. 再创伤风险

| 风险因素 | 说明 | 预防 |
|:---------|:-----|:-----|
| 权力不对等 | 治疗师vs患者 | 共享决策/透明 |
| 身体接触 | 未经同意触碰 | 事先征得同意 |
| 限制自由 | 约束/隔离 | 最小化/替代方案 |
| 不信任环境 | 嘈杂/拥挤/不可预测 | 安静/可预测/安全空间 |
| 忽视创伤史 | 不询问/不记录 | 常规筛查/知情处理 |

---

## 4. 在不同系统中的应用

| 系统 | TIC 实践 |
|:-----|:---------|
| **医疗** | 创伤史筛查/最小化再创伤医疗程序 |
| **学校** | 创伤敏感课堂/行为理解/安全计划 |
| **司法** | 创伤知情法院/替代惩罚/康复导向 |
| **社工** | 优势视角/赋权/系统性支持 |
| **急诊** | 创伤筛查/安全环境/同伴支持 |
| **精神科** | 减少约束/协作安全计划 |

---

## 6. TIC vs 创伤治疗

| 维度 | TIC (创伤知情) | 创伤治疗 (PE/CPT/EMDR) |
|:-----|:---------------|:----------------------|
| 层面 | 组织/系统 | 个体/团体 |
| 目标 | 避免再创伤/创建安全环境 | 处理创伤记忆 |
| 实施者 | 所有工作人员 | 受过培训的治疗师 |
| 范围 | 所有服务接触 | 特定治疗节次 |
| 关系 | 必要前提 | 在 TIC 环境中更有效 |
