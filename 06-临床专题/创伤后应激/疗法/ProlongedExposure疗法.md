# 延长暴露疗法 (Prolonged Exposure Therapy, PE)

> **核心定位**：由 Edna Foa 开发的一线索证创伤治疗。基于**情绪加工理论**，通过**想象暴露**和**现场暴露**帮助患者加工创伤记忆，减少回避和恐惧。

---

## 1. 理论基础：情绪加工理论

| 概念 | 说明 |
|:-----|:-----|
| 恐惧结构 | 创伤形成包含刺激-反应-意义的恐惧网络 |
| 病理性回避 | 回避阻止恐惧结构被激活和修正 |
| 情绪加工 | 激活恐惧结构→提供不兼容信息→修正恐惧结构 |
| 习惯化 | 重复暴露→焦虑自然降低 |
| 认知修正 | "世界完全危险"→"大多数情境是安全的" |

---

## 2. 治疗结构（8-15 次，标准 10 次）

| 节次 | 内容 | 时长 |
|:----:|:-----|:----:|
| 1 | 心理教育/治疗理由/呼吸训练 | 90 min |
| 2 | 现场暴露层级构建/创伤概述 | 90 min |
| 3 | 第一次想象暴露（创伤记忆叙述） | 90 min |
| 4-10 | 想象暴露 + 现场暴露（每周） | 60-90 min |
| 终期 | 回顾进展/复发预防 | 60 min |

---

## 3. 核心技术

### 3.1 想象暴露 (Imaginal Exposure)

| 要素 | 说明 |
|:-----|:-----|
| 方法 | 闭眼大声叙述最痛苦的创伤记忆（现在时态） |
| 时长 | 每次 30-45 分钟 |
| SUD 评分 | 每 5 分钟记录主观痛苦单位（0-100） |
| 重复 | 节内重复+家庭作业听录音 |
| 热记忆 | 聚焦最具情绪冲击力的片段（Hot Spot） |

### 3.2 现场暴露 (In Vivo Exposure)

| 要素 | 说明 |
|:-----|:-----|
| 方法 | 渐进面对安全但被回避的情境 |
| 层级 | 共同构建 SUDS 层级（0-100） |
| 家庭作业 | 每周练习 3-5 次 |
| 升级 | SUDS 降至 ~20 时升级下一项目 |

### 3.3 呼吸再训练

| 技术 | 说明 |
|:-----|:-----|
| 慢呼吸 | 4 秒吸 / 6 秒呼 |
| 用途 | 焦虑管理工具（非暴露替代） |
| 时机 | 第一节教授，贯穿全程 |

---

## 4. 疗效证据

| 研究/数据 | 发现 |
|:----------|:-----|
| APA 指南 (2017) | **强烈推荐** PTSD 治疗 |
| ISTSS 指南 | **一级推荐** |
| 效应量 | d = 1.08-1.26（vs 等待对照） |
| 缓解率 | 60-80% 不再符合 PTSD 诊断 |
| 长期随访 | 2 年维持效果 |
| 跨群体 | 退伍军人/性侵犯/车祸/恐怖袭击幸存者 |

---

## 5. 适应症与禁忌

| 适应症 | 禁忌/需调整 |
|:-------|:-----------|
| 单事件 PTSD | 急性解离（需先稳定化） |
| 多种创伤 PTSD | 正在持续受虐（需先安全） |
| 共病抑郁/焦虑 | 严重自杀意念（需先稳定） |
| 物质滥用（稳定后） | 活动期物质滥用 |
| 复杂 PTSD（先 Phase 1） | 精神病 |

---

## 6. PE vs CPT vs EMDR

| 维度 | PE | CPT | EMDR |
|:-----|:---|:----|:-----|
| 核心机制 | 暴露/习惯化 | 认知重建 | 双侧刺激/AIP |
| 家庭作业 | 录音+现场暴露 | 工作表 | 日志 |
| 创伤叙述 | 大声详细叙述 | 书面影响陈述 | 简略描述 |
| 时长 | 8-15 次 | 12 次 | 8-12 次 |
| 适合人群 | 回避为主 | 认知扭曲为主 | 两者兼可 |
