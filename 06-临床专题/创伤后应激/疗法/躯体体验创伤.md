# 躯体体验疗法 (Somatic Experiencing, SE)

> **核心定位**：由 Peter Levine 开发的躯体导向创伤疗法。核心理念：创伤不是事件本身，而是**神经系统未完成反应**的残留。通过追踪躯体感觉完成被冻结的应激反应。

---

## 1. 理论基础

| 概念 | 说明 |
|:-----|:-----|
| 冻结反应 | 动物在无法战斗/逃跑时"假死"→人类类似 |
| 未完成反应 | 创伤时战斗/逃跑反应被抑制→能量"卡住"在神经系统 |
| 多迷走神经理论 | Porges 三层自主神经系统（社会参与/交感/背侧迷走） |
| 自组织 | 神经系统具有自然完成和释放反应的能力 |
| 标题性体验 | 通过追踪身体感觉（而非认知/情感）释放创伤能量 |

---

## 2. 核心技术

| 技术 | 说明 |
|:-----|:-----|
| **Pendulation** | 在安全感觉和创伤感觉之间来回摆动 |
| **Titration** | 微量接触创伤（避免淹没） |
| **Tracking** | 追踪躯体感觉变化 |
| **Completion** | 完成冻结的应激反应（推/跑/抖/叫） |
| **资源发展** | 建立内在/外在安全资源 |
| **Voo 音** | Levine 腹侧迷走神经激活技术 |

---

## 3. 治疗过程

```
Phase 1: 安全与资源
├─ 建立安全处（内在/外在）
├─ 识别资源（身体/关系/记忆）
└─ 教授 Pendulation 基本技术

Phase 2: 追踪与释放
├─ 微量接触创伤相关感觉
├─ 追踪身体感觉变化（热/冷/紧/松/振动）
├─ 完成冻结反应（推/跑/抖/颤）
└─ 自发性释放（深呼吸/颤抖/温热/流动）

Phase 3: 整合
├─ 整合新的身体体验
├─ 重建健康的自主神经调节
└─ 扩展窗 (Window of Tolerance)
```

---

## 4. 窗口耐受 (Window of Tolerance)

| 状态 | 描述 | SE 干预 |
|:-----|:-----|:--------|
| **过度激活** | 焦虑/惊恐/过度警觉/闪回 | Pendulation 到安全/接地 |
| **窗口内** | 能够处理和整合体验 | 追踪/Titration |
| **低度激活** | 麻木/解离/退缩/无动力 | 激活资源/温和刺激 |

---

## 5. 证据状态

| 数据 | 说明 |
|:-----|:-----|
| RCT 数量 | 有限（vs PE/CPT/EMDR） |
| 初步证据 | Payne et al. 2015：SE 减少 PTSD 症状 |
| 案例研究 | 大量积极案例报告 |
| 机制证据 | 多迷走神经理论支持 |
| 临床接受度 | 广泛用于创伤/疼痛/慢性应激 |
| 批评 | 缺乏大规模 RCT/机制难以验证 |

---

## 6. SE 在治疗体系中的定位

| 定位 | 说明 |
|:-----|:-----|
| 补充 | 与 PE/CPT/EMDR 互补（处理躯体维度） |
| 稳定化 | 复杂创伤 Phase 1 的理想方法 |
| 解离 | 解离倾向患者的首选 |
| 躯体化 | 创伤躯体症状（慢性疼痛/GI） |
| 婴幼儿 | 无法用言语处理创伤的群体 |
