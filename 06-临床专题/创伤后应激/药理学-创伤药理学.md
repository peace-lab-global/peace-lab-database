# 创伤药物治疗

**总体定位**：药物是 PTSD 治疗的**二线/辅助**，一线为创伤聚焦心理治疗（TF-CBT/PE/CPT/EMDR）。药物适用于：(1) 患者拒绝/无法获得心理治疗；(2) 严重共病抑郁/焦虑需药物稳定后才能进入心理治疗；(3) 症状严重干扰心理治疗参与（如严重失眠、激越）；(4) 心理治疗部分反应后的增效。

---

## 一、FDA 批准的一线药物

仅**两种**药物获 FDA 批准用于 PTSD：

| 药物 | 类别 | 起始剂量 | 目标剂量 | 最大剂量 | 机制 |
|------|------|---------|---------|---------|------|
| **舍曲林 Sertraline** | SSRI | 25–50 mg/d | 100 mg/d | 200 mg/d | 抑制 5-HT 再摄取 |
| **帕罗西汀 Paroxetine** | SSRI | 10–20 mg/d | 20–50 mg/d | 60 mg/d | 抑制 5-HT 再摄取（最强 5-HT 再摄取抑制） |

**证据**：舍曲林与帕罗西汀在多项 RCT 中显著优于安慰剂；效应量中等（d ≈ 0.4–0.5）。退伍军人人群效应较弱（VA 合作研究显示舍曲林与安慰剂无显著差异），提示**战斗相关 PTSD 药物反应较差**。

**指南立场**：
- APA (2017)：SSRI 为一线药物推荐。
- ISTSS (2019)：药物推荐 B 级，**低于**创伤聚焦心理治疗。
- NICE NG116 (2018)：**药物不作为首选**；仅在患者拒绝心理治疗或无效时考虑。

## 二、其他 SSRI / SNRI（超适应症但临床常用）

| 药物 | 类别 | 剂量范围 | 备注 |
|------|------|---------|------|
| **氟西汀 Fluoxetine** | SSRI | 20–60 mg/d | 半衰期长，停药反应小；有 RCT 阳性证据 |
| **文拉法辛 Venlafaxine** | SNRI | 75–225 mg/d | 双重再摄取抑制；对伴随焦虑/抑郁有效 |
| **西酞普兰 / 艾司西酞普兰** | SSRI | 10–20 mg/d / 10–20 mg | 耐受性好；超适应症使用 |

**选择原则**：
- 共病抑郁/焦虑/惊恐 → SSRI（舍曲林/艾司西酞普兰耐受佳）。
- 共病慢性疼痛 → SNRI（文拉法辛/度洛西汀）。
- 年老/多种药物相互作用顾虑 → 西酞普兰/舍曲林（CYP 相互作用少）。
- **避免帕罗西汀**于：育龄女性（致畸 D 类）、老年人（抗胆碱、跌倒）、需要快速停药者（短半衰期，戒断重）。

## 三、哌唑嗪（Prazosin）—— 创伤相关噩梦

| 项目 | 内容 |
|------|------|
| **机制** | α-1 肾上腺素受体拮抗剂；降低 REM 期去甲肾上腺素过度激活 |
| **适应症** | PTSD 相关**反复创伤性噩梦**与夜间惊恐（超适应症；无 FDA 批准） |
| **剂量** | 起始 1–2 mg 睡前；每周滴定至目标 5–15 mg 睡前；部分需 10–20 mg |
| **滴定** | 必须缓慢（首剂效应：体位性低血压、晕厥） |
| **证据** | 多项 RCT 阳性；但 **VA PACT 试验（2018）** 显示主要终点阴性 → 争议。临床共识：对**噩梦为主诉**者仍常用，部分患者显著获益 |
| **联用注意** | 与降压药、PDE5 抑制剂联用增加低血压风险 |
| **禁忌** | 严重体位性低血压、左心衰 |

## 四、增效与替代策略

当 SSRI/SNRI 单药 8–12 周部分反应，可考虑：

| 策略 | 药物/方法 | 证据 |
|------|---------|------|
| **非典型抗精神病（低剂量）** | 利培酮 0.5–2 mg、喹硫平 25–100 mg | 用于闪回/解离/激越；证据有限；警惕代谢副作用 |
| **心境稳定剂** | 拉莫三嗪 25–200 mg、托吡酯 | 共病双相/冲动控制；证据中等 |
| **丁螺环酮** | 15–60 mg/d | 共病 GAD；增强 SSRI（ augmentation） |
| **米氮平** | 15–45 mg 睡前 | 共病失眠/体重下降；耐受好 |
| **纳曲酮** | 见 Addiction 专题 | 共病酒精使用障碍时 |

**不推荐/证据不足**：
- 苯二氮䓬类（BZD）：**长期使用争议大**；不改善核心 PTSD 症状、增加解离与物质使用风险、妨碍暴露治疗的恐惧消退。仅用于急性短期危机稳定，**避免长期处方**。
- 单纯去甲肾上腺素能（如阿替洛尔）证据不足。

## 五、黑框警告与安全性

### FDA 黑框警告（适用于所有抗抑郁药，含 SSRI/SNRI）
- ** suicidality**：在儿童、青少年、年轻成人（≤24 岁）中监测自杀意念/行为加重。需在第 1–4 周密切随访。
- 启动药物后**前 4 周**密切监测激越、失眠、冲动、自杀意念。

### SSRI/SNRI 共性副作用与处理

| 副作用 | 处理 |
|--------|------|
| 性功能障碍（性欲↓/高潮延迟） | 与患者讨论；可换安非他酮或加用安非他酮 |
| 胃肠道反应（恶心） | 随餐服；1–2 周耐受 |
| 失眠/嗜睡 | 调整给药时间；舍曲林/氟西汀晨服 |
| 体重变化 | 监测；米氮平增重、安非他酮减重 |
| 低钠血症（SIADH） | 老年人查血钠 |
| 出血风险（5-HT 影响血小板） | 与 NSAID/抗凝联用谨慎 |

### 5-羟色胺综合征（罕见但致命）
- **表现**：自主神经不稳（发热、心动过速、血压波动）、神经肌肉（阵挛、反射亢进、强直）、精神状态改变。
- **触发**：SSRI/SNRI 与 MAOI、曲马多、圣约翰草、5-HT 受体激动剂（如曲坦类）联用。
- **处理**：立即停药、支持治疗、必要时赛庚啶。

### 停药综合征
- 短半衰期药物（帕罗西汀、文拉法辛）尤重：头晕、感觉异常、"电击感"、恶心、焦虑波动、失眠。
- **预防**：缓慢递减（数周至数月）；换用长半衰期氟西汀过渡。

## 六、特殊人群用药

| 人群 | 要点 |
|------|------|
| **儿童/青少年** | TF-CBT 为一线；舍曲林/氟西汀可考虑（监测 suicidality 黑框警告） |
| **老年** | 起始剂量减半；避免帕罗西汀（抗胆碱/跌倒）；查 ECG、血钠；哌唑嗪首剂效应更显著 |
| **妊娠/哺乳** | 帕罗西汀 D 类（心脏致畸）禁用；舍曲林为相对首选（乳汁浓度低）；权衡母体未治疗抑郁/PTSD 风险 |
| **物质使用共病** | 避免苯二氮䓬；整合双重诊断治疗；纳曲酮可处理酒精共病 |
| **战斗相关 PTSD** | SSRI 反应较弱；强调心理治疗 + 共病处理 |

## 七、治疗流程建议

1. **第 0 周**：完整评估（见 [assessment/](评估-创伤评估综合.md)）；与患者共同决策是否用药。
2. **第 0–4 周**：起始 + 滴定至目标剂量；监测副作用与 suicidality。
3. **第 4–8 周**：评估早期反应（PCL-5 下降 ≥ 5–10 分）；若无反应考虑换药或加增效。
4. **第 8–12 周**：评估充分反应；部分反应者考虑增效策略。
5. **维持**：症状缓解后**维持 6–12 个月**（复发预防）；缓慢递减。
6. **全程**：**联用创伤聚焦心理治疗**获得最佳结局；纯药物治疗复发率高。
