# 创伤评估高级 Agent Skill

> **核心定位**：扩展版创伤评估决策树。覆盖 PTSD、CPTSD、道德伤害、解离和童年创伤的综合评估。从筛查到分类、严重度和治疗匹配。

---

## 临床免责声明

> ⚠️ 本技能为**教育和决策支持工具**。创伤诊断需要专业临床评估。急性危机（自杀/自伤/家暴）应立即转介危机服务。

---

## 1. 综合评估决策树

```
START: 用户报告创伤相关困扰
│
├─ Step 0: 安全评估 ──────────────────────────────┐
│   ├─ 当前自杀意念/计划？ → 危机转介              │
│   ├─ 当前处于受虐/家暴环境？ → 安全规划优先       │
│   ├─ 急性解离（无法定向/失去时间）？ → 精神科评估  │
│   └─ 安全 → Step 1                               │
│                                                    │
├─ Step 1: 创伤暴露评估 ──────────────────────────┐
│   ├─ 单一事件 vs 长期/重复？                     │
│   ├─ 儿童期 vs 成人期？                          │
│   ├─ 人际 vs 非人际？                            │
│   ├─ ACE 评分（如童年暴露）                      │
│   └─ 记录 → Step 2                               │
│                                                    │
├─ Step 2: 症状分类 ─────────────────────────────┐
│   ├─ PTSD 核心（再体验/回避/警觉）？             │
│   │   ├─ + DSO（情感/自我/关系）→ CPTSD         │
│   │   └─ 无 DSO → PTSD                          │
│   ├─ 道德痛苦（羞耻/内疚/意义丧失）？→ 道德伤害  │
│   ├─ 解离症状（DES≥30）？→ 解离障碍             │
│   └─ 混合/不典型 → 进一步评估                    │
│                                                    │
├─ Step 3: 严重度评估 ───────────────────────────┐
│   ├─ PCL-5 评分 → 轻度/中度/重度               │
│   ├─ ITQ 评分 → PTSD/CPTSD 分类                 │
│   ├─ 功能评估（工作/关系/日常）                  │
│   └─ 共病评估（抑郁/焦虑/物质/人格）             │
│                                                    │
└─ Step 4: 治疗匹配 ────────────────────────────┐
    ├─ 单事件 PTSD → PE/CPT/EMDR                 │
    ├─ CPTSD → Phase 模型（先稳定化）             │
    ├─ 道德伤害 → Adaptive Disclosure/CFT        │
    ├─ 解离 → 先稳定化/SE → 创伤处理             │
    ├─ 童年创伤 → ARC/TF-CBT/多系统              │
    └─ 复杂共病 → 多学科团队                      │
```

## 3. 治疗选择矩阵

| 诊断 | 一线 | 二线 | 补充 |
|:-----|:-----|:-----|:-----|
| PTSD（单事件） | PE / CPT / EMDR | 药物 (SSRI) | — |
| CPTSD | STAIR-NT / Phase 模型 | EMDR+稳定化 | SE/CFT |
| 复杂 PTSD | 先 Phase 1 | PE/CPT（稳定后） | 团体/家庭 |
| 道德伤害 | Adaptive Disclosure | CFT/叙事 | 灵性/社区 |
| 解离+创伤 | 先稳定化 | SE/EMDR（调整后） | 药物 |
| 儿童创伤 | TF-CBT/ARC | EMDR-Child/CPP | 家庭治疗 |

## 4. 红旗与转介标准

| 红旗 | 转介 |
|:-----|:-----|
| PCL-5 ≥ 50 | 专科创伤治疗 |
| DES ≥ 30 | 解离障碍专科 |
| 当前自杀意念 | 危机服务/急诊 |
| 活动期自伤 | 精神科 |
| 物质滥用共病 | 双重诊断项目 |
| 儿童受虐 | 儿童保护服务 |
