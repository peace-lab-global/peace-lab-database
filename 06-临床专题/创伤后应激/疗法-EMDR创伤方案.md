# EMDR 创伤治疗方案 (Eye Movement Desensitization and Reprocessing)

> **核心定位**：由 Francine Shapiro 开发的循证创伤疗法。基于**适应性信息加工模型**(AIP)，通过**双侧刺激**（眼动/触觉/听觉）激活大脑信息加工系统，促进创伤记忆的再加工。

---

## 1. AIP 模型 (Adaptive Information Processing)

| 概念 | 说明 |
|:-----|:-----|
| 核心假设 | 大脑天生具有信息加工和适应学习能力 |
| 创伤影响 | 创伤事件"卡住"信息加工系统 |
| 未加工记忆 | 以原始感官/情感/认知形式储存（非整合） |
| 再加工 | 双侧刺激激活加工系统→连接适应性信息→整合记忆 |
| 目标 | 记忆不变，但与记忆的**关系**改变（脱敏+认知重建） |

---

## 2. 八阶段方案

| 阶段 | 名称 | 内容 | 节次 |
|:----:|:-----|:-----|:----:|
| 1 | **病史采集** | 创伤史/资源评估/治疗计划 | 1-2 |
| 2 | **准备** | 治疗关系/心理教育/安全处技术 | 2-4 |
| 3 | **评估** | 靶记忆/图像/NC/PC/VoC/SUD/身体 | 4-5 |
| 4 | **脱敏** | 双侧刺激处理靶记忆至 SUD=0 | 5-8 |
| 5 | **植入** | 强化正面认知至 VoC=7 | 8-9 |
| 6 | **身体扫描** | 确认无残留躯体不适 | 9-10 |
| 7 | **结束** | 安全处/容器技术/下次预告 | 每次 |
| 8 | **再评估** | 检查前次加工效果 | 每次起始 |

---

## 3. 关键技术要素

### 3.1 靶记忆评估 (Phase 3)

| 要素 | 说明 |
|:-----|:-----|
| 图像 | 创伤最具代表性的画面 |
| NC (Negative Cognition) | 关于自我的负面信念 |
| PC (Positive Cognition) | 希望拥有的正面信念 |
| VoC (Validity of Cognition) | PC 可信度 1-7 |
| SUD (Subjective Units of Disturbance) | 痛苦程度 0-10 |
| 身体位置 | 不适的身体位置 |

### 3.2 双侧刺激 (BLS)

| 类型 | 说明 |
|:-----|:-----|
| 眼动 | 治疗师手指引导水平眼动（首选/最多研究） |
| 触觉 | 交替拍打蝴蝶拍/手持振动器 |
| 听觉 | 交替声音 |
| 设置 | 每组 24-30 次眼动（约 30 秒） |

### 3.3 认知交织 (Cognitive Interweave)

| 用途 | 当加工"卡住"时，治疗师提供信息促进加工 |
|:-----|:-----|
| 责任交织 | "当时你多大？你能做什么？" |
| 安全交织 | "现在你安全吗？" |
| 选择交织 | "如果朋友经历了同样的事，你会对她说什么？" |

---

## 4. 疗效证据

| 数据 | 发现 |
|:-----|:-----|
| WHO 指南 (2013) | **推荐** PTSD 和 CPTSD 治疗 |
| APA 指南 (2017) | 条件性推荐 |
| ISTSS 指南 | 一级推荐 |
| 效应量 | d = 0.66-1.03（vs 对照） |
| 缓解率 | 70-90%（单事件 PTSD） |
| 节次数 | 8-12 次（单事件），更多（复杂创伤） |
| vs PE/CPT | 效果相当，EMDR 不需要详细叙述和大量家庭作业 |

---

## 5. 适应症与禁忌

| 适应症 | 禁忌/注意 |
|:-------|:---------|
| PTSD（单/多事件） | 严重解离（需先稳定化） |
| 复杂 PTSD（调整后） | 活动期精神病 |
| 恐惧症（有创伤根源） | 严重不稳定生活 |
| 哀伤/丧失 | 癫痫（眼动需调整） |
| 疼痛（心因性） | 眼部疾病（改用触觉 BLS） |

---

## 6. EMDR 争议

| 争议点 | 说明 |
|:-------|:-----|
| 眼动是否必要？ | 元分析：EMDR 有/无眼动效果差异不大 |
| AIP 可证伪性 | 机制假说难以直接验证 |
| vs 暴露疗法 | EMDR 暴露量更少但效果相当 |
| 过度商业化 | 培训认证体系争议 |
| 证据强度 | RCT 数量少于 PE/CPT 但不断增长 |
