# 发展性创伤障碍 (Developmental Trauma Disorder, DTD)

> **核心定位**：由 Bessel van der Kolk 提出的**概念框架**（非正式诊断）。描述儿童期长期人际创伤导致的多系统紊乱，涵盖情感/认知/行为/关系/躯体/自我调节等多维度。DSM-5 未采纳，但深刻影响了临床实践。

---

## 1. 提出背景

| 问题 | 说明 |
|:-----|:-----|
| 诊断不足 | 受虐儿童常被诊断为 ADHD/ODD/DMDD/BPD 等，掩盖创伤根源 |
| 多系统影响 | 创伤影响不止于 PTSD 症状，涉及发展全域 |
| 治疗误导 | 当前诊断导向药物治疗（而非创伤知情照护） |
| DSM-5 争议 | 2011 年提议，2012 年被 DSM-5 工作组拒绝 |

---

## 2. DTD 症状框架（7 领域）

| 领域 | 表现 |
|:-----|:-----|
| **情感调节** | 情绪爆发/麻木/无法自我安抚/述情障碍 |
| **注意力/认知** | 注意力不集中/执行功能受损/解离 |
| **行为控制** | 冲动/攻击/自伤/对立/退缩 |
| **自我概念** | 无价值感/羞耻/内在坏感/破碎身份 |
| **关系** | 不安全依恋/边界混乱/对权威不信任 |
| **躯体** | 慢性疼痛/GI 问题/头痛/感觉异常 |
| **生物调节** | 睡眠障碍/饮食紊乱/HPA 轴失调 |

---

## 3. DTD vs CPTSD vs PTSD（儿童）

| 维度 | 儿童 PTSD | CPTSD (ICD-11) | DTD (概念) |
|:-----|:----------|:---------------|:-----------|
| 诊断状态 | 正式诊断 | 正式诊断 | 概念框架 |
| 核心关注 | 再体验/回避/警觉 | PTSD + DSO | 全发展域 |
| 行为问题 | 不核心 | 不核心 | **核心** |
| 认知/学校 | 不核心 | 不核心 | **核心** |
| 躯体症状 | 不核心 | 不核心 | **核心** |
| 依恋关系 | 不核心 | 关系困难 | **核心**（依恋理论） |

---

## 4. 神经生物学影响

| 系统 | 影响 |
|:-----|:-----|
| HPA 轴 | 皮质醇失调（高→低模式） |
| 杏仁核 | 过度激活/威胁高敏 |
| 前额叶 | 发育不足/执行功能受损 |
| 海马体 | 体积减小/记忆编码异常 |
| 迷走神经 | 腹侧迷走张力降低/社交参与系统受损 |
| 默认模式网络 | 自我参照加工异常 |

---

## 5. 评估

| 工具 | 说明 |
|:-----|:-----|
| UCLA PTSD-RI | PTSD 症状评估 |
| Child Trauma Screening Questionnaire | 创伤暴露筛查 |
| CBCL (Child Behavior Checklist) | 多轴行为评估 |
| Attachment assessments | AAI/CAI/MARS |
| DTD Field Trial Interview | van der Kolk 团队结构化访谈 |
| 发展史 | 里程碑/学校记录/DCP 记录 |

---

## 6. 治疗原则

| 原则 | 说明 |
|:-----|:-----|
| **安全第一** | 物理/情感安全优先于创伤处理 |
| **关系修复** | 依恋修复是治疗核心 |
| **自下而上** | 先躯体调节→再情感→最后认知 |
| **发展适宜** | 根据发展年龄（非实际年龄）设计干预 |
| **多系统** | 家庭/学校/社区协作 |
| **SMART 模型** | Safety, Modulation, Attachment, Reintegration, Transformation |

### 6.1 循证方法

| 方法 | 适用 |
|:-----|:-----|
| TF-CBT (Trauma-Focused CBT) | 学龄儿童首选 |
| ARC (Attachment, Regulation, Competency) | 复杂创伤儿童 |
| CPP (Child-Parent Psychotherapy) | 婴幼儿/幼儿 |
| EMDR (调整版) | 侵入性症状为主 |
| SP (Sensorimotor Psychotherapy) | 躯体症状/解离 |
| 动物辅助治疗 | 关系修复/安全感 |
