# 复杂性 PTSD (Complex PTSD, ICD-11 CPTSD)

> **核心定位**：ICD-11 (6B41) 正式承认的独立诊断。在 PTSD 核心症状基础上，增加**自我组织紊乱**(DSO)三维度。源于长期、重复、无法逃脱的创伤（如童年虐待/家暴/战俘）。

---

## 临床免责声明

> ⚠️ 本文档为教育参考，不替代临床诊断。CPTSD 与 BPD 的鉴别需要专业评估。

---

## 1. ICD-11 诊断标准

### 1.1 PTSD 核心症状（3 项必备）

| 维度 | 表现 |
|:-----|:-----|
| **再体验** | 闪回/噩梦/侵入性记忆（当下感） |
| **回避** | 回避创伤相关思维/感受/外部提示 |
| **过度警觉** | 持续威胁感/惊跳反应/过度警觉 |

### 1.2 自我组织紊乱 (DSO)（3 项新增）

| 维度 | 表现 |
|:-----|:-----|
| **情感调节困难** | 情绪爆发/麻木/无法从痛苦中恢复 |
| **消极自我概念** | 持久的羞耻/内疚/无价值感/失败感 |
| **关系困难** | 难以建立和维持亲密关系/信任困难/感觉与他人隔绝 |

### 1.3 功能受损

> 上述症状导致个人、家庭、社会、教育、职业或其他重要功能领域的**显著损害**。

---

## 2. CPTSD vs PTSD vs BPD

| 维度 | PTSD | CPTSD | BPD |
|:-----|:-----|:------|:----|
| 再体验 | ✓ | ✓ | 可有 |
| 回避 | ✓ | ✓ | 不核心 |
| 过度警觉 | ✓ | ✓ | 可有 |
| 情感调节困难 | 可继发 | **核心** | **核心** |
| 消极自我概念 | 有限 | **核心**（羞耻/无价值） | **核心**（不稳定认同） |
| 关系困难 | 有限 | **核心**（信任/亲密） | **核心**（不稳定/害怕抛弃） |
| 认同紊乱 | 无 | 无 | **核心** |
| 冲动/自伤 | 无 | 少见 | **核心** |
| 理想化-贬低 | 无 | 无 | **核心** |
| 创伤史 | 单一事件可 | 长期/重复创伤 | 常有 |

---

## 3. 流行病学

| 数据 | 值 |
|:-----|:---|
| 一般人群 | 3.8%（Cloitre et al. 2019, WHO World Mental Health） |
| PTSD 中 CPTSD 比例 | ~38% |
| 性别差异 | 女性 > 男性（但差距小于 PTSD） |
| 高风险群体 | 难民/战俘/童年虐待幸存者/家暴受害者 |

## 4. 评估工具

| 工具 | 说明 |
|:-----|:-----|
| **ITQ** (International Trauma Questionnaire) | 18 项自评，同时评估 PTSD 和 CPTSD |
| **ITQ-CA** | 儿童青少年版 |
| **STAIR-NT 评估** | 治疗前全面功能评估 |
| **DES** (Dissociative Experiences Scale) | 解离评估（CPTSD 常见共病） |

---

## 5. 循证治疗

### 5.1 阶段模型 (Herman 1992 / Cloitre)

```
Phase 1: 安全与稳定化
├─ 情感调节技能训练
├─ 安全计划
├─ 建立治疗联盟
└─ 症状稳定化

Phase 2: 创伤记忆处理
├─ 叙事暴露
├─ 认知重建
└─ 记忆整合

Phase 3: 社会重连
├─ 关系技能
├─ 身份重建
└─ 社会参与
```

### 5.2 STAIR-NT (Skills Training in Affective and Interpersonal Regulation + Narrative Therapy)

| 模块 | 内容 | 周数 |
|:-----|:-----|:----:|
| STAIR (Phase 1) | 情感调节/人际技能/情绪觉察 | 8-12 周 |
| NT (Phase 2) | 创伤叙事/认知重建/意义整合 | 8-12 周 |

### 5.3 其他有效方法

| 方法 | 说明 |
|:-----|:-----|
| EMDR + 稳定化 | 双侧刺激+资源发展（适用于侵入性症状为主） |
| NET (Narrative Exposure Therapy) | 生命线叙事（适用于难民/战争创伤） |
| Trauma-Focused CBT | 青少年首选 |
| 躯体方法 (SE/Sensorimotor) | 处理躯体化/解离 |

---

## 6. 共病与鉴别

| 共病 | 频率 | 说明 |
|:-----|:-----|:-----|
| 抑郁症 | 高 | 与消极自我概念重叠 |
| 解离障碍 | 中-高 | DID/DDNOS 需先稳定解离 |
| BPD | 中 | 可同时诊断（ICD-11 允许共病） |
| 物质滥用 | 中 | 自我药疗/回避 |
| 躯体症状 | 中 | 慢性疼痛/GI 症状 |
