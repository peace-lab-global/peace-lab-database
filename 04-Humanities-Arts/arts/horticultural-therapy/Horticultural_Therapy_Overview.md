## 园艺治疗概述

### 定义与发展

| 维度 | 信息 |
|------|------|
| 定义 | 由受训专业人员引导，利用园艺活动作为媒介，促进参与者身心康复的治疗方法 |
| 起源 | 19世纪精神病院开始使用(Benjamin Rush) |
| 专业化 | 1973年美国园艺治疗协会(AHTA)成立 |
| 当前状态 | 全球25+国家开展; 注册园艺治疗师(HTR)认证 |
| 适用场所 | 医院/康复中心/养老院/特殊学校/社区/监狱 |

### 园艺活动分类

| 类型 | 活动示例 | 治疗重点 |
|------|---------|---------|
| 播种/种植 | 播种/移栽/扦插 | 希望/耐心/生命意义 |
| 照料/养护 | 浇水/施肥/除草/修剪 | 责任感/日常结构/精细动作 |
| 收获/加工 | 采摘/烹饪/干花/花艺 | 成就感/社交/感官刺激 |
| 花园设计 | 规划/布局/装饰 | 创造力/决策/空间感 |
| 自然观察 | 植物识别/物候记录 | 注意力/正念/知识 |

### 园艺治疗 vs 治疗性园艺

| 维度 | 园艺治疗(HT) | 治疗性园艺(TH) |
|------|-------------|----------------|
| 引导者 | 注册园艺治疗师 | 园艺爱好者/志愿者 |
| 目标 | 明确治疗目标/评估 | 一般性健康促进 |
| 对象 | 有诊断的个体 | 任何人 |
| 文档 | 治疗计划/进展记录 | 非必需 |
| 设置 | 临床/康复机构 | 社区/家庭 |

## 理论基础

### 核心理论

| 理论 | 提出者 | 核心观点 | 在园艺治疗中的应用 |
|------|--------|---------|-----------------|
| 注意力恢复理论(ART) | Kaplan(1989) | 自然环境恢复定向注意力 | 花园作为恢复性环境 |
| 压力恢复理论(SRT) | Ulrich(1991) | 自然景观引发积极情感/降低应激 | 绿色空间降低皮质醇 |
| 亲生命假说 | Wilson(1984) | 人类天生亲近自然的倾向 | 园艺满足内在需求 |
| 心流理论 | Csikszentmihalyi | 技能与挑战匹配→最优体验 | 园艺活动调节难度 |
| 自我效能理论 | Bandura | 成功体验→自信↑ | 植物生长→掌控感 |
| 感觉统合 | Ayres | 多感官刺激促进神经发展 | 五感花园 |

### 园艺治疗的多维效益

| 维度 | 效益 | 机制 |
|------|------|------|
| 心理 | 抑郁↓ 焦虑↓ 自尊↑ 幸福感↑ | 成就感/自然接触/正念 |
| 认知 | 注意力↑ 记忆↑ 执行功能↑ | ART/多任务/学习 |
| 社会 | 社交技能↑ 孤独↓ 归属感↑ | 团体活动/合作/分享 |
| 身体 | 精细/粗大运动↑ 体力↑ | 挖掘/搬运/精细操作 |
| 灵性 | 意义感↑ 与自然连接↑ | 生命周期/季节/无常 |

## 园艺治疗实践

### 标准治疗流程

| 阶段 | 内容 |
|------|------|
| 评估 | 身体/认知/情绪/社交功能评估 |
| 目标设定 | SMART目标(与个案讨论) |
| 计划 | 选择适当活动/调整难度/安全评估 |
| 实施 | 引导活动/观察/即时调整 |
| 记录 | 参与度/行为变化/目标进展 |
| 评估 | 定期复评/调整计划 |

### 四季活动日历

| 季节 | 活动 | 治疗主题 |
|------|------|---------|
| 春 | 播种/育苗/修剪/花园规划 | 希望/新开始/规划 |
| 夏 | 养护/浇水/收获/插花 | 责任/丰盛/成就 |
| 秋 | 采收/堆肥/种球种植/干花 | 感恩/转化/储备 |
| 冬 | 室内植物/插花/花园设计/种子整理 | 休息/反思/准备 |

### 感官花园设计五要素

| 感官 | 植物/元素 | 治疗效果 |
|------|---------|---------|
| 视觉 | 色彩丰富花卉/季节变化 | 情绪提升/美感 |
| 嗅觉 | 薰衣草/迷迭香/茉莉 | 放松/记忆唤醒 |
| 触觉 | 羊耳朵草/多肉/树皮 | 感觉刺激/接地 |
| 听觉 | 流水/风铃/鸟鸣植物 | 放松/正念 |
| 味觉 | 香草/蔬果/可食用花 | 营养/成就/文化连接 |

## 社区与家庭园艺

### 社区花园的健康效益

| 效益 | 证据 |
|------|------|
| 身体活动↑ | 中等强度体力活动(3-4 METs) |
| 蔬果摄入↑ | 自种→自食(食育) |
| 社会资本↑ | 邻里互动/信任/互助 |
| 心理健康↑ | 压力↓ 生活满意度↑ |
| 犯罪率↓ | 社区凝聚/空间改善(费城研究) |

### 家庭治疗性园艺入门

| 级别 | 活动 | 空间需求 |
|------|------|---------|
| 入门 | 室内盆栽(多肉/绿萝/草本) | 窗台 |
| 初级 | 阳台菜园(番茄/辣椒/香草) | 阳台 |
| 中级 | 庭院花园/社区菜地 | 小型花园 |
| 高级 | 主题花园(疗愈/蝴蝶/感官) | 较大花园 |
