# 沙盘/沙游治疗总览 | Sandplay Therapy Overview

> 沙游治疗(Sandplay Therapy)由多拉·卡尔夫(Dora Kalff, 1966)创立，是基于荣格分析心理学的非言语投射性治疗方法——来访者在沙盘中使用微缩玩具创造"世界"，以象征性方式表达内在心灵。

---

## 1. 理论基础

### 1.1 核心假设

| 假设 | 说明 |
|------|------|
| **心灵具有自我疗愈力** | 在安全/受保护的空间中，心灵的自组织力量会被激活 |
| **象征是潜意识的语言** | 沙盘中的微缩世界是内在心灵的象征性外化 |
| **自由与受保护的空间** | 治疗师创建的安全容器是转化的前提 |
| **自性化(Individuation)** | 治疗的最终目标是自我(Self)的实现和整合 |

### 1.2 理论来源

| 来源 | 贡献 |
|------|------|
| **Jung分析心理学** | 集体无意识/原型/自性化/积极想象 |
| **Margaret Lowenfeld (世界技术)** | "World Technique"——儿童用微缩玩具在沙中创造世界 |
| **Winnicott (过渡空间)** | 潜在空间/过渡客体——沙盘的"既非内部也非外部" |
| **Neumann (发展阶段)** | 儿童心理发展的阶段模型 |

### 1.3 卡尔夫沙游 vs 荣格沙景(Sandtray)

| 维度 | Kalff沙游(Sandplay) | 荣格沙景(Sandtray) |
|------|---------------------|---------------------|
| 哲学 | 荣格+佛教(藏传) | 荣格 |
| 过程 | 非指导、自由创造、不干预 | 可指导性更强 |
| 言语 | 制作过程中不交谈 | 可有对话 |
| 解释 | 延迟解释(不即时分析) | 可即时讨论 |
| 系列 | 多个沙盘构成"过程"(process) | 单次也可独立 |
| 治疗师 | 静默见证者 | 更积极参与 |

## 2. 治疗设置

### 2.1 物理环境

| 要素 | 标准 |
|------|------|
| **沙盘** | 50×40×7.5cm(标准)，内壁涂蓝色(象征水/天) |
| **干沙/湿沙** | 可选——湿沙可塑形(山/河/洞穴) |
| **微缩玩具集** | 500-1000+件，分类存储(见下表) |
| **记录** | 照片(俯视) + 过程笔记 + 来访者描述 |
| **空间** | 安静/私密/光线柔和 |

### 2.2 微缩玩具分类

| 类别 | 内容 |
|------|------|
| **人物** | 各年龄/性别/职业/文化/幻想角色 |
| **动物** | 家畜/野生/海洋/恐龙/昆虫/神话生物 |
| **植物** | 树木/花/草/蔬菜/蘑菇 |
| **建筑** | 房屋/桥/围墙/门/塔/庙宇 |
| **交通工具** | 车/船/飞机/火车/马车 |
| **自然物** | 石头/贝壳/水晶/羽毛/骨头 |
| **幻想/宗教** | 天使/龙/佛/十字架/巫师/宝箱 |
| **日常物品** | 家具/食物/工具/武器/乐器 |
| **围栏/边界** | 栅栏/门/墙/标志 |

### 2.3 治疗过程

```
标准流程(50min):
1. 来访者选择沙(干/湿)
2. 自由选取微缩玩具(不限时)
3. 在沙盘中创造"世界"(10-30min)
4. 来访者描述/命名(可选)
5. 治疗师拍照记录
6. 世界保持不动(直到下次来访前拆除)
```

**重要**: 来访者离开后治疗师**不立即拆除**沙盘——保持"世界"的完整性是对来访者内在世界的尊重。

## 3. 治疗师角色与解读

### 3.1 治疗师立场

| 立场 | 描述 |
|------|------|
| **自由与受保护的空间** | 不评判、不指导、不解释 |
| **母性见证者** | 以"足够好的母亲"态度全然关注 |
| **信任过程** | 相信来访者心灵的自组织能力 |
| **延迟理解** | 不在制作过程中分析/干预 |

### 3.2 沙盘解读维度(训练后使用，非面质来访者)

| 维度 | 观察要点 |
|------|----------|
| **空间使用** | 满/空？集中/分散？左(过去)/右(未来)？上(意识)/下(无意识)？ |
| **主题** | 混乱/秩序？战争/和平？孤立/连接？围栏/开放？ |
| **象征** | 关键象征物的文化/原型意义 |
| **缺失** | 缺少人物？缺少水？缺少桥(连接)？ |
| **过程** | 犹豫/快速？反复移动？破坏？加水？ |
| **系列变化** | 多次沙盘间的主题演变/整合趋势 |

### 3.3 常见主题演变(治愈过程)

```
初期: 混乱/战争/围栏/孤立 → 中期: 出现水/桥/植物 → 后期: 整合/社区/秩序
  (防御/创伤表达)        (转化开始)            (整合/自性化)
```

## 4. 临床应用

### 4.1 循证证据

| 研究 | 人群 | 结果 |
|------|------|------|
| Roesler (2005) | 多种 | 沙游改善行为问题/情绪调节 |
| Hong (2005) | 受虐儿童 | 沙游↓攻击行为/↑自尊 |
| Lu et al. (2016) | 中国文化适应 | 沙游对华人儿童有效 |
| Homeyer & Sweeney (2017) | 综述 | 沙游对创伤/ASD/焦虑有效 |

### 4.2 适用人群

| 人群 | 沙游优势 |
|------|----------|
| **儿童** | 自然游戏媒介→语言表达有限时的替代 |
| **创伤** | 安全距离处理创伤(象征性，非直接回忆) |
| **ASD** | 感官友好、结构化、非社交要求 |
| **跨文化** | 非语言→绕过文化/语言障碍 |
| **成年人** | 潜意识→意识化、灵性探索、中年转型 |
| **家庭** | 家庭成员轮流/合作创建沙盘→关系可视化 |
| **团体** | 团体沙游→团体动力可视化 |

### 4.3 禁忌与注意

| 情况 | 处理 |
|------|------|
| **强迫性整洁(OCD)** | 提供选择，尊重控制需求 |
| **严重解离** | 渐进使用，地面化先行 |
| **急性精神病** | 不适用(需稳定化后) |

## 5. 沙游治疗师培训

| 认证 | 机构 | 要求 |
|------|------|------|
| **ISST(国际沙游治疗学会)** | Dora Kalff创立 | 完成系列培训+个案+督导 |
| **STA(沙游治疗师协会)** | 美国 | 培训+个案时数+论文 |
| **CSAP(中国)** | 申荷永/高岚 | 国际荣格/沙游培训体系 |
| **核心** | 荣格分析+个人分析+督导 | 2-5年 |
