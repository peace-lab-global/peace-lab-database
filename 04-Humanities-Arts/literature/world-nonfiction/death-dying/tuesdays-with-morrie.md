# 《相约星期二》Tuesdays with Morrie

> 米奇·阿尔博姆（Mitch Albom）著，1997年出版。体育记者阿尔博姆与 former 大学教授莫里·施瓦茨在后者患ALS（肌萎缩侧索硬化症）后的14个星期二的重逢记录。全球销量超过1400万册，被改编为同名电影和舞台剧。

## 一、总体评价

这是一本"简单到近乎天真"的书——莫里的智慧不是哲学家的复杂论证，而是老人的朴素洞察。但正是这种朴素，让它触动了数百万人。**面对死亡时，最深刻的真理往往是最简单的。**

莫里的教导围绕着几个主题：爱、工作、社区、衰老、恐惧、家庭、原谅、死亡。每一个主题都是通过两个人的对话来呈现的——阿尔博姆代表"忙碌而迷失的现代人"，莫里代表"知道什么真正重要的将死之人"。

## 二、核心内容

### 莫里的"课程"

| 星期二 | 主题 | 核心教导 |
|--------|------|---------|
| 第一周 | 世界 | 学会与自己和世界和解 |
| 第二周 | 自怜 | 允许自己短暂的自怜，然后继续前进 |
| 第三周 | 遗憾 | 不要遗憾——问自己"如果明天是最后一天，我会做什么？" |
| 第四周 | 死亡 | 学会死，才能学会活 |
| 第五周 | 家庭 | 家庭是你唯一可以依赖的东西 |
| 第六周 | 情感 | 完全地感受你的情感，不要压抑 |
| 第七周 | 衰老 | 衰老不只是衰退，也是收获 |
| 第八周 | 金钱 | 文化误导了我们——金钱不能替代爱 |
| 第九周 | 爱 | 爱是唯一理性的行为 |
| 第十周 | 婚姻 | 尊重对方，懂得妥协，共同的核心价值观 |
| 第十一周 | 文化 | 不要盲从文化——创造你自己的亚文化 |
| 第十二周 | 原谅 | 原谅自己，原谅他人——不要带着怨恨死去 |
| 第十三周 | 完美的一天 | 一个简单的、与所爱的人共处的日子 |
| 第十四周 | 告别 | 哭泣没有关系——这是你在说"我爱你" |

### "学会死，才能学会活"

莫里反复说："一旦你学会了怎样死，你也就学会了怎样活。"他的意思不是"要每天想着死亡"，而是**"意识到生命的有限性，会让你更加珍惜每一个选择"**。当你知道时间不多了，你就不会再浪费时间在无关紧要的事情上。

## 三、阅读体验

**评分：8.5/10** — 这本书的文学性不高，语言简单直接，有些段落接近鸡汤。但它作为"死亡教育的入门读物"几乎是无与伦比的——你可以把它送给任何年龄的人，他们都能读懂，都能被打动。最佳阅读时机：当你觉得生活失去了方向的时候。
