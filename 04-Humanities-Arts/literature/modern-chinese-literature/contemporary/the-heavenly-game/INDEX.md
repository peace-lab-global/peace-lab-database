# The Heavenly Game | 天局

> 矫健短篇小说，1985年发表。讲述一个棋痴在暴风雪的荒野中与天下棋，最终以命为子、以死证道的故事。全文不到两万字，但震撼力极强，被誉为"中国最硬的短篇小说"。
