# 复发预防模型 (Relapse Prevention, RP)

> **核心定位**：由 G. Alan Marlatt 开发的认知行为复发预防模型。核心概念：**失误(lapse)≠复发(relapse)**。通过识别高危情境、发展应对技能和管理生活方式平衡来减少复发。

---

## 1. 核心概念

| 概念 | 说明 |
|:-----|:-----|
| **失误 (Lapse)** | 单次/短暂的物质使用或行为复现 |
| **复发 (Relapse)** | 持续回到治疗前的使用模式 |
| **禁戒违反效应 (AVE)** | 失误后"全完了"思维→真正复发 |
| **高危情境** | 触发使用欲望的内部/外部情境 |
| **应对技能** | 替代使用的应对行为 |
| **生活方式平衡** | 减少"应该"vs"想要"的失衡 |

---

## 2. 高危情境分类

| 类别 | 子类型 | 频率 |
|:-----|:-------|:-----|
| **内部** | 负面情绪（愤怒/焦虑/抑郁/孤独） | ~35% |
| **内部** | 身体不适/疼痛 | ~5% |
| **内部** | 正面情绪（庆祝/自信） | ~5% |
| **外部** | 人际冲突 | ~16% |
| **外部** | 社会压力 | ~10% |
| **外部** | 接触物质/线索 | ~8% |

---

## 3. RP 核心流程

```
Step 1: 识别个人高危情境
├─ 回顾过去复发的"触发链"
├─ HALT: Hungry/Angry/Lonely/Tired
└─ 内部（情绪/认知）+ 外部（人/地/物）

Step 2: 发展应对策略
├─ 认知应对（重新评估/自我对话）
├─ 行为应对（替代活动/放松/运动）
├─ 环境应对（回避/改变路线/移除线索）
└─ 社会应对（求助/支持网络）

Step 3: 应对技能训练
├─ 角色扮演
├─ 放松训练
├─ 问题解决训练
└─ 拒绝技能

Step 4: 生活方式平衡
├─ 增加"想要"的活动
├─ 减少"应该"的负担
├─ 正念/觉察
└─ 自我照顾

Step 5: 失误管理
├─ 正常化失误
├─ 挑战 AVE（"一次失误≠完全失败"）
├─ 紧急计划（联系人/策略）
└─ 从失误中学习
```

---

## 4. RP 疗效

| 数据 | 发现 |
|:-----|:-----|
| 元分析 | 中等效应（减少复发率 14-20%） |
| 酒精 | 有效（尤其联合 MI） |
| 烟草 | 有效 |
| 其他物质 | 有效 |
| 行为成瘾 | 有效 |
| 长期维持 | 技能持续使用则持续有效 |
