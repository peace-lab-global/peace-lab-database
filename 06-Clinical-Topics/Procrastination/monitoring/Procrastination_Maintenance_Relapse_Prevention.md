# 拖延的维持治疗与复发预防 | Maintenance and Relapse Prevention for Procrastination

> **目标**：建立拖延干预后的长期维持框架，预防复发，巩固治疗效果。
> **标准来源**：CBT维持治疗原则、行为改变理论、自我管理研究。

---

## 一、复发模式

### 1.1 常见复发触发

| 触发 | 说明 |
|------|------|
| **生活应激** | 压力增加 → 回归旧的情绪调节模式 |
| **新任务类型** | 未在训练中练习的任务类型 |
| **成功后的放松** | "我已经好了" → 放弃维持策略 |
| **环境变化** | 新工作、新学校、新关系 → 失去原有结构 |
| **季节/节律变化** | 冬季抑郁；假期后 |
| **共病波动** | 抑郁/焦虑复发 → 拖延复发 |

### 1.2 复发信号

| 早期信号 | 中期信号 | 晚期信号 |
|---------|---------|---------|
| 偶尔拖延小任务 | 频繁拖延 | 回避重要任务 |
| 短暂使用手机 | 长时间刷手机 | 整天逃避 |
| "明天再开始"想法 | 反复推迟同一任务 | 完全放弃任务 |
| 轻微焦虑 | 明显焦虑/自责 | 抑郁/绝望 |

## 二、维持策略

### 2.1 核心技能的长期维持

| 技能 | 维持方法 |
|------|---------|
| **5分钟启动** | 遇到困难任务时自动使用 |
| **任务分解** | 面对任何大任务时自动分解 |
| **认知重构** | 定期识别和挑战拖延思维 |
| **环境设计** | 持续维护减少诱惑的环境 |
| **时间块** | 保持日历上的时间块习惯 |

### 2.2 日常维持仪式

| 时间 | 仪式 |
|------|------|
| **早晨** | 回顾今日最重要任务；吃"青蛙" |
| **工作开始时** | 5分钟正念呼吸；设定意图 |
| **每次任务前** | 明确下一步行动 |
| **晚上** | 回顾完成情况；记录拖延触发因素 |
| **周末** | 回顾一周；计划下周 |

## 三、复发应对

### 3.1 "复发急救"步骤

| 步骤 | 行动 |
|------|------|
| **1. 识别** | "我注意到我在拖延"（不加评判） |
| **2. 暂停** | 停止当前拖延行为 |
| **3. 觉察** | "我现在感觉什么？什么想法？" |
| **4. 选择** | "我想继续拖延还是开始任务？" |
| **5. 启动** | 使用5分钟规则立即开始 |
| **6. 支持** | 如果需要，联系 accountability 伙伴 |

### 3.2 复发后"重启"计划

| 时间 | 行动 |
|------|------|
| **第1天** | 不自我批评；选择一个最简单的小任务完成 |
| **第2-3天** | 恢复核心策略（5分钟启动、时间块） |
| **第4-7天** | 增加任务难度；恢复环境设计 |
| **第2周** | 全面恢复维持计划 |
| **第3-4周** | 分析复发原因；调整策略 |

## 四、长期监测

### 4.1 自我监测工具

| 工具 | 频率 | 内容 |
|------|------|------|
| **拖延日志** | 每周 | 记录拖延事件、触发因素、应对 |
| **周回顾** | 每周 | 完成度、情绪、策略使用情况 |
| **月度评估** | 每月 | ISI式自评；趋势分析 |

### 4.2 维持支持

| 支持 | 频率 |
|------|------|
| **自助材料复习** | 每3月 |
| **治疗师"强化"会谈** | 每3-6月或按需 |
| **同伴支持小组** | 持续 |
| **APP/数字工具** | 持续使用 |
