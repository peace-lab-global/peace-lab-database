## 决策科学基础

### 决策的本质

| 维度 | 说明 |
|------|------|
| 定义 | 在不确定性中，从多个选项中做出选择的过程 |
| 核心要素 | 选项/结果/概率/价值/偏好 |
| 好决策≠好结果 | 过程质量>结果质量(结果有随机性) |
| 核心挑战 | 信息不完全/时间有限/认知局限/情绪干扰 |

### 决策类型

| 类型 | 特征 | 适合方法 |
|------|------|---------|
| 简单决策 | 选项少/结果确定 | 直觉/快速判断 |
| 复杂决策 | 多因素/相互依赖 | 结构化分析 |
| 风险决策 | 概率可估计 | 期望值/决策树 |
| 不确定性决策 | 概率不可估计 | 情境规划/稳健策略 |
| 模糊决策 | 连问题是什么都不确定 | 探索/实验/迭代 |
| 群体决策 | 多人参与 | 结构化讨论/投票 |

### 双系统理论(Kahneman)

| 系统 | 特征 | 适用 | 风险 |
|------|------|------|------|
| 系统1(快) | 直觉/自动/情绪/低能耗 | 熟悉/紧急/低风险 | 偏差/过度自信 |
| 系统2(慢) | 分析/审慎/逻辑/高能耗 | 复杂/重要/不熟悉 | 拖延/过度分析 |

## 决策模型与框架

### 经典决策框架对比

| 框架 | 核心 | 适用 | 局限 |
|------|------|------|------|
| 理性决策模型 | 最大化期望效用 | 数据充分/可量化 | 信息成本/认知负荷 |
| 有限理性(Simon) | 满意解 vs 最优解 | 信息不完全/时间压力 | 可能错过更好选项 |
| 前景理论(Kahneman) | 损失厌恶/参照点/框架 | 理解实际行为 | 难以直接指导决策 |
| 自然决策(Klein) | 模式识别/经验 | 专家/时间压力 | 新手不适用 |
| OODA循环 | 观察→定向→决定→行动 | 动态/对抗环境 | 需要迭代能力 |

### 实用决策框架

| 框架 | 步骤 | 最佳适用 |
|------|------|---------|
| 决策矩阵 | 列选项×列标准→加权评分 | 多选项多标准 |
| 决策树 | 图示化选项→概率→结果 | 多阶段/概率已知 |
| 10-10-10法则 | 10分钟/10个月/10年后感受 | 情绪化时 |
| 反转测试 | 如果已做了，你会后悔吗？ | 默认偏差 |
| 预测性后悔 | 想象两个选择→哪个更后悔？ | 二选一 |
| 安全失败实验 | 低成本试错→获取信息 | 不确定性高 |

### 重大人生决策框架

| 步骤 | 内容 |
|------|------|
| 1. 澄清 | 我真正想要什么？(价值观/长期目标) |
| 2. 选项 | 扩大选项池(至少3个) |
| 3. 信息 | 收集足够(非完美)信息 |
| 4. 情绪 | 觉察并管理情绪(但不消除) |
| 5. 分析 | 决策矩阵/预测性后悔/10-10-10 |
| 6. 咨询 | 听取不同视角(尤其反对意见) |
| 7. 决定 | 设定截止时间→做决定 |
| 8. 行动 | 全力投入→评估→调整 |

## 认知偏差与启发式

### 高频偏差清单

| 偏差 | 描述 | 日常示例 |
|------|------|---------|
| 确认偏差 | 只搜索支持已有信念的信息 | 网络搜索只看符合观点的 |
| 可得性启发 | 容易想到的=更常见的 | 飞机事故→高估飞行风险 |
| 锚定效应 | 第一个数字影响判断 | 标价$100→打折$60感觉便宜 |
| 损失厌恶 | 损失的痛苦>等量收益的快乐(约2倍) | 不愿卖亏损股票 |
| 沉没成本 | 因已投入而继续(而非看未来) | 看完不想看的电影 |
| 从众效应 | 跟随大多数人 | "大家都这么做" |
| 过度自信 | 高估自己的判断准确性 | 90%的人认为自己驾驶水平高于平均 |
| 现状偏差 | 倾向于不改变 | 即使更好的选择可用也不切换 |
| 框架效应 | 同一信息不同表述→不同决策 | "90%存活率" vs "10%死亡率" |
| 峰终定律 | 体验评价取决于峰值和结尾 | 旅行体验=最好时刻+结束时刻 |

### Kahneman前景理论核心

| 概念 | 说明 | 决策影响 |
|------|------|---------|
| 参照点 | 评价相对于参照点(非绝对值) | 同样收入在不同基准下感觉不同 |
| 损失厌恶 | 损失权重约为收益的2倍 | 宁愿不冒险也不愿可能损失 |
| 确定性效应 | 偏好确定的小收益vs不确定的大收益 | 选确定的$500>50%的$1200 |
| 可能性效应 | 小概率事件权重被放大 | 买彩票/买保险 |

## 提升决策质量

### 去偏差策略

| 策略 | 目标偏差 | 方法 |
|------|---------|------|
| 红队/魔鬼代言 | 确认偏差 | 指定人专门提出反对意见 |
| 预先验尸(Pre-mortem) | 过度自信 | 假设已失败→分析原因 |
| 参照类预测 | 计划谬误 | 用类似项目的历史数据 |
| 选择架构 | 框架效应 | 多角度重新框架问题 |
| 延迟决策 | 情绪偏差 | 重大决定等待24小时 |
| 机会成本思维 | 沉没成本 | "如果没有之前的投入，我还会选择这个吗？" |
| 基础概率 | 忽视基础概率 | 先查看统计基准数据 |
| 决策日记 | 后见之明偏差 | 记录决策时的理由和信心 |

### 群体决策改进

| 方法 | 说明 |
|------|------|
| 匿名输入 | 先独立思考再讨论(避免从众) |
| 结构化讨论 | 明确议程/时间/发言规则 |
| 六顶思维帽 | de Bono的六种思维角度 |
| 指定反对者 | 轮流扮演"第十个人" |
| 德尔菲法 | 多轮匿名反馈→收敛 |
