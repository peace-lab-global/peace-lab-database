# 睡眠呼吸障碍概述 | Sleep-Related Breathing Disorders Overview

> **目标**：系统阐述阻塞性睡眠呼吸暂停低通气综合征（OSA）及中枢性睡眠呼吸暂停的诊断、评估与治疗。
> **标准来源**：AASM ICSD-3（2014）、AHA/ACC/HRS指南、中文成人OSA诊治指南（2018）。

---

## 一、阻塞性睡眠呼吸暂停低通气（OSA）

### 1.1 流行病学

| 参数 | 数据 |
|------|------|
| **成人患病率** | 男性10-17%；女性3-9%（AHI≥5+症状） |
| | 男性≥30%；女性≥15%（仅AHI≥5） |
| **中老年** | 显著升高（50岁以上>20%） |
| **肥胖** | BMI>30：患病率>40% |
| **未经诊断** | 约80-90%未诊断 |

### 1.2 危险因素

| 不可改变 | 可改变 |
|---------|--------|
| 男性 | 肥胖（首要） |
| 年龄增长 | 颈围增大（男>43cm；女>40cm） |
| 颅面解剖异常 | 饮酒（睡前4小时内） |
| 遗传因素 | 吸烟 |
| 种族（亚裔风险高） | 镇静药物 |
| | 仰卧位睡眠 |

### 1.3 临床表现

**夜间症状：**
- 打鼾（标志性；通常响亮、不规律、中断）
- 目击呼吸暂停（床伴报告）
- 憋醒/窒息感
- 夜间多汗
- 夜尿增多
- 胃食管反流

**日间症状：**
- 日间嗜睡（ESS常>10）
- 晨起头痛
- 注意力不集中、记忆力下降
- 情绪易激惹
- 性欲减退

### 1.4 并发症与合并症

| 系统 | 并发症 |
|------|--------|
| **心血管** | 高血压（难治性）、房颤、心衰、冠心病、卒中 |
| **代谢** | 胰岛素抵抗、2型糖尿病、代谢综合征 |
| **神经认知** | 认知功能下降、痴呆风险增加 |
| **精神** | 抑郁、焦虑、生活质量下降 |
| **交通安全** | 驾驶事故风险增加2-7倍 |
| **围手术期** | 并发症增加、困难气道 |

## 二、诊断与评估

### 2.1 筛查工具

| 工具 | 内容 | 临界值 | 敏感性/特异性 |
|------|------|--------|--------------|
| **STOP-BANG** | 8项（打鼾、疲劳、观察呼吸暂停、血压、BMI、年龄、颈围、性别） | ≥3分：高危 | Sn 93%/Sp 63%（AHI>5） |
| **Berlin问卷** | 3类别（打鼾/窒息、日间嗜睡、高血压/肥胖） | 高风险：2类别阳性 | Sn 86%/Sp 77% |
| **NoSAS** | 5项（颈围、BMI、性别、年龄、打鼾） | ≥8分：高危 | Sn 83%/Sp 65% |

### 2.2 PSG诊断标准

| 严重程度 | AHI（次/小时） | ODI（氧减指数） | 最低SpO₂ |
|---------|---------------|---------------|---------|
| **正常** | <5 | <5 | >90% |
| **轻度** | 5-15 | 5-15 | 85-90% |
| **中度** | 15-30 | 15-30 | 80-85% |
| **重度** | >30 | >30 | <80% |

### 2.3 家庭睡眠呼吸监测（HSAT）

| 适用 | 不适用 |
|------|--------|
| 中重度OSA高危患者 | 合并严重心肺疾病 |
| 无法到睡眠实验室 | 疑似中枢性睡眠呼吸暂停 |
| 疗效随访 | 神经肌肉疾病 |

## 三、治疗

### 3.1 一线治疗：持续气道正压通气（CPAP）

| 参数 | 说明 |
|------|------|
| **机制** | 气动夹板作用，维持上气道开放 |
| **适应证** | 中重度OSA；轻度OSA+症状或合并症 |
| **压力滴定** | 实验室滴定或自动滴定（APAP） |
| **依从性标准** | ≥4小时/夜；≥70%夜晚 |
| **效果** | 症状立即改善；长期降低心血管风险 |

**CPAP依从性促进：**
- 面罩适配（鼻罩、鼻枕、全面罩）
- 加湿器
- 压力缓解（C-Flex/EPR）
- 早期随访（第1周、第1月）
- 远程监测

### 3.2 口腔矫治器（OAT）

| 参数 | 说明 |
|------|------|
| **机制** | 前移下颌/舌，扩大上气道 |
| **适应证** | 轻度-中度OSA；CPAP不耐受 |
| **效果** | AHI降低50%（约）；疗效不如CPAP |
| **注意** | 颞下颌关节问题；牙齿移动 |

### 3.3 体位治疗

| 适应证 | 方法 |
|--------|------|
| 体位性OSA（仰卧位AHI>侧卧2倍） | 背部网球/背包；体位警报器； positional therapy device |

### 3.4 手术治疗

| 手术 | 适应证 | 效果 |
|------|--------|------|
| **悬雍垂腭咽成形术（UPPP）** | 软腭阻塞 | 约40-50%有效；需严格筛选 |
| **颌骨前移术（MMA）** | 颅面畸形 | 高有效率（>80%） |
| **舌下神经刺激（HNS）** | 中度OSA；CPAP不耐受 | AHI降低约50-70% |
| **鼻腔手术** | 鼻腔阻塞 | 辅助作用 |

### 3.5 行为治疗

| 干预 | 效果 |
|------|------|
| **减重** | 体重减轻10%：AHI降低约50% |
| **运动** | 独立于减重改善OSA |
| **戒酒** | 减少呼吸事件频率和持续时间 |
| **戒烟** | 减少上气道炎症 |
| **避免仰卧** | 体位性OSA有效 |

## 四、中枢性睡眠呼吸暂停（CSA）

### 4.1 类型

| 类型 | 特征 | 常见原因 |
|------|------|---------|
| **Cheyne-Stokes呼吸** | 周期性呼吸；渐强-渐弱 | 心衰、卒中、高原 |
| **疾病相关CSA** | 无Cheyne-Stokes | 神经肌肉疾病、脑干病变 |
| **药物/物质所致** | 阿片类药物常见 | 慢性阿片使用 |
| **高海拔CSA** | 海拔>2500m | 高原暴露 |
| **特发性CSA** | 排除上述 | 未知 |

### 4.2 治疗

| 类型 | 治疗 |
|------|------|
| **Cheyne-Stokes（心衰）** | 优化心衰治疗；CPAP（ASV慎用——曾有安全性问题） |
| **药物所致** | 减少/转换阿片 |
| **高海拔** | 下降海拔；乙酰唑胺 |
| **特发性** | CPAP；氧疗；膈肌起搏 |

## 五、OSA与精神障碍

### 5.1 OSA与抑郁

| 关系 | 数据 |
|------|------|
| **共病率** | OSA患者中抑郁患病率15-40% |
| **机制** | 睡眠碎片化；低氧；炎症；生活质量下降 |
| **治疗影响** | CPAP可改善抑郁症状（尤其轻中度） |
| | 抑郁患者CPAP依从性可能较差 |

### 5.2 OSA与焦虑

| 关系 | 数据 |
|------|------|
| **共病率** | OSA患者中焦虑患病率10-30% |
| **特征** | 夜间窒息感；恐夜；睡眠恐惧 |
| **治疗** | CPAP可缓解；必要时加用抗焦虑药 |

### 5.3 OSA与认知障碍

| 关系 | 数据 |
|------|------|
| **影响** | 注意力、执行功能、记忆力下降 |
| **机制** | 间歇性低氧；睡眠剥夺；炎症；血管病变 |
| **治疗** | CPAP可能逆转或减缓认知下降 |

### 5.4 OSA与PTSD

| 关系 | 数据 |
|------|------|
| **共病率** | PTSD患者中OSA患病率高达50-75% |
| **挑战** | CPAP面罩可能触发创伤记忆；依从性差 |
| **策略** | 渐进暴露于CPAP；desensitization；PAP-nap |

## 六、围手术期管理

| 阶段 | 管理 |
|------|------|
| **术前** | OSA筛查；高危→PSG；困难气道评估 |
| **术中** | 区域麻醉优先；避免深度镇静；备好气道管理 |
| **术后** | 持续监护；CPAP恢复使用；避免仰卧；镇痛优化 |
