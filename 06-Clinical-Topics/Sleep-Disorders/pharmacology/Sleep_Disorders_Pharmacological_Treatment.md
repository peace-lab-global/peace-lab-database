# 睡眠障碍药物治疗总览 | Pharmacological Treatment of Sleep-Wake Disorders

> **目标**：系统阐述各类睡眠障碍的药物治疗策略，涵盖药物机制、剂量、适应证与安全性。
> **标准来源**：AASM Clinical Practice Guidelines、FDA approvals、UpToDate。

---

## 一、药物分类总览

| 类别 | 代表药物 | 主要适应证 |
|------|---------|-----------|
| **苯二氮卓类（BZD）** | 艾司唑仑、替马西泮、三唑仑 | 失眠（短期） |
| **非苯二氮卓类（Z-drugs）** | 唑吡坦、右佐匹克隆、扎来普隆 | 失眠 |
| **食欲素受体拮抗剂（DORA）** | 苏沃雷生、莱博雷生、达利雷生 | 失眠 |
| **褪黑素受体激动剂** | 雷美替胺、他司美琼 | 入睡困难、昼夜节律障碍 |
| **镇静抗抑郁药** | 曲唑酮、多塞平、米氮平 | 失眠共病抑郁/焦虑 |
| **抗组胺药** | 苯海拉明、多西拉敏 | 短期/OTC（不推荐常规使用） |
| **兴奋剂** | 莫达非尼、哌甲酯 | 发作性睡病EDS |
| **促觉醒药** | 羟丁酸钠、替洛利生 | 发作性睡病EDS+猝倒 |
| **多巴胺能药物** | 普拉克索、罗匹尼罗 | RLS |
| **其他** | 加巴喷丁、普瑞巴林 | RLS、失眠 |

## 二、失眠药物治疗

### 2.1 选择流程

```
失眠患者
│
├─ 入睡困难为主？
│   ├─ 是 → 短效：唑吡坦/扎来普隆/雷美替胺/苏沃雷生
│   └─ 否 → 睡眠维持困难？
│       ├─ 是 → 中长效：右佐匹克隆/达利雷生/多塞平
│       └─ 否 → 早醒？
│           ├─ 是 → 中长效；考虑共病抑郁
│           └─ 否 → 混合型 → 中长效
│
├─ 共病抑郁/焦虑？
│   └─ 是 → 曲唑酮/米氮平（低剂量）
│
├─ 老年患者？
│   └─ 是 → 避免BZD/Z-drugs；考虑多塞平3-6mg/褪黑素/雷美替胺
│
├─ OSA合并？
│   └─ 是 → 避免BZD；DORA相对安全
│
└─ 慢性失眠？
    └─ CBT-I为首选；药物仅辅助/短期
```

### 2.2 具体药物参数

| 药物 | 剂量范围 | 起效 | 半衰期 | 适应证 | 主要副作用 |
|------|---------|------|--------|--------|-----------|

## 三、发作性睡病药物治疗

### 3.1 EDS治疗

| 药物 | 剂量 | 机制 | 副作用 |
|------|------|------|--------|
| **莫达非尼** | 100-400mg晨 | DA/NE再摄取抑制 | 头痛、失眠、焦虑 |
| **阿莫达非尼** | 150-250mg晨 | 莫达非尼活性异构体 | 同上；半衰期更长 |
| **哌甲酯** | 5-60mg分次 | DA/NE释放 | 心悸、血压升高、焦虑、失眠 |
| **索利氨酯** | 75-150mg晨 | DA/NE再摄取抑制 | 头痛、恶心、食欲下降 |
| **羟丁酸钠** | 4.5-9g分2次 | GABA-B激动 | 恶心、遗尿、抑郁、呼吸抑制 |
| **替洛利生** | 17.8-35.6mg晨 | H3受体反向激动剂 | 失眠、焦虑、恶心 |

### 3.2 猝倒治疗

| 药物 | 剂量 | 机制 | 注意 |
|------|------|------|------|
| **羟丁酸钠** | 4.5-9g | GABA-B激动 | 一线；REMS程序 |
| **文拉法辛** | 37.5-150mg | SNRI | 常用；不可突然停药 |

## 四、不宁腿综合征（RLS）药物治疗

| 药物 | 剂量 | 一线/二线 | 注意 |
|------|------|----------|------|
| **普拉克索** | 0.125-0.75mg睡前 | 一线 | 冲动控制障碍； augmentation |
| **罗匹尼罗** | 0.25-4mg睡前 | 一线 | 同上 |
| **罗替戈汀贴剂** | 1-3mg/24h | 一线 | 贴剂；皮肤反应 |
| **加巴喷丁enacarbil** | 300-1200mg睡前 | 一线（尤其疼痛型） | 头晕、嗜睡 |
| **普瑞巴林** | 75-450mg | 一线/替代 | 体重增加、头晕 |
| **阿片类** | 羟考酮等 | 难治性 | 依赖风险；最后选择 |
| **铁剂补充** | 铁蛋白<75时 | 辅助 | 静脉铁（严重） |

**Augmentation警告：**
- 多巴胺激动剂长期使用可能导致症状加重、提前、扩散（augmentation）
- 处理：换用α2δ配体（加巴喷丁/普瑞巴林）或阿片类

## 五、药物安全性与风险管理

### 5.1 BZD/Z-drugs风险

| 风险 | 管理 |
|------|------|
| **依赖/耐受** | 限制使用≤4周；间歇给药 |
| **戒断/反跳** | 逐渐减量（每1-2周减25-50%） |
| **呼吸抑制** | OSA、COPD、老年人禁用或慎用 |
| **认知损害** | 老年人跌倒风险；避免长期使用 |
| **复杂睡眠行为** | 唑吡坦：梦游、驾驶、进食；FDA黑框警告 |
| **次日残留** | 半衰期长者避免；驾驶警告 |

### 5.2 特殊人群

| 人群 | 推荐 | 避免 |
|------|------|------|
| **老年人** | 多塞平3-6mg；褪黑素；雷美替胺；CBT-I | BZD；Z-drugs（跌倒） |
| **妊娠** | CBT-I；必要时曲唑酮/多塞平 | BZD；Z-drugs |
| **OSA** | DORA；雷美替胺 | BZD；Z-drugs |
| **COPD** | DORA；雷美替胺 | BZD；Z-drugs |
| **肝损害** | 低剂量；避免经肝代谢药物 | 半衰期长者 |
| **肾损害** | 调整剂量 | — |
| **物质滥用史** | CBT-I；非管制药物 | BZD；Z-drugs；兴奋剂 |
