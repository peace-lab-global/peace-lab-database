# 睡眠障碍与精神科共病 | Psychiatric Comorbidity in Sleep-Wake Disorders

> **目标**：系统阐述睡眠障碍与常见精神障碍的双向关系、共病机制与整合治疗策略。
> **标准来源**：APA Practice Guidelines、AASM共识声明、Sleep Medicine Reviews。

---

## 一、概述

睡眠障碍与精神障碍之间存在复杂的双向关系：
- 精神障碍可导致睡眠障碍（如抑郁导致失眠）
- 睡眠障碍可导致精神障碍（如慢性失眠增加抑郁风险）
- 两者可能共享共同病理机制
- 治疗一个可改善另一个

---

## 二、失眠与抑郁

### 2.1 双向关系

| 方向 | 证据 |
|------|------|
| **失眠→抑郁** | 非抑郁症失眠者发展为抑郁的风险增加2-3倍（1-3年内） |
| **抑郁→失眠** | 80-90%抑郁症患者报告睡眠问题 |
| **共同维持** | 失眠残留预测抑郁复发；抑郁残留预测失眠持续 |

### 2.2 整合治疗

| 策略 | 说明 |
|------|------|
| **同时治疗** | CBT-I + 抗抑郁治疗（CBT/药物） |
| **CBT-I优先** | 对于以失眠为主诉的患者 |
| **药物选择** | 米氮平、曲唑酮、阿戈美拉汀（兼具抗抑郁和促睡眠） |
| **注意** | SSRI可能初期加重失眠；可考虑早晨服用 |

---

## 三、失眠与焦虑

### 3.1 双向关系

| 方向 | 证据 |
|------|------|
| **焦虑→失眠** | GAD患者70-90%有失眠；失眠常先于焦虑诊断 |
| **失眠→焦虑** | 慢性失眠者焦虑风险增加 |
| **机制** | 过度觉醒；反刍；条件性焦虑（对睡眠/卧室的焦虑） |

### 3.2 整合治疗

| 策略 | 说明 |
|------|------|
| **CBT-I + CBT** | 同时进行失眠和焦虑的认知行为治疗 |
| **药物** | SSRI/SNRI（ mornings）；必要时加用曲唑酮/多塞平睡前 |
| **放松训练** | 共用于两者 |

---

## 四、OSA与抑郁

### 4.1 共病率

| 数据 | 说明 |
|------|------|
| **OSA→抑郁** | OSA患者中抑郁患病率15-40% |
| **抑郁→OSA** | 抑郁症患者中OSA患病率增加（尤其肥胖、老年） |

### 4.2 机制

| 机制 | 说明 |
|------|------|
| **低氧** | 间歇性低氧影响单胺类神经递质 |
| **睡眠碎片化** | 睡眠结构破坏影响情绪调节 |
| **炎症** | 系统性炎症与抑郁相关 |
| **生活质量** | 日间功能损害导致心理负担 |

### 4.3 治疗

| 策略 | 说明 |
|------|------|
| **CPAP** | 可改善轻中度抑郁症状；重度抑郁仍需抗抑郁治疗 |
| **抗抑郁药** | SSRI/SNRI；注意避免加重OSA的药物（如BZD） |
| **联合治疗** | CPAP + 抗抑郁药（中重度抑郁） |

---

## 五、发作性睡病与抑郁

| 数据 | 说明 |
|------|------|
| **共病率** | 30-50% |
| **原因** | 功能损害；社交孤立；诊断延迟；下丘脑功能异常 |
| **治疗** | 标准抗抑郁治疗；注意与发作性睡病药物的相互作用 |
| **羟丁酸钠注意** | 可能加重抑郁；需监测 |

---

## 六、PTSD与睡眠障碍

### 6.1 常见睡眠问题

| 问题 | 患病率 |
|------|--------|
| **噩梦** | 50-70% |
| **失眠** | 40-60% |
| **OSA** | 50-75%（远高于普通人群） |

### 6.2 整合治疗

| 问题 | 治疗 |
|------|------|
| **噩梦** | IRT（意象排演治疗）为一线；哌唑嗪；EMDR |
| **失眠** | CBT-I（需调整：谨慎暴露于创伤触发） |
| **OSA** | CPAP（依从性挑战；需PTSD敏感的方法） |
| **整体** | CPT/PE（创伤治疗）+ 睡眠靶向干预 |

---

## 七、双相障碍与睡眠

### 7.1 睡眠作为症状与触发因素

| 相态 | 睡眠特征 |
|------|---------|
| **躁狂/轻躁狂** | 睡眠需求减少（核心症状）；入睡困难、早醒 |
| **抑郁** | 失眠或嗜睡；早醒特征性 |
| **混合状态** | 睡眠严重紊乱 |

### 7.2 睡眠作为复发预测因子

| 信号 | 意义 |
|------|------|
| **睡眠减少** | 躁狂发作前最常见的前驱症状 |
| **社交节律紊乱** | 跨时区旅行、轮班可触发发作 |

### 7.3 整合治疗

| 策略 | 说明 |
|------|------|
| **社交节律治疗（SRT）** | 稳定日常节律；预防复发 |
| **睡眠卫生** | 尤其重要；避免睡眠剥夺 |
| **药物** | 心境稳定剂；必要时镇静药（quetiapine等） |
| **光照管理** | 躁狂期减少光照；抑郁期增加 |

---

## 八、精神分裂症与睡眠

### 8.1 常见睡眠问题

| 问题 | 患病率 |
|------|--------|
| **失眠** | 50-80% |
| **昼夜节律紊乱** | 常见 |
| **OSA** | 增加（尤其第二代抗精神病药致肥胖） |
| **RLS** | 增加（抗精神病药相关） |

### 8.2 治疗考量

| 问题 | 策略 |
|------|------|
| **抗精神病药致失眠** | 调整剂量时间；换用镇静型（喹硫平、奥氮平） |
| **OSA** | 体重管理；CPAP |
| **RLS** | 评估抗精神病药；必要时多巴胺激动剂 |

## 九、物质使用障碍与睡眠

### 9.1 各物质对睡眠的影响

| 物质 | 急性使用 | 戒断期 |
|------|---------|--------|
| **酒精** | 促进入睡；后半夜觉醒、REM抑制 | 严重失眠、焦虑、噩梦 |
| **大麻** | 促进入睡；长期使用致耐受 | 失眠、焦虑、梦境 vivid |
| **阿片类** | 嗜睡；睡眠呼吸抑制 | 失眠、RLS、焦虑 |
| **兴奋剂** | 严重失眠 | 过度嗜睡、抑郁 |
| **BZD** | 催眠 | 反跳性失眠、焦虑、癫痫 |
| **尼古丁** | 轻度促醒 | 失眠增加 |

### 9.2 治疗策略

| 原则 | 说明 |
|------|------|
| **先稳定物质使用** | 急性戒断期后再处理睡眠 |
| **非药物优先** | CBT-I；睡眠卫生；放松 |
| **药物谨慎** | 避免交叉依赖（尤其BZD/Z-drugs） |
| **共病精神障碍** | 同时治疗抑郁/焦虑 |

---

## 十、ADHD与睡眠

### 10.1 常见睡眠问题

| 问题 | 患病率 |
|------|--------|
| **入睡困难** | 50-75% |
| **不宁腿/PLMD** | 增加 |
| **昼夜节律延迟** | 常见 |
| **发作性睡病误诊** | 儿童注意鉴别 |

### 10.2 治疗考量

| 问题 | 策略 |
|------|------|
| **兴奋剂致失眠** | 早服药；换用长效制剂；必要时停药 |
| **褪黑素** | 对入睡困难有效（尤其ASD/ADHD儿童） |
| **RLS** | 铁蛋白检查；必要时多巴胺激动剂 |
