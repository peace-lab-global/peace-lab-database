# 嗜睡障碍与发作性睡病 | Hypersomnia Disorders and Narcolepsy

> **目标**：系统阐述日间过度嗜睡的诊断分类、鉴别诊断与治疗策略。
> **标准来源**：DSM-5-TR、AASM ICSD-3、欧洲发作性睡病指南。

---

## 一、概述

日间过度嗜睡（Excessive Daytime Sleepiness, EDS）是睡眠医学中最常见的症状之一，可由多种原因引起。嗜睡障碍特指以EDS为核心表现的疾病。

---

## 二、发作性睡病（Narcolepsy）

### 2.1 分型

| 类型 | 旧称 | 核心特征 | 下丘脑分泌素 |
|------|------|---------|-------------|
| **1型** | 伴猝倒的发作性睡病 | 猝倒 + EDS + 下丘脑分泌素缺乏 | 缺乏（CSF <110 pg/mL） |
| **2型** | 不伴猝倒的发作性睡病 | EDS + MSLT阳性；无猝倒 | 正常 |

### 2.2 核心症状（"嗜睡四联征"）

| 症状 | 描述 | 患病率 |
|------|------|--------|
| **EDS** | 不可抗拒的睡眠发作；短睡（10-30分钟）后恢复清醒 | 100% |
| **猝倒** | 情绪触发（大笑、愤怒）的短暂肌张力丧失；意识清醒 | 1型70-80% |
| **睡眠瘫痪** | 入睡或觉醒时短暂无法活动；意识清醒 | 25-50% |
| **入睡前/觉醒前幻觉** | 生动的视/听/触觉幻觉 | 25-50% |

### 2.3 流行病学

| 参数 | 数据 |
|------|------|
| **患病率** | 1型：0.02-0.05%；2型：0.02-0.05% |
| **发病年龄** | 双高峰：15岁和35岁 |
| **诊断延迟** | 平均8-15年 |

### 2.4 病因与机制

| 因素 | 说明 |
|------|------|
| **自身免疫** | 1型：H1N1感染/疫苗后自身免疫攻击下丘脑分泌素神经元 |
| **HLA关联** | >90% 1型患者为HLA-DQB1*06:02 |
| **下丘脑分泌素缺乏** | 1型核心机制；下丘脑外侧区神经元丧失>90% |
| **遗传** | 2型：家族聚集；多基因 |
| **继发性** | 脑外伤、肿瘤、炎症、血管病变累及下丘脑 |

---

## 三、特发性嗜睡（Idiopathic Hypersomnia, IH）

### 3.1 特征

| 参数 | 说明 |
|------|------|
| **核心** | EDS；非恢复性睡眠（即使睡眠>10小时仍不清醒） |
| **"睡眠醉酒"** | 觉醒困难；觉醒后意识模糊（可持续数小时） |
| **小睡** | 不恢复精力（与发作性睡病不同） |
| **夜间睡眠** | 通常延长（>10小时） |
| **MSLT** | 平均睡眠潜伏期<8分钟；无SOREMP或≤1次 |

### 3.2 与发作性睡病鉴别

| 特征 | 发作性睡病 | 特发性嗜睡 |
|------|-----------|-----------|
| **小睡效果** | 恢复精力 | 不恢复 |
| **夜间睡眠时长** | 正常或片段化 | 延长（>10h） |
| **觉醒困难** | 无 | 显著（睡眠醉酒） |
| **MSLT SOREMP** | ≥2次 | ≤1次 |
| **CSF下丘脑分泌素** | 1型缺乏 | 正常 |
| **HLA-DQB1*06:02** | 1型>90%阳性 | 无关联 |

## 四、Kleine-Levin综合征（KLS）

| 参数 | 说明 |
|------|------|
| **特征** | 反复发作的过度嗜睡（数天-数周）；伴随认知、行为、饮食改变 |
| **发作间期** | 完全正常 |
| **伴随症状** | 食欲亢进（megaphagia）；性欲亢进；现实解体；攻击性 |
| **患病率** | 罕见（1-2/百万） |
| **人群** | 青少年男性为主 |
| **病程** | 平均8-14年；通常自行缓解 |
| **治疗** | 锂盐（预防发作）；兴奋剂（急性期） |

## 五、诊断流程

### 5.1 评估步骤

```
主诉EDS
│
├─ 病史：睡眠时间、质量、打鼾、药物、轮班
├─ ESS评分（>10异常；>16严重）
├─ 睡眠日记（1-2周）
├─ 必要时PSG（排除OSA、PLMD）
│
├─ PSG正常或EDS无法解释
│   └─ MSLT
│       ├─ 平均潜伏期<8分钟 + ≥2次SOREMP
│       │   └─ 发作性睡病1型（有猝倒）或2型（无猝倒）
│       ├─ 平均潜伏期<8分钟 + ≤1次SOREMP
│       │   └─ 特发性嗜睡（排除睡眠不足）
│       └─ 平均潜伏期>8分钟
│           └─ 考虑其他原因：睡眠不足、药物、抑郁、甲减等
```

### 5.2 客观检查

| 检查 | 用途 |
|------|------|
| **PSG** | 排除OSA、PLMD；记录入睡潜伏期和REM潜伏期 |
| **MSLT** | 确诊发作性睡病/特发性嗜睡；平均睡眠潜伏期；SOREMP次数 |
| **MWT** | 维持觉醒能力（职业/驾照评估） |
| **CSF下丘脑分泌素** | 发作性睡病1型确诊 |
| **HLA分型** | 支持性（非诊断性） |
| **Actigraphy** | 记录长期睡眠-觉醒模式 |

## 六、治疗

### 6.1 发作性睡病治疗

**EDS治疗：**

| 药物 | 剂量 | 机制 | 注意 |
|------|------|------|------|
| **莫达非尼** | 100-400mg晨 | DA/NE再摄取抑制 | 一线；头痛；需监测血压 |
| **阿莫达非尼** | 150-250mg晨 | 莫达非尼R-异构体 | 半衰期更长 |
| **哌甲酯** | 5-60mg分次 | DA/NE释放 | 心血管注意；滥用风险 |
| **索利氨酯（Solriamfetol）** | 75-150mg晨 | DA/NE再摄取抑制 | FDA批准 |
| **羟丁酸钠（Sodium oxybate）** | 4.5-9g分2次（睡前+半夜） | GABA-B激动 | 猝倒+EDS均有效；Xyrem REMS |
| **替洛利生（Pitolisant）** | 17.8-35.6mg晨 | H3受体反向激动剂 | 猝倒+EDS；FDA批准 |

**猝倒治疗：**

| 药物 | 剂量 | 机制 |
|------|------|------|
| **羟丁酸钠** | 4.5-9g | GABA-B激动；一线 |
| **抗抑郁药** | — | — |
| ├─ 文拉法辛 | 37.5-150mg | SNRI；常用 |
| ├─ 度洛西汀 | 20-60mg | SNRI |
| ├─ 氯米帕明 | 10-75mg | TCA |
| ├─ 普罗替林 | 2.5-20mg | TCA |
| └─ 氟西汀 | 20-60mg | SSRI |
| **替洛利生** | 17.8-35.6mg | H3受体反向激动剂 |

**注意：**
- 抗抑郁药治疗猝倒时，突然停药可致猝倒反跳
- SSRIs/SNRIs可能抑制REM，但猝倒改善早于REM抑制

**夜间睡眠改善：**
- 羟丁酸钠：改善夜间睡眠片段化
- 苯二氮卓类（谨慎使用）

### 6.2 特发性嗜睡治疗

| 策略 | 说明 |
|------|------|
| **莫达非尼/阿莫达非尼** | 一线；效果不如发作性睡病 |
| **哌甲酯** | 替代 |
| **羟丁酸钠** | 部分有效 |
| **睡眠延长** | 允许长睡眠（不同于失眠治疗） |
| **觉醒困难** | 多个闹钟；定时唤醒设备；咖啡因 |

### 6.3 行为管理

| 策略 | 说明 |
|------|------|
| **规律小睡** | 计划性小睡（15-20分钟）可恢复精力（发作性睡病） |
| **驾驶安全** | 避免长距离驾驶；ESS>16不建议驾驶 |
| **职业调整** | 避免高危操作（高空、水下、重型机械） |
| **睡眠卫生** | 规律作息；充足夜间睡眠 |
| **患者教育** | 疾病知识；支持团体 |

## 七、与精神障碍的关联

### 7.1 抑郁与嗜睡

| 关联 | 说明 |
|------|------|
| **非典型抑郁** | 睡眠过多（hypersomnia）是特征 |
| **鉴别** | 抑郁：情绪低落、兴趣减退；发作性睡病：EDS但情绪可正常 |
| **共病** | 发作性睡病患者抑郁共病率高（30-50%） |

### 7.2 ADHD与发作性睡病

| 关联 | 说明 |
|------|------|
| **共病** | 儿童发作性睡病常被误诊为ADHD |
| **注意** | 哌甲酯可两者共用；但需排除发作性睡病 |
