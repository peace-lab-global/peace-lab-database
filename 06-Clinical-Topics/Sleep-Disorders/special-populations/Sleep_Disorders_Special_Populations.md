# 睡眠障碍特殊人群管理 | Sleep Disorders in Special Populations

> **目标**：系统阐述儿童青少年、老年人、妊娠期、围手术期及神经退行性疾病患者的睡眠障碍管理。
> **标准来源**：AASM Pediatric Guidelines、Geriatric Sleep Medicine、ACOG Guidelines。

---

## 一、儿童青少年睡眠障碍

### 1.1 常见睡眠问题

| 年龄组 | 常见问题 |
|--------|---------|
| **婴幼儿（0-2岁）** | 入睡困难、夜醒、睡眠联想 |
| **学龄前（3-5岁）** | 就寝抵抗、夜惊、噩梦 |
| **学龄期（6-12岁）** | 睡眠不足、OSA、失眠、行为性失眠 |
| **青少年（13-18岁）** | 睡眠相位延迟、睡眠不足、失眠、不宁腿 |

### 1.2 儿童失眠治疗

| 策略 | 年龄 | 方法 |
|------|------|------|
| **行为干预** | 所有年龄 | 一线；父母教育 |
| **消退法** | 婴幼儿 | 减少夜间关注；逐步或彻底 |
| **就寝程序** | 学龄前 | 固定、安静、可预测的睡前活动 |
| **父母在场消退** | 学龄前 | 逐步减少父母在房间的时间 |
| **CBT-I调整** | 青少年 | 适合认知水平的CBT-I |
| **褪黑素** | 儿童 | 3-5mg睡前；尤其ASD/ADHD |
| **避免** | 所有年龄 | BZD；Z-drugs（安全数据不足） |

### 1.3 青少年睡眠相位延迟

| 因素 | 说明 |
|------|------|
| **生理** | 青春期内源性昼夜节律相位延迟 |
| **社会** | 学业压力、电子设备使用、社交活动 |
| **治疗** | 早晨强光（上学前）；褪黑素（睡前1-2小时）；学校开学时间推迟 |

### 1.4 儿童OSA

| 特征 | 说明 |
|------|------|
| **原因** | 腺样体/扁桃体肥大（最常见） |
| **表现** | 打鼾、张口呼吸、生长迟缓、行为问题、ADHD样症状 |
| **治疗** | 腺样体扁桃体切除术（一线）；CPAP（术后残留或不适合手术） |

## 二、老年睡眠障碍

### 2.1 正常老化睡眠变化

| 变化 | 说明 |
|------|------|
| **睡眠结构** | 深睡眠（N3）减少；REM相对稳定 |
| **睡眠效率** | 下降（<85%常见） |
| **睡眠碎片化** | 觉醒次数增加 |
| **昼夜节律** | 相位提前（早睡早醒） |
| **睡眠时间** | 略减少（仍应≥6小时） |

### 2.2 老年失眠治疗

| 原则 | 说明 |
|------|------|
| **首选非药物** | CBT-I；睡眠卫生；光照 |
| **药物谨慎** | 低剂量起始；避免BZD/Z-drugs |
| **推荐药物** | 多塞平3-6mg；褪黑素；雷美替胺 |
| **避免** | 苯海拉明等OTC抗组胺（抗胆碱能：认知损害、便秘、尿潴留） |
| **注意** | 多重用药；药物相互作用；跌倒风险 |

### 2.3 老年RLS

| 特点 | 说明 |
|------|------|
| **患病率** | 10-35%（随年龄增加） |
| **继发原因** | 铁缺乏（常见）；肾功能不全；糖尿病 |
| **治疗** | 铁蛋白检查；补铁；低剂量多巴胺激动剂（注意冲动控制障碍） |

### 2.4 老年OSA

| 特点 | 说明 |
|------|------|
| **表现** | 日间嗜睡可能不如年轻人明显；认知下降、抑郁更突出 |
| **治疗** | CPAP同样有效；依从性可能更好 |

## 三、妊娠期睡眠障碍

### 3.1 妊娠期正常睡眠变化

| 孕期 | 变化 |
|------|------|
| **早孕期** | 嗜睡增加；夜间觉醒增加 |
| **中孕期** | 相对改善 |
| **晚孕期** | 频繁夜醒（排尿、胎动、不适）；打鼾增加；RLS常见 |

### 3.2 妊娠期失眠治疗

| 策略 | 安全性 |
|------|--------|
| **CBT-I** | 安全；首选 |
| **睡眠卫生** | 安全 |
| **放松训练** | 安全 |
| **左侧卧位** | 改善呼吸和循环 |
| **低剂量曲唑酮** | 相对安全（必要时） |
| **低剂量多塞平** | 相对安全（必要时） |
| **避免** | BZD（致畸风险）；Z-drugs（数据不足） |
| **褪黑素** | 数据有限；OTC常见使用 |

### 3.3 妊娠期RLS

| 特点 | 说明 |
|------|------|
| **患病率** | 20-30%（晚孕期） |
| **原因** | 铁缺乏；叶酸缺乏；激素变化 |
| **治疗** | 补铁（铁蛋白<75）；叶酸；非药物措施 |
| **避免** | 多巴胺激动剂（数据不足）；阿片类 |

### 3.4 妊娠期OSA

| 特点 | 说明 |
|------|------|
| **风险** | 妊娠高血压、子痫前期、妊娠糖尿病、胎儿生长受限 |
| **筛查** | 高危孕妇应筛查（肥胖、高血压、打鼾） |
| **治疗** | CPAP安全有效；一线治疗 |

## 四、围手术期睡眠管理

### 4.1 术前评估

| 内容 | 说明 |
|------|------|
| **OSA筛查** | STOP-BANG；高危→PSG |
| **CPAP使用** | 已确诊者术前恢复使用 |
| **镇静药谨慎** | OSA患者对镇静药敏感 |
| **困难气道** | OSA患者困难气道风险增加 |

### 4.2 术后管理

| 策略 | 说明 |
|------|------|
| **持续监测** | 高危OSA患者术后持续血氧监测 |
| **体位** | 侧卧或半卧位；避免仰卧 |
| **镇痛** | 多模式镇痛；减少阿片类 |
| **CPAP恢复** | 尽早恢复使用 |
| **谵妄预防** | 充足睡眠；减少夜间干扰；避免BZD |

## 五、神经退行性疾病

### 5.1 阿尔茨海默病

| 睡眠问题 | 患病率 | 管理 |
|---------|--------|------|
| **日落综合征** | 常见 | 日间光照；日间活动；减少晚间刺激 |
| **夜间觉醒/游走** | 常见 | 非药物优先；确保环境安全 |
| **昼夜节律紊乱** | 常见 | 褪黑素；光照治疗 |
| **药物** | — | 避免抗精神病药（黑框警告）；避免BZD |
| **OSA** | 增加 | CPAP（依从性挑战） |

### 5.2 帕金森病

| 睡眠问题 | 患病率 | 管理 |
|---------|--------|------|
| **RLS/PLMD** | 高 | 多巴胺能药物（注意与帕金森药物协调） |
| **REM睡眠行为障碍（RBD）** | 高 | 氯硝西泮；褪黑素 |
| **日间嗜睡** | 高 | 调整多巴胺药物时间；莫达非尼 |
| **失眠** | 高 | CBT-I；优化帕金森药物 |

### 5.3 多发性硬化

| 睡眠问题 | 管理 |
|---------|------|
| **疲劳** | 鉴别睡眠问题 vs MS疲劳 |
| **RLS** | 常见；多巴胺激动剂 |
| **OSA** | 增加；CPAP |
