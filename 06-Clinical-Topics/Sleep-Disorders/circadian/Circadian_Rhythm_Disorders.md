# 昼夜节律睡眠-觉醒障碍 | Circadian Rhythm Sleep-Wake Disorders

> **目标**：系统阐述昼夜节律睡眠-觉醒障碍的诊断分类、评估与治疗策略。
> **标准来源**：DSM-5-TR、AASM ICSD-3、AASM Clinical Practice Guidelines。

---

## 一、概述

昼夜节律睡眠-觉醒障碍是一组由于内源性昼夜节律与环境/社会需求不同步导致的睡眠-觉醒时序异常，表现为入睡困难、日间嗜睡或两者兼有。

---

## 二、分类与诊断

### 2.1 延迟睡眠-觉醒相位障碍（DSWPD）

| 参数 | 说明 |
|------|------|
| **特征** | 睡眠 onset 和觉醒时间持续延迟（通常>2AM入睡，>10AM起床） |
| **患病率** | 青少年0.4-3%；成人<0.2% |
| **高峰年龄** | 青少年期（生理相位延迟+社会因素） |
| **病程** | 慢性（>3个月） |

**诊断标准：**
- 主诉入睡困难或日间嗜睡
- 睡眠 onset 和觉醒时间持续延迟
- 症状≥3个月
- 非其他障碍所致

### 2.2 提前睡眠-觉醒相位障碍（ASWPD）

| 参数 | 说明 |
|------|------|
| **特征** | 睡眠 onset 和觉醒时间提前（通常<8PM入睡，<5AM起床） |
| **患病率** | 中老年为主；罕见 |
| **遗传** | PER2基因突变（家族性） |

### 2.3 不规则睡眠-觉醒节律障碍（ISWRD）

| 参数 | 说明 |
|------|------|
| **特征** | 睡眠-觉醒模式碎片化、无规律；无明确主睡眠期 |
| **患病率** | 神经退行性疾病患者常见（痴呆50-70%） |
| **原因** | 视交叉上核功能退化；光照不足；活动缺乏 |

### 2.4 非24小时睡眠-觉醒节律障碍（N24SWD）

| 参数 | 说明 |
|------|------|
| **特征** | 内源性昼夜节律周期>24小时；睡眠 onset 每日延迟 |
| **人群** | 盲人（50-70%完全失明者）；少数视力正常人 |
| **机制** | 光-暗信号无法同步生物钟 |

### 2.5 轮班工作睡眠-觉醒障碍（SWSD）

| 参数 | 说明 |
|------|------|
| **特征** | 轮班工作时间与内源性昼夜节律冲突 |
| **患病率** | 轮班工人10-40% |
| **风险** | 事故、心血管代谢疾病、癌症（IARC 2A类致癌物） |

### 2.6 时差障碍

| 参数 | 说明 |
|------|------|
| **特征** | 跨时区旅行后的睡眠-觉醒紊乱 |
| **方向** | 东向飞行比西向飞行更难适应 |
| **恢复** | 每跨越1个时区需1天适应 |

## 三、客观评估

### 3.1 体动记录仪（Actigraphy）

| 用途 | 说明 |
|------|------|
| **DSWPD/ASWPD** | 记录≥7天（含工作日+周末）的睡眠-觉醒模式 |
| **ISWRD** | 显示碎片化、无规律模式 |
| **N24SWD** | 显示每日延迟的睡眠 onset |
| **SWSD** | 记录轮班期间的睡眠片段化 |

### 3.2 核心体温与褪黑素节律

| 方法 | 用途 |
|------|------|
| **核心体温最低点** | 判定昼夜节律相位（通常在睡眠最后1/3） |
| **DLMO（暗光褪黑素释放 onset）** | 金标准；判定内源性相位 |

## 四、治疗

### 4.1 延迟睡眠-觉醒相位障碍（DSWPD）

| 治疗 | 方法 | 证据 |
|------|------|------|
| **时间疗法（Chronotherapy）** | 逐步延迟就寝时间（每2-3天延迟3小时），绕时钟一周后固定 | B级 |
| **晨光暴露** | 觉醒后立即强光（2500-10000 lux，30-60分钟） | A级 |
| **褪黑素** | 目标就寝前5-7小时服用低剂量（0.5-3mg） | A级 |
| **认知行为治疗** | 睡眠卫生；刺激控制；设定固定起床时间 | B级 |
| **避免晚间光暴露** | 限制屏幕蓝光；使用蓝光过滤 | B级 |

### 4.2 提前睡眠-觉醒相位障碍（ASWPD）

| 治疗 | 方法 | 证据 |
|------|------|------|
| **傍晚光暴露** | 晚上7-9点强光 | B级 |
| **晨间褪黑素** | 避免（会加重） | — |
| **行为干预** | 晚间活动安排；避免过早卧床 | C级 |

### 4.3 不规则睡眠-觉醒节律障碍（ISWRD）

| 治疗 | 方法 | 证据 |
|------|------|------|
| **日间光照** | 白天明亮光照（>1000 lux，至少2小时） | A级 |
| **增加日间活动** | 结构化日间活动；减少卧床 | B级 |
| **夜间最小化光照** | 夜间低光照环境 | B级 |
| **褪黑素** | 睡前低剂量（1-3mg） | B级 |
| ** caregiver 教育** | 建立规律护理时间表 | B级 |

### 4.4 非24小时睡眠-觉醒节律障碍（N24SWD）

| 治疗 | 方法 | 证据 |
|------|------|------|
| **他司美琼（Tasimelteon）** | 20mg睡前 | A级（FDA批准） |
| **褪黑素** | 睡前1-5mg | B级 |
| **光疗** | 如视力允许 | C级 |

### 4.5 轮班工作睡眠-觉醒障碍

| 策略 | 方法 |
|------|------|
| **优化轮班制度** | 顺时针轮班（早→午→夜）；最少11小时休息间隔；避免连续夜班>3天 |
| **策略性小睡** | 夜班前小睡（1.5-3小时）；夜班中小睡（20-30分钟） |
| **光疗** | 夜班期间强光（>1000 lux）；下班后戴墨镜减少光暴露 |
| **褪黑素** | 日间睡眠前30分钟服用（1-3mg） |
| **咖啡因** | 夜班开始时适度使用；睡眠前6小时避免 |
| **莫达非尼/阿莫达非尼** | 夜班前服用（FDA批准用于SWSD） |

### 4.6 时差障碍

| 策略 | 东向飞行 | 西向飞行 |
|------|---------|---------|
| **光疗** | 早晨避免光；午后暴露光 | 午后避免光；晚间暴露光 |
| **褪黑素** | 目标 bedtime 前服用 | 目标 bedtime 前服用 |
| **调整睡眠** | 提前就寝 | 延迟就寝 |
| **药物** | 短效催眠药（短期） | 短效催眠药（短期） |

## 五、与精神障碍的关联

### 5.1 DSWPD与抑郁/焦虑

| 关联 | 说明 |
|------|------|
| **共病率** | DSWPD患者中抑郁/焦虑共病率高达50% |
| **机制** | 社会时差；学业/工作功能损害；晨间光照不足 |
| **季节性** | 类似冬季抑郁（低光照暴露） |
| **治疗** | 相位提前治疗可能改善情绪 |

### 5.2 双相障碍与昼夜节律

| 关联 | 说明 |
|------|------|
| **睡眠减少** | 躁狂前兆；睡眠需求减少是核心症状 |
| **社交节律治疗** | 稳定日常节律可预防复发（SRT） |
| **光照管理** | 躁狂期减少光照；抑郁期增加光照 |
