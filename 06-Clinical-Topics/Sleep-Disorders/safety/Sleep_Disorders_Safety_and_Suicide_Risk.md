# 睡眠障碍安全与自杀风险管理 | Safety and Suicide Risk in Sleep Disorders

> **目标**：建立睡眠障碍患者的安全评估框架，特别关注失眠、噩梦与自杀风险的关联。
> **标准来源**：AASM Best Practice Guide、Suicide Prevention Resource Center、WHO guidelines。

---

## 一、睡眠障碍与自杀风险的关联

### 1.1 失眠与自杀

| 证据 | 说明 |
|------|------|
| **独立风险因素** | 失眠是自杀意念和行为的独立预测因子（OR 2.1-2.6） |
| **机制** | 睡眠剥夺 → 前额叶功能下降 → 冲动控制减弱；情绪调节障碍；绝望感增加 |
| **时间关联** | 凌晨2-6点是自杀行为高发时段（与睡眠障碍相关） |
| **治疗影响** | CBT-I可能降低自杀风险（通过改善睡眠和情绪） |

### 1.2 噩梦与自杀

| 证据 | 说明 |
|------|------|
| **PTSD** | 噩梦是PTSD患者自杀的强预测因子 |
| **独立关联** | 非PTSD噩梦也与自杀风险增加相关 |
| **机制** | 噩梦导致睡眠恐惧、回避、绝望；REM睡眠碎片化影响情绪加工 |

### 1.3 其他睡眠障碍

| 障碍 | 风险 |
|------|------|
| **OSA** | 抑郁共病增加自杀风险；日间功能损害 |
| **发作性睡病** | 抑郁共病率高；猝倒导致功能损害 |
| **RLS** | 严重病例抑郁/焦虑增加 |

## 二、安全评估框架

### 2.1 自杀风险评估（每次就诊）

| 评估要素 | 具体内容 |
|---------|---------|
| **自杀意念** | 频率、强度、持续性；是否有计划 |
| **自杀计划** | 具体性、可获得性、致命性 |
| **保护因素** | 社会支持、宗教信仰、责任心、治疗关系 |
| **危险因素** | 既往未遂史、精神障碍、物质滥用、近期应激、睡眠障碍严重程度 |
| **睡眠特异性** | 失眠严重程度；噩梦频率；凌晨觉醒 |

### 2.2 标准化工具

| 工具 | 用途 |
|------|------|
| **C-SSRS（哥伦比亚自杀严重程度评估）** | 标准化自杀风险评估 |
| **PHQ-9（第9项）** | 自杀意念筛查 |
| **ISI** | 失眠严重程度 |
| **噩梦频率量表** | 噩梦评估 |

## 三、安全管理策略

### 3.1 失眠患者

| 策略 | 说明 |
|------|------|
| **优先治疗失眠** | 失眠改善可降低自杀风险 |
| **限制药物数量** | 避免大处方；BZD/Z-drugs限量 |
| **定期随访** | 前4周每周；之后每2-4周 |
| **凌晨时段注意** | 识别凌晨觉醒为高危时段 |
| **共病抑郁筛查** | 每次评估抑郁症状 |

### 3.2 噩梦患者

| 策略 | 说明 |
|------|------|
| **IRT（意象排演治疗）** | 优先治疗噩梦 |
| **评估PTSD** | 噩梦常与PTSD共病 |
| **睡眠安全** | 确保睡眠环境安全（尤其RBD/睡行） |

### 3.3 OSA患者

| 策略 | 说明 |
|------|------|
| **驾驶安全** | ESS>16不建议驾驶；未治疗OSA驾驶风险增加 |
| **职业安全** | 避免操作重型机械、高空作业 |
| **CPAP依从性** | 强调治疗对日间功能和安全的重要性 |

### 3.4 发作性睡病患者

| 策略 | 说明 |
|------|------|
| **驾驶限制** | 未控制EDS不建议驾驶 |
| **职业选择** | 避免高危职业 |
| **猝倒安全** | 避免单独游泳、高空活动 |

## 四、药物安全

### 4.1 催眠药安全处方

| 原则 | 说明 |
|------|------|
| **最小有效剂量** | 从最低剂量开始 |
| **短期处方** | 初始≤2-4周；续方需重新评估 |
| **避免联合** | 不与酒精、阿片类、其他镇静药合用 |
| **老年谨慎** | 优先非药物；避免BZD |
| **防止囤积** | 小量处方；定期清点 |

### 4.2 兴奋剂安全（发作性睡病）

| 原则 | 说明 |
|------|------|
| **心血管监测** | 血压、心率 |
| **滥用风险** | 有物质滥用史者慎用哌甲酯 |
| **羟丁酸钠** | REMS程序；床旁给药；呼吸监测 |

## 五、危机应对

### 5.1 自杀风险升级处理

| 级别 | 表现 | 行动 |
|------|------|------|
| **低危** | 被动意念；无计划；有保护因素 | 加强随访；治疗睡眠；安全计划 |
| **中危** | 主动意念；模糊计划 | 安全计划；精神科会诊；增加随访频率；限制手段 |
| **高危** | 具体计划；手段可获得；缺乏保护因素 | 紧急精神科评估；考虑住院；24小时陪护 |
