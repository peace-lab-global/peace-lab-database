# 睡眠相关运动障碍 | Sleep-Related Movement Disorders

> **目标**：系统阐述睡眠相关运动障碍的诊断、评估与治疗，重点涵盖周期性肢体运动障碍（PLMD）和睡眠磨牙（Sleep Bruxism）。
> **标准来源**：AASM ICSD-3（2014）、AASM Clinical Practice Guidelines。

---

## 一、周期性肢体运动障碍（PLMD）

### 1.1 定义

周期性肢体运动障碍（Periodic Limb Movement Disorder, PLMD）是指睡眠中出现的周期性、重复性、刻板的肢体运动（通常为下肢），导致睡眠片段化和日间功能损害。

**关键：与 RLS 的区别**

| 特征 | RLS | PLMD |
|------|-----|------|
| **时间** | 清醒时（静息） | 睡眠中 |
| **主观感受** | 患者强烈感觉不适 | 患者通常无觉察（床伴发现） |
| **运动** | 活动后缓解 | 周期性、刻板运动 |
| **关系** | 80-90% RLS 患者有 PLM | PLMD 可无 RLS |

### 1.2 ICSD-3 诊断标准

**A.** PSG 显示周期性肢体运动指数（PLMI）≥5次/小时（儿童）或≥15次/小时（成人）。

**B.** 周期性肢体运动导致睡眠紊乱或日间功能损害。

**C.** 不能用其他睡眠障碍、躯体疾病、药物或物质更好地解释。

**D.** 不符合 RLS 的诊断标准（或 RLS 不足以解释症状）。

### 1.3 临床表现

| 特征 | 说明 |
|------|------|
| **运动特征** | 大脚趾背伸；踝背屈；膝/髋屈曲；持续0.5-10秒；间隔5-90秒 |
| **模式** | 呈周期性发作；每晚数次 |
| **主观** | 患者常不知晓；主诉为"睡眠差""日间嗜睡" |
| **床伴报告** | "他/她睡觉时腿抽动" |

### 1.4 评估

| 检查 | 发现 |
|------|------|
| **PSG** | 金标准；记录 PLM 指数 |
| **PLMI** | ≥15/h（成人）；≥5/h（儿童） |
| **伴随觉醒** | PLM 导致皮质觉醒（即使患者未完全醒来） |
| **血检** | 铁蛋白（PLMD 常与低铁相关） |

### 1.5 治疗

| 治疗 | 证据 | 说明 |
|------|------|------|
| **补铁** | B级 | 铁蛋白<75 ng/mL 时补充；静脉铁（严重） |
| **多巴胺激动剂** | A级 | 普拉克索、罗匹尼罗（注意 augmentation） |
| **α2δ配体** | A级 | 加巴喷丁、普瑞巴林 |
| **苯二氮卓类** | B级 | 氯硝西泮（注意耐受） |
| **阿片类** | B级 | 难治性病例 |

## 二、睡眠磨牙（Sleep Bruxism）

### 2.1 定义

睡眠磨牙（Sleep Bruxism）是指睡眠期间的牙齿磨动或紧咬，伴随下颌肌肉的节律性（咀嚼样）或持续性（紧咬）活动。

### 2.2 ICSD-3 诊断标准

**A.** 睡眠期间反复出现的牙齿磨动或紧咬，伴有以下之一：
1. 牙齿异常磨损
2. 与牙齿磨动相关的声音报告（床伴）
3. 下颌肌肉不适、疲劳或疼痛、锁定感
4. 咬肌肥大（咀嚼肌增大）
5. 晨起颞下颌关节（TMJ）僵硬或疼痛

**B.** 不能用其他睡眠障碍、躯体疾病、药物或物质更好地解释。

### 2.3 评估

| 方法 | 说明 |
|------|------|
| **临床检查** | 牙齿磨损；咬肌肥大；TMJ 压痛 |
| **床伴报告** | 夜间磨牙声音 |
| **PSG/EMG** | 咬肌肌电图记录（研究用） |
| **视频记录** | 家用视频记录（临床辅助） |

### 2.4 危险因素

| 因素 | 说明 |
|------|------|
| **应激/焦虑** | 心理应激加重磨牙 |
| **睡眠障碍** | OSA 共病常见；睡眠片段化 |
| **药物** | SSRI、SNRI、兴奋剂等可能加重 |
| **物质** | 酒精、咖啡因、尼古丁 |
| **遗传** | 家族史 |
| **咬合** | 牙齿咬合异常（存在争议） |

### 2.5 治疗

| 治疗 | 证据 | 说明 |
|------|------|------|
| **口腔护具（Occlusal splint）** | A级 | 一线；保护牙齿；减少肌肉活动 |
| **CBT/压力管理** | B级 | 处理应激和焦虑 |
| **肌松弛剂** | C级 | 短期；肌肉松弛 |
| **肉毒素（Botox）** | B级 | 严重病例；咬肌注射 |
| **治疗共病 OSA** | B级 | CPAP 可能改善磨牙 |
| **药物调整** | C级 | 如 SSRI 诱发，考虑换药 |
| **避免刺激物** | C级 | 减少酒精、咖啡因 |

## 三、其他睡眠相关运动障碍（简述）

| 障碍 | 特征 | 治疗 |
|------|------|------|
| **睡眠相关节律性运动障碍（RMD）** | 重复性撞头、摇身体、滚身体；常见于儿童 | 通常随年龄消退；严重可用床垫/药物 |
| **睡眠相关腿痉挛** | 睡眠中突发的下肢痛性痉挛 | 伸展；补镁；奎宁（慎用） |
| **入睡期抽动（Hypnic Jerks）** | 入睡时突发的肢体抽动；良性 | 通常无需治疗；减少咖啡因/应激 |
| ** propriospinal myoclonus** | 躯干肌阵挛；需排除器质性 | 病因治疗 |
| **癫痫相关运动** | 夜间癫痫发作；需 PSG+EEG | 抗癫痫治疗 |

## 四、与精神科共病

| 障碍 | 共病 | 管理 |
|------|------|------|
| **PLMD** | 抑郁、焦虑、RLS | 治疗 PLMD；评估铁缺乏 |
| **睡眠磨牙** | 焦虑、应激、SSRI 使用 | 口腔护具 + 焦虑管理 |
| **RMD** | 自闭症、智力障碍 | 行为干预；保护性环境 |
