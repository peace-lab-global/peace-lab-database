# 图式疗法 (Schema Therapy)

> **核心定位**：Jeffrey Young 发展的整合性长程心理治疗，融合 CBT/精神动力学/格式塔/EMDR 元素。适用于慢性人格障碍（BPD/NPD），通常需 2-3 年。

---

## 1. 核心理论

### 1.1 早期适应不良图式 (EMS) — 18 种

| 领域 | 图式名称 | 核心信念 |
|:-----|:---------|:---------|
| **分离与拒绝** | 被抛弃/不稳定 | "重要的人会离开我" |
| | 不信任/虐待 | "别人会伤害/利用我" |
| | 情感剥夺 | "我的情感需求不会被满足" |
| | 缺陷/羞耻 | "如果人们真正了解我，他们会拒绝我" |
| | 社交孤立 | "我不属于任何群体" |
| **自主与能力** | 依赖/无能 | "我无法独立应对" |
| | 脆弱/伤害 | "随时可能发生灾难" |
| | 纠缠/未发展自我 | "我与某人（常是父母）过度亲密" |
| | 失败 | "我注定失败" |
| **限制受损** | 权利/夸大 | "规则不适用于我" |
| | 不足自控/自律 | "我无法控制自己" |
| **他人导向** | 顺从 | "我必须满足他人" |
| | 自我牺牲 | "我的需求不重要" |
| | 寻求认可 | "我的价值取决于他人认可" |
| **过度警觉/抑制** | 消极/悲观 | "负面结果总是发生" |
| | 情绪抑制 | "表达情绪是危险的" |
| | 苛刻标准 | "我必须完美" |
| | 惩罚 | "错误必须受到惩罚" |

### 1.2 图式风格

| 风格 | 描述 | 举例 |
|:-----|:-----|:-----|
| **顺从 (Surrender)** | 接受图式为真 | 缺陷图式→选择贬低自己的关系 |
| **逃避 (Avoidance)** | 避免触图式的情境 | 不信任→回避亲密关系 |
| **过度补偿 (Overcompensation)** | 表现得好像图式不真 | 缺陷→极端完美主义/夸大 |

---

## 2. 模式工作 (Mode Work)

### 2.1 四种模式类别

| 模式类型 | 子类型 | BPD 中的表现 |
|:---------|:-------|:------------|
| **儿童模式** | 脆弱儿童/愤怒儿童/冲动儿童/快乐儿童 | "受伤的内在小孩" |
| **适应不良的父母模式** | 惩罚性父母/苛求父母 | "内在批评者" |
| **适应不良的应对模式** | 顺从/逃避/过度补偿 | 自动化的应对方式 |
| **健康成人模式** | 健康成人/快乐儿童 | 治疗目标——发展和强化 |

### 2.2 模式工作目标

```
治疗过程：
1. 识别当前激活的模式
2. 理解模式间的冲突
3. 减少适应不良的父母模式和应对模式
4. 抚慰脆弱儿童模式
5. 发展和强化健康成人模式
```

---

## 3. 治疗技术

| 技术类别 | 具体技术 |
|:---------|:---------|
| **认知技术** | 图式日记/证据审查/闪卡 |
| **体验技术** | 意象重塑(imagery rescripting)/椅子对话/信（不发送） |
| **行为技术** | 行为实验/暴露/技能训练 |
| **治疗关系** | 有限再抚育(limited reparenting) |

### 3.1 有限再抚育

| 方面 | 说明 |
|:-----|:-----|
| 定义 | 治疗师在专业范围内满足患者的未满足情感需求 |
| 边界 | 明确的职业边界（非真实养育） |
| 形式 | 温暖/支持/设定限制/灵活度 |
| 目的 | 提供安全基础→修正图式 |

## 4. 适应症与证据

| 适应证 | 证据 | 疗程 |
|:-------|:-----|:-----|
| BPD | 强（Giesen-Bloo et al., 2006） | 2-3年 |
| NPD | 中 | 2-3年 |
| 其他人格障碍 | 中 | 1-2年 |
| 慢性抑郁 | 中-强 | 1-2年 |
| 复杂性 PTSD | 初步 | 2年+ |
