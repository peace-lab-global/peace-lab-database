# 依赖型人格障碍 (Dependent Personality Disorder, DPD)

> **核心定位**：以过度需要被照顾、导致顺从和依附行为、害怕分离为特征。患病率 0.5-0.6%，临床就诊率较高（因焦虑/抑郁共病）。

---

## 1. DSM-5 诊断标准（≥5/8）

| # | 标准 |
|:--|:-----|
| 1 | 没有他人大量建议和保证时难以做出日常决定 |
| 2 | 需要他人为自己生活的大多数主要领域承担责任 |
| 3 | 因害怕失去支持或认可而难以向他人表达不同意见 |
| 4 | 因缺乏自信（而非缺乏动力/精力）而难以独自启动项目 |
| 5 | 为获得他人的照顾和支持而过度付出（甚至做不愉快的事） |
| 6 | 独处时因害怕无法自我照顾而感到不舒服/无助 |
| 7 | 当一段亲密关系结束时急切寻求另一段关系作为支持和照顾来源 |
| 8 | 不现实地害怕被抛弃而不得不自我照顾 |

---

## 2. 鉴别诊断

| DPD vs | 关键区分 |
|:-------|:---------|
| BPD | DPD 顺从依附 vs BPD 不稳定/愤怒交替 |
| AvPD | DPD 寻求依附 vs AvPD 回避关系 |
| 抑郁症 | DPD 为持续性人格模式 vs 抑郁为发作性 |
| 躯体化障碍 | DPD 依赖行为 vs 躯体症状求关注 |

---

## 3. 病因学

| 因素 | 说明 |
|:-----|:-----|
| 依恋理论 | 不安全依恋（焦虑-矛盾型） |
| 家庭教养 | 过度保护/专制型/不允许独立 |
| 气质 | 高伤害回避+低新奇寻求 |
| 文化 | 集体主义文化中患病率更高 |

---

## 4. 治疗

| 方法 | 说明 |
|:-----|:-----|
| CBT | 逐步增加自主决策/减少依赖行为 |
| 心理动力学 | 处理分离-个体化冲突 |
| 图式疗法 | "依赖/无能"和"自我牺牲"图式 |
| 团体治疗 | 练习自主性和人际边界 |
| 治疗关系注意 | 防止过度依赖治疗师/设置边界 |
