## 一、循证等级（ISTSS 2019 / APA 2017 / NICE NG116 2018）

| 疗法 | ISTSS 推荐 | APA 证据 | NICE NG116 | 一线/二线 |
|------|-----------|---------|-----------|----------|
| **创伤聚焦 CBT (TF-CBT)** | A（强烈推荐，成人 & 儿童） | 强 | 强 | 一线 |
| **延长暴露 (PE)** | A | 强 | 强 | 一线 |
| **认知加工治疗 (CPT)** | A | 强 | 强 | 一线 |
| **EMDR** | A | 强 | 强（成人） | 一线 |
| **叙事暴露疗法 (NET)** | B（难民/多创伤） | 中 | 中 | 一线（特定人群） |
| **药物治疗 (SSRI/SNRI)** | B | 中 | 二线（心理治疗失败/拒绝时） | 二线 |
| **躯体体验疗法 (SE)** | C（证据新兴） | 弱 | 未单列 | 辅助/替代 |
| **创伤敏感正念 / MBCT-T** | C | 弱-中 | 未单列 | 稳定化辅助 |

> 共识：**创伤聚焦心理治疗（TF-CBT/PE/CPT/EMDR）是成人 PTSD 的一线选择**；药物用于辅助、共病或拒绝心理治疗者。

## 二、Herman 三阶段模型（Complex PTSD 必备）

| 阶段 | 核心任务 | 主要技术 | 何时进入 |
|------|---------|---------|---------|
| **1. 稳定化与安全** | 安全（环境/人际/内在）、情绪调节、危机应对、减少自伤/物质 | 安全计划、DBT 技能、接地、心理教育、人际节律 | **所有 C-PTSD 必先完成**；不达标不进入阶段 2 |
| **2. 创伤加工** | 在稳定基础上逐步面对与整合创伤记忆 | PE / CPT / EMDR / NET / SE（节律放慢版） | 患者能调节情绪 ≥ 30 分钟、无活跃自杀/物质急性期 |
| **3. 整合与重建** | 重建身份、关系、意义、未来导向 | 意义建构、关系修复、社会参与、PTGI 促进 | 症状缓解 + 自我感恢复后 |

**关键原则**：阶段非严格线性，可在阶段间往返；**再创伤化的核心风险是跳过阶段 1** 直接做暴露。

## 三、四大一线创伤聚焦疗法

### 1. 延长暴露（Prolonged Exposure, PE）—— Foa

| 项目 | 内容 |
|------|------|
| **机制** | 通过**想象暴露**（反复叙述 index trauma）+ **现场暴露**（回避情境）实现恐惧消退、去条件化 |
| **疗程** | 8–15 次，每周 1 次，60–90 分钟/次 |
| **核心组件** (1) 心理教育；(2) 呼吸再训练；(3) 现实情境暴露（in vivo，分级）；(4) 想象暴露（创伤叙述 + 录音回放）；(5) 情绪加工 |
| **适应症** | 单次/多次事件 PTSD（事故、攻击、战斗）效果强 |
| **慎用/禁忌** | 活跃自杀、严重解离（DES 高）、当前严重物质使用、缺乏稳定化能力 |
| **疗效** | 严重度下降 50–70%；约 50% 治疗后不再符合 PTSD 诊断 |

### 2. 认知加工治疗（Cognitive Processing Therapy, CPT）—— Resick

| 项目 | 内容 |
|------|------|
| **机制** | 修正创伤相关的**卡点信念**（stuck points）—— 安全、信任、力量、尊重、亲密 5 维度 |
| **疗程** | **12 次**（标准版），60 分钟；有 6 次简化版 |
| **核心组件** (1) 影响陈述（impact statement）；(2) 与创伤相关想法工作表；(3) 苏格拉底式提问挑战卡点；(4) 信任/亲密/安全模块；(5) 可选：**不写创伤叙述**的纯认知版（CPT-C）适合强烈回避者 |
| **适应症** | 性暴力、军事创伤、负性自我/世界信念强的患者；**适合不想做暴露叙述者** |
| **优势** | 对**羞耻/罪恶感**（D 簇）特别有效；不需要详细叙述创伤 |
| **疗效** | 与 PE 相当，对认知/情绪簇改善更显著 |

### 3. 创伤聚焦 CBT（TF-CBT）—— Cohen, Deblinger, Mannarino

| 项目 | 内容 |
|------|------|
| **机制** | 整合创伤叙事、认知三角、亲子联合、行为管理 |
| **疗程** | **12–25 次**（通常 12–16）；分儿童段 + 家长段 + 联合段 |
| **核心组件（PRACTICE 助记）** Psychoeducation & Parenting skills, Relaxation, Affective expression, Cognitive coping, Trauma narrative & processing, In vivo mastery, Conjoint child-parent sessions, Enhancing safety |
| **适应症** | **儿童青少年 PTSD / 性虐待 / 家庭暴力**一线（详见 [TF-CBT 目录](../../../02-Mind-Psychology/therapy/cognitive-behavioral/tf-cbt-therapy/)） |
| **疗效** | 儿童 PTSD 疗效最一致，效应量大 |

### 4. EMDR（眼动脱敏与再加工）—— Shapiro

| 项目 | 内容 |
|------|------|
| **机制** | 双侧刺激（眼动/触觉/听觉）配合创伤记忆再加工；**工作记忆假说**与 REM 类机制 |
| **疗程** | 6–12 次（单创伤）；复杂创伤需更多 |
| **八阶段**（详见 [EMDR 目录](../../../02-Mind-Psychology/therapy/integrative/emdr-therapy/)） (1) 病史与计划；(2) 准备（安全资源）；(3) 评估（目标记忆）；(4) 脱敏（双侧刺激）；(5) 植入；(6) 身体扫描；(7) 结束；(8) 再评估 |
| **适应症** | 单次创伤尤其有效；**不要求详细叙述**（适合强烈回避/语言表达困难者） |
| **争议** | 双侧刺激的特异性仍有争议，但总体疗效与 PE/CPT 相当 |
| **疗效** | 起效较快；对噩梦/闪回改善显著 |

## 四、复杂/多次创伤人群的辅助疗法

### 叙事暴露疗法（Narrative Exposure Therapy, NET）—— Schauer, Neuner, Elbert
- **机制**：构建完整生命时间线，将离散创伤片段整合为连贯自传叙事。
- **疗程**：4–10 次，每次 60–120 分钟。
- **适应症**：**难民、战争幸存者、多重创伤、酷刑幸存者** —— 当多个 index trauma 难以单一锁定时。
- **疗效**：在难民人群证据最强（ISTSS 推荐 B/A）。

### 躯体体验疗法（Somatic Experiencing, SE）—— Levine
- **机制**：追踪创伤相关的躯体感觉，完成未完成的防御反应（fight/flight/freeze），通过**滴定**（titration）与**摆荡**（pendulation）避免再创伤化。
- **疗程**：灵活，非标准化；常作为稳定化或创伤加工的辅助。
- **适应症**：高度解离/难语言化者、对暴露治疗恐惧者、跨文化偏好身体工作者。
- **证据**：C 级（新兴），详见 [SE 目录](../../../02-Mind-Psychology/therapy/integrative/somatic-experiencing/)。

### 内在家庭系统（Internal Family Systems, IFS）—— Schwartz
- **机制**：识别创伤分裂出的"部分"（exiles/managers/firefighters），发展"自我"（Self）领导。
- **适应症**：C-PTSD、解离倾向、人格结构受影响者。
- **证据**：C 级（新兴，ICD-11 C-PTSD 人群研究增长中）。

## 五、创伤敏感正念与 MBCT-for-trauma

传统 MBSR/MBCT 对创伤人群存在**再创伤化风险**（长时间闭眼身体扫描可能触发闪回）。需改造：

| 改造要点 | 实施 |
|---------|------|
| **睁眼/可选姿势** | 允许睁眼、移动；不强制坐姿 |
| **接地优先** | 每次练习以 5 感接地开始/结束 |
| **窗口化（window of tolerance）** | 监测唤醒水平，超出窗口即暂停 |
| **短时长、可退出** | 单次 3–5 分钟；明确"随时可睁眼/离开" |
| **资源导向** | 优先觉察中性/正性体验（"安全的身体部位"） |

**MBCT for trauma**（MBCT-T，结核反应版）—— 适应 C-PTSD 人群，证据 B 级，与 [MBCT 临床专题](../../mbct/) 整合使用。

## 六、剂量学（Dose）与疗程决策

| 疗法 | 推荐次数 | 单次时长 | 总疗程 |
|------|---------|---------|--------|
| PE | 8–15 | 60–90 分钟 | 2–4 个月 |
| CPT | 12（标准）/ 6（简化） | 60 分钟 | 3 个月 |
| EMDR | 6–12（单创伤）/ 更多（复杂） | 60–90 分钟 | 2–4 个月 |
| TF-CBT（儿童） | 12–25 | 45–60 分钟 | 3–6 个月 |
| NET | 4–10 | 60–120 分钟 | 1–3 个月 |
| **C-PTSD 阶段化综合** | **1–2 年（含稳定化）** | — | 长期 |

**早期反应预测**：PE/CPT 第 4 周症状改善 ≥ 20% 是良好结局的强预测因子；若 6 周无反应（< 10% 改善）应重新评估（不依从、共病、解离、错误 index trauma）。

## 七、治疗选择决策树

```
确认 PTSD 诊断（见 diagnosis/）
│
├─ 单次创伤、愿叙述、能调节情绪 ──────→ PE 或 CPT
│
├─ 强烈羞耻/罪恶感、不想详述 ────────→ CPT（认知版）
│
├─ 不愿/不能详细叙述 ───────────────→ EMDR
│
├─ 多重创伤、难民背景 ──────────────→ NET
│
├─ 儿童/青少年 ────────────────────→ TF-CBT
│
├─ 高度解离（DES≥30）/ 严重不稳定 ──→ 先稳定化（DBT 技能、SE、创伤敏感正念），稳定后再评估
│
├─ 拒绝心理治疗 / 共病严重抑郁 ────→ 药物（见 pharmacology/）+ 稳定化，稳定后再引入创伤聚焦
│
└─ 复杂 PTSD ────────────────────→ 三阶段模型，长期综合
```

## 八、治疗中的关键风险与处理

| 风险 | 表现 | 处理 |
|------|------|------|
| **再创伤化** | 暴露中过度唤醒、闪回、急性解离 | 回到稳定化；滴定暴露；增加接地 |
| **不依从** | 家庭作业不做、脱落 | 动机访谈；可从 CPT-C（不写叙述）入门 |
| **症状恶化** | 自伤、自杀意念增加 | 暂停暴露；强化安全计划；必要时住院 |
| **治疗关系破裂** | 感到被强迫、不信任 | 修复性对话；调整节奏；增加患者控制感 |
| **共病物质使用** | 暴露期物质使用增加 | 整合双重诊断治疗，不可先戒断再治创伤 |

## 九、群体与特殊人群适应

- **退伍军人 / 急救人员**：道德损伤需专门处理；群体同辈支持有效；可结合 ACT。
- **性暴力幸存者**：CPT 对羞耻/自责尤其有效；强调患者控制权与边界。
- **难民/酷刑幸存者**：NET + 文化适应 + 翻译/口译伦理；躯体症状常见。
- **童年创伤/C-PTSD**：三阶段模型；强烈推荐阶段 1 稳定化（DBT/SE/正念）。
- **老年人**：调整暴露强度；注意认知与躯体合并症。
