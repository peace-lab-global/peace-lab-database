# 认知加工疗法 (Cognitive Processing Therapy, CPT)

> **核心定位**：由 Patricia Resick 开发的循证认知行为疗法。通过识别和挑战**卡住点**(stuck points)——关于创伤的错误信念，重建对自我、他人和世界的适应性认知。

---

## 1. 理论基础

| 概念 | 说明 |
|:-----|:-----|
| 同化-调适模型 | 创伤打破既有图式→需调适（而非同化） |
| 卡住点 | 关于安全/信任/权力/亲密/自尊的僵化信念 |
| 5 个主题 | 安全/信任/权力与控制/自尊/亲密 |
| 认知重构 | Socratic 对话挑战卡住点→发展平衡认知 |

---

## 2. 治疗结构（12 次）

| 节次 | 内容 |
|:----:|:-----|
| 1-2 | 心理教育/PTSD 认知模型/识别自动思维 |
| 3 | 创伤影响陈述（写作作业） |
| 4-5 | 识别和挑战卡住点/Socratic 对话 |
| 6-7 | 安全主题 |
| 8-9 | 信任主题 |
| 10 | 权力与控制主题 |
| 11 | 自尊主题 |
| 12 | 亲密主题/回顾/未来规划 |

---

## 3. 核心技术

| 技术 | 说明 |
|:-----|:-----|
| **影响陈述** | 写作创伤对自己的影响（第3节前完成） |
| **ABC 表** | 事件→信念→结果→挑战→替代思维 |
| **Socratic 对话** | 引导式提问挑战卡住点（非对抗） |
| **卡住点日志** | 持续记录和挑战卡住信念 |
| **5 主题模块** | 系统化重建五大认知领域 |

### 3.1 卡住点示例

| 领域 | 卡住点示例 | 挑战方向 |
|:-----|:----------|:---------|
| 安全 | "世界完全不安全" | 评估证据/概率/应对能力 |
| 信任 | "我不能相信任何人" | 区分信任程度/信任判断力 |
| 权力 | "我无力保护自己" | 回顾应对/当下资源 |
| 自尊 | "创伤是我的错" | 责任归属/自我同情 |
| 亲密 | "亲密就意味着受伤" | 区分关系/渐进信任 |

---

## 4. CPT 变体

| 变体 | 说明 |
|:-----|:-----|
| **CPT-C** (Cognitive Only) | 不含创伤叙述写作，纯认知 |
| **CPT+A** (With Written Account) | 包含创伤详细叙述 |
| **Group CPT** | 团体形式（12 周/封闭团体） |

> 研究发现 CPT-C 与 CPT+A 效果相当（Resick et al. 2008）

---

## 5. 疗效证据

| 数据 | 发现 |
|:-----|:-----|
| APA 指南 | **强烈推荐** |
| 效应量 | d = 0.90-1.12 |
| 缓解率 | ~50-70% 不再符合 PTSD |
| 军事人群 | 有效但缓解率略低（~40-50%） |
| 团体 vs 个体 | 效果相当 |
| 远程/视频 | 初步证据有效 |
