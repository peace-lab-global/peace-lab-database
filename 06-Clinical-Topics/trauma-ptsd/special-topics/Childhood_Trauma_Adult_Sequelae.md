# 童年创伤成人后遗症 (Childhood Trauma & Adult Sequelae)

> **核心定位**：基于 ACE (Adverse Childhood Experiences) 研究，系统梳理童年创伤对成人心理健康、躯体健康和行为的长期影响。强调**剂量-效应关系**和**表观遗传学机制**。

---

## 1. ACE 研究 (Felitti et al. 1998)

| 要素 | 说明 |
|:-----|:-----|
| 原始研究 | Kaiser Permanente + CDC, N=17,000+ |
| ACE 类别 | 10 类（虐待/忽视/家庭功能障碍） |
| 核心发现 | ACE 评分↑ → 健康/社会/行为风险↑ |
| 剂量效应 | 线性关系：≥4 ACE 显著增加风险 |
| 患病率 | 67% 至少 1 ACE；12.6% ≥4 ACE |

### 1.1 ACE 10 类

| 类别 | 具体 |
|:-----|:-----|
| **虐待** | 身体虐待/性虐待/情感虐待 |
| **忽视** | 身体忽视/情感忽视 |
| **家庭功能障碍** | 父母精神疾病/物质滥用/家庭暴力/父母离异/家庭成员入狱 |

---

## 2. 心理健康后果

| 后果 | 风险增加 | 说明 |
|:-----|:---------|:-----|
| 抑郁症 | 4.6x | 与 ACE 评分强相关 |
| PTSD | 12x | 尤其性虐待 |
| 自杀企图 | 12x | 4+ ACE |
| 焦虑障碍 | 3-4x | 广泛性焦虑/恐慌/PD |
| 物质滥用 | 7.2x | 静脉注射风险 46x |
| 人格障碍 | 3-5x | 尤其 BPD/AvPD |
| 解离障碍 | 10x+ | DID/DDNOS |

---

## 3. 躯体健康后果

| 系统 | 后果 |
|:-----|:-----|
| 心血管 | 缺血性心脏病 (2.2x) / 卒中 |
| 免疫 | 自身免疫病 / 慢性炎症 |
| 内分泌 | 糖尿病 / 肥胖 / 代谢综合征 |
| 呼吸 | COPD / 哮喘 |
| 消化 | IBS / 肝病 / GI 症状 |
| 疼痛 | 慢性疼痛 / 纤维肌痛 |
| 肿瘤 | 肺癌 / 多种癌症 |
| 寿命 | ≥6 ACE → 寿命缩短 20 年 |

---

## 4. 神经生物学机制

| 机制 | 说明 |
|:-----|:-----|
| HPA 轴编程 | 早期应激永久改变皮质醇反应模式 |
| 表观遗传 | DNA 甲基化/组蛋白修饰改变基因表达 |
| 神经可塑性 | 杏仁核↑/前额叶↓/海马体↓ |
| 炎症 | 慢性低度炎症（CRP↑/IL-6↑/TNF-α↑） |
| 端粒 | 加速端粒缩短（生物学衰老） |
| 迷走神经 | 腹侧迷走发育不足/社交参与受损 |

---

## 5. 社会功能影响

| 领域 | 影响 |
|:-----|:-----|
| 教育 | 学业成就降低 / 辍学率↑ |
| 就业 | 失业 / 收入降低 / 工作不稳定 |
| 关系 | 亲密关系困难 / 家庭暴力 / 离婚 |
| 司法 | 犯罪率↑ / 入狱率↑ |
| 代际传递 | 创伤教养模式代际传递 |

---

## 6. 韧性与保护因素

| 因素 | 说明 |
|:-----|:-----|
| 至少一个稳定成人 | 最强保护因素 |
| 安全依恋关系 | 与替代照顾者的安全关系 |
| 自我调节能力 | 情绪/行为调节 |
| 意义建构 | 创伤后成长 |
| 社区支持 | 学校/邻里/宗教 |
| 早期干预 | 减少累积剂量 |
