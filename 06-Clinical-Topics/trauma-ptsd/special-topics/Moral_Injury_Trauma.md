# 道德伤害 (Moral Injury)

> **核心定位**：不是正式诊断，而是一个**概念框架**。描述经历、目睹或参与违反深层道德信念的行为后产生的持久心理/精神/存在性痛苦。与 PTSD 重叠但核心情感不同（羞耻 vs 恐惧）。

---

## 1. 定义与起源

| 要素 | 说明 |
|:-----|:-----|
| 核心定义 | 违反个人深层道德信念的行为（作为/不作为/见证） |
| 提出者 | Shay (1994) → Litz et al. (2009) 操作化 |
| 核心情感 | **羞耻/内疚/道德困扰**（vs PTSD 的恐惧） |
| 关键区分 | "我做了一件坏事"（道德伤害） vs "可怕的事发生在我身上"（PTSD） |
| 非正式诊断 | 概念框架，无 DSM/ICD 编码 |

---

## 2. 道德伤害事件类型

| 类型 | 示例 |
|:-----|:-----|
| **作为 (Commission)** | 战争中误伤平民/虐待俘虏 |
| **不作为 (Omission)** | 未能救助/未能阻止暴行 |
| **见证 (Witnessing)** | 目睹屠杀/虐待/不公正 |
| **被背叛 (Betrayal)** | 被上级/机构/战友背叛（Shay 的 betrayal-based） |

---

## 3. 高风险群体

| 群体 | 道德伤害来源 |
|:-----|:-----------|
| **退伍军人** | 战争暴行/误伤/战友死亡/被遗弃感 |
| **医护人员** | COVID 分诊/患者死亡/资源不足/无法提供最佳照护 |
| **急救人员** | 无法拯救/目睹死亡 |
| **执法人员** | 使用武力/无法保护 |
| **难民** | 被迫参与/见证暴行 |
| **虐待幸存者** | 被照顾者虐待的道德悖论 |

---

## 4. 道德伤害 vs PTSD

| 维度 | PTSD | 道德伤害 |
|:-----|:-----|:---------|
| 核心情感 | 恐惧/焦虑 | **羞耻/内疚** |
| 核心信念 | "世界危险" | "我是坏的/有罪的" |
| 核心症状 | 再体验/回避/警觉 | 自我惩罚/退缩/意义丧失 |
| 关系 | 回避创伤提醒 | 感觉不配/自我隔离 |
| 行为 | 过度警觉/逃避 | 自毁/自我药疗/自杀 |
| 治疗响应 | PE/CPT/EMDR 有效 | 需**道德/存在性**干预 |

---

## 5. 干预方法

| 方法 | 说明 |
|:-----|:-----|
| **Adaptive Disclosure** | Litz: 暴露+认知+慈悲/修复行为 |
| **Impact of Killing** | Maguen: 专门处理战争中杀戮 |
| **Soul Repair** | 宗教/灵性/社区修复 |
| **慈悲聚焦 (CFT)** | Gilbert: 处理羞耻/自我慈悲 |
| **叙事/存在性** | 意义重建/宽恕/道德重构 |
| **修复行为** | 服务/志愿/弥补/仪式 |
| **团体** | 同伴支持/共同见证/去隔离 |

---

## 6. 评估工具

| 工具 | 说明 |
|:-----|:-----|
| Moral Injury Events Scale (MIES) | 9 项自评 |
| Moral Injury Symptom Scale (MISS-M) | 军人版/医护版 |
| Expressions of Moral Injury Questionnaire | 表达性评估 |
| Trauma-Related Guilt Inventory (TRGI) | 创伤相关内疚 |
