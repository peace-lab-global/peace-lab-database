# 哀伤障碍诊断标准 | Grief-Related Disorders: DSM-5-TR and ICD-11

> **目标**：建立延长哀伤障碍（PGD）及其他哀伤相关障碍的精神科标准诊断框架。
> **标准来源**：DSM-5-TR（2022）、ICD-11（2022）、WHO Prolonged Grief Disorder Guidelines（2022）、APA PGD临床手册。

---

## 一、DSM-5-TR 延长哀伤障碍（Prolonged Grief Disorder, PGD）

### 1.1 核心诊断标准

**A.** 亲近的人死亡（通常≥12个月前；儿童≥6个月）。

**B.** 死亡后持续存在强烈的悲伤/渴望/思念，至少以下**3项中每日/几乎每日出现，持续≥1个月**（儿童：≥6个月）：
1. 身份认同瓦解（感觉自我的一部分已死去）
2. 对死亡明显的不信任
3. 回避提醒物（内部：与逝者的记忆；外部：提醒物）
4. 死亡后情感麻木
5. 死亡后感觉生活无意义
6. 强烈孤独感（感到与他人疏离/脱离）

**C.** 症状导致临床上显著的痛苦或社交、职业或其他重要领域的功能损害。

**D.** 症状不能更好地用重性抑郁障碍、创伤后应激障碍或其他精神障碍解释。

**E.** 症状不能归因于物质/躯体疾病的生理效应。

## 二、ICD-11 延长哀伤障碍

### 2.1 核心特征

| 特征 | 说明 |
|------|------|
| **核心** | 对逝者的持久、 pervasive 的渴望/思念 |
| **时间** | 通常≥6个月（文化考虑允许更长） |
| **严重程度** | 导致功能显著损害 |
| **鉴别** | 不能更好地用抑郁或PTSD解释 |

### 2.2 诊断要件

**A.** 亲近的人死亡。

**B.** 持续强烈的渴望/思念逝者，和/或持续 preoccupation 于逝者。

**C.** 死亡后持续存在强烈的情感痛苦（以下至少1项每日/几乎每日）：
- 悲伤、内疚、愤怒
- 否认/质疑/难以接受死亡
- 感觉失去自我的一部分
- 情感麻木
- 难以参与积极回忆

**D.** 导致显著功能损害（≥1个月）。

**E.** 不能更好地用抑郁障碍、PTSD或其他精神障碍解释。

---

## 三、ICD-11 适应障碍（伴有抑郁反应/其他情绪反应）

| 特征 | 说明 |
|------|------|
| **适用** | 哀伤反应不符合PGD但功能损害显著 |
| **时间** | 症状在可识别的应激源（死亡）后1个月内出现 |
| **持续时间** | 通常不超过6个月（应激源持续时可能更长） |
| **特点** | 以抑郁、焦虑或混合情绪反应为主；程度轻于PGD |

## 四、正常哀伤 vs 病理性哀伤

### 4.1 正常哀伤的特征

| 维度 | 正常哀伤 |
|------|---------|
| **时间** | 数月到数年；逐渐减轻 |
| **强度** | 波动；有改善期 |
| **功能** | 虽有困难但基本维持 |
| **自杀意念** | 可能有"希望死去以重聚"但无计划 |
| **意义感** | 逐渐重建；哀伤融入身份 |
| **与他人连接** | 保持社会联系 |

### 4.2 延长哀伤障碍的特征

| 维度 | PGD |
|------|-----|
| **时间** | 持续≥12月（成人）；无改善趋势 |
| **强度** | 持续强烈；无缓解期 |
| **功能** | 严重损害 |
| **自杀意念** | 可能有具体计划 |
| **意义感** | 持续感到生活无意义 |
| **与他人连接** | 严重退缩；强烈的孤独感 |

### 4.3 鉴别要点总结

| 特征 | 正常哀伤 | 适应障碍 | PGD | 抑郁症 | PTSD |
|------|---------|---------|-----|--------|------|
| **核心** | 思念逝者 | 对应激的情绪反应 | 强烈的渴望/ preoccupation | 广泛情绪低落 | 创伤性再体验 |
| **时间** | 数月-数年 | ≤6月（通常） | ≥12月（成人） | ≥2周 | ≥1月 |
| **对逝者** | 正面/混合回忆 | 不一定相关 | 强烈的渴望/思念 | 不一定 | 可能有侵入 |
| **触发** | 提醒物时悲伤 | 应激源相关 | 持续存在 | 广泛 | 创伤线索 |
| **自我评价** | 保持 | 可能降低 | 身份认同瓦解 | 显著降低 | 可能有罪恶感 |
| **未来取向** | 逐渐恢复 | 可恢复 | 感到无意义 | 无望 | 受限 |

## 五、风险评估

### 5.1 自杀风险（丧亲后高危期）

| 数据 | 说明 |
|------|------|
| **丧偶后** | 自杀风险增加约2倍（尤其老年男性） |
| **时间** | 风险在丧亲后第一年最高 |
| **特殊丧失** | 自杀死亡、儿童死亡的父母自杀风险更高 |

### 5.2 PGD风险因素

| 风险因素 | 说明 |
|---------|------|
| **丧亲类型** | 非预期死亡、暴力死亡、儿童死亡、自杀 |
| **关系** | 依赖关系、冲突关系、未竟事宜 |
| **既往** | 既往PGD、抑郁、PTSD、依恋问题 |
| **社会** | 社会支持缺乏、孤立 |
| **经济** | 经济困难（尤其丧偶） |
| **文化** | 缺乏哀悼仪式 |
