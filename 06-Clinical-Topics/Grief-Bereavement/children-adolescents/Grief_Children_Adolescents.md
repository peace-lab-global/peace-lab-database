# 儿童与青少年的哀伤 | Grief in Children and Adolescents

> **目标**：系统阐述儿童青少年哀伤的发展性特点、评估与年龄适宜的干预。
> **标准来源**：APA Guidelines for Child and Adolescent Grief、DSM-5-TR、Stroebe et al. (2007)。

---

## 一、发展性哀伤反应

### 1.1 婴幼儿（0-2岁）

| 反应 | 说明 |
|------|------|
| **行为** | 食欲下降、睡眠紊乱、哭泣增加、依恋行为增加 |
| **认知** | 无法理解死亡的永久性 |
| **需求** | 稳定的照顾者；一致的 routine |

### 1.2 学龄前（3-5岁）

| 反应 | 说明 |
|------|------|
| **认知** | 可能认为死亡是可逆的； magical thinking（"如果我乖，妈妈会回来"） |
| **行为** | 退行（尿床、吮手指）；游戏再现死亡主题 |
| **情绪** | 短暂、强烈；容易被分散注意 |
| **需求** | 简单、诚实的解释；保持 routine；允许提问 |

### 1.3 学龄期（6-12岁）

| 反应 | 说明 |
|------|------|
| **认知** | 理解死亡的永久性；可能产生恐惧（"我也会死"） |
| **情绪** | 悲伤、愤怒、 guilt（"是我造成的"） |
| **行为** | 学校表现下降；社交退缩；躯体化（头痛、腹痛） |
| **需求** | 参与哀悼（选择参加）；学校支持；艺术/游戏表达 |

### 1.4 青少年（13-18岁）

| 反应 | 说明 |
|------|------|
| **认知** | 抽象思考；思考意义和存在 |
| **情绪** | 强烈的悲伤、愤怒；可能隐藏情绪（"应该坚强"） |
| **行为** | 风险行为增加（物质使用、 reckless 驾驶）；学业变化 |
| **需求** | 同伴支持；自主权；不"被当作小孩"；参与决策 |

---

## 二、评估

### 2.1 年龄适宜评估

| 年龄 | 方法 |
|------|------|
| **0-5岁** | 观察；父母报告；游戏评估 |
| **6-12岁** | 绘画、游戏、故事、量表（儿童版） |
| **13-18岁** | 访谈、量表、日记、艺术表达 |

### 2.2 儿童哀伤量表

| 量表 | 年龄 | 用途 |
|------|------|------|
| **ICG-Revised（儿童版）** | 8-18岁 | 复杂性哀伤 |
| **Texas Revised Inventory of Grief（儿童版）** | 8-18岁 | 哀伤反应 |
| **Child Behavior Checklist（CBCL）** | 6-18岁 | 一般行为/情绪问题 |
| **RCADS** | 8-18岁 | 焦虑/抑郁筛查 |

### 2.3 警示信号（需专业干预）

| 信号 | 说明 |
|------|------|
| **持续功能损害** | >6个月；影响学校/社交/家庭 |
| **自杀意念/行为** | 任何年龄都需立即评估 |
| **创伤症状** | 闪回、噩梦、严重回避 |
| **严重退行** | 持续数月的显著退行行为 |
| **物质使用** | 青少年新发物质使用 |
| **风险行为** | 自伤、 reckless 行为 |

---

## 三、干预

### 3.1 家庭支持

| 策略 | 说明 |
|------|------|
| **诚实沟通** | 使用"死亡""死了"等词；避免委婉语 |
| **允许提问** | 回答简单、诚实；"我不知道"也是答案 |
| **保持 routine** | 一致性提供安全感 |
| **允许参与** | 根据年龄参与葬礼/仪式 |
| **父母自我照顾** | 父母哀伤影响儿童；父母需支持 |

### 3.2 学校支持

| 策略 | 说明 |
|------|------|
| **通知相关人员** | 老师、辅导员知情 |
| **灵活安排** | 允许额外时间、缺勤 |
| **同伴支持** | 教育同学如何支持 |
| **学校纪念** | 适当的纪念活动 |

### 3.3 游戏治疗

| 适用 | 3-12岁 |
|------|--------|
| **方式** | 玩具、绘画、沙盘、故事 |
| **作用** | 表达难以言说的情感；加工丧失 |
| **内容** | 死亡主题的游戏是正常的 |

### 3.4 青少年团体

| 形式 | 同辈支持团体 |
|------|------------|
| **优势** | 同伴理解；减少孤立 |
| **内容** | 分享经历；应对策略； memorial 活动 |

### 3.5 家庭治疗

| 适用 | 家庭功能受损；兄弟姐妹冲突；父母哀伤不同步 |
|------|------------------------------------------|
| **重点** | 沟通；支持彼此；共同纪念 |

---

## 四、父母的特殊关注

### 4.1 父母丧失孩子

| 特征 | 最严重的哀伤之一 |
|------|---------------|
| **婚姻压力** | 哀伤不同步常见；需夫妻治疗 |
| **对兄弟姐妹** | 可能过度保护或忽视 |
| **长期** | 哀伤可能持续终身（但强度变化） |

### 4.2 丧偶后的育儿

| 挑战 | 说明 |
|------|------|
| **单亲压力** | 经济与情感双重负担 |
| **孩子成为"情感支持"** | 避免 parentification |
| **新伴侣** | 需谨慎处理 |
