# 哀伤的精神科共病 | Psychiatric Comorbidity in Grief

> **目标**：系统阐述哀伤过程中常见精神障碍的共病、鉴别与整合治疗。
> **标准来源**：DSM-5-TR、APA PGD临床手册、Lichtenthal et al. (2004)。

---

## 一、共病率总览

| 共病障碍 | PGD患者中共病率 | 一般丧亲人群 |
|---------|--------------|-------------|
| **重性抑郁障碍** | 40-60% | 15-30% |
| **创伤后应激障碍** | 20-40% | 5-15% |
| **广泛性焦虑障碍** | 15-30% | 5-10% |
| **物质使用障碍** | 10-25% | 5-10% |
| **自杀意念/行为** | 20-40% | 5-10% |
| **睡眠障碍** | 50-70% | 30-50% |

---

## 二、哀伤与抑郁

### 2.1 鉴别

| 特征 | 正常哀伤 | PGD | 抑郁症 |
|------|---------|-----|--------|
| **核心** | 思念逝者 | 强烈的渴望/ preoccupation | 广泛情绪低落 |
| **情绪波动** | 有改善期 | 持续强烈 | 持续性 |
| **自尊** | 保持 | 身份认同问题 | 显著降低 |
| **对逝者的情感** | 思念；可回忆美好 | 强烈的渴望/思念 | 可能无特定关联 |
| **未来的希望** | 逐渐恢复 | 感到无意义 | 无望 |
| **快感** | 可体验 | 显著减少 | 广泛丧失 |

### 2.2 共存时的治疗

| 情况 | 策略 |
|------|------|
| **抑郁共病PGD** | CGT + SSRI（中重度抑郁） |
| **单纯抑郁（无PGD）** | 标准抑郁治疗；哀伤支持 |
| **鉴别困难** | 优先治疗更严重的；观察对治疗反应 |

---

## 三、哀伤与PTSD

### 3.1 鉴别

| 特征 | PGD | PTSD | 创伤性哀伤 |
|------|-----|------|-----------|
| **核心** | 渴望/思念 | 创伤再体验 | 两者兼具 |
| **侵入** | 可能有 | 强烈 | 强烈 |
| **回避** | 回避提醒物 | 广泛回避 | 广泛 |
| **警觉** | 无 | 有 | 有 |
| **分离痛苦** | 强烈 | 不一定 | 强烈 |

### 3.2 共存时的治疗

| 情况 | 策略 |
|------|------|
| **PGD + PTSD** | CGT调整 + EMDR/PE；整合创伤和哀伤处理 |
| **单纯PTSD** | 标准PTSD治疗（PE/EMDR/CPT） |

---

## 四、哀伤与焦虑

| 类型 | 表现 | 治疗 |
|------|------|------|
| **GAD** | 广泛担忧；包括对未来的恐惧 | CBT + 必要时SSRI |
| **分离焦虑** | 害怕再丧失；过度依恋 | 暴露 + 安全行为减少 |
| **健康焦虑** | 过度担忧自己/他人的健康 | CBT； reassurance 减少 |
| **存在焦虑** | 对死亡和意义的焦虑 | 存在主义治疗；意义重建 |

## 五、哀伤与物质使用

| 机制 | 说明 |
|------|------|
| **自我药疗** | 用酒精/药物麻痹悲伤 |
| **社交使用** | 社交场合使用增加 |
| **风险** | 物质使用可能加重抑郁/失眠 |

### 治疗

| 策略 | 说明 |
|------|------|
| **评估严重程度** | AUDIT/DAST |
| **先稳定物质** | 急性戒断/依赖者先处理 |
| **哀伤处理** | 处理底层情绪 |
| **替代策略** | 健康的情绪调节替代物质 |

## 六、哀伤与自杀风险

### 6.1 风险因素

| 因素 | 说明 |
|------|------|
| **丧偶** | 老年男性风险最高 |
| **自杀死亡** | 遗族自杀风险增加 |
| **儿童死亡** | 父母自杀风险增加 |
| **既往自杀未遂** | 既往史是强预测因子 |
| **抑郁共病** | 风险叠加 |
| **社会孤立** | 缺乏支持 |

### 6.2 管理

| 策略 | 说明 |
|------|------|
| **每次评估** | 丧亲者每次接触都评估自杀 |
| **安全计划** | 制定个人安全计划 |
| **限制手段** | 移除/限制自杀手段 |
| **增加支持** | 增加联系频率；家庭参与 |
| **危机计划** | 明确危机时联系谁 |

---

## 七、睡眠障碍

| 问题 | 常见 | 治疗 |
|------|------|------|
| **入睡困难** | 思念逝者；焦虑 | CBT-I；曲唑酮 |
| **早醒** | 抑郁特征 | 治疗抑郁；睡眠卫生 |
| **噩梦** | PTSD；思念逝者 | 意象排演治疗；哌唑嗪 |
