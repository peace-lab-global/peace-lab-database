# 预感性哀伤与姑息医学中的哀伤支持 | Anticipatory Grief and Bereavement Support in Palliative Care

> **目标**：系统阐述预感性哀伤的临床特征、评估与姑息医学中的哀伤支持策略。
> **标准来源**：WHO姑息医学指南、APA PGD临床手册、Stroebe & Schut (2010)。

---

## 一、预感性哀伤（Anticipatory Grief）

### 1.1 定义

预感性哀伤是指在预期丧失（通常为死亡）发生之前，面对即将丧失而产生的悲伤反应。它是**对未来的丧失提前进行哀悼**的过程。

### 1.2 与反应性哀伤的比较

| 维度 | 预感性哀伤 | 反应性哀伤（死亡后） |
|------|----------|-------------------|
| **时间** | 死亡前（诊断后、临终期） | 死亡后 |
| **对象** | 预期中的丧失 | 已发生的丧失 |
| **特点** | 既有希望又有悲伤的矛盾 | 确定性的丧失 |
| **功能** | 可能帮助提前心理准备 | 整合已发生的丧失 |
| **复杂性** | 可能因希望而否认；可能因预悲而内疚 | 相对明确 |

### 1.3 常见表现

| 维度 | 表现 |
|------|------|
| **情感** | 悲伤、焦虑、愤怒、内疚、解脱感（矛盾） |
| **认知** | "他走了我怎么办""我还没准备好"反复思考 |
| **行为** | 过度照顾、回避谈论死亡、过早"放手" |
| **关系** | 与患者关系的重新定义；"正在失去"的感觉 |

### 1.4 预感性哀伤的悖论

| 方面 | 说明 |
|------|------|
| **可能有益** | 提前心理准备；死亡后哀伤可能较轻 |
| **可能有害** | 过早"情感撤离"；患者感到被遗弃；照顾质量下降 |
| **矛盾情绪** | 既希望结束痛苦又害怕失去 |
| **内疚** | "我在希望他死"的自我批评 |

## 二、预感性哀伤的评估

### 2.1 评估要点

| 评估 | 内容 |
|------|------|
| **预期时间** | 死亡预期多久？不确定性如何？ |
| **关系质量** | 依恋类型；冲突 vs 亲密 |
| **既往丧失** | 既往哀伤模式 |
| **支持系统** | 家庭、朋友、社区支持 |
| **功能状态** | 照顾者是否还能有效照顾？ |
| **自杀风险** | 照顾者是否有"不想活了"的想法？ |

### 2.2 预感性哀伤量表

| 量表 | 用途 |
|------|------|
| **AGS（Anticipatory Grief Scale）** | 评估预感性哀伤严重程度 |
| **Marwit-Meuser Caregiver Grief Inventory** | 照顾者预悲评估 |
| **Zarit Burden Interview** | 照顾负担（与预悲相关） |

## 三、预感性哀伤的干预

### 3.1 原则

| 原则 | 说明 |
|------|------|
| **正常化** | "在亲人临终前感到悲伤是完全正常的" |
| **允许矛盾** | "既希望他不再痛苦，又害怕失去他，这两种感受可以共存" |
| **保持连接** | 鼓励在患者还在世时继续情感连接，不要过早撤离 |
| **珍惜当下** | 帮助家庭创造有意义的时刻 |
| **自我照顾** | 照顾者需要照顾自己的身心健康 |

### 3.2 具体干预

| 干预 | 方法 |
|------|------|
| **生命回顾（Life Review）** | 与患者一起回顾人生；记录故事 |
| ** Legacy 工作** | 录制视频、写信、制作相册 |
| **未竟事宜处理** | 帮助表达"对不起""谢谢你""我爱你" |
| **关系修复** | 如果有冲突，促进和解 |
| **告别仪式** | 在患者还在世时进行有意义的告别 |
| **照顾者支持** | 喘息服务；支持团体；心理咨询 |

### 3.3 避免过早撤离

| 警告信号 | 干预 |
|----------|------|
| **照顾者情感疏离** | 鼓励继续陪伴；"他还在这里" |
| **减少探访** | 探索原因；不评判 |
| **"我已经准备好了"** | 区分真正的准备 vs 防御性否认 |

## 四、姑息医学中的哀伤支持

### 4.1 团队角色

| 角色 | 哀伤支持职责 |
|------|-------------|
| **医生** | 告知病情；预后沟通；症状管理 |
| **护士** | 陪伴；症状监测；家属支持 |
| **社工** | 实际支持；资源链接；家庭评估 |
| **心理师/咨询师** | 情绪支持；预悲处理；家庭治疗 |
| **灵性照护师** | 意义、希望、信仰支持 |
| **志愿者** | 陪伴；喘息服务 |

### 4.2 告知病情与沟通

| 原则 | SPIKES 模型 |
|------|------------|
| **S**etting | 设置合适的场所 |
| **P**erception | 了解患者/家属对病情的认知 |
| **I**nvitation | 获得告知的许可 |
| **K**nowledge | 传达医学信息 |
| **E**motions | 回应情绪 |
| **S**trategy | 讨论下一步 |

### 4.3 临终关怀中的心理支持

| 阶段 | 家属需求 |
|------|---------|
| **疾病确诊** | 震惊、否认、信息需求 |
| **治疗期** | 希望与焦虑并存 |
| **终末期** | 预感性哀伤、疲劳、决策压力 |
| **濒死期** | "最后一程"的陪伴需求 |
| **死亡后** | 反应性哀伤、实际事务 |

### 4.4 预感性哀伤 vs PGD 风险

| 因素 | 说明 |
|------|------|
| **预悲程度高** | 可能预测死亡后 PGD 风险 |
| **预悲但无准备** | 高预悲 + 无实际准备 → 更高 PGD 风险 |
| **功能维持** | 预悲期间功能维持良好 → 预后较好 |

## 五、丧亲后随访（从姑息医学过渡）

| 时间点 | 行动 |
|--------|------|
| **死亡后1-2周** | 慰问卡/电话；实际支持信息 |
| **死亡后1-2月** | 评估功能；识别高危者；转介 |
| **死亡后6月** | 标准化评估（PG-13）；PGD 筛查 |
| **死亡后12月** | PGD 评估；持续支持 |
| **周年纪念** | 主动联系；纪念活动 |
