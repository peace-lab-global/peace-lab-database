# 哀伤中的自杀风险评估 | Suicide Risk Assessment in Grief and Bereavement

> **目标**：建立丧亲者自杀风险的精神科标准评估框架。
> **标准来源**：C-SSRS、APA Practice Guidelines、自杀预防资源中心。

---

## 一、丧亲后自杀风险

### 1.1 流行病学

| 数据 | 说明 |
|------|------|
| **丧偶后** | 自杀风险增加约2倍 |
| **时间窗口** | 丧亲后第一年风险最高 |
| **性别差异** | 老年丧偶男性风险最高 |
| **特殊丧失** | 自杀死亡、儿童死亡的父母风险更高 |

### 1.2 风险因素

| 分类 | 因素 |
|------|------|
| **丧失特征** | 自杀死亡、暴力死亡、儿童死亡、非预期 |
| **个体特征** | 既往自杀未遂、抑郁、物质使用、人格障碍 |
| **关系特征** | 高度依赖关系、冲突关系 |
| **社会因素** | 社会孤立、缺乏支持、经济困难 |
| **保护因素** | 社会支持、宗教信仰、育儿责任、治疗关系 |

## 二、标准化评估

### 2.1 C-SSRS在哀伤中的应用

| 问题 | 注意 |
|------|------|
| "你是否希望死去以便与逝者重聚？" | 丧亲者常见的被动意念，但需区分是否有主动计划 |
| "你是否觉得没有逝者生活不值得过？" | 无意义感是PGD的核心，但需评估严重程度 |
| "你是否想过伤害自己？" | 直接询问 |

### 2.2 丧亲特异性评估

| 评估 | 说明 |
|------|------|
| **"重聚"愿望** | 希望死去以"重聚"vs 主动自杀计划 |
| **无意义感程度** | 一般 vs 完全的绝望 |
| **功能状态** | 是否完全停止功能 |
| **社会连接** | 是否有继续生活的理由（如孩子） |
| **既往应对** | 既往如何应对压力 |

## 三、风险管理

### 3.1 分层管理

| 级别 | 表现 | 行动 |
|------|------|------|
| **低危** | 被动意念；无计划；有保护因素 | 安全计划；定期随访；治疗哀伤 |
| **中危** | 主动意念；模糊计划；部分保护因素 | 安全计划；增加随访；精神科会诊；限制手段 |
| **高危** | 具体计划；手段可获得；保护因素少 | 紧急精神科评估；考虑住院；24小时陪护 |

## 四、特殊人群的额外注意

### 4.1 老年丧偶者

| 风险 | 管理 |
|------|------|
| **老年男性** | 最高风险群体；需主动接触 |
| **健康下降** | 丧偶后健康常快速下降；需医疗关注 |
| **社交孤立** | 鼓励社交；社区资源 |
| **手段限制** | 关注药物囤积；限制 firearms |

### 4.2 自杀遗族

| 风险 | 管理 |
|------|------|
| **模仿风险** | 遗族自杀风险增加 |
| ** guilt** | 处理"我没能阻止"的 guilt |
| **支持团体** | 自杀遗族支持团体可降低风险 |
