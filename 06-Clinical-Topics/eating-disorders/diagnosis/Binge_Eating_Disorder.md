# 暴食障碍 (Binge Eating Disorder, BED)

> **核心定位**：DSM-5 新增正式诊断，以反复暴食+失控感+痛苦为核心，**无规律补偿行为**。最常见的进食障碍，与肥胖高度交叉但非等同。

---

## 1. 诊断标准

### 1.1 DSM-5-TR 诊断标准 (307.51 / F50.8)

| 标准 | 内容 | 操作化 |
|:-----|:-----|:-------|
| **A. 反复暴食** | 在限定时间内进食量明显大于多数人 + **失控感** | 主观失控感是核心 |
| **B. ≥3项特征** | ①进食速度异常快 ②饱腹不适后仍吃 ③无饥饿感时大量进食 ④因尴尬独自进食 ⑤暴食后厌恶/抑郁/内疚 | 至少3/5 |
| **C. 显著痛苦** | 对暴食行为有显著痛苦 | 区分临床暴食与偶发过量进食 |
| **D. 频率/持续** | ≥1次/周 × 3月 | — |
| **E. 无补偿** | **不**伴随规律性不当补偿行为 | 若有→BN |

### 1.2 严重程度分级

| 等级 | 暴食频率/周 |
|:-----|:-----------|
| 轻度 | 1-3 次 |
| 中度 | 4-7 次 |
| 重度 | 8-13 次 |
| 极重度 | ≥ 14 次 |

### 1.3 BED vs 临床暴食（非障碍）vs 肥胖

| 特征 | BED | 临床暴食（偶发） | 肥胖（无 BED） |
|:-----|:----|:----------------|:--------------|
| 失控感 | **核心** | 偶尔 | 通常无 |
| 频率 | ≥1次/周×3月 | 偶发 | — |
| 痛苦 | **显著** | 轻度 | 可变 |
| 秘密进食 | 常见 | 偶发 | 不常见 |
| 暴食后罪恶感 | **强烈** | 轻微 | 可变 |
| BMI | 可变（常超重/肥胖） | 可变 | ≥25/≥30 |

## 2. 流行病学

| 指标 | 数值 |
|:-----|:-----|
| 女性终生患病率 | 3.0-3.5% |
| 男性终生患病率 | 1.5-2.0% |
| 女:男比 | 1.5-2:1（最接近1:1的 ED） |
| 平均起病年龄 | 20-40 岁 |
| 共病肥胖(BMI≥30) | 40-60% |
| 共病抑郁 | 46-57% |
| 共病焦虑 | 37-45% |
| 共病物质使用 | 15-25% |
| 完全恢复率 | 50-60% |

---

## 3. 病因学

### 3.1 BED 特异性风险因素

| 因素 | 说明 | 证据 |
|:-----|:-----|:-----|
| 遗传 | 遗传率 ~50-60% | 双生子研究 |
| 饮食限制 | 严格节食→暴食反弹 | 实验证据 |
| 情绪调节困难 | 暴食作为情绪调节策略 | 中等证据 |
| 冲动性 | 高冲动特质 | 中等证据 |
| 创伤/逆境 | 童年虐待/忽视 | 中等证据 |
| 肥胖污名 | 体重歧视→内化→暴食 | 新兴证据 |
| 减肥手术 | 术后暴食模式改变 | 手术人群研究 |

### 3.2 BED 与多巴胺奖赏

```
食物线索 → 多巴胺释放（伏隔核） → 渴求/冲动
                                        ↓
                              执行控制抑制（前额叶）不足
                                        ↓
                                   暴食行为
                                        ↓
                              罪恶感/厌恶 → 下一次暴食循环
```

## 4. 评估

### 4.1 评估流程

| 步骤 | 工具/方法 | 目的 |
|:-----|:---------|:-----|
| 1. 筛查 | SCOFF / BEDS-7 | 快速识别 |
| 2. 诊断 | EDE 访谈 / EDE-Q | 暴食频率/失控感/痛苦 |
| 3. 肥胖评估 | BMI / 腰围 / 体脂 | 体重状态 |
| 4. 代谢评估 | 血糖 / HbA1c / 血脂 / 血压 | 代谢综合征风险 |
| 5. 共病 | PHQ-9 / GAD-7 / AUDIT | 抑郁/焦虑/物质 |
| 6. 功能损害 | CIA | 心理社会影响 |

### 4.2 BED 特异性评估要点

| 评估维度 | 关键问题 |
|:---------|:---------|
| 暴食模式 | 频率/食物类型/触发情境/独处vs社交 |
| 失控感 | 能否停止？尝试过什么方法？ |
| 补偿行为 | 呕吐/泻药/过度运动/禁食？（排除BN） |
| 饮食历史 | 节食史/减肥尝试/体重波动史 |
| 情绪功能 | 暴食前情绪/暴食后情绪/情绪觉察能力 |
| 身体活动 | 当前活动水平/运动障碍 |
| 进食障碍态度 | 对肥胖的态度/体重内化污名 |

## 5. 治疗方案

### 5.1 NICE 指南推荐

| 治疗线 | 方案 | 疗程 | 目标 |
|:-------|:-----|:-----|:-----|
| **一线** | 引导自助 CBT-BED | 8-12 周 | 减少暴食 |
| **二线** | CBT-BED（团体/个体） | 16-20 次 | 减少暴食 + 心理改善 |
| **三线** | IPT | 16-20 次 | 人际功能改善 |
| **药物** | lisdexamfetamine 30-70mg/d | 持续 | 减少暴食频率 |
| **药物辅助** | 托吡酯 50-100mg/d | — | 减少暴食 + 减重 |
| **不推荐** | 单独饮食建议/单纯减肥方案 | — | 不能治疗暴食 |

### 5.2 CBT-BED 核心内容

| 阶段 | 目标 | 技术 |
|:-----|:-----|:-----|
| 心理教育 | 理解 BED 模型 | 暴食-限制循环/情绪模型 |
| 自我监测 | 觉察暴食触发 | 食物/情绪/情境日记 |
| 规律进食 | 打破限制-暴食循环 | 3餐+2-3次点心 |
| 认知重构 | 挑战全或无饮食规则 | "禁止食物"→"中性食物" |
| 情绪调节 | 替代暴食的情绪策略 | 问题解决/正念/痛苦耐受 |
| 体像/自尊 | 减少体型过度评价 | 功能欣赏 vs 外貌评价 |
| 复发预防 | 维持长期改变 | 高风险情境/应对计划 |

### 5.3 药物治疗

| 药物 | 剂量 | 效果 | 减重效果 | FDA |
|:-----|:-----|:-----|:---------|:---:|
| **lisdexamfetamine** | 30-70mg/d | 减少暴食 40-50% | 轻度（3-5%） | ✅ |
| 托吡酯 | 50-100mg/d | 减少暴食 30-50% | 中度（5-8%） | 超说明书 |
| SSRI | 标准剂量 | 减少暴食 20-30% | 无/轻微 | 超说明书 |
| GLP-1 RA | 标准剂量 | 减少暴食（早期研究） | 显著（10-15%） | 研究阶段 |

> ⚠️ **关键区分**：BED 治疗目标首先是**减少暴食行为**，减重是次要目标。减重方案不能替代 BED 治疗。

### 5.4 BED + 肥胖：联合治疗模型

```
┌──────────────────────────────────┐
│         BED + 肥胖联合干预        │
├──────────────────────────────────┤
│ 1. 先治 BED（CBT-BED 12-16周）  │
│    → 减少暴食行为                 │
│    → 稳定进食模式                 │
│                                   │
│ 2. 再治肥胖（行为减重方案）       │
│    → 膳食调整 + 运动处方          │
│    → 药物辅助（GLP-1 RA 等）     │
│    → 减重手术评估（BMI≥40）      │
│                                   │
│ 3. 长期维持                       │
│    → 暴食复发监测                 │
│    → 体重维持策略                 │
│    → 代谢综合征管理               │
└──────────────────────────────────┘
```

## 6. 鉴别诊断

| 鉴别 | BED vs | 关键区分 |
|:-----|:-------|:---------|
| BN | BN | 有无规律补偿行为 |
| 肥胖（无 BED） | 肥胖 | 暴食+失控感+痛苦 |
| AN 暴食清除型 | AN | BMI<18.5 + 体像关注 |
| 夜间进食综合征 | NES | 夜间觉醒进食 vs 日间暴食 |
| Prader-Willi 综合征 | PWS | 遗传性食欲亢进/智力障碍 |

## 7. 预后

| 结果 | 比率 |
|:-----|:-----|
| CBT-BED 治疗后暴食停止率 | 50-60% |
| 5年完全恢复率 | 50-60% |
| 复发率 | 20-30% |
| 长期体重改善 | 不显著（除非联合减重干预） |
