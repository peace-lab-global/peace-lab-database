# 神经性厌食 (Anorexia Nervosa)

> **核心定位**：精神障碍中致死率最高的疾病之一（SMR 5-10x），以极低体重、强烈恐胖和体型认知扭曲为三联特征。本文覆盖诊断全流程、亚型分类、临床评估与治疗路径。

---

## 1. 诊断标准

### 1.1 DSM-5-TR 诊断标准 (307.1 / F50.01-02)

| 标准 | 内容 | 临床操作化 |
|:-----|:-----|:-----------|
| **A. 限制能量摄入** | 相对需求而言限制能量摄入→显著低体重 | BMI < 18.5（成人）或 < 第5百分位（儿童青少年） |
| **B. 强烈恐胖** | 对增重/变胖有强烈恐惧，或持续行为妨碍增重 | 行为证据（限制/清除/过度运动）比自述更可靠 |
| **C. 体型认知扭曲** | 对体重/体型的感知、思考或体验障碍 | 自我评价过度受体重体型影响，或否认低体重的严重性 |

### 1.2 DSM-5-TR 亚型分类

| 亚型 | DSM-5-TR 代码 | 核心特征 | 占比 | 预后 |
|:-----|:-------------|:---------|:----:|:-----|
| **限制型 (Restricting)** | F50.01 | 过去3月内**无**规律暴食/清除 | ~50% | 可能转为暴食清除型 |
| **暴食清除型 (Binge-Eating/Purging)** | F50.02 | 过去3月内**有**规律暴食和/或清除行为 | ~50% | 更差：更高共病率/自杀率 |

### 1.3 严重程度分级（基于 BMI）

| 等级 | BMI 范围 | 临床意义 | 治疗建议 |
|:-----|:---------|:---------|:---------|
| 轻度 | ≥ 17 | 功能损害可能存在 | 门诊 |
| 中度 | 16 - 16.99 | 明确功能损害 | 门诊/日间 |
| 重度 | 15 - 15.99 | 严重躯体风险 | 专科门诊/住院 |
| 极重度 | < 15 | 生命威胁 | 住院/强制治疗 |

> **注意**：儿童青少年应使用年龄-BMI 百分位数而非绝对 BMI 值。

### 1.4 缓解标准

| 状态 | 定义 | 说明 |
|:-----|:-----|:-----|
| **部分缓解** | 标准A已不满足（体重恢复），但B或C仍存在 | 体重恢复但恐胖未消 |
| **完全缓解** | 所有标准在一段时间内均不满足 | 持续监测以防复发 |

## 2. 流行病学与风险因素

### 2.1 关键数据

| 指标 | 数值 |
|:-----|:-----|
| 女性终生患病率 | 0.9-1.4% |
| 男性终生患病率 | 0.2-0.3% |
| 女:男比 | 3-10:1 |
| 高峰起病年龄 | 15-19 岁（双峰：14岁 + 18岁） |
| 标准化死亡比 (SMR) | 5.1-10.0 |
| 自杀率（死因构成） | 15-25% |
| 平均病程 | 5-7 年 |
| 完全恢复率 | 46-51% |
| 遗传率 | 48-74% |

### 2.2 特定风险因素

| 风险因素 | 相对风险 | 证据强度 |
|:---------|:---------|:---------|
| 女性 | OR 3-10 | 强 |
| 青春期 | 高峰起病窗口 | 强 |
| 家族史（一级亲属 ED） | OR 4-11 | 强 |
| 1型糖尿病 | OR 2.4 | 中 |
| 完美主义特质 | OR 2-4 | 中 |
| 童年焦虑障碍 | OR 2-3 | 中 |
| 体重相关嘲笑/霸凌 | OR 2-5 | 中 |
| 高风险运动/职业 | OR 3-10 | 强（芭蕾/体操/长跑） |
| 社交媒体/瘦理想暴露 | OR 1.5-3 | 中 |

## 3. 临床评估

### 3.1 评估框架

```
┌────────────────────────────────────────────┐
│            AN 综合评估框架                    │
├────────────────────────────────────────────┤
│ 1. 诊断性访谈                               │
│    → 结构化（EDE 17.0D / SCID-5）          │
│    → 进食行为史 / 体像关注 / 恐胖程度       │
│                                              │
│ 2. 体格检查                                 │
│    → BMI / 生命体征 / 皮肤 / 心脏听诊      │
│    → 低体温 / 心动过缓 / 体毛增多(lanugo)  │
│                                              │
│ 3. 实验室检查                               │
│    → 电解质 / 血常规 / 肝肾功能 / 甲功     │
│    → ECG / 骨密度 / 激素水平               │
│                                              │
│ 4. 心理评估                                 │
│    → 共病筛查（焦虑/抑郁/OCD/自杀风险）    │
│    → 动机评估 / 治疗意愿                    │
│    → 家庭功能评估                           │
│                                              │
│ 5. 营养评估                                 │
│    → 当前摄入 vs 需求 / 营养素缺乏         │
│    → 再喂养风险评估                         │
└────────────────────────────────────────────┘
```

### 3.2 体格检查重点

| 系统 | 检查项目 | 常见异常 |
|:-----|:---------|:---------|
| 一般 | BMI/体重变化曲线 | 进行性下降 |
| 生命体征 | 体温/心率/血压 | 低体温(<35°C)/心动过缓(<40bpm)/低血压 |
| 皮肤 | 干燥/毛发/指甲 | Lanugo 体毛/指甲脆裂/皮肤黄染（胡萝卜素血症） |
| 口腔 | 牙齿/唾液腺 | 牙釉质侵蚀/腮腺肿大（呕吐型） |
| 心脏 | 听诊/ECG | 心律失常/QT延长/ST-T改变 |
| 腹部 | 触诊 | 腹胀/便秘/胃排空延迟 |
| 四肢 | 水肿 | 再喂养期水肿/低蛋白水肿 |
| 骨骼 | 骨密度(DXA) | 骨质疏松/骨减少（60-90% AN 患者） |

### 3.3 住院指征（紧急转介标准）

| 指征 | 阈值 | 紧急程度 |
|:-----|:-----|:---------|
| BMI | < 13（成人）或 < 第2百分位 | 紧急 |
| 心率 | < 40 bpm | 紧急 |
| 血压 | < 80/50 mmHg | 紧急 |
| 体温 | < 35°C | 紧急 |
| 血钾 | < 3.0 mmol/L | 紧急 |
| 血磷 | < 0.5 mmol/L | 紧急（再喂养综合征风险） |
| ECG | QTc > 450ms | 紧急 |
| 自杀风险 | 明确计划/意图 | 紧急 |
| 体重下降速度 | > 20% 在3月内 | 亚紧急 |
| 口服摄入 | < 500 kcal/天 持续 | 亚紧急 |

## 4. 治疗方案

### 4.1 治疗原则

| 原则 | 说明 |
|:-----|:-----|
| **体重恢复优先** | 营养康复是其他治疗的前提（"starved brain can't do therapy"） |
| **多学科协作** | 精神科 + 心理 + 营养 + 内科 团队协作 |
| **分阶段治疗** | 先稳定躯体→再心理治疗→最后维持与预防复发 |
| **动机评估** | AN 患者常为自我协调(ego-syntonic)，治疗动机低 |
| **家庭参与** | 青少年 FBT 为一线；成人也应考虑家庭支持 |

### 4.2 治疗方案循证推荐

| 人群 | 一线治疗 | 二线治疗 | 三线/补充 |
|:-----|:---------|:---------|:----------|
| **青少年** | FBT（Maudsley 方法） | CBT-E-Adapted | 药物辅助（奥氮平） |
| **成人** | CBT-E / SSCM / MANTRA | FBT-Adapted / IPT | 药物（奥氮平/SSRI 共病） |
| **重度住院** | 医学稳定 + 结构化再喂养 + 心理支持 | — | 鼻饲/TPN |

### 4.3 再喂养方案

| 阶段 | 时间 | 热量目标 | 监测要点 |
|:-----|:-----|:---------|:---------|
| **启动** | 第1-3天 | 1,200-1,500 kcal/d（门诊）/ 更低（住院） | 磷/钾/镁 每日监测 |
| **递增** | 第4-7天 | 每2天增加 200 kcal | 再喂养综合征征象 |
| **达标** | 第2-4周 | 2,500-4,000 kcal/d（体重恢复期需求高） | 体重增长 0.5-1.0 kg/周 |
| **维持** | 长期 | 个体化维持体重 | 代谢适应监测 |

> ⚠️ **再喂养综合征**是 AN 治疗中最危险的并发症，表现为低磷血症→心律失常→猝死。必须缓慢启动、密切监测。

## 5. 药物治疗

| 药物 | 适应证 | 证据 | 注意事项 |
|:-----|:-------|:-----|:---------|
| **奥氮平** | 体重恢复辅助/严重体像强迫 | 中（多项 RCT） | 低剂量 2.5-10mg，监测代谢 |
| **SSRI（共病）** | 共病焦虑/抑郁/OCD | 弱（对 ED 本身无效） | 体重恢复后效果更佳 |
| **锌补充** | 味觉改变/食欲恢复 | 弱 | 50mg/d 短期 |
| **雌激素** | 骨密度恢复 | 弱（争议） | 体重恢复是骨密度恢复的最佳策略 |
| **避免** | 安非拉酮(bupropion) | **禁忌** | 降低癫痫阈值 |

## 6. 预后与长期随访

### 6.1 预后数据

| 时间窗 | 完全恢复 | 部分恢复 | 慢性/未恢复 | 死亡 |
|:-------|:--------|:--------|:-----------|:-----|
| 5 年 | 30-40% | 30-35% | 20-30% | 5-8% |
| 10 年 | 46-51% | 25-30% | 15-20% | 8-10% |
| 20 年 | ~50% | ~25% | ~15% | 10-15% |

### 6.2 复发预防

| 策略 | 说明 |
|:-----|:-----|
| 维持治疗 | 体重恢复后继续 CBT-E 12-20 次 |
| 体重监测 | 设定最低安全体重线，低于时主动干预 |
| 压力管理 | 识别复发触发因素（压力/体型变化/生活转变） |
| 共病管理 | 持续治疗焦虑/抑郁/OCD |
| 家庭持续参与 | 青少年治疗后家庭功能维持 |
| 定期随访 | 治疗后 6/12/24 月结构化随访 |

## 7. 特殊考量

### 7.1 男性 AN

| 维度 | 特点 |
|:-----|:-----|
| 患病率 | 约 10%（但低估） |
| 体像关注 | 偏向肌肉不足（而非单纯"胖"）|
| 就诊延迟 | 更长（污名化+性别偏见） |
| 共病 | 更高比例强迫症和物质滥用 |

### 7.2 非典型厌食 (Atypical AN — OSFED)

- 满足所有 AN 标准**但体重在正常范围**
- 医学并发症可以同样严重
- 常被忽视，但患病率可能是典型 AN 的 2-3 倍
- 治疗原则与 AN 相同

### 7.3 法律与伦理

| 议题 | 说明 |
|:-----|:-----|
| 强制治疗 | 生命威胁时可依法强制住院/喂养 |
| 治疗意愿 | AN 常为自我协调，需动机访谈 |
| 未成年人 | 家长可代为同意治疗（各国法律不同） |
| 保密与告知 | 未成年人父母知情权 vs 隐私权平衡 |
