# MBCT 特殊人群临床方案 | MBCT for Special Populations

> **目标**：建立精神科标准的 MBCT 特殊人群改编方案，覆盖青少年、老年、围产期、癌症、慢性病及跨文化适应。
> **原则**：标准MBCT协议需根据 developmental、生理、文化因素进行系统性调整，而非简单缩短或简化。

---

## 一、青少年 MBCT（MBCT-A）

### 1.1 发展性考虑

| 维度 | 成人 | 青少年（12-18岁） |
|------|------|------------------|
| **认知发展** | 抽象思维、元认知成熟 | 元认知仍在发展；抽象概念需具体化 |
| **情绪调节** | 相对成熟的调节策略库 | 情绪强烈且波动；前额叶发育未完善 |
| **注意力** | 可维持40-45分钟专注 | 注意力维持15-20分钟；需更多变化 |
| **同伴影响** | 相对稳定 | 极强；同伴关系为核心 |
| **自主性** | 自主决策 | 仍在争取自主；家长参与复杂 |
| **躯体感受** | 相对稳定 | 青春期身体变化带来强烈躯体觉察 |

### 1.2 MBCT-A 核心改编

| 标准MBCT | MBCT-A改编 | 原理 |
|---------|-----------|------|
| **8周 × 2.5小时** | 8-12周 × 1.5小时 | 注意力限制；学校日程 |
| **45分钟身体扫描** | 15-20分钟身体扫描 | 注意力限制；增加趣味性 |
| **静默静修** | 取消或改为半日活动 | 青少年难以忍受长时间静默；增加互动元素 |
| **团体分享** | 增加配对/小组讨论；使用白板/便利贴 | 同伴互动；减少个体暴露压力 |
| **家庭作业** | 每日15分钟（非45分钟）；使用APP/音频 | 现实可行；技术整合 |
| **认知内容** | 减少"去中心化"抽象概念；增加"想法如泡泡"等隐喻 | 发展适宜性 |
| **家长参与** | 第1周和第8周家长会谈；提供家长教育手册 | 家长支持增强青少年依从性；家长自身压力管理 |
| **主题调整** | 增加社交焦虑、学业压力、网络使用、自我形象 | 发展相关议题 |

### 1.3 评估工具

| 量表 | 用途 |
|------|------|
| MFQ（心境与感受问卷）| 青少年抑郁筛查 |
| RCADS（儿童焦虑抑郁量表修订版）| 青少年焦虑抑郁 |
| CAMM（儿童青少年正念量表）| 青少年正念能力 |
| SMFQ（简版MFQ）| 快速抑郁追踪 |

### 1.4 安全考虑

| 风险 | 管理 |
|------|------|
| **NSSI（非自杀性自伤）** | 入组筛查；安全计划；家长知情；教师培训识别NSSI |
| **同伴模仿** | 团体中避免详细分享自伤方法；强调保密边界 |
| **家长-青少年冲突** | 家长会谈中单独时间与联合时间平衡 |
| **学校转介** | 明确学校知情范围；保护青少年隐私 |

## 二、老年 MBCT（MBCT-L / MBCT for Older Adults）

### 2.1 老年特异性考虑

| 维度 | 考虑 |
|------|------|
| **认知变化** | 处理速度下降；工作记忆减退；可能需要更多重复和复习 |
| **感觉变化** | 视力/听力下降；需更大字体、更清晰的音频、充足光线 |
| **躯体限制** | 关节炎、 balance 问题；身体扫描和瑜伽需大幅调整 |
| **丧失与哀伤** | 丧偶、退休、健康问题；哀伤主题可能自然涌现 |
| **孤独** | 社会网络缩小；团体本身具有治疗价值 |
| **多重用药** | 平均4-6种药物；需了解药物对练习的影响 |
| **污名** | 对"精神疾病"的代际态度；可能抗拒标签 |

### 2.2 MBCT-L 改编

| 标准MBCT | MBCT-L改编 |
|---------|-----------|
| **身体扫描** | 椅子版身体扫描；允许触摸身体部位辅助觉察；缩短至20-30分钟 |
| **正念瑜伽** | 椅子瑜伽；温和伸展；强调安全，避免跌倒风险 |
| **步行冥想** | 重点推荐；适合躯体限制者；可扶着墙或椅子 |
| **认知内容** | 强调"老年不是衰退而是新阶段"；增加生命回顾元素 |
| **主题** | 增加退休适应、健康焦虑、丧亲、代际关系、生命意义 |
| **节奏** | 更慢；更多重复；更短的分段；更多休息时间 |
| **团体规模** | 6-8人（较小）；允许家属旁听或参与部分 |
| **材料** | 大字体手册；高音质音频；高对比度PPT |

### 2.3 认知障碍筛查

| 工具 | 临界值 | 行动 |
|------|--------|------|
| **MMSE** | <24分需进一步评估 | 神经科评估；考虑简化方案 |
| **MoCA** | <22分提示轻度认知障碍 | 神经科评估；调整期望 |
| **GDS-15（老年抑郁量表）** | ≥5分提示抑郁 | 区分抑郁 vs 痴呆；两者可共病 |

## 三、围产期 MBCT（MBCT-Perinatal）

### 3.1 围产期抑郁特点

- 患病率：产前10-15%，产后10-20%
- 特殊性：对药物顾虑（妊娠/哺乳）；睡眠剥夺；角色转变；躯体变化
- 复发风险：既往抑郁史者围产期复发风险30-50%

### 3.2 MBCT-Perinatal 改编

| 标准MBCT | MBCT-Perinatal改编 |
|---------|-------------------|
| **身体扫描** | 包含胎儿/婴儿觉察（如适用）；允许中断（哺乳/照顾婴儿） |
| **主题** | 增加母亲角色转变、身体变化接纳、完美母亲神话、伴侣关系 |
| **团体构成** | 同产期母亲团体（产前/产后分开或整合） |
| **时间安排** | 考虑哺乳时间；允许带婴儿（或提供托婴） |
| **家庭作业** | 与婴儿一起的正念（正念抱持、正念哺乳、正念换尿布） |
| **安全** | 筛查产后精神病（罕见但紧急）；评估母婴安全风险 |
| **药物讨论** | 提供哺乳安全药物信息；减轻药物污名 |

### 3.3 安全筛查

| 风险 | 筛查工具 | 行动 |
|------|---------|------|
| **产后精神病** | 临床访谈；精神状态检查 | 紧急精神科评估；住院指征 |
| **伤害婴儿意念** | 直接但非评判地询问 | 区分"强迫性伤害意念"（常见，不执行）vs "伤害意图"（危险） |
| **母婴依恋问题** | 观察互动；PBQ（产后联结问卷）| 母婴治疗转介 |

## 四、癌症与慢性病 MBCT（MBCT-Life / MBCT-Chronic Illness）

### 4.1 癌症相关应用

| 适应症 | 证据等级 | 调整 |
|--------|---------|------|
| 癌症相关抑郁 | B级 | 标准MBCT适用；增加哀伤主题 |
| 癌症相关焦虑 | B级 | 增加不确定性耐受 |
| 化疗脑雾 | C级 | 缩短练习；增加外部锚定 |
| 癌症复发恐惧 | B级 | 针对"复发焦虑"的认知解离；增加价值导向生活 |

**特殊考虑**：
- 治疗副作用（疲劳、恶心、疼痛）可能限制练习
- 预后不确定性的存在主义议题
- 身体完整性丧失（乳房切除、造口等）
- 死亡焦虑可能在正念中涌现

### 4.2 慢性病通用改编

| 挑战 | 调整 |
|------|------|
| **症状波动** | 允许根据当日状态选择练习类型和时长 |
| **医疗负担** | 正念作为"自我管理"而非额外负担 |
| **疾病认同** | 区分"我有糖尿病"和"我是糖尿病患者"（去中心化应用） |
| **医患关系** | 正念不替代医学治疗；强调互补性 |

## 五、跨文化适应（中国文化情境）

### 5.1 文化特异性议题

| 议题 | 西方MBCT | 中国文化适应 |
|------|---------|-------------|
| **佛教关联** | 部分患者担心"宗教化" | 强调"注意力训练"；说明科学基础；尊重信仰 |
| **情绪表达** | 鼓励直接表达情绪 | 允许间接表达；重视非言语；提供书面选项 |
| **家庭系统** | 聚焦个体 | 承认家庭对决策的影响；必要时家庭会谈 |
| **面子/羞耻** | 个人成长框架 | 强调"养生""自我调节"；减少病理标签 |
| **躯体化** | 将心理痛苦躯体化视为需要矫正 | 接纳躯体表达；从身体觉察切入 |
| **权威关系** | 平等协作 | 尊重治疗师权威；同时逐步引导自主性 |
| **集体主义** | 个人价值澄清 | 整合家庭福祉；价值澄清包含家庭责任 |

### 5.2 中国文化元素整合

| 正念元素 | 中国文化对应 | 整合方式 |
|---------|------------|---------|
| **呼吸觉察** | 气功、太极的"调息" | 引用熟悉概念作为桥梁 |
| **身体扫描** | 中医"内观"、经络觉察 | 允许从中医身体图式切入 |
| **接纳** | 道家"无为"、"顺其自然" | 引用经典增强文化共鸣 |
| **去中心化** | 佛家"不执著" | 对佛教背景患者可用；世俗患者用"心理距离" |
| **自我慈悲** | 儒家"仁爱"扩展至自身 | 从"善待他人"延伸至"善待自己" |
| **静默** | 茶道、书法中的"静" | 引用传统修习增强接受度 |

## 六、性少数群体（LGBTQ+）适应

| 议题 | 调整 |
|------|------|
| **少数群体压力** | 增加歧视、身份隐瞒、出团压力的正念应对 |
| **内部ized homophobia** | 自我慈悲针对内部ized羞耻；去中心化针对内化负面信念 |
| **团体安全** | 确保团体包容性；不假设异性恋；使用包容性语言 |
| **家庭排斥** | 增加"选择家庭"支持网络建设；哀伤工作 |
| **性别不安** | 身体扫描可能触发性别不安；允许调整语言（如不用"身体部位名称"而用"这个区域"） |
