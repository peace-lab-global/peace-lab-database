# MBCT 治疗监测与预后评估 | Treatment Monitoring and Prognosis in MBCT

> **目标**：建立精神科标准的 MBCT 治疗反应定义、监测流程、无应答者识别、脱落预防、复发预警与长期维持方案。

---

## 一、治疗反应定义

### 1.1 临床治愈（Remission）

| 标准 | 定义 | 测量 |
|------|------|------|
| **症状学治愈** | PHQ-9 < 5；或 HAMD-17 < 7 | 量表 |
| **功能恢复** | SDS ≤ 6（功能无损害或极轻度） | 功能量表 |
| **持续时间** | 症状和功能标准维持 ≥ 2个月 | 纵向追踪 |

### 1.2 有效（Response）

| 标准 | 定义 |
|------|------|
| **症状改善** | PHQ-9 下降 ≥ 50%；或 CGI-I ≤ 2（明显改善） |
| **机制改善** | EQ（去中心化）提升 ≥ 1个标准差 |
| **功能改善** | SDS 下降 ≥ 25% |

### 1.3 无应答（Non-Response）

| 标准 | 定义 |
|------|------|
| **症状无改善** | PHQ-9 下降 < 25% |
| **症状恶化** | PHQ-9 上升 ≥ 5分 |
| **机制无变化** | EQ/FFMQ 无显著变化 |
| **功能无改善** | SDS 无变化或恶化 |

### 1.4 复发（Relapse）与再发（Recurrence）

| 术语 | 定义 | MBCT语境 |
|------|------|---------|
| **复发（Relapse）** | 缓解期早期（<6个月）症状再现 | 区分：是"正常波动"还是"需要干预的恶化" |
| **再发（Recurrence）** | 缓解期后期（>6个月）新发作 | MBCT核心预防目标 |
| **假性复发** | 短暂症状波动（<2周），功能未明显受损 | 正念技能可处理；无需升级治疗 |
| **真性复发** | 症状持续 > 2周，功能受损，符合诊断标准 | 需要干预 |

## 二、治疗监测流程

### 2.1 监测时间节点

| 时间点 | 监测内容 | 决策点 |
|--------|---------|--------|
| **入组前（T0）** | 全面评估（症状、功能、机制、风险） | 确定入组合适性；建立基线 |
| **第2周（T1）** | PHQ-9；练习质量；安全扫描 | 早期适应；是否需要调整 |
| **第4周（T2）** | PHQ-9, GAD-7, FFMQ, EQ；临床访谈 | **中期评估**：识别早期无应答者 |
| **第6周（T3）** | PHQ-9；安全扫描；练习参与度 | 后半程支持；复发预防计划启动 |
| **第8周（T4）** | 全面评估（T0全部量表）；CGI-I | 终点疗效；维持计划制定 |
| **3个月（T5）** | PHQ-9, EQ, SCS, RRS；功能评估 | 维持期监测 |
| **6个月（T6）** | PHQ-9, BDI-II, GAD-7, FFMQ | 中期预后 |
| **12个月（T7）** | PHQ-9, C-SSRS, SDS, 复发事件 | 长期预后 |

### 2.2 监测数据可视化

| 指标 | 可视化方式 | 临床用途 |
|------|-----------|---------|
| **PHQ-9 时间曲线** | 折线图：T0-T8-T5-T6-T7 | 直观显示症状轨迹；识别恶化时点 |
| **机制-症状散点图** | X=EQ变化，Y=PHQ-9变化 | 验证去中心化是否是症状改善的机制 |
| **练习-症状关联图** | X=练习频率/质量，Y=症状改善 | 识别"剂量-反应"关系 |
| **风险热力图** | 多维度风险评分（自杀、解离、脱落） | 一目了然的风险监控 |

## 三、无应答者早期识别与处理

### 3.1 早期预警信号（第2-4周）

| 信号 | 可能原因 | 干预 |
|------|---------|------|
| **PHQ-9无变化或上升** | 方案不匹配；共病未识别；参与度低；生物学因素 | 临床访谈；评估共病；检查练习质量；考虑药物 |
| **EQ无变化** | 练习方式错误（如将正念变为新形式的反刍）；认知能力限制 | 个别指导；调整练习类型；简化指导语 |
| **频繁缺课** | 动机问题；交通/时间；团体不适；症状恶化 | 电话跟进；个体会谈；调整时间/形式 |
| **练习日志空白/敷衍** | 抵触；不理解；完美主义导致回避 | 心理教育；降低期望；探索阻力 |
| **团体中表达强烈不满** | 期望不匹配；被触发；对治疗师有负面反应 | 个体会谈；处理移情；调整期望 |

### 3.2 无应答者处理路径

```
第4周评估：PHQ-9下降 < 25%
    │
    ├── 练习质量低（<50%完成率）
    │       → 动机访谈 → 识别障碍 → 调整期望 → 继续
    │       → 若仍无改善：考虑个体MBCT或转介
    │
    ├── 练习质量高但症状无改善
    │       → 重新评估诊断（是否遗漏双相/PTSD/人格障碍？）
    │       → 考虑药物加强
    │       → 考虑更高强度治疗（CBT/DBT/IPT）
    │
    ├── EQ无变化 + 症状无改善
    │       → 个别正念指导（可能练习方式错误）
    │       → 若仍无变化：可能不适合正念取向
    │
    └── 症状恶化（PHQ-9上升 ≥ 5分）
            → 立即精神科评估
            → 考虑药物介入
            → 暂停或退出团体（保护患者和其他成员）
```

### 3.3 换药/升级决策

| 情况 | 升级方案 | 时限 |
|------|---------|------|
| MBCT 4周无应答 + 中度症状 | 联合SSRI/SNRI | 第4-6周决定 |
| MBCT 8周部分应答 + 残留症状 | 维持MBCT维持练习 + 药物优化 | 终点评估后 |
| MBCT 8周无应答 + 严重症状 | 转精神科；考虑ECT/TMS/强化CBT | 终点评估后 |
| MBCT有效但12月内复发 | 复训 + 药物维持 + 长期维持团体 | 复发时 |

## 四、脱落预防（Dropout Prevention）

### 4.1 脱落率与风险因素

| 风险因素 | 脱落率影响 | 预防策略 |
|---------|-----------|---------|
| **症状严重度高** | 高 | 入组前严格筛选；药物辅助；缩短练习 |
| **共病人格障碍** | 高 | 个体化方案；更频繁接触；DBT整合 |
| **交通/时间障碍** | 高 | 在线MBCT选项；灵活时间 |
| **团体不适** | 中高 | 允许少分享；更小团体；个体选项 |
| **对正念期望过高** | 中 | 入组前现实期望设置；心理教育 |
| **男性** | 中 | 去性别化语言；强调技能而非"感受" |
| **低教育水平** | 中 | 简化语言；更多视觉辅助；重复 |
| **缺乏社会支持** | 中 | 团体归属感建设；建立课程外联系 |

### 4.2 脱落预警系统

| 预警信号 | 行动 | 责任人 |
|---------|------|--------|
| 第1次缺课 | 24小时内电话/短信关怀 | MBCT教师 |
| 连续2次缺课 | 电话访谈：了解障碍；提供支持；邀请返回 | MBCT教师 |
| 连续3次缺课 | 个体会谈（面询/视频）；评估是否继续/调整/转介 | MBCT教师 + 临床心理师 |
| 表达退出意愿 | 动机访谈；探索原因；提供替代方案（个体/在线/延期） | MBCT教师 |

## 五、复发预警与早期干预

### 5.1 复发预警模型

**基于MBCT的复发预警信号**：

| 预警级别 | 信号 | 行动 |
|---------|------|------|
| **🟢 绿色（正常波动）** | PHQ-9短暂上升1-3分，<1周；能使用呼吸空间应对 | 自我监控；维持日常练习 |
| **🟡 黄色（早期预警）** | PHQ-9上升3-5分，持续1-2周；反刍增加；睡眠变差；EQ下降 | 增加练习频率；使用复发预防计划；联系MBCT教师；考虑精神科复查 |
| **🟠 橙色（高风险）** | PHQ-9上升 > 5分，持续 > 2周；功能受损；出现自杀意念（被动）；无法使用正念技能 | 紧急精神科评估；药物评估；考虑恢复/加强药物；密集支持 |
| **🔴 红色（复发/危机）** | 符合重性抑郁发作标准；主动自杀意念；功能严重受损 | 危机干预；精神科急诊评估；住院/日间治疗考虑 |

## 六、长期维持方案

### 6.1 维持期阶梯模型

| 阶段 | 时间 | 练习要求 | 支持 | 监测 |
|------|------|---------|------|------|
| **密集维持** | 课程结束后1-3月 | 每日30-45分钟；每周1次团体 | 毕业生团体 | 每月PHQ-9 |
| **标准维持** | 3-12月 | 每日20分钟；每月1-2次团体 | 毕业生团体/在线支持 | 每2月PHQ-9 |
| **自主维持** | 1-2年 | 每日10-20分钟；按需团体 | 自助资源/APP | 每3-6月PHQ-9 |
| **复训触发** | 任何时候预警 | 返回8周课程或密集复训 | 团体 | 密集监测 |

### 6.2 复训（Booster）方案

| 复训类型 | 内容 | 适用 |
|---------|------|------|
| **全天复训** | 1天静默静修 + 核心练习回顾 | 功能良好，希望深化 |
| **周末复训** | 2天核心模块密集回顾 | 有轻微预警信号 |
| **4周复训** | 每周1次，聚焦复发预防 | 有明确预警信号 |
| **完整复训** | 重新参加8周课程 | 复发后或长期中断后 |

## 七、预后因素

### 7.1 良好预后因素

| 因素 | 机制 |
|------|------|
| **复发次数 ≥ 3次** | MBCT对多次发作者效果最强（PREVENT研究） |
| **高练习依从性** | 每日练习 > 20分钟 × ≥ 5天/周 |
| **EQ显著提升** | 去中心化能力建立是复发的保护因素 |
| **良好的社会支持** | 缓冲压力；提供情感资源 |
| **低认知反应性** | 悲伤情绪不易触发消极思维 cascade |
| **无严重共病** | 单一复发性抑郁预后最佳 |

### 7.2 不良预后因素

| 因素 | 风险 |
|------|------|
| **未识别的双相障碍** | 可能诱发躁狂；复发模式不同 |
| **严重人格障碍** | 团体困难；情绪不稳定；高脱落率 |
| **未处理创伤** | 练习中触发；解离风险 |
| **低练习依从性** | 剂量不足；技能未内化 |
| **持续高压环境** | 环境压力超出个人应对资源 |
| **物质使用** | 干扰练习；情绪调节代偿 |
