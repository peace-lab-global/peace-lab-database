# MBCT 精神科标准评估组合 | Psychiatric Assessment Battery for MBCT

> **目标**：建立符合精神科临床标准的 MBCT 入组、过程监测与疗效评估体系。所有量表均为中文验证版本或国际通用版本。
> **适用对象**：精神科医生、临床心理师、注册心理治疗师、MBCT 认证教师。

---

## 一、入组前精神科评估流程（Pre-Treatment Evaluation）

### 1.1 结构化临床访谈

| 访谈模块 | 工具/内容 | 目的 | 执行者 |
|---------|----------|------|--------|
| **轴 I 障碍筛查** | SCID-5-CV（临床版）或 MINI 7.0 | 确认重性抑郁障碍（MDD）诊断及复发次数；排除双相、精神病性障碍、物质使用障碍 | 精神科医生/临床心理师 |
| **自杀风险评估** | C-SSRS（哥伦比亚自杀严重程度量表）+ 临床访谈 | 量化自杀意念、计划、企图；决定是否适合团体 | 精神科医生 |
| **精神病性症状筛查** | PANSS 阳性子量表 或 临床访谈 | 排除活跃期精神病性症状 | 精神科医生 |
| **轻躁狂/躁狂史** | MDQ（心境障碍问卷）+ 临床访谈 | 排除未识别的双相谱系障碍 | 临床心理师 |
| **人格障碍筛查** | SCID-5-PD 或 PID-5 | 识别可能影响治疗的潜在人格病理 | 临床心理师 |
| **创伤史** | PCL-5 或 临床创伤访谈 | 评估未处理创伤对正念练习的潜在影响 | 临床心理师 |
| **躯体疾病** | 病史采集 + 实验室检查建议 | 排除甲状腺功能障碍、贫血、神经系统疾病等 | 精神科医生 |
| **社会功能** | 工作/学业/人际关系功能评估 | 建立基线功能水平 | 临床心理师 |

### 1.2 入组排除标准（Exclusion Criteria）

| 类别 | 具体标准 | 处理建议 |
|------|---------|---------|
| **绝对排除** | 当前活跃重性抑郁发作（HAMD-17 ≥ 24 或 PHQ-9 ≥ 20） | 优先药物治疗；待症状缓解至轻度后评估入组 |
| **绝对排除** | 当前活跃精神病性症状（幻觉、妄想、严重思维混乱） | 转精神科专科治疗；正念为禁忌 |
| **绝对排除** | 当前躁狂或混合发作 | 转精神科；正念可能加剧精神运动性兴奋 |
| **绝对排除** | 主动自杀计划或近期自杀企图（过去3个月） | 立即启动危机干预；住院或日间治疗 |
| **绝对排除** | 严重物质依赖（活跃期，未接受戒断治疗） | 先处理物质问题；稳定后评估 |
| **绝对排除** | 严重认知功能障碍（MMSE < 24 或 MoCA < 22） | 无法跟随团体课程；考虑个体化方案 |
| **相对排除** | 未处理的复杂创伤（C-PTSD，闪回频繁） | 先进行创伤稳定化治疗；后可考虑改良 MBCT |
| **相对排除** | 边缘型人格障碍（BPD，活跃期自伤/自杀行为） | 需 DBT 优先或 DBT-MBCT 整合方案 |
| **相对排除** | 严重躯体疾病限制身体活动 | 调整身体扫描/瑜伽；个体化方案 |
| **相对排除** | 语言障碍（无法理解课程语言） | 提供翻译或母语课程 |

## 二、核心评估量表组合（Core Assessment Battery）

### 2.1 症状严重程度量表

#### PHQ-9（患者健康问卷-9项）

| 项目 | 内容 |
|------|------|
| **用途** | 抑郁症状筛查与追踪 |
| **题数** | 9 |
| **评分** | 0-3四级评分，总分0-27 |
| **临界值** | 5-9轻度；10-14中度；15-19中重度；20-27重度 |
| **MBCT使用** | 入组、每2周、第8周、3月、6月、12月追踪 |
| **特别注意** | 第9题（自杀意念）≥1分需立即临床评估 |

**临床意义分级（MBCT专用）**：
- **适合入组**：PHQ-9 ≤ 14（缓解期或轻度症状）
- **谨慎入组**：PHQ-9 15-19（需精神科评估，可能需药物辅助）
- **暂缓入组**：PHQ-9 ≥ 20（急性期，先稳定化）

#### BDI-II（贝克抑郁量表-II）

| 项目 | 内容 |
|------|------|
| **用途** | 抑郁严重程度深度评估 |
| **题数** | 21 |
| **评分** | 0-3四级评分，总分0-63 |
| **临界值** | 0-13极轻；14-19轻度；20-28中度；29-63重度 |
| **MBCT使用** | 基线、第8周、6个月 |
| **优势** | 对认知症状敏感，适合评估反刍和消极自我概念 |

#### MADRS（蒙哥马利-艾斯伯格抑郁评定量表）

| 项目 | 内容 |
|------|------|
| **用途** | 他评量表，用于精神科临床和研究 |
| **题数** | 10 |
| **评分** | 0-6六级评分，总分0-60 |
| **临界值** | 0-6正常；7-19轻度；20-34中度；35-60重度 |
| **MBCT使用** | 精神科门诊患者基线和终点评估 |
| **优势** | 对变化敏感，适合疗效研究 |

#### GAD-7（广泛性焦虑量表-7项）

| 项目 | 内容 |
|------|------|
| **用途** | 焦虑症状筛查 |
| **题数** | 7 |
| **评分** | 0-3四级评分，总分0-21 |
| **临界值** | 5-9轻度；10-14中度；15-21重度 |
| **MBCT使用** | 基线、第8周、3个月 |

### 2.2 正念特异性量表

#### FFMQ（五因素正念度量表）

| 维度 | 定义 | MBCT靶点 |
|------|------|---------|
| 观察 | 觉察内外经验 | 基础能力 |
| 描述 | 文字标记经验 | 认知标签化 |
| 不反应 | 允许经验存在不过度反应 | **核心靶点** |
| 不评判 | 不对经验评价 | **核心靶点** |
| 有觉察的行动 | 全神贯注当下 | 对抗自动导航 |

**MBCT预期变化**：不反应、不评判、有觉察的行动 三维度提升最显著。

#### EQ（去中心化量表 / Experiences Questionnaire）

| 项目 | 内容 |
|------|------|
| **开发者** | Segal et al. (2002) |
| **核心测量** | 去中心化（Decentering）能力 |
| **临床价值** | EQ变化预测抑郁复发风险降低；是MBCT的**机制验证指标** |
| **关键条目** | "我能够觉察到我的想法只是想法，而非事实" |
| **MBCT使用** | 基线、第4周、第8周、3个月 |

#### SCS（自我慈悲量表）

| 维度 | 正向 | 反向（抑郁常见高分） |
|------|------|-------------------|
| 自我友善 vs 自我评判 | 善待自己 | 严厉自我批评 |
| 共通人性 vs 孤立感 | 理解痛苦是普遍的 | 感到与他人隔绝 |
| 正念 vs 过度认同 | 平衡觉察 | 过度认同负面情绪 |

#### RRS（反刍反应量表）

| 子维度 | 内容 | MBCT靶点 |
|--------|------|---------|
| Brooding（冥思） | 被动、重复思考情绪原因 | **核心靶点** |
| Reflection（反思） | 中性、问题解决导向 | 可保留 |
| 抑郁反刍 | 与症状直接相关 | 需针对性打断 |

### 2.3 自杀风险评估

#### C-SSRS（哥伦比亚自杀严重程度评定量表）

| 评估域 | 内容 | 临床行动阈值 |
|--------|------|-------------|
| 自杀意念 |  wish to be dead, non-specific active suicidal thoughts | 任何肯定回答需进一步评估 |
| 自杀意念（有方法，无计划） | 有具体方法但无实施计划 | 中高风险；安全计划必需 |
| 自杀意念（有具体计划） | 有明确计划和意图 | 高风险；立即精神科评估 |
| 自杀行为 | 实际未遂行为 | 极高风险；危机干预 |
| 自残行为（无自杀意图） | NSSI | 评估功能；安全计划 |

**MBCT入组标准**：C-SSRS 自杀意念部分全部为"否"，且过去3个月无自杀企图。

### 2.4 认知功能与功能评估

| 量表 | 用途 | MBCT应用场景 |
|------|------|-------------|
| **MMSE** | 认知功能筛查 | 老年患者入组前 |
| **MoCA** | 轻度认知障碍筛查 | 老年患者、抑郁伴认知症状 |
| **SDS（Sheehan残疾量表）** | 功能损害评估 | 工作、社交、家庭功能 |
| **WHODAS 2.0** | WHO残疾评定 | 全面功能评估 |
| **Cognitive Reactivity Scale** | 认知反应性 | MBCT特异性：悲伤情绪触发消极思维的程度 |

## 三、MBCT 整合评估时间表

| 时间点 | 症状量表 | 正念量表 | 自杀风险 | 功能评估 | 特殊评估 |
|--------|---------|---------|---------|---------|---------|
| **入组前（T0）** | PHQ-9, BDI-II, GAD-7 | FFMQ, EQ, SCS, RRS | C-SSRS | SDS | SCID, MDQ, MMSE(老年) |
| **第2周末（T1）** | PHQ-9 | — | C-SSRS(简短) | — | 练习质量日志 |
| **第4周末（T2）** | PHQ-9, GAD-7 | FFMQ, EQ | C-SSRS(简短) | — | 中期临床访谈 |
| **第8周末（T3）** | PHQ-9, BDI-II, GAD-7 | FFMQ, EQ, SCS, RRS | C-SSRS | SDS | 终点临床访谈 |
| **3个月（T4）** | PHQ-9, GAD-7 | EQ, SCS, RRS | C-SSRS | SDS | 维持期访谈 |
| **6个月（T5）** | PHQ-9, BDI-II | FFMQ, EQ | C-SSRS | SDS | 长期随访 |
| **12个月（T6）** | PHQ-9 | — | C-SSRS | SDS | 年度复发评估 |

## 四、评估结果的临床决策树

```
基线评估完成
    │
    ├── PHQ-9 ≥ 20 或 C-SSRS 阳性
    │       → 暂缓入组 → 精神科评估 → 药物/住院治疗 → 缓解后重新评估
    │
    ├── PHQ-9 15-19 或 C-SSRS 边缘
    │       → 谨慎入组 → 精神科联合随访 → 缩短练习时长 → 加强安全计划
    │
    ├── PHQ-9 ≤ 14 + C-SSRS 阴性 + 复发史 ≥ 2次
    │       → 标准MBCT入组 → 完整8周课程
    │
    └── PHQ-9 ≤ 9 + 无复发史
            → 预防性MBCT或MBSR → 教育性质团体
```

## 五、评估文档记录标准

| 文档类型 | 必需内容 | 保存要求 |
|---------|---------|---------|
| **入组评估报告** | 诊断、量表分数、排除项核查、风险分层、知情同意 | 加密电子病历，最少7年 |
| **过程监测记录** | 每次PHQ-9、C-SSRS结果、临床观察、练习参与度 | 同步更新 |
| **终点评估报告** | 量表变化分数、临床总体印象（CGI-I）、复发事件 | 归档 |
| **不良事件记录** | 日期、事件描述、严重程度、处理措施、转归 | 强制性报告 |
| **督导记录** | 督导日期、议题、建议、跟进 | 与个案记录一并保存 |
