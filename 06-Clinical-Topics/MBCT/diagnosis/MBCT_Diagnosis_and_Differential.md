# MBCT 诊断与鉴别诊断 | Diagnosis and Differential for MBCT

> **目标**：建立 MBCT 适用人群的精神科诊断标准、鉴别诊断流程与共病评估框架，确保治疗精准性与安全性。
> **诊断标准来源**：DSM-5-TR（2022）、ICD-11（2022）、中国精神障碍分类与诊断标准（CCMD-3 参考）。

---

## 一、MBCT 核心适应症诊断标准

### 1.1 复发性重性抑郁障碍（Recurrent Major Depressive Disorder）— DSM-5-TR

**诊断标准 A**：
> 在同一2周时期内，出现5个（或更多）下列症状，表现出与先前功能相比的变化；其中至少1项是（1）心境抑郁或（2）丧失兴趣或愉悦感。

| 症状 | 说明 |
|------|------|
| 1. 心境抑郁 | 主观报告（如感到悲伤、空虚、无望）或他人观察（如流泪） |
| 2. 兴趣/愉悦感丧失 | 对几乎所有活动兴趣或愉悦感明显减少 |
| 3. 体重显著变化 | 非节食情况下体重减轻/增加 > 5%，或食欲改变 |
| 4. 失眠或嗜睡 | 几乎每天 |
| 5. 精神运动性激越或迟滞 | 他人可观察到的 |
| 6. 疲劳或精力丧失 | 几乎每天 |
| 7. 无价值感或过度内疚 | 可能达到妄想程度 |
| 8. 注意力下降或犹豫不决 | 几乎每天 |
| 9. 反复出现死亡/自杀意念 | 有或没有具体计划 |

**诊断标准 B**：症状引起有临床意义的痛苦，或社交、职业或其他重要功能方面的损害。

**诊断标准 C**：发作不能归因于物质或其他躯体疾病。

**复发性定义**：至少2次重性抑郁发作，两次发作之间至少2个月期间症状不符合重性抑郁障碍的全部标准。

**MBCT适用标准**：
- ✅ 复发性抑郁（≥2次发作， preferably ≥3次）
- ✅ 当前处于缓解期（PHQ-9 ≤ 14，且不符合重性抑郁发作全部标准）
- ❌ 当前活跃重性抑郁发作（PHQ-9 ≥ 20 或 HAMD ≥ 24）

### 1.2 ICD-11 对应编码

| ICD-11 编码 | 诊断名称 | MBCT适用性 |
|------------|---------|-----------|
| 6A70.1 | 复发性抑郁发作，目前为缓解期 | ✅ 核心适应症 |
| 6A70.2 | 复发性抑郁发作，目前为轻度 | ⚠️ 可个体化评估 |
| 6A70.3 | 复发性抑郁发作，目前为中度 | ❌ 需先稳定化 |
| 6A70.4/5 | 复发性抑郁发作，目前为重度/伴精神病性症状 | ❌ 禁忌 |
| 6A71 | 恶劣心境（持续性抑郁障碍） | ⚠️ 可改编方案 |

## 二、关键鉴别诊断

### 2.1 与双相情感障碍的鉴别（Critical Differential）

**鉴别重要性**：双相障碍患者若接受纯MBCT而无心境稳定剂，可能诱发躁狂/轻躁狂发作。

| 特征 | 复发性单相抑郁 | 双相障碍（II型） |
|------|-------------|----------------|
| **发作性质** | 仅抑郁发作 | 抑郁 + 轻躁狂/躁狂发作 |
| **发作突然性** | 通常逐渐起病 | 可能突然起病 |
| **睡眠需求** | 失眠或嗜睡 | 轻躁狂期睡眠需求减少（不感疲倦） |
| **精神病性症状** | 仅严重抑郁期可能出现 | 躁狂期常见 |
| **家族史** | 抑郁障碍家族史 | 双相障碍家族史 |
| **发病年龄** | 通常较晚（25-35岁首发） | 通常较早（<25岁） |
| **抗抑郁药反应** | 通常有效 | 可能诱发轻躁狂/快速循环 |
| **MDQ筛查** | 阴性 | 阳性（7项中≥7项，且功能损害） |

**MBCT入组前必需筛查**：
1. 详细询问既往是否有轻躁狂/躁狂发作史
2. MDQ（心境障碍问卷）筛查
3. 如阳性或不确定 → 转介精神科确诊

### 2.2 与持续性抑郁障碍（恶劣心境）的鉴别

| 特征 | 复发性重性抑郁 | 持续性抑郁障碍（恶劣心境） |
|------|-------------|----------------------|
| **病程** | 发作性，间歇期功能恢复 | 慢性，≥2年（儿童≥1年） |
| **症状严重度** | 发作期严重 | 持续性轻-中度 |
| **发作标准** | 符合重性抑郁发作标准 | 不符合全部重性抑郁标准 |
| **复发概念** | 明确的发作-缓解周期 | 持续低水平症状 |
| **MBCT适用** | 复发预防（核心适应症） | 症状管理（需改编期望） |

### 2.3 与适应障碍的鉴别

| 特征 | 复发性重性抑郁 | 适应障碍伴抑郁情绪 |
|------|-------------|------------------|
| **应激源** | 不一定有明确应激源 | 可识别的应激事件后3个月内起病 |
| **症状标准** | 符合5项以上重性抑郁标准 | 不符合全部重性抑郁标准 |
| **功能损害** | 显著且广泛 | 通常限于应激相关领域 |
| **病程** | 反复发作 | 通常在应激源消除后6个月内缓解 |
| **MBCT适用** | 标准方案 | 可考虑短期正念干预，但非核心适应症 |

### 2.4 与焦虑障碍的共病鉴别

#### 广泛性焦虑障碍（GAD）共病

| 特征 | 纯抑郁 | 抑郁+GAD共病 |
|------|--------|-------------|
| **核心症状** | 低落、兴趣丧失 | 低落 + 过度担忧 |
| **担忧内容** | 过去、自我、失败 | 未来、多种领域、灾难化 |
| **躯体症状** | 疲劳、睡眠障碍 | 肌肉紧张、易惊跳、胃肠道症状 |
| **MBCT调整** | 标准方案 | 增加担忧觉察模块；延长第3-4周 |

#### 社交焦虑障碍共病

- 患者可能在团体中感到极度不适
- MBCT调整：允许初期少发言；提供双人练习选项；增加团体安全感建设

#### 惊恐障碍共病

- 身体扫描可能触发惊恐（对躯体感受过度敏感）
- MBCT调整：从开放式觉察开始，而非身体扫描；允许睁眼练习；教育躯体感受无害性

### 2.5 与创伤后应激障碍（PTSD/C-PTSD）的鉴别

| 特征 | 复发性抑郁 | PTSD/C-PTSD |
|------|-----------|-------------|
| **创伤史** | 不一定有 | 有明确的创伤暴露史 |
| **核心症状** | 情绪低落、兴趣丧失 | 再体验、回避、高警觉、负性认知改变 |
| **正念反应** | 通常耐受良好 | 可能触发闪回、解离、情绪淹没 |
| **MBCT适用** | 标准方案 | 需创伤知情改编或先进行稳定化 |

**评估工具**：PCL-5（PTSD核查表）、DES（解离体验量表）

### 2.6 与人格障碍的共病评估

| 人格障碍类型 | 对MBCT的影响 | 调整策略 |
|------------|------------|---------|
| **边缘型（BPD）** | 情绪剧烈波动；团体中可能触发危机 | 需DBT优先或DBT-MBCT整合；个体化方案 |
| **强迫型（OCPD）** | 过度努力"正确"练习；自我批评严重 | 强调"不努力"态度；减少结构化要求 |
| **回避型（AvPD）** | 回避团体分享；害怕负面评价 | 允许书面替代口头分享；小组规模缩小 |
| **自恋型（NPD）** | 难以接纳脆弱；可能主导团体 | 明确团体边界；个体反馈 |
| **分裂型（SPD）** | 可能加重怪诞思维 | 评估精神病谱系风险；谨慎入组 |

## 三、躯体疾病所致抑郁的排除

| 疾病类别 | 常见疾病 | 筛查建议 | 对MBCT的影响 |
|---------|---------|---------|-------------|
| **内分泌** | 甲状腺功能减退、库欣综合征、糖尿病 | TSH、T3/T4、空腹血糖、HbA1c | 原发病治疗优先 |
| **神经系统** | 帕金森病、癫痫、多发性硬化、脑卒中 | 神经科评估、脑MRI | 需神经科联合管理 |
| **感染/炎症** | 慢性感染、自身免疫病 | CRP、ESR、自身抗体 | 炎症可能降低正念疗效 |
| **肿瘤** | 胰腺癌、脑肿瘤 | 针对性检查 | 需肿瘤科联合管理 |
| **营养缺乏** | B12缺乏、叶酸缺乏、维生素D缺乏 | 血清B12、叶酸、25-OH-D | 补充后重新评估 |
| **药物引起** | 糖皮质激素、干扰素、β受体阻滞剂 | 用药史回顾 | 考虑换药 |

## 四、诊断整合与临床决策

### 4.1 MBCT 适用性决策矩阵

| 诊断 | 缓解期 | 轻度症状 | 中度症状 | 重度/急性 | 备注 |
|------|--------|---------|---------|----------|------|
| 复发性MDD | ✅ 一线 | ✅ 可入组 | ⚠️ 联合药物 | ❌ 禁忌 | NICE指南推荐 |
| 持续性抑郁障碍 | ⚠️ 可尝试 | ✅ 可入组 | ⚠️ 联合治疗 | ❌ 禁忌 | 期望调整 |
| 双相抑郁（缓解期）| ⚠️ 需稳定剂 | ⚠️ 需精神科 | ❌ 禁忌 | ❌ 禁忌 | 需精神科评估 |
| GAD | ✅ 有效 | ✅ 可入组 | ✅ 可入组 | ⚠️ 联合治疗 | 证据B级 |
| 社交焦虑 | ✅ 有效 | ✅ 可入组 | ✅ 可入组 | ⚠️ 联合治疗 | 证据B级 |
| PTSD | ⚠️ 需稳定化 | ⚠️ 需改编 | ❌ 禁忌 | ❌ 禁忌 | 先 trauma-focused |
| BPD | ⚠️ DBT优先 | ⚠️ 需整合 | ❌ 禁忌 | ❌ 禁忌 | DBT-MBCT整合 |

### 4.2 转诊决策树

```
患者申请MBCT
    │
    ├── 当前符合MDD发作标准（≥5项症状，≥2周）
    │       → 转精神科评估 → 药物治疗优先 → 缓解后重新评估
    │
    ├── 有轻躁狂/躁狂史 或 MDQ阳性
    │       → 转精神科确诊双相 → 心境稳定剂 → 稳定后评估
    │
    ├── 有活跃精神病性症状
    │       → 紧急转精神科 → 抗精神病药 → 症状消失后评估
    │
    ├── 有主动自杀计划
    │       → 危机干预 → 住院/日间治疗 → 安全后评估
    │
    ├── PTSD未稳定（频繁闪回/解离）
    │       → 创伤稳定化治疗 → 症状稳定后评估
    │
    └── 缓解期复发性抑郁 / 轻度焦虑 / 功能良好
            → 标准MBCT入组流程
```
