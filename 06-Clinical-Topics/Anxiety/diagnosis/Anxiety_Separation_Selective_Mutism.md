# 分离焦虑障碍与选择性缄默症 | Separation Anxiety and Selective Mutism

> **目标**：建立分离焦虑障碍（SAD）与选择性缄默症（SM）的精神科标准诊断与治疗框架。
> **标准来源**：DSM-5-TR（2022）、APA Practice Guidelines、AACAP 指南。

---

## 一、分离焦虑障碍（Separation Anxiety Disorder）

### 1.1 DSM-5-TR 诊断标准

**A.** 对与依恋对象（如父母、伴侣）分离或预期分离的过度恐惧或焦虑，表现为以下至少**3项**：

1. 与分离相关的痛苦反复且过度
2. 持续且过度担心失去依恋对象或他们可能受到伤害
3. 持续且过度担心不幸事件导致与依恋对象分离
4. 因害怕分离而持续不愿或拒绝外出、离家、上学、上班
5. 持续且过度害怕或拒绝独处或在没有依恋对象的情况下在家或其他场合
6. 持续不愿或拒绝在没有依恋对象陪伴下入睡或离家睡觉
7. 反复做与分离相关的噩梦
8. 与分离相关或预期分离时反复出现躯体症状（头痛、腹痛、恶心、呕吐）

**B.** 恐惧、焦虑或回避持续至少**儿童青少年4周；成人6个月**。

**C.** 引起临床上显著的痛苦或功能损害。

**D.** 不能用其他精神障碍更好地解释。

### 1.2 成人分离焦虑

| 特征 | 说明 |
|------|------|
| **被忽视** | 历史上认为仅儿童疾病；DSM-5 承认成人 SAD |
| **表现** | 过度担心伴侣/父母；拒绝独自旅行；噩梦；躯体症状 |
| **共病** | 惊恐障碍、广场恐惧、抑郁 |
| **影响** | 影响婚姻、工作（拒绝出差） |

## 二、选择性缄默症（Selective Mutism）

### 2.1 DSM-5-TR 诊断标准

**A.** 在预期需要说话的特定社交情境（如学校）中持续无法说话，尽管在其他情境中能够说话。

**B.** 这种障碍干扰了教育或职业成就或社交沟通。

**C.** 障碍持续时间至少**1个月**（不包括入学的第一个月）。

**D.** 无法说话不能归因于对社交情境中使用的语言缺乏知识或感到不适。

**E.** 障碍不能用交流障碍更好地解释（如儿童期发生的流畅障碍）。

**F.** 缺乏言语不能仅出现在孤独症谱系障碍、精神分裂症或其他精神病性障碍的病程中。

### 2.2 诊断特征

| 特征 | 说明 |
|------|------|
| **年龄** | 通常 3-5 岁被发现（入幼儿园/小学时） |
| **表现** | 在家说话正常；在学校/社交场合完全沉默 |
| **非言语沟通** | 可能用点头、手势、表情交流 |
| **焦虑** | 常与社交焦虑障碍共病（>90%） |
| **误解** | 常被误认为"固执""对抗""自闭症" |

---

## 三、鉴别诊断

| 障碍 | SAD 鉴别 | SM 鉴别 |
|------|---------|--------|
| **广场恐惧** | 恐惧情境（独处、公共交通）；非特定依恋对象 | — |
| **社交焦虑** | 恐惧社交评价；不限于分离 | SM 可视为社交焦虑的早期/严重形式 |
| **GAD** | 焦虑内容广泛；不限于分离 | — |
| **抑郁** | 兴趣丧失；能量低；不特定于分离 | — |
| **自闭症** | 社交沟通缺陷广泛；刻板行为 | 自闭症沉默在所有情境；SM 在家正常 |
| **语言障碍** | — | SM 语言能力正常；语言障碍在所有情境 |
| **对立违抗** | — | ODD 的沉默是对抗性的；SM 是焦虑性的 |

## 四、治疗

### 4.1 分离焦虑障碍

#### CBT（儿童/青少年）

| 成分 | 方法 |
|------|------|
| ** psychoeducation** | 解释焦虑模型；正常化反应 |
| **识别焦虑** | 帮助孩子识别焦虑的身体信号 |
| **认知重构** | 挑战"如果妈妈不回来怎么办" |
| **暴露层级** | 逐步增加分离时间；从最容易的开始 |
| **父母训练** | 减少过度保护；一致的反应 |
| **放松训练** | 腹式呼吸；渐进肌肉放松 |

#### 暴露层级示例（上学焦虑）

| 层级 | 情境 | SUDs |
|------|------|------|
| 1 | 妈妈离开房间5分钟 | 20 |
| 2 | 妈妈在家但不同房间1小时 | 30 |
| 3 | 在邻居家30分钟 | 40 |
| 4 | 在邻居家2小时 | 50 |
| 5 | 妈妈送到校门口 | 60 |
| 6 | 妈妈送到教室门口 | 70 |
| 7 | 妈妈送到学校门口后离开 | 80 |
| 8 | 妈妈送到学校后离开并度过整天 | 90 |

#### 药物治疗

| 药物 | 证据 | 说明 |
|------|------|------|
| **SSRI（舍曲林、氟西汀）** | B级 | 中重度；6-12岁起可用 |
| **注意** | — | 轻中度优先 CBT；药物为辅助 |

#### 成人 SAD 治疗

| 方法 | 说明 |
|------|------|
| **CBT** | 类似儿童；增加独立活动暴露 |
| **IPT** | 改善人际关系；减少过度依赖 |
| **SSRI** | 共病焦虑/抑郁时 |

### 4.2 选择性缄默症

#### 核心原则

| 原则 | 说明 |
|------|------|
| **早期干预** | 越早越好；幼儿园/小学早期 |
| **多模式** | 需要家庭、学校、治疗师协作 |
| **不强迫说话** | 不施压；减少焦虑是首要目标 |
| **逐步暴露** | 从最容易的沟通情境开始 |

#### 具体策略

| 策略 | 方法 |
|------|------|
| **刺激渐变** | 从非言语沟通 → 耳语 → 小声 → 正常声音 |
| **同伴中介** | 让健谈的朋友/同学陪伴 |
| **音频/视频** | 让孩子在家录音/录像；在学校播放 |
| **电话练习** | 与熟悉的人电话交谈 |
| **小团体** | 1对1 → 小团体 → 大团体 |
| **强化** | 奖励任何沟通尝试（非仅说话） |

#### 学校干预

| 策略 | 说明 |
|------|------|
| **教师教育** | 不强迫说话；不惩罚沉默 |
| **替代沟通** | 允许点头、手势、写字、卡片 |
| **安全基地** | 指定信任的成人；安全角落 |
| **渐进参与** | 从观察 → 非言语参与 → 言语参与 |
| **同伴支持** | 友善的同伴配对 |

#### 药物治疗

| 药物 | 证据 | 说明 |
|------|------|------|
| **SSRI（氟西汀）** | C级 | 共病严重社交焦虑时；6岁以上 |
| **注意** | — | 优先行为干预；药物为辅助 |

## 五、共病

| 共病 | SAD | SM |
|------|-----|-----|
| **社交焦虑** | 常见 | >90% |
| **GAD** | 常见 | 30-50% |
| **抑郁** | 青少年/成人 | 后期可能出现 |
| **对立违抗** | — | 需鉴别 |
| **语言障碍** | — | 需排除 |

---

## 六、预后

| 障碍 | 预后 |
|------|------|
| **儿童 SAD** | 通常良好；80%在 CBT 后显著改善 |
| **成人 SAD** | 较少研究；CBT + 药物可改善 |
| **SM** | 早期干预预后良好；未治疗可持续数年；部分发展为社交焦虑 |
