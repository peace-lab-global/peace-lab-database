# 焦虑障碍诊断与鉴别诊断 | Anxiety Disorder Diagnosis and Differential

> **目标**：建立精神科标准的焦虑障碍诊断框架，涵盖 DSM-5-TR、ICD-11 诊断标准、各亚型鉴别、与躯体疾病/物质的鉴别及决策树。
> **标准来源**：DSM-5-TR（2022）、ICD-11（2022）、APA Practice Guidelines（2019）。

---

## 一、广泛性焦虑障碍（GAD）

### 1.1 DSM-5-TR 诊断标准

**A. 对多种事件或活动（如工作或学业表现）存在过度的焦虑和担心，个体难以控制这种担心。**

**B. 焦虑和担心与下列6种症状中至少3种相关（儿童只需1种）：**

| 症状 | 临床评估要点 |
|------|-------------|
| 1. 坐立不安或感到紧张/兴奋 | 主观感到"坐不住"、内心烦躁 |
| 2. 容易疲劳 | 即使轻微活动也感疲惫；与抑郁的疲劳鉴别 |
| 3. 注意力难以集中或头脑一片空白 | "脑子像一团乱麻"；注意与ADHD鉴别 |
| 4. 易激惹 | 对小事反应过度；常被误认为脾气差 |
| 5. 肌肉紧张 | 颈肩背部疼痛、紧绷感；常就诊于疼痛科/骨科 |
| 6. 睡眠障碍 | 入睡困难、易醒；与抑郁的早醒鉴别 |

**C. 焦虑、担心或躯体症状引起有临床意义的痛苦，或导致社交、职业或其他重要功能方面的损害。**

**D. 这种障碍不能归因于某种物质或其他躯体疾病。**

**E. 这种障碍不能用其他精神障碍来更好地解释。**

### 1.2 ICD-11 编码

| 编码 | 名称 |
|------|------|
| 6B00 | 广泛性焦虑障碍 |
| 6B01 | 惊恐障碍 |
| 6B02 | 广场恐惧症 |
| 6B03 | 特定恐惧症 |
| 6B04 | 社交焦虑障碍 |
| 6B05 | 分离焦虑障碍 |
| 6B06 | 选择性缄默症 |

---

## 二、惊恐障碍（Panic Disorder）

### 2.1 DSM-5-TR 诊断标准

**A. 反复、不可预期的惊恐发作。**

**惊恐发作的13项症状（急性强烈恐惧/不适，10分钟内达峰，≥4项）：**

| 躯体症状 | 认知症状 |
|---------|---------|
| 心悸、心跳加速 | 濒死感 |
| 出汗 | 失去控制感/"要疯了" |
| 震颤/发抖 | 现实解体（derealization） |
| 气短/窒息感 | 人格解体（depersonalization） |
| 哽噎感 | |
| 胸痛/胸部不适 | |
| 恶心/腹部不适 | |
| 头晕/不稳/晕厥 | |
| 发冷或潮热 | |
| 感觉异常（麻木/刺痛） | |

**B. 至少1次发作后持续1个月以上存在以下1项或2项：**
- 持续担心再次发作或其后果
- 与发作相关的显著适应不良行为改变（如回避运动/密闭空间）

**C. 这种障碍不能归因于物质或其他躯体疾病。**

**D. 这种障碍不能用其他精神障碍来更好地解释。**

### 2.2 惊恐发作 vs 惊恐障碍

| | 惊恐发作 | 惊恐障碍 |
|--|---------|---------|
| **定义** | 症状群（≥4项躯体/认知症状） | 反复不可预期的惊恐发作 + 持续担忧/行为改变 |
| **关联** | 可见于多种障碍（抑郁、PTSD、GAD、躯体疾病） | 独立的焦虑障碍诊断 |
| **临床意义** | 需寻找原因 | 需针对惊恐障碍治疗 |

## 三、社交焦虑障碍（Social Anxiety Disorder）

### 3.1 DSM-5-TR 诊断标准

**A. 对一种或多种社交情境存在显著而持续的恐惧，个体害怕自己的言行或呈现的焦虑症状会导致负性评价。**

**常见恐惧情境**：
- 公开演讲/表演
- 与权威人物互动
- 与陌生人互动
- 被观察（进食、书写、工作）
- 发起对话
- 参加聚会/社交活动

**B. 个体几乎总是能够促发焦虑的社交情境中触发焦虑。**

**C. 主动回避社交情境，或带着强烈的恐惧/焦虑忍受。**

**D. 恐惧/焦虑与情境的社会文化背景下的实际威胁不相称。**

**E. 持续≥6个月。**

**F. 引起有临床意义的痛苦或功能损害。**

### 3.2 亚型

| 亚型 | 特征 | 比例 |
|------|------|------|
| **表演型** | 恐惧限于公开演讲/表演 | 约30% |
| **广泛型** | 恐惧多数社交情境 | 约70% |

---

## 四、特定恐惧症

### 4.1 诊断要点

**A. 对特定物体或情境存在显著而持续的恐惧。**

**常见类型**：

| 类型 | 常见对象 |
|------|---------|
| **动物型** | 蜘蛛、蛇、狗、昆虫 |
| **自然环境型** | 高处、暴风雨、水 |
| **血液-注射-损伤型** | 针头、手术、血液（独特血管迷走反应：先心动过速后心动过缓、血压下降、晕厥） |
| **情境型** | 飞行、电梯、密闭空间、驾驶 |
| **其他** | 呕吐、疾病、噪音、服装 |

**B. 暴露于恐惧对象/情境几乎总是能够促发立即的焦虑反应。**

**C. 主动回避，或带着强烈的恐惧/焦虑忍受。**

**D. 恐惧/焦虑与情境的实际危险不相称。**

**E. 持续≥6个月。**

**F. 引起有临床意义的痛苦或功能损害。**

---

## 五、广场恐惧症

### 5.1 诊断标准

**A. 对下列2种或更多情境存在显著而持续的恐惧：**
- 乘坐公共交通工具
- 处于开放空间
- 处于密闭空间
- 排队或处于人群中
- 独自离家外出

**B. 个体害怕或回避这些情境是因为想到一旦出现惊恐样症状或其他失去能力/尴尬的症状时，逃离困难或无法获得帮助。**

**C. 主动回避，或需要陪伴，或带着强烈的恐惧/焦虑忍受。**

**D. 恐惧/焦虑与情境的实际危险不相称。**

**E. 持续≥6个月。**

**F. 引起有临床意义的痛苦或功能损害。**

## 六、分离焦虑障碍与选择性缄默症

### 6.1 分离焦虑障碍（Separation Anxiety Disorder）

| 特征 | 说明 |
|------|------|
| **核心** | 对与依恋对象分离的过度恐惧 |
| **年龄** | 儿童常见；成人也可诊断（DSM-5新增成人诊断） |
| **症状** | 分离时过度痛苦、担心依恋对象受伤害、拒绝独处/上学、噩梦、躯体症状 |
| **持续时间** | 儿童≥4周；成人≥6个月 |

### 6.2 选择性缄默症（Selective Mutism）

| 特征 | 说明 |
|------|------|
| **核心** | 在特定社交情境中持续不能说话（尽管在其他情境中能说话） |
| **年龄** | 通常5岁前起病 |
| **共病** | 常与社交焦虑障碍共病 |
| **功能** | 干扰学业/职业/社交沟通 |

## 七、关键鉴别诊断

### 7.1 焦虑障碍之间的鉴别

| | GAD | 惊恐障碍 | 社交焦虑 | 特定恐惧 |
|--|-----|---------|---------|---------|
| **核心恐惧** | 多种领域的广泛担忧 | 惊恐发作本身 | 负性评价 | 特定对象/情境 |
| **躯体症状** | 慢性轻中度 | 急性重度 | 社交情境中出现 | 暴露时出现 |
| **回避** | 不典型 | 回避运动/密闭空间等 | 回避社交 | 回避特定对象 |
| **惊恐发作** | 可有，但不是核心 | 核心症状 | 可有 | 可有 |

### 7.2 与抑郁的鉴别与共存

| 特征 | 焦虑为主 | 抑郁为主 | 共存 |
|------|---------|---------|------|
| **核心** | 对未来的恐惧/担忧 | 情绪低落/兴趣丧失 | 两者皆有 |
| **时间焦点** | 未来 | 过去/现在 | 混合 |
| **能量** | 高警觉、紧张 | 疲乏、动力缺乏 | 矛盾感 |
| **睡眠** | 入睡困难 | 早醒 | 均差 |
| **自杀风险** | 中（因绝望） | 高 | 极高 |

**注意**：焦虑与抑郁共病率>50%；GAD常先于抑郁发作。

### 7.3 与躯体疾病的鉴别

| 躯体疾病 | 类似焦虑的表现 | 鉴别要点 |
|---------|--------------|---------|
| **甲状腺功能亢进** | 心悸、震颤、易激惹、失眠 | TSH降低、T3/T4升高；体重下降 |
| **嗜铬细胞瘤** | 阵发性心悸、出汗、血压升高 | 24h尿儿茶酚胺；血/尿甲氧基肾上腺素 |
| **心律失常** | 心悸、胸闷、濒死感 | ECG、Holter、心脏超声 |
| **COPD/哮喘** | 气短、窒息感 | 肺功能检查；发作与过敏原相关 |
| **癫痫（颞叶）** | 发作性恐惧、人格解体 | EEG；发作无明确心理触发 |
| **前庭功能障碍** | 头晕、不稳 | 耳鼻喉科检查 |
| **低血糖** | 心悸、出汗、震颤、焦虑 | 血糖监测；进食后缓解 |
| **药物/物质** | 咖啡因、伪麻黄碱、酒精戒断、苯二氮卓戒断 | 用药史/物质使用史 |

### 7.4 与精神病性障碍的鉴别

| 特征 | 焦虑 | 精神病性障碍 |
|------|------|-------------|
| **恐惧内容** | 与现实有一定联系（过度但非荒谬） | 妄想内容荒谬、不可动摇 |
| **自知力** | 保留（知道担忧过度） | 受损 |
| **现实检验** | 完整 | 受损 |
| **功能** | 因恐惧而受限 | 因精神病性症状而严重受损 |

## 八、诊断决策树

```
患者主诉焦虑/恐惧/担忧
    │
    ├── 症状由物质或躯体疾病解释
    │       → 物质/躯体疾病所致焦虑 → 治疗原发病
    │
    ├── 对多种领域持续担忧 + 肌肉紧张/易疲劳/失眠/易激惹/注意难集中
    │       → GAD评估（≥3项躯体症状，≥6月，难以控制）
    │
    ├── 反复不可预期的惊恐发作 + 持续担忧/行为改变
    │       → 惊恐障碍评估
    │           ├── 回避开放/密闭/公共交通/人群/独自外出
    │           │       → 广场恐惧症共病评估
    │           └── 无广场恐惧
    │                   → 单纯惊恐障碍
    │
    ├── 对社交情境的恐惧（负性评价）+ 回避/忍受
    │       → 社交焦虑障碍（表演型 vs 广泛型）
    │
    ├── 对特定物体/情境的恐惧 + 回避/忍受
    │       → 特定恐惧症（动物/自然/血液-注射-损伤/情境/其他）
    │
    ├── 与依恋对象分离的过度恐惧（儿童/成人）
    │       → 分离焦虑障碍
    │
    ├── 特定社交情境中不能说话（儿童）
    │       → 选择性缄默症
    │
    ├── 反复闯入性思维 + 强迫行为
    │       → OCD评估
    │
    ├── 创伤暴露史 + 再体验/回避/高警觉/负性认知改变
    │       → PTSD评估
    │
    └── 伴情绪低落/兴趣丧失
            → 评估MDD共病
```
