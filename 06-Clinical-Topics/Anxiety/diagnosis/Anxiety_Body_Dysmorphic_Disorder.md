# 身体变形障碍诊断与治疗 | Body Dysmorphic Disorder (BDD)

> **目标**：建立身体变形障碍（BDD）的精神科标准诊断与治疗框架。
> **标准来源**：DSM-5-TR（2022）、APA Practice Guidelines、Phillips (2005)。

---

## 一、DSM-5-TR 诊断标准

### 1.1 诊断要件

**A.** 专注于一处或多处感知到的外貌缺陷或瑕疵，这些缺陷或瑕疵在他人看来不明显或轻微。

**B.** 在病程中，个体进行重复行为（如照镜子、过度修饰、皮肤搔抓、寻求确认）或心理活动（如比较自己与他人在外貌上的差异），以回应对外貌的关注。

**C.** 这种专注引起临床上显著的痛苦或社交、职业或其他重要领域的功能损害。

**D.** 这种专注不能用符合进食障碍诊断标准的对体型/体重的关注来更好地解释。

### 1.2 诊断特征

| 特征 | 说明 |
|------|------|
| **关注部位** | 皮肤（痤疮、疤痕、色斑）；头发（脱发、过多）；鼻子；体重/体型；肌肉（肌肉变形障碍）；面部特征 |
| **重复行为** | 照镜子（过多或回避）、修饰、皮肤搔抓、寻求确认、比较、整容手术 |
| **回避** | 回避镜子、社交场合、拍照 |
| **洞察力** | 多数具有不良洞察力或缺乏洞察力（坚信缺陷真实存在） |
| **功能损害** | 严重社交/职业损害；高失业率；隔离 |

### 1.3 亚型

| 亚型 | 特征 |
|------|------|
| **肌肉变形障碍（Muscle Dysmorphia）** | 认为身体太小或不够肌肉发达；主要见于男性 |
| **良好洞察力** | 认识到外貌关注被夸大了 |
| **不良洞察力** | 认为外貌缺陷看起来和感觉的一样严重 |
| **妄想性信念** | 完全确信缺陷严重且被他人注意到 |

## 二、鉴别诊断

| 障碍 | 鉴别要点 |
|------|---------|
| **OCD** | OCD的强迫思维内容更广泛；BDD 专注限于外貌 |
| **进食障碍** | 进食障碍关注体型/体重；BDD 可关注任何外貌特征 |
| **社交焦虑障碍** | SAD 恐惧社交评价；BDD 恐惧外貌评价；可共存 |
| **抑郁障碍** | 抑郁可有外貌不满；BDD 的 preoccupation 更强烈、更具体 |
| **精神病性障碍** | 妄想性 BDD 需鉴别；BDD 的妄想局限于外貌 |
| **正常外貌不满** | 程度较轻；不造成功能损害 |
| **性别不安** | 对自身性别特征的不适；与 BDD 不同 |

## 三、评估

### 3.1 标准化量表

| 量表 | 项目 | 用途 | 临界值 |
|------|------|------|--------|
| **BDD-YBOCS** | 12项 | 金标准；严重程度 | ≥20分：中度；≥30分：重度 |
| **BDDQ（BDD Questionnaire）** | — | 筛查 | 阳性提示需进一步评估 |
| **DCQ（Dysmorphic Concerns Questionnaire）** | 7项 | 筛查 | — |
| **BBIAS（BDD and Appearance Anxiety Scale）** | — | 评估 | — |

### 3.2 临床访谈要点

| 问题 | 目的 |
|------|------|
| "你对自己外貌有什么不满意的地方？" | 识别关注部位 |
| "你每天花多少时间担心这些？" | 评估 preoccupation 时间 |
| "你做什么来应对这些担心？" | 识别重复行为 |
| "这对你生活有什么影响？" | 评估功能损害 |
| "你是否寻求过整容手术？" | 评估医疗使用 |
| "你是否有肌肉相关的不满？" | 筛查肌肉变形障碍 |

## 四、治疗

### 4.1 药物治疗

| 药物 | 剂量 | 证据 | 说明 |
|------|------|------|------|
| **艾司西酞普兰** | 20-60mg | A级 | SSRI 一线；BDD 常需高剂量 |
| **氟西汀** | 40-80mg | A级 | 高剂量常见 |
| **氟伏沙明** | 150-300mg | A级 | — |
| **舍曲林** | 150-200mg | A级 | — |
| **氯米帕明** | 150-250mg | A级 | TCA；SSRIs 无效时 |
| **文拉法辛** | 150-375mg | B级 | SNRI |

**注意：**
- BDD 的 SSRI 剂量通常高于抑郁症
- 起效时间 4-8 周；需持续 12-16 周评估效果
- 维持治疗 1-2 年；复发常见
- 抗精神病药增效（低剂量利培酮/阿立哌唑）对妄想性 BDD 可能有效

### 4.2 认知行为治疗（CBT）— 一线

| 成分 | 方法 | 说明 |
|------|------|------|
| **暴露与反应预防（ERP）** | 暴露于外貌焦虑情境 + 停止重复行为 | 核心技术 |
| **认知重构** | 挑战外貌相关认知扭曲 | "每个人都在看我"→"大多数人没注意" |
| **知觉再训练** | 减少过度的外貌聚焦 | 练习看整体而非局部 |
| **行为实验** | 验证灾难化预测 | "如果我素颜出门，别人会嘲笑我吗？" |
| **镜子再训练** | 减少强迫性照镜子 | 限制照镜子时间；中性描述 |
| **社交技能训练** | 改善社交功能 | 减少对确认的需求 |

### 4.3 CBT结构（12-20次）

| 阶段 | 次数 | 内容 |
|------|------|------|
| **评估与心理教育** | 1-2 | 评估；BDD 模型；设定目标 |
| **ERP 引入** | 3-6 | 建立暴露层级；开始暴露 |
| **认知重构** | 7-12 | 挑战外貌信念；行为实验 |
| **高级 ERP** | 13-16 | 最具挑战性的暴露；社交情境 |
| **维持** | 17-20 | 复发预防；长期策略 |

### 4.4 ERP 层级示例

| 层级 | 情境 | SUDs |
|------|------|------|
| 1 | 不化妆在家 | 20 |
| 2 | 不化妆见治疗师 | 40 |
| 3 | 不化妆见朋友 | 50 |
| 4 | 穿"不安全感"服装出门 | 60 |
| 5 | 参加社交活动不修饰 | 70 |
| 6 | 拍照并看照片 | 80 |
| 7 | 公开演讲 | 90 |
| 8 | 被他人近距离观察 | 100 |

**反应预防：**
- 不照镜子超过设定时间
- 不寻求外貌确认
- 不比较自己与他人
- 不修饰超过设定时间

### 4.5 肌肉变形障碍特殊注意

| 特征 | 说明 |
|------|------|
| **表现** | 过度锻炼；蛋白质补充；可能的类固醇使用 |
| **回避** | 回避展示身体的情境（如游泳池） |
| **治疗** | 标准 BDD-CBT + 减少锻炼暴露；评估类固醇使用 |

## 五、共病管理

| 共病 | 患病率 | 管理 |
|------|--------|------|
| **抑郁** | 50-80% | SSRI 同时覆盖 |
| **社交焦虑** | 30-50% | ERP 同时覆盖 |
| **OCD** | 20-40% | 标准 OCD 治疗 |
| **物质使用** | 20-30% | 评估类固醇/酒精使用 |
| **自杀** | 高 | 每次评估；安全计划 |

## 六、特殊人群

### 6.1 青少年

| 注意 | 说明 |
|------|------|
| **高发** | 青春期 BDD 发病高峰 |
| **社交媒体** | 滤镜/修图可能加重 BDD |
| **家庭** | 家长可能误解为"虚荣"；需教育 |
| **学校** | 学校回避；欺凌 |

### 6.2 寻求整容者

| 注意 | 说明 |
|------|------|
| **筛查** | 整容前筛查 BDD；BDD 患者整容后通常不满意 |
| **转诊** | 疑似 BDD → 精神科评估 |
