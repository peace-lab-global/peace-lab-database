# 焦虑障碍围手术期与医疗场景 | Anxiety in Perioperative and Medical Settings

> **目标**：系统阐述焦虑在围手术期、牙科、影像检查、ICU等医疗场景中的评估与管理。

---

## 一、围手术期焦虑

### 1.1 术前焦虑

| 患病率 | 约25-80%（取决于手术类型） |
|--------|--------------------------|
| **影响** | 增加术后疼痛、恶心呕吐、延长恢复、增加镇痛药需求 |
| **评估** | VAS焦虑量表（0-10）、STAI-State |

### 1.2 术前焦虑管理

| 干预 | 证据 | 方法 |
|------|------|------|
| **术前教育** | A级 | 手术过程、预期感受、恢复时间 |
| **认知行为准备** | A级 | 信息提供 + 应对技能训练 |
| **放松训练** | B级 | 腹式呼吸、渐进肌肉放松 |
| **音乐** | B级 | 术前30分钟 |
| **药物** | B级 | 苯二氮卓类（咪达唑仑术前30min）；注意老年人慎用 |

### 1.3 术后焦虑

| 原因 | 管理 |
|------|------|
| **疼痛** | 充分镇痛；多模式镇痛 |
| **ICU环境** | 减少噪音/光线；定向力维持；家属陪伴 |
| **谵妄** | 识别与处理；避免BZD（可能加重） |

## 二、牙科焦虑

### 2.1 患病率

- 约10-20%有严重牙科焦虑
- 约5-10%完全回避牙科治疗

### 2.2 管理策略

| 策略 | 证据 |
|------|------|
| **认知行为治疗** | A级 |
| **系统脱敏** | A级 |
| **虚拟现实暴露** | B级 |
| **音乐/分散注意** | B级 |
| **笑气（N2O）** | A级 |
| **口服镇静** | B级（咪达唑仑/Halcion） |
| **静脉镇静** | B级（严重病例） |

---

## 三、ICU焦虑

### 3.1 ICU焦虑与谵妄

| 原因 | 管理 |
|------|------|
| **环境因素** | 降低噪音/光线；昼夜节律维持 |
| **医疗因素** | 充分镇痛；减少约束；早期活动 |
| **心理因素** | 定向力训练；家属沟通；认知刺激 |
| **药物** | 右美托咪定（α2激动剂）；避免BZD |

### 3.2 ICU后综合征（PICS）

- ICU出院后持续焦虑/PTSD/认知障碍
- 约20-30% ICU患者出现
- 预防：ICU日记、早期心理支持

## 四、影像检查焦虑

| 检查 | 焦虑原因 | 管理 |
|------|---------|------|
| **MRI（幽闭恐惧）** | 密闭空间 | 开放MRI；渐进暴露；口服镇静 |
| **CT** | 注射对比剂恐惧 | 解释；放松；分散注意 |
| **内镜检查** | 侵入性、恶心 | 镇静；咽部麻醉；CBT |
