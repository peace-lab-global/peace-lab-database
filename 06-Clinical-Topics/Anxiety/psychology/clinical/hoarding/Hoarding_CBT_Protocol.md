# 囤积障碍 CBT 治疗协议 | CBT Protocol for Hoarding Disorder

> **目标**：提供可直接临床使用的囤积障碍 CBT 标准化协议。
> **标准来源**：Steketee & Frost (2014)、APA、RCT 证据。

---

## 一、治疗概述

| 参数 | 设置 |
|------|------|
| **疗程** | 20-26次个体或团体会谈 |
| **频率** | 每周1-2次（初期）；后期每2周 |
| **时长** | 60-90分钟/次（家中治疗可能更长） |
| **格式** | 个体（首选）或团体（8-12人） |
| **辅助** | 家中治疗（4-8次在治疗师陪同下进行） |

---

## 二、逐次会谈结构

### 第一阶段：评估与动机（1-4次）

| 内容 | 方法 |
|------|------|
| **详细评估** | SI-R、HRS、Clutter Image Rating、功能评估 |
| **家中评估** | 拍摄/绘制 clutter 区域地图 |
| **心理教育** | 囤积模型；CBT 原理 |
| **动机访谈** | 探索矛盾心理；建立个人改变目标 |
| **设定目标** | SMART 目标（如"2周内清理厨房台面"） |

### 第二阶段：技能训练（5-12次）

| 技能 | 训练内容 |
|------|---------|
| **分类练习** | 使用3-5个主要类别（如"保留""捐赠""丢弃""待定"）；练习快速分类 |
| **决策规则** | "1年规则"" duplicates 规则""情绪价值评估" |
| **认知重构** | 识别和挑战"以后可能用到""丢了就忘了"等信念 |
| **非获取练习** | 购物暴露；拒绝免费物品练习 |
| **情绪耐受** | 学习耐受丢弃时的焦虑和悲伤 |

### 第三阶段：家中治疗（13-22次）

| 方法 | 说明 |
|------|------|
| **治疗师陪同** | 到患者家中进行整理和丢弃 |
| **渐进区域** | 从 clutter 最轻的区域开始；逐步到最困难的区域 |
| **决策教练** | 治疗师在旁协助快速决策；防止拖延 |
| **认知重构现场** | 当患者犹豫时进行实时认知重构 |
| **庆祝** | 完成区域后庆祝 |

### 第四阶段：维持与复发预防（23-26次）

| 内容 | 方法 |
|------|------|
| **维持计划** | 每周整理时间；防止 clutter 复发 |
| **非获取策略** | 长期策略防止重新获取 |
| **应对复发** | 识别早期复发信号；快速应对 |
| **家庭支持** | 教育家庭成员如何支持（不批评、不替患者丢弃） |

## 三、核心技术详解

### 3.1 分类系统

```
═══════════════════════════════════════════════════════════════
              囤积物品分类系统
═══════════════════════════════════════════════════════════════

【保留】
□ 过去1年内使用过
□ 有明确的未来使用计划（具体日期）
□ 不可替代的情感价值（仅限少量）
□ 法律/财务需要

【捐赠】
□ 完好但不再需要
□ 有人可以受益
□ 情感价值低

【丢弃/回收】
□ 损坏/过期
□ 无使用价值
□ 重复物品（保留最好的）

【待定】
□ 无法立即决定
□ 放入"待定箱"；设定1周后再决定
□ 限制待定物品数量

【禁止类别】
□ 垃圾（直接丢弃）
□ 过期食品/药品（安全丢弃）

═══════════════════════════════════════════════════════════════
```

### 3.2 常见认知扭曲与重构

| 囤积信念 | 重构替代 |
|---------|---------|
| "以后可能用到" | "过去2年没用到，用到概率<5%；如果真需要可以再买/借" |
| "扔掉是浪费" | "保留不用才是真正的浪费；捐赠可以让别人受益" |
| "这东西有价值" | "实际价值是X元；我的空间和时间更有价值" |
| "丢了就再也找不到了" | "重要文件可以扫描；信息可以重新获取" |
| "这是我的记忆" | "记忆在我心中，不在物品中；我可以保留少量代表性物品" |
| "我需要更多信息才能决定" | "更多信息不会改变结果；快速决策减少痛苦" |

### 3.3 非获取练习

| 练习 | 方法 |
|------|------|
| **购物清单** | 只允许购买清单上的物品 |
| **24小时规则** | 想买非必需品时等待24小时 |
| **免费物品暴露** | 主动拒绝免费物品（如促销品、传单） |
| **商店暴露** | 去商店但不购买；耐受渴望 |

## 四、家中治疗安全指南

| 风险 | 预防 |
|------|------|
| **生物危害** | 戴手套、口罩；发现霉菌/害虫时停止并寻求专业清理 |
| **结构危险** | 注意堆叠物品倒塌；不攀爬 |
| **情绪过载** | 患者情绪过于激动时暂停； grounding 技巧 |
| **治疗师安全** | 告知同事位置；携带手机；有人知道你的行程 |

---

## 五、常见问题处理

| 问题 | 策略 |
|------|------|
| **患者拒绝开始** | 动机访谈；从最小的区域开始；减少威胁感 |
| **"我做不到"** | 分解为更小步骤；5分钟规则 |
| **家庭成员冲突** | 家庭会谈；教育家庭成员不擅自丢弃 |
| **复发** | 正常化；识别触发因素；强化维持策略 |
| **洞察力差** | 使用功能性后果（" clutter 让你不能做饭"）而非批评 |
