## 睡眠焦虑机制

### 什么是睡眠焦虑

睡眠焦虑是对睡不着或睡眠不足的焦虑，形成恶性循环：
- 躺在床上担心睡不着
- 担心导致更睡不着
- 更担心→更睡不着

### 常见思维模式

| 思维 | 问题 |
|------|------|
| "我必须睡够8小时" | 增加了睡眠压力 |
| "今晚睡不着明天就完了" | 灾难化后果 |
| "我是个失眠者" | 身份标签加重焦虑 |
| "已经这么晚了，我应该睡着了" | 与现实抗争增加唤醒 |

## 睡眠卫生

### 环境优化

- 保持卧室黑暗、安静、凉爽
- 床只用于睡眠和亲密，不用于工作或看电视
- 更换舒适的床垫和枕头
- 使用遮光窗帘

### 习惯调整

| 习惯 | 建议 |
|------|------|
| 作息时间 | 每天同一时间睡觉和起床 |
| 咖啡因 | 下午2点后避免 |
| 酒精 | 睡前限制（虽然可能帮助入睡但破坏睡眠质量）|
| 运动 | 避免睡前3小时内剧烈运动 |
| 屏幕 | 睡前1小时不使用电子设备 |

## 睡前放松技术

### 渐进性肌肉放松

睡前进行10-15分钟：
- 从脚到头系统地紧张和放松每组肌肉
- 注意紧张和放松的对比
- 保持呼吸平稳

### 4-7-8呼吸法

1. 躺好，闭眼
2. 用鼻子吸气4秒
3. 屏住呼吸7秒
4. 用嘴呼气8秒
5. 重复4-6次

### 正念冥想

**身体扫描**：
- 从脚趾开始，慢慢向上移动注意力
- 只是观察，不试图改变
- 如果思绪飘走，温和地带回

## 认知重构

### 挑战睡眠神话

| 神话 | 现实 |
|------|------|
| 必须立即入睡 | 需要放松时间，正常 |
| 必须睡够8小时 | 睡眠质量与时长同样重要 |
| 睡不着就什么都做不了 | 躺着休息也有益处 |
| 一晚睡不好会很糟 | 偶尔睡不好不会造成长期伤害 |

### 替代思维

- "偶尔睡不着是正常的"
- "即使没睡着，我的身体也在休息"
- "我不需要强制入睡，只需要放松"
- "睡眠是自然过程，不能用力控制"

## 如果睡不着怎么办

### 20分钟规则

如果躺在床上超过20分钟还没睡着，或者已经超过30分钟：
1. 起床到另一个房间
2. 做放松活动（阅读纸质书、听轻音乐）
3. 等困了再回床上
4. 如果又睡不着，重复这个过程

**为什么**：将床与清醒重新关联，打破"床=清醒"的连接

### 减少对睡眠的关注

- 不要频繁看时间
- 不要计算还剩多少小时可以睡
- 接受睡眠不是可以强迫的
