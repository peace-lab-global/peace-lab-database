# 焦虑障碍复发预防与维持治疗 | Relapse Prevention and Maintenance for Anxiety Disorders

> **目标**：建立精神科标准的焦虑障碍复发预防体系，涵盖维持治疗决策、复发预警信号、长期管理策略。

---

## 一、复发vs复发区别

| 术语 | 定义 | 时间 |
|------|------|------|
| **复发（Relapse）** | 缓解后早期（<6月）症状再现 | 急性期治疗后6月内 |
| **再发（Recurrence）** | 缓解后期（>6月）新发作 | 6月后 |
| **反跳（Rebound）** | 停药后短期症状加重 | 停药后数天-数周 |
| **波动（Fluctuation）** | 轻度、短暂的症状变化 | 任何时间 |

---

## 二、维持治疗决策

### 2.1 各亚型维持建议

| 障碍 | 首次发作 | 复发≥2次 | 慢性 |
|------|---------|---------|------|
| **GAD** | 6-12月 | 1-2年 | 长期 |
| **惊恐障碍** | 6-12月 | 1-2年 | 长期 |
| **社交焦虑** | 12月 | 2-3年 | 长期 |
| **OCD** | 1-2年 | 长期 | 长期 |
| **PTSD** | 6-12月 | 1-2年 | 长期 |

### 2.2 维持期药物

| 原则 | 说明 |
|------|------|
| **有效剂量** | 维持急性期有效剂量 |
| **不减量** | 维持期不减量（除非副作用） |
| **心理治疗维持** | 每月1-2次巩固；维持暴露 |
| **逐渐减停** | 至少6-12月稳定后考虑；极缓慢（数周-数月） |

## 三、复发预警信号

### 3.1 三级预警系统

| 级别 | 信号 | 行动 |
|------|------|------|
| **🟢 绿色** | 偶发担忧；SUDs 1-3；功能正常 | 自我监控；维持日常练习 |
| **🟡 黄色** | 焦虑增加；SUDs 4-6；轻微功能影响；睡眠变差 | 增加应对策略；联系治疗师；强化暴露 |
| **🔴 红色** | 频繁惊恐/严重回避；SUDs 7-10；功能明显受损；安全行为回归 | 紧急联系医生；恢复药物/增加剂量；密集治疗 |

### 3.2 复发高危因素

| 因素 | 说明 |
|------|------|
| **过早停药** | 未达维持期停药 |
| **残留症状** | 治疗不彻底 |
| **重大生活应激** | 丧亲、离婚、失业、搬迁 |
| **共病未处理** | 抑郁、物质使用 |
| **缺乏应对技能** | 未掌握CBT技能 |
| **社会支持丧失** | 孤立 |
| **躯体疾病** | 新发躯体问题 |

## 五、长期管理策略

### 5.1 阶梯维持模型

| 阶段 | 时间 | 频率 |
|------|------|------|
| **密集维持** | 缓解后1-3月 | 每周/每2周 |
| **标准维持** | 3-12月 | 每月 |
| **自主维持** | >12月 | 每3月或按需 |
| **复训触发** | 预警时 | 密集暴露复训 |

### 5.2 复发后应对

| 情况 | 策略 |
|------|------|
| **轻度复发** | 自我CBT技能；增加暴露；联系治疗师 |
| **中度复发** | 恢复药物（如已停）；增加治疗频率 |
| **重度复发** | 精神科评估；恢复急性期治疗强度；考虑住院 |
