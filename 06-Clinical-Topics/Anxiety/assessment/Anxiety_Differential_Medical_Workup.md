# 焦虑障碍躯体疾病鉴别评估 | Differential Medical Workup for Anxiety Disorders

> **目标**：建立精神科标准的躯体疾病所致焦虑的鉴别评估流程，涵盖心血管、内分泌、神经、呼吸、血液、药物/物质等系统。
> **原则**："先排除躯体，再诊断精神" — 尤其对首次发作、年龄较大、无既往焦虑史的患者。

---

## 一、初诊躯体评估决策树

```
患者主诉焦虑/惊恐/心悸/气短
    │
    ├── 年龄>40岁首次发作
    │       → 全面躯体检查 + ECG + 甲功 + 血糖
    │
    ├── 以躯体症状为主诉（心悸、胸痛、气短、头晕）
    │       → 针对性躯体检查
    │
    ├── 症状呈阵发性、突发突止
    │       → 排除心律失常、嗜铬细胞瘤、低血糖、癫痫
    │
    ├── 伴体重变化、怕热/怕冷、震颤
    │       → 甲功检查
    │
    ├── 伴神经系统症状（头痛、视觉改变、晕厥）
    │       → 神经系统检查 + 脑影像学
    │
    ├── 近期用药或物质使用变化
    │       → 毒物/药物筛查
    │
    └── 年轻、慢性、有焦虑史、典型心理触发
            → 重点精神科评估；基础实验室检查
```

## 二、心血管系统

### 2.1 心律失常

| 类型 | 症状 | 检查 | 与焦虑的鉴别 |
|------|------|------|-------------|
| **室上性心动过速（SVT）** | 突发心悸、胸痛、气短 | ECG、24h Holter | 发作时ECG确诊；焦虑时ECG通常正常窦速 |
| **房颤** | 心悸、气短、乏力 | ECG | 脉搏绝对不齐 |
| **频发室性早搏（PVC）** | 心悸、"心跳漏跳" | ECG、Holter | 动态ECG记录PVC |
| **长QT综合征** | 心悸、晕厥 | ECG（QTc>470ms女/450ms男）| 家族史；运动诱发 |
| **Brugada综合征** | 晕厥、夜间猝死 | ECG（V1-V3抬高）| 家族史；东南亚男性多见 |

### 2.2 结构性心脏病

| 疾病 | 症状 | 检查 |
|------|------|------|
| **二尖瓣脱垂** | 心悸、胸痛、气短 | 心脏超声 |
| **肥厚型心肌病** | 胸痛、气短、晕厥 | 心脏超声、ECG |
| **冠心病** | 胸痛（活动诱发） | ECG、运动试验、冠脉CTA |

### 2.3 心脏焦虑综合征（Da Costa综合征/心脏神经症）

- 反复心悸、胸痛，心脏检查正常
- 对正常心脏感觉过度敏感
- 常见于焦虑障碍患者

## 三、内分泌系统

### 3.1 甲状腺功能亢进

| 症状 | 焦虑重叠 | 鉴别要点 |
|------|---------|---------|
| 心悸、震颤、焦虑 | 高度重叠 | 体重下降、怕热、多汗、甲状腺肿大、突眼 |
| 易激惹、失眠 | 高度重叠 | 腹泻、月经紊乱 |
| **检查** | | TSH降低、FT3/FT4升高、TRAb（Graves） |

### 3.2 甲状腺功能减退

| 症状 | 焦虑重叠 | 鉴别要点 |
|------|---------|---------|
| 疲劳、抑郁 | 部分重叠 | 体重增加、怕冷、便秘、皮肤干燥、声音嘶哑 |
| **检查** | | TSH升高、FT4降低 |

### 3.3 嗜铬细胞瘤

| 特征 | 说明 |
|------|------|
| **症状** | 阵发性高血压、心悸、出汗、头痛、焦虑、面色苍白 |
| **与惊恐鉴别** | 伴血压升高（>180/110）；面色苍白（惊恐常面红）；发作后极度疲乏 |
| **检查** | 24h尿儿茶酚胺、血/尿甲氧基肾上腺素、肾上腺CT/MRI |
| **重要性** | 漏诊可致命（高血压危象、心肌病） |

### 3.4 低血糖

| 特征 | 说明 |
|------|------|
| **症状** | 心悸、出汗、震颤、焦虑、饥饿感、头晕 |
| **鉴别** | 进食后迅速缓解；糖尿病患者胰岛素/磺脲类药物过量 |
| **检查** | 发作时血糖<3.9mmol/L |

### 3.5 肾上腺功能不全

| 特征 | 说明 |
|------|------|
| **症状** | 疲劳、焦虑、低血压、皮肤色素沉着 |
| **检查** | 晨起皮质醇、ACTH刺激试验 |

## 四、神经系统

### 4.1 癫痫（特别是颞叶癫痫）

| 特征 | 说明 |
|------|------|
| **症状** | 发作性恐惧、人格解体、现实解体、似曾相识感、自主神经症状 |
| **与惊恐鉴别** | 发作无明确心理触发；持续时间更短（数秒-2分钟）；发作后意识模糊；EEG异常 |
| **检查** | EEG（发作时最佳）、脑MRI（海马硬化） |

### 4.2 前庭功能障碍

| 疾病 | 症状 | 检查 |
|------|------|------|
| **良性阵发性位置性眩晕（BPPV）** | 位置性眩晕、恶心 | Dix-Hallpike试验 |
| **梅尼埃病** | 眩晕、耳鸣、听力下降 | 听力检查 |
| **前庭神经炎** | 急性眩晕、恶心 | 前庭功能检查 |

### 4.3 偏头痛

| 特征 | 说明 |
|------|------|
| **前驱症状** | 焦虑、易激惹、打哈欠（焦虑样表现） |
| **先兆** | 视觉/感觉先兆 |

### 4.4 脑肿瘤/脑血管病

| 警示症状 | 检查 |
|---------|------|
| 新发头痛、视觉改变、局灶性神经体征、认知改变 | 脑MRI/CT |

## 五、呼吸系统

### 5.1 哮喘

| 特征 | 说明 |
|------|------|
| **症状** | 气短、喘息、胸闷、咳嗽 |
| **与焦虑鉴别** | 肺功能检查（FEV1降低）；支气管扩张试验阳性；发作与过敏原/运动相关 |
| **共病** | 哮喘与焦虑高度共病；焦虑可诱发哮喘发作 |

### 5.2 COPD

| 特征 | 说明 |
|------|------|
| **症状** | 慢性气短、咳嗽、咳痰 |
| **检查** | 肺功能（FEV1/FVC<0.7） |

### 5.3 肺栓塞

| 警示症状 | 检查 |
|---------|------|
| 突发呼吸困难、胸痛、咯血、心动过速 | D-二聚体、CT肺动脉造影 |

### 5.4 过度换气综合征

| 特征 | 说明 |
|------|------|
| **机制** | 呼吸频率过快 → CO2降低 → 呼吸性碱中毒 |
| **症状** | 头晕、手足麻木/刺痛、口周麻木、胸闷、手足搐搦 |
| **检查** | 发作时血气分析（pH升高、PaCO2降低） |
| **治疗** | 纸袋呼吸（谨慎）、腹式呼吸再训练 |

## 六、血液系统

### 6.1 贫血

| 类型 | 症状 | 检查 |
|------|------|------|
| **缺铁性贫血** | 疲劳、心悸、气短、头晕 | 血常规（Hb降低、MCV降低）、铁蛋白 |
| **B12缺乏** | 疲劳、认知下降、感觉异常、步态不稳 | 血清B12、同型半胱氨酸 |
| **叶酸缺乏** | 疲劳、情绪低落 | 血清叶酸 |

## 七、药物与物质所致焦虑

### 7.1 药物

| 药物类别 | 代表药物 | 机制 |
|---------|---------|------|
| **兴奋剂** | 哌醋甲酯、安非他酮、莫达非尼 | 增加NE/DA |
| **拟交感神经药** | 伪麻黄碱、去氧肾上腺素 | 激活交感 |
| **茶碱** | 氨茶碱 | 腺苷拮抗 |
| **皮质类固醇** | 泼尼松 | HPA轴激活 |
| **甲状腺素** | 左旋甲状腺素 | 甲状腺功能亢进 |
| **支气管扩张剂** | 沙丁胺醇 | β2受体激活 |
| **多巴胺激动剂** | 普拉克索 | DA受体激活 |
| **某些抗抑郁药** | SSRIs/SNRIs初期、安非他酮 | 5-HT/NE/DA增加 |
| **抗精神病药** | 某些药物（如阿立哌唑初期）| DA激动 |

### 7.2 物质

| 物质 | 急性效应 | 戒断效应 |
|------|---------|---------|
| **咖啡因** | 心悸、焦虑、震颤 | 头痛、疲劳、焦虑 |
| **酒精** | 初期镇静，后期焦虑 | 焦虑、震颤、失眠、癫痫 |
| **尼古丁** | 短暂镇静 | 焦虑、易激惹、注意力不集中 |
| **大麻** | 初期放松，后期焦虑/惊恐 | 焦虑、失眠、食欲下降 |
| **可卡因/冰毒** | 欣快 → 崩溃后焦虑/抑郁 | 严重焦虑、抑郁、渴求 |
| **苯二氮卓类** | 镇静 | 严重焦虑反跳、癫痫 |

## 八、推荐的实验室检查方案

### 8.1 基础检查（所有初诊患者）

| 检查 | 目的 |
|------|------|
| **血常规** | 贫血、感染 |
| **TSH + FT4** | 甲功 |
| **空腹血糖/HbA1c** | 糖尿病/低血糖 |
| **电解质** | 低钾/低钠 |
| **肝肾功能** | 代谢状态；用药安全 |
| **ECG** | 心律失常、QTc |

### 8.2 针对性检查（根据症状）

| 症状 | 检查 |
|------|------|
| **阵发性心悸+高血压** | 24h Holter + 24h尿儿茶酚胺/甲氧基肾上腺素 |
| **发作性恐惧+意识改变** | EEG + 脑MRI |
| **慢性气短** | 肺功能检查 |
| **眩晕** | 前庭功能检查 |
| **胸痛** | 心肌酶、冠脉CTA、心脏超声 |
| **体重下降+震颤** | 甲功全套（TSH/FT3/FT4/TRAb） |
| **怀疑物质所致** | 尿液毒理筛查 |
