# 焦虑障碍精神科标准评估组合 | Psychiatric Assessment Battery for Anxiety Disorders

> **目标**：建立符合精神科临床标准的焦虑障碍评估体系，涵盖结构化访谈、症状量表、功能评估、躯体筛查与疗效监测。

---

## 一、结构化临床访谈

### 1.1 初诊评估清单

| 模块 | 工具/内容 | 目的 |
|------|----------|------|
| **轴I障碍筛查** | SCID-5-CV 或 MINI 7.0 | 确认焦虑障碍诊断；排除抑郁、双相、精神病性障碍 |
| **焦虑亚型评估** | ADIS-5（焦虑障碍访谈 Schedule） | 金标准；详细评估各焦虑亚型 |
| **惊恐发作特征** | 临床访谈（13项症状核查） | 确认惊恐发作；鉴别躯体疾病 |
| **社交焦虑评估** | 社交焦虑情境清单 | 识别恐惧情境（表演型 vs 广泛型） |
| **恐惧对象评估** | 恐惧对象与回避程度 | 特定恐惧症亚型分类 |
| **躯体疾病筛查** | 病史采集 + 体格检查 | 排除甲亢、心律失常、癫痫等 |
| **物质使用筛查** | AUDIT + DAST-10 | 排除物质所致焦虑 |
| **功能评估** | SDS + 工作/社交/家庭功能 | 功能损害程度 |
| **自杀风险评估** | C-SSRS | 焦虑患者自杀风险（尤其伴抑郁） |

### 1.2 必需躯体检查与实验室

| 检查 | 目的 | 适用 |
|------|------|------|
| **甲状腺功能** | 排除甲亢 | 所有伴心悸/震颤/体重下降者 |
| **心电图** | 排除心律失常 | 所有伴心悸/胸痛者 |
| **血常规** | 排除贫血/感染 | 伴疲劳者 |
| **电解质/血糖** | 排除低钾/低血糖 | 伴虚弱/出汗者 |
| **24h Holter** | 捕捉阵发性心律失常 | ECG正常但症状频发者 |
| **24h尿儿茶酚胺/甲氧基肾上腺素** | 排除嗜铬细胞瘤 | 阵发性高血压+心悸+出汗 |
| **脑电图** | 排除颞叶癫痫 | 发作性恐惧/人格解体 |
| **肺功能** | 排除哮喘/COPD | 伴气短/窒息感者 |

## 二、核心评估量表

### 2.1 广泛性焦虑

#### GAD-7（广泛性焦虑量表-7项）

| 项目 | 内容 |
|------|------|
| **用途** | GAD筛查与严重程度追踪 |
| **题数** | 7 |
| **评分** | 0-3四级评分，总分0-21 |
| **临界值** | 5-9轻度；10-14中度；15-21重度 |
| **MCID** | 下降≥4分 |

#### PSWQ（宾州担忧问卷）

| 项目 | 内容 |
|------|------|
| **用途** | 担忧特质评估；区分GAD与正常担忧 |
| **题数** | 16 |
| **特点** | 对GAD特异性高； worry 的强度和频率 |

### 2.2 惊恐障碍

#### PDSS（惊恐障碍严重程度量表）

| 项目 | 内容 |
|------|------|
| **用途** | 惊恐障碍严重程度（临床医生评定） |
| **题数** | 7（惊恐频率、痛苦、 anticipatory anxiety、回避、功能） |
| **评分** | 每项0-4分 |

#### ACQ（焦虑控制问卷）

| 项目 | 内容 |
|------|------|
| **用途** | 评估对焦虑的控制感 |
| **特点** | 控制感低预测治疗反应差 |

#### BSQ（身体感觉问卷）

| 项目 | 内容 |
|------|------|
| **用途** | 评估对身体感觉的恐惧（内感受恐惧） |
| **特点** | 高BSQ预测惊恐障碍 |

### 2.3 社交焦虑

#### LSAS（Liebowitz社交焦虑量表）

| 项目 | 内容 |
|------|------|
| **用途** | 社交焦虑金标准（临床医生评定） |
| **题数** | 24（11项社交互动 + 13项表演情境） |
| **评分** | 恐惧（0-3）+ 回避（0-3），总分0-144 |
| **临界值** | ≥60中度；≥80重度 |

#### SIAS（社交互动焦虑量表）+ SPS（社交恐惧量表）

| 项目 | 内容 |
|------|------|
| **SIAS** | 20项；社交互动焦虑 |
| **SPS** | 20项；被观察/审视恐惧 |

### 2.4 特定恐惧

#### FSS-III（恐惧调查量表-III）

| 项目 | 内容 |
|------|------|
| **用途** | 筛查多种恐惧对象 |
| **题数** | 54项 |

### 2.5 综合焦虑量表

| 量表 | 用途 | 特点 |
|------|------|------|
| **BAI（贝克焦虑量表）** | 焦虑严重程度 | 对躯体症状敏感；21项 |
| **STAI（状态-特质焦虑量表）** | 区分状态焦虑与特质焦虑 | 研究常用；40项 |
| **HAMA（汉密尔顿焦虑量表）** | 他评焦虑严重程度 | 14项；精神科临床 |
| **CGI-S/I** | 临床总体印象 | 严重程度与改善 |

### 2.6 功能与生活质量

| 量表 | 用途 |
|------|------|
| **SDS** | 工作/社交/家庭功能 |
| **QOLIE** | 生活质量 |
| **SF-36** | 健康相关生活质量 |

## 三、整合评估时间表

| 时间点 | 评估内容 | 目的 |
|--------|---------|------|
| **初诊（T0）** | ADIS/SCID + GAD-7/BAI/PDSS/LSAS（根据亚型）+ SDS + C-SSRS + 躯体检查 | 全面基线；诊断确认 |
| **治疗2周** | GAD-7/BAI + 不良反应 | 早期适应 |
| **治疗4周** | 核心量表 + SDS | 疗效评估 |
| **治疗8-12周** | 全面评估 + CGI-I | 急性期终点 |
| **维持期** | 每月→每3月核心量表 | 维持监测 |
