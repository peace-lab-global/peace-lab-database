# 焦虑障碍危机干预与安全 | Anxiety Crisis Intervention and Safety

> **目标**：建立精神科标准的焦虑障碍危机评估与干预体系，涵盖急性惊恐发作处理、严重回避导致的功能丧失、自杀风险评估及安全计划。

---

## 一、急性惊恐发作处理

### 1.1 患者自助技术

| 技术 | 步骤 | 原理 |
|------|------|------|
| **呼吸再训练** | 缓慢腹式呼吸：吸气4秒→屏息2秒→呼气6秒；持续5-10分钟 | 纠正过度换气（降低CO2）；激活副交感神经系统 |
| **5-4-3-2-1 Grounding** | 命名5个看到的物体→4种触摸→3种声音→2种气味→1种味道 | 将注意力从内部躯体感觉转移到外部现实 |
| **认知标签** | "这是惊恐发作，不会伤害我，会过去的" | 减少灾难化解释；增加控制感 |
| **温度刺激** | 冷水洗脸或握冰块 | 激活潜水反射；降低生理唤起 |
| **纸袋呼吸（谨慎使用）** | 仅在确认过度换气时使用；缓慢呼吸入纸袋 | 恢复CO2水平；**不适用于COPD/心脏病** |

### 1.2 临床处理

| 情况 | 处理 |
|------|------|
| **首次惊恐发作（未诊断）** | 排除躯体疾病（ECG、甲功、血糖）；教育；CBT转介 |
| **已知惊恐障碍的急性发作** | 确认无新躯体问题；呼吸训练指导；评估当前治疗方案是否充分 |
| **频繁发作（>1次/周）** | 药物剂量调整；加强CBT；评估共病（抑郁、物质使用） |
| **发作持续时间>30分钟** | 排除躯体疾病；评估是否共病其他障碍 |

### 1.3 需要急诊的情况

| 情况 | 行动 |
|------|------|
| 首次严重胸痛/心悸（未排除心脏问题） | ECG、心肌酶、心脏评估 |
| 发作中意识丧失 | 排除癫痫、心律失常、低血糖 |
| 严重过度换气导致手足搐搦/痉挛 | 呼吸训练；必要时短期镇静 |
| 发作后持续严重躯体症状 | 全面医学评估 |

## 二、严重回避与功能丧失

### 2.1 广场恐惧症的严重回避

| 表现 | 干预 |
|------|------|
| **完全不能离家** | 住院治疗或密集门诊暴露；从家门口开始逐级暴露 |
| **只能在家附近活动** | 设定每日扩展活动范围的暴露作业 |
| **需要陪伴才能外出** | 渐进减少陪伴依赖；从短途独自外出开始 |
| **无法工作/上学** | 病假期间治疗计划；与雇主/学校协调逐步复工 |

### 2.2 社交焦虑的严重回避

| 表现 | 干预 |
|------|------|
| **完全回避社交** | 密集CBT；从最低焦虑情境开始暴露 |
| **无法工作（需人际互动）** | 职业康复；角色扮演；渐进社交暴露 |
| **拒绝上学（儿童）** | 学校回避干预；家长教育；学校协调 |

## 三、自杀风险评估

### 3.1 焦虑障碍患者的自杀风险

| 因素 | 说明 |
|------|------|
| **单纯焦虑** | 自杀风险低于抑郁，但并非零风险 |
| **焦虑+抑郁共病** | 风险显著增加；尤其GAD+MDD |
| **惊恐障碍** | 自杀意念发生率约20%；与抑郁共病时更高 |
| **社交焦虑** | 孤立、孤独感增加风险；物质使用作为应对 |
| **PTSD** | 高自杀风险；需常规评估 |
| **OCD** | 强迫性自杀意念（ego-dystonic）vs 真实自杀意图鉴别 |

### 3.2 评估要点

- 每次评估必须包含C-SSRS
- 特别关注：绝望感、抑郁共病、物质使用、近期应激、社会隔离
- 焦虑患者的自杀常常与"无法忍受的痛苦"而非"想要死去"相关

## 四、安全计划

### 4.1 焦虑障碍专用安全计划调整

| 标准SPI步骤 | 焦虑调整 |
|------------|---------|
| **预警信号** | 增加焦虑特异性信号（心率加快、过度换气、灾难化思维、回避冲动） |
| **内部应对** | 呼吸训练、grounding、认知标签、正念练习 |
| **分散活动** | 散步、运动、冷水刺激、正念活动 |
| **社交支持** | 了解患者焦虑的亲友（避免说"放松"等无效安慰） |
| **专业人士** | 治疗师、精神科医生、危机热线 |
