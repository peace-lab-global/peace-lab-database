# 焦虑障碍药物戒断与停药管理 | Medication Discontinuation and Withdrawal in Anxiety Disorders

> **目标**：建立精神科标准的焦虑障碍药物减停方案，涵盖苯二氮卓类、SSRIs/SNRIs、普瑞巴林的戒断症状识别、减停协议与复发预防。
> **标准来源**：APA Practice Guidelines（2019）、NICE Guidelines（2019）、Lader et al. BZD withdrawal protocols。

---

## 一、苯二氮卓类（BZD）戒断

### 1.1 BZD依赖与戒断流行病学

- 长期使用（>4周）后约30-50%出现戒断症状
- 高剂量、短半衰期、突然停药风险最高
- 戒断症状可持续数周至数月

### 1.2 BZD戒断症状谱

| 时间 | 症状 |
|------|------|
| **急性期（1-3天）** | 焦虑反跳、失眠、易激惹、震颤、出汗、心悸 |
| **早期（3-14天）** | 恶心、体重下降、感觉过敏、知觉扭曲、肌肉痉挛 |
| **高峰期（1-4周）** | 严重焦虑、惊恐发作、抑郁、激越、认知障碍 |
| **迁延期（数周-数月）** | 持续焦虑、失眠、感觉异常、记忆障碍 |
| **严重（罕见）** | 癫痫发作、谵妄、精神病性症状 |

### 1.3 BZD减停协议

**核心原则**：
- **不突然停药**（除非极短使用时间）
- **缓慢递减**（数周至数月）
- **个体化**（根据剂量、使用时间、药物种类、患者状态）

**标准减停方案**：

| 阶段 | 策略 |
|------|------|
| **评估期** | 评估当前剂量、使用时间、既往戒断史、共病、社会支持 |
| **稳定期** | 先稳定剂量；处理多BZD使用（先减短效，保留长效） |
| **转换期（可选）** | 短效BZD → 长效BZD（如阿普唑仑→地西泮/氯硝西泮） |
| **减停期** | 每1-2周减10-25% |
| **平台期** | 出现明显症状时暂停减量，等待稳定 |
| **完成期** | 最后低剂量阶段可能需要更长时间 |

**具体减停速度**：

| 使用时间 | 建议减停时间 |
|---------|-------------|
| < 4周 | 1-2周 |
| 4-12周 | 2-4周 |
| 3-6月 | 1-3月 |
| 6-12月 | 2-4月 |
| > 1年 | 3-12月或更长 |

### 1.4 短效→长效转换方案

| 短效药物 | 等效长效药物 | 转换比例 |
|---------|------------|---------|
| 阿普唑仑 0.5mg | 地西泮 10mg | 1:20 |
| 劳拉西泮 1mg | 地西泮 10mg | 1:10 |
| 奥沙西泮 15mg | 地西泮 10mg | 1.5:1 |

### 1.5 BZD戒断辅助治疗

| 辅助药物 | 剂量 | 证据 |
|---------|------|------|
| **卡马西平** | 200-400mg/d | B级（预防癫痫、减轻戒断） |
| **丙戊酸钠** | 250-500mg/d | B级 |
| **普瑞巴林** | 150-300mg/d | B级（减轻焦虑反跳） |
| **丁螺环酮** | 15-30mg/d | C级 |
| **曲唑酮** | 50-150mg睡前 | C级（失眠） |
| **β受体阻滞剂** | 普萘洛尔10-40mg/d | C级（躯体症状） |

## 二、SSRIs/SNRIs停药症状

### 2.1 停药综合征（Discontinuation Syndrome）

**发生原因**：
- 5-HT受体急性适应性改变
- 胆碱能反跳（帕罗西汀）
- 半衰期短药物更易发生

**常见症状**：

| 症状 | 发生率 | 说明 |
|------|--------|------|
| **头晕/眩晕** | 高 | 感觉异常、电击感（"脑电击"） |
| **感觉异常** | 高 | 肢体麻木/刺痛 |
| **焦虑反跳** | 高 | 焦虑症状加重 |
| **失眠** | 高 | 睡眠障碍加重 |
| **流感样症状** | 中 | 疲劳、肌肉痛、寒战 |
| **恶心** | 中 | 胃肠道症状 |
| **情绪波动/易激惹** | 中 | 情绪不稳定 |
| **头痛** | 中 | |
| **出汗** | 低-中 | |

### 2.2 高风险药物

| 药物 | 风险 | 原因 |
|------|------|------|
| **帕罗西汀** | 最高 | 短半衰期 + 抗胆碱能作用 |
| **文拉法辛** | 高 | 短半衰期 |
| **度洛西汀** | 高 | 短半衰期 |
| **舍曲林** | 中 | 中等半衰期 |
| **艾司西酞普兰** | 中 | 中等半衰期 |
| **氟西汀** | 最低 | 长半衰期（自洗脱效应） |
| **氟伏沙明** | 中 | 短半衰期 |

### 2.3 减停方案

| 药物 | 减停策略 |
|------|---------|
| **帕罗西汀** | 极缓慢：每1-2周减5mg（从20mg开始） |
| **文拉法辛XR** | 每1-2周减37.5mg |
| **度洛西汀** | 每1-2周减30mg |
| **舍曲林** | 每2-4周减25-50mg |
| **艾司西酞普兰** | 每2-4周减5mg |
| **氟西汀** | 可较快：每2-4周减10-20mg |

### 2.4 停药症状 vs 复发鉴别

| 特征 | 停药症状 | 复发 |
|------|---------|------|
| **起病时间** | 停药后1-3天 | 数周至数月 |
| **持续时间** | 通常1-2周（帕罗西汀/文拉法辛可更长） | 持续 ≥2周 |
| **症状性质** | 躯体症状为主（头晕、感觉异常） | 核心焦虑症状 |
| **对药物反应** | 重新服药后24-72h缓解 | 需2-4周缓解 |
| **时间进程** | 逐渐减轻 | 持续或加重 |

## 三、普瑞巴林停药

| 特征 | 说明 |
|------|------|
| **戒断症状** | 失眠、恶心、头痛、焦虑、出汗、腹泻 |
| **减停方案** | 至少1周减量；高剂量需更慢 |
| **注意** | 有滥用潜力；长期使用需监测 |

---

## 四、停药前评估与决策

### 4.1 停药适应症

| 适合停药 | 不适合停药 |
|---------|-----------|
| 症状完全缓解 ≥6-12月 | 症状未完全缓解 |
| 功能完全恢复 | 功能仍有损害 |
| 有良好应对技能 | 缺乏应对技能 |
| 无重大生活应激 | 近期重大应激 |
| 首次发作 | 多次复发 |
| 无共病 | 严重共病 |
| 患者主动要求 | 患者对停药恐惧 |

### 4.2 停药前准备

| 准备 | 内容 |
|------|------|
| **心理教育** | 告知停药症状与复发的区别 |
| **应对技能复习** | 呼吸训练、认知重构、暴露技术 |
| **安全计划更新** | 复发预警信号、应对策略 |
| **随访安排** | 停药后1、2、4、8、12周随访 |
| **家属教育** | 识别复发信号 |
