# 抑郁症自杀风险评估与管理 | Suicide Risk Assessment and Management in Depression

> **目标**：建立精神科标准的自杀风险评估体系、安全计划框架、危机干预流程与长期风险管理策略。
> **原则**：自杀是可预防的；系统性评估、前瞻性计划和协作管理是降低自杀风险的核心。

---

## 一、自杀风险评估框架

### 1.1 评估原则

| 原则 | 说明 |
|------|------|
| **直接询问** | 直接、清晰、非评判地询问自杀意念不会增加自杀风险；反而传递关怀 |
| **逐层深入** | 从被动意念→主动意念→计划→准备→意图→既往企图 |
| **动态评估** | 自杀风险是动态变化的；每次接触都需重新评估 |
| **综合信息** | 结合患者自述、行为观察、家属报告、病史记录 |
| **文档记录** | 所有评估必须详细记录 |

### 1.2 C-SSRS 筛查与评估

**筛查版（Screener）**：

| 问题 | 回答 | 意义 |
|------|------|------|
| 1. 您是否希望自己死去？ | 是/否 | 被动死亡愿望 |
| 2. 您是否有过要杀死自己的实际想法？ | 是/否 | 主动自杀意念 |
| 3. 您是否想过如何自杀或做过准备？ | 是/否 | 计划/准备 |
| 4. 您是否曾试图自杀？ | 是/否 | 既往企图 |

**完整版评估域**：

| 域 | 内容 | 风险分层 |
|----|------|---------|
| **自杀意念强度** | 频率、持续时间、可控性、 deterrents（阻止因素） | 频率越高、控制越差 = 风险越高 |
| **自杀意念具体性** | 是否有具体方法 | 有方法 > 无方法 |
| **自杀计划** | 是否有详细计划 | 有计划 > 无计划 |
| **自杀意图** | 是否有真实实施的意图 | 有意图 = 极高风险 |
| **自杀行为** | 实际未遂行为 | 既往企图是最重要的风险因素之一 |

### 1.3 静态风险因素（相对稳定）

| 因素 | 说明 |
|------|------|
| **性别** | 男性自杀死亡率更高；女性自杀_attempt更多 |
| **年龄** | 青年（15-24）和老年（>65）风险较高 |
| **既往自杀_attempt** | 最重要预测因子之一；既往attempt史增加未来风险10-20倍 |
| **精神障碍诊断** | 抑郁障碍、双相障碍、精神分裂症、物质使用障碍、BPD |
| **躯体疾病** | 慢性疼痛、癌症、HIV、神经系统疾病 |
| **家族史** | 自杀家族史增加独立风险 |
| **童年虐待/创伤** | 增加长期自杀风险 |
| ** LGBTQ+ 身份** | 歧视、家庭排斥增加风险（尤其青少年） |

### 1.4 动态风险因素（可变化）

| 因素 | 说明 |
|------|------|
| **当前自杀意念** | 频率、强度、具体性 |
| **绝望感** | Beck绝望量表；对未来无期望 |
| **急性应激事件** | 丧亲、分手、失业、经济困难、法律问题 |
| **急性症状恶化** | 抑郁/焦虑/精神病性症状加剧 |
| **物质使用** | 急性醉酒或戒断降低抑制力 |
| **社会隔离** | 缺乏社会支持；独居 |
| **近期出院** | 出院后首周-首月风险最高 |
| **治疗变化** | 药物改变、治疗师更换、终止治疗 |
| **接触自杀** | 亲友自杀、媒体报道 |

### 1.5 保护因素

| 因素 | 如何利用 |
|------|---------|
| **社会支持** | 激活支持网络；家庭参与 |
| **对未来的希望** | 认知行为技术；目标设定 |
| **责任感（子女、宠物）** | 强化责任联结 |
| **治疗联盟** | 建立稳固的治疗关系 |
| **文化/宗教信仰** | 尊重并利用（如宗教反对自杀） |
| **问题解决能力** | 技能培训；增强控制感 |
| **安全环境** |  lethal means restriction |

## 二、风险分层与临床决策

### 2.1 三级风险分层

| 层级 | 特征 | 行动 | 环境 |
|------|------|------|------|
| **🟢 低风险** | 无自杀意念；或仅有被动意念（"希望睡着不醒"），无计划、无意图；有强烈保护因素 | 门诊治疗；安全计划；常规随访 | 家中 |
| **🟡 中风险** | 有主动自杀意念；可能有模糊方法但无具体计划；无明确意图；或有既往attempt史目前稳定；或近期重大应激 | 加强门诊（每周1-2次）；精神科评估；安全计划；家属知情； lethal means restriction | 家中（需家属支持） |
| **🔴 高风险** | 有具体自杀计划；有实施意图；或近期自杀_attempt；或命令性幻听命令自杀；或严重物质中毒 | 住院评估；考虑自愿/非自愿住院；24小时陪伴；移除 lethal means；精神科急诊 | 医院（精神科病房/急诊观察） |

### 2.2 住院指征

| 指征 | 说明 |
|------|------|
| 主动自杀计划+意图 | 最高优先级 |
| 近期自杀_attempt（≤1月） | 需要密切监护 |
| 命令性幻听命令自杀 | 精神病性症状 |
| 严重物质中毒+自杀意念 | 判断力受损 |
| 无安全计划执行能力 | 认知障碍、严重解离 |
| 无法保证安全环境 | 独居、无家属、 lethal means 无法控制 |
| 中风险但 outpatient 资源不足 | 门诊无法提供足够支持 |

### 3.2 Lethal Means Restriction（致命手段限制）

| 手段 | 限制策略 |
|------|---------|
| **药物** | 家属保管；每日分发；不囤药；多余药物交回药房 |
| **利器** | 移除或锁起刀具、剃刀；使用安全性刀片 |
| ** firearms** | 移出家中；交由他人保管；枪锁 |
| **高处** | 限制天台/高层窗户可达性；安装防护 |
| **窒息** | 限制绳索/电线；注意衣柜杆 |
| **一氧化碳** | 检查车辆；注意封闭空间 |

**证据**： lethal means restriction 是少数被证明可降低自杀死亡率的干预措施之一。

## 四、危机干预协议

### 4.1 门诊危机响应

| 情况 | 即时响应 | 后续 |
|------|---------|------|
| **患者电话中表达自杀意念** | 保持通话；评估即时风险；不单独挂断；联系紧急联系人或急救 | 危机后24小时内面诊 |
| **门诊中患者突然情绪崩溃** | 不单独留患者；评估即时风险；联系家属；必要时呼叫急救 | 记录事件；调整治疗计划 |
| **收到患者自杀信息（短信/邮件）** | 立即联系患者；如无法联系→联系家属/紧急联系人→必要时报警 welfare check | 记录；后续评估 |

### 4.2 住院患者自杀预防

| 措施 | 说明 |
|------|------|
| **15分钟观察** | 高风险患者每15分钟检查一次 |
| **一对一观察** | 极高风险患者24小时专人陪伴 |
| **移除危险物品** | 皮带、鞋带、玻璃、锐器、过多药物 |
| **环境安全** | 防自缢设施；门锁管理；窗户限制 |
| **出院计划** | 出院前安全计划；家属教育；24-48小时内随访预约 |

### 4.3 出院后高危期管理

| 时间点 | 风险 | 措施 |
|--------|------|------|
| **出院后24-72小时** | 最高 | 电话随访；家属确认安全 |
| **出院后1-2周** | 高 | 面诊随访；安全计划复习 |
| **出院后1月** | 中高 | 密集门诊；药物优化；心理社会支持 |
| **出院后3月** | 中 | 常规随访；复发预防 |

## 五、特殊人群自杀风险

### 5.1 青少年

| 特征 | 管理 |
|------|------|
| **冲动性高** | 缩短评估间隔；增加家庭支持 |
| **社交媒体影响** | 评估网络接触；教育家长监控 |
| **NSSI（非自杀性自伤）** | 区分NSSI与自杀attempt；NSSI是自杀风险因素 |
| **LGBTQ+** | 评估家庭接纳度；学校霸凌；提供 affirmative 支持 |
| **法律要求** | 了解当地强制报告法律 |

### 5.2 老年

| 特征 | 管理 |
|------|------|
| ** lethality高** | 老年自杀attempt更常致命（方法更决绝、躯体脆弱） |
| **丧偶** | 评估新近丧偶；男性丧偶后风险极高 |
| **躯体疾病** | 慢性疾病、疼痛、功能丧失 |
| **社会隔离** | 独居、丧友、退休 |
| **认知障碍** | 评估决策能力 |

### 5.3 围产期

| 特征 | 管理 |
|------|------|
| **产后精神病** | 罕见但极高风险；需住院 |
| **伤害婴儿意念** | 区分强迫性伤害意念（常见，不执行）与真实意图 |
| **治疗决策** | 权衡哺乳、药物、心理治疗 |

## 六、长期风险管理

### 6.1 风险管理文档

| 文档 | 内容 |
|------|------|
| **风险评估记录** | 每次评估的C-SSRS结果、风险分层、依据 |
| **安全计划** | 更新的安全计划；患者和家属签名 |
| **危机预案** | 危机联系人、危机热线、应急程序 |
| ** lethal means 评估** | 家中致命手段清单及限制措施 |
| **治疗计划** | 针对自杀风险的治疗调整 |

### 6.2 治疗调整

| 风险因素 | 治疗调整 |
|---------|---------|
| **持续自杀意念** | 考虑ECT（快速起效）；联合抗精神病药（如伴精神病性特征） |
| **绝望感** | CBT针对 hopeless；意义治疗；问题解决训练 |
| **冲动性** | DBT技能训练；心境稳定剂（锂盐、丙戊酸盐有抗自杀证据） |
| **物质使用** | 物质戒断治疗；降低物质所致判断力下降 |
| **社会隔离** | 团体治疗；家庭治疗；同伴支持；案例管理 |

### 6.3 锂盐的自杀预防证据

| 发现 | 来源 |
|------|------|
| 锂盐可降低自杀风险约60-80% | 多项大型观察性研究和Meta分析 |
| 机制可能独立于情绪稳定 | 可能通过降低冲动性和攻击性 |
| 适用于双相和单相抑郁的长期维持 | 维持期血锂0.6-0.8 |
