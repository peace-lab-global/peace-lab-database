# 抑郁症诊断与鉴别诊断 | Depression Diagnosis and Differential

> **目标**：建立精神科标准的抑郁症诊断框架，涵盖 DSM-5-TR、ICD-11 诊断标准、亚型分类、鉴别诊断流程与疑难病例决策树。
> **标准来源**：DSM-5-TR（2022）、ICD-11（2022）、中国抑郁障碍防治指南（2020）。

---

## 一、重性抑郁障碍（MDD）诊断标准

### 1.1 DSM-5-TR 诊断标准 A

> 在同一2周时期内，出现5项（或更多）下列症状，表现出与先前功能相比的变化；其中至少1项是（1）心境抑郁或（2）丧失兴趣或愉悦感。

| 编号 | 症状 | 临床评估要点 |
|------|------|-------------|
| 1 | **心境抑郁** | 主观报告（悲伤、空虚、无望）或他人观察（流泪、表情凝重）；儿童/青少年可能表现为易激惹 |
| 2 | **兴趣/愉悦感丧失（Anhedonia）** | 对几乎所有活动兴趣或愉悦感明显减少；需询问既往爱好 |
| 3 | **体重显著变化或食欲改变** | 非节食情况下一个月内体重变化 > 5%；食欲增加或减少 |
| 4 | **失眠或嗜睡** | 入睡困难、早醒（特征性）、睡眠过多；早醒比入睡困难更具特异性 |
| 5 | **精神运动性激越或迟滞** | 他人可观察到的活动增多/减少；言语/思维迟缓；搓手、踱步等 |
| 6 | **疲劳或精力丧失** | 即使轻微活动也感疲惫；"连刷牙的力气都没有" |
| 7 | **无价值感或过度/不恰当内疚** | 可达到妄想程度（如坚信自己犯了罪）；需与现实的过错区分 |
| 8 | **注意力下降或犹豫不决** | 主诉"脑子像浆糊"；工作效率明显下降 |
| 9 | **反复出现死亡想法、自杀意念或企图** | 从被动想法（"希望睡着不醒"）到主动计划；必须每次都询问 |

**诊断标准 B**：症状引起有临床意义的痛苦，或社交、职业或其他重要功能方面的损害。

**诊断标准 C**：发作不能归因于物质（如药物滥用、药物治疗）或其他躯体疾病（如甲状腺功能减退）的生理效应。

### 1.2 ICD-11 对应编码

| ICD-11 编码 | 诊断名称 | 关键特征 |
|------------|---------|---------|
| 6A70 | 单次抑郁发作 | 首次发作 |
| 6A70.0 | 轻度抑郁发作 | 2-3项症状；功能轻微受损 |
| 6A70.1 | 中度抑郁发作 | 4-5项症状；功能明显受损 |
| 6A70.2 | 重度抑郁发作，不伴精神病性症状 | 多数症状；功能严重受损 |
| 6A70.3 | 重度抑郁发作，伴精神病性症状 | 伴幻觉/妄想（通常与抑郁主题一致） |
| 6A70.4 | 复发性抑郁发作 | ≥2次发作，间隔≥2个月 |
| 6A71 | 恶劣心境（持续性抑郁障碍） | 慢性轻中度症状，≥2年 |
| 6A72 | 混合性抑郁和焦虑障碍 | 抑郁+焦虑症状均明显，但各自都不够诊断标准 |

### 1.3 抑郁严重度分级（临床实用）

| 指标 | 轻度 | 中度 | 重度 | 伴精神病性特征 |
|------|------|------|------|---------------|
| **症状数** | 5-6项 | 7-8项 | 全部9项 | 全部+精神病性 |
| **功能损害** | 轻微 | 明显 | 严重 | 严重 |
| **PHQ-9** | 5-9 | 10-14 | 15-27 | 通常≥20 |
| **HAMD-17** | 8-16 | 17-23 | ≥24 | ≥24 |
| **MADRS** | 7-19 | 20-34 | 35-60 | 35-60 |
| **工作能力** | 基本维持 | 明显下降 | 无法工作 | 无法工作 |
| **自杀风险** | 低-中 | 中 | 高 | 高 |

## 二、抑郁亚型（Specifiers）

### 2.1 DSM-5-TR 标注

| 标注 | 特征 | 治疗意义 |
|------|------|---------|
| **伴焦虑痛苦** | 焦虑症状突出（紧张、不安、担心） | 可能需要SNRI或联合抗焦虑药 |
| **伴混合特征** | 伴轻躁狂/躁狂症状（活动增多、言语增多等） | 警惕双相谱系；避免单用抗抑郁药 |
| **伴忧郁特征** | 快感完全丧失、晨重夜轻、早醒、精神运动性迟滞、显著体重减轻、过度内疚 | 对TCA/ECT反应较好；可能生物学因素更强 |
| **伴非典型特征** | 情绪反应性保留（好事可暂时改善情绪）、体重增加/食欲增加、嗜睡、铅样麻痹、人际拒绝敏感 | 对MAOI/SSRI反应较好；常见于双相谱系 |
| **伴精神病性特征** | 伴幻觉/妄想（通常与抑郁主题一致，如罪恶妄想、虚无妄想、疑病妄想） | 需联合抗精神病药或ECT |
| **伴紧张症** | 紧张症特征（木僵、蜡样屈曲、缄默等） | ECT为首选；高死亡风险 |
| **伴围产期发生** | 孕期或产后4周内发生 | 需考虑哺乳安全性；母婴安全 |
| **伴季节性模式** | 与特定季节（通常是秋冬）相关，连续2年 | 光照治疗有效；预防性光疗 |

### 2.2 忧郁型 vs 非典型型鉴别

| 特征 | 忧郁型 | 非典型型 |
|------|--------|---------|
| **情绪反应性** | 无（对愉快事件无反应） | 有（可暂时改善） |
| **体重** | 明显减轻 | 增加 |
| **食欲** | 减退 | 增加 |
| **睡眠** | 早醒 | 嗜睡 |
| **精神运动性** | 迟滞为主 | 可激越 |
| **铅样麻痹** | 无 | 有（肢体沉重感） |
| **人际拒绝敏感** | 不明显 | 明显 |
| **治疗反应** | TCA、ECT | MAOI、SSRI |

## 三、关键鉴别诊断

### 3.1 与双相障碍的鉴别（Critical）

**鉴别重要性**：约20%被诊断为单相抑郁的患者实际是双相谱系；单用抗抑郁药可能诱发躁狂/快速循环。

| 特征 | 单相抑郁 | 双相障碍 |
|------|---------|---------|
| **发作性质** | 仅抑郁发作 | 抑郁+轻躁狂/躁狂发作 |
| **发病年龄** | 通常25-35岁 | 通常<25岁 |
| **发作突然性** | 通常逐渐起病 | 可能突然起病（尤其是躁狂） |
| **睡眠需求** | 失眠或嗜睡 | 躁狂/轻躁狂期睡眠需求减少且不感疲倦 |
| **精神病性症状** | 仅重度抑郁期可能有（与情绪一致） | 躁狂期常见（与情绪一致或夸大） |
| **家族史** | 抑郁障碍 | 双相障碍、精神病性障碍 |
| **抗抑郁药反应** | 通常有效 | 可能诱发轻躁狂/快速循环 |
| **MDQ筛查** | 阴性 | 阳性（7项中≥7项+功能损害） |

**筛查工具**：
- **MDQ（心境障碍问卷）**：轻躁狂症状筛查
- **HCL-32（轻躁狂症状清单）**：更敏感的轻躁狂筛查
- **临床访谈**：详细询问既往是否有持续≥4天的精力/活动增加期

### 3.2 与持续性抑郁障碍（恶劣心境）的鉴别

| 特征 | 重性抑郁障碍（MDD） | 持续性抑郁障碍（PDD/恶劣心境） |
|------|-------------------|---------------------------|
| **病程** | 发作性，间歇期功能恢复 | 慢性，≥2年（儿童≥1年） |
| **症状严重度** | 发作期严重（≥5项症状） | 持续性轻-中度（<5项MDD症状） |
| **发作标准** | 符合MDD全部标准 | 不符合全部MDD标准；但可叠加MDD发作（"双重抑郁"） |
| **晨重夜轻** | 常见 | 不明显 |
| **治疗反应** | 急性期治疗+维持 | 需长期治疗；认知行为治疗有效 |

### 3.3 与适应障碍的鉴别

| 特征 | MDD | 适应障碍伴抑郁情绪 |
|------|-----|------------------|
| **应激源** | 不一定有 | 可识别的应激事件后3个月内起病 |
| **症状标准** | 符合5项以上MDD标准 | 不符合全部MDD标准 |
| **功能损害** | 显著且广泛 | 通常限于应激相关领域 |
| **病程** | 反复发作 | 通常在应激源消除后6个月内缓解 |
| **自杀风险** | 可高 | 可高（尤其青少年） |

### 3.4 与躯体疾病所致抑郁的鉴别

| 疾病类别 | 常见疾病 | 筛查建议 | 特征性线索 |
|---------|---------|---------|-----------|
| **内分泌** | 甲减、甲亢、库欣、Addison、糖尿病 | TSH、T3/T4、皮质醇、血糖 | 甲减：疲劳、体重增加、便秘、认知迟缓；甲亢：焦虑、易激惹、失眠 |
| **神经系统** | 帕金森、癫痫、卒中、MS、脑肿瘤 | 神经科评估、脑MRI/CT | 帕金森：运动迟缓先于情绪症状；卒中：病灶定位（左前额叶） |
| **感染/免疫** | 流感、EBV、HIV、自身免疫病 | 病毒学、CRP、自身抗体 | 感染后疲劳持续；自身免疫：多系统症状 |
| **肿瘤** | 胰腺癌、脑肿瘤、淋巴瘤 | 针对性检查 | 胰腺癌：抑郁先于躯体症状；脑肿瘤：局灶性神经体征 |
| **营养缺乏** | B12、叶酸、铁、维生素D | 血清学检查 | B12缺乏：认知下降+情绪症状；铁缺乏：疲劳+抑郁 |
| **药物引起** | 糖皮质激素、干扰素、β阻滞剂、口服避孕药 | 用药史回顾 | 时间关联性（用药后数天-数周出现） |

### 3.5 与物质/药物所致抑郁的鉴别

| 物质 | 机制 | 特征 |
|------|------|------|
| **酒精** | 急性抑制、慢性神经毒性、戒断 | 晨间抑郁（ hangover ）、戒断期焦虑抑郁 |
| **苯二氮卓类** | 长期使用的抑郁效应、戒断 | 剂量增加后症状恶化；戒断期抑郁/焦虑 |
| **阿片类** | 急性欣快后的情绪低落、戒断 | 与用药周期同步的情绪波动 |
| **兴奋剂** | 崩溃期（crash）抑郁 | 可卡因/冰毒使用后数小时-数天 |
| **大麻** | 急性焦虑/解离、慢性动机缺乏 | 长期使用后的快感缺失 |

### 3.6 与精神分裂症/分裂情感性障碍的鉴别

| 特征 | MDD伴精神病性特征 | 精神分裂症 | 分裂情感性障碍 |
|------|------------------|-----------|--------------|
| **精神病性症状** | 仅抑郁发作时出现 | 持续存在，与情绪无关 | 与情绪发作同时存在，但情绪缓解后仍持续 ≥2周 |
| **精神病性内容** | 与情绪一致（罪恶、虚无、疑病） | 怪诞、不系统、与情绪无关 | 可变 |
| **情感症状** | 严重抑郁 | 情感淡漠为主 | 抑郁/躁狂发作明显 |
| **病程** | 发作性 | 持续性 | 发作性精神病+情感发作 |

## 四、抑郁与焦虑的鉴别与共存

### 4.1 GAD 与 MDD 的鉴别

| 特征 | GAD | MDD |
|------|-----|-----|
| **核心症状** | 过度担忧 | 情绪低落/兴趣丧失 |
| **担忧内容** | 多种领域（工作、健康、家庭） | 过去失败、自我、未来无望 |
| **躯体症状** | 肌肉紧张、易疲劳、易惊跳、胃肠道 | 食欲/睡眠改变、精神运动性变化 |
| **时间分布** | 持续性 | 发作性 |
| **晨重夜轻** | 无 | 常见 |

**注意**：GAD与MDD共病率>50%；临床上常需同时处理。

### 4.2 其他焦虑障碍的鉴别

| 障碍 | 与抑郁的区分要点 |
|------|----------------|
| **惊恐障碍** | 以突发惊恐发作为主；发作间期担忧再次发作；抑郁为继发 |
| **社交焦虑** | 恐惧限于社交/表现情境；回避社交；抑郁因社交孤立继发 |
| **OCD** | 以强迫思维/行为为主；抑郁因功能损害继发 |
| **PTSD** | 有创伤暴露史；再体验、回避、高警觉；抑郁常见共病 |

## 五、诊断决策树

```
患者主诉情绪低落/兴趣丧失
    │
    ├── 症状持续 < 2周
    │       → 观察/随访；排除适应反应
    │
    ├── 症状由物质或躯体疾病解释
    │       → 物质/躯体疾病所致抑郁 → 治疗原发病
    │
    ├── 有躁狂/轻躁狂史或MDQ阳性
    │       → 双相障碍评估 → 心境稳定剂治疗
    │
    ├── 有精神病性症状，但与情绪不一致或持续存在
    │       → 精神分裂症/分裂情感性障碍评估
    │
    ├── 症状≥5项，≥2周，功能受损
    │       │
    │       ├── 症状数5-6，功能轻微受损
    │       │       → 轻度MDD
    │       │
    │       ├── 症状数7-8，功能明显受损
    │       │       → 中度MDD
    │       │
    │       ├── 症状数9，功能严重受损
    │       │       → 重度MDD
    │       │
    │       └── 伴幻觉/妄想
    │               → 重度MDD伴精神病性特征
    │
    ├── 症状<5项或不符合全部标准，但慢性（≥2年）
    │       → 持续性抑郁障碍（恶劣心境）
    │
    ├── 症状与可识别应激源相关，<6个月
    │       → 适应障碍
    │
    └── 混合抑郁焦虑，各自都不够标准
            → 混合性抑郁和焦虑障碍（ICD-11）
```
