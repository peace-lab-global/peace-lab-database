# 抑郁症共病管理（精神科视角） | Depression Comorbidity Management — Psychiatric Perspective

> **目标**：从精神科临床视角，系统阐述抑郁症与焦虑障碍、PTSD、人格障碍、物质使用障碍、进食障碍、慢性疼痛、睡眠障碍等常见共病的评估与管理策略。
> **原则**：共病是常态而非例外；治疗方案需针对所有共病诊断进行整合调整。

---

## 一、抑郁-焦虑共病

### 1.1 流行病学与临床意义

- 共病率：50-70%（MDD + GAD/惊恐/社交焦虑最常见）
- 共病患者：病程更长、复发率更高、功能损害更重、治疗反应更慢、自杀风险更高

### 1.2 药物选择

| 共病类型 | 首选药物 | 次选 | 避免 |
|---------|---------|------|------|
| **MDD + GAD** | 舍曲林、艾司西酞普兰、文拉法辛、度洛西汀 | 帕罗西汀、米氮平 | 安非他酮（可能加重焦虑） |
| **MDD + 惊恐障碍** | 舍曲林、艾司西酞普兰 | 文拉法辛 | 安非他酮、米氮平（激活） |
| **MDD + 社交焦虑** | 舍曲林、艾司西酞普兰、文拉法辛 | 帕罗西汀 | — |
| **MDD + OCD** | 舍曲林（高剂量）、氟西汀（高剂量）、氯咪帕明 | 艾司西酞普兰、氟伏沙明 | — |
| **MDD + PTSD** | 舍曲林、帕罗西汀 | 文拉法辛、米氮平 | — |

### 1.3 心理治疗整合

| 共病 | 推荐心理治疗 | 整合策略 |
|------|------------|---------|
| MDD + GAD | CBT + MBCT | 担忧时间技术；区分反刍与担忧 |
| MDD + 惊恐 | CBT（暴露+认知重构）+ MBCT | 呼吸控制；内感受暴露 |
| MDD + PTSD | CPT/PE + MBCT | 稳定化优先；创伤知情正念 |
| MDD + OCD | ERP + ACT/MBCT | 暴露分级；将正念用于"不回应强迫" |

## 二、抑郁-PTSD/C-PTSD共病

### 2.1 评估要点

| 评估工具 | 用途 |
|---------|------|
| **PCL-5** | PTSD症状严重程度 |
| **DES/TSC-40** | 解离症状 |
| **LEC-5** | 创伤暴露史 |
| **DDT（解离诊断访谈）** | 解离性障碍诊断 |

### 2.2 治疗顺序

| 情况 | 策略 |
|------|------|
| **活跃PTSD（频繁闪回/噩梦）** | 先进行PTSD稳定化治疗（如CPT稳定化阶段、STAIR）→ 后整合MBCT/CBT |
| **PTSD稳定期** | 可同时进行抑郁和PTSD治疗 |
| **严重解离（DES>30）** | 先进行解离稳定化；避免早期暴露于身体扫描/静默 |

### 2.3 药物选择

| 药物 | PTSD证据 | 抑郁证据 | 共病适用性 |
|------|---------|---------|-----------|
| **舍曲林** | A级（FDA批准） | A级 | 首选 |
| **帕罗西汀** | A级（FDA批准） | A级 | 注意镇静/停药症状 |
| **文拉法辛XR** | A级（FDA批准） | A级 | 注意血压 |
| **米氮平** | B级 | A级 | 失眠/噩梦有帮助 |
| **喹硫平** | B级 | A级（增效） | 严重失眠/激越 |
| **哌唑嗪** | B级（噩梦） | 无 | 辅助用于噩梦 |
| **托吡酯** | C级 | 无 | 辅助用于闪回/情绪不稳 |

## 三、抑郁-人格障碍共病

### 3.1 边缘型人格障碍（BPD）

**患病率**：约30-50%的BPD患者共病MDD

**治疗调整**：

| 方面 | 调整 |
|------|------|
| **首选心理治疗** | DBT为一线； Schema Therapy、MBT |
| **药物** | 无FDA批准BPD药物；对症处理：SSRI（情绪不稳）、喹硫平（激越）、拉莫三嗪（情绪不稳）、ω-3脂肪酸（冲动性） |
| **抑郁治疗** | 标准抗抑郁药对BPD抑郁效果有限；需整合人格障碍治疗 |
| **自杀/自伤** | DBT技能训练；安全计划；24小时危机热线 |
| **治疗联盟** | 预期理想化-贬低转换；设置清晰边界；避免反移情行动化 |

### 3.2 回避型人格障碍（AvPD）

| 调整 | 策略 |
|------|------|
| **社交回避** | 渐进暴露；社交技能训练；团体治疗需谨慎（可能加重回避） |
| **低自尊** | 认知重构；自我慈悲训练；肯定性训练 |
| **药物** | SSRI（社交焦虑症状）；MAOI（严重社交焦虑） |

### 3.3 强迫型人格障碍（OCPD）

| 调整 | 策略 |
|------|------|
| **完美主义** | 针对"全或无"思维；容忍不确定性训练 |
| **控制需求** | 渐进放弃控制；接受"足够好" |
| **治疗阻抗** | 预期对治疗的"完美执行"要求；强调过程而非结果 |

### 3.4 自恋型人格障碍（NPD）

| 调整 | 策略 |
|------|------|
| **脆弱性接触** | 抑郁常因自恋受损触发；谨慎处理羞耻 |
| **共情缺陷** | 不期待团体中的互惠共情；个别治疗为主 |
| **药物** | 标准抗抑郁药；心境稳定剂（情绪不稳） |

## 四、抑郁-物质使用障碍共病

### 4.1 评估

| 工具 | 用途 |
|------|------|
| **AUDIT** | 酒精使用障碍筛查 |
| **DAST-10** | 药物滥用筛查 |
| **TLFB（时间线回溯法）** | 详细物质使用史 |
| **尿液/血液毒理** | 客观验证 |

### 4.2 治疗顺序

| 情况 | 策略 |
|------|------|
| **活跃物质依赖（严重）** | 先物质戒断/稳定化 → 后抑郁治疗 |
| **物质使用但功能维持** | 可同时治疗；整合方案 |
| **物质所致抑郁** | 戒断后观察2-4周；如抑郁持续则按原发性抑郁治疗 |

### 4.3 整合治疗

| 物质 | 抑郁治疗注意 |
|------|-------------|
| **酒精** | 注意肝功；避免高肝毒性药物；戒酒期癫痫风险（避免安非他酮、TCA） |
| **阿片类** | 美沙酮/丁丙诺啡维持可继续；注意药物相互作用 |
| **苯二氮卓类** | 目标减停BZD（抑郁风险因素）；缓慢 taper |
| **兴奋剂** | 崩溃期抑郁；区分原发性 vs 物质所致 |
| **大麻** | 动机缺乏综合征；认知影响 |

### 4.4 药物选择

| 药物 | 证据 | 注意 |
|------|------|------|
| **舍曲林** | 酒精依赖+抑郁：A级 | 相对安全 |
| **艾司西酞普兰** | 酒精依赖+抑郁：A级 | 相对安全 |
| **度洛西汀** | 酒精依赖+抑郁：B级 | 肝功监测 |
| **纳曲酮 + SSRI** | 酒精依赖+抑郁：B级 | 联合降低渴求和抑郁 |
| **安非他酮** | 尼古丁依赖+抑郁：A级 | 癫痫/进食障碍禁用 |

## 五、抑郁-进食障碍共病

### 5.1 评估

| 障碍 | 与抑郁的关系 | 治疗优先级 |
|------|------------|-----------|
| **神经性厌食（AN）** | 抑郁常见共病；营养不良加重情绪症状 | 营养恢复优先；生命体征稳定 |
| **神经性贪食（BN）** | 抑郁与暴食-清除循环相互维持 | 可同时进行；SSRI为首选（氟西汀60mg为BN一线） |
| **暴食障碍（BED）** | 抑郁常因暴食和体重问题维持 | SSRI（舍曲林、氟西汀）；lisdexamfetamine（BED FDA批准） |
| **ARFID** | 非体重相关；抑郁因营养缺乏/社交隔离继发 | 营养康复；个体化方案 |

### 5.2 药物注意

| 注意 | 说明 |
|------|------|
| **安非他酮** | AN/BN禁用（癫痫/进食障碍禁忌） |
| **米氮平** | 可增加食欲（AN可利用；BED需谨慎） |
| **氟西汀高剂量** | BN一线（60mg）；注意QTc |
| **体重监测** | 所有抗抑郁药对体重的影响需讨论 |

## 六、抑郁-慢性疼痛共病

### 6.1 机制

- 共享神经通路（中枢敏化、炎症、HPA轴失调）
- 疼痛导致活动减少、睡眠差、绝望感 → 抑郁
- 抑郁导致疼痛敏感度增加（疼痛阈值降低）
- "疼痛性抑郁"：躯体症状突出的抑郁亚型

### 6.2 药物选择

| 药物 | 疼痛类型 | 抑郁证据 | 推荐 |
|------|---------|---------|------|
| **度洛西汀** | 纤维肌痛、糖尿病神经病变、慢性腰痛 | A级 | 首选 |
| **文拉法辛** | 神经性疼痛、纤维肌痛 | A级 | 首选 |
| **米那普仑** | 纤维肌痛 | A级（纤维肌痛） | 首选 |
| **TCA（阿米替林、去甲替林）** | 多种慢性疼痛 | A级 | 二线（副作用多） |
| **安非他酮** | 疼痛证据弱 | A级 | 不优先用于疼痛 |

### 6.3 非药物整合

| 干预 | 证据 |
|------|------|
| **CBT-Pain** | A级 |
| **正念减压（MBSR）** | A级 |
| **MBCT-Pain** | B级 |
| **运动** | A级 |
| **物理治疗** | B级 |

## 七、抑郁-睡眠障碍共病

### 7.1 失眠与抑郁的双向关系

- 失眠是抑郁的风险因素（前瞻性研究显示失眠者抑郁发生率增加2-3倍）
- 失眠是抑郁的核心症状
- 失眠残留预示抑郁复发风险增加
- 治疗失眠可改善抑郁预后

### 7.2 药物选择

| 情况 | 策略 |
|------|------|
| **抑郁+入睡困难** | 曲唑酮25-100mg睡前；米氮平15-30mg；曲唑酮/喹硫平低剂量 |
| **抑郁+早醒** | 镇静型SSRI（帕罗西汀、氟伏沙明）；米氮平；加用助眠药 |
| **抑郁+睡眠维持困难** | 米氮平；曲唑酮；喹硫平XR 50-150mg |
| **抑郁+不宁腿综合征** | 普拉克索；加巴喷丁；避免SSRIs（可能加重RLS） |
| **抑郁+睡眠呼吸暂停** | 先CPAP；避免镇静药物加重呼吸抑制 |

### 7.3 CBT-I（失眠认知行为治疗）

| 成分 | 内容 | 抑郁整合 |
|------|------|---------|
| **刺激控制** | 床只用于睡眠；20分钟无法入睡则起床 | 适用于抑郁恢复期 |
| **睡眠限制** | 限制卧床时间至实际睡眠时间 | 重度抑郁期慎用（可能加重疲劳） |
| **认知重构** | 改变对睡眠的灾难化想法 | 与CBT-D整合 |
| **放松训练** | 渐进肌肉放松、呼吸训练 | 与正念整合 |

## 八、共病管理决策树

```
抑郁症患者评估
    │
    ├── 伴焦虑障碍（GAD/惊恐/社交/OCD）
    │       → 首选：SSRI/SNRI（覆盖两者）
    │       → 心理治疗：CBT整合
    │       → 严重焦虑：短期苯二氮卓类（2-4周）
    │
    ├── 伴PTSD
    │       → 首选：舍曲林/帕罗西汀/文拉法辛
    │       → 心理治疗：稳定化→CPT/PE→MBCT
    │       → 解离严重：先稳定化
    │
    ├── 伴BPD
    │       → 心理治疗：DBT为一线
    │       → 药物：对症（SSRI+喹硫平/拉莫三嗪）
    │       → 抑郁：标准抗抑郁药效果有限
    │
    ├── 伴物质使用
    │       → 活跃依赖：先物质治疗
    │       → 稳定期：整合治疗
    │       → 药物：避免肝毒性；注意药物相互作用
    │
    ├── 伴进食障碍
    │       → AN：营养恢复优先
    │       → BN/BED：氟西汀高剂量（BN）；SSRI（BED）
    │       → 避免安非他酮（AN/BN）
    │
    ├── 伴慢性疼痛
    │       → 首选：SNRI（度洛西汀/文拉法辛）
    │       → 整合：CBT-Pain + 运动 + MBSR
    │
    ├── 伴失眠
    │       → 首选：镇静型抗抑郁药（米氮平、曲唑酮）
    │       → 整合：CBT-I + 睡眠卫生
    │       → 注意：失眠残留=复发风险
    │
    └── 无显著共病
            → 标准抗抑郁治疗
```
