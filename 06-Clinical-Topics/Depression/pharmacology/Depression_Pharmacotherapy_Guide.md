# 抑郁症药物治疗手册 | Depression Pharmacotherapy Guide

> **目标**：建立精神科标准的抗抑郁药物治疗体系，涵盖药物选择、剂量滴定、换药/联合/增效策略、副作用管理与特殊人群用药。
> **标准来源**：CANMAT/ISBD指南（2016）、APA Practice Guidelines（2019）、中国抑郁障碍防治指南（2020）。

---

## 一、抗抑郁药分类与选择

### 1.1 一线药物比较

| 类别 | 代表药物 | 起始剂量 | 治疗剂量 | 优势 | 主要不良反应 | 特殊注意 |
|------|---------|---------|---------|------|-------------|---------|
| **SSRI** | 氟西汀 | 20mg | 20-60mg | 安全性高；过量毒性低；证据充分 | 胃肠道、失眠、性功能障碍、激活 | CYP2D6抑制（氟西汀） |
| | 舍曲林 | 50mg | 50-200mg | 药物相互作用较少；可用于心脏病 | 胃肠道、失眠、震颤 | 腹泻较常见 |
| | 帕罗西汀 | 20mg | 20-60mg | 抗焦虑效果好；镇静作用 | 镇静、体重增加、停药症状明显 | 抗胆碱能作用最强SSRIs；避免突然停药 |
| | 艾司西酞普兰 | 10mg | 10-20mg | 疗效/耐受性平衡好；药物相互作用少 | 恶心、头痛、失眠 | QTc延长风险（高剂量） |
| | 氟伏沙明 | 50mg | 100-300mg | 强迫症状共病有效 | 恶心、镇静、药物相互作用多 | CYP1A2/2C19抑制 |
| | 西酞普兰 | 20mg | 20-40mg | 药物相互作用少 | QTc延长（>40mg）| 心脏患者注意QTc |
| **SNRI** | 文拉法辛 | 75mg | 150-375mg | 高剂量时双重机制；严重抑郁有效 | 高血压、恶心、失眠、出汗 | 剂量依赖性血压升高；停药症状明显 |
| | 度洛西汀 | 30mg | 60-120mg | 疼痛共病有效 | 恶心、口干、便秘 | 肝病患者慎用 |
| | 米那普仑 | 25mg | 50-100mg | 去甲肾上腺素再摄取较强 | 恶心、便秘、出汗、心悸 | 较少体重增加 |
| | 左旋米那普仑 | 20mg | 40-120mg | 与米那普仑类似 | 类似 | 肾功能不全需减量 |
| **NaSSA** | 米氮平 | 15mg | 15-45mg | 镇静、食欲增加（适合失眠/消瘦者） | 体重增加、嗜睡、食欲增加 | 高剂量时去甲肾上腺素效应增强，镇静减轻 |
| **NDRI** | 安非他酮 | 150mg | 300-450mg | 无性功能障碍/体重增加；激活 | 失眠、焦虑、头痛、癫痫风险 | 癫痫病史禁用；进食障碍禁用 |
| **SARI** | 曲唑酮 | 50mg | 150-300mg | 镇静（适合失眠） | 镇静、体位性低血压、阴茎异常勃起（罕见） | 低剂量（25-100mg）常用于助眠 |
| **SARIs** | 维拉佐酮 | 10mg | 10-40mg | 5-HT1A部分激动；胃肠耐受好 | 腹泻、恶心 | 需与食物同服 |
| | 伏硫西汀 | 10mg | 10-20mg | 认知症状改善证据 | 恶心、腹泻 | 多模式机制（5-HT3/7拮抗） |

### 1.2 二线/特殊药物

| 类别 | 代表药物 | 特点 | 注意事项 |
|------|---------|------|---------|
| **TCA** | 阿米替林、丙咪嗪、氯咪帕明 | 严重抑郁/忧郁型有效；疼痛共病；便宜 | 心脏毒性（过量致命）；抗胆碱能；癫痫阈值降低 |
| **MAOI** | 苯乙肼、反苯环丙胺 | 非典型抑郁有效；TCA无效时 | 饮食限制（酪胺）；药物相互作用多；高血压危象风险 |
| **RIMA** | 吗氯贝胺 | 可逆性MAOI；饮食限制较轻 | 证据较弱；高剂量时仍需饮食注意 |
| **NASSA** | 米安色林 | 类似米氮平 | 粒细胞缺乏风险（罕见） |
| **SMS** | 噻奈普汀 | 5-HT再摄取增强剂（独特机制） | 法国等欧洲常用；美国未上市 |
| **NMDA调节剂** | 艾司氯胺酮（鼻喷） | 快速抗抑郁（数小时）；难治性抑郁 | 需监测 dissociation、血压；滥用风险；仅医疗机构使用 |

### 1.3 药物选择决策树

```
首次抑郁发作
    │
    ├── 伴显著焦虑/激越
    │       → 舍曲林、艾司西酞普兰、帕罗西汀（镇静型）
    │
    ├── 伴失眠、食欲减退、体重下降
    │       → 米氮平（15-30mg）
    │
    ├── 伴疲乏、动力缺乏、认知迟缓
    │       → 安非他酮、文拉法辛、伏硫西汀
    │
    ├── 伴慢性疼痛
    │       → 度洛西汀、文拉法辛、TCA
    │
    ├── 伴性功能障碍（既往史或高度关注）
    │       → 安非他酮、米氮平、伏硫西汀
    │
    ├── 伴肥胖/体重问题
    │       → 安非他酮、氟西汀
    │
    ├── 心血管疾病
    │       → 舍曲林、艾司西酞普兰（低剂量）、避免TCA/文拉法辛
    │
    ├── 癫痫病史
    │       → 避免安非他酮、TCA；首选SSRI
    │
    ├── 双相障碍（需联合心境稳定剂）
    │       → SSRI（拉莫三嗪为首选单药/联合）
    │
    └── 无特殊共病
            → SSRI（舍曲林或艾司西酞普兰为首选）
```

## 二、剂量滴定与疗程

### 2.1 剂量滴定原则

| 阶段 | 时间 | 策略 |
|------|------|------|
| **起始期** | 第1-2周 | 半量起始（除氟西汀可全量）；减少早期不良反应 |
| **滴定期** | 第2-4周 | 每1-2周增加剂量；监测不良反应 |
| **足量期** | 第4-8周 | 达到治疗剂量；评估疗效 |
| **评估期** | 第6-8周 | 决定是否换药/增效 |

### 2.2 治疗分期

| 分期 | 时间 | 目标 | 剂量 |
|------|------|------|------|
| **急性期** | 8-12周 | 症状缓解（HAMD≤7或PHQ-9<5） | 足量 |
| **巩固期** | 4-9月 | 防止症状复燃 | 急性期有效剂量 |
| **维持期** | 首次发作6-12月；复发≥2次2-3年；≥3次长期 | 防止复发 | 急性期有效剂量 |

## 三、换药、联合与增效策略

### 3.1 换药（Switching）

| 情况 | 策略 |
|------|------|
| **SSRI无效** | 换另一种SSRI（约50%对第二种SSRI有效）；或换SNRI/米氮平/安非他酮 |
| **SSRI不耐受** | 换不同类别（如SSRI→米氮平/安非他酮） |
| **TCA无效** | 换SSRI/SNRI/MAOI |

**换药洗脱期**：
- SSRI→MAOI：需洗脱2-5周（氟西汀需5周，因其半衰期长）
- MAOI→SSRI：需洗脱2周
- SSRI→SSRI：通常无需洗脱，可直接交叉滴定
- TCA→SSRI：谨慎交叉；监测心脏和抗胆碱能副作用叠加

### 3.2 联合治疗（Combination）

| 联合方案 | 证据 | 适用场景 |
|---------|------|---------|
| **SSRI + 米氮平** | A级（Co-Med研究） | 急性期加速缓解；失眠/食欲差 |
| **SSRI + 安非他酮** | A级 | 性功能不良反应；疲乏；戒烟 |
| **文拉法辛 + 米氮平** | B级（California Rocket Fuel） | 严重抑郁；"火箭燃料"组合 |
| **SSRI/SNRI + 非典型抗精神病药** | A级（增效） | 难治性抑郁；伴精神病性特征 |
| **SSRI + 锂盐** | A级 | 难治性抑郁；双相抑郁 |
| **SSRI + 甲状腺素（T3）** | B级 | 难治性抑郁；尤其伴亚临床甲减 |
| **SSRI + 丁螺环酮** | B级 | 焦虑共病；增效 |

### 3.3 增效策略（Augmentation）

| 增效剂 | 剂量 | 证据 | 特殊注意 |
|--------|------|------|---------|
| **阿立哌唑** | 2-15mg | A级（FDA批准） | 激活、静坐不能；代谢影响较小 |
| **喹硫平** | 150-300mg XR | A级（FDA批准） | 镇静、代谢综合征；低剂量镇静助眠 |
| **奥氮平+氟西汀（OFC）** | 奥氮平6-12mg+氟西汀25-50mg | A级 | 代谢影响大；严重抑郁/BPD |
| **利培酮** | 0.5-2mg | B级 | 泌乳素升高；锥体外系反应 |
| **布瑞哌唑** | 0.5-3mg | A级（FDA批准） | 静坐不能；类似阿立哌唑 |
| **鲁拉西酮** | 20-60mg | B级 | 需与食物同服；镇静 |
| **卡利拉嗪** | 1.5-3mg | B级 | 新批准；证据积累中 |
| **锂盐** | 600-1200mg（血锂0.6-1.0） | A级 | 甲状腺/肾/ parathyroid监测；致畸 |
| **甲状腺素（T3）** | 25-50mcg | B级 | 心动过速；骨质疏松（长期） |
| **黛力新（氟哌噻吨+美利曲辛）** | 1-2片 | C级（中国常用） | 锥体外系风险；不宜长期 |
| **安非他酮** | 150-300mg | B级 | 对SSRI性不良反应增效尤佳 |
| **哌醋甲酯** | 5-20mg | C级 | 老年抑郁；快速起效；滥用风险 |
| **莫达非尼** | 100-200mg | C级 | 疲乏、认知症状；睡眠障碍 |
| **S-腺苷甲硫氨酸（SAMe）** | 800-1600mg | B级 | 双相禁用；躁狂风险 |
| **Omega-3（EPA）** | 1-2g EPA | B级 | 抗炎；心血管益处 |
| **N-乙酰半胱氨酸（NAC）** | 1000-2000mg | C级 | 抗氧化；强迫/冲动共病 |
| **镁剂** | 200-400mg | C级 | 焦虑共病；睡眠 |
| **锌剂** | 25-50mg | C级 | 免疫；食欲 |
| **维生素D** | 2000-4000IU | C级 | 缺乏者补充 |
| **光照治疗** | 10000lux，30min/日 | B级 | 季节性抑郁；非季节性也有效 |
| **运动处方** | 中等强度150min/周 | A级 | 联合治疗；复发预防 |

## 四、停药与停药症状管理

### 4.1 停药原则

| 情况 | 建议 |
|------|------|
| **首次发作，症状完全缓解** | 维持6-12月后缓慢减停 |
| **复发≥2次** | 维持2-3年；考虑长期维持 |
| **复发≥3次或慢性抑郁** | 长期维持；可能终身 |
| **妊娠期** | 个体化决策；权衡复发风险与胎儿风险 |

### 4.2 缓慢减停方案

| 药物 | 减量速度 | 特殊注意 |
|------|---------|---------|
| **帕罗西汀、文拉法辛** | 每1-2周减25%（最慢） | 停药症状最明显 |
| **氟西汀** | 可较快（半衰期长，自洗脱） | 停药症状最轻 |
| **其他SSRI/SNRI** | 每2-4周减25-50% | 中等 |
| **TCA/MAOI** | 每1-2周减10-25% | 胆碱能反跳；需特别缓慢 |
| **米氮平** | 每1-2周减7.5-15mg | 失眠反跳；焦虑 |

### 4.3 停药症状 vs 复发鉴别

| 特征 | 停药症状 | 复发 |
|------|---------|------|
| **起病时间** | 停药后数小时-数天 | 逐渐，数周 |
| **持续时间** | 通常1-2周（氟西汀可更长） | 持续 ≥2周 |
| **症状性质** | 流感样、头晕、感觉异常（电击感）、失眠、焦虑 | 抑郁核心症状 |
| **对药物反应** | 重新服药后24-48h缓解 | 需2-4周才缓解 |

## 五、特殊人群用药

### 5.1 儿童青少年（<18岁）

| 药物 | FDA批准 | 注意 |
|------|---------|------|
| **氟西汀** | ≥8岁（MDD/OCD） | 青少年首选；激活/自杀意念监测 |
| **艾司西酞普兰** | ≥12岁（MDD） | 次选 |
| **舍曲林** | ≥6岁（OCD）；≥18岁（MDD off-label） | 常用off-label |
| **文拉法辛** | 不推荐 | 青少年自杀风险增加证据 |
| **帕罗西汀** | 不推荐青少年 | 自杀风险数据不利 |
| **TCA** | 不推荐 | 心脏毒性；过量风险 |

**黑框警告**：24岁以下患者使用抗抑郁药需监测自杀意念增加风险，尤其是治疗初期和剂量调整期。

### 5.2 老年（≥65岁）

| 原则 | 说明 |
|------|------|
| **起始低、滴定慢** | 半量起始；更缓慢滴定 |
| **注意药物相互作用** | 多重用药常见；CYP450相互作用 |
| **避免抗胆碱能药物** | TCA避免；帕罗西汀谨慎 |
| **避免镇静/体位性低血压** | 米氮平低剂量可用；高剂量注意跌倒 |
| **注意QTc延长** | 西酞普兰≤20mg；避免高剂量 |
| **首选** | 舍曲林、艾司西酞普兰、文拉法辛（低剂量） |
| **认知影响** | 抗胆碱能药物增加痴呆风险 |

### 5.3 妊娠期

| 药物 | 风险等级 | 建议 |
|------|---------|------|
| **舍曲林** | C级；相对安全数据最多 | 妊娠期首选 |
| **氟西汀** | C级 | 可用；注意新生儿撤药综合征 |
| **艾司西酞普兰** | C级 | 可用 |
| **帕罗西汀** | D级 | **避免**（心脏畸形风险） |
| **文拉法辛** | C级 | 可用；注意高血压 |
| **度洛西汀** | C级 | 可用 |
| **米氮平** | C级 | 可用；注意体重增加 |
| **安非他酮** | C级 | 可用；注意产后癫痫风险 |
| **TCA** | C级 | 相对安全但副作用多 |
| **MAOI** | C级 | 避免 |
| **锂盐** | D级 | 孕早期避免；孕中晚期可用（需监测） |
| **艾司氯胺酮** | 数据不足 | 避免 |

**原则**：抑郁本身对胎儿有风险（早产、低体重、产后抑郁）； untreated抑郁风险常大于药物风险。个体化决策。

### 5.4 哺乳期

| 药物 | 乳汁浓度 | 建议 |
|------|---------|------|
| **舍曲林、帕罗西汀** | 低 | 首选 |
| **氟西汀** | 相对较高 | 可用但监测婴儿 |
| **TCA（除多塞平）** | 低 | 相对安全 |

### 5.5 肝肾功能损害

| 损害 | 调整 |
|------|------|
| **肝功能损害** | 避免TCA、MAOI、度洛西汀；SSRI中避免高剂量；首选舍曲林、艾司西酞普兰 |
| **肾功能损害** | 度洛西汀（严重肾损禁用）；文拉法辛（减量）；左旋米那普仑（减量） |

## 六、不良反应管理速查

| 不良反应 | 常见药物 | 管理策略 |
|---------|---------|---------|
| **恶心/胃肠道** | SSRI、SNRI | 随餐服用；缓慢滴定；短期用止吐药 |
| **失眠** | SSRI（除帕罗西汀/氟伏沙明）、安非他酮 | 晨间服药；加用曲唑酮/米氮平；睡眠卫生 |
| **镇静** | 帕罗西汀、米氮平、曲唑酮 | 晚间服药；调整剂量 |
| **性功能障碍** | 所有SSRI/SNRI | 换安非他酮/米氮平；加安非他酮；PDE5抑制剂；药物假期（短半衰期药物） |
| **体重增加** | 帕罗西汀、米氮平 | 饮食运动；换药（安非他酮/氟西汀） |
| **焦虑/激活** | SSRI、安非他酮 | 缓慢滴定；短期用苯二氮卓类；换药 |
| **出汗** | SNRI、SSRI | 少见；可短期用抗胆碱能药；换药 |
| **QTc延长** | 西酞普兰、艾司西酞普兰、TCA | ECG监测；避免高剂量；避免其他QTc药物 |
| **低钠血症（SIADH）** | 所有SSRI/SNRI（尤其老年） | 监测电解质；限水；严重时停药 |
| **出血风险** | SSRI（抑制血小板5-HT摄取） | 注意NSAIDs/抗凝药联合；手术前评估 |
| **5-HT综合征** | SSRI+MAOI/曲马多/圣约翰草等 | 高热、肌阵挛、意识改变→急诊；停用5-HT药物；支持治疗 |
| **躁狂诱发** | 所有抗抑郁药（尤其双相） | 筛查双相；联合心境稳定剂；停药 |
| **癫痫发作** | 安非他酮、TCA、氯咪帕明 | 癫痫病史禁用；注意剂量相关 |
