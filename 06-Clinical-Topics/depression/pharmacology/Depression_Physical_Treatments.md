# 抑郁症物理治疗 | Physical Treatments for Depression

> **目标**：系统阐述精神科物理治疗手段，涵盖电休克治疗（ECT）、重复经颅磁刺激（rTMS）、迷走神经刺激（VNS）、深部脑刺激（DBS）、光照治疗及运动处方。
> **标准来源**：APA Practice Guidelines（2019）、APA ECT Task Force Report（2022）、中国抑郁障碍防治指南（2020）。

---

## 一、电休克治疗（ECT）

### 1.1 适应症与推荐等级

| 情况 | 推荐等级 | 说明 |
|------|---------|------|
| **伴精神病性特征的抑郁** | A级 | 伴妄想/幻觉的抑郁，ECT为首选 |
| **伴紧张症的抑郁** | A级 | 木僵、蜡样屈曲等；ECT可救命 |
| **拒食/拒水危及生命** | A级 | 快速恢复进食能力 |
| **自杀风险极高** | A级 | 快速起效（3-5次） |
| **难治性抑郁（TRD）** | A级 | 两种足量足疗程药物无效后 |
| **药物无法耐受** | A级 | 严重不良反应无法继续药物 |
| **产后抑郁严重** | B级 | 需快速恢复母婴功能 |
| **双相抑郁** | A级 | 躁狂和抑郁均有效 |
| **预防复发（维持ECT）** | B级 | 药物+维持ECT vs 单药 |

### 1.2 禁忌症与风险

| 绝对禁忌症 | 相对禁忌症 | 需特殊准备 |
|-----------|-----------|-----------|
| 颅内压增高的占位病变 | 近期心肌梗死（<3月） | 心血管疾病：麻醉科评估 |
| 近期脑出血/卒中（不稳定） | 不稳定心绞痛 | 糖尿病：调整胰岛素 |
| 严重心律失常 | 严重肺部疾病 | 骨质疏松：注意骨折风险 |
| 主动脉瘤 | 妊娠（可实施但需准备） | 牙齿问题：保护牙齿 |
| | 老年（风险增加但可用） | 患者教育：消除恐惧 |

### 1.3 技术参数

| 参数 | 说明 |
|------|------|
| **电极放置** | 双侧（BL-ECT）：疗效更强，认知影响更大；右侧单侧（RUL-ECT）：认知影响小，可能需要更多次数 |
| **刺激剂量** | 基于癫痫阈值滴定；RUL需≥6×阈值剂量；BL需≥1.5×阈值 |
| **麻醉** | 短效麻醉（丙泊酚/美索比妥）+ 肌松药（琥珀酰胆碱）+ 抗胆碱能药（减少分泌物） |
| **治疗频率** | 通常2-3次/周；共6-12次 |
| **疗程** | 急性期6-12次；最多可达20次 |

### 1.4 不良反应

| 不良反应 | 发生率 | 管理 |
|---------|--------|------|
| **急性意识混乱** | 常见（每次治疗后数分钟-数小时） | 监测；休息；通常自限 |
| **顺行性遗忘** | 常见（治疗期间的记忆形成受损） | 通常2周内恢复；注意限度 |
| **逆行性遗忘** | 常见（治疗前数周-数月的记忆模糊） | 多数在数月内恢复；部分永久 |
| **头痛** | 常见 | 对症处理 |
| **肌肉酸痛** | 常见（肌松药相关） | 对症处理 |
| **恶心** | 少见 | 对症处理 |
| **心血管事件** | 罕见 | 监测；麻醉科支持 |

**认知影响最小化策略**：
- 使用RUL-ECT
- 使用超短脉冲刺激（ultrabrief pulse）
- 滴定至最低有效刺激剂量
- 间隔给药（如每周1次维持）

### 1.5 维持ECT（M-ECT）

| 方案 | 频率 | 适用 |
|------|------|------|
| **巩固期** | 每周1次 × 4周 | 急性ECT后过渡 |
| **维持期** | 每2-4周1次 | 高复发风险；药物不耐受 |
| **长期维持** | 每3-8周1次 | 慢性复发性抑郁 |

## 二、重复经颅磁刺激（rTMS）

### 2.1 适应症与推荐等级

| 情况 | 推荐等级 | 说明 |
|------|---------|------|
| **难治性抑郁（一种药物无效）** | A级（FDA批准） | 门诊治疗；无创 |
| **难治性抑郁（两种药物无效）** | A级 | 深部TMS（dTMS）也获FDA批准 |
| **不能耐受药物不良反应** | B级 | |
| **孕期抑郁（不愿/不能用药）** | B级 | 相对安全 |
| **老年抑郁** | B级 | 认知影响小 |
| **维持治疗预防复发** | B级 | 证据积累中 |

### 2.2 技术参数

| 参数 | 标准方案 | 说明 |
|------|---------|------|
| **刺激部位** | 左背外侧前额叶（DLPFC） | 高频（10Hz）；兴奋性刺激 |
| | 右DLPFC | 低频（1Hz）；抑制性刺激 |
| | 背内侧前额叶（DMPFC） | 深部TMS线圈 |
| **刺激强度** | 110-120%运动阈值（MT） | |
| **每次脉冲数** | 3000脉冲 | 约37.5分钟 |
| **治疗频率** | 每日1次，每周5天 | |
| **疗程** | 4-6周（20-30次） | 急性期 |
| **巩固期** | 每周1-2次 × 4-12周 | 预防复发 |

### 2.3 禁忌症

| 绝对禁忌症 | 相对禁忌症 |
|-----------|-----------|
| 颅内金属（铁磁性）植入物 | 植入式医疗设备（起搏器、VNS、DBS、胰岛素泵） |
| 心脏起搏器（旧型） | 癫痫病史/高风险（低频rTMS风险较低） |
| 耳蜗植入 | 妊娠（相对安全但数据有限） |
| 颅内压增高 | 服用降低癫痫阈值药物 |

### 2.4 不良反应

| 不良反应 | 发生率 | 管理 |
|---------|--------|------|
| **头痛/头皮不适** | 常见（10-30%） | 对症；通常数天适应 |
| **面部抽搐** | 常见（治疗中） | 调整线圈位置 |
| **听力变化** | 少见 | 佩戴耳塞 |
| **癫痫发作** | 罕见（<0.01%） | 停止治疗；抗癫痫药；重新评估 |
| **躁狂诱发** | 罕见 | 停止治疗；筛查双相 |

## 三、其他神经调控治疗

### 3.1 迷走神经刺激（VNS）

| 项目 | 内容 |
|------|------|
| **机制** | 植入设备刺激左侧迷走神经；间接影响脑干、边缘系统、皮层 |
| **适应症** | 慢性难治性抑郁（≥4种药物+ECT失败） |
| **疗效** | 起效慢（数月）；长期维持有效；约30-50%应答 |
| **手术** | 需手术植入脉冲发生器（胸部）和电极（颈部） |
| **不良反应** | 声音嘶哑、咳嗽、咽喉疼痛、呼吸困难（刺激时） |
| **现状** | 美国FDA批准；中国应用有限 |

### 3.2 深部脑刺激（DBS）

| 项目 | 内容 |
|------|------|
| **靶点** | 胼胝体下扣带回（SCC/Brodmann 25区）、腹侧内囊/腹侧纹状体（VC/VS） |
| **适应症** | 严重难治性抑郁（所有其他治疗失败） |
| **疗效** | 约40-60%应答；起效数周-数月 |
| **手术** | 立体定向手术植入电极；颅内手术风险 |
| **不良反应** | 手术风险（出血、感染）；刺激相关（情绪高涨、焦虑、轻躁狂） |
| **现状** | 研究阶段；美国FDA未正式批准（人道主义器械豁免） |

### 3.3 磁休克治疗（MST）

| 项目 | 内容 |
|------|------|
| **机制** | 高强度磁刺激诱发癫痫发作（类似ECT但用磁而非电） |
| **优势** | 认知影响可能小于ECT（更聚焦） |
| **现状** | 研究阶段；未广泛应用 |

### 3.4 经颅直流电刺激（tDCS）

| 项目 | 内容 |
|------|------|
| **机制** | 微弱直流电（1-2mA）调节皮层兴奋性；阳极兴奋、阴极抑制 |
| **方案** | 阳极置于左DLPFC，阴极置于右DLPFC或肩部 |
| **疗效** | 轻度-中度抑郁可能有效；证据弱于rTMS |
| **优势** | 极便宜；可家用；安全 |
| **不良反应** | 头皮刺痛、瘙痒、发红（电极下） |
| **现状** | 研究阶段；部分国家家用设备上市 |

## 四、光照治疗

### 4.1 适应症

| 情况 | 证据等级 |
|------|---------|
| **季节性抑郁（SAD）** | A级 |
| **非季节性抑郁** | B级（辅助） |
| **产后抑郁** | B级 |
| **双相抑郁（需谨慎）** | C级 |

### 4.2 技术参数

| 参数 | 标准 |
|------|------|
| **光强度** | 10000 lux |
| **距离** | 约30-60cm |
| **时长** | 30分钟（晨间） |
| **时间** | 醒来后30分钟内最佳 |
| **疗程** | 每日；急性期2-4周见效 |

### 4.3 注意事项

- 避免傍晚使用（可能影响睡眠）
- 视网膜疾病患者咨询眼科
- 双相患者可能诱发躁狂（需联合心境稳定剂）
- 可结合SSRI增强疗效

## 五、运动处方

### 5.1 循证证据

| 运动类型 | 证据 | 效应量 |
|---------|------|--------|
| **有氧运动** | A级 | d=0.6-0.8（轻-中度抑郁与药物相当） |
| **阻力训练** | B级 | d=0.5-0.7 |
| **瑜伽** | B级 | d=0.4-0.6 |
| **太极** | B级 | d=0.4-0.5 |

### 5.2 处方参数

| 参数 | 推荐 |
|------|------|
| **频率** | 每周3-5次 |
| **强度** | 中等强度（最大心率的60-75%；或Borg量表12-14分） |
| **时长** | 每次30-45分钟 |
| **类型** | 有氧运动（快走、慢跑、游泳、骑车）+ 阻力训练 |
| **疗程** | 至少10-12周见效；长期维持 |

### 5.3 特殊人群

| 人群 | 调整 |
|------|------|
| **老年** | 低冲击运动（步行、太极、水中有氧）；注意跌倒风险 |
| **重度抑郁** | 从极短时长开始（5-10分钟）；行为激活策略 |
| **伴躯体疾病** | 医学评估后制定；心脏康复计划 |
| **伴疼痛** | 水中运动、瑜伽、温和伸展 |

## 六、物理治疗选择决策树

```
抑郁症患者
    │
    ├── 伴精神病性特征 / 紧张症 / 拒食危及生命 / 极高自杀风险
    │       → ECT（首选）
    │
    ├── 难治性抑郁（≥2种药物足量足疗程无效）
    │       │
    │       ├── 愿意/能够接受住院/麻醉
    │       │       → ECT（疗效最强证据）
    │       │
    │       └── 倾向门诊/无创治疗
    │               → rTMS（A级证据；门诊；无创）
    │               → 若rTMS无效 → ECT
    │
    ├── 轻-中度抑郁；不愿/不能用药
    │       → 运动处方（A级）
    │       → 光照治疗（尤其季节性）
    │       → tDCS（研究/家用）
    │
    ├── 药物部分有效，需增效
    │       → rTMS联合药物（B级）
    │       → 运动联合药物（A级）
    │
    └── 慢性难治性抑郁（所有药物+ECT失败）
            → VNS（已批准）
            → DBS（研究/试验性）
```
