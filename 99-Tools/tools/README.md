# Peace Lab Tools | 平静实验室工具库

> 本目录包含用于知识库维护、数据抓取、自动化处理的脚本与辅助程序。

## 工具列表 | Tool List

*   **数据辅助工具 (Data Assistants)**: 包含各类抓取与处理脚本。
*   **文档维护 (Document Maintenance)**: 用于自动化链接检查与结构优化。

## 使用指南 | Usage Guide

所有 Python 脚本建议在虚拟环境中运行：
```bash
pip install -r requirements.txt
python <script_name>.py
```
