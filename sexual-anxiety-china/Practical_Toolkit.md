### 常见认知偏差识别

**灾难化思维清单**
- [ ] "如果我在性方面表现不好，我们的关系就完了"
- [ ] "别人一定会觉得我很差劲"
- [ ] "我永远都不会改善这种情况"

**过度概括练习**
- 一次不好的经历 ≠ 永远如此
- 个别情况 ≠ 普遍规律
- 当前困难 ≠ 未来绝望

**黑白思维转换**
- 从"要么完美要么失败"到"可以有不同程度的成功"
- 从"完全好或完全坏"到"有优点也有改进空间"

## 情绪调节技术

### 呼吸调节法

**4-7-8呼吸技术**
1. 用鼻子吸气4秒
2. 屏住呼吸7秒
3. 用嘴呼气8秒
4. 重复4-5轮

**腹式呼吸练习**
- 一只手放在胸部，一只手放在腹部
- 专注让腹部起伏，胸部尽量不动
- 缓慢深长地呼吸，每分钟6-8次

### 渐进性肌肉放松

**全身放松程序**
1. 握紧双拳10秒，然后完全放松
2. 皱紧面部肌肉10秒，然后放松
3. 耸肩靠近耳朵10秒，然后放松
4. 绷紧腹部肌肉10秒，然后放松
5. 收紧臀部肌肉10秒，然后放松
6. 交替绷紧和放松小腿肌肉

### 正念冥想练习

**基础正念练习(10分钟)**
1. 找一个安静舒适的地方坐下
2. 关注呼吸的自然流动
3. 当思绪飘走时，温和地带回注意力
4. 观察但不评判出现的想法和感受
5. 保持这种觉察状态10分钟

## 关系改善工具

### 伴侣沟通指南

**有效沟通原则**
1. 选择合适的时间和地点
2. 使用"I"语句表达感受
3. 积极倾听对方的观点
4. 避免指责和批评
5. 寻求共同解决方案

**沟通练习话题**
- 我们各自对性的期待是什么？
- 什么让我们感到最舒服和最有安全感？
- 我们如何更好地支持彼此？
- 有什么是我们想要一起探索的？

**增进亲密感的活动**

- 创造浪漫和惊喜时刻

- 分享当天的三个积极体验
- 回忆和重温美好时光
- 一起制定未来的计划和梦想

**睡眠优化建议**
- 保持规律的作息时间
- 创造舒适的睡眠环境

## 应急处理预案

### 危机应对流程

**急性焦虑发作处理**
1. 立即停止当前活动
2. 进行深呼吸或放松练习
3. 转移到安静舒适的环境
4. 寻求身边人的支持
5. 必要时联系专业帮助

**自杀风险识别**
- 直接询问是否有自杀想法
- 评估计划的具体性和可行性
- 移除可能的危险物品
- 立即寻求专业紧急干预
- 通知家属或监护人
