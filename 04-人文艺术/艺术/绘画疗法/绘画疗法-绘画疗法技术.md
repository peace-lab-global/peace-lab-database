# 绘画疗法核心技术 | Art Therapy Techniques

> 绘画疗法的技术不是"教画画"，而是借助媒材与任务结构，为内在经验打开表达通道。以下为临床实践中最常用的技术类别。

## 常用技术

| 技术 | 操作 | 主要用途 |
|------|------|---------|
| 自由绘画 | 不限主题，随手而画 | 建立安全感、初诊探索 |
| 主题绘画 | 指定主题（"画一棵树"、"画你的感受"） | 聚焦特定议题 |
| 房树人（H-T-P） | 画房子、树、人 | 评估自我概念与家庭动力；作诊断工具使用时效度有争议 |
| 曼陀罗绘画 | 在圆形结构中涂绘 | 稳定情绪、集中注意；源自荣格对曼陀罗的临床应用 |
| 涂鸦法（Squiggle） | 治疗师与来访者互相把对方的涂鸦发展成图 | 儿童工作；源自温尼科特 |
| 家庭绘画 | 画出"全家人在做一件事" | 呈现家庭关系结构（Kwiatkowska 传统） |
| 拼贴 | 用现成图像剪贴组合 | 降低"不会画"的焦虑，适合成人与团体 |
| 团体壁画 | 多人共作一幅大画 | 团体动力、社区疗愈 |

## 操作要点

1. **媒材即变量**：铅笔可控、水彩流动、黏土具身——媒材的"可控—流动"维度本身就是一个治疗变量，治疗师按来访者的状态选择。
2. **问而不判**：面对作品，标准姿态是开放式提问（"能说说这部分吗？"），而非替来访者解释。意义属于创作者。
3. **过程记录**：关注创作过程中的顺序、停顿、涂改与情绪变化，往往比成品更有信息量。
4. **作品的处置**：作品归属来访者；拍照存档须知情同意；不在他人面前随意展示。

## 实践与应用

- **儿童与青少年**：绘画是其自然语言，适合作为咨询的入门通道。
- **创伤工作**：先用可控媒材建立稳定，再逐步接近创伤主题；避免在稳定化不足时强行"画创伤"。
- **团体设置**：壁画与拼贴适合破冰与呈现动力，但需要清晰的规则与收尾仪式。
