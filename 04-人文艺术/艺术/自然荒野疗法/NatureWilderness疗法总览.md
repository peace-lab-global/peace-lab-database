# 自然与荒野疗愈总览 | Nature & Wilderness Therapy Overview

> 自然疗愈以自然环境为治疗场域，基于"亲生命性假说"(Biophilia Hypothesis, Wilson, 1984)——人类进化于自然环境中，与自然的联结本身具有恢复和疗愈功能。

---

## 1. 理论基础

### 1.1 核心理论

| 理论 | 代表 | 核心主张 |
|------|------|----------|
| **注意力恢复理论(ART)** | Kaplan (1995) | 自然环境提供"柔性迷恋"(soft fascination)→恢复定向注意力疲劳 |
| **压力恢复理论(SRT)** | Ulrich (1991) | 自然视觉刺激→即时副交感激活→压力生理指标下降 |
| **亲生命性假说** | Wilson (1984) | 人类有与生俱来的与自然联结的生物性需求 |
| **生态心理学** | Roszak (1992) | 生态危机与心理危机共享根源——与自然的断裂 |
| **多迷走理论(自然环境)** | Porges | 自然声音/绿色视觉→腹侧迷走激活→安全感 |

### 1.2 循证证据

| 研究 | 发现 |
|------|------|
| **Li Q (2010)** 森林浴 | NK细胞活性↑56%(3天森林暴露)，效果持续30天 |
| **Bratman et al. (2015)** | 90min自然步行↓亚属前额叶活动(rumination神经关联) |
| **White et al. (2019)** 20,000人 | 每周≥120min自然接触→健康/wellbeing显著提升 |
| **Twohig-Bennett & Jones (2018)** Meta | 绿色空间↓皮质醇、血压、心率、II型糖尿病风险 |
| **Barton & Pretty (2010)** Meta | 5min"绿色运动"即可↑自尊/情绪(效应量d=0.54) |
| **Hunter et al. (2019)** | "自然药丸"处方(20-30min自然接触)↓皮质醇 |

## 2. 疗愈形式

### 2.1 森林浴 (Shinrin-yoku / Forest Bathing)

**起源**: 日本林野厅(1982)，由李卿(Qing Li)系统研究

```
标准森林浴(2-4h):
1. 感官打开(10min): 静坐/站立，逐一打开五感
2. 慢步行走(60-90min): 无目的地漫步，速度<1km/h
   - 视觉: 观察光线/色彩/纹理
   - 听觉: 鸟鸣/风声/水流
   - 嗅觉: 芬多精(phytoncides)/泥土/花
   - 触觉: 树皮/苔藓/水
3. 静坐/冥想(20-30min): 选定"坐点"(sit spot)
4. 分享圈(20min): 分享体验，非分析
```

**生理效应**: ↓皮质醇(12-16%)、↓血压、↑HRV、↑NK细胞、↑IgA

### 2.2 荒野治疗 (Wilderness Therapy)

| 要素 | 描述 |
|------|------|
| **远征式** | 多日荒野徒步(7-28天)→脱离日常环境 |
| **基地式** | 固定营地+日间荒野活动 |
| **核心机制** | 挑战- mastery、团体依赖、自然反馈、简化生活 |
| **适用人群** | 青少年行为问题、成瘾、创伤、家庭冲突 |

**循证**: Russell & Hendee (2000); Bettmann et al. (2016) Meta — 荒野治疗对青少年行为/情绪问题有效(d=0.48-0.76)

### 2.3 园艺治疗 (Horticultural Therapy, HT)

| 活动 | 治疗功能 |
|------|----------|
| 播种/培育 | 耐心、责任、生命联结 |
| 收获/烹饪 | 成就感、感官丰富 |
| 修剪/堆肥 | 丧失/死亡接纳、循环观 |
| 花园设计 | 控制感、创造性表达 |
| 感官花园 | 多感官刺激(认知障碍/ASD) |

**循证**: Soga et al. (2017) Meta — HT↓抑郁(d=0.58)、↑QoL

### 2.4 动物辅助治疗 (Animal-Assisted Therapy, AAT)

| 形式 | 适应证 |
|------|--------|
| **马辅助治疗(EAT)** | PTSD/ASD/青少年行为问题——马的生物反馈(心率同步) |
| **犬辅助治疗** | 老年痴呆/住院儿童/创伤——触摸/互动→催产素↑ |
| **农场疗愈** | 成瘾康复/精神分裂——日常照料→结构化+意义 |

### 2.5 蓝色空间疗愈 (Blue Space Therapy)

| 水体 | 效应 |
|------|------|
| **海洋** | ↑情绪、↓心理困扰(White et al., 2020) |
| **河流/湖泊** | 注意力恢复、平静感 |
| **瀑布** | 负离子→可能↑血清素(Yamada, 2009) |
| **水上运动** | 冲浪治疗(PTSD/青少年) |

## 3. 自然处方 (Nature Prescription)

### 3.1 PaRx方案(加拿大BC Parks Foundation)

```
推荐剂量:
- 频率: 每周≥3次
- 时长: 每次≥20min
- 环境: 有树木/水体的绿色空间
- 活动: 非运动性自然接触(坐/走/观察)
- 累计: 每周≥120min
```

### 3.2 日本"森林医学"标准

| 指标 | 森林暴露前→后 |
|------|--------------|
| 皮质醇(唾液) | ↓12.4% |
| 血压(收缩) | ↓5.3mmHg |
| HRV(HF) | ↑8.7% |
| 心率 | ↓3.3bpm |

### 3.3 特殊人群方案

| 人群 | 推荐 | 注意事项 |
|------|------|----------|
| **城市焦虑** | 每日公园步行20min + 周末森林浴 | 手机静音/收纳 |
| **抑郁** | 园艺治疗(8-12周) + 绿色运动 | 渐进参与，不强迫 |
| **PTSD** | 荒野远征(14天) + 马辅助 | 创伤知情框架 |
| **ASD儿童** | 感官花园 + 动物互动 | 可控刺激量 |
| **老年认知** | 日常园艺 + 自然散步 | 防滑/防晒 |
| **企业倦怠** | 森林团建(1-3天) + 绿色办公 | 定期自然接触 |

## 4. 生态焦虑 (Eco-Anxiety)

- 气候变化引发的焦虑/悲伤/无助感——**不是病理，而是对真实威胁的合理反应**
- 治疗方向: 行动(公民参与) + 自然联结 + 社群支持 + 意义建构
