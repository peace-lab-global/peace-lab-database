## 疗愈花园概述

### 定义与分类

| 类型 | 目的 | 特征 |
|------|------|------|
| 疗愈花园(Healing Garden) | 被动恢复 | 观赏为主/减压/恢复 |
| 治疗花园(Therapeutic Garden) | 主动治疗 | 参与性活动/有治疗目标 |
| 冥想花园(Meditation Garden) | 静修/内省 | 简洁/安静/隐私 |
| 感官花园(Sensory Garden) | 多感官刺激 | 五感设计/无障碍 |
| 记忆花园(Memory Garden) | 认知刺激 | 怀旧元素/熟悉植物 |
| 康复花园(Rehabilitation Garden) | 身体康复 | 不同地形/抓握元素 |

### 循证设计(Evidence-Based Design)的核心原则

| 原则 | 来源 | 花园应用 |
|------|------|---------|
| 自然视野降低压力 | Ulrich(1984) | 窗景/自然景观 |
| 选择感与控制感 | 环境心理学 | 多路径/私人空间 |
| 社交支持 | 健康心理学 | 聚会空间/可灵活安排 |
| 积极分散注意力 | 疼痛管理 | 流水/花卉/小动物 |

## 设计原则

### 七大设计原则

| 原则 | 说明 | 实施 |
|------|------|------|
| 可及性 | 所有能力水平均可使用 | 无障碍路径/升高花床 |
| 安全性 | 物理安全+心理安全 | 无毒植物/防滑/围合感 |
| 多感官 | 刺激五感的丰富体验 | 香草/纹理/色彩/声音 |
| 寻路 | 清晰导向/不迷失 | 地标/环形路径/标识 |
| 选择 | 提供多种活动选择 | 安静区/社交区/活动区 |
| 熟悉 | 唤醒记忆的文化元素 | 本地植物/传统花卉 |
| 舒适 | 物理舒适 | 遮阳/座椅/避风 |

### 植物选择标准

| 标准 | 说明 |
|------|------|
| 安全性 | 无毒/无刺/低过敏 |
| 感官丰富 | 色彩/香气/质地多样 |
| 季节性 | 四季有花/有果/有叶色变化 |
| 维护难度 | 低维护/耐候性强 |
| 文化意义 | 与使用者文化背景相关 |
| 互动性 | 可触摸/可采摘/可食用 |

## 功能分区

### 标准分区模型

| 区域 | 功能 | 设计要素 |
|------|------|---------|
| 入口/过渡区 | 从室内到花园的心理准备 | 遮阳廊/香草边植 |
| 观赏/冥想区 | 静坐/观察/放松 | 水景/草坪/隐蔽座椅 |
| 活动/种植区 | 园艺活动/治疗 | 升高花床/工具存储 |
| 社交/聚会区 | 团体活动/交流 | 桌椅/凉亭/烧烤 |
| 步行/运动区 | 散步/康复运动 | 环形路径/不同坡度 |
| 私密/安静区 | 独处/情绪释放 | 藤架/围合空间 |
| 儿童区(如适用) | 探索/游戏/自然教育 | 沙/水/攀爬/菜地 |

### 无障碍设计要点

| 要素 | 标准 |
|------|------|
| 路径宽度 | ≥1.5m(轮椅双向) |
| 路面材质 | 平整/防滑/硬质(非碎石) |
| 花床高度 | 60-80cm(轮椅可及) |
| 坡度 | ≤5%(1:20) |
| 扶手 | 关键位置连续扶手 |
| 休息座椅 | 每30m有座椅(有扶手/靠背) |
| 标识 | 大字体/高对比/图文结合 |

## 特殊人群花园

### 老年/痴呆友好花园

| 特征 | 说明 |
|------|------|
| 环形路径 | 避免死胡同→减少迷失焦虑 |
| 围合安全 | 柔性边界(灌木/花卉)而非铁栅 |
| 记忆元素 | 旧时代花卉/蔬果/晾衣架等 |
| 感官刺激 | 芳香植物/纹理/鸟食台 |
| 可见性 | 从室内可看到花园(邀请进入) |
| 遮阳 | 充足遮阳(皮肤脆弱/药物光敏) |

### 儿童治疗花园

| 特征 | 说明 |
|------|------|
| 自然游戏 | 泥巴/水/沙/攀爬 |
| 微型生态 | 昆虫旅馆/蝴蝶花园/池塘 |
| 种植体验 | 快速生长植物(向日葵/豆类) |
| 安全围合 | 视线可及/无毒植物/柔软地面 |
| 创造空间 | 树屋/柳条隧道/想象游戏 |
