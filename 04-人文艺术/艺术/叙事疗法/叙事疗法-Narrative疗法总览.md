# 叙事疗法总览 | Narrative Therapy Overview

> 叙事疗法由Michael White和David Epston创立，核心信念是"人不是问题，问题才是问题"——通过外化对话和重写生命故事实现疗愈。

---

## 1. 理论基础

### 1.1 哲学根源

| 思想来源 | 核心贡献 |
|----------|----------|
| **后结构主义(Foucault)** | 话语权力、知识建构、主流叙事对人的控制 |
| **社会建构论** | 现实是通过语言和社会互动建构的 |
| **叙事心理学(Bruner)** | 人类是"讲故事的动物"，通过叙事组织经验 |
| **Goffman/Erickson** | 身份是社会表演，非固定本质 |

### 1.2 核心假设

1. 人的身份是被叙事建构的（而非本质的）
2. 问题与人是分开的——**问题是被主流叙事赋予权力的**
3. 每个人都有自己的"在地知识"(local knowledge)——专家是自己生命的专家
4. 治疗是协作性的意义探索，非诊断性干预

---

## 2. 核心技术

### 2.1 外化对话 (Externalizing Conversations)

**将问题从人身上分离**，赋予问题独立的"人格"：

| 外化问句 | 示例 |
|----------|------|
| 命名问题 | "如果给这个'焦虑'起一个名字，你会叫它什么？" |
| 探索影响 | "这个'黑狗'是如何影响你的日常生活的？" |
| 评估影响 | "这些影响中哪些是你愿意接受的？哪些不是？" |
| 论证评估 | "你为什么不愿意接受这个影响？这说明你看重什么？" |

### 2.2 重写对话 (Re-authoring)

寻找"独特结果"(unique outcomes)——与问题故事不一致的例外时刻：

```
问题主导故事 → 寻找例外(独特结果) → 丰厚替代故事 → 巩固新身份
    ↑                                    ↓
    └────── 持续循环 ←────────────────────┘
```

| 重写问句 | 示例 |
|----------|------|
| 发现例外 | "有没有某个时刻，'焦虑'试图控制你但你没有屈服？" |
| 丰厚细节 | "那时候你做了什么？这说明你具有什么能力？" |
| 连接过去 | "你从什么时候开始具备这个能力的？" |
| 展望未来 | "如果你带着这个能力往前走，生活会怎么不同？" |

### 2.3 会员重组 (Re-membering)

| 技术 | 描述 |
|------|------|
| **会员入会** | "谁的生命中会因为你拥有这个能力而感到高兴？" |
| **会员降级** | 将问题声音从"核心会员"降为"边缘成员" |
| **生命俱乐部** | 可视化：你的生命俱乐部中谁是VIP？谁是普通成员？谁应该被请出？ |

### 2.4 治疗性文件 (Therapeutic Documents)

| 文件类型 | 用途 |
|----------|------|
| **声明/证书** | 记录新身份和能力（"勇气证书"） |
| **信件** | 治疗师写给来访者，总结会谈发现 |
| **地图** | 问题影响地图、替代故事时间线 |
| **照片/艺术品** | 见证新故事的证据 |

### 2.5 外部见证人 (Outsider Witnesses)

- 邀请重要他人或同侪倾听来访者的新故事
- 见证者不是评价，而是**共鸣**——"我听到这个故事中什么触动了你？"
- 定义仪式(Definitional Ceremony): Myerhoff (1982)

## 3. 临床应用

### 3.1 循证证据

| 研究 | 人群 | 结果 |
|------|------|------|
| Lopes et al. (2014) Meta | 焦虑/抑郁 | 叙事疗法有效(d=0.67) |
| Etchison & Kleist (2000) SR | 多种人群 | 叙事干预改善自我效能/身份 |
| Beaudoin et al. (2016) RCT | 青少年抑郁 | 叙事疗法↓抑郁(vs TAU) |
| Morgan (2000) | 进食障碍 | 外化"厌食的声音"→显著改善 |

### 3.2 适用场景

| 场景 | 叙事策略 |
|------|----------|
| **创伤后** | 重写创伤叙事：从"受害者"到"幸存者" |
| **进食障碍** | 外化"厌食"/"暴食"→夺回生命控制权 |
| **成瘾** | 外化"渴求"/"成瘾"→发现不使用时的自我 |
| **儿童/青少年** | 绘画/故事/玩偶外化问题→游戏化治疗 |
| **家庭治疗** | 家庭共同重写→打破代际叙事 |
| **文化疗愈** | 殖民叙事的解构→原住民身份重建 |
| **哀伤** | 继续联结(re-membering逝者)→整合丧失 |
| **企业/团队** | 团队叙事重建→从"失败者"到"学习者" |

### 3.3 叙事疗法与其他方法的整合

| 整合 | 方式 |
|------|------|
| **叙事+正念** | 觉察内在叙事→外化→接纳→重写 |
| **叙事+艺术治疗** | 视觉化外化(画出"恐惧")→重写画册 |
| **叙事+戏剧** | 演出替代故事→身体化新身份 |
| **叙事+写作疗愈** | 治疗性写作→生命之书→出版/分享 |

---

## 4. 叙事治疗师核心立场

| 立场 | 说明 |
|------|------|
| **去中心化影响力** | 治疗师不是专家，来访者才是自己生命的专家 |
| **好奇而非诊断** | "我不知道"的立场——保持探索性好奇 |
| **政治觉察** | 关注主流话语(性别/种族/阶级)如何压迫个体 |
| **合作性** | 治疗方向由来访者决定 |
| **去病理化** | 不以DSM标签定义人 |
