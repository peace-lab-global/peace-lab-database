# 戏剧疗愈实操指南 | Drama Therapy Practice Guide

> **文档定位**: 戏剧疗愈的课程设计、暖身活动、团体带领指南
> **关键词**: Warm-up / Session Design / Group Facilitation / Improvisation / Games / Safety

---

## 1. 暖身活动库

### 1.1 身体暖身

| 活动 | 时间 | 操作 | 目的 |
|------|------|------|------|
| 身体抖动 | 2min | 全身放松抖动→逐渐停止→感受静止 | 释放紧张 |
| 空间行走 | 5min | 在空间中行走，变化速度(1-10) | 身体觉察+群体同步 |
| 雕塑传递 | 5min | 一人做姿势→下一人模仿→改变 | 身体表达 |
| 声音+动作 | 3min | 圆圈中轮流做一个声音配一个动作 | 自发性训练 |
| 面部按摩 | 2min | 自我面部按摩→做不同表情 | 面部表达准备 |

### 1.2 关系暖身

| 活动 | 时间 | 操作 | 目的 |
|------|------|------|------|
| 镜像双人 | 5min | 两人面对面，一人带领一人跟随 | 同步+Tele |
| 名字+动作 | 5min | 每人说名字配一个动作，全组重复 | 破冰+认识 |
| 社会测量行走 | 5min | "走到和你最相似/不同的人身边" | 群体关系显化 |
| 信任圆 | 5min | 圆圈中一人闭眼轻摇，周围人接住 | 信任建立 |
| 共同叙事 | 5min | 一人一句编故事 | 创造性协作 |

### 1.3 自发性暖身

| 活动 | 时间 | 操作 | 目的 |
|------|------|------|------|
| Yes, And... | 5min | 即兴对话，每句以"是的，而且..."开始 | 接受+建构 |
| 状态变化 | 5min | 收到"切换"指令立即改变情绪状态 | 情绪灵活性 |
| 完成一句话 | 3min | "我此刻最想..."(快速轮流，不思考) | 自发表达 |
| 声音景观 | 5min | 全组闭眼，每人贡献一个声音 | 集体创造 |

## 2. 团体课程设计

### 2.1 标准90分钟戏剧疗愈课程

| 阶段 | 时间 | 内容 | 治疗师重点 |
|------|------|------|-----------|
| 开场圆圈 | 5min | 签到——一个词描述此刻状态 | 评估群体能量 |
| 身体暖身 | 10min | 物理层面的准备 | 降低防御 |
| 关系暖身 | 10min | 人际层面的连接 | 建立安全感 |
| 主题引入 | 5min | 介绍今天的主题/方法 | 清晰框架 |
| 核心活动 | 35-40min | 戏剧工作(详见各方法) | 引导+观察 |
| 反思 | 10min | 角色退出+体验反思 | 去角色、整合 |
| 闭场圆圈 | 10min | 每人分享一句收获/感受 | 确认+收尾 |

### 2.2 8周入门团体方案

| 周 | 主题 | 核心活动 | 距离 |
|----|------|---------|------|
| 1 | 相遇与安全 | 名字游戏+空间行走+团体守则 | 远 |
| 2 | 身体表达 | 身体雕塑+表情练习+定格画面 | 远 |
| 3 | 即兴基础 | "Yes, And"+状态变化+场景即兴 | 中-远 |
| 4 | 角色探索 | "如果我是..."角色扮演+角色采访 | 中 |
| 5 | 故事工作 | 个人故事戏剧化(自愿) | 中-近 |
| 6 | 面具工作 | 制作半面面具+面具即兴 | 中(投射) |
| 7 | 深度工作 | 心理剧元素(空椅/角色交换) | 近 |
| 8 | 整合告别 | 回顾旅程+仪式性闭场 | 中 |

## 3. 具体方法实操

### 3.1 故事戏剧化 (Story Dramatization)

**步骤**:
1. 参与者分享一个个人故事(或选用文学作品)
2. 提取关键场景和角色
3. 分配角色给团体成员
4. 排练和呈现(可多次修改)
5. 反思讨论

**距离调节**:

| 更远 | 中等 | 更近 |
|------|------|------|
| 使用文学作品 | 用自己的故事但改名 | 直接演自己的故事 |
| 第三人称叙述 | 混合虚实 | 第一人称 |
| 寓言/童话 | "类似的故事" | "就是那件事" |

### 3.2 即兴表演 (Improvisation)

**基本规则**:
- "Yes, And..." — 接受并建构
- 倾听比说话更重要
- 犯错是礼物
- 让搭档好看
- 追随直觉

**治疗性即兴练习**:

| 练习 | 操作 | 治疗功能 |
|------|------|---------|
| 状态变换 | 在对话中突然改变情绪 | 情绪灵活性 |
| 地位游戏 | 练习高地位/低地位 | 权力觉察 |
| 困境场景 | 即兴演出困难情境 | 应对rehearsal |
| 给予接收 | 轮流给予/接收礼物 | 关系模式 |

### 3.3 面具工作 (Mask Work)

**治疗价值**:
- 面具创造安全距离——"不是我在说，是面具在说"
- 投射内在状态到外在形式
- 探索自我的不同面向
- 面具的戏剧性放大效果

**流程**:
1. 制作面具(纸面具/石膏面具)
2. 观察面具——"你看到了什么？"
3. 戴上面具——"面具想说什么？想做什么？"
4. 面具即兴——在空间中行走、互动
5. 摘下面具——回到自己
6. 反思——"戴着面具时，你是谁？"

## 4. 特殊人群适配

### 4.1 儿童戏剧疗愈 (6-12岁)

| 原则 | 说明 |
|------|------|
| 游戏为核心 | 所有活动嵌入游戏框架 |
| 角色保护 | 通过虚构角色处理真实问题 |
| 结构化 | 清晰的开始-中间-结束 |
| 选择权 | 可以选择参与/观看/退出 |
| 动物/超级英雄 | 用非人类角色创造安全距离 |

### 4.2 青少年

| 方法 | 说明 |
|------|------|
| 即兴喜剧 | 降低防御，幽默中探索 |
| 角色面试 | "你是什么样的超级英雄？" |
| 社交场景排练 | 练习困难社交情境 |
| 被压迫者剧场 | 探索校园欺凌、家庭冲突 |

### 4.3 创伤群体

| 安全措施 | 说明 |
|---------|------|
| 高距离起步 | 用虚构故事、寓言、第三人称 |
| 双重觉察 | "一只脚在故事里，一只脚在这间房间里" |
| 退出机制 | 随时可以说"停"或做观众 |
| 去角色严格执行 | 防止角色粘连(Role Contamination) |
| 身体安全 | 不触碰规则(除非明确同意) |

## 5. 带领者能力清单

| 能力域 | 具体要求 |
|--------|---------|
| 戏剧技能 | 即兴表演、导演基础、场景构建 |
| 心理学知识 | 发展心理学、团体动力、创伤知情 |
| 团体带领 | 群体管理、冲突处理、能量调节 |
| 觉察能力 | 读取非言语线索、觉察群体动力 |
| 自我照顾 | 觉察自身反移情、定期督导 |
| 文化敏感 | 尊重多元文化背景和表达方式 |
| 伦理意识 | 保密、边界、知情同意 |
