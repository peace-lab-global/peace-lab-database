# 心理剧核心技术详解 | Psychodrama Core Techniques

> **文档定位**: J.L. Moreno心理剧体系中的核心技术操作详解与临床应用
> **关键词**: Role Reversal / Doubling / Mirroring / Empty Chair / Sociometry / Surplus Reality / Tele

---

## 1. 心理剧三要素

### 1.1 结构要素

| 要素 | 英文 | 角色 | 功能 |
|------|------|------|------|
| 主角 | Protagonist | 核心当事人 | 呈现个人议题 |
| 导演 | Director | 心理剧治疗师 | 引导过程、选择技术 |
| 辅角 | Auxiliary Ego | 团体成员 | 扮演主角生活中的重要他人 |
| 观众 | Audience | 未参与演出的团体成员 | 见证+共鸣+分享 |
| 舞台 | Stage | 演出空间 | 承载行动的物理空间 |

### 1.2 Moreno的核心概念

| 概念 | 英文 | 定义 |
|------|------|------|
| 自发性 | Spontaneity | 在新情境中给出新回应的能力 |
| 创造性 | Creativity | 产生新颖有价值行为/体验的能力 |
| Tele | Tele | 人与人之间的双向感知——吸引/排斥/中性 |
| 社会原子 | Social Atom | 个体最小的社交关系单元 |
| 文化罐头 | Cultural Conserve | 固化的行为模式/刻板反应 |
| 剩余现实 | Surplus Reality | 超越日常现实的戏剧空间(如与逝者对话) |
| 宣泄 | Catharsis | 通过行动获得的情绪释放与整合 |
| 行动洞察 | Action Insight | 在行动中(非言语中)获得的领悟 |

## 2. 核心技术详解

### 2.1 角色交换 (Role Reversal)

**操作**:
- 主角与场景中的他人(辅角扮演)交换位置
- 主角暂时"成为"对方，从对方的视角说话和感受

**治疗功能**:
- 发展同理心——体验对方的立场
- 突破自我中心视角
- 获取新信息——"从对方视角看到什么？"
- 解决人际冲突

**导演指令**: "现在请和他/她换一下位置。你变成了他/她。"

**注意事项**:
- 不适合与施虐者进行角色交换(可能造成再创伤)
- 不适合发展性障碍严重者(心智化能力不足)
- 需要在情感准备充分时才进行

### 2.2 替身技术 (Doubling)

**操作**:
- 治疗师或辅角站在主角身后/旁边
- 表达主角可能想说但未说出的话
- 主角可以确认、修改或拒绝替身的表达

**类型**:

| 类型 | 说明 | 适用 |
|------|------|------|
| 支持性替身 | 表达主角的感受 | 情感阻塞 |
| 矛盾替身 | 表达主角内心的另一面 | 内在冲突 |
| 挑战性替身 | 温和挑战主角的回避 | 阻抗 |
| 身体替身 | 关注并表达身体感受 | 身体化症状 |

**导演指令**: "我站在你身边，替你说出可能在内心的话。如果说对了，请重复；如果不对，请修改。"

### 2.3 镜像技术 (Mirroring)

**操作**:
- 辅角模仿主角的行为、姿势、语气
- 主角退到观众位置观看"自己"

**治疗功能**:
- 从第三人称视角看到自己的模式
- 增强自我觉察
- 突破盲点
- "这就是别人看到的你"

**导演指令**: "请退到观众区，让XX来扮演你。你从外面看一看。"

### 2.4 空椅技术 (Empty Chair)

**操作**:
- 放置一把空椅子代表缺席的人/自我的一部分
- 主角对空椅子说话
- 可加入角色交换——主角坐到空椅子上，成为对方回应

**适用场景**:

| 场景 | 说明 |
|------|------|
| 未完成事件 | 与已去世/失联的人对话 |
| 内在冲突 | 两个自我部分的对话 |
| 决策困难 | 两个选择的对话 |
| 关系修复 | 练习想对某人说但未说的话 |
| 自我慈悲 | 对"内在小孩"说话 |

### 2.5 雕塑技术 (Sculpting)

**操作**:
- 主角用身体姿势/空间距离/方向来呈现关系或内在状态
- 可用团体成员作为"人偶"来摆出家庭/关系图景

**应用**:

| 类型 | 说明 |
|------|------|
| 家庭雕塑 | 用空间距离和姿势呈现家庭关系动力 |
| 情绪雕塑 | 用身体姿势代表当前情绪状态 |
| 时间雕塑 | "过去的家庭" vs "现在的家庭" vs "理想家庭" |
| 组织雕塑 | 团队中的权力结构和关系模式 |

### 2.6 独白 (Soliloquy)

**操作**:
- 主角在场景中暂停行动
- 转向观众方向，大声说出此刻内心真实想法

**治疗功能**:
- 外化内在对话
- 揭示"台词背后的台词"
- 增强情感深度

### 2.7 未来投射 (Future Projection)

**操作**:
- 演出尚未发生的未来场景
- 主角可以"预演"困难对话或决定

**适用**: 求职面试、分手对话、与权威人物沟通、重要决策

## 3. 社会计量学 (Sociometry)

### 3.1 核心概念

**社会计量学** = 测量和映射群体中人际关系结构的方法

| 工具 | 说明 |
|------|------|
| 社会原子 | 画出个人核心关系圈(纸笔) |
| 行动社会计量 | 用身体空间距离呈现群体关系 |
| 社会计量图 | 群体中的选择/排斥网络可视化 |
| 量表暖身 | "站在同意-不同意的连续体上" |
| 旁观者分析 | 观察群体中的隐性结构 |

### 3.2 行动社会计量暖身活动

| 活动 | 指令 | 目的 |
|------|------|------|
| 光谱排列 | "按X维度排成一条线" | 群体差异觉察 |
| 选择与被选择 | "选一个你想和谁合作" | Tele关系显化 |
| 距离圈 | "站到离你最想探索的主题最近的地方" | 议题选择 |
| 锚点 | "谁是你在这个群体中的锚？走向那个人" | 安全连接 |

## 4. 剩余现实 (Surplus Reality)

### 4.1 概念

心理剧可以创造"剩余现实"——超越日常物理现实的治疗性空间：

| 类型 | 例子 |
|------|------|
| 与逝者对话 | "如果你妈妈现在在这里，你想对她说什么？" |
| 未出生的孩子 | 流产后与未出生孩子的对话 |
| 内在部分 | "你的愤怒如果是一个人，他看起来像什么？" |
| 未来自我 | "10年后的你来看看现在的自己" |
| 理想场景 | "如果当时你能做任何事，你会怎么做？" |

### 4.2 治疗价值

- 完成**未完成的事件** (Unfinished Business)
- 表达从未有机会说出的话
- 创造**修复性体验** (Corrective Emotional Experience)
- 与丧失连接——持续纽带 (Continuing Bonds)
- 探索内在世界的多面性

## 5. 心理剧的安全框架

### 5.1 导演的安全职责

| 职责 | 具体做法 |
|------|---------|
| 暖身评估 | 评估主角的准备程度和情绪承受力 |
| 知情同意 | 主角可以随时喊停 |
| 距离控制 | 在过于激烈时使用镜像/独白创造距离 |
| 收尾(De-role) | 演出后帮助辅角退出角色 |
| 分享保护 | 分享阶段禁止分析/评判/建议 |
| 后续关注 | 关注主角在团体结束后的状态 |

### 5.2 去角色 (De-roling)

**关键步骤**:
1. 明确宣布"场景结束"
2. 每个辅角说："我不是XXX(角色名)，我是XXX(本名)"
3. 脱去象征角色的物品(如围巾/帽子)
4. 辅角分享自己在角色中的体验
5. 主角确认"我看到的是演员，不是真人"
