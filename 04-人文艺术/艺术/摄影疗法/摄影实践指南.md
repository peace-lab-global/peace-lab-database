# 摄影疗愈实操与方案 | Photography Therapy Practice Guide

> **文档定位**: 摄影疗愈的具体实践方法、工作坊设计与特殊人群方案

## 1. 摄影疗愈作业设计

### 1.1 经典摄影疗愈作业

| 作业名称 | 指令 | 治疗目标 | 时间 |
|----------|------|---------|------|
| 一日三好 | 每天拍3张让你感到感恩/美好的事物 | 正向注意力偏向训练 | 每日5min×7天 |
| 情绪地图 | 在一周内拍摄代表不同情绪的照片 | 情绪觉察与命名 | 1周 |
| 安全之地 | 拍摄所有让你感到安全的地方/事物 | 安全感资源建立 | 3天 |
| 我的世界 | 拍摄10张代表你当前生活的照片 | 生活现状觉察 | 1周 |
| 过去-现在-未来 | 3组照片分别代表过去、现在和未来的自己 | 时间轴整合 | 2周 |
| 伤疤与力量 | 拍摄代表你伤痛和力量的事物各5张 | 创伤与韧性 | 1周 |
| 告别拍摄 | 拍摄你想告别的事物/地方/习惯 | 放手与过渡 | 1-2天 |
| 身体地景 | 拍摄自己身体的局部（手、脚、影子等） | 身体接纳 | 3天 |
| 家的故事 | 拍摄家中让你有感觉的角落和物品 | 家庭叙事 | 1周 |
| 对立面 | 拍摄对立的事物对（光/暗、新/旧、开/合） | 辩证思维、整合 | 3天 |

### 1.2 作业处理流程 (治疗师用)

| 阶段 | 提问 | 目的 |
|------|------|------|
| 描述 | "告诉我这张照片" | 了解来访者的视角 |
| 情感 | "看这张照片时，你有什么感觉？" | 情绪探索 |
| 意义 | "这张照片对你意味着什么？" | 个人意义建构 |
| 关联 | "这让你想到生活中的什么？" | 连接照片与生活 |
| 行动 | "如果这张照片能说话，它会说什么？" | 深化理解 |

## 2. 自拍像工作 (Self-Portrait Work)

### 2.1 治疗性自拍像方案

**与社交媒体自拍的区别**:

| 维度 | 治疗性自拍像 | 社交媒体自拍 |
|------|------------|------------|
| 目的 | 自我探索与接纳 | 展示与获取认可 |
| 观众 | 自己/治疗师 | 公众 |
| 修图 | 不修图——接纳真实 | 通常修图 |
| 评价 | 非评判探索 | 点赞/评论驱动 |
| 深度 | 内在对话 | 表面呈现 |

### 2.2 自拍像练习序列

| 阶段 | 练习 | 说明 |
|------|------|------|
| 1. 安全建立 | 只拍手/脚/影子 | 从非面部开始，降低暴露焦虑 |
| 2. 环境自我 | 在你喜欢的环境中拍自己（可不露脸） | 自我-环境关系 |
| 3. 情绪表达 | 用表情/姿势表达不同情绪 | 情绪具象化 |
| 4. 面对镜头 | 直视镜头拍一张"真实的我" | 自我直面 |
| 5. 多面自我 | 拍摄不同"角色"的自己 | 自我的多面性 |
| 6. 理想vs现实 | 分别拍"想呈现的我"和"真实的我" | 觉察差距与接纳 |

### 2.3 身体意象摄影工作

适用于进食障碍、身体羞耻感、外貌焦虑等议题：

| 练习 | 说明 | 注意事项 |
|------|------|---------|
| 身体局部赏析 | 拍摄自己喜欢的身体部位 | 先从容易的开始 |
| 功能性视角 | 拍摄身体在做事的照片（手在弹琴、腿在跑步） | 从外貌转向功能 |
| 友善滤镜 | 用拍朋友的心态拍自己 | 练习自我慈悲 |
| 光影身体 | 利用光影创造身体的抽象美 | 艺术化减少评判 |
| 时间线 | 收集从小到大的照片 | 观察身体的变化与恒定 |

## 3. 照片日记 (Photo Journal)

### 3.1 照片日记结构

每日一张照片 + 简短文字反思：

| 要素 | 说明 |
|------|------|
| 日期 | 记录日期 |
| 照片 | 当天最有意义/最吸引你的一个画面 |
| 标题 | 给照片一个标题（1-5个词） |
| 感受 | 拍摄时和此刻的感受各一词 |
| 反思 | 1-3句话——这张照片告诉了我什么？ |

### 3.2 主题照片日记

| 周期 | 主题 | 觉察方向 |
|------|------|---------|
| 30天感恩日记 | 每天拍一件感恩的事 | 正向心理学——培养感恩心态 |
| 21天情绪日记 | 每天拍一张代表当天情绪的照片 | 情绪觉察与追踪 |
| 14天关系日记 | 每天拍一个与人际关系相关的画面 | 关系模式觉察 |
| 7天身体日记 | 每天从不同角度拍自己 | 身体意象工作 |
| 30天正念日记 | 每天一个正念摄影主题 | 深化正念实践 |

## 4. 团体工作坊设计

### 4.1 标准摄影疗愈工作坊 (2小时)

| 阶段 | 时间 | 内容 |
|------|------|------|
| 开场 | 15min | 简介、设定意图、安全守则 |
| 技能mini课 | 10min | 手机摄影基础（构图、光线），不追求完美 |
| 拍摄任务 | 40min | 外出/室内完成指定拍摄作业 |
| 休息 | 10min | 个人翻看照片 |
| 照片选择 | 10min | 每人选出1-3张最有意义的照片 |
| 分享圈 | 30min | 轮流分享照片+故事（自愿） |
| 收尾 | 5min | 一句话总结体验 |

### 4.2 PhotoVoice社区项目 (8-12周)

| 周次 | 活动 | 产出 |
|------|------|------|
| 1-2 | 项目介绍、主题确定、摄影伦理培训 | 参与者知情同意 |
| 3-4 | 首轮拍摄 + 小组讨论 (SHOWeD) | 初始照片集 |
| 5-6 | 第二轮拍摄(深化主题) + 讨论 | 聚焦照片集 |
| 7-8 | 第三轮拍摄(行动导向) + 讨论 | 行动照片集 |
| 9-10 | 策展工作——选照片、写说明、设计展览 | 展览材料 |
| 11-12 | 公开展览/发布 + 政策对话 | 社区倡导 |

## 5. 特殊人群方案

### 5.1 老年人(含轻度认知障碍)

| 方法 | 说明 |
|------|------|
| 照片回忆法 | 翻看老照片→触发长期记忆→建构生命故事 |
| 代际摄影 | 与年轻志愿者合作拍摄→代际连接 |
| 遗产项目 | 拍摄/整理人生重要照片→制作生命故事册 |
| 简化操作 | 使用平板电脑/大屏幕，避免复杂操作 |

### 5.2 青少年

| 方法 | 说明 |
|------|------|
| 身份拼贴 | 用照片拼贴表达"我是谁" |
| 社交媒体觉察 | 分析自己的Instagram→觉察呈现模式 |
| 同伴互拍 | 两人一组互相拍摄→体验被看见 |
| 视觉日记 | 每日一图+一句话→情绪追踪 |

### 5.3 创伤后群体

| 注意事项 | 说明 |
|---------|------|
| 控制权 | 来访者完全控制拍什么/分享什么 |
| 节奏 | 来访者决定进度，治疗师不施压 |
| 安全阀 | 建立信号系统(来访者可随时喊停) |
| 避免再创伤 | 不要求拍摄创伤相关内容 |
| 见证性 | 照片作为"证人"——验证来访者的经历 |
| 美学转化 | 引导将痛苦体验转化为艺术表达 |

## 6. 伦理与安全

### 6.1 摄影疗愈特有的伦理问题

| 问题 | 处理原则 |
|------|---------|
| 照片中的第三方 | 获得被拍摄者知情同意 |
| 照片存储与隐私 | 明确照片归属权和存储方式 |
| 社交媒体分享 | 讨论分享的动机与风险 |
| 自拍与身体意象 | 对进食障碍患者需额外谨慎 |
| 创伤性图像 | 评估是否安全，是否需要心理准备 |
| 儿童照片 | 严格遵守儿童保护法规 |
| 数字遗产 | 去世后照片如何处理 |
