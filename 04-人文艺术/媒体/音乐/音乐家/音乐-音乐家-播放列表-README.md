# Playlists | 精选歌单

> 基于作曲家与演奏家知识库，按情绪与场景编排的古典音乐精选歌单

## 概述

本模块将库中的作曲家、演奏家档案转化为**可按场景直接使用的聆听歌单**。每张歌单围绕一种情绪状态或生活场景（如深夜独处、专注工作、哀伤陪伴）编排，选曲注明作品背景与聆听要点，帮助听者以更有觉知的方式与音乐相处——音乐不只是背景，而是被倾听、被同在的对象。

## 编排原则

- **情绪同频而非情绪矫正**：歌单不急于"驱走"负面情绪，而是选择与当下心境同频的作品，让情绪被理解、被陪伴，再自然流动。
- **作曲家叙事**：每张歌单结合作曲家的生平与创作语境（如肖邦的夜曲与流亡、拉赫玛尼诺夫的大病与康复），让聆听成为一场与生命故事的相遇。
- **疗愈视角**：参考音乐治疗研究中对节奏、调性与情绪调节关系的认识，标注适合的聆听时段与音量建议。
