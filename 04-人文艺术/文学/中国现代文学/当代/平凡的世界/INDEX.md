# The Ordinary World | 平凡的世界

> 路遥长篇小说，1986-1988年创作，1991年获茅盾文学奖。被誉为中国当代文学史上最具影响力的现实主义作品之一。

| 条目 | 内容 |
|------|------|
| 原著 | 《平凡的世界》 |
| 作者 | 路遥（王卫国） |
| 创作时间 | 1986-1988 |
| 出版年份 | 1986（第一部）、1988（第二三部） |
| 类型 | 现实主义/农村题材/社会史诗 |
| 获奖 | 第三届茅盾文学奖（1991） |
| 核心主题 | 奋斗、城乡二元、阶层流动、理想与现实、爱情与命运 |
| 时代背景 | 1975-1985年中国社会转型期 |
