# 《观呼吸》Mindfulness in Plain English

> 德宝法师（Bhante Henepola Gunaratana）著，1992年出版。斯里兰卡上座部佛教僧侣，12岁出家，1968年赴美弘法，在华盛顿特区附近建立了 bhavana 协会（森林修道院）。这本书是上座部佛教视角下最清晰、最实用的内观（Vipassana）禅修手册。

## 一、总体评价

德宝法师的书有一个罕见的特点：**它对初学者极其友好，但对有经验的修行者仍然有价值。** 他从最基础的"如何坐下""如何呼吸"开始，逐步深入到"观无常""观无我""观苦"的核心修法。每一步都有清晰的说明和常见的障碍分析。

与卡巴金的"去宗教化"正念不同，德宝法师明确地在**佛教框架**中教授禅修——但他用极其平实的英语来解释佛教概念，避免了大多数佛教书籍中的术语堆砌。

## 二、核心内容

### "观呼吸"的科学

德宝法师详细描述了"观呼吸"（Anapanasati）的每个阶段：

1. **数息**：吸气数"1"，呼气数"2"，数到10后回到1。目的是安定注意力
2. **随息**：不再数字，只是跟随呼吸的进出
3. **观息**：观察呼吸的质地——长或短、深或浅、顺畅或阻滞
4. **转化**：将呼吸作为基础，扩展到对身体、情绪、念头的全面觉知

### 障碍与应对

德宝法师花了大量篇幅讨论禅修中的常见障碍：

| 障碍 | 表现 | 应对方法 |
|------|------|---------|
| **昏沉** | 打瞌睡、头脑昏沉 | 睁大眼睛、改变姿势、用冷水洗脸 |
| **掉举** | 念头纷飞、坐立不安 | 标记念头（"想""计划""回忆"）、加强呼吸觉察 |
| **疼痛** | 腿痛、背痛 | 观察疼痛的质地、不立即移动、区分"疼痛"和"对疼痛的抗拒" |
| **恐惧** | 突然的恐慌或不安 | 标记"恐惧"、观察它在身体中的位置、知道它会过去 |
| **怀疑** | "这有用吗？""我做对了吗？" | 标记"怀疑"、回到呼吸、信任过程 |

### 对死亡的内观

德宝法师在书的后期讨论了如何用内观面对死亡：
- **观无常**：在每一次呼吸中观察"生起"和"灭去"——这个呼吸来了，走了；下一个呼吸来了，走了。生命也是如此
- **观身体分解**：在冥想中观想身体的各个部分（皮肤、肌肉、骨骼、器官），理解它们的 temporary 性
- **放下执着**：通过持续的观察，发现对"自我"的执着是如何造成痛苦的

## 三、阅读体验

**评分：9.0/10** — 这是最好的"实修手册"之一。德宝法师的声音像一位严厉但慈爱的 grandfather——他会告诉你"不要懒"，但同时会给你具体的工具来克服困难。如果你只想读一本关于"如何禅修"的书，就读这本。
