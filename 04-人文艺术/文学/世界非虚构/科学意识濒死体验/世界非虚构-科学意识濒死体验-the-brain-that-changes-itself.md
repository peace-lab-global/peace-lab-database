# The Brain That Changes Itself 导读 | 诺曼·道伊奇

## 概述

《The Brain That Changes Itself》（2007，中译《重塑大脑》）是加拿大精神病学家诺曼·道伊奇（Norman Doidge）的代表作，也是将"神经可塑性"（neuroplasticity）概念推向大众视野的里程碑式科普著作。本书通过一系列临床案例与科学家访谈，系统推翻了"成年大脑固定不变"的旧教条，论证大脑终生都能依据经验、思想与行为重塑自身的结构与功能。

## 核心内容

全书围绕三条主线展开：

1. **可塑性的科学证据**：梅策尼希的皮层地图重组实验、巴赫-利塔的感官替代装置、陶布的约束诱导运动疗法，分别证明感觉皮层、运动皮层在成年期仍可大规模重组。
2. **可塑性的双刃剑**：同样的机制既带来康复与学习，也可能固化强迫行为、成瘾回路和慢性疼痛。道伊奇以施瓦茨用认知行为技术治疗强迫症为例，说明"重新连接"神经回路的可能。
3. **思想改变物质**：正念冥想、心理治疗等心理层面的干预，可以在脑成像中产生可测量的结构变化——心灵并非大脑的被动产物。

## 实践与应用

- **康复医学**：中风、脑外伤患者的功能恢复训练提供了理论与方法依据。
- **心理治疗**：为 CBT、正念干预的疗效提供神经机制层面的解释框架。
- **教育与自我成长**："用进废退"原则提示持续学习、技能训练对认知储备的价值。
- **批判性阅读提示**：书中偏重戏剧性成功案例，读者需留意神经可塑性的边界与局限。
