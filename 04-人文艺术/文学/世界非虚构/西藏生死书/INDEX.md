# The Tibetan Book of Living and Dying | 西藏生死书

> 索甲仁波切（Sogyal Rinpoche）著，1992年出版。一部将藏传佛教生死观带给现代世界的里程碑式著作，被誉为"当代的西藏度亡经"。

## 基本信息

| 条目 | 内容 |
|------|------|
| 原著 | *The Tibetan Book of Living and Dying* |
| 中文书名 | 《西藏生死书》 |
| 作者 | 索甲仁波切（Sogyal Rinpoche，1947–2019） |
| 出版年份 | 1992（英文原著）；1996（台湾繁体中文版）；1999（中国大陆简体中文版） |
| 类型 | 灵性/佛学/临终关怀/生死学 |
| 核心主题 | 中阴教法、临终关怀、意识转化、生死无惧、心性本然 |
| 关联传统 | 藏传佛教宁玛派大圆满教法；现代临终关怀运动 |
| 全球销量 | 超过300万册，被翻译成56种语言 |
