```
□ 对自己的性别身份感到舒适

```

#### 身份状态测量工具
```
Marcia身份状态评估：

扩散状态识别：
• 缺乏明确的价值观和目标
• 对重要生活领域没有认真思考
• 容易受到他人影响和建议
• 对承诺感到恐惧和回避

早闭状态识别：
• 过早接受他人身份选择
• 缺乏个人探索过程
• 基于权威认同而非自我选择
• 思维方式较为僵化和封闭

延缓状态识别：
• 积极探索不同可能性
• 暂时处于不确定状态
• 开放心态和好奇心强
• 愿意承担探索的风险和挑战

达成状态识别：
• 经过充分探索后的明确承诺
• 基于个人价值观的选择
• 对选择有清晰的理由和理解
• 身份稳定且具有一致性
```

## 🎨 创意表达工具

### 艺术治疗活动包

#### 情绪表达艺术活动
```
绘画情绪地图：
• 材料：大张画纸、彩色笔、颜料
• 活动：用颜色和形状表达一周的情绪变化
• 目标：可视化情绪体验，增强情绪觉察
• 讨论：颜色选择的意义，情绪模式的识别

音乐情绪表达：
• 材料：音乐播放器、各种风格音乐
• 活动：选择能代表当前情绪的音乐并分享理由
• 目标：通过音乐连接和表达内在情感
• 扩展：创作属于自己的情绪主题音乐

身体雕塑表达：
• 材料：无特殊要求
• 活动：用身体姿态表达特定情绪或经历
• 目标：通过身体语言探索和表达内在状态
• 分享：解释雕塑的含义和感受
```
