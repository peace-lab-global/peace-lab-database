# 青少年临床干预实践指南

## 🛠️ 循证干预方法体系

### 辩证行为治疗青少年版 (DBT-A)

#### 核心技能模块

##### 正念技能训练
```
当下觉察培养：

观察技能：
• 不带评判地观察内在体验
• 注意身体感觉、情绪和思维
• 练习专注当下的能力
• 发展对经验的开放态度

描述技能：
• 准确描述观察到的现象
• 使用具体而非笼统的语言
• 避免解释和评判性词汇
• 培养客观描述的习惯

参与技能：
• 全身心投入到当前活动中
• 减少心不在焉的状态
• 增强活动的沉浸感
• 提高体验的丰富性
```

##### 情绪调节技能
```
情绪管理策略：

情绪识别：
• 准确识别和命名情绪状态
• 理解情绪的强度和持续时间
• 觉察情绪的身体伴随反应
• 识别情绪的触发因素

情绪检查：
• 评估情绪是否符合情境
• 分析情绪的合理性和适当性
• 考虑其他可能的解释
• 决定是否需要调节情绪

情绪调节：
• 增加积极情绪活动
• 减少情绪脆弱性因素
• 应对情绪冲动行为
• 建立健康的情绪表达方式
```

##### 人际效能技能
```
关系技能发展：

目标设定：
• 明确人际互动的具体目标
• 区分关系目标和即时目标
• 考虑长远关系维护
• 平衡自我需求和他人需求

策略选择：
• DEAR MAN技巧（Describe, Express, Assert, Reinforce, Mindful, Appear confident, Negotiate）
• GIVE技巧（Gentle, Interested, Validate, Easy manner）
• FAST技巧（Fair, Apologies, Stick to values, Truthful）

效果评估：
• 评估互动结果是否达到目标
• 反思使用的策略是否有效
• 考虑替代策略的可能性
• 为下次互动积累经验
```

##### 痛苦耐受技能
```
危机应对技术：

现实接受：
• ACCEPTS技巧（Activities, Contributing, Comparisons, Emotions, Pushing away, Thoughts, Sensations）
• IMPROVE技巧（Imagery, Meaning, Prayer, Relaxation, One thing, Vacation, Encouragement）

危机生存：

• 专业帮助寻求
```
