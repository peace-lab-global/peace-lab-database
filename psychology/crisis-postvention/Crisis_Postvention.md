# Crisis Postvention & Recovery (危机后干预与康复)

## 核心术语表 (Core Terminology)

| 中文术语 | 英文术语 | 来源语言/词源 | 定义 | 临床意义 |
| :--- | :--- | :--- | :--- | :--- |
| 事后干预 | Postvention | 英语 *post-* + *prevention* | 危机事件发生后对受影响者的干预措施 | 预防连锁反应、促进康复 |
| 危机后康复 | Post-Crisis Recovery | 英语 | 危机事件后个体或系统恢复功能的过程 | 长期目标 |
| 心理减压 | Psychological Debriefing | 英语 | 危机事件后促进情绪表达的结构化谈话 | 争议：可能有害 |
| 心理急救 | Psychological First Aid (PFA) | 英语 | 危机后即时支持的循证方法 | 推荐的即时干预 |
| 复发预防 | Relapse Prevention | 英语 | 预防症状或问题行为复发的策略 | 维持治疗成果 |
| 创伤后成长 | Post-Traumatic Growth (PTG) | 英语 | 经历创伤后的正向心理变化 | 积极适应结果 |
| 复原力 | Resilience | 拉丁语 *resilire* (弹回) | 面对逆境后适应和恢复的能力 | 保护性因素 |
| 连续照护 | Continuum of Care | 英语 | 从急性期到长期康复的连贯服务体系 | 确保无缝衔接 |
| 社区康复 | Community-Based Recovery | 英语 | 在社区环境中进行的康复支持 | 去机构化方向 |
| 同侪支持 | Peer Support | 英语 | 有相似经历者之间的互助支持 | 共同体验的力量 |

## 危机后即时干预

### 心理急救(PFA)核心行动表

| 行动要素 | 英文名称 | 核心内容 | 具体做法 | 目标 |
| :--- | :--- | :--- | :--- | :--- |
| **接触与参与** | Contact and Engagement | 以尊重、非侵入的方式接触 | 自我介绍、询问需求、保持平静 | 建立信任 |
| **安全与舒适** | Safety and Comfort | 确保即时安全与基本需求 | 提供安全环境、满足生理需求 | 稳定化 |
| **稳定化** | Stabilization | 帮助情绪失控者平复 | 接地技术、呼吸练习、陪伴 | 情绪调节 |
| **信息收集** | Information Gathering | 了解当前需求与关注点 | 开放式问题、倾听、不强迫 | 评估需求 |
| **实际帮助** | Practical Assistance | 帮助解决当前实际问题 | 协助联系家人、提供资源信息 | 减轻压力 |
| **连接支持** | Connection with Social Support | 帮助与支持系统连接 | 联系亲友、社区资源 | 激活支持 |
| **应对信息** | Information on Coping | 提供正常化与应对指导 | 心理教育、自我照顾建议 | 赋能 |
| **连接服务** | Linkage with Services | 连接后续专业服务 | 转介信息、协助预约 | 持续支持 |

### PFA与心理减压(Debriefing)对比表

| 对比维度 | 心理急救(PFA) | 心理减压(Debriefing) |
| :--- | :--- | :--- |
| **时机** | 危机后即时（数小时-数天） | 危机后24-72小时 |
| **形式** | 灵活、个别化 | 结构化、常为团体 |
| **核心** | 实际支持、稳定化 | 事件回顾、情绪表达 |
| **证据基础** | 广泛支持 | 争议大、可能有害 |
| **WHO推荐** | 推荐 | 不推荐强制 |
| **培训要求** | 较低 | 较高 |
| **适用范围** | 广泛人群 | 特定团体 |

### 危机后第一时间支持清单表

| 支持维度 | 具体内容 | 提供者 | 时间框架 |
| :--- | :--- | :--- | :--- |
| **安全确认** | 确保人身安全、远离危险 | 急救/安全人员 | 即刻 |
| **医疗需求** | 处理身体伤害 | 医疗人员 | 即刻 |
| **基本需求** | 食物、水、保暖、休息 | 救助人员 | 数小时内 |
| **信息提供** | 发生了什么、接下来怎么办 | 协调人员 | 稳定后 |
| **家属联络** | 联系亲友 | 社工/志愿者 | 尽快 |
| **情绪支持** | 陪伴、倾听 | 受过培训者 | 持续 |
| **资源连接** | 后续服务信息 | 社工/协调员 | 24-48小时 |

## 自杀事后干预专题

### 自杀丧亲者(Suicide Loss Survivors)支持框架表

| 支持阶段 | 时间框架 | 核心任务 | 关键干预 | 注意事项 |
| :--- | :--- | :--- | :--- | :--- |
| **即时支持** | 事发后数小时-数天 | 稳定、信息、实际帮助 | 陪伴、不评判、实际安排协助 | 不催促表达 |
| **早期哀伤** | 事发后数周-数月 | 哀伤处理、正常化 | 心理教育、支持小组、咨询 | 复杂性评估 |
| **持续支持** | 数月-数年 | 意义重建、适应 | 持续咨询、支持团体、纪念 | 周年反应 |
| **长期整合** | 持续 | 整合丧失、继续生活 | 按需支持、连接社区 | 成长可能 |

### 自杀丧亲者特殊哀伤反应表

| 反应类型 | 具体表现 | 与普通丧亲区别 | 支持策略 |
| :--- | :--- | :--- | :--- |
| **强烈内疚** | "我本可以阻止"、反复假设 | 更强烈、更难处理 | 认知处理、正常化 |
| **寻找原因** | 不断追问"为什么" | 更强烈的意义寻找需求 | 容纳不确定性 |
| **愤怒** | 对逝者、对自己、对他人的愤怒 | 更复杂的愤怒对象 | 允许表达、正常化 |
| **羞耻/污名** | 社会污名、隐藏死因 | 独特的社会压力 | 去污名化、社区连接 |
| **创伤反应** | 闪回、噩梦、过度警觉 | 创伤与哀伤交织 | 创伤处理 |
| **自身风险** | 自杀风险增加 | 直接风险因素 | 风险评估与监测 |
| **关系影响** | 家庭动力改变、冲突 | 家庭系统受创 | 家庭支持 |
| **身份重构** | "我是谁"的问题 | 身份改变 | 意义重建 |

### 自杀事件学校/组织事后干预清单表

| 干预领域 | 即刻行动（24-48h） | 短期行动（1-2周） | 长期行动（数月） |
| :--- | :--- | :--- | :--- |
| **信息管理** | 确认信息、准备声明、谣言控制 | 持续澄清、媒体应对 | 纪念指导 |
| **人员支持** | 支持站、心理急救、高危筛查 | 个别咨询、小组支持 | 持续监测、周年关注 |
| **环境调整** | 安全检查、敏感区域处理 | 恢复常规、纪念空间 | 环境优化 |
| **沟通协调** | 内部通报、家长沟通、家属联络 | 持续更新、需求评估 | 反馈总结 |
| **资源调配** | 启动危机团队、外部支援 | 资源协调、转介通道 | 资源整合、能力建设 |

### 自杀事件媒体沟通指南表

| 原则 | 应该做 | 不应该做 | 原因 |
| :--- | :--- | :--- | :--- |
| **语言选择** | "死于自杀" | "自杀成功"、"自杀未遂失败" | 避免暗示成就 |
| **细节披露** | 简短事实性描述 | 详细描述方法、地点 | 防止模仿 |
| **原因归因** | 多因素、复杂性 | 简单归因、责怪 | 准确、减少污名 |
| **资源提供** | 附上求助热线信息 | 不提供任何资源 | 帮助脆弱读者 |
| **框架叙述** | 强调可预防性 | 美化、浪漫化 | 传递希望 |
| **图像使用** | 避免或谨慎使用 | 刺激性图片 | 减少触发 |
| **版面位置** | 适度篇幅 | 头版大标题 | 减少传染效应 |

## 创伤事件后干预

### 关键事件压力管理(CISM)完整体系表

| 组成部分 | 英文名称 | 时间节点 | 目标人群 | 核心内容 |
| :--- | :--- | :--- | :--- | :--- |
| **事前教育** | Pre-Incident Education | 日常 | 全员 | 心理健康素养、预期管理 |
| **现场支持** | On-Scene Support | 事件中 | 现场人员 | 实际支持、稳定化 |
| **个人减压** | Individual Defusing | 事件后数小时 | 受影响个人 | 1:1简短支持 |
| **团体减压** | Group Defusing | 事件后8-12h | 小团体 | 简短团体稳定 |
| **危机事件压力汇报** | CISD (Critical Incident Stress Debriefing) | 事件后24-72h | 团体 | 结构化团体讨论 |
| **家庭支持** | Family Support | 持续 | 家庭成员 | 家庭心理教育、支持 |
| **后续服务** | Follow-up Services | 事件后数周 | 按需 | 个别咨询、转介 |
| **组织咨询** | Organizational Consultation | 持续 | 组织 | 组织层面干预 |

### CISD七阶段结构表

| 阶段 | 英文名称 | 时间 | 核心问题 | 目的 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 介绍** | Introduction | 5-10分钟 | 规则、保密、流程 | 建立安全 |
| **2. 事实** | Fact Phase | 10-15分钟 | 发生了什么？你在哪里？ | 事件重构 |
| **3. 想法** | Thought Phase | 10-15分钟 | 你的第一个想法是什么？ | 认知反应 |
| **4. 反应** | Reaction Phase | 15-20分钟 | 最糟糕的部分是什么？ | 情绪表达 |
| **5. 症状** | Symptom Phase | 10-15分钟 | 你有什么反应/症状？ | 正常化 |
| **6. 教育** | Teaching Phase | 15-20分钟 | 这些反应是正常的、应对方法 | 心理教育 |
| **7. 重入** | Re-entry Phase | 10-15分钟 | 总结、资源、后续 | 闭合 |

### 创伤后正常反应心理教育表

| 反应类型 | 具体表现 | 正常持续时间 | 需关注的情况 | 自助建议 |
| :--- | :--- | :--- | :--- | :--- |
| **侵入性症状** | 闪回、噩梦、闯入性思维 | 数周内渐减 | 4周后仍严重 | 限制媒体、接地 |
| **回避反应** | 不想谈论、回避提示物 | 初期正常 | 长期影响功能 | 渐进面对 |
| **过度警觉** | 惊跳、睡眠困难、易怒 | 数周内 | 持续影响生活 | 放松训练 |
| **情绪波动** | 悲伤、愤怒、恐惧、内疚 | 数周内 | 持续强烈 | 情绪表达 |
| **认知困难** | 注意力下降、记忆困难 | 初期正常 | 长期损害 | 降低要求 |
| **躯体反应** | 头痛、胃肠不适、疲劳 | 数周内 | 持续恶化 | 身体照顾 |
| **社交变化** | 退缩或过度依赖 | 初期正常 | 长期孤立 | 保持连接 |

## 自杀企图后康复

### 自杀企图后康复阶段表

| 康复阶段 | 时间框架 | 核心任务 | 关键干预 | 监测重点 |
| :--- | :--- | :--- | :--- | :--- |
| **急性稳定期** | 企图后0-72小时 | 医学稳定、安全保障 | 医疗处置、1:1监护、评估 | 生命体征、自杀风险 |
| **住院治疗期** | 数天-数周 | 症状控制、安全计划 | 药物调整、心理干预开始 | 症状变化、出院准备 |
| **过渡期** | 出院后1-4周 | 安全过渡、连接服务 | 密集随访、手段限制执行 | 最高风险期 |
| **早期康复期** | 1-3个月 | 治疗建立、技能学习 | 规律治疗、安全计划执行 | 治疗依从、功能恢复 |
| **持续康复期** | 3-12个月 | 巩固成果、复发预防 | 持续治疗、复发预防计划 | 稳定性、生活质量 |
| **长期管理期** | 1年以上 | 维持、成长 | 按需治疗、持续监测 | 预警信号、总体功能 |

### 出院后过渡支持核心要素表

| 要素 | 具体内容 | 执行者 | 时间要求 |
| :--- | :--- | :--- | :--- |
| **出院前会议** | 患者、家属、团队共同计划 | 住院团队 | 出院前1-2天 |
| **安全计划更新** | 根据最新情况更新安全计划 | 治疗师 | 出院前 |
| **手段限制确认** | 确认家庭环境安全 | 社工+家属 | 出院前 |
| **门诊预约** | 确定首次门诊时间 | 协调员 | 出院前确认（7天内） |
| **24小时电话** | 出院后24小时内电话跟进 | 指定人员 | 出院后24h |
| **危机联系卡** | 提供紧急联系信息 | 护士 | 出院时 |
| **家属教育** | 预警信号、手段限制、求助 | 社工/护士 | 出院前 |
| **药物管理** | 药物教育、安全储存 | 药剂师+家属 | 出院前 |

## 复发预防

### 自杀复发预防计划要素表

| 要素 | 具体内容 | 个性化示例 | 更新频率 |
| :--- | :--- | :--- | :--- |
| **个人预警信号** | 识别自己危机恶化的早期信号 | 失眠加重、社交退缩、脑中反复出现某想法 | 每次会谈检查 |
| **高危情境** | 可能触发危机的情境 | 独处过久、酒后、与某人冲突后 | 按需更新 |
| **内部应对策略** | 自己可以做的事情 | 听音乐、洗澡、写日记、深呼吸 | 根据效果调整 |
| **社交支持** | 可以联系的人 | 妈妈、好友小李、支持小组 | 保持更新 |
| **专业资源** | 专业帮助联系方式 | 治疗师电话、热线、急诊 | 确保有效 |
| **手段限制** | 环境安全措施 | 药物由家人保管、无利器在手边 | 持续执行 |
| **活着的理由** | 个人意义与连接 | 孩子、宠物、想实现的目标 | 定期强化 |
| **过去有效的方法** | 以前帮助度过危机的方法 | 运动、与某人聊天、去某个地方 | 积累记录 |

### 危机后复发风险因素监测表

| 风险因素 | 监测指标 | 观察方法 | 响应阈值 |
| :--- | :--- | :--- | :--- |

| **物质使用** | 使用频率、量 | 每次会谈询问 | 使用增加 |
| **社交退缩** | 社交活动、支持联系 | 每次会谈评估 | 明显减少 |
| **治疗依从** | 出席率、用药依从 | 追踪记录 | 脱落风险 |
| **生活压力** | 重大事件、日常压力 | 每次会谈评估 | 累积压力 |
| **支持系统变化** | 关系状况 | 每次会谈评估 | 关键支持丧失 |
| **周年/触发日期** | 企图周年、节日等 | 日历提醒 | 接近日期 |

### 精神科药物康复期管理表

| 管理维度 | 具体内容 | 责任方 | 注意事项 |
| :--- | :--- | :--- | :--- |
| **药物依从** | 规律服药、不自行停药 | 患者+家属+医生 | 不依从是复发主因 |

### 自杀丧亲支持小组运作要素表

| 要素 | 内容 | 最佳实践 |
| :--- | :--- | :--- |
| **小组规模** | 8-12人 | 足够分享，不过于大 |
| **带领者** | 受训的同侪或专业人员 | 混合模式效果好 |
| **频率** | 每周或每两周一次 | 规律性重要 |
| **持续时间** | 每次90-120分钟 | 足够但不过长 |
| **封闭/开放** | 封闭小组（同一批人）或开放 | 封闭组深度更好 |
| **主题** | 哀伤议题、应对技能、资源 | 结构与灵活结合 |
| **规则** | 保密、尊重、不建议、轮流发言 | 建立安全空间 |

## 家庭支持与康复

### 危机后家庭教育核心内容表

| 教育主题 | 核心信息 | 常见误解 | 正确理解 |
| :--- | :--- | :--- | :--- |
| **危机理解** | 危机是可恢复的 | 有过危机就永远有问题 | 多数人可以康复 |
| **预警信号** | 学习识别恶化信号 | 无法预测 | 信号可以学习识别 |
| **沟通方式** | 开放、非评判的沟通 | 谈论会加重问题 | 开放对话有益 |
| **支持边界** | 支持但不过度保护 | 要24小时监控 | 平衡安全与自主 |
| **自我照顾** | 家属也需要照顾自己 | 必须牺牲自己 | 照顾好自己才能帮助他人 |
| **专业帮助** | 知道何时需要专业帮助 | 自己应该能解决 | 专业帮助不是失败 |
| **康复期望** | 康复需要时间，可能有反复 | 应该快速好转 | 康复是过程而非事件 |

### 家庭成员自我照顾策略表

| 策略类别 | 具体做法 | 益处 | 障碍应对 |
| :--- | :--- | :--- | :--- |
| **社会支持** | 联系亲友、加入支持小组 | 减少孤立、获得理解 | "别人不理解" → 找同类经历者 |
| **情绪表达** | 日记、咨询、与信任者交流 | 情绪释放 | "我应该坚强" → 表达不是软弱 |
| **身体健康** | 规律睡眠、运动、营养 | 基础能量 | "没时间" → 短时间也有效 |
| **界限设定** | 明确能做与不能做的边界 | 防止耗竭 | "应该无条件支持" → 边界是健康的 |
| **休息时间** | 允许自己休息、做喜欢的事 | 恢复能量 | "内疚" → 休息后更能帮助 |
| **专业支持** | 如需要，为自己寻求咨询 | 处理复杂情绪 | "我不是患者" → 家属也受影响 |
| **知识学习** | 学习相关知识 | 增加控制感 | "太多信息焦虑" → 按需选择 |
| **现实期望** | 调整对康复的期望 | 减少挫败感 | "应该已经好了" → 康复需时间 |

### 家庭会议/沟通框架表

| 会议要素 | 具体内容 | 技巧 |
| :--- | :--- | :--- |
| **设定时间** | 选择合适、不受打扰的时间 | 预先约定、尊重边界 |
| **明确目的** | 这次讨论的具体目标 | 一次一个议题 |
| **轮流发言** | 每人有发言机会 | 使用发言物 |
| **倾听规则** | 不打断、不评判 | 反映理解 |
| **"我"陈述** | 表达感受而非指责 | "我感到担心当..." |
| **问题解决** | 共同寻找解决方案 | 头脑风暴、协商 |
| **总结行动** | 明确下一步行动 | 谁做什么、什么时候 |
| **结束正向** | 以正向内容结束 | 感谢、肯定 |

## 创伤后成长

### 创伤后成长(PTG)五维度表

| 维度 | 英文名称 | 具体表现 | 促进因素 |
| :--- | :--- | :--- | :--- |
| **与他人关系** | Relating to Others | 更珍惜关系、同理心增加、愿意接受帮助 | 社会支持、分享经历 |
| **新的可能性** | New Possibilities | 发现新的兴趣、方向、角色 | 意义寻找、行动探索 |
| **个人力量** | Personal Strength | 发现自己比想象中更强大 | 成功应对经验、自我效能 |
| **灵性变化** | Spiritual Change | 灵性/存在意义的深化 | 存在性反思、灵性实践 |
| **生命欣赏** | Appreciation of Life | 更珍惜生命、活在当下 | 正念、感恩实践 |

### 促进创伤后成长的干预策略表

| 策略 | 具体方法 | 理论基础 | 注意事项 |
| :--- | :--- | :--- | :--- |
| **叙事建构** | 讲述经历、写作、意义寻找 | 意义重建 | 不强迫、尊重时机 |
| **认知处理** | 挑战负面信念、发展适应性信念 | 认知理论 | 不过早正向重构 |
| **优势聚焦** | 识别应对中展现的优势 | 优势视角 | 同时承认痛苦 |
| **社会支持** | 分享经历、接受支持 | 社会支持理论 | 选择合适的人 |
| **有意义行动** | 利他行为、倡导活动 | 意义治疗 | 尊重个人选择 |
| **灵性实践** | 冥想、宗教实践、存在反思 | 灵性健康 | 文化敏感 |
| **感恩练习** | 感恩日记、积极回顾 | 积极心理学 | 不否定痛苦 |

### 成长与痛苦共存理解表

| 概念 | 说明 | 临床应用 |
| :--- | :--- | :--- |
| **共存性** | 成长和痛苦可以同时存在 | 不要求"好起来"才能成长 |
| **非替代性** | 成长不能替代或否定痛苦 | 承认痛苦的真实性 |
| **非目标性** | 成长是可能的结果而非必须的目标 | 不强加成长期望 |
| **时间性** | 成长通常在一段时间后出现 | 不过早追求 |
| **个体差异** | 不是每个人都会经历成长 | 尊重不同的轨迹 |
