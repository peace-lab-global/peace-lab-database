## 使用说明

### 本指南的定位

本指南提供一套结构化的自我评估工具，帮助个体：
- 初步判断自己是否存在失眠问题及其严重程度
- 识别失眠的具体类型（入睡困难、睡眠维持困难、早醒、睡眠拖延）
- 发现维持失眠的行为和认知因素
- 决定是否需要寻求专业帮助

### 重要声明

- 本指南中的量表和评估工具仅用于**筛查和自我认识**，不能替代专业诊断
- 如果评估结果提示中重度问题，请及时就医
- 评估结果可能受到当前情绪状态、近期事件等因素影响
- 建议在平静状态下完成评估，并回顾过去一个月的整体情况

## 综合自查清单

### 第一步：我的睡眠有问题吗？

请根据过去一个月的情况，回答以下问题：

**核心症状筛查**（回答"是"或"否"）：

| 编号 | 问题 | 是/否 |
|---|---|---|
| 1 | 我经常需要超过 30 分钟才能入睡 | |
| 2 | 我经常在夜间醒来 2 次以上，且难以再次入睡 | |
| 3 | 我经常比预期早醒 1 小时以上，且无法再入睡 | |
| 4 | 我明知应该睡觉，却主动拖延不去上床 | |
| 5 | 我白天经常感到疲劳、困倦或精力不足 | |
| 6 | 我的睡眠问题影响了工作、学习或社交功能 | |
| 7 | 上述问题每周出现 3 次或以上 | |
| 8 | 上述问题已持续 1 个月以上 | |

**解读**：
- 回答"是" **≤ 2 项**：睡眠总体正常，偶有波动属于正常范围
- 回答"是" **3-5 项**：可能存在睡眠问题，建议完成下方详细评估
- 回答"是" **6-8 项**：较大概率存在失眠障碍，建议就诊评估

### 第二步：我的失眠属于什么类型？

根据最困扰你的症状，判断失眠主要类型：

| 类型 | 核心表现 | 典型人群 | 首选干预方向 |
|---|---|---|---|
| **入睡困难型** | 躺在床上 > 30 分钟无法入睡 | 年轻人、焦虑型 | 刺激控制 + 认知重构 |
| **睡眠维持困难型** | 夜间频繁醒来（≥2次），总觉醒 > 30 分钟 | 中老年、疼痛患者 | 睡眠限制 + 排除共病 |
| **早醒型** | 比预期早醒 ≥ 1-2 小时且无法再入睡 | 抑郁症患者、老年人 | 排查抑郁 + 光照治疗 |
| **睡眠拖延型** | 能睡但不想睡，主动推迟入睡时间 | 年轻职场人、学生 | 行为策略 + 需求分析 |
| **混合型** | 以上 2 种或以上同时存在 | 慢性失眠者 | 综合 CBT-I |

## ISI 失眠严重程度指数

### 量表说明
ISI（Insomnia Severity Index）是全球使用最广泛的失眠筛查工具，由 Morin 教授开发，共 7 题，评估过去两周的失眠严重程度。

### 正式评估

请圈选每题最符合你近两周情况的选项（0-4 分）：

**1. 你目前入睡困难的严重程度：**
- 0 = 没有  1 = 轻微  2 = 中度  3 = 严重  4 = 非常严重

**2. 你目前睡眠维持困难（夜间或凌晨醒来）的严重程度：**
- 0 = 没有  1 = 轻微  2 = 中度  3 = 严重  4 = 非常严重

**3. 你目前早醒的严重程度：**
- 0 = 没有  1 = 轻微  2 = 中度  3 = 严重  4 = 非常严重

**4. 你对目前的睡眠模式满意/不满意的程度：**
- 0 = 非常满意  1 = 满意  2 = 一般  3 = 不满意  4 = 非常不满意

**5. 你的睡眠问题在多大程度上干扰了日间功能（如疲劳、情绪、注意力、记忆力、情绪等）：**
- 0 = 没有干扰  1 = 轻微  2 = 中度  3 = 严重  4 = 非常严重

**6. 在他人看来，你的睡眠问题在多大程度上影响了你的生活质量：**
- 0 = 没有影响  1 = 轻微  2 = 中度  3 = 严重  4 = 非常严重

**7. 你对目前睡眠问题的担忧/困扰程度：**
- 0 = 没有  1 = 轻微  2 = 中度  3 = 严重  4 = 非常严重

### 评分与解读

**总分 = 7 题分数之和（范围 0-28 分）**

| 分数范围 | 严重程度 | 建议 |
|---|---|---|
| **0-7 分** | 无临床意义的失眠 | 无需特殊处理，维持良好睡眠习惯 |
| **8-14 分** | 亚临床失眠/轻度失眠 | 可通过睡眠卫生教育和自助策略改善 |
| **15-21 分** | 中度失眠（临床显著） | 建议寻求专业评估，推荐 CBT-I |
| **22-28 分** | 重度失眠 | 强烈建议尽快就诊睡眠专科或精神科 |

## PSQI 匹兹堡睡眠质量指数简化版

### 核心维度快速自评

以下为 PSQI 7 大维度的简化评估，请根据过去一个月情况作答：

| 维度 | 问题 | 0分 | 1分 | 2分 | 3分 |
|---|---|---|---|---|---|
| **主观睡眠质量** | 你如何评价整体睡眠质量？ | 非常好 | 较好 | 较差 | 非常差 |
| **入睡时间** | 通常需要多久才能入睡？ | ≤15min | 16-30min | 31-60min | >60min |
| **睡眠时长** | 每晚实际睡眠时间？ | >7h | 6-7h | 5-6h | <5h |
| **睡眠效率** | 在床时间中实际睡着的比例？ | >85% | 75-84% | 65-74% | <65% |
| **睡眠障碍** | 夜间是否有觉醒、噩梦、疼痛等困扰？ | 无 | 偶尔(1次/周) | 有时(2次/周) | 经常(≥3次/周) |
| **使用助眠药物** | 是否使用药物帮助入睡？ | 无 | <1次/周 | 1-2次/周 | ≥3次/周 |
| **日间功能障碍** | 白天是否困倦或影响工作？ | 无 | 轻微 | 较大 | 非常大 |

**总分 = 7 维度分数之和（范围 0-21 分）**

| 分数 | 判断 |
|---|---|
| **≤ 5 分** | 睡眠质量良好 |
| **6-10 分** | 睡眠质量一般，存在可改善空间 |
| **11-15 分** | 睡眠质量较差，建议采取积极干预 |
| **> 15 分** | 睡眠质量很差，建议就医评估 |

## 入睡困难专项评估

请根据过去一个月的情况作答（0=从不，1=偶尔，2=有时，3=经常，4=几乎每晚）：

| 编号 | 问题 | 得分(0-4) |
|---|---|---|
| 1 | 我躺在床上超过 30 分钟仍无法入睡 | |
| 2 | 睡前我的脑子里充满各种想法，无法平静 | |
| 3 | 我躺在床上感觉身体很紧张 | |
| 4 | 我担心"今晚能不能睡着" | |
| 5 | 我反复看时间，计算还能睡几个小时 | |
| 6 | 我在沙发、车上比在自己床上更容易入睡 | |
| 7 | 越到上床时间我越感到清醒 | |

**评分解读**：
- **0-7 分**：入睡功能基本正常
- **8-14 分**：轻度入睡困难，改善睡眠卫生可能有效
- **15-21 分**：中度入睡困难，建议专业评估和 CBT-I
- **22-28 分**：重度入睡困难，建议尽快就诊

## 睡眠拖延专项评估

请根据过去一个月的情况作答（0=从不，1=偶尔，2=有时，3=经常，4=几乎每晚）：

| 编号 | 问题 | 得分(0-4) |
|---|---|---|
| 1 | 到了应该睡觉的时间，我觉得"再玩一会儿" | |
| 2 | 我明知应该早睡，但就是不想上床 | |
| 3 | 睡着意味着"属于自己的时间结束了"，我很抗拒 | |
| 4 | 我在床上刷手机直到困得不行才放下 | |
| 5 | 白天决心"今晚一定早睡"，到了晚上又做不到 | |
| 6 | 我觉得白天没有足够的自由时间 | |
| 7 | 我推迟睡觉不是因为睡不着，而是因为不想睡 | |

**评分解读**：
- **0-7 分**：睡眠意愿正常
- **8-14 分**：轻度睡眠拖延，建立睡前仪式可改善
- **15-21 分**：中度睡眠拖延，需关注白天生活满意度
- **22-28 分**：重度睡眠拖延，建议评估深层心理需求

**补充判断**：如果第 3、6 题得分较高，可能属于"报复性睡前拖延"，核心问题在于白天自主需求未被满足。

## 睡眠功能失调信念评估

以下是常见的关于睡眠的不合理信念，请判断你对每条的认同程度（1=完全不同意，5=完全同意）：

| 编号 | 信念 | 认同程度(1-5) | 合理替代观点 |
|---|---|---|---|
| 1 | 我必须睡够 8 小时才能正常运作 | | 6-9 小时都是正常范围，因人而异 |
| 2 | 如果晚上没睡好，第二天一定会崩溃 | | 身体有弹性，偶尔睡不好影响有限 |
| 3 | 我应该能像别人一样轻松入睡 | | 每个人入睡特点不同，不存在统一标准 |
| 4 | 失眠可能导致严重的健康问题 | | 偶尔失眠很常见，慢性失眠才需关注 |
| 5 | 我已经失去了正常入睡的能力 | | 入睡能力不会"丧失"，可以通过调整恢复 |
| 6 | 唯一的解决办法是药物治疗 | | CBT-I 是首选治疗，药物并非唯一选择 |
| 7 | 如果躺在床上翻来覆去，应该继续等待入睡 | | 20 分钟未入睡应起床做放松活动 |
| 8 | 白天打盹可以弥补晚上的睡眠不足 | | 过多白天睡眠会削弱夜间睡眠驱动力 |

**评分解读**：
- 总分 **8-16 分**：睡眠认知较为合理
- 总分 **17-28 分**：存在部分不合理信念，可通过学习调整
- 总分 **29-40 分**：存在较多功能失调信念，认知重构是重要干预靶点

## 睡眠日记模板

### 使用说明
连续记录 2 周，每天早起后填写。这是 CBT-I 的核心评估工具，也是改善睡眠的有效自助手段。

### 每日记录表

| 记录项目 | 今日记录 | 说明 |
|---|---|---|
| **日期** | ____年__月__日（周__） | |
| **上床时间** | __:__ | 躺到床上关灯的时间 |
| **入睡时间** | __:__ | 估计入睡的时间 |
| **入睡潜伏期** | ___分钟 | 上床到入睡的估计时间 |
| **夜间觉醒次数** | ___次 | 入睡后醒来的次数 |
| **夜间总觉醒时间** | ___分钟 | 所有夜间清醒时间之和 |
| **最终醒来时间** | __:__ | 最后一次醒来不再入睡的时间 |
| **起床时间** | __:__ | 实际离开床的时间 |
| **总在床时间（TIB）** | ___小时___分钟 | 上床到起床的总时间 |
| **估计总睡眠时间（TST）** | ___小时___分钟 | 实际睡着的时间 |
| **睡眠效率（SE）** | __% | TST / TIB x 100% |
| **睡眠质量评分** | 1-5 | 1=非常差，5=非常好 |
| **白天精力评分** | 1-5 | 1=极差，5=精力充沛 |
| **午睡时间** | __:__ 到 __:__ | 如有 |
| **咖啡因摄入** | 有/无（___时） | 最后一次含咖啡因饮品时间 |
| **酒精摄入** | 有/无（___时） | |
| **运动** | 有/无（___时，___分钟） | |
| **助眠药物** | 有/无（药名:___，剂量:___） | |
| **备注** | | 特殊事件、情绪状态等 |

### 关键指标计算

经过 2 周记录后，计算以下平均值：

| 指标 | 计算方法 | 你的平均值 | 参考范围 |
|---|---|---|---|
| 平均入睡潜伏期 | 所有入睡潜伏期之和 / 天数 | ___分钟 | 正常 < 30 分钟 |
| 平均总睡眠时间 | 所有 TST 之和 / 天数 | ___小时 | 成人 6-9 小时 |
| 平均睡眠效率 | 所有 SE 之和 / 天数 | ___% | 正常 > 85% |
| 夜间平均觉醒次数 | 所有觉醒次数之和 / 天数 | ___次 | 正常 ≤ 1 次 |

## 何时寻求专业帮助

### 就医指征

出现以下任何一种情况，建议及时就医：

**紧急就医**：
- 失眠伴随严重的自杀或自伤念头
- 失眠导致严重的日间功能损害（如无法正常工作、驾驶时困倦）
- 突然出现的、严重的睡眠改变（可能提示躯体疾病）

**尽早就医**：
- ISI 评分 ≥ 15 分（中度及以上失眠）
- 失眠持续 3 个月以上且每周 ≥ 3 晚
- 怀疑有睡眠呼吸暂停（打鼾、呼吸暂停、白天过度嗜睡）
- 怀疑有不宁腿综合征（下肢不适伴运动冲动）
- 失眠伴随显著的焦虑或抑郁症状
- 已经自行尝试多种方法无效

**建议就医**：
- ISI 评分 8-14 分且自助方法改善有限
- 长期依赖安眠药物
- 睡眠问题导致人际关系紧张

## 中国就诊指南

### 科室选择

| 科室 | 适合人群 | 优势 |
|---|---|---|
| **睡眠医学中心/睡眠门诊** | 各类失眠和睡眠障碍 | 最专业，可做 PSG 等检查 |
| **精神心理科** | 失眠伴焦虑、抑郁等心理问题 | 心理评估全面，可开精神类药物 |
| **神经内科** | 疑似神经系统疾病导致的失眠 | 可排除神经系统器质性病变 |
| **中医科** | 偏好中医调理 | 中医辨证论治，副作用较小 |
| **心理咨询中心** | 想做 CBT-I 等心理治疗 | 提供系统的心理行为干预 |

### 就诊准备

就诊前建议准备：
1. **2 周睡眠日记**：最有价值的客观资料
2. **本指南的评估结果**：帮助医生快速了解你的情况
3. **用药清单**：目前服用的所有药物（包括保健品）
4. **问题清单**：想向医生提出的问题

### 推荐就诊机构类型

- **三甲医院睡眠中心**：设备齐全，可进行多导睡眠图检查
- **精神卫生中心**：心理评估全面，心理治疗资源丰富
- **综合医院心理科**：兼顾躯体疾病排查
- **正规心理咨询机构**：提供 CBT-I 等循证心理治疗

### 费用参考（仅供参考，以实际为准）

| 项目 | 大致费用范围 |
|---|---|
| 睡眠门诊挂号 | 50-300 元 |
