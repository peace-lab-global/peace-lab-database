# Delusional Disorder Overview (偏执性精神病概览)

> **偏执性精神病 / 妄想性障碍 (Delusional Disorder / Paranoid Psychosis)**
> 
> *"偏执性精神病的核心困境在于：患者的信念如此真实地嵌入他们的世界，以至于质疑本身就是一种威胁。"*
> *"The core dilemma of delusional disorder is that the patient's beliefs are so convincingly embedded in their world that questioning itself becomes a threat."*
> 
> 偏执性精神病（妄想性障碍）是以一个或多个持续的、非奇异性妄想为核心特征的精神障碍。与精神分裂症不同，患者的整体功能相对保留，行为通常不显著怪异。本文档提供偏执性精神病的分类、诊断标准及鉴别诊断框架。本病是中国国家卫健委六类严重精神障碍之一，详见 [严重精神障碍总览](../Severe_Mental_Illness_Overview.md)。

## 1. 核心术语与词源表 (Core Terminology & Etymology)

| 术语 (中文) | 英文名 (English Term) | 原始语言/词源 (Original Language) | 词源含义 | 精确定义 | 历史演变 | 专业视角 (Professional Perspective) |
|---|---|---|---|---|---|---|
| **偏执性精神病** | Delusional Disorder / Paranoid Psychosis | 希腊语: *Paranoia* (παράνοια, 超越理性) | "超出正常思维" | 以持续的非奇异性妄想为核心，功能相对保留的精神障碍 | 19世纪Kraepelin描述为"偏执狂(Paranoia)" | 中国传统称"偏执性精神病"；DSM/ICD称"妄想性障碍" |
| **非奇异性妄想** | Non-Bizarre Delusion | - | 内容在现实中可能发生 | 妄想内容涉及现实中可能发生的情境（如被跟踪、中毒、配偶不忠） | Jaspers区分了奇异性与非奇异性 | 区别于精分中的奇异性妄想（如思维被控制） |
| **系统性妄想** | Systematized Delusion | - | 结构完整的妄想体系 | 妄想围绕一个中心主题形成逻辑连贯的信念系统 | 偏执性精神病的特征性表现 | 逻辑性强但前提错误 |
| **被害妄想** | Persecutory Delusion | 拉丁语: *Persequi* (追逐) | "被追赶的信念" | 坚信自己被跟踪、监视、陷害或迫害 | 最常见的妄想类型 | 偏执性精神病中最常见亚型 |
| **钟情妄想** | Erotomania | 希腊语: *Erōs* (ἔρως, 爱) + *Mania* | "爱的疯狂" | 坚信某个通常社会地位较高的人深爱着自己 | 又称de Clérambault综合征 | 可涉及跟踪/骚扰行为 |
| **嫉妒妄想** | Jealous Type (Othello Syndrome) | - | - | 坚信配偶或伴侣不忠 | 又称奥赛罗综合征 | 可导致暴力行为；需评估安全 |

## 2. 分类与诊断标准 (Classification & Diagnostic Criteria)

### 表2.1 DSM-5 妄想性障碍诊断标准 (DSM-5 Delusional Disorder Criteria)

| 标准 | 内容 | 详细说明 | 评估要点 |
|---|---|---|---|
| **A** | 一个或多个妄想，持续≥1个月 | 妄想通常为非奇异性 | 评估信念的固定性和可纠正性 |
| **B** | 从未达到精神分裂症标准A | 若有幻觉，则不突出且与妄想主题相关 | 排除精分的其他核心症状 |
| **C** | 功能未明显受损，行为不显著怪异 | 除妄想直接影响的领域外，功能保持良好 | 整体功能评估 |
| **D** | 若有心境发作，其持续时间相对于妄想期较短 | 排除心境障碍伴精神病性 | 时间关系评估 |
| **E** | 排除物质/躯体疾病所致 | 标准排除 | 医学检查 |

### 表2.2 妄想性障碍亚型 (Subtypes of Delusional Disorder)

| 亚型 | 英文名 | ICD-11 | 核心妄想内容 | 患病比例 | 性别倾向 | 特殊风险 |
|---|---|---|---|---|---|---|
| **被害型** | Persecutory Type | 6A24.0 | 被跟踪、监视、投毒、陷害 | ~50%（最常见） | 男女相当 | 可能因"自卫"而攻击"迫害者" |
| **嫉妒型** | Jealous Type | 6A24.1 | 配偶/伴侣不忠 | ~5-10% | 男性多见 | **暴力风险高**——需评估伴侣安全 |
| **钟情型** | Erotomanic Type | 6A24.2 | 某人（通常地位较高）深爱自己 | ~5% | 女性多见 | 跟踪骚扰行为；法律问题 |
| **躯体型** | Somatic Type | 6A24.3 | 身体有缺陷、感染、散发异味 | ~5-10% | 男女相当 | 反复就诊躯体科；不必要的手术 |
| **夸大型** | Grandiose Type | 6A24.4 | 有特殊才能、身份、发现或与重要人物关系 | ~10% | 男性多见 | 需与双相躁狂鉴别 |
| **混合型** | Mixed Type | 6A24.5 | 多于一种妄想主题，无单一主导 | 少见 | - | 更复杂 |
| **未特定型** | Unspecified Type | 6A24.Z | 不符合以上任何亚型 | - | - | - |

## 3. 与精神分裂症的关键区别 (Key Differences from Schizophrenia)

### 表3.1 偏执性精神病 vs 精神分裂症 (Delusional Disorder vs. Schizophrenia)

| 特征 | 偏执性精神病 | 精神分裂症 |
|---|---|---|
| **妄想类型** | 非奇异性（现实可能发生） | 可为奇异性（如思维被植入） |
| **妄想结构** | 系统化、逻辑连贯 | 可能零散、不连贯 |
| **幻觉** | 无或与妄想相关且不突出 | 常见且突出（尤其幻听） |
| **思维形式障碍** | 无 | 可有联想松弛、思维破裂 |
| **阴性症状** | 无或极轻 | 常见且显著 |
| **认知功能** | 基本保留 | 广泛损害 |
| **社会功能** | 妄想领域外保留良好 | 广泛下降 |
| **发病年龄** | 中年（40-55岁多见） | 青年期（18-35岁） |
| **病程** | 相对稳定；少数可缓解 | 慢性波动 |
| **自知力** | 缺乏（对妄想坚信不疑） | 缺乏 |
| **治疗反应** | 对药物反应差；治疗联盟困难 | 对抗精神病药反应较好 |

## 4. 神经生物学与病因 (Neurobiology & Etiology)

### 表4.1 病因学因素 (Etiological Factors)

| 因素 | 证据 | 临床意义 |
|---|---|---|
| **遗传** | 家族聚集性低于精分；与偏执型人格障碍可能共享遗传基础 | 遗传贡献较精分低 |
| **多巴胺** | 纹状体DA功能可能异常（证据有限） | 抗精神病药的理论基础 |
| **社会因素** | 社会隔离、移民、听力障碍、感觉剥夺增加风险 | 社会孤立可能促进妄想形成 |
| **认知偏差** | "跳跃到结论"(Jumping to Conclusions)偏差显著 | 认知干预的靶点 |
| **人格特征** | 多疑、敏感、自我中心、固执 | 与偏执型人格障碍的关系 |
| **年龄相关** | 感觉衰退（听力/视力）增加晚发风险 | 老年患者需评估感觉功能 |

## 5. 流行病学与预后 (Epidemiology & Prognosis)

### 表5.1 流行病学数据 (Epidemiological Data)

| 指标 | 数据 | 备注 |
|---|---|---|
| **终生患病率** | 0.02-0.03% | 远低于精分 |
| **占精神科住院的比例** | 1-4% | 多数在门诊管理 |
| **发病年龄** | 平均40-55岁 | 晚于精分 |
| **性别比** | 男女大致相当（嫉妒型男多；钟情型女多） | 亚型有性别差异 |

### 表5.2 预后 (Prognosis)

| 预后结局 | 比例 | 影响因素 |
|---|---|---|
| **完全缓解** | ~20-30% | 急性起病、有应激源、女性 |
| **部分改善** | ~30-40% | 治疗依从性好 |
| **慢性持续** | ~30-50% | 起病隐匿、被害型、男性 |
| **进展为精分** | <5% | 罕见 |

## 6. 鉴别诊断 (Differential Diagnosis)

### 表6.1 鉴别诊断要点 (Differential Diagnosis)

| 鉴别疾病 | 共同特征 | 鉴别要点 |
|---|---|---|
| **精神分裂症（偏执型）** | 妄想 | 精分有幻觉、阴性症状、认知损害、功能下降 |
| **偏执型人格障碍** | 多疑、敏感 | 人格障碍无固定妄想；模式终生存在 |
| **双相障碍伴精神病性** | 夸大妄想/被害妄想 | 双相妄想仅在心境发作期出现 |
| **强迫症** | 固定信念 | OCD有自知力、视为不合理、有强迫行为 |
| **躯体变形障碍** | 对身体的固定信念 | BDD可被部分说服；关注点为外貌 |
| **疑病障碍** | 对健康的担忧 | 焦虑性质而非妄想性质 |
| **痴呆** | 偏执/多疑 | 有认知下降的客观证据 |
