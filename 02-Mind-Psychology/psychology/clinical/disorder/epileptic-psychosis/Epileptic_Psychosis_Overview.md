# Epileptic Psychosis Overview (癫痫所致精神障碍概览)

> **癫痫所致精神障碍 (Epileptic Psychosis / Psychosis of Epilepsy)**
> 
> *"癫痫与精神病的交汇是神经科学最引人入胜的临床谜题之一——同一个大脑，两种风暴。"*
> *"The intersection of epilepsy and psychosis is one of the most fascinating clinical puzzles in neuroscience — one brain, two storms."*
> 
> 癫痫所致精神障碍是指在癫痫病程中出现的精神病性症状，包括发作期、发作后和发作间期精神病三种类型。其独特之处在于精神病性症状与癫痫发作存在时间关联。本病是中国国家卫健委六类严重精神障碍之一，详见 [严重精神障碍总览](../Severe_Mental_Illness_Overview.md)。

## 1. 核心术语与词源表 (Core Terminology & Etymology)

| 术语 (中文) | 英文名 (English Term) | 原始语言/词源 (Original Language) | 词源含义 | 精确定义 | 历史演变 | 专业视角 (Professional Perspective) |
|---|---|---|---|---|---|---|
| **癫痫所致精神障碍** | Psychosis of Epilepsy (POE) | 希腊语: *Epilepsia* (ἐπιληψία, 抓住) | "被抓住的状态" | 在癫痫病程中出现的精神病性症状 | 古希腊即有癫痫与精神异常的记载 | 需神经科-精神科联合管理 |
| **发作期精神病** | Ictal Psychosis | 拉丁语: *Ictus* (打击/发作) | "发作时的" | 癫痫发作期间出现的精神病性症状（部分复杂发作/非惊厥性癫痫持续状态） | 常被误诊为原发性精神病 | 需EEG确认；治疗以控制发作为主 |
| **发作后精神病** | Postictal Psychosis (PIP) | 拉丁语: *Post-* (之后) + *Ictus* | "发作之后的" | 癫痫发作（尤其丛集发作）后出现的精神病性症状，通常有明确间期 | Logsdail & Toone (1988) 系统描述 | 有特征性"清醒间期"（24-72小时） |
| **发作间期精神病** | Interictal Psychosis (IIP) | 拉丁语: *Inter-* (之间) + *Ictus* | "发作之间的" | 与癫痫发作无直接时间关联的慢性精神病性症状 | 与精分最相似；临床鉴别困难 | 可能需长期抗精神病治疗 |
| **强制正常化** | Forced Normalization (Alternative Psychosis) | - | EEG正常化伴精神病 | 抗癫痫治疗有效控制发作后，EEG恢复正常但出现精神病性症状 | Landolt (1953) 首次描述 | 临床罕见但理论重要；治疗矛盾 |
| **颞叶癫痫** | Temporal Lobe Epilepsy (TLE) | - | 起源于颞叶的癫痫 | 最常与精神病性症状相关的癫痫类型 | 与精神病关联最密切的癫痫亚型 | 颞叶内侧（海马、杏仁核）异常放电 |

## 2. 分类体系 (Classification)

### 表2.1 癫痫所致精神障碍三大类型 (Three Types of Epileptic Psychosis)

| 特征 | 发作期精神病 (Ictal) | 发作后精神病 (Postictal/PIP) | 发作间期精神病 (Interictal/IIP) |
|---|---|---|---|
| **与发作的时间关系** | 发作期间 | 发作后24-72小时（有清醒间期） | 与发作无直接时间关联 |
| **持续时间** | 数秒-数分钟（发作持续时间） | 数天-2周 | 数周-数月-慢性 |
| **发生率** | 癫痫患者中7-10% | 癫痫患者中2-7.8% | 癫痫患者中2-7% |
| **常见症状** | 自动症、凝视、意识模糊中的幻觉/恐惧/被害感 | 幻觉（视觉>听觉）、偏执、激越、情感不稳 | 类精分症状：幻听、被害妄想、思维障碍 |
| **意识状态** | 常有意识模糊/改变 | 意识清楚（清醒间期后） | 意识清楚 |
| **EEG表现** | 发作性异常放电 | 可见发作后变化 | 通常为发作间期异常或正常 |
| **癫痫类型关联** | 部分性发作（颞叶多见） | 丛集发作后；颞叶癫痫多见 | 长期颞叶癫痫（通常病程>10年） |
| **治疗核心** | 控制癫痫发作 | 短期低剂量抗精神病药 | 长期抗精神病药+维持抗癫痫药 |
| **预后** | 随发作控制而缓解 | 通常自限性 | 可为慢性 |

### 表2.2 诊断标准 (Diagnostic Criteria)

| 标准体系 | 分类 | 代码 | 要求 |
|---|---|---|---|
| **DSM-5** | 由其他医学状况引起的精神病性障碍 | 293.81/293.82 | 有癫痫诊断+精神病性症状+因果关系证据 |
| **ICD-11** | 癫痫所致精神障碍 | 6E62 | 同上 |

## 3. 发作后精神病 (Postictal Psychosis) — 详解

### 表3.1 PIP 诊断标准 (Logsdail & Toone, 1988 修订)

| 标准 | 内容 |
|---|---|
| **A** | 有明确的癫痫发作（通常为丛集发作或全面性强直阵挛发作） |
| **B** | 发作与精神病性症状之间有清醒间期（通常24-72小时） |
| **C** | 精神病性症状持续≥15小时但<2个月 |
| **D** | 排除抗癫痫药物毒性、非惊厥性癫痫持续状态、既往发作间期精神病 |

### 表3.2 PIP 临床特征 (Clinical Features)

| 特征 | 表现 | 频率 |
|---|---|---|
| **清醒间期** | 发作后24-72小时的表面正常期 | 特征性（诊断要求） |
| **幻觉** | 视觉幻觉常见（区别于精分的幻听为主） | 60-70% |
| **妄想** | 被害妄想、宗教/夸大妄想 | 50-80% |
| **情感症状** | 情感不稳、恐惧、狂喜 | 40-60% |
| **激越/攻击** | 可出现激越或攻击行为 | 20-30% |
| **意识改变** | 部分患者有轻度意识模糊 | 变化 |
| **自限性** | 通常1-2周内自行缓解 | 大多数 |
| **复发** | 约50%的PIP患者会再次发作 | 高 |

## 4. 神经生物学机制 (Neurobiological Mechanisms)

### 表4.1 癫痫与精神病的神经生物学联系

| 机制 | 说明 | 临床关联 |
|---|---|---|
| **颞叶-边缘系统异常** | 海马硬化、杏仁核异常放电 | TLE与精神病的特殊关联 |
| **多巴胺调节异常** | 反复发作导致DA系统敏化 | 抗精神病药有效的基础 |
| **谷氨酸兴奋毒性** | 反复发作导致神经元损伤 | 长期癫痫→IIP的可能机制 |
| **GABA-谷氨酸失衡** | 抑制-兴奋失衡 | 某些AED可能加重精神病 |
| **强制正常化** | 发作完全控制→异常活动"转向"精神症状 | Landolt理论；治疗矛盾 |
| **双重病理** | 神经发育异常同时影响癫痫和精神病易感性 | 共同的神经发育基础 |

### 表4.2 癫痫相关精神病风险因素 (Risk Factors)

| 风险因素 | 风险增加 | 备注 |
|---|---|---|
| **颞叶癫痫** | 最高 | 尤其左侧颞叶/双侧 |
| **癫痫病程长（>10年）** | 高 | IIP的重要风险 |
| **难治性癫痫** | 高 | 发作频繁增加风险 |
| **丛集发作** | 高（PIP） | PIP的直接触发因素 |
| **精神病家族史** | 中 | 遗传易感性 |
| **双侧EEG异常** | 中 | 比单侧风险高 |
| **抗癫痫药物调整** | 中 | 某些AED减量/停药可触发 |

## 5. 鉴别诊断 (Differential Diagnosis)

### 表5.1 癫痫所致精神病 vs 原发性精神病 (Epileptic vs. Primary Psychosis)

| 特征 | 癫痫所致精神病 | 精神分裂症 |
|---|---|---|
| **癫痫病史** | 有 | 无（但2-4%精分患者有癫痫共病） |
| **精神病性症状起病** | 通常在癫痫诊断后数年 | 青年期独立起病 |
| **与发作的时间关系** | PIP有清晰时间关系 | 无 |
| **幻觉类型** | 视觉幻觉更常见 | 听觉幻觉（幻听）为主 |
| **情感保留** | 情感通常保留较好 | 可有情感平淡 |
| **意识状态** | Ictal和PIP可有意识改变 | 通常意识清楚 |
| **阴性症状** | 轻或无 | 可能显著 |
| **EEG** | 常有癫痫样放电 | 通常正常 |
| **病程** | PIP自限性；IIP可慢性 | 慢性为主 |
