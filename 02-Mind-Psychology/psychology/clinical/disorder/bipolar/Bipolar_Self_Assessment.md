# Bipolar Disorder Self-Assessment & Screening (双相障碍自我评估与筛查指南)

> **双相障碍自我评估与筛查 (Bipolar Disorder Self-Assessment & Screening)**
> 
> *"识别心境的两极，是理解自己的第一步。"*
> *"Recognizing the bipolar extremes of mood is the first step to understanding yourself."*
> 
> 本文档旨在帮助个体初步识别双相障碍相关症状，了解专业评估流程，并提供循证自评工具。**自评工具不能替代专业诊断，仅作为筛查参考。**

---

## 1. 自我诊断的科学边界 (The Science of Self-Diagnosis)

### 表1.1 自我评估vs.专业诊断 (Self-Assessment vs. Professional Diagnosis)

| 维度 | 自我评估 | 专业诊断 |
|---|---|---|
| **目的** | 识别可能的双相障碍倾向；决定是否寻求专业评估 | 确认或排除双相障碍诊断；制定治疗方案 |
| **工具** | 标准化自评量表、症状检核表 | 结构化访谈(DIVA-5/SCID)、多来源评估、知情者访谈 |
| **准确性** | 敏感性较好但特异性有限（假阳性较高） | 更全面和准确 |
| **局限性** | 无法区分双相与其他心境障碍、人格障碍 | 需专业培训；资源可能有限 |
| **价值** | 启动自我认知；为就诊做准备 | 获得确诊；制定规范治疗方案 |
| **风险** | 过度自我诊断或错误归因；忽视其他疾病 | 误诊风险（尤其轻躁狂易被忽视） |

### 表1.2 为什么双相障碍特别需要专业诊断 (Why Professional Diagnosis is Crucial)

| 原因 | 解释 |
|---|---|
| **轻躁狂难以自我识别** | 患者常将轻躁狂视为"最佳状态"或"正常"，不认为是病态 |
| **与单相抑郁混淆** | 约40%的双相患者首诊被误诊为单相抑郁，误诊平均延迟5-10年 |
| **治疗方向截然不同** | 单用抗抑郁药治疗双相抑郁可能诱发躁狂或快速循环 |
| **自杀风险极高** | 双相障碍自杀风险是普通人群的20-30倍，需要及时干预 |
| **共病复杂** | 约60-70%有共病（焦虑、物质使用、ADHD等），需全面评估 |
| **病程评估重要** | 需评估终生发作史，单次截面评估不足以诊断 |
| **功能损害持续** | 即使发作间期也可能有残留症状和功能损害 |

## 2. 心境障碍问卷 (Mood Disorder Questionnaire / MDQ)

> **MDQ**是国际上最广泛使用的双相障碍筛查工具，由Hirschfeld等人开发，已被翻译成多种语言。

### 表2.1 MDQ完整版 (MDQ Full Version)

**指导语**：请回顾您的整个生命历程，回答以下问题。

#### Part A：核心症状（13题）

| 序号 | 问题 | 是 | 否 |
|---|---|---|---|
| **1** | 有没有一段时间，您感觉非常好或情绪高涨，以至于别人认为您不像平时的自己？或者您过度兴奋到惹上麻烦？ | ☐ | ☐ |
| **2** | 有没有一段时间，您非常烦躁，以至于对人大喊大叫、开始争吵或打架？ | ☐ | ☐ |
| **3** | 有没有一段时间，您感觉比平时自信得多？ | ☐ | ☐ |
| **4** | 有没有一段时间，您睡眠比平时少得多，但发现您并不真的需要那么多睡眠？ | ☐ | ☐ |
| **5** | 有没有一段时间，您比平时话多得多，或说话速度比平时快得多？ | ☐ | ☐ |
| **6** | 有没有一段时间，您的思绪飞快，以至于无法跟上，或者脑子里的想法飞速转动？ | ☐ | ☐ |
| **7** | 有没有一段时间，您很容易被周围的事物分心，以至于难以集中注意力或保持专注？ | ☐ | ☐ |
| **8** | 有没有一段时间，您比平时精力充沛得多？ | ☐ | ☐ |
| **9** | 有没有一段时间，您比平时活跃得多，或做的事情比平时多得多？ | ☐ | ☐ |
| **10** | 有没有一段时间，您比平时更善于社交或外向，比如半夜打电话给朋友？ | ☐ | ☐ |
| **11** | 有没有一段时间，您对性的兴趣比平时大得多？ | ☐ | ☐ |
| **12** | 有没有一段时间，您做了一些不寻常的事情，或者别人认为过度、愚蠢或冒险的事情？ | ☐ | ☐ |
| **13** | 有没有一段时间，您花钱让自己或家人陷入麻烦？ | ☐ | ☐ |

#### Part B：症状同时出现

**如果您在上面多个问题中选择了"是"，这些症状是否有在同一时间段内一起出现？**

| ☐ 是 | ☐ 否 |

#### Part C：功能损害程度

**这些症状给您带来了多大程度的问题？**（如：无法工作、家庭/金钱/法律问题、争吵或打架）

| ☐ 无问题 | ☐ 轻微问题 | ☐ 中度问题 | ☐ 严重问题 |

### 表2.2 MDQ评分与解读 (MDQ Scoring & Interpretation)

| 评分标准 | 阳性筛查结果 |
|---|---|
| **Part A** | ≥7项回答"是" |
| **Part B** | 回答"是"（症状同时出现） |
| **Part C** | ≥中度问题 |
| **三项标准同时满足** | MDQ筛查阳性，建议专业评估 |

| 指标 | 数值 |
|---|---|
| **敏感性（精神科样本）** | 约73% |
| **特异性（精神科样本）** | 约90% |
| **敏感性（社区样本）** | 约28%（较低） |
| **注意** | MDQ对双相II型的敏感性较低 |

## 3. 轻躁狂检测表 (Hypomania Checklist-32 / HCL-32)

> **HCL-32**专门设计用于筛查双相II型障碍中容易被忽视的轻躁狂症状，敏感性优于MDQ。

### 表3.1 HCL-32核心条目 (HCL-32 Core Items)

**指导语**：请回想您处于"情绪高涨"或"最佳状态"时期的感受和行为（不是您"正常"状态，而是明显感觉好于平常的时期）。

| 序号 | 在这种"高涨"时期，您是否... | 是 | 否 |
|---|---|---|---|
| **1** | 睡眠需求减少（感觉睡得少但精力仍然充沛） | ☐ | ☐ |
| **2** | 感觉精力更充沛、更活跃 | ☐ | ☐ |
| **3** | 更加自信 | ☐ | ☐ |
| **4** | 更加享受工作 | ☐ | ☐ |
| **5** | 更加善于社交（打更多电话、参加更多聚会） | ☐ | ☐ |
| **6** | 想要旅行，实际上也旅行更多 | ☐ | ☐ |
| **7** | 开车更快，或冒更多风险驾驶 | ☐ | ☐ |
| **8** | 花钱更多、花得更随意 | ☐ | ☐ |
| **9** | 更加冒险（在商业、日常生活中） | ☐ | ☐ |
| **10** | 身体更活跃（运动等） | ☐ | ☐ |
| **11** | 计划更多活动或项目 | ☐ | ☐ |
| **12** | 想法更多 | ☐ | ☐ |
| **13** | 注意力更不容易集中 | ☐ | ☐ |
| **14** | 思维更快 | ☐ | ☐ |
| **15** | 更加不耐烦或易激惹 | ☐ | ☐ |
| **16** | 可能让别人感到疲惫或烦躁 | ☐ | ☐ |
| **17** | 更容易与人发生争执或争论 | ☐ | ☐ |
| **18** | 情绪更好、更乐观 | ☐ | ☐ |
| **19** | 喝更多咖啡 | ☐ | ☐ |
| **20** | 抽更多烟 | ☐ | ☐ |
| **21** | 喝更多酒 | ☐ | ☐ |
| **22** | 服用更多药物（镇静剂、抗焦虑药等） | ☐ | ☐ |
| **23** | 话更多 | ☐ | ☐ |
| **24** | 更有创造力、更多想法 | ☐ | ☐ |
| **25** | 对异性更感兴趣，或性欲增强 | ☐ | ☐ |
| **26** | 更加调情，或性行为更活跃 | ☐ | ☐ |
| **27** | 更加健谈 | ☐ | ☐ |
| **28** | 更加快速思考 | ☐ | ☐ |
| **29** | 开更多玩笑或双关语 | ☐ | ☐ |
| **30** | 更容易分心 | ☐ | ☐ |
| **31** | 参与许多新事物 | ☐ | ☐ |
| **32** | 更加任性或无所顾忌 | ☐ | ☐ |

### 表3.2 HCL-32评分与解读 (HCL-32 Scoring & Interpretation)

| 评分标准 | 解读 |
|---|---|
| **阳性阈值** | ≥14项回答"是" |
| **敏感性** | 约80%（优于MDQ对双相II型的识别） |
| **特异性** | 约51% |
| **最佳用途** | 筛查双相II型、识别轻躁狂史 |

**补充问题评估**：
1. 这种"高涨"状态持续多长时间？（1天 / 2-3天 / 4-7天 / 超过1周 / 超过1月 / 不确定）
2. 这种状态对您的影响是？（正面 / 负面 / 正负兼有 / 无影响）
3. 周围人对这种状态的反应是？（正面 / 困惑/担心/烦躁 / 无反应 / 中性/不确定）

## 5. 双相抑郁vs.单相抑郁筛查 (Bipolar vs. Unipolar Depression Screen)

### 表5.1 双相抑郁高危特征识别 (High-Risk Features for Bipolar Depression)

| 特征类别 | 提示双相的特征 | 更符合单相的特征 |
|---|---|---|
| **发病年龄** | 25岁前发病 | 30岁后发病 |
| **发作频率** | 抑郁发作频繁（≥5次） | 发作较少 |
| **发作特点** | 急性起病、突然缓解 | 渐进起病、缓慢缓解 |
| **症状特征** | 非典型特征（嗜睡、食欲增加、肢体沉重感） | 典型忧郁特征（早醒、食欲下降） |
| **精神运动** | 精神运动迟滞为主 | 可迟滞可激越 |
| **精神病性** | 伴精神病性症状 | 通常无 |
| **产后** | 产后抑郁或精神病 | 无产后发作 |
| **家族史** | 双相障碍家族史 | 单相抑郁家族史 |
| **抗抑郁药反应** | 无效、转躁、快速循环 | 通常有效 |
| **物质使用** | 高共病率 | 共病率较低 |
| **自杀** | 自杀风险更高 | 风险较低（但仍高） |

### 表5.2 双相筛查指数 (Bipolar Specifier Index)

**指导语**：以下特征有多少项符合您的抑郁发作？

| 序号 | 特征 | 是 | 否 |
|---|---|---|---|
| **1** | 25岁前首次抑郁发作 | ☐ | ☐ |
| **2** | 家族中有双相障碍史 | ☐ | ☐ |
| **3** | 抑郁发作时有非典型特征（嗜睡、食欲增加） | ☐ | ☐ |
| **4** | 曾有产后抑郁或精神病 | ☐ | ☐ |
| **5** | 抑郁发作时伴有精神病性症状 | ☐ | ☐ |
| **6** | 抗抑郁药治疗后出现过度激动或躁狂 | ☐ | ☐ |
| **7** | 抑郁发作≥5次 | ☐ | ☐ |
| **8** | 曾有过哪怕短暂的情绪高涨期 | ☐ | ☐ |

**解读**：≥3项"是"，建议进行双相障碍专业评估。

## 6. 自我评估结果的解读指南 (Interpretation Guide)

### 表6.1 自评结果分层解读 (Result Interpretation)

| 筛查结果 | 含义 | 建议行动 | 注意事项 |
|---|---|---|---|
| **低风险**（量表分低于阈值，无明显情绪两极波动史） | 目前症状不太可能是双相障碍 | 如有持续困扰，可观察或咨询 | 可能是单相抑郁或其他问题 |
| **中等风险**（部分量表达到阈值，有一定情绪波动史） | 可能存在双相障碍或相关问题 | 建议预约精神科/心理科评估 | 详细提供病史给医生 |
| **高风险**（MDQ或HCL-32达到阳性阈值，有明确的情绪两极波动史） | 双相障碍可能性较大 | 强烈建议尽快专业评估 | 携带自评结果就诊 |
| **任何结果** | 自评≠诊断 | 有困扰就值得专业评估 | 不要基于自评自行用药或拒绝抗抑郁药 |

### 表6.2 筛查阳性后的准备清单 (Pre-Assessment Preparation)

| 准备事项 | 具体内容 | 为什么重要 |
|---|---|---|
| **情绪日记** | 记录2-4周的情绪变化、睡眠、活动量 | 提供给医生参考 |
| **发作史整理** | 列出所有记得的"高涨期"和"低落期"，标注时间和持续天数 | 双相诊断需要终生病程评估 |
| **知情者** | 邀请家人或亲密朋友陪同就诊 | 他们可能记得您忽略的轻躁狂表现 |
| **用药清单** | 列出既往和当前的精神科用药及反应 | 了解抗抑郁药是否曾诱发转躁 |
| **家族史** | 了解家族中有无双相障碍、自杀、精神疾病史 | 遗传因素重要 |
| **功能影响记录** | 记录情绪问题对工作、学业、人际、财务的具体影响 | 证明功能损害 |
| **物质使用史** | 酒精、药物、咖啡因摄入情况 | 排除物质诱发因素 |

## 7. 专业诊断流程详解 (Professional Diagnostic Process)

### 表7.1 双相障碍标准诊断流程 (Standard Assessment Protocol)

| 步骤 | 内容 | 时间 | 评估者 |
|---|---|---|---|
| **Step 1: 初筛** | MDQ/HCL-32自评、简要访谈 | 30分钟 | 初诊医生/心理师 |
| **Step 2: 详细临床访谈** | 当前症状、终生发作史、功能损害、家族史 | 60-90分钟 | 精神科医生/临床心理师 |
| **Step 3: 结构化诊断访谈** | SCID-5、MINI或专项评估 | 60-120分钟 | 专业评估师 |
| **Step 4: 知情者访谈** | 家属提供发作期行为观察 | 30-60分钟 | 知情者配合 |
| **Step 5: 躁狂/抑郁评估** | YMRS（杨氏躁狂量表）、MADRS（抑郁评估） | 30分钟 | 临床医生 |
| **Step 6: 医学排除** | 甲状腺功能、血常规、必要时神经影像 | 因检查而异 | 医生 |
| **Step 7: 诊断反馈** | 整合所有信息、做出诊断、制定方案 | 30-60分钟 | 主诊医生 |

### 表7.2 DSM-5双相障碍诊断要点 (DSM-5 Diagnostic Criteria Summary)

| 诊断类型 | 核心要求 | 注意事项 |
|---|---|---|
| **双相I型** | ≥1次躁狂发作（≥7天或任何时间需住院） | 不需要抑郁发作；躁狂发作即可诊断 |
| **双相II型** | ≥1次轻躁狂发作（≥4天）+ ≥1次重性抑郁发作 | 不能有躁狂史；轻躁狂不导致严重功能损害 |
| **环性心境障碍** | 2年内反复的轻躁狂症状和轻度抑郁症状 | 症状从未达到躁狂或重性抑郁标准 |

### 表8.3 中国常见误区与应对 (Common Issues in China)

| 常见问题 | 现实情况 | 应对建议 |
|---|---|---|
| **"你只是抑郁症"** | 约40%双相患者被误诊为单相抑郁 | 主动提供轻躁狂/躁狂史；要求MDQ筛查 |
| **"轻躁狂不是病"** | 轻躁狂是双相II型的诊断标志 | 强调轻躁狂后的抑郁和功能损害 |
| **首诊被开抗抑郁药** | 单用抗抑郁药治疗双相抑郁有风险 | 告知医生双相疑虑；要求加用心境稳定剂 |
| **等候时间长** | 大型医院精神科排队较长 | 提前预约；考虑互联网医院初筛 |
| **锂盐获取** | 部分地区锂盐供应紧张 | 正规精神科可处方；备选丙戊酸钠 |
| **病耻感强** | 双相障碍有较高病耻感 | 了解这是脑部疾病，需要治疗如同糖尿病需要胰岛素 |

## 9. 自我评估的常见陷阱 (Common Pitfalls in Self-Assessment)

### 表9.1 自我诊断的认知偏误 (Cognitive Biases in Self-Diagnosis)

| 偏误类型 | 表现 | 危害 | 纠正方法 |
|---|---|---|---|
| **轻躁狂美化** | 将轻躁狂视为"最好的自己"而不报告 | 漏诊双相II型 | 思考这种状态是否异于常态、是否有后果 |
| **只关注抑郁** | 就诊只谈抑郁，不提情绪高涨期 | 被误诊为单相抑郁 | 主动回顾并报告所有情绪异常期 |
| **正常化** | "所有人都有情绪波动" | 延迟寻求帮助 | 评估波动的程度、持续时间、功能影响 |
| **归因偏误** | 将躁狂归因于"性格开朗"或"状态好" | 不认为需要治疗 | 听取知情者的客观观察 |
| **社交媒体影响** | 被网络内容影响自我诊断 | 过度或错误自我诊断 | 使用循证量表、寻求专业评估 |
| **病耻感回避** | 因为对"精神病"的偏见而回避就诊 | 延误治疗、后果严重 | 双相是可治疗的脑部疾病 |

### 表9.2 "类双相症状"的其他常见原因 (Other Causes of Bipolar-Like Symptoms)

| 原因 | 类似双相的症状 | 鉴别线索 | 排除方法 |
|---|---|---|---|
| **边缘型人格障碍** | 情绪不稳定、冲动、人际不稳定 | 情绪波动以小时计；与人际事件相关；无明确躁狂期 | 人格评估 |
| **ADHD** | 注意力问题、冲动、活动增多 | 症状持续存在而非发作性；起病于儿童期 | ADHD评估 |
| **甲状腺功能亢进** | 活动增多、睡眠减少、易激惹 | 伴体重变化、心悸、手抖 | 甲状腺功能检查 |
| **物质诱发** | 兴奋剂使用后的躁狂样症状 | 与物质使用时间相关 | 物质使用史、毒物筛查 |
| **睡眠障碍** | 睡眠不足导致情绪波动 | 改善睡眠后症状改善 | 睡眠评估 |
| **创伤后应激** | 情绪失调、过度警觉 | 有创伤史；症状与创伤相关 | 创伤评估 |
| **焦虑障碍** | 激越、睡眠问题 | 无情绪高涨；以焦虑为主 | 焦虑评估 |
| **单相抑郁伴激越** | 抑郁期的精神运动性激越 | 无情绪高涨期；激越伴抑郁情绪 | 详细病史 |
| **脑部疾病** | 额叶病变可导致去抑制 | 神经系统体征 | 神经影像学 |

## 11. 自评后的行动路径 (Action Pathways After Self-Assessment)

### 表11.1 决策树 (Decision Tree)

| 情境 | 建议行动 | 紧急程度 |
|---|---|---|
| **MDQ/HCL-32阳性 + 有明显功能损害史** | 尽快预约精神科评估 | 高 |
| **有自杀想法或计划** | 立即就诊精神科急诊或拨打危机热线 | 紧急 |
| **正在躁狂发作（明显睡眠减少、冲动行为）** | 尽快就诊精神科，可能需要住院 | 高 |
| **自评临界 + 有抑郁史** | 就诊时告知医生怀疑双相，要求评估 | 中 |
| **自评阴性但仍有情绪困扰** | 可先就诊评估，可能是其他问题 | 低-中 |
| **正在服用抗抑郁药 + 出现兴奋躁动** | 立即联系处方医生 | 高 |

### 表11.2 如果您怀疑自己有双相障碍 (If You Suspect Bipolar Disorder)

| 步骤 | 行动 | 说明 |
|---|---|---|
| **第一步** | 完成MDQ和HCL-32自评 | 获得初步筛查结果 |
| **第二步** | 开始记录情绪日记 | 至少记录2-4周 |
| **第三步** | 收集发作史信息 | 询问家人、查看旧日记/照片 |
| **第四步** | 预约精神科门诊 | 优先选择心境障碍专科 |
| **第五步** | 准备就诊材料 | 自评结果、情绪日记、发作史、家族史 |
| **第六步** | 邀请知情者陪同** | 家人/伴侣/亲密朋友 |
| **第七步** | 如实报告所有症状 | 不仅谈抑郁，也要谈高涨期 |
| **第八步** | 讨论治疗方案 | 了解心境稳定剂vs.抗抑郁药的区别 |

## 12. 早期预警信号与自我管理 (Early Warning Signs & Self-Management)

### 表12.1 躁狂/轻躁狂的早期预警信号 (Mania/Hypomania Early Warning Signs)

| 类别 | 常见预警信号 | 自我干预 |
|---|---|---|
| **睡眠** | 睡眠需求减少但不觉得累 | 强制保持睡眠时间；联系医生 |
| **精力** | 精力异常充沛、难以安静 | 避免新项目；减少刺激 |
| **言语** | 说话加速、难以控制话量 | 自我觉察；请家人提醒 |
| **思维** | 想法涌现、思维加速 | 写下想法但不立即行动 |
| **行为** | 开始新项目、计划增多 | 延迟重大决定24-48小时 |
| **消费** | 购物欲望增加 | 限制信用卡使用；设置消费上限 |
| **社交** | 社交活动增加、联系增多 | 限制社交活动时间 |
| **风险行为** | 冒险冲动增加 | 请家人监督；避免独自做重大决定 |

### 表12.2 抑郁的早期预警信号 (Depression Early Warning Signs)

| 类别 | 常见预警信号 | 自我干预 |
|---|---|---|
| **睡眠** | 睡眠增多或入睡困难 | 保持规律作息；适度运动 |
| **兴趣** | 对平时喜欢的事情失去兴趣 | 强制进行愉悦活动 |
| **能量** | 疲劳感增加、动力下降 | 小步骤行动；不要等"有动力" |
| **社交** | 开始回避社交、取消约会 | 保持最低限度社交 |
| **思维** | 消极想法增多、自我批评 | 识别认知歪曲；寻求支持 |
| **食欲** | 食欲明显变化 | 保持规律饮食 |
| **自我照顾** | 忽视个人卫生、日常任务 | 坚持基本自我照顾 |

### 表12.3 紧急情况处理 (Crisis Management)

| 紧急情况 | 立即行动 | 联系资源 |
|---|---|---|
| **有自杀想法** | 告诉信任的人；移除危险物品 | 危机热线、急诊 |
| **严重躁狂（失控行为、无法睡眠）** | 联系家人；停止驾驶和重大决定 | 精神科急诊 |
| **精神病性症状（幻觉、妄想）** | 联系家人；保持安全 | 精神科急诊 |
| **药物严重副作用** | 联系处方医生 | 急诊（如严重皮疹） |
