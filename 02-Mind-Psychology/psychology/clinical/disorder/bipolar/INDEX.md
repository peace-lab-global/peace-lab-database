# 双相情感障碍 (Bipolar Disorder) | Index

> 双相情感障碍是一种以躁狂/轻躁狂发作与抑郁发作交替或混合出现为特征的慢性精神障碍。全球终生患病率约 1-3%，是致残率最高的精神疾病之一。本专题覆盖双相障碍的临床诊断、自我评估、治疗策略，以及与抑郁鉴别的交叉专题。
>
> **核心原则**：双相障碍是可控的慢性病。心境稳定剂为基础的长期维持治疗、生活方式节律管理和家庭支持是康复的三大支柱。

## 快速参考 | Quick Reference

### 双相障碍类型速查

| 类型 | 核心特征 | 诊断门槛 |
|---|---|---|
| **双相 I 型** | 完全躁狂发作 | ≥7天躁狂或需住院 |
| **双相 II 型** | 轻躁狂 + 重性抑郁 | ≥4天轻躁狂；无完全躁狂 |
| **环性心境** | 慢性波动 | 2年以上反复轻躁狂+轻度抑郁 |

### 自杀风险最高情境

| 情境 | 风险等级 | 行动 |
|---|---|---|
| **混合特征** | 🔴 极高 | 住院评估；避免抗抑郁药 |
| **抑郁发作早期** | 🔴 高 | 密切监测；锂盐优先 |
| **从抑郁向躁狂转化** | 🔴 高 | 识别信号；调整治疗 |
| **快速循环** | 🟡 中高 | 复杂管理；避免抗抑郁药 |
