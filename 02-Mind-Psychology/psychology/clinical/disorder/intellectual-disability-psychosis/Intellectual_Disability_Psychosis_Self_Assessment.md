# Intellectual Disability with Psychosis Self-Assessment (精神发育迟滞伴发精神障碍评估与筛查指南)

> **精神发育迟滞伴发精神障碍评估与筛查 (ID+Psychosis Assessment & Screening)**
> 
> *"对于无法用语言诉说痛苦的人，我们需要学会'阅读'他们的行为。"*
> *"For those who cannot express their suffering in words, we must learn to 'read' their behavior."*
> 
> 本文档面向照护者和专业人员，提供识别智力障碍患者精神病性症状的观察工具和评估方法。**由于ID患者自我报告能力受限，照护者观察是评估的主要信息来源。** 详见 [严重精神障碍总览](../Severe_Mental_Illness_Overview.md)。

---

## 1. 评估的特殊挑战 (Unique Assessment Challenges)

### 表1.1 ID 人群精神病评估面临的挑战

| 挑战 | 原因 | 应对策略 |
|---|---|---|
| **言语报告受限** | 多数中度-重度ID患者无法清晰描述主观体验 | 依赖行为观察和照护者报告 |
| **诊断遮蔽** | 症状被误归因于"就是智力低" | 培训评估者意识到ID患者可以且会有精神障碍 |
| **行为等价物** | 精神病性症状以行为改变形式呈现 | 学习识别行为等价物 |
| **基线差异大** | 每个人的功能基线不同 | 任何评估都需与个人基线对比 |
| **共病复杂** | 常合并癫痫、感觉障碍、躯体疾病 | 系统排除躯体和环境因素 |
| **标准量表不适用** | 多数精神科量表假设正常认知和语言能力 | 使用ID专用评估工具 |

## 2. 照护者行为观察清单 (Caregiver Behavioral Observation Checklist)

### 表2.1 精神病性症状行为等价物清单

> **指导语**：请将以下行为与患者的**日常基线**对比。关注的是"变化"而非"现有行为"。如果多项行为出现新的变化且持续≥2周，建议精神科评估。

**A. 可能提示幻觉的行为变化**

| 观察项目 | ☐ | 与基线对比 | 出现频率 |
|---|---|---|---|
| 自言自语明显增加（超出基线） | ☐新出现/增加 ☐无变化 | 之前：___ 现在：___ | 每天/每周 |
| 对着空气说话/回应/生气 | ☐新出现/增加 ☐无变化 | | |
| 捂耳朵/堵耳朵 | ☐新出现/增加 ☐无变化 | | |
| 盯着特定空无一物的方向 | ☐新出现/增加 ☐无变化 | | |
| 新出现对特定房间/区域的恐惧 | ☐新出现/增加 ☐无变化 | | |
| 不明原因的大笑/哭泣/恐惧 | ☐新出现/增加 ☐无变化 | | |

**B. 可能提示妄想的行为变化**

| 观察项目 | ☐ | 与基线对比 | 出现频率 |
|---|---|---|---|
| 拒绝进食（之前无此问题） | ☐新出现 ☐无变化 | | |
| 新出现对特定人的恐惧/回避/攻击 | ☐新出现 ☐无变化 | | |
| 反复检查锁/门窗 | ☐新出现/增加 ☐无变化 | | |
| 表达被害/被跟踪的想法（如能表达） | ☐新出现 ☐无变化 | | |
| 藏东西/藏食物 | ☐新出现/增加 ☐无变化 | | |

**C. 可能提示阴性症状/抑郁的行为变化**

| 观察项目 | ☐ | 与基线对比 | 出现频率 |
|---|---|---|---|
| 活动参与明显减少 | ☐新出现 ☐无变化 | 之前参与___项活动 → 现在___项 | |
| 自我照料退化（之前能做的现在不做） | ☐新出现 ☐无变化 | | |
| 社交退缩加重 | ☐新出现 ☐无变化 | | |
| 面部表情减少/平淡（超出基线） | ☐新出现/增加 ☐无变化 | | |
| 对之前喜欢的活动/食物失去兴趣 | ☐新出现 ☐无变化 | | |
| 睡眠模式显著改变 | ☐新出现 ☐无变化 | | |

**D. 紧急信号**

| 观察项目 | ☐ | 行动 |
|---|---|---|
| 自伤行为新出现或显著增加 | ☐是 | **立即就医** |
| 攻击他人行为新出现或显著增加 | ☐是 | **立即就医** |
| 拒绝所有进食/饮水 | ☐是 | **立即就医** |
| 持续激越无法安抚 | ☐是 | **立即就医** |

### 表2.2 观察结果建议

| 情况 | 建议 |
|---|---|
| **1-2项轻度变化** | 继续观察2周；记录行为日志 |
| **3项以上变化或任何项持续≥2周** | 安排精神科评估（优先选择ID精神科） |
| **任何D类紧急信号** | 立即就医/急诊 |

## 3. 专业评估工具 (Professional Assessment Tools)

### 表3.1 ID 专用精神科评估工具

| 工具 | 英文/缩写 | 评估者 | 评估内容 | 适用ID严重度 | 说明 |
|---|---|---|---|---|---|
| **PAS-ADD** | Psychiatric Assessment Schedule for Adults with Developmental Disability | 受训临床医师 | 精神障碍全面评估 | 轻-中度 | ID精神科金标准工具 |
| **Mini PAS-ADD** | Mini Psychiatric Assessment Schedule | 照护者/护士 | 精神障碍筛查 | 轻-中度 | PAS-ADD的简化筛查版 |
| **DASH-II** | Diagnostic Assessment for the Severely Handicapped | 照护者 | 重度/极重度ID的精神障碍评估 | 重度-极重度 | 基于行为观察 |
| **ABC** | Aberrant Behavior Checklist | 照护者 | 挑战性行为评估（易怒、退缩、刻板、多动、不当言语） | 全部 | 用于行为监测 |
| **DBC** | Developmental Behaviour Checklist | 照护者 | 行为与情绪问题 | 全部 | 澳大利亚开发 |

### 表3.2 躯体排除检查清单 (Physical Exclusion Checklist)

> **在考虑精神科诊断之前，必须排除以下躯体原因。** ID患者可能无法表达疼痛或不适。

| 检查项目 | 排除 | 优先级 |
|---|---|---|
| **疼痛评估** | 牙痛、耳痛、腹痛、骨折、尿路感染 | 最高——任何行为变化首先排除疼痛 |
| **便秘** | 可导致严重不适和行为变化 | 高——ID人群便秘率极高 |
| **感染** | 尿路感染、中耳炎、肺炎 | 高 |
| **甲状腺功能** | 尤其Down综合征甲减高发 | 中 |
| **癫痫变化** | 新发癫痫或发作模式改变 | 高 |
| **感觉功能** | 听力/视力下降 | 中——感觉丧失可导致类精神病行为 |
| **药物副作用** | 近期药物变化 | 高 |
| **月经/激素** | 女性患者月经相关行为变化 | 中 |

## 4. ABC 行为记录法 (ABC Recording Method)

### 表4.1 ABC 记录模板

> **指导语**：当出现挑战性行为或疑似精神病性症状时，使用ABC方法记录，帮助团队分析行为功能。

| 日期/时间 | A - 前因 (Antecedent) | B - 行为 (Behavior) | C - 结果 (Consequence) |
|---|---|---|---|
| ___ | 发生前发生了什么？（环境、人物、活动、要求） | 具体行为描述（做了什么？持续多久？强度？） | 行为之后发生了什么？（他人反应、行为是否停止？） |
| 示例：3/15 14:00 | 下午活动时间；新来的工作人员在场 | 开始对着墙壁大声说话，持续20分钟；捂耳朵 | 工作人员尝试安抚；患者最终安静但持续捂耳朵 |

## 6. 何时转诊专科 (When to Refer to Specialist Services)

### 表6.1 转诊指征

| 情况 | 转诊至 | 紧急程度 |
|---|---|---|
| **行为变化排除躯体原因后持续≥2周** | ID精神科/精神科 | 较紧急 |
| **疑似精神病性症状（幻觉/妄想行为等价物）** | ID精神科 | 较紧急 |
| **自伤或攻击显著增加** | ID精神科 + 安全评估 | 紧急 |
| **现有药物无效或副作用明显** | ID精神科（药物审查） | 较紧急 |
| **诊断不确定** | ID精神科团队评估 | 非紧急 |
| **22q11.2或其他高精神病风险综合征** | 遗传咨询 + 精神科监测 | 预防性 |
