# 青少年抑郁 (Adolescent Depression) | Index

> 青少年抑郁是全球青少年致残的首要原因。其表现常与成年人不同——易激惹、行为问题、学业骤降、自伤和社交退缩往往比悲伤更突出。约10-20%的青少年在其成长过程中会经历抑郁发作。
>
> **核心原则**：青少年的沉默和愤怒可能是求救的信号。理解、不评判、及时专业求助是成年人最重要的责任。

## 快速参考 | Quick Reference

### 青少年抑郁关键信号（非典型）

| 类别 | 信号 |
|---|---|
| **情绪** | 持续易激惹 > 悲伤；情绪爆发；空虚无聊 |
| **行为** | 自伤（NSSI）；学业骤降；网络沉溺；风险行为；物质使用 |
| **社交** | 社交退缩；回避朋友；闭门不出 |
| **躯体** | 不明原因头痛/腹痛；睡眠剧变；食欲/体重明显变化 |
| **危险** | 赠送珍贵物品；搜索自杀信息；突然情绪「好转」 |

### NSSI vs 自杀鉴别

| 特征 | NSSI | 自杀 |
|---|---|---|
| 意图 | 调节情绪 | 结束生命 |
| 方法 | 切割/烧伤（通常不致命） | 更致命方法 |
| 频率 | 反复、习惯 | 可能一次 |
| 实施风险 | **极低** | **高** |

### 青少年抗抑郁药首选

| 首选 | 次选 | 避免 |
|---|---|---|
| **氟西汀**（≥8岁，FDA批准） | **艾司西酞普兰**（≥12岁，FDA批准） | 帕罗西汀（撤药重；自杀风险数据不佳） |

> **黑框警告**：25岁以下使用抗抑郁药需密切监测自杀意念，尤其在用药前2-4周。但 untreated 抑郁的自杀风险远高于治疗风险。
