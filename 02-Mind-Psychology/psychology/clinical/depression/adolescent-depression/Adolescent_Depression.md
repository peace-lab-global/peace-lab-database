# Adolescent Depression (青少年抑郁专题)

> **青少年抑郁 (Adolescent Depression)**
>
> *"青少年的沉默不是叛逆，而可能是求救的信号。学会读懂他们的语言，是成年人最重要的责任。"*
> *"An adolescent's silence is not always rebellion; it may be a cry for help. Learning to read their language is the most important responsibility of adults."*
>
> 青少年抑郁是全球青少年致残的首要原因。全球 10-19 岁青少年中约 4-5% 患有抑郁障碍，且发病率在青春期急剧上升（尤其是女性）。青少年抑郁与学业失败、自伤自杀、物质滥用、成年期慢性疾病密切相关，是重大的公共卫生挑战。

## 1. 核心概念与发展背景 (Core Concepts & Developmental Context)

### 表1.1 青少年抑郁的特殊性

| 维度 | 儿童期 | 青春期 | 成年期 | 临床意义 |
|---|---|---|---|---|
| **核心情绪** | 易激惹 > 悲伤 | 悲伤 + 易激惹 + 焦虑 | 悲伤/空虚为主 | 青春期易激惹和焦虑常是抑郁的"面具" |
| **表达模式** | 躯体主诉；行为问题 | 情绪波动大；社交退缩；自伤；网络沉溺 | 认知症状突出 | 青少年表达更行为化 |
| **自杀风险** | 较低 | **急剧升高**（尤其15岁后） | 高 | 青春期是自杀风险突增期 |
| **自伤行为** | 罕见 | **高发**（NSSI） | 较低 | NSSI是青少年抑郁的重要信号 |
| **共病模式** | ADHD、焦虑、行为障碍 | **焦虑、进食障碍、物质使用、自伤** | 焦虑、物质使用 | 共病是治疗复杂化的主因 |
| **认知发展** | 具体运算 | 形式运算初期；自我中心（假想观众） | 成熟 | 认知治疗需调整；同伴影响极大 |
| **家庭角色** | 依赖 | 独立-依赖冲突 | 独立 | 家庭治疗核心 |
| **同伴影响** | 有限 | **极强** | 重要 | 同伴既是风险也是保护 |
| **激素变化** | 稳定 | 性激素剧变；HPA轴重塑 | 稳定 | 生物学脆弱期 |
| **大脑发育** | 快速 | 前额叶持续成熟至25岁；边缘系统已成熟 | 成熟 | 情绪-控制失衡 |

> **神经发育视角**：青春期边缘系统（情绪/奖赏）已成熟，而前额叶皮层（执行控制/决策）尚在发育。这一「油门-刹车」失衡解释了青少年的情绪冲动性、风险行为和抑郁易感性。

### 表2.2 性别差异

| 特征 | 女性青少年 | 男性青少年 |
|---|---|---|
| **患病率** | 2-3倍于男性 | 较低 |
| **核心表现** | 悲伤、反刍、焦虑、自伤、进食问题 | 易激惹、愤怒、行为问题、物质使用、冒险 |

| **关键风险** | 身体意象；社交排斥；性暴力 | 学业失败；行为问题；物质使用 |

## 3. 临床表现与识别 (Clinical Presentation & Detection)

### 表3.1 青少年抑郁的非典型信号

| 信号类别 | 具体表现 | 常被误读为 | 识别要点 |
|---|---|---|---|
| **情绪** | 持续烦躁、易怒（尤其对父母） | 叛逆期正常 | 与情境不成比例；持续 > 2周 |
| | 情绪爆发 | 脾气差 | 频率增加；事后后悔 |
| | 空虚感、无聊感 | 青春期正常 | 对任何事物都「没意思」 |
| **行为** | **自伤（NSSI）** | 寻求注意 | 切割、烧伤、抓伤；常隐藏 | 必须严肃对待；评估自杀风险 |
| | 社交退缩 | 内向 | 回避以往朋友；拒绝社交活动 |
| | 学业骤降 | 懒惰 | 成绩突然下滑；缺课增多 |
| | 网络/游戏沉溺 | 正常爱好 | 逃避现实；无法控制 |
| | 风险行为（酒驾、不安全性行为） | 叛逆 | 冲动控制下降；绝望感 |
| | 物质使用（酒精、大麻、电子烟） | 同伴影响 | 自我用药；逃避 |
| **躯体** | 不明原因头痛、腹痛 | 装病逃学 | 医学检查排除后考虑心理 |
| | 睡眠模式剧变 | 青春期晚睡正常 | 凌晨3点后仍不睡；白天嗜睡严重 |
| | 食欲/体重明显变化 | 生长需要 | 短期内显著增减 |
| **认知** | 过度自我批评 | 谦虚 | 达到自我憎恨程度 |
| | 无望感 | 悲观 | 「未来不会好的」 |
| | 注意力涣散 | ADHD | 新发或加重；与情绪低落关联 |
| **危险信号** | 赠送珍贵物品 | 慷慨 | 可能预示自杀计划 |
| | 搜索自杀信息 | 好奇 | 立即评估 |
| | 突然情绪「好转」 | 恢复 | 可能是已做自杀决定的解脱感 |

### 表3.2 自伤（NSSI）与自杀的鉴别

| 特征 | 非自杀性自伤 (NSSI) | 自杀行为 |
|---|---|---|
| **意图** | 调节情绪；缓解痛苦；自我惩罚；感受存在 | 结束生命 |
| **方法** | 切割（最常见）、烧伤、抓伤；通常不致命 | 更致命方法；过量药物；上吊 |
| **频率** | 反复、习惯性 | 可能一次或少数几次 |
| **伤口** | 浅表、可控位置（大腿、手臂内侧） | 任何位置；可能更深 |
| **后续** | 常隐藏伤口；感到短暂缓解 | 可能做准备（留遗书、告别） |
| **自杀风险** | NSSI 是自杀的**独立风险因素**（RR 3-5） | 直接生命威胁 |
| **应对** | 不惩罚；理解功能；替代策略；治疗原发抑郁 | 安全评估；危机干预；保护 |

> **核心原则**：NSSI 不应被轻视或惩罚。它是青少年应对情绪痛苦的「解决方案」（尽管是适应不良的）。理解其功能（情绪调节、自我惩罚、人际沟通、自我认同）是治疗的关键。

### 表3.3 筛查工具

| 工具 | 条目 | 用途 | 特点 |
|---|---|---|---|
| **PHQ-A (青少年版PHQ-9)** | 9项 | 青少年抑郁筛查 | 语言适配青少年；自评 |
| **Kutcher Adolescent Depression Scale (KADS-6)** | 6项 | 青少年抑郁筛查 | 专为青少年设计 |
| **RCADS (Revised Children's Anxiety and Depression Scale)** | 47项（简版25项） | 焦虑+抑郁 | 广泛使用；多维度 |
| **CDI-2 (Children's Depression Inventory)** | 28项 | 儿童+青少年抑郁 | 7-17岁；自评+他评 |
| **SMFQ (Short Mood and Feelings Questionnaire)** | 13项 | 快速筛查 | 简短；临床适用 |
| **C-SSRS (简化版)** | 6项 | 自杀风险筛查 | **每次必做** |
| **SDQ (Strengths and Difficulties Questionnaire)** | 25项 | 广泛行为情绪问题 | 家长/教师/自评 |

## 4. 治疗策略 (Treatment Strategies)

### 4.1 治疗选择决策

| 严重度 | 特征 | 一线治疗 | 辅助 |
|---|---|---|---|
| **轻度** | PHQ-A 5-9；功能基本维持 | **心理治疗**（CBT/IPT/DBT-A）；观察（如果刚起病） | 家庭支持；学校协调 |
| **中度** | PHQ-A 10-14；功能部分受损 | **心理治疗**；考虑SSRI（尤其IPT/CBT不可及或无效时） | 家庭治疗；学校支持 |
| **重度** | PHQ-A 15-20；功能严重受损 | **SSRI + 心理治疗**；住院治疗（如有自杀风险） | 家庭治疗；学校协调；同伴支持 |
| **伴精神病性** | 妄想/幻觉 | 抗精神病药 + 抗抑郁药；住院治疗 | — |
| **伴高自杀风险** | 主动意念/计划 | **住院治疗**；SSRI；强化心理治疗 | 安全计划；限制手段；家庭支持 |

### 4.2 药物治疗

#### 表4.1 青少年抗抑郁药：关键事实

| 药物 | FDA批准 | 适应症年龄 | 疗效证据 | 自杀风险黑框警告 | 推荐度 |
|---|---|---|---|---|---|
| **氟西汀 (Fluoxetine)** | ✅ 是 | ≥8岁 | **最强**（A级） | 有 | ⭐⭐⭐⭐⭐ |
| **艾司西酞普兰 (Escitalopram)** | ✅ 是 | ≥12岁 | 强（A级） | 有 | ⭐⭐⭐⭐⭐ |
| **舍曲林 (Sertraline)** | 标签外 | 青少年 | 中等（B级） | 有 | ⭐⭐⭐⭐ |
| **氟伏沙明 (Fluvoxamine)** | 标签外 | 青少年 | 中等（B级） | 有 | ⭐⭐⭐⭐ |
| **文拉法辛 (Venlafaxine)** | ❌ 否 | — | 弱于SSRI；可能增加自杀意念 | 有 | ⭐⭐ |
| **帕罗西汀 (Paroxetine)** | ❌ 否 | 青少年 | 不建议；疗效弱；撤药症状重 | 有 | ⭐ |
| **西酞普兰 (Citalopram)** | 标签外 | 青少年 | 有限数据 | 有 | ⭐⭐⭐ |

> **黑框警告解析**：2004年FDA发出SSRI在25岁以下人群可能增加自杀意念的警告。但后续数据表明：**治疗抑郁的获益远大于风险**。未治疗的青少年抑郁自杀率远高于治疗组。关键是：用药初期（前2-4周）密切监测自杀风险；心理治疗同步进行。

#### 表4.2 青少年用药原则

| 原则 | 具体做法 |
|---|---|
| **心理治疗优先或同步** | 轻中度首选心理治疗；中重度药物+心理治疗 |
| **氟西汀或艾司西酞普兰首选** | 循证最强；安全性数据最多 |
| **低剂量起始，缓慢滴定** | 起始剂量更低；加量间隔更长 |
| **密切监测初期反应** | 用药后1、2、4周必须随访；监测激活、激越、自杀意念 |
| **家庭知情同意** | 与家长/监护人充分沟通获益与风险 |
| **不单独用药** | 必须配合心理支持和安全监测 |
| **疗程足够** | 急性期8-12周；巩固期6-12月；维持期视复发风险 |
| **避免突然停药** | 青少年撤药症状可能更显著 |
| **注意双相筛查** | 用药前必须筛查轻躁狂/躁狂史（MDQ） |

### 4.3 心理治疗

| 方法 | 证据 | 适用情境 | 青少年调整 |
|---|---|---|---|
| **认知行为治疗 (CBT)** | **A级** | 首选；轻中度抑郁 | 更多行为激活；简化认知重构；使用青少年语言；融入科技元素 |
| **人际心理治疗-青少年版 (IPT-A)** | **A级** | 人际冲突、角色冲突、社交隔离、哀伤 | 聚焦同伴关系、家庭独立、浪漫关系 |
| **辩证行为治疗-青少年版 (DBT-A)** | **B级** | 自伤；情绪失调；冲动；边缘型特质 | 技能训练（痛苦耐受、情绪调节、人际效能、正念）；家庭参与 |
| **家庭治疗 (FBT/FFT)** | **B级** | 家庭冲突；父母抑郁；沟通不良 | 改善家庭沟通；父母管理训练；降低批评和情感过度卷入 |
| **正念认知疗法-青少年改编 (MBCT-A)** | **B级** | 复发预防；反刍；焦虑共病 | 缩短练习时间；增加互动；减少内省深度 |
| **接纳承诺治疗-青少年 (ACT)** | **B级** | 完美主义；价值困惑；慢性疼痛 | 价值澄清；认知解离；承诺行动 |
| **Attachment-Based Family Therapy (ABFT)** | **B级** | 家庭依恋创伤；自杀青少年 | 修复亲子依恋；解决家庭冲突 |
| **团体CBT/IPT** | **B级** | 社交孤立；同伴支持需求 | 同龄人团体；降低病耻感 |
| **线上/数字CBT** | **B级** | 出行困难；偏好自主；偏远地区 | 游戏化；即时反馈；隐私保护 |

### 4.4 自伤（NSSI）的干预

| 策略 | 具体方法 | 目标 |
|---|---|---|
| **功能评估** | 理解NSSI对个体的具体功能（情绪调节？自我惩罚？沟通？） | 个性化干预 |
| **替代策略** | 冰敷、橡皮筋弹手腕、剧烈运动、冷水洗脸、撕纸 | 提供不伤害的替代调节方式 |
| **DBT技能训练** | TIPP（温度、激烈运动、 paced 呼吸、 paired 肌肉放松） | 快速情绪调节 |
| **减少手段** | 移除锋利物品；与信任的人共同管理 | 增加自伤难度 |
| **延迟技术** | 「等15分钟」；计数到100；做一件其他事 | 打破冲动-行为链接 |
| **处理潜在创伤** | 创伤聚焦CBT；EMDR | 如果NSSI与创伤相关 |
| **改善沟通** | 帮助青少年用语言表达需求，而非用身体 | 功能替代 |

## 6. 长期预后与随访 (Long-Term Prognosis & Follow-Up)

### 表6.1 青少年抑郁的纵向轨迹

| 轨迹类型 | 比例 | 特点 | 预测因素 |
|---|---|---|---|
| **单次发作，完全康复** | ~30-40% | 一次抑郁发作后不再复发 | 轻度；良好支持；及时充分治疗 |
| **复发性抑郁** | ~30-40% | 成年期多次发作 | 重度；家族史；早发；残留症状 |
| **慢性/持续性** | ~10-20% | 症状持续或波动 | 共病；逆境；治疗不足 |
| **双相转化** | ~10-20% | 首次抑郁发作后未来出现躁狂 | 混合特征；精神病性；双相家族史 |

### 表6.2 随访计划

| 时间点 | 重点 | 工具 |
|---|---|---|
| **治疗期间** | 每周-每2周 | PHQ-A；副作用；自杀风险；功能 |
| **急性期末（12周）** | 是否达缓解；残留症状；功能恢复 | PHQ-A；CGAS（儿童功能量表） |
| **巩固期（3-6月）** | 维持治疗；复发预防；技能巩固 | PHQ-A；每月 |
| **维持期（6-12月）** | 逐步减药评估；预防复发；学校适应 | PHQ-A；每2-3月 |
| **停药后** | 密切监测6月（复发高风险期） | PHQ-A；每月 |
| **年度随访** | 长期监测至成年早期 | PHQ-A；功能；物质使用 |

> **双相转化警示**：青少年抑郁患者中，约10-20%最终发展为双相障碍。转化预测因素：混合特征、精神病性症状、双相家族史、抗抑郁药诱发轻躁狂、睡眠需求减少。所有青少年抑郁患者应终身监测躁狂症状。
