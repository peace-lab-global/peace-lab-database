# 老年抑郁 (Geriatric Depression) | Index

> 老年抑郁（60岁以上）是识别率最低、治疗最不充分的抑郁类型。其临床表现不典型，常以躯体症状、认知下降和淡漠为主导，且与多种躯体疾病和药物相互作用交织。
>
> **核心原则**：老年抑郁不是衰老的正常部分，而是需要积极识别和治疗的疾病。治疗不仅改善情绪，更是维护认知、功能与生命尊严的关键。

## 快速参考 | Quick Reference

### 假性痴呆 vs 真性痴呆速查

| 特征 | 假性痴呆（抑郁性） | 真性痴呆（如AD） |
|---|---|---|
| 起病 | 急性 | 隐匿 |
| 主诉 | 患者主动抱怨 | 家属发现 |
| 认知测验 | 「我不知道」；不努力 | 努力尝试但做错 |
| 治疗反应 | **抗抑郁可逆转** | 不可逆 |

### GDS-15 快速评分

| 分数 | 解读 |
|---|---|
| 0-4 | 正常 |
| 5-9 | 轻度抑郁 |
| 10+ | 中重度抑郁 |

### 老年用药核心原则

1. **Start Low, Go Slow**
2. **单一用药优先**
3. **注意药物相互作用**
4. **避免苯二氮卓**
5. **关注抗胆碱能负担**
6. **监测低钠血症**
