# Geriatric Depression (老年抑郁专题)

> **老年抑郁 (Late-Life Depression / Geriatric Depression)**
>
> *"老年人的悲伤常常被误认为是衰老的一部分。但抑郁不是正常的老年，它是需要治疗的疾病。"*
> *"Sadness in older adults is often mistaken for normal aging. But depression is not normal aging; it is a condition that demands treatment."*
>
> 老年抑郁指60岁及以上人群发生的抑郁发作。由于其临床表现不典型、与躯体疾病和认知下降重叠，老年抑郁是识别率最低、治疗最不充分的抑郁类型之一。全球60岁以上人群抑郁患病率约 5-7%，但医疗机构中可达 10-15%。

## 1. 核心概念与诊断挑战 (Core Concepts & Diagnostic Challenges)

### 表1.1 老年抑郁与典型抑郁的差异

| 维度 | 典型抑郁（成年期） | 老年抑郁 | 临床意义 |
|---|---|---|---|
| **核心主诉** | 情绪低落、兴趣丧失 | **躯体症状**（疼痛、疲劳、头晕）、认知下降 | 患者常首诊于综合医院非精神科 |
| **情绪表达** | 悲伤、绝望明显 | **情感淡漠、迟钝**多于悲伤；易激惹 | 易被误认为「性格改变」 |
| **焦虑共病** | 常见 | **更常见**；常为主要表现 | 焦虑可能是抑郁的「面具」 |
| **躯体症状** | 伴随 | **主导**；心血管、胃肠、疼痛主诉 | 需大量躯体检查排除器质性疾病 |
| **认知症状** | 注意力不集中 | **假性痴呆**（可逆性认知下降） | 与真性痴呆鉴别困难 |
| **精神病性特征** | 较少见 | **更常见**；妄想多为躯体/被害/虚无 | 需抗精神病药联合 |
| **自杀风险** | 高 | **更高**；计划更周密；成功率更高 | 必须常规评估；限制性手段更常用 |
| **病程特点** | 发作性 | **慢性化、复发性**更突出 | 需长期维持治疗 |
| **功能损害** | 明显 | **极重**；独立性丧失；跌倒风险 | 功能恢复是治疗核心目标 |
| **病耻感** | 存在 | **更强**；「精神病」标签抗拒 | 需去标签化沟通 |

### 表1.2 假性痴呆 vs 真性痴呆鉴别

| 特征 | 假性痴呆（抑郁性认知下降） | 真性痴呆（如阿尔茨海默病） |
|---|---|---|
| **起病** | 相对急性（数周至数月） | 隐匿、渐进 |
| **病程** | 波动性；日间可变化 | 持续进展 |
| **主诉** | 患者主动抱怨记忆力差 | 家属发现，患者常否认或掩饰 |
| **认知测验态度** | 「我不知道」；不努力尝试 | 努力尝试但做错；近事遗忘突出 |
| **定向力** | 通常保留 | 早期即受损 |
| **记忆力** | 近远记忆均差（因注意力不集中） | 近记忆差 >> 远记忆 |
| **执行功能** | 启动困难、精神运动迟滞 | 计划、组织、抽象能力下降 |
| **情感** | 情感痛苦明显；消极内容 | 情感淡漠；欣快或脱抑制 |
| **治疗反应** | **抗抑郁治疗可完全逆转** | 不可逆；对症治疗 |

> **关键原则**：所有老年认知下降患者都应筛查抑郁。约 10-15% 的「痴呆」实为可逆的抑郁性假性痴呆。

| | 社会隔离 | RR 2-3 | 缺乏缓冲；认知刺激减少 | **高**（社会参与） |

| | 经济困难 | RR 1.5-2 | 慢性应激；选择受限 | 中 |
| | 搬迁/机构化 | RR 2-3 | 环境应激；控制感丧失 | 中 |
| | 退休 | RR 1.5-2 | 角色转变；社会网络缩小 | **高**（角色重建） |
| **药物因素** | 多种药物（≥5种） | RR 1.5-2 | 药物相互作用；抑郁作为副作用 | **高**（药物重整） |
| | 苯二氮卓类 | RR 1.5-2 | 本身可致抑郁、认知下降、跌倒 | **高**（减停） |

| | 他汀类（争议） | 不确定 | 可能机制：CoQ10耗竭、炎症 | 观察 |

### 表2.2 血管性抑郁 (Vascular Depression)

| 特征 | 描述 |
|---|---|
| **定义** | 由脑血管病变（大或微小血管）引起的或加重的抑郁 |
| **影像学** | 白质高信号（WMH）；腔隙性梗死；基底节/前额叶病变 |
| **临床特点** | 执行功能损害突出；精神运动迟滞；缺乏明显内疚感；自知力差；对药物反应较慢 |
| **风险因素** | 高血压、糖尿病、高血脂、吸烟、心房颤动 |
| **治疗启示** | 需同时管理血管风险；药物剂量需谨慎；认知康复重要 |
| **预防** | 心血管风险管理是预防血管性抑郁的核心 |

## 4. 治疗策略 (Treatment Strategies)

### 4.1 药物治疗

#### 表4.1 老年抑郁药物选择

| 药物 | 优势 | 风险/注意 | 推荐情境 | 起始剂量 | 目标剂量 |
|---|---|---|---|---|---|
| **艾司西酞普兰 (Escitalopram)** | 耐受性好；药物相互作用少；QTc影响小 | 低钠血症（老年人风险↑） | 一线首选 | 5mg | 10-20mg |
| **舍曲林 (Sertraline)** | 心血管安全性数据好；药物相互作用少 | 胃肠道不适；低钠血症 | 一线；心血管疾病 | 25mg | 50-100mg |
| **西酞普兰 (Citalopram)** | 耐受性好 | QTc延长（老年人更敏感）；≤20mg | 一线（注意剂量） | 10mg | 20mg |
| **米氮平 (Mirtazapine)** | 镇静、促食欲、止吐；改善睡眠和体重 | 体重增加；嗜睡；跌倒风险 | 失眠；食欲不振；体重下降 | 7.5mg | 15-30mg |
| **文拉法辛 (Venlafaxine)** | 双通道；可能对疼痛有效 | 高血压；停药症状；跌倒 | 疼痛共病；SSRIs无效 | 37.5mg | 75-150mg |
| **度洛西汀 (Duloxetine)** | SNRI；疼痛适应症 | 肝毒性；高血压；胃肠道 | 疼痛共病 | 20mg | 60mg |
| **伏硫西汀 (Vortioxetine)** | 认知改善证据；多模式 | 恶心；腹泻 | 认知症状突出 | 5mg | 10-20mg |
| **安非他酮 (Bupropion)** | 无体重/性副作用；激活 | 癫痫风险；失眠；激越 | 疲劳/嗜睡突出；戒烟 | 100mg SR | 150-300mg |
| **TCA（必要时）** | 疗效确切；疼痛有效 | **心脏毒性；抗胆碱能（尿潴留、便秘、谵妄、跌倒）** | 难治性；疼痛 | 极低剂量 | 谨慎滴定 |

#### 表4.2 老年用药核心原则

| 原则 | 具体做法 | 理由 |
|---|---|---|
| **Start Low, Go Slow** | 起始剂量为成人常规起始量的1/2-1/3 | 药代动力学改变（肝血流↓、肾清除↓、体脂↑） |
| **单一用药优先** | 避免多药联用 | 药物相互作用；副作用叠加；依从性 |
| **注意药物相互作用** | 审查所有用药（包括OTC和中药） | 老年人平均服用5-8种药物 |
| **监测副作用** | 特别关注：跌倒、低钠血症、QTc、出血、谵妄 | 老年人对这些副作用更脆弱 |
| **避免苯二氮卓** | 尽量不用；如需极短期 | 跌倒、认知下降、谵妄、依赖、抑郁加重 |
| **关注抗胆碱能负担** | 避免或限制抗胆碱能药物 | 认知损害、谵妄、便秘、尿潴留、视力模糊 |
| **足量但不过量** | 达到有效剂量，但不过度 | 疗效 vs 副作用平衡 |
| **疗程足够长** | 急性期8-12周；维持期至少1-2年（老年复发率高） | 预防复发 |
| **定期药物重整** | 每3-6月审查用药必要性 | 去处方化 (Deprescribing) |

> **低钠血症警示**：SSRIs/SNRIs在老年人中低钠血症风险显著增加（约 10-30%）。基线及用药后1-2周应查血钠。

### 4.2 心理治疗

| 方法 | 证据 | 适用情境 | 调整要点 |
|---|---|---|---|
| **问题解决方法治疗 (PST)** | **A级** | 老年抑郁首选短程心理治疗 | 结构化；聚焦具体问题；灵活会谈时间 |
| **认知行为治疗 (CBT)** | **A级** | 有认知功能者 | 简化认知技术；更多行为激活；慢节奏 |
| **人际心理治疗 (IPT)** | **B级** | 角色丧失、社交隔离、哀伤 | 聚焦晚年角色转变 |
| ** reminiscence 治疗 / 怀旧治疗** | **B级** | 老年群体；自我连续性 | 回顾一生；整合经验；提升自我效能 |
| **行为激活 (BA)** | **B级** | 精神运动迟滞；活动减少 | 从极小步骤开始；结合物理康复 |
| **团体治疗** | **B级** | 社交孤立；正常化 | 同龄人团体； reminiscence 团体 |
| **接纳承诺治疗 (ACT)** | **C级** | 慢性病适应；功能丧失 | 接纳局限；价值导向 |
| **照顾者干预** | **B级** | 家庭支持系统 | 教育；支持；减轻照顾者负担 |
| **线上/电话治疗** | **B级** | 出行困难；偏远地区 | 技术辅助；简化界面 |

### 4.3 躯体与非药物干预

| 干预 | 证据 | 机制 | 适用情境 |
|---|---|---|---|
| **运动** | **A级** | BDNF↑；抗炎；社会参与；功能改善 | 所有可耐受者；尤其功能下降者 |
| **光照治疗** | **B级** | 昼夜节律调节 | 冬季抑郁；机构化老人 |
| **营养干预** | **B级** | B12、维D、ω-3补充 | 缺乏者；营养不良 |
| **社交参与** | **B级** | 社会支持；认知刺激 | 孤立者；机构化老人 |
| **音乐治疗** | **B级** | 情绪调节； reminiscence | 晚期痴呆；沟通困难 |
| **宠物治疗** | **C级** | 陪伴；责任感；触觉刺激 | 独居者；动物爱好者 |
| **rTMS** | **B级** | 非侵入性脑刺激 | 药物不耐受或无效者 |
| **ECT** | **A级** | 严重、精神病性、拒食、高自杀风险 | 老年严重抑郁（非常有效且安全） |

> **ECT在老年抑郁中的地位**：老年抑郁是ECT最明确适应症之一。尤其适用于：严重抑郁、精神病性抑郁、拒食/拒水、高自杀风险、药物无效或不耐受。老年患者对ECT反应良好，认知副作用通常短暂且可逆。

## 5. 自杀预防 (Suicide Prevention)

### 表5.1 老年抑郁自杀风险特征

| 特征 | 描述 | 临床行动 |
|---|---|---|
| **自杀率最高** | 65岁以上男性自杀率最高（尤其85+） | 每次评估必须询问 |
| **计划更周密** | 老年人自杀计划更详细、更致命 | 评估手段可用性 |
| **表达更隐晦** | 不直接说"想死"，而是"不想成为负担" | 倾听弦外之音 |
| **限制性手段** | 更常用枪械、上吊、过量药物 | 讨论手段限制 |
| **躯体疾病驱动** | 常因无法忍受躯体痛苦或不想拖累家人 | 疼痛管理；解决实际担忧 |
| **社交孤立** | 丧偶、独居、社交减少是重要风险 | 社会联系干预 |
| **近期丧亲** | 丧偶后2年内风险最高 | 加强随访 |
| **近期诊断严重疾病** | 新诊断癌症、痴呆、心脏病等 | 疾病适应支持 |
| **近期出院** | 出院后首周风险极高 | 出院计划；紧密随访 |

### 表5.2 老年自杀风险评估与干预

| 评估内容 | 具体询问 | 干预 |
|---|---|---|
| **自杀意念** | "您是否觉得活着没意思？" "您是否想过结束生命？" | 验证感受；不评判；建立联盟 |
| **计划** | "您想过怎么去做吗？" | 手段限制；安全计划 |
| **手段** | "您家里有药物/武器吗？" | 交给家属保管；处理多余药物 |
| **时间** | "您计划在什么时候？" | 危机干预；住院评估 |
| **保护因素** | 什么让您坚持到现在？ | 强化保护因素；连接意义 |
| **既往未遂** | 以前有过吗？ | 既往未遂是最强预测因子 |

## 6. 长期管理与随访 (Long-Term Management)

### 表6.1 老年抑郁维持治疗

| 发作史 | 推荐维持时间 | 理由 |
|---|---|---|
| 首次发作 | ≥1-2 年 | 老年复发率高 |
| 2次发作 | ≥2-3 年 | 高复发风险 |
| ≥3次发作 | 长期/终身 | 极高复发风险 |
| 伴自杀未遂 | 长期 | 风险管理 |
| 伴严重躯体病 | 长期 | 慢性应激持续 |
| 伴认知下降 | 长期 + 认知保护策略 | 抑郁加速认知衰退 |

### 表6.2 随访重点

| 随访内容 | 频率 | 工具/方法 | 目标 |
|---|---|---|---|
| 症状评估 | 每次 | GDS-15 / PHQ-9 | 监测疗效和复发 |
| 副作用监测 | 每次 | 询问 + 血检 | 安全 |
| 认知功能 | 每6-12月 | MoCA | 早期识别痴呆转化 |
| 功能水平 | 每次 | IADL / ADL | 功能维持/恢复 |
| 躯体健康 | 定期 | 体检、血检 | 识别新发病 |
| 药物重整 | 每3-6月 | 全面用药清单 | 减少不必要的药物 |
| 社会支持 | 每次 | 询问 | 减少孤立 |
| 自杀风险 | 每次 | 直接询问 | 安全 |
