## 老年情绪发展

### 老年情绪的悖论

尽管客观上面临更多丧失（健康、亲友、角色），研究一致发现老年人的情绪体验往往更积极：

| 发现 | 描述 | 引用 |
|------|------|------|
| 幸福U型曲线 | 幸福感从中年低谷后回升 | Blanchflower, 2008 |
| 情绪复杂性增加 | 同时体验正负情绪的能力增强 | Carstensen, 2000 |
| 负面情绪减少 | 愤怒/焦虑频率降低 | Charles, 2010 |
| 积极偏向 | 注意力和记忆偏向正面信息 | Mather, 2005 |
| 情绪恢复更快 | 负面事件后情绪恢复速度更快 | Charles, 2010 |

### 情绪调节策略的年龄差异

| 策略 | 年轻人 | 老年人 |
|------|--------|--------|
| 情境选择 | 较少使用 | 积极使用(避免冲突情境) |
| 认知重评 | 常用 | 更熟练、更自动化 |
| 压制 | 常用 | 选择性使用 |
| 注意力部署 | 负面偏向 | 正面偏向 |
| 社交调节 | 广泛社交 | 选择性亲密关系 |

## 社会情绪选择理论

### Carstensen理论核心（SST）

当人们感知剩余时间有限时，目标优先级从知识获取转向情绪调节：

| 时间知觉 | 目标优先 | 社交偏好 | 信息偏好 |
|---------|---------|---------|---------|
| 时间无限(年轻人) | 知识获取 | 广泛社交网络 | 新奇/信息性 |
| 时间有限(老年人) | 情绪意义 | 少而亲密的关系 | 积极/有意义 |

### 实验证据

- **社交伙伴选择**: 老年人更选择情感熟悉的伙伴（vs年轻人选择新奇的）
- **HIV+年轻人**: 感知时间有限的年轻人表现出与老年人类似的偏好模式
- **SARS/COVID期间**: 年轻人也表现出社交网络缩小+亲密关系强化
- **时间框架操纵**: 当让年轻人想象20年后的自己，他们的选择与老年人一致

### 社交网络修剪——不是损失是选择

```

                ↓ 选择性修剪 ↓
老年人网络:  ●  ●  ●  ●  ●          (少但深)
```

## 孤独与社会隔离

### 定义区分

| 概念 | 定义 | 客观/主观 |
|------|------|----------|
| 社会隔离 | 客观上缺乏社会联系 | 客观(可测量) |
| 孤独感 | 主观感受到的社交不满 | 主观(体验性) |
| 独处 | 自愿选择的独自状态 | 可以是正面的 |

### 孤独的健康危害

| 影响 | 数据 | 引用 |
|------|------|------|
| 死亡风险 | 增加26% | Holt-Lunstad, 2015 Meta |
| 等效危害 | 相当于每天吸15支烟 | Holt-Lunstad, 2010 |
| 痴呆风险 | 增加40% | Livingston, 2020 |
| 心脏病 | 增加29% | Valtorta, 2016 |
| 抑郁 | 显著增加 | Cacioppo, 2006 |
| 免疫功能 | 炎症标志物升高 | Cole, 2015 |

### 孤独干预策略

| 策略 | 证据 | 示例 |
|------|------|------|
| 改变不适应认知 | 最有效(Meta支持) | CBT矫正社交负面预期 |
| 增加社交机会 | 中等 | 社区活动、志愿者 |
| 增强社交技能 | 中等 | 社交技能训练 |
| 社会支持 | 中等 | 同伴支持小组 |
| 技术辅助 | 新兴 | 视频通话、社交app |

## 丧亲与哀伤

### 老年丧亲特殊性

| 特点 | 描述 |
|------|------|
| 累积性丧失 | 多次经历丧亲（配偶、兄弟姐妹、朋友） |
| 角色丧失 | 丧偶=失去伴侣角色+可能的照护者角色 |
| 资源减少 | 社交网络已缩小、经济资源可能有限 |
| 韧性普遍 | 约50-60%的丧亲老年人自然恢复 |
| 复杂哀伤 | 约10-15%发展为持续性复杂丧亲障碍 |

### 哀伤的轨迹（Bonanno四轨迹模型）

| 轨迹 | 占比 | 特征 |
|------|------|------|
| 韧性型 | 50-60% | 短暂悲伤后恢复正常功能 |
| 恢复型 | 15-25% | 初期功能下降，6-12月后恢复 |
| 慢性哀伤型 | 10-15% | 持续2年+的严重功能受损 |
| 延迟反应型 | 5-10% | 初期正常，后期出现问题 |

### 持续性复杂丧亲障碍（PCBD, DSM-5）

- 分离痛苦持续12个月以上
- 对逝者的强烈渴望/反复思念
- 回避丧失提醒 或 过度沉浸
- 显著功能受损

### 哀伤支持

| 方法 | 适用 | 证据 |
|------|------|------|
| 正常哀伤支持 | 多数丧亲者 | 社会支持、不干预为佳 |
| 丧亲支持小组 | 渴望同伴支持者 | 中等 |
| 复杂哀伤治疗(CGT) | 复杂哀伤诊断 | 强(Shear) |
| 生命回顾 | 适合整合丧失 | 中等 |

## 老年抑郁

### 老年抑郁特殊性

| 特点 | 描述 |
|------|------|
| 躯体化多 | 以疼痛、疲劳、食欲变化为主诉 |
| 认知症状突出 | 注意力、记忆问题(假性痴呆) |
| 否认情绪 | 不承认"情绪低落" |
| 共病多 | 常与慢性躯体疾病共存 |
| 低就诊率 | 仅10-20%得到适当治疗 |
| 高自杀率 | 老年男性自杀率最高 |

### 治疗选择

| 方法 | 证据 | 注意 |
|------|------|------|
| SSRI | 强 | 起效慢、注意跌倒风险 |
| CBT | 强 | 需适应性修改(步调、重复) |
| 行为激活 | 强 | 适合老年人 |
| 运动(尤其有氧) | 中-强 | d=0.54(vs对照) |
| 人际治疗(IPT) | 中 | 聚焦角色转换/丧亲 |
| 怀旧/生命回顾疗法 | 中 | 老年人特色疗法 |

## 情绪健康促进

### 生命回顾疗法（Life Review Therapy）

| 要素 | 内容 |
|------|------|
| 创始人 | Robert Butler (1963) |
| 方法 | 系统回顾人生经历，寻找意义和连贯性 |
| 结构 | 通常6-8次，按生命阶段回顾 |
| 证据 | Meta分析: 改善抑郁(d=0.57)和生活满意度 |

### 感恩与品味（Savoring）

| 实践 | 方法 | 频率 |
|------|------|------|
| 感恩日记 | 每天记录3件感恩事 | 每日 |
| 品味散步 | 散步时注意并享受感官体验 | 每周2-3次 |
| 回忆分享 | 与他人分享美好记忆 | 每周 |
| 祝福信 | 给重要的人写感恩信 | 每月 |
