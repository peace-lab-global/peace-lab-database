## 认知老化总览

### 认知功能与年龄关系

| 认知功能 | 年龄趋势 | 下降开始 | 备注 |
|---------|---------|---------|------|
| 处理速度 | 线性下降 | 20s后 | 最早下降的能力 |
| 工作记忆 | 逐渐下降 | 30s-40s | 与前额叶功能相关 |
| 情景记忆 | 中度下降 | 40s-50s | 编码和提取均受影响 |
| 语义记忆 | 维持/增长 | 极晚(70s+) | 可持续积累 |
| 词汇知识 | 维持/增长 | 极晚 | 结晶智力的核心 |
| 空间导航 | 中度下降 | 50s | 海马依赖 |
| 执行功能 | 中度下降 | 40s-50s | 前额叶依赖 |
| 注意力(分散) | 中度下降 | 40s | 双任务尤其困难 |
| 注意力(持续) | 轻微下降 | 较晚 | 相对保留 |

### 认知老化的脑机制

| 脑变化 | 描述 | 功能影响 |
|--------|------|---------|
| 灰质体积减少 | 每十年约5%，前额叶最显著 | 执行功能下降 |
| 白质退化 | 髓鞘降解、连接性降低 | 处理速度下降 |
| 海马萎缩 | 每年约1-2%(70岁后加速) | 情景记忆下降 |
| 多巴胺减少 | 每十年约10%的受体丧失 | 动机和奖赏敏感性降低 |
| 神经炎症 | 小胶质细胞激活增加 | 加速退化 |
| 代偿性激活 | 双侧前额叶激活(HAROLD模式) | 功能维持的补偿策略 |

## 正常认知老化

### 正常vs病理老化区分

| 维度 | 正常老化 | MCI | 痴呆 |
|------|---------|-----|------|
| 记忆抱怨 | 偶尔忘事 | 频繁但自知 | 常否认 |
| 日常功能 | 完全保留 | 基本保留 | 明显受损 |
| 客观测试 | 年龄范围内 | 低于年龄匹配1.5SD | 多域显著低于 |
| 进展 | 缓慢稳定 | 每年10-15%进展 | 持续恶化 |
| 提示效果 | 提示后能回忆 | 部分有效 | 基本无效 |

### "正常"遗忘 vs 值得关注的遗忘

| 正常 | 需关注 |
|------|--------|
| 偶尔忘记约会 | 频繁忘记刚发生的事 |
| 叫不出熟人名字（后能想起） | 无法识别熟人 |
| 偶尔找不到钥匙 | 把钥匙放在异常位置 |
| 暂时忘记日期（后想起） | 对年月日持续困惑 |
| 跟不上复杂对话 | 无法跟上简单对话 |

## 轻度认知障碍(MCI)

### 诊断标准（NIA-AA, 2011）

1. 本人或知情者报告认知下降
2. 一个或多个认知域客观损害（低于年龄匹配1-1.5 SD）
3. 日常功能基本保留
4. 不符合痴呆诊断标准

### MCI亚型

| 亚型 | 受损域 | 进展风险 |
|------|--------|---------|
| 遗忘型单域(aMCI) | 仅记忆 | 高风险→AD |
| 遗忘型多域 | 记忆+其他 | 高风险→AD |
| 非遗忘型单域 | 非记忆(如执行/语言) | 中风险→非AD痴呆 |
| 非遗忘型多域 | 多个非记忆域 | 中风险→混合型 |

### MCI管理

| 策略 | 证据 | 建议 |
|------|------|------|
| 有氧运动 | 中-强 | 150分钟/周中等强度 |
| 认知训练 | 中等 | 每周2-3次结构化训练 |
| 地中海饮食 | 中等 | MIND饮食变体最优 |
| 社交参与 | 中等 | 保持活跃社交网络 |
| 睡眠优化 | 中等 | 7-8小时/晚 |
| 心血管风险管理 | 强 | 控制三高 |
| 药物(AChEI) | 弱 | 不推荐用于MCI |

## 痴呆症概览

### 痴呆类型

| 类型 | 占比 | 核心特征 | 早期表现 |
|------|------|---------|---------|
| 阿尔茨海默病(AD) | 60-70% | 淀粉样斑块+Tau缠结 | 情景记忆丧失 |
| 血管性痴呆 | 15-20% | 脑血管病变 | 执行功能障碍 |
| 路易体痴呆 | 5-10% | 路易体沉积 | 视幻觉+帕金森样 |
| 额颞叶痴呆 | 5-10% | 前额/颞叶退化 | 人格改变/语言障碍 |
| 混合型 | 常见 | 多种病理共存 | 混合表现 |

### 照护者负担

| 维度 | 数据 |
|------|------|
| 全球痴呆照护者 | 超过5,000万家庭照护者 |
| 抑郁风险 | 照护者抑郁率30-50% |
| 身体健康 | 免疫功能降低、心血管风险增加 |
| 经济负担 | 全球痴呆年费用超过1.3万亿美元 |

## 认知储备理论

### 定义

认知储备（Cognitive Reserve）指个体通过一生积累的认知资源，使大脑能够更有效地应对病理损伤，维持较好的认知功能（Stern, 2002）。

### 脑储备 vs 认知储备

| 类型 | 含义 | 影响因素 |
|------|------|---------|
| 脑储备(被动) | 脑体积/突触密度等硬件 | 遗传、早期营养 |
| 认知储备(主动) | 灵活使用脑网络的能力 | 教育、职业、活动 |
| 脑维护 | 减少与年龄相关的脑变化 | 生活方式、运动 |

### 认知储备的指标与证据

| 因素 | 储备贡献 | 证据 |
|------|---------|------|
| 教育年限 | 每增加1年→痴呆风险↓7% | Livingston et al., 2020 |
| 职业复杂度 | 高复杂度职业→延迟发病2-4年 | Stern, 2012 |
| 休闲活动 | 阅读/游戏/社交→风险显著降低 | Scarmeas et al., 2001 |
| 双语 | 双语者痴呆发病延迟4-5年 | Bialystok et al., 2007 |
| 社交网络 | 丰富社交→风险降低40% | Fratiglioni et al., 2000 |

## 认知干预与预防

### Lancet委员会可改变风险因素（2020更新）

12个可改变因素解释了约40%的痴呆风险：

| 生命阶段 | 风险因素 | 人群归因分数 |
|---------|---------|------------|
| 早年 | 低教育 | 7.1% |
| 中年 | 听力损失 | 8.2% |
| | 创伤性脑损伤 | 3.4% |
| | 高血压 | 1.9% |
| | 过量饮酒 | 0.8% |
| | 肥胖 | 0.7% |
| 晚年 | 吸烟 | 5.2% |
| | 抑郁 | 3.9% |
| | 社会隔离 | 3.5% |
| | 缺乏运动 | 1.6% |
| | 空气污染 | 2.3% |
| | 糖尿病 | 1.1% |

### FINGER研究（芬兰）

- **设计**: 多域RCT, n=1,260, 60-77岁认知风险人群
- **干预**: 运动+饮食+认知训练+心血管风险管理
- **结果**: 2年后综合认知功能改善25%（vs对照组）
- **意义**: 首个证实多域干预可改善认知轨迹的大型RCT
