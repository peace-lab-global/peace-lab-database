## 儿童创伤概览

### 定义

儿童创伤指超越儿童应对能力的威胁性或有害事件，导致持续的生理、情绪或行为困扰。包括单次创伤(Type I)和复杂/重复创伤(Type II)。

### 创伤类型分类

| 类型 | 示例 | 特征 |
|------|------|------|
| 急性创伤(Type I) | 事故、自然灾害、目睹暴力 | 单次、突然、可识别 |
| 复杂创伤(Type II) | 持续虐待、忽视、家庭暴力 | 重复、长期、关系性 |
| 发展创伤 | 早期依恋破裂、情感忽视 | 影响大脑发育轨迹 |
| 历史/代际创伤 | 战争、种族灭绝后代 | 跨代传递 |
| 医疗创伤 | 侵入性治疗、重症住院 | 与照护系统交叉 |
| 社区创伤 | 校园暴力、社区犯罪 | 安全感基础动摇 |

### 流行病学数据

| 指标 | 数据 |
|------|------|
| 经历≥1项ACE | 约 64% 的成人(美国CDC) |
| 经历≥4项ACE | 约 12.5% |
| 儿童虐待报告率 | 约 9.1‰ (美国, 2019) |
| 儿童性虐待终生患病率 | 女孩 20%, 男孩 5-10% |
| PTSD患病率(创伤后) | 儿童约 15-40%（因创伤类型差异大） |

## ACEs研究与发现

### Felitti ACE研究 (1998)

- **样本**: 17,337名Kaiser Permanente成员
- **方法**: 回溯性调查童年不良经历(0-18岁)
- **发现**: ACE得分与成年健康问题呈强剂量-反应关系

### 十类ACEs

| 类别 | 具体项目 |
|------|---------|
| 虐待(3项) | 身体虐待、情感虐待、性虐待 |
| 忽视(2项) | 身体忽视、情感忽视 |
| 家庭功能障碍(5项) | 家庭暴力、物质滥用、精神疾病、父母入狱、父母离婚/分居 |

### ACE得分与健康结果

| ACE得分 | 抑郁风险倍增 | 自杀未遂 | 酒精依赖 | 心脏病 | 预期寿命 |
|---------|------------|---------|---------|--------|---------|
| 0 | 基准 | 基准 | 基准 | 基准 | 基准 |
| 1 | 1.5x | 2.0x | 1.5x | 1.1x | -2年 |
| 2-3 | 2.5x | 3.7x | 2.0x | 1.4x | -5年 |
| 4+ | 4.6x | 12.2x | 7.4x | 2.2x | -20年 |

### ACEs的生物嵌入机制

```
童年逆境 (ACEs)
     ↓
HPA轴持续激活 → 皮质醇失调
     ↓
慢性炎症 → 免疫系统改变
     ↓
表观遗传改变 → 基因表达变化
     ↓
大脑结构/功能改变
     ↓
成年身心健康问题
```

## 创伤对发展的影响

### 创伤对大脑发展的影响

| 脑区 | 正常发展 | 创伤影响 | 行为表现 |
|------|---------|---------|---------|
| 前额叶皮层 | 执行功能、情绪调节 | 体积减小、活化降低 | 冲动、决策困难 |
| 杏仁核 | 威胁检测 | 过度激活、体积增大 | 过度警觉、恐惧 |
| 海马 | 记忆、学习 | 体积减小 | 记忆问题、学习困难 |
| 胼胝体 | 左右脑连接 | 连接减弱 | 信息整合困难 |
| 小脑 | 运动协调 | 体积改变 | 协调性降低 |

### 创伤对发展各领域的影响

| 发展领域 | 表现 |
|---------|------|
| 依恋 | 无组织型依恋、关系困难、信任缺失 |
| 情绪调节 | 情绪爆发、麻木、快速升级、解离 |
| 认知 | 注意力问题、执行功能缺陷、学业困难 |
| 自我概念 | 低自尊、羞耻感、自我归因("都是我的错") |
| 身体 | 躯体化症状、疼痛敏感、免疫功能降低 |
| 行为 | 攻击、退缩、回避、自伤、物质滥用 |
| 社会 | 社交退缩或过度讨好、边界感缺失 |

### 发展创伤障碍（DTD, van der Kolk提出）

与传统PTSD不同，DTD强调复杂创伤对整体发展的弥漫性影响：
- **调节失调**: 情绪、注意力、身体感觉的调节障碍
- **关系困难**: 信任、边界、亲密感的全面受损
- **自我紊乱**: 身份感、能动感、身体所有权的混乱

## 韧性理论与保护因子

### 韧性定义

韧性（Resilience）指个体在面对显著逆境时保持或恢复良好适应功能的能力（Masten, 2014）。

### 韧性研究三波浪

| 波浪 | 时期 | 关注点 | 代表研究 |
|------|------|--------|---------|
| 第一波 | 1970-1980s | 识别保护因子 | Werner Kauai纵向研究 |
| 第二波 | 1990-2000s | 韧性过程与机制 | Rutter, Masten |
| 第三波 | 2010s+ | 干预促进、表观遗传 | 韧性的生物学基础 |

### Werner Kauai纵向研究（40年追踪）

- **样本**: 夏威夷Kauai岛698名婴儿（1955年出生队列）
- **发现**: 约1/3高风险儿童发展良好，展现韧性
- **保护因子**: 至少一个稳定的照料者、良好的气质、外部支持系统

### 保护因子矩阵

| 层面 | 保护因子 | 机制 |
|------|---------|------|
| 个体 | 良好气质 | 激发照料者积极回应 |
| | 正常智力 | 问题解决、学业成功 |
| | 自我效能感 | 掌控感增强应对 |
| | 情绪调节能力 | 缓冲负面影响 |
| | 内部控制感 | 不过度自责 |
| 家庭 | 至少一个稳定照料者 | 安全依恋基础 |
| | 温暖的亲子关系 | 情感缓冲 |
| | 家庭凝聚力 | 归属感和安全感 |
| | 结构化养育 | 可预测性和边界 |
| 社区 | 积极的学校经历 | 替代性积极关系 |
| | 宗教/社区参与 | 意义感和归属 |
| | 有效的社会服务 | 危机干预和支持 |
| | 导师关系 | 替代性依恋对象 |

### 韧性不是固定特质

- **韧性是动态过程**: 而非固定人格特质
- **领域特异性**: 在某领域有韧性不等于全面韧性
- **可培养**: 通过干预可以增强韧性
- **不意味着无损伤**: 韧性个体也可能有内在挣扎
- **文化嵌入**: 韧性的表现形式受文化影响

## 创伤知情方法

### 创伤知情护理（TIC）核心原则（SAMHSA）

| 原则 | 内容 | 实践 |
|------|------|------|
| 安全 | 身体和心理安全 | 可预测的环境和关系 |
| 信任透明 | 建立信任和清晰沟通 | 解释流程、兑现承诺 |
| 同伴支持 | 利用共同经历的支持 | 同伴辅导/小组 |
| 合作互惠 | 共享权力和决策 | 儿童参与治疗计划 |
| 赋能选择 | 增强掌控感和选择 | 提供选项、尊重偏好 |
| 文化历史性别 | 关注文化和历史因素 | 文化敏感的干预 |

### 创伤知情学校

| 要素 | 做法 |
|------|------|
| 全员培训 | 所有教职工接受创伤知情培训 |
| 安全环境 | 可预测的规则、冷静角、感官工具 |
| 关系优先 | 每个学生有至少一个可信赖的成人 |
| 行为理解 | 行为是沟通，不仅是违规 |
| 连接学习 | 先调节情绪，再进行学习 |

## 循证干预方案

### 主要创伤干预方法

| 方案 | 适用年龄 | 核心方法 | 证据等级 |
|------|---------|---------|---------|
| TF-CBT | 3-18岁 | 认知三角+创伤叙事+安全技能 | A（金标准） |
| EMDR | 6岁+ | 眼动脱敏与再加工 | A |
| CPP | 0-5岁 | 亲子心理治疗+创伤整合 | A |
| PCIT | 2-7岁 | 亲子互动+行为管理 | A |
| ARC | 全年龄 | 依恋-调节-能力框架 | B+ |
| 游戏治疗 | 3-12岁 | 通过游戏处理创伤 | B |

### TF-CBT核心组件（PRACTICE）

| 字母 | 组件 | 内容 |
|------|------|------|
| P | 心理教育 | 创伤反应正常化 |
| R | 放松技能 | 呼吸/肌肉放松/正念 |
| A | 情感调节 | 识别和管理情绪 |
| C | 认知应对 | 纠正创伤认知扭曲 |
| T | 创伤叙事 | 逐步暴露和加工创伤 |
| I | 现场脱敏 | 现实生活中面对提醒物 |
| C | 联合亲子会谈 | 分享创伤叙事、强化沟通 |
| E | 安全提升 | 安全计划和技能巩固 |

### 创后成长（Post-Traumatic Growth）

| 维度 | 表现 |
|------|------|
| 人际关系改善 | 更珍视关系、增强共情 |
| 新的可能性 | 发现新的人生方向和兴趣 |
| 个人力量 | "我比想象中更强大" |
| 精神/灵性改变 | 更深的存在意义感 |
| 对生命的感恩 | 珍惜日常生活、活在当下 |

## 特殊情境创伤

### 家庭暴力目睹

- **影响**: 等同于直接虐待，影响依恋、情绪调节、认知发展
- **患病率**: 约15-25%的儿童曾目睹家庭暴力
- **特殊需求**: 安全规划、忠诚冲突处理、角色重建

### 校园欺凌创伤

| 类型 | 表现 | 干预 |
|------|------|------|
| 身体欺凌 | 推打、抢夺 | 学校反欺凌政策+个别支持 |
| 言语欺凌 | 嘲笑、威胁 | 社交技能训练+认知重构 |
| 关系欺凌 | 排挤、造谣 | 友谊技能+社会支持建设 |
| 网络欺凌 | 线上骚扰、散布信息 | 数字素养+网络安全 |

### 父母离异/丧亲

- **离异**: 约25%的儿童出现适应困难（多数在2年内恢复）
- **丧亲**: 儿童理解死亡随年龄发展（不可逆性→普遍性→因果性）
- **干预重点**: 持续可获得的照料者、情绪表达、日常结构维持
