## 游戏治疗总览

### 定义

游戏治疗（Play Therapy）是利用游戏作为主要沟通媒介的系统心理治疗方法。治疗师通过游戏与儿童建立安全的治疗关系，帮助儿童表达感受、探索经历、解决问题（APT定义）。

### 为什么用游戏治疗儿童

| 原因 | 说明 |
|------|------|
| 语言发展限制 | 儿童难以用语言准确表达内心 |
| 游戏是自然语言 | 游戏是儿童最自然的表达方式 |
| 具象化思维 | 通过玩具和场景具象化内心世界 |
| 安全距离 | 通过角色/玩具间接处理困难体验 |
| 掌控感 | 在游戏中获得日常生活缺乏的掌控 |
| 情绪宣泄 | 安全环境中释放积压情绪 |

### 适用范围

| 适应症 | 证据强度 |
|--------|---------|
| 焦虑/恐惧 | ★★★★☆ |
| 创伤/虐待后 | ★★★★☆ |
| 行为问题/攻击 | ★★★★☆ |
| 依恋问题 | ★★★☆☆ |
| 离婚/丧亲适应 | ★★★☆☆ |
| 自闭症谱系 | ★★★☆☆ |
| ADHD | ★★★☆☆ |
| 选择性缄默 | ★★★☆☆ |
| 社交困难 | ★★★☆☆ |

## 理论基础与流派

### 主要流派对比

| 流派 | 代表人物 | 治疗师角色 | 核心方法 | 结构化程度 |
|------|---------|-----------|---------|-----------|
| 以儿童为中心 | Virginia Axline | 非指导性、跟随 | 无条件积极关注+反映 | 低 |
| 心理动力学 | Melanie Klein | 解释性 | 通过游戏解释无意识 | 中 |
| 认知行为游戏治疗 | Susan Knell | 教导性 | 通过游戏教认知技能 | 高 |
| Theraplay | Ann Jernberg | 指导性、互动 | 结构/参与/滋养/挑战 | 高 |
| 叙事游戏治疗 | — | 共同叙事 | 重新建构故事 | 中 |
| 完形游戏治疗 | Violet Oaklander | 体验性 | 感知觉察+表达 | 中 |
| 阿德勒游戏治疗 | Terry Kottman | 鼓励性 | 生活风格探索 | 中 |

### 以儿童为中心游戏治疗（CCPT）— Axline八原则

| 原则 | 内容 |
|------|------|
| 1 | 快速建立温暖、友好的关系 |
| 2 | 完全接纳儿童当下的状态 |
| 3 | 创造宽容的氛围，让儿童自由表达 |
| 4 | 识别并反映儿童的感受 |
| 5 | 尊重儿童解决问题的能力 |
| 6 | 由儿童引领方向 |
| 7 | 治疗是渐进过程，不催促 |
| 8 | 仅设置必要的限制 |

### CCPT限制设置三步法

1. **认可感受**: "我知道你很想把颜料泼到我身上"
2. **说明限制**: "但是泼到人身上不被允许"
3. **提供替代**: "你可以把颜料泼到纸上"

## 游戏治疗过程

### 标准治疗流程

| 阶段 | 时间 | 内容 |
|------|------|------|
| 评估 | 1-3次 | 家长访谈 + 儿童观察 + 评估工具 |
| 建立关系 | 1-4次 | 建立信任、儿童熟悉游戏室 |
| 深度工作 | 8-20次 | 核心治疗、主题涌现与工作 |
| 整合结束 | 2-4次 | 回顾收获、处理分离 |
| 跟踪随访 | 1-2次 | 1-3个月后随访 |

### 游戏室标准配置

| 区域 | 物品 | 治疗功能 |
|------|------|---------|
| 家庭/角色扮演 | 娃娃屋、厨房、电话、穿戴服装 | 家庭关系探索 |
| 攻击/释放 | 打击玩具、飞镖、黏土 | 愤怒宣泄、力量表达 |
| 创造/表达 | 颜料、蜡笔、纸张、黏土 | 艺术表达、情绪呈现 |
| 沙盘 | 沙盘、微缩模型(100+件) | 内心世界外化 |
| 建构 | 积木、乐高、拼图 | 掌控感、问题解决 |
| 战争/英雄 | 军人、恐龙、超级英雄 | 权力/保护主题 |
| 安抚 | 毛绒动物、毯子、瓶子 | 退行、安全感 |

### 治疗性反应技术

| 技术 | 示例 | 作用 |
|------|------|------|
| 追踪 | "你把红色方块放到了蓝色上面" | 表达关注 |
| 情感反映 | "你看起来为那个娃娃感到担心" | 情绪觉察 |
| 意义赋予 | "你让那只小熊打赢了大恐龙" | 理解潜在主题 |
| 鼓励自我评价 | "你觉得你画得怎么样?" | 建立内在评价系统 |
| 限制设置 | "沙子是留在沙盘里的" | 安全边界 |

## 沙盘疗法

### 概述

沙盘疗法（Sandplay/Sandtray Therapy）由Dora Kalff发展，基于Jung分析心理学，让个体在沙盘中用微缩模型创建场景，外化内心世界。

### 操作流程

1. **引导语**: "在这个沙盘中创建一个世界/场景"
2. **创作过程**: 安静观察、不干预（荣格学派）
3. **完成**: 邀请描述（可选）
4. **拍照记录**: 每次留档
5. **治疗师分析**: 空间使用/主题/原型/变化趋势

### 沙盘解读维度

| 维度 | 关注点 |
|------|--------|
| 空间使用 | 中心/边缘/角落、空间利用率 |
| 对称性 | 平衡/对抗/分裂 |
| 动物/人物 | 攻击性/保护性/受害者角色 |
| 自然元素 | 水/树/山/沙漠→情绪状态 |
| 围栏/界限 | 保护/限制/隔离主题 |
| 系列变化 | 跨次数的演变趋势 |

## 特定问题的游戏治疗

### 创伤后游戏治疗

| 方法 | 描述 | 适用 |
|------|------|------|
| 创伤游戏 | 儿童自发重复创伤场景 | 需辨别创伤性重复 vs 修复性游戏 |
| 结构化创伤叙事 | 通过游戏建构创伤故事 | 能耐受的儿童 |
| EMDR + 游戏 | 眼动脱敏与游戏整合 | 单次创伤 |
| TF-CBT中的游戏 | 认知三角的游戏化教学 | 学龄儿童 |

### 焦虑儿童游戏治疗

- **渐进暴露游戏**: 通过玩偶/角色扮演逐步面对恐惧场景
- **勇气阶梯**: 游戏化设计暴露层级
- **超级英雄技术**: 创建"勇气超级英雄"分身
- **放松英雄**: 教授深呼吸和肌肉放松的游戏版本

### 丧亲/分离儿童

- **记忆盒子**: 收集与逝者/分离者相关的物品
- **再见仪式**: 通过游戏创建告别仪式
- **情绪天气**: 用天气比喻每天的情绪状态
- **故事续写**: 为分离故事创造新的结局

## 家长参与模式

### 亲子游戏治疗（Filial Therapy）

| 要素 | 内容 |
|------|------|
| 原创者 | Bernard & Louise Guerney (1960s) |
| 核心理念 | 训练父母成为治疗性游戏伙伴 |
| 流程 | 10-20次小组培训 + 家庭练习 |
| 父母学习 | 反映性倾听、跟随、限制设置 |
| 证据 | Meta分析支持有效(Bratton, 2005) |

### CPRT (儿童-父母关系治疗)

Landreth & Bratton发展的结构化10次亲子游戏治疗模式：
- 10次小组课程（每周1次、2小时）
- 每周30分钟家庭特别游戏时间
- 录像回放与小组督导
- Meta分析效果量 d=0.80（Ray et al., 2015）

## 循证研究

### Meta分析证据

| 综述 | 纳入 | 总体效果量 | 结论 |
|------|------|-----------|------|
| Ray et al., 2015 | 23 RCTs | d=0.80 | 游戏治疗对内化/外化问题均有效 |
| Bratton et al., 2005 | 93研究 | d=0.80 | 效果量与成人心理治疗相当 |
| Lin & Bratton, 2015 | 52研究 | d=0.47 | 游戏治疗对行为问题有效 |

### 关键发现

- **CCPT vs 指导性游戏治疗**: 两者效果相当，CCPT更适合关系创伤
- **个体 vs 小组**: 个体效果量略高(d=0.93 vs 0.73)
- **最佳疗程**: 14-20次效果最佳
- **治疗师资质**: 受过专门训练的游戏治疗师效果显著优于未受训者
