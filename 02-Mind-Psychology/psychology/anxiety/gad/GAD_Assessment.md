# Generalized Anxiety Disorder Assessment (广泛性焦虑障碍评估)

> **GAD评估 (GAD Assessment / Aestimatio Diagnostica)**
> 
> *"准确的评估是有效治疗的基石。"*
> *"Accurate assessment is the cornerstone of effective treatment."*
> 
> 本文档系统呈现广泛性焦虑障碍的评估工具、诊断访谈技术及临床评估框架，为精准诊断和治疗监测提供专业指导。

---

## 1. 评估术语表 (Assessment Terminology)

| 术语 (中文) | 英文名 (English Term) | 原始语言/词源 (Original Language) | 精确定义 | 临床应用 |
|---|---|---|---|---|
| **筛查** | Screening | 古诺斯语: *Skren* (筛子) | 在一般人群中识别可能病例的简短评估 | 初级保健、社区筛查 |
| **诊断评估** | Diagnostic Assessment | 希腊语: *Diagnosis* (διάγνωσις, 辨别) | 确定是否符合诊断标准的全面评估 | 专科评估 |
| **严重度评估** | Severity Assessment | 拉丁语: *Severus* (严格的) | 量化症状严重程度 | 治疗计划、疗效监测 |
| **功能评估** | Functional Assessment | 拉丁语: *Functio* (执行) | 评估障碍对日常功能的影响 | 治疗目标设定 |
| **共病评估** | Comorbidity Assessment | 拉丁语: *Co-* (共同) + *Morbidus* (疾病) | 评估共存的其他障碍 | 全面治疗规划 |
| **风险评估** | Risk Assessment | — | 评估自伤、自杀等风险 | 安全规划 |
| **个案公式化** | Case Formulation | — | 整合评估信息形成个体化理解 | 治疗靶点确定 |
| **疗效监测** | Treatment Monitoring | — | 治疗过程中持续评估症状变化 | 治疗调整 |
| **心理测量学** | Psychometrics | 希腊语: *Psyche* (ψυχή, 心灵) + *Metron* (μέτρον, 测量) | 心理测量工具的科学属性 | 工具选择依据 |

## 2. GAD专用自评量表 (GAD-Specific Self-Report Scales)

### 表2.1 广泛性焦虑量表-7 (GAD-7)

| 维度 | 内容 |
|---|---|
| **全称** | Generalized Anxiety Disorder 7-item Scale (广泛性焦虑量表-7项版) |
| **开发者** | Spitzer, Kroenke, Williams & Löwe (2006) |
| **条目数** | 7项 |
| **评分** | 0-3分(完全没有/好几天/超过一周/几乎每天) |
| **分数范围** | 0-21分 |
| **施测时间** | 2-3分钟 |
| **评估周期** | 过去2周 |
| **临床切分** | 5分=轻度；10分=中度；15分=重度 |
| **诊断敏感性** | 89% (切分10分) |
| **诊断特异性** | 82% (切分10分) |
| **信度(α)** | 0.92 |
| **主要用途** | 筛查、严重度评估、疗效监测 |
| **优势** | 简短、免费、广泛验证、多语言版本 |
| **局限** | 不能替代诊断访谈；对其他焦虑障碍也敏感 |

### 表2.2 GAD-7条目内容 (GAD-7 Items)

| 条目 | 内容 | 对应DSM标准 |
|---|---|---|
| 1 | 感到紧张、焦虑或烦躁 | 核心焦虑 |
| 2 | 无法停止或控制担忧 | 标准B:难以控制 |
| 3 | 对各种事情担忧过多 | 标准A:过度担忧 |
| 4 | 很难放松 | 标准C:坐立不安 |
| 5 | 烦躁不安以至于坐立难安 | 标准C:坐立不安 |
| 6 | 变得容易恼怒或易激惹 | 标准C:易激惹 |
| 7 | 感到好像会有可怕的事情发生 | 核心焦虑/担忧 |

### 表2.3 宾州担忧问卷 (Penn State Worry Questionnaire, PSWQ)

| 维度 | 内容 |
|---|---|
| **全称** | Penn State Worry Questionnaire (宾州担忧问卷) |
| **开发者** | Meyer, Miller, Metzger & Borkovec (1990) |
| **条目数** | 16项 |
| **评分** | 1-5分(完全不符合→非常符合) |
| **分数范围** | 16-80分 |
| **临床切分** | ≥45分提示病理性担忧；≥62分高度提示GAD |
| **信度(α)** | 0.93 |
| **重测信度** | 0.74-0.93 |
| **主要用途** | 评估病理性担忧特征 |
| **优势** | GAD担忧的金标准测量；治疗敏感 |
| **局限** | 仅评估担忧，不评估躯体症状 |
| **特点** | 含11项正向计分和5项反向计分 |

| **反向计分** | "我很容易就能消除担忧" |

## 7. 临床访谈指南 (Clinical Interview Guide)

### 表7.1 GAD诊断访谈结构 (GAD Diagnostic Interview Structure)

| 访谈阶段 | 目标 | 关键问题 | 时间 |
|---|---|---|---|
| **开放性探索** | 了解主诉和背景 | "是什么让您今天来寻求帮助？" | 5-10分钟 |
| **担忧评估** | 评估担忧特征 | 见下表 | 10-15分钟 |
| **躯体症状评估** | 评估六项躯体症状 | 见下表 | 5-10分钟 |
| **功能影响评估** | 评估对生活各领域的影响 | "焦虑如何影响您的工作/关系/生活？" | 5分钟 |
| **鉴别诊断** | 排除其他焦虑障碍、躯体疾病、物质 | 针对性问题 | 10分钟 |
| **共病筛查** | 筛查抑郁、其他焦虑、物质使用 | 使用筛查量表 | 5-10分钟 |
| **风险评估** | 评估自伤/自杀风险 | 直接询问 | 5分钟 |
| **病史收集** | 既往史、家族史、治疗史 | 标准病史采集 | 5-10分钟 |

### 表7.2 担忧评估核心问题 (Core Questions for Worry Assessment)

| 评估维度 | 问题示例 | 评估目的 |
|---|---|---|
| **担忧存在** | "您是否经常为很多事情担忧？" | 识别担忧 |
| **担忧内容** | "您主要担忧哪些方面？(工作、健康、家庭...)" | 识别担忧领域 |
| **担忧过度性** | "您觉得担忧的程度与实际情况相符吗？" | 评估过度性 |
| **担忧控制** | "您能否让自己停止担忧？" | 评估控制困难 |
| **担忧频率** | "您每周有几天会这样担忧？" | 评估频率 |
| **担忧持续** | "这种担忧持续多久了？" | 评估病程 |
| **担忧功能** | "担忧是否帮助您解决问题，还是主要带来痛苦？" | 功能分析 |

### 表7.3 躯体症状评估问题 (Somatic Symptom Assessment Questions)

| 症状 | 评估问题 |
|---|---|
| **坐立不安** | "您是否经常感到坐立不安、内心紧绷、无法放松？" |
| **易疲劳** | "您是否比平时更容易感到疲倦？" |
| **注意力难集中** | "您是否发现难以集中注意力或大脑一片空白？" |
| **易激惹** | "您是否比往常更容易发脾气或感到烦躁？" |
| **肌肉紧张** | "您是否经常感到肌肉紧绷或酸痛，特别是肩颈背部？" |
| **睡眠障碍** | "您是否有入睡困难、睡眠不安稳或早醒的问题？" |

### 表7.4 鉴别诊断核心问题 (Differential Diagnosis Questions)

| 需排除 | 核心问题 |
|---|---|
| **惊恐障碍** | "您是否有过突然发作的强烈恐惧，伴心跳加速、呼吸困难？" |
| **社交焦虑** | "您的担忧主要是关于社交场合中他人对您的看法吗？" |
| **特定恐惧** | "您的担忧是否主要针对某个特定的对象或情境？" |
| **OCD** | "您是否有反复出现的、不想要的想法，让您必须做某些事来缓解？" |
| **PTSD** | "您是否经历过严重的创伤事件，现在仍受其困扰？" |
| **躯体疾病** | "您最近做过体检吗？有没有发现什么健康问题？" |
| **物质使用** | "您每天喝多少咖啡/酒？使用任何药物或物质吗？" |

## 8. 评估流程与框架 (Assessment Process & Framework)

### 表8.1 GAD评估全流程 (Comprehensive GAD Assessment Process)

| 阶段 | 内容 | 工具/方法 | 结果 |
|---|---|---|---|
| **1. 初筛** | 快速识别可能GAD | GAD-2或GAD-7 | 阳性→进一步评估 |
| **2. 诊断评估** | 确定是否符合标准 | 临床访谈(+SCID/ADIS) | 诊断确立/排除 |
| **3. 严重度评估** | 量化症状严重程度 | GAD-7、PSWQ、HAM-A | 严重度等级 |
| **4. 认知评估** | 识别认知机制 | IUS、MCQ-30、WDQ | 治疗靶点 |
| **5. 功能评估** | 评估功能损害 | SDS、WSAS | 功能恢复目标 |
| **6. 共病评估** | 筛查和评估共病 | PHQ-9、各专项量表 | 共病诊断 |
| **7. 风险评估** | 评估自杀/自伤风险 | C-SSRS、临床访谈 | 安全计划 |
| **8. 个案公式化** | 整合信息形成理解 | 公式化框架 | 治疗计划 |

### 表8.2 分层评估策略 (Stepped Assessment Strategy)

| 层级 | 环境 | 评估工具 | 目的 |
|---|---|---|---|
| **第一层** | 初级保健 | GAD-2 → GAD-7 | 筛查 |
| **第二层** | 初级保健/心理服务 | GAD-7 + PHQ-9 + 临床访谈 | 初步诊断 |
| **第三层** | 专科服务 | 结构化访谈 + 认知量表 + 功能量表 | 全面评估 |
| **第四层** | 研究/复杂病例 | ADIS-5 + 全套量表 | 详细评估 |

### 表8.3 疗效监测时间表 (Treatment Monitoring Schedule)

| 时间点 | 评估内容 | 推荐工具 | 目的 |
|---|---|---|---|
| **基线** | 全面评估 | GAD-7、PSWQ、SDS、PHQ-9 | 基线水平 |
| **每次治疗** | 症状监测 | GAD-7或单项快速评估 | 疗效追踪 |
| **治疗中期(4-6次)** | 中期评估 | GAD-7、PSWQ、SDS | 治疗调整 |
| **治疗结束** | 结局评估 | 全套量表 | 疗效评估 |
| **随访(3-6-12月)** | 维持评估 | GAD-7、PSWQ、SDS | 复发监测 |

## 9. 个案公式化 (Case Formulation)

### 表9.1 GAD个案公式化框架 (GAD Case Formulation Framework)

| 成分 | 内容 | 评估来源 |
|---|---|---|
| **主诉问题** | 当前症状和困扰 | 临床访谈 |
| **触发因素** | 导致当前发作的近因 | 病史采集 |
| **易感因素** | 增加GAD风险的因素 | 发展史、气质评估 |
| **维持因素** | 当前维持症状的因素 | 认知和行为分析 |
| **保护因素** | 缓冲和资源 | 优势评估 |
| **治疗靶点** | 需要干预的关键因素 | 综合分析 |
| **治疗目标** | 具体、可测量的目标 | 协商确定 |

### 表9.2 维持因素分析模板 (Maintaining Factors Analysis)

| 维持机制类别 | 具体因素 | 评估工具 | 干预策略 |
|---|---|---|---|
| **认知因素** | 不确定性不耐受、元认知信念、灾难化 | IUS、MCQ-30 | 认知重构、MCT |
| **行为因素** | 回避、安全行为、寻求保证 | 行为分析 | 行为实验、暴露 |
| **情绪因素** | 情绪回避、情绪失调 | DERS、AAQ-II | 接纳、情绪调节 |
| **生理因素** | 慢性紧张、睡眠问题 | 躯体评估 | 放松、睡眠干预 |
| **环境因素** | 应激源、关系问题 | 环境评估 | 问题解决、支持 |

## 10. 特殊评估考量 (Special Assessment Considerations)

### 表10.1 文化适应性评估 (Culturally Adapted Assessment)

| 考量 | 内容 | 策略 |
|---|---|---|
| **语言** | 工具的语言和文化验证 | 使用经验证的翻译版本 |
| **概念等效性** | "担忧"概念可能因文化而异 | 使用文化相关的表述 |
| **躯体化** | 某些文化更强调躯体症状 | 全面评估躯体症状 |
| **表达方式** | 情绪表达的文化差异 | 开放性询问 |
| **病耻感** | 精神障碍的文化病耻感 | 正常化、去污名化 |

### 表10.2 特殊人群评估调整 (Assessment Adaptations for Special Populations)

| 人群 | 评估调整 | 推荐工具 |
|---|---|---|
| **儿童青少年** | 使用适龄语言；纳入父母报告 | SCARED、RCADS |
| **老年人** | 考虑躯体疾病重叠；认知筛查 | GAI、GDS |
| **妊娠期** | 评估对妊娠的影响 | 标准工具适用 |
| **认知障碍** | 简化评估；依赖观察和照护者报告 | 简化版本 |
| **躯体疾病** | 区分焦虑与疾病症状 | HADS |
