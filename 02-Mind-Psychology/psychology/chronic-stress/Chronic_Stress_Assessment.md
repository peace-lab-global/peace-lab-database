## 1. 评估框架概览 (Assessment Framework Overview)

### 1.1 多维评估原则

| 评估维度 | 内容 | 方法 |
|:---------|:-----|:-----|
| **主观体验** | 感知压力、情绪状态、认知评价 | 自评量表、访谈 |
| **应激源** | 生活事件、慢性困扰、日常麻烦 | 事件量表、日记 |
| **应对资源** | 社会支持、应对策略、韧性 | 资源评估工具 |
| **生理指标** | HPA轴、交感神经、炎症 | 生物样本、生理监测 |
| **功能影响** | 工作、社交、健康行为 | 功能评估、行为监测 |
| **健康后果** | 身心症状、疾病风险 | 症状量表、医学检查 |

### 1.2 评估时机与情境

| 情境 | 评估重点 | 推荐工具 |
|:-----|:---------|:---------|
| **筛查** | 快速识别高压力个体 | PSS-4, 单项视觉模拟量表 |
| **临床初评** | 全面了解压力状态 | PSS-10, DASS-21, 生活事件量表 |
| **深度评估** | 机制分析、干预规划 | 生物标志物 + 多维问卷电池 |
| **研究目的** | 标准化、可比性 | 金标准量表 + 生理指标 |
| **职业健康** | 工作相关压力 | JSS, ERI, MBI |
| **纵向监测** | 变化追踪 | 重复测量、日记法 |

## 2. 自评量表：感知压力 (Self-Report Scales: Perceived Stress)

### 2.1 感知压力量表 (Perceived Stress Scale, PSS)

| 信息 | 详情 |
|:-----|:-----|
| **开发者** | Sheldon Cohen, Tom Kamarck, Robin Mermelstein |
| **年份** | 1983 |
| **版本** | PSS-14, PSS-10, PSS-4 |
| **理论基础** | Lazarus交易压力理论 |
| **测量内容** | 过去一个月感知到的不可预测性、不可控制性、超负荷程度 |
| **题目示例** | "在过去一个月里，你有多经常感到无法控制生活中重要的事情？" |
| **评分** | 0-4分Likert量表，PSS-10范围0-40 |
| **解释** | PSS-10: 0-13低压力, 14-26中等, 27-40高压力 |
| **信效度** | Cronbach's α = 0.78-0.91; 与抑郁、焦虑、健康行为相关 |
| **优点** | 简短、信效度好、跨文化验证广泛 |
| **局限** | 不区分应激源类型、回顾性偏差 |
| **中文版** | 杨廷忠等(2003)验证，信效度良好 |

### 2.2 抑郁-焦虑-压力量表 (DASS-21)

| 信息 | 详情 |
|:-----|:-----|
| **开发者** | Lovibond & Lovibond |
| **年份** | 1995 |
| **题数** | 21题(压力分量表7题) |
| **测量内容** | 压力分量表测量紧张、激惹、过度反应、急躁 |
| **评分** | 0-3分，压力分量表0-21(乘2后0-42) |
| **切分点(压力)** | 正常0-14, 轻度15-18, 中度19-25, 重度26-33, 极重度34+ |
| **优点** | 同时评估抑郁、焦虑、压力；区分度好 |
| **中文版** | 多个版本，广泛使用 |

### 2.3 压力过载量表 (Stress Overload Scale, SOS)

| 信息 | 详情 |
|:-----|:-----|
| **开发者** | Amirkhan |
| **年份** | 2012 |
| **题数** | 30题(短版24题) |
| **测量内容** | 个人脆弱性、事件负荷两个维度 |
| **特点** | 强调压力的"过载"体验 |
| **适用** | 临床和研究 |

## 3. 自评量表：应激源评估 (Stressor Assessment Scales)

### 3.1 生活事件量表

#### 3.1.1 社会再适应评定量表 (Social Readjustment Rating Scale, SRRS)

| 信息 | 详情 |
|:-----|:-----|
| **开发者** | Holmes & Rahe |
| **年份** | 1967 |
| **题数** | 43项生活事件 |
| **理论** | 生活变化需要再适应，累积变化增加疾病风险 |
| **评分** | 每项事件有预设"生活变化单位"(LCU)，总分为累加 |
| **经典事件权重** | 配偶死亡100, 离婚73, 结婚50, 失业47, 退休45 |
| **解释** | <150低风险, 150-299中等风险, ≥300高风险 |
| **局限** | 忽视主观评价、文化差异、正负事件区分 |

#### 3.1.2 生活经历调查 (Life Experiences Survey, LES)

| 信息 | 详情 |
|:-----|:-----|
| **开发者** | Sarason, Johnson, Siegel |
| **年份** | 1978 |
| **改进** | 区分正负性影响、主观评价(–3到+3) |
| **分数** | 可分别计算正性变化分数、负性变化分数、总变化分数 |
| **优点** | 考虑主观感受，区分正负事件 |

#### 3.1.3 日常麻烦量表 (Daily Hassles Scale, DHS)

| 信息 | 详情 |
|:-----|:-----|
| **开发者** | Kanner, Coyne, Schaefer, Lazarus |
| **年份** | 1981 |
| **理念** | 日常小烦恼可能比重大生活事件更影响健康 |
| **测量** | 频率和严重程度 |
| **领域** | 工作、家庭、健康、时间压力、环境、经济等 |
| **研究发现** | 日常麻烦比生活事件更能预测心理健康 |

## 5. 生理指标评估 (Physiological Assessment)

### 5.1 HPA轴评估

#### 5.1.1 皮质醇测量方法

| 方法 | 样本 | 时间窗 | 优点 | 局限 |
|:-----|:-----|:-------|:-----|:-----|
| **血清皮质醇** | 血液 | 即时(结合态+游离态) | 临床标准 | 侵入性、应激反应性、点测量 |
| **唾液皮质醇** | 唾液 | 即时(游离态) | 非侵入、可多点采集 | 受饮食、口腔出血影响 |
| **尿液皮质醇** | 尿液 | 24小时累积 | 日内总量 | 收集不便、受肾功能影响 |
| **发际皮质醇** | 头发 | 1-3个月回顾 | 长期累积暴露 | 受发质、洗护、环境影响 |

#### 5.1.2 皮质醇节律评估协议

| 评估 | 采样时间 | 测量内容 |
|:-----|:---------|:---------|
| **皮质醇觉醒反应(CAR)** | 醒后0, 15, 30, 45, 60分钟 | HPA反应性 |
| **昼夜曲线** | 醒后、中午、下午、晚间、睡前 | 节律模式 |
| **地塞米松抑制试验** | 前晚服药，次日测皮质醇 | HPA负反馈 |
| **ACTH刺激试验** | 注射ACTH后测皮质醇 | 肾上腺储备 |
| **CRH刺激试验** | 注射CRH后测ACTH和皮质醇 | 垂体-肾上腺反应 |

#### 5.1.3 皮质醇指标解释

| 指标 | 计算方法 | 临床意义 |
|:-----|:---------|:---------|
| **CAR幅度** | CAR峰值 - 醒时值 | 正常50-75%增加；钝化或增强均异常 |
| **CAR曲线下面积(AUCg)** | 梯形法计算总量 | 总反应量 |
| **CAR增量面积(AUCi)** | 相对于基线的增量 | 反应性 |
| **昼夜斜率** | (晨峰 - 睡前) / 时间 | 斜率变平与不良健康相关 |
| **日内变异性** | 标准差或变异系数 | 节律完整性 |

### 5.2 自主神经功能评估

#### 5.2.1 心率变异性(HRV)

| 指标 | 域 | 定义 | 意义 |

| **VLF** | 频域 | 极低频功率(<0.04Hz) | 热调节、RAAS等 |

## 6. 异稳态负荷评估 (Allostatic Load Assessment)

### 6.1 经典异稳态负荷指标体系

| 系统 | 指标 | 高风险切点(示例) |
|:-----|:-----|:----------------|
| **心血管** | 收缩压 | >148 mmHg (人群75%tile) |
| **心血管** | 舒张压 | >83 mmHg |
| **心血管** | 静息心率 | >72 bpm |
| **代谢** | 腰臀比 | >0.94(男), >0.84(女) |
| **代谢** | HDL胆固醇 | <37 mg/dL |
| **代谢** | 总胆固醇/HDL | >5.9 |
| **代谢** | 糖化血红蛋白 | >7.1% |
| **神经内分泌** | 尿皮质醇(12h) | 人群前25% |
| **神经内分泌** | 尿肾上腺素(12h) | 人群前25% |
| **神经内分泌** | 尿去甲肾上腺素(12h) | 人群前25% |
| **神经内分泌** | DHEA-S | 人群后25% |
| **炎症** | CRP | >3.19 mg/L |
| **炎症** | 纤维蛋白原 | >336 mg/dL |

### 6.2 计算方法

| 方法 | 描述 | 优缺点 |
|:-----|:-----|:-------|
| **计数法** | 超过高风险切点的指标数目 | 简单；忽略程度差异 |
| **Z分数总和** | 各指标Z分数求和 | 考虑连续性；可能受极端值影响 |
| **规范化四分位法** | 每指标按四分位赋分后求和 | 平衡；常用 |
| **加权法** | 根据重要性加权 | 理论依据；权重确定困难 |
| **主成分分析** | 数据驱动整合 | 统计优化；解释困难 |

## 7. 临床访谈与结构化评估 (Clinical Interview & Structured Assessment)

### 7.1 慢性压力临床访谈框架

| 评估领域 | 关键问题 |
|:---------|:---------|
| **压力感知** | "您觉得目前生活压力有多大？（0-10分）" |
| **主要应激源** | "目前最让您感到压力的是什么？持续多久了？" |
| **功能影响** | "这些压力如何影响您的工作、家庭和社交？" |
| **躯体症状** | "您注意到哪些身体变化或不适？" |
| **情绪状态** | "您最近的情绪状态如何？睡眠、食欲有变化吗？" |
| **应对方式** | "您通常如何应对压力？什么方法有效/无效？" |
| **支持系统** | "您感到压力时可以找谁倾诉或帮助？" |
| **既往史** | "以前有过类似的压力时期吗？您是如何度过的？" |
| **风险评估** | "是否有伤害自己的想法？" |

### 7.2 Trier社会压力测试 (TSST)

| 信息 | 详情 |
|:-----|:-----|
| **目的** | 实验室标准化急性心理社会应激诱发 |
| **程序** | 5分钟准备 + 5分钟演讲(模拟面试) + 5分钟心算，面对评委 |
| **测量** | 皮质醇、ACTH、儿茶酚胺、心率、血压、主观压力 |
| **时间点** | 基线、任务后0/10/20/30/45/60分钟 |
| **应用** | 评估HPA轴反应性、研究个体差异 |
| **变体** | TSST-VR(虚拟现实版)、TSST-C(儿童版)、TSST-G(团体版) |

### 7.3 生态瞬时评估 (Ecological Momentary Assessment, EMA)

| 特征 | 描述 |
|:-----|:-----|
| **定义** | 在自然环境中实时、反复采集数据 |
| **方法** | 智能手机app、电子日记、可穿戴设备 |
| **优点** | 生态效度高、减少回顾偏差、捕捉动态变化 |
| **测量内容** | 瞬时压力、情绪、行为、生理指标 |
| **采样策略** | 时间触发、事件触发、随机信号 |
| **应用** | 压力-症状关系、日常应对过程、干预监测 |

### 8.2 老年人

| 考虑 | 建议 |
|:-----|:-----|
| **认知状态** | 先评估认知，调整方法 |
| **感官功能** | 确保能阅读/听清 |
| **应激源** | 关注健康、丧失、社会孤立、经济 |
| **皮质醇** | 基线和节律随年龄变化 |
| **多病共存** | 分辨压力vs疾病症状 |
| **药物影响** | 许多药物影响HPA轴 |

### 8.3 跨文化考虑

| 考虑 | 建议 |
|:-----|:-----|
| **语言** | 使用经过文化适应的翻译版本 |
| **压力概念** | 不同文化对压力的理解和表达不同 |
| **躯体化** | 某些文化更倾向躯体表达 |
| **耻感** | 心理问题在某些文化中更具耻感 |
| **应对方式** | 宗教、家庭支持在不同文化中角色不同 |
| **常模** | 使用文化相关常模解释结果 |

## 9. 综合评估报告框架 (Integrated Assessment Report Framework)

### 9.1 评估报告结构

```
1. 基本信息与主诉
2. 压力源评估
   - 主要应激源识别
   - 持续时间与严重程度
   - 可控性评估
3. 感知压力水平
   - 主观压力评分
   - 标准化量表结果
4. 生理指标(如有)
   - HPA轴状态
   - 自主神经功能
   - 炎症/代谢标志物
5. 应对资源评估
   - 应对策略分析
   - 社会支持网络
   - 韧性水平
6. 功能影响
   - 工作/学业功能
   - 人际关系
   - 健康行为
   - 生活质量
7. 风险因素与保护因素
8. 诊断考虑(如适用)
9. 综合评估结论
10. 干预建议
```

### 9.2 严重程度分级参考

| 级别 | 感知压力 | 功能影响 | 生理指标 | 建议干预 |
|:-----|:---------|:---------|:---------|:---------|
| **轻度** | PSS<14 | 轻微或无 | 正常范围 | 自我管理、预防教育 |
| **中度** | PSS 14-26 | 部分领域受影响 | 轻度异常 | 结构化压力管理课程 |
| **重度** | PSS 27-40 | 多领域显著影响 | 多指标异常 | 专业干预、可能需药物 |
| **危重** | 极端 | 功能严重损害 | 系统性异常 | 综合治疗、危机干预 |

## 11. 专业视角 (Professional Perspectives)

### 心理测量学视角

> **Sheldon Cohen:** "PSS测量的不是应激源本身，而是个体对生活不可预测性、不可控制性和超负荷程度的感知。这种主观评价比客观事件更能预测健康结局。"

### 生物行为医学视角

> **Bruce McEwen:** "单一生物标志物不足以评估慢性压力的影响。异稳态负荷指标体系通过整合多个系统的指标，能更好地预测健康风险。"

### 临床实践视角

> "综合评估应该整合主观报告和客观指标。主观感受告诉我们患者的体验，生理指标告诉我们身体的状态，两者都是干预规划所必需的。"
