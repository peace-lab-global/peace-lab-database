# 压力状态综合评估技能

## 技能信息

**技能ID**: S_001
**技能名称**: 压力状态综合评估
**功能**: 评估用户压力状态、分类压力类型、判断严重程度
**输出**: 结构化评估报告 + 建议下一步

---

## 评估协议

### 第一步：收集基本信息

收集以下信息（通过提问获取）：

| 信息项 | 问题示例 | 重要性 |
|:-------|:---------|:------:|
| 压力持续时间 | "这种压力感觉持续多久了？" | 必需 |
| 主要压力源 | "目前最主要的压力来自哪里？" | 必需 |
| 身体症状 | "最近有什么身体上的不适吗？" | 必需 |
| 睡眠状况 | "睡眠质量怎么样？" | 必需 |
| 情绪状态 | "最近情绪状态如何？" | 必需 |
| 功能影响 | "压力对你的日常生活影响大吗？" | 重要 |
| 既往史 | "以前有过类似经历吗？" | 参考 |

### 第二步：压力类型分类

根据收集的信息，使用以下决策树分类：

```
压力类型分类决策树
│
├─ 持续时间 < 2周？
│   └─ 是 → 【急性压力】
│
├─ 持续时间 ≥ 2周且 < 3个月？
│   └─ 是 → 【亚慢性压力】
│       ├─ 压力源明确且可解决 → 【可解决型慢性压力】
│       └─ 压力源复杂或多源 → 【复合型慢性压力】
│
└─ 持续时间 ≥ 3个月？
    └─ 是 → 【慢性压力】
        ├─ 主要影响心血管/代谢 → 【躯体化慢性压力】
        ├─ 主要影响情绪/认知 → 【情绪化慢性压力】
        ├─ 疲劳极度严重+劳后恶化 → 【CFS疑似】→ 触发S_006
        └─ 多系统症状 → 【泛化型慢性压力】
```

### 第三步：严重程度评估

使用 **感知压力量表 (PSS-10)** 或等效问题：

**PSS-10 问题**（过去一个月）：
1. 过去一个月，有多少次因意料之外的事情而感到不安？
2. 过去一个月，有多少次感到无法控制生活中重要的事情？
3. 过去一个月，有多少次感到紧张和压力？
4. 过去一个月，有多少次成功应对烦恼的生活事件？
5. 过去一个月，有多少次能解决恼人的事情？
6. 过去一个月，有多少次能掌控所有需要做的事情？
7. 过去一个月，有多少次发现自己一直在思考需要完成的事情？
8. 过去一个月，有多少次感到困难堆积如山而无法克服？

**评分**: 0=从不，1=几乎没有，2=偶尔，3=经常，4=非常频繁
**总分 = 所有问题分数之和**
**反向计分**: 问题4、5、6

| 总分 | 严重程度 | 建议行动 |
|:----:|:--------:|:---------|
| 0-13 | 低压力 | 维持性建议 |
| 14-19 | 中低压力 | 关注预防 |
| 20-26 | 中高压力 | 主动干预 |
| 27-40 | 高压力 | 积极干预+监测 |

### 第四步：HPA轴功能判断

通过症状组合推测HPA轴状态：

| 症状组合 | 推测HPA轴状态 | 建议 |
|:---------|:--------------|:-----|
| 持续疲劳+晨起困难+低皮质醇样症状 | 肾上腺功能低下/耗竭 | 建议检测皮质醇，医疗评估 |
| 焦虑+失眠+夜间皮质醇升高 | HPA轴过度激活 | 优先减压+睡眠干预 |
| 情绪波动+能量不稳+节律紊乱 | HPA轴节律失调 | 节律调节优先 |
| 应激后持续疲劳+难以恢复 | HPA轴反应迟钝 | 适应原+休息 |

## 决策规则

### 紧急转介判断

**立即建议就医** 如果存在以下任一情况：
- 持续严重失眠（>2周）
- 无法控制的自伤/自杀念头
- 严重头痛+视觉障碍+高血压（疑似嗜铬细胞瘤）
- 极度疲劳+无法起床+免疫症状（疑似CFS/ME）
- 恐慌发作频繁

**建议专业帮助** 如果：
- PSS > 26
- 压力持续 > 3个月且自我干预无效
- 出现抑郁或焦虑障碍症状
- 社会功能严重受损

## 使用示例

**用户输入**: "我最近工作压力特别大，已经持续一个月了，每天睡不好，白天很累，注意力也集中不了"

**评估流程**:
1. 收集信息：持续1个月，工作压力源，睡眠障碍，疲劳，注意力问题
2. 类型分类：亚慢性压力 → 复合型慢性压力
3. 严重程度：需要PSS评估，假设中等偏高
4. HPA轴：可能存在节律紊乱和轻度激活

**输出报告**:
```
【压力评估报告】

元数据
- 技能ID: S_001
- 调用时间: 2026-04-10 10:00
- 会话阶段: assessment

一、压力类型: 亚慢性/慢性压力（复合型）
二、严重程度: 中高 (需完成PSS-10确认)
三、主要压力源: 职业压力
四、HPA轴推测状态: 可能有昼夜节律紊乱
五、优先关注症状:
   - 睡眠障碍（入睡困难/早醒）
   - 日间疲劳
   - 注意力下降
六、建议行动:
   1. 完成压力日记记录以明确模式
   2. 优先改善睡眠（睡眠卫生+CBT-I）
   3. 开始每日放松训练（10-15分钟）
   4. 设定工作边界

   - 已完成技能: [S_001]

   - 转介需求: 否
```
