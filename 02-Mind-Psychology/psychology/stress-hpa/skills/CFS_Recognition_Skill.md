# 慢性疲劳综合征识别技能

## 技能信息

**技能ID**: S_006
**技能名称**: 慢性疲劳综合征识别
**功能**: 识别疑似ME/CFS症状，区分普通疲劳与病理性疲劳
**输入**: 用户疲劳相关症状
**输出**: 评估报告 + 建议行动

---

## 关键概念

### ME/CFS定义

肌痛性脑脊髓炎/慢性疲劳综合征（ME/CFS）是一种严重的多系统疾病，以以下核心症状为特征：
- **严重疲劳**：持续≥6个月，休息无法缓解
- **劳后不适（PEM）**：活动后症状恶化，延迟24-72小时出现
- **非恢复性睡眠**：无论睡多久都感觉不清醒
- **认知障碍**：注意力、记忆力、执行功能受损
- **直立不耐受**：站立时症状加重

### 核心特征：劳后不适(PEM)

**PEM是ME/CFS的核心诊断特征**

PEM定义：轻微活动后（甚至日常活动）出现症状显著恶化，通常延迟24-72小时出现，可持续数小时至数天。

**识别PEM的关键问题**:
1. "你在活动后（比如散步、工作、家务）症状会恶化吗？"
2. "这种恶化通常在活动后多久出现？"
3. "恢复需要多长时间？"
4. "如果你哪天多做了一些，第二天会怎样？"

| 如果回答 | 提示 |
|:--------|:-----|
| "活动后确实会累，但休息一下就好了" | 可能是普通疲劳 |
| "活动后会特别累，要休息很久才能恢复" | 疑似PEM，需进一步评估 |
| "如果哪天多做了，第二天肯定崩溃" | 高度疑似PEM |
| "我不敢多做，因为每次都会病好几天" | 典型PEM |

## 核心症状评估

### 1. 严重疲劳

**问题**:
- "你的疲劳有多严重？"
- "休息能缓解疲劳吗？"
- "疲劳对你的日常生活影响有多大？"

**ME/CFS特征**:
- 疲劳严重（日常活动能力显著下降）
- 休息后不能明显缓解
- 不是由持续过度活动引起

### 2. 劳后不适(PEM)

**详细评估问题**:

| 问题 | 评估点 |
|:-----|:-------|
| "轻微活动后（比如散步15-30分钟）症状会加重吗？" | 活动耐受阈值 |
| "症状加重通常在活动后多久出现？" | PEM延迟时间（通常24-72h）|
| "这种加重会持续多久？" | PEM持续时间 |
| "你是否因为担心PEM而避免某些活动？" | 症状性回避 |
| "如果某天工作量超过平时，第二天会怎样？" | PEM触发模式 |

### 3. 非恢复性睡眠

**问题**:
- "你睡醒后感觉怎么样？"
- "不管睡多久都觉得不清醒吗？"
- "有入睡困难或早醒吗？"

### 4. 认知障碍（脑雾）

**问题**:
- "你的注意力/记忆力有问题吗？"
- "有'脑雾'的感觉吗？"
- "做决定或组织事情困难吗？"

### 5. 直立不耐受/神经血管收缩

**问题**:
- "站立时感觉头晕或疲劳吗？"
- "站立时心跳会明显加快吗？"
- "需要坐下或躺下才能缓解吗？"

## 诊断标准对照

### IOM 2015诊断标准（简化）

**必须满足**:
1. 严重疲劳 ≥6个月，新发或明确发作
2. 劳后不适（PEM）
3. 非恢复性睡眠

**至少满足以下一项**:
- 认知障碍
- 直立不耐受

---

## 警示信号（红旗）

**如果存在以下任何一项，需紧急医学评估**:

| 红旗信号 | 考虑诊断 |
|:---------|:---------|
| 体重急剧下降 | 恶性肿瘤、甲状腺功能亢进、艾滋病 |
| 持续发热 | 感染性疾病、炎症性疾病 |
| 严重头痛+颈项强直 | 脑膜炎 |
| 全身淋巴结肿大 | 恶性肿瘤、艾滋病 |
| 肌肉无力进行性加重 | 神经肌肉疾病 |
| 出血性皮疹 | 脑膜炎球菌血症 |
| 呼吸困难 | 心肺疾病 |
| 胸痛+呼吸困难 | 心血管急症 |
| 严重抑郁+自杀念头 | 需要精神科评估 |

## 使用示例

**用户输入**: "我已经连续几个月都感觉很累，不管睡多久都不解乏。有一次我周末多走了点路，结果接下来一周都特别累，感觉要从床上爬起来都很难。去医院查过甲状腺什么的都是正常的。"

**评估过程**:
1. 持续时间：几个月 → ≥6个月需确认
2. PEM：明显存在（多走路→一周崩溃）→ 核心特征符合
3. 睡眠：睡多久都不解乏 → 符合非恢复性睡眠
4. PEM模式：延迟出现（不是立刻累，是之后）→ 典型PEM
5. 其他排查：已排查甲状腺

## 关键提醒

### ME/CFS ≠ 普通疲劳

| 误区 | 正确认知 |
|:-----|:---------|
| "只是累了，休息一下就好" | ME/CFS休息无法缓解，需要特殊管理 |
| "多运动能改善" | graded exercise可能加重PEM |
| "这是心理问题" | ME/CFS是生物医学疾病，非心理疾病 |
| "不活动会更糟" | 过度活动是PEM恶化的主要原因 |

### 如果疑似ME/CFS

1. **不建议**: 独自进行高强度运动或 graded exercise therapy (GET)
2. **建议**: 活动 pacing，保持在能量 envelope 内
3. **建议**: 专业评估（ME/CFS专科或经验丰富的全科医生）
4. **监测**: 记录能量波动和PEM模式

---

*ME/CFS的诊断需要排除其他疾病并由专业医疗人员做出。本技能提供初步识别指导，不替代专业医学评估。*
