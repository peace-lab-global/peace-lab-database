# 皮质醇节律评估技能

## 技能信息

**技能ID**: S_003
**技能名称**: 皮质醇节律评估
**功能**: 评估皮质醇昼夜节律，识别异常模式
**输入**: 能量、睡眠、压力相关症状
**输出**: 皮质醇节律推测 + 调节建议

---

## 皮质醇基础知识

### 正常昼夜节律

| 时段 | 皮质醇水平 | 生理状态 |
|:-----|:----------|:---------|
| 06:00-08:00 | **峰值** (10-20 μg/dL) | 觉醒，警觉性最高 |
| 08:00-12:00 | 高位下降 | 维持日间活动 |
| 12:00-18:00 | 逐渐降低 | 代谢调节 |
| 18:00-24:00 | 低谷前期 | 准备休息 |
| 00:00-04:00 | **谷值** (<3 μg/dL) | 细胞修复，免疫增强 |
| 04:00-06:00 | 快速上升 | 准备觉醒 |

### 皮质醇觉醒反应 (CAR)

- 觉醒后30-45分钟内皮质醇急剧上升50-75%
- 为日间活动准备
- 反映HPA轴反应性

---

## 评估决策树

```
皮质醇节律评估

STEP 1: 评估晨起状态
│
├─ 晨起感觉？
│   ├─ 困难、疲惫、数小时才清醒 → 可能的CAR缺失/肾上腺低下
│   ├─ 逐渐清醒，但需要时间 → 轻度低下可能
│   ├─ 自然醒来，感觉清醒 → 正常CAR可能
│   └─ 醒来时焦虑、过度警觉 → 可能的过度激活
│
STEP 2: 评估日间能量
│
├─ 日间能量模式？
│   ├─ 上午最差，逐渐改善 → 晨峰缺失
│   ├─ 持续稳定 → 正常
│   ├─ 上午好，下午崩溃 → 皮质醇下降过快
│   ├─ 全天疲劳 → 肾上腺功能问题
│   └─ 波动大 → 节律紊乱
│
STEP 3: 评估晚间状态
│
├─ 夜间和睡前状态？
│   ├─ 入睡困难、躺下后脑子停不下来 → 夜间皮质醇高
│   ├─ 正常困倦入睡 → 正常
│   ├─ 凌晨易醒、难以再睡 → 可能是夜间低谷异常
│   └─ 嗜睡、早睡 → 可能的节律相位提前
│
STEP 4: 综合判断
```

## 节律模式分析

### 正常节律

| 特征 | 描述 |
|:-----|:-----|
| 晨起 | 相对清醒，30-60分钟内完全清醒 |
| 日间 | 能量稳定，无极端波动 |
| 午后 | 轻微疲倦（正常下午低谷） |
| 晚间 | 逐渐放松，入睡容易 |
| 睡眠 | 稳定，可能有一次醒来 |

### 异常节律模式

#### 1. 晨峰缺失 (Flattened AM Rise)

| 推测症状 | 你是否有以下表现？ |
|:---------|:------------------|
| 难以起床 | 需要闹钟，醒来后还是有些困 |
| 上午昏沉 | 上午工作效率极低 |
| 能量延迟 | 中午后逐渐清醒 |
| 需要咖啡 | 早上必须喝咖啡才能正常运作 |

**推测机制**: 肾上腺功能低下，CAR缺失

#### 2. 曲线扁平化 (Flattened Curve)

| 推测症状 | 你是否有以下表现？ |
|:---------|:------------------|
| 全天感觉"一样累" | 没有明显的高能量或低能量时段 |
| 难以区分时段 | 不知道什么时候该工作 |
| 情绪平淡 | 感觉没有起伏 |
| 恢复困难 | 压力后难以恢复 |

**推测机制**: HPA轴钝化，常见于慢性疲劳

#### 3. 相位延迟 (Phase Delay)

| 推测症状 | 你是否有以下表现？ |
|:---------|:------------------|
| 晚睡晚起倾向 | 难以在23点前入睡 |
| 夜间皮质醇高 | 睡前感觉不困、脑子活跃 |
| 晨起困难 | 早睡依然晚醒 |
| 晚间警觉 | 晚上是最佳工作时间 |

**推测机制**: 中央时钟紊乱，常见于夜班工作

#### 4. 夜间皮质醇升高 (Nocturnal Rise)

| 推测症状 | 你是否有以下表现？ |
|:---------|:------------------|
| 入睡困难 | 躺下后脑子转不停 |
| 夜间清醒 | 凌晨2-3点醒来并清醒 |
| 睡眠浅 | 容易醒来 |
| 噩梦 | 压力下更明显 |

**推测机制**: 压力持续激活，夜间无法恢复

#### 5. 过度激活 (Hyperactivation)

| 推测症状 | 你是否有以下表现？ |
|:---------|:------------------|
| 过度警觉 | 总是"处于警戒"状态 |
| 心跳快 | 静息心率高 |
| 焦虑 | 无法放松 |
| 反应过激 | 对小事反应过度 |

**推测机制**: HPA轴过度敏感

## 评估问题集

### 晨起评估

| 问题 | 选项 | 评估 |
|:-----|:-----|:-----|
| 你通常几点醒来？ | ____ | 评估生物钟 |
| 醒来后多久感觉清醒？ | <15分钟/15-30分钟/30-60分钟/>1小时 | CAR评估 |
| 需要闹钟吗？ | 是/否 | 评估自然觉醒能力 |
| 醒来时身体感觉？ | 精力充沛/逐渐清醒/疲惫/焦虑 | 基础状态 |

### 日间评估

| 问题 | 选项 | 评估 |
|:-----|:-----|:-----|
| 一天中什么时候能量最高？ | 上午/下午/晚上/差不多 | 峰谷分析 |
| 什么时候能量最低？ | 上午/下午/晚上/差不多 | 低谷分析 |
| 下午特别困吗？ | 经常/有时/很少/从不 | 午后低谷 |
| 咖啡或能量饮料依赖？ | 大量/适量/偶尔/不用 | 肾上腺依赖 |

### 夜间评估

| 问题 | 选项 | 评估 |
|:-----|:-----|:-----|
| 通常几点入睡？ | ____ | 睡眠时间 |
| 入睡前感觉困吗？ | 非常/有点/不困/焦虑 | 夜间皮质醇 |
| 入睡需要多长时间？ | <15分钟/15-30分钟/30-60分钟/>1小时 | 入睡困难 |
| 夜间会醒吗？ | 从不/偶尔/经常（1-2次）/多次 | 睡眠维持 |
| 如果夜间醒来，多久再入睡？ | 马上/几分钟/10分钟以上 | 夜间皮质醇 |

### 压力反应评估

| 问题 | 选项 | 评估 |
|:-----|:-----|:-----|
| 压力后恢复需要多长时间？ | 马上/几小时/1-2天/更久 | HPA轴反应性 |
| 对小压力反应大吗？ | 非常/有点/一般/迟钝 | 反应性 |
| 压力时出现身体症状？ | 头痛/胃痛/心悸/肌肉紧张/无 | 躯体化程度 |

## 节律模式判断表

| 晨起 | 日间 | 夜间 | 推测模式 | 建议 |
|:----:|:----:|:----:|:---------|:-----|
| 困难 | 上午差下午好 | 入睡慢 | 晨峰缺失 | 肾上腺支持 |
| 困难 | 全天低 | 早睡 | 整体低下 | 肾上腺支持 |
| 正常 | 稳定 | 正常 | 正常节律 | 维持 |
| 正常 | 午后崩溃 | 正常 | 下降过快 | 节律调节 |
| 清醒 | 晚上好 | 不困 | 相位延迟 | 节律重置 |
| 早醒 | 下午差 | 早困 | 相位提前 | 节律调节 |
| 困难 | 波动大 | 睡不着 | 紊乱 | 综合调节 |
| 焦虑 | 过度警觉 | 脑子不停 | 过度激活 | 镇静调节 |

## 使用示例

**用户输入**: "我每天晚上都睡不着，躺在床上脑子就停不下来，总是要到2-3点才睡着，第二天睡到10点多才醒，但还是觉得不太清醒。下午3-4点会特别困。"

**评估过程**:
1. 晨起：10点多醒 → 相位延迟明显
2. 日间：下午困 → 午后低谷+晚起导致
3. 夜间：入睡困难，脑子停不下来 → 夜间皮质醇高/相位延迟
4. 综合：相位延迟 + 夜间皮质醇高

**输出报告**:
```
【皮质醇节律评估报告】

元数据
- 技能ID: S_003
- 调用时间: 2026-04-10 10:20
- 会话阶段: depth

一、皮质醇推测模式: 相位延迟 + 夜间升高
   - 晨峰: 缺失/延后（10点多才醒来）
   - 日间: 午后低谷（3-4点困）
   - 夜间: 升高（2-3点才入睡）

二、主要症状对应:
   - 夜间入睡困难（脑子停不下来）
   - 晚睡晚起
   - 午后疲劳

三、节律问题判断:
   - 皮质醇昼夜节律相位严重延迟
```
