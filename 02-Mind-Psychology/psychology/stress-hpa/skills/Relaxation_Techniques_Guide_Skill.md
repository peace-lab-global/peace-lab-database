# 放松技术指导技能

## 技能信息

**技能ID**: S_009
**技能名称**: 放松技术指导
**功能**: 根据用户情况推荐合适的放松技术，并提供具体指导
**输入**: 用户偏好、可用时间、特殊需求
**输出**: 个性化放松方案 + 具体技术指导

## 技术详解

### 1. 4-7-8呼吸法（急性焦虑/睡前）

**最适用**: 急性焦虑发作、睡前无法入睡、即时平复

**步骤**:
1. 舒服地坐着或躺下
2. 用鼻子吸气4秒（数1-2-3-4）
3. 屏住呼吸7秒（数1-2-3-4-5-6-7）
4. 用嘴缓缓呼气8秒（数1-2-3-4-5-6-7-8）
5. 重复3-4次

**机制**: 延长呼气激活副交感神经
**证据等级**: ★★★
**注意**: 屏住呼吸可能有困难，初学者可从3-5-3开始

### 2. 横膈膜呼吸（日常放松）

**最适用**: 每日练习、慢性压力管理、工作间隙

**步骤**:
1. 坐或躺舒适
2. 一只手放胸口，一只放腹部
3. 吸气时让腹部鼓起（胸部少动）
4. 呼气时腹部下降
5. 吸气4秒，呼气6秒
6. 每次练习5-10分钟

### 3. 渐进性肌肉放松（身体紧张）

**最适用**: 肌肉紧张、慢性疼痛、头痛、失眠

**快速版本（5-10分钟）**:
只做主要肌群：脚、腹部、胸部、手、肩膀、脸
- 每组紧张5-7秒 → 突然放松15-20秒

### 4. 身体扫描（失眠/情绪调节）

**最适用**: 失眠、无法停止思考、情绪波动

**步骤（15-30分钟）**:
1. 躺下，闭眼
2. 做几次深呼吸
3. 从脚趾开始，缓慢向上移动注意力
4. 每个部位停留几秒到几十秒
5. 如果思绪飘走，温和地带回

### 5. 正念呼吸冥想（慢性压力）

**最适用**: 长期焦虑、情绪波动、慢性压力

**步骤（10-45分钟）**:
1. 舒服地坐着，保持背部挺直
2. 闭上眼睛或微微下垂
3. 将注意力放在呼吸上
4. 当注意力飘走，温和地标记"走神了"
5. 将注意力轻轻带回呼吸

   - 可用时间: 15-30分钟（睡前）
   - 主要需求: 睡前放松，促进睡眠
   - 偏好: [待确认]

二、推荐技术组合

   主技术: 身体扫描（15-20分钟）
   - 时间: 睡前15-20分钟
   - 频率: 每晚
   - 要点: 不要"努力放松"，只是观察

   辅助技术: 4-7-8呼吸（5分钟）
   - 使用场景: 躺下后仍然清醒时
   - 要点: 如还没睡着，做3-4个循环

   睡前1小时:
   1. 调暗灯光（减少光线→增加褪黑激素）
   2. 关闭屏幕

   睡前30分钟:
   3. 温水澡或泡脚（体温先升后降→困意）

   睡前:
   4. 身体扫描（15-20分钟）
      或
   4-7-8呼吸 + 引导意象

四、效果监测
   - 记录：关灯时间、入睡时间、醒来感觉
   - 目标：30分钟内入睡，早晨精力恢复

五、技能链状态更新:
   - 已完成技能: [S_009]
   - 待完成技能: [无]
   - 用户画像: {relaxation_needs: "睡前+睡眠促进"}

六、安全检查:

## 常见问题处理

| 问题 | 解决方案 |
|:-----|:---------|
| "脑子停不下来" | 先做4-7-8呼吸，再做身体扫描 |
| "做一次没效果" | 放松技术需要练习，通常2-3周见效 |
| "没时间练习" | 从5分钟正念或3次深呼吸开始 |

---

*放松技术需要练习才能掌握。建议从一种技术开始，习惯后再添加其他技术。持续练习4周以上通常能感受到明显改善。*
