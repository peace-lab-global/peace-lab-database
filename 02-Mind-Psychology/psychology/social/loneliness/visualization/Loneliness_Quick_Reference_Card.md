## 正面 | PAGE 1

### 🔴 孤独快速筛查（3分钟）

**UCLA-3 简版**
> 1. 你有多经常感到与周围人缺乏亲密感？
> 2. 你有多经常感到没有人可以求助？
> 3. 你有多经常感到孤独？

| 总分 | 风险 | 行动 |
|------|------|------|
| 3分 | 极低 | 预防资源；年度复查 |
| 4-6分 | 低 | 自助资源；3个月复查 |
| 7-9分 | 中 | **安排详细评估** |
| 10-12分 | 高 | **紧急评估+安全筛查** |

**单条目替代**："过去一周，你有多少天感到孤独？"
- 0-1天=低 | 2-4天=中 | 5-7天=高

### 📊 De Jong Gierveld 11项 阈值速查

| EL(情感孤独 0-5) | SL(社交孤独 0-6) | 类型 | 干预重点 |
|-----------------|-----------------|------|----------|
| 高 | 低 | **情感孤独型** | 深化关系；情感表达训练；依恋治疗 |
| 低 | 高 | **社交孤独型** | 扩大网络；社交技能；团体活动 |
| 高 | 高 | **双重孤独型** | 综合干预；优先情感；防自杀 |
| 低 | 低 | **无孤独** | 维持；预防 |

**严重孤独阈值**：总分 ≥ 6分

### ⚡ 危机响应速查（60秒行动）

```
自杀风险评估 → Columbia Scale (5项)
    │
    ├── 0分 → 继续常规流程
    │
    ├── 1-2分 → 安全计划 + 增加频率 + 24h热线
    │
    ├── 3-4分 → 安全计划 + 精神科会诊 + 家庭参与
    │
    └── 5分 → 安全计划 + 立即精神科评估 + 考虑住院

安全计划5要素：
1. 警告信号识别
2. 内部应对策略
3. 分散注意活动
4. 可联系的人(姓名+电话)
5. 环境安全(移除方法)
```

### 📋 可靠变化指数(RCI)速算

```
UCLA-LS: RCI阈值 ≈ ±9.4分
DJG-11: RCI阈值 ≈ ±2.1分

判定：
• 改善 > +9.4分 → 可靠改善 ✓
• 变化 ±9.4分内 → 无可靠变化 ~
• 恶化 < -9.4分 → 可靠恶化 ✗
```
