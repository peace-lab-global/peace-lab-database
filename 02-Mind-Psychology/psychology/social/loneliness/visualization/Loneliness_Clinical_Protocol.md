## 2. 第一级：快速筛查流程

### 2.1 筛查触发场景

| 场景 | 筛查时机 | 执行者 |
|------|----------|--------|
| **初级医疗就诊** | 年度体检/慢病随访时 | 全科医生/护士 |
| **心理健康初诊** | 首次咨询/评估时 | 心理咨询师 |
| **社区服务接触** | 福利申请/社区活动时 | 社工 |
| **工作场所EAP** | 员工援助计划咨询时 | EAP顾问 |
| **学校咨询** | 学生心理咨询/辅导时 | 学校心理师 |
| **老年服务** | 居家养老评估/社区活动时 | 老年服务专员 |

### 2.2 快速筛查工具（3-5分钟）

**方案A：UCLA-3简版（最常用）**

> 1. 你有多经常感到与周围人缺乏亲密感？
> 2. 你有多经常感到没有人可以求助？
> 3. 你有多经常感到孤独？
>
> 评分：1=从不 2=很少 3=有时 4=总是

| 总分 | 风险等级 | 下一步 |
|------|----------|--------|
| **3分** | 极低风险 | 提供预防资源；年度复查 |
| **4-6分** | 低风险 | 提供自助资源；3个月复查 |
| **7-9分** | 中风险 | 安排第二级详细评估 |
| **10-12分** | 高风险 | 紧急安排详细评估+安全筛查 |

**方案B：单条目筛查（最快速）**

> "在过去一周中，你有多少天感到孤独？"

| 天数 | 风险等级 | 行动 |
|------|----------|------|
| 0-1天 | 低风险 | 预防性资源 |
| 2-4天 | 中风险 | 安排评估 |
| 5-7天 | 高风险 | 紧急评估+安全筛查 |

**方案C：De Jong Gierveld 11项简版（需要区分类型时）**

> 使用情感孤独(EL)和社交孤独(SL)各2个核心条目快速分型。

## 3. 第二级：详细评估流程

### 3.1 评估结构（45-60分钟）

| 模块 | 时间 | 工具/方法 | 目的 |
|------|------|----------|------|
| **A. 建立关系** | 5min | 开放式提问 | 建立信任；评估表达风格 |
| **B. 孤独史** | 10min | 时间线访谈 | 孤独起始、波动、触发因素 |
| **C. 孤独量表** | 10min | UCLA-20 + DJG-11 | 定量评估；EL/SL分型 |
| **D. 社会网络** | 10min | 社会网络图+EGO-NET | 可视化关系结构 |
| **E. 功能评估** | 5min | 工作/学习/日常生活 | 功能损害程度 |
| **F. 安全筛查** | 10min | PHQ-9 + GAD-7 + PCL-5简版 | 精神共病；创伤；自杀风险 |
| **G. 资源评估** | 5min | 社会支持；经济；文化 | 可用资源与障碍 |

### 3.2 社会网络图绘制法

**绘制步骤**：
1. 在纸中央画一个圆代表来访者
2. 让来访者画出所有重要的人（用圆圈表示）
3. 圆圈大小=关系重要性；距离远近=亲密度
4. 连线类型：实线=支持性，虚线=冲突性，双线=非常紧密
5. 标注：情感支持(E) / 实际支持(P) / 信息支持(I) / 陪伴(C)

**分析要点**：
- 网络大小（总人数）
- 密度（互相认识的人的比例）
- 多元性（不同角色类型的数量）
- 冲突/负面关系的比例
- 孤立节点（无连接的重要他人）

### 5.2 按孤独类型选择干预

| 类型 | 特征 | 首选干预 | 辅助干预 | 避免 |
|------|------|----------|----------|------|
| **情感孤独型** | EL高/SL低 | 依恋取向治疗；深化关系技能 | 慈心冥想(LKM)；情感表达训练 | 单纯扩大社交圈 |
| **社交孤独型** | EL低/SL高 | 社交技能训练；团体活动 | 暴露阶梯；志愿服务 | 过深入的情感挖掘 |
| **双重孤独型** | EL高/SL高 | 综合个别治疗优先 | 逐步引入团体；案例管理 | 过早团体暴露 |
| **情境性孤独** | 近期触发 | 问题解决； grief work | 短期支持团体 | 长期治疗承诺 |
| **慢性孤独** | >2年 | 长期个别治疗；人格聚焦 | 正念；自我慈悲 | 短期预期 |
| **存在性孤独** | 意义缺失 | 意义疗法；存在主义治疗 | 哲学咨询；灵性探索 | 单纯社交技能 |

### 5.3 按共病选择干预

| 共病 | 干预调整 | 优先顺序 |
|------|----------|----------|
| **重度抑郁** | 抑郁优先；药物+MBCT；孤独作为次级目标 | 抑郁 > 孤独 |
| **社交焦虑** | 个别开始→逐步团体；社交暴露需缓慢 | 焦虑 = 孤独（并行）|
| **PTSD/复杂创伤** | TF-CBT/EMDR优先；稳定化后再处理孤独 | 创伤 > 孤独 |
| **ASD** | 结构化社交技能；感官友好环境；明确规则 | ASD适应 > 标准孤独干预 |
| **物质使用** | 戒断优先；孤独作为复发风险因素 | 物质使用 > 孤独 |
| **人格障碍** | 长期治疗；心智化/图式治疗；慢节奏 | 人格障碍 > 孤独 |
| **精神病性障碍** | 症状稳定优先；孤独在康复期处理 | 精神病性症状 > 孤独 |

## 7. 随访与复发预防

### 7.1 标准化随访时间表

| 阶段 | 时间点 | 评估内容 | 决策点 |
|------|--------|----------|--------|
| **急性期** | 每1-2周 | UCLA-3；功能；安全 | 是否需要调整干预 |
| **中期** | 每4周 | UCLA-20；DJG-11；进展 | 是否进入维持期 |
| **巩固期** | 每2月 | UCLA-20；复发风险 | 是否结案 |
| **维持期** | 每3-6月 | UCLA-3；生活质量 | 是否需要加强 |
| **长期** | 每年 | 全面评估 | 持续监测 |

### 7.2 可靠变化指数(RCI)判定

```
UCLA-LS的RCI计算：
RCI = (X_post - X_pre) / S_diff

S_diff = √(2 × SE²) = √(2 × (SD × √(1-reliability))²)

对于UCLA-LS (SD≈11, α≈0.91):
S_diff ≈ 4.8
RCI阈值 ≈ ±1.96 × 4.8 ≈ ±9.4

判定：
• RCI < -9.4 → 可靠改善 ✓
• -9.4 ≤ RCI ≤ 9.4 → 无可靠变化 ~
• RCI > 9.4 → 可靠恶化 ✗（需关注）
```

### 7.3 复发预警信号

| 信号 | 级别 | 响应 |
|------|------|------|
| UCLA-3回升2分以上 | 黄灯 | 增加会谈频率 |
| 社交活动减少>50% | 黄灯 | 行为激活 |
| 睡眠恶化(PSQI>10) | 黄灯 | 睡眠干预 |
| 出现自杀意念 | 红灯 | 安全评估 |
| 功能显著下降 | 红灯 | 升级治疗强度 |
| 物质使用复发 | 红灯 | 专项处理 |

### 7.4 结案标准

```
可结案条件（需同时满足）：
□ UCLA-LS ≤ 40（或下降≥20%且RCI<-9.4）
□ DJG-11 ≤ 2
□ 功能恢复（工作/学习/社交）
□ 自杀风险解除（Columbia=0，持续4周+）
□ 来访者主观感到改善
□ 有维持计划和社会支持
□ 了解复发预警信号和求助途径

结案后安排：
• 3个月复查
• 6个月复查
• 1年复查
• 开放预约（需要时可返回）
```

## 8. 危机干预路径

### 8.1 自杀风险评估（Columbia Scale简版）

```
在过去一个月内：

1. 你是否希望死去？
2. 你是否有过实际结束生命的想法？
3. 你有没有想过具体的自杀方法？
4. 你有没有制定过自杀计划？
5. 你有没有任何自杀行为的意图？

评分：0=无 1=是

总分 ≥ 1：需要进一步评估
总分 ≥ 3：高风险，立即安全计划
```

### 8.3 现实感丧失/解离急性发作

| 症状 | 立即响应 | 后续 |
|------|----------|------|
| **现实感丧失(DPD)** | grounding技术（5-4-3-2-1感官练习） | 评估DES-II；创伤知情治疗 |
| **解离发作** | 安全环境；轻柔引导回到当下 | 评估 dissociative disorder |
| **恐慌发作** | 呼吸控制；纸袋呼吸； grounding | 评估 panic disorder |
| **精神病性症状** | 安全环境；避免对抗；紧急精神科 | 评估 psychotic disorder |

### 9.1 团队构成

| 角色 | 职责 | 介入时机 |
|------|------|----------|
| **全科医生/内科** | 躯体健康；药物管理；转介 | 初始评估；共病管理 |
| **精神科医生** | 诊断；药物治疗；复杂病例 | 中度以上共病；药物需求 |
| **临床心理师** | 心理评估；心理治疗；督导 | 核心干预者 |
| **社工** | 社会网络；资源链接；案例管理 | 网络缺乏；资源不足 |
| **职业治疗师** | 社交技能；日常生活；职业 | 功能损害；ASD |
| **精神科护士** | 随访；药物管理；健康教育 | 长期随访；社区管理 |
| **同伴支持者** |  lived experience；希望传递 | 全程；尤其是维持期 |

## 10. 质量控制与督导

### 10.1 治疗师能力清单

| 能力域 | 具体内容 | 评估方式 |
|--------|----------|----------|
| **评估能力** | 熟练运用UCLA/DJG/安全筛查 | 标准化病人考核 |
| **个案概念化** | 5P模型应用 | 书面概念化评审 |
| **干预 fidelity** | 按手册实施CBT/正念/慈心 | 录音编码 |
| **危机处理** | 自杀风险评估与响应 | 模拟场景 |
| **文化能力** | 跨文化敏感性与调整 | 督导讨论 |
| **自我关照** | 防止耗竭；个人练习 | 自我评估 |

### 10.2 督导要点

```
每次督导需讨论：
□ 个案进展与障碍
□ 反移情/个人反应
□ 伦理困境
□ 危机管理
□ 文化议题
□ 干预 fidelity
□ 自我关照状态

督导频率：
• 新手治疗师：每周1次
• 有经验者：每2周1次
• 危机个案：随时可联系
```
