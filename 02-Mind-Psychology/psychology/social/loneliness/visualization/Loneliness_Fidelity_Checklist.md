# 孤独干预 Fidelity 检查清单 | Loneliness Intervention Fidelity Checklist

> **文档类型**：质量控制工具 | Quality Assurance Tool  
> **适用范围**：治疗师督导、研究 fidelity 评估、临床培训、项目质量审计  
> **理论基础**：循证心理治疗的 fidelity 评估框架（Waltz et al., 1993; Perepletchikova et al., 2007）  
> **版本**：v1.0  
> **最后更新**：2026-05

---

## 使用说明

### 评分标准

| 评分 | 含义 | 说明 |
|------|------|------|
| **2分** | 完全达成 | 治疗师完整、熟练地实施了该项 |
| **1分** | 部分达成 | 治疗师实施了该项，但有遗漏或不够熟练 |
| **0分** | 未达成 | 治疗师未实施或明显错误实施了该项 |
| **N/A** | 不适用 | 该次会谈/个案不适用该项 |

###  fidelity 等级判定

| 等级 |  adherence 得分率 | 说明 |
|------|------------------|------|
| **高 fidelity** | ≥ 80% | 可视为标准实施 |
| **中 fidelity** | 60-79% | 需督导反馈改进 |
| **低 fidelity** | < 60% | 需重新培训或调整方案 |

## 模块A：评估阶段 Fidelity（A1-A20）

### A1-A5：第一级快速筛查

| 编号 | 检查项 | 2分标准 | 常见失误 | 评分 |
|------|--------|---------|----------|------|
| A1 | 在适当场景触发筛查 | 在规定的6类场景中主动筛查 | 忘记筛查；只在来访者主诉孤独时才筛查 | |
| A2 | 使用标准化筛查工具 | 使用UCLA-3/单条目/DJG简版之一 | 凭主观印象判断；不使用标准化工具 | |
| A3 | 正确解释筛查分数 | 根据阈值表准确告知风险等级 | 分数解释错误；不告知来访者结果 | |
| A4 | 对中高风险采取规定行动 | 中风险1-2周内安排评估；高风险72h内 | 拖延评估；高风险未立即安全筛查 | |
| A5 | 记录筛查结果 | 在病历中记录工具、分数、日期 | 未记录或记录不完整 | |

### A6-A12：第二级详细评估

| 编号 | 检查项 | 2分标准 | 常见失误 | 评分 |
|------|--------|---------|----------|------|
| A6 | 完成7模块评估结构 | 在45-60分钟内完成全部7模块 | 遗漏模块；超时严重或过于仓促 | |
| A7 | 正确使用UCLA-LS完整版 | 使用20项版本；正确反向计分 | 使用非标准版本；计分错误 | |
| A8 | 正确使用DJG-11 | 区分EL和SL；正确计分 | 与6项版混淆；未区分维度 | |
| A9 | 绘制社会网络图 | 实际绘制或引导来访者绘制 | 仅口头询问未可视化 | |
| A10 | 完成安全筛查组合 | PHQ-9 + GAD-7 + PCL-5简版均完成 | 遗漏任一量表；未评估自杀风险 | |
| A11 | 正确整合评估结果 | 给出明确的孤独类型/严重程度/共病判断 | 无整合结论；仅罗列分数 | |
| A12 | 形成书面评估报告 | 有结构化的评估记录/报告 | 仅有零散笔记；无正式记录 | |

### A13-A20：第三级深入诊断

| 编号 | 检查项 | 2分标准 | 常见失误 | 评分 |
|------|--------|---------|----------|------|
| A13 | 必要时进行人格评估 | 慢性/复杂个案使用MCMI/PID-5 | 对所有个案进行（过度）或对需要的个案不做 | |
| A14 | 必要时进行依恋评估 | 关系困难个案使用ECR-R | 忽视依恋维度；未评估关系模式 | |
| A15 | 使用5P个案概念化 | 呈现/诱发/维持/易感/保护五要素完整 | 概念化不完整；忽视保护因素 | |
| A16 | 评估功能损害程度 | 工作/学习/社交/自我照顾四领域均评估 | 仅评估主观感受；忽视功能 | |
| A17 | 评估生物学指标(如适用) | 研究/深度个案测量CRP/IL-6/HRV等 | 常规临床过度测量；研究个案遗漏 | |
| A18 | 区分孤独类型 | 明确判断情感型/社交型/双重/情境性/慢性 | 未分型；泛化处理 | |
| A19 | 识别并记录保护因素 | 至少识别2项保护因素 | 仅关注问题；忽视资源 | |
| A20 | 制定初步干预计划 | 有明确的目标、方法、频率、时长 | 无计划；或计划模糊不可执行 | |

## 模块B：干预实施 Fidelity（B1-B30）

### B1-B10：核心治疗关系与技术

| 编号 | 检查项 | 2分标准 | 常见失误 | 评分 |
|------|--------|---------|----------|------|
| B1 | 建立协作联盟 | 与来访者共同设定目标；确认来访者理解并同意 | 治疗师单方面决定目标 | |
| B2 | 传递希望与正常化 | 明确告知孤独是可改善的；正常化孤独体验 | 强化来访者的绝望感 | |
| B3 | 使用孤独特异性语言 | 使用"情感孤独"/"社交孤独"等专业概念教育来访者 | 仅使用模糊的"孤独"一词 | |
| B4 | 保持创伤知情立场 | 不强迫暴露；尊重来访者节奏；有grounding准备 | 在不稳定时进行深度探索 | |
| B5 | 文化敏感性实践 | 考虑文化背景对孤独表达的影响；调整语言 | 文化盲点；强加个人主义框架 | |
| B6 | 维持适当的情感距离 | 共情但不过度卷入；保持专业边界 | 过度认同；或情感疏离 | |
| B7 | 及时识别并处理反移情 | 觉察自己对孤独议题的个人反应 | 未觉察自己的孤独经历影响治疗 | |
| B8 | 使用苏格拉底式提问 | 引导来访者自我发现，而非直接给建议 | 说教；过度指导 | |
| B9 | 进行过程评论(如适用) | 在治疗关系中实时评论当下互动 | 只谈内容不谈过程 | |
| B10 | 结束会谈时总结与预告 | 每次会谈结束前5分钟总结要点；预告下次 | 突然结束；无结构收尾 | |

### B11-B20：CBT技术 Fidelity（如使用CBT）

| 编号 | 检查项 | 2分标准 | 常见失误 | 评分 |
|------|--------|---------|----------|------|
| B11 | 识别孤独相关的自动化思维 | 帮助来访者捕捉孤独时的具体想法 | 停留在抽象描述 | |
| B12 | 进行认知重构 | 使用证据检验/替代思维/去灾难化 | 仅否定想法而不提供替代 | |
| B13 | 进行归因重训练 | 帮助来访者从内部/稳定/全局归因转向更灵活的归因 | 忽视归因维度 | |
| B14 | 布置行为实验 | 设计具体的社交行为实验；预测-执行-回顾 | 仅有口头鼓励无具体实验 | |
| B15 | 使用暴露阶梯(社交焦虑共病) | 从低到高系统暴露；有明确的恐惧层级表 | 跳过步骤；暴露过于强烈 | |
| B16 | 进行社交技能训练 | 教授具体技能（目光接触、提问、倾听） | 假设来访者知道如何社交 | |
| B17 | 使用行为激活 | 安排愉悦活动和掌控活动；监测情绪变化 | 仅有建议无结构化安排 | |
| B18 | 进行问题解决训练 | 使用标准化问题解决步骤（D'Zurilla） | 替来访者解决问题 | |
| B19 | 布置结构化家庭作业 | 每次有明确的作业；下次检查完成情况 | 无作业；或作业模糊 | |
| B20 | 使用认知去中心化 | 帮助来访者看到"想法只是想法" | 来访者仍与想法融合 | |

### B21-B25：正念技术 Fidelity（如使用MBSR/LKM/MSC）

| 编号 | 检查项 | 2分标准 | 常见失误 | 评分 |
|------|--------|---------|----------|------|
| B21 | 遵循标准化课程结构 | MBSR 8周/LKM 6周/MSC 8周的标准结构 | 随意压缩/省略核心内容 | |
| B22 | 提供标准化指导语 | 使用经过验证的指导语录音或文字 | 即兴指导；偏离标准 | |
| B23 | 每日练习要求与追踪 | 明确每日练习时长；追踪完成率 | 不追踪练习；不强调每日 | |
| B24 | 处理练习中的困难 | 正常化困难；提供调整建议（坐姿/时长/眼睛） | 忽视困难；批评来访者不努力 | |
| B25 | 区分体验与诠释 | 引导观察体验而不立即诠释 | 过度诠释来访者的体验 | |

## 模块C：婚内/关系孤独干预 Fidelity（C1-C15）

| 编号 | 检查项 | 2分标准 | 常见失误 | 评分 |
|------|--------|---------|----------|------|
| C1 | 评估双方孤独体验 | 分别评估夫妻双方的孤独感（不只问一方） | 仅以来访者视角假设另一方感受 | |
| C2 | 识别追逃循环 | 明确识别焦虑-回避互动模式 | 仅归因于一方"有问题" | |
| C3 | 进行情感反映与验证 | 准确反映双方情感；验证双方体验 | 偏袒一方；或情感反映不准确 | |
| C4 | 引导脆弱性表达 | 帮助双方安全表达需求（非攻击/非逃避） | 让双方进入攻击-防御模式 | |
| C5 | 进行依恋对话 | 引导双方分享依恋恐惧和需求 | 回避深层情感话题 | |
| C6 | 教授修复尝试 | 教授具体的冲突后修复语言和行为 | 假设夫妻知道如何修复 | |
| C7 | 安排结构化夫妻时间 | 具体的"非功能对话"时间安排 | 仅建议"多沟通"无结构 | |
| C8 | 评估性亲密关系 | 温和评估性孤独；不回避性话题 | 完全回避性话题；或过于侵入 | |
| C9 | 处理外遇/背叛(如存在) | 评估影响；不急于原谅；分阶段处理 | 强迫快速修复；忽视创伤 | |
| C10 | 评估子女因素 | 评估共同育儿压力对夫妻关系的影响 | 忽视子女因素；或过度聚焦子女 | |
| C11 | 处理原生家庭议题 | 识别双方原生家庭模式对当前关系的影响 | 仅关注当前互动；忽视历史 | |
| C12 | 进行关系叙事重构 | 帮助夫妻重构关系故事（从失败到成长） | 强化"关系失败"叙事 | |
| C13 | 评估离婚/分居选项 | 中立评估修复可能性；不强迫维持 | 强迫维持有害关系；或轻易建议离婚 | |
| C14 | 制定关系维护计划 | 结案时有具体的维持和复发预防计划 | 无维持计划；突然结案 | |
| C15 | 识别灵性虐待(如涉及修行) | 评估社群是否控制信息/孤立/经济剥削 | 忽视灵性社群的潜在危害 | |
