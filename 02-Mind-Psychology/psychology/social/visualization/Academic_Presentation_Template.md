## Slide 9: 12周联合干预方案

```
四阶段整合模型

Week 1-2   建立期    评估 + 关系建立 + 个案概念化
Week 3-6   连接期    社交行为激活 + 认知重构 + 微独处嵌入
Week 7-9   深化期    中独处训练 + 自我关怀 + 个人节奏设计
Week 10-12 整合期    动态平衡 + 复发预防 + 维持计划

核心创新: 不将孤独与独处对立，而是协同干预
目标人群: 慢性孤独伴独处困难成人(18-65岁)
```

**设计建议**: 横向时间轴，四个节点用不同颜色，下方简短说明

## Slide 12: 案例摘要

```
典型案例: 婚内孤独

张女士, 42岁, 结婚15年
表面模范夫妻 → 每天对话<10句 → 3年无性

评估: 中度孤独 + 严重婚姻失调 + 低独处能力

干预:
  第1-4周   个体工作: 自我分化 + 独处能力
  第5周     邀请伴侣 → 转介EFT夫妻治疗
  第9-12周  联合干预: 每日15分钟对话 + 独处协议

结果: 独处能力 2→4, 婚姻质量显著提升, 未离婚
```

## Slide 14: 结论

```
核心信息

1️⃣ 孤独是信号, 不是缺陷
   → 提醒我们: 需要连接

2️⃣ 独处是能力, 不是逃避
   → 让我们: 在任何情况下保持内心平静

3️⃣ 两者可以协同干预
   → 从病理缓解走向成长促进

"孤独是人类的共通体验, 独处是人类的共通能力"
```

**设计建议**: 三个要点用大号数字标出，底部一句引言作为情感收尾
