# 教学设计心理学 | Teaching Design Psychology

> **学科定位**: 教学设计 × 认知科学 × 学习心理学
> **核心主张**: 有效教学建立在对认知过程深入理解的基础上

---

## 一、认知学徒制（Cognitive Apprenticeship）

### 1.1 核心方法

| 方法 | 说明 | 示例 |
|------|------|------|
| 示范（Modeling） | 专家展示思维过程 | 教师"出声思维"解题 |
| 指导（Coaching） | 学生练习时提供引导 | 观察+即时反馈 |
| 脚手架（Scaffolding） | 提供渐进式支持 | 提示卡→线索→独立 |
| 清晰化（Articulation） | 让学生表达思维过程 | 解释自己的解题步骤 |
| 反思（Reflection） | 比较自己与专家的差异 | 自评+同伴评价 |
| 探索（Exploration） | 鼓励独立发现 | 开放性问题/项目 |

### 1.2 学习迁移设计

| 迁移类型 | 说明 | 促进策略 |
|---------|------|---------|
| 近迁移 | 相似情境间 | 变式练习 |
| 远迁移 | 不同领域间 | 抽象原则教学 |
| 正迁移 | 旧知促进新学 | 类比、先行组织者 |
| 负迁移 | 旧知干扰新学 | 明确对比差异 |

## 二、多媒体学习认知理论（Mayer）

### 2.1 核心原则

| 原则 | 内容 | 效应量 |
|------|------|--------|
| 多媒体 | 图文优于纯文字 | d = 1.39 |
| 空间邻近 | 相关文图靠近呈现 | d = 1.10 |
| 时间邻近 | 图文同时而非先后 | d = 1.30 |
| 一致性 | 去除无关信息 | d = 0.86 |
| 信号 | 关键信息加标记 | d = 0.46 |
| 冗余 | 图+旁白优于图+旁白+字幕 | d = 0.72 |
| 分段 | 分段呈现优于连续 | d = 0.70 |
| 预训练 | 先介绍关键概念 | d = 0.46 |
| 个人化 | 对话体优于正式体 | d = 0.79 |

## 三、问题式学习（PBL）

### 3.1 标准流程

| 步骤 | 活动 | 心理过程 |
|------|------|---------|
| 1. 呈现问题 | 真实情境、非结构化 | 激活先前知识 |
| 2. 分析问题 | 小组讨论、界定问题 | 元认知规划 |
| 3. 形成假设 | 提出可能的解释 | 推理、假设生成 |
| 4. 识别知识缺口 | 确定需学习的内容 | 自我监控 |
| 5. 自主学习 | 查找资源、个人研究 | 自我调节学习 |
| 6. 综合应用 | 回到问题、整合知识 | 迁移、综合 |
| 7. 反思评估 | 评估过程与成果 | 元认知反思 |

### 3.2 PBL循证证据

| 研究 | 发现 |
|------|------|
| Hmelo-Silver (2004) | PBL发展灵活知识、问题解决和自主学习能力 |
| Strobel & van Barneveld (2009) | 长期知识保持和技能应用优于传统教学 |
| 局限性 | 对新手学生可能认知负荷过高（Kirschner et al., 2006） |

## 四、翻转课堂心理学

| 阶段 | 课前（在家） | 课上（教室） |
|------|------------|-------------|
| 认知层次 | 记忆、理解（低阶） | 应用、分析、创造（高阶） |
| 活动 | 观看视频、阅读材料 | 讨论、练习、项目 |
| 教师角色 | 内容制作者 | 引导者、教练 |
| 心理原理 | 自定步调、可重复 | 社会建构、即时反馈 |

## 五、教育技术心理学

| 技术 | 心理基础 | 应用 |
|------|---------|------|
| 间隔重复系统 | 间隔效应+测试效应 | Anki、Quizlet |
| 自适应学习 | 个性化ZPD | 智能教学系统 |
| 游戏化学习 | 内在动机+即时反馈 | 积分、徽章、排行榜 |
| 虚拟现实 | 具身认知+情境学习 | 沉浸式模拟 |
| AI辅助教学 | 个性化+即时反馈 | ChatGPT辅助、智能批改 |
