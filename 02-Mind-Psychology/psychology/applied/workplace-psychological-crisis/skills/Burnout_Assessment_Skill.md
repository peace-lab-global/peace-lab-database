# 职业倦怠评估技能

## 技能信息

**技能ID**: S_014
**技能名称**: 职业倦怠评估 (Occupational Burnout Assessment)
**功能**: 系统评估用户职业倦怠状态，识别倦怠维度与阶段，判断严重程度，提供分级干预建议
**适用人群**: 主诉工作疲劳、动力丧失、职业意义感缺失的用户
**理论基础**: Maslach倦怠量表 (MBI)；Freudenberger倦怠阶段模型；工作需求-资源模型 (JD-R Model)
**输出**: 结构化评估报告 + 倦怠维度分析 + 严重程度分级 + 分层干预建议

---

## 评估协议

### 第一步：MBI 倦怠量表评估 (Maslach Burnout Inventory)

通过结构化提问评估三个核心维度。每个项目采用 0-6 分制（0=从不，6=每天）。

#### 维度一：情绪耗竭 (Emotional Exhaustion) — 9项

评估工作中的情感资源耗竭程度。

| 编号 | 评估项目 | 评分(0-6) |
|:-----|:---------|:---------:|
| EE-1 | 下班后感觉精疲力竭 | __ |
| EE-2 | 早上想到又要工作一天就感到疲惫 | __ |
| EE-3 | 和来访者/同事相处让我感到有压力 | __ |
| EE-4 | 感觉工作让我筋疲力尽 | __ |
| EE-5 | 感觉工作快要让我崩溃了 | __ |
| EE-6 | 对工作感到厌倦 | __ |
| EE-7 | 对他人（同事/客户）感到不耐烦 | __ |
| EE-8 | 工作中的情感压力让我难以承受 | __ |
| EE-9 | 感觉自己在工作中已经山穷水尽 | __ |

**计分规则**:
- 低倦怠: 0-16分
- 中度倦怠: 17-26分
- 高度倦怠: 27-54分

#### 维度二：去人格化 (Depersonalization) — 5项

评估对工作对象的冷漠和疏离程度。

| 编号 | 评估项目 | 评分(0-6) |
|:-----|:---------|:---------:|
| DP-1 | 对服务对象/同事变得越来越冷漠 | __ |
| DP-2 | 担心工作会让自己变得麻木 | __ |
| DP-3 | 不关心服务对象/同事发生了什么 | __ |
| DP-4 | 把服务对象/同事当成物品来对待 | __ |
| DP-5 | 和服务对象/同事接触越多，越不想理会他们 | __ |

**计分规则**:
- 低倦怠: 0-6分
- 中度倦怠: 7-12分
- 高度倦怠: 13-30分

#### 维度三：个人成就感降低 (Reduced Personal Accomplishment) — 8项

评估对工作成就和价值的否定程度。注意：此维度反向计分。

| 编号 | 评估项目 | 评分(0-6) |
|:-----|:---------|:---------:|
| PA-1 | 能有效解决工作中的问题（反向） | __ |
| PA-2 | 感到自己在做有价值的工作（反向） | __ |
| PA-3 | 对他人有积极影响（反向） | __ |
| PA-4 | 感到精力充沛（反向） | __ |
| PA-5 | 能轻松理解服务对象/同事的感受（反向） | __ |
| PA-6 | 和同事相处时能营造轻松氛围（反向） | __ |
| PA-7 | 能够创造性地完成工作（反向） | __ |
| PA-8 | 对同事有积极影响（反向） | __ |

**计分规则**（反向计分：0→6, 1→5, 2→4, 3→3, 4→2, 5→1, 6→0）:
- 高成就感: 39-48分（反向计分后，表明低倦怠）
- 中度: 32-38分
- 低成就感: 0-31分（表明高倦怠风险）

#### MBI 综合判定

| 严重程度 | 情绪耗竭 | 去人格化 | 个人成就感 |
|:---------|:--------:|:--------:|:----------:|
| 高度倦怠风险 | ≥27 | ≥13 | ≤31 |
| 中度倦怠风险 | 17-26 | 7-12 | 32-38 |
| 低度倦怠风险 | ≤16 | ≤6 | ≥39 |

> 注：MBI 为标准化自评量表，完整版权由 Mind Garden Inc. 持有。此处为评估导向的结构化提问框架，非量表原文复制。

---

### 第二步：倦怠阶段评估 (Freudenberger 12阶段模型)

根据用户描述，定位其处于倦怠发展的哪个阶段。

| 阶段 | 特征 | 关键表现 |
|:-----|:-----|:---------|
| 1 | 强迫性证明自己 | 过度努力证明自身价值，承担过多任务 |
| 2 | 加倍努力 | 不惜代价追求成就，工作量显著增加 |
| 3 | 忽视个人需求 | 压抑正常需求，忽略社交、休息和娱乐 |
| 4 | 冲突转移 | 开始回避根本问题，用其他方式排解压力 |
| 5 | 价值观修订 | 淡化朋友和爱好，社交活动减少 |
| 6 | 否认问题 | 对问题持否认态度，对他人缺乏耐心，出现社交隔离 |
| 7 | 社交退缩 | 社交生活极度缩减，几乎不与人交往 |
| 8 | 行为改变 | 出现明显的外在行为变化，旁人开始察觉 |
| 9 | 去人格化 | 失去对自我的感知，不再关心自己或他人 |
| 10 | 内心空虚 | 感到空虚，可能用成瘾行为（食物、酒精、购物）来填补 |
| 11 | 抑郁 | 出现绝望感、冷漠和完全疲惫 |
| 12 | 倦怠崩溃 | 身心崩溃，可能需要医疗干预 |

**评估提问**: "您觉得自己的状态大概像是上述哪个阶段？哪些描述和您的感受最接近？"

---

### 第三步：工作需求-资源模型评估 (JD-R Model)

评估工作需求与工作资源之间的失衡状况。

#### 工作需求 (Job Demands) — 评估负荷

| 需求类型 | 评估问题 | 水平(1-5) |
|:---------|:---------|:---------:|
| 工作量 | "您的工作任务量是否超出个人承受范围？" | __ |
| 时间压力 | "您是否经常面临紧迫的截止日期？" | __ |
| 情绪要求 | "工作是否要求您持续付出大量情感劳动？" | __ |
| 角色冲突 | "您是否经常面临相互矛盾的工作要求？" | __ |
| 工作不安全感 | "您是否担心失业或岗位变动？" | __ |

#### 工作资源 (Job Resources) — 评估支撑

| 资源类型 | 评估问题 | 水平(1-5) |
|:---------|:---------|:---------:|
| 自主性 | "您在工作中有多大的自主决策空间？" | __ |
| 上级支持 | "您的上级是否给予足够的支持和认可？" | __ |
| 同事关系 | "您与同事之间有良好的合作关系吗？" | __ |
| 发展机会 | "您是否有学习和职业成长的机会？" | __ |
| 反馈 | "您是否能收到关于工作表现的积极反馈？" | __ |

**JD-R 失衡判定**:
- 需求总分 ≥ 20 且 资源总分 ≤ 10: 高危失衡
- 需求总分 ≥ 15 且 资源总分 ≤ 15: 中度失衡
- 需求-资源差值 ≤ 5: 基本平衡

---

### 第四步：职业价值观匹配度评估

评估个人核心职业价值观与当前工作现实之间的匹配程度。

| 价值观维度 | 评估问题 | 匹配度(1-5) |
|:-----------|:---------|:-----------:|
| 意义感 | "您的工作让您感到有意义吗？" | __ |
| 成就感 | "您能在工作中获得成就感吗？" | __ |
| 自主性 | "您在工作中感到足够的自主和自由吗？" | __ |
| 公平感 | "您觉得组织对您的付出给予了公平回报吗？" | __ |
| 归属感 | "您在团队中感到被接纳和重视吗？" | __ |
| 成长性 | "当前工作能帮助您成长为想成为的人吗？" | __ |

**价值观匹配度判定**:
- 平均分 ≥ 4: 高匹配（低倦怠风险）
- 平均分 2.5-3.9: 中度匹配（需关注）
- 平均分 < 2.5: 严重不匹配（高倦怠风险）

## 干预建议

### Level 1: 轻度倦怠 — 工作边界设定 + 恢复活动

适用条件: MBI单项维度中度偏高；Freudenberger 1-3阶段；JD-R轻度失衡

**短期行动**:
- 设立明确的工作时间边界，避免非工作时间处理工作事务
- 每天安排至少1项恢复性活动（运动、社交、兴趣爱好）
- 练习"心理脱离"(Psychological Detachment)：下班后有意识地将注意力从工作中转移

**中期规划**:
- 评估并减少不必要的工作承诺
- 建立固定的午休和微休息习惯
- 每周至少1次与工作无关的社交活动

---

### Level 2: 中度倦怠 — 认知重构 + 组织层面干预

适用条件: MBI双维度高度偏高；Freudenberger 4-8阶段；JD-R中度失衡；价值观匹配度中等

**认知层面**:
- 识别并挑战与工作相关的不合理信念（如"我必须做到完美"、"休息就是偷懒"）
- 练习认知重评 (Cognitive Reappraisal)，重新定义工作中的"成功"标准
- 学习区分"可控"与"不可控"的工作因素

**行为层面**:
- 与上级进行坦诚的工作负荷沟通
- 明确角色边界，学会委派和拒绝
- 制定可执行的工作优先级清单

**组织建议**:
- 推动建立弹性工作制度
- 倡导定期的工作反馈与认可机制
- 建议组织提供员工心理支持资源

---

### Level 3: 重度倦怠 — 工作调整 + 专业支持

适用条件: MBI三维度全部高度偏高；Freudenberger 9-12阶段；JD-R严重失衡；价值观严重不匹配

**紧急措施**:
- 评估是否需要暂时从工作中休息（如休假、病假）
- 识别并停止所有加重倦怠的行为模式
- 建立稳定的基础生活保障（睡眠、饮食、运动）

**专业支持**:
- 建议寻求职业心理咨询或心理治疗
- 考虑进行正式的心理健康评估（排除抑郁障碍等共病）
- 与职业规划师讨论职业转型的可能性

**长期规划**:
- 评估当前职业路径是否需要调整
- 探索与个人价值观更匹配的工作方向
- 建立持续的职业自我关怀 (Professional Self-Care) 计划

## 转介红线

以下情况需要立即转介至专业危机干预或精神科评估:

| 红线信号 | 行动 | 转介目标 |
|:---------|:-----|:---------|
| 用户表达自杀念头或计划 | 立即评估风险等级，执行危机干预 | 危机干预技能 / 急诊 |
| 出现严重抑郁症状（持续2周以上的情绪低落、兴趣丧失） | 跳转抑郁评估技能 | Depression Assessment Skill |
| 物质滥用（酒精、药物依赖） | 建议专业评估与戒断支持 | 精神科评估 |
| 躯体症状严重（心悸、胸痛、呼吸困难） | 建议就医排除躯体疾病 | 医疗转介 |
| 现实感丧失或人格解体 | 建议紧急精神科评估 | 精神科评估 |

## 使用示例

**场景**: 用户说"每天上班都像上坟，感觉自己已经枯竭了，做什么都提不起劲"

**评估流程**:
1. 确认触发: 用户表达"上班像上坟""枯竭""提不起劲" → 命中 entry_trigger
2. Step 1: 通过MBI结构化提问评估三个维度得分
3. Step 2: 通过12阶段模型定位倦怠发展阶段
4. Step 3: 通过JD-R模型评估工作需求-资源平衡
5. Step 4: 评估职业价值观匹配度
6. 综合判定倦怠严重程度 → 给出对应级别干预建议
7. 若检测到转介红线 → 立即执行转介协议
