## 暴露原理

### 抑制性学习

暴露的本质是建立新的学习：
- 大脑原有学习：情境=危险→回避
- 新学习：情境=安全→可以接近

### 暴露起效的条件

**情绪处理理论**：
- 必须在暴露中等待情绪自然下降
- 情绪必须真正得到处理，而非压抑
- 回避和安全行为会阻止学习

### 关键暴露成分

| 成分 | 说明 |
|------|------|
| 投入程度 | 全心进入暴露情境 |
| 暴露长度 | 足够让焦虑下降到50%以下 |
| 频率 | 足够频繁以巩固学习 |
| 不使用安全行为 | 让学习真正发生 |

## 暴露类型

### 想象暴露

**适合情境**：难以在现实中创造的情境、未来事件、过去创伤

**执行要点**：
- 闭上眼睛，尽可能生动地想象
- 包括所有感官（视觉、听觉、嗅觉、身体感觉）
- 持续足够长时间（直到焦虑开始下降）

**想象暴露示例**：
想象在重要面试中：
1. 等待面试房间外
2. 走进房间
3. 与面试官目光接触
4. 回答问题时紧张
5. 忘记要说什么
6. 面试官表现出不耐烦

### 现实暴露

**适合情境**：可以通过真实体验学习的情境

**执行要点**：
- 从低难度开始
- 逐渐增加难度
- 记录SUDs变化

### 体内暴露（刺激控制）

**适合情境**：对身体感觉恐惧（如心悸恐惧）

**方法**：
- 通过运动、咖啡等安全地引发身体感觉
- 练习在这种感觉下保持冷静
- 学习到这些感觉是无害的

## 暴露计划制定

### 创建暴露阶梯

**步骤**：
1. 列出所有让你焦虑的情境
2. 对每个情境评估SUDs（0-10）
3. 从低SUDs情境开始
4. 确保每个步骤之间难度差距合理

### 暴露计划模板

**社交焦虑暴露阶梯**：

| 周次 | 情境 | 目标SUDs | 实际SUDs变化 | 备注 |
|------|------|----------|-------------|------|
| 1 | 对镜子练习说话 | 3 | 5→2 | |
| 1 | 对手机录音说话 | 4 | 6→3 | |
| 2 | 与朋友视频通话 | 5 | 7→4 | |
| 2 | 在Zoom会议发言 | 6 | 8→5 | |
| 3 | 在小组会发言 | 7 | 8→4 | |
| 3 | 在部门会发言 | 8 | 9→5 | |

### 频率建议

- 理想：每天或隔天暴露
- 最低：每周2-3次
- 持续：直到情境不再引发显著焦虑

## 暴露执行要点

### 执行检查清单

**暴露前**：
- [ ] 确认情境/任务
- [ ] 确认不使用安全行为
- [ ] 设定手机计时器（暴露期间）
- [ ] 准备好记录表

**暴露中**：
- [ ] 完全投入情境
- [ ] 监控SUDs（每5分钟）
- [ ] 留在情境中直到下降
- [ ] 不使用安全行为

**暴露后**：
- [ ] 记录SUDs变化过程
- [ ] 记录实际结果
- [ ] 反思学习
