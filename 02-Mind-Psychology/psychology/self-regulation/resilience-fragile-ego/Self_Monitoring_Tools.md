## 触发事件日志

### 触发事件日志模板

| 日期/时间 | 情境描述 | 触发因素 | 情绪反应（强度1-10） | 使用技能 | 效果评估 |
|------------|----------|----------|---------------------|----------|----------|
| | | | | | |

**填写指南**：

- **情境描述**：客观描述发生了什么，不要加入评价
- **触发因素**：识别什么具体内容触发了反应（批评/拒绝/忽视/比较）
- **情绪反应**：标注具体情绪和强度
- **使用技能**：本次使用了哪种韧性技术
- **效果评估**：使用技能后情绪变化（1-10）

### 触发因素分类编码

| 编码 | 类型 | 示例 |
|------|------|------|
| A | 批评 | 被指出错误/被负面评价 |
| B | 拒绝 | 被排斥/被拒绝/被忽视 |
| C | 比较 | 看到他人成功/发现他人更好 |
| D | 预期焦虑 | 担心可能的负面结果 |
| E | 完美主义 | 害怕不够好/不敢开始 |
| F | 其他 | 注明具体 |
