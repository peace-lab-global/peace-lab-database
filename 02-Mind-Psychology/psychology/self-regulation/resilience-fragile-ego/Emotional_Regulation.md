## 情绪调节基础

### 情绪的本质

情绪是个体对内外刺激的主观体验和生理反应，具有以下特点：
- 进化适应性（恐惧帮助逃离危险）
- 短暂性（会自然消退）
- 多维度（认知评估+身体反应+行为冲动）

### 情绪调节的定义

情绪调节（Emotion Regulation）指个体影响自身情绪的类型、发生时间、体验方式和表达方式的能力。良好的情绪调节不是压抑情绪，而是：
- 在适当的时候表达适当强度的情绪
- 能够缩短情绪恢复时间
- 减少情绪对功能的负面影响

### 情绪调节策略分类

| 策略类型 | 策略名称 | 效果 |
|----------|----------|------|
| 先行聚焦 | 情境选择 | 避免触发情绪的情境 |
| 先行聚焦 | 情境修正 | 改变情境以降低影响 |
| 先行聚焦 | 注意分配 | 将注意力转向或远离刺激 |
| 先行聚焦 | 认知改变 | 改变对情境的解读 |
| 反应聚焦 | 反应调节 | 调节情绪表达和行为反应 |

玻璃心干预主要聚焦于**认知改变**和**反应调节**策略。

## 即时调节技术

### STOP技术

当感到被批评触发强烈情绪时使用：

- **S**top（暂停）：停止当前行为，不立即反应
- **T**ake breath（呼吸）：进行3次深呼吸
- **O**bserve（观察）：命名当前情绪和身体感觉
- **P**roceed（继续）：选择有意识的行为回应

### 身体锚定技术

**冷刺激法**：
- 用冷水洗脸或含冰块
- 激活迷走神经，降低情绪唤醒
- 适合急性焦虑或愤怒

**接地技术（5-4-3-2-1）**：
- 说出5样看到的东西
- 4样触摸到的东西
- 3样听到的声音
- 2种闻到的气味
- 1种尝到的味道

### 呼吸调节

**4-7-8呼吸法**：
1. 用鼻子吸气4秒
2. 屏住呼吸7秒
3. 用嘴呼气8秒
4. 重复3-4次

**交替鼻孔呼吸法**：
1. 用右手拇指按住右鼻孔
2. 左鼻孔吸气4秒
3. 拇指和无名指同时按住
4. 右鼻孔呼气4秒
5. 右鼻孔吸气，左鼻孔呼气
6. 交替进行5分钟

## 情绪觉察训练

### 身体扫描

**每日身体扫描练习（10分钟）**：

1. 找一个舒适的姿势闭上眼睛
2. 从脚趾开始，慢慢向上移动注意力
3. 关注每个部位的感觉（温度、压力、刺痛等）
4. 注意到情绪可能对应的身体感觉
5. 不要试图改变，只是观察

### 情绪命名练习

**情绪颗粒度训练**：

研究表明，能够精确命名情绪（情绪颗粒度 high emotional granularity）的人情绪调节能力更强。

| 基本分类 | 精细化命名 |
|----------|------------|
| 不舒服 | 焦虑不安、忐忑、紧张、担忧、恐慌 |
| 难过 | 失落、沮丧、悲伤、哀伤、忧郁、绝望 |
| 生气 | 恼怒、愤怒、怨恨、受挫、不满 |

**练习方法**：每天记录情绪时，尝试使用至少一个精细化情绪词汇。

### 情绪日记模板

| 项目 | 填写内容 |
|------|----------|
| 时间 | 2024-XX-XX XX:XX |
| 情境 | 正在发生什么？ |
| 触发 | 什么触发了这个情绪？ |
| 情绪名称 | 使用精细化词汇 |
| 情绪强度 | 1-10 |
| 身体感觉 | 哪里有什么感觉？ |
| 自动化反应 | 想做什么/说什么？ |
| 有意识的回应 | 实际做了什么？ |
| 效果 | 这样回应有帮助吗？ |

## 长期调节策略

### 正念练习

**正念观情绪**：

1. 以不评判的态度注意情绪
2. 不要试图推开或抓住情绪
3. 只是观察它如云朵般来去
4. 记住：情绪会来，也会走

**每日正念练习（10-20分钟）**：
- 固定时间练习（早晨或睡前）
- 从5分钟开始，逐渐增加
- 使用指导音频或App辅助
- 关键是每日坚持

### 自我关怀实践

**自我关怀三要素**：

**1. 自我友善（vs 自我批评）**：
- 当犯错或失败时，像对待好朋友一样对待自己
- 问自己："我会对朋友说什么？"
- 给自己说一些温暖鼓励的话

**2. 共同人性（vs 孤立感）**：
- 记住：每个人都有不完美的时刻
- "我不是唯一一个感到..."的感受
- 痛苦是人类共同经验的一部分

**3. 正念觉察（vs 过度认同）**：
-觉察当下的痛苦，但不过度卷入
- 既不压抑也不放大情绪
- 平衡地观察情绪的来去

### 睡眠、运动与营养

**睡眠**：
- 7-9小时高质量睡眠
- 睡眠不足会加剧情绪敏感性
- 建立规律睡眠时间

**运动**：
- 每周150分钟中等强度运动
- 运动释放的内啡肽有自然抗焦虑作用
- 快走、跑步、游泳、瑜伽都是好选择

**营养**：
- 避免过多咖啡因（加剧焦虑）
- 保持血糖稳定
- Omega-3脂肪酸有益情绪健康
