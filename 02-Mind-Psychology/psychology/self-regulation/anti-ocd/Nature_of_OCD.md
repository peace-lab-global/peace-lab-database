## 强迫症的定义

### OCD概述

强迫症（Obsessive-Compulsive Disorder, OCD）是一种以强迫思维和/或强迫行为为特征的心理障碍：
- 强迫思维：反复出现、不受欢迎、引发焦虑的闯入性思维
- 强迫行为：重复性行为或 mental acts，旨在减少焦虑

### 诊断标准（简化）

根据DSM-5：
1. 强迫思维、强迫行为或两者都有
2. 消耗大量时间（每天>1小时）或造成显著痛苦/功能损害
3. 症状不是物质或医疗状况的直接效应
4. 症状不能用其他心理障碍更好解释

### 强迫症与强迫型人格的区别

| 特征 | 强迫症(OCD) | 强迫型人格(OCPD) |
|------|-------------|------------------|
| 自知力 | 通常知道自己不合理 | 通常认为自己的方式是好的 |
| 痛苦 | 高焦虑痛苦 | 较低焦虑，更关注效率 |
| 功能 | 常常损害功能 | 有时反而帮助功能 |
| 核心 | 侵入性思维+仪式 | 完美主义+控制 |

## 强迫症的心理学模型

### 认知模型

**认知因素在OCD中的作用**：

| 认知因素 | 说明 |
|----------|------|
| 责任过度 | 高估自己对伤害事件的责任 |
| 思维-行动融合 | 认为想到坏事就等同于会做坏事 |
| 过度重要性 | 高估担忧思维的重要性 |
| 控制思维的需要 | 认为必须控制思维才能安全 |
| 不确定性不耐受 | 无法容忍模糊和不确定性 |

### 三因素模型

```
前因因素 ← → 强迫思维 → 焦虑 → 强迫行为 → 暂时缓解 → 强化
   ↓                                        ↓
  诱发因素 ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
```

**前因因素**：
- 遗传易感性
- 童年经历
- 脑神经机制差异

**诱发因素**：
- 压力事件
- 特定情境或物体
- 想法（任何想法都可能触发）

## 强迫思维与强迫行为

### 强迫思维（Obsessions）

**定义**：反复、持续、不受欢迎的闯入性思维、意象或冲动，造成显著焦虑。

**常见类型**：

| 类型 | 示例 |
|------|------|
| 污染型 | "我的手被脏了，会生病" |
| 伤害型 | "我会伤害自己或他人" |
| 对称型 | "东西必须排列整齐，否则会不舒服" |
| 迷信型 | "如果不说这个词，会有坏事发生" |
| 性相关 | 不想要的性相关侵入性思维 |
| 宗教/道德 | 不想要的宗教或道德相关思维 |

**关键特征**：
- 闯入性（不是自愿选择的）
- 不受欢迎（引起痛苦）
- 去中心化（不代表本人的意愿）

### 强迫行为（Compulsions）

**定义**：重复性行为或精神活动，个体感到必须执行或规则化以减少焦虑。

**常见类型**：

| 类型 | 示例 |
|------|------|
| 清洗/清洁 | 反复洗手、清洁物品 |
| 检查 | 反复检查门、炉子、锁 |
| 排序/整理 | 反复调整物品位置 |
| 计数 | 对特定数字有仪式化行为 |
| Mental acts | 重复默念词语、祈祷、计数 |
| 回避 | 回避引发强迫思维的情境 |

## 维持因素分析

### 强迫行为的短期效益与长期代价

**短期效益**：
- 焦虑立即下降
- 感觉"更安全"
- 不确定感减少

**长期代价**：
- 强迫行为被强化，变得更频繁
- 对不确定性的耐受越来越低
- 功能严重受损
- 自尊心下降
- 生活质量下降

### 核心维持循环

```
强迫思维 → 焦虑 → 强迫行为 → 焦虑暂时缓解 → 强迫行为强化
                ↑                              ↓
← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
```

### 反强迫的核心原理

打破循环的关键：
1. 识别强迫思维（不是威胁，是OCD的症状）
2. 减少或消除强迫行为（不让安全行为强化循环）
3. 通过暴露学习：不安会自然消退
4. 容忍不确定性

## 反强迫的核心原则

### 原则一：区分强迫思维与现实

| 强迫思维 | 实际意义 |
|----------|----------|
| "我会伤害孩子" | 不代表你想要或将会伤害孩子 |
| "手被污染了" | 实际上手可能是干净的 |
| "门没锁" | 可能实际上已经锁好了 |

**关键理解**：强迫思维是OCD的症状，不是真实的威胁指示。

### 原则二：不让强迫行为"解决"焦虑

强迫行为短期缓解焦虑，但：
- 告诉大脑"这是危险的，需要保护"
- 阻止学习到"不安会自然消退"
- 使不安阈值越来越高

### 原则三：焦虑会自然消退

**消退原理**：
- 焦虑是一种波动性反应，会自然升降
- 不使用强迫行为，焦虑会自行下降
- 等待足够长的时间（通常20-30分钟）
- 反复暴露会降低整体焦虑水平

### 原则四：选择价值而非安全

价值导向意味着：
- 即使感到不安全，也做符合价值的事
- 接受"不确定"是生活的正常部分
- 选择生活质量而非短暂的安全感
