## 强迫思维识别

### 强迫思维的特征

强迫思维具有以下特征：
- 闯入性：突然出现，不是主动想的
- 不受欢迎：你想让它停止，但它不停止
- 引发焦虑：导致强烈的不适感
- 过度夸张：灾难化风险

### 强迫思维工作表

| 问题 | 你的回答 |
|------|----------|

| 思维出现时你如何反应？ | |
| 这些思维多久出现一次？ | |

### 常见强迫思维类型

| 类型 | 典型思维 |
|------|----------|
| 污染型 | "如果我碰到这个，会生很严重的病" |
| 伤害型 | "我会不小心伤害自己或他人" |
| 对称型 | "如果东西不整齐，会发生坏事" |
| 计数型 | "我必须做X次，否则Y会发生" |
| 道德/宗教型 | "我有一个不虔诚的想法，我会被惩罚" |
| 性相关型 | 不想要的性侵入性思维 |
| 失去控制型 | "我会失控，做出可怕的事" |

## 强迫行为识别

### 显性强迫行为

**可见的重复行为**：
- 反复洗手（有时洗到皮肤破损）
- 反复检查（门、炉子、锁）
- 重复性动作（坐下站起多次）
- 整理/清洁仪式
- 回避某些地方或物品

### 隐形强迫行为

**mental acts（精神行为）**：
- 默念特定词语或祈祷
- 在心里重复特定短语
- 计数
- 回忆特定场景以"中和"
- 努力不想某事（反而更想）

### 强迫行为清单

| 行为 | 频率 | 平均时长 | 触发情境 | 感觉到的需要程度 |
|------|------|----------|----------|------------------|
| 洗手 | 20次/天 | 每次5分钟 | 碰到脏东西 | 非常强烈 |
| 检查门锁 | 10次/天 | 每次2分钟 | 离开家时 | 强烈 |
| 默念祈祷 | 无数次 | 每次1-2分钟 | 强迫思维出现时 | 强烈 |

## 触发情境分析

### 识别你的触发点

**情境性触发**：
- 特定物体（门把手、厕所）
- 特定场所（医院、墓地）
- 特定时间（夜晚、独自一人）
- 特定活动（做饭、开车）

**事件性触发**：
- 压力增加
- 疲劳
- 生病
- 重大生活变化

### 触发-强迫-焦虑循环图

```
触发事件 → 强迫思维 → "危险！"反应 → 强迫行为 → 焦虑暂时缓解

```
