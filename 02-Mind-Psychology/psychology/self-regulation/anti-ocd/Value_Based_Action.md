## 价值澄清

### 什么是价值

价值是你内心深处真正在乎的东西，是你想要你的生活代表的东西。价值不是目标（目标可以完成），而是持续的指引方向。

### 价值领域

| 领域 | 价值示例 |
|------|----------|
| 家庭 | 成为支持、爱护的家庭成员 |
| 友谊 | 成为可靠、关怀的朋友 |
| 亲密关系 | 建立深厚的情感连接 |
| 工作/学习 | 追求成长、做出贡献 |
| 健康 | 照顾身体和心理 |
| 个人成长 | 持续学习、发展潜能 |
| 休闲 | 享受生活、体验乐趣 |
| 社会贡献 | 帮助他人、回馈社会 |

### 价值澄清练习

**问题**：
- 如果没有OCD，我想过什么样的生活？
- 我想成为什么样的人？
- 什么对我真正重要？
- 我希望我的生活代表什么？
- 如果我80岁了，我会后悔没做什么？

**深入探索**：
- 为什么这个价值对我重要？
- 如果我按照这个价值生活，会怎样？
- 描述一个体现这个价值的完美一天

## 价值vs强迫vs安全

### 三种行动的对比

| 类型 | 特征 | 感觉 |
|------|------|------|
| 强迫行为 | 感觉必须做、为了安全、做不完 | 沉重、焦虑、束缚 |
| 回避行为 | 为了安全、阻止不安、但也阻止生活 | 受限、遗憾 |
| 价值行动 | 主动选择、为了生活、可以灵活 | 自由、满足、有意义 |

### 区分练习

**情况**：需要出门上班

| 行动 | 类型 | 原因 |
|------|------|------|
| 检查门锁20次 | 强迫行为 | "如果不检查会出事" |
| 跳过同事聚会 | 回避行为 | "和人接触太焦虑" |
| 检查门锁1-2次，正常出门 | 价值行动 | "我需要准时上班，和同事相处是重要的" |

### 价值声明模板

基于____（价值），即使感到焦虑，我也愿意____（具体行动）。

示例：
- 基于"家庭关系"，即使感到焦虑，我也愿意和家人共进晚餐
- 基于"个人成长"，即使感到不确定，我也愿意尝试新事物
- 基于"健康"，即使感到不舒服，我也愿意进行日常活动

## 以值为导向的承诺行动

### 承诺 vs 勉强

| 勉强行动 | 承诺行动 |
|----------|----------|
| "我必须做" | "我选择做" |
| 为了避免焦虑 | 为了价值 |
| 被迫的感觉 | 主动的选择 |
| 做完就停止 | 持续的方向 |

### 价值导向的每日行动

**格式**：
- 价值：________
- 承诺行动：________
- 时间：________
- 可能的障碍：________
- 如何克服：________

**示例**：
- 价值：友谊
- 承诺行动：周五晚上和朋友视频
- 时间：周五7pm
- 障碍：担心自己说错话
- 克服：提醒自己"OCD在夸大风险"

### 行动vs完美

| 完美主义（OCD） | 价值导向 |
|------------------|----------|
| 必须正确做 | 做比不做强 |
| 不能有任何焦虑 | 可以带着不适行动 |
| 必须万无一失 | 允许不确定 |
| 做了就完成 | 持续进行 |

## ACT核心技术

### 接纳技术

**接纳定义**：允许不舒服的情绪、想法、感觉存在，不抗拒、不斗争、不试图控制。

**接纳练习**：
- "焦虑可以在这里"
- "我不喜欢这个感觉，但我可以带着它"
- "试图推开焦虑只会让它更强"

### 认知解离

（参见 Cognitive_Defusion.md）

### 当下时刻

- 将注意力带回当下
- 不沉浸于过去，不担忧未来
- 此刻我可以做什么？

### 观察者自我

- 发展一个稳定的"观察者"视角
- "我不仅仅是OCD，我是经历OCD的人"
- 你的价值比你以为的更多

### 价值承诺行动

**以值为导向的承诺公式**：
"我选择____（价值），即使____（不舒服），我愿意____（行动）"

---

*价值导向行动是长期维持改善的关键。只减少强迫行为是不够的，必须同时增加有价值的活法。*
