# 自尊与自我价值评估 | Self-Esteem & Self-Worth Assessment

## 技能信息

### 概述

本技能提供一套系统化的自尊与自我价值评估框架，整合 Rosenberg
自尊量表（RSES）、自我慈悲量表（SCS）、Beck 认知三角及内在
批评者类型识别四大核心模块。旨在帮助用户客观认识自身自尊水平，
识别导致低自尊的认知模式，并提供分级干预建议。

### 理论基础

- **Rosenberg 自尊理论** (1965): 自尊是个体对自我价值的整体评价
- **Neff 自我慈悲理论** (2003): 自我友善、共同人性与正念三大支柱
- **Beck 认知模型** (1976): 认知三角——对自我、世界与未来的负性信念
- **内在批评者理论** (Earley, 2004): 五种内在批评者亚型

### 适用人群

- 持续感到自卑或自我否定的个体
- 过度在意他人评价的人群
- 经历"不配得感"的来访者
- 情感敏感、容易受伤（"玻璃心"）的个体
- 希望提升自信与自我价值感的人群

### 禁忌与限制

- 本评估不替代临床诊断
- 若评估结果提示严重抑郁或自伤风险，应立即转介专业人员
- 本工具适用于16岁以上青少年及成人

### Step 1: RSES 自尊量表评估

#### 量表说明

- **全称**: Rosenberg Self-Esteem Scale (Rosenberg, 1965)
- **项目数**: 10 题
- **评分方式**: Likert 4 级评分（1=非常不同意 ~ 4=非常同意）
- **分数范围**: 10-40 分
- **信效度**: Cronbach α = 0.77-0.88，跨文化验证良好

#### 评估题目

请用户对以下10个陈述进行评分（1=非常不同意，2=不同意，
3=同意，4=非常同意）：

| 编号 | 题目                                               | 方向 |
|------|----------------------------------------------------|------|
| 1    | 我感到自己是一个有价值的人，至少与别人相等         | 正向 |
| 2    | 我觉得自己有许多优点                               | 正向 |
| 3    | 总的来说，我倾向于认为自己是一个失败者             | 反向 |
| 4    | 我能把事情做得和大多数人一样好                     | 正向 |
| 5    | 我觉得自己没有什么值得自豪的                       | 反向 |
| 6    | 我对自己持肯定态度                                 | 正向 |
| 7    | 总的来说，我对自己感到满意                         | 正向 |
| 8    | 我希望能更尊重自己                                 | 反向 |
| 9    | 我有时确实感到自己很没用                           | 反向 |
| 10   | 我有时认为自己一无是处                             | 反向 |

#### 计分规则

- 正向题目 (1,2,4,6,7): 直接计分 1-4
- 反向题目 (3,5,8,9,10): 反向计分 (5-原始分)
- 总分 = 10项得分之和，范围 10-40

#### 解读标准

```
总分 < 25   → 低自尊 (Low Self-Esteem)
              可能存在显著的自我否定模式，建议深入评估
总分 25-35  → 正常范围 (Normal Range)
              自尊水平基本健康，可关注特定触发情境
总分 > 35   → 高自尊 (High Self-Esteem)
              自我价值感良好，注意是否为防御性高自尊
```

#### 特殊注意

- 若总分 ≤ 15，需立即评估是否存在抑郁或自伤风险
- 第3、5、9、10题若单项得分=1（反向后=4），提示核心负面自我信念
- 注意"防御性高自尊"：总分>35但Step 3存在显著负性信念

### Step 2: 自我批评程度评估

#### 量表说明

- **全称**: Self-Compassion Scale - Short Form (SCS-SF; Neff, 2003)
- **维度**: 自我友善 vs 自我批评、共同人性 vs 孤立、正念 vs 过度认同
- **聚焦**: 本评估重点关注"自我批评"维度，快速筛查内在批评程度

#### 评估题目（核心4项）

请用户评分（1=几乎从不 ~ 5=几乎总是）：

| 编号 | 题目                                           | 维度           | 方向 |
|------|------------------------------------------------|----------------|------|
| 1    | 当我失败或犯错时，我会严厉地批评和责备自己     | 自我批评       | 反向 |
| 2    | 当我痛苦时，我会告诉自己很多人都有类似经历     | 共同人性       | 正向 |
| 3    | 当事情出错时，我会反复想"为什么会是我"         | 过度认同       | 反向 |
| 4    | 当我感到难过时，我会试着用平衡的态度看待问题   | 正念           | 正向 |

#### 解读标准

```
自我批评维度均分:
  ≥ 4.0  → 高度自我批评，内在批评者活跃
  3.0-3.9 → 中度自我批评，存在改善空间
  ≤ 2.9  → 自我慈悲水平良好

若题1得分=5且题2得分≤2: 标记"强烈自我否定+孤立感"模式
若题3得分≥4: 标记"反刍型自我批评"倾向
```

### Step 3: 核心信念识别

#### 理论框架

基于 Beck 认知三角 (Cognitive Triad)，识别用户是否存在对以下
三个维度的系统性负性核心信念：

```
         自我 (Self)
          /    \
         /      \
  世界 (World) — 未来 (Future)
```

#### 评估方法

通过以下引导性问题探查核心信念：

**对自我的信念**
- "你觉得你是一个怎样的人？"
- "你认为自己有哪些根本性的缺陷或不足？"
- 关键词捕捉: "我不够好"、"我是多余的"、"我没有价值"

**对世界的信念**
- "你觉得这个世界/周围的人是怎样的？"
- "你觉得别人会怎么看你？"
- 关键词捕捉: "世界是不安全的"、"别人都比我强"、"没人真心对我"

**对未来的信念**
- "你觉得你的未来会怎样？"
- "你认为事情会好转吗？"
- 关键词捕捉: "不会变好的"、"我不可能成功"、"没有希望"

#### 负性信念严重程度评级

```
轻度: 1个维度存在负性信念，频率低，可自行调节
中度: 2个维度存在负性信念，或单一维度信念强烈且固定
重度: 3个维度均存在负性信念，形成完整认知三角，需专业干预
```

#### 常见核心信念清单

| 信念类型 | 典型表述 | 所属维度 |
|----------|----------|----------|
| 无价值感 | "我没有任何价值" | 自我 |
| 无能感   | "我什么都做不好" | 自我 |
| 不被爱   | "没有人会真正爱我" | 自我/世界 |
| 危险世界 | "世界处处是危险" | 世界 |
| 他人敌意 | "别人总是会伤害我" | 世界 |
| 无望感   | "我的未来没有希望" | 未来 |
| 无助感   | "我无法改变任何事" | 未来 |

### Step 4: 内在批评者类型识别

#### 理论基础

内在批评者 (Inner Critic) 是内化了早期重要他人的批评声音，
根据其功能和特征可分为五种亚型。识别具体类型有助于精准干预。

#### 五种内在批评者类型

**1. 任务监工 (Taskmaster)**
- 内心独白: "你必须做得更多、更好，否则你就是废物"
- 核心驱动: 生产力与成就
- 表现: 过度工作、无法休息、成就焦虑
- 来源: 常内化于高期望的家庭环境

**2. 完美主义者 (Perfectionist)**
- 内心独白: "这还不够好，差一点就是失败"
- 核心驱动: 零缺陷标准
- 表现: 拖延、回避尝试、细节纠结
- 来源: 常内化于条件性认可的成长环境

**3. 暗中破坏者 (Underminer)**
- 内心独白: "你根本做不到，别自不量力了"
- 核心驱动: 破坏行动与自信
- 表现: 自我破坏、主动放弃、回避挑战
- 来源: 常内化于忽视或贬低的成长经历

**4. 内疚制造者 (Guilt-tripper)**
- 内心独白: "你又让别人失望了，你真自私"
- 核心驱动: 道德控制与内疚
- 表现: 过度讨好、无法拒绝、自我牺牲
- 来源: 常内化于情感操控的家庭模式

**5. 从众者 (Conformist)**
- 内心独白: "你必须和别人一样，否则会被排斥"
- 核心驱动: 归属与安全
- 表现: 压抑真实自我、随大流、害怕与众不同
- 来源: 常内化于强调服从的教养环境

#### 识别方法

引导用户回忆最近一次自我批评的经历，分析：

1. **批评内容**: 批评的具体内容指向哪种类型？
2. **情绪反应**: 配套的主要情绪是什么？
   - 任务监工 → 焦虑、紧迫感
   - 完美主义者 → 不满、挫败感
   - 暗中破坏者 → 绝望、无力感
   - 内疚制造者 → 内疚、羞耻感
   - 从众者 → 恐惧、不安感
3. **行为模式**: 批评导致的行为倾向是什么？

#### 输出格式

```
主要内在批评者类型: [类型名称]
强度评级: 强/中/弱
触发情境: [具体描述]
配套情绪: [主要情绪]
典型内心独白: "[引用用户的原话]"
次要类型: [如有]
```

## 干预建议

### Level 1: 自我慈悲练习 (适用于低自尊 + 轻度自我批评)

**适用条件**: RSES 20-25分，自我批评中度，核心信念轻度

**推荐练习**:

1. **自我慈悲信**
   - 给自己写一封充满理解与接纳的信
   - 以好朋友的口吻，回应你正在经历的困难
   - 每周2-3次，每次10分钟

2. **慈悲身体扫描**
   - 将手放在心脏位置，感受温暖
   - 对自己说: "这一刻是痛苦的，痛苦是人生的一部分，
     愿我善待自己"
   - 每日睡前5分钟

3. **共同人性冥想**
   - 想象此刻世界上有很多人正经历类似的痛苦
   - 感受与他人的连接，减少孤立感
   - 每周3次，每次5-10分钟

4. **积极自我对话日志**
   - 每天记录3件自己做得好的事（不论多小）
   - 用客观事实替代负面自我评价
   - 持续4周以上

---

### Level 2: 认知重构 (适用于低自尊 + 中重度自我批评)

**适用条件**: RSES < 20分，自我批评高度，核心信念中度

**推荐方法**:

1. **核心信念工作表**
   - 识别自动思维 → 找到底层核心信念
   - 收集支持与反对该信念的证据
   - 形成更平衡的替代信念
   - 示例:
     - 原信念: "我一无是处"
     - 证据支持: [列出]
     - 证据反对: [列出]
     - 替代信念: "我有一些不足，但也有自己的价值和长处"

2. **内在批评者对话技术**
   - 识别批评者声音 → 给它命名
   - 理解它的"保护意图"（它试图保护你免受什么？）
   - 用成人自我的声音回应:
     "我听到你了，你是在担心______。但现在我可以用更
      温和的方式照顾自己。"
   - 每日练习，逐步建立新的内在声音

3. **行为实验**
   - 设计小实验验证负性信念是否成立
   - 例: 信念"如果我说不，别人会讨厌我"
     → 实验: 在安全情境中练习温和拒绝
     → 记录实际结果 vs 预期结果
   - 每周1-2个行为实验

4. **价值澄清练习**
   - 列出对你最重要的5个核心价值
   - 评估当前生活在多大程度上与这些价值一致
   - 制定一个与核心价值更一致的小行动计划

---

### Level 3: 深度心理治疗转介 (适用于严重低自尊 + 结构性负性信念)

**适用条件**: RSES ≤ 15分，三维度均存在负性信念，或存在以下
任何一项:
- 评估中浮现童年创伤经历
- 存在自伤或自杀意念
- 严重的社会功能受损
- 自我否定伴随解离症状

**推荐方案**:

1. **转介专业心理治疗**
   - CBT 认知行为治疗（针对核心信念重建）
   - SFT 图式治疗（针对早期适应不良图式）
   - EMDR（若伴随创伤记忆）
   - IFS 内在家庭系统治疗（针对内在批评者工作）

2. **危机资源**（若有自伤风险）
   - 全国心理援助热线: 400-161-9995
   - 北京心理危机研究与干预中心: 010-82951332
   - 生命热线: 400-821-1215

3. **治疗期间支持性措施**
   - 每日情绪监测（推荐使用简版PHQ-9）
   - 建立安全计划（safe plan）
   - 指定信任联系人

## 使用示例

### 示例一: 轻度低自尊

```
用户输入: "我总觉得自己不够好，但日常还能正常工作生活。"

评估过程:
Step 1 - RSES 总分: 26/40 (正常偏低)
Step 2 - 自我批评均分: 3.2 (中度)
Step 3 - 核心信念: 对自我轻度负性("不够好")
Step 4 - 主要批评者: 完美主义者 (强度: 中)

综合分析: 用户自尊处于正常偏低水平，存在中度自我批评，
核心信念指向"不够好"的无能感，完美主义倾向是主要驱动。

干预建议: Level 1 自我慈悲练习
→ 自我慈悲信 (每周2次)
→ 积极自我对话日志 (每日)
→ 4周后复评
```

### 示例二: 严重低自尊

```
用户输入: "我觉得我活在这个世界上就是多余的，没有人需要我。"

评估过程:
Step 1 - RSES 总分: 13/40 (严重低自尊)
Step 2 - 自我批评均分: 4.5 (高度)
Step 3 - 核心信念:
  - 对自我: "我是多余的" (强烈负性)
  - 对世界: "没有人真正需要我" (强烈负性)
  - 对未来: "不会有人在乎我的存在" (强烈负性)
Step 4 - 主要批评者: 暗中破坏者 (强度: 强)
         次要批评者: 内疚制造者 (强度: 中)

综合分析: 用户呈现严重低自尊，完整的认知三角负性信念已形成，
内在批评者高度活跃，存在无价值感与被遗弃感。需评估自伤风险。

干预建议: Level 3 深度心理治疗转介
→ 立即评估自伤/自杀风险
→ 推荐CBT或图式治疗
→ 提供危机热线资源
→ 建立安全计划
```

### 示例三: 高自尊但存在隐性自我否定

```
用户输入: "我在人前表现得很自信，但独处时总会陷入自我怀疑。"

评估过程:
Step 1 - RSES 总分: 34/40 (正常偏高)
Step 2 - 自我批评均分: 3.8 (中度偏高)
Step 3 - 核心信念: 对自我存在隐性负性信念("真实的我不够好")
Step 4 - 主要批评者: 从众者 (强度: 中)

综合分析: 用户表面自尊水平正常偏高，但自我批评维度揭示了
隐性的自我否定模式，可能属于"防御性高自尊"——通过外在表现
掩盖内在的不安全感。从众者批评者驱动了"必须表现完美"的行为。

干预建议: Level 2 认知重构
→ 核心信念工作表 (探索真实自我)
→ 内在批评者对话技术
→ 价值澄清练习
```
