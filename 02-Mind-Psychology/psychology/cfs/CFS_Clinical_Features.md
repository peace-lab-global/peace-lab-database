# 慢性疲劳综合征临床表现 (CFS Clinical Features)

> ME/CFS的临床表现复杂多样，涉及多个器官系统。核心特征包括严重疲劳、劳后不适(PEM)、认知障碍和睡眠问题，同时可累及神经系统、免疫系统、心血管系统和胃肠系统。了解这些临床表现对于正确诊断和管理至关重要。

---

## 1. 症状学术语 (Symptomatology Terminology)

### 1.1 核心症状术语

| 中文 | 英文 | 原始语言/词源 | 临床定义 |
|------|------|--------------|---------|
| 疲劳 | Fatigue | 拉丁语: fatigare (使疲倦) | 持续的、压倒性的疲惫感，非单纯体力不足 |
| 劳后不适 | Post-Exertional Malaise (PEM) | 拉丁语: post + exertio + 法语: malaise | 活动后症状系统性恶化，延迟出现 |
| 崩溃 | Crash / Flare | 古英语: cræsc | 症状急剧恶化期 |
| 脑雾 | Brain Fog | — | 认知功能障碍的通俗描述 |
| 非恢复性睡眠 | Unrefreshing Sleep | — | 睡眠后仍感疲倦 |
| 直立不耐受 | Orthostatic Intolerance | 希腊语: ὀρθός (直立) + 拉丁语: intolerantia | 站立时症状加重 |
| 感觉过敏 | Sensory Hypersensitivity | 拉丁语: sensus + 希腊语: ὑπέρ (hyper, 过度) | 对光、声、触觉等过度敏感 |
| 活动能力包络 | Energy Envelope | — | 可用能量的边界范围 |
| 波动性 | Fluctuation | 拉丁语: fluctuare (波动) | 症状严重度的变化特性 |
| 恶化 | Exacerbation | 拉丁语: exacerbare (加剧) | 症状加重期 |
| 缓解 | Remission | 拉丁语: remittere (放松) | 症状减轻期 |

### 1.2 疼痛相关术语

| 中文 | 英文 | 原始语言/词源 | 定义 |
|------|------|--------------|------|
| 肌痛 | Myalgia | 希腊语: μῦς (mys, 肌肉) + ἄλγος (algos, 痛) | 肌肉疼痛 |
| 关节痛 | Arthralgia | 希腊语: ἄρθρον (arthron, 关节) + ἄλγος | 关节疼痛(非炎症性) |
| 头痛 | Cephalalgia | 希腊语: κεφαλή (kephale, 头) + ἄλγος | 头部疼痛，ME/CFS中常见新型或加重头痛 |
| 纤维肌痛 | Fibromyalgia | 拉丁语: fibra (纤维) + 希腊语: μῦς + ἄλγος | 广泛性肌肉骨骼疼痛，高度共病 |
| 异常性疼痛 | Allodynia | 希腊语: ἄλλος (allos, 其他) + ὀδύνη (odyne, 痛) | 非痛刺激引起疼痛 |
| 痛觉过敏 | Hyperalgesia | 希腊语: ὑπέρ + ἄλγος | 痛刺激引起过度疼痛反应 |

### 1.3 自主神经相关术语

| 中文 | 英文 | 词源 | 定义 |
|------|------|------|------|
| 体位性心动过速综合征 | POTS | — | 站立时HR↑≥30bpm(成人)或≥40bpm(青少年) |
| 神经介导性低血压 | Neurally Mediated Hypotension (NMH) | — | 长时间站立后血压下降 |
| 血管迷走性晕厥 | Vasovagal Syncope | 拉丁语: vas (血管) + vagus (迷走神经) | 迷走神经介导的晕厥 |
| 体温调节障碍 | Thermoregulatory Dysfunction | 希腊语: θέρμη (therme, 热) | 体温调节异常 |
| 多汗/少汗 | Hyperhidrosis / Hypohidrosis | 希腊语: ὑπέρ/ὑπο + ἱδρώς (hidros, 汗) | 出汗异常 |
| 心悸 | Palpitation | 拉丁语: palpitare (跳动) | 心跳异常感知 |

## 2. 核心症状详解 (Core Symptoms)

### 2.1 疲劳 (Fatigue)

#### 2.1.1 疲劳特征

| 特征 | 描述 | 与普通疲劳区别 |
|------|------|---------------|
| **严重程度** | 压倒性、使人虚弱 | 普通疲劳：可通过意志克服 |
| **持续时间** | 持续≥6个月 | 普通疲劳：休息后消失 |
| **与活动关系** | 非劳累引起，活动后加重 | 普通疲劳：劳累后出现，休息后恢复 |
| **休息效果** | 休息不能缓解 | 普通疲劳：休息后明显改善 |
| **功能影响** | 显著降低活动水平(通常≥50%) | 普通疲劳：轻度影响 |
| **性质** | 身体虚弱感，非单纯困倦 | 普通疲劳：主要为困倦 |

#### 2.1.2 疲劳的多维度

| 维度 | 表现 | 评估要点 |
|------|------|---------|
| **躯体疲劳** | 肢体沉重、无力、活动困难 | 与活动能力直接相关 |
| **认知疲劳** | 思考费力、难以集中注意力 | 智力活动后加重 |
| **情绪疲劳** | 情绪调节能力下降 | 社交活动后加重 |
| **感觉疲劳** | 对感觉刺激耐受性下降 | 声光环境敏感 |
| **睡眠疲劳** | 即使充足睡眠仍疲劳 | 睡眠质量不能改善 |

### 2.2 劳后不适 (Post-Exertional Malaise, PEM)

#### 2.2.1 PEM核心特征

| 特征 | 描述 | 临床意义 |
|------|------|---------|
| **触发因素** | 体力、认知、情绪或社交活动 | 多种活动均可触发 |
| **延迟出现** | 通常24-72小时后达峰 | 与即时疲劳不同 |
| **持续时间** | 数天到数周甚至更长 | 恢复缓慢 |
| **症状性质** | 所有症状系统性恶化 | 非单一症状 |
| **阈值低** | 极少活动即可触发 | 轻微活动如淋浴、交谈 |
| **不可预测性** | 相同活动反应可不同 | 难以预测 |

#### 2.2.2 PEM触发活动类型

| 活动类型 | 具体例子 | 风险等级 |
|---------|---------|---------|
| **体力活动** | 步行、家务、购物、运动 | 高风险 |
| **认知活动** | 阅读、写作、计算、决策 | 中-高风险 |
| **社交活动** | 交谈、会议、聚会 | 中-高风险 |
| **情绪活动** | 争论、压力事件 | 高风险 |
| **感觉刺激** | 噪音、强光、拥挤环境 | 中风险 |
| **直立活动** | 长时间站立、坐 | 中风险 |
| **日常自理** | 淋浴、穿衣、进食 | 重度患者高风险 |

#### 2.2.3 PEM时间特征

```
PEM时间进程

活动 → 可能感觉尚可 → 延迟期(24-72h) → 症状恶化高峰 → 缓慢恢复
 │         │              │                │              │
 ▼         ▼              ▼                ▼              ▼
Day 0    Day 0-1       Day 1-3          Day 2-5      Day 5-14+

特征表现:
• Day 0: 活动中可能无明显不适
• Day 0-1: 延迟期，开始出现疲劳
• Day 1-3: 症状逐渐加重
• Day 2-5: 症状高峰期，可能卧床
• Day 5-14+: 缓慢恢复，可能数周
```

#### 2.2.4 PEM症状表现

| 症状类型 | 具体表现 |
|---------|---------|
| 疲劳加重 | 极度虚弱，无法起床 |
| 认知恶化 | 脑雾加重，无法思考 |
| 疼痛加重 | 肌肉酸痛、头痛加剧 |
| 流感样症状 | 咽喉痛、淋巴结压痛、寒战 |
| 睡眠障碍 | 失眠或嗜睡，睡眠质量更差 |
| 自主神经症状 | 心悸、头晕加重 |
| 情绪变化 | 焦虑、抑郁、易激惹 |
| 感觉过敏 | 光敏、声敏加重 |

### 2.3 认知功能障碍 (Cognitive Dysfunction)

#### 2.3.1 "脑雾"特征

| 认知域 | 表现 | 日常影响 |
|--------|------|---------|
| **注意力** | 难以集中、易分散 | 无法专注工作、阅读 |
| **工作记忆** | 短期记忆受损、"进房间忘了做什么" | 忘记任务、对话内容 |
| **信息处理速度** | 思维迟缓、反应慢 | 跟不上对话、决策困难 |
| **词汇提取** | 找词困难、话在嘴边说不出 | 表达困难 |
| **执行功能** | 计划、组织、多任务能力下降 | 无法管理复杂任务 |
| **空间导航** | 方向感差、易迷路 | 熟悉环境中迷失 |

#### 2.3.2 认知症状特点

| 特点 | 描述 | 与其他疾病区别 |
|------|------|---------------|
| 波动性 | 随疲劳程度、时间变化 | 痴呆：进行性恶化 |
| 活动相关 | 认知活动后加重 | 抑郁：相对稳定 |
| 可恢复性 | 休息后部分改善 | 器质性疾病：持续 |
| 主观vs客观 | 主观感受可能超过客观测试 | 需要敏感测试 |
| 与警觉度相关 | 低警觉状态更差 | — |

### 2.4 睡眠障碍 (Sleep Disturbance)

#### 2.4.1 睡眠问题类型

| 类型 | 表现 | 发生率 |
|------|------|--------|
| **非恢复性睡眠** | 睡眠后仍感疲倦，核心特征 | >90% |
| **入睡困难** | 难以入睡 | 40-60% |
| **睡眠维持困难** | 频繁醒来、早醒 | 50-70% |
| **睡眠节律紊乱** | 昼夜节律延迟或紊乱 | 常见 |
| **多梦/噩梦** | 睡眠质量差 | 常见 |
| **嗜睡** | 白天过度困倦 | 常见 |
| **睡眠时间改变** | 过多或过少睡眠 | 常见 |

#### 2.4.2 睡眠-疲劳恶性循环

```
非恢复性睡眠恶性循环

睡眠质量差
     ↓
疲劳加重 ←───────────────────┐
     ↓                      │
白天活动减少                 │
     ↓                      │
生物钟紊乱 ──→ 进一步睡眠障碍─┘
     ↓
情绪影响(焦虑/抑郁)
     ↓
睡眠质量更差
```

## 3. 系统性症状 (Systemic Symptoms)

### 3.1 自主神经系统症状

#### 3.1.1 直立不耐受 (Orthostatic Intolerance)

| 亚型 | 诊断标准 | 主要症状 |
|------|---------|---------|
| **POTS** | 站立10分钟内HR↑≥30bpm(成人)或≥40bpm(青少年)，无显著血压下降 | 心悸、头晕、疲劳、认知障碍、视觉变化 |
| **神经介导性低血压(NMH)** | 倾斜试验中SBP下降≥20mmHg或DBP下降≥10mmHg | 头晕、恶心、晕厥前兆、虚弱 |
| **血管迷走性晕厥** | 心率和血压同时下降导致晕厥 | 晕厥、面色苍白、出汗 |
| **延迟性直立不耐受** | 站立>3分钟后出现症状 | 症状逐渐加重 |

#### 3.1.2 其他自主神经症状

| 症状 | 表现 | 机制 |
|------|------|------|
| **体温调节异常** | 怕冷、怕热、体温波动、低体温 | 下丘脑/自主神经调节异常 |
| **出汗异常** | 多汗、少汗、夜间盗汗 | 交感神经功能障碍 |
| **心悸** | 心跳感知增强、不规则感 | 自主神经敏感性增加 |
| **血压波动** | 低血压、高血压或波动 | 血管调节异常 |
| **瞳孔反应异常** | 光敏感、调节困难 | 瞳孔自主神经 |
| **排尿异常** | 尿频、尿急、排尿困难 | 膀胱自主神经 |
| **胃肠运动障碍** | 见胃肠系统详述 | 肠神经系统 |

### 3.2 免疫系统症状

#### 3.2.1 类流感症状

| 症状 | 特点 | 发生情境 |
|------|------|---------|
| **咽喉痛** | 反复发作、非感染性 | 常在PEM期间 |
| **淋巴结压痛** | 颈部、腋下，触痛但不肿大 | 可波动 |
| **低热/畏寒** | 低热(<38°C)或畏寒感 | 无感染证据 |
| **全身不适** | 类似发病初期的"病感" | PEM期间明显 |
| **肌肉酸痛** | 类似流感后酸痛 | 活动后加重 |

#### 3.2.2 免疫功能表现

| 表现 | 描述 | 临床意义 |
|------|------|---------|
| 反复感染 | 呼吸道感染频繁、恢复慢 | NK细胞功能下降 |
| 过敏/敏感性增加 | 新发过敏、化学物质敏感 | 免疫失调 |
| 自身免疫现象 | 部分患者有自身抗体 | 自身免疫亚型 |
| 慢性炎症症状 | 持续低度炎症表现 | 免疫激活 |

### 3.3 疼痛症状

#### 3.3.1 疼痛类型与特点

| 疼痛类型 | 位置 | 特点 |
|---------|------|------|
| **肌肉痛** | 广泛或局限 | 深部酸痛、活动后加重 |
| **关节痛** | 多关节 | 无肿胀、游走性 |
| **头痛** | 多种类型 | 新发或模式改变的头痛 |
| **神经痛** | 肢体、躯干 | 烧灼感、刺痛、麻木 |
| **胸痛** | 胸壁 | 非心源性，可能为肌肉骨骼 |
| **腹痛** | 腹部 | 与IBS相关 |

#### 3.3.2 头痛详解

| 头痛类型 | 特征 | 发生率 |
|---------|------|--------|
| 紧张型 | 双侧、压迫感、轻中度 | 最常见 |
| 偏头痛样 | 单侧、搏动、伴恶心光敏 | 常见 |
| 颈源性 | 颈部起源、向头部放射 | 常见 |
| 压力性 | 颅内压感、站立加重 | 部分患者 |
| 混合型 | 多种特征 | 常见 |

### 3.4 胃肠系统症状

#### 3.4.1 胃肠症状谱

| 症状 | 表现 | 与IBS重叠 |
|------|------|----------|
| **腹痛/腹部不适** | 痉挛性或隐痛 | 核心IBS症状 |
| **腹胀** | 腹部膨胀感 | 常见 |
| **排便改变** | 便秘、腹泻或交替 | IBS亚型相关 |
| **恶心** | 持续或间歇 | 自主神经相关 |
| **食欲改变** | 减退或异常 | 常见 |
| **吞咽困难** | 功能性吞咽困难 | 部分患者 |
| **胃食管反流** | 烧心、反酸 | 常见 |
| **食物不耐受** | 新发食物敏感 | 增加 |

#### 3.4.2 肠易激综合征(IBS)共病

| 特征 | ME/CFS + IBS | 临床意义 |
|------|-------------|---------|
| 共病率 | 35-92% | 高度重叠 |
| 可能机制 | 肠脑轴异常、自主神经功能障碍 | 共同病理 |
| 治疗影响 | 需要同时管理 | 整合治疗 |
| 饮食因素 | 低FODMAP可能有帮助 | 营养干预 |

### 3.5 神经感觉症状

#### 3.5.1 感觉过敏

| 类型 | 表现 | 影响 |
|------|------|------|
| **光敏感** | 强光不耐受、需戴墨镜 | 避免明亮环境 |
| **声敏感** | 噪音不耐受、需耳塞 | 社交困难 |
| **触觉敏感** | 轻触不适 | 衣物选择受限 |
| **嗅觉敏感** | 气味过敏 | 化学物质不耐受 |
| **温度敏感** | 极端温度不耐受 | 环境控制需求 |

#### 3.5.2 感觉异常

| 症状 | 描述 | 分布 |
|------|------|------|
| 感觉异常(Paresthesia) | 麻木、刺痛、针扎感 | 肢体远端常见 |
| 振动感异常 | 内部振动感 | 全身 |
| 平衡障碍 | 不稳、眩晕 | 与前庭功能相关 |
| 视觉症状 | 模糊、对焦困难、飞蚊 | 眼部 |
| 听觉症状 | 耳鸣、听觉过敏 | 听觉系统 |

## 4. 特殊人群临床表现 (Special Populations)

### 4.1 儿童与青少年

| 特点 | 描述 | 临床意义 |
|------|------|---------|
| 发病年龄 | 可低至5岁，青春期高峰 | 学龄期重要考虑 |
| 主要表现 | 疲劳、学校出勤困难、认知问题 | 影响教育 |
| 直立不耐受 | POTS更常见 | 诊断标准不同(HR↑≥40bpm) |
| 症状波动 | 可能比成人更大波动 | 可能被误解为"逃避" |
| 心理影响 | 社交隔离、身份认同问题 | 需要心理支持 |
| 家庭影响 | 照护者负担、家庭功能 | 家庭支持重要 |
| 预后 | 可能比成人稍好 | 早期识别重要 |

#### 青少年ME/CFS特异性表现

| 表现 | 详情 |
|------|------|
| 学业困难 | 无法完成学业、需要调整 |
| 同伴关系 | 社交减少、孤立 |
| 活动减少 | 无法参加体育、课外活动 |
| 睡眠节律 | 延迟睡眠相综合征常见 |
| 情绪问题 | 继发性焦虑、抑郁 |
| 身体发育 | 可能影响生长发育 |

### 4.2 老年患者

| 特点 | 描述 | 挑战 |
|------|------|------|
| 诊断困难 | 与年龄相关疾病重叠 | 需排除更多疾病 |
| 多病共存 | 心血管、代谢疾病等 | 症状归因困难 |
| 多药物使用 | 药物相互作用 | 治疗选择受限 |
| 功能储备 | 基础功能储备下降 | 恢复潜力可能受限 |
| 照护需求 | 更可能需要照护支持 | 社会服务需求 |

### 4.3 重度与极重度患者

#### 4.3.1 严重度分级

| 级别 | 功能水平 | 典型表现 |
|------|---------|---------|
| **轻度** | 活动能力下降约50%，可部分工作/学习 | 需大量休息，避免过度活动 |
| **中度** | 大部分时间居家，无法工作 | 主要在家活动，外出困难 |
| **重度** | 大部分时间卧床，自理困难 | 需要日常照护，活动极受限 |
| **极重度** | 完全卧床，完全依赖照护 | 可能需管饲，光敏声敏极端 |

#### 4.3.2 重度患者特殊表现

| 特征 | 表现 |
|------|------|
| **极度感觉过敏** | 需黑暗安静环境，无法耐受交谈 |
| **吞咽困难** | 可能需要流质饮食或管饲 |
| **完全卧床** | 无法坐起、翻身困难 |
| **极度PEM** | 任何活动(包括交谈)触发崩溃 |
| **认知障碍** | 可能无法沟通 |
| **体位限制** | 无法耐受直立位 |
| **照护依赖** | 完全依赖他人 |

### 4.4 性别差异

| 方面 | 女性 | 男性 |
|------|------|------|
| 患病率 | 2-4倍高于男性 | 相对较低 |
| 发病年龄 | 可能与激素变化相关 | — |
| 症状特点 | 可能更多自主神经症状 | 可能更多认知症状 |
| 月经影响 | 经期可能症状加重 | — |
| 诊断 | 可能更易被误诊为心身疾病 | 可能更难获得诊断 |
| 代谢特征 | 特异性代谢模式 | 不同的代谢模式 |

## 5. 症状波动与诱发因素 (Symptom Fluctuation & Triggers)

### 5.1 症状波动特征

| 波动类型 | 时间尺度 | 特点 |
|---------|---------|------|
| 日内波动 | 小时 | 上午常较差，下午可能改善 |
| 日间波动 | 天 | 好日子与坏日子交替 |
| 周期波动 | 周-月 | 相对稳定期与恶化期 |
| 季节波动 | 季节 | 部分患者有季节模式 |
| 长期趋势 | 年 | 总体稳定、改善或恶化 |

### 5.2 常见诱发/加重因素

| 因素类别 | 具体因素 | 管理建议 |
|---------|---------|---------|
| **体力活动** | 运动、家务、购物、步行 | 活动管理/Pacing |
| **认知活动** | 工作、阅读、屏幕使用 | 认知pacing |
| **社交活动** | 聚会、交谈、冲突 | 限制社交负荷 |
| **感染** | 感冒、流感等 | 预防、早期治疗 |
| **睡眠不足** | 睡眠质量/数量不足 | 睡眠卫生 |
| **应激/情绪** | 心理压力、情绪波动 | 压力管理 |
| **环境因素** | 极端温度、高海拔、天气变化 | 环境控制 |
| **感觉刺激** | 强光、噪音、拥挤 | 感觉保护 |
| **饮食因素** | 某些食物、酒精、咖啡因 | 识别个人触发 |
| **药物** | 某些药物可能加重 | 药物审查 |
| **手术/创伤** | 手术、受伤 | 谨慎评估必要性 |
| **激素变化** | 月经、更年期 | 激素管理 |

### 5.3 保护因素

| 因素 | 作用 | 实施建议 |
|------|------|---------|
| 适当休息 | 预防PEM | 定期休息，不仅是累了才休息 |
| 活动管理 | 维持在能量包络内 | 使用心率监测、活动日志 |
| 稳定作息 | 维持生物钟 | 固定起床时间 |
| 营养均衡 | 支持身体功能 | 规律进食、充足水分 |
| 社会支持 | 心理健康 | 理解支持的社交网络 |
| 压力管理 | 减少触发 | 正念、放松技术 |

## 6. 症状模式与表型 (Symptom Patterns & Phenotypes)

### 6.1 症状聚类研究

| 亚型 | 主要特征 | 可能机制 |
|------|---------|---------|
| **高炎症型** | 更多免疫症状、流感样 | 免疫激活为主 |
| **自主神经型** | POTS、直立不耐受突出 | 自主神经功能障碍为主 |
| **认知型** | 脑雾、认知障碍为主 | 神经炎症/代谢 |
| **疼痛型** | 纤维肌痛样、广泛疼痛 | 中枢敏化 |
| **混合型** | 多系统均衡受累 | 多机制 |

### 6.2 发病模式

| 发病类型 | 比例 | 特点 |
|---------|------|------|
| 急性感染后 | ~70% | 明确感染后发病，最常见 |
| 渐进发病 | ~20% | 无明确触发，逐渐加重 |
| 应激/创伤后 | ~10% | 重大生活事件、手术后 |

### 6.3 病程模式

| 模式 | 描述 | 比例 |
|------|------|------|
| 波动稳定型 | 症状波动但总体稳定 | ~40-50% |
| 逐渐改善型 | 总体趋势改善 | ~20-30% |
| 逐渐恶化型 | 总体趋势恶化 | ~10-20% |
| 复发缓解型 | 明显缓解期与恶化期 | ~10-15% |
| 完全康复 | 恢复正常功能 | ~5-10% |

## 7. 功能影响评估 (Functional Impact Assessment)

### 7.1 日常功能影响

| 领域 | 轻度影响 | 中度影响 | 重度影响 |
|------|---------|---------|---------|
| **工作/学业** | 部分工作，需调整 | 无法工作/学习 | — |
| **家务** | 基本可完成 | 需要帮助 | 完全依赖 |
| **个人卫生** | 独立完成 | 需要休息 | 需要帮助 |
| **社交** | 有限社交 | 很少社交 | 基本无法 |
| **休闲活动** | 有限参与 | 很少活动 | 无法参与 |
| **外出** | 可以外出，需休息 | 很少外出 | 完全居家/卧床 |

### 7.2 SF-36生活质量对比

| SF-36维度 | ME/CFS患者 | 一般人群 | 其他慢性病 |
|-----------|-----------|---------|-----------|
| 躯体功能 | 显著降低 | 高 | 中等降低 |
| 躯体角色 | 极度降低 | 高 | 降低 |
| 躯体疼痛 | 降低 | 较高 | 不同程度降低 |
| 总体健康 | 显著降低 | 较高 | 降低 |
| 活力 | 极度降低 | 中高 | 降低 |
| 社会功能 | 显著降低 | 高 | 中等降低 |
| 情感角色 | 降低 | 高 | 不同程度 |
| 心理健康 | 降低 | 较高 | 不同程度 |

**ME/CFS患者的生活质量评分低于许多其他慢性疾病，包括心力衰竭、类风湿关节炎、多发性硬化症等。**

## 8. 心理与情绪表现 (Psychological & Emotional Manifestations)

### 8.1 继发性心理问题

| 问题 | 发生率 | 性质 |
|------|--------|------|
| 抑郁症状 | 30-50% | 继发于慢性疾病和功能丧失 |
| 焦虑症状 | 30-40% | 与不确定性、症状相关 |
| 悲伤/哀悼 | 普遍 | 对失去功能、角色的哀悼 |
| 挫败感 | 普遍 | 对限制的反应 |
| 孤立感 | 常见 | 社交减少、不被理解 |

### 8.2 心理表现与原发性精神疾病区别

| 特征 | ME/CFS继发心理问题 | 原发性抑郁/焦虑 |
|------|------------------|----------------|
| 发病顺序 | 躯体症状先于心理症状 | 心理症状可独立出现 |
| 对活动的态度 | 想做但做不了(Wishing to but unable) | 可能不想做(缺乏兴趣/动机) |
| PEM | 存在 | 不存在 |
| 活动后 | 症状恶化 | 活动可能改善情绪 |
| 躯体症状 | 多系统、特异性 | 非特异性躯体化 |
| 对治疗反应 | 抗抑郁药可能帮助情绪但不改善疲劳 | 抗抑郁药可改善核心症状 |

### 8.3 创伤与ME/CFS

| 方面 | 描述 |
|------|------|
| 既往创伤 | 部分研究显示早年创伤可能增加风险 |
| 疾病本身的创伤 | 被诊断、被否定、功能丧失均为创伤性体验 |
| 医疗创伤 | 不当治疗(如GET)、不被相信可造成创伤 |
| 心理支持 | 创伤知情的支持很重要 |

## 9. 临床表现评估要点 (Clinical Assessment Points)

### 9.1 病史采集要点

| 领域 | 关键问题 |
|------|---------|
| **发病** | 何时开始？有无明确触发(感染等)？ |
| **疲劳特征** | 性质、严重度、休息能否缓解？ |
| **PEM** | 活动后症状恶化吗？延迟多久？持续多久？ |
| **睡眠** | 睡眠后感觉恢复吗？睡眠问题类型？ |
| **认知** | 注意力、记忆、思维有何问题？ |
| **直立耐受** | 站立时有何症状？ |
| **功能水平** | 与发病前相比下降多少？日常活动能力？ |
| **症状波动** | 好日子/坏日子模式？诱发因素？ |
| **治疗史** | 尝试过什么治疗？效果如何？ |
| **影响** | 对工作、社交、家庭的影响？ |

### 9.2 体格检查要点

| 检查 | 目的 | ME/CFS可能发现 |
|------|------|---------------|
| 一般状态 | 评估严重度 | 可能看起来"正常"或明显疲态 |
| 生命体征 | 基础血压心率 | 可能正常或轻度异常 |
| 直立试验 | POTS/OI | HR增加≥30bpm(成人) |
| 淋巴结 | 压痛 | 可能有颈部/腋下压痛 |
| 咽喉 | 充血 | 可能轻度充血(无感染) |
| 肌肉骨骼 | 压痛点 | 类似纤维肌痛压痛点 |
| 神经系统 | 常规检查 | 通常正常，可能有细微异常 |

### 9.3 红旗症状

**以下症状需要进一步检查以排除其他疾病：**

| 红旗症状 | 考虑疾病 |
|---------|---------|
| 体重显著下降(非刻意) | 恶性肿瘤、甲亢、糖尿病 |
| 持续发热(>38°C) | 感染、恶性肿瘤、风湿病 |
| 淋巴结肿大(非压痛) | 淋巴瘤、感染 |
| 局灶性神经体征 | 神经系统疾病 |
| 严重头痛(雷击样/新发) | 颅内病变 |
| 关节肿胀 | 炎症性关节炎 |
| 进行性恶化 | 需重新评估诊断 |

## 10. 专家临床观点 (Expert Clinical Perspectives)

### 10.1 诊断要点

> **"PEM是ME/CFS的标志性特征。如果患者没有PEM，应该考虑其他诊断。"**
> — NICE 2021指南

> **"疲劳只是冰山一角。ME/CFS是一种多系统疾病，需要全面评估认知、睡眠、直立耐受和疼痛。"**
> — Lucinda Bateman, MD

### 10.2 常见误区

| 误区 | 事实 |
|------|------|
| "看起来很好所以没病" | ME/CFS是隐形疾病，外观可能正常 |
| "休息会好的" | 休息不能缓解疲劳，需要活动管理 |
| "应该多运动" | 运动可能有害，需要谨慎的活动管理 |
| "主要是心理问题" | 是生物医学疾病，心理问题是继发的 |
| "缺乏客观证据" | 有大量研究证据，只是缺乏临床检测 |
| "会自己好的" | 多数患者不会完全康复 |

### 10.3 患者体验的重要性

**理解患者体验对于临床照护至关重要：**

- **"今天好一点"不意味着"病好了"** — 症状波动是疾病特征
- **活动的代价** — 患者可能为某项活动付出数天的代价
- **隐形的努力** — 基本日常活动需要巨大努力
- **失去的人生** — 患者失去的不仅是健康，还有职业、社交、身份
- **不被相信的痛苦** — 长期不被理解是重大心理负担
