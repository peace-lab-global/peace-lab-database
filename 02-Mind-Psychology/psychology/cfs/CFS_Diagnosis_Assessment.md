# 慢性疲劳综合征诊断与评估 (CFS Diagnosis & Assessment)

> ME/CFS的诊断主要基于临床表现和排除性诊断，目前尚无确定性生物标志物。本文档详细介绍各种诊断标准、评估工具、实验室检查和鉴别诊断流程，为临床医生提供系统的诊断框架。

---

## 2. 主要诊断标准详解 (Diagnostic Criteria in Detail)

### 2.1 诊断标准演变概览

| 标准 | 年份 | 制定机构 | 特点 | 主要用途 |
|------|------|---------|------|---------|
| Holmes/CDC | 1988 | CDC | 首个正式标准，过于宽泛 | 历史意义 |
| Oxford | 1991 | 英国研究者 | 仅要求疲劳，最宽松 | 研究(已不推荐) |
| Fukuda/CDC | 1994 | CDC | 最广泛使用，研究金标准 | 研究 |
| 加拿大共识(CCC) | 2003 | 临床专家组 | 强调PEM，较严格 | 临床 |
| 国际共识(ICC) | 2011 | 国际专家组 | 最严格，PEM必须 | 研究/临床 |
| IOM/SEID | 2015 | IOM | 简化临床诊断 | 临床 |
| NICE 2021 | 2021 | 英国NICE | 强调PEM，否定GET | 临床指南 |

### 2.2 IOM/SEID 2015 诊断标准 (临床推荐)

#### 2.2.1 诊断要求

**必须同时满足以下全部条件：**

├─────────────────────────────────────────────────────────────┤
│                                                             │

│     • 持续≥6个月                                            │
│     • 非终生存在                                            │

│     • 非过度劳累引起                                        │

│                                                             │

### 2.3 加拿大共识标准 (CCC) 2003

#### 2.3.1 诊断要求

| 类别 | 要求 |
|------|------|
| **疲劳** | 新发、无法解释的持续/反复躯体和精神疲劳，显著降低活动水平 |
| **劳后不适/疲劳** | 活动后症状恶化，异常丧失体力/认知/社交耐力，恢复时间延长(≥24h) |
| **睡眠障碍** | 非恢复性睡眠或睡眠节律紊乱 |
| **疼痛** | 显著肌痛和/或关节痛，和/或头痛 |
| **神经/认知** | ≥2项：思维混乱、注意力受损、记忆障碍 |
| **自主神经/神经内分泌/免疫** | ≥1项：每个类别的症状 |

#### 2.3.2 CCC详细症状要求

**神经/认知症状 (至少2项):**
- 思维混乱
- 注意力集中困难
- 短期记忆障碍
- 词汇检索困难
- 信息处理/分类/检索困难
- 感知/感觉障碍
- 动作失序
- 计算困难
- 新信息获取困难

**自主神经症状 (至少1项):**
- 直立不耐受(NMH, POTS, 延迟性低血压)
- 头晕
- 极端面色苍白
- 恶心/肠易激
- 尿频/膀胱功能障碍
- 心悸(有/无心律失常)
- 呼吸费力

**神经内分泌症状 (至少1项):**
- 体温调节失调
- 出汗异常
- 发热感/寒战反复
- 极端冷热不耐受
- 食欲/体重明显改变
- 对应激、情绪、疾病耐受性下降

**免疫症状 (至少1项):**
- 淋巴结压痛
- 反复咽喉痛
- 反复流感样症状
- 全身不适
- 新发对食物、药物、化学物质的敏感

### 2.4 国际共识标准 (ICC) 2011

#### 2.4.1 核心要求

| 类别 | 具体要求 |
|------|---------|
| **A. 劳后神经免疫性疲乏 (PENE)** | 必须满足，是ME的核心特征 |
| **B. 神经系统损害** | 至少3个类别的1个症状 |
| **C. 免疫/胃肠/泌尿生殖损害** | 至少3个类别的1个症状 |
| **D. 能量产生/运输损害** | 至少1个症状 |

#### 2.4.2 PENE详细标准

**劳后神经免疫性疲乏(PENE)必须满足：**
1. 体力或认知活动后明显快速疲乏
2. 劳后症状恶化(如急性流感样症状)
3. 劳后疲乏可在活动后立即出现或延迟数小时/数天
4. 恢复期延长(通常≥24小时)
5. 症状严重度与致病活动相关程度低

### 2.5 NICE 2021 更新标准

#### 2.5.1 疑似ME/CFS标准

**所有以下症状存在≥6周时应疑似ME/CFS：**
- 使人虚弱的疲劳
- 活动后症状恶化(PEM)
- 非恢复性睡眠
- 认知困难("脑雾")

#### 2.5.2 确诊ME/CFS

**症状持续≥3个月 + 排除其他疾病 = 确诊**

**NICE 2021关键更新：**
- 强调PEM是标志性特征
- **明确不推荐GET(分级运动疗法)**
- 推荐活动管理/Pacing
- 强调以患者为中心的个体化照护

### 2.6 诊断标准比较

| 特征 | Fukuda | CCC | ICC | IOM | NICE 2021 |
|------|--------|-----|-----|-----|-----------|
| PEM必须 | 否(4/8之一) | 是 | 是(PENE) | 是 | 是 |
| 睡眠障碍必须 | 否(4/8之一) | 是 | 否 | 是 | 是 |
| 认知障碍必须 | 否(4/8之一) | 是 | 是(部分) | 是(或OI) | 是 |
| 直立不耐受 | 不包含 | 是 | 是 | 是(或认知) | 不特别强调 |
| 疼痛必须 | 否 | 是 | 否 | 否 | 否 |
| 时间要求 | ≥6个月 | ≥6个月 | 无特定 | ≥6个月 | ≥3个月 |
| 严格程度 | 低 | 中 | 高 | 中 | 中 |

## 3. 临床评估流程 (Clinical Assessment Process)

### 3.1 初步评估

#### 3.1.1 详细病史采集

| 领域 | 关键问题 | 评估目的 |
|------|---------|---------|
| **发病情况** | 何时开始？有无明确触发因素(感染等)？发病模式(急性/渐进)？ | 识别发病类型 |
| **疲劳特征** | 性质描述？严重程度(1-10)?与发病前比较？休息能否缓解？ | 评估核心症状 |
| **PEM** | 活动后症状恶化吗？延迟多久？持续多久？什么活动触发？ | 核心诊断特征 |
| **睡眠** | 睡眠后感觉恢复吗？入睡/维持困难？睡眠时长？质量？ | 评估睡眠障碍 |
| **认知功能** | 注意力、记忆、思维速度、词汇检索有何问题？ | 评估脑雾 |
| **直立症状** | 站立时有何症状？头晕？心悸？需要坐下？ | 评估OI |
| **疼痛** | 肌肉痛？关节痛？头痛？神经痛？位置和性质？ | 评估疼痛负担 |
| **其他症状** | 免疫症状？胃肠症状？感觉过敏？ | 全面症状评估 |
| **功能水平** | 工作能力？日常活动？社交？与发病前比较下降多少？ | 评估功能影响 |
| **病程** | 症状波动模式？加重/缓解因素？总体趋势？ | 了解病程特点 |
| **既往史** | 既往疾病？家族史？药物/过敏史？ | 排除其他诊断 |
| **心理社会** | 情绪状态？社会支持？疾病影响？ | 全面评估 |

#### 3.1.2 体格检查

| 检查项目 | 方法 | ME/CFS可能发现 | 需排除情况 |
|---------|------|---------------|-----------|
| **一般状态** | 观察 | 可能外观正常或明显疲态 | — |
| **生命体征** | 测量BP、HR、体温 | 可能正常；低体温 | 发热提示感染 |
| **卧立位血压心率** | 卧位3分钟后站立测量 | HR↑≥30bpm提示POTS | — |
| **淋巴结** | 触诊颈部、腋下 | 可能有压痛(非肿大) | 淋巴结病需进一步检查 |
| **咽喉** | 视诊 | 可能轻度充血 | 明显感染体征 |
| **甲状腺** | 触诊 | 通常正常 | 甲状腺肿大 |
| **心肺** | 听诊 | 通常正常 | 心脏杂音、肺部异常 |
| **腹部** | 触诊 | 通常正常 | 肝脾肿大、肿块 |
| **肌肉骨骼** | 压痛点检查 | 可能有纤维肌痛样压痛 | 关节肿胀/炎症 |
| **神经系统** | 常规检查 | 通常正常或细微异常 | 局灶性神经体征 |
| **皮肤** | 视诊 | 通常正常 | 皮疹、苍白 |

### 3.2 直立不耐受评估

#### 3.2.1 简易主动站立试验 (Active Standing Test)

**方法：**
1. 患者仰卧休息5-10分钟
2. 测量卧位血压和心率
3. 患者站立(无支撑)
4. 每2分钟测量血压心率，共10分钟
5. 记录症状

**判读：**

| 指标 | POTS | NMH | 正常 |
|------|------|-----|------|
| HR变化 | ↑≥30bpm(成人)或≥40bpm(青少年) | 可增加 | ↑<30bpm |
| SBP变化 | 通常稳定 | ↓≥20mmHg | 轻度↓或稳定 |
| DBP变化 | 可能轻度↑ | ↓≥10mmHg | 轻度变化 |
| 症状 | 头晕、心悸、疲劳 | 头晕、恶心、晕厥前兆 | 轻度或无 |

#### 3.2.2 NASA 10分钟倾斜试验 (简化版)

**方法：**
- 使用倾斜台，70度角
- 监测心率、血压、症状
- 持续10分钟或至症状无法耐受

**优势：** 标准化，可重复，更敏感

### 3.3 两日心肺运动试验 (2-Day CPET)

#### 3.3.1 方法与意义

| 方面 | 描述 |
|------|------|
| **方法** | 第1天进行最大心肺运动试验；24小时后重复测试 |
| **意义** | 客观评估PEM，ME/CFS患者第2天表现显著下降 |
| **测量指标** | 峰值耗氧量(VO2peak)、无氧阈(AT)、工作能力 |
| **典型发现** | 第2天VO2peak、AT下降；恢复时间延长 |
| **限制** | 可能诱发严重PEM，需谨慎；非常规临床检查 |

#### 3.3.2 2-Day CPET典型结果

| 指标 | 正常人 | ME/CFS患者 |
|------|--------|-----------|
| 第2天VO2peak | 与第1天相似 | 显著下降(5-22%) |
| 第2天无氧阈 | 与第1天相似 | 显著下降 |
| 第2天工作能力 | 相似或轻度改善 | 下降 |
| 症状恢复 | 快速 | 延迟(数天) |

## 4. 症状评估量表 (Symptom Assessment Scales)

### 4.1 疲劳评估量表

#### 4.1.1 Chalder疲劳量表 (Chalder Fatigue Scale, CFQ)

| 属性 | 描述 |
|------|------|
| 项目数 | 11项 |
| 维度 | 躯体疲劳(7项)、精神疲劳(4项) |
| 计分 | 二分法(0-11)或Likert(0-33) |
| 用途 | 评估疲劳严重度和类型 |
| 信效度 | 良好，广泛使用 |

**示例项目：**
- 您是否有肌肉无力感？
- 您是否需要更多休息？
- 您是否感到困倦或嗜睡？
- 您是否难以集中注意力？

| 属性 | 描述 |
|------|------|

| 总分 | 9-63，平均分≥4提示显著疲劳 |
| 用途 | 评估疲劳对功能的影响 |
| 信效度 | 良好 |

#### 4.1.3 多维疲劳量表 (Multidimensional Fatigue Inventory, MFI-20)

| 属性 | 描述 |
|------|------|
| 项目数 | 20项 |

| 计分 | 5点量表，各维度4-20分 |
| 用途 | 多维度评估疲劳 |

| 20 | 卧床不起，需帮助自理 |
| 10 | 完全卧床，依赖照护 |
| 0 | 完全失能，完全依赖 |

#### 4.4.3 WHO失能评定量表 (WHODAS 2.0)

| 属性 | 描述 |
|------|------|
| 项目数 | 36项(完整版)或12项(简版) |
| 维度 | 认知、活动、自理、相处、生活活动、参与 |
| 用途 | 标准化失能评估 |

### 4.5 认知功能评估

#### 4.5.1 症状问卷中的认知项目

**DSQ认知维度：**
- 注意力集中困难
- 短期记忆问题
- 找词困难
- 信息处理慢
- 多任务困难

#### 4.5.2 客观认知测试

| 测试 | 评估功能 | ME/CFS发现 |
|------|---------|-----------|
| 数字广度 | 工作记忆 | 可能下降 |
| Stroop测试 | 注意力、执行功能 | 反应时间延长 |
| 连线测试 | 视觉搜索、处理速度 | 时间延长 |
| 词汇流畅性 | 词汇检索 | 可能下降 |
| N-back | 工作记忆 | 负荷增加时下降 |

**注意：** 客观测试可能低估症状，因患者可能在测试时耗尽能量，之后出现PEM

### 4.6 睡眠评估

#### 4.6.1 匹兹堡睡眠质量指数 (PSQI)

| 属性 | 描述 |
|------|------|
| 项目数 | 19项 |
| 维度 | 主观睡眠质量、入睡时间、睡眠时长、睡眠效率、睡眠障碍、安眠药使用、日间功能 |
| 总分 | 0-21，>5提示睡眠质量差 |
| 用途 | 筛查睡眠问题 |

#### 4.6.2 非恢复性睡眠评估

**关键问题：**
- 睡眠后您感觉恢复精力了吗？(0-10)
- 您醒来时感觉精神焕发还是仍然疲倦？
- 您需要多久才能完全清醒？

### 4.7 直立不耐受评估

#### 4.7.1 直立分级评估量表 (OGS)

| 级别 | 描述 |
|------|------|
| 0 | 无直立不耐受 |
| 1 | 轻度，可正常站立，轻度不适 |
| 2 | 中度，站立受限，需要支撑或坐下 |
| 3 | 重度，难以长时间站立，频繁需要躺下 |
| 4 | 极重度，几乎无法站立 |

#### 4.7.2 自主神经症状问卷 (COMPASS-31)

| 属性 | 描述 |
|------|------|
| 项目数 | 31项 |
| 维度 | 直立不耐受、血管舒缩、分泌功能、胃肠、膀胱、瞳孔 |
| 用途 | 综合评估自主神经功能 |

## 5. 实验室与辅助检查 (Laboratory & Auxiliary Tests)

### 5.1 基本排除性检查

#### 5.1.1 必要检查

| 检查 | 目的 | ME/CFS预期结果 |
|------|------|---------------|
| **血常规** | 排除贫血、感染、血液病 | 通常正常 |
| **血沉(ESR)** | 炎症筛查 | 通常正常 |
| **C反应蛋白(CRP)** | 炎症筛查 | 通常正常或轻度升高 |
| **血糖/HbA1c** | 排除糖尿病 | 正常 |
| **甲状腺功能(TSH)** | 排除甲状腺疾病 | 正常(可能处于正常低限) |
| **肝肾功能** | 排除器官功能异常 | 通常正常 |
| **电解质** | 排除电解质紊乱 | 正常 |
| **尿常规** | 筛查泌尿系统疾病 | 正常 |

#### 5.1.2 根据临床情况考虑的检查

| 检查 | 适应情况 |
|------|---------|
| 铁蛋白/铁代谢 | 疲劳，尤其女性 |
| 维生素D | 疲劳、肌肉痛 |
| 维生素B12/叶酸 | 神经症状、贫血 |
| 抗核抗体(ANA) | 关节痛、皮疹 |
| 类风湿因子 | 关节症状 |
| 皮质醇(晨起) | 疲劳、低血压 |
| 乳糜泻筛查 | 胃肠症状 |
| HIV筛查 | 风险因素存在 |
| 单核细胞增多症筛查 | 急性感染后发病 |

### 5.2 专科检查 (根据症状)

#### 5.2.1 心血管检查

| 检查 | 适应情况 | 目的 |
|------|---------|------|
| 心电图 | 心悸、晕厥 | 排除心律失常 |
| Holter | 心悸频繁 | 评估心律 |
| 超声心动图 | 心脏症状 | 排除结构性心脏病 |
| 倾斜试验 | 直立不耐受明显 | 诊断POTS/NMH |

#### 5.2.2 神经科检查

| 检查 | 适应情况 | 目的 |
|------|---------|------|
| 头部MRI | 头痛、神经症状 | 排除结构性病变 |
| 睡眠多导图(PSG) | 睡眠障碍明显 | 排除睡眠呼吸暂停等 |
| 神经传导/肌电图 | 明显肢体麻木无力 | 排除周围神经病 |

### 5.3 研究性生物标志物

#### 5.3.1 潜在生物标志物

| 标志物 | 研究状态 | 临床可用性 |
|--------|---------|-----------|
| NK细胞功能 | 多项研究支持下降 | 研究用，非常规 |
| 细胞因子谱 | 变化报告，但不一致 | 研究用 |
| 代谢组学特征 | Naviaux等研究有发现 | 研究阶段 |
| 自身抗体 | β2AR抗体等发现 | 研究阶段 |
| 脑脊液标志物 | 部分研究有发现 | 侵入性，研究用 |
| 红细胞变形性 | 部分研究异常 | 研究阶段 |
| 微生物组 | 菌群改变报告 | 研究阶段 |

#### 5.3.2 影像学研究标志物

| 技术 | 研究发现 | 临床应用 |
|------|---------|---------|
| fMRI | 功能连接异常 | 研究用 |
| PET(TSPO) | 小胶质细胞激活 | 研究用 |
| MR光谱 | 代谢物异常 | 研究用 |
| 灌注成像 | 脑血流改变 | 研究阶段 |

**注意：** 目前尚无可用于临床诊断的确定性生物标志物

## 6. 鉴别诊断 (Differential Diagnosis)

### 6.1 必须排除的疾病

#### 6.1.1 按系统分类

| 系统 | 需排除疾病 | 关键鉴别点 |
|------|-----------|-----------|
| **内分泌** | 甲状腺功能减退 | TSH↑，T4↓ |
| | 甲状腺功能亢进 | TSH↓，T4↑ |
| | 肾上腺功能不全 | 低皮质醇，电解质异常 |
| | 糖尿病 | 血糖、HbA1c异常 |
| **风湿免疫** | 系统性红斑狼疮 | ANA+，多系统表现 |
| | 类风湿关节炎 | 关节肿胀，RF+ |
| | 干燥综合征 | 干眼干口，抗体+ |
| | 炎性肌病 | CK升高，肌无力 |
| **感染** | 慢性活动性EBV | 病毒载量持续高 |
| | 莱姆病 | 流行病学史，抗体+ |
| | HIV/AIDS | HIV抗体+ |
| | 慢性肝炎 | 肝炎病毒标志物+ |
| **血液** | 贫血 | Hb↓ |
| | 恶性肿瘤 | 影像学、活检 |
| **睡眠** | 睡眠呼吸暂停 | PSG确诊 |
| | 发作性睡病 | PSG/MSLT确诊 |
| **神经** | 多发性硬化 | MRI特征性改变 |
| | 重症肌无力 | 抗体+，肌无力模式 |
| | 帕金森病 | 运动症状，神经科检查 |
| **心血管** | 心力衰竭 | 超声心动图，BNP |
| **精神** | 重性抑郁障碍 | 见下详述 |
| **其他** | 慢性肾病 | 肾功能异常 |
| | 肝硬化 | 肝功能、影像 |

### 6.2 ME/CFS与重性抑郁障碍鉴别

| 特征 | ME/CFS | 重性抑郁障碍 |
|------|--------|-------------|
| **对活动的态度** | 想做但做不了 | 不想做(兴趣缺乏) |
| **PEM** | 存在，核心特征 | 不存在 |
| **活动后** | 症状恶化 | 可能改善情绪 |
| **认知症状** | 脑雾，可随疲劳波动 | 负性思维，相对稳定 |
| **躯体症状** | 多系统、特异性(POTS等) | 非特异性躯体化 |
| **发病** | 常有感染等触发 | 可有心理社会因素 |
| **睡眠** | 非恢复性睡眠 | 早醒更常见 |
| **治疗反应** | 抗抑郁药不改善核心疲劳 | 抗抑郁药有效 |

**重要：** ME/CFS与抑郁可共病，需同时评估和治疗

### 6.3 ME/CFS与纤维肌痛鉴别

| 特征 | ME/CFS | 纤维肌痛 |
|------|--------|---------|
| **主诉** | 疲劳为主 | 广泛疼痛为主 |
| **PEM** | 核心特征 | 可存在但非核心 |
| **压痛点** | 可存在 | 特征性18个压痛点 |
| **疼痛分布** | 可有但非主要 | 广泛、多部位 |
| **重叠** | 30-70%共病 | 30-70%共病 |

**临床意义：** 两者高度重叠，可能为同一谱系疾病，治疗策略有相似之处

### 6.4 可能共存的疾病

| 疾病 | 共病率 | 关系 |
|------|--------|------|
| 纤维肌痛 | 30-70% | 重叠谱系 |
| POTS | 30-70% | 自主神经共病 |
| 肠易激综合征 | 35-92% | 肠脑轴共病 |
| 偏头痛 | 高 | 共同机制 |
| 化学物质敏感 | 未明确 | 共病报告增加 |
| 肥大细胞活化综合征 | 未明确 | 新兴关注 |
| 埃勒斯-当洛斯综合征(hEDS) | 未明确 | 共病报告增加 |

**这些疾病不排除ME/CFS诊断，可同时存在**

### 7.2 诊断延迟的原因与解决

| 延迟原因 | 解决方案 |
|---------|---------|
| 医生不熟悉ME/CFS | 教育与培训 |
| 认为是心理问题 | 基于循证的认识更新 |
| 缺乏确定性检测 | 临床诊断能力培养 |
| 症状非特异 | 系统性评估流程 |
| 多科就诊 | 统一诊疗路径 |

### 7.3 诊断要点总结

**支持ME/CFS诊断的特征：**
- ✓ 明确的发病点(尤其感染后)
- ✓ PEM存在且典型
- ✓ 非恢复性睡眠
- ✓ 认知障碍("脑雾")
- ✓ 直立不耐受(尤其POTS)
- ✓ 多系统症状
- ✓ 波动性病程
- ✓ 常规检查正常或轻度异常

**不支持ME/CFS诊断的特征：**
- ✗ 实验室检查有明确异常可解释症状
- ✗ 无PEM(活动后症状不恶化)
- ✗ 持续发热(>38°C)
- ✗ 局灶性神经体征
- ✗ 体重持续下降
- ✗ 淋巴结肿大(非压痛)
- ✗ 进行性恶化无波动

## 8. 严重度评估与分级 (Severity Assessment)

### 8.1 严重度分级标准

| 级别 | 功能水平 | Bell评分 | 典型表现 |
|------|---------|---------|---------|
| **轻度** | 活动能力下降~50%，可部分工作/学习 | 50-70 | 需要大量休息；可能需要减少工作时间；停止非必要活动 |
| **中度** | 大部分时间居家，无法工作 | 30-50 | 活动能力严重受限；大部分时间在家；可能仍能独立自理 |
| **重度** | 大部分时间卧床，自理困难 | 10-30 | 大部分时间卧床；需要轮椅；日常活动需帮助 |
| **极重度** | 完全卧床，完全依赖照护 | 0-10 | 完全卧床；可能需管饲；极度光敏声敏；无法交流 |

### 8.2 功能评估维度

| 维度 | 评估内容 |
|------|---------|
| **日常生活活动(ADL)** | 洗浴、穿衣、进食、如厕、转移 |
| **工具性日常生活活动(IADL)** | 购物、做饭、家务、财务管理 |
| **社会参与** | 社交活动、外出、工作/学习 |
| **认知功能** | 阅读、交流、决策能力 |
| **活动耐量** | 可耐受的活动类型和时长 |

### 8.3 监测与随访

| 监测内容 | 频率 | 方法 |
|---------|------|------|
| 症状严重度 | 每次随访 | DSQ或类似量表 |
| 功能水平 | 定期 | Bell评分、SF-36 |
| PEM模式 | 持续 | 活动日志 |
| 共病状态 | 定期 | 针对性评估 |
| 治疗反应 | 持续 | 症状和功能变化 |

## 9. 特殊情况诊断 (Special Diagnostic Situations)

### 9.1 儿童与青少年

| 方面 | 特殊考虑 |
|------|---------|
| **诊断标准** | 可使用成人标准，时间可缩短(≥3个月) |
| **POTS标准** | HR增加≥40bpm(而非成人30bpm) |
| **表现特点** | 学校出勤困难、学业下降、社交减少 |
| **鉴别诊断** | 排除青少年特有疾病 |
| **诊断挑战** | 可能被误解为"逃避"或心理问题 |

### 9.2 老年患者

| 方面 | 特殊考虑 |
|------|---------|
| **鉴别诊断** | 需排除更多年龄相关疾病 |
| **共病** | 更多慢性病共存 |
| **症状归因** | 避免将症状归因于"年龄" |
| **功能评估** | 考虑基础功能水平 |

### 9.3 Long COVID与ME/CFS

| 方面 | 考虑要点 |
|------|---------|
| **诊断时机** | COVID-19感染后≥3个月 |
| **重叠程度** | 30-50% Long COVID符合ME/CFS标准 |
| **区分** | Long COVID可能有更多呼吸道症状 |
| **管理** | 类似原则，活动管理为基础 |

## 10. 专家诊断建议 (Expert Diagnostic Recommendations)

### 10.1 诊断原则

> **"ME/CFS是临床诊断。没有必要等待所有检查完成才开始治疗。早期识别和活动管理可以改善预后。"**
> — NICE 2021

> **"PEM是ME/CFS的标志性特征。仔细评估PEM的存在、特征和严重度是诊断的关键。"**
> — IOM 2015

### 10.2 实践建议

| 建议 | 原因 |
|------|------|
| 认真倾听患者 | 患者是疾病体验的专家 |
| 不要等待排除所有疾病 | 基本检查正常即可开始管理 |
| 尽早诊断 | 早期活动管理可预防恶化 |
| 详细评估PEM | 是诊断的关键 |
| 考虑直立不耐受 | 常被忽视但重要 |
| 评估严重度 | 指导管理强度 |
| 识别共病 | 改善治疗效果 |
| 避免有害干预 | GET可能造成损害 |

### 10.3 诊断后的下一步

1. **教育患者和家属** — 关于疾病性质、预期、管理原则
2. **评估严重度** — 指导管理计划
3. **识别共病** — 纤维肌痛、POTS、IBS等
4. **开始活动管理** — Pacing是基础
5. **症状管理** — 针对主要症状
6. **定期随访** — 监测变化、调整计划
7. **心理支持** — 处理疾病带来的心理影响
8. **社会支持** — 工作调整、福利申请等
