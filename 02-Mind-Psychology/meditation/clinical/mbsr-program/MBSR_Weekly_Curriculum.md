# MBSR 八周课程详细设计 | MBSR Weekly Curriculum

> **领域**：正念减压（Mindfulness-Based Stress Reduction）
> **关键词**：课程设计（Curriculum Design）、每周主题（Weekly Themes）、教学大纲（Syllabus）、家庭作业（Home Practice）、教学指引（Teaching Guide）
> **上次更新**：2026-04

---

## 导向会（Orientation Session, Week 0）

| 项目 | 详情 |
|------|------|
| **时长** | 2-2.5小时 |
| **目的** | 介绍MBSR、建立期望、筛查评估、知情同意 |

**内容清单**：
1. MBSR项目简介与历史
2. 什么是正念（体验式介绍）
3. 课程结构与时间承诺说明
4. 家庭练习要求（每日45分钟）
5. 参与者自我介绍与参加动机
6. 知情同意与筛查
7. 前测评估（问卷）

---

## 第1周：觉察与自动导航

| 项目 | 详情 |
|------|------|
| **主题** | There is More Right with You Than Wrong（你身上的对比错多得多） |
| **核心概念** | 自动导航（Autopilot）、正念的操作定义、初心（Beginner's Mind） |

**课堂流程（2.5小时）**

| 时间 | 活动 | 说明 |
|------|------|------|
| 0:00-0:20 | 欢迎与签到 | 简短自我介绍、参加动机 |
| 0:20-0:40 | **葡萄干练习**（Raisin Exercise） | 正念饮食体验练习 |
| 0:40-1:00 | 探询：你注意到了什么？ | 引出正念的核心特征 |
| 1:00-1:15 | 正念定义与核心态度 | 七种态度介绍 |
| 1:15-1:20 | 休息 | — |
| 1:20-1:55 | **身体扫描引导练习**（35min） | 完整身体扫描体验 |
| 1:55-2:20 | 身体扫描探询 + 小组讨论 | 体验分享 |
| 2:20-2:30 | 家庭作业说明 | 发放音频/资料 |

**家庭作业**：
- [ ] 每日身体扫描（45分钟，跟随音频）
- [ ] 正念饮食（至少1餐/天）
- [ ] 九点连线练习（觉察思维定势）

---

## 第2周：感知与创造性回应

| 项目 | 详情 |
|------|------|
| **主题** | How We Perceive（我们如何感知世界） |
| **核心概念** | 感知（Perception）、反应模式（Reactive Patterns）、觉察选择点 |

**课堂流程（2.5小时）**

| 时间 | 活动 |
|------|------|
| 0:00-0:35 | 身体扫描（35min）+ 探询 |
| 0:35-1:00 | 家庭练习回顾：障碍、发现、问题 |
| 1:00-1:15 | 感知练习（视觉错觉/翻转图） |
| 1:15-1:20 | 休息 |
| 1:20-1:50 | **坐禅入门**（呼吸觉察，20min）+ 探询 |
| 1:50-2:20 | 压力与感知：我们的"镜片"如何染色体验 |
| 2:20-2:30 | 家庭作业说明 |

**家庭作业**：
- [ ] 交替练习：身体扫描（隔日）+ 坐禅10-20分钟（隔日）
- [ ] **愉悦事件日志**（每天记录1个愉悦时刻）
- [ ] 选择一项常规活动做正念练习（刷牙/走路/洗碗）

---

## 第3周：身体中的正念

| 项目 | 详情 |
|------|------|
| **主题** | There is a Pleasure in the Pathless Woods（无路之林中有欢乐） |
| **核心概念** | 身体智慧（Body Wisdom）、温和的边界（Gentle Edges）、一心多用vs正念 |

**课堂流程（2.5小时）**

| 时间 | 活动 |
|------|------|
| 0:00-0:10 | 正念运动热身 |
| 0:10-0:45 | **正念瑜伽**（Mindful Yoga, 35min） |
| 0:45-1:10 | 探询：身体、边界、自我关怀 |
| 1:10-1:15 | 休息 |
| 1:15-1:40 | **坐禅**（呼吸+身体觉察，25min） |
| 1:40-2:10 | 愉悦事件日志分享 + 规律觉察 |
| 2:10-2:30 | 讨论：愉悦与当下的关系；家庭作业 |

**家庭作业**：
- [ ] 交替练习：正念瑜伽 + 坐禅（各隔日，每次30-45分钟）
- [ ] **不愉悦事件日志**（每天记录1个不愉悦时刻）
- [ ] 觉察自动反应的时刻

---

## 第4周：压力——反应还是回应？

| 项目 | 详情 |
|------|------|
| **主题** | Responding vs. Reacting to Stress（回应vs反应） |
| **核心概念** | 压力生理学（Stress Physiology）、战或逃反应、STOP练习 |

**课堂流程（2.5小时）**

| 时间 | 活动 |
|------|------|
| 0:00-0:35 | **坐禅**（呼吸+身体+声音，35min） |
| 0:35-0:55 | 探询 + 家庭练习回顾 |
| 0:55-1:15 | 不愉悦事件日志分享 + 反应性讨论 |
| 1:15-1:20 | 休息 |
| 1:20-1:50 | **压力教学**：压力反应生理学 + 视频/图示 |
| 1:50-2:05 | **经行（Walking Meditation, 15min）** |
| 2:05-2:20 | STOP练习介绍 + 练习 |
| 2:20-2:30 | 家庭作业：全日静修准备 |

**家庭作业**：
- [ ] 交替练习：坐禅 + 正念瑜伽（每次45分钟）
- [ ] **压力反应觉察日志**
- [ ] STOP练习：每天至少3次
- [ ] 准备全日静修（着装、饮食、心理准备）

---

## 全日静修（All-Day Silent Retreat）

| 项目 | 详情 |
|------|------|
| **时间** | 第4-5周之间的周末，6-7小时 |
| **形式** | 全程静默（Noble Silence） |
| **目的** | 深化练习、体验持续正念、培养内在静默 |

**全日流程**

| 时间 | 活动 | 时长 |
|------|------|------|
| 9:00 | 到达、安顿 | 15min |
| 9:15 | 引导坐禅（呼吸+身体） | 30min |
| 9:45 | 正念瑜伽 | 45min |
| 10:30 | 经行 | 20min |
| 10:50 | 坐禅（声音+念头） | 30min |
| 11:20 | 身体扫描 | 45min |
| 12:05 | **正念午餐**（静默进食） | 45min |
| 12:50 | 自由正念时间（坐/走/立） | 30min |
| 13:20 | **慈心禅引导** | 30min |
| 13:50 | 坐禅（开放觉察） | 25min |
| 14:15 | 经行 | 15min |
| 14:30 | 结束坐禅 | 15min |
| 14:45 | **打破静默** + 分享 | 30min |
| 15:15 | 回顾与闭幕 | 15min |

---

## 第5周：压力应对——以正念回应

| 项目 | 详情 |
|------|------|
| **主题** | Finding the Eye of the Hurricane（找到飓风之眼） |
| **核心概念** | 困难情绪的正念（Mindfulness of Difficult Emotions）、RAIN模型 |

**课堂流程（2.5小时）**

| 时间 | 活动 |
|------|------|
| 0:00-0:40 | **坐禅**（含困难情绪邀请，40min） |
| 0:40-1:10 | 探询 + 全日静修体验分享 |
| 1:10-1:15 | 休息 |
| 1:15-1:45 | **RAIN模型教学**：Recognize-Allow-Investigate-Non-identification |
| 1:45-2:10 | 小组练习：选择一个困难情境用RAIN探索 |
| 2:10-2:30 | 大组分享 + 家庭作业 |

**家庭作业**：
- [ ] 每日坐禅（30-45分钟，含困难情绪探索）
- [ ] RAIN练习应用于日常困难时刻
- [ ] **沟通觉察日志**（为下周准备）

---

## 第6周：正念沟通

| 项目 | 详情 |
|------|------|
| **主题** | Mindful Communication（正念沟通） |
| **核心概念** | 人际压力、习惯性沟通模式、正念倾听、慈心 |

**课堂流程（2.5小时）**

| 时间 | 活动 |
|------|------|
| 0:00-0:30 | **坐禅**（声音+念头，30min） |
| 0:30-0:50 | 家庭练习回顾 |
| 0:50-1:10 | **慈心禅引导**（20min） |
| 1:10-1:15 | 休息 |
| 1:15-1:50 | **正念沟通练习**（双人组） |
| 1:50-2:15 | 人际压力讨论：Aikido隐喻——迎接而非抵抗 |
| 2:15-2:30 | 反馈 + 家庭作业 |

**家庭作业**：
- [ ] 自选每日练习组合（坐禅/瑜伽/身体扫描，45分钟）
- [ ] 慈心禅练习（隔日）
- [ ] 正念沟通练习（每天至少1次觉察性对话）

---

## 第7周：自我照顾与生活方式

| 项目 | 详情 |
|------|------|
| **主题** | Taking Care of Yourself（照顾自己） |
| **核心概念** | 正念生活方式、滋养vs消耗活动、个人化练习计划 |

**课堂流程（2.5小时）**

| 时间 | 活动 |
|------|------|
| 0:00-0:35 | **自选练习**（教师引导多种选项，学员选择） |
| 0:35-0:55 | 探询：你如何选择？什么影响了你的选择？ |
| 0:55-1:15 | 滋养vs消耗活动清单 |
| 1:15-1:20 | 休息 |
| 1:20-1:50 | 山的冥想 / 湖的冥想（意象冥想） |
| 1:50-2:15 | 个人化练习计划设计 |
| 2:15-2:30 | 为最后一周做准备 |

**家庭作业**：
- [ ] **不使用引导音频**的独立练习（45分钟/天）
- [ ] 实施个人化练习计划
- [ ] 反思：8周以来发生了什么变化？

---

## 第8周：继续前行

| 项目 | 详情 |
|------|------|
| **主题** | Keeping Up Your Momentum（保持动力） |
| **核心概念** | 整合、长期练习承诺、资源网络、告别 |

**课堂流程（2.5小时）**

| 时间 | 活动 |
|------|------|
| 0:00-0:40 | **完整身体扫描**（40min，呼应第1周） |
| 0:40-1:00 | 探询：与第1周的身体扫描有什么不同？ |
| 1:00-1:15 | 休息 |
| 1:15-1:45 | 8周回顾：每周主题串联、个人变化 |
| 1:45-2:00 | 长期练习资源介绍（App/社群/闭关/进阶课程） |
| 2:00-2:20 | **告别仪式**：每人一句话分享 |
| 2:20-2:30 | 后测评估 + 结业 |
