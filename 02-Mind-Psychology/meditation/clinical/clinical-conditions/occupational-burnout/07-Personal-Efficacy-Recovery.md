# 07. 个人效能感恢复 | Personal Efficacy Recovery

> **文档定位**：低个人成就感是倦怠的"认知后果"——长期耗竭和去人格化最终导致"我做什么都没用"的自我否定。本章提供从自我怀疑到自我效能的正念重建路径。

---

## 7.1 低效能感的心理机制

### 7.1.1 习得性无助的工作版本

```
反复的努力 → 未获得预期回报（认可、晋升、成就感）
    ↓
归因模式改变："不是我的努力不够，而是我这个人不行"
    ↓
泛化："不仅这件事我做不好，所有事我都做不好"
    ↓
行为退缩：停止尝试 → 确实没有成果 → 强化"我不行"的信念
    ↓
习得性无助
```

**正念的切入点**：
正念不直接改变外部环境或结果，但它可以改变**归因模式**——从"我是失败者"（稳定、整体、内部归因）转向"我注意到有一个'我是失败者'的想法"（去中心化），并基于价值而非结果行动。

## 7.2 优势识别与正念

### 7.2.1 优势觉察日志

倦怠者的大脑被负面信息垄断，完全看不到自己的能力和贡献。此练习强制激活"优势回路"。

```markdown
【每日优势记录】

日期：________

1. 今天我做了一件小事，体现了我的什么品质/能力？
   （例：耐心回复了一封棘手的邮件 → 耐心、专业）
   ___________________________________________________________

2. 今天有人对我表达了感谢或认可（无论多小）？
   （例：同事说"谢谢你帮我"；客户没有投诉）
   ___________________________________________________________

3. 今天我克服了一个困难（无论多小）？
   （例：虽然很累，还是完成了报告；忍住没有发脾气）
   ___________________________________________________________

4. 如果我的朋友面临我今天的处境，我会如何评价TA的表现？
   （例："TA已经很努力了""TA处理得很专业"）
   ___________________________________________________________

【规则】
- 不评判内容大小（"只是日常工作"也是成就）
- 允许重复（耐心可以每天都写）
- 若完全写不出，使用"最低标准"："我今天来上班了""我今天呼吸了"
```

### 7.2.2 身体优势锚定

```markdown
【操作步骤】

1. 回忆一次你感到"胜任""被认可""做得好"的经历。
   不需要是惊天动地的大事——可以是"成功修好了一个打印机"
   或"一个客户对我说谢谢"。

2. 闭上眼睛，尽可能生动地回忆那个场景：
   - 你在哪里？
   - 周围有什么？
   - 你做了什么？
   - 别人说了什么？

3. 特别关注**身体感受**：
   - 当时你的姿势是怎样的？（挺直？放松？）
   - 你的呼吸是怎样的？（深？浅？）
   - 你的面部表情是怎样的？（微笑？平静？）
   - 你身体的哪个部位感到最"对劲"？

4. 现在，有意识地重现这个身体姿态：
   - 调整坐姿或站姿
   - 调整呼吸
   - 做一个微小的表情（如轻轻微笑）

5. 对自己说：
   "这个身体知道如何感到胜任。
    我曾经做到过，我的身体记得。
    我可以再次访问这个状态。"

【日常应用】
在需要效能感的时刻（如会议前、汇报前），先做这个身体锚定。
```

## 7.3 小胜 savoring

### 7.3.1 为什么要 savoring？

抑郁和倦怠的共同特征是"快感缺失"（anhedonia）——无法体验愉悦。不是因为没有愉悦事件，而是因为大脑不再对它们反应。

**Savoring（品味）** 是有意识地延长和深化积极体验的过程。Fredrickson（2001）的"积极情绪拓展-建构理论"指出，积极情绪的累积效应可拓展认知范围、建构持久资源。

### 7.3.2 工作场景 savoring 练习

```markdown
【任务完成 savoring】

每完成一个任务（无论大小）：

1. 停顿 3-5 秒，不立即跳到下一个任务。
2. 感受"完成"的身体信号：
   - 肩膀微微下沉
   - 一次稍微深一点的呼气
   - 面部肌肉放松
3. 内心说一句：
   - "完成了。"
   - "我做了这件事。"
   - "这是一个小胜利。"
4. 若愿意，做一个微小的庆祝动作：
   - 轻轻握拳（"Yes"）
   - 微笑
   - 喝一口茶/水
   - 在清单上打勾（感受笔尖划过纸面的触感）

【会议/互动 savoring】

一次困难的会议或互动结束后：

1. 问自己："这次互动中，有一个我处理得还不错的瞬间是什么？"
   （即使整体很糟，也找一个瞬间）
2. 感受那个瞬间的身体记忆。
3. 对自己说："在那个瞬间，我是胜任的。"
```

## 7.4 成长型思维正念

### 7.4.1 从"固定型"到"成长型"

Carol Dweck 的成长型思维（Growth Mindset）指出：相信能力可以通过努力发展的人，比相信能力固定的人更能应对挑战。

倦怠者常陷入固定型思维："我就是不擅长这个""我没有这方面的天赋"。

### 7.4.2 正念化的成长型语句

| 固定型（倦怠思维） | 成长型（正念重构） |
|------------------|------------------|
| "我失败了。" | "我注意到这个结果不符合我的期待。这是一个学习的数据点。" |
| "我不擅长公开演讲。" | "我注意到公开演讲让我紧张。每一次练习，我的神经系统都在适应。" |
| "这个反馈说明我很差。" | "我收到了反馈。反馈是关于我的行为，不是关于我的全部。" |
| "我永远改不了。" | "改变需要时间。我的大脑有神经可塑性——科学已经证明了这一点。" |
| "别人天生就比我强。" | "每个人都在自己的旅程上。我只和自己昨天比较。" |

### 7.4.3 正念化的成长型练习

```markdown
【每日一句】

每天早晨，选择一句成长型语句，写在便签上，贴在电脑边。

在这一天中，每当感到自我怀疑时：
1. 看一眼便签。
2. 做 1 次呼吸。
3. 对自己说这句话——不是说给自己"听"，而是说给自己"体验"。

【每周回顾】

周末回顾：
- 这周哪句成长型语句对我最有用？
- 这周我有一个"差点放弃但坚持了下来"的时刻吗？
- 下周我想练习哪一句？
```

## 7.5 价值澄清与承诺行动

### 7.5.1 工作价值评估

ACT 框架下的价值澄清，帮助倦怠者重新连接"为什么选择这份工作"。

```markdown
【工作价值评估表】

在以下每个领域，用1-10分评估你目前的工作与该价值的匹配度：

| 价值领域 | 我的定义 | 现状(1-10) | 理想(1-10) | 差距 |
|---------|---------|-----------|-----------|------|
| 帮助他人/服务 | | | | |
| 创造/创新 | | | | |
| 学习/成长 | | | | |
| 联结/合作 | | | | |
| 公平/正义 | | | | |
| 自主/独立 | | | | |
| 稳定/安全 | | | | |
| 认可/成就 | | | | |

差距最大的 1-2 个领域 = 倦怠的核心来源。
差距最小的 1-2 个领域 = 可利用的"意义锚点"。
```

### 7.5.2 微承诺行动

```markdown
【操作框架】

选择一个价值领域，设定一个极小的、本周可执行的下一步：

- 帮助他人：本周主动询问一位同事"有什么我可以帮忙的？"
- 创造：本周花15分钟尝试一个新方法/工具。
- 联结：本周与一位同事进行5分钟非工作对话。
- 学习：本周读一篇与专业相关的短文章。
- 公平：本周为一位受到不公平对待的同事说一句话。

关键原则：
- 行动与情绪解耦——不是"等我感觉好了再做"，而是"即使感到无力，我仍然可以做一点点"。
- 微小但朝向价值——5分钟的行动 > 5小时的计划。
- 庆祝完成——倦怠会贬低任何成就。有意识地肯定自己。
```
