## 12.1 评估量表

### 12.1.1 单条目倦怠筛查

```markdown
【简化倦怠自评】

请用1-10分评估以下三个维度：

1. 能量耗竭："我感到 emotionally drained by my work"
   1-10分：____

2. 心理距离："我感到 cynical about my work"
   1-10分：____

3. 职业效能："我感到 I am effective at my work"
   1-10分：____

【解读】
- 任一项 ≥7分：提示需要关注和干预
- 两项 ≥7分：提示中度倦怠，建议寻求专业支持
- 三项 ≥7分：提示重度倦怠，建议全面评估和休假
```

## 12.3 脚本与引导语

### 12.3.1 3次呼吸重置（30秒版）

```markdown
【自助脚本】

停下手中事。
鼻吸——3秒——感受空气进入。
呼气——5秒——感受肩膀下沉。
（重复3次）

对自己说："我回来了。"
```

### 12.3.2 下班边界仪式（5分钟版）

```markdown
【自助脚本】

1. 坐在工作区域，双手放桌面。
2. 3次深慢呼吸。
3. 回顾今天完成的一件事（无论大小）。
4. 写下未完成的，合上笔记本。
5. 设定意图："现在，我离开工作模式。接下来的时间属于恢复。"
6. 做一个"关门"动作（关电脑/整理桌面/双手合十）。
7. 起身，有意识地迈向下一段生活。
```

### 12.3.3 慈心冥想简化版（5分钟）

```markdown
【自助脚本】

手放心口。

对自己："愿我平安。愿我对自身温和。"
对恩人："愿你平安。愿你对自身温和。"
对困难者："愿你平安。愿你对自身温和。"
对一切众生："愿所有生命平安。愿所有生命对自身温和。"

（每句重复2-3次，感受词语背后的意图）
```

## 12.5 组织与家庭工具

### 12.5.1 团队正念微会议（10分钟版）

```markdown
【会议流程】

1. 开场呼吸（2分钟）
   全员闭眼，3次共同呼吸。

2. 状态检查（3分钟）
   每人用1个词描述今天的状态（轮流或自愿）。
   不解释、不讨论、不评判。

3. 意图设定（2分钟）
   主持人或自愿者提出今日团队意图：
   "今天我们选择______（如：专注、协作、耐心）。"

4. 共同祝福（2分钟）
   "愿我们今天的努力对自己和他人都有益。"
   全员睁眼，开始工作。

【频率建议】每日晨会或每周一/五
```
