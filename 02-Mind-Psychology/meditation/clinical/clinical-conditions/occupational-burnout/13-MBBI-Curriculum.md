# 13. 职业倦怠正念干预课程方案（MBBI）| MBBI 8-Week Curriculum

> **文档定位**：本章提供一套标准化的 8 周职业倦怠正念干预课程方案（Mindfulness-Based Burnout Intervention, MBBI），可直接用于临床培训、EAP项目或研究。

---

## 13.1 MBBI 设计原则

| 原则 | 说明 |
|------|------|
| **针对性** | 不直接套用 MBSR/MBCT，而是针对倦怠三维度（EE/DP/PA）设计专属技术 |
| **工作嵌入** | 所有练习均可嵌入工作日，不依赖大块空闲时间 |
| **渐进性** | 从微恢复（1-5分钟）→ 正式练习（15-20分钟）→ 深度恢复（ retreat） |
| **组织协同** | 课程包含组织层面的建议，不仅聚焦个体 |
| **循证基础** | 整合 MBSR、ACT、MSC 的循证技术 |

---

## 13.2 八周课程大纲

### Week 1：自动导航与能量觉察

| 元素 | 内容 |
|------|------|
| **主题** | 识别工作中的"自动导航"模式；建立能量觉察基础 |
| **核心练习** | 葡萄干练习（工作版：用一杯茶/咖啡替代）；每日能量审计启动 |
| **微练习** | 3次呼吸重置；每日3次 STOP |
| **作业** | 连续3天记录"能量高峰与低谷"时刻 |
| **组织建议** | 管理层发送支持邮件；允许弹性工时 |

### Week 2：身体信号与边界设定

| 元素 | 内容 |
|------|------|
| **主题** | 学习识别身体的疲劳信号；建立工作-生活边界 |
| **核心练习** | 身体扫描缩短版（20分钟）；下班边界仪式 |
| **微练习** | 每小时1分钟脚底接地；肩颈释放 |
| **作业** | 每日执行1次下班边界仪式；记录身体信号 |
| **组织建议** | 推行"无会议时段"；鼓励午间离开工位 |

### Week 3：情绪耗竭与呼吸调控

| 元素 | 内容 |
|------|------|
| **主题** | 理解情绪耗竭的机制；学习呼吸作为能量调节工具 |
| **核心练习** | 共振呼吸（10分钟）；4-7-8呼吸 |
| **微练习** | 午后共振呼吸5分钟；箱式呼吸（高压前） |
| **作业** | 每日1次正式呼吸练习；记录呼吸后的能量变化 |
| **组织建议** | 设置安静室/冥想角；提供呼吸练习音频 |

### Week 4：去人格化与关系正念

| 元素 | 内容 |
|------|------|
| **主题** | 识别去人格化信号；重建与人的连接 |
| **核心练习** | 慈心冥想（从恩人开始）；目光接触正念 |
| **微练习** | 会议前1次意图设定；困难对话前的慈心祝福 |
| **作业** | 对3位同事/客户进行"共同 humanity 觉察" |
| **组织建议** | 团队建设活动；非工作性质的社交时间 |

### Week 5：犬儒主义与意义重建

| 元素 | 内容 |
|------|------|
| **主题** | 转化犬儒态度；重建工作意义 |
| **核心练习** | 价值澄清；意义微连接；工作叙事重写 |
| **微练习** | 每完成一个任务，进行 savoring（5秒） |
| **作业** | 书写"我的工作价值声明"；识别3个小意义 |
| **组织建议** | 分享客户/用户正面反馈；连接工作与更大价值 |

### Week 6：效能感与成长型思维

| 元素 | 内容 |
|------|------|
| **主题** | 恢复个人成就感；建立成长型思维 |
| **核心练习** | 优势觉察日志；身体优势锚定；小胜 savoring |
| **微练习** | 每日记录1个优势；成长型语句替换 |
| **作业** | 完成"我的优势档案"；设定1个微承诺行动 |
| **组织建议** | 管理者提供具体、及时的认可反馈 |

### Week 7：整合与系统优化

| 元素 | 内容 |
|------|------|
| **主题** | 整合个人工具；优化工作系统（Job Crafting） |
| **核心练习** | 任务/关系/认知三维 Job Crafting；能量预算 |
| **微练习** | 每日能量审计；评估任务给予/消耗比 |
| **作业** | 制定个人"工作再设计"计划；与上级沟通1个调整 |
| **组织建议** | 支持弹性工作；工作再设计对话 |

### Week 8：维持与复发预防

| 元素 | 内容 |
|------|------|
| **主题** | 建立长期维持计划；识别复发信号 |
| **核心练习** | 个人正念工具包制作；复发预警信号识别；行动计划卡 |
| **微练习** | 自选组合：每日至少2个微练习 |
| **作业** | 完成"维持计划书"；预约3个月随访 |
| **组织建议** | 建立同伴支持小组；季度倦怠筛查 |

## 13.3 课程实施规范

### 13.3.1 带领者要求

| 要求 | 说明 |
|------|------|
| **资质** | 完成MBSR/MBCT教师培训；或具备临床心理学/精神科背景 + 正念培训 |
| **经验** | 至少带领过1轮完整的MBSR或MBCT |
| **督导** | 新手需在督导下带领；建议每2周督导1次 |
| **个人练习** | 每日正念练习 + 至少1次 retreat 经验 |

### 13.3.2 团体设置

| 参数 | 建议 |
|------|------|
| **人数** | 8-15人 |
| **时长** | 每周1次，2小时 |
| **总周期** | 8周 |
| **地点** | 安静、私密、有自然光的房间；或高质量视频会议 |
| **材料** | 音频引导、工作手册、练习日志、评估量表 |

### 13.3.3 评估方案

| 时间点 | 评估内容 |
|--------|---------|
| **入组前** | MBI-GS + PHQ-9 + GAD-7 + FFMQ + SCS |
| **第4周（中点）** | MBI-GS + PHQ-9 |
| **第8周（结束）** | MBI-GS + PHQ-9 + GAD-7 + FFMQ + SCS + 满意度 |
| **3个月随访** | MBI-GS + PHQ-9 + 练习频率日志 |
| **6个月随访** | MBI-GS + PHQ-9 + 维持计划执行情况 |
