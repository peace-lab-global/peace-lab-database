## 7. 临床实施框架

### 7.1 分层干预模型

```
急性期（重度抑郁发作）
    └── 优先药物治疗和/或CBT
    └── 不建议引入正式正念训练（注意力困难；可能加重自责）
    
缓解期（症状减轻，功能恢复）
    └── 引入MBCT预防复发
    └── 可同时进行维持治疗
    
维持期（无症状或残留症状）
    └── MBCT作为核心维持策略
    └── 日常正念练习
    
高风险期（识别早期预警信号）
    └── 加强练习频率
    └── 3分钟呼吸空间作为"急救"
```

### 7.2 筛查与评估

| 评估内容 | 工具 | 目的 |
|---------|------|------|
| **抑郁严重程度** | PHQ-9, BDI-II, MADRS | 确定是否适合正念干预 |
| **反刍倾向** | RRS（Ruminative Responses Scale） | 识别高反刍个体（MBCT特别受益） |
| **自杀风险** | C-SSRS, 临床访谈 | 安全筛查 |
| **认知功能** | 简要认知筛查 | 评估是否能参与课程 |
| **既往发作史** | 临床访谈 | 确定MBCT适应症 |

**入组标准**：
- 既往至少2次抑郁发作
- 当前症状缓解或轻度（PHQ-9 < 10）
- 无活跃自杀风险
- 能够参与团体课程

### 7.3 药物-正念整合

| 情境 | 建议 |
|------|------|
| **急性期药物治疗** | 正念干预延迟至缓解期 |
| **维持期药物 + MBCT** | 可安全联合；MBCT结束后可尝试谨慎减药 |
| **药物不耐受** | MBCT作为维持治疗替代 |
| **残留症状** | 联合治疗优于单一治疗 |
