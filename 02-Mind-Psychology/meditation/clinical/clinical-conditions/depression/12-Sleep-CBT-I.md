## 12. 睡眠正念与失眠认知行为疗法（CBT-I）整合

> **说明**：抑郁与失眠存在双向因果关系——失眠是抑郁发作的独立风险因素，也是抑郁复发的最强预测指标之一。正念与 CBT-I（失眠认知行为疗法，一线循证治疗）的整合，可同时打破"抑郁→失眠→更抑郁"的恶性循环。

---

### 12.1 抑郁-失眠的双向机制

```
抑郁症状
    ↓
反刍思维（夜间思维活跃度增加）
    ↓
生理唤醒升高（HPA轴激活、皮质醇夜间峰值）
    ↓
入睡困难 / 夜间觉醒
    ↓
睡眠剥夺（REM睡眠过多、慢波睡眠不足）
    ↓
情绪调节能力下降、前额叶功能受损
    ↓
抑郁加重（对负面刺激更敏感、快感缺失加剧）
    ↓
循环强化...
```

**关键数据**：
- 约 75% 的抑郁患者报告失眠症状（入睡困难、维持睡眠困难、早醒）。
- 失眠患者发生抑郁的风险是正常睡眠者的 **2-10 倍**（Baglioni et al., 2011）。
-  residual insomnia（抑郁缓解后仍残留失眠）是抑郁复发的最强预测因子之一。

---

### 12.2 睡眠卫生正念版

传统睡眠卫生（sleep hygiene）建议（如"睡前不喝咖啡""保持卧室黑暗"）对慢性失眠效果有限，但结合正念后可提升依从性和效果。

| 传统睡眠卫生 | 正念增强版 | 抑郁特定调整 |
|------------|-----------|------------|
| 固定就寝时间 | **觉察身体的睡眠信号**：不是"到点了必须睡"，而是学会识别自己的困倦信号（眼皮沉重、眨眼变慢、思维模糊） | 抑郁患者常伴"晨重夜轻"，就寝时间不应过早（防止床上反刍），建议固定起床时间而非固定入睡时间 |
| 卧室只用于睡眠 | **建立床的"睡眠条件反射"**：如果躺20分钟未入睡，起床到另一个房间进行 grounding/呼吸，有睡意再回床 | 避免"床 = 反刍场所"的负面联结；抑郁期尤其重要 |
| 睡前放松 | **睡前30分钟正念过渡仪式**：渐进式身体放松 + 呼吸觉察 + 感恩回顾3件事 | 抑郁患者对"感恩"可能有抵抗，改为"今天我有3个瞬间没有很痛苦" |
| 避免蓝光 | **正念使用电子设备**：如果必须看手机，设置黑白模式，缓慢滑动，觉察手指动作 | 不强迫"完全不使用"（会引发焦虑），而是有觉察地使用 |
| 避免酒精助眠 | **酒精的正念觉察**：记录饮酒后睡眠质量（易醒、早醒），建立因果觉察 | 抑郁患者酒精使用风险高；正念替代为"晚间自我照顾仪式" |

### 12.3 刺激控制与睡眠限制简化方案

CBT-I 的两大核心技术，可简化为抑郁患者友好的家用版本。

#### 12.3.1 刺激控制（Stimulus Control）

**核心原则**：重建"床 = 睡眠"的经典条件反射，打破"床 = 反刍/焦虑/清醒"的负面联结。

```markdown
【五规则简化版】

1. 只在困倦时上床
   - 不是"到点睡觉"，而是"感到困了才上床"。
   - 若计划22:00睡但21:30已困倦：提前上床。
   - 若22:00不困：延迟上床，在客厅进行低唤醒活动（正念呼吸、听轻音乐、看纸质书）。

2. 20分钟规则
   - 上床后20分钟仍未入睡（不要看表，凭感觉估算），立即起床。
   - 离开卧室，在昏暗灯光下进行 grounding 或呼吸练习。
   - 有睡意时再回床。
   - 重复此过程，直到入睡。

3. 床只用于睡眠（和性活动）
   - 不在床上工作、看手机、吃东西、反刍、争吵。
   - 抑郁患者尤其要注意：不要躺在床上"思考人生"。

4. 固定起床时间
   - 无论前一晚睡得多差，每天同一时间起床（包括周末）。
   - 这是重置生物钟的最强信号。
   - 抑郁患者晨重夜轻时：起床后可立即进行 grounding + 光照。

5. 避免日间小睡
   - 若必须小睡：限制在20分钟内，且在下午15:00前。
   - 抑郁伴严重疲劳：小睡可允许，但需在睡眠日志中记录其对夜间睡眠的影响。
```

#### 12.3.2 睡眠限制（Sleep Restriction）简化版

**原理**：减少在床上的清醒时间，提高"睡眠效率"（睡着时间/在床时间），重建睡眠驱动力。

```markdown
【家用简化版步骤】

步骤1：记录1周睡眠日志
  - 记录：上床时间、入睡时间、夜间觉醒次数/时长、起床时间、日间小睡。
  - 计算：平均实际睡眠时间（而非在床时间）。

步骤2：设定初始"在床时间"
  - 初始在床时间 = 平均实际睡眠时间（但不低于5小时）。
  - 例：平均只睡5.5小时，但躺床上8小时 → 初始在床时间设为5.5小时。

步骤3：固定起床时间，倒推上床时间
  - 若固定起床时间为6:30，在床时间5.5小时 → 上床时间为1:00。
  - 这意味着：即使很困，也要熬到1:00才上床。
  - 初期会有轻度睡眠剥夺，但这会积累"睡眠驱动力"，让后续入睡更快。

步骤4：每周调整
  - 若睡眠效率 > 85%（睡着时间/在床时间）：下周增加15分钟在床时间。
  - 若睡眠效率 < 80%：下周减少15分钟在床时间。
  - 逐步扩展，直到达到目标睡眠时长（通常7-8小时）。

【抑郁特定调整】
- 睡眠限制初期的轻度睡眠剥夺可能暂时加重情绪低落。
- 因此：
  - 初始在床时间不低于5小时（避免过度剥夺）。
  - 在增加在床时间时，优先增加"早晨"（提前起床不现实，可逐步提前上床）。
  - 密切监测自杀意念（睡眠剥夺可能暂时增加风险）。
  - 与医生讨论：是否暂时调整抗抑郁药服药时间（如将镇静型药物放在睡前）。
```

### 12.4 夜间觉醒的反刍应对方案

夜间觉醒是抑郁患者的核心痛点：醒来后，反刍思维立即涌入，再也无法入睡。

#### 12.4.1 夜间 grounding 协议（2分钟版）

```markdown
【触发】：夜间觉醒，发现思维开始活跃

步骤1：不要看时间（避免计算"还能睡多久"的焦虑）
步骤2：脚趾抓地5次（本体感觉 grounding）
步骤3：3次深慢呼吸（鼻吸4秒，口呼6秒）
步骤4：内心默念："觉醒是正常的。许多人此刻也醒着。"
步骤5：若仍有反刍：起身，到卫生间或客厅，进行5分钟 grounding
       （5-4-3-2-1 或冷水洗脸）
步骤6：有睡意时回床；若无睡意，在沙发上 resting（不强迫自己睡）
```

#### 12.4.2 " worry time" 的正念版本

传统 CBT 建议设立每日"worry time"来处理担忧。正念版本：

```markdown
【傍晚 worry time（15分钟）】

1. 设定计时器，在固定时间（如18:00）进行。
2. 写下所有担忧和待办事项。
3. 对每个项目，问自己：
   - "此刻我能做什么？" → 写下一个微小行动。
   - "此刻我无能为力？" → 标记为"接受"，放在一旁。
4. 结束时，将纸折起来，放入抽屉——物理上与思维分离。
5. 对自己说："我已经在今天照顾了这些担忧。夜间醒来时，它们可以等待。"

【夜间觉醒时】
若担忧再次升起："这个担忧已在 worry time 中被听到。我可以明天再处理。"
```

### 12.5 睡前正念引导脚本（15分钟版）

```markdown
【准备】
- 关灯或仅留微弱夜灯。
- 仰卧，被子覆盖（深压觉增强安全感）。
- 设定15分钟音频引导，或使用手机计时器。

【阶段1：日间放下（3分钟）】
"现在，是放下白天的时候了。
无论你今天经历了什么——成就或失误、愉快或痛苦——
它们属于今天，不属于这个夜晚。
想象你在把白天的事情一件一件放入一个盒子，
盖上盖子。盒子放在床边，明天早上再打开。"

【阶段2：身体沉降（5分钟）】
从头顶开始，逐部位默念：
"额头...放松..."
"眼睛...放松..."
"下巴...放松..."
（依次向下：肩膀、手臂、胸腹、背部、臀部、大腿、小腿、双脚）

不追求完全放松，只是邀请每个部位"可以休息了"。

【阶段3：呼吸摇篮（5分钟）】
将注意力放在呼吸上。
想象每一次吸气，是摇篮轻轻摇起；
每一次呼气，是摇篮轻轻落下。
你不需要控制呼吸，只是躺在呼吸的摇篮里。

若思绪飘走：温和地带回，如同轻轻把摇篮中的婴儿重新安置。

【阶段4：允许不睡（2分钟）】
"即使我此刻没有睡着，也没关系。
躺在这里，呼吸，本身就是一种休息。
睡眠会来，当它准备好的时候。
我不需要追赶它。"

【结束】
不刻意结束。若音频结束仍未入睡，保持姿势，继续呼吸。
或启动20分钟规则（起床 grounding）。
```
