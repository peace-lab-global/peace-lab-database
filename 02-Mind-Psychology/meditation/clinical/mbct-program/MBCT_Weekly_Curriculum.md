# MBCT 八周课程逐周详解 | MBCT Weekly Curriculum

> **领域**：正念认知疗法（Mindfulness-Based Cognitive Therapy）
> **关键词**：课程设计（Curriculum Design）、每周主题（Weekly Themes）、3分钟呼吸空间（3-Minute Breathing Space）、家庭作业（Home Practice）、教学指引（Teaching Guide）、全日静修（All-Day Retreat）
> **上次更新**：2026-04

## 课程总览

```
前置导入                    8周主课程                                  后续维持
┌────────┐  ┌─────────────────────────────────────────────┐  ┌──────────┐
│ 导向会  │→ │ W1 → W2 → W3 → W4 → 全日 → W5 → W6 → W7 → W8 │→ │ 毕业后支持  │
│(2小时)  │  │      每周2小时 × 8 + 全日静修7小时           │  │ 练习计划   │
└────────┘  └─────────────────────────────────────────────┘  └──────────┘
                    总计约 23 小时课堂时间
              + 每日40-50分钟家庭练习 × 56天 ≈ 37-47小时
```

**MBCT课程核心设计原则**：
1. **体验优先**：通过直接体验而非说教来传达正念
2. **认知整合**：将正念体验与CBT对思维-情绪-身体关系的理解结合
3. **去中心化训练**：系统培养"将体验视为心理事件"的元认知能力
4. **复发预防导向**：始终围绕"识别并退出抑郁模式"这一核心目标

## 课前：导向会（Pre-Course Orientation Session）

### 基本信息

| 项目 | 详情 |
|------|------|
| **时长** | 2小时 |
| **目的** | 介绍MBCT、建立期望、筛查评估、知情同意、建立团体凝聚力 |
| **必要人员** | 2名认证MBCT教师（一男一女，如可能） |
| **最大人数** | 12人（确保每位参与者有足够的关注与分享时间） |

### 导向会流程

| 时间段 | 活动 | 说明 |
|--------|------|------|
| 0:00-0:15 | 欢迎与签到 | 营造安全、包容的氛围；非正式交流 |
| 0:15-0:35 | **葡萄干练习**（Raisin Exercise） | 核心引入体验：从"自动导航"到"正念觉察" |
| 0:35-0:55 | 探询：你注意到了什么？ | 引导参与者自发发现正念的特征 |
| 0:55-1:10 | MBCT简介：什么是MBCT？ | 历史、目标、与MBSR/CBT的区别；**不深入理论** |
| 1:10-1:25 | 课程结构与时间承诺 | 强调每日家庭练习的重要性；讨论常见障碍 |
| 1:25-1:45 | 参与者自我介绍与参加动机 | 简短分享；教师示范"去中心化"倾听 |
| 1:45-1:55 | 知情同意与筛查 | 签署知情同意书；完成前测问卷 |
| 1:55-2:00 | 答疑与结束 | 发放家庭练习音频/资料 |

### 导向会筛查清单

```markdown
## MBCT 参与者入组筛查清单

### 包含标准（Inclusion Criteria）
- [ ] 年龄18-65岁
- [ ] 既往至少有2次（优选3次及以上）重度抑郁发作（Major Depressive Episodes）
- [ ] 目前处于缓解期（Remission），BDI-II ≤ 13 或 HAM-D ≤ 7
- [ ] 有能力参加全部8周课程与全日静修
- [ ] 愿意承诺每日家庭练习（40-50分钟/天）
- [ ] 能理解课程语言（或提供翻译支持）

### 排除标准（Exclusion Criteria）
- [ ] 当前符合重度抑郁发作诊断标准（BDI-II > 20）
- [ ] 当前存在精神病性症状（Psychotic Symptoms）
- [ ] 当前有活跃的物质依赖（Active Substance Dependence）
- [ ] 当前有高风险自杀意念或近期自杀未遂
- [ ] 诊断为边缘性人格障碍（BPD）且未接受专门治疗
- [ ] 近期（6个月内）经历重大创伤事件且未稳定
- [ ] 严重躯体疾病无法参与温和身体活动

### 需谨慎评估（Require Careful Assessment）
- [ ] 有创伤史（PTSD/C-PTSD）：需评估是否适合标准MBCT或需要创伤知情改编
- [ ] 当前使用抗抑郁药：记录药物种类与剂量；讨论药物与MBCT的关系
- [ ] 有双相障碍病史：需精神科确认处于稳定期；排除当前躁狂/轻躁狂
- [ ] 有焦虑障碍共病：通常可纳入，但需监测焦虑激活反应
```

## 第1周：自动导航与身体扫描（Week 1: Autopilot & Body Scan）

### 本周主题与核心概念

| 项目 | 详情 |
|------|------|
| **主题** | 从自动导航中醒来（Waking Up from Autopilot） |
| **核心概念** | 自动导航（Autopilot）、正念的操作定义、身体扫描（Body Scan） |
| **核心练习** | 身体扫描（Body Scan, 30-40分钟） |
| **隐喻** | "我们大部分生活是在自动驾驶模式下度过的" |

### 课堂流程（2小时）

| 时间段 | 活动 | 教学要点 |
|--------|------|----------|
| 0:00-0:10 | 欢迎与签到 | 简短正念时刻；建立本周氛围 |
| 0:10-0:30 | **葡萄干练习** 或 **正念饮食** | 深化导向会的体验；讨论"觉察vs自动" |
| 0:30-0:45 | 探询与小组讨论 | "当你有意识地吃葡萄干时，你注意到了什么？" |
| 0:45-1:00 | 正念的定义与态度 | 卡巴金定义；引入七种态度（非评判、耐心、初心等） |
| 1:00-1:05 | 休息 | — |
| 1:05-1:45 | **身体扫描引导练习**（35分钟） | 系统地从脚到头觉察身体感觉；反复将注意力带回 |
| 1:45-1:55 | 身体扫描探询 | "你注意到身体扫描中的什么？困难？发现？" |
| 1:55-2:00 | 家庭作业说明 | 发放音频；强调每日练习的重要性 |

### 本周家庭作业

- [ ] **每日身体扫描**（30-40分钟，跟随音频）
- [ ] **正念饮食**（至少1餐/天，有意识地进食）
- [ ] **记录一次日常活动的"自动导航"时刻**：例如开车回家却完全没注意路程

### 第1周检查清单（教师用）

```markdown
## Week 1 教学检查清单

### 必须覆盖的内容
- [ ] 引导至少一次完整的身体扫描练习
- [ ] 探询环节至少邀请3-4位参与者分享
- [ ] 明确说明"没有正确或错误的体验"
- [ ] 介绍"自动导航"概念并联系参与者生活
- [ ] 解释家庭练习的重要性与承诺要求

### 教学注意事项
- [ ] 对身体扫描中可能出现的困难（如无聊、烦躁、睡着）进行正常化
- [ ] 避免将身体扫描描述为"放松练习"（它是觉察练习）
- [ ] 对有创伤史的参与者保持敏感：身体扫描可能触发不安
- [ ] 不深入讨论抑郁理论（留待后续周次）
- [ ] 鼓励"好奇与友善"的态度，而非"努力做好"

### 常见参与者反应与应对
| 反应 | 教师回应 |
|------|----------|
| "我做不到，一直在走神" | "走神是正常且重要的发现——觉察到走神本身就是正念" |
| "我觉得很无聊" | "无聊也是一种体验——注意它的身体感觉是什么" |
| "我总是睡着" | "这也常见；试着睁开眼睛练习，或调整练习时间" |
| "身体扫描让我更焦虑" | "可以缩短时间，或从较安全的身体部位开始；我们提供替代方案" |
```

## 第2周：将觉知带入身体（Week 2: Bringing Awareness to the Body）

### 本周主题与核心概念

| 项目 | 详情 |
|------|------|
| **主题** | 身体：觉察的锚点（The Body as an Anchor） |
| **核心概念** | 身体觉察（Body Awareness）、愉悦-不愉悦反应（Pleasant-Unpleasant Reactivity） |
| **核心练习** | 身体扫描 + 正念瑜伽入门（Mindful Yoga） |
| **新增内容** | 愉悦事件日志（Pleasant Events Calendar） |

### 课堂流程（2小时）

| 时间段 | 活动 | 教学要点 |
|--------|------|----------|
| 0:00-0:35 | 身体扫描（30分钟）+ 探询 | 深化练习；关注身体感觉的微妙变化 |
| 0:35-0:55 | 家庭练习回顾 | 分享障碍、发现、问题；教师示范非评判倾听 |
| 0:55-1:10 | **愉悦事件日志**介绍 | 每天记录1个愉悦时刻及身心反应；发现"愉悦也有身体感觉" |
| 1:10-1:15 | 休息 | — |
| 1:15-1:45 | **正念瑜伽**（30分钟） | 温和的身体伸展；强调觉察而非"锻炼" |
| 1:45-1:55 | 正念瑜伽探询 + 讨论 | 觉察身体的边界、限制、温和的坚持 |
| 1:55-2:00 | 家庭作业说明 | 强调交替练习的重要性 |

### 本周家庭作业

- [ ] **交替练习**：身体扫描（隔日）+ 正念瑜伽（隔日），各30分钟
- [ ] **愉悦事件日志**：每天记录1个愉悦时刻，注意：
  - 事件是什么？
  - 身体感觉如何？
  - 情绪如何？
  - 念头是什么？
- [ ] **选择一项日常活动做正念练习**（如刷牙、走路、洗碗）

## 第3周：呼吸空间——核心技术（Week 3: The Breathing Space）

### 本周主题与核心概念

| 项目 | 详情 |
|------|------|
| **主题** | 3分钟呼吸空间：危机中的正念（The Breathing Space: Mindfulness in Crisis） |
| **核心概念** | 3分钟呼吸空间（3-Minute Breathing Space）、反应vs回应（Reacting vs Responding） |
| **核心练习** | **3分钟呼吸空间**（3-Minute Breathing Space）— MBCT标志性技术 |
| **新增内容** | 不愉悦事件日志（Unpleasant Events Calendar） |

### 3分钟呼吸空间详解

```
┌─────────────────────────────────────────────┐
│         3分钟呼吸空间（3-Minute               │
│         Breathing Space）                     │
│                                             │
│   步骤1：觉察（Awareness）          约1分钟   │
│   ┌─────────────────────────────────────┐   │
│   │  暂停，问自己："此刻，我的体验是什么？"  │   │
│   │  • 身体感觉（Bodily Sensations）        │   │
│   │  • 情绪（Emotions）                     │   │
│   │  • 念头（Thoughts）                     │   │
│   │  不试图改变，只是觉察                  │   │
│   └─────────────────────────────────────┘   │
│                                             │
│   步骤2：聚焦（Gathering）          约1分钟   │
│   ┌─────────────────────────────────────┐   │
│   │  将注意力集中在呼吸上                  │   │
│   │  感受呼吸在身体中的感觉                │   │
│   │  用呼吸作为锚点，将散乱的心带回        │   │
│   └─────────────────────────────────────┘   │
│                                             │\n│   步骤3：扩展（Expanding）          约1分钟   │
│   ┌─────────────────────────────────────┐   │
│   │  将觉察从呼吸扩展至整个身体            │   │
│   │  感受身体作为一个整体的呼吸            │   │
│   │  对任何不适保持"允许"的态度            │   │
│   └─────────────────────────────────────┘   │
│                                             │
│   口诀：A-G-E（Awareness-Gathering-Expanding）│
│        或 觉-聚-扩                           │
└─────────────────────────────────────────────┘
```

**3分钟呼吸空间的三种使用时机**：

| 类型 | 使用时机 | 目的 |
|------|----------|------|
| **回应性呼吸空间**（Responsive） | 当感到压力、烦躁、情绪低落时 | 在反应冲动与行动之间创造停顿 |
| **过渡性呼吸空间**（Transitional） | 在日常活动转换之间（如工作→回家） | 清除上一活动的"残留"，进入当下 |
| **计划性呼吸空间**（Planned） | 每天固定时间练习（如餐前、睡前） | 建立正念习惯，强化神经通路 |

### 课堂流程（2小时）

| 时间段 | 活动 | 教学要点 |
|--------|------|----------|
| 0:00-0:35 | 正念瑜伽（30分钟）+ 探询 | 深化身体觉察；注意愉悦与不愉悦的身体信号 |
| 0:35-0:50 | 家庭练习回顾 | 重点讨论愉悦事件日志的发现 |
| 0:50-1:05 | **引入3分钟呼吸空间** | 理论讲解：为什么需要"微型正念"？ |
| 1:05-1:10 | 休息 | — |
| 1:10-1:25 | **3分钟呼吸空间练习**（3次） | 第一次：标准引导；第二次：简化引导；第三次：自我引导 |
| 1:25-1:40 | 不愉悦事件日志介绍 | 与愉悦日志配对；注意"不愉悦≠不好" |
| 1:40-1:55 | 呼吸空间在日常中的应用讨论 | 什么时候用？遇到什么障碍？ |
| 1:55-2:00 | 家庭作业说明 | 强调每日3次呼吸空间（计划性+回应性） |

### 本周家庭作业

- [ ] **交替练习**：正念瑜伽（隔日）+ 身体扫描（隔日），各30分钟
- [ ] **3分钟呼吸空间**：每日至少3次（建议：晨起、午间、睡前 + 任何需要时）
- [ ] **不愉悦事件日志**：每天记录1个不愉悦时刻，注意身心反应模式

## 第4周：识别厌恶反应（Week 4: Recognizing Aversion）

### 本周主题与核心概念

| 项目 | 详情 |
|------|------|
| **主题** | 厌恶：痛苦的放大器（Aversion: The Amplifier of Suffering） |
| **核心概念** | 厌恶（Aversion / Pali: *Dosa*）、痛苦公式（Pain × Resistance = Suffering） |
| **核心练习** | 坐禅（Sitting Meditation）：呼吸+身体+声音；呼吸空间 |
| **新增内容** | 引入"思维-情绪-身体"关系地图 |

### 课堂流程（2小时）

| 时间段 | 活动 | 教学要点 |
|--------|------|----------|
| 0:00-0:35 | 坐禅（30分钟）：呼吸+身体+声音 | 扩大觉察领域；注意对声音的厌恶反应 |
| 0:35-0:50 | 家庭练习回顾 | 讨论呼吸空间的日常应用 |
| 0:50-1:05 | **厌恶反应教学** | 讲解：痛苦+抗拒=苦难；抑郁中的厌恶模式 |
| 1:05-1:10 | 休息 | — |
| 1:10-1:30 | **3分钟呼吸空间** + 探询 | 特别关注呼吸空间第三步：扩展与允许 |
| 1:30-1:50 | **思维-情绪-身体关系地图**练习 | 用近期不愉悦事件填写三要素循环 |
| 1:50-1:55 | 小组讨论 | "你看到了自己的什么模式？" |
| 1:55-2:00 | 家庭作业说明 | 引入"困难情境中的呼吸空间" |

## 第5周：允许/顺其自然（Week 5: Allowing / Letting Be）

### 本周主题与核心概念

| 项目 | 详情 |
|------|------|
| **主题** | 允许如其所是（Allowing / Letting Things Be） |
| **核心概念** | 允许（Allowing / Acceptance）、顺其自然（Letting Be）、与体验的接近（Turning Toward） |
| **核心练习** | 坐禅：引入困难情绪与念头；呼吸空间 |
| **关键转变** | 从"想要改变"到"允许存在" |

### 课堂流程（2小时）

| 时间段 | 活动 | 教学要点 |
|--------|------|----------|
| 0:00-0:35 | 坐禅（30分钟）：呼吸+身体+声音+念头 | 将觉察扩展至念头领域 |
| 0:35-0:50 | 家庭练习回顾 | 讨论第4周关系地图的发现 |
| 0:50-1:05 | **允许与接纳教学** | 澄清：允许≠放弃；允许≠喜欢；允许是减少挣扎 |
| 1:05-1:10 | 休息 | — |
| 1:10-1:35 | **"接近困难"练习**（Turning Toward Difficulty） | 选择轻-中度困难体验，有意识地接近而非回避 |
| 1:35-1:50 | 探询与讨论 | "接近困难时，你注意到了什么？" |
| 1:50-1:55 | 全日静修预告 | 说明下周全日静修的安排与准备 |
| 1:55-2:00 | 家庭作业说明 | 继续每日练习；准备全日静修 |

## 全日静修日（All-Day Retreat, 通常第5-6周之间）

### 基本信息

| 项目 | 详情 |
|------|------|
| **时长** | 7-8小时（通常为上午9:00至下午4:00或5:00） |
| **形式** | 以静默为主；教师提供引导 |
| **目的** | 深化练习；体验持续的正念觉察；整合前5周所学 |

## 第6周：思维不是事实（Week 6: Thoughts Are Not Facts）

### 本周主题与核心概念

| 项目 | 详情 |
|------|------|
| **主题** | 将念头视为心理事件（Seeing Thoughts as Mental Events） |
| **核心概念** | 去中心化（Decentering）、认知去融合（Cognitive Defusion）、"思维不是事实" |
| **核心练习** | 坐禅：念头作为念头；思维-情绪-身体关系地图进阶 |
| **关键教学** | 从"思维内容"转向"与思维的关系" |

### 课堂流程（2小时）

| 时间段 | 活动 | 教学要点 |
|--------|------|----------|
| 0:00-0:05 | 简短开场 | 欢迎从全日静修回来的参与者 |
| 0:05-0:35 | 坐禅（30分钟）：念头作为念头 | 将念头视为"心理事件"而非"真相" |
| 0:35-0:55 | 全日静修体验分享 | 简短分享；鼓励但不过度强迫 |
| 0:55-1:10 | **"思维不是事实"教学** | 核心MBCT认知组件；隐喻：念头像天空中的云 |
| 1:10-1:15 | 休息 | — |
| 1:15-1:35 | **念头识别练习** | "我有一个想法..."技术；给念头贴标签 |
| 1:35-1:50 | 小组练习：关系地图回顾 | 用"我有一个想法..."重新检视之前的困难情境 |
| 1:50-1:55 | 讨论 | "当你把念头视为念头时，有什么变化？" |
| 1:55-2:00 | 家庭作业说明 | 引入"早期预警信号"识别 |

| **核心概念** | 维持计划（Maintenance Plan）、复发预防作为生活方式、持续成长 |
| **核心练习** | 参与者自选练习引导（由一位教师引导，另一位观察） |
| **关键教学** | 告别团体；建立毕业后支持 |

### 课堂流程（2小时）

| 时间段 | 活动 | 教学要点 |
|--------|------|----------|
| 0:00-0:10 | 欢迎与开场 | 回顾8周旅程；肯定参与者的努力 |
| 0:10-0:40 | **参与者引导练习** | 邀请1-2位参与者引导3分钟呼吸空间或身体扫描片段 |
| 0:40-1:00 | 8周旅程回顾 | 逐周回顾核心学习；参与者分享最重要的收获 |
| 1:00-1:10 | 休息 | — |
| 1:10-1:30 | **维持正念的挑战与策略** | 常见障碍与应对；将正念从"课程"转为"生活方式" |
| 1:30-1:45 | **毕业后资源** | 介绍进阶课程、静修、在线资源、本地团体 |
| 1:45-1:55 | 感恩与告别仪式 | 简短慈心发送；团体圈结束 |
| 1:55-2:00 | 后测评估 | 完成课程后问卷 |

### 毕业后维持练习建议

| 练习频率 | 建议内容 | 目的 |
|----------|----------|------|
| **每日** | 3分钟呼吸空间（3次/天） | 维持觉察习惯；应对日常压力 |
| **每周3-5次** | 身体扫描或坐禅（20-40分钟） | 深化并维持正式练习能力 |
| **每周1-2次** | 正念瑜伽或正念行走 | 身体觉察与动态正念 |
| **需要时** | 关系地图、早期预警信号检查 | 主动预防复发 |
| **每月** | 参加正念团体或一日静修 | 社会支持；深化练习 |
| **每年** | 参加3-7日密集静修 | 重置与深化；维持长期动力 |
