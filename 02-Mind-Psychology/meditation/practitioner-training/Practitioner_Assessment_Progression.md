# 冥想执行师评估与进阶体系 | Practitioner Assessment & Progression

> **领域**：执行师培训体系（Practitioner Training System）
> **关键词**：能力评估（Competency Assessment）、MBI-TAC（正念干预教学评估标准）、进阶路径（Progression Pathway）、反思性实践（Reflective Practice）、督导（Supervision）
> **上次更新**：2026-04

## 1. 评估哲学与原则

### 1.1 核心理念

冥想教学能力评估不同于一般技能测试，需要同时评估**知识（Knowing）、技能（Doing）和存在品质（Being）**三个维度。

| 评估原则 | 说明 |
|----------|------|
| **过程导向** | 重视成长过程而非单一时间点的表现 |
| **多维评估** | 知识、技能、存在品质缺一不可 |
| **情境嵌入** | 在真实或高仿真教学情境中评估 |
| **发展取向** | 评估服务于发展，而非筛选 |
| **自我评估优先** | 培养持续的自我反思能力 |
| **关系性评估** | 评估者与被评估者的关系本身也是考察内容 |

### 1.2 评估的三重功能

```
诊断性评估（Diagnostic）        形成性评估（Formative）        总结性评估（Summative）
├─ 入训前能力基线               ├─ 培训过程中持续反馈          ├─ 阶段结业考核
├─ 识别学习需求                 ├─ 微调学习路径               ├─ 认证资格判定
└─ 个性化培训计划               └─ 建立自我评估习惯           └─ 进入下一阶段的门槛
```

## 2. MBI-TAC 教学评估标准详解

### 2.1 框架概述

**MBI-TAC**（Mindfulness-Based Interventions: Teaching Assessment Criteria）由 Rebecca Crane 等人于班戈大学（Bangor University）开发，是目前最广泛使用的正念教学评估框架。

### 2.2 六大评估域

| 域 | 名称 | 英文 | 评估焦点 |
|----|------|------|----------|
| 1 | 课程组织与教学 | Coverage, Pacing & Organisation | 课程结构、时间管理、教学内容覆盖 |
| 2 | 关系性技能 | Relational Skills | 真诚、温暖、好奇、尊重、教学在场 |
| 3 | 正念练习体现 | Embodiment of Mindfulness | 教师自身正念品质的体现 |
| 4 | 引导正念练习 | Guiding Mindfulness Practices | 引导词质量、节奏、语调、空间 |
| 5 | 传递主题内容 | Conveying Course Themes | 心理教育内容的清晰度和深度 |
| 6 | 正念探询 | Holding the Learning Environment | 团体促进、探询技巧、回应学员 |

### 2.3 六级能力等级

| 等级 | 名称 | 描述 | 对应阶段 |
|------|------|------|----------|
| **1** | 不胜任（Incompetent） | 未展现相关能力，可能造成伤害 | 不适合教学 |
| **2** | 初学（Beginner） | 有基本意识但实施不一致 | 培训早期 |
| **3** | 进阶初学（Advanced Beginner） | 部分情境下能胜任，需指导 | 培训中期 |
| **4** | 胜任（Competent） | 多数情境下能可靠地教学 | 可独立教学 |
| **5** | 熟练（Proficient） | 灵活应对复杂情境，深度整合 | 资深教师 |
| **6** | 高级（Advanced） | 卓越的教学在场，创新与引领 | 培训师/督导 |

> **认证门槛**：通常要求所有六个域均达到 Level 4（胜任）及以上。

## 3. 多维评估工具箱

### 3.1 知识维度评估

| 工具 | 形式 | 评估内容 | 适用阶段 |
|------|------|----------|----------|
| 笔试 | 选择/简答 | 理论知识、禁忌症、伦理规范 | 基础阶段 |
| 案例分析 | 书面报告 | 临床推理、方案设计、伦理决策 | 中级阶段 |
| 文献综述 | 论文 | 研究证据理解与批判性思维 | 进阶阶段 |
| 口头答辩 | 面试 | 整合性理解、即时应变 | 认证阶段 |

### 3.2 技能维度评估

| 工具 | 形式 | 评估内容 | 适用阶段 |
|------|------|----------|----------|
| 引导练习录音 | 音频 | 引导词、语速、语调、停顿 | 全程 |
| 教学录像 | 视频 | 完整课堂呈现、互动质量 | 中后期 |
| 标准化教学演示 | 现场 | 20-30分钟模拟课堂 | 认证阶段 |
| OSCE | 结构化 | 标准化情境应对（含困难场景） | 临床方向 |
| 同伴互评 | 互评 | 同学之间的建设性反馈 | 全程 |

### 3.3 存在品质（Being）维度评估

| 方法 | 说明 |
|------|------|
| 个人冥想日志 | 持续记录个人练习体验与反思 |
| 反思性随笔 | 对教学经历和个人成长的深度反思 |
| 督导面谈 | 督导对教师在场品质的观察和反馈 |
| 360度反馈 | 学员、同伴、督导多角度评价 |
| 闭关参与记录 | 密集练习的频率和深度 |

## 4. 进阶路径体系

### 4.1 五级发展阶段

| 阶段 | 名称 | 时长（参考） | 核心任务 | 教学权限 |
|------|------|-------------|----------|----------|
| **Level 0** | 稳固练习者 | 2年+ 个人练习 | 建立稳定的个人冥想练习 | 无 |
| **Level 1** | 见习教师 | 培训第1年 | 理论学习 + 观摩教学 | 仅辅助 |
| **Level 2** | 实习教师 | 培训第2年 | 督导下教学实践 | 督导下教学 |
| **Level 3** | 合格教师 | 认证后 | 独立教学 + 持续督导 | 独立教学 |
| **Level 4** | 资深教师 | 5年+ 教学经验 | 培训新教师 + 督导 | 培训师/督导 |

### 4.2 各阶段进阶要求

**Level 0 → Level 1（进入培训）**
- [ ] 至少2年规律个人冥想练习（每周5天以上）
- [ ] 至少1次5天以上住宿式闭关经历
- [ ] 相关领域学历或工作经验
- [ ] 个人陈述 + 推荐信
- [ ] 面试通过

**Level 1 → Level 2（开始教学实践）**
- [ ] 完成理论课程模块（通常200+学时）
- [ ] 理论考试达标
- [ ] 观摩教学满 100 小时
- [ ] 至少完成2次密集闭关（合计10天以上）
- [ ] 督导评估通过

**Level 2 → Level 3（独立认证）**
- [ ] 督导下教学实践满 300 小时
- [ ] 完成至少2轮完整课程教学
- [ ] 教学录像评估（MBI-TAC 全域 ≥ Level 4）
- [ ] 反思性论文/案例报告
- [ ] 累计闭关满 30 天
- [ ] 个人练习持续不间断
- [ ] 认证委员会最终评审

**Level 3 → Level 4（培训师/督导）**
- [ ] 独立教学满 5 年 / 1000+ 小时
- [ ] 接受过督导培训
- [ ] 持续个人督导经历
- [ ] 在专业领域有贡献（研究/出版/创新）
- [ ] 同行认可与推荐

## 5. 督导体系

### 5.1 督导模式

| 模式 | 频率 | 形式 | 适用对象 |
|------|------|------|----------|
| 个人督导 | 月1-2次 | 一对一 | 实习/新手教师 |
| 小组督导 | 月1次 | 3-6人小组 | 所有阶段教师 |
| 同伴督导 | 按需 | 2-3人互督 | 合格教师之间 |
| 直播督导 | 按需 | 现场或视频观摩 | 实习阶段重点 |
| 录像回顾督导 | 按需 | 回看教学录像 | 所有阶段 |

### 5.2 督导核心要素

| 要素 | 具体内容 |
|------|----------|
| **教学过程回顾** | 发生了什么？你做了什么？结果如何？ |
| **体现性探究** | 你在那个时刻身体/情绪的感受是什么？ |
| **盲点识别** | 你可能忽略了什么？有什么模式在重复？ |
| **边缘成长** | 哪些情境让你感到不确定或超出舒适区？ |
| **个人练习连接** | 你的个人修行如何影响你的教学？ |
| **伦理审视** | 有没有伦理灰色地带需要讨论？ |

### 5.3 督导中的反思性问题（示例）

1. 在那个困难时刻，你注意到自己内在发生了什么？
2. 如果重来，你会做什么不同的选择？
3. 你的反应中，有多少来自你自己的"材料"（个人议题）？
4. 学员的困难是否触发了你自身的某些模式？
5. 你如何平衡"做"（Doing）和"在"（Being）？

## 6. 持续专业发展（CPD）

### 6.1 年度CPD要求（参考标准）

| 类别 | 最低要求 | 示例活动 |
|------|----------|----------|
| 个人练习 | 每日练习 + 年度闭关 | 日常坐禅30-45分钟 + 5天闭关 |
| 持续教育 | 30小时/年 | 工作坊、研讨会、在线课程 |
| 督导 | 12小时/年 | 个人督导 + 小组督导 |
| 同伴学习 | 12小时/年 | 同伴互督、教学观摩 |
| 阅读/研究 | 持续 | 跟踪最新研究文献 |

### 6.2 CPD记录模板

```
年度持续专业发展记录

姓名：____________  年度：____________

一、个人冥想练习
   日均练习时间：_____ 分钟
   年度闭关记录：[日期、地点、时长、传统]

二、教学活动
   课程教授：_____ 轮次 / _____ 总学时
   服务学员：_____ 人次

三、继续教育
   [日期] [活动名称] [时长] [主办方]

四、督导
   [日期] [督导者] [模式] [主题]

五、年度反思
   本年度最大的成长是什么？
   最大的挑战是什么？
   明年的发展目标是什么？
```
