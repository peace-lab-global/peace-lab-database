# 中国传统静坐功法 | Chinese Sitting Meditation Practices

> **领域**：中国本土冥想与禅修（Chinese Indigenous Meditation）
> **关键词**：静坐法（Quiet-Sitting Method）、因是子静坐法、打坐姿势、调身调息调心、七支坐法
> **上次更新**：2026-04

---

## 1. 静坐基础：调身、调息、调心

### 1.1 三调体系

中国传统将静坐准备归纳为"三调"：

| 调 | 内容 | 要点 |
|----|------|------|
| **调身** | 坐姿、身体放松 | 姿势端正但不僵硬 |
| **调息** | 呼吸调整 | 从粗到细、从急到缓 |
| **调心** | 心念安住 | 从散乱到专注 |

### 1.2 七支坐法（毗卢遮那七支坐法）

| 支 | 要求 | 说明 |
|----|------|------|
| 1. 足 | 双盘/单盘/散盘 | 根据柔韧度选择 |
| 2. 手 | 定印（右手在上，左手在下，拇指相触） | 或结其他手印 |
| 3. 背 | 脊柱自然直立 | 如叠铜钱般正直 |
| 4. 肩 | 双肩放松下沉 | 略向后张开 |
| 5. 颈 | 微收下巴 | 后颈微微伸展 |
| 6. 舌 | 舌抵上腭 | 连接任督二脉 |
| 7. 眼 | 微闭或半开 | 目光落在鼻尖前方 |

### 1.3 坐姿选择

| 坐姿 | 难度 | 说明 | 适合人群 |
|------|------|------|----------|
| **双盘（全跏趺）** | ⭐⭐⭐⭐⭐ | 双脚放在对侧大腿上 | 柔韧性好的长期练习者 |
| **单盘（半跏趺）** | ⭐⭐⭐ | 一脚放在对侧大腿上 | 有一定基础者 |
| **散盘（自然盘坐）** | ⭐⭐ | 双脚自然交叉 | 初学者 |
| **缅甸坐** | ⭐⭐ | 双脚前后平放地面 | 膝盖不适者 |
| **正坐/跪坐** | ⭐⭐ | 日式正坐，可用蒲团 | 腿部不便者 |
| **椅坐** | ⭐ | 端坐椅上，双脚平放 | 身体限制者/办公场景 |

> **注意**：不应追求高难度坐姿而忽视身体信号。疼痛不是"业力消除"的标志，而是需要调整的提示。

## 2. 因是子静坐法

### 2.1 概述

蒋维乔（号因是子，1873-1958）于1914年出版《因是子静坐法》，是中国近现代第一部系统的世俗静坐指南。

| 项目 | 详情 |
|------|------|
| **作者** | 蒋维乔（因是子） |
| **年代** | 1914年（初版）/ 1955年（续编） |
| **特色** | 融合道家养生与佛教禅修，以现代语言阐述，去宗教化 |
| **影响** | 开启中国现代静坐运动，影响几代知识分子 |

### 2.2 核心方法

**第一步：姿势准备**
1. 选择安静场所，空气流通
2. 采用散盘或单盘坐姿
3. 宽松衣物，解开腰带
4. 端身正坐，两手叠放

**第二步：呼吸调整**
- 初期：自然呼吸，不刻意控制
- 进阶：腹式呼吸（深长缓匀）
- 高阶：胎息（呼吸若有若无）

**第三步：意念安住**
- 初期：意守丹田（脐下三指处）
- 中期：随息法（心随呼吸）
- 高阶：无念（心无所住）

### 2.3 因是子的现代意义

| 贡献 | 说明 |
|------|------|
| 去宗教化 | 首次将静坐从宗教修行中解放，定位为健康养生法 |
| 科学化 | 用生理学语言解释静坐效果 |
| 平民化 | 让普通人也能学习和实践 |
| 系统化 | 提供了完整的入门到进阶路径 |

## 3. 禅堂静坐规矩

### 3.1 传统禅堂一天

| 时间 | 活动 |
|------|------|
| 4:00 | 起板（起床） |
| 4:30 | 早课 |
| 5:30 | 早粥 |
| 6:30-11:00 | 坐香（坐禅）+ 行香（经行）交替 |
| 11:00 | 午斋 |
| 12:00-13:00 | 养息 |
| 13:00-17:00 | 坐香 + 行香交替 |
| 17:00 | 药石（晚餐） |
| 18:00-21:00 | 坐香 + 行香 |
| 21:00 | 开静（就寝） |

### 3.2 坐香与行香

**坐香**（静坐冥想）
- 一支香约45分钟-1小时
- 引磬开始，木鱼收尾
- 全程止语、不动

**行香**（经行/走圈）
- 坐香间隙的活动环节
- 围绕禅堂内圈行走
- 从慢走到快走到"跑香"
- 重新激活血液循环

### 3.3 禅堂礼仪

| 规矩 | 说明 |
|------|------|
| 止语 | 禅堂内全程不说话 |
| 不串寮 | 不随意走动到他人位置 |
| 眼观鼻 | 目光低垂，不左顾右盼 |
| 轻动作 | 一切动作轻缓 |
| 尊师 | 听从维那（管堂师）指令 |

## 4. 常见身心反应与处理

### 4.1 身体反应

| 反应 | 原因 | 处理 |
|------|------|------|
| 腿麻/疼 | 血液循环受阻 | 调整坐姿、交替双腿、逐渐延长时间 |
| 背痛 | 肌力不足或姿势不当 | 加强核心肌群、使用靠背、检查坐姿 |
| 昏沉/瞌睡 | 疲劳、消化、氧气不足 | 开窗通风、适当运动、调整饮食 |
| 身体发热/出汗 | 气血运行加快 | 正常现象、不必紧张、注意保暖 |
| 身体震动/摇晃 | 肌肉放松反应 | 通常自行消失、若持续需咨询老师 |

### 4.2 心理反应

| 反应 | 说明 | 处理 |
|------|------|------|
| 散乱 | 杂念纷飞 | 正常过程，温和地回到所缘 |
| 昏沉 | 意识模糊 | 睁大眼睛、稍提精神 |
| 掉举 | 坐立不安 | 放松身体、关注呼吸 |
| 幻境 | 出现光影、声音等 | "见怪不怪，其怪自败"——不执著 |
| 喜悦/轻安 | 安详喜悦感 | 好征兆但不执著 |

### 4.3 传统警示："走火入魔"

| 类型 | 症状 | 原因 | 预防 |
|------|------|------|------|
| **走火** | 头痛、胸闷、气逆 | 呼吸用力过猛、意念过于集中 | 自然放松、不强迫 |
| **入魔** | 幻觉执著、妄想膨胀 | 执著异常体验、缺乏正见 | 正确知见、明师指导 |
