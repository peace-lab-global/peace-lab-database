## 1. 抑郁症的临床谱系

### 1.1 DSM-5 抑郁症分类

| 障碍类型 | 核心特征 | 正念干预适用性 |
|---------|---------|-------------|
| **重性抑郁障碍 (MDD)** | 持续2周以上的抑郁发作 | ★★★★☆ 中高（MBCT预防复发最强） |
| **持续性抑郁障碍 (PDD/dysthymia)** | 慢性轻度抑郁，持续2年以上 | ★★★★☆ 中高 |
| **经前烦躁障碍 (PMDD)** | 月经周期相关的严重情绪症状 | ★★★★☆ 中高 |
| **破坏性心境失调障碍 (DMDD)** | 儿童慢性易激惹 | ★★★☆☆ 中 |
| **物质/药物所致抑郁** | 由物质或药物引发 | ★★★☆☆ 中（需先处理原发因素） |
| **其他医学状况所致抑郁** | 由躯体疾病引发 | ★★★☆☆ 中（需联合医学治疗） |

### 1.2 全球流行病学

| 指标 | 数据 | 来源 |
|------|------|------|
| 终身患病率（全球） | 10-15% | WHO, 2021 |
| 女性:男性比例 | 约 2:1 | 多项流行病学研究 |
| 复发率（首次发作后） | 50% | 1年内 |
| 复发率（三次发作后） | 90% | — |
| 全球致残原因排名 | 第1位（精神疾病中） | WHO |
| 自杀相关死亡比例 | 约 60-70% 涉及抑郁 | — |

### 1.3 抑郁的维持机制

```
负性生活事件 / 生物易感性
    ↓
负性认知偏向（注意、记忆、解释）
    ↓
情绪低落 → 行为退缩（减少活动、社交隔离）
    ↓
躯体变化（睡眠、食欲、精力下降）
    ↓
反刍思维（Rumination）：对情绪及其原因的反复思考
    ↓
认知融合（"我很失败" = 事实）
    ↓
抑郁螺旋自我维持
```

**核心洞察**：反刍思维（rumination）是抑郁维持的关键机制——不是抑郁情绪本身，而是对抑郁情绪的**过度思考**使症状持续。冥想通过培养**去中心化（decentring）**能力来打断这一循环。

## 2. 抑郁的神经生物学机制

### 2.1 抑郁相关的脑网络

| 脑区/网络 | 在抑郁中的角色 | 冥想的调节作用 |
|----------|------------|--------------|
| **默认模式网络 (DMN)** | 过度活跃；自我参照反刍 | 降低DMN活动；增强DMN-SN动态切换 |
| **前额叶皮层 (PFC)** | 背外侧PFC功能下降（认知控制） | 增强执行控制功能 |
| **杏仁核** | 对负性刺激过度反应 | 降低杏仁核对负性刺激的反应性 |
| **海马体** | 体积缩小；神经发生减少 | 可能促进海马神经发生 |
| **前扣带皮层 (ACC)** | 情绪调节功能异常 | 增强情绪监测和调节 |
| **纹状体** | 奖赏加工减弱（快感缺失） | 可能增强对自然奖赏的敏感性 |

### 2.2 神经可塑性改变

**Siegle et al. (2007)** 发现：
- 抑郁患者在反刍时，内侧前额叶皮层和海马之间的连接过度活跃
- 正念训练可以改善这种异常的连接模式

**Gotlib et al. (2010)** 发现：
- 抑郁风险个体在 DMN 上显示出过度连接
- 这种连接模式在正念干预后可被调节

## 3. 冥想干预抑郁的核心机制

### 3.1 反刍的打断机制

| 机制 | 描述 | 对应抑郁病理 |
|------|------|------------|
| **去中心化（Decentring）** | 将思维视为心理事件而非现实 | "我很失败" → "我注意到我有一个'我很失败'的想法" |
| **注意重定向** | 将注意力从反刍内容转移到当下锚点 | 打破反刍循环 |
| **接纳（Acceptance）** | 允许情绪存在而不抗拒 | 减少"对抑郁的抑郁"（二级情绪） |
| **身体觉察** | 从思维转向身体感受 | 打破思维-情绪循环 |
| **自我慈悲（Self-Compassion）** | 以温和态度对待自己的痛苦 | 替代自我批评（抑郁的核心认知特征） |

### 3.2 正念 vs. CBT：关键区分与互补

| 维度 | CBT | 正念/MBCT |
|------|-----|----------|
| **目标** | 改变负性思维内容 | 改变与思维的关系 |
| **对反刍的策略** | 认知重构（挑战想法） | 去中心化（观察想法而不卷入） |
| **对情绪的态度** | 识别情绪背后的想法 | 直接觉察和接纳情绪 |
| **预防复发** | 良好 | 优秀（MBCT = 维持药物效果） |
| **适用阶段** | 急性期（症状明显时） | 缓解期/维持期（预防复发最强） |

## 4. 循证干预方案

### 4.1 关键荟萃分析

| 研究 | 设计 | 结论 | 效应量 |
|------|------|------|--------|
| **Hofmann et al. (2010）** | 元分析, 39项RCT | 正念对焦虑和抑郁有中到大效应 | g = 0.63 |
| **Piet & Hougaard (2011）** | MBCT 元分析, 6项RCT | MBCT 显著降低抑郁复发风险 | HR = 0.66 |
| **Kuyken et al. (2016）** | RCT, MBCT vs 维持药物 | MBCT 非劣效于维持药物；成本更低 | — |
| **Goldberg et al. (2018）** | 元分析, 163项研究 | 正念对抑郁有中等效应，随访期维持 | g = 0.30-0.50 |
| **Parsons et al. (2017）** | 元分析, 13项fMRI研究 | 正念改变抑郁相关的神经活动模式 | — |

### 4.2 MBCT = 维持药物治疗（里程碑研究）

**Kuyken et al. (2015), The Lancet**：

| 发现 | 意义 |
|------|------|
| 424名复发性抑郁患者，随机分配至MBCT+渐停药物 vs 维持药物 | 最大规模的MBCT预防复发研究 |
| 2年随访：MBCT组复发率与维持药物组无显著差异 | MBCT可作为维持药物的替代 |
| MBCT组生活质量评分更高 | 不仅仅是等效，还有额外益处 |
| MBCT组成本更低 | 卫生经济学优势 |

> **NICE指南（英国）**：推荐MBCT作为复发性抑郁的三线治疗（在CBT和药物治疗之后）。

## 5. MBCT详解：预防抑郁复发

### 5.1 MBCT 的核心创新

MBCT（Mindfulness-Based Cognitive Therapy）由 **Zindel Segal, Mark Williams, John Teasdale** 开发，将 MBSR 与认知治疗技术整合。

**核心洞察**：
> "抑郁复发的关键不在于负性思维的内容，而在于个体**陷入**负性思维模式的方式。当情绪低落时，过去抑郁时的思维模式（'我毫无价值'）会自动被激活。MBCT教人们在模式激活的**早期**就识别它，从而避免坠入抑郁的深渊。"

### 5.2 八周课程结构

| 周次 | 主题 | 核心正念练习 | 认知元素 |
|------|------|------------|---------|
| **W1** | 自动导航与觉察 | 身体扫描；葡萄干练习 | 识别自动化模式 |
| **W2** | 障碍的存在 | 身体扫描；坐姿冥想 | "另一个视角"——去中心化 |
| **W3** | 呼吸作为锚点 | 呼吸觉察（3分钟呼吸空间引入） | 将觉察带入日常 |
| **W4** | 困难情绪 | 开放觉察身体感受 | 情绪与想法的区分 |
| **W5** | 反应 vs. 回应 | 困难情绪的正念 | "想法不是事实" |
| **W6** | 沟通与关系 | 慈心冥想引入 | 人际中的自动化反应 |
| **W7** | 自我照顾模式 | 自选练习；愉悦/掌控活动表 | 识别"行动模式"vs"存在模式" |
| **W8** | 整合与未来 | 回顾；维持计划 | 预防复发行动计划 |

### 5.3 核心认知技术："想法不是事实"

**技术演示**：

| 情境 | 自动化想法 | MBCT 回应 |
|------|-----------|----------|
| 工作出错 | "我是个彻底的失败者" | "我注意到我有一个'我是个失败者'的想法" |
| 朋友没回消息 | "他讨厌我了" | "我注意到我在猜测别人的想法" |
| 感到疲倦 | "我的抑郁又回来了" | "我注意到我感到疲倦，这是当下的感受" |

### 5.4 3分钟呼吸空间（针对抑郁的改良版）

**标准版**：觉察 → 聚焦 → 扩展

**抑郁改良版**：
1. **觉察（1分钟）**：承认当下的情绪（"我感到低落/沮丧/空虚"）
2. **聚焦（1分钟）**：将注意力收窄到呼吸的物理感觉
3. **扩展（1分钟）**：扩展觉察到整个身体；对自己升起慈悲

## 6. 不同抑郁亚型的冥想策略

### 6.1 复发性抑郁（MBCT核心适应症）

| 特征 | 策略 |
|------|------|
| 有3次及以上发作史 | 标准MBCT 8周课程 |
| 正在服用抗抑郁药 | 可在维持用药同时进行MBCT；MBCT后可谨慎减药（需医生指导） |
| 对药物不耐受/不愿长期服药 | MBCT作为维持药物的替代 |
| 残留症状 | 日常3分钟呼吸空间；身体扫描 |

### 6.2 慢性抑郁（PDD/dysthymia）

| 特征 | 策略 |
|------|------|
| 长期低度抑郁 | 更长周期的正念训练（12周+） |
| 高反刍倾向 | 重点训练去中心化和注意力重定向 |
| 快感缺失突出 | 结合正念愉悦觉察（savoring）练习 |
| 社交退缩 | 团体MBCT提供社交支持 |

### 6.3 季节性抑郁（SAD）

| 特征 | 策略 |
|------|------|
| 冬季发作 | 结合光照疗法 |
| 嗜睡/食欲增加 | 身体扫描改善身体觉察；正念饮食 |
| 预防 | 在秋季开始MBCT课程 |

### 6.4 产后抑郁

| 特征 | 策略 |
|------|------|
| 睡眠剥夺加重症状 | 短版正念练习（5-10分钟） |
| 自责/内疚 | 自我慈悲冥想 |
| 时间限制 | 将正念融入育儿活动 |
| 母婴互动 | 正念育儿（Mindful Parenting） |

## 7. 临床实施框架

### 7.1 分层干预模型

```
急性期（重度抑郁发作）
    └── 优先药物治疗和/或CBT
    └── 不建议引入正式正念训练（注意力困难；可能加重自责）
    
缓解期（症状减轻，功能恢复）
    └── 引入MBCT预防复发
    └── 可同时进行维持治疗
    
维持期（无症状或残留症状）
    └── MBCT作为核心维持策略
    └── 日常正念练习
    
高风险期（识别早期预警信号）
    └── 加强练习频率
    └── 3分钟呼吸空间作为"急救"
```

### 7.2 筛查与评估

| 评估内容 | 工具 | 目的 |
|---------|------|------|
| **抑郁严重程度** | PHQ-9, BDI-II, MADRS | 确定是否适合正念干预 |
| **反刍倾向** | RRS（Ruminative Responses Scale） | 识别高反刍个体（MBCT特别受益） |
| **自杀风险** | C-SSRS, 临床访谈 | 安全筛查 |
| **认知功能** | 简要认知筛查 | 评估是否能参与课程 |
| **既往发作史** | 临床访谈 | 确定MBCT适应症 |

**入组标准**：
- 既往至少2次抑郁发作
- 当前症状缓解或轻度（PHQ-9 < 10）
- 无活跃自杀风险
- 能够参与团体课程

### 7.3 药物-正念整合

| 情境 | 建议 |
|------|------|
| **急性期药物治疗** | 正念干预延迟至缓解期 |
| **维持期药物 + MBCT** | 可安全联合；MBCT结束后可尝试谨慎减药 |
| **药物不耐受** | MBCT作为维持治疗替代 |
| **残留症状** | 联合治疗优于单一治疗 |

## 8. 实操工具包

### 8.1 "RAIN" 应对低落情绪

| 步骤 | 全称 | 操作 |
|------|------|------|
| **R** | Recognize（识别） | "我注意到低落情绪正在升起" |
| **A** | Allow（允许） | "我允许这种感受存在，不试图赶走它" |
| **I** | Investigate（探究） | 温和地觉察情绪在身体中的感觉 |
| **N** | Non-identify（不认同） | "这是抑郁的感受，不是我本身" |

### 8.2 反刍打断技术

**当发现自己陷入反刍时**：

1. **命名它**："我注意到我正在反刍"
2. **3分钟呼吸空间**：立即执行
3. **身体觉察**：将注意力转移到脚底与地面的接触
4. **行动**：有意识地开始一个具体的、愉悦的活动

### 8.3 愉悦事件觉察（Pleasant Events Awareness）

抑郁的快感缺失使个体对愉悦不敏感。此练习重新训练奖赏觉察：

> 每天记录一个愉悦时刻：
> - 发生了什么？
> - 身体有什么感受？
> - 情绪如何？
> - 脑海中有什么想法？
> - 是否在享受时也在想别的事？

### 8.4 自我慈悲练习（针对抑郁的自我批评）

```
将手放在心口...
对自己说：
"此刻我很痛苦"
"痛苦是生活的一部分"
"愿我对自己温和"
"愿我接纳自己本来的样子"
```

## 9. 共病考量

### 9.1 抑郁 + 焦虑（最常见共病）

| 特点 | 干预要点 |
|------|---------|
| 高反刍 + 高担忧 | MBCT/MBSR均适用；强调去中心化 |
| 躯体症状突出 | 身体扫描；内感受觉察 |
| 睡眠问题 | 瑜伽尼德拉；睡前正念 |

### 9.2 抑郁 + 创伤

| 特点 | 干预要点 |
|------|---------|
| 创伤驱动抑郁 | 创伤知情正念；可能触发闪回 |
| 解离风险 | 保持接地；睁眼冥想 |
| 专业监督 | 必须在有经验的治疗师指导下进行 |

### 9.3 抑郁 + 物质使用

| 特点 | 干预要点 |
|------|---------|
| 自我用药 | 识别物质使用的情绪调节功能 |
| 双重诊断 | 整合MBCT与物质使用治疗 |
| 复发风险 | 物质复发可能触发抑郁复发 |

### 9.4 抑郁 + 慢性疼痛

| 特点 | 干预要点 |
|------|---------|
| 疼痛-抑郁循环 | 区分疼痛信号和痛苦反应 |
| 功能下降 | 行为激活 + 正念 |
|  catastrophizing | 认知解离 + 接纳 |

## 10. 禁忌症与安全注意事项

### 10.1 相对禁忌症

| 情况 | 风险 | 建议 |
|------|------|------|
| **急性重度抑郁** | 注意力困难；自责加重 | 优先药物/CBT；延迟正念至缓解期 |
| **活跃自杀风险** | 独处冥想可能恶化反刍 | 仅在专业人员密切监督下进行 |
| **精神病性抑郁** | 妄想/幻觉可能受冥想影响 | 禁忌；优先药物治疗 |
| **严重人格障碍** | 团体中人际困难 | 个体正念治疗优先 |
| **双相抑郁（未稳定）** | 冥想可能诱发躁狂 | 仅在情绪稳定后引入；避免过长时间 |

### 10.2 不良反应监测

| 信号 | 可能含义 | 行动 |
|------|---------|------|
| 冥想后情绪显著恶化 | 未处理的情绪材料涌现 | 减少强度；增加支持性会谈 |
| 反刍在冥想中加剧 | "元反刍"（对反刍的反刍） | 教授注意力转移；缩短时长 |
| 感到更加自责 | "我应该能正念" | 重新教育；强调自我慈悲 |
| 解离感 | 过度内省 | grounding技术；外部锚点 |

### 10.3 教师特殊要求

| 要求 | 说明 |
|------|------|
| **临床培训** | MBCT教师需完成牛津正念中心或同等培训 |
| **督导** | 在督导下带领初期课程 |
| **危机处理** | 能够识别和应对自杀风险 |
| **个人练习** | 教师自身需有稳定的正念练习 |
| **对抑郁的理解** | 理解抑郁的主观体验（最好是个人经历或深入培训） |
