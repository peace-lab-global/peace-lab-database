## 1. 成瘾的临床谱系与神经生物学

### 1.1 成瘾障碍的 DSM-5 分类

| 障碍类型 | 核心特征 | 正念干预适用性 |
|---------|---------|-------------|
| **酒精使用障碍 (AUD)** | 失控饮酒；耐受性；戒断症状 | ★★★★★ 高（MBRP证据最强） |
| **阿片类物质使用障碍** | 对阿片类药物的强迫性使用 | ★★★★☆ 中高 |
| **兴奋剂使用障碍** | 可卡因/安非他命成瘾 | ★★★★☆ 中高 |
| **烟草使用障碍** | 尼古丁依赖 | ★★★★★ 高 |
| **行为成瘾**（赌博、网络、性） | 奖赏系统失调；强迫性行为 | ★★★★☆ 中高 |
| **大麻使用障碍** | 尽管有后果仍继续使用 | ★★★☆☆ 中 |

### 1.2 成瘾的神经生物学

```
正常奖赏回路
    自然奖赏（食物、社交） → 伏隔核多巴胺释放 → 满足 → 停止

成瘾奖赏回路
    药物/行为 → 伏隔核多巴胺大量释放（2-10倍） → 强烈快感
        ↓
    前额叶皮层功能抑制（冲动控制下降）
        ↓
    耐受性发展（需要更多才能获得同等效果）
        ↓
    戒断/渴望状态（负性强化驱动使用）
        ↓
    自动化使用模式（习惯回路主导）
```

| 脑区/系统 | 成瘾中的改变 | 正念的调节作用 |
|----------|------------|--------------|
| **伏隔核 (NAcc)** | 对药物线索过度敏感 | 可能降低线索反应性 |
| **前额叶皮层 (PFC)** | 执行控制功能减弱 | 增强冲动控制和决策能力 |
| **杏仁核** | 戒断期的负性情绪放大 | 改善情绪调节 |
| **岛叶** | 内感受信号（渴望）的突显化 | 改变与渴望的关系 |
| **默认模式网络 (DMN)** | 自我参照反刍（"我需要药物"） | 降低DMN与渴望相关的活动 |

### 1.3 渴望的神经科学

**渴望（Craving）不是意志力失败，而是可测量的神经事件**：

- **内感受觉察**：岛叶编码"我需要某物"的身体感觉
- **条件反射**：环境线索（人、地点、物品）触发多巴胺预期释放
- **负性强化**：戒断的不适驱动使用以"感觉正常"

> **核心洞察**：正念不试图消除渴望，而是改变个体与渴望的关系——从"我必须满足这个渴望"转变为"我觉察到渴望正在升起，我可以选择不行动"。

## 2. 冥想干预成瘾的核心机制

### 2.1 "冲浪"模型（Urge Surfing）

由 **G. Alan Marlatt** 提出，基于正念的渴望应对模型：

```
渴望升起
    ↓
[常规反应: 自动化满足 → 短期缓解 → 长期维持成瘾]

[正念反应:]
    ↓
觉察渴望（"我注意到渴望正在升起"）
    ↓
观察渴望的特征（强度、位置、变化）
    ↓
允许渴望存在而不反应
    ↓
注意到渴望的波动性（"渴望像波浪一样，有升有降"）
    ↓
渴望自然消退
    ↓
强化非使用选择
```

### 2.2 机制矩阵

| 机制 | 描述 | 对应成瘾病理 |
|------|------|------------|
| **去自动化（De-automatization）** | 打破刺激-反应的自动化链条 | 成瘾使用的高度自动化 |
| **再感知（Reperceiving）** | 将渴望视为短暂的心理事件而非命令 | "我需要使用"的绝对化认知 |
| **耐受不适（Distress Tolerance）** | 提高对负性情绪的耐受阈值 | 负性强化驱动的使用 |
| **注意控制（Attentional Control）** | 将注意力从药物线索转移 | 线索引发的注意偏向 |
| **自我慈悲（Self-Compassion）** | 减少对复发的自我批判 | "破堤效应"（Abstinence Violation Effect） |

### 2.3 正念与12步程序的整合

| 12步原则 | 正念对应 | 整合方式 |
|---------|---------|---------|
| **第一步：承认无力** | 接纳（Acceptance） | 正念强化对现实的非评判觉察 |
| **第三步：交托更高力量** | 放下（Letting Go） | 冥想中的"不控制"练习 |
| **第四步：道德盘点** | 觉察（Awareness） | 正念增强对行为模式的觉察 |
| **第十一步：祈祷与冥想** | 正念冥想本身 | 结构化正念练习作为"第11步" |
| **第十二步：传递信息** | 慈心冥想/服务 | 慈悲练习增强利他动机 |

## 3. 循证干预方案

### 3.1 关键荟萃分析

| 研究 | 设计 | 结论 | 效应量 |
|------|------|------|--------|
| **Bowen et al. (2014)** | MBRP RCT, 物质使用障碍 | MBRP 显著降低物质使用和渴望 | d = 0.45-0.60 |
| **Li et al. (2017)** | 元分析, 22项RCT | 正念对成瘾有中等到大效应 | g = 0.55 |
| **Sancho et al. (2018)** | 元分析, 30项研究 | 正念降低渴望和物质使用频率 | SMD = 0.42 |
| **Grant et al. (2017)** | fMRI 研究 | 正念练习者岛叶对渴望反应降低 | — |
| **Davis et al. (2014)** | MBRP 对吸烟 RCT | MBRP 优于标准戒烟治疗 | OR = 1.8 |

### 3.2 主要干预方案对比

| 方案 | 全称 | 时长 | 目标人群 | 核心内容 |
|------|------|------|---------|---------|
| **MBRP** | Mindfulness-Based Relapse Prevention | 8周 | 物质使用障碍康复期 | 复发预防；渴望冲浪 |
| **MBCT-A** | MBCT for Addiction | 8周 | 抑郁症共病成瘾 | 整合MBCT与成瘾治疗 |
| **MoreCRA** | Mindfulness-Oriented Recovery Enhancement | 10周 | 处方阿片类成瘾 | 重新定义奖赏；疼痛管理 |
| **Moment-by-Moment** | — |  varies | 酒精依赖 | 即时觉察训练 |

## 4. MBRP：正念复发预防方案详解

### 4.1 方案概述

MBRP 由 **Sarah Bowen, Neha Chawla, G. Alan Marlatt** 在西雅图华盛顿大学开发，是将正念与复发预防（RP）整合的8周团体课程。

**理论基础**：
- 正念增强对高风险情境的觉察
- 改变与内在体验（渴望、负性情绪）的关系
- 打破自动化反应链条

### 4.2 8周课程结构

| 周次 | 主题 | 核心正念练习 | 复发预防元素 |
|------|------|------------|------------|
| **W1** | 自动导航与成瘾 | 身体扫描；呼吸觉察 | 识别自动化使用模式 |
| **W2** | 觉察触发因素 | 身体扫描；坐姿冥想 | 触发因素-渴望-使用链条分析 |
| **W3** | 渴望冲浪 | 渴望冲浪冥想 | Urge Surfing 技术学习 |
| **W4** | 困难情绪 | 困难情绪的正念 | HALT（Hungry, Angry, Lonely, Tired）识别 |
| **W5** | 压力与应对 | 压力反应的正念 | 正念应对策略替代物质使用 |
| **W6** | 人际关系与沟通 | 慈心冥想 | 高风险人际情境的觉察 |
| **W7** | 自我关怀与维持 | 自我慈悲练习 | 预防复发计划 |
| **W8** | 整合与未来 | 回顾与维持 | 长期维持策略 |

### 4.3 核心练习：渴望冲浪（Urge Surfing）

**引导脚本框架**：

> "现在，将注意力带到此刻任何出现的渴望或冲动...
> 
> 不需要刻意制造渴望——如果此刻没有，回想最近一次渴望的体验...
> 
> 注意渴望在你身体中的位置...可能是胃部、胸部、喉咙...
> 
> 注意渴望的强度...从0到10，给它一个数字...
> 
> 现在，像一个冲浪者观察海浪一样，观察这个渴望...
> 
> 注意渴望的变化——它可能在增强、减弱、转移位置...
> 
> 记住：渴望是一种波浪，它会升起，也会消退...
> 
> 你不需要满足它，也不需要对抗它...
> 
> 只是观察，只是觉察...
> 
> [暂停 1-2 分钟]
> 
> 注意渴望现在如何了？...它可能更强了、更弱了、或保持不变...
> 
> 无论它如何，这都是可以的..."

## 5. 不同物质/行为成瘾的冥想策略

### 5.1 酒精使用障碍

| 特点 | 冥想策略 |
|------|---------|
| 社交情境触发强 | 社交情境预演冥想；角色扮演 |
| 负性情绪触发 | 情绪调节正念；RAIN技术 |
| 酒精线索无处不在 | 暴露-正念整合（在安全环境中想象酒精线索+正念觉察） |

### 5.2 阿片类物质使用障碍

| 特点 | 冥想策略 |
|------|---------|
| 强烈的身体戒断 | 身体扫描（从远端开始）；疼痛正念 |
| 高复发率 | 密集的渴望冲浪训练；24小时热线支持 |
| 共病慢性疼痛 | MoreCRA方案；重新定义疼痛与药物的关系 |

### 5.3 尼古丁依赖（吸烟）

| 特点 | 冥想策略 |
|------|---------|
| 高频次触发 | "一分钟呼吸空间"作为每次渴望的即时应对 |
| 手-口习惯 | 手部觉察冥想；替代行为的正念 |
| 社交吸烟 | 社交情境的正念觉察；识别"社交"vs."真正渴望" |

**关键研究**：Davis et al. (2014) 发现，MBRP 对吸烟的效应在**4个月随访时**优于标准戒烟治疗，但在1年随访时差异减弱，提示维持练习的重要性。

### 5.4 行为成瘾（赌博、网络、性）

| 特点 | 冥想策略 |
|------|---------|
| 认知扭曲（"我能赢回来"） | 认知解离："我注意到我有'我能赢'的想法" |
| 多巴胺寻求行为 | 日常愉悦的正念（ savoring ）；重新定义奖赏 |
| 隐蔽性（易于隐藏） | 增强的觉察训练；诚实/透明练习 |

## 6. 临床实施框架

### 6.1 分阶段干预

```
急性戒断期（0-30天）
    └── 优先医学管理（如需要）
    └── 简单接地技术；呼吸调节
    └── 不建议深度内省性冥想

早期康复期（1-6个月）
    └── 引入MBRP或类似结构化课程
    └── 重点：渴望冲浪；触发因素识别
    └── 频率：每周团体 + 每日个人练习

维持期（6个月+）
    └── 维持练习团体
    └── 重点：预防复发；生活意义建构
    └── 鼓励成为冥想练习的"传递者"
```

### 6.2 团体设置要点

| 要素 | 建议 |
|------|------|
| **团体规模** | 6-12人 |
| **时长** | 2小时/周（8周）+ 一日静修 |
| **教师配置** | 正念教师 + 成瘾治疗师（联合带领） |
| **成员同质性** | 建议按物质类型或康复阶段分组 |
| **缺席政策** | 灵活但鼓励连续性；缺席后个别补课 |

### 6.3 药物辅助-正念整合

| 药物 | 与正念的关系 | 注意事项 |
|------|------------|---------|
| **美沙酮/丁丙诺啡** | 可安全联合；正念辅助减药 | 注意药物镇静效果可能影响冥想警觉度 |
| **纳曲酮** | 可联合；正念增强纳曲酮效果 | — |
| **阿坎酸** | 可联合 | — |
| **抗抑郁药** | 共病抑郁时常用 | 注意药物副作用对躯体觉察的影响 |

## 7. 实操工具包

### 7.1 "SOBER" 渴望应对法

| 字母 | 全称 | 操作 |
|------|------|------|
| **S** | Stop（暂停） | 在自动化反应前暂停 1 秒钟 |
| **O** | Observe（观察） | 觉察当下的身体感受、情绪、想法 |
| **B** | Breathe（呼吸） | 三次有意识的呼吸 |
| **E** | Expand（扩展） | 扩展觉察到整个身体 |
| **R** | Respond（回应） | 有意识地选择回应方式，而非自动化反应 |

### 7.2 触发因素日记（正念版）

| 时间 | 触发因素 | 渴望强度(0-10) | 身体感受 | 情绪 | 想法 | 正念回应 | 结果 |
|------|---------|---------------|---------|------|------|---------|------|
| | | | | | | | |

### 7.3 复发后的自我慈悲练习

> "破堤效应"（Abstinence Violation Effect, AVE）：
> "我已经破戒了，不如彻底放弃"

**自我慈悲干预**：
1. **正念**："我注意到我有强烈的自责和失望"
2. **共同人性**："康复是一个过程， setbacks 是常见的"
3. **自我慈悲**："愿我对自己温和；愿我从这次经历中学习"
4. **行动**：尽快回到正轨；联系支持者；回顾触发因素

### 7.4 高风险情境预演冥想

1. 在安全环境中，闭眼想象一个高风险情境
2. 注意身体中升起的反应
3. 练习渴望冲浪
4. 想象成功地不使用的场景
5. 强化自我效能感

## 8. 共病考量

### 8.1 成瘾 + 创伤（最常见共病）

| 特点 | 干预要点 |
|------|---------|
| 创伤驱动使用 | 创伤知情正念；优先稳定化 |
| 解离风险 | 避免长时间静默；保持接地 |
| 情绪调节困难 | 情绪耐受训练渐进进行 |

### 8.2 成瘾 + 抑郁症

| 特点 | 干预要点 |
|------|---------|
| 双重诊断 | MBCT-A 或整合方案 |
| 自杀风险 | 定期评估；安全计划 |
| 动机缺乏 | 小步骤目标；增强内在动机 |

### 8.3 成瘾 + 慢性疼痛

| 特点 | 干预要点 |
|------|---------|
| 疼痛驱动的阿片类使用 | MoreCRA 方案 |
| 疼痛与情绪交织 | 区分"疼痛信号"和"痛苦反应" |
| 运动恐惧 | 温和的身体觉察恢复 |

## 9. 禁忌症与安全注意事项

### 9.1 急性戒断期的注意事项

| 物质 | 戒断风险 | 冥想建议 |
|------|---------|---------|
| **酒精** | 震颤谵妄（DTs）；癫痫 | 医学监督戒断；稳定后再引入冥想 |
| **苯二氮卓类** | 癫痫；精神症状 | 逐渐减量；医学监督 |
| **阿片类** | 严重不适（非致命） | 可尝试轻度呼吸冥想缓解焦虑 |

### 9.2 冥想被误用的风险

| 误用形式 | 识别信号 | 应对 |
|---------|---------|------|
| **冥想作为回避** | "我不需要处理这个问题，我只需要冥想" | 强调正念是面对而非逃避 |
| **灵性逃避（Spiritual Bypassing）** | 用"接纳"为借口继续使用 | 区分健康的接纳与消极的放任 |
| **强迫性冥想** | 每天必须冥想数小时；不冥想就焦虑 | 强调灵活性和自我慈悲 |

### 9.3 复发危机协议

| 信号 | 行动 |
|------|------|
| 渴望强度持续 >8/10 | 立即联系支持者；前往安全环境 |
| 计划/准备使用 | 启动"打破链条"干预；联系治疗师 |
| 实际使用 | 非评判性回应；尽快返回康复轨道；分析触发因素 |
