## 1. 基础步法详解

### 1.1 极慢行禅（Theravada 传统）

**特点**：速度最慢；对动作的分解觉察最精细

**步骤**：

```
1. 站立准备
   - 双脚平行，与肩同宽
   - 双手自然交叠于腹前或背后
   - 微垂双眼

2. 抬脚
   - 觉察右脚跟开始抬起
   - 感受脚底前部的压力增加
   - 继续抬起，觉察整个脚离地

3. 前移
   - 脚在空中缓慢前移
   - 觉察腿部肌肉的活动
   - 觉察身体重心的微妙调整

4. 落脚
   - 脚跟先接触地面
   - 感受脚跟与地面的接触
   - 逐渐过渡到全脚掌

5. 重心转移
   - 身体重心向前移到右脚
   - 觉察左脚开始变轻

6. 重复左脚
```

**速度**：每步约 5-10 秒；往返 10 米约 10-15 分钟

### 1.2 中速行走（MBSR 标准版）

**特点**：自然但稍慢的步行速度；适合初学者

**技术要点**：

| 阶段 | 觉察内容 |
|------|---------|
| **站立** | 双脚与地面的接触；身体平衡 |
| **起步** | "提起-前移-放下"的简化觉察 |
| **行走中** | 脚步的节奏；手臂的自然摆动；呼吸 |
| **转身** | 转身的动作序列；重心的变化 |
| **停止** | 最后几步；回到站立觉察 |

**引导语示例**：
> "现在，开始慢慢行走...
> 觉察提起右脚的感觉...前移...放下...
> 觉察提起左脚的感觉...前移...放下...
> 不需要控制速度——让速度自然发生...
> 只是觉察每一步的提起、移动、放下..."

### 1.3 正常速度行走（日常整合版）

**特点**：正常步行速度；将正念带入日常行走

**简化觉察**：
- 不必分解每一步
- 保持对"正在行走"的整体觉察
- 偶尔将注意力细化到脚步感受

## 2. 进阶技术

### 2.1 呼吸同步行走

**方法**：将呼吸与脚步协调

| 比例 | 适用 |
|------|------|
| **1:1** | 吸气2步，呼气2步 | 日常平静状态 |
| **1:2** | 吸气2步，呼气4步 | 需要放松时 |
| **自然同步** | 不控制，只是觉察呼吸与脚步的自然关系 | 通用 |

**示例**：
> "现在，让行走和呼吸自然协调...
> 不需要强迫——注意它们如何自然同步...
> 也许是吸气时走两步，呼气时走两步...
> 也许是另一种节奏...
> 只是觉察这个自然的协调..."

### 2.2 标签法（Noting）

在南传内观传统中使用：

| 动作 | 标签 |
|------|------|
| 提起脚跟 | "提起" |
| 脚在空中 | "提起" 或 "移动" |
| 脚落下 | "放下" |
| 重心转移 | "触" 或 "压" |

**进阶标签**（更精细）：
- "提起-提起-放下-触-压-提起..."

### 2.3 慈心行走

将慈心冥想与行走结合：

```
每走几步，在心中发送一次慈心祝愿：

"愿我平安"（走4-6步）
"愿我快乐"（走4-6步）
"愿我健康"（走4-6步）
"愿我自在"（走4-6步）

然后扩展：
"愿[特定人]平安"...
"愿所有人平安"...
```

### 2.4 自然行禅（户外）

**环境选择**：
- 公园小径、森林步道、海滩
- 避开交通繁忙的区域
- 相对平坦的地面

**觉察扩展**：
| 对象 | 觉察内容 |
|------|---------|
| **脚步** | 脚底与不同地面（泥土、草地、沙子）的接触 |
| **身体** | 行走中身体的整体运动 |
| **环境** | 风声、鸟鸣、光线、温度 |
| **视野** | 颜色、形状、运动 |
| **整体** | 作为自然环境一部分的存在的感受 |

### 2.5 绕圈行走（Kinhin 简化版）

**设置**：
- 圆形路径或绕房间/庭院行走
- 直径约 3-5 米

**技术**：
- 单手立掌于胸前（左手包右手拇指，右手包左手拇指）
- 另一只手自然下垂或轻握拳于腰侧
- 步伐稳健、有节奏
- 目光微垂，注视前方约 1-2 米

## 3. 场景化应用

### 3.1 MBSR 课程中的行禅（标准版）

| 课程周次 | 引入内容 | 时长 |
|---------|---------|------|
| **W3** | 首次引入正念行走 | 10-15分钟 |
| **W4-5** | 深化；户外行禅 | 15-20分钟 |
| **W6-7** | 结合慈心或开放觉察 | 20-30分钟 |
| **一日静修** | 行禅与坐禅交替 | 全天 |

### 3.2 团体行禅带领

**设置**：
- 路径：室内直线（10-15米）或圆形
- 人数：5-20人（前后间隔1-2米）
- 时间：15-30分钟

**流程**：
1. 站立觉察（1-2分钟）
2. 慢速行禅（10-20分钟）
3. 站立觉察（1-2分钟）

### 3.3 户外带领行禅

**安全要点**：
- 选择平坦、无障碍的路径
- 提前踩点
- 告知参与者注意脚下
- 准备应对天气变化的方案

## 4. 引导词撰写原则

### 4.1 语言原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **现在时态** | 创造即时性 | "你现在正在提起右脚..." |
| **过程导向** | 关注动作而非结果 | "感受提起的过程..."（而非"提起你的脚"） |
| **开放式** | 不预设体验 | "注意脚底有什么感受..." |
| **节奏匹配** | 引导语的节奏与行走节奏同步 | 在提起时说"提起"，在放下时说"放下" |
| **暂停** | 在关键动作后留出感受空间 | 说完"放下"后暂停2-3秒 |

### 4.2 结构原则

```
开始：站立觉察 → 建立基础
    ↓
发展：行走觉察 → 逐步细化
    ↓
深化：扩展觉察 → 环境/呼吸/整体
    ↓
结束：站立觉察 → 整合体验
```

## 5. 特殊人群改良

### 5.1 老年人/平衡障碍

| 标准做法 | 改良 | 理由 |
|---------|------|------|
| 直线往返 | 沿墙壁或扶手行走 | 增加安全感 |
| 闭眼或微垂 | 保持正常视线 | 防止跌倒 |
| 慢速 | 自然舒适的速度 | 避免失去平衡 |
| 独自练习 | 团体陪伴；照护者在旁 | 应急支持 |

### 5.2 帕金森病/运动障碍

| 改良 | 说明 |
|------|------|
| **极短路径** | 3-5米直线 |
| **视觉提示** | 地面贴彩色胶带作为步伐标记 |
| **听觉节拍** | 使用节拍器辅助节奏 |
| **辅助器具** | 允许使用手杖或助行器 |
| **冻结应对** | 教授在冻结时使用正念觉察 |

### 5.3 视力障碍

| 改良 | 说明 |
|------|------|
| **熟悉路径** | 在非常熟悉的环境中练习 |
| **触觉引导** | 沿墙壁、栏杆或盲道行走 |
| **听觉觉察** | 增强对环境声音的觉察 |
| **引导者陪伴** | 轻触引导者的手臂或肩部 |

### 5.4 ADHD/注意力困难

| 改良 | 说明 |
|------|------|
| **较快速度** | 比标准行禅稍快，匹配高能量 |
| **较短时长** | 5-10分钟开始 |
| **结构化** | 使用标签法增加结构 |
| **户外** | 自然环境提供更多感官锚点 |

## 6. 常见问题与应对

| 问题 | 原因 | 应对 |
|------|------|------|
| **行走时思绪纷飞** | 正常；移动增加刺激 | 比坐禅更频繁地将注意力带回脚步 |
| **感到尴尬或不自然** | 社会 conditioning | 私密环境开始；理解这是学习过程 |
| **速度无法控制** | 习惯性快步 | 从极慢开始；使用节拍器 |
| **头晕或失衡** | 慢速改变平衡模式 | 扶墙/扶手；睁眼；缩短时间 |
| **无聊** | 期望过高 | 接受无聊；细化觉察 |
| **脚痛** | 长时间慢走 | 穿舒适鞋子；在柔软地面练习 |
| **无法安静** | 环境干扰 | 接受声音为觉察对象；换环境 |

## 7. 实操脚本示例

### 7.1 10分钟室内行禅引导

```
"请站起来，双脚与肩同宽...
轻轻闭上眼睛，或微垂双眼...

感受你的双脚与地面的接触...
感受身体重心的分布...
觉察此刻的呼吸...

[暂停 30 秒]

现在，开始慢慢行走...
觉察右脚的提起...
前移...
放下...
接触地面...

觉察左脚的提起...
前移...
放下...
接触地面...

继续行走...
每一步，只是觉察：提起、移动、放下...
不需要控制速度...
让速度自然发生...

[暂停 1-2 分钟]

如果你注意到注意力跑开了——
也许在想事情，也许在听声音——
这很正常...
温柔地将注意力带回脚步...
回到提起、移动、放下...

[暂停 1-2 分钟]

现在，如果愿意，扩展你的觉察...
不仅觉察脚步，也觉察整个身体在行走中的感受...
手臂的自然摆动...
肩膀的放松...
脊柱的挺直...

[暂停 1-2 分钟]

现在，再次将注意力收窄到脚步...
准备停止...
再走几步...

慢慢停下...
回到站立...
感受双脚与地面的接触...

轻轻活动手指...
慢慢睁开眼睛..."
```

### 7.2 户外自然行禅引导（15分钟版）

```
"现在，我们开始户外行走...
不需要交谈...
只是行走...

感受脚底与地面的接触...
泥土的柔软...
石头的坚硬...
草地的弹性...

[行走 3-5 分钟]

扩展觉察...
感受微风拂过皮肤...
听到周围的声音...
鸟叫...风声...树叶的沙沙声...

[行走 3-5 分钟]

觉察光线...
阳光透过树叶的斑驳...
阴影和明亮的交替...

[行走 3-5 分钟]

现在，感受整个身体在移动...
感受你是这个自然环境的一部分...
不是分离的观察者...
而是自然的一部分...

[行走 2-3 分钟]

慢慢停下...
回到站立...
感受此刻的完整..."
```
