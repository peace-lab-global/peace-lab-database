| 要素 | 英文名称 | 具体内容 | 重要性 | 注意事项 |
| :--- | :--- | :--- | :--- | :--- |

| **无努力** | Effortlessness | 不强迫、不控制 | 核心原则 | 允许思维漂移 |
| **规律性** | Regularity | 每天2次、每次20分钟 | 累积效应 | 早晚固定时间 |
| **舒适姿势** | Comfortable Posture | 坐姿舒适、背部支撑 | 辅助 | 不必莲花坐 |
| **闭眼** | Eyes Closed | 双眼轻闭 | 基本 | 减少外界干扰 |
| **环境安静** | Quiet Environment | 相对安静的空间 | 辅助 | 不必绝对寂静 |

## 批评与争议

### TM主要争议点表

| 争议类型 | 批评内容 | TM组织回应 | 客观评估 |
| :--- | :--- | :--- | :--- |
| **商业化** | 收费过高、商业模式 | 维持教师、质量保证 | 确实有商业性质，但有奖学金 |
| **秘密性** | 真言保密、神秘化 | 个性化需要、避免误用 | 保密有争议，技术可标准化 |
| **科学性** | 部分研究质量低、偏倚 | 600+研究、同行评审 | 有高质量研究，也有低质量 |
| **宗教性** | 传授仪式、印度教色彩 | 技术非宗教、仪式是传统 | 确有印度教背景，但技术可世俗 |
| **夸大宣传** | 过度宣称效果 | 基于研究发现 | 部分宣传确实夸大 |
| **TM-Sidhi** | "瑜伽飞行"等超自然宣称 | 主观体验、意识发展 | 极具争议、证据不足 |
| **组织控制** | 中央集权、排他性 | 质量控制需要 | 确实控制严格 |
| **邪教指控** | 被部分人称为邪教 | 完全否认 | 不符合邪教特征，但有争议 |
