# 6-3 第六周：与情绪共处：冥想练习：如实观照心念

知识精讲

本次课程进入止观冥想的观照阶段，核心讲解如实观照心念的内涵、练习方法与底层逻辑，强调必须先建立专注能力再开展观照练习，通过去除内心染污获得心灵的自由与清明。

课程整体框架与练习次第

- 课程进度说明：本次是正式课程第5周，前四周完成了止观冥想4D中止的阶段练习，上一周为向观阶段过渡的视觉化冥想练习，本周正式进入观照心念的主题。
  - 🚩重点：观照心念的能力必须建立在专注能力之上，只有先获得心一境性的稳定，才能更好地观照心念。如果观照练习效果不佳，一定要反过来巩固前一阶段的专注练习。
  - 🚩重点：练习必须遵循止而后观的次第：先获得禅定的稳定心力，搭建观照的平台，才能脱离外界干扰、惯性思维与情绪牵引，获得清明的内在状态，进而穿透思维表象，看到问题的根源，自然获得解决方法。

如实观照的核心内涵

- 观照和观看/观察的区别：观照如同照镜子，一方面你作为镜子如实照见对象，另一方面对象也作为镜子如实照见你自身的情结，核心要求是如实——反映事物本来面貌。
- 为何如实很难做到：大多数时候我们内心这面镜子并非干净清澈，会被主观视角、过往经验、预先判断、固有观念、分别心、执着等染污，我们会用预设的“模具”套在对象上，最终看到的是被自己染污后的失真版本，而非事物本来面貌，即产生了扭曲的事实。
  - 举例：沟通中，如果你是在“等着说”而非倾听，就无法听到对方真实的表达；亲密关系中，如果你拿“理想伴侣”的预设模具要求对方，就无法看到真实的对方，最终带来关系的遗憾。
- 观照的状态类比：好的观照如同平静澄澈的水面：当心如止水时，水面既能清晰映照水面上的事物，也能让人看清水下的一切；当情绪波动、突发事冲入生活，就如同风吹动水面泛起涟漪、脏物进入水中让水变浑浊。
- 应对扰动的方法：要让浑浊的水重新变清澈，不需要额外做什么，只需要允许扰动自然发生：等待风吹过，让涟漪自然散去，让脏物自然沉淀，相信内心本自澄明，只要给时间让澄明自然呈现即可。

观照练习的具体方法

- 保持观察者视角：在具体情境中，和你的感觉、感受、感知、行为、情绪、想法保持距离，只是如实观看它们发生，允许它们出现。
- 穿透表象看根源：在保持距离观看的基础上，要进一步穿透表象，看到表象背后引发情绪感受的根源：初始的感觉是真实的，后续的感受、判断都已经有自我意识的叠加和染污，观照就是要把这些叠加看清。
- 常见的内心染污类型：
  1. 主观视角与经验：把过往对同类事件的认知直接套用到当下事件上；
  2. 预设判断：用固有观念的“模具”框定对象，不符合预设就产生抗拒；
  3. 二元分别心：用“我/你”“好/坏”“喜欢/不喜欢”的对立视角看问题；
  4. 🚩重点：执着——认定自己的观点不可动摇，不接纳其他可能性，这是对自由束缚最大的染污：很多时候限制你自由的不是环境或他人，而是你自己对固有观念的执着，一直走不出固化模式就是一种“轮回”，打破执着就能获得解脱自在。

核心思想案例佐证

- 王阳明心学“心外无物”解读：王阳明典故“你未看此花时，此花与汝心同归于寂；你来看此花时，此花颜色一时明白起来，便知此花不在你心外”。事物本身是否存在是一回事，对你而言，只有你感知、认同的事物才会在你的世界中存在，影响你的体验。
  - 延伸：若他人向你释放恶意，你不接受这份恶意，它就无法对你产生影响，如同别人送你礼物你不接，礼物依然属于对方，这就是心本有的自由。
- 🚩重点：观照的智慧：不是刻意忽视问题，而是看清所有事实，但不认同影响你的部分，不抓取连接，回到自己心性本身。所有的体验都是自心的造作，心习惯依附外物，找外在对象指责，但真正的解决之道是松开你内在的认同。
- 认知真相的本质：我们的本然智慧本就能够认知真相，习性染污遮住了镜子，让我们只能间接认知；去除染污之后，本然智慧自然会直接呈现事物原貌，不需要额外分析推导，我们要做的只是去除染污。
- 奥德修斯与海妖的比喻：内心强大的惯性心绪如同塞壬海妖的歌声，会诱惑你和它纠缠，这个时候你要像奥德修斯一样，把自己“绑在锚点上”，回到专注的定境，保持距离不被带走，定力是深度观照的基础。

练习目标提示

- 不需要强求在短时间内达成深度直接认知，每次练习去掉一点点染污，能多看到事物一分真实面相，就是成长进步；随着练习深入，自然会体验到更放松、更开阔、更自在的心境，也会对自己和他人更有理性与接纳度。

[00:05:32](https://getnotes.seek:332) 解释如实难以实现的原因
大多数时候我们内心这面镜子并不干净，会被各种色彩、滤镜覆盖，甚至本身就是哈哈镜或预设形状的模具，我们在和对象互动时，已经带着很多主观添加，最终呈现的是被染污后的样子，而非真实样貌，这就会导致事实扭曲失真。

[00:06:56](https://getnotes.seek:416) 举例说明失真的日常场景
沟通时，如果你是“等着说”而非认真倾听，你就听不到对方真实的表达，自然没有真实的连接和互动；亲密关系中常见的遗憾，就是爱人在身边，你却不想了解真实的对方，只是用“理想伴侣”的标准去要求、对照对方，或是注意力放在其他地方，没有给对方真实的关注，自然无法呈现真实的对方。

[00:08:50](https://getnotes.seek:530) 用水面类比观照的状态
成为镜子如同成为平静澄澈的水面：当心如止水时，水面既可以清晰映照所有倒映的事物，也能让人透过水面看清水下的一切；此时你既可以深度了解内在，也可以如实呈现对方、呈现事物本来样貌。当情绪波动、突发事冲入生活，就如同风吹动水面、脏物落入水中，水面会变得浑浊不再澄明。

[00:10:16](https://getnotes.seek:616) 说明应对内在扰动的方法
要让一杯浑浊的水重新变清澈，不需要额外搅动或做其他操作，只需要慢慢等待，允许风吹过，让涟漪自然散去，让落叶被水流带走，让污垢自然沉淀；练习就是陪伴自己和情绪共处，如实看见发生了什么，不刻意做什么，只是允许扰动自然流动，相信内心本自澄明，澄明会自然呈现，不需要额外添加。

[00:11:45](https://getnotes.seek:705) 拆解观照的对象层次
观照需要你以观察者视角，在具体情境中观看自己的所有反应，包括：感觉、感受、感知、行为、情绪、想法：感觉是刺激产生的初始讯号，比如触碰肌肤首先产生触碰感；之后才会升起感受，比如触碰后慢慢感觉到疼痛；再之后产生感知，识别出“锋利指甲触碰柔软肌肤”，随后产生行为（揉一揉）、情绪和想法。我们需要做的就是如实观看这些内容依次出现，允许它们发生，和它们保持观察者的距离。

[00:13:09](https://getnotes.seek:789) 强调观照要穿透表象看根源
观照的“照见”自带光明，要穿透表象的遮蔽，看到引发情绪感受的根源：同样的触碰，如果你带着好奇心，就会产生完全不同的情绪感受，只有初始的触碰感觉是真实的，后续所有内容都已经有自我意识的叠加，已经产生了分别，观照就是要看清这些叠加和染污。

[00:14:32](https://getnotes.seek:872) 梳理常见的内心染污类型
染污可以是：①你的主观视角，你看待事情的固有观点；②过往经验，遇到类似事件会直接调出旧有认知；③预先判断，基于分析得出的固有结论，这些都会变成你看待对象的预设。我们常常用预设的“模具”套对象：比如你心里有理想伴侣的形状，就会把对方往模具里塞，不合适就会要求对方改变、争吵，其实这都是你的自我叠加，和真实的对方无关。还有二元分别心，用喜欢/不喜欢、好/坏、我的/你的对立视角看问题，最终都会带来执念。

[00:16:23](https://getnotes.seek:983) 重点说明执着对自由的束缚
前面的视角、经验都可以灵活切换，但执着是对自由影响最大的染污：执着就是认定自己的观点不可动摇，无法接纳其他声音，用固定视角看所有事情；很多人觉得自由被环境或他人限制，其实真正限制你的是你自己，是你对固有观念的执着；一直陷在固化模式里打转就是一种轮回，打破执着就能获得解脱自在。

[00:18:43](https://getnotes.seek:1123) 用王阳明心学典故佐证核心观点
借助王阳明与友人观花的典故解释心外无物：你未看此花时，此花与汝心同归于寂；你来看此花时，此花颜色一时明白起来，便知此花不在你心外。对你而言，没见过的花，在你的世界里就是不存在的，只有你看到、认同，它才会在你的世界里存在；你认定某件事、某个人的态度代表什么，本质是你内在认同这个判断，如果你不认同，它就不会对你产生影响。

[00:20:52](https://getnotes.seek:1252) 回应常见疑问：对方真的有问题怎么办
如果对方确实带着恶意、做得不对，你依然可以选择不接这份恶意：就像别人送你礼物你不接受，礼物依然属于对方，你内在不认同，恶意就不会影响你，这就是心本有的自由；不是刻意忽视问题，而是你看清所有事实，不会因为不确定产生额外波动，同时你知道：如果对方对你产生影响，一定是你内在有对应部分认同了它，松开这份认同，就能获得真正的自由。

[00:23:10](https://getnotes.seek:1390) 总结核心认知：体验源自自心造作
所有体验都是自心的造作，心习惯抓取外物依附，找外在对象发泄不满，但问题的本质不在对错的表象，而在于松开你内在的认同，不和影响你的部分连接，一切回到你自己的心性本身。东西方先贤都指向同一个目标：苏格拉底说“认识你自己”，佛陀说“认识你的心”，止观冥想练习最终就是通过认识自心，借由自身的经验体验生命，所以认识你的心是最重要的第一步。

[00:24:52](https://getnotes.seek:1492) 明确观照的核心方向
我们要观照：你的心在抓取什么？在依附什么？你给心喂养了什么内容？你获得的是生命滋养还是消耗？尤其要警惕那些让你执着、消耗你的部分，始终保持观照。

[00:25:27](https://getnotes.seek:1527) 总结观照练习的本质
观照练习就是不断让自己成为清明的镜子，不断去除染污，照见浑浊的部分；正知正见就是内在的光明，把光明带入内在的阴霾，只要持续观照，阴霾会自然散开，执着会自然松开；我们要做的只是保持观察者的距离，不跟心念情绪纠缠。这个阶段的观照比之前的觉察更深，要看表象背后的内容，但🚩注意：不要让头脑分析介入，分析会搅动内心，让水再次变浑浊。

[00:26:52](https://getnotes.seek:1612) 强调定力对观照的重要性
深度观照越深入，越容易和心念情绪纠缠，这个阶段需要更扎实的专注定力基础，相当于冥想练习的“打大boss”阶段，难度更高，更依赖前面的基础积累。

[00:27:28](https://getnotes.seek:1648) 用奥德修斯过海妖的故事比喻定力的作用
希腊英雄奥德修斯经过塞壬海妖的海域时，海妖用优美歌声诱惑听者，让船触礁沉没。奥德修斯让船员用蜡封住耳朵，把自己绑在风帆支柱上，要求船员无论自己发出什么指令都不要解开，最终顺利通过海域。我们内在强大的惯性心绪，就像海妖的歌声，总会诱惑你和它纠缠，当诱惑出现时，你要想到这个绳索，回到你的专注锚点，保持距离，不要被带走，所以定力非常重要。

[00:29:25](https://getnotes.seek:1765) 说明认知真相的原理
我们本自具有认知真相的智慧，当镜子清明，本然智慧自然会工作；因为习性染污遮蔽了镜子，我们只能通过染污间接认知事物；当我们在观照中清理掉染污，事物就会如实呈现，本然智慧会直接看到事物原貌，直接认知是自然呈现的结果，不需要刻意操作，我们只要做好去除染污就够了。

[00:30:38](https://getnotes.seek:1838) 给出练习的合理预期
不需要强求在学习期间就达成深度的直接认知，每次练习去掉一点点染污，就能多看到一分事物的真实，这就是成长进步；随着练习深入，你自然会体验到更多放松、心境开阔、内心自在，也会对自己和他人多一份理性与接纳。

💡 核心概念

- 如实观照：以如同镜子照物的方式，保持观察者距离，去除内心染污，如实呈现事物与心念本来面貌，穿透表象看到问题根源的冥想练习。
- 止而后观：止观冥想的练习次第，先通过练习获得专注稳定的心力，再开展观照练习，没有稳定的专注就无法开展深度观照。
- 心外无物：对你而言，只有你感知、认同的事物才会在你的主观世界中存在，影响你的体验，你不认同的内容无法对你产生作用。
