## 1. 内观禅修概述

### 1.1 基本定义

| 维度 | 详情 |
|------|------|
| **巴利文** | Vipassanā（vi-特殊/深入 + passanā-观看） |
| **含义** | 如实观察事物的本来面目 |
| **核心法则** | 观察三法印：无常（Anicca）、苦（Dukkha）、无我（Anattā） |
| **历史** | 佛陀所教导的核心禅修方法，2500年+ |
| **当代复兴** | 缅甸传统：Ledi Sayadaw → U Ba Khin → S.N. Goenka |

### 1.2 两大当代传承

| 传承 | 代表 | 方法特色 | 全球分布 |
|------|------|----------|----------|
| **Goenka传承** | S.N. Goenka (1924-2013) | 身体感受扫描、10天标准课程 | 全球300+中心 |
| **马哈希传承** | Mahasi Sayadaw | 腹部起伏标记、观智阶段 | 东南亚为主 |

## 3. 核心技术详解

### 3.1 安那般那（Ānāpāna）— 前三天

- 关注**鼻孔以下、上唇以上**的三角区域
- 观察呼吸进出时的自然感受
- 不控制呼吸，只是观察
- 逐渐缩小关注区域
- 目的：锐化注意力（像磨刀），为内观做准备

### 3.2 内观（Vipassana）— 第四天起

**核心练习流程**：
1. 从头顶开始，系统地扫描全身感受
2. 注意每个区域的**任何感受**：温度、压力、刺痛、麻、痒、沉重、轻盈等
3. 以**等量心**（Equanimity）对待所有感受：不贪恋愉悦感受，不厌恶不愉悦感受
4. 持续觉知**无常**：所有感受都在变化（Anicca）
5. 逐渐从部位扫描 → 同时扫描 → 贯穿扫描

### 3.3 等量心（Upekkhā）的核心角色

```
愉悦感受 → 不贪（不执著、不渴望更多）
不愉悦感受 → 不嗔（不厌恶、不渴望消除）
中性感受 → 不痴（保持觉知、不陷入无聊或昏沉）
```

> "你在学习的不是逃避痛苦或追逐快乐，而是以平等心观察一切现象的生灭。" — S.N. Goenka

## 4. 适合人群与注意事项

### 4.1 适合人群

- 愿意投入10天密集练习的认真修行者
- 对佛教禅修有兴趣但不一定是佛教徒
- 寻求深层自我探索和心灵净化的人
- 有一定心理稳定性的成年人

### 4.2 不适合/需谨慎

| 情况 | 建议 |
|------|------|
| 严重精神疾病（精神分裂、双相等） | 不建议参加 |
| 急性创伤（近期） | 暂缓，先寻求治疗 |
| 严重抑郁/自杀倾向 | 不建议参加 |
| 身体严重残障 | 需提前沟通 |
| 药物依赖中 | 不建议 |
| 年龄<18岁 | 有专门的青少年课程 |
