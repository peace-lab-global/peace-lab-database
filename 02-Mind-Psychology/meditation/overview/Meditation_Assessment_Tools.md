## 1. 冥想效果评估框架

### 1.1 评估维度矩阵

| 维度 | 评估内容 | 常用工具 |
|------|---------|---------|
| **症状改善** | 焦虑、抑郁、压力、疼痛 | GAD-7, PHQ-9, PSS-10, VAS |
| **冥想能力** | 正念水平、去中心化能力 | FFMQ, MAAS, TMS |
| **心理功能** | 注意力、情绪调节、认知灵活性 | ANT, DERS, WCST |
| **生理指标** | 自主神经功能、应激激素 | HRV, 皮质醇, EMG |
| **生活质量** | 整体功能、满意度 | SF-36, WHOQOL |
| **不良反应** | 冥想相关负面体验 | MIOME, MACE |

### 1.2 评估时间点

| 时间点 | 目的 |
|--------|------|
| **基线（T0）** | 建立干预前状态；确定适应症 |
| **中期（T4周）** | 监测进展；调整方案 |
| **干预结束（T8周）** | 评估即时效果 |
| **随访（T3月/6月）** | 评估维持效果 |

## 2. 主观量表工具

### 2.1 压力与情绪

#### PSS-10（感知压力量表，10项版）

| 项目 | 说明 |
|------|------|
| **开发者** | Cohen, Kamarck, Mermelstein (1983) |
| **用途** | 评估过去1个月的感知压力水平 |
| **评分** | 0-4分/项；总分0-40 |
| **分界** | 0-13（低）；14-26（中）；27-40（高） |
| **应用** | MBSR/MBCT研究中最常用的压力指标 |

#### PHQ-9（患者健康问卷-9）

| 项目 | 说明 |
|------|------|
| **开发者** | Kroenke et al. (2001) |
| **用途** | 评估抑郁严重程度 |
| **评分** | 0-3分/项；总分0-27 |
| **分界** | 1-4（极轻）；5-9（轻度）；10-14（中度）；15-19（中重度）；20-27（重度） |
| **应用** | 筛查抑郁；监测治疗反应 |

#### GAD-7（广泛性焦虑量表-7）

| 项目 | 说明 |
|------|------|
| **开发者** | Spitzer et al. (2006) |
| **用途** | 评估焦虑严重程度 |
| **评分** | 0-3分/项；总分0-21 |
| **分界** | 0-4（极轻）；5-9（轻度）；10-14（中度）；15-21（重度） |
| **应用** | 焦虑筛查；疗效监测 |

### 2.2 睡眠质量

#### PSQI（匹兹堡睡眠质量指数）

| 项目 | 说明 |
|------|------|
| **开发者** | Buysse et al. (1989) |
| **用途** | 评估过去1个月的睡眠质量 |
| **评分** | 7个维度；总分0-21 |
| **分界** | >5分提示睡眠质量差 |
| **应用** | 睡眠障碍筛查；冥想睡眠研究 |

#### ISI（失眠严重度量表）

| 项目 | 说明 |
|------|------|
| **开发者** | Bastien et al. (2001) |
| **用途** | 评估失眠严重程度 |
| **评分** | 7项；总分0-28 |
| **分界** | 0-7（无）；8-14（亚临床）；15-21（中度）；22-28（重度） |

### 2.3 疼痛评估

#### VAS（视觉模拟量表）

| 项目 | 说明 |
|------|------|
| **用途** | 主观疼痛强度 |
| **评分** | 0-100mm或0-10分 |
| **应用** | 疼痛干预前后的快速评估 |

#### BPI（简明疼痛量表）

| 项目 | 说明 |
|------|------|
| **开发者** | Cleeland & Ryan (1994) |
| **用途** | 评估疼痛强度和对功能的干扰 |
| **应用** | 慢性疼痛的综合性评估 |

## 3. 冥想特质与能力评估

### 3.1 正念水平

#### FFMQ（五因素正念问卷）

| 维度 | 说明 |
|------|------|
| **开发者** | Baer et al. (2006) |
| **用途** | 评估正念的五个维度 |
| **分量表** | 观察、描述、有觉察地行动、不评判、不反应 |
| **评分** | 39项；总分39-195 |
| **应用** | 正念研究中最全面的正念测量工具 |

**五维度详解**：

| 维度 | 高分特征 | 低分特征 |
|------|---------|---------|
| **观察** | 善于觉察内外感受 | 忽视内在体验 |
| **描述** | 能用语言准确标记体验 | 难以表达感受 |
| **有觉察地行动** | 专注于当下活动 | 自动导航 |
| **不评判** | 对体验不评判 | 对自己批评性强 |
| **不反应** | 情绪不自动反应 | 容易被情绪淹没 |

#### MAAS（正念注意觉知量表）

| 项目 | 说明 |
|------|------|
| **开发者** | Brown & Ryan (2003) |
| **用途** | 评估日常注意和觉察状态 |
| **特点** | 单维度；反向计分（低分=高正念） |
| **评分** | 15项；总分1-6 |
| **应用** | 最简单的正念测量工具 |

#### TMS（多伦多正念量表）

| 维度 | 说明 |
|------|------|
| **开发者** | Lau et al. (2006) |
| **用途** | 评估冥想状态的正念品质 |
| **分量表** | 好奇心、去中心化 |
| **应用** | 区分冥想经验者和新手 |

### 3.2 冥想能力

#### SDMS（去中心化量表）

| 项目 | 说明 |
|------|------|
| **用途** | 评估去中心化能力（将思维视为事件而非事实） |
| **应用** | MBCT研究中的核心机制变量 |

#### Self-Compassion Scale（自我慈悲量表）

| 项目 | 说明 |
|------|------|
| **开发者** | Kristin Neff (2003) |
| **用途** | 评估自我慈悲的六个维度 |
| **分量表** | 自我友善、共通人性、正念、自我评判、孤立、过度认同 |
| **应用** | 自我慈悲干预的效果评估 |

### 3.3 冥想不良反应

#### MIOME（冥想相关不良体验量表）

| 项目 | 说明 |
|------|------|
| **开发者** | Lindahl et al. (2020) |
| **用途** | 系统评估冥想练习中的不良反应 |
| **维度** | 认知、感知、情感、躯体、社交 |
| **应用** | 冥想安全性研究 |

#### MACE（冥想不良事件检查表）

| 项目 | 说明 |
|------|------|
| **开发者** | Schlosser et al. (2019) |
| **用途** | 评估冥想相关的不良事件 |
| **应用** | 临床试验中的安全性监测 |

## 4. 生理测量工具

### 4.1 心率变异性（HRV）

| 指标 | 说明 | 正常范围 | 冥想效果 |
|------|------|---------|---------|
| **RMSSD** | 相邻R-R间期差值的均方根 | 20-50ms（年龄相关） | 通常增加 |
| **pNN50** | 相邻R-R间期差>50ms的比例 | 3-15% | 通常增加 |
| **HF Power** | 高频功率（0.15-0.4Hz） | 随年龄变化 | 通常增加 |
| **LF/HF Ratio** | 低频/高频比值 | 1-2 | 通常降低 |

**测量方法**：
- **心电图（ECG）**：金标准
- **光电容积描记（PPG）**：消费级设备（如Polar、Apple Watch）
- **建议**：研究使用ECG；日常监测使用PPG

### 4.2 皮质醇

| 指标 | 说明 | 采集方式 |
|------|------|---------|
| **觉醒皮质醇反应（CAR）** | 醒后30-45分钟皮质醇峰值 | 唾液；醒后0/30/45分钟采集 |
| **日间斜率** | 全天皮质醇下降模式 | 唾液；多次采集 |
| **夜间皮质醇** | 睡前水平 | 唾液；睡前采集 |

### 4.3 其他生理指标

| 指标 | 工具 | 冥想中的变化 |
|------|------|------------|
| **肌电图（EMG）** | 表面电极 | 肌肉紧张度降低 |
| **皮肤电反应（GSR/EDA）** | 皮肤电极 | 交感神经活动降低 |
| **呼吸频率** | 胸带/呼吸带 | 通常降低 |
| **血压** | 袖带/连续监测 | 通常降低 |
| **脑电图（EEG）** | 脑电帽/干电极 | α/θ波增加；γ波变化 |

## 5. 神经影像与电生理

### 5.1 功能性磁共振成像（fMRI）

| 应用 | 常用范式 | 冥想相关发现 |
|------|---------|------------|
| **静息态功能连接** | 闭眼静息 | DMN连接改变；SN-EN连接增强 |
| **任务态激活** | 情绪任务/注意任务 | 杏仁核反应降低；PFC激活增强 |
| **结构MRI** | 高分辨率T1 | 海马/岛叶/ACC灰质变化 |

### 5.2 脑电图（EEG）

| 频段 | 频率 | 冥想相关变化 |
|------|------|------------|
| **δ波** | 0.5-4 Hz | 深度冥想/瑜伽尼德拉中增加 |
| **θ波** | 4-8 Hz | 冥想状态标志；与放松和潜意识可及相关 |
| **α波** | 8-12 Hz | 闭眼放松时增加；长期冥想者基线增加 |
| **β波** | 12-30 Hz | 清醒活跃状态；某些冥想中降低 |
| **γ波** | 30-100 Hz | 高级冥想者中可能增加（存在争议） |

### 5.3 其他影像技术

| 技术 | 应用 | 说明 |
|------|------|------|
| **PET** | 神经递质变化 | 多巴胺释放（Kjaer et al., 2002） |
| **SPECT** | 脑血流 | 特定脑区血流变化 |
| **MRS** | 神经代谢物 | GABA等神经递质浓度 |
| **fNIRS** | 便携式脑功能成像 | 前额叶血氧变化 |

## 6. 临床实践中的评估流程

### 6.1 标准化评估方案（MBSR/MBCT）

| 时间点 | 量表组合 | 时长 |
|--------|---------|------|
| **入组前（T0）** | PHQ-9 + GAD-7 + PSS-10 + FFMQ +  demographics | 15-20分钟 |
| **第4周（T4）** | PHQ-9 + GAD-7 + PSS-10（简短监测） | 5-10分钟 |
| **第8周（T8）** | PHQ-9 + GAD-7 + PSS-10 + FFMQ + 满意度问卷 | 15-20分钟 |
| **3月随访（T3M）** | PHQ-9 + GAD-7 + PSS-10 + FFMQ + 练习日记 | 15分钟 |
| **6月随访（T6M）** | PHQ-9 + GAD-7 + PSS-10 + FFMQ | 10分钟 |

### 6.2 每日练习日志

| 记录内容 | 目的 |
|---------|------|
| **练习日期/时间** | 追踪规律性 |
| **练习类型** | 身体扫描/呼吸觉察/瑜伽/其他 |
| **练习时长** | 剂量-反应关系 |
| **专注度（1-10）** | 主观质量 |
| **情绪变化（1-10）** | 即时效果 |
| **困难/障碍** | 个性化调整 |
| **备注** | 补充信息 |

### 6.3 团体课程中的快速评估

| 工具 | 用途 | 时间 |
|------|------|------|
| **情绪温度计** | 课程开始/结束时的情绪自评（0-10） | 30秒 |
| **一分钟纸** | 课后写下最关键的学习点和问题 | 1分钟 |
| **举手投票** | 快速了解群体状态 | 30秒 |

## 7. 常见问题清单

### 7.1 教师自评清单

| 领域 | 检查点 |
|------|--------|
| **知识** | 是否了解正念的理论基础和科学依据？ |
| **个人练习** | 是否有稳定的每日正念练习？ |
| **教学经验** | 是否接受过正规的正念教师培训？ |
| **伦理** | 是否了解正念教学的伦理边界？ |
| **督导** | 是否有定期的教学督导？ |
| **继续教育** | 是否持续学习和更新知识？ |

### 7.2 课程质量评估

| 维度 | 指标 |
|------|------|
| **内容 fidelity** | 是否按照标准协议实施？ |
| **教师能力** | 引导语的清晰度和稳定性 |
| **团体氛围** | 安全、接纳、非评判 |
| **参与者满意度** | 课程评价问卷 |
| **退出率** | 无故退出比例（目标<20%） |
| **完成率** | 完成全部课程的比例（目标>70%） |
