## 1. 安全管理体系总览

### 1.1 安全管理原则

| 原则 | 内容 | 实践要点 |
|------|------|----------|
| **非伤害（Ahimsa）** | 不对学员造成身心伤害 | 充分告知风险，尊重自主权 |
| **知情同意** | 学员了解练习内容与风险 | 课程前书面/口头知情同意 |
| **边界清晰** | 明确教学边界，不越界诊疗 | 出现病理症状及时转介 |
| **持续监测** | 全程关注学员状态变化 | 课前评估、课中观察、课后跟进 |
| **最小风险** | 从低强度开始渐进引导 | 不急于推进深层体验 |

### 1.2 安全管理流程

```
课前阶段                    课中阶段                    课后阶段
┌─────────────┐         ┌─────────────┐         ┌─────────────┐
│ 学员筛查评估 │────────→│ 持续状态监测 │────────→│ 反馈与跟进  │
│ 风险分级标注 │         │ 异常即时响应 │         │ 不良反应追踪│
│ 环境安全检查 │         │ 动态调整强度 │         │ 转介与记录  │
│ 知情同意签署 │         │ 紧急预案启动 │         │ 经验复盘    │
└─────────────┘         └─────────────┘         └─────────────┘
```

## 2. 学员筛查与风险评估

### 2.1 入学前健康问卷（核心条目）

| 类别 | 询问内容 | 风险等级 |
|------|----------|----------|
| 精神健康 | 是否有焦虑症、抑郁症、PTSD、精神分裂等诊断史 | 高 |
| 精神健康 | 是否正在服用精神科药物 | 中 |
| 精神健康 | 近6个月是否有自杀意念或自伤行为 | 极高 |
| 创伤经历 | 是否经历过重大创伤事件（暴力、虐待、灾难等） | 高 |
| 创伤经历 | 是否正在进行创伤治疗 | 中 |
| 身体健康 | 是否有癫痫、心脏病、严重慢性疼痛 | 中 |
| 冥想经历 | 过去冥想练习中是否有过不适或不良反应 | 中 |
| 物质使用 | 是否有酒精或药物依赖/戒断 | 高 |
| 近期重大变故 | 近3个月是否有重大生活事件（丧亲、离婚、失业等） | 中 |

### 2.2 风险分级与处理策略

| 风险等级 | 判定标准 | 处理策略 |
|----------|----------|----------|
| **绿色（低风险）** | 无精神健康诊断、无急性压力源、冥想经验正常 | 正常教学流程 |
| **黄色（中风险）** | 有心理治疗史但稳定、轻度焦虑/抑郁、有冥想不适经历 | 加强监测、课后跟进、可能需要医疗许可 |
| **橙色（较高风险）** | 近期心理危机、正在服药调整期、复杂创伤史 | 需治疗师许可、一对一教学、降低练习强度 |
| **红色（高风险）** | 活跃自杀意念、精神病性症状、急性创伤反应 | **不建议参加团体冥想**，转介专业治疗 |

## 3. 不良反应识别与处理

### 3.1 常见不良反应分类

| 类别 | 表现 | 发生率 | 严重程度 |
|------|------|--------|----------|
| **焦虑/恐慌** | 心悸、呼吸困难、过度换气、恐惧感 | 5-8% | 中 |
| **解离** | 感到不真实、与身体脱节、时空混乱 | 3-5% | 中高 |
| **情绪淹没** | 大量负面情绪涌现、无法自控的哭泣 | 8-12% | 中 |
| **身体不适** | 头痛、恶心、麻木、刺痛、疼痛加剧 | 5-10% | 低-中 |
| **睡眠障碍** | 失眠、噩梦、嗜睡 | 4-7% | 低-中 |
| **自我感丧失** | 持续的"我"感觉模糊、存在感动摇 | 1-3% | 高 |
| **闪回/创伤再现** | 创伤记忆以画面/身体感受重新出现 | 2-5% | 高 |
| **躁动/不安** | 无法静坐、强烈想要逃离、烦躁 | 6-10% | 中 |
| **错觉/幻觉** | 光影、声音、身体变形感 | 1-2% | 高 |

### 3.2 即时处理流程

```
发现异常信号
    │
    ├── 轻度（躁动、轻微不适）
    │   └── 轻声引导"张开眼睛，回到房间"
    │       └── 提供饮水、允许改变姿势或暂停
    │           └── 课后个别跟进
    │
    ├── 中度（情绪淹没、解离）
    │   └── 暂停引导，使用接地技术（Grounding）
    │       │   "感觉双脚踩在地上，按压椅子扶手"
    │       │   "说出你看到的5样东西"
    │       └── 课后1对1谈话，评估是否需要转介
    │
    └── 重度（恐慌发作、闪回、解体感）
        └── 停止课程或安排助教处理其他学员
            └── 启动个别危机干预
                ├── 确保物理安全
                ├── 接地与稳定化
                ├── 呼吸协助（不要求闭眼）
                └── 必要时联系紧急联系人/医疗机构
```

### 3.3 接地技术工具箱

| 技术 | 方法 | 适用场景 |
|------|------|----------|
| **5-4-3-2-1感官法** | 说出5样看到的、4样摸到的、3样听到的、2样闻到的、1样尝到的 | 解离、恐慌 |
| **双脚接地** | 感受双脚与地面的接触，缓慢踩踏 | 焦虑、不真实感 |
| **3-3-3呼吸** | 吸气3秒、屏息3秒、呼气3秒 | 轻度焦虑 |
| **身体按压** | 双手交叉抱臂交替按压（蝴蝶拍） | 情绪淹没、闪回 |
| **温度刺激** | 冰水洗手、握冰块 | 急性解离、恐慌 |
| **安全锚定** | 提前设定安全词/安全动作，危机时使用 | 已知创伤史学员 |

## 4. 创伤知情教学框架

### 4.1 创伤知情六大原则

1. **安全（Safety）**：物理环境与心理空间都要让人感到安全
2. **信任（Trustworthiness）**：透明、一致、可预测的教学方式
3. **选择（Choice）**：尊重学员的自主选择权，随时可以暂停
4. **协作（Collaboration）**：教学是合作关系，不是权威指令
5. **赋权（Empowerment）**：肯定学员的力量与能力
6. **文化敏感（Cultural Sensitivity）**：尊重身份认同与文化背景

### 4.2 创伤知情教学实践

| 教学环节 | 创伤知情做法 | 避免行为 |
|----------|-------------|----------|
| 开场 | 说明"可以随时改变姿势、睁眼、暂停" | "请务必闭上眼睛" |
| 引导 | 提供多个选择（"可以选择关注呼吸或声音"） | 唯一指令式引导 |
| 身体扫描 | 鼓励"从外部观察"，不要求深入感受 | 要求深入感受痛点区域 |
| 闭眼引导 | "可以选择闭眼或柔和地注视前方一点" | 强制闭眼 |
| 触碰 | 不触碰学员，除非得到明确许可 | 未经同意触碰调整姿势 |
| 闭关静修 | 提前告知可能的强度，提供退出机制 | 强调"坚持就是胜利" |

## 5. 紧急预案

### 5.1 紧急情况分类与响应

| 情况 | 识别信号 | 响应步骤 |
|------|----------|----------|
| **恐慌发作** | 呼吸急促、颤抖、表述"我要死了" | 停止引导→接地技术→辅助呼吸→评估医疗需求 |
| **解离发作** | 眼神空洞、无反应、自言自语 | 轻声叫名字→物理接地→不触碰→缓慢引导回神 |
| **情绪崩溃** | 大声哭泣、无法自控、蜷缩 | 给予空间→递纸巾/水→不急于安慰→等情绪自然流动 |
| **闪回发作** | 惊恐表情、身体僵硬、重复性动作 | 安全提醒"你在这里，现在是安全的"→接地→不追问内容 |
| **身体急症** | 胸痛、晕厥、抽搐 | 呼叫急救→保持安全体位→不离场→记录时间线 |
| **自杀意念表达** | 课后透露绝望、想消失 | 认真对待→不保密承诺→联系危机热线/紧急联系人→记录备案 |

### 5.2 场地安全检查清单

| 检查项 | 标准 | 频率 |
|--------|------|------|
| 出口通道 | 无遮挡，标识清晰 | 每次课前 |
| 急救包 | 在有效期内，位置明确 | 每周检查 |
| 紧急联系人信息 | 学员紧急联系人表就位 | 每次课前 |
| 温度与通风 | 20-24°C，空气流通 | 每次课前 |
| 光线 | 可调光，有柔和照明选项 | 每次课前 |
| 座椅/坐垫 | 稳固、无破损 | 每次课前 |
| 噪音控制 | 无突发噪音源 | 每次课前 |

## 6. 安全事件记录与报告

### 6.1 事件记录模板

```
【安全事件报告】
日期：            时间：            地点：
记录人：          涉及学员（代号）：

事件类型：□ 恐慌发作 □ 解离 □ 情绪崩溃 □ 闪回
          □ 身体急症 □ 自伤/自杀风险 □ 其他

事件描述（客观事实）：
─────────────────────
─────────────────────

即时处理措施：
─────────────────────

后续跟进计划：
─────────────────────

是否需要转介：□ 是 → 转介至：___________
               □ 否

学员当前状态：□ 已恢复 □ 需持续关注 □ 已转介
```

### 6.2 报告制度

| 事件级别 | 报告对象 | 时间要求 |
|----------|----------|----------|
| 轻度（课后自行恢复） | 督导 | 7天内 |
| 中度（需要课后跟进） | 督导+机构负责人 | 48小时内 |
| 重度（需要转介/医疗） | 督导+机构负责人+保险 | 24小时内 |
| 危急（自杀/严重伤害） | 机构负责人+保险+法律顾问 | 立即 |
