## 1. 职业伦理准则

### 1.1 核心伦理原则

| 原则 | 内涵 | 实践指南 |
|------|------|----------|
| **无害（Non-harming）** | 不对学员造成身心伤害 | 充分评估风险、及时转介、不强制推进练习 |
| **善行（Beneficence）** | 以学员福祉为核心 | 持续提升专业能力、基于学员需求调整教学 |
| **自主（Autonomy）** | 尊重学员的选择权 | 知情同意、不操控、允许退出 |
| **公正（Justice）** | 公平对待所有学员 | 不歧视、无偏见、平等服务 |
| **诚实（Honesty）** | 真实呈现自己与课程 | 不夸大效果、不虚报资质、如实说明局限 |
| **保密（Confidentiality）** | 保护学员隐私 | 分享内容保密、资料加密存储、例外情况告知 |

### 1.2 伦理决策框架

```
面对伦理困境时的决策流程：

1. 识别问题
   └── 具体是什么伦理冲突？
       └── 涉及哪些原则？

2. 收集信息
   └── 相关事实是什么？
       └── 各方利益相关者是谁？

3. 参考标准
   └── 行业伦理守则如何规定？
       └── 法律要求是什么？

4. 咨询督导/同行
   └── 寻求外部视角
       └── 记录咨询过程

5. 做出决定
   └── 选择最小伤害方案
       └── 记录决策理由

6. 反思评估
   └── 结果是否合理？
       └── 是否需要调整？
```

## 2. 边界管理

### 2.1 专业边界清单

| 边界类别 | 合规行为 | 越界行为（应避免） |
|----------|----------|-------------------|
| **身体边界** | 拍肩前询问许可 | 未经同意的触碰、不恰当的身体接触 |
| **情感边界** | 关心但不卷入学员情绪 | 与学员恋爱、过度情感依赖 |
| **财务边界** | 透明收费、合同清晰 | 向学员借钱、推销非相关产品 |
| **时间边界** | 合理的课程与咨询时间 | 课后长时间私下交流、随叫随到 |
| **关系边界** | 专业教学关系 | 双重关系（如同时是学员的房东、雇主） |
| **权力边界** | 尊重学员自主性 | 利用"灵性权威"操控学员决定 |
| **社交媒体** | 适度互动 | 窥探学员私生活、不当评论 |

### 2.2 灵性权威陷阱

| 模式 | 表现 | 危害 | 预防策略 |
|------|------|------|----------|
| **灵性绕道** | 用冥想概念回避真实问题 | 延误治疗、压抑情绪 | 识别超出能力范围的问题，及时转介 |
| **权威依赖** | 学员过度依赖执行师指导 | 丧失自主性、退行 | 强调学员内在权威，避免"导师崇拜" |
| **情感操控** | "你不够精进""你不够放下" | 自责、羞耻、丧失自信 | 从不评判学员的练习体验 |
| **神秘化** | 用玄学解释正常现象 | 误导、恐惧、依赖 | 科学解释优先，尊重但不鼓励神秘体验 |
| **排他性** | "只有这种方法才正确" | 狭隘、排斥其他有效方法 | 尊重多元方法，推荐学员多方探索 |

### 2.3 社交媒体使用伦理

| 场景 | 建议做法 | 禁止行为 |
|------|----------|----------|
| 发布课程信息 | 专业、真实、简洁 | 夸大疗效、使用"治愈"等医疗用语 |
| 分享学员故事 | 获得书面授权、匿名化处理 | 未经同意使用学员照片/故事 |
| 在线互动 | 回应一般性询问 | 提供个性化诊断或治疗建议 |
| 个人生活分享 | 适度展示专业形象 | 过度展示奢华生活造成攀比 |
| 评论争议话题 | 可表达专业观点 | 强加个人价值观于学员 |

## 3. 保密原则

### 3.1 保密范围与例外

| 类别 | 保密内容 | 保密例外（可披露） |
|------|----------|-------------------|
| **个人信息** | 姓名、联系方式、健康信息 | 学员书面授权 |
| **课程内容** | 分享的具体内容、情绪表达 | 有伤害自身或他人的风险 |
| **评估数据** | 量表结果、评估记录 | 法律强制要求 |
| **不良反应** | 练习中出现的不适 | 需要医疗转介时（最低限度披露） |

### 3.2 保密实践指南

| 环节 | 做法 |
|------|------|
| **课前** | 在知情同意书中说明保密范围与例外 |
| **课中** | 引导分享时提醒"在这里分享的内容请留在课堂" |
| **记录** | 学员档案加密存储、使用代号而非真名 |
| **督导** | 督导中讨论案例时隐去可识别信息 |
| **营销** | 使用学员故事/评价前获得书面授权 |
| **线上** | 课程录像需告知学员并获得同意 |

## 4. 法律合规

### 4.1 资质与许可

| 地区 | 冥想教学资质要求 | 相关法规 |
|------|------------------|----------|
| **中国大陆** | 无国家级准入门槛 | 需遵守《职业教育法》《心理健康服务规范》 |
| **美国** | 无统一联邦要求 | 各州规定不同，涉及治疗需州执照 |
| **英国** | 无强制要求 | 建议加入 BACP / UKCP 获得行业认可 |
| **澳大利亚** | 无强制要求 | 建议遵循 PACFA / ACA 伦理准则 |
| **日本** | 无专门法规 | 建议获得相关协会认证 |

### 4.2 必备法律文件

| 文件 | 用途 | 建议内容 |
|------|------|----------|
| **知情同意书** | 学员签署 | 课程说明、风险告知、保密范围、紧急联系人 |
| **服务协议** | 明确双方权利义务 | 服务内容、价格、退款政策、取消规则 |
| **隐私政策** | 数据保护合规 | 收集哪些数据、如何使用、如何保护 |
| **免责声明** | 限定责任范围 | 冥想不能替代医疗、不保证特定效果 |
| **保险单** | 风险转移 | 职业责任险、公众责任险 |

### 4.3 知情同意书模板

```
【冥想课程知情同意书】

一、课程说明
本课程为冥想引导与正念练习课程，由 [执行师姓名] 提供。
冥想是一种注意力训练方法，不构成医疗诊断或治疗。

二、健康告知
请如实告知以下情况（信息将被保密）：
□ 我有精神健康诊断史
□ 我正在服用精神科药物
□ 我有癫痫或严重心血管疾病
□ 我近期经历了重大创伤事件
□ 我正在接受心理治疗
□ 我有冥想练习中不适的经历

三、风险了解
我了解冥想练习可能出现暂时的情绪波动、身体不适等现象，
我同意在出现不适时及时告知执行师。

四、保密协议
执行师将对我的个人信息和课堂分享保密，以下情况除外：
（1）存在伤害自身或他人的风险
（2）法律强制要求披露

五、权利确认
- 我有权随时中止练习并不需要说明理由
- 我不会被强制做任何不舒服的事情
- 我可以提出问题并获得回答

学员签名：          日期：
紧急联系人：        联系电话：
```

## 5. 保险与风险管理

### 5.1 保险类型

| 保险类型 | 覆盖范围 | 是否必须 | 参考费用 |
|----------|----------|----------|----------|
| **职业责任险** | 教学过程中导致的人身/精神损害 | 强烈建议 | ¥2000-8000/年 |
| **公众责任险** | 场地内意外伤害 | 如有自有场地必须 | ¥1000-5000/年 |
| **收入保障险** | 因伤病无法工作的收入补偿 | 推荐 | 视保额而定 |
| **设备保险** | 教学设备的意外损失 | 视设备价值 | ¥500-2000/年 |

### 5.2 风险矩阵

| 风险 | 可能性 | 影响 | 预防措施 | 应急预案 |
|------|--------|------|----------|----------|
| 学员恐慌发作 | 中 | 中 | 课前筛查、创伤知情教学 | 接地技术、课后跟进 |
| 学员指控不当行为 | 低 | 极高 | 清晰边界、避免独处 | 法律顾问、保险理赔 |
| 学员自杀/自伤 | 极低 | 极高 | 风险评估、知情同意 | 紧急联系人、危机热线 |
| 版权纠纷 | 中 | 中 | 原创内容、标注引用 | 下架争议内容 |
| 数据泄露 | 低 | 高 | 加密存储、最小化收集 | 通知当事人、整改 |

## 6. 投诉处理

### 6.1 投诉处理流程

```
收到投诉
    │
    ├── 1. 记录（24小时内书面记录投诉内容）
    │
    ├── 2. 确认（向投诉人确认收到，承诺回应时间）
    │
    ├── 3. 调查
    │   ├── 收集事实（双方陈述、证人、记录）
    │   ├── 咨询督导/法律顾问
    │   └── 评估严重程度
    │
    ├── 4. 回应（在承诺时间内给予回复）
    │   ├── 说明调查结果
    │   ├── 提出补救措施
    │   └── 如有错误，真诚道歉
    │
    ├── 5. 改进（记录教训，调整流程）
    │
    └── 6. 归档（保密存档，保留3-5年）
```

### 6.2 投诉处理原则

- **认真对待每一条投诉**：不辩解、不轻视
- **及时回应**：24小时内确认，7天内实质性回复
- **公平公正**：听取双方陈述
- **保密处理**：限制知情范围
- **持续改进**：每次投诉都是改善的机会
