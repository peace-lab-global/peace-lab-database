## 一、七维能力速记 (7DCM)

```
        C = Concentration 专注力
           ↓ 支撑
        SC = Sensory Clarity 感官清晰度
           ↓ 精细化
        MA = Metacognitive Awareness 元认知觉察
           ↓ 觉察到
        E  = Equanimity 平静度
           ↓ 调节
        SR = Self-Regulation 自我调节
           ↓ 整合
        CL = Compassion & Loving-Kindness 慈悲能力
           ↓ 体现
        BQ = Being Quality 存在品质
```

**各维度一句话定义**：

| 维度 | 核心问句 | 关键量表 | 神经标记 |
|------|----------|----------|----------|
| **C** | "你能把注意力稳定地放在一个地方多久？" | ANT/SART/呼吸计数 | EEG α同步；注意网络效率 |
| **SC** | "你能多精细地分辨身体感受和情绪？" | MAIA/心跳检测 | 前岛叶灰质密度 |
| **E** | "情绪来了，你能不跟着跑吗？" | 情绪Stroop/冷压/CPT | 杏仁核反应性↓ |
| **MA** | "你知道自己正在想什么吗？" | ESM/思维探针/MAAS | DMN活性↓ |
| **BQ** | "你的在场本身是否能传递安定？" | MBI-TAC D3/360度反馈 | 自主神经同步性 |
| **SR** | "你能主动调节自己的注意和情绪吗？" | DERS/Go-No-Go/HRV | vmPFC-杏仁核调节 |
| **CL** | "你能在痛苦中对自己和他人保持善意吗？" | SCS/CS/CFQ | vmPFC/隔区关怀回路 |

## 二、九级水平速查 (9LM)

### 2.1 快速判定卡

| 层级 | 时长 | 核心能力标志 | 风险 | 快速判断法 |
|------|------|-------------|------|-----------|
| **L0** | <20h | 能静坐5min，跟随呼吸 | 🟢低 | 初学者，需引导 |
| **L1** | 20-100h | 规律练习，觉察走神-拉回 | 🟢低 | 问："你能觉察到自己走神吗？" |
| **L2** | 100-500h | 2种技术，日常正念初现 | 🟡低-中 | 问："你能区分专注和强迫吗？" |
| **L3** | 500-2000h | 稳定专注≥15min，FFMQ≥135 | 🟡中 | 问："你能带领他人基础冥想吗？" |
| **L4** | 2k-5kh | 深度禅定/开放觉知≥30min | 🟠中-高 | 问："你完成过7天闭关吗？" |
| **L5** | 5k-10kh | 非二元glimpses，NADA≥3.5 | 🔴高 | 问："你有过'无我'体验吗？" |
| **L6** | 10k-20kh | 非二元常见，觉知渗透睡眠 | 🟡中 | 问："睡眠中保持觉知吗？" |
| **L7** | >20kh | 非二元默认，任运自在 | 🟢低 | 需多位导师验证 |
| **L8** | 超越时间 | 不可评估 | — | 传承持有者确认 |

### 2.2 发展序列口诀

> **"专清先，觉平跟，调慈整，存在成"**
>
> 早期练专注和清晰度 → 中期练觉察和平静 → 中晚期整合调节和慈悲 → 高级体现存在品质

## 三、量表选择决策树

```
第一步：被评估者是谁？
│
├─ 儿童(6-12) → CAMM / MAAS-C + 父母报告
├─ 青少年(13-18) → MAAS-A / CHIME-A + SCS-Y
├─ 成人(18-65)
│   └─ 第二步：什么目的？
│       ├─ 基础筛查/大规模研究 → MAAS 或 FFMQ-15（5分钟）
│       ├─ 临床干预前后测 → FFMQ + PSS-10 + PHQ-9 + GAD-7 + MIOME（25分钟）
│       ├─ 能力进阶评估 → FFMQ + NADA-T + MAIA + SDMS + SCS（40分钟）
│       ├─ 高阶传统修习 → NADA-T/S + MEQ-30 + FMI + 现象学访谈（90分钟）
│       └─ 安全监测 → MIOME + MACE（每次随访）
├─ 老年人(65+) → MoCA(必做) + FFMQ + GDS-15 + 适应性任务
├─ 孕期/产后 → EPDS(必做) + 调整HRV基线
└─ 临床人群 → 见下方特殊人群调整
```

| **MAAS** | 15 | 3min | L0-L4 | .82-.87 | 36/54/72 |

| **GAD-7** | 7 | 2min | 临床 | .89-.92 | 5/10/15 |
| **MAIA** | 32 | 6min | L1-L5 | .66-.87 | 各子维度 |

## 四、安全筛查红灯清单

### 4.1 绝对红灯（暂停水平判定，仅保留安全监测）

| 红灯 | 筛查工具 | 阈值 | 立即行动 |
|------|----------|------|----------|
| 🔴 **主动自杀意念** | PHQ-9第9项 | >0 | 危机评估 + 安全计划 + 转介精神科 |
| 🔴 **急性躁狂** | MDQ/HCL-32 | 阳性 | 改期至稳定期；精神科评估 |
| 🔴 **未控制精神病** | PSYRATS/PANSS | 阳性 | 精神科评估；禁止开放监控 |
| 🔴 **严重解离危机** | DES-II | ≥15 + 功能受损 | 临床心理师紧急评估 |
| 🔴 **MIOME重度** | 任何维度 | ≥4分 | 启动不良反应协议 |
| 🔴 **急性物质中毒** | 自报 | 评估前48h内 | 改期 |

### 4.2 黄灯（可评估但需调整路径）

| 黄灯 | 调整 |
|------|------|
| 🟡 PHQ-9 10-14（中度抑郁） | 简化评估；增加安全监测频率 |
| 🟡 GAD-7 10-14（中度焦虑） | 同上 |
| 🟡 PCL-5 ≥33（PTSD） | 创伤知情路径；禁止密集闭关评估 |
| 🟡 CTQ任一维度≥中度 | 创伤知情路径 |
| 🟡 MIOME中度（11-30） | 调整练习方案；增加督导 |
| 🟡 孕期 | EPDS必做；调整体位和HRV基线 |
| 🟡 老年MoCA<26 | 排除认知衰退混淆；简化认知任务 |

## 五、风险响应速查卡

### 5.1 不良反应响应流程

```
MIOME任何维度≥4 或 总分>30
    ↓
┌─────────────────┐
│ ① 立即减量/暂停  │
│ ② 启动不良反应协议│
└────────┬────────┘
         ↓
┌─────────────────────────────┐
│ 按维度响应：                   │
│ 认知≥4 → 评估解离/精神病性     │
│ 感知≥4 → 评估感知紊乱/癫痫     │
│ 情感≥4 → 评估情绪危机/自杀     │
│ 躯体≥4 → 评估躯体疾病/自主神经 │
│ 存在≥4 → 评估存在危机/灵性危机 │
└─────────────────────────────┘
         ↓
┌─────────────────┐
│ ③ 专业评估      │
│ ④ 调整方案      │
│ ⑤ 48h内安全电话 │
└─────────────────┘
```

### 5.2 修行黑夜(Dark Night)识别与响应

| 特征 | 普通困难 | 修行黑夜 | 临床危机 |
|------|----------|----------|----------|
| 恐惧/存在焦虑 | 轻微，可耐受 | 强烈但可被容纳 | 压倒性，无法忍受 |
| 自我感动摇 | 表层 | 深层但整合中 | 崩溃性 |
| 功能影响 | 最小 | 暂时降低 | 严重受损 |
| 持续时间 | 数天 | 数周至数月 | 持续恶化 |
| 支持响应 | 同伴支持 | **有经验的导师+可能的心理治疗** | **临床干预** |
| 是否停止练习 | 通常不必 | 可能需要减量 | 通常需要暂停 |

### 5.3 灵性绕道(Spiritual Bypassing)识别

| 标志 | 说明 |
|------|------|
| 用"不评判"回避愤怒 | 不处理冲突，只说"我不该生气" |
| 用"放下"回避承诺 | 逃避责任，说"我要放下执着" |
| 用"空性"回避情感 | 情感隔离，说"一切都是空" |
| 过度追求"高阶体验" | 忽视日常关系和责任 |
| 评判"低阶"练习者 | 灵性傲慢 |
| 情感麻木被误认为平静 | 无法感受悲伤或愤怒 |

**响应**：转介整合性心理治疗师；减少冥想；增加心理探索。

## 六、特殊人群调整要点

### 6.1 一句话调整

| 人群 | 核心调整 |
|------|----------|
| **儿童** | 仅L0-L3；时长≤30min；游戏化；父母报告 |
| **青少年** | 关注身份认同；同伴影响；隐私保护 |
| **老年人** | MoCA必做；靠背椅默认；趋势>绝对值 |
| **孕期** | EPDS必做；左侧卧/靠背椅；调整HRV基线 |
| **ADHD** | ANT/SART简化；注意力稳定性专项；行为任务优先 |
| **ASD** | 感官友好；书面回答；照顾者报告权重高 |
| **PTSD** | 创伤知情路径；禁止开放监控；grounding优先 |
| **精神病性** | 活跃期禁止开放监控；PANSS必做；精神科共同评估 |
| **双相** | 躁狂期禁止评估；稳定≥6月后可做 |
| **进食障碍** | 避免身体扫描；EDE-Q增加；关注身体觉察风险 |
| **军人/急救** | mTBI筛查；道德伤害评估；保密需求高 |

### 6.2 月经周期（育龄女性）

> **卵泡期 = 最佳评估窗口** | 黄体期HRV自然降低，避免基线测量

## 七、评估禁区一览

**以下情形绝对禁止进行水平判定：**

1. ❌ 急性精神健康危机（自杀/躁狂/精神病活跃期）
2. ❌ 未获得知情同意
3. ❌ 评估者存在利益冲突（同时销售课程/浪漫关系）
4. ❌ 缺乏合格现象学访谈员却判定L4+
5. ❌ 仅基于单一方法判定
6. ❌ 急性MIOME危机（任何维度≥4）
7. ❌ 未控制解离障碍<6个月稳定期
8. ❌ 评估者文化胜任力不足
9. ❌ 未成年人<18岁判定L5+
10. ❌ 商业竞争/社群排名目的
11. ❌ 评估被用作强制留团/晋升/惩罚条件
12. ❌ 数据未经同意用于AI训练

## 八、关键数值记忆卡

### 8.1 心理测量学底线

```
Cronbach's α ≥ 0.70    （内部一致性最低）
重测信度 r ≥ 0.70      （间隔2-4周）
ICC ≥ 0.80             （量表评分者间）
ICC ≥ 0.75             （MBI-TAC录像评审）
Kappa ≥ 0.70           （现象学编码）
效应量 d ≥ 0.50        （量表敏感性最低）
RCI > 1.96             （个体可靠变化）
```

### 8.2 HRV关键值

```
RMSSD基线 <20ms     → 慢性应激/就医检查
RMSSD冥想中↑>20%   → 有效副交感激活
RMSSD无变化/降低    → 可能过度努力/焦虑
LF/HF比 <2.0        → 交感-副交感平衡改善
呼吸频率 <8次/分    → 深度放松状态
```

### 8.3 环境标准

```
背景噪音 ≤35 dB
室温 22-24°C
照明 200-300 lux
咖啡因禁用时 评估前≥6小时
酒精禁用时 评估前≥24小时
剧烈运动禁用时 评估前≥2小时
睡眠要求 前夜≥6小时
```

### 8.4 水平判定黄金法则

```
水平 = MAX(时长下限, 能力指标≥80%, 神经标记, 传统验证)
      × 无严重不良反应
      × 无否决条件
```

### 8.5 概率化判定

```
某水平后验概率 ≥70% 且 差距 ≥30% → 明确判定
最高概率 <70% 或 差距 <30%      → 边界判定（过渡期）
非典型模式                      → 专家委员会复核
```

### 8.6 纵向追踪时间点

```
T0   基线      → 全套评估
T1M  1个月    → FFMQ-15 + ESM
T3M  3个月    → FFMQ + MAAS + MIOME
T6M  6个月    → 半套评估 + 访谈(L3+)
T12M 12个月   → 全套评估（年度确认）
特殊 MIOME任何维度≥3 → 专项评估
```
