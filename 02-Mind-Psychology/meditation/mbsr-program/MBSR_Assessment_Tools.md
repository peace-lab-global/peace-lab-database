## 1. 评估框架

### 1.1 三层评估模型

基于 [冥想水平与能力评估总纲](../overview/Meditation_Level_Ability_Assessment_Standard.md) 的 **5DCM模型**，MBSR评估聚焦于**L1-L3**水平的能力发展：

```
第一层：结果评估（Outcome）        → 课程是否有效？
        └─ 压力、焦虑、抑郁、生活质量
        
第二层：过程评估（Process）         → 正念能力是否提升？
        └─ FFMQ各维度、MAAS、自我慈悲
        
第三层：教学评估（Teaching Quality） → 教师/课程质量如何？
        └─ MBI-TAC、学员满意度、课程fidelity
```

### 1.2 MBSR专用评估原则

| 原则 | 说明 |
|------|------|
| **标准化** | 使用经学术验证的量表，确保结果可比较 |
| **最小负担** | 选择最简有效工具组合，避免评估疲劳 |
| **安全优先** | 每次评估包含不良反应筛查（MIOME） |
| **循证解释** | 基于已发表的效应量解释个体变化 |

## 2. 正念能力测量工具

### 2.1 五因素正念量表（FFMQ）— MBSR金标准

| 项目 | 详情 |
|------|------|
| **全称** | Five Facet Mindfulness Questionnaire |
| **作者** | Baer et al. (2006) |
| **题目数** | 39题（简版24题 / FFMQ-15） |
| **评分** | 5点Likert量表 |
| **信度** | alpha = 0.75-0.91；重测信度 r=0.81 |

**MBSR预期变化（基于Carmody & Baer, 2008 元分析）**：

| 维度 | 预期效应量(d) | 8周变化幅度 | 临床意义 |
|------|--------------|-------------|----------|
| 观察 | 0.40-0.60 | +5-10分 | 中等 |
| 描述 | 0.50-0.70 | +5-10分 | 中-大 |
| 觉知行动 | 0.60-0.90 | +8-15分 | 大 |
| 不评判 | 0.50-0.80 | +5-12分 | 中-大 |
| 不反应 | 0.60-0.90 | +3-8分 | 大 |
| **总分** | **0.60-0.90** | **+15-35分** | **大** |

> **使用建议**：前测/后测/随访标配量表；推荐39项版用于研究，FFMQ-15用于临床快速筛查。

### 2.2 正念注意觉知量表（MAAS）

| 项目 | 详情 |
|------|------|
| **全称** | Mindful Attention Awareness Scale |
| **作者** | Brown & Ryan (2003) |
| **题目数** | 15题 |
| **特点** | 单维度；全部反向计分 |
| **适用** | 快速筛查、大规模研究 |

**MBSR预期变化**：效应量 d = 0.50-0.80；8周后平均增加 8-15分。

### 2.3 其他正念量表

| 量表 | 缩写 | 题数 | 特色 | MBSR适用性 |
|------|------|------|------|-----------|
| Toronto Mindfulness Scale | TMS | 13 | 状态正念 | 单次课程前后 |
| Philadelphia Mindfulness Scale | PHLMS | 20 | 觉察+接纳 | 替代选项 |
| Cognitive Affective Mindfulness Scale | CAMS-R | 12 | 简短 | 快速筛查 |
| Freiburg Mindfulness Inventory | FMI | 14/30 | 适合有经验者 | MBSR毕业后追踪 |

| 评估项 | MBSR标准 | 评估方法 |
|--------|----------|----------|
| **课程时长** | 8周 × 2.5小时 + 全日 retreat | 出勤记录 |

| **主题内容** | 8周标准主题（自动导航、压力反应等） | 课程大纲核对 |

## 6. 数据分析与解释

### 6.1 统计分析方法

| 分析类型 | 方法 | 适用 | 效应量指标 |
|----------|------|------|-----------|
| 前后比较 | 配对t检验 / Wilcoxon符号秩 | 单组前后测 | Cohen's d |
| 组间比较 | 混合模型 / ANCOVA | RCT设计 | Hedges' g |
| 效应量 | Cohen's d / Hedges' g | 效果大小 | — |
| 临床显著性 | 可靠变化指数(RCI) | 个体层面变化 | RCI > 1.96 |
| 剂量效应 | 回归分析/相关分析 | 练习量与效果 | r / beta |
| 网络分析 | 网络心理测量学 | 症状-正念关系 | 边权重 |

### 6.2 MBSR效应量参考标准

基于最新元分析（Kral et al., 2018; Creswell et al., 2019）：

| 结局指标 | 短期效应(d) | 长期效应(d, 3-6月) | 临床意义 |
|----------|------------|---------------------|----------|
| 压力(PSS) | 0.70-1.00 | 0.50-0.70 | 大 |
| 焦虑 | 0.60-0.90 | 0.40-0.70 | 中-大 |
| 抑郁 | 0.60-0.90 | 0.40-0.70 | 中-大 |
| 正念(FFMQ) | 0.60-0.90 | 0.40-0.60 | 中-大 |
| 疼痛 | 0.30-0.60 | 0.20-0.50 | 中 |
| 睡眠质量 | 0.50-0.80 | 0.30-0.60 | 中-大 |
| 生活质量 | 0.50-0.70 | 0.30-0.50 | 中 |

### 6.3 个体层面变化解释

**可靠变化指数（RCI）计算**：

$$
RCI = \frac{X_{post} - X_{pre}}{S_{diff}}
$$

其中 $S_{diff} = \sqrt{2 \times (SE)^2}$，$SE = SD \times \sqrt{1 - r_{xx}}$

| RCI值 | 解释 |
|-------|------|
| < -1.96 | 可靠恶化 |
| -1.96 至 +1.96 | 无可靠变化 |
| > +1.96 | 可靠改善 |

## 7. 临床实践整合

### 7.1 每日练习日志（MBSR推荐）

| 记录内容 | 目的 | 示例 |
|---------|------|------|
| 练习日期/时间 | 追踪规律性 | 2026-05-01 07:00 |
| 练习类型 | 技术多样性 | 身体扫描 / 坐姿冥想 / 瑜伽 |
| 练习时长 | 剂量-反应关系 | 45分钟（正式）+ 多次非正式 |
| 专注度（1-10） | 主观质量 | 6/10 |
| 情绪变化（-5至+5） | 即时效果 | +2（更平静） |
| 困难/障碍 | 个性化调整 | 昏沉；第20分钟特别困 |
| 非正式练习 | 日常整合 | 正念进食早餐、正念步行上班 |
| 备注 | 补充信息 | 全日 retreat 后第三天 |

### 7.2 不良反应响应流程

```
MIOME评估
│
├─ 无不良反应（0分）
│   └─ 继续标准方案
│
├─ 轻度（1-10分）
│   └─ 记录；调整练习强度；下次随访复查
│
├─ 中度（11-30分）
│   └─ 调整练习类型（如从身体扫描改为坐姿）
│   └─ 增加督导支持
│   └─ 必要时减量
│
├─ 重度（31-60分）
│   └─ 暂停或大幅减量
│   └─ 一对一评估
│   └─ 考虑转介心理健康专业人员
│
└─ 严重（>60分 或 任何自杀意念）
    └─ 立即停止冥想练习
    └─ 紧急风险评估
    └─ 转介精神科/心理治疗
    └─ 建立安全计划
```

### 7.3 特殊人群评估调整

| 人群 | 调整建议 | 额外量表 |
|------|----------|----------|
| **创伤史** | 使用创伤知情版本；增加安全确认 | PCL-5（PTSD筛查） |
| **重度抑郁** | 增加PHQ-9频率；自杀风险监测 | C-SSRS（自杀筛查） |
| **慢性疼痛** | 疼痛相关量表为核心 | PCS + VAS + BPI |
| **癌症患者** | 关注疲劳、恐惧复发 | FCR（恐惧复发）+ FACIT-F |
| **青少年** | 使用年龄适配量表 | CAMM（儿童青少年正念） |
| **老年人** | 简化流程；关注认知变化 | MoCA（认知筛查） |
