# MBSR 评估工具汇编 | MBSR Assessment Tools

> **领域**：正念减压（Mindfulness-Based Stress Reduction）
> **关键词**：正念量表（Mindfulness Scales）、效果评估（Outcome Assessment）、FFMQ、PSS、程序评估（Program Evaluation）
> **上次更新**：2026-04

---

## 1. 评估框架

### 1.1 三层评估模型

```
第一层：结果评估（Outcome）         → 课程是否有效？
第二层：过程评估（Process）          → 正念能力是否提升？
第三层：教学评估（Teaching Quality）  → 教师/课程质量如何？
```

## 2. 正念能力测量工具

### 2.1 五因素正念量表（FFMQ）

| 项目 | 详情 |
|------|------|
| **全称** | Five Facet Mindfulness Questionnaire |
| **作者** | Baer et al. (2006) |
| **题目数** | 39题（简版24题） |
| **评分** | 5点Likert量表 |

**五个维度**：

| 维度 | 英文 | 示例题目 | 测量内容 |
|------|------|----------|----------|
| 观察 | Observing | "我注意到食物和饮料对我身体的影响" | 对内外刺激的注意 |
| 描述 | Describing | "我善于用语言描述我的感受" | 给内在体验贴标签 |
| 觉知行动 | Acting with Awareness | "我做事时很少走神"（反向） | 当前行为的觉察 |
| 不评判 | Non-judging | "我批评自己有不理性的想法"（反向） | 对内在体验不评判 |
| 不反应 | Non-reactivity | "我感知到情绪，但不被卷入" | 对内在体验的不反应性 |

**使用建议**：前测/后测/随访标配量表

### 2.2 正念注意觉察量表（MAAS）

| 项目 | 详情 |
|------|------|
| **全称** | Mindful Attention Awareness Scale |
| **作者** | Brown & Ryan (2003) |
| **题目数** | 15题 |
| **特点** | 单维度（注意觉察）；全部反向计分；简短易用 |
| **适用** | 快速筛查、大规模研究 |

### 2.3 其他正念量表

| 量表 | 缩写 | 题数 | 特色 |
|------|------|------|------|
| Philadelphia Mindfulness Scale | PHLMS | 20 | 觉察+接纳双维度 |
| Cognitive Affective Mindfulness Scale | CAMS-R | 12 | 简短；认知情感聚焦 |
| Toronto Mindfulness Scale | TMS | 13 | 状态正念（State Mindfulness） |
| Freiburg Mindfulness Inventory | FMI | 14/30 | 适合冥想经验者 |

## 4. 教学质量评估

### 4.1 MBI-TAC（正念干预教学评估标准）

| 维度 | 评估焦点 | 等级 |
|------|----------|------|
| 课程覆盖与组织 | 结构、节奏、内容覆盖 | 1-6 |
| 关系性技能 | 温暖、尊重、真诚 | 1-6 |
| 正念体现 | 教师自身正念品质 | 1-6 |
| 引导练习 | 引导词质量、节奏 | 1-6 |
| 传递主题 | 心理教育清晰度 | 1-6 |
| 学习环境 | 探询、团体促进 | 1-6 |

> 详细说明见 [冥想执行师评估与进阶](../practitioner-training/Practitioner_Assessment_Progression.md)

### 4.2 学员满意度评估

**课程满意度问卷（推荐内容）**：

| 维度 | 示例问题 |
|------|----------|
| 总体满意度 | "总体而言，您对MBSR课程的满意度是？"（1-10） |
| 教师评价 | "教师的教学能力如何？" |
| 内容适切性 | "课程内容与您的需求相关吗？" |
| 练习实用性 | "您在日常生活中能运用所学吗？" |
| 推荐意愿 | "您会推荐此课程给他人吗？"（NPS） |
| 开放反馈 | "您最大的收获是什么？""有什么改进建议？" |

## 5. 推荐评估方案

### 5.1 标准评估套件（Standard Battery）

| 时间点 | 量表 | 目的 |
|--------|------|------|
| **前测** | PSS-10 + FFMQ + PHQ-9 + GAD-7 + SF-36 | 基线 |
| **中期（W4）** | PSS-10（简版） | 进度监测 |
| **后测** | PSS-10 + FFMQ + PHQ-9 + GAD-7 + SF-36 + 满意度 | 效果评估 |
| **随访（3月）** | PSS-10 + FFMQ + 练习维持度 | 长期效果 |

### 5.2 精简评估套件（Brief Battery）

| 时间点 | 量表 | 用时 |
|--------|------|------|
| 前测 | PSS-10 + MAAS + PHQ-9 | 约10分钟 |
| 后测 | PSS-10 + MAAS + PHQ-9 + 满意度 | 约15分钟 |

### 5.3 数据分析建议

| 分析类型 | 方法 | 适用 |
|----------|------|------|
| 前后比较 | 配对t检验/Wilcoxon | 单组前后测 |
| 组间比较 | 混合模型/ANCOVA | RCT设计 |
| 效应量 | Cohen's d / Hedges' g | 效果大小 |
| 临床显著性 | 可靠变化指数（RCI） | 个体层面变化 |
| 剂量效应 | 回归分析 | 练习量与效果关系 |
