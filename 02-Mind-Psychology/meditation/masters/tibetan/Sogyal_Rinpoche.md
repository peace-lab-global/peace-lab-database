# 索甲仁波切（Sogyal Rinpoche / བསོད་ནམས་རྒྱལ་མཚན）

> 《西藏生死书》的作者——从畅销书大师到性侵被告的坠落

---

## 一、概述（Overview）

| 项目 | 信息 |
|------|------|
| 全名 | Sogyal Rinpoche (Sogyal Lakar) |
| 生卒 | 1947-2019 |
| 出生地 | 西藏 |
| 传统 | 宁玛派 |
| 代表作 | 《The Tibetan Book of Living and Dying》(1992) |
| 创立组织 | Rigpa International |
| 核心教导 | Dzogchen（大圆满）入门、死亡与临终关怀 |

---

## 二、《西藏生死书》的影响

- 全球销量 300 万+，翻译成 34 种语言
- 使"中阴"（Bardo）概念进入西方公众意识
- 推动了临终关怀中引入灵性维度的运动
- 被达赖喇嘛作序推荐
- 至今仍是藏传佛教最畅销的入门书之一

---

## 三、性侵丑闻与组织解体

### 3.1 指控时间线

| 时间 | 事件 |
|------|------|
| 1994 | 首次被曝光性行为不当（BBC 纪录片） |
| 2011 | Dakini Power 一书中有间接提及 |
| 2017 | 8 名前学员联名公开信详述身体/性/情感虐待 |
| 2018 | 辞去 Rigpa 所有职务 |
| 2019 | 去世 |

### 3.2 行业教训

- 灵性权威不应免于伦理审查
- 组织透明度和问责机制的必要性
- "疯智"（Crazy Wisdom）不能成为虐待的借口
- 受害者保护机制在灵性组织中同样重要
