## 1. 教学设计原则

### 1.1 冥想教学核心原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **体验先行** | 先引导体验，再讲解理论 | 先做5分钟呼吸冥想，再解释注意力机制 |
| **渐进递进** | 从短到长、从简单到复杂 | 5min→10min→20min→30min |
| **多元通道** | 视觉、听觉、身体、想象结合 | 呼吸（身体）+意象（视觉）+引导词（听觉） |
| **自主授权** | 强调学员的内在权威 | "注意你自己的感受，而不是我说了什么" |
| **安全容器** | 创造包容、不评判的空间 | "没有正确的冥想方式" |

### 1.2 课程节奏设计

```
标准60分钟课程节奏

0-5min    开场与安定
          ├── 签到、简短互动
          ├── 说明今日主题
          └── 邀请安定坐姿、初步调息

5-10min   简短引导冥想（5min）
          ├── 定心练习（呼吸/声音）
          └── 为后续教学建立内在空间

10-25min  主题教学
          ├── 理论讲解（简洁、要点化）
          ├── 示范/体验性练习
          └── 关键概念解析

25-40min  深度练习（15min）
          ├── 核心冥想引导
          └── 学员独立练习时间

40-50min  反思与分享
          ├── 引导性反思问题
          ├── 配对/小组分享（可选）
          └── 常见问题回应

50-55min  收束与整合
          ├── 今日要点总结
          ├── 课后练习建议
          └── 下次预告

55-60min  缓冲与个别交流
          ├── 学员可个别提问
          └── 观察需要跟进的学员
```

## 2. 引导技术

### 2.1 引导词撰写要领

| 要素 | 原则 | 示例 |
|------|------|------|
| **语速** | 比日常慢30-50%，留出空间 | "慢慢地……将注意力……带回到呼吸上" |
| **声调** | 温暖、稳定、不带命令感 | 邀请式而非指令式："你可以尝试……" |
| **停顿** | 关键指令前后留3-5秒空白 | "感受你的呼吸……（停顿3秒）……自然的节奏" |
| **意象** | 具体而非抽象 | "想象一团温暖的光"优于"想象正能量" |
| **选择** | 提供替代选项 | "如果关注呼吸不舒服，可以倾听周围的声音" |
| **回锚** | 定期拉回注意力 | 每隔2-3分钟重新锚定注意力 |

### 2.2 不同冥想技术的引导框架

#### 呼吸冥想（Breath Meditation）

```
1. 安定（Settling）- 2min
   "找到一个舒适的坐姿……感受身体的重量"

2. 锚定（Anchoring）- 3min
   "将注意力轻轻放在呼吸上……不需要改变什么……只是注意"

3. 深化（Deepening）- 5min
   "可以选择关注鼻尖的凉意……或者腹部的起伏……"

4. 扩展（Expanding）- 3min
   "在关注呼吸的同时……也觉察周围的声音……整个身体的感觉"

5. 回归（Returning）- 2min
   "慢慢地……将注意力带回这个房间……感受你坐在这里"
```

#### 身体扫描（Body Scan）

```
1. 全身概览 - 1min
   "从头到脚快速扫描一遍……注意到哪里紧张……哪里放松"

2. 逐区深入 - 12min
   头部→面部→颈部→肩膀→手臂→胸腔→腹部→臀部→大腿→小腿→双脚
   每个区域："注意到这里的感觉……不评判……只是观察"

3. 整体融合 - 2min
   "感受整个身体作为一个整体……坐在那里……呼吸着"
```

#### 慈心禅（Loving-Kindness）

```
1. 自我慈悲 - 3min
   "愿我平安……愿我快乐……愿我健康……愿我自在"

2. 挚爱之人 - 3min
   "想象一个你关心的人……愿您平安……"

3. 中性之人 - 2min
   "一个不太熟悉的人……愿您平安……"

4. 困难之人（可选，需评估学员准备度）- 2min
   "一个你感到困难的人……"（注意：有创伤史的学员可能需要跳过）

5. 一切众生 - 2min
   "愿一切众生平安……快乐……健康……自在"
```

### 2.3 特殊场景引导调整

| 场景 | 调整策略 | 注意事项 |
|------|----------|----------|
| **焦虑人群** | 缩短闭眼时间，增加外部锚定 | 避免"放空"类引导 |
| **儿童（6-12岁）** | 游戏化引导，缩短至3-5分钟 | 用故事和想象替代抽象概念 |
| **青少年（13-18岁）** | 加入自我认同、情绪管理主题 | 尊重自主性，不强加参与 |
| **老年人** | 椅子坐姿为主，关注身体安全 | 简化指令，增加重复 |
| **企业团体** | 效率导向、数据支撑 | 10-15分钟短冥想，不使用灵性语言 |
| **疼痛人群** | 不聚焦疼痛区域，关注安全区域 | 声音或想象锚定替代身体感知 |
| **线上教学** | 增加视觉辅助，更频繁回锚 | 提前测试技术，预留缓冲 |

## 3. 课堂管理

### 3.1 团体动力学管理

| 阶段 | 特征 | 执行师策略 |
|------|------|-----------|
| **形成期** | 学员谨慎、试探 | 建立安全感、明确规范、破冰 |
| **风暴期** | 可能出现质疑、冲突 | 保持中立、开放讨论、不防御 |
| **规范期** | 信任建立、投入增加 | 鼓励分享、深化练习 |
| **表现期** | 深度参与、自发性高 | 适度放手、引导自主探索 |
| **结束期** | 不舍、整合 | 正式收束仪式、庆祝成长 |

### 3.2 常见课堂挑战与应对

| 挑战 | 表现 | 应对方式 |
|------|------|----------|
| **学员迟到** | 打断课堂氛围 | 提前设规矩"迟到5分钟内可安静进入"，超时不建议进入 |
| **学员打瞌睡** | 头低下、呼吸变深 | 不批评，下课前轻声提醒可站立 |
| **学员频繁看手机** | 注意力外散 | 开场说明"手机静音"，课中不强制但以身作则 |
| **学员过度分享** | 占用大量时间 | 温和截断"谢谢你的分享，我们回到练习中" |
| **学员质疑挑战** | "这有用吗？" | 认可提问、邀请直接体验验证、不辩论 |
| **学员情绪激动** | 突然哭泣 | 递纸巾、不急于安慰、允许情绪流动、课后跟进 |
| **学员之间冲突** | 分享时观点对立 | 中立主持、强调"这是分享不是辩论"、课下调解 |

### 3.3 班级规模与教学策略

| 规模 | 人数 | 教学策略 | 助教需求 |
|------|------|----------|----------|
| **私密组** | 3-6人 | 高互动、个性化调整、深度分享 | 无 |
| **标准组** | 8-15人 | 平衡教学与互动、配对练习 | 建议有 |
| **大型组** | 20-40人 | 结构化教学、减少个人分享、书面反馈 | 需要1名 |
| **工作坊** | 40-100人 | 演讲+大型体验、分组活动 | 需要2-3名 |
| **线上课** | 不限 | 增加互动频率（投票/聊天框）、缩短练习时间 | 需要1名管理聊天区 |

## 4. 学员评估与反馈

### 4.1 评估维度

| 维度 | 评估方式 | 评估频率 |
|------|----------|----------|
| **练习参与度** | 出勤、课堂投入观察 | 每次课 |
| **技能发展** | 冥想时长、专注力稳定性、情绪调节能力 | 每个课程周期 |
| **知识理解** | 口头问答、书面反思 | 课程中段 |
| **心理变化** | 标准化量表（PSS、MAAS、PHQ-9等） | 课前/课后 |
| **满意度** | 匿名反馈问卷 | 每个课程周期结束 |

### 4.2 课后反馈收集模板

```
【课程反馈问卷】

1. 今天课程中你最有收获的部分是什么？
   __________________

2. 有哪些部分让你感到困惑或不舒服？
   __________________

3. 你会在课后继续练习吗？□ 是 □ 暂时不会
   如果是，计划练习频率：□ 每天 □ 每周3-5次 □ 每周1-2次

4. 课程节奏：□ 太快 □ 适中 □ 太慢

5. 课程难度：□ 太难 □ 适中 □ 太简单

6. 执行师的教学：□ 非常满意 □ 满意 □ 一般 □ 不满意

7. 你对下次课程有什么建议？
```

## 5. 教学材料与资源管理

### 5.1 教学材料清单

| 类别 | 必备 | 推荐 | 补充 |
|------|------|------|------|
| **音频** | 引导冥想录音 | 自然声/音乐背景 | 学员练习音频 |
| **视觉** | PPT/关键概念图 | 海报/信息图 | 冥想姿势示意图 |
| **文字** | 课程大纲、练习指南 | 学员手册/工作簿 | 推荐阅读书单 |
| **道具** | 坐垫、毛毯 | 眼罩、精油（可选） | 颂钵/音叉（可选） |
| **评估** | 反馈问卷 | 前后测量表 | 练习日志模板 |

### 5.2 音频录制标准

| 参数 | 标准 | 说明 |
|------|------|------|
| 采样率 | 44.1kHz/16bit | CD音质 |
| 环境 | 安静室内、混响适中 | 避免空旷回声 |
| 麦克风 | 电容麦克风、防喷罩 | 保证声音温暖 |
| 后期 | 轻度压缩、去噪 | 不过度处理 |
| 背景音 | 音量低于引导声10-15dB | 不喧宾夺主 |
| 格式 | MP3 192kbps / WAV | 分发用MP3，存档用WAV |
