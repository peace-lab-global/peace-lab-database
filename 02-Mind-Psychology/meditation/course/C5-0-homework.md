# 答题
03 觉察与沉思
1.记录一次你在生活中连接对象品质进入专注冥想的练习体验
答：
我在房间角落打坐，面壁坐好，点了一支尼泊尔线香。一缕烟在眼前慢慢升起来，我就先睁着眼看着它。烟是软的、轻的，一直在变，但不乱，盯着盯着心就自己静下来了。过了几分钟我把眼神往回收一点，放在我和烟中间，再完全松开——不去看了，但烟还在视野里。从主动盯着到被动看着的那个过渡，我这次做得比以前顺。闭上眼之后，烟的那种轻、那种飘劲儿已经在心里了，我不再想着烟，而是让心跟着鼻息走，香味一进来呼吸自己就变深变长，完全不用刻意控制。借个具体东西把心境勾出来，然后把画面扔掉，只留下心境。整座坐下来身子很稳，脑子里念头也有，像天上的云一样飘过去，我知道它在飘，但我这座山没动。

观察一下，生活中是否有某个事物、某种场景会为你带来积极正向的连接体验
答：
有三个场景对我特别有牵引力：
- 清晨焚香：尼泊尔线香、塔香、鼠尾草，烟的升腾与气味能直接把呼吸的深度和长度拉开。
- 房间的冥想角：面壁、有边界、有仪式感，是我能进入精微觉察的茅棚，有一个能遮风的茅棚就可以起观。
- 走路时路边的草木：正念走路，风一吹、叶子摇，会自动提醒我自在如风。

你从中感受到怎样的品质意涵
答：
- 线香的烟 → 轻盈、流动、不执、消融自我。
- 冥想角落 → 稳定、承托、归属、仪式。
- 路边草木 → 扎根与伸展同在，既安稳又开放。
统摄成一句话，就是稳如山、轻如风、容如空。这恰好补足我平时的短板：我习惯收紧肚腩、习惯评判自己，而这些品质都在教我松而不垮、包容一切。

当你把这种意涵心境带入到冥想中，是否给你带来了新的体验与发现
答：
按老师点评把包容的目光从路人收回到自己身上，最直接的体验是肚腩羞耻松了：我把我不应该有肚腩这心念当成食物来观察，看见是脑子因为怕失去减肥成功者的人设在吃，身体其实只想好好呼吸、胖也可以瘦也可以。试着只是让肚子自然起伏，呼吸立刻变长，比我以前硬告诉自己有肚腩也很好的包容有用得多，它不是说服自己，而是认出脑子正在吃一口不该吃的心念。

2.达成止定的三种方式（锚点）是什么，结合你的练习体会谈谈三种方式给你带来的帮助
答：
三种锚点由粗到细、由外到内依次是身体、呼吸、心境。

（1）身体锚点是最重的所缘，用坐姿、触感去绑住心。我现在把舌尖、两指尖、两髋部做成一个五边形，刚上座散乱最大时它先把我钉住；单盘腿麻也是延伸，痛和麻只要不被带走，它自己就是定力在长。

（2）呼吸锚点中等精细，从数息、观呼吸到只感受小腹起伏，逐步把重调轻。焚香是我最好的外援，香味自然把呼吸拉长拉深，不必刻意控制。点上香之后，松不会散、紧不会僵，一紧就散乱、一松就昏沉的循环被破了。

（3）心境锚点最精细，借意象引出心境，或直接锚定在意涵上，心境出来了就把画面扔掉。最近让我突破最多：上座先用山锚定不动，起坐行走时换成风锚定不抓，定不再是死水，而是带弹性的安住。更重要的是它让我看见我的定之所以易破，是因为在用定对抗肚腩羞耻和慢心，而空与容的心境直接把对抗的对象拿掉，心自然就稳了。

三者的关系：散乱时用身体→稳定后用呼吸→熟练后用心境，由重到轻层层松手，是同一次练习里的阶梯而不是三选一。

1. 舌抵上腭的加持。我有一个遗传，和我爸一样，时不时摸耳朵然后嘴巴就吸奶一样，其实是舌顶上颚，这个小时候被家人笑了很多年，他耳垂都磨出老茧了。切实感受是加强沉静感，是一种放松。但是冥想中，舌抵上腭就有了一种陌生感，因为很少不摸耳垂抵上腭的。我现在把舌尖，两指尖，两髋部，定成一个稳定的五边形，求定，求稳。

2. 数息时焚香的加持。我是精油和焚香的爱好者，因为香味对我有很强的牵引力，我也需要强的牵引。尼泊尔线香，塔香，鼠尾草都是很适合冥想的，主要是气味可以加深呼吸的深度和长度的感知。

3. 呼吸为锚点时的肚腩羞耻，转化或者加入一个包容的冥想主题。我习惯瘦肚子，减肥五十斤之后，对于残留的肚腩，走到哪里都要收肚子，有羞耻心。所以放松肚腩非常不自然，觉得膨大起来非常悲哀的感觉。这里要先包容自己，我可以成为任何形状，我可以包容一切。

4. 腿麻为练习的标尺。散盘还是容易散乱，单盘还好我就单盘，现在单盘坐感觉更稳了，因为我会通过挺直腰，这样可以彻底去放松髋部。我买了 MOCICI 的冥想垫，屁股高一些，这个动作很正。但是站起来就痛苦了，会麻痛，整个人要忍住，和抽筋一样。这个时间目前是 20 分钟，这个间隙有逐步加长，我认为算是一个精进的过程。

5. 仪式感的重要性。我之前早晨通勤，无论行走坐卧，都在听引导词，因为有志于此，所以每天三个冥想 APP 轮流打卡，也有五个月了，潮汐，KEEP，MOCICI 小程序刚开始。但是最近我尝试了一下在我房间的角落，房间乱我就面壁，有一个圆垫和冥想垫，这个状态下我是能体会到更精微的专注和变化。准备后续尽可能早睡，这样早起来些，把每天 30 分钟左右的打卡从相对粗糙的练习，先变成一种生活仪式，才能融入生活，而不是反过来。

6. 包容的主题。上周我有一个心态的波动起伏，按老师的建议，加入包容的主题。我尝试使用我熟悉的行走入定的过程中，加入这个主题。不过失败了，当且仅当我把路上见到的所有人，当成两只脚的两坨肉的时候，我才可以不加评判，不加滤镜，全然包容。这里有很强的慢心，也就是傲慢。慈心是建立在平等的基础上的，俯视产生不了真正的连接。把人看成一坨肉这个就是顽空，这也断了慈悲的。我要练习的是：我看到了你我，你我都不完美，我接受了这个事实。下次行走我会试着把路人当成另一个自己，而不是一坨肉，期待做到更柔软的定。
