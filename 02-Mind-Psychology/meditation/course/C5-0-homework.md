# 第五周 作业

# 20260504 亚仑 分享
1. 舌抵上腭的加持。我有一个遗传，和我爸一样，时不时摸耳朵然后嘴巴就吸奶一样，其实是舌顶上颚，这个小时候被家人笑了很多年，他耳垂都磨出老茧了。切实感受是加强沉静感，是一种放松。但是冥想中，舌抵上腭就有了一种陌生感，因为很少不摸耳垂抵上腭的。我现在把舌尖，两指尖，两髋部，定成一个稳定的五边形，求定，求稳。

2. 数息时焚香的加持。我是精油和焚香的爱好者，因为香味对我有很强的牵引力，我也需要强的牵引。尼泊尔线香，塔香，鼠尾草都是很适合冥想的，主要是气味可以加深呼吸的深度和长度的感知。

3. 呼吸为锚点时的肚腩羞耻，转化或者加入一个包容的冥想主题。我习惯瘦肚子，减肥五十斤之后，对于残留的肚腩，走到哪里都要收肚子，有羞耻心。所以放松肚腩非常不自然，觉得膨大起来非常悲哀的感觉。这里要先包容自己，我可以成为任何形状，我可以包容一切。

4. 腿麻为练习的标尺。散盘还是容易散乱，单盘还好我就单盘，现在单盘坐感觉更稳了，因为我会通过挺直腰，这样可以彻底去放松髋部。我买了 MOCICI 的冥想垫，屁股高一些，这个动作很正。但是站起来就痛苦了，会麻痛，整个人要忍住，和抽筋一样。这个时间目前是 20 分钟，这个间隙有逐步加长，我认为算是一个精进的过程。

5. 仪式感的重要性。我之前早晨通勤，无论行走坐卧，都在听引导词，因为有志于此，所以每天三个冥想 APP 轮流打卡，也有五个月了，潮汐，KEEP，MOCICI 小程序刚开始。但是最近我尝试了一下在我房间的角落，房间乱我就面壁，有一个圆垫和冥想垫，这个状态下我是能体会到更精微的专注和变化。准备后续尽可能早睡，这样早起来些，把每天 30 分钟左右的打卡从相对粗糙的练习，先变成一种生活仪式，才能融入生活，而不是反过来。

6. 包容的主题。上周我有一个心态的波动起伏，按老师的建议，加入包容的主题。我尝试使用我熟悉的行走入定的过程中，加入这个主题。不过失败了，当且仅当我把路上见到的所有人，当成两只脚的两坨肉的时候，我才可以不加评判，不加滤镜，全然包容。这里有很强的慢心，也就是傲慢。慈心是建立在平等的基础上的，俯视产生不了真正的连接。把人看成一坨肉这个就是顽空，这也断了慈悲的。我要练习的是：我看到了你我，你我都不完美，我接受了这个事实。下次行走我会试着把路人当成另一个自己，而不是一坨肉，期待做到更柔软的定。
