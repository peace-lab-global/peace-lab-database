# C3 第三周作业

# 作业
1. 记录一次你在练习中对身体的觉察以及应对方式
（1）身体中出现的感觉是什么？
（2）你对感觉的想法是什么？
（3）你是否做出了什么行动回应这种感觉或想法？
（4）这是必须的么？

2. 从练习止的六个关键中选择 1-2 个，谈谈你对它的理解，以及这种态度如何在练习中支持你？

# 回答
1. 记录一次你在练习中对身体的觉察以及应对方式
问：身体中出现的感觉是什么？
答：身体扫描中觉察到双肩与颈后侧紧结沉重、肩胛骨区域发痒发麻、腰背酸胀伴随呼吸循环微微起伏、双脚掌与地面接触处有温热压实的触感。

问：你对感觉的想法是什么？
答：第一反应是把紧结和发痒发麻解读为“必须马上调整掉的不舒服”，并产生“自己身体还是太僵硬、练习不够到位”的批判性评价。

问：你是否做出了什么行动回应这种感觉或想法？
答：下意识耕肩扰动姿势、挑剔痒处、努力深呼吸或更换所缘镜来消除不适，试图用动作快速结束感受本身。

问：这是必须的么？
答：并不是必须的，按照“不大惊小怪、不受二见、主从分明”的原则，观察感受自然生灭即可，回到所缘镜而不加解读、不扰动泥水，感受自然会因缘和合消散。

2. 从练习止的六个关键中选择 1-2 个，谈谈你对它的理解，以及这种态度如何在练习中支持你？
答：选择“活在过程”与“一期一会”：前者让我放下追求入定的结果性期待、安住于吸呼交替的每一个当下，后者让我以庄严尊重的心态将全部生命投入这一座，二者共同消解了“坐不好下一次再补”的拖延与抢结果的焦虑，在练习中形成不后悔、不纠结的稳定支撑。

# 20260420 周一 亚仑作业
1. 学习情况
还是特别繁忙高压且会议众多的周一，越忙越要抽出时间听课，保持劳逸结合的节奏，才能提高专注的质量，结果往往更好。今天中午学习了杨硕诚老师的冥想与止观第三讲：练习止的关键字，好好坐着，不看其他，安静听课，甘之若饴。定力比之前非要走路走起来才可以听进去要好很多。

2. 学习感悟
（1）动中定力核心：活在过程即活在当下。活在过程就是清楚知道自己正在做什么，让心与当下的动作共处，自然就能达到活在当下的状态。动中定力的锻炼比静中定力锻炼更重要，同时方法要领更简单，普通人更容易上手掌握。
（2）动中禅定核心：慢跑，瑜伽、八段锦、太极拳、写书法，猫步（略微屈膝保持弯曲不伸直，每一步缓慢踏出，前脚站稳后再抬起后脚，重心缓慢前移，初始可关注脚底接触地板的触感，熟练后不用低头看脚，靠足部感受即可）
（3）静坐核心：准备工作分为调身、调息、调心三个步骤，不建议初学者勉强双盘。只需要知道自己吸气、呼气的过程，不需要刻意控制呼吸节奏，完成3个深呼吸后就可以进入调心阶段。
（4）正念行走的身体感受： 我觉得高压到喘不过气的时候，下楼走路是快速调节情绪，脱离反刍的grounding 接地技术。新鲜空气迎面扑来击中肺部的时候，立刻就会有打开的松弛感。如果是在树林里走路，10 分钟以上grounding 接地感更加强烈，身体重心下沉，步幅慢而稳。若持续下去，DMN 逐步消退，会 TPN 到某些难题上，自然会有解决方法。近期加入止观练习，可以主动的DMN 拉回 TPN，切换如同大梦初醒的恍惚感，随着行走时间加长，更容易进入 TPN，心念觉察更加容易，无念的心流状态更加持久。整体下来，身体的拉练带来了心理的深度放松，如同充满电一样。

# 20260421 周二 亚仑作业
1. 学习情况
今天中午学习杨硕诚老师的冥想与止观：练习止的方式。关于一期一会的内容非常精彩，并尝试在下午的工作中运用，近期整体的安排也会以一期一会的思路去做事情。

2. 学习感悟
（1）一期一会：以庄严尊重的心态面对当下每一刻，将全部生命投入当下的事项，不会让当下匆匆流逝。每一刻都尽全力面对，即使结果不理想也已经发挥了当下的最高水平，因此不容易后悔、纠结，也不会愧疚。每次上座打坐前，要告诉自己当下这一座是全生命的第一座，也是最后一座，以一期一会的心态进入打坐。
（2）打坐的自然发展阶段：找到所缘境陪伴后，会自然依次出现「喜」「乐」「心一境性」三个阶段，这三个阶段都是自然结果，不是刻意模仿就能获得。喜是放松稳定的满足感，出现喜就代表打坐上轨道；乐是充足满足感，心不会向外追逐其他事物；心一境性是心念自然稳定，不需要用力扣住所缘境就能维持平稳状态。
（3）四种数息方法：鼻下数吸不数呼、鼻下数呼不数吸、小腹数吸不数呼、小腹数呼不数吸，每种方法都从一数到十循环往复。
（4）个人感受：主从分明，就像走路去目的地，清楚知道主目标，遇到干扰只是暂时偏离，不会彻底偏离目标，一定能走到终点。只要是有畏难情绪的事情，以一期一会的全力的专注去做，就会有相对满意的结果。日常都有十个左右的任务，难事不一而足，有时出现拖延或者痛苦，庄严尊重的心态面对困难的当下，心理韧性强了，也得到个人满意的进展。我也要把一期一会融入生活中。

# 20260422 周三 亚仑作业
1. 学习情况
今天是周四，昨天周三只做一件事情，主动放松身体，主动入睡。

 2. 学习感悟
* 主动入睡是最近一两年我都无法做到的事情，早于 1 点都算早。 
* 我回忆一下昨晚的情况：10 点我就开始主动放松身体，认真感受放松的过程。不加入任何目的性，甚至不寻求放松，只是感受身体的各个部位和床垫的接触，感受床垫对我身体的支撑。放不下就放弃，放弃思考，放弃得出什么结论，放弃这段时间要有什么收获，放弃对明天的期待和担忧；有点像主动进入一次死亡。身体越来越沉，再一睁眼就是第二天八点了，闹钟八点半。
* 整个早晨明显情绪的韧性的好的，就是我清楚的知道今天发生任何事情我都可以 hold 住。难以想象如果天天保持优质的睡眠，我会有哪种生命体验。

# 20260424 周五 亚仑作业
Q：老师，想要提问今天再跟练安定的身体空间时然后越让身体放松就会越紧张，身体很难放松，该怎么调整呢。刚开始坐着，后来我试着躺下，但还是会紧张，好像一安静我就会紧张。这样是可以的，但是如果我跟随录音，想让身体放松就会很紧张。那这个时候我是继续跟随音频还是不继续了呢。继续跟着音频，觉察到紧张，但不管，继续紧张着跟随练习嘛。嗯嗯好的，不去纠正要求这个很重要，我总想纠正不舒服，觉得冥想应该是舒服的，忽略了看到了就是看到了，让它走就好了。
A：我个人感受哈，好比人的眼睛只能感受光谱中的某段，也就是它其实并没有看到桌子，只是阳光的某些频段从桌子表面折射到眼里，眼睛才能看到。不安的情绪就是一个事实真实的情况（桌子），通过空气（观念经验判断）被眼睛（心）看到，也就是如实的情况我们只是透过透明的空气，看到一个桌子，但我们会在空气中加入雾气毒气，或者桌子成了一个怪物，自然就会不安。我个人的话，比如有些事情必须完成但出现拖延，心里就好像我拿着刀子追杀自己，那一定是坐立不安的。所以可能是某个课程之外或者课程本身的投射，有应激反应。课程本身投射可能是提前获得，心里已经想象得到了老师的赞赏，回到当下就会觉得如果进不了状态怎么办，我就会失去老师的赞赏，进入对学习本身的焦虑，捅自己刀子，而心念管理本身就是精微的，婴儿学步，状态不稳是常态。
我觉得张雪机车的张雪肯定不冥想，看了很多他的表达，很中正，认知高又没有认知高的累赘和痛苦，真是难得。有人说 无知是最大的恶，但无知可能是对广大生灵的一种保护，当广大生灵本身它不具备心念管理能力的时候，他具备强大的信念和认知，对他有什么好处呢。小狗不自杀，你可以鄙视他认知低 但他就是不自杀。
