# 亲密关系质量评估 (Intimate Relationship Quality Assessment)

## 技能信息

| 属性       | 值                                    |
| ---------- | ------------------------------------- |
| 技能ID     | S_017                                 |
| 版本       | 1.0                                   |
| 角色       | 入口技能 (entry)                      |
| 分类       | 评估 (assessment)                     |
| 证据等级   | A (随机对照试验/系统综述支持)         |
| 预计时长   | 15-20分钟                             |
| 适用人群   | 18岁以上处于亲密关系中的个体          |
| 触发关键词 | 感情、婚姻、伴侣、吵架、冷战、出轨、离婚、亲密关系、恋爱、分手 |

### 理论基础

本评估协议整合以下循证工具：

- **CSI 关系满意度量表** (Couple Satisfaction Index) — Funk & Rogge, 2007
- **ECR-R 亲密关系经历量表修订版** — Fraley, Waller & Brennan, 2000
- **Gottman 四骑士模型** — Gottman & Silver, 1999
- **关系暴力筛查** — WHO 亲密伴侣暴力筛查指南

### 适用条件

- 用户主动描述亲密关系困扰
- 关键词触发进入评估流程
- 用户年龄 ≥ 18 岁
- 用户愿意接受评估

### 禁忌条件

- 用户处于即时人身危险中 → 直接转介危机干预
- 用户未满 18 岁 → 转介青少年关系技能
- 用户无自知力或严重精神症状 → 转介精神科评估

### Step 1: 关系满意度评估

**工具**: CSI 关系满意度量表 (Couple Satisfaction Index, 16项版)
**信效度**: Cronbach α = 0.94, 区分效度优于 DAS
**评分方式**: 1-7 分 Likert 量表
**预计用时**: 5分钟

#### 评估指令

> 接下来我会询问你 16 个关于你和伴侣关系的问题。请根据你的实际感受，
> 在 1-7 分之间选择最符合的分数。
>
> 1 = 完全不符合  |  4 = 有些符合  |  7 = 完全符合

#### 评估项目

| 编号 | 项目内容                                     | 评分 |
| ---- | -------------------------------------------- | ---- |
| C01  | 我的伴侣让我感到被珍惜                      | 1-7  |
| C02  | 我对我们的关系感到满意                      | 1-7  |
| C03  | 我们的感情非常好                            | 1-7  |
| C04  | 我的伴侣理解我                              | 1-7  |
| C05  | 我对伴侣的承诺感到安心                      | 1-7  |
| C06  | 我们的相处充满温暖                          | 1-7  |
| C07  | 我能向伴侣倾诉内心的困扰                    | 1-7  |
| C08  | 和伴侣在一起我很开心                        | 1-7  |
| C09  | 我经常想要改善我们的关系（反向计分）        | 1-7  |
| C10  | 我有时觉得关系中有困难（反向计分）          | 1-7  |
| C11  | 伴侣总是让我失望（反向计分）                | 1-7  |
| C12  | 我们经常争吵（反向计分）                    | 1-7  |
| C13  | 我对这段关系感到幸福                        | 1-7  |
| C14  | 伴侣是我最好的朋友                          | 1-7  |
| C15  | 我愿意为这段关系付出努力                    | 1-7  |
| C16  | 总体而言，我非常满意我们的关系              | 1-7  |

#### 计分与解读

| 总分范围 | 满意度等级   | 描述                               |
| -------- | ------------ | ---------------------------------- |
| 81-112   | 高满意       | 关系质量良好，有稳固的情感基础     |
| 57-80    | 中等满意     | 关系存在改善空间，建议关注沟通     |
| 36-56    | 低满意       | 关系存在明显问题，建议专业介入     |
| 16-35    | 极低满意     | 关系严重受损，强烈建议专业帮助     |

### Step 2: 依恋风格评估

**工具**: ECR-R 亲密关系经历量表修订版 (Experiences in Close Relationships-Revised)
**信效度**: Cronbach α (焦虑) = 0.91, α (回避) = 0.93
**评分方式**: 1-7 分 Likert 量表
**预计用时**: 5分钟

#### 评估指令

> 接下来我会询问你一些在亲密关系中的感受和行为方式。
> 请根据你在当前或最近一段关系中的典型体验作答。

#### 核心项目 (简化版, 8项)

**焦虑维度 (4项):**

| 编号 | 项目内容                                     | 评分 |
| ---- | -------------------------------------------- | ---- |
| A01  | 我担心伴侣不像我在乎他/她那样在乎我         | 1-7  |
| A02  | 我害怕被抛弃                                  | 1-7  |
| A03  | 我需要伴侣不断确认他/她对我的爱              | 1-7  |
| A04  | 伴侣不回应我时，我会感到焦虑不安             | 1-7  |

**回避维度 (4项):**

| 编号 | 项目内容                                     | 评分 |
| ---- | -------------------------------------------- | ---- |
| V01  | 我觉得向伴侣敞开心扉很困难                   | 1-7  |
| V02  | 我不喜欢在情感上过于依赖伴侣                 | 1-7  |
| V03  | 伴侣太亲近时我会感到不舒服                   | 1-7  |
| V04  | 我更倾向于自己解决问题而非向伴侣求助         | 1-7  |

#### 依恋风格分类

```
                    回避维度
                低 ──────── 高
            ┌──────────┬──────────┐
     高     │  焦虑型   │  混乱型   │
焦         │  (痴迷)   │  (恐惧)   │
虑         ├──────────┼──────────┤
维         │  安全型   │  回避型   │
度  低     │  (安全)   │  (疏离)   │
            └──────────┴──────────┘
```

| 依恋风格 | 焦虑均分 | 回避均分 | 核心特征                           |
| -------- | -------- | -------- | ---------------------------------- |
| 安全型   | < 4      | < 4      | 信任伴侣，能自在表达需求和情感     |
| 焦虑型   | ≥ 4      | < 4      | 渴望亲密但害怕被拒，情绪波动大     |
| 回避型   | < 4      | ≥ 4      | 重视独立，回避深层情感交流         |
| 混乱型   | ≥ 4      | ≥ 4      | 既渴望又恐惧亲密，关系模式不稳定   |

### Step 3: 沟通模式评估

**工具**: Gottman 四骑士模型 (The Four Horsemen of the Apocalypse)
**理论来源**: John Gottman, 华盛顿大学婚姻实验室
**预计用时**: 3分钟

#### 评估指令

> 请回想你和伴侣在发生分歧或冲突时的典型互动方式。
> 以下四种沟通模式，请评估它们在你们关系中出现的频率。

#### 四骑士评估

| 骑士       | 描述                                     | 从不 | 偶尔 | 经常 | 总是 |
| ---------- | ---------------------------------------- | ---- | ---- | ---- | ---- |
| 批评       | 对伴侣人格的攻击而非针对具体行为         | 0    | 1    | 2    | 3    |
| (Criticism)| 例如："你总是这样自私！"                  |      |      |      |      |
| 蔑视       | 表达对伴侣的厌恶、嘲讽、翻白眼           | 0    | 1    | 2    | 3    |
| (Contempt) | 例如："你真可悲，这种事都做不好。"        |      |      |      |      |
| 防御       | 拒绝承担责任，反过来指责对方             | 0    | 1    | 2    | 3    |
| (Defensiveness)| 例如："这不是我的问题，是你太敏感了。"|      |      |      |      |
| 冷战       | 关闭沟通，回避互动，冷暴力               | 0    | 1    | 2    | 3    |
| (Stonewalling)| 例如：沉默不语、离开房间、拒绝对话     |      |      |      |      |

#### 四骑士风险等级

| 总分 | 风险等级   | 解读                                           |
| ---- | ---------- | ---------------------------------------------- |
| 0-3  | 低风险     | 沟通模式基本健康，偶有冲突属正常范围           |
| 4-6  | 中等风险   | 需要学习替代沟通策略，建议学习非暴力沟通       |
| 7-9  | 高风险     | 沟通模式严重损害关系，强烈建议专业伴侣咨询     |
| 10-12| 极高风险   | 关系处于危机状态，需要立即介入                 |

#### 四骑士的解药

| 骑士 | 解药             | 具体做法                           |
| ---- | ---------------- | ---------------------------------- |
| 批评 | 温和的开场       | 用"我"开头表达感受，而非指责      |
| 蔑视 | 建立欣赏文化     | 每天表达感恩和赞赏                |
| 防御 | 承担责任         | 承认自己在冲突中的那部分责任      |
| 冷战 | 自我安抚         | 暂停时约定返回时间，而非无限期沉默|

### Step 4: 关系暴力筛查

**工具**: WHO 亲密伴侣暴力筛查指南改编版
**重要性**: 安全优先原则 — 此步骤不可跳过
**预计用时**: 2分钟

#### 安全声明

> 以下问题涉及关系中的安全状况。你的回答将被保密。
> 如果你现在处于危险中，请拨打 110 或全国妇女维权热线 12338。

#### 筛查项目

| 编号 | 筛查领域   | 筛查问题                                           | 是/否 |
| ---- | ---------- | -------------------------------------------------- | ----- |
| V01  | 身体暴力   | 伴侣是否曾推搡、打、踢或伤害你的身体？            | □     |
| V02  | 情感暴力   | 伴侣是否经常贬低你、羞辱你或控制你的行为？        | □     |
| V03  | 性暴力     | 伴侣是否曾在你不愿意的情况下强迫性行为？          | □     |
| V04  | 经济控制   | 伴侣是否控制你的经济来源或阻止你工作？            | □     |
| V05  | 社交隔离   | 伴侣是否阻止你与朋友或家人联系？                  | □     |
| V06  | 威胁恐吓   | 伴侣是否威胁要伤害你、孩子、宠物或自己？          | □     |
| V07  | 监视跟踪   | 伴侣是否监视你的行踪、手机或社交媒体？            | □     |
| V08  | 破坏物品   | 伴侣是否摔砸物品或威胁要破坏财物？                | □     |

#### 暴力风险评估

| 阳性项目数 | 风险等级   | 即时行动                               |
| ---------- | ---------- | -------------------------------------- |
| 0          | 无明显风险 | 继续常规评估                           |
| 1-2        | 中等风险   | 提供安全信息，制定安全计划             |
| 3-4        | 高风险     | 立即提供危机资源，强烈建议安全计划     |
| ≥5         | 极高风险   | 启动紧急转介，协助联系庇护资源         |

## 综合分析与干预建议

### 综合评估矩阵

根据四步骤评估结果，生成综合报告：

```
┌────────────────────────────────────────────────────┐
│             综合评估报告                            │
├────────────────────────────────────────────────────┤
│  关系满意度 (CSI):    ___/112  │ 等级: _______     │
│  依恋风格 (ECR-R):    焦虑___  回避___  → _______  │
│  沟通模式 (四骑士):   ___/12   │ 风险: _______     │
│  暴力筛查:            ___/8    │ 风险: _______     │
├────────────────────────────────────────────────────┤
│  综合风险等级:        ☐低  ☐中  ☐高  ☐极高         │
│  建议干预级别:        Level ___                     │
└────────────────────────────────────────────────────┘
```

### 干预建议

#### Level 1: 自助干预 (低风险)

**适用条件**: 满意度中高 + 安全依恋 + 四骑士低 + 无暴力

**推荐策略**:

- 阅读关系自助书籍
  - 《亲密关系》(罗兰·米勒)
  - 《爱的五种语言》(盖瑞·查普曼)
  - 《非暴力沟通》(马歇尔·卢森堡)
- 学习非暴力沟通四步骤
  1. 观察：描述事实而非评判
  2. 感觉：表达真实情绪
  3. 需要：识别内在需求
  4. 请求：提出具体可行的请求
- 每日关系维护练习
  - 6秒亲吻
  - 每日欣赏表达
  - 减压对话 (每天20分钟)

**输出**: 推荐资源清单 + 自助练习计划

#### Level 2: 伴侣咨询 (中等风险)

**适用条件**: 满意度低 + 不安全依恋 + 四骑士中高 + 无暴力

**推荐方案**:

- **情绪聚焦伴侣治疗 (EFT)** — 证据等级 A
  - 适用于依恋问题为核心的关系困扰
  - 通常 8-20 次
- **Gottman 方法伴侣治疗** — 证据等级 A
  - 适用于沟通模式问题
  - 聚焦四骑士替代策略
- **认知行为伴侣治疗 (CBCT)** — 证据等级 A
  - 适用于认知扭曲导致的关系问题

**转介渠道**:
- 当地心理咨询中心
- 线上伴侣咨询平台
- 医院心理科

#### Level 3: 安全计划 (高/极高风险)

**适用条件**: 暴力筛查阳性 + 满意度极低 + 严重安全威胁

**立即行动**:

1. 确保用户当前安全
2. 提供紧急联系方式
   - 110 (报警)
   - 12338 (全国妇女维权热线)
   - 12355 (青少年服务热线，涉及子女时)
3. 协助制定个人安全计划
   - 紧急联系人
   - 安全撤离路线
   - 重要物品清单
   - 重要文件副本
4. 提供庇护资源信息

**安全计划模板**:

```
┌─────────────────────────────────────┐
│  个人安全计划                        │
├─────────────────────────────────────┤
│  紧急联系人:  ____________          │
│  紧急电话:    ____________          │
│  安全地点:    ____________          │
│  重要物品:    □证件 □现金 □钥匙     │
│  安全信号:    ____________          │
│  撤离路线:    ____________          │
└─────────────────────────────────────┘
```

## 转介红线

以下情况需**立即转介**，不适用常规评估流程：

### 绝对转介 (跳过评估，直接行动)

| 红线情况           | 转介目标                     | 联系方式          |
| ------------------ | ---------------------------- | ----------------- |
| 家庭暴力正在发生   | 公安机关 + 庇护机构         | 110 / 12338       |
| 自杀风险 (有计划)  | 危机干预热线                 | 400-161-9995      |
| 虐待儿童           | 未成年人保护机构             | 12355             |
| 命令性幻听         | 精神科急诊                   | 120               |
| 即时人身安全威胁   | 公安机关                     | 110               |

### 识别信号

- 用户提及"不想活了"、"想结束一切"
- 用户描述遭受殴打、性侵
- 用户提及子女被虐待或目睹暴力
- 用户表达正在被跟踪、监视

```

```

```

```

Agent: 无论你想倾诉感受，还是想了解自己在关系中的

```

## 注意事项

1. **安全第一**: 暴力筛查为必做步骤，任何评估都不可跳过安全筛查
2. **去病理化**: 依恋风格不是标签，而是理解关系模式的工具
3. **文化敏感**: 考虑不同文化背景下的关系期待和表达方式
4. **保密原则**: 评估结果仅用于帮助用户，不会被分享
5. **非诊断**: 本评估为筛查工具，不构成临床诊断
6. **知情同意**: 开始评估前需告知用户评估目的和流程
