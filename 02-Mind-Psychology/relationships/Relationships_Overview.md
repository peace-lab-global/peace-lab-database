# 人际关系治疗体系总览

## 📋 体系架构

### 核心理论基础
- **依恋理论** - 安全基地与情感纽带
- **社会学习理论** - 关系模式的学习与传承  
- **系统家庭理论** - 关系网络的动力学
- **人际神经生物学** - 大脑-关系-发展的整合视角

### 治疗目标层次
1. **安全依恋建立** - 培养健康的情感连接能力
2. **沟通技能提升** - 有效表达与倾听技巧训练
3. **边界意识强化** - 个人空间与相互尊重的平衡
4. **冲突化解能力** - 建设性处理分歧的方法
5. **亲密关系深化** - 深层情感连接与信任建立
