## 一、内科疾病案例

### 1.1 消化系统疾病

#### 慢性胃炎案例

```
病例基本信息：

患者资料：
• 姓名：张某某，女，45岁
• 职业：会计
• 主诉：胃脘胀痛反复发作3年余
• 现病史：3年来胃脘部胀满疼痛，饭后加重，伴有嗳气反酸，纳差便溏
• 既往史：无重大疾病史，否认药物过敏史
• 体格检查：神志清楚，营养中等，上腹部轻压痛，舌淡苔白腻，脉弦滑

中医诊断：
脾胃虚弱证：
• 症候分析：脾虚失运，胃失和降
• 病机要点：脾气虚弱，运化失司，胃气上逆
• 治法治则：健脾和胃，理气止痛

治疗方案：
艾灸取穴：
主穴：
• 中脘：胃之募穴，和胃降逆
• 足三里：胃之下合穴，健脾和胃
• 内关：宁心安神，理气止痛
• 公孙：健脾和胃，通调冲任

配穴：
• 脾俞：健脾益气
• 胃俞：和胃降逆
• 神阙：温阳健脾

操作方法：
• 温和灸：每穴灸15-20分钟
• 雀啄灸：中脘、足三里采用雀啄法
• 隔姜灸：神阙穴隔姜片施灸
• 频率：隔日1次，10次为1疗程

治疗过程：
第一疗程：
• 第1-3次：胃脘胀痛明显减轻，嗳气减少
• 第4-6次：食欲改善，大便成形
• 第7-10次：症状基本缓解，精神状态好转

第二疗程：
• 巩固治疗：继续原方案治疗
• 症状变化：偶有轻微不适，很快缓解
• 体质改善：面色红润，精力充沛

疗效评价：
近期疗效：
• 症状评分：治疗前后VAS评分从7.2降至1.8
• 生活质量：SF-36评分显著提升
• 实验室检查：胃镜复查显示黏膜炎症明显改善
• 患者满意度：非常满意，愿意继续治疗

远期随访：
• 6个月随访：症状稳定，无明显复发
• 12个月随访：生活质量良好，偶有轻微不适
• 预防措施：饮食调理，规律作息，适度运动
```

### 1.2 呼吸系统疾病

#### 支气管哮喘案例

```
病例基本信息：

患者资料：
• 姓名：李某某，男，32岁
• 职业：程序员
• 主诉：反复喘息气促8年，加重1周
• 现病史：8年来每逢气候变化或接触过敏原即发喘息，1周前因感冒诱发加重
• 既往史：过敏性鼻炎病史，青霉素过敏
• 体格检查：呼吸急促，双肺可闻及哮鸣音，舌红苔黄腻，脉滑数

中医诊断：
痰热壅肺证：
• 症候分析：外邪引动伏痰，痰热壅肺
• 病机要点：痰热互结，肺失宣降
• 治法治则：清热化痰，宣肺平喘

治疗方案：
艾灸取穴：
急性期：
• 肺俞：宣肺化痰
• 定喘：止咳平喘特效穴
• 膻中：宽胸理气
• 丰隆：化痰降逆
• 太渊：补肺益气

缓解期：
• 肾俞：补肾纳气
• 命门：温补肾阳
• 足三里：培土生金
• 太溪：滋肾养阴

操作方法：
急性期治疗：
• 雀啄灸：肺俞、定喘采用雀啄法
• 温和灸：膻中、太渊温和施灸
• 时间：每次20-25分钟
• 频率：每日1次，连续5-7天

缓解期治疗：
• 隔附子饼灸：肾俞、命门
• 温针灸：足三里、太溪
• 时间：每次30分钟
• 频率：隔日1次，15次为1疗程

治疗过程：
急性期处理：
• 第1-2天：喘息症状明显缓解
• 第3-5天：哮鸣音减少，呼吸平稳
• 第6-7天：症状基本控制，可正常活动

缓解期调理：
• 第1-5次：体质逐步改善
• 第6-10次：肺功能指标好转
• 第11-15次：免疫力提升，发作频率减少

疗效评价：
肺功能改善：
• FEV1：治疗前65%预计值，治疗后85%
• PEF：治疗前70%预计值，治疗后90%
• 症状评分：哮喘控制测试(ACT)评分从12分提升至24分

生活质量：
• 日常活动：恢复正常工作生活
• 睡眠质量：夜间症状明显改善
• 情绪状态：焦虑抑郁情绪缓解
• 社会功能：重返正常社交活动

预防复发：
• 避免过敏原：远离花粉、尘螨等过敏原
• 增强体质：规律锻炼，提高免疫力
• 情志调节：保持心情舒畅，避免情绪波动
• 饮食调理：清淡饮食，忌辛辣刺激食物
```

## 二、疼痛性疾病案例

### 2.1 骨关节疾病

#### 膝骨关节炎案例

```
病例基本信息：

患者资料：
• 姓名：王某某，女，68岁
• 职业：退休教师
• 主诉：双膝疼痛僵硬5年余，加重半年
• 现病史：5年前开始双膝疼痛，上下楼梯困难，半年来症状加重，行走受限
• 既往史：高血压病史，规律服药控制良好
• 体格检查：双膝轻度肿胀，活动受限，浮髌试验阳性，舌暗苔白，脉沉涩

中医诊断：
肝肾亏虚，瘀血阻络证：
• 症候分析：年老体衰，肝肾不足，筋骨失养
• 病机要点：虚实夹杂，本虚标实
• 治法治则：补益肝肾，活血通络，消肿止痛

治疗方案：
艾灸取穴：
局部取穴：
• 内外膝眼：膝关节局部经验穴
• 阳陵泉：筋会阳陵，舒筋活络
• 血海：活血化瘀，养血润燥
• 梁丘：郄穴止痛，消肿散结

远端取穴：
• 足三里：强壮保健，调理脾胃
• 三阴交：肝脾肾三经交会穴
• 肾俞：补肾强筋骨
• 委中：腰背委中求，疏通膀胱经

操作方法：
温和灸法：
• 直接灸：内外膝眼、阳陵泉直接施灸
• 隔盐灸：神阙穴隔盐施灸温阳
• 温针灸：足三里、三阴交温针结合
• 铺灸：督脉铺灸补肾壮阳

治疗参数：
• 温度控制：42-45°C为宜
• 时间安排：每穴15-20分钟
• 频率密度：隔日1次，15次为1疗程
• 疗程间隔：两个疗程间休息1周

治疗过程：
第一疗程：
• 第1-3次：疼痛有所缓解，活动度增加
• 第4-7次：肿胀减轻，上下楼梯较前轻松
• 第8-10次：疼痛明显改善，可正常行走
• 第11-15次：症状基本控制，生活质量提升

第二疗程：
• 巩固疗效：继续原方案治疗
• 功能锻炼：配合膝关节功能锻炼
• 体质调理：加强肝肾调补
• 预防复发：指导日常保健方法

疗效评价：
疼痛改善：
• VAS评分：治疗前7.8分，治疗后2.1分
• WOMAC评分：疼痛维度显著改善
• 日常活动：上下楼梯、蹲起动作基本正常
• 步行能力：6分钟步行距离从320米增至480米

影像学改善：
• X线检查：关节间隙无明显变化
• MRI检查：软骨信号有所改善
• 超声检查：关节积液明显减少
• 炎症指标：ESR、CRP水平下降

功能恢复：
• 关节活动度：屈曲角度从90°增至120°
• 肌力测试：股四头肌力量增强
• 平衡能力：静态和动态平衡改善
• 生活质量：SF-36各维度评分提升
```

### 2.2 神经系统疾病

#### 三叉神经痛案例

```
病例基本信息：

患者资料：
• 姓名：赵某某，女，52岁
• 职业：家庭主妇
• 主诉：右侧面部阵发性剧痛3年余
• 现病史：3年前无明显诱因出现右侧面部闪电样剧痛，刷牙、说话时诱发
• 既往史：无特殊病史，否认药物过敏
• 体格检查：神志清楚，痛苦面容，右侧面部触痛敏感，舌红苔薄黄，脉弦数

中医诊断：
风火上扰，经络瘀阻证：
• 症候分析：风火相煽，上扰清窍，经络不通
• 病机要点：实证为主，风火痰瘀互结
• 治法治则：祛风清热，活血通络，止痛解痉

治疗方案：
艾灸取穴：
局部取穴：
• 下关：面痛要穴，疏通面部经络
• 颊车：疏风清热，消肿止痛
• 四白：明目止痛，疏风通络
• 迎香：通鼻窍，疏风热

远端取穴：
• 合谷：面口合谷收，疏风止痛
• 太冲：平肝熄风，清泄肝火
• 风池：祛风清热，通络止痛
• 外关：疏风解表，通络止痛

特殊穴位：
• 阿是穴：疼痛最敏感点
• 太阳：疏风清热，明目止痛
• 印堂：宁神止痛，通络开窍

操作方法：
温和灸法：
• 雀啄灸：下关、颊车采用雀啄法
• 回旋灸：合谷、太冲回旋施灸
• 隔蒜灸：阿是穴隔蒜片施灸
• 温和灸：其余穴位温和施灸

治疗参数：
• 温度控制：40-43°C，避免烫伤
• 时间安排：每穴10-15分钟
• 频率密度：每日1次，急性期连续治疗
• 疗程设置：10次为1疗程，间隔3-5天

治疗过程：
急性期治疗：
• 第1-3天：疼痛频率减少，程度减轻
• 第4-7天：发作间歇期延长，疼痛缓解
• 第8-10天：症状明显改善，可正常生活

缓解期调理：
• 巩固治疗：继续温和灸法
• 体质调理：配合中药调理
• 生活指导：避免诱发因素
• 预防复发：建立健康生活习惯

疗效评价：
疼痛控制：
• 疼痛频率：从每日5-8次降至每周1-2次
• 疼痛强度：VAS评分从8.5分降至2.0分
• 发作持续时间：从每次1-2分钟缩短至30秒内
• 无痛间歇期：从数小时延长至数天

生活质量改善：
• 日常活动：洗脸、刷牙等动作可正常进行
• 睡眠质量：夜间疼痛减少，睡眠改善
• 情绪状态：焦虑抑郁情绪明显缓解
• 社会功能：恢复正常社交和工作

远期随访：
• 3个月：偶有轻微发作，很快缓解
• 6个月：症状稳定，生活质量良好
• 12个月：基本无发作，偶因劳累轻微不适
• 预防措施：规律作息，避免情绪激动，注意保暖
```

## 三、妇科疾病案例

### 3.1 月经不调

#### 功能性子宫出血案例

```
病例基本信息：

患者资料：
• 姓名：刘某某，女，28岁
• 职业：护士
• 主诉：月经淋漓不尽20余天
• 现病史：月经周期紊乱2年，此次月经持续20余天未净，量时多时少
• 既往史：人流术后月经不调，否认其他重大疾病
• 体格检查：面色萎黄，神疲乏力，妇科检查无明显异常，舌淡苔薄白，脉细弱

中医诊断：
脾肾亏虚，冲任不固证：
• 症候分析：脾虚失统，肾虚不固，冲任失调
• 病机要点：虚证为主，气血两虚
• 治法治则：健脾补肾，固冲止血，调经止带

治疗方案：
艾灸取穴：
止血要穴：
• 隐白：脾经井穴，收敛止血
• 三阴交：肝脾肾三经交会，调经止带
• 血海：活血调经，养血润燥
• 地机：理血调经，健脾渗湿

固摄穴位：
• 关元：小肠募穴，补益元气
• 气海：肓之原穴，补气固脱
• 足三里：强壮保健，健脾和胃
• 肾俞：补肾强腰，固摄下元

调理穴位：
• 脾俞：健脾益气，统血摄血
• 胃俞：和胃降逆，健脾化湿
• 太溪：滋肾养阴，补肾固精
• 神阙：温阳健脾，调理冲任

操作方法：
温和灸法：
• 隔附子饼灸：关元、气海温补元气
• 隔姜灸：神阙温中健脾
• 温和灸：三阴交、血海调经止血
• 雀啄灸：隐白、地机收敛止血

治疗参数：
• 温度控制：42-45°C温和施灸
• 时间安排：每穴15-20分钟
• 频率密度：每日1次，出血期间连续治疗
• 疗程设置：出血停止后继续治疗5-7次巩固

治疗过程：
止血阶段：
• 第1-3天：出血量逐渐减少
• 第4-7天：出血明显减少，色转淡红
• 第8-10天：出血基本停止，仅少量褐色分泌物

调理阶段：
• 第11-15天：体质逐步恢复
• 第16-20天：月经周期趋向规律
• 第21-25天：面色红润，精力充沛

疗效评价：
止血效果：
• 出血时间：从20余天缩短至10天内止血
• 出血量：从大量出血转为少量分泌物
• 血红蛋白：从85g/L恢复至115g/L
• 临床症状：头晕乏力等症状明显改善

月经调节：
• 周期恢复：从紊乱恢复至28-30天规律
• 经量正常：从过多或过少恢复正常
• 经期长短：从延长恢复至5-7天正常
• 痛经改善：伴随的痛经症状缓解

体质改善：
• 面色改善：从萎黄转为红润
• 精神状态：从疲乏转为精力充沛
• 食欲增加：消化功能改善
• 睡眠质量：夜间睡眠安稳

远期随访：
• 3个月：月经周期规律，无异常出血
• 6个月：体质明显改善，工作生活正常
• 12个月：月经正常，无复发
• 预防措施：规律作息，情绪调节，适度运动
```
