# 内部家庭系统治疗 (IFS) | Index

> **专题定位**：Richard Schwartz 创立的"部分工作"（parts work）疗法，将人的内心视为由管理者、消防员、流放者组成的家庭系统，以"Self 领导"为疗愈核心。
> **适用对象**：创伤治疗师、心理咨询师、对内在探索感兴趣的自我成长者。

## 核心概念速查

- **部分 (Parts)**：人的心智由多个有意图的子人格组成（非病理）
- **Self（自我）**：完整未损的核心，具 8C 特质（好奇/慈悲/清晰等）
- **流放者 (Exiles)**：承载创伤的年轻部分
- **管理者 (Managers)**：预防性保护者
- **消防员 (Firefighters)**：紧急反应保护者
- **6F 步骤**：Find / Focus / Flesh out / Feel toward / Befriend / Fear
