# MBCT 正念认知疗法 RCT 证据摘要

> **状态**: 实证 (A级证据)
> **最后更新**: 2026-05-18
> **证据等级**: A — 强证据（多项高质量RCT及多项元分析支持）

---

## 代表性 RCT 证据汇总

### 1. Kuyken et al. (2015) — 英国多中心RCT

| 字段 | 内容 |
|------|------|
| **第一作者** | Kuyken, W |
| **发表年份** | 2015 |
| **期刊** | The Lancet, 386(9988), 2384-2392 |
| **研究设计** | 多中心RCT（英国） |
| **样本量** | N = 617（MBCT组 n=314，常规治疗组 n=303） |
| **研究周期** | 8周干预 + 12个月随访 |
| **研究地区** | 英国 |

**研究人群**
- 诊断: 复发性抑郁（≥3次发作），当前缓解期
- 年龄: 18-75岁（均值 49岁）
- 排除: 当前抑郁发作（BDI < 9）、双相、精神病性障碍

**干预方案**
- MBCT团体（8周，每周2.5小时）+ 常规治疗
- 包含正念冥想、身体扫描、认知技术

**主要结局**

| 指标 | 测量工具 | 结果 |
|------|---------|------|
| 抑郁复发率 | DSM-IV 诊断 | MBCT组 47%，对照组 60%（HR=0.69, p=0.04） |
| 残留症状 | PHQ-9 | MBCT组显著低于对照组（d=0.37） |
| 复发时间 | — | MBCT组中位复发时间更长 |

**效应量**: d = 0.37（残留症状），HR = 0.69（复发风险）
**证据等级**: A

---

### 2. Segal et al. (2010) — 加拿大RCT

| 字段 | 内容 |
|------|------|
| **第一作者** | Segal, ZV |
| **发表年份** | 2010 |
| **期刊** | Archives of General Psychiatry, 67(4), 347-355 |
| **研究设计** | RCT（加拿大） |
| **样本量** | N = 169（MBCT组 n=86，MAST组 n=83） |
| **研究周期** | 8周干预 + 12个月随访 |

**主要结局**
- 抑郁复发率: MBCT组 37%，MAST组 56%（p=0.01）
- 残留症状改善更显著

**证据等级**: A

---

### 3. Williams et al. (2008) — 英国RCT

| 字段 | 内容 |
|------|------|
| **第一作者** | Williams, JM |
| **发表年份** | 2008 |
| **期刊** | Journal of Consulting and Clinical Psychology, 76(3), 466-475 |

**主要结局**
- 抑郁复发率显著降低（p<0.001）
- 对高复发风险患者效果更显著

**证据等级**: A

---

### 4. Geschke et al. (2022) — 元分析

| 字段 | 内容 |
|------|------|
| **研究类型** | 系统性元分析 |
| **样本量** | 14项RCT，N=2452 |
| **结论** | MBCT显著降低抑郁复发风险（RR=0.69） |

**证据等级**: A

## 临床应用结论

**一级推荐**（NICE指南，BPS指南）:
- 复发性抑郁预防（≥3次发作）
- 残留症状管理
- 缓解期患者

**效应量**: 中等（d=0.30-0.40）
**安全性**: 不良事件率低，无严重不良事件报告
