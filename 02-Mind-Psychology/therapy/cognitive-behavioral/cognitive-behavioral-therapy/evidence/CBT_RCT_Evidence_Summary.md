# CBT 认知行为疗法 RCT 证据摘要

> **状态**: 实证 (A级证据)
> **最后更新**: 2026-05-19
> **证据等级**: A — 强证据（多项高质量RCT及多项元分析支持）

---

## 代表性 RCT 证据汇总

### 1. Butler et al. (2006) — CBT 元分析

| 字段 | 内容 |
|------|------|
| **第一作者** | Butler, AC |
| **发表年份** | 2006 |
| **期刊** | Journal of Consulting and Clinical Psychology, 74(4), 651-659 |
| **研究设计** | 系统综述与元分析 |
| **样本量** | 16项RCT，N=2,653 |
| **研究周期** | 多项研究汇总 |

**研究人群**
- 诊断: 多种焦虑障碍（广泛性焦虑、社交焦虑、惊恐障碍等）
- 比较: CBT vs 等待名单对照

**主要结局**

| 指标 | CBT组 vs 对照组 | 效应量 |
|------|----------------|--------|
| 焦虑症状 | 显著降低 | d=0.82 |
| 抑郁症状 | 显著降低 | d=0.73 |
| 整体功能 | 显著改善 | d=0.68 |

**效应量**: d = 0.73-0.82（大效应量）
**证据等级**: A

---

### 2. Hofmann et al. (2012) — CBT 疗效综合元分析

| 字段 | 内容 |
|------|------|
| **第一作者** | Hofmann, SG |
| **发表年份** | 2012 |
| **期刊** | Cognitive Therapy and Research, 36(5), 427-440 |
| **研究设计** | 系统综述与元分析 |
| **样本量** | 106项研究，N=11,253 |
| **研究周期** | 多项研究汇总 |

**研究人群**
- 涵盖: 焦虑症、抑郁症、强迫症(OCD)、创伤后应激障碍(PTSD)、物质滥用
- 比较: CBT vs 对照条件

**主要结局**

| 病症 | 效应量 | 结论 |
|------|--------|------|
| 焦虑障碍 | 大 | CBT显著优于对照 |
| 抑郁症 | 大 | CBT显著优于对照 |
| OCD | 中-大 | CBT（含ERP）有效 |
| PTSD | 中-大 | CBT（含暴露疗法）有效 |
| 物质滥用 | 中 | CBT作为辅助治疗有效 |

**效应量**: 大效应量（跨病症）
**证据等级**: A

### 3. Cuijpers et al. (2019) — CBT vs 抗抑郁药物

| 字段 | 内容 |
|------|------|
| **第一作者** | Cuijpers, P |
| **发表年份** | 2019 |
| **期刊** | World Psychiatry, 18(3), 243-255 |
| **研究设计** | 系统综述与网络元分析 |
| **样本量** | 115项RCT，N=15,000+ |
| **研究周期** | 急性期 + 长期随访 |

**研究人群**
- 诊断: 重度抑郁症(MDD)
- 比较: CBT vs 药物治疗 vs 联合治疗

**主要结局**

| 比较 | 急性期疗效 | 长期随访 |
|------|-----------|---------|
| CBT vs 抗抑郁药 | 等效 | CBT更优（复发率低） |
| CBT+药物 vs 单一治疗 | 联合更优 | 联合更优 |
| CBT vs 等待名单 | 大效应量 | — |

**效应量**: 与药物等效，长期复发预防更优
**证据等级**: A

### 4. Clark et al. (2006) — CBT 治疗社交焦虑障碍

| 字段 | 内容 |
|------|------|
| **第一作者** | Clark, DM |
| **发表年份** | 2006 |
| **期刊** | Archives of General Psychiatry, 63(12), 1313-1321 |
| **研究设计** | 三臂RCT |
| **样本量** | N=144（CBT n=48，氟西汀 n=48，安慰剂 n=48） |
| **研究周期** | 14周干预 + 12个月随访 |
| **研究地区** | 英国 |

**研究人群**
- 诊断: DSM-IV 社交焦虑障碍
- 排除: 严重共病、物质滥用

**干预方案**
- CBT：个体治疗，14次，聚焦认知重建与行为实验
- 氟西汀：20-40mg/日
- 安慰剂对照

**主要结局**

| 时间点 | CBT | 氟西汀 | 安慰剂 |
|--------|-----|--------|--------|
| 14周（治疗结束） | 显著改善 | 显著改善 | 轻微改善 |
| 12个月随访 | 持续改善 | 部分复发 | — |

**关键发现**: CBT在12个月随访时显著优于氟西汀，复发率更低
**证据等级**: A

### 5. Driessen et al. (2015) — CBT 治疗慢性抑郁症

| 字段 | 内容 |
|------|------|
| **第一作者** | Driessen, E |
| **发表年份** | 2015 |
| **期刊** | American Journal of Psychiatry, 172(7), 650-658 |
| **研究设计** | RCT + 安慰剂对照 |
| **样本量** | N=132 |
| **研究周期** | 8周急性期 |

**研究人群**
- 诊断: 慢性抑郁症（持续性抑郁障碍）
- 比较: CBASP vs 安慰剂心理治疗

**干预方案**
- CBASP（认知行为分析系统心理治疗）：每周2次个体治疗
- 针对慢性抑郁症的专门化CBT方案

**主要结局**

| 指标 | CBASP组 | 对照组 | p值 |
|------|---------|--------|-----|
| 应答率 | 62% | 29% | p<0.01 |
| 缓解率 | 36% | 15% | p<0.05 |
| 功能改善 | 显著 | — | — |

**效应量**: 中-大效应量
**证据等级**: A

## 临床应用结论

**一级推荐**:
- 焦虑障碍（广泛性焦虑、社交焦虑、惊恐障碍）
- 抑郁症（重度抑郁症、慢性抑郁）
- 强迫症（结合ERP暴露反应预防）
- PTSD（结合创伤聚焦暴露）

**二级推荐**:
- 物质使用障碍（作为辅助治疗）
- 饮食障碍（结合其他干预）
- 失眠症（CBT-I方案）

**效应量**: 大（d=0.70-1.1）
**安全性**: 无严重不良事件报告，副作用极小
**长期优势**: 复发率低于单纯药物治疗
