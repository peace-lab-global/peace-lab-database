## 桩功训练

### 核心桩法

#### 1. 无极桩（基础站桩）

- **姿势**: 双脚与肩同宽、自然站立、双手垂于体侧
- **要领**: 百会上领、下颏微收、舌抵上腭、双肩松沉
- **意念**: 排除杂念、全身放松、体会重力自然下沉
- **时间**: 初始 3-5 分钟 → 逐渐增至 15-20 分钟
- **功效**: 培养松静、体会地心引力、建立身体觉知

#### 2. 太极桩（混元桩）

- **姿势**: 双脚略宽于肩、膝微曲、双手抱球于胸前
- **要领**: 含胸拔背、沉肩坠肘、松腰敛臀、气沉丹田
- **高度分级**:
  - 高桩: 膝弯曲约 120° — 初学者
  - 中桩: 膝弯曲约 90-120° — 进阶者
  - 低桩: 膝弯曲约 90° — 资深者
- **时间**: 初始 5 分钟 → 目标 20-30 分钟
- **功效**: 增强下肢力量、培养内劲、训练呼吸

#### 3. 开合桩

- **动作**: 在太极桩基础上，双手做开合运动
- **开**: 双手向外缓慢展开至肩宽，吸气
- **合**: 双手向内缓慢合拢至胸前，呼气
- **速度**: 约 6-8 次/分钟
- **功效**: 训练内劲感知、协调呼吸与动作

#### 4. 升降桩

- **动作**: 双手从体侧缓慢上升至肩高（吸气），再缓慢下降（呼气）
- **要领**: 手臂上升时想象水中浮起、下降时想象按水入地
- **功效**: 打通上下、培养升降劲路

### 桩功训练注意事项

| 注意事项 | 说明 |
|---------|------|
| 循序渐进 | 初始时间短、逐渐延长，避免过度 |
| 膝不过趾 | 膝关节弯曲时不超过脚尖，保护关节 |
| 身体不适即停 | 出现头晕、胸闷等不适应立即停止 |
| 避免空腹/饱腹 | 进食后1小时以上再练 |
| 环境选择 | 空气清新、地面平整、避风向阳 |

## 基本功与身法训练

### 手法基本功

| 手型 | 描述 | 训练方法 |
|------|------|---------|
| 太极掌 | 五指微展、掌心微含 | 静态保持、感受掌心气感 |
| 太极拳 | 四指卷曲、拇指扣食指 | 松握不紧、拳面平整 |
| 太极勾 | 五指捏拢、指尖朝下 | 腕部放松、指尖聚拢 |

### 步法基本功

| 步型 | 重心分配 | 训练要点 |
|------|---------|---------|
| 弓步 | 前7后3 | 前膝不过趾、后腿自然伸直 |
| 虚步 | 后9前1 | 后腿承重、前脚点地 |
| 仆步 | 侧9对1 | 蹲腿膝对脚尖、仆腿伸直 |
| 独立步 | 单腿100% | 支撑腿微曲、提膝过腰 |
| 马步 | 左5右5 | 双膝外撑、脚尖微外展 |

### 步法转换训练

1. **定步换重心**: 原地弓步前后移重心
2. **进步练习**: 上步、弓步、推掌
3. **退步练习**: 撤步、虚步、按掌
4. **侧步练习**: 横移步、云手步法
5. **转身练习**: 以脚跟/脚掌为轴旋转

### 腰胯训练

| 练习 | 方法 | 功效 |
|------|------|------|
| 转腰摆臂 | 双脚不动、以腰带动手臂左右摆动 | 松腰活胯 |
| 松腰沉坐 | 膝微曲、重心下沉如坐高凳 | 体会松沉 |
| 画圈运动 | 以腰为轴水平画圆 | 活化腰椎 |
| 涮胯训练 | 骨盆做前后左右及环形运动 | 松开髋关节 |

## 单式训练

### 太极拳八大基本式

| 式名 | 劲别 | 训练重点 | 难度 |
|------|------|---------|------|
| 起势 | 松沉 | 放松、沉气、入静 | ★☆☆☆☆ |
| 野马分鬃 | 掤捋 | 弓步转换、手臂分展 | ★★☆☆☆ |
| 白鹤亮翅 | 开合 | 虚实转换、上下对拉 | ★★☆☆☆ |
| 搂膝拗步 | 按劲 | 弓步推按、腰胯协调 | ★★☆☆☆ |
| 手挥琵琶 | 捋劲 | 虚步坐腕、前后对拉 | ★★★☆☆ |
| 倒卷肱 | 退步 | 退步连续、手脚协调 | ★★★☆☆ |
| 云手 | 旋转 | 侧步移动、腰带手运 | ★★★☆☆ |
| 单鞭 | 掤劲 | 全身整合、对拉拔长 | ★★★★☆ |

### 单式训练方法

1. **定式静练**: 定在最终姿势不动，体会劲力方向
2. **慢动反复**: 单式反复练习、每次关注不同身体部位
3. **左右对称**: 每个式子左右两侧都要练习
4. **配合呼吸**: 开/展/起 — 吸气；合/收/落 — 呼气
5. **组合串联**: 2-3 个单式连接成小组合

- **第9-10周**: 海底针至收势（式19-24）
- **第11-12周**: 全套连贯、呼吸配合、内劲贯穿

## 呼吸与内功

### 太极拳呼吸法

| 呼吸方式 | 方法 | 阶段 |
|---------|------|------|
| 自然呼吸 | 不刻意控制、随意呼吸 | 初学（0-1年） |
| 腹式呼吸 | 吸气腹部膨胀、呼气腹部收缩 | 入门（1-2年） |
| 逆腹式呼吸 | 吸气腹部收缩、呼气腹部膨胀 | 进阶（2-5年） |
| 拳势呼吸 | 呼吸与动作完全配合 | 高级（5年+） |
| 胎息 | 呼吸微细若存若亡 | 精深（10年+） |

### 呼吸与动作配合原则

- **起吸落呼**: 手臂上升吸气、下降呼气
- **开吸合呼**: 身体展开吸气、合拢呼气
- **蓄吸发呼**: 蓄劲时吸气、发力时呼气
- **虚吸实呼**: 重心后移吸气、前移呼气

## 推手入门

### 推手分类

| 类型 | 描述 | 训练目的 |
|------|------|---------|
| 单推手 | 单手搭手、一攻一守 | 感知对方力量方向 |
| 双推手 | 双手搭手、掤捋挤按 | 四正劲的实际运用 |
| 定步推手 | 双脚不动 | 在固定位置化发 |
| 活步推手 | 可移动步法 | 在运动中化发 |
| 大捋推手 | 四隅劲训练 | 采挒肘靠的运用 |
| 自由推手 | 无固定套路 | 综合运用、接近实战 |

### 推手核心——听劲

"听劲"是太极推手核心能力，通过皮肤接触感知对方意图：

| 能力层级 | 描述 | 训练方法 |
|---------|------|---------|
| 觉知 | 能感受到对方用力方向 | 闭眼搭手、静心感受 |
| 辨别 | 区分劲力大小和虚实 | 反复对练、体会变化 |
| 预判 | 预感对方即将发力 | 长期推手经验积累 |
| 引化 | 引导对方力量偏离 | 以小力化大力训练 |
| 发放 | 找准时机借力发力 | 粘黏连随中发放 |

## 常见问题与纠偏

| 常见问题 | 原因 | 纠正方法 |
|---------|------|---------|
| 膝关节疼痛 | 膝过趾/内扣 | 调整弓步角度、减少弯曲 |
| 架子不稳 | 下盘力量不足 | 增加桩功时间 |
| 动作僵硬 | 过度用力 | 减力练习、体会松沉 |
| 上下不协调 | 腰胯不松 | 加强腰胯训练 |
| 呼吸紊乱 | 过早配合呼吸 | 回归自然呼吸 |
| 速度不均 | 心急求快 | 用节拍器辅助均速 |
