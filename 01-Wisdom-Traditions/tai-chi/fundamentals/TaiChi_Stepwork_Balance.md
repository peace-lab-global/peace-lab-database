# 步法与平衡 | Stepwork & Balance

> "太极先看步"——步法是太极拳的根基。没有稳固的下盘，一切身法、手法、内劲都无从谈起。

---

## 一、太极拳步法的总原则

### 1.1 四大原则

| 原则 | 说明 |
|------|------|
| 虚实分明 | 重心清楚分配，非双重（两腿均实） |
| 轻起轻落 | 脚如猫行、起落无声 |
| 裆走下弧 | 重心转换走下弧线，非水平移动 |
| 先落脚再移重心 | 脚跟先着地→全脚掌→再移重心 |

### 1.2 步法与身法的关系

- **脚为根**: 脚底的蹬地力是一切力量的起点
- **腰为轴**: 腰部旋转控制步法方向
- **手为梢**: 步法到手法的完整力量传导链

## 二、五种基本步法

### 2.1 弓步（Gong Bu）

**定义**: 前腿屈膝弓出、后腿伸直蹬地

**要领:**
1. 前膝弯曲、膝盖不超过脚尖
2. 前小腿垂直或微前倾
3. 后腿自然伸直（不锁死膝关节）
4. 两脚横向间距约一肩宽（避免走在一条线上）
5. 前脚尖朝前、后脚外展约45度
6. 重心前七后三（偏前）

**常见错误:**

| 错误 | 表现 | 纠正 |
|------|------|------|
| 膝过脚尖 | 前膝前伸超过脚尖 | 调整步幅、身体不前倾 |
| 双重 | 重心5:5分配 | 明确前七后三 |
| 走钢丝 | 两脚在一条直线上 | 保持一肩宽间距 |
| 后脚跟离地 | 后脚不稳 | 后脚踏实、微外展 |

### 2.2 虚步（Xu Bu）

**定义**: 后腿承重、前脚虚点

**要领:**
1. 后腿屈膝承重约90%
2. 前脚以脚尖或前脚掌轻点地面
3. 身体重心偏后
4. 前腿微屈（不完全伸直）
5. 保持身法中正

**功能**: 灵活变换方向、蓄势待发

### 2.3 仆步（Pu Bu）

**定义**: 一腿全蹲、一腿平铺

**要领:**
1. 蹲腿：大腿尽量平行地面、脚跟不离地
2. 铺腿：全腿贴地、脚尖内扣
3. 两脚全脚掌着地
4. 上体保持中正、不前俯

**注意**: 对膝关节要求高，初学者不宜过深

### 2.4 歇步（Xie Bu）

**定义**: 两腿交叉、后腿跪于前腿后侧

**要领:**
1. 前腿屈膝、大腿近水平
2. 后腿膝盖靠近前腿膝窝后方
3. 后脚跟离地、以脚尖支撑
4. 臀部接近后脚跟
5. 上体中正

**太极中的应用**: 下势（海底针）、歇步推掌

### 2.5 独立步（Du Li Bu）

**定义**: 单腿站立、另一腿提起

**要领:**
1. 支撑腿微屈（不完全锁死）
2. 提起腿膝高过腰、脚尖自然下垂
3. 虚灵顶劲、保持平衡
4. 目视前方一点

**代表招式**: 金鸡独立、白鹤亮翅（变体）

## 三、重心转换技术

### 3.1 虚实转换的核心原则

- **渐变**: 重心转换是渐进的、非突变的
- **裆走下弧**: 重心从一腿移至另一腿时，胯部走下弧线
- **先实后移**: 先完全实于一腿，再缓缓移向另一腿

### 3.2 重心转换的四步法

```
1. 实腿稳定 → 确认支撑腿完全稳定
2. 虚腿轻提 → 虚脚脚跟先离地
3. 轻落虚处 → 虚脚脚跟先着地
4. 缓缓移重 → 重心如水流般缓缓转移
```

### 3.3 双重之病

**定义**: 双重是指两腿重心分配不清（5:5），导致行动迟缓、被推易倒

**王宗岳原文**: "偏沉则随、双重则滞"

**避免方法:**
- 每个动作都明确虚实（如 7:3 或 8:2）
- 即使是"定势"也有明确的虚实分配
- 在转换过程中，虚实不断交替、如波浪起伏

## 四、猫步训练法

### 4.1 什么是猫步

猫步（Cat Walk）是太极拳最核心的步法训练，模拟猫行走时的轻盈、稳定与协调。

### 4.2 猫步练习步骤

**基础猫步（前进）:**
1. 站立、重心在右腿（实）
2. 左腿轻轻提起（如猫抬爪）
3. 左脚向前迈出一步（脚跟先着地）
4. 缓缓将重心从右腿移至左腿
5. 左脚全掌着地、右脚跟提起
6. 右脚跟进、收至左脚旁
7. 换右腿重复

**进阶猫步（后退）:**
- 后退时脚尖先着地→全掌→移重心
- 保持裆走下弧

**猫步质量评估:**

| 标准 | 评估方法 |
|------|---------|
| 无声 | 脚步落地无声音 |
| 无波 | 身体高度无起伏 |
| 稳定 | 任何时刻都能停下来保持平衡 |
| 轻灵 | 如履薄冰、如猫行雪 |

## 五、平衡训练进阶

### 5.1 三级平衡训练

**初级：静态平衡**
- 单腿站立（睁眼）：目标30秒
- 单腿站立（闭眼）：目标15秒
- 弓箭步定势：目标30秒

**中级：动态平衡**
- 猫步前进/后退：10步往返
- 云手配合步法移动
- 倒撵猴（后退步法）

**高级：对抗平衡**
- 推手中的平衡维持
- 单腿站桩+外力干扰
- 器械练习（剑/刀的平衡要求更高）

### 5.2 平衡改善的科学机制

| 机制 | 说明 |
|------|------|
| 本体感觉增强 | 脚踝/膝关节感受器敏感度提升 |
| 前庭功能改善 | 头部位置变化时的稳定能力 |
| 下肢肌力增加 | 股四头肌、小腿肌群力量 |
| 视觉整合优化 | 视觉-前庭-本体感觉三重系统协调 |
| 反应时间缩短 | 失衡时的快速纠正反应 |

### 5.3 每日平衡训练方案（15分钟）

| 内容 | 时间 |
|------|------|
| 猫步前进 | 3分钟 |
| 猫步后退 | 3分钟 |
| 单腿站立（左右各） | 2分钟×2 |
| 弓步/虚步转换 | 3分钟 |
| 云手+步法 | 2分钟 |
