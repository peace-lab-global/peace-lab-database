# 太极器械 | Tai Chi Weapons Forms

> 器械是拳法的延伸。太极器械以太极身法为基础，将内劲通过器械传导至更远的空间，是太极修炼的高阶内容。

---

## 一、器械修炼的意义

| 价值 | 说明 |
|------|------|
| 劲力延伸 | 器械是手臂的延长，训练远距离发力 |
| 身法强化 | 器械的重量与长度要求更精准的身法 |
| 专注力 | 持械练习需要更高的注意力 |
| 传统传承 | 器械套路保留了传统武术的完整性 |
| 表演价值 | 器械表演更具观赏性 |

---

## 二、太极剑

### 2.1 概述

太极剑是太极器械中最重要、练习人数最多的器械。

**基本规格:**
- 长度: 反手持剑时剑尖齐耳
- 重量: 竞赛用剑约300-500g，练习用稍重
- 握法: 反手持剑（剑刃朝上/下灵活切换）

### 2.2 太极剑十三法

| 剑法 | 说明 | 代表动作 |
|------|------|---------|
| 抽 | 向后拉剑 | 怀中抱月 |
| 带 | 顺势牵引 | 左右拦扫 |
| 提 | 向上提起 | 提膝提剑 |
| 格 | 格挡对方 | 格剑式 |
| 击 | 直线出击 | 仙人指路 |
| 刺 | 剑尖直刺 | 弓步直刺 |
| 点 | 剑尖点啄 | 燕子衔泥 |
| 崩 | 弹起剑尖 | 崩剑式 |
| 劈 | 下劈 | 劈剑式 |
| 截 | 横向截断 | 截剑式 |
| 抹 | 弧形横抹 | 抹剑式 |
| 撩 | 由下向上 | 撩剑式 |
| 挂 | 挂住对方器械 | 挂剑式 |

### 2.3 主要套路

| 套路 | 式数 | 基础流派 | 特点 |
|------|------|---------|------|
| 32式太极剑 | 32式 | 杨式 | 入门竞赛套路 |
| 67式太极剑 | 67式 | 杨式（传统） | 传统长套路 |
| 陈式太极剑 | 49式 | 陈式 | 含缠丝与发力 |
| 武当太极剑 | 多种 | 武当 | 道教风格 |

## 三、太极刀

### 3.1 概述

太极刀以单刀为主，刀法刚猛有力，比太极剑更具武术气息。

**基本规格:**
- 长度: 从地面到腰际
- 特点: 单面刃、刀背较厚
- 左手: 配"刀手"（掌法配合）

### 3.2 太极刀核心刀法

| 刀法 | 说明 |
|------|------|
| 缠头裹脑 | 刀绕头部旋转（防御） |
| 劈刀 | 由上向下劈砍 |
| 砍刀 | 斜方向劈砍 |
| 撩刀 | 由下向上 |
| 扎刀 | 刀尖直刺 |
| 抹刀 | 弧形横抹 |
| 斩刀 | 横向斩切 |

### 3.3 陈式太极单刀

- 共22式，动作刚猛
- 保留传统武术刀法精华
- 含跳跃、转身等高难动作

---

## 四、太极枪/杆

### 4.1 太极杆（大杆/大枪）

**概述:**
- 长度: 3-4米的白蜡杆
- 重量: 较重，训练臂力与整劲
- 功能: 培养太极长劲的最佳器械

**核心训练方法——抖杆:**
1. 双手持杆（前手松、后手紧）
2. 以腰发力、向前抖出
3. 杆尖发出"啪"声（脆劲）
4. 训练整劲与穿透力

**杆法:**
- 拦: 横向拦截
- 拿: 控制对方枪杆
- 扎: 直线刺出

### 4.2 太极十三枪

传统十三枪法：开、合、崩、劈、点、扎、拨、撩、缠、带、滑、截、挑

## 五、太极扇

### 5.1 概述

- 参见: [太极功夫扇](../schools-lineage/TaiChi_Contemporary_Styles.md) 详述
- 扇法: 开扇、合扇、撩、劈、云、穿、点
- 特色: 开扇时的声响增加气势

### 5.2 太极双扇

- 双手各持一扇
- 对协调性要求更高
- 表演性强、受众广泛

---

## 六、其他太极器械

### 6.1 太极双钩

- 器械: 一对虎头钩
- 特点: 钩法独特、含锁拿技法
- 难度: 较高

### 6.2 太极拂尘

- 器械: 拂尘（源自道家）
- 特点: 柔软器械、以柔克刚
- 与武当太极关系密切

### 6.3 太极双剑/双刀

- 双手各持一械
- 对协调能力要求极高
- 属高级器械

---

## 七、器械修炼的进阶路径

```
阶段1：徒手（拳架基础）
  → 至少1年拳架基础后才开始器械
阶段2：太极剑（首选器械）
  → 32式太极剑入门→67式传统剑
阶段3：太极刀
  → 刀法刚猛、训练爆发力
阶段4：太极杆/枪
  → 培养长劲与整劲
阶段5：其他器械
  → 扇、钩、双器械等
```
