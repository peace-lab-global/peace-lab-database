# 前行法 (Ngöndro / Preliminary Practices)

> **前行法 (Ngöndro)**：藏传佛教密法修行前的基础净化与积资修法，通过大礼拜、金刚萨埵、献曼达、上师瑜伽等四加行，为正行打下坚实基础。
> 
> *"前行比正行更重要，因为没有前行的净化与积资，正行无法生效。" —— 巴楚仁波切*
> 
> *藏语：སྔོན་འགྲོ (Ngöndro) / "前行" / 梵语：Pūrvaka (पूर्वक)*
> 
> 前行法是藏传佛教各派的共同基础，通常要求完成十万遍的大礼拜、金刚萨埵百字明、献曼达供、上师瑜伽，部分传承还包括皈依、发心等额外加行。

### 表5.2 前行修行注意事项 (Practice Cautions)

| 注意事项 | 内容 | 建议 |
|---|---|---|
| **传承** | 必须从上师处获得传承 | 先找具格上师 |
| **次第** | 按顺序修完 | 不可跳过 |
| **质量** | 质量重于数量 | 每遍专注 |
| **持续** | 每日不间断 | 养成习惯 |
| **回向** | 每座结束回向 | 功德圆满 |
| **记数** | 准确记录数量 | 使用念珠 |
| **身体** | 大礼拜注意保护膝盖 | 使用垫子 |

### 表5.3 常见问题与解决 (Common Problems & Solutions)

| 问题 | 原因 | 解决方法 |
|---|---|---|
| **膝盖疼痛** | 大礼拜姿势不当 | 使用护膝、调整姿势 |
| **散乱** | 缺乏专注 | 放慢速度、加强观想 |
| **懈怠** | 缺乏动力 | 思维前行功德、找共修 |
| **怀疑** | 信心不足 | 学习传承、祈请上师 |
| **时间不够** | 生活繁忙 | 早起、调整优先级 |
| **数量不对** | 记错数 | 认真记数、使用计数器 |
