# 四阿含经（Āgama Sūtras）

## 基本信息

| 项目 | 内容 |
|------|------|
| **性质** | 原始佛教根本经典，佛陀教法最直接记录 |
| **译者** | 佛陀耶舍、竺佛念等（后秦时期） |
| **来源** | 《大正新修大藏经》第1-2册 |
| **对应** | 南传巴利三藏尼柯耶（Nikāya） |
| **语言** | 汉译梵语/巴利语佛典 |
| **时代** | 公元前5-前4世纪结集，公元4-5世纪汉译 |

四阿含经是佛教最早期的经典结集，记录了佛陀及其直接弟子们的言教，被公认为最接近佛陀本怀的教法。南传佛教将相应经典称为"尼柯耶"（Nikāya），北传佛教则称为"阿含"（Āgama，意为"传承"或"结集"）。

---

## 四部阿含分别介绍

### 一、长阿含经（Dīrgha Āgama）

| 项目 | 内容 |
|------|------|
| **卷数** | 22卷 |
| **经数** | 30经 |
| **对应** | 南传《长部》（Dīgha Nikāya，34经） |
| **特点** | 篇幅较长，多为佛陀与外道论师对话 |
| **主题** | 世界起源、宇宙论、修行次第、佛陀生平 |

#### 主要内容：
- **《游行经》**（第2-4卷）：记录佛陀最后游行教化、入涅槃及八王分舍利等事迹
- **《大会经》**：描述三十三天众神大会，佛陀升天为母说法
- **《小缘经》**：说明四种姓平等及婆罗门起源
- **《长阿含经》**：论述劫数、世界成坏、众生住处等宇宙观
- **《起世经》**：详细描述世界生成与众生起源

#### 原文摘录：
> **《游行经》卷中（佛临涅槃）**
> 
> 尔时，世尊告阿难曰："汝今知不？我昔与汝共于金山，修集梵行，于时汝度于我，我亦度于汝。如是展转，经八万四千由旬，于生死中无穷已。"
> 
> "阿难，我当为汝说微妙法，初中后善，善义善味，纯一满净，梵行清白，所谓《四十二章经》。汝当谛听，善思念之。"
> 
> 佛告阿难："我自忆念，过去世时，于金山中修集梵行，尔时汝以阿耨多罗三藐三菩提心，先行动导。阿难，汝今亦当如是，发阿耨多罗三藐三菩提心，劝进于我。"

---

### 二、中阿含经（Madhyama Āgama）

| 项目 | 内容 |
|------|------|
| **卷数** | 60卷 |
| **经数** | 222经（现行本） |
| **对应** | 南传《中部》（Majjhima Nikāya，152经） |
| **特点** | 篇幅适中，义理深入 |
| **主题** | 分别诸法深义，系统阐述教理 |

#### 主要内容：
中阿含经是四阿含中义理最为系统的一部，详细阐述：
- 禅定修持次第（四禅八定）
- 四圣谛的深入解说
- 十二因缘的详细分析
- 五蕴、十二处、十八界的解析
- 声闻道、缘觉道、菩萨道的修行方法

#### 原文摘录：
> **《分别圣谛经》**
> 
> 佛告诸比丘："有四圣谛。何等为四？谓苦圣谛、苦集圣谛、苦灭圣谛、苦灭道迹圣谛。若比丘于此四圣谛，如实知见，明了解脱，是名阿罗汉，诸漏已尽，所作已办，舍于重担，逮得己利，尽诸有结，正智心解脱。"
> 
> "比丘，云何苦圣谛？谓生苦、老苦、病苦、死苦、恩爱别苦、怨憎会苦、所求不得苦、略五受阴苦，是谓苦圣谛。"
> 
> "云何苦集圣谛？谓当来有爱，喜欲共俱，贪喜于彼，彼彼乐着，是谓苦集圣谛。"
> 
> "云何苦灭圣谛？谓当来有爱，喜欲共俱，贪喜于彼，彼彼乐着无余断，吐尽、离欲、灭、没，是谓苦灭圣谛。"
> 
> "云何苦灭道迹圣谛？谓八圣道：正见、正志、正语、正业、正命、正方便、正念、正定，是谓苦灭道迹圣谛。"

---

### 三、杂阿含经（Saṃyukta Āgama）

| 项目 | 内容 |
|------|------|
| **卷数** | 50卷 |
| **经数** | 1362经（现行本） |
| **对应** | 南传《相应部》（Saṃyutta Nikāya，7762经） |
| **特点** | 篇幅短小，义理相应 |
| **主题** | 五蕴、六入、因缘、道品 |

#### 主要内容：
杂阿含经是四阿含中篇幅最多、内容最丰富的一部，按主题分为相应：
- **阴相应**：五蕴（色、受、想、行、识）的分析
- **入处相应**：十二处（六根、六尘）的观察
- **因缘相应**：十二因缘、缘起法的深入
- **道品相应**：三十七道品的修持
- **声闻相应**：声闻弟子修道证果的事迹

#### 原文摘录：
> **《杂阿含经》卷二（无常经）**
> 
> 如是我闻：一时，佛住舍卫国祇树给孤独园。尔时，世尊告诸比丘："色无常，无常即苦，苦即非我，非我者即非我所。如是观者，名真实正观。如是受、想、行、识无常，无常即苦，苦即非我，非我者即非我所。如是观者，名真实正观。"
> 
> "圣弟子如是观者，于色解脱，于受、想、行、识解脱。我说是等解脱于生老病死、忧悲苦恼。"
> 
> 时，诸比丘闻佛所说，欢喜奉行。

> **《杂阿含经》卷十（缘起经）**
> 
> 佛告比丘："云何缘起法？谓此有故彼有，此生故彼生。谓缘无明行，缘行识，缘识名色，缘名色六入处，缘六入处触，缘触受，缘受爱，缘爱取，缘取有，缘有生，缘生老死、忧悲恼苦，如是纯大苦聚集。"
> 
> "云何缘生法？谓无明、行、识、名色、六入处、触、受、爱、取、有、生、老死，是名缘生法。"
> 
> "比丘，若佛出世，若未出世，此法常住，法住法界，彼如来自所觉知，成等正觉，为人演说，开示显发，谓缘无明行，乃至缘生有老死。若佛出世，若未出世，此法常住，法住法界，彼如来自觉知，成等正觉，为人演说，开示显发。"

### 四、增一阿含经（Ekottara Āgama）

| 项目 | 内容 |
|------|------|
| **卷数** | 51卷 |
| **经数** | 472经（现行本） |
| **对应** | 南传《增支部》（Aṅguttara Nikāya，9557经） |
| **特点** | 法数递增，由一至十 |
| **主题** | 菩萨道、天人数目、修行阶位 |

#### 主要内容：
增一阿含经以数字为次第组织教法：
- **一法**：一乘道、一甘露门、一归依处
- **二法**：二甘露门（止观）、二种果报
- **三法**：三宝、三法印、三皈依
- **四法**：四圣谛、四念处、四正勤
- **五法**至**十法**：依次递增

与其他三部阿含相比，增一阿含含有更多的菩萨道思想，显示了大乘佛教的萌芽。

#### 原文摘录：
> **《增一阿含经》卷一（序品）**
> 
> 尔时，世尊告诸比丘："若有比丘成就一法，不能自在，诸根驰散。何等为一？谓无正念。如是，比丘，若不成就此一法者，则诸根驰散。若复比丘成就一法，便得自在，诸根不驰散。何等为一？谓正念。如是，比丘，若成就此一法，便得自在，诸根不驰散。"
> 
> "复有二法，二业、二足、二行、二缘、二虑、二念、二定。云何为二法？谓一法喜、一法忧。云何二业？谓一善业、一恶业。云何二足？谓一福足、一慧足。云何二行？谓一止、一观。云何二缘？谓一因、一缘。云何二虑？谓一内、一外。云何二念？谓一正念、一邪念。云何二定？谓一有漏定、一无漏定。"

> **《增一阿含经》卷四十四（十不善业道）**
> 
> 世尊告诸比丘："有十恶业，云何为十？所谓身三、口四、意三。云何身三？谓杀、盗、淫。云何口四？谓两舌、恶口、妄言、绮语。云何意三？谓嫉妒、嗔恚、邪见。是谓，比丘，十恶业，如此十业，皆恶不善。"

## 核心教义

### 一、四圣谛（Cattāri Ariyasaccāni）

四圣谛是佛陀在鹿野苑初转法轮所宣说的根本教法：

| 圣谛 | 内涵 | 比喻 |
|------|------|------|
| **苦谛** | 世间一切本质是苦 | 如病 |
| **集谛** | 苦的原因（贪嗔痴） | 如病因 |
| **灭谛** | 苦的止息（涅槃） | 如病愈 |
| **道谛** | 灭苦的方法（八正道） | 如药方 |

> **原文摘录**（《杂阿含经》卷十五）：
> 
> "比丘，于四圣谛平等正觉，名为如来、应、等正觉。何等为四？所谓苦圣谛、苦集圣谛、苦灭圣谛、苦灭道迹圣谛。于此四圣谛平等正觉，名为如来、应、等正觉。"

---

### 二、八正道（Ariyo Aṭṭhaṅgiko Maggo）

八正道是灭苦的具体修行方法，分为三学：

```
八正道
├── 戒学（道德基础）
│   ├── 正见（Right View）
│   ├── 正志/正思维（Right Intention）
│   ├── 正语（Right Speech）
│   ├── 正业（Right Action）
│   └── 正命（Right Livelihood）
├── 定学（禅定修持）
│   ├── 正方便/正精进（Right Effort）
│   ├── 正念（Right Mindfulness）
│   └── 正定（Right Concentration）
└── 慧学（智慧解脱）
    └── 贯穿前三学
```

> **原文摘录**（《中阿含经》卷三）：
> 
> "云何八正道？谓正见、正志、正语、正业、正命、正方便、正念、正定。云何正见？谓此见有施、有斋、亦有言，有善、恶、业、有报，有此世、有他世，有父、有母，世有真人，往至善处，善逝，向于善，此世他世，自知、自觉，自作证成就游。是谓正见。"

---

### 三、十二因缘（Paṭicca-samuppāda）

十二因缘说明轮回流转与解脱还灭的因果关系：

**流转门（生死轮回）**：

| 支 | 含义 | 说明 |
|----|------|------|
| 无明 | 无知 | 不明因果、不明四谛 |
| ↓ 行 | 造作 | 身口意三业 |
| ↓ 识 | 了别 | 业识投胎 |
| ↓ 名色 | 精神与物质 | 胎儿初成 |
| ↓ 六入 | 六根 | 眼耳鼻舌身意 |
| ↓ 触 | 接触 | 根尘相接 |
| ↓ 受 | 感受 | 苦乐舍三受 |
| ↓ 爱 | 贪爱 | 对乐受的执着 |
| ↓ 取 | 执取 | 追求占有 |
| ↓ 有 | 业因 | 造作新业 |
| ↓ 生 | 出生 | 再受生 |
| ↓ 老死 | 衰老死亡 | 忧悲苦恼 |

**还灭门（解脱涅槃）**：
灭无明则行灭，行灭则识灭……生灭则老死忧悲苦恼灭。

> **原文摘录**（《长阿含经》卷十）：
> 
> "缘无明有行，缘行有识，缘识有名色，缘名色有六入，缘六入有触，缘触有受，缘受有爱，缘爱有取，缘取有有，缘有有生，缘生有老死忧悲苦恼，大苦阴集。此经名为《大缘方便》"

---

### 四、五蕴无我（Pañcakkhandhā Anattā）

五蕴是构成生命现象的五个要素，皆无常、苦、无我：

| 蕴 | 含义 | 分析 |
|----|------|------|
| **色蕴** | 物质 | 四大（地水火风）及所造色 |
| **受蕴** | 感受 | 苦受、乐受、不苦不乐受 |
| **想蕴** | 取相 | 了知、分别、概念 |
| **行蕴** | 造作 | 意志、心理活动 |
| **识蕴** | 了别 | 六识（眼耳鼻舌身意识） |

> **原文摘录**（《杂阿含经》卷二）：
> 
> "比丘，云何观察色如聚沫？比丘，所有色，若过去、若未来、若现在，若内、若外，若粗、若细，若好、若丑，若远、若近，彼一切如实观察，无常、苦、非我、非我所，如实知。如是观者，于色厌离，心得解脱。"
> 
> "比丘，云何观察受如泡？比丘，所有受，若过去、若未来、若现在，若内、若外，若粗、若细，若好、若丑，若远、若近，彼一切如实观察，无常、苦、非我、非我所，如实知。如是观者，于受厌离，心得解脱。"

---

### 五、涅槃寂静（Nibbāna/Nirvāṇa）

涅槃是佛教修行的终极目标，可分为：

| 类型 | 梵/巴利语 | 说明 |
|------|----------|------|
| **有余涅槃** | Sopādisesa-nibbāna | 断尽烦恼，仍有肉身 |
| **无余涅槃** | Anupādisesa-nibbāna | 断尽烦恼，肉身亦灭 |
| **自性涅槃** |  | 一切众生本具的清净本性 |

> **原文摘录**（《杂阿含经》卷十八）：
> 
> "涅槃者，贪欲永尽，嗔恚永尽，愚痴永尽，一切诸烦恼永尽，是名涅槃。"
> 
> "如是无闻愚痴凡夫，于苦生苦，于苦灭生乐，于乐生乐，于乐灭生苦，如是不得出离苦边。圣弟子不然，于苦如实知苦，于苦灭如实知苦灭，于乐如实知乐，于乐灭如实知乐灭，如是则能出离苦边，是名圣弟子如实知苦、苦集、苦灭、苦灭道迹。"

## 重要单经详解

### 一、《游行经》——佛陀的最后教化

《游行经》是《长阿含经》的核心经典，详细记录了佛陀入涅槃前后的重要事迹：

**主要内容**：
1. **最后游行**：佛陀从王舍城出发，经那烂陀、波吒离村等地，向拘尸那竭城行进
2. **度化最后弟子**：于路途中度化铁匠纯陀（Cunda）
3. **纯陀供食**：接受纯陀供养的栴檀树耳（Sukara-maddava），引发疾病
4. **遗教嘱咐**：在波波村为众比丘说《遗教经》，咐嘱以戒为师
5. **度化须跋陀罗**：于娑罗双树间度化最后一位弟子
6. **入般涅槃**：于拘尸那竭城娑罗双树间入灭
7. **八王分舍利**：佛灭后八国争分舍利，起塔供养

> **原文摘录**（《游行经》卷下）：
> 
> 佛告阿难："我自觉悟成最正觉，从是已来，所可说法、所可戒律，尽当护持，勿令朽败。阿难，我成佛道已来，所说经戒，即是汝护，是汝所持。阿难，汝当于后授我四部之众，汝当精勤，广设方便，令法久住。"
> 
> 佛告阿难："我般涅槃后，汝等当尊重珍敬波罗提木叉，如暗遇明、贫人得宝，当知此则是汝大师。若我住世，无异此也。持净戒者，不得贩卖贸易、安置田宅、畜养人民奴婢畜生，一切种植及诸财宝，皆当远离如避火坑。"

---

### 二、《弊宿经》——轮回论证

《弊宿经》见于《长阿含经》第七卷，是佛陀与婆罗门弊宿（Pāyāsi）关于轮回的论辩：

**主要论点**：
弊宿婆罗门持断灭见，认为人死后身心俱灭，无有后世。佛陀以种种善巧方便破其邪见：

1. **日月的比喻**：如日月运行于天上，人眼不能见其运行轨迹，不能说日月不动
2. **盗贼的比喻**：如盗贼被捉，财物被夺，不能说没有因果
3. **梦喻**：如人梦中见种种境界，不能说完全没有
4. **天眼论证**：修行禅定者可得天眼，能见众生生死轮回

> **原文摘录**（《弊宿经》）：
> 
> 弊宿言："沙门，我坐此处，忽见有人，犯王法者，为王所捉，反缚两手，令唤罪人，以刀斫头，我即往观。我观此人，临欲死时，语我言：'汝今见我死，身中神识从此而出，上于空中，我欲见汝。'我即时观，求其神识，周遍求之，而不相见。"
> 
> 佛言："居士，譬如有人，以毒置少肉中，以闭口瓮覆之，持诣深林，置于树下，即便舍去。时，有毒蛇，闻肉香故，便入瓮中，食彼毒肉，毒发即死。如是，居士，人死神识，从此舍身，受他身者，亦复如是。居士，汝今云何于肉眼中求那罗延身？"

### 三、《布吒婆楼经》——识的论述

《布吒婆楼经》（Pottapāda Sutta）见于《长阿含经》第八卷，是佛陀与布吒婆楼婆罗门的对话，详细讨论心识的本质：

**主要内容**：
1. **三界唯心**：论述心识与境界的关系
2. **识住九种**：详细说明九种识住（Vijñānasthiti）
3. **想灭定**：说明想受灭定（Saññāvedayitanirodha）的境界
4. **我与我所**：破斥神我观念，阐明无我正见

**九种识住**：
1. 众生有种种身、种种想（欲界人天）
2. 众生有种种身、一想（初禅梵天）
3. 众生有一身、种种想（光音天）
4. 众生有一身、一想（遍净天）
5. 空无边处天
6. 识无边处天
7. 无所有处天
8. 非想非非想处天
9. 想受灭定

> **原文摘录**（《布吒婆楼经》）：
> 
> 佛言："布吒婆楼，云何识住？我言说识住，有九种。或有众生，有种种身、种种想，谓人及欲界天，是名第一识住。或有众生，有种种身、一想，谓初禅天，是名第二识住。或有众生，有一身、种种想，谓光音天，是名第三识住。或有众生，有一身、一想，谓遍净天，是名第四识住。"
> 
> "或有众生，度一切色想，灭有对想，不念种种想，入无边空，空无边处具足住，是名第五识住。或有众生，度一切空无边处，入无边识，识无边处具足住，是名第六识住。或有众生，度一切识无边处，入无所有，无所有处具足住，是名第七识住。或有众生，度一切无所有处，入非想非非想处具足住，是名第八识住。"
> 
> "或有众生，度一切非想非非想处，入想知灭，身作证具足住，是名第九识住。布吒婆楼，是名九识住。"

## 阿含经的现代意义

### 修行指导价值

1. **正念修习**：四念处（观身不净、观受是苦、观心无常、观法无我）为现代正念疗法（MBSR、MBCT）的理论源头
2. **情绪管理**：八正道中的正语、正业、正命为现代伦理学提供基础
3. **压力释放**：观呼吸、安那般那念等禅修方法广泛应用于现代心理治疗

### 学术研究价值

1. **比较宗教学**：阿含经是研究早期佛教与印度其他宗教（婆罗门教、耆那教等）关系的重要资料
2. **佛教史研究**：记录佛陀时代的社会、政治、文化背景
3. **语言学研究**：汉译阿含保留了大量早期佛教术语

### 原文结语

> **《杂阿含经》卷十**
> 
> "诸比丘，若于我法之中，能勤修学者，必自知、自觉，自作证：我生已尽，梵行已立，所作已办，自知不受后有。"
