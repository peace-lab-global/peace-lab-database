# 净土宗五经

净土宗以「五经一论」为核心理论依据，其中「五经」是修行净土法门的根本经典。这五部经典分别从不同角度阐述了阿弥陀佛的大愿、西方极乐世界的庄严、以及往生净土的方法。

---

## 一、佛说无量寿经

### 基本信息

| 项目 | 内容 |
|------|------|
| **译者** | 康僧铠（梵名 Śaṅghavarman） |
| **年代** | 三国曹魏嘉平四年（252年） |
| **卷数** | 二卷 |
| **全名** | 《佛说无量寿经》卷上、卷下 |
| **别称** | 大阿弥陀经、大经、双卷经 |

### 核心内容

《无量寿经》是净土宗的根本经典，被誉为「净土第一经」。本经详细记载了阿弥陀佛在因地作法藏比丘时，发四十八大愿，经过无量劫修行，成就西方极乐世界，以及十方众生往生净土的方法。

本经的核心思想可以概括为「愿行果」：
- **发愿**：法藏比丘发四十八大愿
- **修行**：无量劫积功累德
- **证果**：成就西方极乐世界，接引众生

### 四十八大愿

四十八大愿是阿弥陀佛在因地作法藏比丘时，向世自在王如来所发的宏大誓愿。这些愿力可分为三大类：

#### （一）摄法身愿（1-3愿）

| 愿号 | 愿名 | 内容 |
|------|------|------|
| 第一 | 无三恶趣愿 | 设我得佛，国有地狱饿鬼畜生者，不取正觉 |
| 第二 | 不更恶趣愿 | 设我得佛，国中天人不住定聚，必至灭度者，不取正觉 |
| 第三 | 悉皆金色愿 | 设我得佛，国中天人不悉真金色者，不取正觉 |

#### （二）摄净土愿（4-13愿）

阐述极乐世界的庄严相好，包括没有男女之别、天人皆具六神通、光明无量、寿命无量等。

#### （三）摄众生愿（14-48愿）

这是四十八大愿的核心，特别是第十八愿最为重要。

### 第十八愿——十念必生愿

> **「设我得佛，十方众生，至心信乐，欲生我国，乃至十念，若不生者，不取正觉。唯除五逆，诽谤正法。」**

这是净土宗最核心的愿力，被称为「本愿王」或「王本愿」。

**第十八愿的要义：**

1. **至心信乐**——具足深信切愿
2. **欲生我国**——发愿往生
3. **乃至十念**——下至十念、上至多念，执持名号
4. **若不生者，不取正觉**——阿弥陀佛的保证

善导大师在《观经疏》中解释：
> 「一者决定深信：自身现是罪恶生死凡夫，旷劫以来常没常流转，无有出离之缘；二者决定深信：彼阿弥陀佛四十八愿，摄受众生，无疑无虑，乘彼愿力，定得往生。」

### 重要原文摘录

**法藏菩萨发愿：**

> 「我建超世愿，必至无上道，斯愿不满足，誓不成等觉。
> 我于无量劫，不为大施主，普济诸贫苦，誓不成等觉。
> 我至成佛道，名声超十方，究竟有不闻，誓不成等觉。」

**极乐世界庄严：**

> 「设我得佛，国中菩萨，不住定聚，必至灭度者，不取正觉。」

> 「彼佛国土，微风吹动诸宝行树及宝罗网，出微妙音，譬如百千种乐，同时俱作。」

**三辈往生：**

经中将往生众生分为上辈、中辈、下辈三类：
- **上辈**：舍家弃欲而作沙门，发菩提心，专念无量寿佛
- **中辈**：不能出家，当发无上菩提之心，一向专念无量寿佛
- **下辈**：不能作诸功德，当发无上菩提之心，一向专意，乃至十念，念无量寿佛

### 净土宗修行指导

《无量寿经》的修行要点：

1. **发菩提心**：发愿成就佛道，广度众生
2. **一向专念**：专一称念阿弥陀佛名号
3. **修诸功德**：行善积德，回向净土
4. **发愿往生**：至诚发愿，愿生彼国

## 二、佛说观无量寿经

### 基本信息

| 项目 | 内容 |
|------|------|
| **译者** | 畺良耶舍（Kālayāśas） |
| **年代** | 南朝宋元嘉年间（424-442年） |
| **卷数** | 一卷 |
| **全名** | 《佛说观无量寿佛经》 |
| **别称** | 观经、十六观经 |

### 核心内容

《观无量寿经》主要讲述佛陀应韦提希夫人之请，开示往生西方极乐世界的方法。经中详细说明了「十六观法」，即十六种观想极乐世界的修行方法，以及「九品往生」的阶次。

本经的缘起：韦提希夫人因儿子阿阇世王囚禁其父频婆娑罗王，并欲杀害母亲，心生忧恼。佛陀从耆阇崛山现身王宫，为夫人开示往生净土的法门。

### 十六观法

十六观法是观想极乐世界的修行次第，分为两类：

#### （一）依报观（1-13观）

观想极乐世界的庄严环境：

| 观号 | 观名 | 观想内容 |
|------|------|----------|
| 第一 | 日想观 | 观想西方落日，令心坚住，专想不移 |
| 第二 | 水想观 | 观想清水澄净，转想成冰，再转想成琉璃地 |
| 第三 | 地想观 | 观想琉璃大地，黄金为绳，界道分明 |
| 第四 | 宝树观 | 观想极乐国土七宝行树，行行相值，茎茎相望 |
| 第五 | 宝池观 | 观想八功德水，七宝池中莲花弥覆其上 |
| 第六 | 宝楼观 | 观想众宝国土，一一界上有五百亿宝楼 |
| 第七 | 华座观 | 观想阿弥陀佛所坐莲花宝座 |
| 第八 | 像想观 | 先观想阿弥陀佛、观世音菩萨、大势至菩萨像 |
| 第九 | 真身观 | 观想阿弥陀佛真实身相，高六十万亿那由他恒河沙由旬 |
| 第十 | 观音观 | 观想观世音菩萨身长八十万亿那由他由旬 |
| 第十一 | 势至观 | 观想大势至菩萨身量大小等与观音相同 |
| 第十二 | 普想观 | 观想自己生于西方极乐世界，于莲花中结跏趺坐 |
| 第十三 | 杂想观 | 观想阿弥陀佛现大身满虚空，或现小身丈六八尺 |

#### （二）正报观（14-16观）

观想往生净土的众生：

| 观号 | 观名 | 内容 |
|------|------|------|
| 第十四 | 上辈观 | 观想上品上生、上品中生、上品下生 |
| 第十五 | 中辈观 | 观想中品上生、中品中生、中品下生 |
| 第十六 | 下辈观 | 观想下品上生、下品中生、下品下生 |

### 九品往生

九品往生是根据众生临终时的功德和念佛功夫，将往生者分为九等：

#### 上品上生
- **条件**：受持读诵大乘方等经典，解第一义，深信因果，愿生彼国
- **临终瑞相**：阿弥陀佛与观音、势至及无数化佛、百千比丘来迎
- **往生去处**：生彼国已，见佛色身众相具足，即悟无生法忍

#### 上品中生
- **条件**：不必受持读诵方等经典，但善解义趣，深信因果，信乐大乘
- **临终瑞相**：阿弥陀佛与观音、势至、无量大众持金莲台来迎
- **往生去处**：生经七日，遇观音、势至，闻法欢喜，得阿鞞跋致

#### 上品下生
- **条件**：信乐大乘，不谤正法，但发无上道心
- **临终瑞相**：阿弥陀佛与观音、势至及诸眷属持金莲华来迎
- **往生去处**：生经一日一夜，莲华乃开，七日乃见佛

#### 中品上生
- **条件**：受持五戒，持八戒斋，修行诸戒，不造五逆，无众过患
- **临终瑞相**：阿弥陀佛与诸比丘眷属放金色光，持七宝莲华来迎
- **往生去处**：生经七日，遇观音、势至，闻法欢喜，得须陀洹

#### 中品中生
- **条件**：一日一夜持八戒斋，或一日一夜持沙弥戒，或一日一夜持具足戒
- **临终瑞相**：阿弥陀佛与诸眷属放金色光，持七宝莲华来迎
- **往生去处**：生经七日，莲华乃开，闻观音、势至说法，得须陀洹

#### 中品下生
- **条件**：孝养父母，行世仁慈，临终遇善知识，为说极乐国土事
- **临终瑞相**：命终时，遇善知识为说阿弥陀佛乐事，即生彼国
- **往生去处**：生经七日，遇观音、势至，闻法欢喜，得须陀洹

#### 下品上生
- **条件**：作众恶业，虽不诽谤方等经典，但多造恶法，无有惭愧
- **临终瑞相**：临终遇善知识，为说大乘十二部经首题名字，教令称念南无阿弥陀佛
- **往生去处**：生经七七日，莲华乃开，见观音、势至，闻法欢喜，发菩提心得无生忍

#### 下品中生
- **条件**：毁犯五戒、八戒及具足戒，偷僧祇物，不净说法，无有惭愧
- **临终瑞相**：临终遇善知识，为说阿弥陀佛十力威德，称念南无阿弥陀佛
- **往生去处**：生经六劫，莲华乃开，见观音、势至，闻法欢喜，发菩提心得无生忍

#### 下品下生
- **条件**：作不善业，五逆十恶，具诸不善，应堕恶道
- **临终瑞相**：临终遇善知识，为说妙法，教令念佛，称南无阿弥陀佛
- **往生去处**：生满十二大劫，莲华方开，见观音、势至，闻说法已，欢喜信乐，发菩提心得无生忍

### 重要原文摘录

**韦提希夫人发愿：**

> 「世尊！是诸佛土，虽复清净，皆有光明，我今乐生极乐世界阿弥陀佛所。」

**日想观：**

> 「正坐西向，谛观于日欲没之处，令心坚住，专想不移。见日欲没，状如悬鼓。」

**真身观：**

> 「阿弥陀佛身量无边，非是凡夫心力所及。彼佛圆光，如百亿三千大千世界。」

**下品下生：**

> 「若念「佛」者，当知此人，则是人中芬陀利华。观世音菩萨、大势至菩萨，为其胜友，当坐道场，生诸佛家。」

### 净土宗修行指导

《观经》的修行要点：

1. **观想念佛**：依十六观法，观想极乐世界依正庄严
2. **称名念佛**：下至一声南无阿弥陀佛，皆可往生
3. **回向往生**：将一切功德回向净土
4. **临终正念**：保持正念，迎接佛来接引

## 三、佛说阿弥陀经

### 基本信息

| 项目 | 内容 |
|------|------|
| **译者** | 鸠摩罗什（Kumārajīva） |
| **年代** | 后秦弘始四年（402年） |
| **卷数** | 一卷 |
| **全名** | 《佛说阿弥陀经》 |
| **别称** | 小阿弥陀经、小经、一经 |

### 核心内容

《阿弥陀经》是净土三经中最简短、流传最广的一部。经中佛陀无问自说，直接向舍利弗等众弟子宣说西方极乐世界的依正庄严，以及持名念佛往生净土的方法。

本经的特点是：
- **无问自说**：不待人问，佛自宣说
- **简明扼要**：篇幅短小，义理明了
- **持名为主**：强调称念阿弥陀佛名号
- **六方诸佛证明**：六方诸佛出广长舌相，赞叹此经

### 重要原文摘录

**极乐世界距离：**

> 「从是西方，过十万亿佛土，有世界名曰极乐，其土有佛，号阿弥陀。」

**七宝庄严：**

> 「彼佛国土，微风吹动诸宝行树及宝罗网，出微妙音，譬如百千种乐，同时俱作。闻是音者，自然皆生念佛、念法、念僧之心。」

**持名念佛：**

> 「若有善男子善女人，闻说阿弥陀佛，执持名号，若一日、若二日、若三日、若四日、若五日、若六日、若七日，一心不乱，其人临命终时，阿弥陀佛与诸圣众，现在其前。是人终时，心不颠倒，即得往生阿弥陀佛极乐国土。」

**不可以少善根福德因缘：**

> 「舍利弗，不可以少善根福德因缘，得生彼国。舍利弗，若有善男子善女人，闻说阿弥陀佛，执持名号……」

**六方诸佛赞叹：**

> 「如我今者，赞叹阿弥陀佛不可思议功德之利。东方亦有阿閦鞞佛、须弥相佛、大须弥佛……各于其国，出广长舌相，遍覆三千大千世界，说诚实言：『汝等众生，当信是称赞不可思议功德，一切诸佛所护念经。』」

### 净土宗修行指导

《阿弥陀经》的修行要点：

1. **信愿行具足**：深信、切愿、实行缺一不可
2. **执持名号**：一心不乱，专念阿弥陀佛
3. **一心不乱**：从事一心到理一心，层次递进
4. **回向功德**：将功德回向一切众生，同生净土

## 四、普贤行愿品

### 基本信息

| 项目 | 内容 |
|------|------|
| **出处** | 《华严经》第四十卷（别行本） |
| **译者** | 般若（Prajñā） |
| **年代** | 唐贞元十一年（795年） |
| **卷数** | 一卷 |
| **全名** | 《大方广佛华严经入不思议解脱境界普贤行愿品》 |
| **别称** | 普贤行愿品、四十华严 |

### 核心内容

《普贤行愿品》原本是《华严经》的最后一品，由唐代般若三藏重译。此品以普贤菩萨的十大愿王为核心，阐述菩萨修行的圆满法门，最终导归西方极乐世界。

本经被净土宗列为五经之一，是因为普贤菩萨以十大愿王导归极乐，显示华严圆教与净土法门的圆融无碍。

### 普贤十大愿王

| 愿号 | 愿名 | 内容 |
|------|------|------|
| 第一 | 礼敬诸佛 | 以清净身语意，遍礼尽法界、虚空界一切诸佛 |
| 第二 | 称赞如来 | 以甚深胜解，现前知见一切如来，出微妙音称赞功德 |
| 第三 | 广修供养 | 以种种上妙供具，供养尽法界、虚空界一切如来 |
| 第四 | 忏悔业障 | 于一切诸佛菩萨众前，诚心忏悔无始以来所造业障 |
| 第五 | 随喜功德 | 随喜尽法界、虚空界一切菩萨及众生所有功德 |
| 第六 | 请转法轮 | 请诸如来转妙法轮，宣说微妙法音 |
| 第七 | 请佛住世 | 请诸如来久住于世，度脱众生 |
| 第八 | 常随佛学 | 常随一切如来修学，成就圆满功德 |
| 第九 | 恒顺众生 | 随顺众生种种根性，利益安乐一切众生 |
| 第十 | 普皆回向 | 将所有功德回向一切众生，同生极乐净土 |

### 导归极乐

**重要原文摘录：**

> 「愿我临欲命终时，尽除一切诸障碍，
> 面见彼佛阿弥陀，即得往生安乐刹。
> 我既往生彼国已，现前成就此大愿，
> 一切圆满尽无余，利乐一切众生界。」

> 「我此普贤殊胜行，无边胜福皆回向，
> 普愿沉溺诸众生，速往无量光佛刹。」

这显示普贤菩萨的十大愿王最终都导归西方极乐世界，由阿弥陀佛的愿力加持，速证无上菩提。

### 净土宗修行指导

《普贤行愿品》的修行要点：

1. **修十大愿**：以普贤十大愿为修行准则
2. **导归极乐**：将一切功德回向净土
3. **华严圆教**：与净土法门圆融无碍
4. **速证菩提**：往生净土后速证无上道

## 五、楞严经大势至菩萨念佛圆通章

### 基本信息

| 项目 | 内容 |
|------|------|
| **出处** | 《大佛顶首楞严经》第五卷 |
| **译者** | 般剌蜜帝（Prāmukhā） |
| **年代** | 唐神龙元年（705年） |
| **卷数** | 一卷（别行本） |

### 核心内容

《念佛圆通章》是《楞严经》中二十五圆通之一，由大势至菩萨宣说。此章以「念佛三昧」为圆通法门，阐述持名念佛的方法，被净土宗视为念佛修行的重要理论依据。

本章的核心思想是「都摄六根，净念相继」，即收摄眼、耳、鼻、舌、身、意六根，令念佛之心念念相续，最终证得念佛三昧。

### 大势至菩萨念佛圆通

**忆佛念佛，是心作佛：**

> 「我忆往昔恒河沙劫，有佛出世，名无量光。十二如来，相继一劫。其最后佛，名超日月光。彼佛教我，念佛三昧。譬如有人，一专为忆，一人专忘，如是二人，若逢不逢，或见非见。二人相忆，二忆念深，如是乃至从生至生，同于形影，不相乖异。十方如来，怜念众生，如母忆子。若子逃逝，虽忆何为？子若忆母，如母忆时，母子历生，不相违远。若众生心，忆佛念佛，现前当来，必定见佛，去佛不远，不假方便，自得心开。」

这段经文阐明：
1. 佛常忆念众生，如母忆子
2. 众生若能忆佛念佛，必能见佛
3. 忆念深切，从生至生，不相远离

### 都摄六根，净念相继

**重要原文摘录：**

> 「如染香人，身有香气，此则名曰香光庄严。我本因地，以念佛心，入无生忍。今于此界，摄念佛人，归于净土。佛问圆通，我无选择，都摄六根，净念相继，得三摩地，斯为第一。」

**「都摄六根，净念相继」的修行方法：**

- **都摄六根**：收摄眼、耳、鼻、舌、身、意，不令外缘
- **净念相继**：清净念佛之心，念念相续，无有间断
- **得三摩地**：成就念佛三昧，即心即佛

### 净土宗修行指导

《念佛圆通章》的修行要点：

1. **忆佛念佛**：心心念念，忆念阿弥陀佛
2. **都摄六根**：收摄六根，不攀外缘
3. **净念相继**：清净念佛，念念相续
4. **不假方便**：专一念佛，无需其他法门
5. **自得心开**：自然开悟，证无生忍

## 六、净土宗祖师论述

### （一）善导大师（613-681）

善导大师是净土宗的实际创始人，被尊为净土宗二祖（或高祖）。他的著作《观经疏》被誉为「楷定疏」，确立了净土宗的基本教义。

**主要著作：**
- 《观无量寿佛经疏》（观经疏/四帖疏）
- 《往生礼赞偈》
- 《观念法门》
- 《法事赞》
- 《般舟赞》

**核心思想：**

1. **正由称名**

> 「望佛愿意者，唯劝正念称名，往生义疾，不同余行之杂疏难成。……正由称名易故，相续即生。」

2. **六字释**

善导大师对「南无阿弥陀佛」六字的开示：

> 「言『南无』者，即是归命，亦是发愿回向之义；言『阿弥陀佛』者，即是其行：以斯义故，必得往生。」

3. **正定业**

> 「一心专念，弥陀名号，行住坐卧，不问时节久近，念念不舍者，是名正定之业，顺彼佛愿故。」

4. **二种深信**

> 「一者决定深信：自身现是罪恶生死凡夫，旷劫以来常没常流转，无有出离之缘；二者决定深信：彼阿弥陀佛四十八愿，摄受众生，无疑无虑，乘彼愿力，定得往生。」

### （二）昙鸾大师（476-542）

昙鸾大师是南北朝时期净土宗的重要祖师，被净土宗尊为初祖。他著有《往生论注》，对世亲菩萨的《往生论》进行详细注释。

**主要著作：**
- 《往生论注》
- 《赞阿弥陀佛偈》
- 《略论安乐净土义》

**核心思想：**

1. **他力本愿**

昙鸾大师强调净土法门是他力法门，依靠阿弥陀佛的大愿力往生：

> 「难易之道，天渊不同。难易者，总有二言：一者难行道，二者易行道。……易行道者，谓但以信佛因缘，愿生净土，乘佛愿力，便得往生彼清净土。」

2. **五念门**

根据《往生论》，昙鸾大师阐述五念门修行：
- **礼拜门**：身业礼拜阿弥陀佛
- **赞叹门**：口业赞叹阿弥陀佛
- **作愿门**：意业专念，发愿往生
- **观察门**：智慧观察净土庄严
- **回向门**：将所有功德回向众生

3. **速满成佛**

> 「无量寿经三辈往生中，虽行有优劣，莫不皆发无上菩提心。」

### （三）道绰禅师（562-645）

道绰禅师是唐代净土宗的重要祖师，继承昙鸾大师的净土思想，大力提倡持名念佛，对净土宗的发展有重大影响。

**主要著作：**
- 《安乐集》

**核心思想：**

1. **圣净二门**

道绰禅师在《安乐集》中提出「圣道门」与「净土门」的分别：

> 「若一念称阿弥陀佛，即能灭除八十亿劫生死之罪。」

> 「在此起心立行，愿生净土，皆是自力；临命终时，阿弥陀佛与诸圣众，亲来迎接，即是佛力。」

2. **念佛计数**

道绰禅师以豆计数念佛，日课七万声，为后世佛数珠的起源。他教导信众：

> 「初学念佛之人，当用数珠，记其念佛数目，令心有所系，不散不乱。」

3. **末法时代**

道绰禅师根据佛经，判定当时已进入末法时代，主张众生应修净土法门：

> 「当今末法，现是五浊恶世，唯有净土一门，可通入路。」

## 七、净土宗修行总结

### 信愿行三资粮

净土宗修行以「信、愿、行」为三资粮，缺一不可：

| 资粮 | 内容 | 说明 |
|------|------|------|
| **信** | 深信阿弥陀佛的大愿力 | 相信四十八愿真实不虚，十念必生 |
| **愿** | 切愿往生西方极乐世界 | 厌离娑婆，欣求极乐 |
| **行** | 持名念佛，一向专念 | 行住坐卧，念念不舍 |

### 修行方法

1. **持名念佛**：称念「南无阿弥陀佛」名号
2. **观想念佛**：依《观经》十六观法观想
3. **观像念佛**：观想阿弥陀佛圣像
4. **实相念佛**：观佛法身，悟入实相

### 回向文

> 「愿以此功德，平等施一切，
> 同发菩提心，往生安乐国。」
