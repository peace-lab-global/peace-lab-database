# 密宗重要经典（Vajrayana Sutras）

本文档介绍密宗（金刚乘）的重要经典，包括密续经典、仪轨经典、陀罗尼及藏传佛教核心经典。

---

## 一、密续经典（金刚界）

密续（Tantra）是密宗修行的根本依据，阐述本尊瑜伽、曼荼罗、灌顶等核心教法。

### 1. 大日经

| 项目 | 内容 |
|------|------|
| 中文全称 | 大毗卢遮那成佛神变加持经 |
| 梵文名称 | Mahavairocana Sutra / Mahavairocana Abhisambodhi Tantra |
| 译者 | 善无畏（Subhakarasiṃha）、金刚智（Vajrabodhi）、一行（译语） |
| 译出时间 | 唐开元年间（公元724年前后） |
| 卷数 | 七卷三十一品 |

**主要内容：**
- 阐述大日如来（毗卢遮那佛）的法身遍照法界
- 建立菩提心为因、大悲为根本、方便为究竟的修行纲领
- 详述胎藏界曼荼罗的建立与观想方法
- 介绍身、口、意三密相应的修行法门

**在密宗中的地位：**
大日经是唐密胎藏界的根本经典，与金刚顶经并称两部大经。它奠定了密宗即身成佛的理论基础，确立了胎藏界五佛（大日如来、阿閦佛、宝生佛、阿弥陀佛、不空成就佛）的体系。

---

### 2. 金刚顶经

| 项目 | 内容 |
|------|------|
| 中文全称 | 金刚顶一切如来真实摄大乘现证大教王经 |
| 梵文名称 | Sarvatathagata Tattvasamgraha Mahayanabhisamaya Mahakalaraja Tantra |
| 译者 | 不空（Amoghavajra） |
| 译出时间 | 唐天宝年间（公元753年） |
| 卷数 | 三卷（略出本），另有十八会广本未全译 |

**主要内容：**
- 建立金刚界曼荼罗的九会结构（成身会、三昧耶会、微细会等）
- 阐述五智（法界体性智、大圆镜智、平等性智、妙观察智、成所作智）
- 详述金刚萨埵等三十七尊的观想与手印
- 介绍金刚界灌顶仪轨与修持次第

**在密宗中的地位：**
金刚顶经是唐密金刚界的根本经典，代表密宗果上起修的教法。金刚界与胎藏界合称金胎不二，是唐密最高修行法门。此经也是日本真言宗（东密）的核心经典。

---

### 3. 苏悉地羯罗经

| 项目 | 内容 |
|------|------|
| 中文全称 | 苏悉地羯罗经 / 苏悉地羯啰经 |
| 梵文名称 | Susiddhikara Sutra |
| 译者 | 善无畏 |
| 译出时间 | 唐代 |
| 卷数 | 三卷 |

**主要内容：**
- 详细说明密宗供养、灌顶、护摩等仪轨的具体操作方法
- 介绍各种成就法（悉地）的修持要领
- 阐述护摩（火供）的分类与用途
- 说明曼荼罗布置、法器使用等实务

**在密宗中的地位：**
此经被称为密宗的修行手册，是大日经和金刚顶经的具体实践指南。它填补了理论经典与实修之间的空隙，对于密宗仪轨的规范化具有重要作用。

---

## 二、密宗仪轨经典

仪轨经典详细规定密宗修持的具体步骤、手印、真言及观想方法。

### 4. 陀罗尼集经

| 项目 | 内容 |
|------|------|
| 梵文名称 | Dharani Samgraha Sutra |
| 译者 | 阿地瞿多（Atikuta） |
| 译出时间 | 唐永徽年间（公元653-654年） |
| 卷数 | 十二卷 |

**主要内容：**
- 汇集各种重要陀罗尼（真言咒语的集合）
- 记载佛顶尊胜陀罗尼、一切如来心陀罗尼等
- 介绍陀罗尼的持诵方法、功德利益
- 详述坛城建立、供养仪轨等

**在密宗中的地位：**
陀罗尼集经是陀罗尼密教的重要经典，也是唐密形成期的重要文献。它标志着陀罗尼咒语从简到繁、从散到集的发展过程，为后世密宗的建立奠定了基础。

---

### 5. 妙臂菩萨所问经

| 项目 | 内容 |
|------|------|
| 梵文名称 | Subahu Pariprccha Tantra / Subahu-pariprccha-nama-tantra |
| 译者 | 菩提流志（Bodhiruci） |
| 译出时间 | 唐代（公元693年） |
| 卷数 | 四卷 |

**主要内容：**
- 妙臂菩萨向佛陀请问密乘修行的种种问题
- 详细说明护摩法的各种分类（息灾、增益、敬爱、降伏）
- 介绍各种手印的制作与使用
- 阐述选择修行地点、时间、方位等要诀

**在密宗中的地位：**
此经是护摩法的专门经典，对密宗火供仪轨的完备起到重要作用。其问答形式使得复杂的仪轨内容更加清晰易懂，是实修者的重要参考。

---

### 6. 金刚寿命陀罗尼经

| 项目 | 内容 |
|------|------|
| 梵文名称 | Vajrayusa Dharani Sutra |
| 译者 | 不空 |
| 译出时间 | 唐代 |
| 卷数 | 一卷 |

**主要内容：**
- 宣说金刚寿命陀罗尼及其功德
- 介绍延寿、除障、增福的修持方法
- 阐述陀罗尼与护摩结合的应用
- 说明供养、持诵的仪轨

**在密宗中的地位：**
此经属于息灾法的经典，在密宗延寿祈福法门中具有重要地位。金刚寿命陀罗尼被广泛运用于祈福延寿的法会中。

---

## 三、重要陀罗尼

陀罗尼（Dharani）意为总持，能持善遮恶，是密宗修行的核心法门。

### 7. 佛顶尊胜陀罗尼

| 项目 | 内容 |
|------|------|
| 梵文名称 | Usnisa Vijaya Dharani |
| 全称 | 最胜佛顶陀罗尼 / 佛顶尊胜陀罗尼 |
| 译者 | 佛陀波利、义净等多人译 |
| 出处 | 佛顶尊胜陀罗尼经（佛陀波利译本） |

**主要内容：**
- 陀罗尼具有消除业障、增福延寿的殊胜功德
- 可消除地狱、饿鬼、畜生等恶道之苦
- 能破一切恶道，回向善道
- 命终可往生西方极乐世界或善道

**在密宗中的地位：**
佛顶尊胜陀罗尼是汉传佛教最流行的陀罗尼之一，是密宗息灾法的核心法门。此陀罗尼在唐代极为盛行，成为祈福、超度、延寿的主要修持法门。

---

### 8. 宝箧印陀罗尼

| 项目 | 内容 |
|------|------|
| 梵文名称 | Sarvadurgati Parisodhana Usnisa Dharani |
| 全称 | 一切如来心秘密全身舍利宝箧印陀罗尼 |
| 译者 | 不空 |
| 出处 | 一切如来心秘密全身舍利宝箧印陀罗尼经 |

**主要内容：**
- 陀罗尼出自宝箧印塔中的法舍利
- 能消除一切恶业、破地狱之苦
- 具有不可思议的加持力
- 与宝箧印塔的供养结合

**在密宗中的地位：**
宝箧印陀罗尼与宝箧印塔密切相关，是密宗塔供养法的核心。此陀罗尼在东亚地区广为流传，宝箧印塔成为重要的佛教建筑形式。

---

### 9. 一切如来心秘密全身舍利宝箧印陀罗尼经

| 项目 | 内容 |
|------|------|
| 梵文名称 | Sarvatathagata Hrdaya Guhya Dhatu Garbha Usnisa Dharani Sutra |
| 译者 | 不空 |
| 译出时间 | 唐代 |
| 卷数 | 一卷 |

**主要内容：**
- 佛陀讲述宝箧印陀罗尼的起源与功德
- 详述宝箧印塔的建立方法与供养仪轨
- 说明陀罗尼的持诵方法与功德利益
- 介绍礼拜、供养、绕塔等修行方法

**在密宗中的地位：**
此经是宝箧印陀罗尼的根本经典，详细说明了宝箧印塔供养的完整仪轨。它对于理解密宗塔供养法具有核心意义，也是研究唐代密宗建筑的重要依据。

---

### 10. 毗尼日用切要

| 项目 | 内容 |
|------|------|
| 编撰者 | 读体律师（见月老人） |
| 成书时间 | 明末清初 |
| 性质 | 戒律与陀罗尼合编的日常修持手册 |

**主要内容：**
- 收录出家人日常所用的各种陀罗尼
- 包括华严经净行品、佛顶尊胜陀罗尼、楞严咒心等
- 说明穿衣、吃饭、走路、睡觉等各场合的持诵仪轨
- 融戒律规范与密咒修持为一体

**在密宗中的地位：**
毗尼日用切要是汉传佛教寺院日常课诵的重要参考，体现了戒律与密咒结合的修行特色。它将陀罗尼修持融入日常生活，使密法成为日常修行的一部分。

## 四、藏传佛教经典

藏传佛教（藏密）拥有独特的经典体系，以藏译梵本为主。

### 11. 时轮金刚根本续

| 项目 | 内容 |
|------|------|
| 藏文名称 | Dus kyi 'khor lo'i rgyud / Kalachakra Tantra |
| 梵文名称 | Kalachakra Tantra |
| 译者 | 月贤王（传为佛陀亲授），后由印度传入西藏 |
| 传承 | 时轮金刚法系 |

**主要内容：**
- 阐述时轮金刚（Kaalachakra）的生起次第与圆满次第
- 介绍外在的宇宙时轮、内在的身心时轮、本尊瑜伽时轮
- 详述六支瑜伽（摄心、专注、呼吸、持风、忆念、三摩地）
- 包含独特的历法、占星、医学等内容

**在密宗中的地位：**
时轮金刚根本续是无上瑜伽部母续的重要经典，在藏传佛教格鲁派、萨迦派、宁玛派中均有传承。它被视为佛陀最后的教法，具有极强的完整性和系统性。

---

### 12. 密集金刚根本续

| 项目 | 内容 |
|------|------|
| 藏文名称 | Gsang ba 'dus pa'i rgyud / Guhyasamaja Tantra |
| 梵文名称 | Guhyasamaja Tantra |
| 译者 | 吉祥狮子等传入西藏 |
| 传承 | 密集金刚法系 |

**主要内容：**
- 阐述密集金刚五方佛聚会的坛城
- 介绍生起次第的五相成身观、圆满次第的光明观
- 详述五次第（隔离轮、秘密轮、智慧轮、喜乐轮、不坏轮）
- 建立父续灌顶与修持的完整体系

**在密宗中的地位：**
密集金刚根本续是无上瑜伽部父续的根本经典，对藏传佛教父续法系的形成具有决定性影响。它是龙树菩萨所传密集法门的根本依据，也是格鲁派五大本尊之一密集金刚的核心经典。

---

### 13. 胜乐金刚根本续

| 项目 | 内容 |
|------|------|
| 藏文名称 | Bde mchog rgyud / Chakrasamvara Tantra |
| 梵文名称 | Chakrasamvara Tantra / Shriheruka Abhidhana Tantra |
| 译者 | 由印度大成就者传入西藏 |
| 传承 | 胜乐金刚法系 |

**主要内容：**
- 阐述胜乐金刚（Heruka）与金刚亥母的瑜伽
- 介绍六十二尊坛城的观想与修持
- 详述拙火、明点、气脉等圆满次第修法
- 建立母续灌顶与修持体系

**在密宗中的地位：**
胜乐金刚根本续是无上瑜伽部母续的重要经典，是藏传佛教各派共修的大本尊之一。它对大手印、那洛六法等修持体系有深远影响，也是噶举派的核心本尊法门。

---

### 14. 大幻化网（玛哈玛雅）

| 项目 | 内容 |
|------|------|
| 藏文名称 | Sgyu 'phrul chen mo'i rgyud / Mayajala Tantra |
| 梵文名称 | Mahamaya Tantra / Guhyagarbha Tantra（相关） |
| 译者 | 由印度传入西藏 |
| 传承 | 宁玛派大圆满法系 |

**主要内容：**
- 阐述大圆满（Dzogchen）的见、修、行、果
- 介绍普贤王如来、金刚萨埵等本尊坛城
- 详述直断（Trekcho）与顿超（Togal）的修行方法
- 说明一切众生本具佛性的究竟见地

**在密宗中的地位：**
大幻化网是宁玛派（红教）的根本经典之一，属于阿底瑜伽（无上瑜伽）的心髓教法。它代表了大圆满见地，对藏传佛教宁玛派的独特传承具有核心意义。

---

### 15. 入菩萨行论

| 项目 | 内容 |
|------|------|
| 梵文名称 | Bodhicaryavatara |
| 藏文名称 | Byang chub sems dpa'i spyod pa la 'jug pa |
| 作者 | 寂天菩萨（Shantideva / Zhi ba lha） |
| 译出时间 | 公元8世纪传入西藏 |

**主要内容：**
- 第一品：菩提心的功德与利益
- 第二品：忏悔罪业
- 第三品：受持菩提心
- 第四品：不放逸（守护菩提心）
- 第五品：护正知（守护正念正知）
- 第六品：安忍（忍辱波罗蜜）
- 第七品：精进（精进波罗蜜）
- 第八品：静虑（禅定波罗蜜）
- 第九品：般若（智慧波罗蜜）
- 第十品：回向

**在密宗中的地位：**
入菩萨行论是大乘佛教修行菩提心的根本论典，被藏传佛教各派共尊为核心教材。它虽然不是密续，但被视为进入密乘修行的基础。宗喀巴大师、华智仁波切等大成就者都曾为此论作注，是显密圆融的典范著作。

## 五、经典体系总览

### 汉传唐密经典体系

```
根本经典
├── 大日经（胎藏界）
├── 金刚顶经（金刚界）
└── 苏悉地羯罗经（仪轨指南）

仪轨经典
├── 陀罗尼集经
├── 妙臂菩萨所问经
└── 金刚寿命陀罗尼经

陀罗尼法门
├── 佛顶尊胜陀罗尼
├── 宝箧印陀罗尼
└── 毗尼日用切要（日常修持）
```

### 藏传佛教经典体系

```
无上瑜伽部
├── 父续
│   └── 密集金刚根本续
├── 母续
│   ├── 时轮金刚根本续
│   └── 胜乐金刚根本续
└── 心髓教法
    └── 大幻化网（大圆满）

共修基础
└── 入菩萨行论（菩提心修习）
```

## 六、结语

密宗经典体系博大精深，涵盖显密圆融的完整修行路径。无论是汉传的唐密经典，还是藏传的密续体系，都指向同一个核心：通过身口意三密的修持，即身证悟佛的果位。

修行者在学习这些经典时，应当：
1. **依止具格上师**：密法修持必须有上师口传心授
2. **次第修学**：先学显教基础，再入密乘深法
3. **发菩提心**：一切密法修持以利益众生为根本
4. **严守戒律**：密乘戒律是成就的根本保障

愿一切众生皆得妙法，速证菩提。
