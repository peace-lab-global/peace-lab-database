# 佛教论藏重要论典

佛教论藏（Abhidharma Pitaka）是三藏之一，主要阐述佛法义理，包括各部派的阿毗达磨论书以及大乘佛教的各类论典。本文档系统介绍中观派、瑜伽行派及阿毗达磨各部的重要论典。

---

## 一、中观派论典（Mādhyamaka）

中观派由龙树（Nāgārjuna）创立，以《中论》为核心，主张一切法缘起性空，为印度大乘佛教两大主流之一。

### 1. 大智度论（Mahāprajñāpāramitāśāstra）

| 项目 | 内容 |
|------|------|
| **作者** | 龙树（Nāgārjuna）菩萨造 |
| **译者** | 鸠摩罗什（Kumārajīva）于后秦弘始四年至七年（402-405年）译出 |
| **卷数** | 100卷 |
| **核心内容** | 本论为《摩诃般若波罗蜜经》的释论，详释般若波罗蜜多的深义，广明菩萨道修行阶位、六波罗蜜、诸法实相、十八空等义理。内容博大精深，涵盖显密教法，被誉为"论中之王"。 |
| **主要教义** | 1. 详细阐释十八空：从内空到一切法空<br>2. 系统建立菩萨十地修行体系<br>3. 广说六波罗蜜的实践方法<br>4. 融通大小乘，会通各派学说<br>5. 对佛身、净土、神通等亦有详论 |
| **历史地位** | 《大智度论》是汉传佛教最重要的论典之一，对天台宗、三论宗的形成影响深远。鸠摩罗什的翻译使中观学说得以系统传入中国，成为此后中国佛教义学发展的理论基础。本论也是研究印度大乘佛教思想发展史的重要文献。 |

---

### 2. 中论（Mūlamadhyamakakārikā）

| 项目 | 内容 |
|------|------|
| **作者** | 龙树（Nāgārjuna，约150-250年）菩萨造 |
| **译者** | 鸠摩罗什于后秦弘始十一年（409年）译出 |
| **卷数** | 4卷（27品，446颂） |
| **核心内容** | 中观学派的根本论书，以缘起性空为核心，通过归谬论证（ reductio ad absurdum ）破斥当时印度各派（包括佛教内部各部派及外道）的实体见，包括自性、他性、共性、无因等诸种执著，确立一切法无自性的中道正见。 |
| **主要教义** | 1. **八不中道**：不生亦不灭，不常亦不断，不一亦不异，不来亦不出<br>2. **缘起性空**：缘起故空，空故缘起，二者不一不二<br>3. **二谛说**：世俗谛与胜义谛的辩证关系<br>4. **无自性**：人无我、法无我的究竟义<br>5. **涅槃即世间**：生死与涅槃无差别 |
| **历史地位** | 《中论》是中观学派的奠基之作，被誉为"佛法的心脏"。其空性思想不仅影响印度佛教发展，更通过汉传、藏传两条路线深刻影响了整个东亚佛教。青目释、佛护释、月称释等各派注疏丰富，形成中观自续派与应成派的分野。现代佛学研究中，本论亦是理解佛教哲学的必读经典。 |

---

### 3. 十二门论（Dvādaśanikāyaśāstra / Dvādaśamukhaśāstra）

| 项目 | 内容 |
|------|------|
| **作者** | 龙树（Nāgārjuna）菩萨造 |
| **译者** | 鸠摩罗什于后秦弘始十一年（409年）译出 |
| **卷数** | 1卷 |
| **核心内容** | 本论为《中论》的入门书，以十二门（十二种观察角度）阐释缘起性空的道理。十二门包括：观因缘门、观有果无果门、观缘门、观相门、观有相无相门、观一异门、观有无门、观性门、观因果门、观作者门、观三时门、观生门。 |
| **主要教义** | 1. 从十二种角度观察诸法，层层深入，破除各种实体见<br>2. 以简洁方式阐明缘起无自性的核心义理<br>3. 每一门都包含立论、破论、结论三段式论证<br>4. 特别重视对因果关系的分析<br>5. 为初学者提供系统学习空观的方法 |
| **历史地位** | 《十二门论》与《中论》《百论》合称"三论"，为三论宗所依的根本论典。本论篇幅精要，义理明晰，是理解《中论》的重要阶梯。在中国佛教史上，三论宗的吉藏大师对此论有详细注疏，对后世影响深远。 |

---

### 4. 百论（Śataśāstra）

| 项目 | 内容 |
|------|------|
| **作者** | 提婆（Āryadeva，约170-270年）菩萨造。提婆为龙树菩萨的嫡传弟子，被誉为"千部论主"。 |
| **译者** | 鸠摩罗什于后秦弘始十四年（412年）译出（原本有二十品，什师删略为十品） |
| **卷数** | 2卷（十品，每品十颂，共百颂） |
| **核心内容** | 本论以破邪显正为宗旨，前九品破斥外道（数论、胜论等）及佛教内部执著，第十品《破空品》则防止对空性的执著。全论以辩论方式展开，针对当时印度各派的实体见进行系统批判。 |
| **主要教义** | 1. **破神我**：破斥外道对永恒自我的执著<br>2. **破一异**：破斥事物一性、异性等实体见<br>3. **破情尘**：破斥认识主体与客体的实有<br>4. **破因中无果论**：破斥胜论派的因果观<br>5. **破空执著**：最后品防止堕入恶取空 |
| **历史地位** | 《百论》继承并发展了龙树的空性思想，特别擅长论辩技巧。与《中论》《十二门论》并称"三论"，对三论宗的形成有重要影响。提婆本人也是印度佛教史上著名的辩论大师，其思想通过论著和弟子传承，使中观学派在印度佛教界取得主导地位。 |

---

### 5. 十住毗婆沙论（Daśabhūmika-vibhāṣā-śāstra）

| 项目 | 内容 |
|------|------|
| **作者** | 龙树（Nāgārjuna）菩萨造 |
| **译者** | 鸠摩罗什于后秦年间译出 |
| **卷数** | 17卷（现存） |
| **核心内容** | 本论为《华严经·十地品》的注释书，详释菩萨十地修行的阶位、内容与方法。特别对初地（欢喜地）的修行有详细说明，包含著名的《易行品》，介绍阿弥陀佛净土念佛法门，是印度论师中最早系统论述净土思想的重要文献。 |
| **主要教义** | 1. **十地修行体系**：详释从欢喜地到法云地的修行内容<br>2. **难行与易行**：对比菩萨修行的两种途径<br>3. **净土思想**：《易行品》阐述阿弥陀佛本愿及念佛法门<br>4. **檀波罗蜜**：特别强调布施波罗蜜的实践<br>5. **菩萨愿行**：建立以利他为核心的大乘修行观 |
| **历史地位** | 本论是研究龙树菩萨思想的重要文献，也是理解《华严经》十地学说的关键著作。《易行品》中的净土思想对后世中国净土宗的形成影响深远，昙鸾、道绰、善导等净土宗祖师均重视此论。本论也是研究印度大乘佛教修行体系的重要资料。 |

---

## 二、瑜伽行派论典（Yogācāra）

瑜伽行派由弥勒（Maitreya）传出，无著（Asaṅga）、世亲（Vasubandhu）弘扬，主张万法唯识，与中观派并称印度大乘佛教两大主流。

### 1. 瑜伽师地论（Yogācārabhūmi-śāstra）

| 项目 | 内容 |
|------|------|
| **传出者** | 弥勒（Maitreya）菩萨说，无著（Asaṅga，约310-390年）菩萨记 |
| **译者** | 玄奘法师于唐贞观二十年至二十二年（646-648年）译出 |
| **卷数** | 100卷 |
| **核心内容** | 瑜伽行派的根本论书，系统阐述瑜伽修行的阶位（地）与内容。全书分为五分：本地分（十七地）、摄决择分、摄释分、摄异门分、摄事分。内容涵盖三乘（声闻、独觉、菩萨）修行的全部理论与实践，是大乘佛教最系统的修行论典。 |
| **主要教义** | 1. **十七地**：五识身相应地、意地、有寻有伺地等十七种修行阶位<br>2. **三性说**：遍计所执性、依他起性、圆成实性的详细阐释<br>3. **唯识说**：八识（前五识、第六意识、第七末那识、第八阿赖耶识）的建立<br>4. **菩萨道**：六度四摄、三十七道品等修行方法<br>5. **止观修习**：四禅八定、四念处等具体修法 |
| **历史地位** | 《瑜伽师地论》是汉传佛教最重要的论典之一，法相唯识宗的根本所依。玄奘法师西行求法的主要目的就是学习本论。本论体系完备、组织严密，对理解印度大乘佛教的思想全貌具有不可替代的价值。也是研究印度瑜伽修行传统的重要文献。 |

---

### 2. 唯识三十颂（Triṃśikā-vijñaptimātratā-siddhiḥ）

| 项目 | 内容 |
|------|------|
| **作者** | 世亲（Vasubandhu，约320-400年）菩萨造。世亲为无著之弟，原属说一切有部，后转学大乘，造论千部，被誉为"千部论主"。 |
| **译者** | 玄奘法师译（另有真谛、菩提流支等不同译本） |
| **卷数** | 1卷（三十颂） |
| **核心内容** | 唯识学的纲要性著作，以三十首偈颂简明阐述唯识学的核心教义。内容涵盖：异熟能变（第八识）、思量能变（第七识）、了别能变（前六识）、一切唯识、三性、转依果位等，是理解唯识学的入门与总结性著作。 |
| **主要教义** | 1. **三能变识**：异熟能变（阿赖耶识）、思量能变（末那识）、了别能变（前六识）<br>2. **四分说**：相分、见分、自证分、证自证分的认识结构<br>3. **三性**：遍计所执性、依他起性、圆成实性<br>4. **转依**：转识成智，证得菩提与涅槃<br>5. **五位百法**：心法、心所法、色法、心不相应行法、无为法的分类 |
| **历史地位** | 《唯识三十颂》是唯识学最精要的概论，后世注疏极多。印度有十大论师（护法、安慧、难陀等）分别作释，玄奘法师糅译十大论师释为《成唯识论》。本论对东亚佛教影响深远，是理解唯识学的必读经典，也是现代佛学研究中译本最多的唯识学著作之一。 |

---

### 3. 唯识二十论（Viṃśatikā-vijñaptimātratā-siddhiḥ）

| 项目 | 内容 |
|------|------|
| **作者** | 世亲（Vasubandhu）菩萨造 |
| **译者** | 玄奘法师译（另有真谛、瞿昙般若流支译本） |
| **卷数** | 1卷（二十颂及长行释） |
| **核心内容** | 本论以论辩方式论证唯识无境的道理，针对外道及小乘对唯识说的各种质疑（如：若无外境，如何解释世间现象？为何有共同所见？如何解释业力因果？等）一一作答，确立唯识学的理论基础。 |
| **主要教义** | 1. **破外境**：论证所缘境不离识，实无外境<br>2. **释疑难**：回答对唯识说的各种质疑<br>3. **共相识**：解释为何众生有共同所见<br>4. **梦喻**：以梦境比喻说明识变似境<br>5. **二取空**：能取所取皆空，唯识实性即真如 |
| **历史地位** | 《唯识二十论》与《唯识三十颂》是唯识学的双璧，前者偏重论辩破执，后者偏重建立体系。本论是理解唯识学理论基础的重要文献，也是研究印度佛教认识论与论理学的重要资料。玄奘法师的译本准确流畅，是研究本论的最佳版本。 |

---

### 4. 摄大乘论（Mahāyāna-saṃparigraha-śāstra）

| 项目 | 内容 |
|------|------|
| **作者** | 无著（Asaṅga）菩萨造。无著是瑜伽行派创始人，原属说一切有部，经弥勒菩萨点拨转入大乘，其著作奠定了瑜伽行派的理论基础。 |
| **译者** | 先后有三个译本：佛陀扇多译（1卷）、真谛译（3卷）、玄奘译（3卷） |
| **卷数** | 3卷（玄奘译本）/ 1-3卷（其他译本） |
| **核心内容** | 本论为《阿毗达磨大乘经·摄大乘品》的释论，系统阐述瑜伽行派的大乘思想。全论分为十相：依止胜相、应知胜相、应知入胜相、入所知胜相、彼入因果胜相、彼因果修差别胜相、此中增上戒学等三学胜相、彼果断胜相、彼果智胜相。 |
| **主要教义** | 1. **阿赖耶识**：详细建立第八识作为轮回与解脱的所依<br>2. **三性说**：遍计所执、依他起、圆成实的系统阐述<br>3. **唯识观**：从有分别影像观到无分别影像观的修行次第<br>4. **六度修行**：以般若波罗蜜为核心的菩萨道<br>5. **佛果功德**：无住涅槃与四智的圆成 |
| **历史地位** | 《摄大乘论》是瑜伽行派最重要的论典之一，对南北朝佛教影响深远。真谛译本流传于南方，形成摄论学派，对地论宗、华严宗、天台宗的形成都有影响。玄奘译本更为完整，是法相唯识宗的重要所依。本论是理解瑜伽行派思想发展的重要文献。 |

---

### 5. 大乘庄严经论（Mahāyāna-sūtrālaṃkāra-śāstra）

| 项目 | 内容 |
|------|------|
| **传出者** | 弥勒（Maitreya）菩萨说，无著（Asaṅga）菩萨记 |
| **译者** | 波罗颇蜜多罗（Prabhākaramitra）于唐贞观年间译出；另有藏译本 |
| **卷数** | 13卷（24品） |
| **核心内容** | 本论属弥勒五论之一，以诗颂体阐述大乘佛法的庄严（美好特质）。内容涵盖发菩提心、菩萨行、六度、十地、如来功德等，特别强调大乘佛法相较于小乘的殊胜之处，故名"庄严"。 |
| **主要教义** | 1. **菩提心**：发心的种类、体性、功德与保护<br>2. **六度**：布施、持戒、忍辱、精进、禅定、般若的详细阐述<br>3. **十地**：菩萨修行的十个阶位<br>4. **如来功德**：佛的十力、四无畏、十八不共法等<br>5. **大乘殊胜**：为何大乘优于小乘 |
| **历史地位** | 本论是弥勒五论中篇幅最长者，文学性较强，对理解印度大乘菩萨道思想有重要价值。在藏传佛教中，本论是弥勒五论之一，受到高度重视。汉传佛教中，本论对理解菩萨修行体系有重要参考价值，也是研究印度佛教文学的重要文献。 |

---

### 6. 辩中边论（Madhyānta-vibhāga-śāstra）

| 项目 | 内容 |
|------|------|
| **传出者** | 弥勒（Maitreya）菩萨说，世亲（Vasubandhu）菩萨释 |
| **译者** | 玄奘法师译（另有真谛译《中边分别论》） |
| **卷数** | 3卷（或作2卷、1卷） |
| **核心内容** | 本论属弥勒五论之一，"辩中边"即"辨明中道与边见"。全论分为五品：辩相品、辩障品、辩真实品、辩修对治品、辩修分位品、辩得果品、辩无上乘品。系统阐述远离断常二边的中道正见。 |
| **主要教义** | 1. **中道**：离增益（常见）与损减（断见）的中道<br>2. **三性中道**：依他起为中性，遍计执为杂染，圆成实为清净<br>3. **十障**：修行中需要克服的十种障碍<br>4. **十真实**：从根本到究竟的十种真实<br>5. **修行位次**：资粮位、加行位、通达位、修习位、究竟位 |
| **历史地位** | 本论是理解瑜伽行派中道思想的重要文献，对唯识学的三性说有系统阐述。玄奘译本与真谛译本各有所长，对照研究有助于深入理解原文。在藏传佛教中，本论是弥勒五论之一，宗喀巴大师等对其有重要注疏。本论也是研究印度佛教中观与唯识关系的重要资料。 |

---

## 三、阿毗达磨论典（Abhidharma）

阿毗达磨（Abhidharma）意译为"对法"、"无比法"，是佛教各部派系统整理、研究佛法的论书。

### 1. 大毗婆沙论（Mahāvibhāṣā-śāstra）

| 项目 | 内容 |
|------|------|
| **编集者** | 说一切有部（Sarvāstivāda）论师集体编集，传说由迦多衍尼子（Kātyāyanīputra）等人主持 |
| **译者** | 玄奘法师于唐显庆元年至四年（656-659年）译出 |
| **卷数** | 200卷 |
| **核心内容** | 本论为《发智论》的广释，是说一切有部的根本论书。全书对佛教的各种概念、法相进行系统分类与解释，涵盖色法、心法、心所法、心不相应行法、无为法等一切法，并对当时各部的异说进行评判。 |
| **主要教义** | 1. **法相分析**：系统建立五位七十五法的分类体系<br>2. **三世实有**：过去、现在、未来一切法皆实有<br>3. **因果学说**：六因、四缘、五果的详细阐述<br>4. **修行阶位**：四向四果、见道、修道等<br>5. **破斥异部**：对大众部、经量部等各部异说的批判 |
| **历史地位** | 《大毗婆沙论》是现存最庞大的佛教论书，是研究说一切有部思想的最重要文献，也是理解部派佛教发展史的必读经典。本论保存了大量早期佛教的珍贵资料，对研究原始佛教思想有重要参考价值。玄奘法师的翻译历时四年，参与译场者达千余人，是唐代译经史上的盛事。 |

---

### 2. 俱舍论（Abhidharma-kośa-śāstra）

| 项目 | 内容 |
|------|------|
| **作者** | 世亲（Vasubandhu）菩萨造。此论为世亲由小乘转入大乘前的代表作。 |
| **译者** | 真谛译《阿毗达磨俱舍释论》（22卷）、玄奘译《阿毗达磨俱舍论》（30卷） |
| **卷数** | 30卷（玄奘译本） |
| **核心内容** | 本论以颂文概述说一切有部的教义，再以长行解释，最后往往以经量部的观点评判有部。全书分为八品：界品、根品、世间品、业品、随眠品、贤圣品、智品、定品，涵盖佛教的世界观、人生观、伦理观与解脱观。 |
| **主要教义** | 1. **五位七十五法**：色法十一、心法一、心所法四十六、心不相应行法十四、无为法三<br>2. **四谛十六行相**：苦、集、灭、道四谛的详细观修<br>3. **十二因缘**：三世两重的缘起观<br>4. **修道阶位**：从暖、顶、忍、世第一法到阿罗汉果<br>5. **禅修次第**：四禅八定、灭尽定等 |
| **历史地位** | 《俱舍论》被誉为"聪明论"，组织严密、义理清晰，是研究部派佛教的最佳入门书。玄奘译本最为完整准确，唐代设有专门的俱舍宗。本论在印度、中国、西藏、日本都有深远影响，是连接南北传佛教的重要桥梁，也是现代佛学研究中译本最多、研究最深入的佛教论书之一。 |

---

### 3. 成实论（Satyasiddhi-śāstra）

| 项目 | 内容 |
|------|------|
| **作者** | 诃梨跋摩（Harivarman），约公元3-4世纪，原属说一切有部，后批判有部思想，综合各部学说而成此论。 |
| **译者** | 鸠摩罗什于后秦弘始十三年（411年）至十四年译出 |
| **卷数** | 16卷（或作20卷） |
| **核心内容** | 本论以"成实"（成立四谛真实）为宗旨，批判说一切有部的实有法体论，主张人法二空。全书分为五聚：发聚、苦谛聚、集谛聚、灭谛聚、道谛聚，系统阐述四谛法门，被认为是从小乘向大乘过渡的重要论典。 |
| **主要教义** | 1. **二谛**：世界悉檀与第一义谛<br>2. **人法二空**：人无我与法无我的初步建立<br>3. **四谛**：苦谛（五受阴）、集谛（业与烦恼）、灭谛（涅槃）、道谛（三十七道品）<br>4. **五受阴**：色、识、想、受、行蕴的分析<br>5. **涅槃**：灭尽心业，证得毕竟空的境界 |
| **历史地位** | 《成实论》在南北朝时期影响极大，形成成实学派，与俱舍宗、涅槃宗等并称。本论被视为小乘向大乘过渡的桥梁，其思想与般若学有相通之处，对三论宗的形成有一定影响。隋代吉藏大师批判本论为"小乘内之附庸"，此后成实学派渐衰，但本论作为研究部派佛教思想演变的重要文献，仍具有重要学术价值。 |

---

## 结语

佛教论藏是佛法智慧的结晶，各派论典从不同角度阐释佛陀教法：

- **中观派**以缘起性空为核心，破除一切执著，建立中道正见
- **瑜伽行派**以万法唯识为纲要，详细分析心识与法相，建立修行体系
- **阿毗达磨**以法相分析为基础，系统整理佛教教义，为修行提供理论依据

这些论典不仅是佛学研究的宝贵资料，更是实修者深入经藏、指导修行的重要依归。愿读者能由此入门，深入佛法大海，究竟觉悟彼岸。
