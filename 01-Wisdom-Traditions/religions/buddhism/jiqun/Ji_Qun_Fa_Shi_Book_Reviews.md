## 学术定位

济群法师属于现代中国佛教**戒律学**与**佛教心理学**方向的代表学者。其学术特点在于：

| 学术特点 | 说明 |
| :--- | :--- |
| **戒律学专长** | 现代南山律学传承，对《四分律》有深入研究 |
| **心理学视角** | 将唯识学与现代心理学结合，开创佛教心理学应用 |
| **人间佛教** | 继承太虚大师、印顺导师人间佛教理念 |
| **深入浅出** | 将深奥佛学义理转化为通俗语言，适合现代读者 |

#### 基本信息

#### 核心内容

| 主题 | 核心观点 | 实践方法 |
| :--- | :--- | :--- |
| **般若空性** | "色不异空，空不异色"——诸法无自性 | 观照缘起，超越分别 |

| **止观双运** | 理事并重，解行相应 | 止中有观，观中有止 |

#### 特色与价值

| 特色 | 说明 |
| :--- | :--- |
| **深入浅出** | 将复杂的般若义理用通俗易懂的语言表达 |
| **对应现实** | 每段经文都配合现代人的困惑与需求 |
| **实修指导** | 不仅停留在理论层面，更有实修操作指引 |
| **学术严谨** | 依经文原义展开，不偏离正统般若思想 |
