# 佛教历史发展年表

---

## 一、佛陀时期（公元前6-5世纪）

| 时间 | 事件 |
|------|------|
| **公元前565/623年** | **佛陀诞生** — 释迦牟尼出生于古印度迦毗罗卫国（今尼泊尔境内），为净饭王太子 |
| **公元前531/588年** | **佛陀成道** — 35岁时于菩提伽耶菩提树下证悟无上正等正觉 |
| 公元前528/585年 | **初转法轮** — 佛陀在鹿野苑（Sarnath）为五比丘说法，开示四圣谛、八正道，佛教僧团正式成立 |
| **公元前486/543年** | **佛陀涅槃** — 80岁时于拘尸那揭罗（Kushinagar）入灭 |
| 公元前486/543年 | **第一次结集** — 摩诃迦叶（Mahākāśyapa）主持，于王舍城（Rajgir）集结佛陀教法，阿难诵经、优波离诵律 |

---

## 二、部派佛教时期（公元前4世纪-公元1世纪）

| 时间 | 事件 |
|------|------|
| 公元前386年 | **第二次结集** — 耶舍（Yasa）主持，于毗舍离（Vaishali）举行，讨论戒律争议，上座部与大众部分歧初现 |
| 公元前3世纪 | **上座部与大众部分裂** — 佛教首次分裂为根本二部，此后逐渐形成十八部派（或说二十部） |
| **公元前250年** | **第三次结集** — 阿育王（Ashoka）支持，目犍连子帝须（Moggaliputta Tissa）主持，于华氏城（Pataliputra）举行，编纂《论事》（Kathavatthu），确定上座部正统 |
| **公元前247年** | **佛教传入斯里兰卡** — 阿育王之子摩哂陀（Mahinda）长老将上座部佛教传入锡兰（今斯里兰卡），建立大寺（Mahavihara）传统 |
| 公元前1世纪 | 巴利三藏首次书写记录 — 此前皆以口传背诵传承 |

---

## 三、大乘佛教兴起（公元1-6世纪）

| 时间 | 事件 |
|------|------|
| 公元1世纪 | **初期大乘经典出现** — 《般若经》《法华经》《无量寿经》等大乘经典开始流通，提倡菩萨道与普度众生 |
| **150-250年** | **龙树菩萨与中观学派** — 龙树（Nagarjuna）著《中论》，阐述缘起性空思想，创立中观学派（Madhyamaka），弟子提婆（Aryadeva）继承发扬 |
| 320-400年 | **无著、世亲与瑜伽行派** — 无著（Asanga）、世亲（Vasubandhu）兄弟创立瑜伽行派（Yogacara），阐述唯识学说，著《瑜伽师地论》《唯识三十颂》等 |
| 320-550年 | **笈多王朝佛教兴盛** — 印度笈多王朝时期，佛教艺术（如犍陀罗艺术、马图拉艺术）蓬勃发展，那烂陀寺成为学术中心 |
| **427年** | **那烂陀寺建立** — 于摩揭陀国建立，后发展为古印度最高佛教学府，玄奘、义净等曾留学于此 |
| 5-6世纪 | 陈那（Dignaga）、法称（Dharmakirti）创立佛教因明学（逻辑学） |

---

## 四、佛教传入中国（公元1-10世纪）

| 时间 | 事件 |
|------|------|
| **公元67年** | **佛教正式传入中国** — 汉明帝夜梦金人，遣使西行求法，摄摩腾、竺法兰以白马驮经至洛阳，建白马寺，为中国第一座佛寺 |
| 148-170年 | 安世高、支娄迦谶来华译经 — 早期小乘禅法与大乘般若经典开始汉译 |
| **401-413年** | **鸠摩罗什译经** — 后秦迎请鸠摩罗什（Kumarajiva）至长安，译出《法华经》《金刚经》《中论》等三百余卷，奠定中国佛教义学基础 |
| 5-6世纪 | 南北朝佛教鼎盛 — 寺院林立，开凿云冈、龙门石窟，竺道生倡"一切众生皆有佛性" |
| **629-645年** | **玄奘西行** — 玄奘法师西行求法十七年，游学印度，于那烂陀寺学习，归国后译出《大般若经》《瑜伽师地论》等千余卷，创立法相宗 |
| 600-760年 | **禅宗六祖** — 达摩祖师传法入华，经慧可、僧璨、道信、弘忍，至**六祖惠能**（638-713年）开创南宗顿悟法门，著《六祖坛经》 |
| 7-8世纪 | **八大宗派形成** — 天台宗（智顗）、三论宗、法相宗（玄奘）、华严宗（法藏）、禅宗、净土宗、律宗、密宗（善无畏、金刚智） |
| 8世纪 | 义净西行（671-695年） — 赴印度求法，归国后译经，著《大唐西域求法高僧传》 |

---

## 五、藏传佛教（公元7-14世纪）

| 时间 | 事件 |
|------|------|
| 公元7世纪 | **佛教传入吐蕃** — 松赞干布（617-650年）统一西藏，迎娶文成公主、尺尊公主，建立大昭寺、小昭寺，佛教开始传入西藏 |
| 641年 | 文成公主入藏 — 携带佛像、佛经，促进汉藏文化交流 |
| **747年** | **莲花生大士入藏** — 乌仗那国大师莲花生（Padmasambhava）受邀入藏，降伏本土苯教神祇，建立桑耶寺，为藏传佛教奠基，开创宁玛派（红教） |
| 8-9世纪 | 前弘期佛教 — 赤松德赞、赤祖德赞大力支持佛教，邀请寂护（Shantarakshita）大师入藏，建立僧团，开展译经事业 |
| 838-842年 | 朗达玛灭佛 — 吐蕃末代赞普禁佛，佛教在西藏一度衰落 |
| 10世纪 | 后弘期开始 — 佛教从阿里、安多地区重新传入西藏，形成多种传承 |
| **1042年** | **阿底峡尊者入藏** — 印度超戒寺高僧阿底峡（Atisha）受邀入藏，弘法十五年，著《菩提道灯论》，整顿佛教戒律与修行次第，对噶当派及格鲁派影响深远 |
| 1073-1150年 | 萨迦派（花教）兴起 — 昆·贡却杰布创建萨迦寺，萨迦班智达、八思巴等发扬光大 |
| 11-12世纪 | 噶举派（白教）建立 — 玛尔巴译师、米拉日巴、冈波巴等传承，注重密法实修 |
| **1357-1419年** | **宗喀巴大师** — 创立格鲁派（黄教），强调戒律清净与次第修行，建甘丹寺，著《菩提道次第广论》《密宗道次第广论》，其弟子达赖、班禅转世系统延续至今 |

## 六、佛教传播世界（公元19-21世纪）

| 时间 | 事件 |
|------|------|
| 19世纪中叶 | **佛教传入欧美** — 英国殖民者发现巴利文献，西方学者开始佛教研究；1879年《大乘起信论》英译本出版；达摩波罗（Anagarika Dharmapala）赴欧美弘法 |
| 1881年 | 巴利圣典会（Pali Text Society）成立 — 英国里斯·戴维斯（T.W. Rhys Davids）创建，系统整理翻译巴利三藏 |
| 1893年 | 世界宗教议会（芝加哥）— 达摩波罗、释宗演等东方佛教代表参会，佛教正式进入西方主流视野 |
| 20世纪初 | 禅宗传入西方 — 铃木大拙（D.T. Suzuki）向西方介绍禅宗思想，影响深远 |
| 1950年 | 世界佛教徒联谊会（WFB）成立 — 于斯里兰卡科伦坡成立，推动全球佛教徒团结 |
| 1956年 | 印度安贝德卡尔皈依佛教 — 数十万贱民集体皈依，印度佛教复兴 |
| 1960年代 | 西方禅修运动兴起 — 马哈希、葛印卡等内观禅修大师在西方教授禅法；藏传佛教四大教派上师流亡海外弘法 |
| 1980年代后 | **人间佛教蓬勃发展** — 台湾佛光山（星云大师）、法鼓山（圣严法师）、慈济（证严法师）等推行人间佛教，强调生活化、国际化 |
| 21世纪 | 佛教数字化与全球化 — 网络弘法、电子佛典、线上禅修课程普及；佛教环保、社会公益、心灵关怀等新领域发展 |
