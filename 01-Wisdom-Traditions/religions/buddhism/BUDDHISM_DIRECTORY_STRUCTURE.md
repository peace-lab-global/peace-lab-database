# 佛教专业内容体系目录结构说明 (Buddhism Professional Content Directory Structure)

## 总体架构 (Overall Architecture)

佛教专业内容体系按照"核心哲学-传统实践-心理学应用-修行方法-高级主题"的逻辑结构组织，形成了完整而系统的知识框架。

```
buddhism/
├── foundations/               # 佛教基础教义
├── history/                   # 历史与文献体系
├── core-philosophy/           # 核心哲学理论
├── traditions/                # 各宗派传统总论
├── theravada/                 # 南传上座部专题
├── vajrayana/                 # 金刚乘专题
├── pure-land/                 # 净土宗专题
├── tiantai/                   # 天台宗专题
├── dzogchen/                  # 大圆满专题
├── mahamudra/                 # 大手印专题
├── meditation/                # 禅修实践方法
├── sutras/                    # 核心经典
├── rituals/                   # 佛教仪轨
├── psychology/                # 心理学应用
├── ethics/                    # 戒律与伦理学
├── advanced/                  # 高级修行主题
├── vasana/                    # 熏习理论
├── nan-huaijin/               # 南怀瑾专题
├── jiqun/                     # 济群法师专题
├── dzongsar-khyentse/         # 宗萨仁波切书评
└── modern-applications/       # 佛教现代应用交叉索引
```
