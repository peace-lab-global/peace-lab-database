# 汉传佛教水陆法会仪轨 (Chinese Water-Land Dharma Assembly)

> **水陆法会 (Water-Land Dharma Assembly / Shuilu Fahui)**：汉传佛教最盛大、最庄严的超度法会，全称"法界圣凡水陆普度大斋胜会"。历时七至四十九天，超度水陆空一切孤魂亡灵，普济六道众生。
>
> 不唯度亡，亦且资生；不唯利己，亦且利人。 —— 《水陆仪轨会本》序

---

## 1. 水陆法会概论 (Overview)

### 表1.1 法会历史

| 时期 | 发展 | 关键人物 |
|---|---|---|
| **梁武帝时期（6世纪）** | 创始 | 梁武帝、宝志禅师 |
| **唐代** | 发展完善 | 玄宗时期制度化 |
| **宋代** | 成熟定型 | 杨锷撰《水陆仪轨》 |
| **明清** | 普及兴盛 | 祩宏大师修订 |
| **近现代** | 复兴传承 | 各丛林定期举行 |

### 表1.2 法会基本信息

| 项目 | 内容 |
|---|---|
| **全称** | 法界圣凡水陆普度大斋胜会 |
| **简称** | 水陆法会、水陆道场 |
| **时间** | 七昼夜（最普遍）、四十九昼夜（最大型） |
| **目的** | 超度水陆空一切亡灵、普济六道 |
| **主法** | 金刚上师/方丈/住持 |
| **参与** | 数十至数百僧众，成千上万信众 |
| **费用** | 庞大（现代数十万至数百万人民币） |

## 2. 内坛与外坛 (Inner and Outer Altars)

### 表2.1 外坛（六个）

| 外坛 | 内容 | 功能 |
|---|---|---|
| **大坛** | 礼拜《梁皇宝忏》二十四部 | 忏悔主坛 |
| **法华坛** | 诵《法华经》二十四部 | 圆成佛道 |
| **净土坛** | 诵《阿弥陀经》、念佛 | 往生净土 |
| **华严坛** | 诵《华严经》一部 | 法界缘起 |
| **楞严坛** | 诵《楞严咒》二十四部 | 破魔护法 |
| **诸经坛** | 诵《金刚经》、《药师经》等 | 诸经功德 |

### 表2.2 内坛（核心）

| 项目 | 内容 |
|---|---|
| **位置** | 法会中心，最庄严处 |
| **布置** | 悬挂水陆画、设上堂十席、下堂十四席 |
| **主法** | 金刚上师主持 |
| **核心仪轨** | 结界、发符、悬幡、上堂、下堂、圆满供 |
| **时间** | 七昼夜不间断 |

## 3. 七昼夜完整仪轨 (Seven-Day Ritual Schedule)

### 3.1 第一天：结界洒净

| 时间 | 内容 | 意义 |
|---|---|---|
| 上午 | **洒净** | 以大悲咒水洒净坛场 |
| 下午 | **结界** | 观想金刚墙围困会场 |
| 晚间 | **发符悬幡** | 向诸天发邀请符 |

### 3.2 第二天：奉请上堂

| 时间 | 内容 | 意义 |
|---|---|---|
| 上午 | **奉请上堂** | 迎请诸佛菩萨、缘觉、声闻 |
| 下午 | **诵经** | 各坛诵经不断 |
| 晚间 | **上堂斋** | 供养圣众 |

### 3.3 第三天：奉请下堂

| 时间 | 内容 | 意义 |
|---|---|---|

| 下午 | **诵经** | 各坛诵经 |
| 晚间 | **瑜伽焰口** | 放焰口施食 |

### 3.4 第四天：上堂大斋

| 时间 | 内容 | 意义 |
|---|---|---|
| 上午 | **上堂大斋** | 最隆重供养 |
| 下午 | **诵经** | 各坛诵经 |
| 晚间 | **焰口** | 施食 |

### 3.5 第五天：忏悔

| 时间 | 内容 | 意义 |
|---|---|---|
| 全天 | **拜梁皇宝忏** | 大忏悔 |
| 晚间 | **焰口** | 施食 |

### 3.6 第六天：上圆满供

| 时间 | 内容 | 意义 |
|---|---|---|
| 上午 | **圆满供** | 供养所有圣凡 |
| 下午 | **诵经回向** | 功德回向 |
| 晚间 | **焰口** | 最后大焰口 |

### 3.7 第七天：圆满送圣

| 时间 | 内容 | 意义 |
|---|---|---|
| 上午 | **烧圆满香** | 法会圆满 |
| 下午 | **送圣** | 送诸佛菩萨、众生各归本位 |
| 晚间 | **圆满回向** | 总回向 |

## 5. 水陆画 (Water-Land Paintings)

### 5.1 水陆画概述

| 项目 | 内容 |
|---|---|
| **数量** | 一套138幅（传统） |
| **尺寸** | 约150-200cm高 |
| **材质** | 绢本或布本设色 |
| **悬挂** | 内坛两侧，对称悬挂 |
| **功能** | 代表所邀请的圣凡对象 |

### 5.2 水陆画分类

| 类别 | 数量 | 内容 |
|---|---|---|
| **佛部** | 约20幅 | 诸佛、菩萨 |
| **明王部** | 约10幅 | 忿怒本尊、护法 |
| **罗汉部** | 约15幅 | 阿罗汉、缘觉 |
| **诸天部** | 约25幅 | 天王、诸天善神 |
| **鬼神部** | 约30幅 | 鬼神、夜叉 |
| **圣贤部** | 约15幅 | 祖师、高僧 |
| **众生部** | 约23幅 | 六道众生、亡魂 |
