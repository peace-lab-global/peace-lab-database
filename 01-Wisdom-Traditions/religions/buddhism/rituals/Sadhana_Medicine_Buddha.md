# 药师佛完整修持仪轨 (Medicine Buddha Sadhana / Bhaisajyaguru)

> **药师佛 (Bhaisajyaguru / Medicine Buddha / Yaoshi Fo)**：东方净琉璃世界教主，发愿消除众生病苦、延长寿命、增进福报。此仪轨涵盖药师佛观想、药师咒修持、十二大愿发愿、药师法门的完整体系。
>
> 愿我来世得菩提时，身如琉璃，内外明彻，净无瑕秽，光明广大，功德巍巍。 —— 药师佛第一大愿

## 2. 药师佛法相观想 (Visualization)

### 2.1 药师佛身相

| 部位 | 特征 |
|---|---|
| **身色** | 身如琉璃，内外明彻，呈绀琉璃色 |
| **相好** | 三十二相，八十种好 |
| **手印** | 右手施无畏印，持药诃子；左手定印，持药钵 |
| **坐姿** | 全跏趺坐于莲花宝座 |
| **背光** | 身光普照，犹如琉璃 |

### 2.2 药师佛观想

**完整观想流程：**
1. 观想前方虚空中有大宝莲花座
2. 莲花上有月轮，月轮上有药师佛种子字 "bhai"（藏文）或 "hrīḥ"
3. 种子字放光，化现药师佛完整身相
4. 身如琉璃，光明照耀十方
5. 佛顶有宝塔，塔中安住多宝如来
6. 周围有日光遍照菩萨、月光遍照菩萨
7. 八大菩萨、十二药叉大将围绕

## 3. 药师灌顶真言修持 (Medicine Buddha Mantra Sadhana)

### 3.1 药师灌顶真言

**梵音（汉传常用）：**
```
南无薄伽伐帝 鞞杀社 窭噜薛琉璃
钵喇婆 喝啰阇也 怛他揭多也
阿啰喝帝 三藐三勃陀耶 怛侄他
唵 鞞杀逝 鞞杀逝 鞞杀社
三没揭帝 莎诃
```

**罗马转写：**
```
Namo bhagavate bhaiṣajye guru vaiḍūrya
prabha rājāya tathāgatāya
arhate samyaksambuddhāya tadyathā
Oṃ bhaiṣajye bhaiṣajye bhaiṣajya
samudgate svāhā
```

## 4. 药师忏修持 (Medicine Buddha Confession)

### 4.1 药师忏流程

| 顺序 | 内容 | 说明 |
|---|---|---|
| 1 | 净坛 | 洒净 |
| 2 | 皈依 | 皈依佛法僧 |
| 3 | 药师赞 | 赞颂药师佛 |
| 4 | 诵《药师经》 | 全经或择要 |
| 5 | 持药师咒 | 108遍以上 |
| 6 | 发愿 | 发十二大愿 |
| 7 | 忏悔 | 忏悔业障 |
| 8 | 回向 | 回向一切众生 |

### 4.2 药师忏发愿文

```
愿消三障诸烦恼 愿得智慧真明了
普愿罪障悉消除 世世常行菩萨道
愿我来世得菩提时 身如琉璃 内外明彻
净无瑕秽 光明广大 功德巍巍
```
