# 禅宗打七仪轨 (Zen Seven-Day Retreat / Chan Qi)

> **禅七 (Chan Qi / Zen Sesshin)**：禅宗最核心的集体修行仪轨，源于佛陀在菩提树下七日证悟的典故。修行者七日之内禁语、禁足、禁卧，专一参究话头或默照禅，是禅宗「克期取证」的极致体现。
>
> 七日为期，寸香过堂；不悟不疑，不参不食。 —— 丛林古规

---

## 1. 禅七概论 (Overview)

### 表1.1 禅七的历史源流

| 时期 | 发展 | 关键人物 |
|---|---|---|
| **佛陀时代** | 结夏安居 | 佛陀制定雨季三月禁足 |
| **唐代** | 禅堂制度确立 | 百丈怀海禅师立清规 |
| **宋代** | 打七制度化 | 长芦宗赜禅师完善 |
| **明清** | 规约细化 | 《百丈清规》、《禅门日诵》 |
| **近现代** | 普化与简化 | 虚云老和尚、来果禅师复兴 |

### 表1.2 禅七的种类

| 类型 | 时间 | 内容 | 对象 |
|---|---|---|---|
| **禅七** | 七日 | 参话头或默照 | 禅宗行者 |
| **禅五** | 五日 | 简化版禅七 | 初学者 |
| **方便七** | 周末两日 | 现代适应版 | 居士 |
| **精进七** | 十四日或更长 | 深修 | 资深行者 |
| **生死七** | 七日不倒单 | 极限精进 | 大修行者 |

## 4. 坐禅仪轨 (Zazen)

### 4.1 入堂坐禅

| 步骤 | 动作 | 要点 |
|---|---|---|
| 1 | **闻引磬入堂** | 闻引磬三声，进入禅堂 |
| 2 | **绕佛一圈** | 沿东单或西单绕至座位 |
| 3 | **站立** | 于座前站立，合掌 |
| 4 | **礼座** | 以右手抚座，表示让众生先坐 |
| 5 | **展具** | 展开坐具，放置坐垫上 |
| 6 | **就坐** | 端身正坐，结跏趺或半跏趺 |
| 7 | **调身** | 头正、肩平、背直 |
| 8 | **调息** | 深呼吸三次，转为自然呼吸 |
| 9 | **调心** | 提起话头或默照 |
| 10 | **静候开静** | 等待引磬声出定 |

## 6. 参话头 (Huatou Practice)

### 6.1 常见话头

| 话头 | 来源 | 用功方法 |
|---|---|---|
| **念佛是谁** | 最普遍 | 念佛时反观能念者是谁 |
| **万法归一，一归何处** | 赵州禅师 | 追究「一」归何处 |
| **未出娘胎谁是我** | 大慧宗杲 | 追究本来面目 |
| **狗子无佛性** | 赵州公案 | 追究「无」字 |
| **庭前柏树子** | 赵州公案 | 追究话头真意 |
| **干屎橛** | 云门文偃 | 追究话头 |

### 6.2 参话头方法

```
第一步：提起话头
在心中反复念「念佛是谁」，如猫捕鼠，紧盯不放。

第二步：起疑情
对「是谁」产生大疑情，如丧考妣，如救头燃。

第三步：追穷到底
步步紧跟，穷追不舍，不令间断。

第四步：疑成一团
疑情打成一片，行住坐卧不离此话头。

第五步：桶底脱落
疑至极处，如桶底脱，豁然开朗。
```

## 7. 禅堂规矩 (Zen Hall Rules)

### 7.1 四止语

| 止语 | 内容 |
|---|---|
| **口止语** | 不开口说话 |
| **眼止语** | 不左顾右盼 |
| **身止语** | 不妄动身体 |
| **意止语** | 不散乱妄想 |

### 7.2 警策（打香板）

| 情况 | 处理方式 |
|---|---|
| **昏沉** | 香板轻点肩部，提醒清醒 |
| **姿势不正** | 香板纠正 |
| **严重昏沉** | 请下座，站立或跑香 |
