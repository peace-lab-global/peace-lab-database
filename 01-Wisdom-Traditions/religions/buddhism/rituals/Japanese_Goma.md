# 日本真言宗护摩仪轨 (Japanese Shingon Goma Ritual)

> **护摩（Goma）**：日本真言宗最核心的仪轨之一，每日于高野山等寺院定时举行。以智慧火焚烧供品，供养诸佛菩萨、护法圣众，具息灾、增益、调伏、敬爱四大功能。
>
> 一茎草上，建法王坛；一茎草上，转大法轮。 —— 空海大师

---

## 1. 护摩坛结构 (Goma Altar Structure)

### 1.1 外坛

| 要素 | 位置 | 象征 |
|---|---|---|
| **护摩坛** | 中央 | 大日如来的心殿 |
| **贤瓶** | 坛左 | 甘露水源 |
| **供品盘** | 坛前 | 妙欲供品 |
| **六器** | 坛周 | 六波罗蜜 |
| **烛台** | 坛两侧 | 智慧光明 |
| **香炉** | 坛前 | 戒定真香 |

### 1.2 内坛（火炉）

| 要素 | 说明 |
|---|---|
| **火炉** | 铜制，高约一尺，圆形 |
| **火口** | 向东或向南 |
| **火床** | 炉底设火床，置护摩木 |
| **炉口本尊** | 观想本尊坐于炉口 |
| **火中天** | 观想火神阿耆尼于火中 |

## 3. 不动明王护摩完整仪轨 (Fudo Myoo Goma)

### 3.1 前行

| 顺序 | 内容 | 说明 |
|---|---|---|
| 1 | **着法衣** | 著袈裟、挂络子 |
| 2 | **净身** | 以水洒净身心 |
| 3 | **着甲胄印** | 结护身印，观想自身被光明甲胄围绕 |
| 4 | **净三业** | 净身口意三业 |
| 5 | **设坛** | 布置护摩坛，供养诸尊 |

### 3.2 正行

| 顺序 | 内容 | 真言 |
|---|---|---|
| 1 | **净火炉** | 以真言净火炉 |
| 2 | **点火** | 点护摩木，观想智慧火 |
| 3 | **迎请火天** | 迎请火神阿耆尼 | 唵 阿祇尼耶 娑婆诃 |
| 4 | **供养火天** | 供养火天 | 火天真言 |
| 5 | **迎请本尊** | 迎请不动明王 | 不动明王真言 |
| 6 | **供养本尊** | 以护摩木供养 | 本尊真言 |
| 7 | **投供品** | 依次投入供品 | 护摩真言 |
| 8 | **祈请事业** | 祈请息增怀诛 | 事业真言 |

### 3.3 不动明王真言

**梵音：**
```
南无 三曼多 缚日罗 喃 憾
```

**罗马转写：**
```
Namo Samanta Vajranam Ham
```

**含义：** 皈命普遍诸金刚。

### 3.4 结行

| 顺序 | 内容 |
|---|---|
| 1 | **供养诸尊** | 供养护摩坛一切圣众 |
| 2 | **送圣** | 送圣众各归本位 |
| 3 | **回向** | 回向一切众生 |
| 4 | **收坛** | 撤收护摩坛 |
