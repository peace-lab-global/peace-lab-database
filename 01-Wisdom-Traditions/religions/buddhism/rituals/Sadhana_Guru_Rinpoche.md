# 莲花生大士完整修持仪轨 (Guru Rinpoche / Padmasambhava Sadhana)

> **莲花生大士 (Padmasambhava / Guru Rinpoche / Lianhua Sheng Da Shi)**：藏传佛教创始祖师，被尊为第二佛。此仪轨涵盖莲师七句祈请、莲师心咒、莲师会供、莲师护法的完整修持体系，为宁玛派最核心的修持法门。
>
> 给我你的心，我将加持你；给我你的信仰，我将保护你。 —— 莲花生大士

### 2.2 莲师七句祈请文（最重要祈请文）

**藏音：**
```
吽
欧坚 耶吉 努 向 贡
巴玛 萨瓦 东波 拉
雅拉 却格 朱古 杜
巴玛 炯尼 些些 拉
沽汝 色吉 所 所 克
巴玛 炯尼 些些 索
```

**梵音罗马转写：**
```
Hūṃ
Oḍḍiyāna ye gi nub gyi nang
Padma ge sar dong po la
Ya mtshan mchog gi sgrub pa nyid
Padma sam bhava zhes su grags
Khor du ma ṇḍa ra wa dang
Sras kyi mchog gi tshogs kyis bskor
Khyed kyi rjes su bdag bsgyur nas
Gsol ba deb so byin gyis brlobs
Hūṃ
```

**汉译：**
```
吽
邬金刹土西北隅
莲茎花胚之座上
稀有殊胜成就者
世称名号莲花生
空行眷属众围绕
我随汝尊而修持
为赐加持祈降临
格日贝玛思德吽
```

### 2.3 七句祈请文观修法

| 步骤 | 观想 |
|---|---|
| 1 | 观想前方虚空中有彩虹光蕴 |
| 2 | 光中出现铜色吉祥山（莲师净土） |
| 3 | 山上有大宝莲花座 |
| 4 | 莲师安坐其上，周围有空行、护法围绕 |
| 5 | 诵七句祈请文时，莲师微笑 |
| 6 | 从莲师心间放光，融入自身 |
| 7 | 自身与莲师无二无别 |

## 3. 莲师心咒修持 (Guru Rinpoche Mantra Sadhana)

### 3.1 莲师心咒

**藏音：**
```
唵 阿 吽 班匝 格热 贝玛 色德 吽
```

**罗马转写：**
```
Oṃ Āḥ Hūṃ Vajra Guru Padma Siddhi Hūṃ
```

**含义：**
| 音节 | 含义 |
|---|---|
| **Oṃ (唵)** | 三宝之身体 |
| **Āḥ (阿)** | 三宝之语言 |
| **Hūṃ (吽)** | 三宝之心灵 |
| **Vajra (班匝/金刚)** | 不可摧毁的智慧 |
| **Guru (格热/上师)** | 具德导师 |
| **Padma (贝玛/莲花)** | 莲花中诞生 |
| **Siddhi (色德/成就)** | 共通与殊胜成就 |
| **Hūṃ (吽)** | 祈请加持 |

### 3.2 莲师心咒修持仪轨

| 顺序 | 内容 | 次数 |
|---|---|---|
| 1 | 观想莲师于顶门 | — |
| 2 | 七句祈请文 | 3/7/21遍 |
| 3 | 莲师心咒 | 108/1000/10000遍 |
| 4 | 观想莲师心间放光 | 持续 |
| 5 | 光融入自身 | 收摄 |
| 6 | 回向 | 1 |

### 3.3 莲师心咒功德

据经典记载，持诵莲师心咒可得：
- 消除内外障碍
- 增长寿命福德
- 避免一切灾难
- 事业顺利成就
- 临终往生莲师净土（铜色吉祥山）
