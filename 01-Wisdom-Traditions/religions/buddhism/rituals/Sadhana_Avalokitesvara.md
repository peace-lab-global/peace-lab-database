# 观世音菩萨完整修持仪轨 (Avalokitesvara Bodhisattva Sadhana)

> **观世音菩萨 (Avalokitesvara / Guanyin / Chenrezig)**：大慈大悲的化身，汉传佛教最受欢迎的菩萨。此仪轨涵盖大悲咒修持、六字大明咒、观音法门、千手观音观想的完整体系。
>
> 若有众生，多于淫欲，常念恭敬观世音菩萨，便得离欲；若多嗔恚，常念恭敬观世音菩萨，便得离嗔；若多愚痴，常念恭敬观世音菩萨，便得离痴。 —— 《观世音菩萨普门品》

## 3. 六字大明咒修持 (Six-Syllable Mantra Sadhana)

### 3.1 六字大明咒

**藏音：**
```
唵 嘛呢叭咪 吽
```

**梵音罗马转写：**
```
Oṃ Maṇi Padme Hūṃ
```

**含义：**
| 字 | 含义 | 净化 |
|---|---|---|
| **唵 (Oṃ)** | 佛部心 | 净化天道 |
| **嘛 (Ma)** | 宝部心 | 净化阿修罗道 |
| **呢 (ṇi)** | 莲花部心 | 净化人道 |
| **叭 (Pad)** | 金刚部心 | 净化畜生道 |
| **咪 (me)** | 事业部心 | 净化饿鬼道 |
| **吽 (Hūṃ)** | 佛部心 | 净化地狱道 |

### 3.2 六字大明咒观修法

**四臂观音观想：**
| 部位 | 特征 |
|---|---|
| **身色** | 白色 |
| **面容** | 一面，双目微垂，慈悲注视 |
| **前二手** | 合掌于心间，持摩尼宝珠 |
| **左上** | 持白莲花 |
| **右上** | 持水晶珠 |
| **坐姿** | 全跏趺坐于月轮莲台 |

**持咒观想：**
1. 观想四臂观音于顶门或前方
2. 观音心间放白光
3. 光入自身，净化六道习气
4. 自身化为四臂观音
5. 持咒不断，光不断放射，利益众生

## 4. 千手千眼观音观想 (Thousand-Armed Avalokitesvara)

### 4.1 身相

| 特征 | 描述 |
|---|---|
| **面** | 十一面，前三面慈悲，左三面嗔怒，右三面白牙上出，后一面暴大笑，顶一面如来 |
| **手** | 千手，每手掌有一眼 |
| **主手** | 二手合掌，余九百九十八手各持法器 |
| **身色** | 白色或金色 |
| **坐姿** | 全跏趺坐于大莲花上 |

### 4.2 四十种法器

千手观音手持四十种法器，代表救度众生的各种善巧方便。

## 5. 观音法门修持 (Guanyin Dharma Practice)

### 5.1 《普门品》持诵

**核心段落：**
```
佛告无尽意菩萨：善男子，若有无量百千万亿众生，
受诸苦恼，闻是观世音菩萨，一心称名，
观世音菩萨即时观其音声，皆得解脱。
```

### 5.2 耳根圆通（楞严经）

```
初于闻中，入流亡所。
所入既寂，动静二相，了然不生。
如是渐增，闻所闻尽。
尽闻不住，觉所觉空。
空觉极圆，空所空灭。
生灭既灭，寂灭现前。
```
