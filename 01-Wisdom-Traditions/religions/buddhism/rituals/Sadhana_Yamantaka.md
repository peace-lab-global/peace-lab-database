# 大威德金刚完整修持仪轨 (Yamantaka / Vajrabhairava Sadhana)

> **大威德金刚 (Yamantaka / Vajrabhairava)**：文殊菩萨的忿怒化身，藏传佛教格鲁派三大本尊之首。具九面三十四臂十六足，为水牛首忿怒相，象征降伏死魔的终极力量。
>
> 大威德者，大威有力，能降恶魔；大德无边，能护善士。 —— 格鲁派祖师语

---

## 1. 大威德金刚概论 (Overview)

### 表1.1 身份与意义

| 层面 | 说明 |
|---|---|
| **本质** | 文殊菩萨的忿怒化身 |
| **功能** | 降伏死魔、烦恼魔、天魔、五阴魔 |
| **事业** | 智慧降伏、护持正法 |
| **级别** | 无上瑜伽部父续 |
| **教派** | 格鲁派三大本尊之首 |

### 表1.2 身相特征

| 部位 | 特征 |
|---|---|
| **主面** | 水牛头，极忿怒，双目圆睁，獠牙外露 |
| **上面** | 文殊菩萨寂静面（头顶） |
| **左右面** | 共九面，五色各异 |
| **手臂** | 三十四臂，各持法器 |
| **足** | 十六足，踩踏八禽八兽 |
| **身色** | 深蓝黑色 |
| **拥抱明妃** | 拥抱金刚起尸母（Vetali） |
