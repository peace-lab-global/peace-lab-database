# 唐密/东密护摩仪轨 (Esoteric Homa / Goma Rituals)

> **护摩 (Homa / Goma)**：源于古印度吠陀祭火传统，佛教密宗化后成为以火为媒介供养本尊的殊胜法门。唐密（汉传密宗）与东密（日本真言宗）护摩体系最为完备，具息灾、增益、调伏、敬爱、钩召五大功能。
>
> 火能烧除一切烦恼，亦能成熟一切善法。 —— 《大日经疏》

---

## 1. 护摩概论 (Overview)

### 表1.1 护摩历史源流

| 时期 | 发展 | 关键人物 |
|---|---|---|
| **吠陀时代** | 火祭起源 | 婆罗门火神阿耆尼祭祀 |
| **印度密教** | 佛教化 | 龙树菩萨、龙智菩萨 |
| **唐代** | 传入中国 | 善无畏、金刚智、不空三藏 |
| **日本平安时代** | 东密确立 | 空海大师（弘法大师） |
| **日本现代** | 制度化 | 各总本山定期举行 |

### 表1.2 五大护摩

| 类型 | 梵语 | 功能 | 火色 | 本尊 |
|---|---|---|---|---|
| **息灾护摩** | Santaika | 消除灾难、疾病、业障 | 白色 | 佛顶尊胜、大日如来 |
| **增益护摩** | Paustika | 增长福寿、智慧、财富 | 黄色 | 大日如来、虚空藏 |
| **调伏护摩** | Abhicaraka | 降伏魔怨、恶人 | 黑色 | 大威德明王、军荼利 |
| **敬爱护摩** | Vashikarana | 怀柔人天、增进关系 | 红色 | 爱染明王、观音 |
| **钩召护摩** | Agharana | 召请圣众、摄受众生 | 杂色 | 多罗菩萨、金刚萨埵 |

## 2. 护摩坛城 (Homa Mandala)

### 2.1 护摩坛结构

| 要素 | 内容 | 象征 |
|---|---|---|
| **火炉（护摩炉）** | 铜制或土制，圆形或方形 | 佛心/宇宙中心 |
| **火口** | 炉口向东或向南 | 出生与成长 |
| **莲花座** | 炉底刻莲花 | 清净 |
| **金刚界** | 炉内观想金刚界 | 坚固不坏 |
| **火神（阿耆尼）** | 炉中观想火神 | 转化之力 |
| **本尊座** | 炉上观想本尊 | 主尊 |
| **贤瓶** | 盛香水置于炉侧 | 甘露 |
| **供品盘** | 置于炉前 | 妙欲 |

## 3. 护摩仪轨完整流程 (Complete Ritual Process)

### 3.1 前行准备

| 顺序 | 内容 | 说明 |
|---|---|---|
| 1 | **净身** | 沐浴净身，著新衣 |
| 2 | **净坛** | 以真言水洒净护摩坛 |
| 3 | **供养** | 设八供、十供或百供 |
| 4 | **发愿** | 发菩提心，明确护摩目的 |
| 5 | **观想本尊** | 观想本尊降临护摩炉 |

### 3.2 正行

| 顺序 | 内容 | 说明 |
|---|---|---|
| 1 | **点火** | 以智慧火点燃护摩木 |
| 2 | **供养火天** | 先供养火神阿耆尼 |
| 3 | **供养本尊** | 以护摩木投入火中，供养本尊 |
| 4 | **持咒** | 诵本尊真言及护摩真言 |
| 5 | **投入供品** | 依次投入五谷、香草、油、蜜等 |
| 6 | **观想** | 观想供品化为无量，上供圣众，下施众生 |
| 7 | **事业成就** | 祈请息增怀诛事业成就 |

### 3.3 结行

| 顺序 | 内容 |
|---|---|
| 1 | **供养诸尊** | 供养护摩坛一切圣众 |
| 2 | **送圣** | 送圣众各归本位 |
| 3 | **回向** | 回向功德 |
| 4 | **散坛** | 撤收护摩坛 |

## 4. 护摩真言 (Homa Mantras)

### 4.1 通用护摩真言

**火天真言（阿耆尼）：**
```
唵 阿祇尼耶 娑婆诃
（Om Agnaye Svaha）
```

**护摩真言：**
```
唵 嚩日罗 俱路驮耶 吽 癹吒
（Om Vajra Kulodhaya Hum Phat）
```

## 5. 唐密与东密护摩差异 (Tangmi vs Shingon Homa)

### 表5.1 主要差异

| 对比 | 唐密（汉传） | 东密（日本真言宗） |
|---|---|---|
| **传承** | 唐代传入，宋元后衰微 | 空海大师传自惠果阿阇梨 |
| **现状** | 近现代复兴中 | 日本完整保留 |
| **护摩坛** | 较简单 | 极精致，各总本山有固定护摩坛 |
| **供品** | 简化 | 严格依传统 |
| **频率** | 不定期 | 每日定时举行（如高野山） |
| **服饰** | 海青或袈裟 | 严密装束，威仪具足 |
| **咒语** | 汉语发音 | 日语梵音 |
