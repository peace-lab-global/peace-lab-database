# 南传佛教出家受戒仪轨 (Theravada Ordination Rituals)

> **出家受戒 (Pabbajjā & Upasampadā)**：南传佛教中最神圣的仪轨之一，标志着一个世俗人正式成为佛陀的弟子，踏上解脱之路。此仪轨传承两千五百余年，至今仍在斯里兰卡、缅甸、泰国、柬埔寨、老挝等地严格遵行。
>
> *"出家非为逃避，乃为面对；非为断离，乃为深入。"*

---

## 1. 出家制度概论 (Ordination System Overview)

### 表1.1 出家阶位体系

| 阶位 | 巴利语 | 英文 | 年龄要求 | 戒律数量 | 仪式名称 | 身份特征 |
|---|---|---|---|---|---|---|
| **沙弥** | *Sāmaṇera* | Novice Monk | 7-20岁 | 10条 | *Sāmaṇera-pabbajjā* | 初学者，学习基础 |
| **式叉摩那** | *Sikkhamānā* | Probationary Nun | 18-20岁 | 六法 | *Sikkhamānā-upasampadā* | 女众见习期（2年） |
| **比丘** | *Bhikkhu* | Fully Ordained Monk | 20岁以上 | 227条 | *Upasampadā* | 完全出家 |
| **比丘尼** | *Bhikkhunī* | Fully Ordained Nun | 20岁以上 | 311条 | *Bhikkhunī-upasampadā* | 女众完全出家（目前在南传传统中已中断） |

### 表1.2 出家资格要求

| 要求类别 | 具体内容 | 目的 |
|---|---|---|
| **年龄** | 沙弥满7岁，比丘满20岁 | 身心成熟 |
| **父母同意** | 需父母书面同意（已婚需配偶同意） | 家庭和谐 |
| **无债务** | 无未偿还债务 | 无牵无挂 |
| **无官司** | 非在押犯或通缉犯 | 法律清白 |
| **无重大疾病** | 无传染病或严重残疾 | 能自理生活 |
| **非奴隶** | 非他人财产 | 人身自由 |
| **非军人** | 非现役军人 | 避免冲突 |
| **非王族特殊身份** | 非需要继承王位者 | 责任履行 |
| **非负债潜逃者** | 非逃避债务 | 诚信 |
| **非他人强迫** | 出自自愿 | 真实发心 |

## 2. 沙弥剃度仪轨 (Sāmaṇera Ordination)

### 2.1 剃度前准备

| 准备事项 | 具体内容 | 时间 |
|---|---|---|
| **请求出家** | 向寺院住持或比丘表达出家意愿 | 提前数周 |
| **师长考察** | 师长观察求度者发心与品行 | 1-4周 |
| **父母同意书** | 取得父母签字同意 | 剃度前 |
| **准备袈裟** | 准备沙弥袈裟（二衣） | 剃度前 |
| **准备钵盂** | 准备乞食用钵 | 剃度前 |
| **沐浴净身** | 剃度当日清晨沐浴 | 当日 |
| **穿白衣** | 穿着白色世俗服装最后一次 | 当日 |

### 2.2 剃度仪轨流程

| 顺序 | 环节 | 巴利语 | 内容 | 象征意义 |
|---|---|---|---|---|
| 1 | **礼佛** | *Vandana* | 至大殿礼佛三拜 | 皈依之始 |
| 2 | **请求出家** | *Pabbajjā-yācanā* | 向剃度师请求出家 | 正式请求 |
| 3 | **三皈依** | *Ti-saraṇa* | 皈依佛、法、僧 | 入门标志 |
| 4 | **受十戒** | *Dasa-sīla* | 受持沙弥十戒 | 行为规范 |
| 5 | **剃发** | *Munda-kamma* | 剃除头发 | 舍弃世俗装饰 |
| 6 | **披袈裟** | *Cīvara-dāna* | 披沙弥袈裟 | 著如来衣 |
| 7 | **授钵** | *Patta-dāna* | 授与钵盂 | 乞食器 |
| 8 | **法名授予** | *Nāma-karaṇa* | 赐予法名 | 新身份 |
| 9 | **教诲** | *Ovāda* | 剃度师开示教诲 | 修行指导 |
| 10 | **回向** | *Patti-dāna* | 回向功德 | 愿力 |

### 2.3 沙弥十戒

| 序号 | 戒条 | 巴利语 | 内容 |
|---|---|---|---|
| 1 | 离杀生 | *Pāṇātipātā veramaṇī* | 不伤害有情生命 |
| 2 | 离不与取 | *Adinnādānā veramaṇī* | 不偷盗 |
| 3 | 离非梵行 | *Abrahmacariyā veramaṇī* | 完全禁欲 |
| 4 | 离妄语 | *Musāvādā veramaṇī* | 不说谎 |
| 5 | 离饮酒 | *Surāmeraya... veramaṇī* | 不饮酒及麻醉品 |
| 6 | 离非时食 | *Vikāla-bhojanā veramaṇī* | 过午不食 |
| 7 | 离歌舞伎乐 | *Nacca-gīta... veramaṇī* | 不观听娱乐 |
| 8 | 离香花严身 | *Gandha-vilepana... veramaṇī* | 不涂抹香料、不戴花环 |
| 9 | 离高广大床 | *Uccāsayana... veramaṇī* | 坐卧简朴 |
| 10 | 离金银钱宝 | *Jātarūpa-rajata... veramaṇī* | 不持有金钱 |

### 2.4 剃度请求文

**巴利语请求文：**
```
Suṇātu me, bhante, saṅgho.
Ahaṃ, bhante, pabbajjaṃ yācāmi.
Ullumpatu maṃ, bhante, saṅgho anukampaṃ upādāya.
```

**中文翻译：**
```
尊者，愿僧团倾听我。
尊者，我请求出家。
愿僧团以慈悲摄受我。
```

## 3. 比丘具足戒仪轨 (Bhikkhu Upasampadā)

### 3.1 受具足戒前要求

| 要求 | 说明 | 最短期限 |
|---|---|---|
| **沙弥期** | 需先受沙弥戒 | 至少数年（视传承而定） |
| **年满20岁** | 按周岁计算 | — |
| **三衣钵具** | 拥有完整的比丘三衣和钵 | — |
| **师长推荐** | 剃度师同意并推荐 | — |
| **学识考核** | 能背诵基本经文和戒律 | — |
| **僧团同意** | 经僧团羯磨通过 | — |
| **界内清净** | 在清净的戒场内进行 | — |

### 3.2 具足戒仪轨流程（白四羯磨）

| 阶段 | 环节 | 巴利语 | 内容 | 次数 |
|---|---|---|---|---|
| **预备阶段** | 鸣钟召集 | *Ghāṇṭā* | 通知僧众 | — |
| | 入戒坛 | *Sīmā-okkanti* | 进入戒场 | — |
| | 求度者礼拜 | *Vandana* | 向僧团顶礼 | — |
| | 盘问遮难 | *Antarāyika-dhamma-pucchā* | 询问13种障碍 | 1次 |
| | 请求受戒 | *Upasampadā-yācanā* | 求度者请求 | 3次 |
| | 托钵验衣 | *Cīvara-paṭiggahaṇa* | 展示三衣一钵 | — |

| | 第三羯磨 | *Tatiya-kamma-vācā* | 第三次宣告 | 1次 |
| | 第四羯磨 | *Catuttha-kamma-vācā* | 最后宣告，默许通过 | 1次 |
| **完成阶段** | 授四依 | *Nissaya-dāna* | 授予四种依处 | — |
| | 开示依止师 | *Upajjhāya-ācikkhana* | 告知依止阿阇梨 | — |
| | 教诲 | *Ovāda* | 阿阇梨教诲 | — |
| | 回向 | *Patti-dāna* | 回向功德 | — |

### 3.3 白四羯磨原文

**求戒者请求文（三遍）：**
```
Suṇātu me, bhante, saṅgho.
Ahaṃ, bhante, upasampadaṃ yācāmi.
Ullumpatu maṃ, bhante, saṅgho anukampaṃ upādāya.
```

**僧团羯磨文（四遍，前三遍问有无异议，第四遍默许）：**
```
Suṇātu me, bhante, saṅgho.
Ayaṃ Itthannāmo Itthannāmassa āyasmato upasampadāpekkho.
Yadi saṅghassa pattakallaṃ, saṅgho Itthannāmaṃ upasampādeyya,
Itthannāmena upajjhāyena.
Esā ñatti.
```

### 3.4 四依（四种依止）

受戒后，新比丘被告知四种基本依止：

| 依止 | 巴利语 | 内容 | 意义 |
|---|---|---|---|
| **依树下坐** | *Rukkha-mūla-nissaya* | 以树下为住处 | 少欲知足 |
| **依穿粪扫衣** | *Paṃsukūla-cīvara-nissaya* | 以废弃布为衣 | 不拣择衣物 |
| **依乞食** | *Piṇḍiyālopanissaya* | 以乞食为生 | 知足 |
| **依陈弃药** | *Pūtimuttabhesajja-nissaya* | 以尿（发酵物）为药 | 不耽著药物 |

## 4. 短期出家仪轨 (Temporary Ordination)

### 4.1 泰国短期出家传统

| 项目 | 内容 |
|---|---|
| **传统名称** | *Buat phra* (泰语) |
| **常见年龄** | 20岁左右，通常在成年礼或结婚前 |
| **期限** | 通常3个月至数年不等 |
| **目的** | 积累功德、学习佛法、体验出家生活 |
| **社会意义** | 泰国男子普遍经历，被视为成熟标志 |

### 4.2 短期出家流程

| 阶段 | 内容 | 时长 |
|---|---|---|
| **准备阶段** | 父母同意、准备物资、学习基本礼仪 | 1-2周 |
| **剃度仪式** | 同沙弥剃度仪轨 | 1日 |
| **出家生活** | 遵守戒律、学习经典、禅修 | 3个月或更长 |
| **还俗仪式** | 向师长告假、脱下袈裟、恢复白衣 | 1日 |

### 4.3 缅甸短期出家（沙弥期）

| 项目 | 内容 |
|---|---|
| **传统名称** | *Shinbyu* (缅甸语) |
| **参与对象** | 所有缅甸男孩，通常在7-14岁 |
| **仪式规模** | 家庭盛大庆典，社区参与 |
| **期限** | 数日、数周或数月 |

## 6. 现代应用与注意事项 (Modern Considerations)

### 6.1 出家前的心理准备

| 准备维度 | 具体内容 |
|---|---|
| **动机净化** | 确认出家动机非为逃避现实、贪求供养、追求神秘体验 |
| **现实认知** | 了解出家生活的艰辛：早起、简陋、禁欲、规律 |
| **心理健康** | 排除未处理的心理创伤、精神疾病 |
| **经济安排** | 世俗事务处理完毕，无经济牵挂 |
| **人际关系** | 与家人充分沟通，获得理解支持 |
| **修行基础** | 有一定的禅修或佛法学习基础 |

### 6.2 出家与心理治疗

| 方面 | 说明 |
|---|---|
| **可能的益处** | 规律生活、正念训练、社群支持、意义感 |
| **潜在风险** | 用出家逃避问题、灵性绕道、解离加重 |
| **建议** | 重大决定前咨询有经验的师长和心理专业人士 |
| **整合视角** | 出家是选择而非逃避，需面对而非回避 |
