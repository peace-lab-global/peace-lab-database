# 文殊师利完整修持仪轨 (Manjusri / Jamyang Sadhana)

> **文殊师利 (Manjusri / Jamyang / Wenshu Shili)**：智慧本尊，诸佛之师，七佛之师。身黄色，持宝剑与般若经函，象征般若智慧能断一切烦恼。此仪轨涵盖文殊五字真言、文殊心咒、文殊法相观想及智慧开显修持。
>
> 文殊师利，大智如海；般若宝剑，斩断无明。 —— 文殊赞

## 3. 文殊心咒与五字真言 (Mantras)

### 3.1 文殊心咒

**梵音：**
```
Oṃ A Ra Pa Ca Na Dhiḥ
```

**藏音：**
```
唵 阿 Ra 巴 扎 那 德
```

**逐字含义：**
| 字 | 含义 | 净化 |
|---|---|---|
| **A (阿)** | 一切法本不生 | 净化无明 |
| **Ra (Ra)** | 一切法离垢 | 净化嗔恨 |
| **Pa (巴)** | 一切法无显 | 净化傲慢 |
| **Ca (扎)** | 一切法无说 | 净化贪欲 |
| **Na (那)** | 一切法无因缘 | 净化嫉妒 |
| **Dhiḥ (德)** | 智慧 | 净化愚痴 |

### 3.2 文殊五字真言（心咒核心）

```
阿 Ra 巴 扎 那
```

**功能：** 开启智慧、增强记忆、消除愚痴。

### 3.3 修持方法

| 方法 | 内容 |
|---|---|
| **日常持诵** | 每日108遍以上 |
| **考前修持** | 考试前持诵一万遍 |
| **配合观想** | 观想宝剑斩断愚痴 |
| **文殊剑观想** | 观想智慧宝剑从心轮发出 |
