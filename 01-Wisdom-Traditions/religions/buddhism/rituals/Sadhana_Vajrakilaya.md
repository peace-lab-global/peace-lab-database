# 普巴金刚/金刚橛完整修持仪轨 (Vajrakilaya / Dorje Phurba Sadhana)

> **普巴金刚 (Vajrakilaya / Dorje Phurba / Pubajingang)**：藏传佛教宁玛派极为重要的忿怒本尊，身持金刚橛（三棱杵），象征钉住一切烦恼与障碍。为莲花生大士主要修法之一，具强大降伏力量，能破除一切邪术、诅咒与魔障。
>
> 普巴三棱，钉住三毒；一杵直下，穿透无明。 —— 宁玛派伏藏口诀

### 3.2 心咒

**普巴金刚心咒：**
```
Oṃ Vajra Kili Kilaya Sarva Vighnam Bam Hum Phat
```

**藏音：**
```
唵 班匝 格利 格啦雅 萨尔瓦 比嘎南 班 吽 呸
```

**含义：** 皈命金刚橛，钉住一切障碍。

### 3.3 金刚橛修法

| 步骤 | 内容 |
|---|---|
| 1 | 观想金刚橛于心间 |
| 2 | 橛尖放光，刺向一切魔障 |
| 3 | 观想魔障被钉住，不能动摇 |
| 4 | 持普巴心咒 |
| 5 | 观想魔障化为光，融入空性 |
