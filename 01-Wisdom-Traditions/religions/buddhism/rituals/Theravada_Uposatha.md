# 南传佛教布萨仪轨 (Theravada Uposatha Rituals)

> **布萨 (Uposatha)**：源于古印度吠陀传统的斋日制度，佛教化后成为僧团每半月诵戒、居士受持八戒的核心仪轨，是维系僧团清净与居士修行的重要制度。
>
> *"布萨日，月之明暗相交之时，亦是修行者反省身心之日。"*

---

## 1. 布萨制度概论 (Uposatha System Overview)

| **满月布萨** | *Pannarasī* / *Pūrimā* | 农历每月十五 | 满月 | 诵戒、受八戒 | 僧俗共修 |

## 2. 僧团诵戒仪轨 (Monastic Patimokkha Recitation)

### 2.1 诵戒流程

| 顺序 | 环节 | 巴利语 | 内容 | 时长 |
|---|---|---|---|---|
| 1 | **鸣钟召集** | *Ghāṇṭā* | 通知僧众集合 | 5分钟 |
| 2 | **进入戒坛** | *Sīmā* | 进入布萨界（戒场） | 5分钟 |
| 3 | **礼佛三拜** | *Vandana* | 向佛礼敬 | 3分钟 |
| 4 | **请求诵戒** | *Pāṭimokkha-uddesa-yācanā* | 由资深比丘请求 | 2分钟 |
| 5 | **诵念戒本** | *Pāṭimokkha-bhaṇana* | 完整诵念227条戒 | 45-60分钟 |
| 6 | **忏悔环节** | *Pāpadesanā* | 有过失者当众忏悔 | 10-15分钟 |
| 7 | **清净宣布** | *Parisuddhi* | 宣布僧团清净 | 2分钟 |
| 8 | **回向功德** | *Patti-dāna* | 回向 | 3分钟 |

### 2.2 戒本结构

| 类别 | 巴利语 | 数量 | 性质 | 忏悔方式 |
|---|---|---|---|---|
| **波罗夷** | *Pārājika* | 4条 | 极重罪（性罪） | 不可忏悔，逐出僧团 |
| **僧残** | *Saṅghādisesa* | 13条 | 重罪 | 僧团羯磨忏悔 |
| **不定** | *Aniyata* | 2条 | 视情况定 | 依实情处理 |
| **舍堕** | *Nissaggiya-pācittiya* | 30条 | 财物过失 | 舍财+忏悔 |
| **单堕** | *Pācittiya* | 92条 | 轻罪 | 自责忏悔 |
| **悔过** | *Pāṭidesanīya* | 4条 | 饮食过失 | 向居士忏悔 |
| **众学** | *Sekhiya* | 75条 | 威仪规范 | 自觉改正 |
| **灭诤** | *Adhikaraṇa-samathā* | 7条 | 纠纷解决 | 依法裁决 |

## 3. 居士八戒仪轨 (Lay Eight Precepts Ritual)

### 3.1 受戒前准备

| 准备事项 | 说明 | 意义 |
|---|---|---|
| **沐浴净身** | 受戒前一日或当日清晨沐浴 | 身体清净 |
| **穿白衣** | 穿着素色或白色整洁衣服 | 象征清净 |
| **断食** | 受八戒当日上午可食，过午不食 | 预先适应 |
| **至寺院** | 前往僧团所在的寺院 | 三宝前受戒 |
| **准备供品** | 香、花、灯、食等供养 | 积累资粮 |

### 3.2 八戒受戒仪轨流程

| 顺序 | 环节 | 巴利语 | 内容 | 示范文 |
|---|---|---|---|---|
| 1 | **礼佛** | *Vandana* | 向佛三拜 | — |
| 2 | **三皈依** | *Ti-saraṇa* | 念诵三皈依 | 见日常功课 |
| 3 | **请求受戒** | *Yācanā* | 向比丘请求 | "尊者，我请求受八戒。" |
| 4 | **诵八戒** | *Aṭṭha-sīla* | 逐条传授受持 | 见下文 |
| 5 | **发愿** | *Paṇidhāna* | 发愿一日一夜清净 | "我今发愿，一日一夜清净持戒。" |
| 6 | **回向** | *Patti-dāna* | 功德回向 | 通用回向文 |

### 3.3 八戒完整仪轨文

**请求受戒文（中文）：**
```
尊者，我名某某，今于佛前、法前、僧前，
恭敬顶礼，请求受一日一夜八关斋戒。
愿尊者慈愍，传授于我。
```

**八戒传授文（巴利语逐条）：**
```
1. Pāṇātipātā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离杀生学处）

2. Adinnādānā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离不与取学处）

3. Abrahmacariyā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离非梵行学处——完全禁欲）

4. Musāvādā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离妄语学处）

5. Surāmeraya-majja-pamādaṭṭhānā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离放逸之因的诸酒类学处）

6. Vikāla-bhojanā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离非时食学处——过午不食）

7. Nacca-gīta-vādita-visūka-dassanā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离歌舞伎乐学处）

8. Uccāsayana-mahāsayanā veramaṇī-sikkhāpadaṃ samādiyāmi.
   （离高广大床学处）
```

### 3.4 八戒日修行安排

| 时段 | 活动 | 内容 |
|---|---|---|
| 清晨 (5:00-7:00) | 受戒 | 至寺院受八戒 |
| 上午 (7:00-11:00) | 听闻佛法 | 参加法会、听法师开示 |
| 中午 (11:00-12:00) | 日中一食 | 正午后不再进食 |
| 下午 (12:00-17:00) | 禅修/诵经 | 坐禅、经行、诵念经典 |
| 傍晚 (17:00-19:00) | 晚课 | 集体诵经、回向 |
| 夜间 (19:00-21:00) | 禅修 | 继续静坐或经行 |
| 睡前 (21:00) | 回向发愿 | 回向功德、发愿 |
| 次日清晨 | 舍戒 | 自然舍戒或至寺院舍戒 |

## 4. 布萨日通宵禅修 (Uposatha Night Meditation)

### 4.1 通宵禅修安排

| 时段 | 禅修方式 | 时长 | 要点 |
|---|---|---|---|
| 19:00-21:00 | 坐禅（入出息念） | 2小时 | 安定身心 |
| 21:00-22:00 | 经行 | 1小时 | 活动身体 |
| 22:00-24:00 | 坐禅（观禅） | 2小时 | 深入观察 |
| 00:00-01:00 | 休息/躺禅 | 1小时 | 右侧卧，保持正念 |
| 01:00-03:00 | 坐禅（慈心禅） | 2小时 | 培养慈悲 |
| 03:00-04:00 | 经行 | 1小时 | 清醒身体 |
| 04:00-06:00 | 坐禅（观禅） | 2小时 | 清晨定力最佳 |

### 4.2 通宵禅修注意事项

| 事项 | 说明 |
|---|---|
| **地点** | 寺院禅堂或安静处 |
| **姿势交替** | 坐禅与经行交替，避免身体僵硬 |
| **饮食** | 过午不食，可饮清水 |
| **保暖** | 夜间注意保暖，备毯子 |
| **照明** | 保持柔和灯光 |
| **伴修** | 建议集体共修，相互策励 |

### 5.2 忏悔仪轨

**个人忏悔文：**
```
尊者，我今忏悔，
从过去乃至今日，
身口意所造诸恶业，
一切我今皆忏悔。
愿罪业消灭，不复更造。
```
