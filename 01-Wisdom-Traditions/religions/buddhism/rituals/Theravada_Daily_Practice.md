# 南传佛教日常功课仪轨 (Theravada Daily Practice Rituals)

> **南传日常功课 (Dina-pūjā)**：南传上座部佛教最基本的修持仪轨，涵盖从清晨到夜晚的完整修行安排，是僧俗二众每日必修的功课体系。
>
> *"一日不作，一日不食；一日不修，一日虚度。" —— 南传丛林古训*

---

## 4. 核心诵念仪轨详解 (Core Chanting Rituals)

### 4.1 三皈依仪轨

**巴利语原文：**
```
Buddhaṃ saraṇaṃ gacchāmi.
Dhammaṃ saraṇaṃ gacchāmi.
Saṅghaṃ saraṇaṃ gacchāmi.
Dutiyampi Buddhaṃ saraṇaṃ gacchāmi.
Dutiyampi Dhammaṃ saraṇaṃ gacchāmi.
Dutiyampi Saṅghaṃ saraṇaṃ gacchāmi.
Tatiyampi Buddhaṃ saraṇaṃ gacchāmi.
Tatiyampi Dhammaṃ saraṇaṃ gacchāmi.
Tatiyampi Saṅghaṃ saraṇaṃ gacchāmi.
```

**中文翻译：**
```
我皈依佛。
我皈依法。
我皈依僧。
第二次我皈依佛。
第二次我皈依法。
第二次我皈依僧。
第三次我皈依佛。
第三次我皈依法。
第三次我皈依僧。
```

**仪轨要点：**
- 每念一句，合掌一次
- 念诵时观想三宝功德
- 三次重复表示坚定决心
- 结束后三拜顶礼

### 4.2 五戒仪轨

**巴利语原文：**
```
Pāṇātipātā veramaṇī-sikkhāpadaṃ samādiyāmi.
Adinnādānā veramaṇī-sikkhāpadaṃ samādiyāmi.
Kāmesumicchācārā veramaṇī-sikkhāpadaṃ samādiyāmi.
Musāvādā veramaṇī-sikkhāpadaṃ samādiyāmi.
Surāmeraya-majja-pamādaṭṭhānā veramaṇī-sikkhāpadaṃ samādiyāmi.
```

**中文翻译：**
```
我受持离杀生学处。
我受持离不与取学处。
我受持离欲邪行学处。
我受持离妄语学处。
我受持离放逸之因的诸酒类学处。
```

### 4.3 八关斋戒仪轨

**适用时间：** 每月四次布萨日（新月、满月及上下半月）及特别斋日

**八戒内容：**
| 序号 | 戒条 | 巴利语 | 说明 |
|---|---|---|---|
| 1 | 离杀生 | *Pāṇātipātā veramaṇī* | 包括不伤害一切有情 |
| 2 | 离不与取 | *Adinnādānā veramaṇī* | 不偷盗、不侵占 |
| 3 | 离非梵行 | *Abrahmacariyā veramaṇī* | 完全禁欲 |
| 4 | 离妄语 | *Musāvādā veramaṇī* | 不说谎、不两舌等 |
| 5 | 离谷酒花果酒 | *Surāmeraya-majja...* | 不饮酒及麻醉品 |
| 6 | 离非时食 | *Vikāla-bhojanā veramaṇī* | 过午不食（正午至次日黎明） |
| 7 | 离歌舞伎乐 | *Nacca-gīta-vādita...* | 不观听娱乐 |
| 8 | 离高广大床 | *Uccāsayana-mahāsayanā veramaṇī* | 坐卧简朴 |

**受戒仪轨流程：**
1. 先受三皈依
2. 在比丘或佛像前请求受戒
3. 比丘逐条传授
4. 受戒者逐条复述受持
5. 发愿一日一夜清净持戒
6. 回向功德

## 6. 禅修仪轨 (Meditation Rituals)

### 6.1 坐禅仪轨

**准备阶段：**
| 步骤 | 动作 | 要点 |
|---|---|---|
| 1 | 选择坐处 | 安静、干净、通风 |
| 2 | 铺设坐垫 | 垫高约10-15cm |
| 3 | 端身正坐 | 双盘、单盘或散盘 |
| 4 | 双手结定印 | 右手在上，拇指轻触 |
| 5 | 闭眼或垂帘 | 目光微内收 |
| 6 | 三次深呼吸 | 放松身心 |
| 7 | 发愿 | 为利益众生而修 |

**禅修正行：**
- 入出息念：专注鼻端呼吸
- 四念处：观身、受、心、法
- 观禅：观察生灭无常

**结行：**
- 缓慢睁开眼睛
- 搓手、按摩面部
- 缓慢起身
- 回向功德

### 6.2 经行仪轨

| 步骤 | 内容 | 要点 |
|---|---|---|
| 1 | 选择经行道 | 约20-30步距离 |
| 2 | 站立调息 | 觉知全身 |
| 3 | 缓慢起步 | 先迈右脚（或左） |
| 4 | 步步觉知 | "右、左、右、左..." |
| 5 | 至尽头停步 | 转身、继续 |
| 6 | 持续经行 | 30分钟至数小时 |

## 7. 回向仪轨 (Merit Dedication)

### 7.1 通用回向文

**巴利语：**
```
Idaṃ me puññaṃ, āsavakkhayāvahaṃ hotu.
Idaṃ me puññaṃ nibbānassa paccayo hotu.
Mama puñña-bhāgaṃ sabbasattānaṃ bhājemi;
Te sabbe me samaṃ puñña-bhāgaṃ labhantu.
Sādhu! Sādhu! Sādhu!
```

**中文：**
```
愿以此功德，导向诸漏尽。
愿以此功德，为证涅槃缘。
我此功德分，回向诸众生；
愿彼一切众，同得功德分。
善哉！善哉！善哉！
```
