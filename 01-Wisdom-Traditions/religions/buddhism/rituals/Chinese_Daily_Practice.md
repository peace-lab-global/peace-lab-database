# 汉传佛教早晚课仪轨 (Chinese Buddhist Daily Practice)

> **朝暮课诵 (Morning and Evening Service)**：汉传佛教寺院每日必修的功课，是两千年来汉地佛教修行最基本、最普遍的仪轨形式。以念诵、礼拜、回向为核心，集忏悔、发愿、持咒、念佛于一体。
>
> 朝暮不勤，道业难成；功课不辍，戒定慧增。 —— 汉地丛林规约

---

## 1. 早晚课概论 (Overview)

### 表1.1 早晚课历史源流

| 时期 | 发展 | 代表 |
|---|---|---|
| **东晋** | 课诵制度萌芽 | 慧远大师结社念佛 |
| **唐宋** | 朝暮课诵定型 | 《百丈清规》确立丛林制度 |
| **明清** | 课诵本统一 | 《禅门日诵》通行全国 |
| **近现代** | 课诵内容调整 | 增《普贤行愿品》、删部分咒语 |

### 表1.2 早晚课结构对比

| 时段 | 核心内容 | 氛围 | 功能 |
|---|---|---|---|
| **早课** | 楞严咒、大悲咒、十小咒、心经、赞佛偈 | 清净、发愿 | 开启一日修行 |
| **晚课** | 阿弥陀经、八十八佛大忏悔文、蒙山施食 | 忏悔、回向 | 收摄一日修行 |

## 2. 早课详解 (Morning Service)

### 2.1 早课完整流程（禅宗丛林传统）

| 顺序 | 内容 | 时长 | 说明 |
|---|---|---|---|
| 1 | **打板召集** | 5分钟 | 行者（晨三点半）打板通知 |
| 2 | **钟鼓齐鸣** | 5分钟 | 晨钟暮鼓，召集大众 |
| 3 | **集众排班** | 5分钟 | 僧众按戒腊排班入殿 |
| 4 | **楞严神咒** | 15-20分钟 | 早课核心，破魔护法 |
| 5 | **大悲咒** | 5分钟 | 观世音菩萨根本咒 |
| 6 | **十小咒** | 10分钟 | 十首短咒合集 |
| 7 | **心经** | 3分钟 | 般若心要 |
| 8 | **赞佛偈** | 5分钟 | 赞颂佛德 |
| 9 | **绕佛** | 10分钟 | 绕佛三匝或更多 |
| 10 | **回向发愿** | 5分钟 | 国泰民安、众生离苦 |
| 11 | **礼佛三拜** | 3分钟 | 早课圆满 |

### 2.2 楞严神咒（完整梵音）

**咒首：**
```
南无楞严会上佛菩萨（三称）
妙湛总持不动尊 首楞严王世稀有
销我亿劫颠倒想 不历僧祇获法身
愿今得果成宝王 还度如是恒沙众
...
```

**咒心（最核心部分，共五会）：**
```
第一会：
南无萨怛他 苏伽多耶 阿罗诃帝 三藐三菩陀写
（Namah sarpatah sugataya arahate samyaksambuddhaya）
...

第五会（最著名咒心）：
唵 阿那隶 毗舍提 鞞罗 跋阇罗 陀唎 盘陀盘陀你 跋阇罗谤尼泮 虎吽 都嚧瓮泮 莎婆诃
（Om anale visate virajah bhaiṣajye bhaiṣajye mahābhaiṣajya-samudgate svāhā）
```

**注：** 楞严咒全长427句，2620字，为汉传佛教最长咒语，具降伏魔外、护持正法的殊胜功德。

### 2.3 大悲咒（完整梵音）

**全称：** 《千手千眼观世音菩萨广大圆满无碍大悲心陀罗尼》

**咒首：**
```
南无喝罗怛那 哆罗夜耶
南无阿唎耶 婆卢羯帝 烁钵罗耶
菩提萨埵婆耶 摩诃萨埵婆耶
摩诃迦卢尼迦耶
唵
```

**咒心（最常用的简化版，共40句）：**
```
唵 萨皤罗罚曳 数怛那怛写
南无悉吉栗埵 伊蒙阿唎耶
婆卢吉帝 室佛罗愣驮婆
南无那罗谨墀 醯利摩诃皤哆沙咩
...
娑婆诃
```

**功能：** 灭罪除障、治病救难、满愿成就。共84句，代表观音菩萨的84种化身。

## 3. 晚课详解 (Evening Service)

### 3.1 晚课完整流程

| 顺序 | 内容 | 时长 | 说明 |
|---|---|---|---|
| 1 | **钟鼓召集** | 5分钟 | 晚钟通知 |
| 2 | **阿弥陀经** | 10分钟 | 净土根本经典 |
| 3 | **往生咒** | 3分钟 | 拔业障生净土 |
| 4 | **礼赞阿弥陀佛** | 5分钟 | 赞佛偈 |
| 5 | **八十八佛大忏悔文** | 20分钟 | 礼拜八十八佛、忏悔 |
| 6 | **蒙山施食** | 15分钟 | 施食饿鬼 |
| 7 | **回向偈** | 5分钟 | 普贤行愿回向 |
| 8 | **三皈依** | 3分钟 | 皈依佛法僧 |
| 9 | **祝韦驮** | 3分钟 | 赞韦驮菩萨护法 |

### 3.2 阿弥陀经（姚秦鸠摩罗什译本）

**核心段落：**
```
舍利弗，彼土何故名为极乐？
其国众生，无有众苦，但受诸乐，故名极乐。
...
舍利弗，若有善男子善女人，
闻说阿弥陀佛，执持名号，
若一日、若二日、若三日、若四日、若五日、若六日、若七日，
一心不乱，其人临命终时，
阿弥陀佛与诸圣众，现在其前。
是人终时，心不颠倒，即得往生阿弥陀佛极乐国土。
```

### 3.3 八十八佛大忏悔文

**五十三佛（过去佛）：**
```
南无普光佛、南无普明佛、南无普净佛、
南无多摩罗跋旃檀香佛、南无旃檀光佛、
南无摩尼幢佛、南无欢喜藏摩尼宝积佛、
...
南无一切法常满王佛
```

**三十五佛（现在佛，出自《佛说三十五佛名礼忏文》）：**
```
南无释迦牟尼佛、南无金刚坚固能摧佛、
南无宝焰佛、南无龙自在王佛、
...
南无宝华游步佛、南无宝莲华善住娑罗树王佛
```

**忏悔文核心：**
```
往昔所造诸恶业 皆由无始贪嗔痴
从身语意之所生 一切我今皆忏悔
```

### 3.4 晚课回向文

```
愿生西方净土中 九品莲花为父母
花开见佛悟无生 不退菩萨为伴侣
```

## 4. 过堂仪轨 (Meal Ceremony)

### 4.1 二时临斋仪

| 顺序 | 内容 | 说明 |
|---|---|---|
| 1 | **排班入斋堂** | 按戒腊排班 |
| 2 | **供养偈** | 粥/饭有十利/五利 |
| 3 | **食存五观** | 修行人用餐心法 |
| 4 | **静默用餐** | 不语、端身 |
| 5 | **结斋偈** | 饭后回向 |

### 4.2 食存五观

```
一、计功多少，量彼来处；
二、忖己德行，全缺应供；
三、防心离过，贪等为宗；
四、正事良药，为疗形枯；
五、为成道业，应受此食。
```
