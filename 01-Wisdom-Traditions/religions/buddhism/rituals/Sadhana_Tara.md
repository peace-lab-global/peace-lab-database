# 度母完整修持仪轨 (Tara Sadhana)

> **度母 (Tara / Sgrol Ma / Dumu)**：观世音菩萨眼泪化现的女身佛，藏传佛教最受尊崇的女性本尊。绿度母救度八难，白度母延寿增福，二十一度母各有特殊功能。此仪轨涵盖绿度母与白度母的完整修持。
>
> 顶礼救度母，度尽诸苦难；一念度母名，八难悉消除。 —— 度母赞

---

| **黑度母** | 黑蓝色 | 忿怒相 | 降伏魔障 |
| **黄度母** | 黄色 | 持宝瓶 | 财富、增益 |
| **蓝度母** | 蓝色 | 持金刚杵 | 除障、力量 |

## 3. 度母心咒修持 (Tara Mantra Sadhana)

### 3.1 绿度母心咒

**梵音/藏音：**
```
Oṃ Tāre Tuttāre Ture Svāhā
```

**汉音：**
```
唵 达咧 度达咧 度咧 梭哈
```

**含义：**
| 音节 | 含义 |
|---|---|
| **Oṃ (唵)** | 皈依三宝 |
| **Tāre (达咧)** | 救度者（度母） |
| **Tuttāre (度达咧)** | 救度一切苦难 |
| **Ture (度咧)** | 速疾救度 |
| **Svāhā (梭哈)** | 圆满成就 |

### 3.2 白度母心咒

**梵音：**
```
Oṃ Tāre Tuttāre Ture Mama Ayuḥ Punya Jñāna Puṣṭiṃ Kuru Svāhā
```

**汉音：**
```
唵 达咧 度达咧 度咧 玛玛 阿优尔 布尼亚 嘉那 布斯塔 姑如 梭哈
```

**含义：** 祈愿赐予我寿命、福德、智慧与增长。

### 3.3 二十一度母赞

**藏传常用赞颂（二十一度母礼赞经）：**
```
顶礼救度母，救度诸苦难
...
（全文共二十一首偈颂，各赞一度母）
```
