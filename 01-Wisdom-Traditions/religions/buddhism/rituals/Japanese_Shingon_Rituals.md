# 日本真言宗仪轨 (Japanese Shingon Buddhist Rituals)

> **真言宗 (Shingon-shu)**：日本八宗之一，由弘法大师空海（774-835）于唐代从中国长安青龙寺惠果阿阇梨处受法后传入日本。以「即身成佛」为最高教义，以「身口意三密相应」为核心修法。
>
> 六大无碍，四曼不离，三密相应，即身成佛。 —— 空海大师《即身成佛义》

### 3.2 光明真言

**梵字：**
```
唵 阿谟伽 尾罗左曩 摩诃 素噜娑嚩 耶 吽 怛罗 吒 含 鍐
```

**罗马转写：**
```
Oṃ Amogha Vairocana Maha Sūra Sava Ya Hum Traṭ Hāṃ Maṃ
```

**功能：** 超度亡灵、消除业障、增长光明。

### 3.3 御宝号

```
南无大师遍照金刚
（Namo Daishi Henjo Kongo）
```

**含义：** 皈依大师（空海）遍照金刚（大日如来）。

## 5. 阿字观（核心禅修） (Ajikan Meditation)

### 5.1 阿字观修法

**梵字「阿」：**
```
अ （梵文天城体）
```

**观想步骤：**
| 步骤 | 内容 |
|---|---|
| 1 | 观想心间有满月 |
| 2 | 满月上有红色「阿」字 |
| 3 | 「阿」字放红光 |
| 4 | 红光遍照全身 |
| 5 | 自身与宇宙合一 |
| 6 | 证悟「本不生」之理 |

### 5.2 「阿」字含义

| 层面 | 含义 |
|---|---|
| **发音** | 开口第一音，一切声音之本 |
| **意义** | 本不生（anutpada），一切法本来不生不灭 |
| **哲理** | 表法界体性，即大日如来的法身 |
| **修行** | 阿字即心，心即阿字 |
