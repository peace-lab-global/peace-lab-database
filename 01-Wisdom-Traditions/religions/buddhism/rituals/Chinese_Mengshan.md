# 汉传佛教蒙山施食仪轨 (Chinese Mengshan Food Offering)

> **蒙山施食 (Mengshan Offering)**：汉传佛教日常度亡施食仪轨，源于四川蒙山德异禅师（非今名山县蒙山，而是雅安县蒙山）。每日晚课中必行，是汉地寺院最普及的施食法。
>
> 每日一施，普利幽冥；一念慈悲，功不唐捐。 —— 蒙山施食仪轨缘起

---

## 1. 蒙山施食概论 (Overview)

### 表1.1 历史源流

| 时期 | 发展 | 关键人物 |
|---|---|---|
| **宋代** | 始创 | 甘露法师（四川蒙山） |
| **明代** | 普及 | 祩宏大师推崇 |
| **清代** | 定型 | 纳入《禅门日诵》 |
| **近现代** | 通行 | 所有汉地寺院晚课必修 |

### 表1.2 蒙山施食与焰口的区别

| 对比 | 蒙山施食 | 焰口 |
|---|---|---|
| **时间** | 每日晚课 | 专门法会 |
| **时长** | 15-30分钟 | 1-5小时 |
| **法师** | 一位即可 | 需金刚上师 |
| **坛场** | 简单，法座即可 | 需专门坛场 |
| **仪轨** | 简化 | 完整繁复 |
| **对象** | 一切幽冥 | 饿鬼道为主 |

### 2.2 破地狱真言

**梵音：**
```
唵 伽罗帝耶 娑婆诃
```

**功能：** 破除地狱，令受苦众生得解脱。

### 2.3 召请文

```
一心召请：
累朝帝主，历代侯王，
九卿百辟，文武官僚，
...（略）
一心召请：
姑娘姊妹，妯娌婆媳，
孤魂等众，惟愿承三宝力，
仗秘密言，此夜今时，来临法会。
```

### 2.4 普施食文

```
阿弥陀佛身金色 相好光明无等伦
白毫宛转五须弥 绀目澄清四大海
光中化佛无数亿 化菩萨众亦无边
四十八愿度众生 九品咸令登彼岸
南无西方极乐世界大慈大悲阿弥陀佛
```

### 2.5 回向偈

```
施食功德殊胜行 无边胜福皆回向
普愿沉溺诸众生 速往无量光佛刹
十方三世一切佛 一切菩萨摩诃萨
摩诃般若波罗蜜
```
