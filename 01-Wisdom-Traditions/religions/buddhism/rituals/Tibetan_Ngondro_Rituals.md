# 藏传佛教加行仪轨 (Tibetan Ngondro Rituals)

> **加行 (Ngondro)**：藏传佛教密乘修行的基础功课，意为前行、预备。包含四种或五种基础修持，是进入无上瑜伽密续前必须完成的资粮积累。传统要求各修十万遍以上。
>
> 高楼万丈平地起，加行如地基；地基不牢，大厦必倾。

---

## 2. 皈依磕大头仪轨

### 2.1 磕大头的方法

| 步骤 | 动作 | 要点 | 意义 |
|---|---|---|---|
| 1 | **站立合掌** | 双手合十于顶 | 身口意三门皈依 |
| 2 | **触额** | 双手触额 | 身业清净 |
| 3 | **触喉** | 双手触喉 | 口业清净 |
| 4 | **触心** | 双手触心 | 意业清净 |
| 5 | **五体投地** | 双膝、双手、额触地 | 五毒转为五智 |
| 6 | **起身** | 快速起立 | 精进不懈 |

### 2.2 磕大头时的念诵

**念诵内容（每磕一次）：**
```
喇嘛 千巴 美 措 吉 恭 贝 拉
桑吉 丘吉 措 吉 秋 南 拉
向却 措吉 卓瓦 卓瓦 真 拉
向却 瓦巴 吉 度 度
```

**中文：**
```
皈依上师、本尊、空行众
皈依佛、法、僧三宝
皈依一切圣众及护法
直至菩提我皈依
```

### 2.3 磕大头修行安排

| 项目 | 建议 |
|---|---|
| **每日数量** | 100-500遍 |
| **完成时间** | 3-6个月 |
| **辅助工具** | 磕大头垫、计数器、手套 |
| **注意事项** | 保护膝盖和手掌，循序渐进 |

## 3. 金刚萨埵忏悔仪轨

### 3.1 金刚萨埵观想

| 要素 | 内容 |
|---|---|
| **本体** | 金刚萨埵佛，身白色，一面二臂 |
| **右手** | 持金刚杵于心间 |
| **左手** | 持金刚铃于胯侧 |
| **坐姿** | 全跏趺坐于月轮莲台 |
| **位置** | 观想于自己顶门 |

### 3.2 百字明咒

**梵文（罗马转写）：**
```
Om Vajrasattva samayam anupalaya,
Vajrasattva tvenopa tistha,
drdho me bhava,
Sutosyo me bhava,
Suposyo me bhava,
Anurakto me bhava,
Sarva siddhim me prayaccha,
Sarva karma su ca me cittam sriyam kuru,
Hum Ha Ha Ha Ha Hoh,
Bhagavan sarva tathagata vajra ma me munca,
Vajri bhava mahasamaya sattva,
Ah
```

### 3.3 忏悔观想

1. 观想金刚萨埵于顶门
2. 念百字明时，从金刚萨埵心间流出甘露
3. 甘露流入自身，洗净一切恶业
4. 恶业化为黑水、脓血从毛孔排出
5. 下方大地裂开，业障魔被悉皆消除

| **香水海** | 内圈香水 | 清净 |
| **七金山** | 七重金山 | 七觉支 |
| **四大洲** | 东西南北洲 | 四方众生 |
| **日月** | 日轮月轮 | 方便智慧 |
| **珍宝** | 七宝、八吉祥等 | 一切妙欲 |
| **天界** | 欲界天宫殿 | 天界福报 |

### 4.2 三十七供曼茶罗

**念诵文（传统）：**
```
萨耶 布切 肖 木 秋吉 恭巴
日阿那 曼折拉 布涅 果杯 阿吽
```

**汉译：**
```
须弥山王 四大洲 日月等
观想为佛土 供养曼茶罗
```

| 步骤 | 内容 |
|---|---|
| 1 | 用布擦拭曼茶罗盘（象征净除我执） |
| 2 | 涂抹香水（象征清净） |
| 3 | 放置粮食（米、宝石等） |
| 4 | 观想须弥山、四大洲、日月等 |
| 5 | 念诵曼茶罗供养咒 |
| 6 | 观想供养给上师本尊 |
| 7 | 收摄观想 |

## 5. 上师瑜伽仪轨

### 5.1 上师瑜伽核心

| 要素 | 内容 |
|---|---|
| **皈依境** | 观想根本上师化现本尊相 |
| **祈请** | 念诵上师祈祷文 |
| **接收加持** | 观想上师心间放光融入自身 |
| **合一** | 自身与上师无二无别 |

### 5.2 莲花生大士上师瑜伽（宁玛派常用）

**七句祈祷文：**
```
吽
欧坚 耶吉 努 向 贡
巴玛 萨瓦 东波 拉
雅拉 却格 朱古 杜
巴玛 炯尼 些些 拉
沽汝 色吉 所 所 克
```

**汉译：**
```
吽
邬金刹土西北隅
莲茎花胚之座上
稀有殊胜成就者
世称名号莲花生
空行眷属众围绕
我随汝尊而修持
为赐加持祈降临
格日贝玛思德吽
```

### 5.3 接收加持观想

1. 观想上师于顶门或前方虚空中
2. 上师心间种子字放光
3. 光入自身心间
4. 自身充满加持与光明
5. 自身与上师无二
