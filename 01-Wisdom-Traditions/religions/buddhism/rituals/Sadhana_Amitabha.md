# 阿弥陀佛完整修持仪轨 (Amitabha Buddha Sadhana)

> **阿弥陀佛 (Amitabha Buddha / Wuliangshou Fo)**：西方极乐世界教主，接引念佛人往生净土。汉传佛教影响最大的本尊，净土宗立宗根本。此仪轨涵盖观想、持名、发愿、回向的完整修持。
>
> 从是西方，过十万亿佛土，有世界名曰极乐，其土有佛，号阿弥陀。 —— 《佛说阿弥陀经》

---

## 1. 阿弥陀佛概论 (Overview)

### 表1.1 阿弥陀佛三身

| 身 | 名称 | 所在 | 特征 |
|---|---|---|---|
| **法身** | 无量光佛 (Amitabha) | 法界 | 光明无量 |
| **报身** | 无量寿佛 (Amitayus) | 极乐净土 | 寿命无量 |
| **化身** | 接引佛 | 十方世界 | 接引众生 |

### 表1.2 四十八愿（核心五愿）

| 序号 | 愿名 | 内容 |
|---|---|---|
| **第十八愿** | 十念必生愿 | 至心信乐，欲生我国，乃至十念，若不生者，不取正觉 |
| **第十九愿** | 临终接引愿 | 临终时，我与圣众，现身其前 |
| **第二十愿** | 系念定生愿 | 闻我名号，系念我国，植众德本，至心回向，欲生我国 |
| **第十二愿** | 光明无量愿 | 光明无量，普照十方 |
| **第十三愿** | 寿命无量愿 | 寿命无量，国中声闻天人无数 |

## 2. 阿弥陀佛法相观想 (Visualization)

### 2.1 观经十六观（日观、水观、地观、宝树观等）

**第一观：日想观**
```
正坐西向，谛观于日，令心坚住，专想不移。
见日欲没，状如悬鼓，既见日已，闭目开目，皆令明了。
```

**第八观：阿弥陀佛观想**
```
无量寿佛，身量无边，非是凡夫心力所及。
彼佛圆光，如百亿三千大千世界。
于圆光中，有百万亿那由他恒河沙化佛。
一一化佛，亦有众多无数化菩萨，以为侍者。
```

**阿弥陀佛身相：**
| 部位 | 特征 |
|---|---|
| **身色** | 身金色 |
| **相好** | 三十二相、八十种好 |
| **白毫** | 眉间白毫，宛转如五须弥山 |
| **眼目** | 绀眼澄清，如四大海水 |
| **圆光** | 圆光化佛，无数化菩萨 |
| **手印** | 定印（弥陀印）或接引印 |
| **持物** | 有时持金莲华 |

## 3. 持名念佛仪轨 (Buddha Recitation Sadhana)

### 3.1 念佛方法

| 方法 | 说明 | 适用 |
|---|---|---|
| **出声念** | 高声或低声念 | 初学、昏沉时 |
| **金刚念** | 唇齿微动，默念 | 日常、公共场所 |
| **观想念** | 观想佛相而念 | 定力较好者 |
| **观想念** | 观佛功德而念 | 智慧型行者 |
| **追顶念** | 一句追一句，不间断 | 精进期 |
| **礼拜念** | 绕佛或拜佛而念 | 忏悔、发愿时 |

### 3.2 十念法（慈云遵式大师所传）

```
每日清晨，面西而立，合掌，连声称阿弥陀佛，尽一气为一念。
如是十气，名为十念。
```

### 3.3 持名仪轨流程

| 顺序 | 内容 | 说明 |
|---|---|---|
| 1 | **净身口意** | 洗手、漱口、静心 |
| 2 | **礼佛三拜** | 向西方顶礼 |
| 3 | **称圣号** | 南无本师释迦牟尼佛（三称） |
| 4 | **开经偈** | 无上甚深微妙法... |
| 5 | **诵《阿弥陀经》** | 或诵《无量寿经》、《观经》 |
| 6 | **诵往生咒** | 拔一切业障根本得生净土陀罗尼 |
| 7 | **赞佛偈** | 阿弥陀佛身金色... |
| 8 | **正行念佛** | 南无阿弥陀佛（数百至数万声） |
| 9 | **回向** | 愿生西方净土中... |
| 10 | **三皈依** | 皈依佛法僧 |

### 3.4 往生咒（完整梵音）

**全称：** 拔一切业障根本得生净土陀罗尼

**梵音：**
```
南无阿弥多婆夜 哆他伽多夜 哆地夜他
阿弥利都婆毗 阿弥利哆 悉耽婆毗
阿弥利哆 毗迦兰帝 阿弥利哆 毗迦兰多
伽弥腻 伽伽那 枳多迦利 娑婆诃
```

**罗马转写：**
```
Namo amitabhaya tathagataya tadyatha
amrte vyam bhe amrte
siddham bhe amrte
vikrante amrte vikranta
gamine gagana kirti kare svaha
```

## 4. 净土发愿文 (Pure Land Aspiration)

### 4.1 慈云净土发愿文

```
一心归命，极乐世界，阿弥陀佛。
愿以净光照我，慈誓摄我。
我今正念，称如来名，
为菩提道，求生净土。
佛昔本誓：若有众生，欲生我国，
至心信乐，乃至十念，若不生者，不取正觉。
...
愿我临终，预知时至，身无病苦，
心不贪恋，意不颠倒，如入禅定。
阿弥陀佛，与观音、势至，
及诸圣众，放光接引，
垂手提携，于一念顷，生极乐国。
```

### 4.2 回向偈

```
愿生西方净土中 九品莲花为父母
花开见佛悟无生 不退菩萨为伴侣
```

## 5. 临终助念仪轨 (Deathbed Recitation)

### 5.1 临终助念要点

| 要点 | 说明 |
|---|---|
| **环境** | 保持安静、庄严，设阿弥陀佛像 |
| **佛号** | 轻声、持续地念"南无阿弥陀佛" |
| **提醒** | 提醒临终者观想阿弥陀佛 |
| **避免** | 勿移动身体、勿哭泣、勿换衣 |
| **时间** | 断气后至少助念八小时 |

### 5.2 临终四句

```
阿弥陀佛 来接引你
随佛光明 往生净土
```
