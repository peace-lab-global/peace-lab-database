## 教学中的同意与安全

### 物理调整 (Hands-on Assists) 伦理

| 维度 | 最佳实践 | 不当做法 |
| :--- | :--- | :--- |
| **同意获取** | 每节课开始前询问；提供同意卡片(Yes/No翻转卡) | 未经询问直接触碰 |
| **触碰区域** | 骨骼突出点(肩膀、髋骨、膝盖外侧)为安全区 | 躯干中段、大腿内侧、颈部前方 |
| **力度** | 引导性轻触，非强制性推压 | 过度用力"推"入体式 |
| **沟通** | 先口头提示→再轻触引导→询问感受 | 无声突然调整 |
| **创伤意识** | 理解触碰可能触发创伤反应 | 忽视学生的退缩或冻结反应 |
| **替代方案** | 口头引导、演示、使用辅具 | 坚持所有学生必须接受调整 |

## 瑜伽商业基础

### 商业模式对比

| 模式 | 描述 | 优势 | 挑战 | 启动成本 |
| :--- | :--- | :--- | :--- | :--- |
| **独立工作室** | 自有/租赁空间，线下授课 | 完全控制、社区感 | 高固定成本、地域限制 | 高 |
| **健身房/机构教师** | 在健身房、企业、学校教课 | 低风险、稳定收入 | 低课时费、缺乏自主权 | 低 |
| **线上教学** | 直播/录播线上课程 | 全球可达、低成本 | 竞争激烈、缺乏个人互动 | 低-中 |
| **混合模式** | 线上+线下结合 | 灵活、多元收入 | 精力分散 | 中 |
| **静修/工作坊** | 定期举办深度活动 | 高客单价、深度体验 | 不稳定、筹备耗时 | 中-高 |
| **私人教学** | 一对一/小组 | 高课时费、深度关系 | 时间受限、难以规模化 | 低 |
| **瑜伽治疗** | C-IAYT认证临床服务 | 专业溢价、意义感 | 认证门槛高、保险问题 | 中 |

## 法律与保险

### 瑜伽教师法律责任

| 法律领域 | 关键问题 | 防范措施 |
| :--- | :--- | :--- |
| **人身伤害** | 学生因教学不当受伤 | 专业保险、免责声明、安全教学 |
| **职业资质** | 超范围执业(如声称治疗疾病) | 明确瑜伽不替代医疗 |
| **知识产权** | 序列版权、品牌商标 | 注明来源、原创内容注册 |
| **数据保护** | 学生个人健康信息 | 保密协议、安全存储 |
| **合同** | 教师-工作室、教师-学生协议 | 书面合同、明确条款 |
| **税务** | 收入申报、抵扣 | 专业财务咨询 |

### 保险类型

| 保险类型 | 覆盖范围 | 必要性 |
| :--- | :--- | :--- |
| **职业责任险** | 教学失误导致的学生伤害索赔 | 必需 |
| **公众责任险** | 教学场所内的意外事故 | 必需 |
| **财产保险** | 教学设备、辅具损失 | 建议(有工作室时) |
| **商业中断险** | 因不可抗力无法营业 | 建议(有固定场所时) |
| **网络安全险** | 线上业务数据泄露 | 建议(线上教学时) |
