## 瑜伽饮食哲学

### 三Guṇa食物分类体系

#### 悦性食物 (Sāttvika Āhāra)

| 食物类别 | 具体食物 | 品质 | 效果 |
| :--- | :--- | :--- | :--- |
| **谷物** | 大米、小麦、燕麦、小米、藜麦 | 新鲜、全谷 | 持久能量、稳定情绪 |
| **豆类** | 绿豆(Mung)、扁豆、鹰嘴豆 | 易消化 | 蛋白质、不易胀气 |
| **乳制品** | 新鲜牛奶、酥油(Ghī)、酸奶、新鲜奶酪 | 有机、少加工 | 滋养Ojas(活力精华) |
| **蔬果** | 绿叶蔬菜、当季水果、根茎类 | 新鲜、有机 | 维生素、矿物质、Prāṇa |
| **坚果/种子** | 杏仁、腰果、芝麻、亚麻籽 | 浸泡催芽更佳 | 健康脂肪、微量元素 |
| **甜味剂** | 蜂蜜(生)、枣糖、甘蔗汁 | 天然、未加工 | 温和滋补 |
| **香料** | 姜黄、姜、小茴香、胡芫荽 | 新鲜 | 助消化、抗炎 |
| **油脂** | 酥油(Ghī)、芝麻油、椰子油 | 冷压初榨 | 润滑、滋养 |

#### 变性食物 (Rājasika Āhāra)

| 食物类别 | 具体食物 | 品质 | 效果 |
| :--- | :--- | :--- | :--- |
| 刺激性 | 咖啡、浓茶、巧克力 | 兴奋神经系统 | 短暂提神后疲劳 |
| 辛辣 | 辣椒、芥末、过量黑胡椒 | 过度刺激Agni | 烦躁、失眠 |
| 盐味过重 | 加工食品、腌制品 | 水钠潴留 | 血压升高、浮肿 |
| 酸味过重 | 过量柑橘、醋 | 刺激胃酸 | Pitta失衡 |
| 速食 | 快餐、微波食品 | 方便但缺乏Prāṇa | 躁动不安 |

#### 惰性食物 (Tāmasika Āhāra)

| 食物类别 | 具体食物 | 品质 | 效果 |
| :--- | :--- | :--- | :--- |
| 肉类 | 红肉、加工肉 | 难消化、残留毒素 | 昏沉、钝重 |
| 陈腐 | 剩菜(超过数小时)、罐头 | Prāṇa流失 | 惰性增加 |
| 过度加工 | 白糖、白面粉、精制油 | 营养流失 | 炎症、血糖波动 |
| 酒精/药物 | 各类酒精、烟草 | 毒性、成瘾性 | 意识模糊 |
| 发酵过度 | 某些发酵食品 | 传统认为增加Tamas | 视情况而定 |
| 油炸 | 重复油炸食物 | 产生自由基 | 消化负担 |

### 经典饮食原则

**《薄伽梵歌》(Bhagavad Gītā) 17.8-10：**

> 悦性之人喜爱的食物增长寿命、力量、健康、幸福与满足，它们多汁、油润、稳定、令心欢喜。
> 变性之人喜爱的食物过苦、过酸、过咸、过热、过辣、过干且令人灼烧，带来痛苦、忧愁和疾病。
> 惰性之人喜爱的食物烹饪超过三小时、缺乏味道、腐败发臭、隔夜、残余、不洁。

**《哈他瑜伽灯论》(Haṭha Yoga Pradīpikā) 饮食指导：**
