## 2. 核心职业路径

### 2.1 教学路径（最常见）

```
个人练习(1-2年) → 200小时YTT → 团体课教师(兼职)
    → 300小时进阶培训 → 全职教师/私教
    → 500小时RYT → 资深教师/工作坊讲师/教师培训师
```

**适合人群**：热爱教学、有沟通天赋的修行者
**核心资质**：RYT-200 → RYT-500 → E-RYT-500 + YACEP
**关键挑战**：初期收入不稳定，需要建立口碑和学生群体

### 2.2 治疗路径

```
瑜伽教学经验(3年+) → 瑜伽治疗培训(800+小时)
    → 认证瑜伽治疗师(C-IAYT)
    → 与医疗团队合作 → 临床应用/研究
```

**适合人群**：有医学/心理学背景或对疗愈有深度兴趣的教师
**核心资质**：C-IAYT认证 + 相关临床经验
**收入潜力**：高于普通教学路径（¥500-3000/小时）

### 2.3 专项特色路径

```
基础教师资质 → 专项培训（孕产/儿童/运动员/老年/创伤知情）
    → 专项认证 → 利基市场专家
    → 专项教师培训师/内容创作者
```

**适合人群**：希望建立差异化竞争优势的教师
**核心专项**：

| 专项 | 培训时长 | 目标人群 | 市场需求 |
|------|----------|----------|----------|
| 孕产瑜伽 | 85+小时 | 孕期/产后女性 | 高（中国二孩/三孩政策后） |
| 儿童瑜伽 | 95+小时 | 3-12岁儿童 | 增长中 |
| 老年瑜伽 | 50-100小时 | 60岁以上 | 高（老龄化社会） |
| 运动员瑜伽 | 50-100小时 | 专业/业余运动员 | 中等 |
| 创伤知情瑜伽 | 100-300小时 | 创伤幸存者 | 增长中 |
| 椅子瑜伽 | 20-50小时 | 办公室/行动不便者 | 中等 |

### 2.4 商业/创业路径

```
教学经验(3-5年) → 商业培训/MBA
    → 瑜伽馆创业/合伙
    → 多店经营/加盟体系
    → 品牌化/平台化/国际化
```

**适合人群**：有商业头脑的瑜伽爱好者
**核心能力**：教学能力 + 运营管理 + 营销推广 + 财务管理

### 2.5 内容创作/数字路径

```
深入瑜伽练习 + 内容创作能力
    → 社媒账号/YouTube频道/播客
    → 在线课程平台合作
    → 独立品牌/App → KOL/意见领袖
```

**适合人群**：有内容创作天赋、镜头感好的瑜伽人
**核心资质**：深入的个人练习 + 内容创作能力 + 可信度建设
**代表案例**：Adriene Mishler（Yoga With Adriene, 1200万+YouTube订阅者）

## 3. 收入模式与定价

### 3.1 收入来源矩阵

| 收入来源 | 模式 | 可扩展性 | 收入潜力 |
|----------|------|----------|----------|
| 团体课教学 | 按节课/按月 | 低 | 低-中 |
| 1对1私教 | 时间换钱 | 低 | 中 |
| 企业培训 | B2B | 中 | 高 |
| 工作坊/密集课 | 活动式 | 中 | 中-高 |
| 在线课程 | 一次创建，持续销售 | 高 | 高 |
| 静修营 | 活动式 | 低-中 | 中-高 |

## 5. 中国市场特色

### 5.1 中国瑜伽市场概况
- **市场规模**：2025年中国瑜伽市场预计超过500亿人民币
- **练习者规模**：3000万+活跃瑜伽练习者
- **教师供给**：20万+注册瑜伽教师（含兼职）
- **主要平台**：线下工作室 + 健身App（Keep、每日瑜伽）+ 短视频（抖音、小红书）

### 5.2 中国市场独特路径
| 路径 | 特点 | 机遇 |
|------|------|------|
| **线下工作室** | 仍是主流教学场景 | 社区化、精品化趋势 |
| **企业健康** | 大型企业员工福利 | 健康中国战略推动 |
| **母婴市场** | 孕产瑜伽需求强劲 | 三孩政策+消费升级 |
| **老年健康** | 适老化瑜伽 | 银发经济+医养结合 |
| **线上教学** | 疫后加速发展 | 平台经济+社交电商 |
| **文旅融合** | 瑜伽+旅游/民宿 | 乡村振兴+健康旅游 |

### 5.3 中国认证现状
- 目前中国尚无统一的国家级瑜伽教师认证体系
- Yoga Alliance（美国）认证在中国市场仍具较高认可度
- 中国瑜伽行业协会等组织在推动本土认证标准
- 部分高校（如北京体育大学）开设了瑜伽相关学位课程
