# 冲正弘（Masahiro Oki / 沖正弘）

> 日本瑜伽之父——将瑜伽与东方身心修行融合的先驱

---

## 一、概述（Overview）

| 项目 | 信息 |
|------|------|
| 全名 | 沖正弘 / Masahiro Oki |
| 生卒 | 1921-1985 |
| 出生地 | 日本 |
| 传统 | Oki Yoga（沖瑜伽） |
| 核心贡献 | 创建日本本土化的瑜伽体系，融合东方身心修行传统 |
| 创立组织 | 沖瑜伽修道场（静岡県三島市） |

---

## 二、Oki Yoga 体系

### 2.1 核心理念

冲正弘的瑜伽体系融合了多种东方传统：

| 元素 | 来源 |
|------|------|
| 瑜伽体式与哲学 | 印度瑜伽传统 |
| 禅 | 日本禅宗 |
| 武术 | 空手道、合气道原理 |
| 东方医学 | 中医经络理论、指压（Shiatsu） |
| 断食疗法 | 自然疗法传统 |
| 呼吸法 | 印度 Pranayama + 日本呼吸法 |

### 2.2 Oki Yoga 的特色练习

- **修正体操** —— 通过特定动作纠正身体歪斜
- **内脏体操** —— 针对内脏器官的活化练习
- **断食指导** —— 系统性的断食与排毒
- **冥想** —— 禅坐与瑜伽冥想的融合
- **自然生活** —— 饮食、睡眠、运动的整合指导

---

## 三、影响与遗产

- 被尊为"日本瑜伽之父"
- 对日本替代医学和自然疗法运动有深刻影响
- Oki Yoga 在日本、西班牙和意大利有传承社区
- 他的东西融合方法为后来的跨文化瑜伽实践开辟了道路
- 证明了瑜伽可以与本土身心传统有机融合
