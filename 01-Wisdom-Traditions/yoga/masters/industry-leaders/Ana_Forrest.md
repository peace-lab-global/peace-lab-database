# 安娜·福瑞斯特（Ana Forrest）

> Forrest Yoga 的创造者——以疗愈创伤为核心的当代瑜伽先锋

---

| 项目 | 信息 |
|------|------|

| 生卒 | 1959- |
| 出生地 | 美国加州 |
| 流派 | Forrest Yoga |
| 创立时间 | 1982年起发展 |
| 师承 | Iyengar, Sivananda, 原住民疗愈传统 |
| 代表作 | 《Fierce Medicine》(2011) |
| 核心特色 | 创伤疗愈、情绪释放、强力体式、深度呼吸 |

---

## 二、Forrest Yoga 核心特征

### 2.1 四大支柱

| 支柱 | 说明 |
|------|------|
| **呼吸（Breath）** | 深层腹式呼吸是所有练习的基础 |
| **力量（Strength）** | 特别强调核心力量和上肢力量 |
| **正直（Integrity）** | 诚实面对身体和情感真相 |
| **精神（Spirit）** | 与内在智慧和灵性连接 |

### 2.2 独特教学方法

- 大量核心激活练习
- 长时间保持体式（培养内在力量）
- 课堂温度较高（但不是传统热瑜伽）
- 强调"感受而非思考"
- 直接处理情绪和创伤在身体中的表现
- 融合美洲原住民的疗愈仪式

### 2.3 个人故事的力量

Ana Forrest 公开分享了自己的创伤经历：
- 童年虐待
- 酒精和药物成瘾
- 身体损伤
- 通过瑜伽获得疗愈

这种极度的个人透明度成为她教学的核心力量，也推动了瑜伽界对创伤敏感教学的讨论。

## 三、影响与贡献（Impact）

| 影响 | 说明 |
|------|------|
| 创伤知情瑜伽 | Forrest Yoga 是最早系统性处理创伤的瑜伽流派之一 |
| 原住民文化整合 | 将美洲原住民传统融入瑜伽实践（也引发文化挪用讨论） |
| 教师培训 | 培养了数百名 Forrest Yoga 认证教师 |
| 身体真实性 | 挑战了瑜伽界的身体完美主义 |
