## 二、历史与传说（History & Legend）

### 2.1 生平传说

- 传说是 Matsyendranath 用灰烬（vibhuti）创造的孩子
- 另一传说：从牛粪堆中被 Matsyendranath 发现并收为弟子
- "Goraksha" 意为"牧牛者"（Go = 牛，Raksha = 保护）
- 据说他"拯救"了沉溺于 Kaula 密教享乐中的 Matsyendranath
- 这个故事被解读为弟子修行超越了师父

### 2.2 历史争议

- 学者对他的确切年代争议极大（从 9 世纪到 15 世纪不等）
- 可能不是单一历史人物，而是多位 Natha 大师的综合化身
- 但 Natha 教团的组织化确实在 11-12 世纪加速

## 三、核心教导（Core Teachings）

### 3.1 Hatha Yoga 的系统化

Gorakhnath 的最大贡献是将散落在密教、苦行和瑜伽传统中的身体修炼方法系统化：

| 领域 | 贡献 |
|------|------|
| 体式（Asana） | 将体式从少数冥想坐姿扩展为系统的身体练习 |
| 呼吸法（Pranayama） | 建立系统的呼吸控制方法 |
| 收束法（Bandha） | Mula, Uddiyana, Jalandhara 三大收束 |
| 洁净法（Shatkarma） | 六种净化方法的系统化 |
| 手印/身印（Mudra） | 能量控制的手势和身体姿态 |
| 脉轮（Chakra） | 精微能量体系的详细描述 |

### 3.2 六种净化法（Shatkarma）

Gorakhnath 传统特别强调身体净化：
1. **Dhauti** —— 消化道清洁
2. **Basti** —— 肠道清洁
3. **Neti** —— 鼻腔清洁
4. **Trataka** —— 凝视练习（眼和心灵净化）
5. **Nauli** —— 腹部搅动
6. **Kapalabhati** —— 头骨清明呼吸

### 3.3 身体观念

Gorakhnath 发展了一套完整的"身体哲学"：
- **Pinda**（微观身体）反映 **Brahmanda**（宏观宇宙）
- 身体是一个修行的实验室
- 通过练习可以实现"不坏之身"（Divya Deha）
- 物质身体与精神解脱并不矛盾

### 3.4 Natha 哲学核心

| 概念 | 含义 |
|------|------|
| Sahaja（自然状态） | 超越造作的本然状态 |
| Unmani（超心） | 超越心意的觉知状态 |
| Shunya（空） | 不同于佛教空义的 Natha 空性 |
| Amaraugha（不死之流） | 通过练习获得的永恒状态 |

## 五、Natha 教团（Natha Order）

### 5.1 教团结构

Gorakhnath 建立的 Natha 教团是印度历史上最重要的修行组织之一：
- 分为 12 个支派（Panth）
- 不限种姓——革命性的平等主义
- 包括苦行者（Yogi）和在家修行者
- 在北印度建立了大量寺庙和修行中心

### 5.2 Natha 修行者特征

| 标志 | 意义 |
|------|------|
| 穿耳（Kanphata） | Natha Yogi 的标志性身体改造，象征超越感官 |
| 灰烬涂身 | 象征世俗欲望的焚烧 |
| 号角（Singnad） | 宣告法音 |
| 黑羊毛座 | 修行用座 |
| 项链（Rudraksha） | 湿婆的象征 |

### 5.3 社会影响

- Natha 教团跨越种姓和宗教界限
- 影响了中世纪印度的虔信运动（Bhakti）
- 多位重要圣人（如 Kabir、Guru Nanak）受到 Natha 思想影响
- 北印度政治中也发挥了作用（Gorakhpur 城市以他命名）

## 六、传承影响（Legacy）

### 6.1 从 Gorakhnath 到现代瑜伽

```
Gorakhnath (10-12c) → Natha 教团发展
        ↓
Svatmarama (15c) → 《Hatha Yoga Pradipika》
        ↓
中世纪 Hatha Yoga 传统
        ↓
Krishnamacharya (20c) → 现代瑜伽
```

### 6.2 当代影响

- Gorakhpur 是印度最重要的 Natha 宗教中心
- Natha 传统至今仍有活跃的修行社区
- 2017 年以来，印度政界人物（如 UP 首席部长 Yogi Adityanath）与 Gorakhnath Math 的关联引发了对 Natha 传统政治化的讨论
- 国际上，Hatha Yoga 的全球传播是 Gorakhnath 遗产的最广泛体现
