### 重点体式详解

#### Tāḍāsana (山式) — 所有站立位的基础

| 方面 | 详细说明 |
| :--- | :--- |
| **足部** | 双脚并拢或髋宽；Pada Bandha(足弓三点激活)；十趾展开压地 |
| **腿部** | 股四头肌微收(不锁膝)；大腿微内旋；膝盖对齐第二趾 |
| **骨盆** | 中立位(ASIS与耻骨在同一垂直面)；尾骨微下沉 |
| **躯干** | 腹部轻度启动；肋骨不外翻；胸骨上提 |
| **肩/手臂** | 肩下沉远离耳朵；肩胛微内收下沉；手臂自然垂于两侧 |
| **头/颈** | 下巴微收；头顶上提(如悬线牵引)；目光水平前方 |
| **呼吸** | 自然深长呼吸；感受从脚底到头顶的能量线 |
| **常见错误** | 膝盖过伸(锁膝)；骨盆前倾(凸腹)；耸肩；重心偏移 |
| **治疗应用** | 姿势矫正；本体感觉训练；焦虑时的接地练习 |

#### Vīrabhadrāsana II (战士二式) — 力量与开放的典范

| 方面 | 详细说明 |
| :--- | :--- |
| **前腿** | 膝盖正对第二趾(不内扣)；大腿尽量平行地面(90°为目标) |
| **后腿** | 后脚外旋约60°；外侧脚缘压实；后腿完全伸直 |
| **骨盆** | 面向侧方开放；两侧髂骨等高 |
| **躯干** | 直立不倾斜；肋骨在骨盆正上方 |
| **手臂** | 平行地面伸展；肩下沉；手指积极延展 |
| **Dṛṣṭi** | 前手指尖(Hastāgra Dṛṣṭi) |
| **常见错误** | 前膝内扣→膝MCL压力；躯干前倾；后脚塌陷；耸肩 |
| **变体** | 反转战士(Viparīta Vīrabhadrāsana)→侧弯；绑定变体 |
