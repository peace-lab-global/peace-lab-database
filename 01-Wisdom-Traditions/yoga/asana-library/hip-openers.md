### 髋关节的六方向运动

| 运动方向 | 相关肌群 | 对应体式 | 限制因素 |
| :--- | :--- | :--- | :--- |
| **屈曲 (Flexion)** | 髂腰肌、股直肌 | 前屈类、高弓步(前腿) | 腘绳肌紧张；腰椎代偿 |
| **伸展 (Extension)** | 臀大肌、腘绳肌 | 低弓步(后腿)、战士I(后腿) | 髂腰肌紧张(久坐) |
| **外展 (Abduction)** | 臀中肌、阔筋膜张肌 | 坐角式、蛙式 | 内收肌紧张 |
| **内收 (Adduction)** | 内收肌群(长/短/大/耻骨肌) | 鹰式(腿) | 外展肌紧张 |
| **外旋 (External Rotation)** | 梨状肌、臀深六肌 | 鸽子式、莲花座、束角式 | 骨骼结构(前倾角/颈干角) |
| **内旋 (Internal Rotation)** | 臀中肌(前纤维)、阔筋膜张肌 | 牛面式(下腿)、英雄坐 | 常被忽视——现代人普遍受限 |

### 髋部与情绪的关联

| 现象 | 理论基础 | 临床观察 | 教学应对 |
| :--- | :--- | :--- | :--- |
| **开髋时哭泣** | 髂腰肌是"fight-or-flight肌肉"——存储恐惧/创伤 | 鸽子式/蛙式中常见 | 正常化；允许而不分析；提供安全空间 |
| **深度抗拒** | 骨盆区域存储生存恐惧(Mūlādhāra)和性创伤(Svādhiṣṭhāna) | 部分练习者持续回避开髋 | 不强迫；渐进式接近 |
| **科学视角** | 筋膜存储机械压力记忆(Schleip)；迷走神经与骨盆肌群的连接(Porges) | 解释可能涉及内感受和身体记忆 | 创伤知情教学(Trauma-informed) |

### 安全要点：保护膝关节

| 原则 | 说明 |
| :--- | :--- |
| **旋转来自髋** | 莲花座/鸽子式等外旋体式——旋转必须来自髋臼，绝不在膝关节扭转 |
| **膝盖疼痛=停止** | 任何膝关节的尖锐疼痛意味着立即退出体式 |
| **骨骼限制** | 髋关节的骨骼形态(前倾角/颈干角)决定了外旋的最大范围——这是不可改变的(Clark, 2016) |
| **辅具使用** | 膝下放毯子/砖块支撑——减少膝关节承受的扭力 |
