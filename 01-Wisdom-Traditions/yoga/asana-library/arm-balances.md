# Arm Balances 手臂平衡体式

| 10 | **Tolāsana** | 秤式 | Scale Pose | ★★★ | 肱三头、腹肌、髂腰肌 | 莲花座(或簡易坐) | 手掌下压→整个身体提离 |

### 手臂平衡的核心原则

| 原则 | 描述 |
| :--- | :--- |
| **手基础** | 手指展开均匀承重；指根(尤其食指根)压实——不塌入手腕 |
| **肩胛稳定** | 前锯肌启动(推离地面)——防止翼状肩胛和肩撞击 |
| **核心是真正的力量** | 手臂提供平台，核心(腹横肌+髂腰肌)提供升力 |
| **目光前方** | 视线落于前方地面(非正下方)——重心前移是起飞的关键 |
| **摔倒是学习** | 准备好安全落地方向；用柔软地面练习 |

### 腕关节保护

| 策略 | 描述 |
| :--- | :--- |
| **腕部热身** | 练习前进行腕圈、腕屈伸、前臂拉伸 |
| **手指展开** | 5指均匀展开→负荷分散(非全集中于手掌根) |
| **握拳变体** | 腕痛者可用拳头或前臂支撑替代手掌 |
| **渐进负荷** | 从板式→四柱→鹤式逐步增加承重 |
