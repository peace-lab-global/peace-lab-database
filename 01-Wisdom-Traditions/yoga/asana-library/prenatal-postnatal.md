# Prenatal & Postnatal Yoga 孕产期体式

## 孕期各阶段安全体式指南

### 第一孕期 (1-12周)

| 体式 | 安全性 | 调整 | 注意 |
| :--- | :--- | :--- | :--- |
| 站立位(战士、三角) | 安全 | 减少保持时间 | 注意头晕(血压变化) |
| 温和后弯(猫牛式) | 安全 | 不深度后弯 | 荷尔蒙变化致关节松弛 |
| 温和扭转(开放式) | 安全 | 仅开放式；避免闭合 | 保护腹部空间 |
| 倒立(倒箭式) | 安全 | 继续原有练习；不新开始 | 如已有倒立习惯可继续 |
| 剧烈Vinyāsa | 谨慎 | 降低强度；多休息 | 避免核心温度过高 |
| 强力核心(船式) | 避免 | 用侧板式替代 | 保护着床 |

### 第二孕期 (13-27周)

| 体式 | 安全性 | 调整 | 注意 |
| :--- | :--- | :--- | :--- |
| 宽距站立位 | 推荐 | 双脚宽于髋——给腹部空间 | 重心改变→注意平衡 |
| 猫牛式 | 强烈推荐 | 优秀的脊柱维护 | 缓解背痛 |
| 蹲式(Mālāsana) | 推荐 | 用砖或靠墙支撑 | 打开骨盆出口 |
| 支撑侧卧 | 推荐 | 左侧卧优先(下腔静脉保护) | 替代仰卧Śavāsana |
| 俯卧位 | 禁止 | 用猫牛式/手膝位替代 | 压迫腹部 |
| 闭合扭转 | 禁止 | 仅开放式扭转 | 压迫子宫 |

### 第三孕期 (28-40周)

| 体式 | 安全性 | 调整 | 注意 |
| :--- | :--- | :--- | :--- |
| 温和站立(靠墙) | 安全 | 使用墙壁/椅子支撑 | 平衡力显著下降 |
| 骨盆运动(猫牛+环绕) | 强烈推荐 | 在生球上做 | 准备分娩 |
| 蝴蝶式(束角) | 推荐 | 靠墙坐；膝下支撑 | 骨盆底意识 |
| 支撑婴儿式(宽膝) | 推荐 | 膝盖大幅分开 | 休息位 |
| 仰卧位(16周后) | 避免 | 左侧卧替代 | 仰卧位低血压综合征 |
| 全倒立 | 禁止 | 倒箭式(侧卧变体) | 摔倒风险 |
| 深后弯 | 禁止 | 温和猫牛式 | 腹直肌分离风险 |

### 产后恢复阶段

| 阶段 | 时间 | 推荐练习 | 避免 |
| :--- | :--- | :--- | :--- |
| **早期(0-6周)** | 产后立即 | 盆底觉察(非收缩)；深呼吸；温和行走 | 所有体式(等待医生许可) |
| **恢复期(6-12周)** | 医生许可后 | 温和猫牛；桥式(无重量)；盆底渐进激活 | 卷腹(腹直肌分离检查先)；深后弯 |
| **重建期(3-6月)** | 持续恢复 | 站立位(温和)；侧板(修正版)；完全呼吸 | 强力核心(先确认腹直肌愈合) |
| **回归期(6月+)** | 基本恢复 | 逐步回到产前练习水平 | 不急于恢复到产前强度 |

### 腹直肌分离 (Diastasis Recti) 检查与管理

| 方面 | 说明 |
| :--- | :--- |
| **检查方法** | 仰卧屈膝→微卷起头肩→手指按压肚脐上下的中线→测量间隙(>2指宽=分离) |
| **避免** | 卷腹、船式、板式(未愈合时)、所有使腹部"凸出"的动作 |
| **治疗性练习** | 腹式呼吸+轻柔腹横肌激活(呼气时"拉拢"腹壁)；侧卧起身 |
| **恢复时间** | 大多数在产后6-12个月自然恢复；严重者需物理治疗 |
