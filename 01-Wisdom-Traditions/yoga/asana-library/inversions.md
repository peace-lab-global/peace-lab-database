### 倒立的分级系统

| 级别 | 体式 | 心脏-头部关系 | 适合人群 |
| :--- | :--- | :--- | :--- |
| **微倒立** | 下犬式、宽距前屈 | 心脏略高于头部 | 所有人(除绝对禁忌) |
| **半倒立** | 倒箭式(靠墙)、支撑桥式 | 心脏明显高于头部(被动) | 大多数人包括高血压(受控) |
| **全倒立(冷却)** | 肩倒立、犁式 | 完全倒置(副交感主导) | 中级以上；无颈椎问题 |
| **全倒立(加热)** | 头倒立、前臂倒立、手倒立 | 完全倒置(交感参与) | 高级；无绝对禁忌 |

### 倒立安全核心原则

| 原则 | 说明 | 关键提示 |
| :--- | :--- | :--- |
| **颈椎保护** | 肩倒立必须用毯子垫高肩膀(2-3层折叠毯) | 头部不承重——是肩膀和上臂承重 |
| **核心先行** | 上下倒立都通过核心控制，不靠惯性踢腿 | 先掌握半倒立→墙辅助→独立 |
| **出入控制** | 能控制地上去=能控制地下来 | 如果只能踢上去而不能控制下来，说明还没准备好 |
| **时长渐进** | 从30秒开始→渐增至3-5分钟 | 下来后休息等量时间(尤其头倒立) |
| **经期争议** | 传统禁止经期倒立(能量逆转)；现代观点分歧 | 尊重个人选择；提供信息而非强制 |
