# 瑜伽纪录片完整指南 | Yoga Documentary Guide

> 精选瑜伽相关优质纪录片，涵盖瑜伽历史传承、当代流派、大师人物传记、练习科学与争议反思，从多维视角呈现瑜伽作为身心灵科学体系的全貌。

## 一、瑜伽历史与传承 (Yoga History & Heritage)

### 1. 《诸神的呼吸》/ Breath of the Gods

| 字段 | 信息 |
|------|------|
| **英文名** | Breath of the Gods: A Journey to the Origins of Modern Yoga |
| **年份** | 2012 |
| **导演** | Jan Schmidt-Garre |
| **时长** | 105分钟 |
| **类型** | 历史传记纪录片 |

**核心内容**：
- 追溯**现代瑜伽之父 T. Krishnamacharya**（克里希那马查亚）的生平与教学
- 其三大弟子如何将瑜伽带向世界：
  - **B.K.S. Iyengar** → 艾扬格瑜伽
  - **Pattabhi Jois** → 阿斯汤加瑜伽
  - **T.K.V. Desikachar** → 维尼瑜伽
- 迈索尔宫廷瑜伽传统的历史还原
- 瑜伽从印度本土实践到全球运动的传播路径

**疗愈价值**：
- 理解当代主流瑜伽流派的共同根源
- 展现瑜伽作为活态传承的历史脉络
- 对"瑜伽只是体操"的偏见的深度纠正

> **关联知识**：[克里希那马查亚](../masters/modern-pioneers/Krishnamacharya.md) · [B.K.S. 艾扬格](../masters/contemporary/BKS_Iyengar.md) · [帕塔比·乔伊斯](../masters/contemporary/Pattabhi_Jois.md) · [瑜伽历史与传统](../philosophy-history/Yoga_History_Tradition.md)

### 2. 《瑜伽揭秘》/ Yoga Unveiled

| 字段 | 信息 |
|------|------|
| **英文名** | Yoga Unveiled |
| **年份** | 2004 |
| **导演** | Gita Desai |
| **时长** | 两部（各约60分钟） |
| **类型** | 文化历史纪录片 |

**核心内容**：
- **第一部**：瑜伽的古代起源与哲学基础
  - 从吠陀时代到帕坦伽利《瑜伽经》
  - 瑜伽八支体系的历史形成
  - 印度文化背景中的瑜伽位置
- **第二部**：瑜伽的现代发展与全球传播
  - 辨喜(Vivekananda)将瑜伽带到西方
  - 20世纪瑜伽大师的贡献
  - 当代瑜伽产业的形成

**适合人群**：想系统了解瑜伽5000年历史的学习者

> **关联知识**：[瑜伽历史与传统](../philosophy-history/Yoga_History_Tradition.md) · [瑜伽哲学](../philosophy-history/Yoga_Philosophy.md) · [帕坦伽利](../masters/ancient/Patanjali.md) · [辨喜](../masters/pre-modern/Vivekananda.md)

## 二、瑜伽大师人物传记 (Yoga Master Biographies)

### 3. 《觉醒：尤迦南达的一生》/ Awake: The Life of Yogananda

| 字段 | 信息 |
|------|------|
| **英文名** | Awake: The Life of Yogananda |
| **年份** | 2014 |
| **导演** | Paola di Florio, Lisa Leeman |
| **时长** | 87分钟 |
| **类型** | 人物传记纪录片 |
| **获奖** | 多个国际电影节奖项 |

**核心内容**：
- 帕拉宏撒·尤迦南达(Paramahansa Yogananda)的完整生平
- 1920年代将瑜伽与冥想带到美国的开创之旅
- 其经典著作《一个瑜伽行者的自传》(Autobiography of a Yogi)的影响
- Kriya Yoga（克利亚瑜伽）的传承与实践
- 乔布斯（Steve Jobs）葬礼上每人一册此书的故事
- 东西方灵性对话的先驱角色

**疗愈价值**：
- 展现瑜伽作为完整灵性道路（而非仅仅是体式练习）的深度
- 探讨东方智慧与西方科学的对话可能性
- 对自我实现(Self-Realization)概念的深入探索

> **关联知识**：[尤迦南达](../masters/modern-pioneers/Yogananda.md) · [瑜伽哲学](../philosophy-history/Yoga_Philosophy.md) · [昆达里尼与高级修炼](../meditation-consciousness/Yoga_Advanced_Kriya_Kundalini.md)

## 三、瑜伽科学与疗愈 (Yoga Science & Healing)

### 5. 《论瑜伽：和平的建筑》/ On Yoga: The Architecture of Peace

| 字段 | 信息 |
|------|------|
| **英文名** | On Yoga: The Architecture of Peace |
| **年份** | 2017 |
| **导演** | Heitor Dhalia |
| **时长** | 104分钟 |
| **类型** | 文化探索纪录片 |

**核心内容**：
- 基于摄影师Michael O'Neill的同名影集
- 走访印度、纽约等地的瑜伽大师与修行者
- 瑜伽如何在不同文化中被实践和理解
- 体式之美与灵性深度的视觉呈现
- 瑜伽作为"和平建筑"的哲学隐喻

**视听特色**：
- 摄影极其精美，体式拍摄堪称艺术
- 印度圣地的壮丽景观
- 将瑜伽的美学与精神性完美融合

> **关联知识**：[瑜伽哲学](../philosophy-history/Yoga_Philosophy.md) · [瑜伽历史与传统](../philosophy-history/Yoga_History_Tradition.md)

### 6. 《瑜伽的科学》/ The Science of Yoga

| 字段 | 信息 |
|------|------|
| **类型** | 科学纪录片 / 教学系列 |
| **常见平台** | 各大纪录片平台 |

**核心内容**：
- 瑜伽对身体各系统的科学影响

- 瑜伽与慢性疼痛管理的循证研究
- 瑜伽对心理健康的影响机制
- 现代科学如何验证古老实践

## 四、当代瑜伽现象与反思 (Contemporary Yoga & Critical Reflection)

### 7. 《Bikram：瑜伽士、大师、掠食者》

| 字段 | 信息 |
|------|------|
| **英文名** | Bikram: Yogi, Guru, Predator |
| **年份** | 2019 |
| **平台** | Netflix |
| **导演** | Eva Orner |
| **时长** | 86分钟 |
| **类型** | 调查性纪录片 |

**核心内容**：
- Bikram Choudhury创立热瑜伽帝国的过程
- 多名受害者关于性侵和权力滥用的证词
- 培训中的邪教式控制手段
- 瑜伽体式版权争议与法律判决
- 行业对灵性领袖权力监督的缺失

**反思价值**：

| 议题 | 启示 |
|------|------|
| 权力与滥用 | 灵性权威如何转化为控制与剥削 |
| 师生界限 | 瑜伽教学中的伦理边界 |
| 品牌去Bikram化 | 大量工作室更名为"热瑜伽"或"26&2" |
| 行业改革 | 推动更透明的培训环境和学员保护机制 |

> ⚠️ **注意**：此片涉及性侵话题，观看时注意自我照顾

> **关联知识**：[Bikram Choudhury](../masters/industry-leaders/Bikram_Choudhury.md) · [瑜伽伦理与专业发展](../modern-professional/Yoga_Ethics_Business_Professional.md) · [瑜伽安全](../safety/)
