# 拉伸与康复动作 (Stretch & Rehabilitation Exercises)

## 概述

本文档适用于**急性期（48小时）后**的康复训练，提供系统的拉伸和强化动作序列，帮助恢复颈部功能、缓解肌肉紧张。

### 开始训练的前提条件

**在开始以下训练前，请确认**：

```
✓ 急性期已过（发病超过48小时）
✓ 疼痛明显减轻（评分下降50%以上）
✓ 颈部活动度有所改善
✓ 无红旗症状（手臂麻木、放射痛等）
✗ 如有疑问，请先阅读 [医疗转诊指南](Neck_Pain_Medical_Referral_Guide.md)
```

### 训练原则

```
渐进性原则：
阶段一（第3-5天）：温和激活 → 以等长收缩为主
阶段二（第5-7天）：逐步拉伸 → 增加活动范围
阶段三（第7天后）：功能强化 → 加入抗阻训练

安全性原则：
• 所有动作应在无痛或轻微不适范围内进行
• 疼痛评分控制在 3-4/10 以内
• 如疼痛加剧，立即停止
• 动作缓慢、平滑，避免突然发力
• 保持正常呼吸，不憋气
```

## 阶段一：温和激活（第3-5天）

### 动作1：颈部等长收缩训练

**这是急性期后首选的康复动作**，可安全激活颈部肌肉而不增加关节压力。

#### 科学原理

**什么是等长收缩？**

等长收缩是指肌肉收缩时长度不变，关节角度不变的收缩方式。例如：用手推墙，肌肉在用力但手臂位置不变。

**为什么适合康复期？**

1. **安全性高**：不产生关节运动，不增加关节压力
2. **激活稳定肌**：可选择性激活深层稳定肌群
3. **改善血液循环**：肌肉收缩促进局部血液流动
4. **重建神经控制**：改善神经肌肉协调性

#### 目标肌群

- 深层颈屈肌（颈长肌、颈长头肌）
- 颈部伸肌
- 颈侧屈肌群

#### 详细执行方法

**前侧等长收缩**（激活深层颈屈肌）

```
准备：
1. 坐直或站立，目视前方
2. 双脚平放地面（坐姿）或与肩同宽（站姿）
3. 肩膀放松下沉
4. 胸部挺起，但不过度

动作：
1. 右手掌心轻放在前额
2. 手的位置：手掌根部在眉毛上方，手指向上
3. 头部尝试向前点（下巴向前），手给予轻微阻力
4. 关键：头部不实际移动
5. 感受颈部前方深层肌肉收紧
6. 保持5秒
7. 缓慢放松3秒
8. 重复5次

要点：
• 收缩强度约30-40%最大力量
• 仅感受肌肉收缩，不追求强烈疲劳
• 保持正常呼吸，不要憋气
• 颈后不应感到压力增加
```

**后侧等长收缩**（激活颈部伸肌）

```
准备：
1. 同上坐姿或站姿
2. 双手交叉置于后脑勺（枕骨区域）
3. 手掌贴住头骨底部

动作：
1. 头部尝试向后仰，手给予阻力
2. 头部不动，仅感受颈后肌肉工作
3. 保持5秒
4. 缓慢放松3秒
5. 重复5次

要点：
• 不要用力过猛
• 颈后应有温和的收缩感，不紧张
• 肩膀保持下沉，不耸肩
```

**侧面等长收缩**（激活颈侧屈肌群）

```
针对左侧疼痛的版本：

右侧等长收缩（先做健侧）：
1. 左手掌轻放在头部右侧（耳上）
2. 头部尝试向右侧倾，手给予阻力
3. 头部不动
4. 保持5秒
5. 放松3秒
6. 重复5次

左侧等长收缩（后做患侧）：
1. 右手掌轻放在头部左侧
2. 头部尝试向左侧倾，手给予阻力
3. 强度更温和（约20-30%）
4. 保持5秒
5. 放松3秒
6. 重复5次

要点：
• 先做健侧，再做患侧
• 患侧收缩强度更温和
• 如疼痛明显，减少收缩强度或次数
```

#### 训练参数

| 参数 | 建议 |
|------|------|
| 保持时间 | 5秒（初期）→ 8秒（后期） |
| 放松时间 | 3秒 |
| 重复次数 | 每方向5次（初期）→ 10次（后期） |
| 每日频率 | 3-4次 |
| 总时长 | 约5-8分钟/次 |

#### 注意事项

```
✓ 动作缓慢、平稳
✓ 保持正常呼吸
✓ 仅感受肌肉收缩，不追求强烈疲劳
✓ 如某个方向疼痛加剧，减少强度或跳过

✗ 不要突然发力
✗ 不要憋气
✗ 不要过度用力（应能轻松保持）
✗ 颈椎间盘突出者谨慎进行后侧收缩
```

### 动作2：下巴回缩运动

**这是矫正前头姿势、激活深层颈屈肌的核心动作**。

#### 科学原理

**什么是下巴回缩？**

下巴回缩（Chin Tuck/Chin Retraction）是将下巴水平向后缩的动作，类似于形成"双下巴"。

**为什么要做？**

1. **激活深层颈屈肌**：颈长肌、颈长头肌是维持颈椎稳定的关键
2. **拉长枕骨下肌群**：缓解后头部紧张
3. **矫正前头姿势**：改善头部前移的不良姿势
4. **减轻颈椎压力**：头部回到中立位，减少后侧肌群负担

#### 目标肌群

- 深层颈屈肌（颈长肌、颈长头肌）
- 枕骨下肌群（被拉长）
- 胸锁乳突肌（被抑制）

#### 详细执行方法

**基础版本**

```
准备：
1. 坐直或站立，目视前方
2. 肩膀放松下沉
3. 找一个参考点：正前方的物体或镜子中的自己

动作：
1. 保持头部水平（不低头或抬头）
   - 想象头顶有一根线向上牵引
   - 眼睛保持看正前方
   
2. 想象有人从后方轻推你的下巴
   - 将下巴水平向后缩
   - 形成"双下巴"的姿态
   
3. 感受变化：
   - 颈部后方拉长
   - 颈部前方深层肌肉收紧
   - 后头部可能有轻微拉伸感
   
4. 保持3-5秒
   
5. 缓慢放松，回到起始位置
   
6. 重复10-15次

关键：
• 动作应水平进行，不是低头
• 下巴向后缩，不是向下
• 头顶保持向上顶的感觉
```

**进阶版本：加阻力**

```
准备：
1. 同上坐姿或站姿

动作：
1. 将食指或两根手指轻放在下巴上
2. 手指方向向上
3. 进行下巴回缩，同时手指提供轻微阻力
4. 下巴推手指向后
5. 保持5秒
6. 放松
7. 重复10次

要点：
• 手指阻力很轻微（约10-20%力量）
• 主要目的是增加本体感觉反馈
• 不是真正对抗手指力量
```

**动态版本**

```
准备：
1. 同上坐姿或站姿

动作：
1. 从头部轻微前移的位置开始
   - 很多人日常就是这种姿势
   
2. 缓慢将下巴回缩至中立位
   - 用时约2秒
   
3. 保持2秒
   
4. 缓慢放松至起始位置
   - 用时约2秒
   
5. 重复15次

节奏：2秒回缩 → 2秒保持 → 2秒放松
```

#### 训练参数

| 参数 | 建议 |
|------|------|
| 保持时间 | 3-5秒（基础）→ 5-8秒（进阶） |
| 重复次数 | 10-15次 |
| 每日频率 | 每小时1次 |
| 总时长 | 约2分钟/次 |

#### 视觉化提示

```
想象方式：
• "让你的耳朵远离肩膀"
• "延长颈部后侧"
• "头顶有线向上牵引，下巴水平后移"
• "把后脑勺向后墙靠近"
• "想有一本书在头顶，不要掉下来"

对镜练习：
• 面对镜子练习，观察下巴水平移动
• 正确：下巴前后移动，不下移
• 错误：下巴向下低头
```

#### 常见错误与纠正

| 错误 | 表现 | 纠正方法 |
|------|------|---------|
| 低头 | 下巴向下移动 | 保持眼睛看正前方；想象头顶向上顶 |
| 耸肩 | 肩膀抬起 | 先主动沉肩，再进行下巴回缩 |
| 过度后缩 | 颈后压力增加 | 减小幅度，仅需达到中立位 |
| 憋气 | 呼吸停止 | 保持正常呼吸，边做边数秒 |

### 动作3：肩胛骨回缩运动

**激活菱形肌和中下斜方肌，改善圆肩**。

#### 科学原理

**为什么落枕后要练肩胛骨？**

1. **肌筋膜链连续性**：颈部和肩胛骨通过斜方肌、肩胛提肌相连
2. **代偿模式**：颈部问题常伴随肩胛骨位置异常
3. **整体矫正**：改善圆肩可间接改善颈部姿势

#### 目标肌群

- 菱形肌（大、小菱形肌）
- 中斜方肌
- 下斜方肌
- 前锯肌

#### 详细执行方法

**基础回缩**

```
准备：
1. 坐直，双脚平放地面
2. 双手自然垂于身体两侧
3. 或轻放大腿上
4. 肩膀放松

动作：
1. 想象肩胛骨是两扇门，要向后关闭
2. 将肩胛骨向脊柱中线靠拢（内收）
3. 同时肩膀下沉（下压）
4. 保持胸部挺起，不要过度弓背
5. 感受背部中央肌肉收紧
6. 保持5秒
7. 缓慢放松
8. 重复10次

要点：
• 肩胛骨向后向内移动
• 不是夹紧到疼痛
• 胸部自然挺起，不是挺肚子
• 颈部保持中立，不过度后仰
```

**进阶版本：W-Pull**

```
准备：
1. 坐直
2. 双手抬至肩膀高度
3. 肘关节屈曲90°
4. 前臂垂直向上，如"投降"姿势

动作：
1. 保持肘关节角度（90°）
2. 将双肘向后拉
3. 肩胛骨用力内收
4. 挤压背部中央
5. 形成"W"字形
6. 保持5秒
7. 缓慢放松
8. 重复10-15次

要点：
• 肘部高度不变
• 肩胛骨主导运动
• 感受肩胛骨之间收紧
```

**动态版本**

```
准备：
1. 坐直，双手垂于体侧

动作：
1. 从肩膀前伸位置开始（圆肩姿势）
2. 缓慢回缩至肩胛骨内收位
   - 用时约1秒
3. 最大内收位保持2秒
4. 缓慢放松至起始位置
   - 用时约2秒
5. 重复15次

节奏：1秒回缩 → 2秒保持 → 2秒放松
```

#### 提示语

```
• "把肩膀装进后裤口袋"
• "挤压肩胛骨之间的铅笔"
• "骄傲地挺起胸膛"
• "肩膀向后向下"
• "打开胸廓"
```

#### 训练参数

| 参数 | 建议 |
|------|------|
| 保持时间 | 5-10秒（静态） |
| 重复次数 | 10次（静态）或 15次（动态） |
| 每日频率 | 3-4次 |
| 总时长 | 约3-5分钟/次 |

## 阶段二：逐步拉伸（第5-7天）

### 动作4：上斜方肌拉伸

**这是缓解颈肩交界处紧张的关键拉伸**。

#### 解剖学基础

**上斜方肌位置**：
- 起点：枕骨外隆凸、项韧带
- 止点：锁骨外侧1/3
- 功能：肩胛骨上提、上旋

**为什么要拉伸？**

落枕和长期伏案工作常导致上斜方肌过度紧张，形成"铁肩膀"。

#### 详细执行方法

**基础拉伸**

```
准备：
1. 坐在椅子上，双脚平放地面
2. 背部挺直，目视前方

固定肩膀（关键步骤）：
- 右手从背后绕过，握住座椅右侧底部
- 这固定了右侧肩膀，防止拉伸时耸肩
- 也可以坐在右手上

拉伸动作：
1. 左手越过头顶
2. 轻放在头部右侧（耳上区域）
3. 将头部缓慢向左侧倾斜
4. 同时下巴稍向左前方旋转约30°
   - 旋转是为了对准肌肉纤维方向
5. 感受右侧颈肩交界处的温和拉伸

进阶：
- 右侧肩膀主动下沉
- 想象肩胛骨向下滑动
- 增加拉伸强度

保持：
- 保持30秒
- 保持呼吸平稳
- 在呼气时可稍加深拉伸

结束：
- 缓慢回到中立位
- 换另一侧

重复：
- 左右各2-3次
```

**拉伸感觉定位**

```
正确的感觉：
• 右侧颈肩交界处有"舒适的拉长感"
• 拉伸感沿着颈部侧面延伸
• 评分约 3-4/10 的拉伸感

错误的感觉（需调整）：
• 锐痛或刺痛 → 减小幅度或停止
• 肩膀前侧疼痛 → 可能压迫神经，调整角度
• 手臂麻木 → 立即停止，可能神经受压
```

#### 训练参数

| 参数 | 建议 |
|------|------|
| 保持时间 | 30秒（初期）→ 45秒（后期） |
| 重复次数 | 每侧2-3次 |
| 每日频率 | 2-3次 |
| 总时长 | 约3-5分钟/次 |

#### 注意事项

```
✓ 拉伸应缓慢进入，避免快速牵拉
✓ 感受"舒适的拉长感"，不疼痛
✓ 保持肩膀下沉，避免耸肩
✓ 呼吸平稳，可在呼气时加深拉伸
✓ 手仅引导，不过度用力拉

✗ 不要快速弹震式拉伸
✗ 不要拉到疼痛（>5/10）
✗ 不要耸起被拉伸侧的肩膀
✗ 不要憋气
```

### 动作5：胸大肌拉伸

**改善胸前肌肉紧张，间接缓解颈部压力**。

#### 解剖学基础

**胸大肌位置**：
- 起点：锁骨内侧半、胸骨、第1-6肋软骨
- 止点：肱骨大结节嵴
- 功能：肩关节内收、内旋

**与颈肩问题的关系**：

胸大肌缩短导致肩膀前倾（圆肩），进而影响颈部姿势，形成上交叉综合征。

#### 详细执行方法

**座椅版本**（办公室可行）

```
准备：
1. 坐在椅子前端
2. 双手向后扶住座椅后侧边缘
3. 手臂伸直，手指朝下

动作：
1. 双手固定位置，身体缓慢向前移动
2. 挺起胸部
3. 肩胛骨向后夹紧
4. 感受胸前拉伸感
5. 保持20-30秒
6. 缓慢回到起始位置
7. 重复2-3次

要点：
• 手臂位置不变
• 胸部主动挺起
• 感受胸前拉伸，不疼痛
```

**门框版本**（家中或休息时）

```
准备：
1. 站立于门框内
2. 选择左手或右手开始

动作：
1. 左手抬至肩膀高度
2. 前臂靠在门框左侧
3. 保持肘关节90°
4. 右脚向前迈一步
5. 身体重心缓慢前移，穿过门框
6. 感受左侧胸前温和拉伸
7. 保持30秒
8. 换右侧重复

变化：
- 高位拉伸（针对锁骨部）：手抬高至头顶高度
- 低位拉伸（针对胸肋下部）：手降低至腰部高度
```

#### 训练参数

| 参数 | 建议 |
|------|------|
| 保持时间 | 20-30秒（座椅版）或 30-60秒（门框版） |
| 重复次数 | 2-3次 |
| 每日频率 | 2-3次 |

### 动作6：颈部时钟运动

**恢复颈椎全方位活动度**。

#### 科学原理

颈部时钟运动综合了颈椎的多维度活动：
- 屈曲-伸展
- 侧屈
- 旋转
- 圆周运动

这种多维活动可促进关节液循环、减少关节僵硬。

#### 详细执行方法

```
想象概念：
- 想象面前有一个巨大的时钟
- 12点在正上方
- 6点在正下方
- 3点在右侧
- 9点在左侧

执行步骤：
1. 坐直，目视前方（起始位置）

2. 缓慢将下巴向胸部方向点
   - 向6点方向移动
   - 到达舒适的终末位

3. 缓慢向右侧移动
   - 经过5点、4点位置
   - 到达3点位置

4. 继续向右上方移动
   - 经过2点位置
   - 向上延伸到右上方
   - 接近12点方向

5. 跨越正上方
   - 从右侧12点附近
   - 向左上方移动

6. 继续向左侧移动
   - 经过10点、9点位置

7. 向左下方移动
   - 经过8点、7点位置

8. 回到6点位置
   - 完成一圈

9. 反方向重复
   - 从6点 → 7点 → 8点 → ... → 5点 → 6点

要点：
• 动作缓慢、平滑，如同缓慢画圈
• 每个"小时"位置可停留1-2秒
• 活动范围由小到大，逐渐增加
• 保持肩膀放松，仅颈部活动
```

#### 训练参数

| 参数 | 建议 |
|------|------|
| 单圈时间 | 30-40秒 |
| 方向 | 顺时针2圈，逆时针2圈 |
| 每日频率 | 2-3次 |
| 总时长 | 约3-5分钟/次 |

#### 注意事项

```
✓ 动作极慢，不要快速转动
✓ 保持肩膀放松
✓ 活动范围循序渐进

✗ 如某个方向出现疼痛或眩晕：
   - 减小活动范围
   - 或跳过该角度
   - 或停止该运动

✗ 不要快速转动，避免颈动脉压迫
✗ 老年人或有颈椎问题者谨慎进行
```

## 阶段三：功能强化（第7天后）

### 动作7：YTWL肩胛稳定训练

**这是肩胛骨稳定训练的经典序列，常用于肩颈康复**。

#### 目标肌群

| 动作 | 主要肌群 | 次要肌群 |
|------|---------|---------|
| Y字 | 下斜方肌、前锯肌 | 菱形肌 |
| T字 | 中斜方肌、菱形肌 | 后三角肌 |
| W字 | 菱形肌、中下斜方肌 | 肩袖肌群 |
| L字 | 冈下肌、小圆肌 | 肩胛下肌 |

#### 详细执行方法

**Y字形**

```
准备：
1. 俯卧在床或瑜伽垫上
2. 额头轻触垫子或转向一侧
3. 双臂自然下垂

动作：
1. 双臂向前上方伸展
2. 与躯干呈约135°角
3. 形成"Y"字形
4. 拇指向上（旋后位）
5. 肩胛骨后缩
6. 将手臂离开地面5-10厘米
7. 保持3-5秒
8. 缓慢放下
9. 重复10-15次

要点：
• 由肩胛骨带动，不是手臂用力
• 感受肩胛骨向脊柱靠拢
• 颈部放松，不抬起头
```

**T字形**

```
准备：
1. 同上俯卧位

动作：
1. 双臂向两侧伸展
2. 与躯干成90°
3. 形成"T"字形
4. 拇指向上
5. 肩胛骨内收
6. 抬起手臂
7. 保持3-5秒
8. 缓慢放下
9. 重复10-15次

要点：
• 肩胛骨向脊柱中线挤压
• 手臂高度不超过肩膀
```

**W字形**

```
准备：
1. 同上俯卧位

动作：
1. 肘关节屈曲90°
2. 上臂外展，与躯干约60-90°
3. 形成"W"字形
4. 肩胛骨用力内收
5. 抬起肘部和手部
6. 保持3-5秒
7. 缓慢放下
8. 重复10-15次

要点：
• 这是肩胛骨内收感最强的动作
• 想象挤压肩胛骨之间的铅笔
```

**L字形**

```
准备：
1. 同上俯卧位

动作：
1. 上臂贴近身体
2. 肘关节90°屈曲
3. 前臂垂直于地面（起始位）
4. 保持肘部位置不变
5. 前臂向外旋转
   - 如同打开门的动作
6. 保持3-5秒
7. 缓慢回到起始位
8. 重复10-15次

要点：
• 肘部位置固定
• 仅前臂旋转
• 感受肩关节后外侧工作
```

#### 办公室坐姿版本

```
坐姿T字强化：
1. 坐在椅子前端，身体前倾30°
2. 双臂自然下垂
3. 手持轻哑铃（1-2公斤）或水瓶
4. 保持肘部微屈
5. 将手臂向两侧抬起
6. 肩胛骨内收，挤压背部中央
7. 抬至肩膀高度
8. 2秒抬起，3秒放下
9. 重复12-15次

坐姿W字强化：
1. 坐直
2. 双臂抬至肩膀高度
3. 肘部屈曲90°
4. 将肘部向后拉，形成W字
5. 最大程度挤压肩胛骨
6. 保持2秒，缓慢放松
7. 重复15-20次
```

#### 训练参数

| 参数 | 建议 |
|------|------|
| 每个动作次数 | 10-15次（初期）→ 15-20次（后期） |
| 保持时间 | 3-5秒 |
| 组数 | 1-2组（初期）→ 3组（后期） |
| 每日频率 | 1次（初期）→ 1-2次（后期） |
| 总时长 | 约10-15分钟/次 |

## 完整训练序列

### 5分钟快速序列（每日多次）

```
1. 颈部等长收缩（前-后-侧）     约1.5分钟
   - 每方向5次，每次5秒

2. 下巴回缩                     约1分钟
   - 15次

3. 肩胛骨回缩                   约1分钟
   - 10次静态，每次5秒

4. 上斜方肌拉伸                 约1.5分钟
   - 左右各2次，每次30秒
```

### 15分钟完整序列（每日1-2次）

```
热身（2分钟）：
1. 颈部时钟运动：左右各1圈，约1分钟
2. 温和的肩部绕圈：约1分钟

激活（5分钟）：
1. 颈部等长收缩：每方向10次，约2分钟
2. 下巴回缩：20次，约2分钟
3. 肩胛骨回缩：15次动态，约1分钟

拉伸（5分钟）：
1. 上斜方肌拉伸：左右各2次，每次30秒，约3分钟
2. 胸大肌拉伸：2次，每次30秒，约1.5分钟
3. 颈部时钟运动：左右各1圈，约30秒

强化（3分钟）：
1. YTWL序列：各1组×10次，约3分钟
```

## 训练进度表

### 第1周（第3-7天）

```
每日基础：
- 5分钟快速序列：3-4次
- 15分钟完整序列：1次（从第5天开始）

重点：

- 熟悉动作要领
- 观察身体反应

- 强力拉伸
- 大幅度活动
- 过度训练
```

```
每日基础：
- 5分钟快速序列：2-3次
- 15分钟完整序列：1-2次

新增：
- 加入YTWL强化训练
- 增加保持时间和次数
- 加入进阶版本

重点：
- 强化训练
- 姿势矫正
- 日常习惯建立
```

```
- 保持良好姿势

长期目标：
- 预防复发
- 建立健康习惯
```

## 常见问题解答

### Q1: 训练后疼痛加重怎么办？

```
轻微酸痛（2-3/10）：
- 正常反应，继续训练
- 可减少强度或次数
- 配合热敷

中度疼痛（4-6/10）：
- 减少训练强度50%
- 避免引起疼痛的动作
- 增加休息时间
- 如持续超过24小时，暂停训练

明显疼痛（>6/10）：
- 立即停止训练
- 可能需要就医评估
- 排除其他问题
```

### Q2: 多久能看到效果？

```
个体差异较大，一般：

第1-2天：
- 可能感觉轻微改善
- 主要是活动度提升

第3-5天：
- 疼痛明显减轻
- 活动度继续改善

第1周后：
- 大部分症状改善
- 可进行日常活动

第2-3周：
- 接近完全恢复
- 进入预防维护阶段

如2周后仍无改善，建议就医。
```

### Q3: 能否只做喜欢的动作？

```
不推荐：

原因：
- 需要整体平衡
- 拉伸和强化需结合
- 代偿模式需要全面矫正

建议：
- 至少完成每个类别各1个动作
- 激活 + 拉伸 + 强化
- 或按完整序列执行
```
