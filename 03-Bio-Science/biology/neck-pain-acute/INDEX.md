## 处理原则 | Treatment Principles

### 急性期核心原则

```
R.I.C.E. 原则改良版：

R - Rest (休息)：避免加重损伤的活动
I - Ice (冷敷)：前24小时冷敷减轻炎症
C - Compression (压迫)：颈部不适用
E - Elevation (抬高)：不适用

附加原则：
P - Protection (保护)：保持舒适体位
A - Avoid (避免)：避免主动拉伸、强力按摩
```

### 康复期核心原则

```
渐进性原则：
1. 等长收缩 → 主动活动 → 抗阻训练
2. 小幅度 → 中幅度 → 全幅度
3. 短时间 → 中时间 → 长时间

安全性原则：
• 疼痛评分控制在 3-4/10 以内
• 如疼痛加剧立即停止
• 出现红旗症状立即就医
```

## 警示信号 | Warning Signs

### 需立即就医的红旗症状

```
🚨 神经压迫征象
   • 手臂放射性疼痛、麻木
   • 手指特定区域感觉异常
   • 握力下降、物品掉落

🚨 脊髓压迫征象
   • 四肢无力、步态不稳
   • 精细动作困难
   • 大小便功能障碍

🚨 血管性症状
   • 转头时眩晕
   • 视力模糊、复视
   • 突发剧烈头痛

🚨 感染征象
   • 发热伴颈痛
   • 颈部红肿热痛
   • 无法下巴触胸

详见 → [医疗转诊指南](Neck_Pain_Medical_Referral_Guide.md)
```
