## 手法分类体系 (Manipulation Classification System)

### 基本手法分类 (Basic Technique Classification)

#### 按力学特性分类 (Mechanical Characteristic Classification)

##### 摆动类手法 (Swinging Techniques)
| 手法名称 | 英文术语 | 动作特点 | 力学机制 | 适应病症 | 操作要点 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **一指禅推法** | One-Finger Meditation Pushing | 拇指指端着力，腕部摆动 | 前臂主动摆动带动腕关节屈伸 | 头面部疾病、内脏疾病 | 沉肩坠肘、悬腕、掌虚指实 |
| **㨰法** | Rolling Technique | 手背尺侧着力，前臂旋转 | 前臂旋转带动腕关节屈伸 | 颈肩腰腿痛、软组织损伤 | 吸定、拖动、移动三点连一线 |
| **揉法** | Kneading Technique | 掌指或鱼际着力，腕关节回旋 | 腕关节为主动关节的回旋运动 | 腹部疾病、软组织扭伤 | 轻柔节律、带动皮下组织 |
| **摩法** | Friction Technique | 掌面或指面贴附体表往返摩擦 | 手掌与体表之间的相对滑动 | 胸腹部保健、皮肤美容 | 压力适中、频率均匀 |


#### 临床应用案例数据库
##### 典型病例分析模板
###### 腰椎间盘突出症推拿治疗
**患者资料**: 男性，35岁，办公室工作者，L4-5椎间盘突出

**治疗方案**:
1. **预备阶段** (5分钟)
   - 轻柔掌摩腰背部，频率80-100次/分
   - 力度控制在15-20N，使局部温度上升2-3°C
   
2. **主要治疗** (15分钟)
   ```
   操作序列: 拿法(腰部) → 按法(肾俞、大肠俞) → 点法(委中、承山) → 推法(膀胱经)
   力学参数: 压强30-40N/cm²，频率60-80次/分，持续时间2-3秒/点
   ```

3. **结束阶段** (5分钟)
   - 轻柔揉法放松，配合深呼吸指导
   - 被动直腿抬高测试评估疗效

**疗效评估**:
- VAS疼痛评分: 治疗前7分 → 治疗后3分
- 直腿抬高角度: 30° → 60°
- 日常活动能力: 显著改善

**注意事项**:
- 避免暴力旋转手法
- 急性期配合卧床休息
- 定期复查MRI影像学变化

## 特色流派手法 (Characteristic School Techniques)

### 一指禅推拿流派 (One-Finger Meditation Tuina School)

#### 流派特色
- **技术核心**: 以一指禅推法为代表，强调"柔中带刚、刚中有柔"
- **理论基础**: 脏腑辨证、经络学说相结合
- **手法特点**: 动作轻柔、持久、有力、均匀、深透
- **适应病症**: 内科杂病、妇科疾病、儿科疾病

#### 标准操作流程
```
1. 预备式: 沉肩坠肘、悬腕、掌虚指实
2. 起势: 拇指指端着力，其余四指自然弯曲
3. 摆动: 前臂主动摆动，带动腕关节屈伸
4. 推进: 沿经络走向或病变部位推进
5. 收势: 逐渐减力，轻柔结束
```

**技术参数**:
- 频率: 120-160次/分
- 压力: 15-25N
- 幅度: 1-2cm
- 持续时间: 每个部位3-5分钟

### 㨰法推拿流派 (Rolling Tuina School)

#### 流派特色
- **技术核心**: 以㨰法为特色手法，强调"吸定、拖动、移动"三要点
- **理论基础**: 软组织损伤理论、生物力学原理
- **手法特点**: 力量深透、接触面广、适应性强
- **适应病症**: 骨伤科疾病、软组织损伤、运动系统疾病

#### 标准操作流程
```
1. 定位: 手背尺侧小指侧着力点对准治疗部位
2. 吸定: 手法与体表紧密贴合，产生吸附力
3. 拖动: 前臂旋转带动腕关节屈伸，产生拖动感
4. 移动: 手法沿一定方向缓慢移动
5. 连贯: 三个动作连贯进行，无缝衔接
```

**技术参数**:
- 频率: 100-120次/分
- 压力: 25-40N
- 接触面积: 手背尺侧5-8cm²
- 移动速度: 1-2cm/秒

### 小儿推拿流派 (Pediatric Tuina School)

#### 流派特色
- **技术核心**: 以轻柔手法为主，强调"轻快柔和、平稳着实"
- **理论基础**: 小儿生理病理特点、脏腑娇嫩理论
- **手法特点**: 力度轻柔、频率较快、时间适中
- **适应病症**: 小儿常见病、消化系统疾病、呼吸系统疾病

#### 标准操作流程
```
1. 开天门: 两拇指交替推眉心至前发际
2. 推坎宫: 两拇指分推眉心至眉梢
3. 运太阳: 一手固定头部，另一手运太阳穴
4. 揉耳后高骨: 拇指揉耳后乳突下方凹陷处
5. 拿风池: 拇食中三指拿风池穴
```

**技术参数**:
- 频率: 200-300次/分
- 压力: 5-15N
- 持续时间: 每个手法1-3分钟
- 治疗总时长: 10-15分钟
