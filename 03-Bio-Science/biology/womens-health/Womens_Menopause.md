## 更年期基础

### 核心概念

| 术语 | 定义 |
|------|------|
| 围绝经期 | 月经周期开始变化至绝经后12月 |
| 绝经 | 最后一次月经后满12月(回顾性诊断) |
| 绝经后 | 绝经确认后 |
| 平均绝经年龄 | 51岁(45-55岁正常范围) |
| 早发绝经 | 40-45岁 |
| 卵巢早衰(POI) | <40岁 |
| 围绝经期持续 | 平均4-8年 |

### 激素变化

| 激素 | 变化 | 临床意义 |
|------|------|---------|
| 雌二醇(E2) | 波动→逐渐↓ | 大多数症状的根源 |
| FSH | ↑↑ | >25 IU/L提示围绝经期 |
| 孕酮 | ↓→无(无排卵) | 子宫内膜保护↓ |
| 睾酮 | 缓慢↓ | 性欲/能量影响 |
| DHEA | ↓ | 肾上腺来源 |
| 抑制素B | ↓ | 卵巢储备↓的早期标志 |

### STRAW+10分期

| 阶段 | 月经特征 | FSH | 持续 |
|------|---------|-----|------|
| -2 早期围绝经 | 周期长度变化>7天 | 可变 | 不定 |
| -1 晚期围绝经 | 闭经≥60天 | >25 | 1-3年 |
| +1a 早期绝经后 | 无月经 | ↑↑ | 2年 |
| +1b | 无月经 | 稳定↑ | 3-6年 |
| +2 晚期绝经后 | 无月经 | 稳定↑ | 终身 |

## 更年期症状

### 症状分类与时间线

| 类别 | 症状 | 发生率 | 出现时间 |
|------|------|--------|---------|
| 血管舒缩 | 潮热/盗汗 | 75% | 围绝经期-绝经后5年 |
| 泌尿生殖 | 阴道干燥/性交痛/尿频 | 50% | 绝经后渐进加重(不自愈) |
| 睡眠 | 失眠/入睡困难/早醒 | 40-60% | 围绝经期 |
| 情绪 | 抑郁/焦虑/易怒 | 30-40% | 围绝经期 |
| 认知 | 记忆↓/注意力↓/"脑雾" | 60% | 围绝经期(多数可逆) |
| 骨骼 | 骨密度↓/骨折 | 逐渐 | 绝经后(前5年最快) |
| 心血管 | CVD风险↑ | 逐渐 | 绝经后 |
| 代谢 | 体重↑(中心性)/胰岛素抵抗 | 常见 | 围绝经期 |
| 关节 | 关节痛/僵硬 | 50% | 围绝经期 |

### 泌尿生殖综合征(GSM)

| 维度 | 信息 |
|------|------|
| 定义 | 绝经后雌激素↓导致的泌尿生殖道萎缩 |
| 阴道症状 | 干燥/烧灼/分泌物↓/性交痛 |
| 泌尿症状 | 尿频/尿急/反复UTI |
| 特点 | 不自愈，渐进加重 |
| 一线治疗 | 局部雌激素(阴道制剂) |
| 非激素选择 | 保湿剂/润滑剂/DHEA阴道制剂 |

## 激素替代疗法

### MHT(更年期激素治疗)核心原则

| 原则 | 说明 |
|------|------|
| 适应证 | 中重度血管舒缩症状/GSM/骨质疏松预防 |
| 时间窗 | 绝经<10年或<60岁开始(获益>风险) |
| 个体化 | 最低有效剂量/最短有效时间 |
| 有子宫 | 必须加孕激素(保护子宫内膜) |
| 无子宫 | 单用雌激素 |
| 定期评估 | 每年重新评估继续/停药 |

### MHT获益与风险

| 获益 | 证据 |
|------|------|
| 潮热缓解 | 75%↓(vs安慰剂40%) |
| GSM改善 | 显著 |
| 骨折预防 | 髋骨折↓34%(WHI) |
| 结肠癌↓ | 雌孕合剂 |
| 全因死亡 | <60岁开始→↓(时间窗假说) |

| 风险 | 证据 |
|------|------|
| 乳腺癌 | 雌孕合剂>5年: RR 1.26(WHI) |
| 血栓/中风 | 口服>经皮(经皮低风险) |
| 胆囊疾病 | 口服↑ |
| 子宫内膜癌 | 单用雌激素(有子宫者) |

### MHT给药方式

| 方式 | 优点 | 缺点 |
|------|------|------|
| 口服 | 便利/经济 | 肝脏首过/血栓↑ |
| 经皮(贴片/凝胶) | 无首过/血栓不增加 | 皮肤过敏 |
| 阴道(局部) | 仅局部效应/全身低吸收 | 仅治GSM |
| 缓释宫内(LNG-IUS) | 子宫内膜保护+避孕 | 需置入 |

## 非激素管理

### 血管舒缩症状的非激素选择

| 方法 | 证据 | 效果 |
|------|------|------|
| SSRI/SNRI | A | 潮热↓50-60%(帕罗西汀/文拉法辛) |
| 加巴喷丁 | B | 潮热↓ |
| Fezolinetant(NK3拮抗) | A | 2023年获批 潮热↓显著 |
| CBT | B | 改善对潮热的应对 |
| 正念/放松 | B | 改善主观困扰 |
| 有氧运动 | B | 整体改善(但对潮热争议) |
| 大豆异黄酮 | B-C | 轻度改善(安慰剂效应显著) |
| 针灸 | B-C | 短期改善(质量不一) |
| 黑升麻 | C | 证据不一致 |

### 运动处方

| 目标 | 运动类型 | 建议 |
|------|---------|------|
| 骨密度 | 负重/力量训练 | 3-5次/周 |
| 心血管 | 有氧运动 | 150min中等/周 |
| 体重管理 | 有氧+力量 | 热量适当限制+运动 |
| 关节灵活 | 瑜伽/太极/拉伸 | 2-3次/周 |
| 跌倒预防 | 平衡训练 | 太极拳特别推荐 |
