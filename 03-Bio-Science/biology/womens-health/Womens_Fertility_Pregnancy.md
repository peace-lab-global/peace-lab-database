## 女性生育力

### 生育力基本数据

| 指标 | 数据 |
|------|------|
| 卵子总数(出生) | ~100-200万 |
| 青春期卵子数 | ~30-40万 |
| 排卵总数(一生) | ~400个 |
| 最佳生育年龄 | 20-30岁 |
| 生育力显著下降 | 35岁后加速 |
| 自然受孕概率(每月) | 25-30%(25岁) → 10-15%(35岁) |

### 年龄与生育力

| 年龄 | 每月受孕率 | 自然流产率 | 染色体异常率 |
|------|-----------|-----------|------------|
| <30 | 25-30% | 10% | 2-3% |
| 30-35 | 15-20% | 15% | 5% |
| 35-40 | 10-15% | 20-25% | 10% |
| 40-45 | 5% | 30-40% | 20-30% |
| >45 | <1% | 50%+ | 50%+ |

### 不孕症

| 维度 | 信息 |
|------|------|
| 定义 | 规律性生活12月未孕(35岁以上6月) |
| 患病率 | 10-15%育龄夫妇 |
| 女性因素 | 40%(排卵/输卵管/子宫/子宫颈) |
| 男性因素 | 30%(精子数量/质量) |
| 双方/不明 | 30% |

## 备孕准备

### 备孕清单

| 维度 | 建议 |
|------|------|
| 叶酸 | 至少孕前3月开始 0.4-0.8mg/天 |
| 疫苗 | 风疹/水痘/流感/COVID(确认免疫) |
| 药物审查 | 停用致畸药物(异维A酸等) |
| 基础检查 | 血常规/甲功/血糖/血型/TORCH |
| 生活方式 | 戒烟戒酒/健康体重/规律运动 |
| 口腔健康 | 牙周病→早产风险↑ |
| 心理准备 | 伴侣沟通/角色期望/支持系统 |

### 优化受孕的策略

| 策略 | 说明 |
|------|------|
| 排卵期识别 | LH试纸/基础体温/宫颈粘液 |
| 受孕窗口 | 排卵前5天至排卵后1天(共6天) |
| 性交频率 | 受孕窗口内每1-2天 |
| 男方准备 | 避免高温(桑拿/紧身裤)/补锌/戒烟酒 |
| 压力管理 | 过度焦虑可能影响排卵 |

## 妊娠健康

### 三孕期概览

| 孕期 | 时间 | 胎儿发育 | 母体变化 | 注意事项 |
|------|------|---------|---------|---------|
| 第一孕期 | 1-12周 | 器官形成/心跳 | 孕吐/疲劳 | 致畸高风险期 |
| 第二孕期 | 13-27周 | 快速生长/胎动 | 能量恢复/"蜜月期" | 唐筛/排畸超声 |
| 第三孕期 | 28-40周 | 肺成熟/脂肪储存 | 腰痛/水肿/失眠 | GDM筛查/分娩准备 |

### 妊娠期运动建议(ACOG)

| 建议 | 说明 |
|------|------|
| 推荐 | 150min中等强度有氧/周 |
| 安全运动 | 步行/游泳/固定自行车/低冲击有氧/瑜伽(孕妇) |
| 避免运动 | 接触性运动/高跌倒风险/仰卧(20周后)/高温环境 |
| 禁忌 | 前置胎盘/先兆早产/严重贫血/心肺疾病 |
| 益处 | GDM↓/先兆子痫↓/剖宫产↓/产后恢复↑ |

### 围产期心理健康

| 状态 | 发生率 | 特征 | 干预 |
|------|--------|------|------|
| 产前焦虑 | 15-20% | 对分娩/胎儿/角色的担忧 | CBT/支持 |
| 产后忧郁(Baby Blues) | 50-80% | 产后3-5天哭泣/情绪波动 | 自限(2周内) |
| 产后抑郁 | 10-20% | 持续≥2周的抑郁/无兴趣/内疚 | 心理治疗/药物 |
| 产后焦虑 | 10-15% | 过度担忧婴儿安全 | CBT/药物 |
| 产后精神病 | 0.1-0.2% | 精神病性症状/急性发作 | 紧急精神科 |

## 产后恢复

### 产后恢复时间表

| 时间 | 恢复内容 |
|------|---------|
| 产后即刻 | 子宫收缩/恶露开始/母乳建立 |
| 1-2周 | 伤口愈合/休息为主 |
| 2-6周 | 恶露停止/开始轻度活动 |
| 6-12周 | 运动渐进恢复/盆底评估 |
| 3-6月 | 体能接近孕前/月经恢复(非哺乳) |
| 6-12月 | 全面恢复/情绪稳定 |

### 盆底康复

| 阶段 | 内容 |
|------|------|
| 评估 | 产后6周盆底功能评估 |
| 基础训练 | 凯格尔运动(每日3组×10次) |
| 进阶训练 | 盆底肌力+耐力+协调 |
| 专业康复 | 生物反馈/电刺激(症状严重时) |
| 注意 | 避免过早高冲击运动 |
