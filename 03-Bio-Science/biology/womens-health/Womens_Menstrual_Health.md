## 月经生理

### 正常月经参数

| 参数 | 正常范围 |
|------|---------|
| 周期长度 | 24-38天 |
| 经期持续 | 3-8天 |
| 出血量 | 5-80mL/周期 |
| 初潮年龄 | 10-16岁(平均12.5) |
| 绝经年龄 | 45-55岁(平均51) |

### 月经周期四阶段

| 阶段 | 天数(28天周期) | 主导激素 | 子宫内膜 |
|------|-------------|---------|---------|
| 月经期 | 1-5 | 雌孕激素低 | 剥脱出血 |
| 卵泡期 | 6-13 | FSH↑ 雌激素↑ | 增生 |
| 排卵期 | 14(±2) | LH峰 雌激素峰 | 峰值厚度 |
| 黄体期 | 15-28 | 孕酮↑ | 分泌期转化 |

## 月经周期与激素

### 激素变化对身心影响

| 阶段 | 能量 | 情绪 | 认知 | 运动建议 |
|------|------|------|------|---------|
| 月经期(冬) | 低 | 内省/安静 | 直觉↑ | 轻柔运动/瑜伽/散步 |
| 卵泡期(春) | 上升 | 乐观/外向 | 学习力↑ | 逐渐增加强度 |
| 排卵期(夏) | 最高 | 自信/社交 | 沟通↑ | 高强度训练/社交运动 |
| 黄体期(秋) | 下降 | 易怒/敏感 | 细节↑ | 中等强度/渐降 |

### 经前综合征(PMS)

| 维度 | 信息 |
|------|------|
| 发生率 | 75%女性有某程度PMS |
| 时间 | 黄体晚期(经前1-2周) |
| 躯体症状 | 乳房胀痛/腹胀/头痛/水肿/疲劳 |
| 情绪症状 | 易怒/焦虑/抑郁/情绪波动 |
| 行为症状 | 食欲↑(碳水渴求)/失眠/注意力↓ |

### 经前期情绪障碍(PMDD)

| 维度 | 信息 |
|------|------|
| 发生率 | 3-8% |
| 诊断 | DSM-5: 黄体期显著情绪症状→经后缓解×2周期 |
| 严重度 | 显著功能损害(工作/关系/日常) |
| 生物机制 | GABA受体对孕酮代谢物(ALLO)异常敏感 |
| 治疗 | SSRI(连续或黄体期)/口服避孕药/CBT |

### PMS/PMDD管理策略

| 策略 | 证据 | 建议 |
|------|------|------|
| 有氧运动 | A | 30min 3-5次/周 |
| 钙补充 | A | 1200mg/天 |
| SSRI | A | 连续或黄体期给药(PMDD) |
| CBT | B | 认知重构+行为策略 |
| 维生素B6 | B | 50-100mg/天 |
| 镁 | B | 200-400mg/天 |
| 减少咖啡因/盐/糖 | C | 黄体期特别注意 |
| 正念/放松 | C | 每日练习 |

## 月经相关疾病

### 痛经

| 分类 | 特征 | 治疗 |
|------|------|------|
| 原发性 | 无器质性病变/前列腺素过多 | NSAIDs/口服避孕药/热敷/运动 |
| 继发性 | 子宫内膜异位/腺肌症/肌瘤 | 治疗原发病 |

### 月经不调

| 类型 | 定义 | 常见原因 |
|------|------|---------|
| 闭经(原发) | 15岁无月经 | 解剖异常/内分泌 |
| 闭经(继发) | 已有月经→停经≥3周期 | PCOS/压力/低体重/妊娠 |
| 月经过多 | >80mL/周期 | 肌瘤/内膜病变/凝血障碍 |
| 月经稀发 | 周期>38天 | PCOS/甲状腺/高泌乳素 |
| 月经频发 | 周期<24天 | 黄体功能不全 |

### 运动相关月经紊乱(RED-S)

| 维度 | 信息 |
|------|------|
| 原名 | 女运动员三联征(现扩展为RED-S) |
| 核心问题 | 能量可用性不足(EA<30kcal/kg FFM/天) |
| 三联征 | 低能量可用性→月经紊乱→骨密度↓ |
| 高危运动 | 体操/芭蕾/长跑/花滑 |
| 管理 | 增加能量摄入/降低训练量/多学科团队 |

## 月经与生活方式

### 周期同步训练(Cycle Syncing)

| 阶段 | 运动类型 | 营养重点 |
|------|---------|---------|
| 月经期 | 恢复性运动/轻瑜伽/散步 | 铁/维C(补充铁丢失) |
| 卵泡期 | 渐进增量/尝试新运动 | 蛋白/复合碳水 |
| 排卵期 | 高强度/团体运动/竞赛 | 充足热量 |
| 黄体早期 | 力量训练/中等有氧 | 镁/B6/omega-3 |
| 黄体晚期 | 降低强度/瑜伽/游泳 | 减盐减咖啡因 |
