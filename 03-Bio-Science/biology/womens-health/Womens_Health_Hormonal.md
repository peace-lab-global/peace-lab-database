## 月经周期优化

### 月经周期的激素变化

月经周期（Menstrual Cycle）是女性生殖内分泌系统的核心节律，平均28天（21-35天正常范围）。理解周期中激素的动态变化是优化健康、运动和生活方式的基础。

**四阶段激素图谱**：

| 阶段 | 天数 | 主要激素变化 | 身体状态 |
|------|------|------------|---------|
| 月经期 | 1-5天 | E2↓P4↓，FSH开始升高 | 能量较低，需要休息 |
| 卵泡期 | 6-13天 | FSH→E2↑↑ | 能量↑，精力充沛，社交意愿↑ |
| 排卵期 | 13-16天 | LH峰值→排卵 | 能量最高，性欲峰值 |
| 黄体期 | 17-28天 | P4↑↑，E2先升后降 | 能量逐渐↓，PMS可能出现 |

### 周期同步生活方式

**运动优化**：

| 阶段 | 推荐运动 | 原因 |
|------|---------|------|
| 月经期 | 轻度运动（瑜伽、散步、拉伸） | 恢复期，避免高强度冲击 |
| 卵泡期 | 高强度训练（HIIT、力量训练、跑步） | E2↑→能量↑、疼痛阈值↑、恢复能力↑ |
| 排卵期 | 峰值训练（竞技运动、挑战新目标） | 力量和速度峰值 |
| 黄体期 | 中等强度（游泳、骑行、普拉提） | 体温↑0.3-0.5°C，避免过热 |

**营养优化**：

| 阶段 | 营养重点 | 具体建议 |
|------|---------|---------|
| 月经期 | 补铁、抗炎 | 红肉/深绿叶菜（铁），姜茶（抗炎），充足水分 |
| 卵泡期 | 支持雌激素代谢 | 十字花科蔬菜（DIM/I3C），亚麻籽，发酵食品 |
| 排卵期 | 抗氧化支持 | 浆果、深色蔬果，支持卵子健康 |
| 黄体期 | 血清素支持、缓解PMS | 复合碳水化合物、黑巧克力、镁（200-400mg） |

### 月经周期异常的识别

| 异常类型 | 定义 | 可能原因 |
|---------|------|---------|
| 月经过少（Oligomenorrhea） | 周期>35天 | PCOS、甲状腺问题、过度运动、进食障碍 |
| 月经过多（Menorrhagia） | 出血量>80mL/周期或>7天 | 子宫肌瘤、凝血障碍、激素失衡 |
| 闭经（Amenorrhea） | >3个月无月经 | 低体重、过度运动、压力、PCOS |
| 经间出血 | 非经期出血 | 排卵出血（正常）或息肉、内膜问题 |
| 痛经（Dysmenorrhea） | 严重经期疼痛 | 原发性（前列腺素）或继发性（内膜异位症） |

## 多囊卵巢综合征 PCOS

### 概述

多囊卵巢综合征（Polycystic Ovary Syndrome, PCOS）是育龄女性最常见的内分泌疾病，影响约8-13%的女性。PCOS不仅影响生殖健康，还与代谢综合征、心理健康问题和长期心血管风险相关。

**诊断标准（Rotterdam共识）**：

需满足以下三条中的两条：
1. 排卵障碍（稀发排卵或无排卵）
2. 高雄激素表现（临床或生化）
3. 超声显示多囊卵巢形态
4. 排除其他疾病（甲状腺疾病、高泌乳素血症等）

### PCOS的表型分类

| 表型 | 高雄激素 | 排卵障碍 | 多囊卵巢 | 占比 | 特点 |
|------|---------|---------|---------|------|------|
| A（经典型） | ✓ | ✓ | ✓ | 约50% | 症状最重，代谢风险最高 |
| B（经典无PCO） | ✓ | ✓ | ✗ | 较少 | 同A但卵巢形态正常 |
| C（排卵型） | ✓ | ✗ | ✓ | 约20% | 高雄激素但排卵正常 |
| D（非高雄型） | ✗ | ✓ | ✓ | 约20% | 排卵障碍但雄激素正常 |

### PCOS的健康影响

| 领域 | 具体影响 |
|------|---------|
| 生殖 | 不孕、流产风险↑、妊娠并发症↑ |
| 代谢 | 胰岛素抵抗（70%）、2型糖尿病风险↑、血脂异常 |
| 心血管 | 高血压、动脉粥样硬化风险↑ |
| 心理 | 抑郁（↑3倍）、焦虑、体像困扰、进食障碍 |
| 肿瘤 | 子宫内膜癌风险↑（长期无排卵→内膜增生） |
| 睡眠 | 阻塞性睡眠呼吸暂停风险↑ |

### PCOS的综合管理

**生活方式干预**（一线治疗）：

| 干预 | 目标 | 具体建议 |
|------|------|---------|
| 体重管理 | 减重5-10%可显著改善症状 | 减少精制碳水，增加蛋白质和纤维 |
| 运动 | 改善胰岛素敏感性 | 每周≥150分钟有氧+2次力量训练 |
| 饮食模式 | 抗炎、低GI | 地中海饮食、低升糖指数饮食 |
| 睡眠 | 7-9小时，规律 | 改善 leptin/ghrelin 平衡 |
| 压力管理 | 降低皮质醇 | 正念、瑜伽、社交支持 |

**药物治疗**（需医生指导）：

| 药物 | 适应症 | 机制 |
|------|--------|------|
| 二甲双胍（Metformin） | 胰岛素抵抗 | 改善胰岛素敏感性→降低雄激素 |
| 复方口服避孕药 | 调节周期、痤疮/多毛 | 抑制卵巢雄激素产生 |
| 螺内酯（Spironolactone） | 痤疮/多毛 | 雄激素受体拮抗剂 |
| 来曲唑（Letrozole） | 促排卵（有生育需求） | 芳香化酶抑制剂 |
| Myo-肌醇 | 辅助改善 | 改善胰岛素信号和卵子质量 |

## 子宫内膜异位症

### 概述

子宫内膜异位症（Endometriosis）是指子宫内膜组织生长在子宫腔以外的位置，是一种慢性炎症性疾病。影响约10%的育龄女性，平均诊断延迟7-10年。

**常见异位位置**：

| 位置 | 发生率 | 症状特点 |
|------|--------|---------|
| 卵巢 | 最常见（子宫内膜瘤/巧克力囊肿） | 盆腔痛、不孕 |
| 直肠子宫陷凹 | 常见 | 深部性交痛、排便痛 |
| 子宫骶韧带 | 常见 | 慢性盆腔痛 |
| 膀胱 | 较少见 | 经期血尿、排尿痛 |
| 肠道 | 较少见 | 经期排便痛、便血 |
| 罕见位置 | 极少见 | 肺、脑等（周期性症状） |

### 症状与诊断

**核心症状**：

| 症状 | 发生率 | 特点 |
|------|--------|------|
| 痛经 | >90% | 进行性加重，常需止痛药 |
| 深部性交痛 | 50-70% | 月经前最明显 |
| 慢性盆腔痛 | 40-60% | 非经期也持续 |
| 不孕 | 30-50% | 机械性+炎症性因素 |
| 排便/排尿痛 | 20-30% | 与经期相关 |
| 疲劳 | 60-80% | 慢性炎症导致 |

**诊断延迟的原因**：
- 症状被归为"正常痛经"（痛觉正常化）
- 缺乏特异性生物标志物
- 影像学检查可能遗漏早期病灶
- 需要腹腔镜手术确认（金标准）
- 医患双方对疼痛的轻视

### 综合管理

| 层次 | 方法 | 适用阶段 |
|------|------|---------|
| 生活方式 | 抗炎饮食、规律运动、热敷、减压 | 辅助所有阶段 |
| 止痛 | NSAIDs、对乙酰氨基酚 | 轻中度疼痛 |
| 激素治疗 | 复方避孕药、孕激素、GnRH激动剂 | 抑制病灶生长 |
| 手术 | 腹腔镜切除/消融病灶 | 中重度、不孕 |
| 生育管理 | ART/IVF | 合并不孕 |
| 心理支持 | CBT、支持小组、心理教育 | 慢性疾病管理 |

## 围绝经期管理

### 围绝经期的定义与阶段

围绝经期（Perimenopause）是女性从生育期过渡到绝经期的阶段，通常从40多岁开始，平均持续4-8年。这一时期卵巢功能逐渐衰退，激素水平波动剧烈。

| 阶段 | 平均年龄 | 激素特征 | 主要症状 |
|------|---------|---------|---------|
| 早期围绝经期 | 40-45岁 | 月经周期变化±7天，FSH波动 | 月经不规律开始 |
| 中期围绝经期 | 45-50岁 | FSH↑↑，E2波动大 | 潮热、情绪波动 |
| 晚期围绝经期 | 48-52岁 | E2↓↓，FSH持续↑ | 月经间隔>60天 |
| 绝经后 | 52岁+ | E2极低，FSH>30 IU/L | 长期健康管理 |

### 围绝经期症状管理

**潮热与盗汗（Vasomotor Symptoms）**：

| 干预方法 | 效果 | 说明 |
|---------|------|------|
| 激素替代治疗（MHT） | 最有效（↓80-90%） | 适用于60岁以下/绝经10年内 |
| SSRIs/SNRIs | 中等（↓50-60%） | Paroxetine已获FDA批准 |
| 加巴喷丁 | 中等 | 非激素选择 |
| 可乐定 | 轻度至中等 | 二线选择 |
| 认知行为治疗（CBT） | 中等 | 改善应对和主观感受 |
| 黑升麻（Black Cohosh） | 争议 | 部分研究有效 |
| 植物雌激素 | 轻度 | 大豆异黄酮等 |
| 针灸 | 轻度至中等 | 部分RCT支持 |

**激素替代治疗（MHT）的风险与获益**：

| 方面 | 获益 | 风险 |
|------|------|------|
| 血管舒缩症状 | 显著改善 | — |
| 骨密度 | 骨质疏松预防 | — |
| 心血管 | "时间窗口"内启动可能有保护 | >60岁启动可能↑风险 |
| 乳腺癌 | — | 结合雌激素+孕激素→轻微↑（5年使用） |
| 静脉血栓 | — | 口服制剂风险↑（经皮风险较小） |
| 卒中 | — | 60岁以上↑风险 |

### 绝经后长期健康管理

| 关注领域 | 管理策略 | 筛查建议 |
|---------|---------|---------|
| 骨密度 | 负重运动+维生素D/钙+必要时药物治疗 | 65岁起DEXA扫描 |
| 心血管 | 生活方式+风险因素管理 | 每年评估 |
| 泌尿生殖道 | 局部雌激素、盆底肌训练 | 定期评估 |
| 认知功能 | 运动、社交、认知刺激 | 定期评估 |
| 心理健康 | 抑郁/焦虑筛查、支持 | 定期PHQ-9 |

## 激素与情绪效应

### 激素对情绪的生物学机制

女性激素对情绪的影响远超传统认识。雌激素和孕酮通过多种神经机制直接影响大脑的情绪调节系统。

**雌激素（Estradiol, E2）的神经效应**：

| 机制 | 效应 | 情绪影响 |
|------|------|---------|
| 5-HT系统增强 | ↑TPH2（5-HT合成酶）、↓MAO-A | 抗抑郁效应 |
| 神经营养因子 | ↑BDNF表达 | 神经可塑性、认知保护 |
| GABA调节 | 月经周期依赖性调节 | 焦虑水平波动 |
| 多巴胺系统 | ↑D2受体表达、↑DA合成 | 动机、奖赏、愉悦感 |
| HPA轴调节 | 抑制CRH过度反应 | 应激反应缓冲 |
| 海马保护 | 促进突触形成 | 记忆和情绪调节 |

**孕酮（Progesterone, P4）的神经效应**：

| 代谢产物 | 机制 | 情绪影响 |
|---------|------|---------|
| 孕酮→别孕烯醇酮（Allopregnanolone） | GABA-A受体正向别构调节剂 | 类似苯二氮卓的镇静/抗焦虑效应 |
| P4快速下降期 | Allopregnanolone↓→GABA功能↓ | 经前烦躁障碍（PMDD）的机制 |
| 黄体期P4↑ | GABA-A受体脱敏 | 部分女性出现焦虑而非镇静 |

### 激素相关情绪障碍

**经前综合征（PMS）与经前烦躁障碍（PMDD）**：

| 对比 | PMS | PMDD |
|------|-----|------|
| 患病率 | 约75%女性有某种程度 | 3-8%育龄女性（严重） |
| 情绪症状 | 轻度情绪波动、易怒 | 严重抑郁、极度易怒、绝望感 |
| 功能影响 | 轻度影响日常 | 显著影响工作/社交/关系 |
| 诊断 | 月经前出现身体/情绪症状 | DSM-5正式诊断，≥5项特定症状 |
| 治疗 | 生活方式调整 | SSRI（一线）、OCP、CBT |

**PMDD的神经生物学**：

近年的研究揭示PMDD并非"激素水平异常"，而是大脑对正常激素波动的"过敏"反应：

| 发现 | 研究者 | 含义 |
|------|--------|------|
| PMDD女性对allopregnanolone异常敏感 | Schmidt et al. | GABA系统的差异反应 |
| ESC/E(Z)基因复合体变异 | Dubey et al. (2017) | 首个确认的PMDD遗传基础 |
| 激素"开关"效应 | 停止GnRH激动剂→症状重现 | 确认激素依赖性 |
| SSRI快速起效 | 不同于抑郁症的2-4周 | 提示不同神经机制 |

### 激素与心理健康的时间线

| 生命阶段 | 激素特征 | 心理健康风险窗口 | 保护策略 |
|---------|---------|----------------|---------|
| 青春期 | E2↑↑，波动大 | 抑郁风险↑（女孩2倍于男孩） | 情绪教育、支持系统 |
| 生育期 | 周期性波动 | PMS/PMDD、产后抑郁 | 周期追踪、早期干预 |
| 围产期 | E2/P4剧变 | 产后抑郁（10-20%）、产后焦虑 | 筛查、支持、治疗 |
| 围绝经期 | E2↓↓，波动剧烈 | 抑郁风险↑2-4倍 | 激素评估、心理支持 |
| 绝经后 | E2低而稳定 | 认知功能下降、心血管风险 | 运动、社交、认知训练 |
