# Sexually Transmitted Diseases Public Health (性传播疾病公共卫生)

> 🌍 **公共卫生指南**: 本文档详细阐述性传播疾病的流行病学监测、防控策略、政策框架和全球合作机制。建议结合[临床](STD_Clinical_Features.md)和[预防](STD_Prevention_Strategies.md)文档进行系统学习。

## 公共卫生框架体系 (Public Health Framework System)

### 流行病学监测体系 (Epidemiological Surveillance System)

#### 监测指标体系 (Surveillance Indicator System)
**核心监测指标**
```
疾病负担指标：
• 发病率 (Incidence Rate) 新发病例数/暴露人口数×10万
• 患病率 (Prevalence Rate) 现患病例数/调查人口数×10万
• 死亡率 (Mortality Rate) 死亡病例数/总人口数×10万
• 病死率 (Case Fatality Rate) 死亡病例数/发病病例数×100%

传播动力学指标：
• 基本再生数 (R₀) 一个感染者平均传播人数
• 有效再生数 (Rt) 实际传播能力动态变化
• 潜伏期分布 感染到发病的时间分布
• 传染期长度 能够传播给他人的时期

高危行为指标：
• 安全套使用率 安全性行为保护水平
• 多性伴侣比例 性行为风险暴露程度
• 性病检测率 高危人群筛查覆盖情况
• 治疗依从率 规范治疗完成程度
```

**分层监测体系**
| 监测层级 | 覆盖范围 | 监测内容 | 数据来源 | 更新频率 |
| :--- | :--- | :--- | :--- | :--- |
| **国家级** | 全国范围 | 总体流行趋势、政策效果 | 法定传染病报告系统 | 月度/季度 |
| **省级** | 省市区域 | 地区流行特征、资源配置 | 省级疾控中心数据 | 月度 |
| **地市级** | 城市地区 | 本地流行情况、服务利用 | 地市卫生统计年报 | 季度 |
| **县区级** | 基层单位 | 社区流行状况、干预效果 | 基层医疗机构报告 | 月度 |

#### 监测方法技术 (Surveillance Methods and Technologies)


### 疾病负担评估 (Disease Burden Assessment)

#### 全球疾病负担分析 (Global Burden of Disease Analysis)
**DALY指标体系**
```
伤残调整生命年 (Disability Adjusted Life Years)：
• 早死所致生命年损失 (YLL) 死亡导致的预期寿命损失
• 失能所致生命年损失 (YLD) 疾病导致的健康寿命损失
• 总体疾病负担 DALY = YLL + YLD

权重系数设定：
• 死亡权重 根据年龄别死亡权重
• 残疾权重 0-1之间的健康状态权重
• 时间贴现率 考虑时间价值的贴现
• 年龄权重 反映不同年龄的健康价值
```

**区域负担差异**
```
高负担地区：
• 撒哈拉以南非洲 HIV、梅毒负担最重
• 东南亚地区 淋病、衣原体感染高发
• 拉丁美洲地区 综合性性病负担较高
• 东欧中亚地区 HIV疫情快速增长

中等负担地区：
• 中国及东亚地区 整体控制较好但局部高发
• 中东北非地区 传统文化影响报告率
• 南亚地区 贫困和医疗资源限制
• 太平洋岛国 地理隔离和资源匮乏

低负担地区：
• 北美欧洲地区 相对控制较好
• 澳新地区 良好的公共卫生体系
• 日本韩国等 发达亚洲国家
```

```

• 一级预防 健康促进、风险干预
• 二级预防 早期筛查、及时诊断
• 三级预防 规范治疗、康复管理
• 四级预防 并发症预防、质量改善
```
