# 内分泌系统与疗愈整合 | Endocrine System & Healing Integration

> 内分泌系统通过激素调控全身几乎所有生理过程，是身心交互的核心中介——情绪、压力、睡眠、代谢、免疫、生殖均受其调控。理解内分泌是泛疗愈实践的生物医学基础。

---

## 1. 主要内分泌腺与激素

### 1.1 下丘脑-垂体轴 (HPA/HPG/HPT)

```
下丘脑(Hypothalamus) — "总指挥"
  ├── CRH → 垂体 → ACTH → 肾上腺皮质 → 皮质醇 (HPA轴: 压力)
  ├── GnRH → 垂体 → FSH/LH → 性腺 → 雌激素/睾酮 (HPG轴: 生殖)
  ├── TRH → 垂体 → TSH → 甲状腺 → T3/T4 (HPT轴: 代谢)
  └── GHRH → 垂体 → GH → 全身 (生长)
```

### 1.2 核心内分泌腺一览

| 腺体 | 主要激素 | 功能 | 疗愈关联 |
|------|----------|------|----------|
| **下丘脑** | CRH/TRH/GnRH | 整合神经-内分泌 | 压力/情绪的中枢 |
| **垂体** | ACTH/TSH/GH/PRL/ADH | 调控下游腺体 | "主腺" |
| **肾上腺皮质** | 皮质醇/DHEA | 压力适应/抗炎 | HPA轴核心 |
| **肾上腺髓质** | 肾上腺素/去甲 | 战逃反应 | 急性压力 |
| **甲状腺** | T3/T4 | 代谢速率 | 能量/情绪/体重 |
| **胰腺** | 胰岛素/胰高血糖素 | 血糖调控 | 代谢/情绪波动 |
| **松果体** | 褪黑素 | 昼夜节律 | 睡眠/时差 |
| **性腺** | 雌激素/孕酮/睾酮 | 生殖/第二性征 | 性欲/情绪/骨骼 |
| **脂肪组织** | 瘦素/脂联素 | 饱腹/代谢 | 体重/炎症 |
| **肠道** | GLP-1/Ghrelin/5-HT(95%) | 消化/饱腹/情绪 | 肠-脑轴 |

## 2. HPA轴与压力反应

### 2.1 急性压力(适应性)

```
威胁感知 → 下丘脑(CRH) → 垂体(ACTH) → 肾上腺(皮质醇↑)
  → 血糖↑ / 免疫↑(短期) / 注意力↑ / 记忆增强
  → 负反馈 → 恢复
```

### 2.2 慢性压力(病理性)

| 阶段 | HPA变化 | 后果 |
|------|---------|------|
| **过度激活** | 皮质醇持续↑ | 免疫抑制/海马萎缩/代谢综合征/焦虑 |
| **功能失调** | 昼夜节律紊乱(扁平曲线) | 晨起疲惫/夜间兴奋 |
| **衰竭/低皮质醇** | 肾上腺"疲劳"(HPA下调) | 慢性疲劳/纤维肌痛/PTSD亚型 |

**McEwen异稳态负荷(Allostatic Load)**: 慢性压力→累积生物磨损→加速衰老/疾病

### 2.3 疗愈干预对HPA的影响

| 干预 | 证据 |
|------|------|
| **正念冥想** | ↓皮质醇(基线)、↓皮质醇觉醒反应(CAR) (Pascoe et al., 2017) |
| **瑜伽** | ↓皮质醇、↑GABA (Streeter et al., 2012) |
| **太极** | ↓皮质醇/↑DHEA (Wang et al., 2014) |
| **运动** | 急性↑皮质醇(适应性)→长期↓基线 (Hill et al., 2008) |
| **社会支持** | 缓冲HPA激活(Heinrichs et al., 2003) |
| **自然暴露** | ↓皮质醇(12-16%) (Li, 2010) |
| **音乐** | ↓皮质醇(术后/术前) (Nilsson, 2008) |

## 3. 关键内分泌紊乱与疗愈

### 3.1 甲状腺功能异常

| 类型 | 症状 | 疗愈注意 |
|------|------|----------|
| **甲减** | 疲劳/体重↑/抑郁/怕冷/便秘 | 运动+营养+药物(左旋甲状腺素) |
| **甲亢** | 焦虑/心悸/体重↓/失眠 | 放松+正念+药物(甲巯咪唑) |
| **桥本** | 自身免疫性甲减 | 抗炎饮食+压力管理+药物 |

### 3.2 胰岛素抵抗与代谢综合征

```
高GI饮食/久坐/压力 → 胰岛素持续↑ → 受体下调 → 胰岛素抵抗
  → 血糖↑/脂肪↑/炎症↑ → II型糖尿病/心血管/PCOS

干预: 低GI饮食 + 运动(改善胰岛素敏感性) + 压力管理(↓皮质醇→↓血糖)
```

### 3.3 性激素与疗愈

| 激素 | 疗愈关联 |
|------|----------|
| **雌激素↓**(围绝经期) | 潮热/情绪波动/骨质疏松/性欲↓ → HRT/运动/正念 |
| **睾酮↓**(男性) | 疲劳/抑郁/性欲↓/肌肉↓ → 运动/睡眠/营养 |
| **催产素** | "联结激素"——拥抱/抚摸/社交→↑催产素→↓皮质醇/↑信任 |

### 3.4 褪黑素与睡眠

- 松果体在黑暗中分泌褪黑素→诱导睡眠
- 蓝光(380-500nm)抑制褪黑素分泌→睡前避免屏幕
- 外源褪黑素(0.5-3mg): 时差/延迟性睡眠障碍(非长期方案)
- 参见: [睡眠科学](../sleep-science/)

## 4. 肠-脑-内分泌轴

| 通路 | 机制 |
|------|------|
| **迷走神经** | 肠道→迷走→脑干(95%传入) |
| **肠道激素** | GLP-1/Ghrelin/PYY→影响食欲/情绪 |
| **肠道血清素** | 肠道合成95%血清素→影响情绪/胃肠运动 |
| **微生物代谢物** | SCFA(短链脂肪酸)/色氨酸代谢→影响大脑 |
| **免疫-内分泌** | 肠道炎症→细胞因子→HPA轴→皮质醇 |

## 5. 内分泌友好型生活方式

| 维度 | 建议 |
|------|------|
| **光照** | 晨起30min自然光(↑皮质醇峰值) + 夜间减少蓝光(↑褪黑素) |
| **运动** | 力量训练(↑睾酮/GH) + 有氧(↓胰岛素抵抗) + 瑜伽(↓皮质醇) |
| **营养** | 低GI/抗炎/优质蛋白/健康脂肪(Omega-3) |
| **睡眠** | 7-9h、固定时间、黑暗/凉爽环境 |
| **压力** | 每日正念/呼吸/自然接触→HPA调节 |
| **社交** | 正向社会互动→催产素↑→HPA缓冲 |
