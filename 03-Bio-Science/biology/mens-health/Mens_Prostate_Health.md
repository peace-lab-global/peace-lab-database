## 前列腺解剖与功能

### 基本信息

| 特征 | 描述 |
|------|------|
| 位置 | 膀胱下方，围绕尿道起始部 |
| 大小 | 约核桃大小(正常~20g) |
| 功能 | 分泌前列腺液(精液30%) |
| 前列腺液 | 含PSA/锌/柠檬酸/蛋白酶 |
| 神经支配 | 盆腔自主神经(勃起/排尿相关) |

### 分区(McNeal分区)

| 分区 | 体积占比 | 临床意义 |
|------|---------|---------|
| 外周区 | 70% | 前列腺癌好发区(75%) |
| 中央区 | 25% | 精囊管通过 |
| 移行区 | 5% | BPH好发区 |
| 前纤维肌肉基质 | — | 无腺体 |

## 良性前列腺增生

### BPH核心信息

| 维度 | 信息 |
|------|------|
| 定义 | 前列腺移行区腺体增生→尿道压迫 |
| 患病率 | 40岁~25%; 60岁~50%; 80岁~90% |
| 病因 | DHT/雌激素/生长因子/炎症 |
| 关键激素 | DHT(5α-还原酶将T→DHT) |

### LUTS症状分类

| 类型 | 症状 |
|------|------|
| 储尿期 | 尿频/尿急/夜尿↑/急迫性尿失禁 |
| 排尿期 | 尿流弱/排尿犹豫/间断尿流/尿末滴沥 |
| 排尿后 | 排尿不尽感/排尿后滴沥 |

### 评估工具

| 工具 | 用途 |
|------|------|
| IPSS评分(0-35) | 症状严重度(轻0-7/中8-19/重20-35) |
| 最大尿流率(Qmax) | <10mL/s提示梗阻 |
| 残余尿量 | >100mL需关注 |
| 前列腺体积 | TRUS测量(>30mL有意义) |

### 治疗阶梯

| 阶段 | 措施 | 适用 |
|------|------|------|
| 观察等待 | 生活方式调整/定期随访 | IPSS≤7 |
| 药物治疗 | α-受体阻断剂/5ARI/联合 | IPSS 8-19 |
| 微创手术 | UroLift/水蒸气消融/前列腺栓塞 | 药物不耐受/中等症状 |
| 手术 | TURP/激光/开放手术 | IPSS≥20或并发症 |

## 前列腺炎

### 分类(NIH)

| 类型 | 特征 | 比例 |
|------|------|------|
| I. 急性细菌性 | 发热/尿路感染/急性 | <5% |
| II. 慢性细菌性 | 反复UTI/持续>3月 | 5-10% |
| III. 慢性盆腔疼痛(CPPS) | 盆腔痛>3月/无感染证据 | 90% |
| IIIa. 炎症型 | 精液/前列腺液WBC↑ | — |
| IIIb. 非炎症型 | 无WBC升高 | — |
| IV. 无症状炎症性 | 偶然发现 | — |

### CPPS的多模态治疗

| 方法 | 证据 | 机制 |
|------|------|------|
| α-受体阻断剂 | B | 缓解排尿症状 |
| 物理治疗(盆底) | B | 肌筋膜释放/放松训练 |
| 认知行为治疗 | B | 疼痛灾难化↓ |
| 热疗 | B-C | 缓解疼痛/痉挛 |
| 针灸 | B-C | 镇痛/肌肉放松 |
| 运动 | C | 整体健康改善 |

## 前列腺癌

### 流行病学

| 指标 | 数据 |
|------|------|
| 全球发病率 | 男性第二大癌症 |
| 终生风险 | ~11%(每9名男性1人) |
| 5年生存率(局限性) | >98% |
| 5年生存率(转移性) | ~30% |
| 风险因素 | 年龄>50/家族史/黑人/BRCA2 |

### 筛查与诊断

| 方法 | 说明 |
|------|------|
| PSA | >4ng/mL需进一步评估(敏感非特异) |
| DRE(直肠指检) | 触诊异常硬结 |
| MRI(mpMRI) | PI-RADS评分(≥3需活检) |
| 前列腺活检 | 12针系统活检/MRI融合靶向活检 |
| Gleason评分 | 分级分组1-5(GG1=低风险) |

### 前列腺癌治疗选择

| 风险 | 治疗 | 说明 |
|------|------|------|
| 极低/低风险 | 主动监测 | PSA+MRI定期随访; 避免过度治疗 |
| 中风险 | 根治手术或放疗 | 手术(机器人辅助)/外放射/近距离放疗 |
| 高风险 | 多模式治疗 | 手术+放疗+激素治疗 |
| 转移性 | 系统治疗 | ADT+化疗/新型抗雄 |

### 前列腺癌与心理

| 维度 | 影响 |
|------|------|
| 诊断冲击 | 焦虑/恐惧/不确定感 |
| 治疗副作用 | ED/尿失禁→自尊↓/关系↓ |
| 主动监测焦虑 | "带癌生存"的心理负担 |
| ADT副作用 | 热潮红/抑郁/认知↓/体成分变化 |
| 干预 | 心理支持/运动/伴侣辅导 |

## 前列腺保护策略

### 生活方式建议

| 策略 | 证据 | 建议 |
|------|------|------|
| 运动 | B | 中等有氧150min/周(BPH/PCa预防) |
| 健康体重 | B | BMI↑与高级别PCa相关 |
| 番茄红素 | B-C | 番茄制品(抗氧化) |
| 减少红肉/加工肉 | B | WHO/IARC建议 |
| 绿茶 | C | 儿茶素(初步证据) |
| 大豆/异黄酮 | C | 亚洲人群低发病率(相关性) |
| 减少乳制品 | C | 高钙摄入可能↑风险 |
