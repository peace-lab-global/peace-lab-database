## 睾酮基础

### 核心数据

| 指标 | 参考值 |
|------|--------|
| 总睾酮(成年男性) | 300-1000 ng/dL (10.4-34.7 nmol/L) |
| 游离睾酮 | 5-21 pg/mL |
| 日变化 | 晨起最高，下午/晚间降低 |
| 年龄下降 | 30岁后每年↓1-2% |
| 分泌来源 | 睾丸95% + 肾上腺5% |

### HPG轴(下丘脑-垂体-性腺轴)

```
下丘脑
  ↓ GnRH(脉冲式分泌)
垂体前叶
  ↓ LH → 睾丸Leydig细胞 → 睾酮
  ↓ FSH → 睾丸Sertoli细胞 → 精子生成
     ↑
  (负反馈) ← 睾酮/雌二醇/抑制素B
```

### 睾酮代谢

| 代谢物 | 酶 | 功能 |
|--------|-----|------|
| DHT(二氢睾酮) | 5α-还原酶 | 前列腺/毛囊/皮肤(10倍活性) |
| 雌二醇(E2) | 芳香化酶 | 骨密度/脂代谢/脑功能 |
| SHBG结合型 | — | 无活性(占60-70%) |
| 白蛋白结合型 | — | 弱结合(占25-35%) |
| 游离睾酮 | — | 有生物活性(占1-3%) |

## 睾酮的生理功能

### 多系统效应

| 系统 | 作用 |
|------|------|
| 肌肉骨骼 | 蛋白合成↑/肌肉量↑/骨密度维持 |
| 生殖 | 精子生成/性欲/勃起功能 |
| 代谢 | 脂肪分布/胰岛素敏感性/红细胞生成 |
| 心血管 | 血管扩张(低剂量)/血脂影响(复杂) |
| 认知 | 空间记忆/注意力/决策 |
| 情绪 | 自信/动力/攻击性调节 |
| 免疫 | 免疫调节(一般抑制性) |

### 睾酮与身体成分

| 睾酮水平 | 肌肉 | 脂肪 | 代谢 |
|---------|------|------|------|
| 正常高值 | 合成代谢↑ | 脂肪分解↑ | 代谢率↑ |
| 正常低值 | 维持 | 中性 | 基础 |
| 低于正常 | 肌肉↓ | 内脏脂肪↑ | 胰岛素抵抗↑ |

## 低睾酮症

### 诊断标准

| 维度 | 标准 |
|------|------|
| 症状 | 性欲↓/ED/疲劳/肌肉↓/情绪低/认知↓ |
| 实验室 | 晨起总睾酮<300 ng/dL(至少2次) |
| 游离睾酮 | <5 pg/mL |
| 排除 | 急性疾病/垂体疾病/药物影响 |

### 低睾酮的原因

| 分类 | 原因 | 特征 |
|------|------|------|
| 原发性(睾丸) | Klinefelter/损伤/感染 | LH↑ FSH↑ T↓ |
| 继发性(中枢) | 垂体瘤/肥胖/药物 | LH↓(或正常) T↓ |
| 迟发性(LOH) | 年龄相关 | 渐进性↓ 症状轻微 |
| 功能性 | 肥胖/糖尿病/压力/睡眠不足 | 可逆(去除病因) |

### 功能性低睾酮的常见原因

| 因素 | 机制 | 睾酮影响 |
|------|------|---------|
| 肥胖 | 芳香化酶↑→T→E2转化↑ | ↓20-50% |
| 慢性压力 | 皮质醇↑抑制GnRH | ↓15-30% |
| 睡眠不足(<5h) | GnRH脉冲减少 | ↓10-15% |
| 过度运动 | 能量赤字/皮质醇↑ | 显著↓ |
| 高糖饮食 | 急性降低(75g糖负荷) | 急性↓25% |
| 酒精过量 | 睾丸毒性/HPG抑制 | 剂量依赖↓ |
| 阿片类药物 | GnRH抑制 | 显著↓ |

## 自然优化睾酮

### 循证方法

| 方法 | 效果量 | 机制 | 建议 |
|------|--------|------|------|
| 力量训练 | ★★★★★ | 急性升高+长期改善体成分 | 复合动作/大肌群/中高负荷 |
| 减脂(肥胖者) | ★★★★★ | 芳香化酶↓/SHBG正常化 | 每↓1kg BMI→T↑~2% |
| 充足睡眠 | ★★★★☆ | GnRH脉冲优化 | 7-9h/晚 |
| 压力管理 | ★★★★☆ | 皮质醇-睾酮拮抗↓ | 冥想/放松/认知重构 |
| 维生素D | ★★★☆☆ | VDR在睾丸表达 | 维持>30ng/mL |
| 锌补充(缺乏者) | ★★★☆☆ | 精子生成/Leydig细胞 | 15-30mg/天 |
| 镁补充 | ★★★☆☆ | SHBG结合竞争 | 200-400mg/天 |
| 限制酒精 | ★★★☆☆ | 减少睾丸抑制 | ≤2杯/天 |
| 健康脂肪摄入 | ★★★☆☆ | 胆固醇→睾酮前体 | 脂肪占热量25-35% |

### 力量训练优化睾酮的原则

| 原则 | 建议 |
|------|------|
| 动作选择 | 深蹲/硬拉/卧推(复合多关节) |
| 强度 | 70-85% 1RM |
| 组数/次数 | 3-5组, 6-12次 |
| 间歇 | 60-90秒 |
| 频率 | 3-4次/周 |
| 注意 | 避免过度训练(皮质醇↑→T↓) |

## 睾酮替代疗法

### TRT适应证与禁忌

| 适应证 | 禁忌证 |
|--------|--------|
| 症状性低睾酮(<300ng/dL, 2次) | 前列腺癌/乳腺癌 |
| LOH伴显著症状 | 红细胞增多症(Hct>54%) |
| 原发性/继发性性腺功能减退 | 未控制的心力衰竭 |
| — | 未治疗的睡眠呼吸暂停 |
| — | 备孕(TRT抑制精子生成) |

### TRT给药方式

| 方式 | 频率 | 优点 | 缺点 |
|------|------|------|------|
| 肌肉注射 | 1-2周/次 | 经济/有效 | 水平波动 |
| 凝胶/贴片 | 每日 | 稳定水平 | 皮肤传递风险/过敏 |
| 皮下埋植 | 3-6月 | 稳定/便利 | 需小手术/感染 |
| 鼻腔凝胶 | 每日2-3次 | 无皮肤传递 | 鼻腔刺激 |
| 口服(十一酸) | 每日2次 | 方便 | 需随餐/吸收变异 |

### TRT监测

| 指标 | 频率 | 目标 |
|------|------|------|
| 总睾酮 | 3-6月 | 400-700 ng/dL |
| 红细胞压积 | 3-6月 | <54% |
| PSA | 3-6月→年度 | 无异常升高 |
| 骨密度 | 基线+1-2年 | 改善或稳定 |
| 肝功能 | 基线+年度 | 正常范围 |
| 血脂 | 年度 | 正常范围 |
