## 前列腺健康

### 前列腺疾病概览

前列腺（Prostate）是男性特有的器官，位于膀胱下方、包绕尿道。随年龄增长，前列腺疾病几乎影响所有男性，是男性健康最重要的关注点之一。

| 疾病 | 患病率 | 好发年龄 | 严重程度 |
|------|--------|---------|---------|
| 良性前列腺增生（BPH） | 50岁以上约50%，80岁以上>80% | 50岁+ | 中度（影响生活质量） |
| 前列腺炎 | 约50%男性一生中经历 | 各年龄段 | 轻度至中度 |
| 前列腺癌 | 男性第二大癌症 | 65岁+高峰 | 可严重（早期可治愈） |

### 前列腺癌筛查

**PSA（前列腺特异性抗原）筛查指南**：

| 年龄 | 建议 | PSA参考值 |
|------|------|----------|
| 40-49岁 | 有家族史或非裔族群建议筛查 | <2.5 ng/mL |
| 50-54岁 | 建议与医生讨论利弊 | <3.5 ng/mL |
| 55-69岁 | 建议每2年筛查一次 | <4.0 ng/mL |
| 70岁+ | 个体化决策 | 视整体健康状态 |

**PSA解读注意事项**：
- PSA升高不等于前列腺癌（BPH、前列腺炎、射精、骑行均可升高）
- PSA速率（每年上升>0.75 ng/mL）比单次数值更重要
- 游离PSA/总PSA比值可辅助判断（<25%提示较高恶性风险）
- 前列腺MRI是多参数评估的重要工具

### 前列腺健康生活方式

| 策略 | 具体建议 | 证据 |
|------|---------|------|
| 饮食 | 富含番茄红素（番茄）、硒（巴西坚果）、绿茶 | 降低前列腺癌风险 |
| 运动 | 每周150分钟中等强度有氧运动 | BPH风险↓30% |
| 体重管理 | BMI<25，腰围<90cm | 肥胖→前列腺癌进展风险↑ |
| 补充剂 | 讨论5α还原酶抑制剂的预防性使用 | 需医生评估 |
| 性生活 | 规律射精（≥21次/月） | 前列腺癌风险↓（哈佛研究） |
| 避免久坐 | 每1小时起身活动 | 减少前列腺充血 |

## 睾酮管理

### 睾酮概述

睾酮（Testosterone, T）是男性最重要的雄性激素，对性功能、肌肉量、骨密度、红细胞生成、情绪和认知功能都有深远影响。

| 年龄段 | 总睾酮正常范围 | 游离睾酮趋势 |
|--------|--------------|------------|
| 20-30岁 | 300-1000 ng/dL | 最高峰 |
| 30-40岁 | - | 开始每年下降约1% |
| 40-50岁 | - | 可能出现明显下降 |
| 50岁+ | - | 20-30%男性低于正常 |

### 睾酮缺乏（男性性腺功能减退）

**症状评估**：

| 症状类别 | 具体表现 |
|---------|---------|
| 性功能 | 性欲↓、勃起功能障碍、晨勃减少 |
| 身体变化 | 肌肉量↓、体脂↑（特别是腹部）、乳房发育 |
| 精力 | 持续疲劳、耐力下降、恢复变慢 |
| 情绪 | 抑郁、易怒、动力不足、自信下降 |
| 认知 | 注意力下降、记忆力减退 |
| 骨骼 | 骨密度↓→骨质疏松风险↑ |
| 代谢 | 胰岛素敏感性↓、血脂异常 |

**诊断标准**：
- 存在上述症状（≥2个以上）
- 两次晨间总睾酮<300 ng/dL（或游离睾酮低于参考范围）
- 排除急性疾病/药物影响
- 评估SHBG（性激素结合球蛋白）水平

### 睾酮替代治疗（TRT）

| 项目 | 内容 |
|------|------|
| 适应症 | 确诊睾酮缺乏+症状明显 |
| 给药方式 | 凝胶（日常）、注射（每1-2周）、贴片、皮下植入 |
| 预期效果 | 2-4周开始改善性欲和情绪；3-6个月改善身体成分 |
| 监测指标 | 睾酮水平、PSA、红细胞比容（HCT）、肝功能、血脂 |

**TRT的风险与争议**：

| 方面 | 争议点 | 当前共识 |
|------|--------|---------|
| 心血管风险 | 早期研究提示↑；后续研究未确认 | 无确切证据显示TRT增加心血管风险，但需监测 |
| 前列腺癌 | 理论上的担忧 | TRT不会导致前列腺癌，但可能促进已有癌变 |
| 生育影响 | 外源性睾酮→精子生成抑制 | 有生育需求者不应使用TRT，可考虑HCG/Clomiphene |
| 红细胞增多 | HCT↑→血栓风险 | 定期监测，必要时放血或调整剂量 |

### 自然提升睾酮的方法

| 方法 | 具体建议 | 预期效果 |
|------|---------|---------|
| 力量训练 | 复合动作（深蹲、硬拉），中高强度 | 短期T↑15-30% |
| 充足睡眠 | 7-9小时/晚，规律作息 | 睡眠剥夺→T↓10-15% |
| 体重管理 | 减少体脂至<20% | 肥胖→芳香化酶↑→T→E2 |
| 锌补充 | 15-30mg/天（缺乏者） | 锌缺乏→T↓ |
| 维生素D | 维持血清25(OH)D >30ng/mL | VDD与低T相关 |
| 减压 | 冥想、运动、社交 | 皮质醇↑→T↓（负相关） |
| 减少酒精 | 适量或不饮酒 | 过量酒精→T↓ |
| 避免环境内分泌干扰 | 减少接触BPA、邻苯二甲酸酯 | 异种雌激素→T↓ |

## 男性更年期

### 概念与争议

男性更年期（Andropause / Late-Onset Hypogonadism, LOH）是指中老年男性因睾酮水平逐渐下降而出现的一系列身心症状。与女性更年期的急剧激素变化不同，男性睾酮下降是渐进的。

| 对比 | 女性更年期 | 男性更年期 |
|------|-----------|-----------|
| 激素变化 | 雌激素在1-3年内急剧下降 | 睾酮每年下降约1%，渐进过程 |
| 发生率 | 几乎100%女性 | 约20-30%男性有明显症状 |
| 诊断 | 停经12个月 | 症状+低睾酮（需两次确认） |
| 不适程度 | 可非常显著 | 通常较轻但可能被忽视 |

### 男性更年期症状

| 维度 | 症状 | 发生机制 |
|------|------|---------|
| 体能 | 疲劳、肌力下降、耐力下降 | T↓→肌肉蛋白合成↓ |
| 性功能 | 性欲↓、ED、射精量减少 | T↓+血管功能↓ |
| 情绪 | 抑郁、易怒、动力丧失 | T↓→5-HT/DA系统影响 |
| 认知 | 记忆力↓、注意力↓ | T对海马的保护作用减弱 |
| 代谢 | 腹部脂肪↑、胰岛素抵抗↑ | T↓→脂肪分布改变 |
| 骨骼 | 骨密度↓ | T↓→骨形成↓ |
| 睡眠 | 失眠、睡眠质量↓ | T/褪黑素交互影响 |

### 管理策略

**分层管理方案**：

| 层次 | 策略 | 适用对象 |
|------|------|---------|
| 第一层 | 生活方式优化（运动、睡眠、饮食、减压） | 所有中年男性 |
| 第二层 | 针对性补充（锌、维生素D、DHEA评估） | 有轻度症状者 |
| 第三层 | 药物治疗（Clomiphene/HCG刺激自身T产生） | 有生育需求者 |
| 第四层 | 睾酮替代治疗（TRT） | 确诊LOH且无禁忌症者 |
| 第五层 | 多学科管理（心理+运动+营养+药物） | 复杂病例 |

## 心血管风险预防

### 男性心血管风险评估

心血管疾病（CVD）是男性第一大死因。男性比女性早10-15年出现心血管事件，这主要归因于睾酮对代谢的双刃剑效应、更高的吸烟率和风险行为模式。

**风险评估工具**：

| 工具 | 适用 | 内容 |
|------|------|------|
| Framingham风险评分 | 一般人群 | 10年CVD风险百分比 |
| ASCVD风险计算器 | 美国指南 | 10年动脉粥样硬化性CVD风险 |
| QRISK3 | 英国指南 | 包含更多风险因素 |
| 冠状动脉钙化评分（CAC） | 中等风险者 | 直接评估动脉粥样硬化负担 |

### 关键风险因素管理

| 风险因素 | 目标值 | 干预措施 |
|---------|--------|---------|
| 血压 | <130/80 mmHg | DASH饮食、运动、减盐、药物 |
| LDL胆固醇 | <100 mg/dL（高危<70） | 地中海饮食、运动、他汀类药物 |
| 血糖 | HbA1c <5.7% | 饮食控制、运动、减重 |
| BMI | 18.5-24.9 | 热量控制、运动 |
| 腰围 | <90cm | 减少内脏脂肪 |
| 吸烟 | 戒烟 | 尼古丁替代、行为支持 |
| 体力活动 | ≥150分钟/周中等强度 | 有氧+力量训练 |

### 男性心血管健康的时间线

| 年龄 | 重点 |
|------|------|
| 20-30岁 | 建立健康习惯（运动、饮食、不吸烟），基线检查 |
| 30-40岁 | 每2年体检，关注血压和血脂，管理压力 |
| 40-50岁 | 开始心血管风险评估，PSA讨论，代谢监测 |
| 50-60岁 | 每年体检，低剂量阿司匹林评估，心脏筛查 |
| 60岁+ | 全面心血管评估，多重用药管理，功能维护 |

## 心理健康筛查

### 男性心理健康的特殊性

男性心理健康面临独特的挑战：社会对"男子气概"的期望导致男性更少寻求帮助、更多使用不健康的应对策略。

| 指标 | 数据 |
|------|------|
| 男性自杀率 | 女性的3-4倍（全球） |
| 男性寻求心理帮助率 | 约为女性的一半 |
| 男性物质滥用率 | 显著高于女性 |
| 男性抑郁漏诊率 | 约50-65%的男性抑郁被漏诊 |

### 男性抑郁的特殊表现

男性抑郁常以非典型方式表现，导致漏诊：

| 传统抑郁症状 | 男性常见表现 |
|------------|------------|
| 情绪低落 | 易怒、攻击性↑ |
| 哭泣/悲伤 | 情感麻木、冷漠 |
| 体重变化 | 过度工作/运动 |
| 失眠 | 酗酒/物质使用 |
| 疲劳 | 冒险行为↑ |
| 自我评价低 | 过度竞争/控制行为 |

### 筛查工具

| 工具 | 特点 | 适用场景 |
|------|------|---------|
| PHQ-9 | 9项，标准化评分 | 初级保健常规筛查 |
| GAD-7 | 7项焦虑筛查 | 焦虑症状评估 |
| AUDIT | 10项饮酒评估 | 酒精使用问题筛查 |
| PHQ-2 | 2项快速筛查 | 快速筛查大门 |
| Kessler-6 | 6项心理困扰 | 社区筛查 |
| 男性抑郁量表（MDI） | 特异性关注男性表现 | 男性特异性评估 |

### 系统性筛查建议

**全生命周期心理健康筛查**：

| 年龄 | 建议筛查频率 | 关注重点 |
|------|------------|---------|
| 18-30岁 | 每年 | 学业/职业压力、人际关系、物质使用 |
| 30-50岁 | 每年 | 职业倦怠、婚姻/家庭关系、中年危机 |
| 50-65岁 | 每年 | 更年期情绪变化、空巢综合征、退休规划 |
| 65岁+ | 每6个月 | 认知功能、社会孤立、悲伤/丧失 |

### 男性心理健康促进

| 策略 | 具体建议 | 原理 |
|------|---------|------|
| 运动处方 | 每周3-5次，30-60分钟 | 运动→BDNF↑、5-HT↑、内啡肽↑ |
| 社交连接 | 维护至少2-3个深度友谊 | 社会隔离→抑郁/自杀风险↑ |
| 情绪素养 | 学习识别和命名情绪 | 情绪觉察→更好的应对 |
| 压力管理 | 正念冥想、呼吸练习 | 降低皮质醇、增强情绪调节 |
| 专业求助 | 将寻求帮助视为力量而非弱点 | 早期干预效果更好 |
| 睡眠优化 | 7-9小时/晚，规律作息 | 睡眠与心理健康双向影响 |
| 减少酒精 | ≤14标准杯/周 | 酒精是中枢神经抑制剂→加重抑郁 |
| 意义与目的 | 志愿服务、兴趣培养、代际传承 | 存在意义感→心理韧性↑ |
