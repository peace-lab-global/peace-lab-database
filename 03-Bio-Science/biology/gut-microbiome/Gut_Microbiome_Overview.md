## 微生物组基础

### 核心数据

| 指标 | 数据 |
|------|------|
| 肠道微生物细胞数 | ~38万亿（与人体细胞数量接近） |
| 微生物基因数 | ~330万（人类基因约2万） |
| 总重量 | 约1.5-2 kg |
| 物种数 | 1,000-1,500种（个体500-1,000种） |
| 主要定植部位 | 结肠（密度最高: 10¹¹ CFU/mL） |

### 肠道微生物组 vs 微生物群

| 概念 | 定义 |
|------|------|
| 微生物群(Microbiota) | 特定环境中的微生物群落 |
| 微生物组(Microbiome) | 微生物群+其基因组+代谢产物+环境 |
| 宏基因组(Metagenome) | 环境样本中所有微生物的基因组总和 |
| 代谢组(Metabolome) | 微生物产生的所有代谢产物 |

## 菌群组成与多样性

### 主要菌门

| 菌门 | 占比 | 代表属 | 功能特点 |
|------|------|--------|---------|
| 厚壁菌门(Firmicutes) | 40-60% | Lactobacillus, Clostridium, Ruminococcus | 短链脂肪酸产生 |
| 拟杆菌门(Bacteroidetes) | 20-40% | Bacteroides, Prevotella | 多糖降解 |
| 放线菌门(Actinobacteria) | 3-10% | Bifidobacterium | 免疫调节 |
| 变形菌门(Proteobacteria) | 1-5% | E. coli, Helicobacter | 过多提示失调 |
| 疣微菌门(Verrucomicrobia) | 1-3% | Akkermansia | 黏液层维护 |

### 多样性指标

| 指标 | 定义 | 健康意义 |
|------|------|---------|
| Alpha多样性 | 单个样本内的物种丰富度和均匀度 | 高→通常更健康 |
| Beta多样性 | 不同样本间的菌群组成差异 | 用于比较人群差异 |
| Shannon指数 | 综合丰富度和均匀度 | 常用Alpha指标 |
| F/B比值 | 厚壁菌/拟杆菌比值 | 比值过高与肥胖相关 |

### 关键有益菌

| 菌属 | 功能 | 来源 |
|------|------|------|
| Lactobacillus | 乳酸产生、免疫调节 | 发酵食品 |
| Bifidobacterium | 短链脂肪酸、免疫发育 | 母乳、发酵食品 |
| Akkermansia muciniphila | 黏液层修复、代谢调节 | 高纤维饮食培养 |
| Faecalibacterium prausnitzii | 丁酸产生、抗炎 | 高纤维饮食培养 |

## 菌群建立与生命周期

| 阶段 | 时期 | 关键事件 |
|------|------|---------|
| 宫内 | 孕期 | 争议中——可能有低水平微生物暴露 |
| 出生 | 分娩 | 顺产获得阴道+肠道菌；剖宫产获得皮肤+环境菌 |
| 哺乳期 | 0-6月 | 母乳HMO喂养Bifidobacterium |
| 辅食引入 | 6-12月 | 菌群多样性快速增加 |
| 成熟期 | 2-3岁 | 菌群趋于成人模式 |
| 成人期 | 18-65岁 | 相对稳定、个体差异大 |
| 老年期 | 65岁+ | 多样性下降、Proteobacteria增加 |

### 分娩方式的影响

| 指标 | 顺产 | 剖宫产 |
|------|------|--------|
| 早期优势菌 | Lactobacillus, Bacteroides | Staphylococcus, Clostridium |
| Bifidobacterium定植 | 早(1周内) | 延迟(1-6月) |
| 多样性 | 较高 | 较低(6月内) |
| 长期差异 | 基准 | 多数在1-2岁后趋同 |

## 微生物组功能

| 功能类别 | 具体功能 | 机制 |
|---------|---------|------|
| 营养代谢 | 膳食纤维发酵→SCFA | 丁酸/丙酸/醋酸 |
| 维生素合成 | 维生素K、B族维生素 | 细菌合成 |
| 屏障保护 | 竞争性排斥病原体 | 定植抗力 |
| 免疫训练 | 免疫系统发育和调节 | Treg细胞诱导 |
| 肠-脑轴 | 神经递质合成(5-HT/GABA) | 迷走神经信号 |
| 药物代谢 | 影响药物效果和毒性 | 微生物酶活性 |

### 短链脂肪酸（SCFA）

| SCFA | 主要功能 | 产生菌 |
|------|---------|--------|
| 丁酸(Butyrate) | 肠上皮能源(70%)、抗炎、屏障修复 | Faecalibacterium, Roseburia |
| 丙酸(Propionate) | 糖异生调节、饱腹感信号 | Bacteroides, Propionibacterium |
| 醋酸(Acetate) | 外周组织能源、脂代谢 | 多种菌广泛产生 |

## 检测与评估

| 方法 | 原理 | 优势 | 局限 |
|------|------|------|------|
| 16S rRNA测序 | 扩增细菌标记基因 | 成本低、标准化 | 只到属级、功能推断 |
| 宏基因组测序(WGS) | 全部DNA测序 | 种级分辨、功能分析 | 成本高、分析复杂 |
| 代谢组学 | 测量SCFA等代谢产物 | 功能性信息 | 技术要求高 |
| 培养组学 | 高通量培养 | 获得活菌 | 只能培养30-50% |

## 影响菌群的因素

| 因素 | 影响方向 | 证据强度 |
|------|---------|---------|
| 饮食(最重要) | 高纤维→多样性↑ | ★★★★★ |
| 抗生素 | 多样性急剧↓ | ★★★★★ |
| 运动 | 多样性↑ Akkermansia↑ | ★★★★☆ |
| 压力 | 多样性↓ Proteobacteria↑ | ★★★☆☆ |
| 睡眠 | 昼夜节律影响菌群 | ★★★☆☆ |
| 年龄 | 老年后多样性↓ | ★★★★☆ |
| 药物 | PPI/二甲双胍/NSAID均影响 | ★★★★☆ |
