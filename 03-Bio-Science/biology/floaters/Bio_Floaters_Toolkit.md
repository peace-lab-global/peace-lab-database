### 危险信号识别卡

**需要紧急就医的症状：**
```
🔴 红色警报（立即就医）：
□ 突然出现大量新漂浮物（短时间内成倍增加）
□ 持续性闪光感或光幻视
□ 固定位置的视野缺损或暗影
□ 视力急剧下降（看不清手指或人脸）
□ 伴有眼痛、头痛或恶心呕吐

🟡 黄色警报（尽快就医）：
□ 症状持续加重超过一周
□ 影响日常生活和工作效率
□ 出现焦虑、抑郁等心理症状
□ 有高度近视或其他眼部疾病史
□ 家族中有视网膜脱离病史

🟢 绿色提示（定期观察）：
□ 症状稳定无明显变化
□ 不影响正常生活功能
□ 无其他眼部不适症状
□ 定期眼科检查结果正常
```

## 日常管理检查清单

### 每日护眼习惯清单

** morning routine (晨间护眼)：**
```
🌅 起床后护眼：
□ 用温水清洗眼部周围
□ 进行简单的眼球运动（上下左右各10次）
□ 确保充足睡眠（7-9小时）
□ 早餐包含富含维生素A的食物
□ 避免起床后立即看手机屏幕
```

** daytime care (日间护眼)：**
```
☀️ 工作时间护眼：
□ 遵循20-20-20法则（每20分钟休息20秒）
□ 调整屏幕亮度与环境光线匹配
□ 保持正确的坐姿和用眼距离
□ 定时眨眼，避免干眼症
□ 适量饮水，保持身体水分
□ 午休时闭眼放松10-15分钟
```

** evening routine (晚间护眼)：**
```
🌙 睡前护眼：
□ 睡前1小时停止使用电子设备
□ 进行眼部热敷5-10分钟
□ 做简单的放松冥想或深呼吸
□ 确保卧室光线柔和不刺眼
□ 使用护眼模式或蓝光过滤
□ 涂抹适合的眼霜进行眼部护理
```

### 环境优化检查表

**工作环境设置：**
```
💼 办公室护眼配置：

□ 建立专门的放松休息区域

## 应急处理指南

### 急性症状处理流程

**突发症状应急响应：**
```
🚨 紧急情况处理步骤：

第一步：立即停止当前活动
□ 停止使用电子设备
□ 找安全地方坐下或躺下
□ 避免剧烈头部运动

第二步：症状快速评估
□ 记录症状出现的具体时间
□ 描述漂浮物的数量和特征
□ 注意是否伴有其他症状（闪光、视野缺损等）

第三步：寻求医疗帮助
□ 如有危险信号，立即前往急诊科
□ 拨打急救电话（120）或眼科急诊热线
□ 准备好病历和用药记录

第四步：就医前准备
□ 散瞳眼药水（如有）
□ 身份证件和医保卡
□ 详细症状描述记录
□ 紧急联系人信息
```

### 常见问题应急方案

**轻微不适缓解方法：**
```
😌 日常不适处理：
□ 眼部疲劳：闭眼休息5-10分钟
□ 轻微干涩：使用人工泪液滴眼
□ 注意力过度集中：转移注意力到远处景物
□ 轻微焦虑：进行深呼吸放松练习
□ 睡眠不佳：改善睡眠环境和习惯
```

## 护眼环境优化指南

### 光环境设置标准

**照明参数推荐：**
```
💡 理想光环境指标：
工作区域照明：
├── 照度标准：500-750 lux
├── 色温范围：4000-5000K（自然白光）
├── 显色指数：Ra≥90
├── 眩光控制：UGR≤19
└── 照明均匀度：≥0.7

休息区域照明：
├── 照度标准：150-300 lux
├── 色温范围：2700-3000K（暖白光）
├── 显色指数：Ra≥80
├── 氛围营造：柔和漫反射
└── 夜间模式：≤50 lux
```

### 电子设备护眼设置

**显示屏参数优化：**
```
🖥️ 显示器护眼设置：
屏幕参数：
├── 亮度调节：与环境光线匹配
├── 对比度设置：适中偏高
├── 色彩温度：6500K标准色温
├── 刷新率：≥75Hz（推荐120Hz）
└── 分辨率：原生分辨率显示

护眼功能启用：
├── 蓝光过滤：开启50-70%强度
├── 护眼模式：根据时间自动调节
├── 闪烁控制：DC调光或高频PWM
├── 自动亮度：环境光感应调节
└── 字体放大：125-150%显示比例
