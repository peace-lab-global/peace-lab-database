# 疼痛科学总览 | Pain Science Overview

> **学科定位**: 神经科学 × 临床医学 × 心理学
> **核心主张**: 疼痛是大脑构建的保护性体验，不等于组织损伤，理解疼痛机制是有效管理的基础

---

## 一、核心概念

### 1.1 关键术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 伤害性感受 | Nociception | 感觉神经末梢检测有害刺激的过程 |
| 疼痛 | Pain | 与实际或潜在组织损伤相关的不愉快感觉和情绪体验 |
| 痛觉过敏 | Hyperalgesia | 对正常疼痛刺激的反应增强 |
| 异常性疼痛 | Allodynia | 对正常无痛刺激产生疼痛反应 |
| 中枢敏化 | Central Sensitization | 中枢神经系统放大疼痛信号的状态 |
| 外周敏化 | Peripheral Sensitization | 外周神经末梢敏感度增加 |
| 风上涌现象 | Wind-up | 反复C纤维刺激导致脊髓背角神经元逐渐增强反应 |
| 灾难化 | Pain Catastrophizing | 对疼痛的夸大、反刍和无助感 |

### 1.2 IASP 疼痛定义（2020修订）

> "与实际或潜在的组织损伤相关联的、或类似于此的不愉快的感觉和情感体验。"

**六条附注**:
1. 疼痛始终是个人体验
2. 疼痛不同于伤害性感受
3. 人们通过生活经验学习疼痛
4. 个人对疼痛的报告应被尊重
5. 疼痛通常具有适应性功能
6. 无法交流并不否认疼痛的存在

## 二、疼痛分类

### 2.1 按时间分类

| 类型 | 持续时间 | 功能 | 治疗重点 |
|------|---------|------|---------|
| 急性疼痛 | <3个月 | 保护性警报 | 病因治疗 |
| 亚急性疼痛 | 3-6个月 | 过渡期 | 预防慢性化 |
| 慢性疼痛 | >3个月 | 失去保护功能 | 多模式管理 |

### 2.2 按机制分类（ICD-11）

| 类型 | 英文 | 机制 | 示例 |
|------|------|------|------|
| 伤害性疼痛 | Nociceptive | 组织损伤激活伤害性感受器 | 骨折、关节炎 |
| 神经病理性疼痛 | Neuropathic | 躯体感觉神经系统损伤 | 糖尿病神经痛、带状疱疹后 |
| 可塑性疼痛 | Nociplastic | 中枢疼痛处理异常 | 纤维肌痛、慢性腰痛 |
| 混合型 | Mixed | 多机制并存 | 复杂区域疼痛综合征 |

## 三、疼痛的神经科学

### 3.1 上行疼痛通路

| 阶段 | 结构 | 功能 |
|------|------|------|
| 1. 转导 | 外周伤害性感受器（Aδ/C纤维） | 将有害刺激转化为电信号 |
| 2. 传导 | 脊髓背角 → 脊髓丘脑束 | 信号从外周传至脊髓 |
| 3. 调节 | 脊髓背角门控 | 闸门理论（Melzack & Wall, 1965） |
| 4. 投射 | 丘脑 → 大脑皮层 | 信号传至意识层面 |
| 5. 感知 | 疼痛矩阵（多脑区网络） | 产生疼痛体验 |

### 3.2 疼痛矩阵（Pain Matrix/Neuromatrix）

| 脑区 | 功能 |
|------|------|
| 初级体感皮层(S1) | 疼痛位置、强度 |
| 次级体感皮层(S2) | 疼痛质量 |
| 岛叶 | 情绪-感觉整合 |
| 前扣带回(ACC) | 疼痛的不愉快感/情绪维度 |
| 前额叶皮质 | 认知评估、注意力 |
| 杏仁核 | 恐惧/焦虑与疼痛 |
| 导水管周围灰质(PAG) | 下行抑制 |

### 3.3 闸门控制理论

> Melzack & Wall (1965) — 疼痛科学的革命性理论

| 机制 | 说明 |
|------|------|
| 闸门开放 | C纤维活动 → 增加疼痛信号传递 |
| 闸门关闭 | Aβ纤维活动（触摸、振动） → 抑制疼痛传递 |
| 下行调节 | 大脑可以从上而下开/关闸门 |
| 临床意义 | 揉搓撞伤处减痛、TENS疗法、注意力转移 |

## 四、慢性疼痛的生物心理社会模型

### 4.1 三维度模型

| 维度 | 因素 | 示例 |
|------|------|------|
| 生物 | 组织损伤、神经敏化、炎症、遗传 | 椎间盘突出、中枢敏化 |
| 心理 | 信念、情绪、行为、应对方式 | 灾难化、运动恐惧、抑郁 |
| 社会 | 文化、社会支持、工作环境、医疗系统 | 残疾补偿、工作压力 |

### 4.2 疼痛慢性化风险因素（Yellow Flags）

| 类别 | 风险因素 |
|------|---------|
| 认知 | 灾难化思维、恐惧-回避信念、低自我效能 |
| 情绪 | 抑郁、焦虑、愤怒、创伤经历 |
| 行为 | 过度休息或过度活动、被动应对 |
| 社会 | 缺乏支持、工作不满、赔偿诉讼 |
| 生活方式 | 睡眠差、缺乏运动、肥胖 |

## 六、疼痛教育

### 6.1 疼痛神经科学教育（PNE）

> 也称治疗性疼痛神经科学教育（TPNE）

| 维度 | 内容 |
|------|------|
| 定义 | 教导患者理解疼痛的神经生物学机制 |
| 核心信息 | 疼痛≠损伤、大脑在建构疼痛、中枢敏化可逆转 |
| 证据 | 减少灾难化(d=0.39)、减少恐惧-回避(d=0.43)、改善功能 |
| 适用 | 慢性疼痛、术前教育、运动恐惧 |

### 6.2 核心教育要点

| 要点 | 简述 |
|------|------|
| 疼痛是保护系统 | 疼痛的目的是保护，不是报告损伤 |
| 影像不等于疼痛 | 很多影像异常（如椎间盘突出）在无痛人群中也常见 |
| 大脑预测疼痛 | 疼痛受预期、情绪、注意力、上下文影响 |
| 运动是良药 | 渐进式运动可以减敏化，而非造成损伤 |
| 敏化可逆转 | 中枢敏化不是永久性的，可以通过治疗改善 |
