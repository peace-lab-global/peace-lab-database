# 慢性疼痛心理干预 | Pain Psychology Intervention

> **学科定位**: 健康心理学 × 行为医学 × 疼痛康复
> **核心主张**: 心理干预是慢性疼痛多模式管理的必要组成部分，可改变疼痛体验与功能

---

## 一、认知行为疗法与疼痛（CBT for Pain）

### 1.1 概述

| 维度 | 内容 |
|------|------|
| 证据级别 | 慢性疼痛心理干预的金标准 |
| 效应量 | 疼痛强度 d=0.30-0.50；功能障碍 d=0.40-0.60 |
| 疗程 | 通常 8-12 次（个体或团体） |
| 机制 | 改变疼痛相关不良认知、行为和情绪 |

### 1.2 CBT疼痛管理核心模块

| 模块 | 内容 | 目标 |
|------|------|------|
| 心理教育 | 疼痛的生物心理社会模型 | 概念重构 |
| 认知重构 | 识别和挑战灾难化思维 | 减少灾难化 |
| 活动分级 | 渐进性增加活动水平 | 克服回避行为 |
| 放松训练 | 渐进性肌肉放松、腹式呼吸 | 降低交感兴奋 |
| 注意力管理 | 分散注意力技术、注意力转移 | 减少疼痛聚焦 |
| 睡眠卫生 | 睡眠限制、刺激控制 | 改善睡眠质量 |
| 目标设定 | SMART目标、价值导向活动 | 功能恢复 |
| 复发预防 | 应对计划、自我管理策略 | 维持获益 |

### 1.3 恐惧-回避模型

> Vlaeyen & Linton (2000) — 慢性疼痛核心心理模型

| 阶段 | 过程 |
|------|------|
| 损伤/疼痛 | 初始疼痛体验 |
| 灾难化解释 | "这一定是严重的问题" |
| 疼痛相关恐惧 | 恐惧运动(Kinesiophobia) |
| 回避行为 | 减少活动、过度保护 |
| 废用/残疾 | 肌肉萎缩、功能下降、抑郁 |
| 更多疼痛 | 去调适(Deconditioning)加重疼痛 → 恶性循环 |

**干预切入点**: 分级暴露（Graded Exposure in Vivo）— 从最低恐惧活动逐步暴露

## 二、接纳承诺疗法与疼痛（ACT for Pain）

### 2.1 概述

| 维度 | 内容 |
|------|------|
| 核心理念 | 不追求消除疼痛，而是在有疼痛的情况下活出有意义的生活 |
| 目标 | 增加心理灵活性（Psychological Flexibility） |
| 证据 | 与CBT效果相当；在接纳和价值维度可能更优 |

### 2.2 ACT六大核心过程在疼痛中的应用

| 过程 | 英文 | 疼痛中的应用 |
|------|------|-------------|
| 接纳 | Acceptance | 愿意体验疼痛而不与之斗争 |
| 认知解离 | Defusion | "我有一个想法：疼痛会毁了我的生活"（去字面化） |
| 此时此刻 | Present Moment | 觉察当下疼痛体验而非预期未来 |
| 觉察自我 | Self as Context | 我不是"慢性疼痛患者"，我是体验疼痛的人 |
| 价值澄清 | Values | 什么对我最重要？（家庭/工作/健康/关系） |
| 承诺行动 | Committed Action | 即使有疼痛，也朝价值方向行动 |

## 三、其他心理干预

### 3.1 正念基础干预（MBI for Pain）

| 项目 | 特点 | 证据 |
|------|------|------|
| MBSR | 8周标准课程，含身体扫描、正念瑜伽 | ★★★★ |
| MBCT | 结合认知疗法，预防疼痛复发 | ★★★ |
| 正念冥想 | 独立正念练习 | ★★★ |

**Zeidan et al. (2011)**: 仅4天正念训练即显著降低疼痛不愉快感(57%)和疼痛强度(40%)。

### 3.2 催眠与疼痛

| 维度 | 内容 |
|------|------|
| 机制 | 改变疼痛感知的神经加工 |
| 证据 | 急性和慢性疼痛均有效(d=0.50-0.80) |
| 神经影像 | 催眠建议可选择性改变ACC（情绪维度）或S1（感觉维度）活动 |
| 适用 | 手术疼痛、癌症疼痛、IBS、纤维肌痛 |

### 3.3 生物反馈

| 类型 | 监测指标 | 疼痛适应证 |
|------|---------|-----------|
| EMG反馈 | 肌电活动 | 紧张性头痛、颞下颌关节 |
| 温度反馈 | 皮肤温度 | 偏头痛、雷诺病 |
| HRV反馈 | 心率变异 | 慢性疼痛全面管理 |
| 神经反馈 | EEG | 纤维肌痛、慢性疼痛 |

## 四、运动与疼痛

### 4.1 分级运动暴露

| 阶段 | 内容 |
|------|------|
| 评估 | 建立恐惧-回避层级表（PHODA） |
| 教育 | 解释疼痛≠损伤，运动是安全的 |
| 暴露 | 从最低恐惧活动开始 |
| 推进 | 逐步挑战更高恐惧等级的活动 |
| 泛化 | 将获益迁移到日常生活 |

### 4.2 分级活动（Graded Activity）

| 原则 | 说明 |
|------|------|
| 基线 | 确定当前可耐受的活动水平 |
| 配额制 | 按时间/数量计划，非按疼痛停止 |
| 渐进 | 每次增加小幅度(5-10%) |
| 一致性 | 好日子不多做，坏日子不少做 |
