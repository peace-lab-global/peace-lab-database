# 疼痛的身心疗法 | Pain Mind-Body Approaches

> **学科定位**: 整合医学 × 身心医学 × 补充替代疗法
> **核心主张**: 身心疗法通过调节自主神经、改变疼痛加工、增强自我效能来管理慢性疼痛

---

## 一、正念与疼痛

### 1.1 正念疼痛管理机制

| 机制 | 说明 | 神经基础 |
|------|------|---------|
| 感觉-情感解耦 | 将疼痛感觉与情绪反应分离 | 降低前扣带回与岛叶耦合 |
| 去灾难化 | 减少对疼痛的夸大反应 | 增强前额叶对杏仁核的调控 |
| 注意力再分配 | 从自动反刍转向此刻觉察 | 改变默认模式网络活动 |
| 接纳态度 | 减少与疼痛的心理抗争 | 降低ACC活动 |
| 自我调节 | 增强身体觉察和自主调节 | 增强岛叶内感受觉知 |

### 1.2 身体扫描与疼痛

| 步骤 | 说明 |
|------|------|
| 1 | 舒适仰卧，温和闭眼 |
| 2 | 从脚趾开始，逐步扫描全身 |
| 3 | 遇到疼痛区域时，不评判地觉察 |
| 4 | 将呼吸"送入"疼痛区域 |
| 5 | 观察疼痛的质量、形状、温度变化 |
| 6 | 不试图改变，只是观察 |
| 7 | 继续向上扫描 |

### 1.3 MBSR疼痛项目

| 周次 | 主题 | 与疼痛关联 |
|------|------|-----------|
| 1-2 | 身体扫描 | 觉察疼痛而非反应 |
| 3-4 | 坐禅、行走冥想 | 与疼痛共处 |
| 5-6 | 正念瑜伽 | 安全的身体活动 |
| 7 | 全天静修 | 深入疼痛觉察 |
| 8 | 整合与维持 | 日常疼痛管理 |

## 二、瑜伽与疼痛

### 2.1 瑜伽疼痛管理证据

| 疼痛类型 | 瑜伽类型 | 证据级别 | 效应量 |
|---------|---------|---------|--------|
| 慢性腰痛 | Iyengar/Viniyoga | ★★★★ | d=0.44-0.73 |
| 颈部疼痛 | 温和哈他瑜伽 | ★★★ | d=0.33-0.52 |
| 骨关节炎 | 椅子瑜伽/温和瑜伽 | ★★★ | d=0.30-0.50 |
| 纤维肌痛 | 瑜伽尼德拉/恢复性瑜伽 | ★★★ | d=0.28-0.45 |
| 偏头痛 | 呼吸法+冥想 | ★★ | 待确认 |
| 类风湿关节炎 | 温和瑜伽 | ★★ | d=0.20-0.35 |

### 2.2 瑜伽疼痛管理原则

| 原则 | 说明 |
|------|------|
| 不痛原则 | 练习不应增加疼痛 |
| 个体化 | 根据疼痛类型和程度调整 |
| 呼吸优先 | 呼吸比体位更重要 |
| 渐进性 | 从最温和的形式开始 |
| 整合性 | 结合体位、呼吸、冥想 |
| 自主性 | 培养自我管理能力 |

## 三、太极与疼痛

### 3.1 证据基础

| 疼痛类型 | 太极类型 | 证据 | 关键研究 |
|---------|---------|------|---------|
| 膝骨关节炎 | 杨式太极 | ★★★★ | Wang et al. (2016): 52周太极≥物理治疗 |
| 纤维肌痛 | 杨式太极 | ★★★★ | Wang et al. (2018): 太极优于有氧运动 |
| 慢性腰痛 | 简化太极 | ★★★ | 10周显著改善 |
| 类风湿关节炎 | 适应太极 | ★★ | 改善功能和疼痛 |

### 3.2 太极疼痛管理机制

| 机制 | 说明 |
|------|------|
| 运动恐惧消除 | 缓慢温和的运动降低运动恐惧 |
| 本体感觉增强 | 改善身体觉察和姿势控制 |
| 放松反应 | 副交感神经激活 |
| 社交支持 | 团体练习的心理益处 |
| 自我效能 | 增强对疼痛管理的信心 |

## 四、催眠与疼痛

### 4.1 催眠镇痛机制

| 层面 | 机制 |
|------|------|
| 认知 | 改变疼痛注意力、预期和解释 |
| 情感 | 减少疼痛相关焦虑和不愉快感 |
| 神经 | 选择性调节ACC（情绪维度）或S1（感觉维度）活动 |
| 下行抑制 | 激活PAG下行疼痛抑制通路 |
| 内啡肽 | 部分通过内源性阿片系统介导 |

### 4.2 催眠疼痛干预证据

| 疼痛类型 | 效应量 | 特点 |
|---------|--------|------|
| 手术疼痛 | d=0.60-0.80 | 减少术后镇痛药物需求 |
| 癌症疼痛 | d=0.50-0.70 | 改善生活质量 |
| 肠易激综合征 | d=0.60-0.90 | 长期缓解率75% |
| 纤维肌痛 | d=0.40-0.60 | 改善疼痛和睡眠 |
| 烧伤换药 | d=0.70-1.00 | 显著减少操作性疼痛 |

## 五、其他身心疗法

### 5.1 针灸

| 维度 | 内容 |
|------|------|
| 机制 | 激活下行疼痛抑制、释放内啡肽、减少炎症 |
| 证据 | 慢性疼痛优于假针灸(d=0.20)，优于无治疗(d=0.50) |
| 适应证 | 偏头痛、腰痛、膝骨关节炎、颈痛 |

### 5.2 呼吸训练

| 技术 | 方法 | 疼痛机制 |
|------|------|---------|
| 缓慢深呼吸 | 4-6次/分钟 | 激活迷走神经、降低交感兴奋 |
| 交替鼻孔呼吸 | Nadi Shodhana | 平衡自主神经 |
| 4-7-8呼吸 | 吸4-屏7-呼8 | 快速放松反应 |

### 5.3 音乐疗法

| 维度 | 内容 |
|------|------|
| 机制 | 分散注意力、情绪调节、释放多巴胺 |
| 证据 | 急性疼痛(d=0.40)、术后疼痛(d=0.50)、慢性疼痛(d=0.30) |
| 选择 | 患者自选音乐效果优于研究者指定 |
