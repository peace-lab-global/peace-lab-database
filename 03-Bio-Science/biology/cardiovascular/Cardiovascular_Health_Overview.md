## 心血管系统基础

### 核心数据

| 指标 | 数据 |
|------|------|
| 心脏每日泵血量 | ~7,500升 |
| 血管总长度 | ~96,000公里 |
| 心率(静息) | 60-100次/分 |
| 心输出量 | 约5升/分(静息) |
| CVD全球死亡 | 每年约1,790万(占总死亡31%) |

### 心血管疾病(CVD)分类

| 类型 | 描述 | 占CVD死亡比例 |
|------|------|-------------|
| 冠心病(CAD) | 冠状动脉粥样硬化 | ~45% |
| 脑血管病 | 中风(缺血性/出血性) | ~35% |
| 高血压性心脏病 | 长期高血压致心脏损害 | ~5% |
| 心力衰竭 | 心脏泵血功能不全 | ~5% |
| 外周动脉疾病 | 四肢动脉狭窄 | ~3% |
| 心律失常 | 房颤等异常心律 | ~3% |
| 其他 | 瓣膜病/先心病等 | ~4% |

### 动脉粥样硬化进程

```
内皮损伤(高血压/吸烟/高血糖)
     ↓
LDL胆固醇沉积于内膜下
     ↓
氧化LDL → 巨噬细胞吞噬 → 泡沫细胞 → 脂纹
     ↓
平滑肌增生 → 纤维帽形成 → 斑块
     ↓
斑块破裂 → 血栓形成 → 急性事件(心梗/中风)
```

## 心血管疾病风险因素

### 不可改变因素

| 因素 | 影响 |
|------|------|
| 年龄 | 男>45岁、女>55岁后风险显著增加 |
| 性别 | 男性风险更高(绝经前女性有雌激素保护) |
| 家族史 | 一级亲属早发CVD(男<55岁/女<65岁) |
| 种族 | 黑人/南亚裔风险较高 |

### 可改变因素

| 因素 | 风险增加 | 干预效果 |
|------|---------|---------|
| 高血压 | 2-4倍 | 降压10mmHg→事件↓25% |
| 高LDL胆固醇 | 2-3倍 | LDL每↓1mmol/L→事件↓22% |
| 吸烟 | 2-4倍 | 戒烟1年→风险↓50% |
| 糖尿病 | 2-4倍 | 血糖控制→微血管并发症↓ |
| 肥胖(BMI>30) | 1.5-2倍 | 减重5-10%→显著改善 |
| 久坐不动 | 1.5-2倍 | 规律运动→风险↓30-50% |
| 不良饮食 | 1.5-2倍 | 地中海饮食→事件↓30% |
| 心理压力 | 1.5-2倍 | 压力管理→改善多项指标 |
| 过量饮酒 | 1.5-2倍 | 适度或不饮→降低风险 |

### 新兴风险因素

| 因素 | 机制 | 证据 |
|------|------|------|
| 慢性炎症(CRP) | 内皮损伤、斑块不稳定 | CANTOS试验证实 |
| 睡眠不足(<6h) | HPA轴激活、内皮功能障碍 | 大型队列研究 |
| 空气污染(PM2.5) | 氧化应激、全身炎症 | WHO认定 |
| 孤独/社会隔离 | 等效每天吸15支烟 | Meta分析支持 |
| 肠道菌群失调 | TMAO产生增加 | 新兴证据 |
| 抑郁症 | 自主神经失调、炎症 | 独立风险因素 |

## 心血管健康评估

### 常用评估工具

| 工具 | 评估内容 | 适用人群 |
|------|---------|---------|
| Framingham风险评分 | 10年CVD事件风险 | 40-79岁无CVD史 |
| ASCVD风险计算器 | 10年ASCVD风险 | 40-79岁(ACC/AHA) |
| SCORE2 | 10年致死CVD风险 | 欧洲人群 |
| 冠状动脉钙化评分 | 亚临床动脉硬化 | 中等风险人群 |
| 踝肱指数(ABI) | 外周动脉疾病 | 疑似PAD |

### 关键指标参考值

| 指标 | 理想值 | 边界值 | 高风险 |
|------|--------|--------|--------|
| 总胆固醇 | <5.2 mmol/L | 5.2-6.2 | >6.2 |
| LDL-C | <2.6 mmol/L | 2.6-3.4 | >3.4 |
| HDL-C | >1.0(男)/1.3(女) | — | <1.0 |
| 甘油三酯 | <1.7 mmol/L | 1.7-2.3 | >2.3 |
| 血压 | <120/80 mmHg | 120-139/80-89 | ≥140/90 |
| 空腹血糖 | <5.6 mmol/L | 5.6-6.9 | ≥7.0 |
| HbA1c | <5.7% | 5.7-6.4% | ≥6.5% |
| 静息心率 | 60-80 bpm | 80-100 | >100 |
| BMI | 18.5-24.9 | 25-29.9 | ≥30 |
| 腰围 | <90cm(男)/<80cm(女) | — | 超标 |

## 生活方式干预

### AHA Life's Essential 8

| 维度 | 最佳标准 | CVD风险降低 |
|------|---------|------------|
| 饮食 | DASH/地中海饮食 | 30% |
| 运动 | 150min中等/75min高强度/周 | 30-50% |
| 不吸烟 | 从不/已戒烟 | 50%(戒烟后) |
| 睡眠 | 7-9小时/晚 | 20-30% |
| BMI | 18.5-24.9 | 20-30% |
| 血压 | <120/80 | 25% |
| 血脂 | LDL<2.6 | 22%/每↓1mmol |
| 血糖 | 空腹<5.6 | 显著 |

### 运动处方

| 类型 | 推荐 | 心血管效益 |
|------|------|-----------|
| 有氧运动 | 150min中等或75min高强度/周 | 心输出量↑ 血压↓ HDL↑ |
| 力量训练 | 2-3次/周 | 胰岛素敏感↑ 血压↓ |
| 柔韧性 | 2-3次/周 | 血管顺应性↑ |
| 太极拳 | 3-5次/周 | 血压↓9mmHg HRV↑ |

### 心血管友好饮食

| 饮食模式 | 核心特征 | 证据 |
|---------|---------|------|
| 地中海饮食 | 橄榄油/鱼/蔬果/全谷 | PREDIMED: 事件↓30% |
| DASH饮食 | 低钠/高钾钙镁 | 血压↓11/6mmHg |
| 北欧饮食 | 全谷/浆果/鱼/菜籽油 | 炎症↓ 血脂改善 |
| 植物性饮食 | 全食物植物基 | CAD风险↓25% |

## 心血管保护策略

### 按年龄段建议

| 年龄 | 重点 | 筛查 |
|------|------|------|
| 20-39 | 建立健康习惯、戒烟 | 基线血脂/血压(每5年) |
| 40-59 | 风险评估、积极干预 | 年度血脂/血压/血糖 |
| 60-79 | 二级预防、药物管理 | 年度全面评估 |
| 80+ | 个体化管理、避免过度治疗 | 根据功能状态评估 |
