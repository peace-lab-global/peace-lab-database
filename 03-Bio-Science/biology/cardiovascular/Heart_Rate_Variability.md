## HRV基础

### 定义
心率变异性（Heart Rate Variability, HRV）指连续心跳间隔(R-R间期)的时间变化。高HRV通常反映自主神经系统灵活性好，是整体健康的标志。

### 自主神经系统与HRV

| 分支 | 功能 | 对心率影响 | 对HRV影响 |
|------|------|-----------|----------|
| 交感神经(SNS) | 战斗或逃跑 | 心率↑ | HRV↓ |
| 副交感神经(PNS) | 休息与恢复 | 心率↓ | HRV↑ |
| 平衡状态 | 灵活适应 | 适度变化 | HRV最优 |

### HRV的生理意义

```
高HRV = 自主神经灵活 = 适应能力强
  ↓
心血管保护 + 情绪调节良好 + 免疫功能好 + 认知功能佳

低HRV = 自主神经僵化 = 适应能力差
  ↓
CVD风险↑ + 压力耐受↓ + 炎症↑ + 抑郁焦虑风险↑
```

## HRV指标体系

### 时域指标

| 指标 | 全称 | 含义 | 正常参考 |
|------|------|------|---------|
| SDNN | R-R间期标准差 | 总变异性(交感+副交感) | >100ms |
| RMSSD | 连续R-R差值均方根 | 副交感(迷走)活性 | >20ms |
| pNN50 | 相邻R-R差>50ms的百分比 | 副交感活性 | >3% |
| 平均心率 | — | 总心率水平 | 60-80bpm |

### 频域指标

| 指标 | 频率范围 | 反映 | 解读 |
|------|---------|------|------|
| VLF | 0.003-0.04 Hz | 体温调节/RAAS | 低→预后差 |
| LF | 0.04-0.15 Hz | 交感+副交感(压力反射) | 混合指标 |
| HF | 0.15-0.4 Hz | 副交感(呼吸性窦性心律不齐) | 迷走活性 |
| LF/HF比值 | — | 交感/副交感平衡(争议) | <2为宜 |

### 非线性指标

| 指标 | 含义 |
|------|------|
| SD1 | 短期变异性(Poincare图) |
| SD2 | 长期变异性(Poincare图) |
| 样本熵(SampEn) | 复杂度/不可预测性 |
| DFA α1 | 短期分形特征 |

## HRV与健康

### HRV与疾病关系

| 疾病/状态 | HRV变化 | 临床意义 |
|-----------|---------|---------|
| 心梗后 | SDNN<70ms | 全因死亡率↑2-4倍 |
| 心力衰竭 | 显著降低 | 预后不良指标 |
| 糖尿病 | 降低(自主神经病变) | 早期筛查价值 |
| 抑郁症 | 降低(RMSSD/HF↓) | 反映情绪调节障碍 |
| PTSD | 降低+反应性异常 | 迷走刹车失灵 |
| 慢性压力 | 逐渐降低 | 自主神经耗竭 |
| 肥胖 | 降低 | 代谢风险标志 |

### HRV的年龄趋势

| 年龄段 | RMSSD参考 | 趋势 |
|--------|----------|------|
| 20-29 | 40-80 ms | 最高 |
| 30-39 | 30-60 ms | 开始下降 |
| 40-49 | 25-50 ms | 持续下降 |
| 50-59 | 20-40 ms | 明显下降 |
| 60+ | 15-35 ms | 进一步下降 |

### 多迷走理论（Porges）

Stephen Porges的多迷走理论将HRV与社会参与系统联系：

| 系统 | 迷走状态 | HRV | 行为 |
|------|---------|-----|------|
| 社会参与 | 腹侧迷走(有髓鞘) | 高 | 平静、社交、安全 |
| 战斗/逃跑 | 交感激活 | 低 | 焦虑、防御 |
| 冻结/关闭 | 背侧迷走(无髓鞘) | 极低 | 解离、麻木 |

## HRV训练与提升

### 循证提升方法

| 方法 | 效果量 | 机制 | 建议 |
|------|--------|------|------|
| 呼吸训练(6次/分) | ★★★★★ | 呼吸窦性心律不齐最大化 | 每日10-20min |
| 有氧运动 | ★★★★★ | 迷走张力↑ | 150min/周 |
| HRV生物反馈 | ★★★★☆ | 共振频率训练 | 10-20次训练 |
| 正念冥想 | ★★★★☆ | 副交感激活 | 每日15-30min |
| 瑜伽 | ★★★★☆ | 呼吸+运动+冥想 | 3-5次/周 |
| 太极拳 | ★★★★☆ | 慢呼吸+运动 | 3-5次/周 |
| 冷暴露 | ★★★☆☆ | 迷走神经刺激 | 渐进式冷水 |
| 良好睡眠 | ★★★★☆ | 夜间副交感恢复 | 7-9h/晚 |
| Omega-3脂肪酸 | ★★★☆☆ | 抗炎/心脏电稳定 | 2g EPA+DHA/天 |

### 共振呼吸训练(6次/分)

| 步骤 | 内容 |
|------|------|
| 1 | 找到舒适坐姿 |
| 2 | 吸气5秒 |
| 3 | 呼气5秒 |
| 4 | 持续10-20分钟 |
| 5 | 每日1-2次 |
| 原理 | 6次/分呼吸频率(0.1Hz)与LF频段共振，最大化HRV振幅 |

### HRV生物反馈协议

| 阶段 | 内容 | 次数 |
|------|------|------|
| 评估 | 确定个人共振频率(4.5-7次/分) | 1次 |
| 训练 | 以共振频率呼吸+实时反馈 | 8-10次 |
| 迁移 | 日常应用、无反馈练习 | 持续 |
| 维持 | 每日家庭练习 | 终身 |

## HRV监测工具

### 设备对比

| 设备 | 精度 | 易用性 | 价格 | 适用 |
|------|------|--------|------|------|
| 心电图(ECG) | ★★★★★ | ★★☆☆☆ | 高 | 临床金标准 |
| 胸带心率计(Polar) | ★★★★☆ | ★★★☆☆ | 中 | 运动/研究 |
| 指夹式(Elite HRV) | ★★★☆☆ | ★★★★☆ | 低 | 日常监测 |
| 智能手表(Apple Watch) | ★★★☆☆ | ★★★★★ | 中 | 日常趋势 |
| Oura Ring | ★★★☆☆ | ★★★★★ | 中 | 夜间/恢复 |

### 日常HRV监测建议

| 建议 | 说明 |
|------|------|
| 固定时间 | 每日晨起、同一姿势测量 |
| 关注趋势 | 7天移动平均比单日更有意义 |
| 个体基线 | 与自己比较，而非绝对值 |
| 综合解读 | 结合主观感受、睡眠、运动负荷 |
| 避免过度 | 不要因一天低HRV而焦虑 |
