## 甲沟炎的病因与治疗

### 定义与分类

甲沟炎（Paronychia）是指甲周围软组织的炎症或感染，是最常见的指甲相关感染性疾病。根据病程分为急性甲沟炎（Acute Paronychia）和慢性甲沟炎（Chronic Paronychia），两者的病因、治疗策略和预后有显著差异。

### 急性甲沟炎

**特征**：起病急（数小时至数天），通常为细菌感染

| 项目 | 内容 |
|------|------|
| 主要病原菌 | 金黄色葡萄球菌（Staphylococcus aureus）最常见 |
| 其他病原菌 | 链球菌、铜绿假单胞菌、混合感染 |
| 诱发因素 | 倒刺撕扯、过度修剪指甲、刺伤、咬指甲 |
| 好发人群 | 经常接触水者、糖尿病患者、免疫力低下者 |

**临床表现分级**：

| 分期 | 症状 | 治疗 |
|------|------|------|
| 早期（I期） | 甲襞红肿、触痛 | 温水浸泡+外用抗生素 |
| 中期（II期） | 脓肿形成、搏动性疼痛 | 切开引流+口服抗生素 |
| 晚期（III期） | 脓肿扩展至甲床、指甲松动 | 部分拔甲+抗生素治疗 |

**急性甲沟炎的家庭处理**：

1. **温水浸泡**：每天3-4次，每次15-20分钟，可加入少量盐
2. **外用消毒**：碘伏或氯己定溶液涂抹
3. **保持干燥**：浸泡间隔期保持手部干燥
4. **避免挤压**：不要试图挤压脓肿
5. **抬高患指**：减轻肿胀和疼痛

### 慢性甲沟炎

**特征**：病程超过6周，反复发作，多为混合感染或真菌感染

| 项目 | 内容 |
|------|------|
| 主要病原体 | 白色念珠菌（Candida albicans）最常见 |
| 其他病原体 | 混合菌群：细菌+真菌 |
| 核心病理 | 甲襞角质层（Cuticle）受损→保护屏障丧失→反复感染 |
| 好发人群 | 经常接触水者（厨师、清洁工、医护人员）、糖尿病患者 |

**慢性甲沟炎的综合管理**：

| 措施 | 具体内容 | 原理 |
|------|---------|------|
| 保持干燥 | 戴防水手套做家务 | 消除真菌滋生的湿润环境 |
| 保护角质层 | 不推不剪角质层 | 恢复天然保护屏障 |
| 外用抗真菌 | 克霉唑/酮康唑乳膏 | 针对真菌感染 |
| 外用抗炎 | 弱效皮质类固醇 | 减轻慢性炎症 |
| 避免刺激 | 减少接触洗涤剂和化学物质 | 防止进一步损伤屏障 |

### 甲沟炎的预防策略

| 策略 | 操作 |
|------|------|
| 正确修剪 | 直线修剪，略留长度，不深入甲沟 |
| 保护角质层 | 不推不剪，涂抹护手霜保湿 |
| 手部保湿 | 洗手后及时涂抹护手霜 |
| 防水保护 | 接触水/化学物质时戴手套 |
| 戒除坏习惯 | 不咬指甲、不撕倒刺 |
| 糖尿病管理 | 控制血糖，定期足部和手部检查 |

## 指甲作为健康窗口

### 指甲改变与系统性疾病

指甲（Nail）的生长速度约每天0.1毫米，完全替换一个指甲约需3-6个月。指甲的变化可以反映数周至数月前的健康状况，因此被临床医生称为"健康的窗口"。

### 常见指甲异常及其临床意义

**颜色变化**：

| 指甲颜色 | 可能原因 | 临床意义 |
|---------|---------|---------|
| 白色（白甲） | 创伤、低蛋白血症、肾病、肝病 | 部分白甲常见于创伤；全白甲需排查系统疾病 |
| 黄色（黄甲） | 真菌感染、黄甲综合征、吸烟 | 黄甲综合征（Yellow Nail Syndrome）与淋巴系统异常相关 |
| 蓝色 | 银中毒、Wilson病、缺氧 | 银质沉着症（Argyria）罕见但明确 |
| 绿色 | 铜绿假单胞菌感染 | 常见于指甲外伤后水分滞留 |
| 棕/黑色 | 甲下血肿、黑色素瘤、药物 | 需排除甲黑色素瘤（Subungual Melanoma） |
| 红色 | 甲下出血、心内膜炎 | 裂片状出血（Splinter Hemorrhages） |

**形态变化**：

| 异常形态 | 可能原因 | 临床关联 |
|---------|---------|---------|
| 杵状指（Clubbing） | 慢性缺氧、肺癌、心脏病、炎症性肠病 | 常与严重系统性疾病相关 |
| 匙状甲（Koilonychia） | 缺铁性贫血、血色病、甲状腺功能减退 | 反甲，最常见于缺铁 |
| 甲横沟（Beau's Lines） | 严重疾病/应激后生长暂停 | 反映过去3-6个月内的重大健康事件 |
| 甲纵裂 | 老化、甲状腺问题、银屑病 | 常见纵向裂纹 |
| 甲剥离（Onycholysis） | 甲状腺疾病、银屑病、创伤 | 指甲与甲床分离 |
| 泰里甲（Terry's Nails） | 肝硬化、心衰、糖尿病、衰老 | 近端80%变白，远端正常 |
| 米氏线（Mees' Lines） | 砷/铊中毒、严重系统疾病 | 横向白色条纹 |

**特别警示信号**：

以下指甲变化需要立即就医评估：

1. **新出现的黑色纵纹**或已有纵纹发生变化——需排除甲黑色素瘤
2. **指甲周围持续红肿**——可能为甲沟炎或更严重的感染
3. **指甲突然全部变暗**——需排除系统性原因
4. **甲下不明原因出血**——需排除心内膜炎等严重疾病
5. **指甲进行性变厚变黄**伴异味——可能为甲真菌病（Onychomycosis）

## 指甲结构与生理

### 指甲解剖结构

| 结构 | 英文 | 功能 |
|------|------|------|
| 甲板 | Nail Plate | 可见的坚硬角质化结构 |
| 甲床 | Nail Bed | 甲板下方的皮肤，提供营养 |
| 甲母质 | Nail Matrix | 指甲生长的"工厂"，位于甲根下方 |
| 甲上皮 | Eponychium | 甲根处覆盖的薄膜（角质层） |
| 甲半月 | Lunula | 甲母质远端可见的白色半月形 |
| 甲沟 | Nail Fold | 包围甲板的皮肤皱褶 |
| 甲下皮 | Hyponychium | 甲板远端下方的皮肤 |

### 指甲生长特征

| 特征 | 手指甲 | 脚趾甲 |
|------|--------|--------|
| 生长速度 | 约3mm/月 | 约1mm/月 |
| 完全更新时间 | 3-6个月 | 12-18个月 |
| 影响因素 | 年龄、营养、季节、健康状态 | 同左，但更新更慢 |

**影响指甲生长的因素**：

| 因素 | 效应 |
|------|------|
| 年龄 | 随年龄增长减慢 |
| 营养 | 蛋白质、铁、锌、生物素充足时正常 |
| 季节 | 夏季生长较快 |
| 妊娠 | 孕期生长加速 |
| 疾病 | 严重疾病/发热减缓生长（→Beau's Lines） |
| 手利 | 惯用手指甲生长较快 |
| 血液循环 | 外周循环良好时生长正常 |

## 指甲日常护理指南

### 基本护理原则

**清洁**：
- 温和洗手，避免过度使用强力清洁剂
- 使用软毛指甲刷清洁甲下区域
- 洗手后彻底擦干，特别是甲沟区域
- 避免用尖锐物品清理甲下——可能导致甲床分离

**修剪**：
- 手指甲：沿自然弧线直线修剪，略微圆角
- 脚趾甲：直线修剪，不深入角落（防止嵌甲）
- 使用锋利的指甲剪/钳，避免拉扯
- 修剪后用指甲锉轻轻修整边缘

**保湿**：
- 每日涂抹护手霜，特别注意甲沟和角质层
- 角质层油（Cuticle Oil）可保持角质层柔软
- 含尿素（Urea）的产品对干燥脆弱的指甲特别有效

### 营养与指甲健康

| 营养素 | 推荐食物 | 对指甲的作用 |
|--------|---------|------------|
| 蛋白质 | 鸡蛋、鱼、豆类 | 指甲的主要成分（角蛋白 Keratin） |
| 生物素（B7） | 蛋黄、坚果、燕麦 | 改善指甲厚度和硬度 |
| 铁 | 红肉、菠菜、豆类 | 缺铁导致匙状甲 |
| 锌 | 牡蛎、南瓜子、牛肉 | 缺锌导致白点和生长缓慢 |
| 钙 | 乳制品、深绿叶菜 | 维持指甲硬度和结构 |
| 维生素C | 柑橘、猕猴桃、甜椒 | 胶原蛋白合成必需 |
| Omega-3 | 鱼油、亚麻籽 | 保持指甲水润，防止干燥脆弱 |

### 美甲与健康

**美甲风险**：

| 类型 | 主要风险 | 建议 |
|------|---------|------|
| 普通指甲油 | 溶剂干燥效应，色素沉着 | 使用底油保护，定期休息 |
| 凝胶甲（Gel） | UV照射、过敏、甲板变薄 | 每次间隔2-4周，使用防晒手套 |
| 水晶甲（Acrylic） | 甲板损伤、真菌感染风险 | 选择有资质的美甲师，保持清洁 |
| 甲油胶（Shellac） | 去除过程损伤 | 专业去除，不自行撕剥 |
| 延长甲 | 胶水过敏、机械损伤 | 避免过度延长 |

## 何时需要就医

### 必须就医的情况

| 紧急程度 | 情况 | 可能诊断 |
|---------|------|---------|
| 立即就医 | 甲沟脓肿伴发热/红肿扩散 | 严重感染/蜂窝织炎 |
| 立即就医 | 指甲外伤后持续出血/剧烈疼痛 | 甲床裂伤/骨折 |
| 一周内就医 | 新出现黑色纵纹或原有纵纹变化 | 甲黑色素瘤需排除 |
| 一周内就医 | 指甲持续变形/增厚/变色 | 甲真菌病/银屑病 |
| 两周内就医 | 慢性甲沟炎反复发作 | 需要系统治疗评估 |
| 常规就诊 | 不明原因指甲改变持续>2个月 | 系统性疾病筛查 |

### 指甲相关专科

| 科室 | 适用情况 |
|------|---------|
| 皮肤科 | 甲真菌病、银屑病甲改变、甲黑色素瘤 |
| 普通外科 | 急性甲沟炎切开引流、拔甲 |
| 手外科 | 复杂甲床损伤、慢性甲沟炎手术 |
| 内科 | 怀疑系统性疾病导致的指甲改变 |
| 营养科 | 营养缺乏导致的指甲异常 |

### 就诊前的准备

1. 记录症状开始时间和发展过程
2. 拍摄照片记录变化（尤其颜色变化）
3. 列出正在使用的药物和补充剂
4. 回忆可能的诱发因素（创伤、新美甲产品等）
5. 准备好个人和家族病史信息
