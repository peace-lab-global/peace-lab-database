# Death Education Workshop Design (死亡教育工作坊设计指南)

> *"死亡教育的最高境界不是消除恐惧，而是让人在有限性的觉知中活出更真实、更完整的人生。"*
> — 综合自 Herman Feifel, Robert Kastenbaum

## 一、工作坊设计的核心原则

### 1.1 安全容器原则

| 原则 | 操作 | 为什么重要 |
|---|---|---|
| **知情同意** | 参与者事先知道将讨论死亡 | 减少意外触发 |
| **自愿参与** | 不强迫任何人分享 | 尊重个人边界 |
| **保密协议** | "这里说的，留在这里" | 建立信任 |
| **退出权** | 随时可以离开或暂停 | 自主权 |
| **触发预案** | 准备处理强烈情绪的人员和流程 | 安全 |
| ** grounding 技术** | 提供简单的 grounding 方法 | 帮助参与者回到当下 |

### 1.2 渐进暴露原则

```
第1层：抽象/远距离 ──→ 他人之死、文化仪式、哲学讨论
      ↓
第2层：中距离    ──→ 电影/文学中的死亡、案例讨论
      ↓
第3层：近距但间接 ──→ 家族死亡史、记忆分享
      ↓
第4层：直接面对  ──→ 写自己的悼词、设计葬礼、伦理遗嘱
      ↓
第5层：深度整合  ──→ 冥想体验、临终模拟
```

## 二、按受众设计的工作坊

### 2.1 大学生工作坊（2-3小时）

| 目标 | 帮助年轻人在生命早期建立健康的死亡意识，澄清价值观 |
|---|---|
| **时间** | 2.5小时 |
| **人数** | 15-25人 |

| 时间 | 环节 | 内容 | 方法 |
|---|---|---|---|
| 0:00-0:10 | 开场与契约 | 介绍工作坊；保密协议；退出权 | 讲解 |
| 0:10-0:20 | 破冰 | "你对死亡的第一印象是什么？" | 小组分享 |
| 0:20-0:40 | 死亡态度自测 | DAP-R简化版；了解自己的死亡态度 | 自评+分享 |
| 0:40-1:00 | 文化中的死亡 | 不同文化如何看待死亡 | 小组讨论 |
| 1:00-1:10 | 休息 | | |
| 1:10-1:30 | 电影片段 | 《遗愿清单》或《寻梦环游记》片段 | 观看+讨论 |
| 1:30-1:50 | 价值澄清 | "如果你的生命只剩一年，你会做什么不同？" | 个人写作+自愿分享 |
| 1:50-2:10 | 伦理遗嘱入门 | 写一段给家人的话 | 个人写作 |
| 2:10-2:30 | 总结与 grounding | 5分钟正念呼吸；资源介绍 | 引导冥想 |

### 2.2 医护人员工作坊（4小时）

| 目标 | 提升医护人员的死亡沟通能力；处理职业性死亡暴露 |
|---|---|
| **时间** | 4小时 |
| **人数** | 10-20人 |

| 时间 | 环节 | 内容 |
|---|---|---|
| 0:00-0:20 | 开场 | 医护工作中的死亡经历分享 |
| 0:20-0:50 | 死亡沟通模型 | SPIKES / NURSE / Ask-Tell-Ask / REMAP |
| 0:50-1:20 | 案例讨论 | 3个临床案例（告知坏消息、处理家属情绪、预立指示讨论） |
| 1:20-1:40 | 休息 | |
| 1:40-2:10 | 医护人员的自我照顾 | 替代性创伤； burnout；意义维持 |
| 2:10-2:50 | 角色扮演 | 模拟坏消息告知；同伴反馈 |
| 2:50-3:10 | 休息 | |
| 3:10-3:40 | 灵性评估 | FICA工具实操 |
| 3:40-4:00 | 总结 | 资源；同行支持网络 |

### 2.3 企业员工工作坊（2小时）

| 目标 | 提升员工的生命意义感；为可能的职场丧亲做准备 |
|---|---|
| **时间** | 2小时 |
| **人数** | 20-40人 |

| 时间 | 环节 | 内容 |
|---|---|---|
| 0:00-0:15 | 开场 | "我们为什么在这里谈论死亡？" |
| 0:15-0:35 | 生命的有限性 | 平均寿命计算；"你还有多少个夏天？" |
| 0:35-0:55 | 价值澄清 | "工作在你生命中的位置"；"什么是你真正在乎的？" |
| 0:55-1:05 | 休息 | |
| 1:05-1:25 | 预立指示入门 | 为什么需要；如何开始 |
| 1:25-1:45 | 职场丧亲支持 | 如果同事去世了，我们如何支持彼此？ |
| 1:45-2:00 | 总结 | 行动步骤；公司EAP资源 |

### 2.4 普通公众死亡咖啡馆（1.5小时）

| 目标 | 为公众提供开放、无结构的死亡讨论空间 |
|---|---|
| **时间** | 1.5小时 |
| **人数** | 8-15人 |

| 原则 | 说明 |
|---|---|
| **无议程** | 没有固定主题；参与者带话题 |
| **无专家** | 没有"正确答案"；每个人都是专家 |
| **茶与蛋糕** | 轻松的氛围；非正式 |
| **讨论规则** | 保密；不强迫分享；尊重不同观点 |
| **引导者** | 确保每个人都有机会发言；管理时间 |

## 三、体验式活动设计

### 3.1 经典活动库

| 活动 | 时间 | 目标 | 操作 | 安全注意 |
|---|---|---|---|---|
| **写自己的悼词** | 20-30分钟 | 价值澄清；生命反思 | "想象你的葬礼，你希望谁来？他们会说你什么？" | 可能触发强烈情绪；提供 grounding |
| **设计自己的葬礼** | 20-30分钟 | 了解个人价值观；文化表达 | 音乐、地点、参与者、仪式 | 允许创意和幽默 |
| **伦理遗嘱写作** | 30-40分钟 | 精神遗产传承 | 使用模板；写给特定的人 | 可能触发未竟事务 |
| **生命时间线** | 15-20分钟 | 生命回顾；意义识别 | 画出人生高低点 | 创伤史需温和处理 |
| **空椅技术（改良版）** | 20-30分钟 | 未竟事务处理 | 与逝去的亲人"对话" | **仅在有专业人员时进行** |
| **电影讨论** | 30-45分钟 | 替代性体验；情感共鸣 | 观看+讨论 | 选择适合受众的影片 |
| **死亡冥想（L1）** | 15-20分钟 | 正念+死亡觉知 | 引导式死亡冥想（温和版） | **禁忌症筛查**；不强迫闭眼 |
| **四道人生练习** | 20-30分钟 | 关系修复； closure | 向在场的重要他人表达四道 | 征得同意 |
| **预立指示填写** | 30-45分钟 | 实际准备 | 提供表格；协助填写 | 提供法律咨询资源 |

### 3.2 电影/文学选用指南

| 受众 | 推荐 | 主题 | 注意 |
|---|---|---|---|
| 大学生 | 《遗愿清单》 | 生命意义；遗愿 | 轻松入门 |
| 大学生 | 《寻梦环游记》 | 记忆与死亡；家庭 | 文化敏感（墨西哥） |
| 医护人员 | 《爱》(Amour) | 老年照护；死亡 | 沉重；需处理时间 |
| 医护人员 | 《Being Mortal》纪录片 | 医学与善终 | 基于Gawande的书 |
| 青少年 | 《死亡诗社》 | 青春；反叛；自杀 | 需讨论自杀预防 |
| 普通公众 | 《告别》(The Farewell) | 文化差异；死亡告知 | 跨文化讨论 |
| 冥想者 | 《西藏生死书》纪录片 | 中阴；转世 | 宗教框架 |

## 四、课堂安全协议

### 4.1 触发反应的预防

| 预防策略 | 操作 |
|---|---|
| **课前筛查** | "是否有任何与死亡相关的近期经历可能影响你的参与？"（可选匿名问卷） |
| **渐进暴露** | 从抽象到具体；不突然跳入深度体验 |
| **选择暴露** | 总是给参与者选择："你可以选择不写/不分享" |
| **信号系统** | 提供一个"暂停"信号（如举手），任何人可以随时使用 |

### 4.2 触发反应的处理

| 反应程度 | 表现 | 处理 |
|---|---|---|
| **轻度** | 眼眶湿润、沉默、轻微不安 | 正常化；"这些感受是完全正常的" |
| **中度** | 哭泣、呼吸急促、身体颤抖 | 邀请其离开房间（如有同伴陪同）；提供水；grounding |
| **重度** | 惊恐发作、解离、无法停止哭泣 | 立即停止活动；安排专人陪同；评估是否需要进一步专业帮助 |

### 4.3 Grounding 技术（现场可用）

| 技术 | 操作 | 时间 |
|---|---|---|
| **5-4-3-2-1感官 grounding** | "说出5个你看到的东西、4个触摸到的、3个听到的…" | 2分钟 |
| **呼吸 grounding** | "深吸4秒，屏息4秒，呼气6秒" | 2分钟 |
| **身体扫描** | "感觉你的脚接触地面…感觉你的背靠在椅子上…" | 3分钟 |
| **现实检验** | "你现在在哪里？今天是几号？你安全吗？" | 1分钟 |

## 五、讨论引导技术

### 5.1 有效提问

| 类型 | 示例 | 效果 |
|---|---|---|
| **开放性问题** | "你对死亡有什么想法？" | 鼓励自由表达 |
| **假设性问题** | "如果你的生命只剩一年，你会改变什么？" | 降低防御；促进反思 |
| **对比性问题** | "你的文化如何看待死亡？与你个人看法一致吗？" | 促进批判性思考 |
| **连接性问题** | "这与你之前分享的经历有什么联系？" | 促进深度整合 |

### 5.2 困难时刻的引导

| 情境 | 引导策略 |
|---|---|
| **参与者说"我不怕死"** | "那很好。我好奇：是什么让你有这种平静？" |
| **参与者极度恐惧** | "谢谢你分享这种恐惧。它是什么感觉？" |
| **参与者分享创伤** | "谢谢你信任这个空间。你现在的感受如何？"（不深入探索；转介） |
| **争论发生** | "我们有不同的观点是自然的。让我们听听每个人的看法。" |
| **有人沉默** | "沉默也是参与的一种方式。你没有义务分享。" |
| **有人主导讨论** | "谢谢你分享了很多。我想听听还没发言的人的看法。" |

## 六、效果评估

### 6.1 过程评估

| 维度 | 问题 | 工具 |
|---|---|---|
| **安全感** | "你在这个空间中感到安全吗？" | 5点量表 |
| **参与度** | "你愿意参与讨论吗？" | 5点量表 |
| **触发程度** | "讨论是否触发了你不舒服的情绪？" | 5点量表 |
| **有用性** | "你认为这个工作坊有价值吗？" | 5点量表 |

### 6.2 结果评估

| 时间 | 评估内容 | 工具 |
|---|---|---|
| **工作坊前** | 死亡态度（DAP-R简化版）；生命意义（MLQ） | 自评问卷 |
| **工作坊后** | 同上 + 满意度 | 自评问卷 |
| **1个月后** | 行为改变（是否采取了任何行动？） | 随访问卷 |
