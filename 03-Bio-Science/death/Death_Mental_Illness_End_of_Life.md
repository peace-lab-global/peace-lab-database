# Mental Illness & End-of-Life Care (精神疾病患者的临终关怀)

> *"精神疾病不是临终关怀的排除标准，而是需要更细致、更人性化的照护信号。"*
> — 综合自 Hospice Foundation of America, APA Guidelines

---

## 核心术语与词源表 (Core Terminology & Etymology)

| 术语 (中文) | 英文名 | 词源/来源 | 精确定义 |
|---|---|---|---|
| **决策能力** | Decisional Capacity | 医学伦理 | 做出特定医疗决定所需的理解、欣赏、推理、表达能力 |
| **代理决策** | Substituted Judgment | 医学伦理 | 代理人在患者无能力时，基于患者已知价值观做出决定 |
| **最佳利益** | Best Interest Standard | 医学伦理 | 当患者意愿未知时，基于客观福祉做出决定 |
| **能力波动** | Fluctuating Capacity | 临床术语 | 精神疾病患者的决策能力可能随症状波动而变化 |
| **提前指示** | Advance Statement / Ulysses Contract | 精神卫生法 | 患者在有能力时对未来治疗（包括拒绝治疗）的预先声明 |
| **严重精神疾病** | Severe Mental Illness (SMI) | 精神病学 | 精神分裂症、双相障碍、重度抑郁等导致严重功能损害的疾病 |
| **共病** | Comorbidity | 医学 | 精神疾病与躯体疾病同时存在 |
| **治疗联盟** | Therapeutic Alliance | 心理治疗 | 患者与治疗者之间的合作关系 |
| **强制治疗** | Compulsory Treatment | 精神卫生法 | 在非自愿情况下对患者进行治疗 |
| **恢复导向** | Recovery-Oriented | 精神卫生 | 关注患者的潜能、希望和有意义的生活，而非仅聚焦症状 |

## 一、精神疾病患者临终的双重困境

### 1.1 双重污名

```
躯体疾病患者临终
    ├── 社会同情
    ├── 充分的医疗资源
    └── 家庭支持通常充分

精神疾病患者
    ├── 社会污名（"疯子""危险的"）
    ├── 医疗资源不足（精神科与躯体科割裂）
    └── 家庭支持可能断裂（因疾病导致的家庭冲突）

精神疾病 + 临终
    ├── 双重污名：既是"精神病人"又是"将死之人"
    ├── 沟通困难：症状可能影响表达
    ├── 决策复杂：能力评估困难
    └── 哀伤特殊：可能被忽视
```

### 1.2 数据

| 数据 | 说明 |
|---|---|
| **预期寿命差距** | 严重精神疾病患者的预期寿命比普通人群短10-20年 |
| **死因** | 心血管疾病、糖尿病、癌症、呼吸系统疾病（与普通人相同，但诊断更晚、治疗更差） |
| **自杀** | 约5-10%的精神分裂症患者死于自杀；抑郁症患者的自杀风险是普通人群的20倍 |
| **安宁疗护使用率** | 精神疾病患者获得安宁疗护的比例显著低于普通人群 |

## 二、决策能力评估

### 2.1 能力评估的四要素

| 要素 | 评估问题 | 精神疾病的特殊挑战 |
|---|---|---|
| **理解** | 患者是否理解病情和治疗选项？ | 妄想可能影响对病情的理解 |
| **欣赏** | 患者是否理解这些信息适用于自己？ | 疾病失认（Anosognosia）常见 |
| **推理** | 患者是否能权衡各选项的利弊？ | 认知症状可能影响推理 |
| **表达选择** | 患者是否能清晰表达决定？ | 思维障碍可能影响表达 |

### 2.2 能力波动的评估策略

| 策略 | 操作 |
|---|---|
| **多次评估** | 在不同时间、不同症状状态下评估 |
| **简化信息** | 使用简单、具体的语言 |
| **视觉辅助** | 图表、图片帮助理解 |
| **重复确认** | 确保患者的决定是稳定的 |
| **记录最佳状态** | 在患者状态最佳时进行深度讨论 |

### 2.3 不同疾病的能力特点

| 疾病 | 能力特点 | 评估注意 |
|---|---|---|
| **精神分裂症** | 妄想/幻觉可能影响；认知症状可能损害推理 | 区分妄想内容与合理的恐惧/偏好 |
| **双相障碍** | 躁期可能过度乐观；郁期可能绝望 | 在稳定期评估；不基于单一情绪状态做判断 |
| **重度抑郁** | 绝望感可能扭曲判断；但不是自动无能力 | 评估是否为"理性的"拒绝 vs. 抑郁驱动的绝望 |
| **人格障碍** | 情绪不稳定可能影响；但通常保留能力 | 区分情绪驱动的决定 vs. 持久的价值观 |
| **焦虑障碍** | 恐惧可能放大；但通常保留能力 | 正常化恐惧；区分焦虑 vs. 合理的担忧 |
| **痴呆** | 随病程逐渐丧失 | 早期尽可能完成预立指示 |

## 三、不同精神疾病的临终关怀

### 3.1 精神分裂症患者的临终

| 挑战 | 干预 |
|---|---|
| **妄想** | 不直接挑战妄想；在安全范围内工作；如患者认为"医生要毒害我"，可换其他方式给药 |
| **幻觉** | 评估幻觉是否引起痛苦；如痛苦，使用低剂量抗精神病药 |
| **阴性症状** | 患者可能缺乏动力表达需求；需要主动评估症状 |
| **社会孤立** | 长期的社会孤立；需要主动建立支持网络 |
| **药物相互作用** | 抗精神病药与镇痛药、镇静药的相互作用 |
| **安宁疗护的接受** | 患者可能对陌生人（安宁团队）有偏执；需要缓慢建立信任 |

### 3.2 双相障碍患者的临终

| 挑战 | 干预 |
|---|---|
| **情绪波动的管理** | 继续情绪稳定剂；避免触发因素 |
| **躁期的风险** | 冲动决定（如拒绝治疗、冲动消费）；需要代理决策 |
| **郁期的风险** | 自杀风险升高；被动死亡愿望 |
| **药物依从性** | 确保情绪稳定剂的持续使用 |
| **家属教育** | 帮助家属理解情绪波动是疾病的一部分，不是"态度问题" |

### 3.3 重度抑郁患者的临终

| 挑战 | 干预 |
|---|---|
| **抑郁性绝望 vs. 理性拒绝** | 这是**核心鉴别**：抑郁导致的"想死" vs. 因痛苦而理性的"不想继续治疗" |
| **抗抑郁药的继续** | 通常继续抗抑郁药；但注意与临终药物的相互作用 |
| **心理治疗** | 意义疗法、CBT可能仍有帮助 |
| **ECT** | 在极少数情况下，严重抑郁可考虑ECT；但在临终阶段通常不适用 |
| **家属的哀伤** | 家属可能在患者生前即开始" anticipatory mourning " |

### 3.4 焦虑障碍患者的临终

| 挑战 | 干预 |
|---|---|
| **死亡恐惧的放大** | 焦虑障碍可能放大死亡恐惧；需要区分"正常的"临终焦虑 vs. 病理性焦虑 |
| **苯二氮䓬类的使用** | 在临终阶段，苯二氮䓬类可用于焦虑控制；但注意呼吸抑制 |
| **正念** | 改良版正念可能帮助；但注意创伤史 |
| **暴露疗法** | 不适用；临终不是暴露的好时机 |

## 四、MAiD与精神疾病：伦理雷区

### 4.1 核心争议

| 立场 | 论证 |
|---|---|
| **支持** | 不可缓解的精神痛苦与躯体痛苦在伦理上等价；自主权应包括精神病患者 |
| **反对** | 精神疾病的"不可治愈"极难判定；新疗法持续出现；可能沦为弱势群体被迫的"选择" |

### 4.2 临床决策框架

| 步骤 | 内容 |
|---|---|
| **1. 排除可治疗的抑郁** | 精神病患者表达"想死"时，首先评估是否为抑郁性自杀意念 |
| **2. 评估决策能力** | 多次评估；多位独立专家 |
| **3. 尝试所有治疗** | 确认已尝试所有循证治疗（药物、心理治疗、ECT等） |
| **4. 时间稳定性** | 意愿是否长期稳定（通常要求数年） |
| **5. 独立评估** | 通常需要多位独立精神科医师评估 |
| **6. 等待期** | 法律要求的等待期 |
| **7. 持续评估** | 在等待期内持续评估意愿和能力 |

### 4.3 加拿大案例的启示

| 时间线 | 事件 |
|---|---|
| 2016 | MAiD合法化，限于躯体疾病 |
| 2021 | 扩大到非终末期疾病 |
| 原计划2023 | 扩大到精神疾病 |
| **推迟至2027** | 因评估能力不足、伦理争议而推迟 |
| **启示** | 需要充分的评估体系和专业培训 |

## 五、安宁疗护中的整合照护

### 5.1 精神科与安宁疗护的整合模式

| 模式 | 说明 | 优势 |
|---|---|---|
| **咨询模式** | 安宁团队需要时咨询精神科 | 灵活；资源要求低 |
| **共同照护模式** | 精神科医师加入安宁团队 | 连续性；专业知识 |
| **专科安宁模式** | 专门的"精神安宁疗护"团队 | 最专业；但资源要求高 |

### 5.2 症状管理的特殊考量

| 症状 | 非药物干预 | 药物干预 | 注意 |
|---|---|---|---|
| **疼痛** | 正念、音乐、按摩 | 阿片类 | 精神病患者可能对药物更敏感 |
| **焦虑** | grounding、呼吸、音乐 | 苯二氮䓬类（短期） | 依赖风险；呼吸抑制 |
| **躁动** | 环境调整、陪伴 | 低剂量抗精神病药 | 避免高剂量；EPS风险 |
| **抑郁** | 意义对话、生命回顾 | 抗抑郁药 | 起效慢；临终阶段可能有限 |
| **幻觉/妄想** | 不直接挑战；转移注意 | 低剂量抗精神病药 | 避免过度镇静 |
| **睡眠障碍** | 睡眠卫生、放松 | 褪黑素、小剂量镇静药 | 避免苯二氮䓬类长期用 |

### 5.3 家庭支持

| 挑战 | 干预 |
|---|---|
| **长期照护的疲惫** | Respite care；照护者支持小组 |
| **病耻感** | 心理教育；正常化 |
| **关系断裂** | 家庭治疗；四道人生 |
| **经济压力** | 社会资源链接 |
| **哀伤复杂性** | 预感性哀伤支持；复杂性哀伤预防 |

## 六、法律与制度考量

### 6.1 预立指示在精神疾病中的应用

| 挑战 | 建议 |
|---|---|
| **能力波动** | 在稳定期完成；详细、具体；定期复核 |
| **提前指示的拒绝治疗** | 部分国家允许患者在有能力时拒绝未来的强制治疗 |
| **代理人的选择** | 选择理解精神疾病、尊重患者价值观的人 |

### 6.2 强制治疗与临终关怀的张力

| 张力 | 平衡 |
|---|---|
| **患者拒绝治疗（因妄想）** | 如为延长生命的治疗（如抗生素），尊重拒绝 vs. 最佳利益？ |
| **患者要求停止治疗** | 评估能力；如能力保留 → 尊重；如无能力 → 代理决策 |
| **家属要求继续治疗 vs. 患者拒绝** | 以患者意愿为优先（如能力保留） |
