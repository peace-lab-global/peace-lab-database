# Assessment & Measures in Death Studies (死亡专题评估与测量整合手册)

> *"如果你不能测量它，你就不能管理它。"*
> — Peter Drucker (管理学格言，同样适用于临床与研究)

---

## 核心术语与词源表 (Core Terminology & Etymology)

| 术语 (中文) | 英文名 | 词源/来源 | 精确定义 |
|---|---|---|---|
| **死亡焦虑** | Death Anxiety | 心理学 | 对死亡及其相关议题（过程、死后状态、不存在）的恐惧与担忧 |
| **死亡恐惧** | Fear of Death | 心理学 | 对死亡本身的情绪反应（死亡焦虑的子维度） |
| **死亡回避** | Death Avoidance | 心理学 | 回避死亡话题与相关情境的倾向 |
| **死亡接纳** | Death Acceptance | 心理学 | 对死亡事实的认知与情感整合（分接近/中性/逃避三型） |
| **死亡态度** | Death Attitude | 心理学 | 对死亡的总体心理取向，包含恐惧、回避、接纳等多维度 |
| **临终质量** | Quality of Dying / Quality of Death | 医疗质量 | 临终患者在生命最后阶段的身体、心理、社会、灵性福祉 |
| **复杂性哀伤** | Complicated Grief (CG) / Prolonged Grief Disorder (PGD) | DSM-5-TR/ICD-11 | 持续6个月以上、功能严重受损的病理性情反应 |
| **灵性痛苦** | Spiritual Distress | NANDA/安宁疗护 | 个体在意义、目的、信仰、超越性方面经历的痛苦 |
| **预立指示完成度** | Advance Directive Completion | 医学伦理 | 个人预立医疗指示文件的完整性与法律有效性 |
| **心理测量信度** | Reliability (Cronbach's α) | 心理测量学 | 量表测量结果的一致性与稳定性 |
| **心理测量效度** | Validity | 心理测量学 | 量表测量其所声称测量构念的程度 |

## 一、死亡态度与焦虑量表

### 1.1 坦普勒死亡焦虑量表 (Templer Death Anxiety Scale, DAS)

| 维度 | 内容 |
|---|---|
| **作者/年份** | Donald Templer, 1970 |
| **维度** | 单维度 |
| **条目数** | 15 |
| **计分方式** | 是/否（0/1） |
| **信度** | α = .76-.87 |
| **适用人群** | 成人通用 |
| **特点** | 最广泛使用的死亡焦虑量表；简短；但单维度限制了信息量 |
| **代表性条目** | "我非常害怕死亡" "死亡的想法经常进入我的脑海" |

### 1.2 死亡态度描绘-修订版 (Death Attitude Profile-Revised, DAP-R)

| 维度 | 条目数 | 核心内容 | 临床意义 |
|---|---|---|---|
| **恐惧死亡** | 7 | 对死亡本身的情绪反应 | 核心死亡焦虑 |
| **死亡回避** | 5 | 回避死亡话题与情境 | 防御机制评估 |
| **中性接纳** | 5 | 平静地接受死亡为生命的一部分 | 适应性态度 |
| **趋近接纳** | 5 | 相信死后有更好存在 | 宗教/灵性资源 |
| **逃避接纳** | 5 | 因生活痛苦而渴望死亡 | **风险信号**：需鉴别自杀意念 |
| **信度** | α = .65-.97（各维度不同） | | |
| **适用人群** | 成人；研究与临床均可 | | |

### 1.3 多维死亡恐惧量表 (Multidimensional Fear of Death Scale, MFODS)

| 维度 | 条目数 | 内容 | 临床关联 |
|---|---|---|---|
| **对死者的恐惧** | 5 | 对尸体、葬礼的恐惧 | 特定恐惧症 |
| **对死后未知的恐惧** | 5 | 对死后状态的担忧 | 存在性焦虑 |
| **对临终的恐惧** | 5 | 对死亡过程的恐惧 | 临终焦虑 |
| **对早死的恐惧** | 5 | 对英年早逝的担忧 | 疑病症/健康焦虑 |
| **对被遗忘的恐惧** | 5 | 担心死后无人记得 | 意义危机 |
| **对意识的恐惧** | 5 | 对意识终止的恐惧 | 虚无主义 |
| **对身体的恐惧** | 5 | 对身体腐败的恐惧 | 躯体变形恐惧 |
| **对孤立的恐惧** | 5 | 死亡时的孤独感 | 依恋创伤 |
| **总条目** | 42 | | |
| **信度** | α = .65-.90（各维度） | | |

### 1.4 科利特-雷斯特死亡恐惧量表 (Collett-Lester Fear of Death Scale, CLFDS)

| 维度 | 条目数 | 内容 |
|---|---|---|
| **对自身死亡的恐惧** | 7 | 核心死亡恐惧 |
| **对死亡过程的恐惧** | 7 | 濒死恐惧 |
| **对他人死亡的恐惧** | 7 | 丧亲恐惧 |
| **对死后未知的恐惧** | 7 | 存在性恐惧 |
| **总条目** | 28（2003修订版） | |
| **信度** | α = .91 | |
| **特点** | 区分自身vs他人死亡；研究用 | |

### 1.5 存在焦虑问卷 (Existential Anxiety Questionnaire, EAQ)

| 维度 | 条目数 | 内容 |
|---|---|---|
| **死亡焦虑** | 6 | Yalom四大终极关怀之一 |
| **自由与责任焦虑** | 6 | |
| **孤独焦虑** | 6 | |
| **无意义焦虑** | 6 | |
| **总条目** | 24 | |
| **信度** | α = .84 | |
| **适用人群** | 青少年/成人 | |

### 1.6 死亡接纳量表 (Death Acceptance Scale, DAS)

| 维度 | 内容 | 测量目标 |
|---|---|---|
| **自然接纳** | 接受死亡为自然过程 | 适应性死亡态度 |
| **超然接纳** | 相信死后有灵性存在 | 宗教/灵性资源 |
| **趋近接纳** | 因生活痛苦而接受死亡 | **风险鉴别** |
| **代表性工具** | Death Attitude Profile-Revised中的接纳维度；独立量表如Dickstein的Death Acceptance Scale | |

## 二、临终质量评估

### 2.1 FAMCARE量表

| 维度 | 内容 |
|---|---|
| **用途** | 测量家属对安宁疗护服务的满意度 |
| **维度** | 信息提供、症状控制、家属支持、协调、可及性 |
| **条目数** | 20 |
| **计分** | Likert 5点量表 |
| **适用** | 安宁疗护质量评估；研究 |

### 2.2 CANHELP量表

| 维度 | 内容 |
|---|---|
| **用途** | 测量患者/家属对临终关怀的满意度 |
| **维度** | 医患关系、沟通、共享决策、症状管理、临终准备 |
| **条目数** | 21 |
| **特点** | 患者版与家属版；可比较视角差异 |

### 2.3 VOICE量表

| 维度 | 内容 |
|---|---|
| **用途** | 测量家属对癌症临终照护的体验 |
| **维度** | 症状控制、沟通、尊重、情感支持、协调 |
| **条目数** | 31 |
| **特点** | 聚焦癌症患者；详细 |

### 2.4 善终指标 (Good Death Indicator, GDI)

| 维度 | 评估内容 | 测量方式 |
|---|---|---|
| **身体舒适** | 疼痛控制、呼吸困难、恶心等症状 | 患者自评+家属观察 |
| **心理平静** | 焦虑、抑郁、接受度 | 患者访谈 |
| **社会完整** | 关系修复、家庭和谐 | 家属访谈 |
| **灵性安宁** | 意义感、无重大遗憾 | 患者访谈 |
| **自主实现** | 死亡地点、医疗决策符合意愿 | 记录复核 |
| **家庭准备** | 家属理解并接受、有告别机会 | 家属访谈 |

## 四、哀伤评估工具

### 4.1  prolonged grief disorder (PGD) 评估

| 工具 | 条目数 | 用途 | 诊断标准 |
|---|---|---|---|
| **ICG-R (Inventory of Complicated Grief-Revised)** | 19 | 复杂性哀伤筛查与诊断 | DSM-5-TR PGD标准 |
| **PG-13 (Prolonged Grief-13)** | 13 | 快速筛查 | ICD-11 PGD标准 |
| **TRIG (Texas Revised Inventory of Grief)** | 21 | 哀伤强度评估 | 研究用 |
| ** Hogan Grief Reaction Checklist (HGRC)** | 61 | 全面哀伤反应评估 | 研究用 |

### 4.2 DSM-5-TR Prolonged Grief Disorder 诊断标准

| 维度 | 标准 |
|---|---|
| **时间** | 丧亲后至少12个月（儿童6个月） |
| **核心症状** | 对逝者的强烈渴望和/或沉浸性思念 |
| **附加症状（需≥3项）** | 身份瓦解、意义感丧失、逃避提醒、情感麻木、难以信任、孤独感、生命无趣 |
| **功能损害** | 社交、职业或其他重要功能领域显著受损 |
| **鉴别** | 不能更好地用抑郁、PTSD、物质使用解释 |

## 五、儿童死亡概念与哀伤评估

### 5.1 儿童死亡概念评估

| 方法 | 年龄 | 操作 | 评估内容 |
|---|---|---|---|
| **访谈法** | 4+岁 | "人死了会怎么样？""死了还会回来吗？""所有人都会死吗？""为什么会死？" | Nagy四维度：永久性、普遍性、非功能性、因果性 |
| **绘画分析** | 3+岁 | "画一个死去的人"/"画出死亡" | 对死亡的理解、情感反应、 magical thinking |
| **游戏观察** | 3-8岁 | 提供医院/葬礼玩具，观察游戏内容 | 对死亡的认知、情感表达、 coping 模式 |
| **木偶/故事完成** | 4-10岁 | "小狗的好朋友小猫死了，小狗会…" | 投射性评估 |

### 5.2 儿童哀伤量表

| 工具 | 年龄 | 维度 | 条目数 |
|---|---|---|---|
| **CGIC (Childhood Grief and Importance of Circumstances)** | 6-18岁 | 哀伤反应、情境因素 | 20 |
| **CRIES (Children's Revised Impact of Event Scale)** | 8+岁 | 创伤性哀伤（闯入、回避、觉醒） | 13 |
| **WGRO (Working Model of the Child Interview - Grief)** | 学龄前 | 依恋与哀伤 | 访谈 |

## 六、自杀风险评估工具

### 6.1 标准化量表

| 工具 | 适用人群 | 时间 | 特点 |
|---|---|---|---|
| **C-SSRS (Columbia Suicide Severity Rating Scale)** | 全年龄（有各版本） | 5-10分钟 | 金标准；筛查+风险评估；广泛用于临床、学校、军队 |
| **ASQ (Ask Suicide-Screening Questions)** | 青少年/成人 | 2分钟 | 4个是/否问题；适合急诊快速筛查 |
| **BSS (Beck Scale for Suicidal Ideation)** | 成人 | 20分钟 | 半结构化访谈；详细评估意念、计划、准备 |
| **SIQ (Suicidal Ideation Questionnaire)** | 青少年 | 10分钟 | 自评；频率、态度、计划 |
| **RSIQ (Reynolds SIQ)** | 青少年 | 10分钟 | 自评；频率、计划、暴露 |
| **NGASR (Nurses' Global Assessment of Suicide Risk)** | 成人 | 10分钟 | 护士用；15个风险因素清单 |

### 6.2 C-SSRS 简化筛查版

| 问题 | 回答"是"的含义 |
|---|---|
| 1. 你是否希望死去？ | 被动死亡愿望 |
| 2. 你是否有过结束自己生命的想法？ | 主动自杀意念 |
| 3. 你有没有想过具体的自杀方法？ | 自杀计划 |
| 4. 你有没有打算真的去实施？ | 自杀意图 |
| 5. 你有没有采取过任何准备行动？ | 自杀行为准备 |
| 6. 你有没有尝试过自杀？ | 自杀未遂史 |
| 7. 你是否有过自伤行为？ | 非自杀性自伤 |

## 八、测量选择的临床决策树

```
评估目的
    ├─ 筛查 → 简短工具（ASQ 2分钟 / DAS 15条目）
    ├─ 诊断 → 金标准工具（BSS / ICG-R / C-SSRS完整版）
    ├─ 疗效评估 → 前后测敏感工具（DAP-R / MLQ / FFMQ）
    └─ 研究 → 多维度、高信效度工具（MFODS / CLFDS）

评估对象
    ├─ 儿童 → 发展适宜工具（访谈+绘画 / CGIC / CRIES）
    ├─ 青少年 → 青少年版本（C-SSRS Youth / SIQ）
    ├─ 成人 → 通用工具
    ├─ 老年人 → 考虑认知负荷（简化版 / 访谈优先）
    └─ 临终患者 → 简短、不疲劳（FICA / 单条目筛查）

评估维度
    ├─ 死亡焦虑 → DAS / DAP-R / MFODS
    ├─ 死亡接纳 → DAP-R接纳维度 / Death Acceptance Scale
    ├─ 存在焦虑 → EAQ / Yalom访谈
    ├─ 临终质量 → FAMCARE / CANHELP / VOICE
    ├─ 灵性 → FICA / SPIRIT / HOPE / FACIT-Sp
    ├─ 哀伤 → ICG-R / PG-13 / TRIG
    ├─ 自杀风险 → C-SSRS / ASQ / BSS
    └─ 正念/意义 → FFMQ / MLQ / AWE-S
```
