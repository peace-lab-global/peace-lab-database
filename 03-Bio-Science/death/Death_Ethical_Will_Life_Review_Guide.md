# Ethical Will & Life Review Guide (伦理遗嘱与生命回顾实操指南)

> *"我们每个人都有两个遗产：一个是我们留给世界的物质遗产，另一个是我们留给所爱之人的精神遗产。"*
> — Barry Baines, *Ethical Wills*

---

## 核心术语与词源表 (Core Terminology & Etymology)

| 术语 (中文) | 英文名 | 词源/来源 | 精确定义 |
|---|---|---|---|
| **伦理遗嘱** | Ethical Will | 犹太传统 / Barry Baines | 书面或录制的文件，传递价值观、人生教训、宽恕请求、对家人的期望，而非财产分配 |
| **生命回顾** | Life Review | Robert Butler, 1963 | 老年人自然发生的（或治疗引导的）对整个人生的回顾过程 |
| **生平回顾疗法** | Life Review Therapy | Robert Butler / James Birren | 结构化的治疗性生命回顾，旨在整合人生经验、解决未竟事务 |
| **四道人生** | Four Things That Matter Most | Ira Byock | 道歉、道谢、道爱、道别——临终前最重要的四句 |
| **预立指示** | Advance Directive | 医学伦理 | 对未来医疗决策的预先安排 |
| **持久授权书** | Power of Attorney | 法律 | 指定他人在自己无决策能力时代为决策 |
| **数字遗产** | Digital Legacy | 数字时代 | 个人在数字世界的遗留物（社交账号、照片、文档、AI模拟） |
| **记忆遗产** | Memory Legacy | 叙事医学 | 通过故事、访谈、录音录像保存的个人历史 |
| **持续性联结** | Continuing Bonds | Klass et al., 1996 | 丧亲后与逝者的关系以新的形式继续存在 |
| **遗产项目** | Legacy Project | 安宁疗护 | 患者创作的、将作为其精神遗产留存的项目 |

## 一、伦理遗嘱（Ethical Will）

### 1.1 什么是伦理遗嘱？

| 对比维度 | 法律遗嘱（Legal Will） | 伦理遗嘱（Ethical Will） |
|---|---|---|
| **内容** | 财产分配、法律事务 | 价值观、人生教训、宽恕、爱、期望 |
| **法律效力** | 有 | 无（但具有深远的情感/精神价值） |
| **形式** | 法律文件 | 书信、视频、音频、艺术作品 |
| **读者** | 继承人、执行人 | 家人、后代、朋友 |
| **目的** | 物质财富的传承 | 精神财富的传承 |
| **时间** | 任何年龄 | 任何年龄；临终时尤其有意义 |

### 1.2 伦理遗嘱的历史

| 传统 | 形式 | 特点 |
|---|---|---|
| **犹太传统** | *Zava'ah*（伦理遗嘱） | 可以追溯到圣经时代；父亲向儿子传递道德教诲 |
| **基督教传统** | 灵性遗嘱 | 圣徒和修士的灵性遗言 |
| **伊斯兰传统** | *Wasiyya* | 包含宗教义务和道德教诲 |
| **东方传统** | 家书、家训 | 颜氏家训、曾国藩家书等 |
| **现代形式** | 视频伦理遗嘱、数字伦理遗嘱 | 多媒体、可广泛分享 |

### 1.3 伦理遗嘱写作指南

#### 步骤一：准备

| 准备项 | 操作 |
|---|---|
| **选择形式** | 书信、视频、音频、绘画、音乐 |
| **选择受众** | 特定的家人、后代、朋友、或"未来的你" |
| **设定氛围** | 安静的时间；可能有点仪式感（点蜡烛、放音乐） |
| **心理准备** | 这不是告别；这是爱的传递 |

#### 步骤二：内容框架

```
伦理遗嘱结构模板

【开场】
"亲爱的[名字]，当我写这封信时，我在想…"

第一部分：我是谁（身份与价值观）
- 我最核心的3-5个价值观是什么？
- 这些价值观从何而来？
- 我如何在生命中实践它们？

第二部分：我的人生教训
- 我犯过的最大错误是什么？我学到了什么？
- 我最骄傲的时刻是什么？为什么？
- 如果我能重来，我会怎么做不同？

第三部分：我对你的期望（不是要求）
- 我希望你记住的关于我的事
- 我对你的希望和祝福
- 我相信你能成为什么样的人

第四部分：宽恕与道歉
- 我后悔说过的话或做过的事
- 我请求你的原谅
- 我也原谅你（如果有需要）

第五部分：爱与感谢
- 你给我的生命中带来了什么
- 我爱你，因为…
- 谢谢你，因为…

第六部分：告别（可选）
- 如果这是我最后的留言，我想让你知道…
- 我的希望、我的梦想、我的祝福

【结尾】
"永远爱你的，[名字]，[日期]"
```

#### 步骤三：写作提示

| 主题 | 提示问题 |
|---|---|
| **家庭** | "关于我们的家庭，我最珍视的是什么？" |
| **工作** | "我的工作教会了我什么关于人生的事？" |
| **失败** | "我最痛苦的失败是什么？它如何塑造了现在的我？" |
| **信仰** | "我的信仰/灵性如何帮助我度过困难时期？" |
| **快乐** | "我最快乐的记忆是什么？" |
| **遗憾** | "我最大的遗憾是什么？" |
| **智慧** | "如果我只能给你一个建议，那会是什么？" |
| **爱** | "爱对我意味着什么？" |

### 1.4 伦理遗嘱示例（节选）

> **致我的孩子们**
>
> 我最亲爱的孩子们：
>
> 当你们读到这封信时，我可能已经不在了。但我想让你们知道，我从未停止爱你们，即使在那些我表现得很糟糕的时刻。
>
> **关于我的价值观**：我这辈子最在乎的三件事是——诚实、善良、和勇气。诚实不是说出所有的真相，而是不说谎。善良不是做圣人，而是在可以伤害别人时选择不伤害。勇气不是不害怕，而是害怕时仍然去做正确的事。
>
> **我的教训**：我最大的错误是曾经为了工作错过了你们成长中的太多时刻。我希望你们记住：没有任何工作值得牺牲与所爱之人在一起的时间。我学到的太晚了，但你们还可以早学。
>
> **我的请求**：我希望你们彼此照顾。不是因为我要求你们这样做，而是因为我知道，当你们在一起时，你们是最强大的。
>
> **道歉**：我后悔曾经对你们大声吼叫。我后悔曾经把压力发泄在你们身上。这些不是你们的错，是我的。请你们原谅我。
>
> **感谢**：谢谢你们教会我做父亲。谢谢你们给我的每一次拥抱、每一张涂鸦、每一次"爸爸我爱你"。这些是我这辈子收到的最珍贵的礼物。
>
> 永远爱你们的，爸爸

## 二、生命回顾（Life Review）

### 2.1 Robert Butler 的生命回顾理论

> *"生命回顾是老年人普遍经历的一种自然过程，通过回忆过去来整合人生经验、准备面对死亡。"*
> — Robert N. Butler, 1963

| 维度 | 自然生命回顾 | 治疗性生命回顾 |
|---|---|---|
| **发生** | 自然发生 | 由治疗师引导 |
| **结构** | 无结构 | 有结构 |
| **目标** | 自发的整合 | 解决未竟事务、减少抑郁、提升福祉 |
| **深度** | 可深可浅 | 系统地覆盖人生各阶段 |

### 2.2 治疗性生命回顾的步骤

| 步骤 | 操作 | 时间 |
|---|---|---|
| **1. 建立关系** | 解释生命回顾的目的；建立信任 | 1-2次会谈 |
| **2. 人生时间线** | 画出人生时间线：关键事件、转折、关系 | 1次会谈 |
| **3. 阶段回顾** | 按阶段回顾：童年、青少年、成年早期、中年、老年 | 每次会谈一个阶段 |
| **4. 主题探索** | 识别反复出现的主题、模式、核心信念 | 贯穿全程 |
| **5. 未竟事务** | 识别并处理未解决的情感、关系、事件 | 中期 |
| **6. 整合与意义** | 帮助来访者构建一个连贯的人生叙事 | 后期 |
| **7. 遗产与传承** | 讨论希望留下的遗产；可能开始伦理遗嘱 | 最后阶段 |

### 2.3 结构化访谈指南（Birren & Cochran 的 Guided Autobiography）

| 主题 | 引导问题 |
|---|---|
| **分支点（Branching Points）** | "你人生中最重要的决定是什么？如果选择不同，人生会如何？" |
| **家庭** | "你的家庭如何塑造了你？你继承了什么？你改变了什么？" |
| **金钱/工作** | "钱对你意味着什么？你的工作如何定义了你？" |
| **健康/身体** | "你的身体如何影响了你的人生？" |
| **爱与关系** | "谁是你生命中最重要的人？为什么？" |
| **损失/死亡** | "你失去了什么？这些丧失如何改变了你？" |
| **意义/灵性** | "什么给了你生命的意义？这个意义如何随时间变化？" |
| **未来/遗产** | "你希望如何被记住？" |

### 2.4 生命回顾中的困难情绪处理

| 情绪 | 表现 | 干预 |
|---|---|---|
| **内疚** | "我本该做得更好" | 认知重构；宽恕练习；四道人生 |
| **愤怒** | "他们伤害了我" | 表达性写作；空椅技术； forgiveness 不是 forget |
| **悲伤** | "我失去了太多" | 哀伤正常化；持续性联结 |
| **后悔** | "如果当初…" | 接受不完美；寻找"遗憾中的智慧" |
| **骄傲** | "我做到了" | 庆祝； legacy 确认 |

## 三、四道人生的实操

### 3.1 Ira Byock 的四道

| 道 | 话语 | 功能 | 实操脚本 |
|---|---|---|---|
| **道歉** | "请原谅我" | 释放内疚；修复关系 | "我知道我伤害了你，我很抱歉。请你原谅我。" |
| **道谢** | "谢谢你" | 确认价值；表达感激 | "谢谢你为我做的一切。特别是[具体事例]。" |
| **道爱** | "我爱你" | 确认情感连接 | "我爱你。我一直爱你。即使我没有总是说出来。" |
| **道别** | "再见/保重" | 完成分离仪式 | "我要走了。请你好好照顾自己。再见。" |

### 3.2 四道人生的家庭会议指南

| 步骤 | 操作 | 注意 |
|---|---|---|
| **准备** | 与各方沟通四道的意义；征得同意 | 不强迫任何人 |
| **设定场景** | 安静、私密的空间；足够时间 | 避免匆忙 |
| **轮流进行** | 每个人向患者表达四道 | 允许沉默、哭泣 |
| **患者的回应** | 如患者有能力，也允许其回应 | 不强迫 |
| **结束** | 以身体接触（握手、拥抱）结束 | 尊重文化 |

## 四、预立指示的实操清单

### 4.1 预立指示准备清单

| 类别 | 问题 | 记录 |
|---|---|---|
| **代理人** | 谁是我的医疗决策代理人？备选是谁？ | 姓名、电话、关系 |
| **生命维持** | 如果我心跳停止，我希望心肺复苏吗？ | 是/否 |
| **人工呼吸** | 如果我无法自主呼吸，我希望使用呼吸机吗？ | 是/否/短期 |
| **管饲** | 如果我无法进食，我希望管饲吗？ | 是/否/短期 |
| **疼痛控制** | 我希望疼痛被充分控制吗？即使可能影响意识？ | 是/否 |
| **透析** | 如果肾衰竭，我希望透析吗？ | 是/否 |
| **地点** | 我希望在哪里度过最后时光？家/医院/安宁疗护机构？ | |
| **灵性** | 我希望什么样的灵性/宗教支持？ | |
| **遗体** | 我希望遗体如何处理？火化/土葬/捐献？ | |
| **葬礼** | 我希望什么样的纪念仪式？ | |
| **器官捐献** | 我希望捐献器官吗？ | 是/否 |

### 4.2 家庭会议 facilitation 指南

| 步骤 | 操作 |
|---|---|
| **开场** | "我们今天聚在一起，是因为[患者]希望和我们讨论他/她对未来医疗的意愿。这不是一个容易的话题，但[患者]希望我们知道他/她的想法。" |
| **患者发言** | 让患者先表达（如可能） |
| **提问与澄清** | 家人可以提问，澄清患者的意愿 |
| **情感支持** | 允许情感表达；正常化困难情绪 |
| **记录** | 指定一人记录；会后整理成书面文件 |
| **后续** | 安排法律手续（如需要）；定期复核 |

## 五、遗产项目（Legacy Projects）

### 5.1 遗产项目类型

| 类型 | 描述 | 适合人群 |
|---|---|---|
| **口述历史** | 录制人生故事 | 善于讲述的人 |
| **食谱书** | 收集家族食谱及背后的故事 | 热爱烹饪的人 |
| **记忆 quilts** | 用衣物制作 quilts | 手工爱好者 |
| **照片书** | 整理人生照片并配文字 | 有丰富照片的人 |
| **视频留言** | 为未来的孙子/曾孙录制视频 | 所有人群 |
| **慈善基金** | 以逝者名义设立慈善基金 | 经济条件允许的人 |
| **植树/花园** | 种植纪念树或花园 | 热爱自然的人 |
| **信件时间胶囊** | 为未来的家人写时间胶囊信件 | 所有人群 |

### 5.2 遗产项目的治疗价值

| 价值 | 说明 |
|---|---|
| **意义感** | "我的生命有价值；我的故事值得被记住" |
| **控制感** | 在失控的疾病中，仍能控制"如何被记住" |
| **关系修复** | 遗产项目可以成为与家人合作的契机 |
| ** legacy 确认** | "我已经留下了一些东西" |
| **哀伤准备** | 家属在参与项目时提前开始哀伤工作 |
