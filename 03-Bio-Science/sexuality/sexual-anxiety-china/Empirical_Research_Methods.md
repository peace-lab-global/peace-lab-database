# 性焦虑研究的实证方法论体系

## 研究设计框架

### 混合方法研究设计

**定量研究组件**
**横断面调查研究**
- 大规模代表性样本调查(n≥5000)
- 分层随机抽样策略
- 标准化测量工具应用
- 信效度检验程序

**纵向追踪研究**
- 队列研究设计(3-5年追踪)
- 关键时间节点设置
- 测量工具稳定性验证
- 缺失数据分析处理

**实验研究设计**
- 随机对照试验(RCT)标准流程
- 准实验设计适应性调整
- 中介效应和调节效应分析
- 效应量计算和meta分析

**定性研究组件**
**深度访谈研究**
- 半结构化访谈提纲设计
- 现象学研究方法应用
- 扎根理论编码分析
- 参与者triangulation验证

**焦点小组研究**
- 同质性分组策略
- 主持人中立性保障
- 互动过程录音录像
- 内容分析和主题提取

**民族志研究**
- 参与观察法实施
- 文化context深度理解
- 厚描述(thick description)原则
- reflexive journal维护

### 测量工具体系

**核心量表开发**
**性焦虑量表(SAS-Chinese)**
```
维度结构：
1. 性表现焦虑(Performance Anxiety) - 7题
2. 性道德焦虑(Moral Anxiety) - 5题  
3. 关系性焦虑(Relational Anxiety) - 6题
4. 身体形象焦虑(Body Image Anxiety) - 4题
5. 安全性焦虑(Safety Anxiety) - 3题

评分标准：5点李克特量表(1-5分)
信效度指标：
- Cronbach's α ≥ 0.90
- 重测信度 ≥ 0.85
- 内容效度CVI ≥ 0.80
- 结构效度CFI ≥ 0.90
```

**辅助测量工具**
- 性功能指数(SFI)中英文版本
- 焦虑敏感性指数(ASI-3)文化适应
- 亲密关系质量量表(IRQS)验证
- 生活质量量表(WHOQOL-BREF)应用

### 样本代表性保障

**分层抽样策略**
```
地理分层：
- 一线城市(北京、上海、广州、深圳)
- 二线城市(省会城市及计划单列市)
- 三四线城市(地级市)
- 农村地区(县乡镇)

人口分层：
- 年龄段：15-25岁、26-35岁、36-45岁、46-55岁、55岁以上
- 性别：男性、女性、其他性别认同
- 教育水平：初中及以下、高中/职高、大专、本科、研究生及以上
- 职业类别：学生、白领、蓝领、自由职业、退休
```

**样本量计算**
- 总体样本量：≥10,000人
- 各亚群体样本量：≥500人
- 统计检验力：≥0.80
- 置信水平：95%
