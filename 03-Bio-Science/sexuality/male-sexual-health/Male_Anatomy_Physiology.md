# Male Anatomy and Physiology (男性解剖与生理基础)

## 概述

男性性健康的生物学基础包括生殖系统结构、神经血管调控以及内分泌轴的协同作用。本主题为后续“勃起唤起”“射精与高潮”“生育力”等模块提供共同的生理框架。

## 机制与生理基础

### 关键结构与功能

| 结构 | 解剖要点 | 主要功能 | 临床关注点 |
| :--- | :--- | :--- | :--- |
| **阴茎海绵体** | 富含平滑肌与血窦 | 勃起时充血维持硬度 | 血管内皮功能、静脉漏 | 
| **前列腺** | 围绕尿道的腺体组织 | 分泌前列腺液、参与射精 | 炎症、增生、敏感性差异 |
| **精囊腺** | 后上方成对腺体 | 分泌精浆成分 | 炎症与射精痛 |
| **睾丸** | 生精小管 + 莱迪希细胞 | 产生精子与睾酮 | 温度调控、曲张静脉 |
| **盆底肌群** | 球海绵体肌、坐骨海绵体肌 | 控制射精与快感强度 | 肌张力失衡、训练不足 |

### 性反应周期（经典阶段）

| 阶段 | 主要生理变化 | 主导系统 | 关键指标 |
| :--- | :--- | :--- | :--- |
| **欲望** | 睾酮与多巴胺驱动 | 中枢神经系统 | 性动机、注意偏向 |
| **唤起** | 海绵体血流上升 | 副交感神经 | 硬度、充血 |
| **平台期** | 神经血管稳定 | 中枢-周围协同 | 心率、肌张力 |
| **高潮** | 盆底节律性收缩 | 脊髓反射弓 | 快感峰值 |
| **消退** | 血流回落 | 交感神经 | 不应期 |

### 内分泌轴与性功能

| 轴 | 关键激素 | 作用 | 影响因素 |
| :--- | :--- | :--- | :--- |
| **下丘脑-垂体-性腺轴** | GnRH、LH、FSH、睾酮 | 生精与性欲维持 | 睡眠、压力、代谢状态 |
| **肾上腺轴** | DHEA、皮质醇 | 性欲与应激调节 | 慢性压力、抑郁 |
| **甲状腺轴** | T3/T4 | 代谢与能量 | 甲亢/甲减 |

## 临床与实践要点

- **基础评估**：体检 + 激素谱（总/游离睾酮、LH、FSH、PRL）+ 生活方式评估。
- **生活方式**：规律睡眠、体重管理、阻力训练与有氧结合可改善激素与血管功能。
- **盆底肌训练**：适度训练可改善射精控制与快感感知。

## 风险、伦理与注意事项

- **避免自行诊断**：性功能变化可能与心血管、内分泌或心理因素相关。
- **健康不等于“性能”**：强调功能与满意度，而非单一指标。
- **隐私与尊重**：临床沟通需避免羞耻化表达。

## 研究与证据

- 男性性功能研究强调**生物-心理-社会**模型，多数问题非单一器质性原因。
- 生理指标与主观满意度并非线性关系，应联合评估。
