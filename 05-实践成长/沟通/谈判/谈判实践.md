## 二、经典谈判案例

### 2.1 橙子谈判

> 谈判学最经典的教学案例

| 要素 | 内容 |
|------|------|
| 情境 | 两姐妹争一个橙子 |
| 立场型解决 | 把橙子切两半，各拿一半 |
| 利益型解决 | 姐姐要橙皮做蛋糕，妹妹要橙汁 → 各取所需 |
| 教训 | 立场背后的利益可能不同，探索利益可以创造双赢 |

### 2.2 薪资谈判模拟

| 角色 | 信息 |
|------|------|
| 求职者 | 目标薪资30万，BATNA是现有工作25万，市场中位数28万 |
| 招聘方 | 预算范围22-32万，候选人是最佳人选 |
| 练习要点 | 锚定、有条件让步、扩展议题(奖金/期权/假期) |

### 2.3 商业谈判模拟

| 角色 | 信息 |
|------|------|
| 供应商 | 希望长期合同、较高单价 |
| 采购方 | 希望低价、灵活条款 |
| 练习要点 | 打包交易、数量折扣、付款条件、独家条款 |

## 三、困难谈判策略

### 3.1 应对强硬谈判者

| 战术 | 应对策略 |
|------|---------|
| 威胁 | 不回应威胁，重回利益讨论 |
| 最后通牒 | 测试其真实性，提出替代方案 |
| 好警察/坏警察 | 识别战术，直接指出 |
| 沉默压力 | 保持耐心，不急于填补 |
| 虚假截止日期 | 验证真实性 |
| 情绪爆发 | 保持冷静，标注对方情绪 |

### 3.2 僵局突破

| 技术 | 说明 |
|------|------|
| 暂时搁置 | 先跳过争议点，处理其他议题 |
| 重新框架 | 改变问题的定义方式 |
| 引入调解 | 请中立第三方协助 |
| 休息 | 给双方冷静和重新评估的时间 |
| 改变代表 | 换一个谈判代表 |
| 非正式渠道 | 在正式谈判外建立沟通 |

## 四、跨文化谈判

### 4.1 文化维度与谈判

| 文化特征 | 高语境(中/日/韩) | 低语境(美/德/澳) |
|---------|---------------|-----------------|
| 沟通风格 | 间接、含蓄、读空气 | 直接、明确、言语为主 |
| 关系优先级 | 先建关系再谈事 | 先谈事后建关系 |
| 时间观 | 弹性、注重过程 | 线性、注重效率 |
| 决策方式 | 共识导向、集体决策 | 个人权限、快速决策 |
| 合同地位 | 关系起点、可调整 | 最终约束、严格执行 |
| 面子意识 | 极高、避免公开冲突 | 较低、可以直接disagreement |

### 4.2 中国商务谈判要点

| 要点 | 说明 |
|------|------|
| 关系先行 | 通过饭局、茶叙建立信任 |
| 面子 | 绝不让对方公开丢面子 |
| 中间人 | 利用共同认识的人引荐 |
| 耐心 | 中国谈判通常需要更多时间 |
| 层级 | 尊重对方的决策层级 |
| 暧昧性 | 接受"差不多"和模糊承诺 |

## 五、谈判伦理

### 5.1 伦理灰色地带

| 行为 | 伦理性 | 说明 |
|------|--------|------|
| 隐藏BATNA | 可接受 | 不必透露替代方案 |
| 隐藏保留价格 | 可接受 | 这是谈判的正常部分 |
| 虚假声明 | 不道德 | "另一家出价更高"(实际没有) |
| 虚假让步 | 灰色 | 让步本不值那么多 |
| 威胁 | 灰色-不道德 | 取决于是否有能力和意愿执行 |
| 欺骗 | 不道德 | 关于实质事实的欺骗 |

### 5.2 长期主义谈判观

| 原则 | 说明 |
|------|------|
| 声誉资本 | 诚信谈判建立长期声誉 |
| 关系维护 | 今天的对手可能是明天的合作伙伴 |
| 可持续协议 | 不公平的协议难以持续执行 |
| 双赢倾向 | 双方满意的协议更稳固 |
