> **学科定位**: 05-实践成长 > personal-development > financial-literacy

## 一、个人财务健康评估

### 1.1 财务健康四维度

| 维度 | 指标 | 健康标准 |
|------|------|---------|
| **消费管理** | 收支比 | 支出 < 收入的80% |
| **储蓄保障** | 紧急基金月数 | ≥ 6个月生活费 |
| **债务控制** | 债务收入比 (DTI) | < 36% (住房贷除外 < 28%) |
| **资产增值** | 投资回报 vs 通胀 | 实际回报 > 0 |

### 1.2 个人财务报表

#### 资产负债表 (Balance Sheet)

| 资产类型 | 示例 | 负债类型 | 示例 |
|---------|------|---------|------|
| 流动资产 | 现金、存款、货基 | 短期负债 | 信用卡、消费贷 |
| 投资资产 | 基金、股票、债券 | 中期负债 | 车贷 |
| 固定资产 | 房产、车辆 | 长期负债 | 房贷 |
| 个人资产 | 珠宝、收藏品 | | |

**净资产 = 总资产 - 总负债**

#### 现金流量表 (Cash Flow Statement)

| 收入类型 | 支出类型 |
|---------|---------|
| 工资/奖金 | 固定支出 (房租/房贷/保险) |
| 投资收入 | 可变必需 (食物/交通/水电) |
| 副业/兼职 | 可变非必需 (娱乐/购物) |
| 被动收入 | 储蓄/投资/偿债 |

## 二、预算编制系统

### 2.1 主流预算方法对比

| 方法 | 原则 | 适合人群 | 工具 |
|------|------|---------|------|
| **50/30/20** | 需求/欲望/储蓄三分法 | 入门者 | 简单计算 |
| **零基预算** | 每分钱都有去处，收支=0 | 细致管理者 | YNAB |
| **信封法** | 各类支出放入不同信封 | 现金消费者 | 实体/虚拟信封 |
| **反向预算** | 先储蓄再消费 (先支付自己) | 自动化爱好者 | 自动转账 |
| **极简预算** | 只跟踪3-5个大类 | 不喜细节者 | 简单app |

### 2.2 预算执行的心理学

| 挑战 | 心理机制 | 解决策略 |
|------|---------|---------|
| 预算疲劳 | 意志力消耗 | 自动化扣款、减少决策 |
| 意外消费 | 计划谬误 | 预留"缓冲金"(5-10%) |
| 社交压力消费 | 从众/面子 | 提前设定限额、替代活动 |
| 冲动购买 | 多巴胺驱动 | 24小时冷却法则 |
| 月末放松 | 自我许可效应 | 周预算而非月预算 |

### 2.3 消费决策框架

**购买决策三问**:
1. **需要 vs 想要**: 这是生活必需还是情感欲望？
2. **每日成本**: 价格 ÷ 预计使用天数 → 每天值多少？
3. **时间成本**: 需要工作多少小时才能买到？

## 三、储蓄策略

### 3.1 储蓄的心理障碍与对策

| 障碍 | 心理学解释 | 对策 |
|------|-----------|------|
| "没有多余的钱" | 帕金森定律: 支出膨胀至收入 | 工资到账即自动转存 |
| "存了也没用" | 习得性无助 | 设定具体储蓄目标+可视化 |
| "以后再说" | 双曲贴现 | 利用默认选项+自动递增 |
| "生活要享受" | 即时满足偏好 | 对比表: 存30年的复利增长 |

### 3.2 储蓄加速策略

| 策略 | 方法 | 年增储蓄 |
|------|------|---------|
| **自动递增** | 每次加薪自动增加储蓄比例50% | 渐进增长 |
| **找钱挑战** | 将零散收入全额储蓄 | 数千元 |
| **无消费日** | 每周设1-2天不额外消费 | 数千元 |
| **退订审计** | 季度清理不用的订阅 | 数百-千元 |
| **延迟购买** | 非必需品等30天 | 减少20-30%冲动消费 |

## 四、债务管理

### 4.1 好债 vs 坏债

| 类型 | 特征 | 示例 |
|------|------|------|
| **好债** | 利率低、税收优惠、增值资产 | 合理房贷、教育贷 |
| **中性债** | 必要但需控制 | 适度车贷 |
| **坏债** | 高利率、贬值资产、消费驱动 | 信用卡分期、消费贷 |
| **有毒债** | 极高利率、不透明 | 高利贷、不规范网贷 |

### 4.2 债务偿还计划

#### 雪球法 (Dave Ramsey)

| 步骤 | 行动 |
|------|------|
| 1 | 列出所有债务，按余额从小到大排序 |
| 2 | 最低还款额支付所有债务 |
| 3 | 将所有多余资金投向最小余额 |
| 4 | 还清一笔后，金额"滚雪球"到下一笔 |

#### 雪崩法 (数学最优)

| 步骤 | 行动 |
|------|------|
| 1 | 列出所有债务，按利率从高到低排序 |
| 2 | 最低还款额支付所有债务 |
| 3 | 将所有多余资金投向最高利率债务 |
| 4 | 还清一笔后，金额转向下一个最高利率 |

### 4.3 信用管理

| 维度 | 建议 |
|------|------|
| 征信报告 | 每年至少查一次个人征信 |
| 信用卡使用 | 利用率保持在30%以下 |
| 按时还款 | 设置自动还款，避免逾期 |
| 信用历史 | 保持最老信用卡不销户 |

## 五、保险基础

### 5.1 个人保险配置优先级

| 优先级 | 险种 | 作用 | 关键参数 |
|--------|------|------|---------|
| ★★★★★ | 社会医保 | 基础医疗保障 | 必须参保 |
| ★★★★★ | 重疾险 | 大病收入替代 | 保额≥3-5年收入 |
| ★★★★ | 医疗险 (百万医疗) | 大额医疗费用 | 免赔额1万以内 |
| ★★★★ | 意外险 | 意外伤害/伤残 | 杠杆高、费率低 |
| ★★★ | 定期寿险 | 家庭责任 | 保额覆盖负债+5年收入 |
| ★★ | 年金/养老险 | 退休补充 | 中后期考虑 |

### 5.2 保险购买原则

| 原则 | 说明 |
|------|------|
| 先保障后理财 | 保障型保险优先于储蓄型/投资型 |
| 先大人后小孩 | 家庭经济支柱优先 |
| 先高频后低频 | 医疗/意外先于特定疾病 |
| 保额充足 | 保额比保费重要 |
| 如实告知 | 健康告知必须诚实 |

## 六、税务基础

### 6.1 个人所得税基础概念

| 概念 | 说明 |
|------|------|
| 应税收入 | 总收入 - 扣除项 |
| 累进税率 | 收入越高, 边际税率越高 |
| 专项扣除 | 社保/公积金 |
| 专项附加扣除 | 子女教育/继续教育/住房/赡养老人/大病医疗 |
| 合法节税 | 充分利用扣除项和税收优惠政策 |

### 6.2 主要合法节税策略

| 策略 | 方法 |
|------|------|
| 充分扣除 | 确保所有专项附加扣除都已填报 |
| 年终奖规划 | 利用年终奖单独计税政策优化 |
| 公积金上限 | 在允许范围内提高公积金缴存比例 |
| 商业健康险 | 税优健康险可抵扣 |
| 长期持有 | 持股超12个月分红减税 |

## 七、财务规划里程碑

### 7.1 按年龄段的财务目标

| 年龄 | 关键目标 | 行动清单 |
|------|---------|---------|
| 20-25 | 建立财务基础 | 开设储蓄账户、开始记账、建立1月紧急基金 |
| 25-30 | 资产积累起步 | 3月紧急基金、开始定投、获得适当保险 |
| 30-35 | 家庭财务规划 | 6月紧急基金、房产规划、配偶财务融合 |
| 35-45 | 资产增长期 | 多元投资、子女教育金、提高储蓄率 |
| 45-55 | 退休准备 | 加速储蓄、降低投资风险、遗嘱准备 |
| 55-65 | 退休过渡 | 退休收入测算、医疗保障、生活方式调整 |
| 65+ | 退休生活 | 安全提取率(4%法则)、遗产分配 |
