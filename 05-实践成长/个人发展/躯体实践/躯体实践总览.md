# 身体工作实践总览 | Somatic Practice Overview

> 身体工作(Somatic Practice)是一系列以身体觉察为核心的疗愈方法，强调"自下而上"(bottom-up)的身体智慧——创伤与情绪不仅存储在认知中，更深深刻写在筋膜、肌肉和神经系统里。

---

## 1. 核心理论

### 1.1 身体-心理连续体

| 概念 | 说明 |
|------|------|
| **躯体化(Somatization)** | 未处理的情绪/创伤通过身体症状表达 |
| **身体记忆** | 内隐记忆(程序性/情绪性)存储在基底节/杏仁核/小脑 |
| **冻结反应** | 未完成的战/逃反应被冻结在神经系统中 (Levine, 1997) |
| **内感受(Interoception)** | 感知身体内部状态的能力→情绪觉察的基础 (Craig, 2009) |
| **多迷走阶梯** | 腹侧迷走(安全)→交感(激活)→背侧迷走(冻结/解离) (Porges, 2011) |

### 1.2 自下而上 vs 自上而下

| 维度 | 自上而下(Top-Down) | 自下而上(Bottom-Up) |
|------|-------------------|---------------------|
| 代表 | CBT、正念、谈话疗法 | SE、Feldenkrais、Sensorymotor |
| 起点 | 认知/思维/意义 | 感觉/运动/身体反应 |
| 路径 | 前额叶→边缘系统→脑干 | 脑干→边缘系统→前额叶 |
| 优势 | 认知重构、意义建构 | 处理前语言创伤、自动模式 |
| 局限 | 难以触及身体冻结反应 | 需配合认知整合 |

## 2. 核心方法

### 2.1 躯体体验 (Somatic Experiencing, SE)

**创始人**: Peter Levine (1997)

**核心概念**:
- **创伤不是事件本身，而是神经系统中未完成的生物反应**
- 动物在危险后通过颤抖/抖动"释放"冻结能量；人类往往抑制这一过程
- 治疗目标：帮助神经系统完成被中断的防御反应

| 核心技术 | 描述 |
|----------|------|
| **Pendulation(摆荡)** | 在资源(安全/舒适)区域和激活(不适)区域之间缓慢摆荡 |
| **Titration(滴定)** | 极小剂量接触创伤激活→避免再创伤 |
| **完成反应** | 允许身体完成冻结的防御动作(推/跑/颤抖) |
| **身体觉察** | 持续追踪感觉(温度/紧绷/振动/流动) |
| **Voo练习** | 深长的低频发声→振动迷走神经→腹侧迷走激活 |

**循证**: Brom et al. (2017) RCT — SE显著↓PTSD症状(d=1.1)

### 2.2 费登奎斯方法 (Feldenkrais Method)

**创始人**: Moshe Feldenkrais (1904-1984)

| 形式 | 描述 |
|------|------|
| **ATM (Awareness Through Movement)** | 团体课，引导者口述指令→学生探索运动 |
| **FI (Functional Integration)** | 个体课，治疗师用手轻触引导运动 |

**核心原则**:
- **Weber-Fechner定律**: 刺激越小→觉察越敏锐
- **有机学习**: 像婴儿学习一样——通过探索而非指令
- **差异觉察**: 关注"什么不同"而非"什么正确"
- **无目标**: 不追求"正确姿势"，而是扩展运动选项

**循证**: 
- Buchanan & Jull (2002) — 改善慢性颈痛、平衡
- Connors et al. (2011) SR — 改善运动功能、疼痛、QoL

### 2.3 感觉运动心理治疗 (Sensorimotor Psychotherapy)

**创始人**: Pat Ogden (2006)

| 阶段 | 干预 |
|------|------|
| **第一阶段(稳定化)** | 身体资源建设、地面化(grounding)、姿势觉察 |
| **第二阶段(创伤处理)** | 追踪身体激活→完成防御动作→整合感觉 |
| **第三阶段(整合)** | 新行为实验、关系中的身体表达、日常整合 |

**与SE的区别**: SP整合了依恋理论，更强调关系中的身体模式

### 2.4 身体-心智中心化 (Body-Mind Centering, BMC)

**创始人**: Bonnie Bainbridge Cohen

- 基于胚胎发展和婴儿运动发展序列
- 通过觉察特定身体系统(骨骼/肌肉/器官/流体)来重组运动模式
- 用于创造性运动和深层身体觉察

### 2.5 Alexander Technique (亚历山大技术)

| 核心概念 | 说明 |
|----------|------|
| **Use(使用方式)** | 我们"使用"身体的习惯模式决定功能 |
| **Primary Control** | 头-颈-背关系是整体协调的关键 |
| **Inhibition** | 暂停自动反应→创造选择空间 |
| **Direction** | 给予建设性方向(如"让颈部自由") |

循证: Cacciatore et al. (2011) — 慢性背痛改善(APT试验)

### 2.6 其他身体工作

| 方法 | 核心技术 | 适应证 |
|------|----------|--------|
| **Rolfing(罗尔夫)** | 筋膜手法重组(10系列) | 姿势/慢性疼痛 |
| **Bioenergetics(生物能)** |  grounding/呼吸/表达 | 情绪阻塞/抑郁 |
| **Hakomi** | 正念+身体实验 | 自我探索/创伤 |
| **Rubenfeld Synergy** | 轻触+对话+意象 | 慢性疼痛/情绪 |
| **Trauma-Informed Yoga** | 正念运动+内感受 | PTSD/焦虑 |

## 3. 临床整合方案

### 3.1 按人群选择

| 人群 | 推荐方法 | 理由 |
|------|----------|------|
| **PTSD(复杂)** | SE → SP | 自下而上处理冻结反应 |
| **慢性疼痛** | Feldenkrais + Alexander | 运动模式重组 |
| **焦虑/恐慌** | SE(摆荡) + 瑜伽 | 神经系统调节 |
| **解离障碍** | SP(稳定化阶段) | 渐进身体重新居住 |
| **姿势/办公** | Feldenkrais ATM | 低门槛团体课 |
| **创伤后成长** | BMC + Authentic Movement | 深层身体探索 |
| **ASD儿童** | 感觉运动游戏 | 感觉整合 |

### 3.2 日常身体实践(最小可行方案)

```
晨间(5min): 仰卧身体扫描 → 注意温度/紧绷/流动
午间(3min): 站立 grounding → 脚底感知 → 骨盆摇摆
晚间(5min): Voo发声(3次) → 自主抖动(2min) → 静躺
```
