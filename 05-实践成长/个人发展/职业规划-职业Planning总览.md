## 职业发展理论

### 经典理论矩阵

| 理论 | 提出者 | 核心观点 | 适用情境 |
|------|--------|---------|---------|
| 特质-因素理论 | Parsons(1909) | 人-环境匹配 | 初次职业选择 |
| 职业发展理论 | Super(1957) | 生涯彩虹/生命广度 | 全生涯规划 |
| Holland类型论 | Holland(1959) | 6种人格-环境类型 | 职业匹配 |
| 社会认知理论 | Lent(1994) | 自我效能/结果期望/目标 | 职业兴趣发展 |
| 混沌理论 | Pryor(2003) | 复杂/非线性/偶然事件 | 不确定时代 |
| 生涯建构理论 | Savickas(2005) | 叙事/意义/适应性 | 后现代生涯 |
| 计划偶然理论 | Mitchell(1999) | 创造/利用偶然机遇 | 非线性职业路径 |

### Holland六角模型(RIASEC)

| 类型 | 特征 | 典型职业 |
|------|------|---------|
| R 实际型 | 动手/机械/户外 | 工程师/农民/技工 |
| I 研究型 | 分析/理论/好奇 | 科学家/研究员 |
| A 艺术型 | 创意/表达/直觉 | 艺术家/设计师/作家 |
| S 社会型 | 助人/教导/合作 | 教师/咨询师/社工 |
| E 企业型 | 领导/说服/风险 | 企业家/销售/律师 |
| C 事务型 | 组织/精确/规则 | 会计/行政/程序员 |

### Super生涯彩虹

| 阶段 | 年龄 | 任务 |
|------|------|------|
| 成长 | 0-14 | 自我概念形成/幻想/兴趣 |
| 探索 | 15-24 | 试探/过渡/初步承诺 |
| 建立 | 25-44 | 稳定/巩固/晋升 |
| 维持 | 45-64 | 保持/更新/创新 |
| 衰退 | 65+ | 减速/退休规划/角色转换 |

## 职业探索与评估

### 自我评估维度

| 维度 | 工具 | 说明 |
|------|------|------|
| 兴趣 | Holland SDS/Strong | 你喜欢做什么？ |
| 能力/技能 | CliftonStrengths/技能盘点 | 你擅长什么？ |
| 价值观 | Schwartz价值观/Life Values | 什么对你最重要？ |
| 人格 | MBTI/Big Five | 你的工作风格？ |
| 锚 | Schein职业锚 | 你不愿放弃什么？ |

### Schein八种职业锚

| 职业锚 | 核心需求 | 典型选择 |
|--------|---------|---------|
| 技术/功能 | 专业精通 | 专家/技术岗 |
| 管理 | 领导/整合 | 管理者/总监 |
| 自主/独立 | 自由/灵活 | 自由职业/咨询 |
| 安全/稳定 | 可预测/保障 | 公务员/大企业 |
| 创业 | 创造/控制 | 创业者/发明家 |
| 服务/奉献 | 帮助他人/社会贡献 | 非营利/医疗/教育 |
| 挑战 | 克服困难/竞争 | 战略咨询/投行 |
| 生活方式 | 工作-生活平衡 | 弹性工作/远程 |

## 职业决策

### 职业决策困难

| 类型 | 表现 | 干预 |
|------|------|------|
| 缺乏准备 | 缺乏动机/犹豫不决/信念不合理 | 动机探索/认知重构 |
| 缺乏信息 | 不了解自己/不了解职业/不知流程 | 信息收集/评估/实习 |
| 信息不一致 | 内部冲突/外部冲突 | 价值观澄清/优先排序 |

### Ikigai职业整合模型

```
         你热爱的
        ╱         ╲
   激情           使命
      ╲         ╱
你擅长的 — IKIGAI — 世界需要的
      ╱         ╲
   专业           天职
        ╲         ╱
       你能获得报酬的
```

| 交集 | 含义 |
|------|------|
| 激情(Passion) | 热爱 + 擅长 |
| 使命(Mission) | 热爱 + 世界需要 |
| 天职(Vocation) | 世界需要 + 获得报酬 |
| 专业(Profession) | 擅长 + 获得报酬 |
| Ikigai | 四者交汇 = 人生意义 |

## 职业发展阶段

### 职场新人(0-3年)

| 维度 | 建议 |
|------|------|
| 技能 | T型技能(广+深)/快速学习/反馈循环 |
| 关系 | 寻找导师/建立专业网络 |
| 心态 | 成长型思维/接受不确定/主动探索 |
| 品牌 | 开始建立专业声誉 |

### 职业中期(4-15年)

| 维度 | 建议 |
|------|------|
| 深化 | 成为某领域专家 或 转向管理 |
| 网络 | 从"被介绍"→"介绍他人" |
| 平衡 | 工作-生活整合(非平衡) |
| 意义 | 从成就驱动→意义驱动 |

### 职业后期(15年+)

| 维度 | 建议 |
|------|------|
| 传承 | 指导/培养下一代 |
| 转型 | 组合式职业/顾问/教育 |
| 遗产 | 思考"我想留下什么？" |
| 退休 | 渐进式退休/安可职业(Encore Career) |

## 当代职业挑战

### VUCA时代的职业策略

| 特征 | 应对 |
|------|------|
| 波动性(Volatility) | 敏捷适应/多技能储备 |
| 不确定性(Uncertainty) | 持续学习/实验心态 |
| 复杂性(Complexity) | 系统思维/跨域连接 |
| 模糊性(Ambiguity) | 舒适区扩展/容忍模糊 |

### 未来工作趋势

| 趋势 | 影响 |
|------|------|
| AI/自动化 | 重复性工作↓ 创意/情感↑ |
| 远程/混合工作 | 工作与生活边界模糊 |
| 零工经济 | 组合式职业/多元收入 |
| 终身学习 | 每3-5年需技能更新 |
| 健康/意义 | 工作意义>薪酬(代际变化) |
