# 职场语言霸凌与PUA：法律法规汇编 (Legal References for Workplace Verbal Bullying & PUA)

> **定位**：汇总与职场语言霸凌、羞辱性管理、孕产歧视、报复性管理、违法调岗、劝退协商相关的中国大陆常见法律法规与制度依据
> **重要提醒**：以下内容为一般性整理，不构成法律意见；具体争议应结合个案事实、证据和当地裁审口径判断

## 一、法律适用的基本思路

职场语言霸凌和PUA往往不是单独以“PUA”名义进入法律处理，而是通过以下路径进入正式规则：
- **人格权侵害**：侮辱、名誉、隐私、健康、性骚扰等
- **劳动权益侵害**：违法调岗、降薪、违法解除、未尽劳动保护义务
- **平等就业与反歧视**：性别、婚育、孕产、残障等差别待遇
- **争议处理程序**：内部申诉、劳动仲裁、诉讼、监察投诉

## 二、核心法律法规清单

| 法规 | 主要适用问题 | 可支持的主张方向 |
| --- | --- | --- |
| 《中华人民共和国民法典》 | 人格权、名誉权、隐私权、身体权、健康权、性骚扰防治 | 证明侮辱、人格侵害、精神损害相关风险 |
| 《中华人民共和国劳动法》 | 劳动保护、工资、工作条件、休息休假 | 作为基础劳动权益保护依据 |
| 《中华人民共和国劳动合同法》 | 劳动条件变更、规章制度、解除、经济补偿 | 处理违法调岗、违法解除、程序瑕疵 |
| 《中华人民共和国就业促进法》 | 平等就业、反就业歧视 | 处理招聘录用、晋升、机会差别对待 |
| 《中华人民共和国妇女权益保障法》 | 性别歧视、婚育歧视、孕产保护、反性骚扰 | 支持孕产歧视和女性职场权益保护 |
| 《女职工劳动保护特别规定》 | 孕期、产期、哺乳期劳动保护 | 支持孕产节点的特殊保护主张 |
| 《中华人民共和国劳动争议调解仲裁法》 | 劳动争议处理程序、时效、提交路径 | 明确仲裁与证据准备框架 |
| 《中华人民共和国残疾人保障法》 | 残障相关平等就业与保护 | 适用于残障歧视等问题 |
| 《中华人民共和国治安管理处罚法》及相关刑事规则 | 威胁、骚扰、暴力、猥亵等严重行为 | 涉及人身安全时的行政或刑事路径 |

## 三、按争议类型映射法规

## 3.1 公开羞辱、持续贬低、人格化攻击
可重点关注：
- 《中华人民共和国民法典》中的人格权保护逻辑
- 公司内部行为准则、反骚扰或反霸凌制度

重点并非“对方脾气差”，而是：
- 是否构成持续侮辱
- 是否造成明显人格或健康损害
- 公司是否有义务采取保护措施

## 3.2 孕产歧视与婚育偏见
可重点关注：
- 《中华人民共和国妇女权益保障法》
- 《中华人民共和国就业促进法》
- 《女职工劳动保护特别规定》

关键判断点：
- 是否因怀孕、产假、哺乳、婚育计划而受不利待遇
- 是否存在晋升、录用、薪酬、项目机会差别对待
- 是否将歧视包装成“业务需要”

## 3.3 报复性低绩效、报复性调岗
可重点关注：
- 《中华人民共和国劳动合同法》
- 《中华人民共和国劳动法》
- 《中华人民共和国劳动争议调解仲裁法》

关键判断点：
- 绩效与调岗是否具有事实基础与程序基础
- 是否与申诉、举报或拒绝不当要求存在时间关联
- 是否因此造成薪酬、岗位、机会等实质损害

## 3.4 违法调岗与劝退协商
可重点关注：
- 《中华人民共和国劳动合同法》关于劳动合同履行、变更、解除的相关规则
- 劳动争议程序规则

关键判断点：
- 是否协商一致
- 是否属于合理调配还是实质不利变更
- 是否通过高压协商规避单位责任

## 四、内部制度同样重要

在很多案件中，企业内部规则同样具有重要意义，包括：
- 员工手册
- 行为准则
- 反骚扰/反歧视制度
- 绩效管理制度
- 调岗审批制度
- 举报与申诉保护机制

如果公司内部制度已经明确反霸凌、反报复或申诉保护，那么这些制度本身就是重要依据。

## 五、证据与法规的对应关系

| 证据类型 | 可证明的问题 | 常见对应规则 |
| --- | --- | --- |
| 邮件/聊天记录 | 原话、口径变化、申诉与报复时间点 | 人格权、劳动条件、内部制度 |
| 会议纪要/录音 | 公开羞辱、施压、劝退、调岗沟通 | 人格权、程序合法性 |
| 劳动合同/岗位说明 | 原岗位与约定职责 | 劳动合同变更争议 |
| 绩效记录/周报 | 实际表现与程序过程 | 绩效处理合法性 |
| 调岗通知/离职协议 | 实质变更和协商内容 | 调岗、解除、补偿争议 |
| 医疗/心理记录 | 身心损害与功能影响 | 健康权、损害后果 |

## 六、维权路径简表

### 6.1 内部路径
- 直属管理沟通
- 隔级沟通
- HR/员工关系
- 工会/申诉委员会/合规热线

### 6.2 外部路径
- 劳动保障监察/人社投诉
- 劳动争议仲裁
- 民事诉讼
- 公安报案（涉及威胁、暴力、猥亵等）

## 七、常见误区提醒

- 误以为没有“PUA专门法条”就无法处理
- 只强调情绪伤害，不说明劳动权益损害
- 法规罗列很多，但诉求和证据不对应
- 只依赖口头投诉，不同步固定书面材料

## 八、法规使用建议

更实用的表达方式不是“我要告你PUA”，而是：
- 存在持续人格化羞辱与侮辱行为
- 存在因婚育/孕产产生的不合理差别待遇
- 存在缺乏事实和程序依据的报复性绩效/调岗
- 存在高压劝退、违法解除或不当协商风险
