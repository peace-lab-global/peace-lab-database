# PIP周报/复盘模板 (PIP Weekly Report & Retrospective Templates)

> **定位**：提供适用于绩效改进计划执行期的周报、周会纪要、中期复盘和结案复盘模板
> **适用场景**：进入PIP后，需要持续向直属上级、HR、业务负责人同步目标进展、障碍、支持请求和阶段性结论
> **使用原则**：事实先行、指标清晰、动作具体、口径一致、全程留痕

## 一、填写说明

建议所有模板统一保留以下字段：
- 员工姓名：________
- 部门/岗位：________
- 直属上级：________
- PIP周期：自____年__月__日至____年__月__日
- 当前周次：第____周 / 共____周
- 文档日期：____年__月__日

书写要求：
- 只写可验证事实，不写情绪化判断
- 所有指标尽量量化，如完成率、时点、交付件、质量分
- 凡是依赖资源、跨部门支持或口头变更，均单独列示
- 每次复盘都要写清下一步动作和责任人

## 六、PIP状态红黄绿标记模板

```text
当前状态：绿灯 / 黄灯 / 红灯
判断依据：
- 绿灯：目标明确、按周推进、支持到位、结论一致
- 黄灯：部分目标模糊、存在资源卡点或优先级变化
- 红灯：标准频繁变化、关键支持缺失、存在明显不利留痕风险

本周状态：________
主要原因：________
建议动作：________
```

## 八、常见误区提醒

- 只写“我很努力”，不写可核验结果
- 只写问题，不写原因和改进动作
- 对资源/标准变化不单独留痕
- 会上讲清了，会后不发纪要
- 周报内容与口头表达前后不一致
