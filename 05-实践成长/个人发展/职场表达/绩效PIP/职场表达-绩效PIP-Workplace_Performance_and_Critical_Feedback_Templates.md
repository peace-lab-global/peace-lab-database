# 一对一绩效/批评谈话模板 (1:1 Performance & Critical Feedback Templates)

> **定位**：用于绩效回顾、一对一批评、问题纠偏、改进要求和会后确认的实用谈话模板
> **核心目标**：把容易伤人的评价对话，转化为围绕事实、影响、标准和下一步的建设性对话

## 一、适用场景

- 直属上级与你做绩效回顾
- 你需要接受一对一批评或纠偏
- 你作为管理者，要向下属做负向反馈
- 某项结果不达标，需要单独拉齐标准
- 你需要在谈话中保护边界，避免人身化表达

## 二、谈话前准备模板

在进入一对一谈话前，建议先准备以下四项：
- **事实**：发生了什么
- **影响**：造成什么后果
- **标准**：原先要求是什么
- **动作**：接下来怎么改

**准备清单**
- 这次谈话要解决的唯一核心问题是什么？
- 有哪些客观事实，而不是主观评价？
- 我希望对方会后改变什么行为？
- 这次谈话的结论如何留痕？

## 三、你作为“接受反馈的一方”时

### 3.1 开场模板
- "我希望先听您最核心的反馈点。"
- "为了便于我调整，我们能否先聚焦最重要的一项问题？"

### 3.2 澄清模板
- "您说的问题，具体是指哪次场景或哪类行为？"
- "我想确认一下，您期待的标准具体是什么？"
- "这次反馈里，您最希望我优先改的是哪一点？"

### 3.3 接受但不自我贬低
- "这个反馈我收到，我会把问题拆开处理。"
- "如果问题在结果和动作层面，我愿意承担并调整。"

### 3.4 防止反馈变成人格评价
- "如果是工作结果问题，我希望我们聚焦具体表现和改进要求。"
- "我接受针对任务的反馈，但希望避免过于笼统的人格化评价。"

### 3.5 收尾模板
- "我复述一下我接下来的改进动作，您看是否准确。"
- "我会在________前完成调整，并把结果同步给您。"

## 四、你作为“给出反馈的一方”时

### 4.1 温和开场版
```text
我想和你单独聊一下最近的一项工作表现，目的是帮助我们把结果拉回标准，而不是否定你个人。
我先说具体事实，再说影响和后续期望。
```

### 4.2 标准反馈版
```text
在________这个任务上，出现了________情况。
它带来的影响是________。
原先的标准/预期是________。
接下来我希望你在________前完成________。
如果你需要支持，我们现在可以一起确认。
```

### 4.3 较为严肃但不羞辱版
```text
这次结果没有达到应有标准，我需要直接指出来。
问题主要不在态度，而在________。
接下来我们不讨论泛泛的感受，先把标准、动作和检查点定下来。
```

## 五、绩效回顾场景模板

### 5.1 上级对下属做绩效回顾
```text
过去这个周期，你做得好的部分有：________。
明显需要提升的部分有：________。
对团队和结果的影响是：________。
下一阶段我希望你重点提升：________。
我们会在________做一次复盘。
```

### 5.2 你向上级回应绩效反馈
```text
谢谢您的反馈，我先确认一下重点。
我理解您认可的是________，需要我改进的是________。
我计划接下来通过________来改进，并在________前给您阶段性结果。
如果有偏差，请您现在纠正我。
```

## 六、困难情境模板

### 6.1 对方上来就是批评，没有具体事实
- "为了让我真正改进，您能否举一个最典型的具体例子？"
- "如果要让我优先调整，我想先对齐一个最关键场景。"

### 6.2 对方开始人格化评价
- "如果是工作问题，我希望我们聚焦行为、结果和标准。"
- "我愿意讨论改进，但如果评价过于笼统，我会不容易抓到重点。"

### 6.3 你作为管理者情绪上来了
- "我先停一下，避免表达方式影响效果。我们先回到事实。"
- "我想解决问题，不想把对话做成人身化。"

### 6.4 对方情绪很强
- "我理解这段反馈不容易听，我们先只讨论一件最重要的事。"
- "如果你现在需要一点时间消化，我们可以先把结论和下一步定下来。"

## 七、会后确认模板

### 7.1 你是员工，发给上级
```text
根据今天一对一沟通，我理解当前需要我重点改进的事项有：
1. ________
2. ________
3. ________

我会在________前完成第一轮调整，并于________向您同步进展。
如我理解有偏差，请您指正。
```

### 7.2 你是管理者，发给下属
```text
根据今天谈话，我们确认如下：
1. 当前问题：________
2. 目标标准：________
3. 改进动作：________
4. 完成时间：________
5. 下次复盘时间：________

如果需要支持，请在________前提出。
```

## 八、三个底线提醒

- 批评行为，不贴人格标签
- 说标准和动作，不说含糊判断
- 给改进路径，不只给压力
