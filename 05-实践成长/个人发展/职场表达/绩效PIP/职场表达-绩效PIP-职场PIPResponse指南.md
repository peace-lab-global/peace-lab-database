# 绩效改进计划(PIP)应对指南 (Performance Improvement Plan Response Guide)

> **定位**：帮助在被纳入PIP、被暗示“主动离开”、或被用绩效工具施压时，稳住节奏、识别风险、保护证据并进行专业沟通
> **核心目标**：先确认规则，再确认标准；先保留证据，再谈结论；先做书面留痕，再决定去留
> **重要提醒**：以下内容以中国大陆常见劳动法实务为主，不构成法律意见；请结合劳动合同、员工手册、绩效制度和当地最新法规使用

## 一、先明确：PIP不是法律概念，而是企业管理工具

PIP本身不是法定程序，而是公司常见的绩效改进机制。它可能有三种真实用途：
- **真改进型**：希望你在明确标准、资源支持和复盘机制下改善表现
- **观察淘汰型**：通过一段观察期，为后续低绩效处理准备材料
- **谈判施压型**：用PIP制造心理压力，促使你主动离职或接受低补偿方案

所以，面对PIP，不要先问“我是不是完了”，而要先问：
1. 规则依据是什么？
2. 标准是否清晰、可衡量？
3. 公司是否给足了资源、培训和岗位支持？
4. 结论会如何形成、由谁确认、以什么书面形式留痕？

## 二、PIP流程全面解析

## 2.1 常见流程图

### 阶段一：进入前信号
常见信号包括：
- 绩效评语突然变差，但之前没有充分预警
- 上级频繁放大单点失误，却不谈整体产出
- 你被排除在关键项目、会议和资源之外
- 开始出现“你要不要考虑别的机会”“你最近状态不太适合团队”等话术

### 阶段二：正式启动会议
通常会出现以下动作：
- 上级或HR通知你进入PIP
- 提供一份PIP文件或口头说明改进目标
- 约定持续时间，常见为30/60/90天
- 指定评估人、复盘频率和结果出口

### 阶段三：执行期
这段时间最关键：
- 每周/双周是否有固定review
- 标准有没有被临时改写
- 是否有资源支持、培训或岗位调整
- 你是否持续用书面方式确认目标、进展和障碍

### 阶段四：中期复盘
中期复盘应回答四个问题：
- 哪些目标已完成？
- 哪些目标未完成？原因是什么？
- 是否存在资源、依赖或标准问题？
- 后半程的重点动作是什么？

### 阶段五：结案
常见结局：
- 通过PIP，回到正常绩效周期
- 延长PIP，继续观察
- 调岗/培训后继续观察
- 公司主张你“不能胜任工作”，准备解除或协商离开

## 2.2 你要特别盯住的五个关键字段

拿到PIP后，重点看：
- **起止时间**：从哪天开始，到哪天结束
- **目标指标**：是否具体、可衡量、可验证
- **支持条件**：培训、资源、协作人、权限是否明确
- **复盘机制**：多久review一次，谁参加，谁定结论
- **结案标准**：达到什么程度算通过，未通过会怎样

如果以上五项不清楚，PIP就很容易变成“模糊标准 + 事后追责”。

## 三、收到PIP后的第一时间动作

## 3.1 当场不要做三件事

不要：
- 立刻情绪化争辩
- 在未读清文件前仓促签字
- 口头承认“我确实不胜任”“那我考虑离职”

## 3.2 第一时间做六件事

### 第一步：要书面版本
可用话术：
- "为了确保我准确理解改进要求，请把PIP目标、周期和评估方式发我书面版本。"
- "我会认真配合，但希望先基于正式文件理解标准。"

### 第二步：确认评价依据
要问清：
- 本次PIP依据的是哪份绩效制度、岗位说明或目标约定？
- 是哪一个考核周期的数据支持当前结论？
- 是否有前置反馈、辅导或提醒记录？

### 第三步：拆解目标
把每个目标改写成：
- 具体任务
- 衡量口径
- 截止时间
- 所需资源
- 依赖方

### 第四步：要求固定review节奏
推荐：
- 每周一次进展review
- 每次会后都发确认邮件/纪要
- 把“我做了什么、还差什么、缺什么支持”写清楚

### 第五步：建立个人证据包
建议单独整理：
- 劳动合同、岗位说明、OKR/KPI、绩效记录
- 历史表扬、项目结果、客户反馈
- PIP文件、会议纪要、聊天记录、邮件
- 资源受限、目标变化、临时加码的书面证据

### 第六步：同步判断自己目标
你要尽早判断自己处于哪种策略：
- **策略A：争取通过PIP，留在当前岗位**
- **策略B：边完成PIP边准备外部机会**
- **策略C：围绕补偿、离职日期和背调口径做谈判**

## 四、常见陷阱与应对策略

## 4.1 陷阱一：目标模糊，永远无法达标

典型表现：
- "提升主人翁意识"
- "增强协作感"
- "让团队更信任你"

应对方式：
- 要求把抽象要求改成行为标准和结果标准
- 让对方举例说明“达标”长什么样

话术模板：
- "为了便于执行，我希望把‘提升协作’进一步明确成几个具体行为或结果指标。"
- "您期待我在未来30天内具体做到哪三件事，才算达到标准？"

## 4.2 陷阱二：只给压力，不给资源

典型表现：
- 目标加码，但权限、协作人、时间不变
- 上级要求改善，却不愿澄清优先级

应对方式：
- 把资源、范围、时间放在同一张表里确认
- 明确哪些结果依赖跨部门支持

话术模板：
- "如果目标维持不变，我需要同步确认资源支持和跨部门接口，否则执行风险会很高。"
- "我可以承诺先完成A和B；如果同时保留C，建议补充________支持。"

## 4.3 陷阱三：口头改标准，事后追责

典型表现：
- 开会说一套，结论文件写另一套
- 上级临时改优先级，最后按旧标准评价你

应对方式：
- 每次会后立即发书面确认
- 让对方纠正，而不是靠自己记忆对抗

会后确认模板：
```text
根据今天的PIP沟通，我理解当前改进重点为：
1. ________
2. ________
3. ________

对应时间点分别为：________。
如我的理解有偏差，请您直接修改，我会按确认后的版本执行。
```

## 4.4 陷阱四：PIP与“主动离职”捆绑

典型表现：
- "你自己走会更体面"
- "如果你现在提离职，后面大家都省事"
- "你不一定非要经历完整PIP"

应对方式：
- 不要被带入“马上表态”
- 先问正式方案、法律口径、补偿安排

话术模板：
- "我先按正式PIP要求推进，关于去留问题，我希望基于完整书面方案再做判断。"
- "如果公司有协商方案，请书面说明，我会统一评估。"

## 4.5 陷阱五：把历史问题一次性堆到你头上

典型表现：
- 过去未明确指出的问题，在PIP时集中爆发
- 把团队系统性问题都记到你个人绩效里

应对方式：
- 区分“我可控的部分”和“外部依赖/组织问题”
- 对历史记录逐项核对，不默认接受笼统归责

## 4.6 陷阱六：在PIP期间诱导你做不利表述

高风险表述包括：
- "我确实不适合这个岗位"
- "我接受公司任何安排"
- "我愿意主动离职"

更稳妥的表达：
- "我愿意配合改进，但需要基于明确标准来执行。"
- "我会先按流程完成PIP，并以书面记录为准。"

## 五、关键时间节点把控

## 5.1 Day 0：收到通知当天
你要完成：
- 记录会议时间、参会人和原话
- 索要正式文件
- 不仓促签署不利文本
- 把自己的初步理解发成邮件/消息留痕

## 5.2 24小时内
你要完成：
- 把PIP目标拆成任务清单
- 整理历史绩效证据和项目成果
- 识别哪些指标不清楚、哪些资源不足
- 预约第一次正式review

## 5.3 第3-5天
你要完成：
- 发出一版书面执行计划
- 明确每周交付节奏
- 开始同步保留每周进度、阻碍和支持请求

## 5.4 每周review后
必须做：
- 用邮件/企业微信发纪要
- 写明已完成、未完成、原因、下周动作
- 明确哪些问题因资源/优先级变更产生

## 5.5 中期复盘时
重点动作：
- 要求中期结论书面化
- 如果标准被改、资源被撤、范围被放大，马上记录并提出异议
- 评估自己是否需要同步推进转岗/求职/法律咨询

## 5.6 结案前
重点动作：
- 索要最终评估依据
- 不接受只有口头结论、没有书面说明
- 如果出现解除/协商离职，转入补偿与证据策略，不要混在同一次情绪性沟通里处理

## 六、沟通谈判技巧与实用话术

## 6.1 启动会议中的稳态回应
```text
我理解公司希望我在这个周期内完成改进，我愿意配合。
为了确保执行有效，我希望先把目标、衡量标准、支持资源和review节奏明确下来。
我会基于书面确认后的版本来推进。
```

## 6.2 要求把抽象评价改成具体标准
```text
我希望把“沟通不足”“执行力不够”这类评价进一步落到具体场景。
请您帮我确认：哪几类行为需要改变、对应标准是什么、以什么结果作为完成依据？
```

## 6.3 争取资源支持的谈法
```text
对于当前目标，我可以承担执行责任。
但若希望在PIP周期内达标，我需要同步确认以下支持：
1. ________
2. ________
3. ________
否则我担心会把资源问题误判成个人绩效问题。
```

## 6.4 中期复盘时的主动表达
```text
截至目前，我已完成________。
未完成部分主要受________影响。
我希望今天把后半程最关键的两项优先级定清楚，并确认当前标准是否有调整。
```

## 6.5 结案前要求明确结论
```text
为了避免理解偏差，我希望最终评估能以书面形式确认：
1. 已达成项；
2. 未达成项；
3. 依据材料；
4. 后续安排。
这样我可以准确理解下一步。
```

## 6.6 对“你不如自己走”的回应
```text
我理解您在讨论后续安排。
但关于离职或协商方案，我希望先收到完整书面信息，再统一评估。
在此之前，我会继续按当前PIP要求推进工作。
```

## 七、常见误区提醒

- **误区一：把PIP等同于必然辞退**
  - 并非所有PIP都会走向解除，但你必须按“可能升级为劳动争议”来做证据准备
- **误区二：只顾解释，不做留痕**
  - 真正有价值的不是现场说得多漂亮，而是会后写得多清楚
- **误区三：一上来就全面对抗**
  - 先拿规则、标准、资源和证据，再决定是争取通过、争取转岗还是谈补偿
- **误区四：轻易签“自愿离职”**
  - 一旦签成个人主动离职，后续补偿和争议空间可能显著缩小
- **误区五：只盯工资，不看背调口径和离职证明**
  - 如果PIP后转入离职谈判，补偿、离职日期、离职原因、证明措辞都要一起谈

## 八、法律风险防范要点

## 8.1 先看公司制度是否真实存在且已公示
PIP的程序合法性，通常离不开：
- 员工手册或绩效制度是否存在
- 规则是否经过合理制定、公示或告知
- 岗位标准、考核口径是否事先可知

## 8.2 “不能胜任工作”通常不能只靠一句评价成立
在中国大陆常见劳动法实务中，若公司主张员工“不能胜任工作”，通常更看重：
- 是否有明确、客观、稳定的岗位标准
- 是否有持续、可核验的绩效证据
- 是否做过培训、辅导或调岗安排
- 是否给过合理改进期，而非临时拼凑材料

## 8.3 警惕三类高风险文件
不要轻易签：
- 自愿离职申请
- 认定自己不胜任工作的承诺书
- 未看清内容的协商解除协议或无争议承诺

## 8.4 证据保存要合法
建议保留：
- 与本人劳动关系和绩效相关的合法记录
- 邮件、聊天、纪要、绩效表、岗位说明等原始材料

避免：
- 擅自带走商业机密、客户隐私、源代码等受保护信息
- 通过侵入账户、非法窃听等方式取证

## 8.5 出现这些情况，建议尽快咨询律师或劳动专业人士
- 公司以PIP为名长期施压，但不给标准和资源
- 要求你先签自愿离职再谈补偿
- PIP结案后直接解除，但未说明依据
- 涉及降薪、调岗、停职、停发奖金、背调风险等连带问题

## 九、相关法律法规依据（中国大陆常见规则）

- **《中华人民共和国劳动合同法》**
  - 涉及劳动报酬、岗位调整、协商解除、经济补偿、规章制度适用、解除条件等
- **《中华人民共和国劳动法》**
  - 涉及工资支付、工作时间、休息休假、劳动保护等基础规则
- **《中华人民共和国劳动争议调解仲裁法》**
  - 涉及劳动争议处理路径、仲裁时效、证据提交与程序衔接
- **公司劳动合同、岗位说明书、员工手册、绩效管理制度**
  - 是判断PIP流程是否一致、标准是否明确的重要内部依据
