# 职场维权申诉邮件与仲裁请求模板 (Complaint Email & Arbitration Request Templates)

> **定位**：提供适用于职场霸凌、歧视、骚扰、报复性管理等场景的正式申诉邮件模板、情况说明模板和劳动仲裁请求示例
> **适用场景**：向HR、员工关系、合规部门、隔级领导提交正式申诉；就相关劳动争议准备仲裁申请材料框架
> **重要提醒**：以下模板为通用示例，不构成法律意见；正式仲裁请求需根据个案事实、证据和当地最新规则调整

- 已掌握证据目录：________

## 二、正式申诉邮件模板

## 2.1 向HR/员工关系部门提交申诉邮件

```text
主题：关于________事项的正式申诉与处理请求

________您好：
我系________部门________岗位员工，现就近期发生的________事项，提交正式申诉并请求公司启动相应处理流程。

一、基本情况
1. 涉及人员：________；
2. 事件发生时间：________；
3. 事件发生场景：________；
4. 主要问题性质：职场霸凌 / 歧视 / 羞辱性管理 / 报复性处理 / 其他________。

二、事实陈述
1. 自____年__月__日起，出现以下情况：
   - ________；
   - ________；
   - ________。
2. 上述行为对我造成的影响包括：
   - 工作执行影响：________；
   - 身心状态影响：________；
   - 职业评价/团队协作影响：________。

三、已掌握材料
1. 邮件/聊天记录：________；
2. 会议纪要/录音/截图：________；
3. 绩效/调岗/处分记录：________；
4. 其他相关证据：________。

四、已采取的内部沟通
我已于________通过________方式进行沟通/反馈，结果为________。
如仍未改善或已出现进一步不利后果，具体表现为________。

五、处理请求
基于以上情况，我正式请求：
1. 对相关事实启动调查或访谈；
2. 要求停止相关不当行为，并明确后续沟通边界；
3. 视情况评估汇报关系、工作接口或保护性安排；
4. 对处理过程及结论给予书面反馈。

六、法律与制度依据（可按实际情况选择）
- 《中华人民共和国劳动合同法》关于用人单位劳动保护、规章制度与劳动条件管理的相关规定；
- 《中华人民共和国劳动法》关于劳动保护及劳动者合法权益保障的相关规定；
- 《中华人民共和国就业促进法》关于平等就业、反就业歧视的相关原则；
- 《中华人民共和国妇女权益保障法》关于招聘录用、晋升、薪酬、孕产及反性骚扰保护的相关规定；
- 公司员工手册、行为准则、反骚扰/反歧视制度等内部规则。

如需我进一步补充时间线、附件材料或配合访谈，请直接告知。

此致
敬礼！

姓名：________
工号：________
联系方式：________
日期：________
```

## 四、劳动仲裁请求文件示例（通用示意版）

> 说明：以下为结构示意，具体仲裁请求必须与实际争议类型一致，可根据是否涉及解除、降薪、调岗、歧视性待遇、报复性绩效等情形调整。

```text
劳动仲裁申请书

申请人：________，性别：________，身份证号：________，住址：________，联系电话：________。
被申请人：________公司，住所地：________，法定代表人：________，联系电话：________。

仲裁请求：
1. 请求裁决被申请人________；
2. 请求裁决被申请人________；
3. 请求裁决被申请人承担本案仲裁相关责任/费用（如适用，按当地规则填写）；
4. 其他请求：________。

事实与理由：
申请人于____年__月__日入职被申请人，担任________岗位，双方签订劳动合同，约定________。
在劳动关系存续期间，自____年__月__日起，被申请人/相关管理人员存在________行为，具体包括：
1. ________；
2. ________；
3. ________。

申请人曾于____年__月__日通过________方式向公司内部进行反馈/申诉，但被申请人未予合理处理/后续出现报复性措施，具体表现为________。
上述行为导致申请人遭受________后果，并形成以下争议：________。

申请人认为：
1. 根据《中华人民共和国劳动合同法》相关规定，被申请人应依法保障劳动者合法权益，不得违法调整劳动条件、违法解除劳动关系或以不当管理方式侵害劳动者权益；
2. 如本案涉及平等就业、婚育歧视、性别差别待遇等问题，可结合《中华人民共和国就业促进法》《中华人民共和国妇女权益保障法》等相关规定主张；
3. 如本案同时涉及侮辱、人格权侵害或性骚扰问题，申请人保留依据《中华人民共和国民法典》等规定另行主张权利的权利。

为维护申请人合法权益，特依法申请仲裁，请予支持。

此致
________劳动人事争议仲裁委员会

申请人：________
日期：____年__月__日
```

## 五、常见仲裁请求表达示例

> 以下仅为写法示例，需按个案事实选择，避免机械套用。

### 5.1 涉及违法调岗/降薪时

```text
1. 请求确认被申请人于____年__月__日作出的调岗/降薪决定违法；
2. 请求裁决被申请人恢复申请人原岗位/按原薪酬标准支付劳动报酬；
3. 请求裁决被申请人补足自____年__月起少发的工资差额共计________元。
```

### 5.2 涉及报复性低绩效处理时

```text
1. 请求确认被申请人作出的________绩效处理缺乏事实及程序依据；
2. 请求裁决被申请人撤销由该绩效处理所引发的相关不利后果；
3. 请求裁决被申请人支付因________造成的工资/奖金差额共计________元。
```

### 5.3 涉及解除劳动关系争议时

```text
1. 请求确认被申请人于____年__月__日解除劳动关系的行为违法；
2. 请求裁决被申请人支付违法解除劳动合同赔偿金________元；
3. 请求裁决被申请人出具合法、准确的离职证明并办理相关手续。
```

## 六、证据目录模板

```text
证据目录

证据一：劳动合同
证明目的：证明双方存在劳动关系及岗位约定。

证据二：工资流水/个税记录
证明目的：证明工资标准及实际支付情况。

证据三：邮件/聊天记录
证明目的：证明相关霸凌、歧视、申诉或不利处理事实。

证据四：会议纪要/录音/截图
证明目的：证明具体事件经过、在场人员及口径变化。

证据五：绩效记录/调岗通知/处分通知
证明目的：证明后续不利处理与争议事实之间的关联。

证据六：申诉邮件及回复
证明目的：证明申请人已进行内部反馈以及被申请人的处理情况。
```

## 七、常见误区提醒

- 情绪性控诉很多，但事实时间线不完整
- 法律依据堆很多，但与诉求不对应
- 仲裁请求写得过泛，无法执行
- 缺少证据目录，导致材料杂乱
- 申诉邮件里只表达痛苦，不写具体处理请求
