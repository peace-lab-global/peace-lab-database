# 乱伦临床工具箱 (Incest Clinical Toolkit)

> 🛠️ **实用工具集**: 本工具箱提供乱伦创伤临床工作中必需的评估量表、治疗模板、安全协议和危机干预工具，帮助专业人员高效开展临床服务。

## 核心评估工具

### 1. 综合创伤评估量表

#### 乱伦创伤症状评估表 (ITSA-25)
```
评估维度：创伤重现、回避行为、过度警觉、情感麻木、身体症状、人际关系、自我认知

评分标准：0=从未，1=很少，2=有时，3=经常，4=总是

关键项目示例：
1. 我会突然想起创伤经历（0-4分）
2. 我避免与创伤相关的地点或活动（0-4分）
3. 我容易受到惊吓或感到紧张（0-4分）
4. 我对曾经喜欢的活动失去兴趣（0-4分）
5. 我感到与他人疏远或隔离（0-4分）

总分解释：
0-25分：轻微症状
26-50分：中度症状
51-75分：重度症状
76-100分：极重度症状
```

#### 家庭功能评估工具 (FFA-15)
```
评估家庭系统健康状况，包括：
- 沟通模式
- 边界清晰度
- 角色分配
- 情感表达
- 问题解决能力

评估方法：结构化访谈 + 家庭绘图技术
```

### 2. 安全风险评估矩阵

#### 儿童安全风险评估表
| 风险因子 | 低风险(1分) | 中风险(2分) | 高风险(3分) | 极高风险(4分) |
|---------|------------|------------|------------|--------------|
| **即时安全** | 环境安全，有保护者 | 环境基本安全 | 环境存在隐患 | 立即危险 |
| **家庭支持** | 家庭功能良好 | 家庭功能一般 | 家庭功能受损 | 家庭功能严重障碍 |
| **创伤严重度** | 单次轻微接触 | 反复轻度接触 | 严重或持续接触 | 极端暴力接触 |
| **心理症状** | 轻微焦虑 | 中度症状 | 严重症状 | 危机状态 |
| **社会资源** | 资源充足 | 资源一般 | 资源有限 | 缺乏支持 |

**风险等级判定：**
- 4-8分：低风险，门诊治疗
- 9-12分：中风险，加强监控
- 13-16分：高风险，住院评估
- 17-20分：极高风险，紧急干预

```

□ 移除危险物品
□ 安排安全陪伴
□ 联系紧急服务

□ 使用接地技术
□ 教授深呼吸方法
□ 提供情感支持

第三步：评估风险(1小时内)
□ 自杀风险评估

□ 功能损害评估

□ 短期安全措施
□ 支持系统激活
□ 后续治疗安排
```
