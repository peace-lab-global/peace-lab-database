## 二、🔴 P0 — 镜像歼灭战 (`MIRROR-CLEAN`)

### 2.1 任务 P0-01：镜像文件扫描与清单生成

| 属性 | 内容 |
|:-----|:-----|
| **目标** | 生成 231 组镜像文件的完整清单，标注已分歧/未分歧状态 |
| **输入** | `02-Mind-Psychology/` 与 `06-Clinical-Topics/` 全文件比对 |
| **方法** | 文件大小 + 首 20 行哈希 + 末 20 行哈希快速比对 |
| **输出** | `Tools/plans/mirror-cleanup-inventory.md` |
| **验收** | 清单包含：源路径、镜像路径、文件大小、最后修改日期、分歧状态 |
| **工时** | 2h |
| **依赖** | 无 |

```bash
# 快速扫描脚本思路
find 02-Mind-Psychology -name '*.md' -exec md5sum {} + > 02-hashes.txt
find 06-Clinical-Topics -name '*.md' -exec md5sum {} + > 06-hashes.txt
# 交叉比对生成清单
```

### 2.2 任务 P0-02：已分歧文件裁决（96 组）

| 属性 | 内容 |
|:-----|:-----|
| **目标** | 对 96 组已分歧文件裁定保留版本 |
| **裁决原则** | 1. 取最新修改日期版本 2. 取行数更多版本 3. 02 源文件优先于 06 副本 4. 临床聚合内容保留在 06 |
| **输出** | `Tools/plans/mirror-divergence-rulings.md`（每组的裁决 + 理由） |
| **验收** | 全部 96 组有明确裁决，无一遗漏 |
| **工时** | 4h |
| **依赖** | P0-01 |

### 2.3 任务 P0-03：未分歧镜像删除 + symlink 替换

| 属性 | 内容 |
|:-----|:-----|
| **目标** | 删除 135 组未分歧的 06 物理副本，改为 symlink 指向 02 源文件 |
| **方法** | `ln -s` 创建符号链接，保留 06 的文件路径以维持外部引用 |
| **风险** | Windows 环境 symlink 支持有限；需测试 Git 对 symlink 的处理 |
| **输出** | 06 目录下 135 个 `.md` 文件变为 symlink |
| **验收** | `readlink` 验证全部 symlink 指向正确；`git status` 确认变更可追踪 |
| **工时** | 3h |
| **依赖** | P0-02 |

### 2.4 任务 P0-04：已分歧镜像合并与同步

| 属性 | 内容 |
|:-----|:-----|
| **目标** | 按 P0-02 裁决，将保留版本同步到 02 和 06（或删除 06 副本） |
| **方法** | 对裁决保留 02 版的：用 02 版覆盖 06 版 → 转 symlink；对裁决保留 06 版的：用 06 版覆盖 02 版 → 转 symlink |
| **输出** | 96 组分歧文件合并为统一版本 |
| **验收** | `diff` 验证 02 和 symlink 指向的文件内容完全一致 |
| **工时** | 6h |
| **依赖** | P0-03 |

- [ ] 分歧裁决无遗漏（P0-02）
- [ ] 未分歧镜像全部 symlink 化（P0-03）
- [ ] 已分歧镜像全部合并同步（P0-04）

## 七、风险与应对

| 风险 | 概率 | 影响 | 应对策略 |
|:-----|:----:|:----:|:---------|
| Windows 环境 symlink 不支持 | 中 | 高 | 方案 B：不用 symlink，06 保留 INDEX 链接 + 删除物理副本 |

| IFS/SE 内容医学准确性 | 中 | 高 | 人工审核所有临床内容；标注"基于现有文献整理，不替代专业培训" |

| 新空壳在修复期间累积 | 中 | 低 | 执行期间暂停新增模块，聚焦修复；P2-01 机制上线后恢复 |
