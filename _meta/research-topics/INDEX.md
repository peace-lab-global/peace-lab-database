# research-topics — 跨领域课题研究

> **板块定位**：跨五大支柱与临床专题的**前沿交叉研究平台**，以课题为导向，打通学科壁垒，构建面向未来的知识生产体系。
> **适用对象**：跨学科研究者、博士生、独立研究者、AI 智能体研究者、知识策展人。

---

## 设计理念

本板块不同于 01-06 板块的**知识编目**逻辑，采用**课题驱动（Research-Topic-Driven）**模式：

| 维度 | 01-06 板块 | research-topics |
|:-----|:-----------|:-------------------|
| 组织逻辑 | 学科分类（Discipline） | 研究课题（Research Topic） |
| 知识方向 | 编目 → 学习 | 交叉 → 生产 |
| 产出形态 | 综述、索引、指南 | 研究框架、假说、实验设计、文献综合 |
| 跨域深度 | 单支柱为主，交叉引用 | 多支柱深度整合 |
| 评估标准 | 完整性、准确性 | 创新性、可验证性、跨域解释力 |

```
01-智慧传统 ──┐

05-实践成长     ──┤

```

research-topics 是**跨板块的研究聚合与创新层**：
- 每个课题深度调用 01-06 板块的既有知识
- 产出的研究框架反哺 01-06 板块的内容更新
- 为 `_元信息/学习路径/` 和 `_元信息/topic-maps/` 提供前沿课题输入

## 课题研究模板

每个课题建议包含以下结构：

```
课题名/
├── INDEX.md              # 课题总览与文献地图
├── research-questions.md # 核心研究问题与假说
├── literature-review.md  # 文献综述（跨学科）
├── methodology.md        # 研究方法论
├── evidence-synthesis.md # 证据综合与发现
└── future-directions.md  # 未来方向与开放问题
```
