### 《原子习惯》(Atomic Habits) - James Clear

#### 作者背景

| 项目 | 详情 |
| :--- | :--- |
| **作者** | James Clear |
| **背景** | 行为心理学专家、演讲家、习惯研究者 |
| **个人经历** | 高中时被棒球棒击中头部重伤，通过微小习惯恢复并成为优秀运动员 |
| **网站** | jamesclear.com，订阅用户超过100万 |
| **成就** | 全球销量超过1500万册，被译成50多种语言 |

#### 核心内容

| 主题 | 核心观点 | 应用方法 |
| :--- | :--- | :--- |
| **1%进步复利** | 每天进步1%，一年后提升37倍 | 关注系统而非目标 |
| **身份层改变** | 结果→过程→身份三层，身份最深 | "我是[某种人]"而非"我想[做某事]" |
| **四定律** | 显而易见、有吸引力、简单易行、令人满足 | 正向应用建立好习惯，反向打破坏习惯 |
| **习惯堆叠** | 在[当前习惯]后，我将[新习惯] | 利用已有神经通路 |
| **环境设计** | 环境比意志力更重要 | 让好选择显而易见，坏选择隐而不见 |

#### 名言金句

| 金句 | 主题 |
| :--- | :--- |
| "你不会上升到目标的高度，而是下降到系统的水平。" | 系统vs目标 |
| "每次行动都是对你想成为的人的投票。" | 身份改变 |
| "习惯不限制自由，它创造自由。" | 习惯的价值 |
| "改变习惯最有效的方法是改变身份。" | 身份优先 |

### 《深度工作》(Deep Work) - Cal Newport

#### 作者背景

| 项目 | 详情 |
| :--- | :--- |
| **作者** | Cal Newport |
| **背景** | 乔治城大学计算机科学教授、畅销书作家 |
| **研究领域** | 分布式算法理论、数字极简主义 |
| **特点** | 从不使用社交媒体，践行自己所写 |
| **其他著作** | 《数字极简主义》《优秀到不能被忽视》 |

#### 核心内容

| 主题 | 核心观点 | 应用方法 |
| :--- | :--- | :--- |
| **深度工作定义** | 无干扰状态下的高认知要求专业活动 | 创造高价值、稀缺且难以复制 |
| **浅层工作** | 低认知要求、常在干扰中进行的后勤任务 | 批量处理、限制时间 |
| **注意力残留** | 任务切换后前任务仍占认知资源 | 完成任务或仪式性结束 |
| **四种深度哲学** | 修道式、双模式、节律式、记者式 | 根据职业选择适合的模式 |
| **深度工作量** | 深度工作时长 × 专注强度 | 追踪深度小时数 |

#### 名言金句

| 金句 | 主题 |
| :--- | :--- |
| "深度工作是21世纪的超级力量。" | 深度工作价值 |
| "如果你不生产，你就不会成功。" | 产出导向 |
| "人们在浅薄的工作上忙碌，以逃避深入思考。" | 忙碌陷阱 |
| "最大化深度的量，最小化浅层的量。" | 工作分配 |

### 《活出意义来》(Man's Search for Meaning) - Viktor Frankl

#### 作者背景

| 项目 | 详情 |
| :--- | :--- |
| **作者** | Viktor E. Frankl (1905-1997) |
| **背景** | 奥地利神经学家、精神病学家 |
| **经历** | 纳粹集中营幸存者，失去父母、妻子和兄弟 |
| **贡献** | 创立意义疗法(Logotherapy) |
| **影响** | 全球销量超过1600万册，被译成50多种语言 |

#### 核心内容

| 主题 | 核心观点 | 应用方法 |
| :--- | :--- | :--- |
| **意志的意义** | 人类最主要的动机是寻找意义 | 超越快乐原则和权力原则 |
| **三种发现意义的途径** | 创造(工作)、体验(爱/美)、态度(苦难中的选择) | 即使无法改变境遇，仍可选择态度 |
| **悲剧乐观主义** | 在悲剧中找到意义与成长 | 苦难→成就、罪疚→改变、死亡→行动 |
| **去反射技术** | 过度关注问题会强化问题 | 将注意力从症状转向目标 |
| **自我超越** | 真正的自我实现来自对他人的贡献 | 意义在自我之外 |

#### 名言金句

| 金句 | 主题 |
| :--- | :--- |
| "那些有理由活下去的人，几乎能承受任何方式的生活。" (引用尼采) | 意义的力量 |
| "在刺激与反应之间，人有选择的自由。" | 选择的自由 |
| "苦难不再是苦难的那一刻，是我们找到它的意义的那一刻。" | 苦难与意义 |
| "不要追问生命的意义，而要回答生命对你的提问。" | 责任 |
