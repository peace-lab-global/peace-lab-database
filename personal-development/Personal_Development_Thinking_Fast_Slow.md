## 核心内容与观点 (Core Content & Perspectives)

### 两个思维系统

| 维度 | 系统1 (快) | 系统2 (慢) |
| :--- | :--- | :--- |
| **速度** | 快速、自动 | 缓慢、刻意 |
| **努力程度** | 不费力 | 费力 |
| **控制** | 不由自主 | 有意识控制 |
| **典型活动** | 面孔识别、简单计算、情绪反应 | 复杂计算、逻辑推理、专注任务 |
| **优势** | 效率高、处理日常决策 | 能够处理复杂问题 |
| **劣势** | 容易犯系统性错误 | 耗费认知资源、容易懒惰 |
| **隐喻** | 自动驾驶 | 手动驾驶 |

### 主要认知偏差

| 偏差 | 描述 | 示例 | 应对策略 |
| :--- | :--- | :--- | :--- |
| **锚定效应** | 最初接触的信息过度影响判断 | 谈判中的首次出价 | 意识到锚点，主动设定锚点 |
| **可得性启发** | 容易想到的事件被认为更常见 | 飞机失事后害怕坐飞机 | 查看统计数据 |
| **代表性启发** | 根据相似性而非概率判断 | 忽视基准率 | 考虑基准率 |
| **损失厌恶** | 损失的痛苦大于等量收益的快乐 | 不愿卖出亏损股票 | 从整体组合角度思考 |
| **框架效应** | 同一信息的不同表述影响决策 | "90%存活率"vs"10%死亡率" | 重新框架问题 |
| **确认偏误** | 倾向于寻找支持自己观点的信息 | 只看支持自己立场的新闻 | 主动寻找反面证据 |
| **过度自信** | 高估自己的知识和预测能力 | 创业者高估成功率 | 使用预测基准率 |
| **后见之明** | 事后觉得结果是可预测的 | "我早就知道会这样" | 记录预测并回顾 |
| **规划谬误** | 低估完成任务所需时间和资源 | 项目超时超预算 | 使用外部视角和参考类别 |

### 前景理论 (Prospect Theory)

| 概念 | 描述 | 传统经济学观点 | 行为经济学发现 |
| :--- | :--- | :--- | :--- |
| **参考点** | 结果相对于参考点评估 | 绝对财富水平 | 相对于现状的变化 |
| **损失厌恶** | 损失的权重大于收益 | 损失和收益等价 | 损失痛苦是收益快乐的2倍 |
| **敏感性递减** | 边际变化的影响递减 | 线性效用 | S型价值曲线 |
| **概率加权** | 小概率被高估，大概率被低估 | 客观概率 | 主观概率权重 |

### 两个自我

| 自我 | 定义 | 关注点 | 测量方式 |
| :--- | :--- | :--- | :--- |
| **体验自我** | 当下体验快乐或痛苦的自我 | 每一刻的感受 | 实时情绪采样 |
| **记忆自我** | 事后回忆和评价经历的自我 | 整体印象 | 回顾性评价 |

### 峰终定律 (Peak-End Rule)

| 概念 | 描述 | 示例 | 应用 |
| :--- | :--- | :--- | :--- |
| **峰终定律** | 对体验的记忆主要取决于峰值和结束时的感受 | 结肠镜检查实验 | 优化体验的高峰和结尾 |
| **持续时间忽视** | 体验的持续时间对记忆影响小 | 长时间的平淡不如短时间的精彩 | 质量重于数量 |

### WYSIATI原则

| 概念 | 描述 | 影响 | 应对 |
| :--- | :--- | :--- | :--- |
| **What You See Is All There Is** | 系统1只根据现有信息构建故事 | 忽视缺失信息、过度自信 | 主动问"我还不知道什么？" |

## 实践建议 (Practical Recommendations)

### 改善决策的策略

| 策略 | 描述 | 操作方法 |
| :--- | :--- | :--- |
| **预先验尸** | 假设项目失败，分析原因 | 在开始前问"一年后这失败了，为什么？" |
| **外部视角** | 使用类似情况的统计数据 | 查找参考类别的基准率 |
| **延迟决策** | 给系统2参与的时间 | 重大决策前睡一晚 |
| **检查清单** | 系统性地考虑各种因素 | 使用决策检查清单 |
| **考虑相反观点** | 主动寻找反面证据 | 问"如果我错了呢？" |
| **分散评估** | 分开评估不同选项 | 避免单独评估 |

### 认知偏差自检清单

| 偏差 | 自检问题 |
| :--- | :--- |
| **锚定效应** | 我是否受到了最初信息的过度影响？ |
| **可得性偏差** | 我是否因为容易想到就认为更可能发生？ |
| **过度自信** | 我对自己预测的准确度有多大把握？历史上我的预测准确率如何？ |
| **确认偏误** | 我是否只在寻找支持我观点的信息？ |
| **损失厌恶** | 我是否因为害怕损失而做出非理性决策？ |
| **框架效应** | 用不同方式表述这个问题，我的选择会改变吗？ |

### 与其他实践的整合

| 整合领域 | 结合方式 | 增强效果 |
| :--- | :--- | :--- |
| **深度工作** | 深度工作激活系统2 | 做出更理性的决策 |
| **正念冥想** | 正念增加对系统1的觉察 | 识别自动反应 |
| **习惯系统** | 用习惯将好决策自动化 | 减少决策疲劳 |
