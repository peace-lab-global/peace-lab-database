## 核心内容与观点 (Core Content & Perspectives)

### 幸福公式

```
H = S + C + V

H = 持久幸福水平
S = 设定范围（遗传决定的幸福基线，约50%）
C = 生活环境（约10%）
V = 自愿控制的因素（约40%）
```

### 幸福的三个层面

| 层面 | 英文 | 描述 | 持久性 |
| :--- | :--- | :--- | :--- |
| **愉悦的生活** | Pleasant Life | 追求积极情绪和感官享受 | 短暂、容易适应 |
| **投入的生活** | Engaged Life | 运用优势进入心流状态 | 较持久 |
| **有意义的生活** | Meaningful Life | 服务于超越自我的目标 | 最持久 |

### 24种品格优势 (VIA分类)

| 美德类别 | 品格优势 | 描述 |
| :--- | :--- | :--- |
| **智慧与知识** | 创造力、好奇心、开放思维、热爱学习、洞察力 | 获取和运用知识 |
| **勇气** | 勇敢、坚持、正直、热情 | 面对困难追求目标 |
| **人道** | 爱、善良、社会智慧 | 关爱他人 |
| **正义** | 团队精神、公平、领导力 | 健康的社区生活 |
| **节制** | 宽恕、谦虚、谨慎、自律 | 防止过度 |
| **超越** | 审美、感恩、希望、幽默、灵性 | 与宇宙的联系 |

### 乐观解释风格

| 维度 | 悲观解释 | 乐观解释 |
| :--- | :--- | :--- |
| **持久性** | "这会永远持续" | "这是暂时的" |
| **普遍性** | "这会影响所有事情" | "这只影响这件事" |
| **个人化** | "都是我的错" | "有多种因素" |

### 积极情绪的扩展-建构理论

| 概念 | 描述 | 效果 |
| :--- | :--- | :--- |
| **扩展** | 积极情绪扩展思维和行动范围 | 更有创造力、更开放 |
| **建构** | 扩展带来持久资源的建设 | 身体、智力、社会、心理资源 |
| **螺旋上升** | 积极情绪带来更多积极情绪 | 良性循环 |

### 幸福干预策略

| 策略 | 操作 | 效果 |
| :--- | :--- | :--- |
| **三件好事** | 每晚写下今天发生的三件好事及原因 | 增加幸福感、减少抑郁 |
| **感恩拜访** | 写信感谢曾帮助过你的人并亲自送达 | 大幅提升幸福感 |
| **运用优势** | 每天用新方式运用你的首要优势 | 持续提升幸福 |
| **品味** | 有意识地品味积极体验 | 延长积极情绪 |
| **积极回应** | 积极热情地回应他人的好消息 | 增强关系 |

### 习得性乐观

| 步骤 | 描述 | 示例 |
| :--- | :--- | :--- |
| **识别信念** | 识别消极自动思维 | "我总是失败" |
| **挑战信念** | 寻找反面证据 | "我上次成功了" |
| **替换信念** | 建立更平衡的信念 | "有时成功有时失败是正常的" |
| **能量化** | 用乐观信念增加能量 | 采取行动 |
