## 核心内容与观点 (Core Content & Perspectives)

### 汉堡模型：四种人生原型

| 类型 | 隐喻 | 当下体验 | 未来收益 | 特征 |
| :--- | :--- | :--- | :--- | :--- |
| **享乐主义型** | 垃圾汉堡 | 好吃 | 有害 | 追求即时满足，忽视未来 |
| **忙碌奔波型** | 素食汉堡 | 难吃 | 健康 | 为未来牺牲当下，延迟满足 |
| **虚无主义型** | 最糟糕汉堡 | 难吃 | 有害 | 对当下和未来都绝望 |
| **感悟幸福型** | 理想汉堡 | 好吃 | 健康 | 当下愉悦且有益未来 |

### 幸福的定义

```
幸福 = 愉悦 + 意义
     (当下的积极体验) + (对未来有目的感)
```

### 幸福与成功的关系

| 传统观点 | 研究发现 | 启示 |
| :--- | :--- | :--- |
| 成功→幸福 | 幸福→成功 | 先追求幸福，成功随之而来 |
| 外在成就带来幸福 | 幸福带来外在成就 | 幸福是因，成功是果 |
| "到达谬误" | 到达后幸福感短暂 | 享受旅程，而非只看终点 |

### 设定幸福目标

| 自我和谐目标特征 | 描述 | 检验问题 |
| :--- | :--- | :--- |
| **内在动机** | 因为想做而非应该做 | "这是我真正想要的吗？" |
| **意义感** | 与价值观一致 | "这对我有意义吗？" |
| **愉悦感** | 追求过程也令人愉悦 | "我享受追求的过程吗？" |

### 快乐提升器

| 策略 | 描述 | 频率 |
| :--- | :--- | :--- |
| **表达感恩** | 写下感恩的人和事 | 每天或每周 |
| **记录快乐时刻** | 记录让你快乐的日常瞬间 | 每天 |
| **简化生活** | 减少承诺，专注于重要的事 | 持续 |
| **运动** | 定期身体锻炼 | 每周3-4次 |
| **冥想** | 正念练习 | 每天 |
| **社交联系** | 与亲朋好友相处 | 经常 |

### 工作中的幸福

| 概念 | 描述 | 应用 |
| :--- | :--- | :--- |
| **MPS过程** | 意义(Meaning)+愉悦(Pleasure)+优势(Strengths) | 找到三者交集的工作 |
| **工作塑造** | 调整工作方式以增加意义和愉悦 | 在现有工作中寻找或创造意义 |
| **心流体验** | 在工作中进入沉浸状态 | 调整挑战与技能的匹配 |

### 关系中的幸福

| 原则 | 描述 | 实践 |
| :--- | :--- | :--- |
| **核心自我** | 了解和接受真实的自己 | 分享真实的想法和感受 |
| **无条件积极关注** | 爱TA本身而非TA的行为 | 接受而非试图改变对方 |
| **被知道** | 渴望被真正理解 | 深度对话、自我暴露 |
| **积极回应** | 积极热情地回应好消息 | 主动庆祝他人的成功 |

### 幸福的冥想

| 类型 | 描述 | 效果 |
| :--- | :--- | :--- |
| **感恩冥想** | 专注于感恩的人和事 | 增强积极情绪 |
| **慈悲冥想** | 发送善意给自己和他人 | 增强联结感 |
| **正念冥想** | 专注于当下体验 | 增强觉察和接纳 |
