## 核心内容与观点 (Core Content & Perspectives)

### 福格行为模型 (B=MAP)

| 要素 | 定义 | 问题 | 当要素不足时 |
| :--- | :--- | :--- | :--- |
| **B (Behavior)** | 行为本身 | 你想要什么行为发生？ | 行为不会发生 |
| **M (Motivation)** | 做行为的动力 | 你想要做这件事吗？ | 即使简单也不做 |
| **A (Ability)** | 执行行为的能力 | 你能做这件事吗？ | 即使想做也做不到 |
| **P (Prompt)** | 触发行为的信号 | 什么提醒你去做？ | 想做也能做但忘了做 |

### 行为设计三步骤

| 步骤 | 描述 | 操作方法 | 示例 |
| :--- | :--- | :--- | :--- |
| **1. 明确愿望** | 找出你想实现的愿望 | 问自己"我想要什么？" | "我想更健康" |
| **2. 探索行为选项** | 列出能实现愿望的所有行为 | 头脑风暴行为云 | 跑步、健身、走路、游泳... |
| **3. 匹配黄金行为** | 找出既有效又可行的行为 | 使用焦点映射法 | 每天步行10分钟 |

### 焦点映射法

| 维度 | 高影响力 | 低影响力 |
| :--- | :--- | :--- |
| **高可行性** | 黄金行为 (优先选择) | 可做但效果有限 |
| **低可行性** | 有效但难以持续 | 不值得做 |

### 微习惯设计配方

| 组成部分 | 描述 | 示例 |
| :--- | :--- | :--- |
| **锚点 (Anchor)** | 已有的习惯或事件 | "刷牙后..." |
| **微行为 (Tiny Behavior)** | 极小的新行为 | "...我会用牙线清理一颗牙" |
| **庆祝 (Celebration)** | 即时的积极情感 | "...然后对自己说'太棒了！'" |

### 完整配方公式

```
在我 [锚点行为] 之后，我会 [微行为]，然后我会 [庆祝]。
```

### 锚点类型

| 锚点类型 | 描述 | 示例 |
| :--- | :--- | :--- |
| **晨间锚点** | 早晨固定发生的事 | 闹钟响、起床、刷牙 |
| **常规锚点** | 日常固定行为 | 吃饭、喝水、开会 |
| **珍珠锚点** | 新习惯变成下一个习惯的锚点 | 习惯链 |

### 庆祝的力量

| 概念 | 描述 | 为何重要 | 示例 |
| :--- | :--- | :--- | :--- |
| **闪耀时刻 (Shine)** | 完成行为后的积极情感 | 情感编码帮助行为进入大脑 | 微笑、击掌、说"Yes!" |
| **即时性** | 庆祝必须紧随行为之后 | 大脑需要即时反馈 | 完成后立即庆祝 |
| **真实感受** | 必须是真正的积极情感 | 假装的庆祝无效 | 找到让你真正开心的方式 |

### 能力链 (Ability Chain)

| 环节 | 问题 | 优化方向 |
| :--- | :--- | :--- |
| **时间** | 你有时间做吗？ | 缩短行为时间 |
| **金钱** | 你负担得起吗？ | 降低成本 |
| **体力** | 你有体力吗？ | 减少体力需求 |
| **脑力** | 你需要多少思考？ | 简化决策 |
| **常规** | 是否与现有习惯冲突？ | 融入现有流程 |

### 打破坏习惯的方法

| 策略 | 操作 | 对应B=MAP | 示例 |
| :--- | :--- | :--- | :--- |
| **减少动机** | 让坏习惯变得不那么有吸引力 | 降低M | 将垃圾食品联想到负面后果 |
| **增加难度** | 让坏习惯变得更难执行 | 降低A | 将手机放在另一个房间 |
| **移除提示** | 消除触发坏习惯的信号 | 移除P | 删除社交媒体App |

## 实践建议 (Practical Recommendations)

### Tiny Habits配方模板

| 锚点行为 | 微行为 | 庆祝方式 |
| :--- | :--- | :--- |
| 刷完牙后 | 用牙线清理一颗牙 | 对镜微笑 |
| 坐到办公桌前 | 打开工作文档 | 说"开工啦！" |
| 喝完早晨第一杯水后 | 做一个深呼吸 | 握拳说"Yes!" |
| 躺到床上后 | 读一句书 | 闭眼微笑 |
| 午饭后 | 走50步 | 心中说"太棒了" |

### 庆祝方式清单

| 类型 | 示例 |
| :--- | :--- |
| **身体动作** | 击掌、握拳、跳一下、微笑 |
| **声音** | "Yes!"、"太棒了！"、哼小曲 |
| **想象** | 想象观众鼓掌、想象自己站在领奖台 |
| **内心感受** | 深吸一口气、感受骄傲感 |
