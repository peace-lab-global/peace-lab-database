## Carol Dweck 成长型思维理论 (Growth Mindset Theory)

### 固定型vs成长型思维对比

| 维度 | 固定型思维 (Fixed Mindset) | 成长型思维 (Growth Mindset) |
| :--- | :--- | :--- |
| **核心信念** | 智力和才能是固定的天赋 | 智力和才能可通过努力发展 |
| **对努力的看法** | 努力说明能力不足 | 努力是通往精通的路径 |
| **面对挑战** | 回避，因为失败会暴露能力不足 | 欢迎，因为挑战是成长机会 |
| **面对挫折** | 放弃，证明自己不够聪明 | 坚持，从错误中学习 |
| **面对批评** | 防御或忽视 | 从中学习改进 |
| **面对他人成功** | 感到威胁 | 受到激励和学习 |
| **自我评价** | 不断证明自己的价值 | 关注学习与进步 |
| **目标导向** | 表现目标（看起来聪明） | 学习目标（变得更好） |
| **神经基础** | 对错误反应较少关注 | 错误后脑部活动增强 |

### 思维模式在不同情境的表现

| 情境 | 固定型思维反应 | 成长型思维反应 |
| :--- | :--- | :--- |
| **考试失败** | "我就是不够聪明" | "我需要改变学习方法" |
| **新技能学习** | "我没有天赋，算了" | "开始时困难是正常的，继续练习" |
| **收到批评** | "他们在否定我" | "这是改进的机会" |
| **同事升职** | "为什么不是我？不公平" | "我可以学习他/她的成功之处" |
| **复杂项目** | "太难了，我做不到" | "我还不会，但我可以学" |
| **犯了错误** | "我真糟糕" | "错误是学习的一部分" |

### 培养成长型思维的策略

| 策略 | 英文 | 操作方法 | 科学依据 |
| :--- | :--- | :--- | :--- |

| **重构挑战** | Reframing Challenges | 将困难视为成长机会而非威胁 | 压力重评研究 |
| **学习目标** | Learning Goals | 设定学习提升目标而非表现目标 | 目标导向理论 |
| **神经可塑性教育** | Teaching Neuroplasticity | 了解大脑可改变的科学事实 | 干预研究 |
| **错误正常化** | Normalizing Mistakes | 将错误视为学习过程的必要部分 | 创造心理安全 |
| **努力归因** | Effort Attribution | 将成功归因于努力而非天赋 | 归因理论 |
| **自我慈悲** | Self-Compassion | 失败时善待自己，减少自我批评 | Neff研究 |

## 心智模型概览 (Mental Models Overview)

### 什么是心智模型

| 属性 | 描述 |
| :--- | :--- |
| **定义** | 对现实运作方式的简化内部表征 |
| **功能** | 帮助理解世界、预测结果、做出决策 |
| **来源** | 经验、教育、文化、进化 |
| **局限** | 简化必然意味着不完整，可能导致盲点 |
| **可塑性** | 可通过学习和经验更新与扩展 |

### 关键心智模型分类

| 类别 | 来源领域 | 代表模型 | 核心洞见 |
| :--- | :--- | :--- | :--- |
| **系统思维** | 系统论 | 反馈循环、二阶效应、涌现 | 世界是相互连接的复杂系统 |
| **概率思维** | 统计学 | 贝叶斯更新、回归均值、样本偏差 | 在不确定性中做出更好判断 |
| **经济思维** | 经济学 | 机会成本、边际效用、激励 | 理解人类行为的经济逻辑 |
| **进化思维** | 生物学 | 自然选择、适应、竞争与合作 | 理解变化与适应 |
| **物理思维** | 物理学 | 惯性、杠杆、临界质量 | 理解基本的因果关系 |
| **心理思维** | 心理学 | 认知偏差、框架效应、锚定 | 理解人类心理的系统性偏差 |
| **哲学思维** | 哲学 | 第一性原理、奥卡姆剃刀 | 清晰思考的基础工具 |

## 核心心智模型详解 (Key Mental Models Detailed)

### 系统思维模型

| 模型 | 英文 | 定义 | 应用示例 |
| :--- | :--- | :--- | :--- |
| **反馈循环** | Feedback Loops | 系统输出返回影响输入 | 正：病毒传播；负：恒温器 |
| **二阶效应** | Second-Order Effects | 行动的间接后果 | 政策的意外副作用 |
| **涌现** | Emergence | 整体大于部分之和 | 蚁群智能、城市文化 |
| **临界点** | Tipping Points | 小变化引发大转变的阈值 | 病毒爆发、社会运动 |
| **系统思维** | Systems Thinking | 看到整体而非孤立部分 | 组织变革、生态问题 |
| **瓶颈** | Bottlenecks | 限制整体产出的约束点 | 生产流程、项目管理 |

### 概率与决策模型

| 模型 | 英文 | 定义 | 应用示例 |
| :--- | :--- | :--- | :--- |
| **贝叶斯思维** | Bayesian Thinking | 根据新证据更新信念概率 | 医学诊断、投资决策 |
| **期望值** | Expected Value | 概率加权的平均结果 | 赌博、投资、决策 |
| **回归均值** | Regression to Mean | 极端表现后趋向平均 | 运动员表现、治疗效果 |
| **基础概率** | Base Rates | 在具体信息前考虑一般概率 | 避免代表性偏差 |
| **确认偏差** | Confirmation Bias | 只寻找支持现有信念的证据 | 研究、投资决策 |
| **可得性偏差** | Availability Heuristic | 易想起的事物被高估概率 | 恐惧飞行、媒体影响 |

### 经济与激励模型

| 模型 | 英文 | 定义 | 应用示例 |
| :--- | :--- | :--- | :--- |
| **机会成本** | Opportunity Cost | 选择的真实成本是放弃的最佳替代 | 时间分配、投资选择 |
| **边际效用递减** | Diminishing Marginal Utility | 额外单位带来的满足递减 | 消费决策、工作加班 |

| **沉没成本** | Sunk Cost | 已发生不可收回的成本 | 避免沉没成本谬误 |
| **比较优势** | Comparative Advantage | 专注于相对优势领域 | 分工、外包决策 |
| **博弈论** | Game Theory | 策略互动中的最优决策 | 谈判、竞争策略 |

### 思维清晰度模型

| :--- | :--- | :--- | :--- |

## 认知偏差图谱 (Cognitive Bias Atlas)

### 决策与判断偏差

| 偏差 | 英文 | 定义 | 去偏策略 |
| :--- | :--- | :--- | :--- |
| **确认偏差** | Confirmation Bias | 寻找支持现有信念的信息 | 主动寻找反证、红队思维 |
| **后见之明** | Hindsight Bias | 事后觉得结果是可预见的 | 事前记录预测、概率思维 |
| **过度自信** | Overconfidence | 高估自己的知识和能力 | 校准练习、寻求反馈 |
| **计划谬误** | Planning Fallacy | 低估任务时间和难度 | 参考基础概率、外部视角 |
| **禀赋效应** | Endowment Effect | 高估自己拥有的东西 | 考虑机会成本 |
| **现状偏见** | Status Quo Bias | 偏好保持现状 | 强制考虑替代方案 |
| **峰终定律** | Peak-End Rule | 以峰值和结束时刻评价体验 | 关注整体体验 |

| 偏差 | 英文 | 定义 | 去偏策略 |
| :--- | :--- | :--- | :--- |
| **基本归因错误** | Fundamental Attribution Error | 高估性格、低估情境因素 | 考虑情境约束 |
| **自利归因** | Self-Serving Bias | 将成功归因于自己，失败归因于外部 | 平衡归因、自我反思 |

## 决策框架 (Decision-Making Frameworks)

### 理性决策流程

| 步骤 | 操作 | 常见错误 | 优化方法 |
| :--- | :--- | :--- | :--- |
| **1. 定义问题** | 明确真正需要解决的问题 | 解决错误问题、症状而非根因 | 5个为什么、问题重构 |
| **2. 设定目标** | 明确决策标准和优先级 | 目标模糊、标准冲突 | SMART目标、权重排序 |
| **3. 生成选项** | 产生可能的解决方案 | 选项过少、过早收敛 | 头脑风暴、逆向思考 |
| **4. 评估选项** | 根据标准评估各选项 | 确认偏差、忽视风险 | 决策矩阵、红队思考 |
| **5. 选择决策** | 选择最佳选项 | 分析瘫痪、直觉冲动 | 设定截止日期、承诺机制 |
| **6. 执行行动** | 实施决策 | 执行不力、缺乏跟进 | 行动计划、问责机制 |
| **7. 评估反馈** | 评估结果，学习改进 | 忽视反馈、后见之明 | 决策日志、事后复盘 |

### 逆向思维框架 (Inversion)

| 步骤 | 操作 | 示例 |
| :--- | :--- | :--- |
| **1. 明确目标** | 定义你想要达成的结果 | "我想要一个成功的项目" |
| **2. 反向思考** | 问"如何确保失败？" | "如何确保项目失败？" |
| **3. 列出失败因素** | 列出所有导致失败的因素 | 沟通不畅、资源不足、目标模糊... |
| **4. 避免这些因素** | 将避免失败转化为行动 | 建立沟通机制、确保资源、明确目标 |

### 费曼学习法 (Feynman Technique)

| 步骤 | 操作 | 目的 |
| :--- | :--- | :--- |
| **1. 选择概念** | 选择要学习/理解的概念 | 明确学习目标 |
| **2. 教给小孩** | 用简单语言解释给想象中的孩子 | 检验真正理解程度 |
| **3. 识别差距** | 发现无法简单解释的部分 | 找到理解的薄弱点 |
| **4. 回去学习** | 回到资料，填补知识差距 | 深化理解 |
| **5. 简化类比** | 用更简单的语言和类比重新解释 | 巩固并内化知识 |

| **孩子失败** | "没关系，不是每个人都擅长数学" | "这说明你需要尝试不同的方法" |
| **孩子遇到困难** | "这对你太难了" | "这是有挑战性的，你还在学习" |
| **比较他人** | "你比他/她聪明" | "每个人都在按自己的节奏成长" |
| **赞美** | 赞美天赋和结果 | 赞美努力、策略和进步 |

### 心理治疗整合

| 治疗取向 | 与成长型思维的整合 | 应用 |
| :--- | :--- | :--- |
| **认知行为治疗** | 挑战固定思维的认知扭曲 | 重构"我做不到"为"我还在学习" |
| **接纳承诺疗法** | 接纳当前能力，承诺成长行动 | 价值导向的技能发展 |
| **正念疗法** | 觉察固定思维模式，不认同 | 观察而非相信限制性信念 |
| **叙事疗法** | 重写能力叙事 | 从"我是失败者"到"我是学习者" |
| **解决聚焦疗法** | 关注小进步与例外 | 发现已有的成长证据 |
