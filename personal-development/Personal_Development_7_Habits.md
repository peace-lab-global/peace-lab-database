### 成熟连续体

### 七个习惯详解

#### 习惯一：主动积极 (Be Proactive)

| 维度 | 被动消极 | 主动积极 |
| :--- | :--- | :--- |
| **语言** | "我没办法"、"他让我..." | "我选择"、"我可以..." |
| **关注点** | 关注圈（无法控制的事） | 影响圈（能够影响的事） |
| **责任感** | 外界决定我的生活 | 我选择我的反应 |
| **核心** | 刺激→反应 | 刺激→选择→反应 |

#### 习惯二：以终为始 (Begin with the End in Mind)

| 概念 | 描述 | 操作 |
| :--- | :--- | :--- |
| **两次创造原则** | 所有事物先在脑中创造，再实际创造 | 先规划后行动 |
| **个人使命宣言** | 明确人生目标和价值观 | 写下你的人生宪法 |
| **葬礼练习** | 想象在自己的葬礼上希望别人怎么评价你 | 以此为生活指南 |
| **角色与目标** | 明确各个角色的目标 | 平衡各领域 |

#### 习惯三：要事第一 (Put First Things First)

| 象限 | 特征 | 示例 | 建议 |
| :--- | :--- | :--- | :--- |
| **Ⅰ 重要且紧急** | 危机、问题、截止日期 | 紧急项目、客户投诉 | 处理但减少 |
| **Ⅱ 重要但不紧急** | 预防、规划、关系建设 | 学习、运动、战略规划 | 增加时间 |
| **Ⅲ 紧急但不重要** | 打断、一些电话、邮件 | 某些会议、他人的紧急事务 | 减少或委托 |
| **Ⅳ 不重要不紧急** | 琐事、消遣 | 刷手机、无聊活动 | 避免 |

#### 习惯四：双赢思维 (Think Win-Win)

| 模式 | 描述 | 何时适用 |
| :--- | :--- | :--- |
| **赢/输** | 我赢你输 | 体育比赛等竞争环境 |
| **输/赢** | 我让步你赢 | 讨好型人格，长期有害 |
| **输/输** | 两败俱伤 | 报复心态，应避免 |
| **赢** | 只关注自己赢 | 独立任务 |
| **双赢** | 寻求互利方案 | 大多数人际关系 |
| **双赢或不成交** | 找不到双赢就放弃交易 | 重要关系中 |

#### 习惯五：知彼解己 (Seek First to Understand, Then to Be Understood)

| 倾听层次 | 描述 | 效果 |
| :--- | :--- | :--- |
| **忽视** | 根本不听 | 破坏关系 |
| **假装倾听** | "嗯，是的" | 虚伪 |
| **选择性倾听** | 只听部分 | 误解 |
| **专注倾听** | 集中注意力在话语上 | 表面理解 |
| **同理心倾听** | 理解对方的感受和意图 | 深层理解和信任 |

#### 习惯六：统合综效 (Synergize)

| 概念 | 描述 | 结果 |
| :--- | :--- | :--- |
| **1+1>2** | 整体大于部分之和 | 创造性合作 |
| **第三选择** | 不是你的方案也不是我的，而是更好的新方案 | 突破性解决方案 |
| **差异的价值** | 差异是机会而非威胁 | 互补与创新 |

#### 习惯七：不断更新 (Sharpen the Saw)

| 维度 | 描述 | 更新活动 |
| :--- | :--- | :--- |
| **身体** | 身体健康和精力 | 运动、营养、休息 |
| **心智** | 持续学习和成长 | 阅读、写作、学习新技能 |
| **社会/情感** | 人际关系和服务 | 服务他人、建立信任关系 |
| **精神** | 价值观和目的 | 冥想、阅读经典、亲近自然 |

### 情感账户

| 概念 | 描述 | 存款行为 | 取款行为 |
| :--- | :--- | :--- | :--- |
| **情感账户** | 人际关系中的信任储蓄 | 理解、守信、期望清晰、善意、道歉 | 不理解、失信、期望模糊、恶意、傲慢 |
