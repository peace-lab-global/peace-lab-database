# 南传上座部佛教概览 (Theravada Buddhism Overview)

> **上座部佛教 (Theravada Buddhism)**：佛教最古老的存续传承，以巴利三藏为经典依据，强调原始佛陀教法的忠实传承与阿罗汉道的解脱实践。
> 
> *"诸比丘！当以自己为洲岛，以自己为皈依处；当以法为洲岛，以法为皈依处，不以其他为皈依处。" —— 《大般涅槃经》*
> 
> *巴利语：Theravāda (थेरवाद) / "长老之道" / 梵语：Sthaviravāda (स्थविरवाद)*
> 
> 上座部佛教是现存最接近原始佛教的传承，主要分布于斯里兰卡、缅甸、泰国、柬埔寨、老挝等南亚与东南亚国家，以戒定慧三学为核心，以证得阿罗汉果为修行目标。
