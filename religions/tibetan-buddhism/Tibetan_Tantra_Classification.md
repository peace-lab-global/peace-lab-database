# 密续分类体系 (Tantra Classification System)

> **密续 (Tantra)**：金刚乘佛教的核心经典与修行体系，通过本尊瑜伽、真言持诵、坛城观想等方便法门，实现即身成佛的快速解脱道。
> 
> *"密续者，相续不断之义，以方便善巧令心相续转化为觉悟。" —— 《密续总义》*
> 
> *梵语：Tantra (तन्त्र) / "相续、织布机" / 藏语：རྒྱུད (Gyü) / "续"*
> 
> 藏传佛教将密续分为四部(格鲁派)或六部(宁玛派)，形成了从事续到无上瑜伽续的完整分类体系，每一层次都有其特定的见地、修法与成就。

### 表1.2 密续与显教比较 (Tantra vs. Sūtra)

| 比较维度 | 显教 (Sūtra) | 密教 (Tantra) |
|---|---|---|

| **方便** | 六度万行 | 三密相应 |
| **转化对象** | 压制烦恼 | 转化烦恼为智慧 |

| **五毒** | 对治断除 | 转化为五智 |
| **传授方式** | 公开说法 | 需要灌顶传承 |
| **修行条件** | 相对宽松 | 严格戒律与誓言 |
| **主要经典** | 经(Sūtra) | 续(Tantra) |

## 2. 新译派四续分类 (Four Tantra Classes - New Translation)

### 表2.1 四续总览 (Overview of Four Tantras)

| 续部 | 梵语 | 藏语 | 英文 | 特征 | 代表本尊 | 修行重点 |
|---|---|---|---|---|---|---|
| **事续** | *Kriyātantra* | བྱ་རྒྱུད (Ja Gyü) | Action Tantra | 外在仪轨为主 | 观音、度母 | 净化、仪式 |
| **行续** | *Caryātantra* | སྤྱོད་རྒྱུད (Chö Gyü) | Performance Tantra | 内外平衡 | 大日如来 | 平衡修持 |
| **瑜伽续** | *Yogatantra* | རྣལ་འབྱོར་རྒྱུད (Naljor Gyü) | Yoga Tantra | 内在瑜伽为主 | 金刚界如来 | 观想为主 |
| **无上瑜伽续** | *Anuttarayogatantra* | བླ་ན་མེད་པའི་རྣལ་འབྱོར་རྒྱུད | Highest Yoga Tantra | 生圆二次第 | 胜乐、密集、大威德 | 即身成佛 |

### 表2.2 事续 (Kriyātantra)

| 项目 | 内容 |
|---|---|
| **定义** | 以外在净化行为与仪轨为主的密续 |
| **特征** | 强调沐浴、持斋、清净、供养等外在行为 |
| **见地** | 本尊如主人，行者如仆人 |
| **修法** | 持咒、供养、礼拜、洁净仪轨 |
| **代表本尊** | 观音菩萨、绿度母、白度母、文殊菩萨 |
| **代表密续** | 《大悲心陀罗尼经》《度母续》 |
| **适合根器** | 重视外在清净者 |
| **成就时间** | 传统说十六生 |

| 项目 | 内容 |
|---|---|

| **特征** | 外在仪轨与内在瑜伽并重 |
| **见地** | 本尊如朋友，平等相待 |
| **修法** | 仪轨与观想并行 |
| **代表本尊** | 大日如来 (毗卢遮那佛) |
| **代表密续** | 《大日经》(毗卢遮那成佛神变加持经) |
| **适合根器** | 内外兼修者 |
| **成就时间** | 传统说七生 |

### 表2.4 瑜伽续 (Yogatantra)

| 项目 | 内容 |
|---|---|
| **定义** | 以内在瑜伽观想为主的密续 |
| **特征** | 强调内在三摩地、本尊瑜伽 |
| **见地** | 行者与本尊无二 |
| **修法** | 四印(大印、法印、羯磨印、三昧耶印) |
| **代表本尊** | 金刚界如来、金刚萨埵 |
| **代表密续** | 《金刚顶经》《真实摄经》 |
| **适合根器** | 重视内在观修者 |
| **成就时间** | 传统说三生 |

### 表2.5 无上瑜伽续 (Anuttarayogatantra)

| 项目 | 内容 |
|---|---|
| **定义** | 最高层次的密续，生圆二次第完整 |
| **特征** | 转化身心，即身成佛 |
| **见地** | 轮涅不二，烦恼即菩提 |
| **修法** | 生起次第、圆满次第、气脉明点 |
| **三分类** | 父续(方便续)、母续(智慧续)、无二续 |
| **代表本尊** | 密集金刚、胜乐金刚、大威德金刚、喜金刚、时轮金刚 |
| **适合根器** | 上根利器者 |
| **成就时间** | 即身成佛 |
