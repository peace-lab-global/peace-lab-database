## 研究设计方法学 (Research Design Methodology)

### 实验研究设计

#### 随机对照试验(RCT)

##### 经典RCT设计

```
标准RCT结构：

实验设计要素：
- 随机分组：计算机生成随机序列
- 对照组设置：等待列表对照/活性对照
- 盲法实施：单盲/双盲/三盲设计
- 样本量计算：基于效应量的统计power

核心测量时间点：
基线评估(T0)：治疗前全面评估
中期评估(T1)：治疗中期效果监测
终点评估(T2)：治疗结束时主要结局
随访评估(T3)：治疗后3-6个月维持效果

质量控制措施：
- 标准化操作程序(SOP)
- 数据监查委员会(DMC)
- 中间分析计划
- 安全性监测
```

#### 准实验设计

##### 非随机对照试验

| 设计类型 | 特征描述 | 适用情况 | 局限性 |
| :--- | :--- | :--- | :--- |
| **前后对照设计** | 同一组被试治疗前后比较 | 资源有限的小规模研究 | 历史效应和成熟效应 |
| **非等效对照组** | 不同组间比较，无随机分组 | 自然分组的研究情境 | 选择偏倚风险高 |
| **中断时间序列** | 多个时间点的重复测量 | 政策或干预效果评估 | 外部变量影响控制困难 |
| **回归不连续设计** | 基于临界值的分组比较 | 有明确cut-off标准的情况 | 临界值附近的选择偏倚 |

## 测量工具与评估方法 (Measurement Tools & Assessment Methods)

### 标准化量表

#### 核心症状评估量表

##### 恋物障碍量表(Fetishism Scale)

```
量表开发规范：

项目构建：
- 基于DSM-5诊断标准条目化
- 专家咨询和内容效度检验
- 预测试和项目分析
- 最终版本确定

 psychometric properties：
- 信度：Cronbach's α ≥ 0.85
- 效度：结构效度、效标效度
- 敏感性：能detect临床显著变化
- 特异性：正确识别非病例的能力

使用指导：
- 适用人群：18岁以上成人
- 施测时间：15-20分钟完成
- 评分标准：5点李克特量表
- 解释准则：临床分界值确定
```

## 数据分析方法 (Data Analysis Methods)

### 定量数据分析

#### 统计分析策略

##### 基础统计方法

```
常用统计技术：

描述性统计：
- 集中趋势：均值、中位数、众数
- 离散程度：标准差、四分位距
- 分布形态：偏度、峰度指标
- 频数分布：百分比、构成比

推断性统计：
- 参数检验：t检验、方差分析
- 非参数检验：Mann-Whitney U、Wilcoxon
- 相关分析：Pearson、Spearman相关
- 回归分析：线性、logistic回归
```
