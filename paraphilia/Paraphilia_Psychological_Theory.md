## 心理动力学理论 (Psychodynamic Theory)

### 经典精神分析观点

#### 弗洛伊德的恋物理论

##### 核心概念框架

```
恋物形成的经典理论：

阉割焦虑假说：
- 男孩在俄狄浦斯期面临阉割威胁
- 为缓解焦虑，将注意力转向中性对象
- 足部作为生殖器的安全替代品
- 通过象征性满足避免直接性焦虑

防御机制分析：
- 压抑(repression)：将恐惧意识排除
- 升华(sublimation)：将性冲动转向社会可接受形式
- 投射(projection)：将内在冲突外化
- 合理化(rationalization)：为行为寻找合理解释

发展固着概念：
- 肛欲期固着：控制与释放的冲突
- 性器期转移：从生殖器关注转向其他身体部位
- 潜伏期压抑：通过恋物表达被压抑的性冲动
```

#### 客体关系理论解释

##### 内化客体关系

```
客体关系发展模型：

早期母婴关系：
- 完美客体(internalized ideal object)
- 部分客体关系(partial object relations)
- 分裂机制(splitting)的作用
- 投射性认同(projective identification)

关系模式内化：
- 照顾者足部作为关爱象征
- 安全感与依恋的建立
- 分离个体化的挑战
- 补偿性关系模式形成
```

## 认知行为理论 (Cognitive Behavioral Theory)

### 学习理论基础

#### 经典条件反射

##### Pavlovian conditioning model

```
条件反射形成过程：

初始配对阶段：
- 中性刺激(足部) + 无条件刺激(性唤起)
- 反复同时呈现建立关联
- 神经通路的强化连接

泛化和分化：
- 对相似刺激的反应扩展
- 刺激辨别能力的发展
- 反应强度的变化规律

消退和自发恢复：
- 条件刺激单独呈现时反应减弱
- 时间间隔后的重新出现
- 永久性消除的困难性
```

#### 操作性条件反射

##### Skinnerian reinforcement theory

| 强化类型 | 机制原理 | 恋物维持 | 干预策略 |
| :--- | :--- | :--- | :--- |
| **正强化** | 行为后获得愉悦刺激 | 快感强化恋物行为 | 移除强化物 |
| **负强化** | 行为后移除不愉快刺激 | 焦虑缓解维持行为 | 替代应对策略 |
| **惩罚** | 行为后出现负面后果 | 社会谴责的矛盾效应 | 建设性后果 |

### 认知理论模型

#### Beck的认知歪曲

##### 功能不良思维模式

| 认知歪曲类型 | 具体表现 | 维持机制 | 认知重构 |
| :--- | :--- | :--- | :--- |
| **全或无思维** | "要么完全控制，要么彻底失控" | 极端化思维维持焦虑 | 寻找中间地带 |
| **灾难化** | "如果不能控制就会毁灭" | 灾难性预期增强恐惧 | 现实检验训练 |
| **读心术** | "别人一定认为我很奇怪" | 社会焦虑的恶性循环 | 证据检验技术 |
| **选择性抽象** | "只关注负面评价，忽略正面反馈" | 注意偏向维持问题 | 注意力重新训练 |
