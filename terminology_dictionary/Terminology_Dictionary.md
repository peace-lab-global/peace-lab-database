# 平静实验室专业术语词典 (Peace Lab Professional Terminology Dictionary)

## 📋 术语标准化规范

本词典旨在为平静实验室知识库建立统一的专业术语标准，确保术语使用的一致性和准确性。词典涵盖心理学、东方传统智慧、神经科学、治疗方法等多个领域。

> 💡 **新手友好提示**：如果您是初次接触这些概念，建议先阅读下面的"大众化术语解释"部分，那里用更简单的语言解释了这些专业术语。

## 🌟 大众化术语解释 (Popular Terms Explanation)

这部分专门为不了解心理学和疗愈领域的朋友们准备，用最简单直白的语言来解释专业术语。

### 🧠 心理健康基础概念

#### 情绪与心理状态
| 日常用语 | 专业术语 | 简单解释 | 生活举例 |
|---------|---------|----------|----------|
| 心情不好 | 抑郁情绪 | 持续的低落情绪状态 | 连续几天都不想出门、对什么都提不起兴趣 |
| 过度担心 | 焦虑 | 对未来事情的过度担忧 | 总是担心工作出错、害怕社交场合 |
| 容易发火 | 情绪失调 | 情绪波动大，难以控制 | 一点小事就大发脾气，过后又很后悔 |
| 想不开 | 认知扭曲 | 思考问题的方式出了偏差 | 总觉得别人在针对自己，把小问题想得很严重 |
| 压力山大 | 慢性压力 | 长期承受的压力状态 | 工作、家庭、经济多重压力让人喘不过气 |
| 总是回想痛苦事 | 创伤后应激 | 经历创伤后反复回忆痛苦经历 | 车祸后总是梦到事故场景 |
| 对什么都不感兴趣 | 快感缺失 | 失去享受快乐的能力 | 连以前喜欢的爱好也不想做了 |
| 总觉得自己不行 | 低自尊 | 对自己评价过低 | 认为自己不值得被爱、没有价值 |
| 害怕独处 | 分离焦虑 | 离开熟悉环境或人时的恐惧 | 上学时哭闹不愿离开父母 |
| 控制不住自己 | 冲动行为 | 难以抑制的强烈行为冲动 | 明知有害还要暴饮暴食、疯狂购物 |

#### 心理治疗方法
| 通俗说法 | 专业名称 | 简单解释 | 适用情况 |
|---------|---------|----------|----------|
| 聊天治疗 | 心理咨询 | 和专业人士谈话来解决心理问题 | 遇到困难、情绪困扰时 |
| 想法调整 | 认知行为疗法(CBT) | 学会用更积极的方式思考问题 | 总是往坏处想、自我怀疑时 |
| 接受现状 | 接受承诺疗法(ACT) | 学会接受无法改变的事情，专注能控制的部分 | 面对疾病、失去等不可改变的事实 |
| 情绪管理 | 辩证行为疗法(DBT) | 学会更好地处理强烈情绪 | 情绪起伏很大、容易冲动时 |
| 冥想放松 | 正念疗法 | 专注当下，不被杂念带走 | 心思散乱、难以集中注意力时 |
| 眼动治疗 | 眼动脱敏再加工(EMDR) | 通过眼球运动处理创伤记忆 | 创伤后总是回想痛苦经历时 |
| 逐步面对 | 暴露疗法 | 渐进式地接触害怕的事物 | 害怕某种事物或情境时 |
| 身体释放 | 躯体体验疗法 | 通过身体感受释放创伤压力 | 身体紧张、创伤后身体不适时 |
| 家庭和睦 | 家庭治疗 | 改善家庭成员间的关系 | 家庭矛盾、亲子关系问题时 |
| 理解自己 | 心理动力学治疗 | 探索内心深处的想法和感受 | 想深入了解自己、解决深层问题时 |

### 🧘‍♀️ 冥想与正念相关

#### 冥想练习
| 日常理解 | 专业术语 | 简单解释 | 入门方法 |
|---------|---------|----------|----------|
| 静坐放松 | 冥想/禅修 | 安静地坐着，专注呼吸或其他对象 | 每天花5-10分钟，找个安静的地方坐下 |
| 注意当下 | 正念 | 专心注意正在做的事情 | 吃饭时专心品尝味道，走路时感受脚步 |
| 慈悲祝愿 | 慈心禅 | 希望自己和他人都幸福快乐 | 默念"愿我平安快乐，愿你平安快乐" |
| 身体扫描 | 身体觉察 | 依次注意身体各个部位的感受 | 躺下后从脚趾开始，慢慢注意到头顶 |
| 观察呼吸 | 呼吸觉察 | 专注观察自己的呼吸过程 | 数呼吸次数，吸气时数1，呼气时数2 |
| 不评判观察 | 觉察练习 | 不带好坏评价地观察内心活动 | 看到杂念来了又走，不去抓取也不排斥 |
| 专注训练 | 注意力练习 | 训练集中注意力的能力 | 专注于一个点，比如蜡烛火焰或声音 |
| 慢慢行走 | 行禅 | 走路时专注每一步的感受 | 散步时感受脚底与地面的接触 |
| 感恩练习 | 感恩冥想 | 专注感恩的人和事 | 每天想想值得感谢的三件事 |
| 睡前放松 | 睡眠冥想 | 睡前进行放松练习帮助入睡 | 躺在床上做渐进式肌肉放松 |

#### 冥想益处
| 生活改善 | 专业效果 | 简单说明 |
|---------|---------|----------|
| 睡得更好 | 改善睡眠质量 | 冥想能让人更容易入睡，睡得更深 |
| 压力减轻 | 降低皮质醇水平 | 减少压力荷尔蒙，让人更放松 |
| 情绪稳定 | 调节情绪反应 | 不那么容易被小事激怒或难过 |
| 注意力提升 | 增强专注力 | 能够更长时间集中注意力做事 |
| 人际关系改善 | 增强共情能力 | 更能理解别人的感受和需要 |
| 身体健康 | 降低血压心率 | 改善心血管功能，增强免疫力 |
| 创造力增强 | 促进发散思维 | 放松状态下容易产生新想法 |
| 疼痛缓解 | 提高疼痛阈值 | 改变对疼痛的感知和反应 |
| 自我认知 | 增强内省能力 | 更清楚地了解自己的想法和感受 |
| 生活满意度 | 提升主观幸福感 | 对生活有更积极的体验和评价 |

## 🔧 使用规范

### 术语选择原则
1. **准确性优先** - 选择最精确表达概念的术语
2. **一致性原则** - 同一概念在所有文档中使用相同术语
3. **权威性标准** - 优先使用学术界公认的标准术语
4. **实用性考量** - 考虑术语的易理解性和使用便利性

### 术语更新流程
1. 发现新术语或术语争议时，提交术语委员会审议
2. 定期审查术语词典，确保与时俱进
3. 建立术语变更记录，追踪演进历史
4. 通知相关文档维护人员同步更新

### 交叉引用规范
- 术语定义后应提供相关文档的超链接
- 建立术语间的关联关系网络
- 标注术语的使用频率和重要程度
