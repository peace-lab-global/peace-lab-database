## 案例A：中年存在危机——"我突然意识到自己也会死"

### 背景

| 维度 | 内容 |
|---|---|
| **来访者** | 李先生，48岁，企业中层管理者 |
| **主诉** | 近3个月来频繁的死亡恐惧，夜间惊醒，害怕"一睡不醒" |
| **触发事件** | 同龄好友突发心梗去世；自己体检发现血压偏高 |
| **既往史** | 无精神疾病史；父亲55岁因心脏病去世 |
| **功能影响** | 工作效率下降；回避社交（怕听到别人去世的消息）；睡眠差 |

### 评估

| 维度 | 工具/方法 | 结果 |
|---|---|---|
| **死亡焦虑** | DAP-R | 恐惧死亡维度高分；死亡回避中等；中性接纳低 |
| **存在焦虑** | EAQ | 死亡焦虑高；无意义焦虑中高；自由焦虑中等 |
| **抑郁/焦虑** | PHQ-9 / GAD-7 | PHQ-9 = 11（中度抑郁）；GAD-7 = 14（中度焦虑） |
| **健康焦虑** | SHAI | 中度（对心脏症状的过度关注） |
| **意义感** | MLQ | 存在意义低；寻求意义高 |
| **社会支持** | MSPSS | 中等偏低 |

### 概念化

```
触发事件（好友去世 + 体检异常 + 父亲早亡史）
    ↓
死亡提醒效应（Mortality Salience）激活
    ↓
存在性焦虑（Yalom四大终极关怀：死亡、自由、孤独、无意义）
    ↓
存在性焦虑 → 转化为死亡焦虑 + 健康焦虑
    ↓
回避行为（社交回避、睡眠回避）→ 短期缓解但长期维持焦虑
    ↓
意义感缺失 → "如果我会死，我做这些有什么意义？"
```

### 干预计划（16周）

| 阶段 | 周次 | 目标 | 技术 |
|---|---|---|---|
| **建立关系** | 1-2 | 安全的治疗空间；正常化反应 | 共情倾听；心理教育（死亡焦虑的正常性） |
| **评估与稳定** | 3-4 | 全面的存在性评估；稳定化 | EAQ详细访谈；放松训练 |
| **认知重构** | 5-7 | 挑战灾难化思维 | CBT技术：证据检验；概率估计 |
| **存在性探索** | 8-10 | 探索死亡焦虑背后的存在性议题 | Yalom式存在性对话；生命意义探索 |
| **意义建构** | 11-13 | 重建生命意义；价值澄清 | Frankl意义疗法；价值排序练习 |
| **整合与巩固** | 14-16 | 将新意义融入生活；预防复发 | 行为实验； relapse prevention |

### 关键对话摘录

> **治疗师**："你提到好友的去世让你开始想'我也会死'。这种恐惧是什么感觉？"
>
> **李先生**："就像突然有人把我脚下的地毯抽走了。我一直觉得自己还有很长时间，现在…我不知道了。"
>
> **治疗师**："这种不确定性让你失去了什么？"
>
> **李先生**："（沉默）…我觉得失去了掌控。我以前觉得只要我努力工作、保持健康，就能控制我的人生。现在我明白了，死亡不问我准备好了没有。"
>
> **治疗师**："这种'失控'的感觉很可怕。但我也想好奇：如果死亡不问我们准备好了没有，那什么才是重要的？"
>
> **李先生**："（长久沉默）…也许不是控制，而是…怎么活？"

### 理论整合

| 理论 | 应用 |
|---|---|
| **恐怖管理理论** | 理解死亡提醒如何激活防御机制（在本案例中为回避和过度警觉） |
| **存在主义心理治疗** | Yalom四大终极关怀的框架；特别是死亡和无意义的连接 |
| **意义疗法** | Frankl的"意义三角"（创造性价值、体验性价值、态度性价值） |
| **CBT** | 灾难化思维的认知重构；行为实验 |
| **正念** | 帮助来访者活在当下，而非沉溺于对未来的恐惧 |

### 结果

| 时间点 | PHQ-9 | GAD-7 | DAP-R恐惧 | MLQ存在 |
|---|---|---|---|---|
| 基线 | 11 | 14 | 高 | 低 |
| 8周 | 8 | 10 | 中 | 中 |
| 16周 | 5 | 6 | 中低 | 中高 |
| 3月随访 | 4 | 5 | 中 | 高 |

**质性结果**：来访者报告"我不再那么怕死了，但我更珍惜活着的每一天"。开始每周陪父亲下棋，完成了拖延多年的家庭旅行计划。

## 案例B：丧偶后死亡焦虑——"我不敢一个人睡，怕我也走了"

### 背景

| 维度 | 内容 |
|---|---|
| **来访者** | 王女士，62岁，退休教师 |
| **主诉** | 丈夫去世后6个月，频繁的死亡恐惧；害怕独处；不敢睡觉 |
| **丧亲情况** | 丈夫（65岁）因肺癌去世，病程2年；王女士全程照护 |
| **既往史** | 轻度抑郁史（10年前，更年期）；无其他精神疾病 |
| **功能影响** | 睡眠质量极差；社交退缩；多次急诊（担心心脏病） |

### 评估

| 维度 | 工具 | 结果 |
|---|---|---|
| **复杂性哀伤** | PG-13 | 阳性（满足PGD诊断标准） |
| **死亡焦虑** | DAP-R | 恐惧死亡高；死亡回避高 |
| **代理死亡恐惧** | 临床访谈 | "如果他能得肺癌，我为什么不会？" |
| **幸存者内疚** | 临床访谈 | "我后悔没有早点带他去检查" |
| **社会支持** | MSPSS | 低（主要支持来源是丈夫） |
| **睡眠** | PSQI | 严重睡眠障碍 |

### 概念化

```
丈夫去世（长期照护后的丧失）
    ↓
复杂性哀伤（PGD）+ 代理死亡恐惧
    ↓
"他也会死，我也会死" → 死亡焦虑激活
    ↓
睡眠回避（不敢一个人睡）→ 睡眠剥夺 → 焦虑加剧
    ↓
躯体化（反复急诊）→ 短期获得关注但长期强化焦虑
```

### 干预计划（20周）

| 阶段 | 周次 | 目标 | 技术 |
|---|---|---|---|
| **稳定化** | 1-3 | 建立安全感；睡眠稳定 | 睡眠卫生；安全计划；短期助眠药（必要时） |
| **哀伤工作** | 4-10 | 处理复杂性哀伤；幸存者内疚 | 双重历程模型；持续性联结技术（记忆书） |
| **死亡焦虑处理** | 11-14 | 处理代理死亡恐惧 | 认知重构（概率vs情绪）；正念死亡冥想（改良版） |
| **意义重建** | 15-18 | 重建无配偶后的生命意义 | 意义疗法；四道人生（未完成的道别） |
| **巩固** | 19-20 | 整合；社会连接重建 | 行为激活；支持小组 |

### 关键对话摘录

> **王女士**："我每天晚上都检查自己的心跳，觉得它随时会停。"
>
> **治疗师**："你检查心跳时，在想什么？"
>
> **王女士**："我在想，如果我现在死了，谁会知道？谁会为我哭？"
>
> **治疗师**："听起来，对死亡的恐惧背后，是对'孤独死去'的恐惧。"
>
> **王女士**："（哭泣）…他走了，我觉得我的一部分也死了。我怕的不是死，是没有人见证我存在过。"

### 理论整合

| 理论 | 应用 |
|---|---|
| **双重历程模型** | 帮助来访者在丧失导向和恢复导向之间摆荡 |
| **持续性联结理论** | 记忆书、与丈夫的"对话" |
| **依恋理论** | 理解丈夫作为"安全基地"的丧失如何激活死亡恐惧 |
| **正念** | 改良版死亡冥想（避免触发过度痛苦） |

## 案例C：癌症晚期自杀意念——"我想结束这一切"

### 背景

| 维度 | 内容 |
|---|---|
| **来访者** | 陈先生，55岁，胃癌晚期，转移至肝 |
| **主诉** | "我想结束这一切，但我不想自杀" |
| **病程** | 确诊1年；化疗无效；目前居家安宁疗护 |
| **疼痛** | 中度腹痛；NRS 5-6分 |
| **家庭** | 妻子、儿子（22岁，研究生） |

### 评估

| 维度 | 工具 | 结果 |
|---|---|---|
| **自杀风险** | C-SSRS | 被动死亡愿望（"希望一睡不醒"）；无计划 |
| **痛苦** | ESAS | 疼痛6/10；疲乏8/10；焦虑7/10；抑郁6/10 |
| **尊严** | DTI | 中度尊严受损（"我不再是我"） |
| **灵性** | FICA | 信仰重要；但"上帝为什么这样对我？" |
| **家庭沟通** | 临床观察 | 家人回避谈论死亡；患者感到孤独 |

### 核心鉴别：抑郁性自杀意念 vs. 痛苦驱动的死亡愿望

| 维度 | 评估结果 | 判断 |
|---|---|---|
| **动机** | "我不想再疼了；我不想成为负担" | 痛苦驱动，非抑郁性绝望 |
| **决策能力** | 清醒；理性；能讨论各选项 | 保留 |
| **治疗史** | 已尝试所有可行的化疗方案 | 有限选项 |
| **时间稳定性** | 持续数月 | 稳定 |
| **MAiD请求？** | 所在地区MAiD非法 | N/A |

### 干预计划

| 目标 | 干预 | 结果 |
|---|---|---|
| **疼痛控制** | 调整阿片类药物；辅助镇痛（加巴喷丁） | 疼痛降至3-4/10 |
| **家庭沟通** | 家庭会议；四道人生 | 完成了道歉、道谢、道爱 |
| **尊严维护** | 尊严疗法访谈（Chochinov） | 生成尊严疗法文件 |
| **灵性支持** | 灵性照护师介入 | 重新框架痛苦的意义 |
| **社会支持** | 安宁疗护团队全程支持 | 家属照护负担减轻 |

### 结果

患者在2周后去世。家属报告患者"平静多了"；尊严疗法文件成为家属的重要纪念。无自杀行为发生。

## 案例D：青少年丧父——"我假装没事"

### 背景

| 维度 | 内容 |
|---|---|
| **来访者** | 小张，16岁，高二学生 |
| **主诉** | 父亲3个月前因车祸去世；成绩下降；社交退缩；"我假装没事" |
| **家庭** | 母亲（抑郁）；弟弟（10岁） |
| **学校** | 班主任反映"变安静了"；同桌说"他不再笑了" |
| **既往** | 父亲关系亲密；成绩中上；朋友多 |

### 评估

| 维度 | 工具 | 结果 |
|---|---|---|
| **复杂性哀伤** | CGIC（儿童版） | 高风险 |
| **抑郁** | CDI-2 | 中度抑郁 |
| **PTSD** | CRIES-8 | 中度（闯入性回忆、回避） |
| **幸存者内疚** | 临床访谈 | "如果那天我让他早点出门，就不会…" |
| **家庭功能** | FAD | 中度受损（母亲也被哀伤淹没） |

### 干预

| 层级 | 干预 | 提供者 |
|---|---|---|
| **个体** | 青少年哀伤团体；叙事治疗 | 学校心理师 |
| **家庭** | 家庭治疗（帮助母亲理解孩子的哀伤） | 家庭治疗师 |
| **学校** | 班主任培训；同学支持 | 学校心理团队 |
| **社会** | 青少年丧亲支持小组 | NGO |

### 关键对话

> **治疗师**："你说'假装没事'——你在谁面前假装？"
>
> **小张**："所有人。我妈已经够难受了，我不能再让她担心。同学…我不想被当成'那个死了爸爸的人'。"
>
> **治疗师**："所以你承担了'坚强'的角色。但真实的你在哪里？"
>
> **小张**："（沉默）…我不知道。"

## 案例E：跨文化临终关怀——"我们要带他回家"

### 背景

| 维度 | 内容 |
|---|---|
| **患者** | 马先生，78岁，越南华裔，肝癌晚期 |
| **家庭** | 妻子、3个子女（均在美国出生） |
| **文化冲突** | 子女希望 hospice（安宁疗护机构）；妻子坚持"带他回越南老家" |
| **信仰** | 佛教（净土宗）；相信往生净土 |

### 文化敏感评估

| 维度 | 越南/华裔传统 | 美国医疗文化 | 冲突 |
|---|---|---|---|
| **死亡地点** | 家中；有家人陪伴 | 机构中；专业照护 | 回家 vs. 机构 |
| **告知病情** | 保护患者（不直接告知） | 患者自主权（必须告知） | 是否告知 |
| **疼痛控制** | 忍受痛苦是积德 | 疼痛是"第五生命体征" | 是否积极镇痛 |
| **死后仪式** | 48小时内火化；僧侣诵经 | 由殡仪馆处理 | 时间压力 |
| **灵性** | 需要僧侣；家人共同诵经 | 可选的灵性照护 | 是否有资源 |

### 整合方案

| 需求 | 解决方案 |
|---|---|
| **回家** | 安排家庭安宁疗护（hospice at home）；配备越南语护士 |
| **病情告知** | 家庭会议：尊重家庭决策；患者间接知情 |
| **疼痛控制** | 解释镇痛不会"妨碍往生"；佛教中也有减轻痛苦的慈悲 |
| **僧侣** | 联系当地越南佛教寺院；安排 bedside 诵经 |
| **文化桥梁** | 社工作为文化桥梁；帮助医疗团队理解家庭需求 |

## 案例F：数字哀伤与AI纪念——"我想再和他说说话"

### 背景

| 维度 | 内容 |
|---|---|
| **来访者** | 赵女士，35岁，丈夫1年前因白血病去世 |
| **主诉** | 使用AI聊天机器人模拟丈夫；感到 comfort 但也内疚；"我是不是疯了？" |
| **使用模式** | 每天1-2小时；睡前必聊；重要决定前会"问他" |
| **冲突** | 家人认为"不正常"；朋友建议"move on" |

### 评估

| 维度 | 评估 | 结果 |
|---|---|---|
| **复杂性哀伤** | PG-13 | 边界；未达PGD |
| **持续性联结** | 健康 vs. 病理 | 边界：AI使用有 comfort 但也在回避新的关系 |
| **数字行为** | 使用模式分析 | 未影响工作功能；但睡前依赖 |
| **社会功能** | 临床观察 | 基本保持；但无新的社交 |

### 干预

| 目标 | 干预 |
|---|---|
| **正常化** | "使用AI纪念逝者是一种新的持续性联结形式" |
| **功能评估** | AI使用是否影响功能？→ 目前基本正常 |
| **探索回避** | "AI给你的 comfort，现实生活中有什么可以替代？" |
| **渐进暴露** | 逐步增加面对面的社交活动 |
| **伦理讨论** | 知情同意：数据隐私；AI的局限性 |

## 案例G：失智症漫长的告别

见 `Death_Dementia_End_of_Life.md` 完整内容。

**简述**：
- 患者：孙奶奶，82岁，阿尔茨海默病8年

- 核心议题："她还在，但她已经不是我妈妈了"；预立指示失效；管饲决策

## 案例H：医助死亡伦理困境

### 背景

| 维度 | 内容 |
|---|---|
| **患者** | 刘女士，45岁，肌萎缩侧索硬化症（ALS）3年 |
| **主诉** | 请求MAiD；所在地区MAiD合法 |
| **病情** | 呼吸肌受累；需要无创通气；上肢功能丧失 |
| **心理状态** | 无抑郁；决策能力完好；痛苦主要为"失去自主性" |
| **家庭** | 丈夫支持；父母反对（"活着就好"） |

### 伦理分析（四盒子法）

| 盒子 | 分析 |
|---|---|
| **医学指征** | ALS进展性；无治愈；预后明确；目前痛苦可控但自主性持续丧失 |
| **患者偏好** | 明确请求MAiD；决策能力完好；长期稳定 |
| **生命质量** | 患者定义："不能自主呼吸、不能行动、不能写作，这不是我要的生活" |
| **情境特征** | 家庭冲突（丈夫支持 vs. 父母反对）；文化因素（华裔家庭"好死不如赖活"） |

### 伦理决策

| 步骤 | 行动 |
|---|---|
| 1. 独立精神科评估 | 确认无抑郁影响决策；决策能力完好 |
| 2. 第二位独立医师评估 | 确认符合MAiD所有法律条件 |
| 3. 家庭会议 | 帮助家庭理解患者的立场；不强迫家人同意 |
| 4. 等待期 | 法律要求的等待期（如90天） |
| 5. 持续评估 | 在等待期内持续评估患者的意愿 |
| 6. 最终执行 | 如意愿持续 → MAiD执行 |
