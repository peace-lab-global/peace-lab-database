## 男性健康概述

### 核心数据

| 指标 | 数据 |
|------|------|
| 男性平均寿命差距 | 比女性短4-7年(全球) |
| 男性自杀率 | 女性的3-4倍 |
| 主动就医率 | 仅40%定期体检 |
| 心血管死亡(男性) | 第一大死因 |
| 前列腺癌 | 男性第二大癌症 |

### 男性健康的性别差异视角

| 维度 | 男性特点 | 健康影响 |
|------|---------|---------|
| 生物学 | 睾酮影响行为/代谢/体成分 | 肌肉量↑ 内脏脂肪↑ |
| 行为模式 | 风险行为↑ 求助行为↓ | 意外伤害↑ 延迟诊断 |
| 社会角色 | "强壮"期望/情感压抑 | 心理问题隐匿化 |
| 就医行为 | 倾向自我诊断/忽视症状 | 错过早期干预 |
| 社交网络 | 深度友谊较少 | 社会支持不足 |

## 男性特有健康风险

### 男性高发疾病

| 疾病 | 男性vs女性风险比 | 发病年龄 |
|------|----------------|---------|
| 冠心病 | 2:1(55岁前) | 早10-15年 |
| 肺癌 | 1.5:1 | 与吸烟相关 |
| 肝病(酒精性) | 2:1 | 40-60岁 |
| 2型糖尿病 | 1.5:1(低BMI即发) | 40岁+ |
| 肾结石 | 2-3:1 | 30-60岁 |
| 痛风 | 3-4:1 | 40岁+ |
| 睡眠呼吸暂停 | 2-3:1 | 40岁+ |

### 可预防的男性健康风险

| 风险因素 | 男性患病率 | 干预 |
|---------|-----------|------|
| 吸烟 | 高于女性 | 戒烟(最大单一干预) |
| 过量饮酒 | 2-3倍于女性 | 适量/戒酒 |
| 肥胖(内脏型) | 腰围>90cm | 减重/运动 |
| 久坐 | 办公室/驾驶久坐 | 每30min活动 |
| 加工食品 | 红肉/高钠摄入 | 饮食结构调整 |

## 常见男性健康问题

### 前列腺健康

| 疾病 | 特征 | 发生率 |
|------|------|--------|
| 良性前列腺增生(BPH) | 尿频/排尿困难/夜尿 | 50岁50%/80岁90% |
| 前列腺炎 | 疼痛/排尿困难 | 终生患病率10-15% |
| 前列腺癌 | 常无早期症状 | 男性第二大癌症 |

### 前列腺癌筛查争议

| 立场 | 论点 |
|------|------|
| 支持筛查 | 早期发现→治愈率↑; PSA↓死亡率(ERSPC研究) |
| 反对普筛 | 过度诊断/过度治疗; 低危癌症可能不影响寿命 |
| 共识 | 50岁起知情决策(高风险45岁); 与医生讨论利弊 |

### 勃起功能障碍(ED)

| 维度 | 信息 |
|------|------|
| 患病率 | 40岁40%有某程度ED; 70岁70% |
| 作为预警 | ED可能是CVD的早期信号(先于心梗3-5年) |
| 生物因素 | 血管/神经/激素/药物 |
| 心理因素 | 焦虑/抑郁/关系问题/表现焦虑 |
| 生活方式干预 | 运动(有氧)→ED改善(Meta分析支持) |

### 男性不育

| 因素 | 占比 | 常见原因 |
|------|------|---------|
| 精子生成障碍 | 50-60% | 精索静脉曲张/隐睾/感染 |
| 梗阻性 | 10-15% | 输精管阻塞/先天性缺如 |
| 内分泌 | 5-10% | 低促性腺激素 |
| 不明原因 | 20-30% | — |

## 男性生命周期健康

### 各年龄段重点

| 年龄 | 健康重点 | 推荐筛查 |
|------|---------|---------|
| 18-30 | 建立健康习惯/性健康/心理调适 | STI筛查/自我检查 |
| 30-40 | 代谢健康/体重管理/压力管理 | 血脂/血压/血糖(基线) |
| 40-50 | CVD预防/前列腺关注/睾酮下降 | 年度体检/PSA讨论 |
| 50-60 | 癌症筛查/慢病管理/骨密度 | 结肠镜/PSA/年度血检 |
| 60-70 | 功能维持/跌倒预防/认知 | 全面年度评估 |
| 70+ | 生活质量/慢病共管 | 个体化筛查 |

### 男性更年期(迟发性性腺功能减退)

| 维度 | 信息 |
|------|------|
| 术语 | 迟发性性腺功能减退(LOH)/男性更年期(Andropause) |
| 睾酮下降 | 30岁后每年↓1-2% |
| 症状 | 疲劳/性欲↓/情绪波动/肌肉↓/腹部脂肪↑ |
| 诊断 | 症状+晨起总睾酮<300ng/dL(至少2次) |
| 与女性更年期区别 | 渐进(非突然停止)/生育力可延续 |

## 男性健康促进策略

### 综合建议

| 维度 | 建议 |
|------|------|
| 运动 | 有氧150min+力量2-3次/周(维持睾酮/体成分) |
| 饮食 | 地中海模式/足够蛋白/限制酒精/补充锌镁 |
| 心理 | 学会求助/建立情感支持网络/压力管理 |
| 睡眠 | 7-9h/晚(睡眠对睾酮至关重要) |
| 定期检查 | 遵循年龄段筛查建议 |
| 社交 | 维持有意义的社交连接 |
