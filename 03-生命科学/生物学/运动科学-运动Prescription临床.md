## 一、临床运动处方概论 (Clinical Exercise Prescription Overview)

### 1.1 "运动即良药" (Exercise is Medicine, EIM)

美国运动医学会 (ACSM) 于 2007 年发起的全球倡议，核心主张：

> **运动应像药物一样被开具处方：有明确的类型、剂量、频率、疗程和注意事项。**

| 药物处方 vs 运动处方 |
|---------------------|

| 要素 | 药物处方 | 运动处方 |
|------|---------|---------|
| **适应症** | 诊断 | 健康评估结果 |
| **药物/类型** | 药物名称 | 运动形式 (有氧/抗阻/柔韧) |
| **剂量** | mg/次 | 强度 (%HRmax/RPE)、时间 (min) |
| **频率** | 次/天 | 次/周 |
| **疗程** | 周/月 | 周/月 |
| **禁忌症** | 药物禁忌 | 运动禁忌 |
| **不良反应** | 药物副作用 | 运动损伤风险 |
| **随访** | 复诊评估 | 定期体适能评估 |

### 1.2 运动处方的 FITT-VP 框架

| 要素 | 英文 | 临床考量 |
|------|------|---------|
| **频率** | Frequency | 慢性病患者起步 2-3 次/周 |
| **强度** | Intensity | 从低强度开始，渐进增加 |
| **时间** | Time | 可分段累积 (如 3×10 min) |
| **类型** | Type | 根据禁忌症选择关节友好型 |
| **容量** | Volume | 周总量 = 频率 × 时间 × 强度 |
| **进阶** | Progression | 每 2-4 周评估并调整 |

## 二、主要慢性病的运动处方 (Exercise Prescriptions for Chronic Diseases)

### 2.1 2 型糖尿病 (Type 2 Diabetes Mellitus)

| 参数 | 推荐 | 证据/注意事项 |
|------|------|-------------|
| **有氧运动** | 150 min/周中等强度 或 75 min/周高强度 | 降低 HbA1c 0.5-0.7% |
| **抗阻训练** | ≥2 次/周，8-10 个动作，2-3 组×10-15 次 | 改善胰岛素敏感性 |
| **联合训练** | 有氧 + 抗阻优于单一形式 | Church et al., 2010 |
| **避免间隔** | 连续 2 天不运动 | 胰岛素敏感性 24-72h 衰减 |
| **时机** | 餐后 1-3h 运动最利于血糖控制 | 利用胰岛素敏感窗口 |
| **禁忌** | 血糖 >250 mg/dL 伴酮症 | 运动可加重高血糖 |
| **低血糖预防** | 携带快糖、同伴运动、监测血糖 | 特别是使用胰岛素/磺脲类药物时 |

### 2.2 高血压 (Hypertension)

| 参数 | 推荐 | 证据/注意事项 |
|------|------|-------------|
| **有氧运动** | 5-7 次/周，30-60 min/次，中等强度 | 降低 SBP 5-7 mmHg |
| **抗阻训练** | 2-3 次/周，中等负荷，多次数 | 降低 SBP 2-3 mmHg |
| **等长握力训练** | 每次 2 min × 4 组，3 次/周 | 降低 SBP 可达 10 mmHg (新证据) |
| **避免** | Valsalva 动作 (屏气用力) | 血压急剧升高 |
| **注意** | 使用 β-受体阻滞剂者心率反应减弱 | 用 RPE 而非心率监测强度 |
| **紧急停止** | SBP >250 或 DBP >115 mmHg | 立即停止运动 |

> **交叉引用**: 血压详细知识参见 → [biology/blood-pressure/](../血压/)

### 2.3 冠心病 (Coronary Heart Disease)

| 参数 | 推荐 | 证据/注意事项 |
|------|------|-------------|
| **阶段** | 心脏康复 I-IV 期分阶段处方 | 需心脏康复团队参与 |
| **有氧运动** | 3-5 次/周，20-60 min，40-80% HRR | 从低端开始 |
| **抗阻训练** | 2-3 次/周 (稳定期) | 低-中负荷、高次数 |
| **监测** | 心电、血压、RPE | 特别是初期需 ECG 监测 |
| **禁忌** | 不稳定心绞痛、未控制心衰、急性心肌炎 | 绝对禁忌 |
| **危险信号** | 胸痛、头晕、心悸、异常呼吸困难 | 立即停止并就医 |

### 2.4 肥胖 (Obesity)

| 参数 | 推荐 | 证据/注意事项 |
|------|------|-------------|
| **初始目标** | 150 min/周中等强度 | 减重期 |
| **维持目标** | 200-300 min/周 | 防止体重反弹 |
| **抗阻训练** | ≥2 次/周 | 保存/增加瘦体重、提升 RMR |
| **类型** | 低冲击 (游泳/骑行/椭圆机) | 保护关节 |
| **起步** | 10-20 min/次、低强度 | 避免过快进阶致受伤/放弃 |
| **能量赤字** | 运动 + 饮食联合 > 单纯运动 | 500-1000 kcal/天赤字 |
| **心理支持** | 行为改变策略、自我效能建设 | 防止放弃 |

> **交叉引用**: 减肥详细知识参见 → [biology/weight-loss/](../体重减/)

### 2.5 骨质疏松 (Osteoporosis)

| 参数 | 推荐 | 证据/注意事项 |
|------|------|-------------|
| **负重有氧** | 散步、慢跑、爬楼梯、跳舞 | 刺激骨骼重建 |
| **抗阻训练** | 中-高负荷，2-3 次/周 | 刺激成骨细胞 |
| **平衡训练** | 太极拳、单腿站立 | 跌倒预防 |
| **避免** | 脊柱前屈 (仰卧起坐)、高冲击 | 椎体骨折风险 |
| **游泳/骑行** | 心肺有益但对骨密度帮助有限 | 非负重运动 |

### 2.6 关节炎 (Arthritis)

| 类型 | 推荐运动 | 强度 | 注意事项 |
|------|---------|------|---------|
| **骨关节炎** (OA) | 水中运动、骑行、太极、瑜伽 | 低-中等 | 避免高冲击、急性发作期减量 |
| **类风湿性关节炎** (RA) | 低冲击有氧 + 低负荷抗阻 | 低-中等 | 活动期调整、关注关节肿胀 |
| **通用原则** | "Start low, go slow" | 渐进 | 疼痛后 2h 内可接受 |

> **交叉引用**: 太极拳循证参见 → [01-智慧传统/tai-chi/](../../01-智慧传统/太极拳)

## 三、特殊人群运动处方 (Exercise for Special Populations)

### 3.1 老年人 (Older Adults, 65+)

| 组成 | 推荐 | 目标 |
|------|------|------|
| **有氧** | 150-300 min/周中等强度 | 心肺健康 |
| **抗阻** | ≥2 次/周, 主要肌群 | 对抗肌少症 |
| **平衡** | ≥3 次/周 (太极/单腿站) | 跌倒预防 |
| **柔韧** | 每日、主要肌群 | 维持 ROM |

**跌倒预防运动核心**：
- 太极拳 (证据最强：跌倒风险降低 19-43%)
- 奥塔戈运动计划 (Otago Exercise Programme)
- 平衡训练渐进：双脚 → 半脚距 → 单脚 → 闭眼

### 3.2 孕妇 (Pregnancy)

| 阶段 | 推荐 | 禁忌 |
|------|------|------|
| **孕早期 (T1)** | 散步、游泳、产前瑜伽 | 恶心严重时调整 |
| **孕中期 (T2)** | 同上 + 低冲击有氧 | 仰卧位运动 (T2后避免) |
| **孕晚期 (T3)** | 散步、水中运动、骨盆底训练 | 高强度、碰撞运动 |

**绝对禁忌**：
- 前置胎盘 (26 周后)、宫颈机能不全
- 先兆早产、胎膜早破
- 严重贫血、未控制的心脏病/高血压

### 3.3 儿童青少年 (Children & Adolescents)

| 年龄组 | 推荐 | 注意事项 |
|--------|------|---------|
| **5-12 岁** | ≥60 min/天中-高强度体育活动 | 以玩耍为主、多样化 |
| **13-17 岁** | ≥60 min/天 + 抗阻训练 3 次/周 | 可引入结构化训练 |
| **通用** | 含高强度有氧 + 肌肉骨骼强化 | 限制屏幕时间 ≤2h/天 |

**青少年力量训练安全原则**：
- 技术优先于重量
- 有合格教练指导
- 避免 1RM 测试 (用 6-15 RM)
- 关注生长板保护

## 四、运动禁忌与风险分层 (Contraindications & Risk Stratification)

### 4.1 运动绝对禁忌 (Absolute Contraindications)

| 状况 | 说明 |
|------|------|
| 急性心肌梗死 (48h 内) | 心肌尚未稳定 |
| 不稳定型心绞痛 | 运动可诱发心肌缺血 |
| 未控制的严重心律失常 | 猝死风险 |
| 急性心内膜炎/心肌炎 | 心肌炎症活动期 |
| 急性主动脉夹层 | 血压升高加剧撕裂 |
| 症状性重度主动脉狭窄 | 晕厥/猝死风险 |
| 急性肺栓塞/深静脉血栓 | 栓子脱落风险 |
| 急性全身感染 (发热) | 心肌炎风险↑ |

### 4.2 运动相对禁忌 (Relative Contraindications)

需医学评估后决定：

- 未控制的高血压 (SBP >180 或 DBP >110)
- 未控制的代谢性疾病 (甲亢/糖尿病酮症)
- 肥厚性心肌病
- 高度房室传导阻滞
- 严重贫血
- 电解质紊乱

### 4.3 运动中的危险信号 (Warning Signs During Exercise)

**立即停止运动的情况**：

- [ ] 胸痛、胸闷或放射至左臂/下颌的疼痛
- [ ] 头晕、黑矇或晕厥前兆
- [ ] 严重呼吸困难 (与运动强度不成比例)
- [ ] 心悸伴头晕
- [ ] 面色苍白、冷汗
- [ ] 下肢突然肿胀/疼痛 (DVT 征兆)
- [ ] 关节突然锁定或卡顿
