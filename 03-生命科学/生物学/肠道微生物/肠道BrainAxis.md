## 肠-脑轴概述

肠-脑轴（Gut-Brain Axis, GBA）是指肠道微生物组与中枢神经系统之间的双向通讯系统，也称为微生物-肠-脑轴（Microbiota-Gut-Brain Axis）。

### 双向通讯

```
       中枢神经系统 (CNS)
        ↑↓   ↑↓   ↑↓
    自主神经  HPA轴  免疫
     (迷走)  (皮质醇) (细胞因子)
        ↑↓   ↑↓   ↑↓
       肠道微生物组
        ↑↓   ↑↓
     代谢产物  肠道屏障
    (SCFA/色氨酸) (通透性)
```

## 通讯通路

### 五大通路

| 通路 | 机制 | 速度 | 代表信号 |
|------|------|------|---------|
| 神经通路 | 迷走神经直接传导 | 秒级 | 5-HT信号 |
| 内分泌通路 | 激素进入血循环 | 分钟级 | 皮质醇、GLP-1 |
| 免疫通路 | 细胞因子全身传播 | 小时级 | IL-6, TNF-α |
| 代谢通路 | SCFA、色氨酸代谢物 | 小时级 | 丁酸、犬尿氨酸 |
| 屏障通路 | 肠道通透性改变 | 天级 | LPS泄漏 |

### 迷走神经——信息高速公路

| 特征 | 描述 |
|------|------|
| 方向 | 80%传入(肠→脑)、20%传出(脑→肠) |
| 功能 | 传递饱腹感、情绪信号、炎症状态 |
| 关键证据 | 迷走神经切断后，益生菌抗焦虑效果消失(Bravo, 2011) |
| 迷走神经刺激 | FDA批准用于难治性抑郁和癫痫 |

### 肠道神经递质合成

| 神经递质 | 肠道产生占比 | 产生菌 | 功能 |
|---------|------------|--------|------|
| 5-羟色胺(5-HT) | ~95% | 肠嗜铬细胞(菌群调节) | 情绪、肠蠕动 |
| GABA | 大量 | Lactobacillus, Bifidobacterium | 抗焦虑 |
| 多巴胺 | ~50% | Bacillus, Serratia | 动机、奖赏 |
| 去甲肾上腺素 | 少量 | E. coli, Bacillus | 唤醒、警觉 |
| 乙酰胆碱 | 少量 | Lactobacillus | 肌肉控制、记忆 |

## 肠道神经系统

### "第二大脑"

| 特征 | 描述 |
|------|------|
| 神经元数量 | ~5亿（比脊髓还多） |
| 独立性 | 可独立于CNS运作 |
| 神经递质 | 使用>30种神经递质 |
| 与CNS相似性 | 含胶质细胞、屏障系统、类似回路 |

### 肠道屏障——"肠漏"

| 状态 | 描述 | 后果 |
|------|------|------|
| 健康屏障 | 紧密连接完整、选择性通透 | 正常免疫 |
| 肠漏(Leaky Gut) | 紧密连接破坏、通透性增加 | LPS入血→全身炎症 |
| 诱因 | 压力、高脂饮食、酒精、NSAIDs、菌群失调 | — |
| 后果 | 全身低度炎症→脑炎症→行为变化 | 抑郁、焦虑、认知下降 |

## 微生物-免疫-脑通路

### 炎症通路

```
菌群失调 → 屏障破坏 → LPS入血 → TLR4激活 → 促炎细胞因子↑
                                                    ↓
                               穿过血脑屏障 → 小胶质细胞激活
                                                    ↓
                               神经炎症 → 5-HT↓ BDNF↓ → 抑郁/焦虑
```

### SCFA的脑效应

| SCFA | 脑效应 | 机制 |
|------|--------|------|
| 丁酸 | 抗抑郁、BDNF↑ | HDAC抑制 |
| 丙酸 | 过多→行为异常 | 直接神经毒性(高浓度) |
| 醋酸 | 食欲调节 | 下丘脑信号 |

## 临床意义

### 菌群失调相关疾病谱

| 疾病 | 菌群特征 | 证据强度 |
|------|---------|---------|
| 肠易激综合征(IBS) | 多样性↓、Firmicutes变化 | ★★★★★ |
| 炎症性肠病(IBD) | 显著失调、多样性大幅↓ | ★★★★★ |
| 肥胖 | F/B比值↑、多样性↓ | ★★★★☆ |
| 2型糖尿病 | Akkermansia↓、丁酸菌↓ | ★★★★☆ |
| 抑郁症 | Lactobacillus/Bifidobacterium↓ | ★★★☆☆ |
| 自闭症(ASD) | 多样性↓、Clostridium↑ | ★★★☆☆ |
| 帕金森病 | Prevotella↓、口腔菌↑ | ★★★☆☆ |
| 阿尔茨海默病 | 促炎菌↑、SCFA产生菌↓ | ★★☆☆☆ |
