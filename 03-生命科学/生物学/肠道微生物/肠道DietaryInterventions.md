## 饮食与菌群

### 饮食对菌群的影响时间线

| 时间 | 变化 |
|------|------|
| 24小时 | 饮食改变→菌群代谢产物变化 |
| 3-4天 | 菌群组成开始显著变化 |
| 2周 | 主要菌群结构调整 |
| 1-3月 | 相对稳定的新菌群格局 |
| 恢复原饮食 | 菌群趋向恢复原始状态 |

### 饮食成分对菌群的影响

| 饮食成分 | 菌群效应 | 健康影响 |
|---------|---------|---------|
| 膳食纤维 | Bifidobacterium↑ SCFA↑ 多样性↑ | 抗炎、屏障修复 |
| 多酚 | Akkermansia↑ 抗氧化菌↑ | 抗炎、抗氧化 |
| 饱和脂肪 | Bilophila↑ 多样性↓ | 促炎、屏障损伤 |
| 精制糖 | 多样性↓ 致病菌↑ | 促炎、代谢紊乱 |
| 人工甜味剂 | 菌群紊乱、糖耐量↓ | 代谢风险 |
| 红肉(过量) | TMAO↑ | 心血管风险 |
| Omega-3脂肪酸 | 抗炎菌↑ | 抗炎 |
| 发酵食品 | 多样性↑ 炎症标志物↓ | 免疫调节 |

## 益生菌干预

### 益生菌选择指南

| 目标 | 推荐菌株 | 剂量 | 证据 |
|------|---------|------|------|
| 一般肠道健康 | L. rhamnosus GG / B. lactis BB-12 | 10⁹ CFU/天 | 强 |
| IBS | B. infantis 35624 / L. plantarum 299v | 10⁹-10¹⁰ CFU/天 | 强 |
| 抗生素后恢复 | S. boulardii | 250-500mg/天 | 强 |
| 情绪改善 | L. helveticus R0052 + B. longum R0175 | 10⁹ CFU/天 | 中 |
| 免疫支持 | L. rhamnosus GG + B. lactis | 10⁹ CFU/天 | 中 |

### 益生菌使用注意

| 原则 | 建议 |
|------|------|
| 菌株特异性 | 益处是菌株特异的，不是属级通用 |
| 剂量足够 | 通常需≥10⁹ CFU/天 |
| 持续使用 | 停用后效果可能减退 |
| 配合饮食 | 益生菌+益生元(合生元)效果更好 |
| 储存条件 | 部分需冷藏，注意保质期 |
| 安全性 | 免疫抑制/重症患者慎用 |

## 益生元与膳食纤维

### 益生元定义

益生元（Prebiotics）: 不被宿主消化、但能选择性刺激有益菌生长的膳食成分。

### 主要益生元类型

| 类型 | 食物来源 | 培养的菌 | 每日建议量 |
|------|---------|---------|-----------|
| 菊粉(Inulin) | 菊苣根、洋葱、大蒜 | Bifidobacterium | 5-10g |
| FOS(低聚果糖) | 香蕉、洋葱、大蒜 | Bifidobacterium, Lactobacillus | 5-10g |
| GOS(低聚半乳糖) | 豆类、母乳 | Bifidobacterium | 3-5g |
| 抗性淀粉 | 冷却米饭/土豆、青香蕉 | 丁酸产生菌 | 15-30g |
| β-葡聚糖 | 燕麦、蘑菇 | 多种有益菌 | 3-5g |
| 多酚 | 茶、浆果、可可、红酒 | Akkermansia | 变化大 |

### 膳食纤维推荐

| 指标 | 推荐 | 现实摄入 |
|------|------|---------|
| WHO推荐 | 25-30g/天 | 全球平均仅约15g |
| 菌群最优 | 30-40g/天 | 发达国家平均12-18g |
| 纤维种类 | ≥30种不同植物/周 | 多数人<15种 |
| 渐进增加 | 每周增加5g，避免胀气 | — |

## 发酵食品

### 常见发酵食品

| 食品 | 主要微生物 | 每日建议 |
|------|-----------|---------|
| 酸奶 | Lactobacillus, Streptococcus | 200g |
| 开菲尔 | 多种乳酸菌、酵母菌 | 200mL |
| 泡菜/酸菜 | Lactobacillus | 50-100g |
| 味噌 | Aspergillus | 1-2汤匙 |
| 康普茶 | SCOBY(多种菌) | 200-350mL |
| 纳豆 | Bacillus subtilis | 50g |
| 豆豉 | Aspergillus, Rhizopus | 适量 |

### Stanford发酵食品研究（2021）

- **Wastyk et al., Cell**: 36名健康成人，10周饮食干预
- **高发酵食品组(6份/天)**: 菌群多样性显著↑，19种炎症标志物↓
- **高纤维组**: 菌群SCFA产能↑，但短期内多样性未显著增加
- **结论**: 发酵食品比单纯高纤维更快速增加菌群多样性

## 饮食模式推荐

### 菌群友好饮食排名

| 排名 | 饮食模式 | 菌群效应 | 心理健康证据 |
|------|---------|---------|------------|
| 1 | 地中海饮食 | 多样性↑ SCFA↑ | SMILES试验: 抑郁缓解率32% |
| 2 | MIND饮食 | 抗炎菌↑ | 认知衰退风险↓53% |
| 3 | 传统日本饮食 | 多样性高 | 与低抑郁率相关 |
| 4 | 素食/半素食 | Prevotella↑ SCFA↑ | 有益但需注意B12 |
| 5 | 西方饮食(反面) | 多样性↓ 致病菌↑ | 抑郁风险↑ |

### SMILES试验（2017, Jacka et al.）

- **设计**: RCT, n=67, 中重度抑郁患者
- **干预**: 地中海饮食指导(12周) vs 社会支持
- **结果**: 饮食组32.3%达到缓解 vs 对照组8%
- **NNT**: 4.1（每治疗4人有1人受益）
- **意义**: 首个证实饮食干预可治疗抑郁的RCT

## 实用指南

### 7天菌群重置计划

| 天数 | 饮食调整 |
|------|---------|
| 第1天 | 添加1份发酵食品(酸奶/泡菜) |
| 第2天 | 增加2种新蔬菜 |
| 第3天 | 加入全谷物替代精白米面 |
| 第4天 | 添加豆类(扁豆/鹰嘴豆) |
| 第5天 | 增加1份坚果/种子 |
| 第6天 | 添加第2种发酵食品 |
| 第7天 | 回顾+规划持续方案 |

### 每周30种植物挑战

目标每周吃30种不同植物（包括蔬菜、水果、谷物、豆类、坚果、种子、香料、香草）：

| 类别 | 目标种类 | 示例 |
|------|---------|------|
| 蔬菜 | 10+ | 西蓝花、胡萝卜、菠菜、番茄... |
| 水果 | 5+ | 苹果、蓝莓、橙子、猕猴桃... |
| 全谷物 | 4+ | 燕麦、糙米、荞麦、小米... |
| 豆类 | 3+ | 黑豆、鹰嘴豆、扁豆... |
| 坚果种子 | 4+ | 核桃、杏仁、亚麻籽、南瓜子... |
| 香料草本 | 4+ | 姜黄、生姜、肉桂、罗勒... |
