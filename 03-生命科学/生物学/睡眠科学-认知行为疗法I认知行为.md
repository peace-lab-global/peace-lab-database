# CBT-I 认知行为疗法 | Cognitive Behavioral Therapy for Insomnia

> CBT-I是慢性失眠的一线治疗（ACP 2016强推荐），效果优于药物且长期获益更持久。

---

## 1. CBT-I理论基础

### 1.1 失眠的维持模型

CBT-I基于Spielman的3P模型，聚焦**维持因素**（使失眠慢性化的行为和认知）：

| 维持因素 | 行为层面 | 认知层面 |
|----------|----------|----------|
| **延长卧床** | 早上赖床、白天补觉 | "我需要补回来" |
| **不规律作息** | 每天入睡/起床时间波动>2h | "什么时候困了什么时候睡" |
| **条件性觉醒** | 床=清醒/焦虑/思考 | 一进卧室就清醒 |
| **灾难化思维** | — | "再不睡明天就完了" |
| **安全行为** | 提前上床、取消活动 | "我得保证8小时" |
| **过度监控** | 反复看钟、计算还能睡多久 | "我已经只睡了3小时" |

### 1.2 核心治疗逻辑

```
维持因素(目标) → 干预技术 → 预期效果
─────────────────────────────────────
延长卧床/不规律 → 睡眠限制 → 增强S过程驱动
条件性觉醒 → 刺激控制 → 重建床-睡眠联结
灾难化认知 → 认知重构 → 降低睡前焦虑
过度觉醒 → 放松训练 → 降低HPA/交感活动
安全行为 → 行为实验 → 纠正错误信念
```

## 2. CBT-I六大核心组分

### 2.1 睡眠教育 (Psychoeducation)

**目标**: 建立对睡眠的科学认知，降低焦虑

- 双过程模型：解释S过程和C过程如何调节睡眠
- 睡眠结构：NREM/REM周期，夜间变化规律
- 年龄相关变化：SWS随年龄减少是正常的
- 「8小时」迷思：个体差异(6-9h)，睡眠质量>时长
- 失眠的后果：短期影响可逆，打破灾难化预期

### 2.2 刺激控制 (Stimulus Control)

**原理**: Bootzin (1972) 基于经典条件反射——打破「床=觉醒/焦虑」的条件化

| 规则 | 说明 |
|------|------|
| ① 仅困倦时上床 | 不在清醒状态下躺床 |
| ② 床仅用于睡眠和性 | 不看书/不刷手机/不工作/不打电话 |
| ③ 20min未入睡→离床 | 去另一个房间做放松活动(阅读/冥想/轻音乐) |
| ④ 困了再回床 | 如果20min又未入睡→重复③④ |
| ⑤ 固定起床时间 | 无论前夜睡眠如何，每天同一时间起床(±30min) |
| ⑥ 避免白天午睡 | 或限制为下午3点前20min的power nap |

**效果量**: d = 0.89-1.20，CBT-I中效果最大的单一组分

### 2.3 睡眠限制 (Sleep Restriction)

**原理**: Spielman et al. (1987) 通过压缩卧床时间增加睡眠稳态驱动

**操作流程**:
1. 2周睡眠日记→计算平均TST(实际睡眠时间)和SE(睡眠效率)
2. 设定初始卧床时间 = TST（最低5.5h，最高不超过基线卧床时间）
3. 固定起床时间（与个人社会时间一致）→倒推入睡时间
4. 每周评估：
   - SE ≥ 85%（连续5天）→ 卧床增加15-30min（就寝提前）
   - SE 80-85% → 维持不变
   - SE < 80% → 卧床减少15min（就寝推迟）
5. 目标：SE > 90%且TST达个人需求

**示例**:
```
基线: 卧床10h(23:00-9:00), 实际睡6h, SE=60%
第1周: 卧床6h(2:00-8:00固定起), 预期SE↑
第2周: SE=88% → 卧床6.25h(1:45-8:00)
第3周: SE=91% → 卧床6.5h(1:30-8:00)
...逐步扩展至7.5h且SE>85%
```

**注意事项**:
- 初始日间嗜睡可能加重→警告驾驶/操作机械
- 躁狂风险患者慎用（睡眠限制可能触发）
- 癫痫未控患者慎用

### 2.4 认知重构 (Cognitive Restructuring)

**目标**: 识别并挑战关于睡眠的功能不良信念

| 功能不良信念 | 挑战 | 替代认知 |
|-------------|------|----------|
| "我必须睡满8小时" | 个体差异大，6-9h均正常 | "我的身体会找到合适的时长" |
| "睡不着就完了" | 一晚失眠不影响长期功能 | "即使睡不好，明天也能应对" |
| "失眠会毁掉我的健康" | 短期失眠可逆，身体有弹性 | "睡眠问题可以改善" |
| "我应该控制睡眠" | 睡眠是自动过程，越控制越觉醒 | "放下控制，让睡眠自然发生" |
| "昨晚那么差，今晚肯定更差" | 每晚独立，不存在必然递进 | "每晚都是新的开始" |

**技术**:
- **思维记录表**: 情境→自动思维→情绪→证据→替代想法
- **苏格拉底式提问**: "最坏会发生什么？""过去类似经历的实际情况？"
- **行为实验**: 一晚不刻意追求睡眠→观察次日实际功能
- **矛盾意向法**: "今晚请努力保持清醒"→减少入睡表现焦虑

### 2.5 放松训练

| 技术 | 方法 | 机制 |
|------|------|------|
| **渐进性肌肉放松(PMR)** | 16组肌群交替紧张5s→放松20s | 降低躯体觉醒，注意力转移 |
| **腹式呼吸** | 4s吸-7s停-8s呼(4-7-8) | 延长呼气激活副交感(迷走神经) |
| **正念身体扫描** | 从脚趾到头顶逐步觉察 | 降低认知反刍，接受当下 |
| **引导意象** | 想象平静场景(海滩/森林) | 占据工作记忆，阻断焦虑思维 |
| **自生训练** | "我的手臂沉重温暖" | 自我暗示降低交感张力 |

### 2.6 矛盾意向法 (Paradoxical Intention)

- 指示来访者"尽可能保持清醒，不要睡着"
- 消除"我必须入睡"的表现焦虑
- 证据: 部分RCT显示可缩短SOL (Ascher & Efran, 1975)
- 适用: 高睡眠焦虑、过度努力型失眠者

## 3. CBT-I疗程设计

### 3.1 标准6-8周方案

| 周次 | 主题 | 内容 |
|------|------|------|
| **W1** | 评估与教育 | 睡眠日记开始、双过程模型、睡眠卫生 |
| **W2** | 刺激控制+睡眠限制 | 设定卧床窗口、20min规则、固定起床 |
| **W3** | 调整与放松 | 根据SE调整卧床、PMR/腹式呼吸 |
| **W4** | 认知重构 | 思维记录、挑战灾难化信念 |
| **W5** | 深化与扩展 | 卧床时间扩展、矛盾意向法 |
| **W6** | 复发预防 | 应对挫折计划、维持策略、自我监测 |
| **W7-8** | 巩固(可选) | 微调、处理残余问题、逐步退出 |

### 3.2 简版CBT-I (4次会谈)

- **BBTI (Brief Behavioral Treatment for Insomnia)**: Buysse et al.
- 4次会谈(2面诊+2电话)，聚焦刺激控制+睡眠限制
- RCT: 与完整CBT-I效果相当 (Buysse et al., 2011)

## 4. CBT-I的数字化

| 平台 | 类型 | 循证 | 特点 |
|------|------|------|------|
| **Sleepio** | 全自动6周 | 3项RCT (Espie et al.) | 虚拟治疗师、个性化 |
| **Somryst** | FDA-DTx | RCT (Roth et al., 2019) | 处方级、保险覆盖 |
| **SHUTi** | 交互式6模块 | RCT (Ritchie et al., 2017) | 学术背景、适配研究 |
| **CBT-i Coach** | 辅助工具(非独立) | 基于证据 | VA出品、配合面诊使用 |

**dCBT-I vs 面诊CBT-I**: 效果量相当 (van Straten et al., 2018 Meta)，dCBT-I可扩展性更强

## 5. 共病情境下的CBT-I调整

| 共病 | 调整 |
|------|------|
| **抑郁症** | CBT-I可独立改善抑郁(SMD=-0.53)；与抗抑郁药协同 |
| **慢性疼痛** | 睡眠限制保守(≥6h)；整合疼痛认知教育 |
| **PTSD** | 先处理噩梦(IRT/哌唑嗪)→再CBT-I；避免过度觉醒期做睡眠限制 |
| **物质使用** | 戒断初期失眠常见；CBT-I预防复发(失眠→渴求→使用) |
| **老年** | 保守睡眠限制(≥6.5h)；日间活动增加；光照疗法辅助 |
