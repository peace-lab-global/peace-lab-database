## 女性健康概述

### 核心数据

| 指标 | 数据 |
|------|------|
| 女性平均寿命 | 比男性长4-7年(全球) |
| 产后抑郁发生率 | 10-20% |
| 子宫内膜异位症 | 约10%育龄女性 |
| 乳腺癌 | 女性第一大癌症 |
| 骨质疏松 | 50岁以上女性1/3 |
| 自身免疫病 | 女性占78% |

### 女性健康的性别差异视角

| 维度 | 女性特点 | 健康意义 |
|------|---------|---------|
| 激素周期 | 月经/妊娠/更年期的激素波动 | 影响情绪/代谢/免疫 |
| 自身免疫 | 免疫活性更强 | 自身免疫病风险↑ |
| 心血管 | 绝经前雌激素保护 | 绝经后风险快速追赶男性 |
| 骨骼 | 骨量较低+绝经后快速丢失 | 骨质疏松高风险 |
| 疼痛 | 慢性疼痛患病率↑ | 纤维肌痛/偏头痛/盆腔痛 |
| 心理 | 抑郁/焦虑发病率↑ | 激素波动期为脆弱窗口 |

## 女性特有健康领域

### 妇科常见疾病

| 疾病 | 患病率 | 核心症状 |
|------|--------|---------|
| 子宫肌瘤 | 30-70%(40岁) | 异常出血/盆腔痛/压迫 |
| 子宫内膜异位症 | 10% | 痛经/慢性盆腔痛/不孕 |
| PCOS | 6-12% | 月经不调/多毛/痤疮/不孕 |
| 外阴阴道炎 | 75%终生≥1次 | 分泌物异常/瘙痒 |
| 盆底功能障碍 | 25%(产后) | 尿失禁/器官脱垂 |

### 乳腺健康

| 维度 | 信息 |
|------|------|
| 乳腺癌终生风险 | ~12.5%(每8人1人) |
| 筛查(钼靶) | 40-50岁起，每1-2年(指南不一) |
| 自我检查 | 乳房自我意识(awareness)优于定期自检 |
| 风险因素 | 年龄/家族史/BRCA/肥胖/酒精/HRT |
| 保护因素 | 运动/健康体重/母乳喂养/限酒 |

### 骨质疏松

| 维度 | 信息 |
|------|------|
| 绝经后骨丢失 | 前5年丢失20-30%松质骨 |
| DXA筛查 | 65岁起(高风险者更早) |
| 预防 | 钙1200mg+VitD 800IU/天 |
| 运动 | 负重运动+力量训练 |
| 药物 | 双膦酸盐/SERM/Denosumab |

## 女性生命周期健康

### 各阶段重点

| 阶段 | 年龄 | 健康重点 |
|------|------|---------|
| 青春期 | 10-18 | 月经建立/身体形象/HPV疫苗 |
| 育龄期 | 18-40 | 避孕/生育规划/宫颈筛查 |
| 围产期 | 孕前-产后 | 孕前准备/产前保健/产后恢复 |
| 围绝经期 | 40-55 | 月经变化/血管舒缩/骨密度 |
| 绝经后 | 55+ | CVD预防/骨质疏松/认知 |

### 女性心理脆弱窗口

| 时期 | 激素变化 | 心理风险 |
|------|---------|---------|
| 经前期 | 孕酮↓ 雌激素↓ | PMS/PMDD |
| 产后 | 雌孕激素骤降 | 产后抑郁/焦虑/精神病 |
| 围绝经期 | 雌激素波动→↓ | 抑郁↑ 焦虑↑ 认知↓ |
| 手术绝经 | 突然雌激素↓ | 急性心理适应 |

## 女性健康促进

### 综合建议

| 维度 | 建议 |
|------|------|
| 运动 | 有氧+力量+柔韧(骨密度/心血管/情绪) |
| 营养 | 铁/钙/叶酸充足; 地中海饮食模式 |
| 筛查 | 宫颈(21岁起)/乳腺(40岁起)/骨密度(65岁起) |
| 心理 | 关注激素脆弱窗口/社会支持 |
| 盆底 | 产后盆底康复/凯格尔运动 |
| 睡眠 | 7-9h/晚(关注围绝经期睡眠障碍) |
