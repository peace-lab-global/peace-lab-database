## 一、免疫系统基础 (Immune System Fundamentals)

### 1.1 免疫系统的三道防线

```
第一道防线 (物理/化学屏障):
  皮肤 → 黏膜 → 胃酸 → 泪液/唾液中的溶菌酶

第二道防线 (先天免疫/非特异性):
  巨噬细胞 → 中性粒细胞 → NK细胞 → 补体系统 → 炎症反应

第三道防线 (适应性免疫/特异性):
  T细胞 (细胞免疫) → B细胞 (体液免疫) → 抗体 → 免疫记忆
```

### 1.2 先天免疫 vs 适应性免疫

| 特征 | 先天免疫 (Innate) | 适应性免疫 (Adaptive) |
|------|-------------------|---------------------|
| **反应速度** | 即时 (分钟-小时) | 慢 (天-周) |
| **特异性** | 非特异性 (广谱) | 高度特异性 |
| **免疫记忆** | 无 | 有 (疫苗原理) |
| **主要细胞** | 巨噬细胞、NK细胞、中性粒细胞 | T细胞、B细胞 |
| **分子介质** | 补体、干扰素、炎症因子 | 抗体 (IgG/IgM/IgA/IgE) |
| **第一反应** | ✅ 总是第一个启动 | 需要先天免疫信号激活 |

### 1.3 关键免疫细胞图谱

| 细胞类型 | 英文 | 主要功能 | 临床意义 |
|---------|------|---------|---------|
| **巨噬细胞** | Macrophage | 吞噬、抗原呈递、分泌细胞因子 | 炎症调控核心 |
| **中性粒细胞** | Neutrophil | 细菌感染第一反应 | 感染标志 |
| **NK 细胞** | Natural Killer | 杀伤病毒感染细胞/肿瘤细胞 | 免疫监视 |
| **树突细胞** | Dendritic Cell | 抗原呈递 (连接先天与适应性) | 疫苗设计核心 |
| **辅助 T 细胞** | CD4+ T Cell | 调控免疫应答方向 | HIV 靶细胞 |
| **杀伤 T 细胞** | CD8+ T Cell | 杀伤病毒感染细胞/肿瘤 | 癌症免疫治疗 |
| **调节 T 细胞** | Treg | 抑制过度免疫反应 | 自身免疫调控 |
| **B 细胞** | B Cell | 产生抗体 | 疫苗效果 |
| **肥大细胞** | Mast Cell | 释放组胺 | 过敏反应 |

> **Professional Perspective**: Medzhitov (2008, Nature) 指出，免疫系统不仅是"防御系统"，更是维持组织稳态的"感知-调控系统"。慢性炎症本质上是这一调控系统失衡的表现。

## 二、免疫系统的调控网络 (Regulatory Networks)

### 2.1 Th1/Th2 平衡模型

| 极化方向 | 主导细胞因子 | 主要功能 | 失衡后果 |
|---------|------------|---------|---------|
| **Th1** | IFN-γ, IL-2, TNF-α | 细胞免疫、抗病毒/肿瘤 | 自身免疫 (RA, MS, T1DM) |
| **Th2** | IL-4, IL-5, IL-13 | 体液免疫、抗寄生虫 | 过敏/哮喘 |
| **Th17** | IL-17, IL-22 | 黏膜防御、中性粒细胞召集 | 自身免疫/慢性炎症 |
| **Treg** | IL-10, TGF-β | 免疫耐受、抑制过度反应 | 耐受缺失→自身免疫 |

### 2.2 细胞因子关键图谱

| 细胞因子 | 主要来源 | 功能 | 临床意义 |
|---------|---------|------|---------|
| **TNF-α** | 巨噬细胞 | 促炎、发热、组织破坏 | RA/IBD 靶向治疗 |
| **IL-1β** | 巨噬细胞 | 促炎、发热 | 痛风靶向治疗 |
| **IL-6** | 多种细胞 | 急性期反应、促/抗炎双重 | CRP 诱导、运动后释放 |
| **IL-10** | Treg/巨噬细胞 | 抗炎、免疫抑制 | 自身免疫治疗靶点 |
| **IFN-γ** | T细胞/NK细胞 | 抗病毒、激活巨噬细胞 | 抗肿瘤免疫 |
| **TGF-β** | 多种细胞 | 组织修复、免疫调节 | 纤维化/肿瘤微环境 |

## 三、免疫系统与生活方式因素 (Lifestyle & Immunity)

### 3.1 影响免疫功能的因素

| 因素 | 增强免疫 ✅ | 削弱免疫 ❌ |
|------|-----------|-----------|
| **睡眠** | 7-9h 充足睡眠 → NK细胞活性↑ | <6h 睡眠 → 感染风险↑ 4.2倍 |
| **运动** | 中等强度规律运动 → 免疫监视↑ | 过度训练 → 免疫抑制窗口 |
| **压力** | 短暂急性压力 → 免疫动员 | 慢性压力 → 皮质醇持续↑ → 免疫抑制 |
| **营养** | 多样化饮食 → 微量营养素充足 | 营养不良/过度限制 → 免疫缺陷 |
| **社交** | 良好社交关系 → 炎症标志物↓ | 社交隔离 → 炎症水平↑ |
| **年龄** | 青壮年免疫功能最强 | 免疫衰老 (Immunosenescence) |
| **肠道** | 微生物多样性高 → 免疫调节好 | 菌群失调 → 免疫紊乱 |

### 3.2 运动与免疫的 J 型曲线

```
感染风险
  高 ┤ ●                            ●
     │  ╲                         ╱
     │   ╲                       ╱
  中 ┤    ╲                     ╱
     │     ╲         ●        ╱
     │      ╲       ╱ ╲      ╱
  低 ┤       ╲     ╱   ╲    ╱
     │        ╲   ╱     ╲  ╱
     │         ● ╱       ●
     ┼────────────────────────────
     久坐    适量运动   过度训练

→ 中等强度规律运动：感染风险最低
→ 过度训练/马拉松后：存在"免疫开放窗口" (Open Window, 3-72h)
```

> **Professional Perspective**: Nieman & Wentz (2019, JSSM) 的最新观点修正了经典 J 型曲线，认为高强度运动后的免疫抑制窗口可能被高估。真正的风险因素是长期大训练量+恢复不足+心理压力+营养缺陷的"多重打击"(Multiple Hits)，而非单次高强度运动。

## 四、免疫系统评估指标 (Assessment Markers)

### 4.1 常用免疫功能指标

| 指标 | 英文 | 正常范围 | 临床意义 |
|------|------|---------|---------|
| **白细胞计数** | WBC | 4,000-11,000/μL | 感染/免疫状态 |
| **C-反应蛋白** | CRP | <3 mg/L | 全身炎症标志 |
| **红细胞沉降率** | ESR | 男<15, 女<20 mm/h | 慢性炎症 |
| **免疫球蛋白** | IgG/IgA/IgM | 各有参考值 | 体液免疫功能 |
| **NK 细胞活性** | NK Activity | — | 免疫监视能力 |
| **CD4/CD8 比值** | T Cell Ratio | 1.0-3.0 | 免疫调控状态 |
| **IL-6** | Interleukin-6 | <7 pg/mL | 急性/慢性炎症 |
| **TNF-α** | 肿瘤坏死因子 | <8.1 pg/mL | 慢性炎症强度 |
