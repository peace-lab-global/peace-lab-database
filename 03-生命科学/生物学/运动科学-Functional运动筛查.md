## 一、功能性动作筛查概论 (FMS Overview)

### 1.1 什么是 FMS

功能性动作筛查 (Functional Movement Screen, FMS) 由 Gray Cook 和 Lee Burton 于 1995 年开发，是一套标准化的 7 个动作测试体系，用于评估基本动作模式的质量。

**核心理念**：

> "在力量和体能之前，先确保动作质量。在一个功能障碍的基础上增加力量，就像在一个裂缝的地基上建房子。"  
> — Gray Cook, *Movement*

```
FMS 的层级模型:

                ┌────────────┐
                │  运动表现  │ ← 技巧/力量/速度/爆发力
                ├────────────┤
                │  功能性动作│ ← FMS 评估层级 (你在这里)
                ├────────────┤
                │  基本动作  │ ← 关节活动度/稳定性
                └────────────┘
```

### 1.2 FMS 的价值与局限

| 优势 | 局限 |
|------|------|
| 标准化评分系统 (0-3 分) | 不能诊断具体病理 |
| 快速 (~10 min) | 评分者间信度中等 (ICC 0.76-0.91) |
| 识别左右不对称 | 与损伤预测的关联仍有争议 |
| 为矫正训练提供方向 | 不评估力量和有氧能力 |
| 建立基线数据 | 需要培训才能正确执行 |

> **Professional Perspective**: Kiesel et al. (2007) 发现 FMS 总分 ≤14 分的职业橄榄球运动员受伤风险比 >14 分者高 11 倍。但后续研究 (Dorrel et al., 2015 meta-analysis) 表明 FMS 的损伤预测能力在一般人群中不如在运动员中强。FMS 更适合作为动作质量评估工具，而非损伤预测工具。

## 二、FMS 七项测试详解 (The Seven FMS Tests)

### 2.1 测试总览

| # | 测试名称 | 英文 | 评估维度 | 关键肌群/关节 |
|---|---------|------|---------|-------------|
| 1 | 深蹲 | Deep Squat | 双侧对称活动度+稳定性 | 踝/膝/髋/肩/胸椎 |
| 2 | 跨栏步 | Hurdle Step | 单腿站立稳定性+步态 | 髋屈曲/踝背屈/核心 |
| 3 | 直线弓步蹲 | In-Line Lunge | 减速/稳定性/对称性 | 髋/踝/核心 |
| 4 | 肩部灵活性 | Shoulder Mobility | 肩关节综合活动度 | 肩内外旋/胸椎伸展 |
| 5 | 主动直腿抬高 | Active Straight Leg Raise | 腘绳肌柔韧/骨盆稳定 | 腘绳肌/髋屈肌/核心 |
| 6 | 躯干稳定俯卧撑 | Trunk Stability Push-up | 核心稳定性 | 核心抗伸展/上肢推 |
| 7 | 旋转稳定性 | Rotary Stability | 多平面核心稳定 | 核心抗旋转/协调 |

### 2.2 评分标准

| 分数 | 定义 | 含义 |
|------|------|------|
| **3** | 完美完成 | 无代偿动作 |
| **2** | 可以完成但存在代偿 | 有动作质量问题需关注 |
| **1** | 无法完成动作模式 | 存在显著功能限制 |
| **0** | 执行时出现疼痛 | 需转介医学评估 |

**总分解读**：

| 总分 (满分 21) | 解读 | 建议 |
|---------------|------|------|
| 18-21 | 优秀动作质量 | 可进行高强度训练 |
| 15-17 | 良好，存在小问题 | 在训练中融入矫正动作 |
| 12-14 | 中等，存在明显代偿 | 优先矫正后再增加负荷 |
| <12 | 较差，功能限制明显 | 以矫正训练为主 |

**额外规则**：
- 左右不对称差 ≥1 分 → 优先解决不对称
- 任何项目得 0 分 → 转介物理治疗/医学评估
- 任何项目得 1 分 → 该动作模式暂不负重训练

## 三、常见动作代偿模式 (Common Compensation Patterns)

### 3.1 深蹲代偿模式

| 代偿表现 | 可能原因 | 筛查方法 | 矫正策略 |
|---------|---------|---------|---------|
| 脚跟离地 | 踝背屈受限 | 膝过墙测试 | 踝关节松动术、小腿拉伸 |
| 膝盖内扣 (Valgus) | 臀中肌无力/踝内翻 | 单腿蹲测试 | 侧向弹力带走、蚌式 |
| 过度前倾 | 核心弱/髋屈肌紧 | 高脚杯深蹲测试 | 前蹲、死虫 |
| 腰椎过度后凸 (Butt Wink) | 髋屈曲受限/腘绳肌紧 | 仰卧髋屈曲测试 | 90/90 拉伸、箱式深蹲 |
| 手臂前移 | 肩/胸椎活动度不足 | 壁上天使测试 | 胸椎旋转、肩部松动 |

### 3.2 上交叉综合征 (Upper Crossed Syndrome)

由 Vladimir Janda 提出的常见姿势失衡模式：

```
           紧张/过度活跃                 无力/被抑制
        ┌─────────────────┐       ┌─────────────────┐
        │ 上斜方肌/肩胛提肌│       │ 深层颈屈肌       │
        │ 胸大肌/胸小肌    │       │ 下斜方肌/菱形肌  │
        │ 枕下肌群         │       │ 前锯肌           │
        └─────────────────┘       └─────────────────┘
        
表现: 头前倾 + 圆肩 + 驼背 + 胸椎后凸增加
```

**矫正策略**：

| 步骤 | 目标 | 练习 |
|------|------|------|
| 1. 抑制 (Inhibit) | 放松过度紧张肌群 | 胸大肌筋膜球松解、上斜方肌拉伸 |
| 2. 延长 (Lengthen) | 增加柔韧性 | 门框胸肌拉伸、颈伸肌拉伸 |
| 3. 激活 (Activate) | 唤醒无力肌群 | 下斜方肌 Y-T-W、深层颈屈曲 |
| 4. 整合 (Integrate) | 功能性模式训练 | 面拉、壁上天使、高位划船 |

> **交叉引用**: 办公肩颈问题参见 → [biology/office-neck-shoulder/](../办公颈部肩/)

### 3.3 下交叉综合征 (Lower Crossed Syndrome)

```
           紧张/过度活跃                 无力/被抑制
        ┌─────────────────┐       ┌─────────────────┐
        │ 髋屈肌 (髂腰肌)  │       │ 臀大肌            │
        │ 竖脊肌 (腰段)    │       │ 腹肌 (深层核心)   │
        └─────────────────┘       └─────────────────┘
        
表现: 骨盆前倾 + 腰椎过伸 + 腹部突出 + 臀肌失活
```

**矫正策略**：

| 步骤 | 目标 | 练习 |
|------|------|------|
| 1. 抑制 | 髋屈肌放松 | 髂腰肌筋膜放松、股直肌拉伸 |
| 2. 延长 | 增加髋伸展 | 半跪姿髋屈肌拉伸、Thomas 拉伸 |
| 3. 激活 | 臀肌唤醒 | 臀桥、蚌式、四足伸髋 |
| 4. 整合 | 功能性模式 | 罗马尼亚硬拉、台阶上步 |

> **交叉引用**: 腰痛问题参见 → [biology/lower-back-pain/](../下背疼痛/)

## 四、核心稳定性评估与训练 (Core Stability Assessment & Training)

### 4.1 核心功能的四维模型

| 维度 | 英文 | 功能 | 代表练习 |
|------|------|------|---------|
| **抗伸展** | Anti-Extension | 抵抗腰椎过伸 | 平板支撑、死虫 |
| **抗屈曲** | Anti-Flexion | 抵抗躯干前屈 | 硬拉、早安式 |
| **抗旋转** | Anti-Rotation | 抵抗躯干旋转 | Pallof Press、单臂农夫走 |
| **抗侧屈** | Anti-Lateral Flexion | 抵抗侧向弯曲 | 侧平板、手提箱走 |

### 4.2 核心训练渐进方案

```
Level 1 (初级 — 稳定性基础):
  死虫 (Dead Bug) → 鸟狗 (Bird Dog) → 平板支撑 (Plank)
  
Level 2 (中级 — 增加负荷/不稳定):
  Pallof Press → 单臂农夫走 → 半跪姿Chop/Lift
  
Level 3 (高级 — 动态/多平面):
  土耳其起立 (TGU) → 负重行走 → 旋转药球抛
```

### 4.3 McGill "Big Three" (Stuart McGill 核心三大)

| 练习 | 英文 | 目标 | 处方 |
|------|------|------|------|
| 卷腹 | McGill Curl-Up | 前方核心 | 3×6-8, 保持 10s |
| 侧桥 | Side Bridge | 侧方核心 | 3×15-30s 每侧 |
| 鸟狗 | Bird Dog | 后方核心 | 3×6-8 每侧, 保持 10s |

> **Professional Perspective**: McGill (2015) 强调，核心训练的目标是"刚度" (Stiffness) 而非"力量"，即核心应作为力量传递的稳定平台。仰卧起坐 (Sit-up) 因对腰椎施加约 3,300N 压力已不被推荐为核心训练。

## 五、动作筛查与矫正训练的实施流程 (Implementation Protocol)

### 5.1 标准评估流程

```
Step 1: 病史/运动史/疼痛史问卷 (10 min)
   ↓
Step 2: 静态姿势评估 — 前/后/侧面观察 (5 min)
   ↓
Step 3: FMS 七项动作筛查 (10-15 min)
   ↓
Step 4: 补充评估 — 针对问题区域深入 (5-10 min)
   ├── 踝背屈受限 → 膝过墙测试
   ├── 肩活动度受限 → Apley搔背测试
   └── 核心不稳 → 侧平板耐力测试
   ↓
Step 5: 报告与矫正训练处方 (5 min)
```
