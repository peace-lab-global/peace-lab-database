# 睡眠卫生与昼夜节律 | Sleep Hygiene & Chronobiology

---

## 1. 睡眠卫生：循证行为处方

### 1.1 光照管理

光是最强的授时因子，通过ipRGC(含黑视素的内在光敏视网膜神经节细胞)→SCN通路调节昼夜节律。

| 时段 | 光照策略 | 参数 |
|------|----------|------|
| **晨起 (起床后30min)** | 户外自然光暴露 | ≥30min, >2500lux(阴天可达10,000+) |
| **日间** | 保持充足照明 | 办公≥500lux，靠窗最佳 |
| **傍晚 (日落前2h)** | 降低光照强度 | <30lux, 暖色调(<3000K) |
| **睡前 (1-2h)** | 最低光照 | <10lux, 避免直视屏幕 |
| **夜间觉醒** | 红光/琥珀光 | <5lux, 不开顶灯 |

**蓝光机制**:
- 峰值敏感波长460-480nm
- 抑制褪黑素分泌：2h屏幕暴露可延迟DLMO约1.5h (Chang et al., 2015)
- Night-shift/蓝光过滤降低50%抑制但非消除
- 实用策略：纸质书替代屏幕、Kindle Paperwhite低亮度、f.lux软件

### 1.2 温度调节

| 策略 | 机制 | 推荐 |
|------|------|------|
| **卧室温度** | 核心体温需下降1-1.5°F促进入睡 | 18-19.5°C (65-67°F) |
| **热水浴/浴足** | 外周血管扩张→核心体温下降 | 40-42°C, 睡前90min, 10-20min |
| **冷枕/降温垫** | 头部散热降低额叶代谢 | 33-35°C枕面(Nofzinger et al., 2016 RCT) |
| **被褥选择** | 透气材质、分层盖被 | 避免过热→夜间觉醒 |
| **手足保暖** | 远端血管扩张→核心体温下降 | 穿袜入睡可缩短SOL (Krauchi et al., 1999) |

### 1.3 饮食与睡眠

| 物质 | 影响 | 建议 |
|------|------|------|
| **咖啡因** | 腺苷A1/A2A拮抗；半衰期5-6h(基因差异) | 下午2点后避免；日摄入<400mg |
| **酒精** | 缩短SOL但碎片化REM→WASO↑ | 睡前3h内避免；不作为助眠手段 |
| **尼古丁** | 兴奋剂，缩短SWS/REM | 戒烟或至少睡前4h避免 |
| **大餐** | 消化负担→觉醒/GERD | 睡前2-3h避免 |
| **色氨酸食物** | 5-HT前体→褪黑素 | 火鸡/牛奶/坚果+碳水(助吸收) |
| **酸樱桃汁** | 含天然褪黑素+花青素 | 240ml bid (Howatson et al., 2012) |
| **猕猴桃** | 含5-HT/抗氧化物 | 睡前1h 2颗 (Lin et al., 2011) |
| **镁** | NMDA/GABA调节 | 甘氨酸镁200-400mg睡前 |
| **甘氨酸** | 降低核心体温 | 3g睡前 (Yamadera et al., 2007) |

### 1.4 运动与睡眠

| 运动类型 | 时间 | 对睡眠的影响 |
|----------|------|-------------|
| **规律有氧** | 下午/傍晚 | SWS↑12-25%、SOL↓、SE↑ (Kredlow et al., 2015 Meta) |
| **高强度** | 睡前2h内 | 交感激活→SOL延长 |
| **瑜伽/拉伸** | 睡前30-60min | 副交感激活→SOL缩短 |
| **抗阻训练** | 下午 | SWS↑，但避免睡前3h |

**最佳策略**: 下午4-6时进行中高强度运动，睡前做温和拉伸/瑜伽

## 2. 昼夜节律优化实践

### 2.1 识别个人睡眠时型 (Chronotype)

| 时型 | MEQ评分 | 特征 | 占比 |
|------|---------|------|------|
| 明确晨型 | 70-86 | 自然早起、晨间高效 | 13% |
| 中度晨型 | 59-69 | 偏好早起但可调节 | 27% |
| 中间型 | 42-58 | 灵活适应 | 40% |
| 中度晚型 | 31-41 | 偏好晚睡晚起 | 15% |
| 明确晚型 | 16-30 | 自然入睡晚(1-3am) | 5% |

### 2.2 建立稳固的昼夜节律锚点

| 锚点 | 做法 | 重要性 |
|------|------|--------|
| **固定起床时间** | 每天同一时间(±30min)，含周末 | 最强锚点，驱动C过程同步 |
| **晨间光照** | 起床后30min内接受光照 | SCN授时，抑制褪黑素 |
| **规律进食** | 三餐定时，早餐在起床1h内 | 外周时钟同步(肝/肠) |
| **日间活动** | 保持足够的日间身体和认知活动 | 增强S过程积累 |
| **睡前例行** | 每晚相同的30-60min放松程序 | 条件化"准备入睡"信号 |

### 2.3 社会时差管理

社会时差 = 工作日与休息日中位睡眠时间的差异

| 社会时差 | 健康影响 | 干预 |
|----------|----------|------|
| <1h | 最小影响 | 维持 |
| 1-2h | 轻度影响 | 周末起床延后≤1h |
| >2h | 代谢/心血管/情绪风险显著 | 严格固定起床+晨光照 |

### 2.4 时差调整 (Jet Lag)

| 方向 | 策略 |
|------|------|
| **东飞(提前)** | 出发前3天逐步提前就寝+晨光照；到达后避免傍晚光 |
| **西飞(延后)** | 出发前3天逐步延迟就寝+傍晚光照；到达后避免晨间光 |
| **通用** | 褪黑素0.5-3mg(到达后当地就寝时间)；短途<4天不必严格调整 |

## 3. 睡前例行程序 (Bedtime Routine)

### 3.1 推荐60分钟放松程序

```
T-60min  停止工作/刺激性活动；降低室内光照
T-45min  热水淋浴/浴足 (40-42°C)
T-30min  轻柔活动：拉伸/瑜伽/正念呼吸
T-20min  放松活动：阅读(纸质)/日记/轻音乐
T-10min  准备就寝：刷牙、调温、黑暗环境
T-0      上床 → 腹式呼吸/身体扫描 → 入睡
```

### 3.2 认知"关机"技术

| 技术 | 方法 | 适用人群 |
|------|------|----------|
| **忧虑日记** | 睡前1h写下担忧+明日行动项→合上本子 | 反刍型 |
| **感恩练习** | 写下今天3件感恩的事 | 焦虑/悲观型 |
| **认知洗牌** | 随机想不相关词语(每词想象几秒) | 思维奔逸型 |
| **4-7-8呼吸** | 4s吸-7s屏-8s呼 × 4-8次 | 躯体紧张型 |
| **身体扫描** | 从脚趾到头顶渐进放松 | 全身紧张型 |
