## 运动与心血管健康

### 运动的心血管保护机制

| 机制 | 效应 | 证据强度 |
|------|------|---------|
| 血管内皮功能↑ | NO生成↑→血管扩张 | 强 |
| 动脉顺应性↑ | 血管弹性改善 | 强 |
| HDL-C↑ LDL-C↓ | 血脂谱改善 | 强 |
| 血压↓ | 静息血压↓5-8mmHg | 强 |
| 胰岛素敏感性↑ | 代谢改善 | 强 |
| 炎症标志物↓ | CRP/IL-6↓ | 中等 |
| 心脏重塑 | 心室容量↑ 壁厚适度↑ | 强 |
| HRV↑ | 迷走张力↑ | 中等 |
| 凝血功能改善 | 纤溶活性↑ | 中等 |
| 心理效益 | 抑郁焦虑↓ 压力耐受↑ | 强 |

### 运动量效关系

| 运动量(MET-h/周) | CVD风险降低 | 等效活动 |
|-------------------|------------|---------|
| 0 (久坐) | 基准 | — |
| 7.5 (最低推荐) | ~20% | 快走150min/周 |
| 15 (双倍推荐) | ~30% | 快走300min/周 |
| 25+ (高量) | ~35% | 跑步150min/周 |
| 50+ (极高量) | ~35-40%(边际递减) | 马拉松训练 |

**关键发现**: 从久坐到少量运动的风险降低最大(非线性关系)。

### 久坐行为的独立危害

| 指标 | 影响 |
|------|------|
| 每日久坐>8h | CVD死亡率↑(独立于运动) |
| 每30min打断一次 | 代谢指标改善 |
| 站立办公 | 每日额外消耗~54kcal |
| 每增加1h电视 | CVD风险↑18% |

## 心血管运动处方

### 一般人群推荐(AHA/ACC 2019)

| 运动类型 | 频率 | 强度 | 时间 | 备注 |
|---------|------|------|------|------|
| 有氧运动 | ≥5天/周 | 中等(64-76% MHR) | ≥30min | 或3天高强度各≥25min |
| 力量训练 | 2-3天/周 | 中等-高 | 每组8-12次 | 8-10种主要肌群 |
| 柔韧性 | ≥2天/周 | 拉伸至轻微不适 | 每次30-60s | 主要关节 |
| 减少久坐 | 每天 | — | 每30min活动 | 即使已达运动量 |

### 高血压患者运动处方

| 参数 | 推荐 | 效果 |
|------|------|------|
| 有氧 | 5-7天/周, 30-60min, 中等强度 | 收缩压↓5-8mmHg |
| 力量 | 2-3天/周, 中等负荷 | 收缩压↓2-3mmHg |
| 等长握力 | 3天/周, 4×2min | 收缩压↓5-10mmHg |
| 注意 | 血压>180/110暂停运动 | 运动前评估 |

### 高强度间歇训练(HIIT)的心血管效益

| 维度 | HIIT vs 持续中等强度 |
|------|---------------------|
| VO2max提升 | HIIT优势(+2-3 mL/kg/min) |
| 血压改善 | 相当 |
| 血脂改善 | HIIT可能更优 |
| 时间效率 | HIIT显著优势 |
| 安全性 | 需评估后进行(心脏患者) |
| 依从性 | 因人而异 |

## 心脏康复运动

### 心脏康复四阶段

| 阶段 | 时间 | 目标 | 运动内容 |
|------|------|------|---------|
| I期(住院) | 急性期 | 早期活动 | 床边活动/步行 |
| II期(门诊) | 2-12周 | 体能恢复 | 监护下运动(12-36次) |
| III期(维持) | 3-6月 | 独立运动 | 社区运动计划 |
| IV期(终身) | 终身 | 维持/提升 | 自主运动习惯 |

### 心梗后运动处方

| 参数 | 推荐 |
|------|------|
| 开始时间 | 事件后4-6周(经评估) |
| 运动试验 | 运动前症状限制性运动试验 |
| 有氧强度 | 40-80% 心率储备(渐进) |
| 频率 | 3-5天/周 |
| 持续时间 | 20-60min(含热身冷却) |
| 力量训练 | 8-12周后，低阻力高重复 |
| 监测 | 心率/血压/RPE/症状 |

### 运动中心脏风险管理

| 风险信号 | 处理 |
|---------|------|
| 胸痛/压迫感 | 立即停止，就医 |
| 异常气短 | 降低强度或停止 |
| 头晕/晕厥 | 停止，平卧 |
| 心悸/不规则心跳 | 停止，评估 |
| 异常疲劳 | 降低强度 |

## 心血管营养策略

### 地中海饮食详解(PREDIMED)

| 成分 | 推荐 | 心血管机制 |
|------|------|-----------|
| 特级初榨橄榄油 | ≥4汤匙/天 | 多酚抗氧化/抗炎 |
| 坚果(核桃杏仁) | 30g/天 | α-亚麻酸/精氨酸 |
| 鱼类 | ≥3次/周 | EPA/DHA/抗心律失常 |
| 蔬菜 | ≥2份/天 | 膳食硝酸盐/钾 |
| 水果 | ≥3份/天 | 抗氧化/纤维 |
| 全谷物 | 替代精制谷物 | 纤维/改善血糖 |
| 豆类 | ≥3次/周 | 植物蛋白/纤维/镁 |
| 红酒(可选) | ≤1杯/天 | 白藜芦醇(争议) |
| 限制: 红肉 | <1次/周 | 减少饱和脂肪/TMAO |
| 限制: 加工食品 | 尽量避免 | 减少钠/反式脂肪 |

**PREDIMED结果**: 主要心血管事件↓30%(HR 0.70, P=0.001)

### DASH饮食详解

| 成分 | 推荐 | 降压机制 |
|------|------|---------|
| 钠 | <2300mg/天(理想<1500) | 容量↓ |
| 钾 | 4700mg/天 | 钠钾平衡 |
| 钙 | 1250mg/天 | 血管平滑肌 |
| 镁 | 500mg/天 | 血管扩张 |
| 纤维 | 30g/天 | 代谢改善 |
| 蔬果 | 8-10份/天 | 综合效应 |
| 低脂奶 | 2-3份/天 | 钙/蛋白 |
| 全谷物 | 7-8份/天 | 纤维/镁 |

**DASH结果**: 收缩压↓11mmHg(高血压者), ↓6mmHg(正常者)

### 关键营养素与心血管健康

| 营养素 | 心血管效应 | 食物来源 |
|--------|-----------|---------|
| Omega-3(EPA/DHA) | TG↓/抗心律失常/抗炎 | 深海鱼/亚麻籽 |
| 膳食纤维 | LDL↓/血糖控制 | 全谷/蔬果/豆类 |
| 硝酸盐 | NO↑→血管扩张→血压↓ | 甜菜/菠菜/芹菜 |
| 类黄酮 | 抗氧化/内皮保护 | 可可/浆果/茶 |
| 钾 | 降压/心脏保护 | 香蕉/土豆/豆类 |
| 镁 | 血压↓/抗心律失常 | 坚果/深绿蔬菜/全谷 |
| 叶酸 | 同型半胱氨酸↓ | 深绿蔬菜/豆类 |

## 营养补充剂

### 心血管补充剂证据

| 补充剂 | 证据等级 | 效果 | 建议 |
|--------|---------|------|------|
| Omega-3鱼油(>2g) | A | TG↓20-30% | 高TG人群 |
| CoQ10 | B | 心衰辅助/他汀肌痛 | 100-300mg/天 |
| 植物固醇 | B | LDL↓6-15% | 2g/天 |
| 可溶性纤维 | B | LDL↓5-10% | 5-10g/天 |
| 镁 | B | 血压↓2-3mmHg | 缺镁者 |
| 红曲米 | B-C | 类他汀效应(天然洛伐他汀) | 需质控 |
| 维生素D | C | 争议(缺乏者补充) | 无明确心血管益处 |
| 大蒜提取物 | C | 血压轻微↓ | 证据不一致 |
| 维生素E | D | 无益甚至可能有害 | 不推荐 |
| β-胡萝卜素 | D | 可能增加吸烟者肺癌风险 | 不推荐 |

### 补充剂使用原则

| 原则 | 说明 |
|------|------|
| 食物优先 | 营养素优先从食物中获取 |
| 针对性补充 | 明确缺乏后再补充 |
| 与医生沟通 | 特别是正在服药的患者 |
| 注意交互 | 鱼油+抗凝药→出血风险↑ |
| 循证选择 | 选择有临床证据支持的补充剂 |
