## 心理心脏病学概述

### 定义
心理心脏病学(Psychocardiology)研究心理因素与心血管疾病之间的双向关系。心理因素既是CVD的风险因素，CVD也对心理健康产生深远影响。

### 核心关系图

```
心理因素 ←→ 生理路径 ←→ 心血管结局
  ↕              ↕              ↕
抑郁焦虑     HPA/SNS激活     动脉硬化
愤怒敌意     炎症/凝血       心梗/中风
社会隔离     内皮功能障碍    心力衰竭
工作压力     代谢紊乱        心律失常
```

### 里程碑研究

| 年份 | 研究 | 发现 |
|------|------|------|
| 1959 | Friedman & Rosenman | A型行为模式与CHD |
| 1988 | WCGS随访 | 敌意(而非整体A型)是核心因子 |
| 1993 | Barefoot纵向研究 | 抑郁独立预测CAD |
| 2004 | INTERHEART | 心理社会因素贡献33% PAR |
| 2012 | Lancet综述 | 抑郁确认为CVD独立风险因素 |
| 2017 | Kivimäki Meta | 工作压力→CHD RR=1.23 |

## 心理因素与CVD

### 抑郁与CVD

| 维度 | 证据 |
|------|------|
| 新发CVD风险 | RR=1.6-2.0(独立风险因素) |
| 心梗后抑郁发生率 | 20-25% |
| 心梗后抑郁预后 | 死亡率↑2-3倍 |
| 生物机制 | HPA轴激活/SNS过度/炎症↑/血小板活化/HRV↓ |
| 行为机制 | 服药依从性↓/运动↓/吸烟↑/社交退缩 |

### 焦虑与CVD

| 类型 | CVD关联 |
|------|--------|
| 广泛性焦虑 | CHD风险↑26%(Meta分析) |
| 恐慌障碍 | 急性心脏事件风险↑ |
| 心脏性焦虑 | 过度关注心脏症状→功能受限 |
| 创伤后应激 | CVD风险↑55%(Edmondson 2013) |

### 愤怒、敌意与A型行为

| 维度 | 说明 |
|------|------|
| A型行为(Type A) | 竞争性/时间紧迫感/敌意 |
| 毒性核心(Toxic Core) | 敌意(Hostility)是真正致病因素 |
| 愤怒触发心脏事件 | 心梗后2小时内风险↑4.7倍 |
| 生理路径 | 反复SNS激活→内皮损伤→动脉硬化 |
| 循证干预 | 愤怒管理→CAD事件↓(Williams) |

### D型人格(Type D)

| 特征 | 说明 |
|------|------|
| 定义 | 消极情感(NA)高 + 社交抑制(SI)高 |
| 提出者 | Denollet (1996) |
| CVD影响 | 不良预后风险↑3倍 |
| 评估工具 | DS14量表(14题) |
| 机制 | 持续负性情绪+不表达→慢性应激 |

### 社会心理因素

| 因素 | CVD影响 |
|------|--------|
| 社会隔离/孤独 | 风险↑29%(Holt-Lunstad) |
| 低社会支持 | 预后差2-3倍 |
| 工作压力 | CHD风险↑23%(Kivimäki) |
| 婚姻冲突 | 心脏事件风险↑ |
| 低社经地位 | CVD发病率/死亡率均↑ |
| 照护者负担 | CVD风险↑(配偶照护者) |

## 心脏事件的心理影响

### 心梗后心理反应

| 阶段 | 反应 | 发生率 |
|------|------|--------|
| 急性期(住院) | 焦虑/恐惧/否认 | 50-70% |
| 恢复期(1-3月) | 抑郁/愤怒/哀伤 | 20-30% |
| 适应期(3-12月) | 功能恢复/心脏性焦虑 | 10-20%持续困难 |
| 长期 | 创伤后成长/慢性心理困扰 | 因人而异 |

### 心脏性焦虑(Cardiac Anxiety)

| 维度 | 表现 |
|------|------|
| 恐惧 | 害怕再次发作/猝死 |
| 过度警觉 | 反复检查脉搏/关注胸部感觉 |
| 回避行为 | 避免运动/性活动/外出 |
| 躯体化 | 非心源性胸痛/心悸 |
| 评估 | 心脏焦虑问卷(CAQ) |

### 心脏手术后的心理影响

| 状态 | 发生率 | 表现 |
|------|--------|------|
| 术后谵妄 | 15-50% | 意识混乱/幻觉 |
| 术后认知下降 | 20-40% | 记忆/注意力受损 |
| 术后抑郁 | 20-30% | 情绪低落/兴趣丧失 |
| 术后PTSD | 10-15% | 闪回/噩梦/回避 |

## 心理干预在心脏病中的应用

### 循证心理干预

| 干预 | 证据等级 | 效果 |
|------|---------|------|
| 心脏康复(含心理) | A | 全因死亡↓20-26% |
| CBT治疗心脏抑郁 | A | 抑郁↓(ENRICHD试验) |
| 压力管理训练 | B | CVD事件↓(Gulliksson 2011) |
| 正念减压(MBSR) | B | 焦虑/抑郁/血压↓ |
| 生物反馈(HRV) | B | HRV↑ 焦虑↓ |
| 社会支持干预 | B | 改善预后 |
| 冥想/放松 | B | 血压↓ 压力↓ |

### 心脏康复心理组件

| 组件 | 内容 |
|------|------|
| 心理评估 | PHQ-9/GAD-7筛查 |
| 健康教育 | 疾病知识/风险因素 |
| 压力管理 | 放松/认知重构 |
| 行为改变 | 运动/饮食/戒烟计划 |
| 社会支持 | 团体/配偶/互助小组 |
| 回归计划 | 工作/性生活/日常活动 |

## 心理行为改变

### 心血管行为风险改变策略

| 行为 | 干预策略 | 成功率 |
|------|---------|--------|
| 戒烟 | 动机访谈+NRT/药物 | 25-35% |
| 运动习惯 | 渐进目标+社会支持 | 40-50%(6月维持) |
| 饮食改变 | 正向替换+环境设计 | 30-40% |
| 服药依从 | 简化方案+提醒系统 | 提高20-30% |
| 压力管理 | MBSR+认知重构 | 显著改善 |

### 动机访谈(MI)在心脏病中的应用

| 原则 | 在心脏病中的应用 |
|------|----------------|
| 表达共情 | 理解患者对改变的矛盾心理 |
| 发展差异 | 当前行为vs健康目标的差距 |
| 避免争辩 | 不直接对抗阻抗 |
| 支持自我效能 | 强调过往成功经验 |

### 积极心理学在心脏健康中的应用

| 因素 | CVD保护效应 |
|------|------------|
| 乐观主义 | CVD事件↓35%(Kubzansky 2001) |
| 目的感 | 全因死亡↓(Hill & Turiano 2014) |
| 积极情绪 | 独立于负性情绪的保护效应 |
| 感恩 | HRV↑ 炎症↓(初步证据) |
| 社交联结 | 保护效应等同戒烟 |
