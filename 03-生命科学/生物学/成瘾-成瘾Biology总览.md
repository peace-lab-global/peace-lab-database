# 物质成瘾生物学总览 | Substance Addiction Biology Overview

> 成瘾是一种慢性脑疾病，涉及奖赏、动机、记忆及相关回路的神经生物学改变。

---

## 1. 成瘾的神经生物学模型

### 1.1 三阶段循环模型 (Koob & Volkow, 2010/2016)

| 阶段 | 脑回路 | 神经递质 | 体验 |
|------|--------|----------|------|
| **狂醉/中毒 (Binge/Intoxication)** | 基底节(伏隔核/背侧纹状体) | DA↑↑ / 阿片类↑ / GABA↑ | 欣快、奖赏 |
| **戒断/负性情感 (Withdrawal/Negative Affect)** | 杏仁核扩展区 | CRF↑ / 去甲肾上腺素↑ / DA↓ / 阿片↓ | 焦虑、烦躁、痛苦 |
| **渴求/复发 (Preoccupation/Anticipation)** | 前额叶(眶额叶/前扣带/岛叶) | 谷氨酸↑ / DA条件化 | 渴求、强迫性寻药 |

**核心洞见**: 成瘾从正性强化(追求快感)转变为负性强化(逃避痛苦)——反奖赏理论(Anti-Reward)

### 1.2 多巴胺与奖赏预测误差

| 概念 | 说明 |
|------|------|
| **基线DA** | 伏隔核基础多巴胺水平 |
| **药物诱导DA** | 可卡因↑DA 300%、甲基苯丙胺↑DA 1200%、酒精↑DA 40-200% |
| **奖赏预测误差(RPE)** | 实际奖赏 > 预期 → DA↑(正误差)；实际 < 预期 → DA↓(负误差) |
| **敏化(Incentive Sensitization)** | Robinson & Berridge: 反复用药→中脑边缘系统对药物线索超敏化→"想要"(wanting)≠"喜欢"(liking) |
| **耐受(Tolerance)** | 受体下调→需要更多药物达到同等效果 |

### 1.3 关键脑结构

| 结构 | 成瘾中的角色 | 变化 |
|------|-------------|------|
| **伏隔核(NAc)** | 奖赏中枢、动机驱动 | DA释放增加→后期下调 |
| **腹侧被盖区(VTA)** | DA神经元胞体 | 药物诱导长时程可塑性(LTP) |
| **杏仁核** | 情绪记忆、负性情感 | CRF系统过度激活(戒断期) |
| **前额叶(PFC)** | 决策、抑制控制、渴求 | 灰质减少、功能低下→失控 |
| **岛叶(Insula)** | 内感受觉察、渴求体验 | 激活→渴求感 |
| **海马** | 情境记忆→药物关联记忆 | 环境线索触发渴求 |

## 2. 主要成瘾物质的生物学

### 2.1 酒精 (Alcohol)

| 维度 | 内容 |
|------|------|
| **主要靶点** | GABA-A↑(抑制性)、NMDA↓(兴奋性)、DA↑、阿片↑、5-HT↑ |
| **急性效应** | 低剂量：去抑制/社交增强；高剂量：镇静/共济失调/呼吸抑制 |
| **戒断** | 震颤→焦虑→幻觉→DT(震颤性谵妄)→癫痫→死亡 |
| **戒断时间线** | 6-12h: 焦虑/震颤 → 12-48h: 幻觉 → 48-72h: DT风险 |
| **基因** | ADH1B/ALDH2多态性(亚洲flush反应)→酒精不耐受→保护因素 |
| **致死量** | BAC > 0.40% (约400mg/dL) |

### 2.2 尼古丁 (Nicotine)

| 维度 | 内容 |
|------|------|
| **主要靶点** | α4β2烟碱型乙酰胆碱受体→VTA DA释放 |
| **急性效应** | 警觉↑、焦虑↓、认知增强(短暂) |
| **戒断** | 烦躁、注意力差、食欲↑、渴望(持续数周到数月) |
| **成瘾性** | 所有物质中最高的——约67%使用者发展为依赖 |
| **致死** | 全球每年800+万死亡(含二手烟) |

### 2.3 阿片类 (Opioids)

| 物质 | μ受体亲和力 | 风险 |
|------|-----------|------|
| **海洛因** | 极高(经代谢为吗啡) | 过量→呼吸抑制→死亡 |
| **芬太尼** | μ受体强效(吗啡50-100×) | 极小量致死(2mg) |
| **处方阿片(羟考酮等)** | 中等 | iatrogenic成瘾 |
| **美沙酮** | 长半衰期μ激动剂 | 维持治疗 |
| **丁丙诺啡** | 部分μ激动+κ拮抗 | 封顶效应，过量风险低 |

**戒断**: 流涕/流泪→肌痛→腹泻→焦虑→失眠(持续7-10天)；不致死但极痛苦

### 2.4 大麻 (Cannabis)

| 维度 | 内容 |
|------|------|
| **主要靶点** | CB1受体(中枢)/CB2受体(外周) |
| **THC** | 精神活性成分→DA间接释放→欣快 |
| **CBD** | 非精神活性→抗焦虑/抗炎(潜在治疗) |
| **戒断** | 烦躁、失眠、食欲↓(轻度，持续1-2周) |
| **争议** | 9%使用者发展为依赖(青少年17%)；高THC品种→精神病风险↑ |

### 2.5 兴奋剂 (Stimulants)

| 物质 | 机制 | 特征 |
|------|------|------|
| **可卡因** | DAT阻断→突触DA↑↑ | 短效(30-60min)→强迫性复用 |
| **甲基苯丙胺(Meth)** | DAT逆转+VMAT2抑制→DA大量释放 | 长效(8-24h)、神经毒性 |
| **ADHD药物(滥用)** | 低剂量DAT/NET抑制 | 非处方使用时成瘾风险低 |

## 3. 成瘾的遗传与表观遗传

### 3.1 遗传度

| 物质 | 遗传度(h²) | 关键基因 |
|------|-----------|----------|
| 酒精 | 50-60% | ADH/ALDH、GABRA2、OPRM1 |
| 尼古丁 | 40-70% | CHRNA5-CHRNA3、CYP2A6 |
| 可卡因 | 30-60% | DRD2、DAT1、BDNF |
| 阿片 | 40-50% | OPRM1(A118G)、CYP2D6 |
| 大麻 | 30-50% | CNR1、FAAH |

### 3.2 表观遗传机制

- DNA甲基化：慢性用药→基因表达持久改变(如BDNF启动子超甲基化→BDNF↓)
- 组蛋白修饰：可卡因→伏隔核H3乙酰化增加→基因表达重塑
- 跨代传递：父代药物暴露→精子表观遗传改变→后代易感性↑(动物证据)

## 4. 成瘾的治疗生物学

### 4.1 药物治疗

| 物质 | 药物 | 机制 |
|------|------|------|
| **酒精** | 纳曲酮(Naltrexone) | μ阿片拮抗→减少渴求/欣快 |
| | 阿坎酸(Acamprosate) | 谷氨酸/GABA平衡→减少负性情感 |
| | 双硫仑(Disulfiram) | ALDH抑制→饮酒后不适(厌恶疗法) |
| **阿片** | 美沙酮(Methadone) | μ全激动→替代维持 |
| | 丁丙诺啡(Buprenorphine) | μ部分激动→封顶效应 |
| | 纳曲酮(长效) | μ拮抗→预防复发 |
| **尼古丁** | 尼古丁替代(NRT) | 渐进减量 |
| | 伐尼克兰(Varenicline) | α4β2部分激动 |
| | 安非他酮(Bupropion) | DA/NE再摄取抑制 |
| **大麻** | 无FDA批准药物 | 心理治疗为主；Nabiximols研究中 |

### 4.2 心理治疗与神经可塑性

| 干预 | 机制 |
|------|------|
| **动机访谈(MI)** | 增强内在改变动机→前额叶-纹状体回路激活 |
| **CBT** | 识别触发→应对策略→前额叶控制增强 |
| **正念复发预防(MBRP)** | 渴求觉察+不反应→岛叶-PFC功能改善 (Witkiewitz et al., 2014) |
| **列联管理(CM)** | 行为强化→纹状体奖赏回路重塑 |

## 5. 共病：成瘾与精神障碍

| 精神障碍 | 物质使用共病率 | 双向关系 |
|----------|-------------|----------|
| **抑郁症** | 30-50% | 自我药疗假说；戒断诱发抑郁 |
| **焦虑症** | 30-40% | 苯二氮卓/酒精短期缓解→长期恶化 |
| **PTSD** | 40-60% | 创伤→物质使用→创伤暴露↑ |
| **双相障碍** | 50-60% | 躁狂期物质使用↑；物质触发躁狂 |
| **精神分裂** | 40-50% | 大麻使用↑精神病风险(尤其青少年) |
| **ADHD** | 25-40% | 冲动性→早期物质使用→ADHD症状加重 |

| 工具 | 项目 | 用途 |
|------|------|------|
| **AUDIT** | 10项 | 酒精使用障碍筛查(≥8有风险) |
