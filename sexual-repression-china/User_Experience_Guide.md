## 🛠️ 实用工具使用指南 (Practical Tools Usage Guide)

### 评估工具使用流程

#### 个体评估操作步骤
```
准备工作:
1. 熟悉评估工具内容和使用说明
2. 准备评估环境和所需材料
3. 向被评估者说明评估目的和流程

实施评估:
1. 按照标准化流程进行评估
2. 记录评估过程中的重要信息
3. 确保评估的客观性和准确性

结果解释:
1. 根据评分标准进行结果判定
2. 结合个案具体情况综合分析
3. 制定个性化的干预建议
```

#### 团体评估注意事项
```
组织实施:
→ 确保团体成员的同质性
→ 营造安全和支持的团体氛围
→ 明确团体规则和保密原则

数据分析:
→ 使用适当的统计分析方法
→ 注意群体差异和个体特点
→ 结合理论框架进行结果解释
```
