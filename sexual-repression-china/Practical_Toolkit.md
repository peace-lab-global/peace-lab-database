## 📋 临床评估工具包 (Clinical Assessment Toolkit)

### 标准化评估量表

#### 性压抑程度评估量表 (Sexual Repression Scale)

**量表结构**:
```
性压抑程度评估量表 (SRS-Chinese Version)
共5个维度，30个条目，5点李克特量表(1-5分)

维度1: 性表达抑制 (Items 1-6)
维度2: 性焦虑体验 (Items 7-12)
维度3: 性羞耻认知 (Items 13-18)
维度4: 性功能障碍 (Items 19-24)
维度5: 社会适应困难 (Items 25-30)
```

**使用指南**:
- **适用人群**: 18岁以上成年人
- **施测时间**: 15-20分钟
- **评分标准**: 总分30-150分，分数越高压抑程度越重
- **结果解释**:
  - 30-60分: 轻度性压抑
  - 61-90分: 中度性压抑
  - 91-120分: 重度性压抑
  - 121-150分: 极重度性压抑

#### 创伤史筛查问卷 (Trauma History Screening Questionnaire)

**核心条目**:
```
创伤史筛查问卷 (THSQ)
共12个核心问题，是/否回答

A. 家庭环境创伤
1. 儿童期是否经历过性相关的不当对待？
2. 是否目睹过家庭中的性暴力事件？
3. 家长是否对性话题采取严厉禁止态度？

B. 教育环境创伤
4. 在学校是否遭受过性相关的羞辱或惩罚？
5. 是否因为性相关行为受到过严重的纪律处分？

C. 社会文化创伤
6. 是否因为性取向或性别认同遭受过社会歧视？
7. 是否在职场中经历过性骚扰或性歧视？

D. 数字环境创伤
8. 是否在网络环境中遭遇过性相关的网络暴力？
9. 是否因为性相关内容被不当曝光而遭受困扰？

E. 亲密关系创伤
10. 是否在恋爱或婚姻关系中经历过性控制或性暴力？
11. 是否因为性问题导致重要关系破裂？

F. 自我认同创伤
12. 是否对自己的性身份或性取向感到深度困扰？
```

### 临床访谈指南

#### 结构化访谈提纲

**第一阶段: 建立关系 (10分钟)**
```
1. 自我介绍和保密承诺
2. 解释访谈目的和流程
3. 确认来访者舒适度和安全感受
4. 建立治疗联盟和信任关系
```

**第二阶段: 问题探索 (20分钟)**
```
1. 性发展史采集
   - 早期性认知和体验
   - 关键发展节点和转折事件
   - 重要他人影响(父母、同伴、老师)

2. 压抑体验评估
   - 具体压抑情境描述
   - 情绪反应和身体感受
   - 应对策略和适应方式

3. 影响范围分析
   - 对亲密关系的影响
   - 对工作学习的影响
   - 对身心健康的影响
```

**第三阶段: 目标设定 (10分钟)**
```
1. 共同制定治疗目标
2. 商讨治疗计划和时间安排
3. 确定后续跟进机制
```
