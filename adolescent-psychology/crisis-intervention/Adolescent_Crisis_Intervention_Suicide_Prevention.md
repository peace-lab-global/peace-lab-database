# 青少年危机干预与自杀预防专业指南

## ⚠️ 青少年危机风险识别

### 危机预警信号系统

#### 自杀意念识别指标

##### 言语警告信号
```
直接表达类：
• "我不想活了"、"活着没意思"
• "死了算了"、"一了百了"
• "这个世界没有我更好"
• "我再也受不了这种痛苦了"

间接暗示类：
• "我累了"、"我受够了这一切"
• "如果我不在了会怎样"
• "我做了安排"、"一切都结束了"
• "你们要好好照顾自己"

绝望表达类：
• "没有任何希望"、"一切都完了"
• "没有人理解我"、"没有人关心我"
• "我是个负担"、"我不值得被爱"
• "我永远无法摆脱这些问题"
```

##### 行为变化预警
```
社交行为改变：
• 突然的社交退缩
• 与朋友和家人疏远
• 取消之前的计划和承诺
• 赠送珍贵物品给他人

日常习惯变化：
• 睡眠模式显著改变
• 食欲明显增加或减少
• 个人卫生习惯恶化
• 学业表现急剧下降

危险行为增加：
• 自伤行为频率上升
• 寻找自杀方法和工具
• 酒精或药物滥用增加
• 参与高风险活动
```

#### 危险因素评估框架

##### 个体风险因素
```
心理健康史：
• 既往抑郁或焦虑病史
• 既往自杀企图史
• 精神疾病诊断史
• 心理创伤经历

人格特征：
• 冲动性人格特质
• 完美主义倾向
• 悲观思维模式
• 低自尊水平

应对能力：
• 问题解决技能缺乏
• 情绪调节困难
• 社会支持网络薄弱
• 应对资源匮乏
```

##### 环境风险因素
```
家庭环境：
• 家庭冲突和暴力
• 父母离异或丧亲
• 家庭成员自杀史
• 缺乏家庭支持和理解

学校环境：
• 学业压力过大
• 校园欺凌经历
• 师生关系紧张
• 同伴关系问题

社会环境：
• 经济困难和贫困
• 社会歧视和排斥
• 文化和宗教压力
• 媒体和网络影响
```

### 危机严重程度分级

#### 风险等级评估

##### 一级风险（紧急危险）
```
立即危险指标：

明确自杀计划：
• 具体的自杀方法和时间安排
• 获取了实施自杀的工具或手段
• 选择了特定的地点和时机
• 向他人透露了详细计划

高度准备状态：
• 已经开始了自杀行为的准备
• 表现出立即实施的意图
• 情绪极度绝望和无助
• 拒绝接受任何形式的帮助
```

##### 二级风险（高度危险）
```
严重风险表现：

强烈的自杀意念：
• 频繁出现死亡和自杀想法
• 对自杀方法有详细了解
• 表达强烈的死亡愿望
• 认为自己无法继续生存

功能严重受损：
• 日常生活功能严重受损
• 无法维持基本的自我照顾
• 社会关系完全破裂
• 学业和工作能力丧失
```

##### 三级风险（中度危险）
```
中等风险特征：

持续的情绪困扰：
• 长期的抑郁和绝望情绪
• 反复出现的自杀念头
• 情绪调节能力严重不足
• 应对压力的能力下降

部分功能受损：
• 学业和社交功能受到影响
• 日常生活有一定困难
• 需要额外的支持和帮助
• 存在一定的保护性因素
```

## 🛡️ 危机干预核心原则

### 干预基本原则

#### 立即安全性优先
```
首要干预目标：

生命安全保障：
• 立即消除自杀风险
• 确保当事人人身安全
• 移除可及的危险物品
• 建立24小时监护

环境安全控制：
• 创造安全的物理环境
• 限制独处时间和空间
• 建立紧急联系网络
• 制定详细的监护计划
```

#### 专业合作网络
```
多学科团队协作：

专业人员组成：
• 精神科医生和心理医生
• 危机干预专家
• 社会工作者
• 学校心理咨询师

协作工作机制：
• 信息共享和协调
• 统一的干预策略
• 定期的进展评估
• 持续的专业督导
```

### 干预技术体系

#### 危机评估技术

##### 结构化评估工具
```
标准化评估量表：

哥伦比亚自杀严重程度评定量表(C-SSRS)：
• 自杀意念频率和强度
• 自杀行为历史和计划
• 危险因素和保护因素
• 即刻风险评估

贝克自杀意念量表(BSI)：
• 主动自杀欲望评估
• 自杀计划完备程度
• 自杀准备状态检查
• 阻止因素识别
```

##### 临床访谈技巧
```
评估性会谈要点：

建立治疗联盟：

• 创造安全和信任的氛围
• 使用开放式提问技术
• 积极倾听和共情回应

风险评估内容：
• 自杀想法的性质和频率
• 自杀计划的具体程度
• 可获得的自杀手段
• 保护性因素和支持系统
```

```
现实检验：

• 回顾过往的成功应对经验
```

```
情绪急救方法：

情感验证：

• 表达对其痛苦的理解和同情
• 避免评判和说教式的安慰
• 提供无条件的积极关注

情绪调节：
```
