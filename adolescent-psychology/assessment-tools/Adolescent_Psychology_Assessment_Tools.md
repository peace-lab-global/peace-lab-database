# 青少年心理学评估工具专业指南

## 📊 青少年评估体系概述

青少年心理学评估需要考虑独特的身心发展特点、社会文化影响和风险因素。本指南提供专门针对青少年的评估工具和方法，涵盖心理健康筛查、危机风险评估、身份认同测量等方面。

## 🔍 心理健康筛查工具

### 抑郁与焦虑评估

#### 患者健康问卷-青少年版 (PHQ-A)

##### 评估结构与内容
```
核心症状维度：

抑郁核心症状：
• 情绪低落和兴趣丧失
• 睡眠障碍和疲劳感
• 自我评价低下和注意力问题
• 自杀意念和行为倾向

焦虑相关症状：
• 紧张不安和担心
• 躯体化焦虑症状
• 恐慌发作倾向
• 社交焦虑表现
```

##### 量表项目构成
| 项目 | 内容 | 评估频率 | 评分标准 |
|------|------|----------|----------|
| **兴趣丧失** | 对平时喜欢的事情失去兴趣 | 过去2周 | 0-3分 |
| **情绪低落** | 感到沮丧、忧郁或绝望 | 过去2周 | 0-3分 |
| **睡眠问题** | 睡眠困难或睡眠过多 | 过去2周 | 0-3分 |
| **疲劳乏力** | 感到疲倦或精力不足 | 过去2周 | 0-3分 |
| **食欲变化** | 食欲不振或暴饮暴食 | 过去2周 | 0-3分 |
| **自我否定** | 觉得自己很糟或让自己失望 | 过去2周 | 0-3分 |
| **专注困难** | 注意力难以集中 | 过去2周 | 0-3分 |
| **行动迟缓** | 动作缓慢或焦躁不安 | 过去2周 | 0-3分 |
| **自杀想法** | 有自伤或自杀的想法 | 过去2周 | 0-3分 |

##### 临床应用要点
```
筛查标准：

轻度抑郁：
• 总分≥10分
• 至少5个项目≥1分
• 无自杀风险

中度抑郁：
• 总分≥15分
• 功能损害明显
• 需要进一步评估

重度抑郁：
• 总分≥20分
• 或有自杀风险
• 需要紧急干预
```

#### 广泛性焦虑障碍量表-青少年版 (GAD-7-A)

##### 焦虑症状评估
```
焦虑核心维度：

过度担忧：
• 对多种事情的持续担心
• 难以控制的忧虑
• 担心的频率和强度

躯体焦虑：
• 肌肉紧张和不安
• 易疲劳和注意力困难
• 睡眠障碍和易激惹

功能影响：
• 日常活动受限
• 学业表现下降
• 社交回避行为
```

##### 评估项目设置
| 项目 | 症状内容 | 时间范围 | 严重程度 |
|------|----------|----------|----------|
| **担心过度** | 担心很多事情 | 过去2周 | 0-3分 |
| **控制困难** | 无法停止或控制担心 | 过去2周 | 0-3分 |
| **紧张不安** | 感到紧张、焦急或不安 | 过去2周 | 0-3分 |
| **放松困难** | 很难放松 | 过去2周 | 0-3分 |
| **焦躁不安** | 坐立不安，很难保持不动 | 过去2周 | 0-3分 |
| **易怒烦躁** | 容易生气或易怒 | 过去2周 | 0-3分 |
| **恐惧担心** | 感到害怕，好像即将有可怕的事情发生 | 过去2周 | 0-3分 |

### 自杀风险评估工具

#### 哥伦比亚自杀严重程度评定量表 (C-SSRS-A)

##### 评估核心要素
```
自杀意念评估：

被动自杀意念：
• 希望死去但无具体计划
• 认为死了会更好
• 对死亡的模糊愿望

主动自杀意念：
• 有自杀的具体想法
• 考虑自杀的方法
• 制定自杀的计划

自杀行为评估：
• 自杀未遂历史
• 自杀准备行为
• 自杀威胁和姿态
```

##### 评估流程设计
| 阶段 | 评估内容 | 询问方式 | 记录要点 |
|------|----------|----------|----------|
| **筛查** | 是否有过自杀想法 | 开放式询问 | 具体时间、频率 |
| **深入** | 自杀想法的性质 | 结构化探询 | 意念强度、持续时间 |
| **风险** | 自杀计划和准备 | 直接询问 | 方法、时间、手段 |
| **历史** | 既往自杀行为 | 详细回顾 | 次数、严重程度、结果 |

##### 风险等级判定
| 等级 | 标准 | 干预级别 | 后续措施 |
|------|------|----------|----------|
| **低风险** | 仅有被动意念 | 观察随访 | 定期监测 |
| **中风险** | 主动意念无计划 | 加强关注 | 增加支持 |
| **高风险** | 有具体计划 | 紧急干预 | 专业治疗 |
| **极高风险** | 准备实施行为 | 立即处理 | 危机干预 |

#### 青少年自杀风险评估量表 (SAD PERSONS)

##### 风险因素评估
```
十个核心风险因子：

S - Sex (性别)：男性自杀成功率更高
A - Age (年龄)：青少年期自杀风险增加
D - Depression (抑郁)：抑郁症状是主要风险因子
P - Previous attempt (既往自杀史)：最强预测因子
E - EtOH/Substance abuse (酒精/物质滥用)：增加冲动性
R - Rational thinking loss (理性思维丧失)：判断力受损
S - Social supports lacking (缺乏社会支持)：孤立无援
O - Organized plan (有组织的计划)：预谋性强
N - No spouse (无配偶)：缺乏亲密关系支持
S - Sickness (疾病)：慢性疾病增加风险
```

##### 评分与干预建议
| 风险等级 | 得分 | 特征 | 干预策略 |
|----------|------|------|----------|
| **低风险** | 0-2分 | 风险因素较少 | 常规心理健康服务 |
| **中风险** | 3-5分 | 多个风险因素 | 加强监测和预防 |
| **高风险** | 6-8分 | 高度危险状态 | 专业评估和治疗 |
| **极高风险** | 9-10分 | 紧急危险状况 | 立即危机干预 |

##### 状态分类系统

|----------|----------|----------|----------|

## 🧪 特殊问题评估工具

### 网络使用与成瘾评估

#### 青少年网络成瘾量表 (YIAS)

##### 成瘾行为特征
```
核心成瘾症状：

耐受性表现：
• 需要不断增加上网时间
• 对网络内容的需求升级
• 停止使用的困难增加

戒断症状：
• 停止上网时的焦虑不安
• 渴望重新上网的强烈冲动
• 情绪低落和烦躁易怒

功能损害：
• 学业成绩明显下降
• 社交活动显著减少
• 日常作息严重紊乱
```

##### 评估维度构成
| 维度 | 评估内容 | 项目数 | 信度系数 |
|------|----------|--------|----------|
| **强迫性使用** | 无法控制的上网冲动 | 6题 | α=0.82 |
| **戒断反应** | 停止使用时的不适症状 | 5题 | α=0.78 |
| **耐受性增加** | 需要更多时间满足需求 | 4题 | α=0.75 |
| **功能损害** | 对生活各方面的负面影响 | 7题 | α=0.85 |
| **时间管理** | 上网时间控制困难 | 3题 | α=0.72 |

```
认知重评：
```

| 策略类型 | 适应性效果 | 使用情境 | 发展特点 |

## 📊 综合评估实施框架

```
三个核心阶段：

初步筛查阶段：
• 快速识别高风险个体
```
