##### 身体认知类活动 (Body Awareness Activities)

```
活动1: 身体探险家游戏 (Body Explorer Game)
适用年龄: 5-8岁
活动目标: 准确认识身体部位及其功能
所需时间: 20-25分钟
参与人数: 6-12人小组

活动材料:
• 人体模型或大幅人体图
• 彩色贴纸(红、黄、蓝、绿)
• 身体部位卡片
• 小镜子(每人一面)
• 活动记录表

活动流程:
1. 导入环节(3分钟)
   • 歌曲导入:"头肩膀膝盖脚"
   • 提问互动:你知道身体有哪些部位吗？

2. 探索环节(10分钟)
   • 分组观察人体模型
   • 使用镜子观察自己的身体
   • 贴纸标记认识的身体部位
   • 教师讲解各部位功能

3. 巩固环节(8分钟)
   • 身体部位卡片配对游戏
   • 小组竞赛:看谁说得又快又准
   • 制作个人身体认知图

4. 总结环节(4分钟)
   • 分享学习收获
   • 强调身体保护重要性
   • 布置家庭延伸活动

评估要点:
✓ 身体部位识别准确率≥80%
✓ 能说出2-3个身体部位功能
✓ 积极参与活动，乐于分享
✓ 初步建立身体保护意识

变式调整:
• 年龄较小(5-6岁): 增加歌曲和游戏比重
• 年龄较大(7-8岁): 增加功能认知深度
• 有特殊需求: 提供触觉模型和辅助材料
```

##### 安全教育类活动 (Safety Education Activities)

```
活动2: 安全守护者训练营 (Safety Guardian Training Camp)
适用年龄: 6-10岁
活动目标: 识别安全/不安全情境，掌握求助技能
所需时间: 30-35分钟
参与人数: 8-15人小组

活动材料:
• 安全/危险情境图片卡(20张)
• 安全守护者徽章(制作材料)
• 求助电话卡片
• 安全计划制定表
• 情境角色扮演道具

活动流程:
1. 情境认知(8分钟)
   • 展示各种情境图片
   • 分类讨论:安全vs危险
   • 总结安全情境特征

2. 技能训练(12分钟)
   • 求助电话使用练习
   • 安全求助话语训练
   • 紧急情况应对演练

3. 角色扮演(10分钟)
   • 分组进行情境模拟
   • 练习拒绝和求助技能
   • 同伴反馈和改进建议

4. 承诺仪式(5分钟)
   • 制作安全守护者徽章
   • 签署安全承诺书
   • 分享个人安全计划

评估要点:
✓ 能准确识别80%以上的安全/危险情境
✓ 掌握至少2种求助方式
✓ 能清楚表达拒绝和求助
✓ 建立基本的安全防护意识

注意事项:
• 营造安全支持的活动氛围
• 及时处理孩子可能出现的焦虑情绪
• 强调寻求帮助是勇敢的表现
• 提供后续支持和跟进服务
```

##### 关系技能类活动 (Relationship Skills Activities)

```
活动3: 健康关系工作坊 (Healthy Relationship Workshop)
适用年龄: 13-16岁
活动目标: 识别健康关系特征，培养沟通技能
所需时间: 45-50分钟
参与人数: 10-20人小组

活动材料:
• 健康/不健康关系特征卡片
• 沟通技巧练习情境卡
• 关系温度计评估工具
• 边界设定练习材料
• 小组讨论记录表

活动流程:
1. 特征识别(12分钟)
   • 头脑风暴:什么是健康关系？
   • 卡片分类:健康vs不健康特征
   • 小组讨论和分享观点

2. 沟通练习(15分钟)
   • 积极倾听技巧训练
   • "我"语句表达练习
   • 非暴力沟通模拟

3. 边界设定(12分钟)
   • 个人边界识别练习
   • 边界协商角色扮演
   • 尊重他人边界训练

4. 应用反思(6分钟)
   • 关系现状自我评估
   • 制定关系改善计划
   • 同伴支持网络建立

评估要点:
✓ 能识别健康关系的核心特征
✓ 掌握基本的沟通技巧
✓ 理解边界设定的重要性
✓ 具备初步的关系问题处理能力

进阶版本:
• 增加冲突解决策略训练
• 引入数字时代关系挑战
• 添加长期关系维护技能
• 包含家庭关系沟通技巧
```

##### 价值澄清类活动 (Value Clarification Activities)

```
活动4: 价值观拍卖会 (Values Auction)
适用年龄: 14-18岁
活动目标: 澄清个人核心价值观，培养批判思维
所需时间: 40-45分钟
参与人数: 8-15人小组

活动材料:
• 价值观卡片(50张不同价值观)
• 虚拟货币(筹码或点数)
• 拍卖锤或计时器
• 个人价值观反思表
• 小组讨论引导问题

活动流程:
1. 价值认知(8分钟)
   • 介绍拍卖规则和流程
   • 展示价值观卡片示例
   • 讨论价值观的含义和重要性

2. 个人反思(10分钟)
   • 独立浏览所有价值观卡片
   • 思考哪些对自己最重要
   • 初步确定个人价值优先级

3. 拍卖环节(15分钟)
   • 模拟拍卖会进行
   • 每人有一定数量虚拟货币
   • 竞拍自己认为重要的价值观
   • 记录竞拍结果和花费

4. 分享讨论(12分钟)
   • 分享竞拍策略和思考过程
   • 讨论价值观冲突和权衡
   • 反思个人价值体系
   • 制定价值实践行动计划

评估要点:
✓ 能清晰表达个人核心价值观
✓ 理解价值观冲突和选择困难
✓ 具备价值澄清和排序能力
✓ 初步形成个人价值体系

深度延伸:
• 价值观与行为一致性分析
• 文化背景对价值观的影响
• 价值观在人生决策中的作用
• 价值体系的动态发展过程
```

```
青少年期技能观察量表(13-18岁):

观察维度1: 沟通表达能力
评估指标:
• 表达清晰度(语言组织能力)
• 倾听专注度(注意力集中表现)
• 情感表达准确性(情绪传达效果)

• 小组讨论参与度观察

• 问题识别准确性
• 信息收集全面性
• 风险评估合理性
• 选择论证充分性

观察方法:
• 案例分析表现观察
• 决策过程记录分析
• 问题解决策略评估
• 后果预见能力判断

评分标准参照上述等级体系
```

#### 危机处理操作手册 (Crisis Management Operations Manual)

```
紧急情况处理流程:

1. 即时响应程序
识别信号:
• 学生行为异常表现
• 情绪状态剧烈变化
• 直接或间接求助表达
• 同伴报告相关信息

初步评估:
• 判断紧急程度和风险等级
• 确定是否需要立即干预
• 评估是否涉及法律问题
• 决定是否需要专业转介

2. 干预实施步骤
安全保护:
• 确保当事人人身安全
• 提供情感支持和安慰
• 联系相关专业人员
• 通知必要的情况下家长

信息收集:
• 详细了解事件经过
• 收集相关证据材料
• 记录关键时间节点
• 保护当事人隐私信息

资源协调:
• 联系学校心理咨询服务
• 协调医疗健康资源
• 沟通家长和监护人
• 必要时联系外部专业机构

3. 后续跟进支持
短期支持:
• 持续情感关怀陪伴
• 学习和生活适应帮助
• 同伴支持网络建立
• 定期状况跟踪了解

长期规划:
• 制定个人发展计划
• 建立长效支持机制
• 促进身心健康发展
• 预防类似问题再发生

记录归档:
• 详细事件处理记录
• 干预措施效果评估
• 经验教训总结反思
• 案例资料妥善保管
```
