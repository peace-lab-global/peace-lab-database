## 核心术语表 (Core Terminology)

| 术语 (中文) | 英文名 (English Term) | 原始语言/术语 (Original Language) | 精确定义 (Technical Definition) | 区分与备注 (Differentiation & Notes) | 专业视角 (Professional Perspective) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **恋物癖** | Fetishism | 希腊: *Fetishismos* (偶像崇拜) | 以非生命物体或身体特定部位为主要性唤起源，且这种唤起模式持续至少6个月并对日常生活造成显著困扰或损害。 | 不同于正常的性偏好，恋物癖达到病理程度，符合DSM-5诊断标准。 | 需要区分适应性性偏好与病理性性偏离，关注功能损害程度。 |
| **性偏离障碍** | Paraphilic Disorders | 希腊: *Para* (异常) + *Philos* (爱) | 一类以非典型性唤起模式为特征的精神障碍，包括恋物癖、露阴癖、窥阴癖等多种类型。 | 包含paraphilia(性偏离倾向)和paraphilic disorder(性偏离障碍)两个概念层次。 | 现代诊断强调痛苦(distress)和损害(harm)双重标准。 |
| **性唤起模式** | Sexual Arousal Pattern | 英: Sexual Arousal Pattern | 个体在性活动中产生性兴奋和性快感的特定方式和触发条件组合。 | 包括视觉、触觉、听觉、嗅觉等多种感官通道的整合激活。 | 具有个体差异性，在正常范围与病理范围之间存在连续谱。 |
| **功能损害** | Functional Impairment | 英: Functional Impairment | 恋物行为对个体的社会功能、职业表现、人际关系或心理健康造成的负面影响。 | 是区分正常性偏好与病理性恋物癖的关键诊断标准之一。 | 需要综合评估多个生活领域的功能状态。 |
| **性偏离倾向** | Paraphilia | 英: Paraphilia | 个体表现出非典型的性唤起模式，但尚未达到造成显著痛苦或损害的程度。 | 与paraphilic disorder(性偏离障碍)相区别，后者必须伴有临床显著的痛苦或功能损害。 | 研究中常作为连续变量处理，而非绝对的类别划分。 |

## 恋物癖分类体系表 (Classification System)

| 分类维度 | 子类别 | 典型对象 | 核心特征 | 临床意义 | 研究现状 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **身体部位类** | 恋足癖 | 足部、脚趾 | 对足部形态、气味、触感的强烈性唤起 | 最常见的恋物类型之一 | 研究较为充分，有明确的神经机制假说 |
| | 恋手癖 | 手部、手指 | 对手部动作、触感、形状的关注 | 常与其他恋物类型共存 | 临床案例相对较少 |
| | 恋发癖 | 头发、发丝 | 对头发质地、颜色、长度的痴迷 | 多见于女性恋物者 | 社会接受度相对较高 |
| **服饰配件类** | 恋高跟鞋 | 高跟鞋、鞋跟 | 对高跟鞋形状、材质、穿着者的联想 | 具有明显的性别指向性 | 与权力、性感的社会建构相关 |
| | 恋丝袜 | 丝袜、连裤袜 | 对丝袜质感、光泽、包裹感的关注 | 常伴随恋足或其他腿部恋物 | 涉及触觉和视觉双重刺激 |
| | 恋内衣 | 内衣、睡衣 | 对贴身衣物的材质、款式、穿着状态的迷恋 | 多与私密性、禁忌感相关 | 临床意义相对较轻 |
| **年龄相关类** | 恋童癖 | 前青春期儿童 | 对未成年个体的性吸引(病理形式) | 严重的刑事违法行为 | 需要严格法律监管和治疗干预 |
| | 恋萝莉 | 幼态化成人 | 对具有幼态特征的成人的性吸引 | 存在法律灰色地带 | 涉及审美偏好与伦理边界的复杂问题 |
| **物品材质类** | 恋皮革 | 皮革制品 | 对皮革质地、气味、光泽的性唤起 | 常与BDSM文化相关 | 具有一定的亚文化认同 |
| | 恋橡胶 | 橡胶制品 | 对橡胶材质、光滑表面的迷恋 | 多见于特殊性癖好群体 | 涉及感官刺激的独特性 |

| :--- | :--- | :--- | :--- | :--- | :--- |
| **总体患病率** | 社区调查(n=2,424) | 成年男性，平均年龄36岁 | 恋物癖终生患病率约15-25% | 自报数据可能存在低估 | 大部分为亚临床水平，不构成障碍 |

| **年龄特征** | 纵向追踪研究 | 青少年至中老年 | 恋物倾向通常在青春期(12-16岁)形成 | 回忆偏倚，缺乏早期数据 | 关键发展期的神经可塑性影响 |
| **共病情况** | 临床流行病学 | 恋物障碍患者 | 抑郁症(45%)、焦虑症(38%)、物质滥用(28%) | 横断面研究，因果关系不明 | 需要综合治疗方案 |
