# 佛教禅修实践体系详论 (Buddhist Meditation Practice System)

> **佛教禅修 (Buddhist Meditation Practice)**：通过系统化的修行方法，培养心的安定与智慧，最终达到解脱和觉悟的目标。涵盖止禅、观禅、生活禅等多种修习方式。

> *"修习止观，如车二轮，如鸟二翼，缺一不可。" —— 《大智度论》*

> *梵语：ध्यान (Dhyāna) / भावना (Bhāvanā)*
> *巴利语：झान (Jhāna) / भावना (Bhāvanā)*
> *藏语：བསྒོམ་པ (BsGom-pa) / སྒོམ་ཚུལ (BsGom-tshul)*

> 佛教禅修是连接理论与实践的桥梁，是将佛教智慧转化为生命体验的核心方法。现代科学研究已证实禅修对身心健康具有显著的积极影响。
