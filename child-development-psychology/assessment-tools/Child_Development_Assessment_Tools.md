# 儿童发展心理学评估工具专业指南

## 📊 评估工具体系概述

儿童发展心理学评估是识别发展需求、制定干预计划和监测进展的核心环节。本指南整合了标准化评估工具、发展筛查方法和诊断性评估技术，为专业实践提供科学依据。

## 🔍 标准化评估量表

### 智力与认知能力评估

#### 韦氏儿童智力量表第五版 (WISC-V)

##### 评估结构体系
```
核心指数构成：

言语理解指数 (VCI)：
• 词汇理解能力
• 抽象概念掌握
• 语言推理水平
• 知识应用能力

知觉推理指数 (PRI)：
• 非言语推理能力
• 视觉空间处理
• 图形分析技能
• 抽象问题解决

工作记忆指数 (WMI)：
• 短时记忆保持
• 信息处理速度
• 注意力维持能力
• 认知灵活性

加工速度指数 (PSI)：
• 视觉运动协调
• 反应速度
• 注意力集中度
• 执行效率

全量表智商 (FSIQ)：
• 综合认知能力
• 整体智力水平
• 发展潜力评估
• 学业成就预测
```

##### 适用年龄与信效度
| 指标 | 数值 | 说明 |
|------|------|------|
| **适用年龄** | 6-16岁 | 分为6-7岁、8-9岁、10-11岁、12-13岁、14-16岁五个年龄段 |
| **重测信度** | 0.85-0.92 | 各指数重测相关系数 |
| **内部一致性** | 0.80-0.90 | 分半信度和α系数 |
| **效标效度** | 0.70-0.85 | 与学业成就、教师评定的相关 |

```
评估实施注意事项：

```
