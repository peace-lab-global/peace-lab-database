# Fasting Therapy Overview (断食疗法概览)

### Core Terminology of Fasting (断食核心术语总览表)
| 术语 (中文) | 英文名 (English Term) | 原始语言/术语 (Sanskrit/Arabic/Latin) | 精确定义 (Technical Definition) | 区分与备注 (Differentiation & Notes) | 专业视角 (Professional Perspective) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **断食** | Fasting | 梵: **Upavāsa** (उपवास) | 在限定时间内自愿停止能量性食物与含热量饮品的摄入，以触发代谢与精神层面的重组。 | 与"节食"不同，断食强调时间窗口与代谢状态，而非仅仅减少总热量。 | 应视为一种代谢干预与心智训练工具，而不是惩罚性节食。 |
| **间歇性断食** | Intermittent Fasting (IF) | - | 周期性在进食与禁食之间切换的总称，包括16:8、18:6、5:2等模式。 | 不是单一方法，而是一系列时间结构的家族概念。 | 对现代久坐人群最可行的长期基础方案，有丰富临床与流行病学数据支持。 |
| **时间限制进食** | Time-Restricted Feeding (TRF) | - | 在24小时内将进食限制在固定窗口（如8小时内），不必刻意降低总热量。 | TRF 是 IF 的一个子类，强调昼夜节律 (Circadian Rhythm)。 | 更接近生物钟医学，将“何时吃”置于与“吃什么”同等重要的地位。 |
| **隔日断食** | Alternate-Day Fasting (ADF) | - | 进食日与极低热量日/完全断食日交替出现（如一日正常、一日500 kcal）。 | 与 5:2 饮食在结构上相近，但频率更高，依从性要求更严。 | 更适合在专科门诊中短期使用，不建议非专业人群长期自行实施。 |
| **完全禁食** | Absolute Fasting | 拉: **Ieiunium** | 在特定时间内完全不摄入任何食物或含热量饮品，仅可饮水或在宗教中甚至连水都限制。 | 医疗语境下通常允许饮水；“干断食”属高风险极端形式。 | 对大多数现代人来说，>48 小时的完全禁食应置于严密医疗监护之下。 |
| **辟谷** | Bigu | 中: 辟谷；道教术语 | 通过气功、导引、服气与意念训练，减少甚至停绝五谷摄入，以“气”代“谷”。 | 传统文本强调“食气”、“服气”，并非简单不吃饭。 | 更接近“能量效率训练”和精神修炼，不应被简化为减肥方法。 |
| **过午不食** | No Solid Food After Noon | 巴利/梵: **Vikāla-bhojana** | 比丘戒律中，在正午之后不再进食固体食物，仅少量饮品，为长期持守的日常戒律。 | 与短期减脂类 IF 不同，其目标是约束贪欲与支持禅修。 | 在在家众中可作为温和的 TRF 形式，但需兼顾体力与工作需求。 |
| **斋戒（宗教）** | Fasting as Ṣawm | 阿: **Ṣawm** (صوم) | 以伊斯兰斋月为代表，从日出到日落禁止饮食、吸烟及部分感官欲望。 | 进食窗口集中在夜间，与代谢最佳节律并不完全一致。 | 强调“意向与虔敬”的心灵重塑，对代谢的利弊与作息质量高度相关。 |
| **Ekādaśī 斋日** | Ekādaśī Fasting | 梵: **Ekādaśī** (एकादशी) | 印度教与部分瑜伽传统在阴历每月第11天进行的断食或简化饮食。 | 常配合持咒、礼拜与静心练习。 | 兼具天文周期观与身心净化逻辑，适合理解为“精神与代谢同步调谐日”。 |

### Typology of Fasting Modes (断食模式分型与适配表)
| 分型维度 | 子类型名称 (中文) | English Name | 典型时间结构 (Fasting:Feeding) | 能量/营养规则 | 主要目标 (Primary Goal) | 适用人群 | 专业视角 (Professional Perspective) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **按时间结构** | 14:10 轻度时间窗 | 14:10 TRF | 连续禁食 14 小时，进食窗 10 小时。 | 不刻意限量，只限制进食时间。 | 温和改善胰岛素敏感性与睡眠。 | 初学者、体重正常但有代谢风险者。 | 常作为所有断食计划的入门级“适应期”，降低失败率。 |
| | 16:8 经典模式 | 16:8 IF | 禁食 16 小时，进食窗 8 小时。 | 建议在进食窗内 2–3 餐，避免持续性零食。 | 减脂、改善代谢综合征。 | BMI 略高、生活节奏规律者。 | 目前证据最丰富的模式之一，可视作“代谢重置”的基础方案。 |
| | 18:6 深度时间窗 | 18:6 IF | 禁食 18 小时，进食窗 6 小时。 | 需提高每餐营养密度与蛋白质比例。 | 强化脂肪动员与生酮。 | 有一定 IF 经验者、胰岛素抵抗明显者。 | 建议配合阻力训练与电解质补充，以防肌肉流失与低血压。 |
| **按频率与周期** | 5:2 轻断食 | 5:2 Diet | 每周 5 天正常饮食，2 天控制在 500–600 kcal。 | 断食日以高蛋白蔬菜为主。 | 体重管理与代谢缓慢纠正。 | 工作节奏难以长时间 TRF 者。 | 依从性较好，但易在“正常日”过量补偿，需要心理与行为策略。 |
| | 隔日断食 | Alternate-Day Fasting | 正常日与≤25% 热量日交替。 | 断食日最多 1 轻餐。 | 强力减脂与脂肪肝逆转。 | 有强动机、无进食障碍史者。 | 适合作为医师监督下的短期强化工具，而非长期生活方式。 |
| **按允许摄入** | 水断食 | Water-Only Fasting | 禁食期间仅饮水与电解质。 | 0 kcal；可补充电解质与少量微量元素。 | 深度自噬与免疫重启。 | 健康成人，在评估后短期实施。 | >72 小时水断食属于“高剂量干预”，应像药物一样严肃看待。 |
| | 模拟断食饮食 | Fasting-Mimicking Diet (FMD) | 连续 4–5 天极低热量植物性饮食。 | 约为基础代谢的 30–50%，多为植物脂肪与纤维。 | 模拟水断食的自噬与抗衰老效果。 | 代谢综合征、家族肿瘤风险人群。 | 以 Valter Longo 研究为代表，适合在功能医学或营养门诊中周期实施。 |
| | 宗教斋戒 | Religious Fasting | 依传统而定，如日出到日落不食。 | 允许水与特定饮品，通常不限夜间进食量。 | 精神净化、群体认同。 | 有宗教背景的实践者。 | 若想兼顾健康，应特别关注夜间进食质量与睡眠卫生。 |

### Goals & Mechanisms Overview (断食目标与机制一览表)
| 目标领域 (Domain) | 主要目标 (Primary Goals) | 关键机制 (中文/English) | 推荐模式 | 监测指标 (Biomarkers) | 专家建议 (Expert Tips) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **体重与组成** | 降低体脂、保持肌肉。 | 能量负平衡 (Energy Deficit)、提高代谢灵活性 (Metabolic Flexibility)。 | 16:8 IF + 每周 2–3 次阻力训练。 | 体重、腰围、体脂率、主观精力。 | 相比极端节食，以断食+高蛋白+力量训练更容易保护瘦体重。 |
| **血糖与胰岛素** | 改善胰岛素敏感性，逆转早期 2 型糖尿病。 | 减少胰岛素分泌负担；AMPK 激活；GLUT4 上调。 | 医师监督下的 16:8 或 18:6 + 碳水集中在进食窗前半段。 | 空腹血糖、空腹胰岛素、HOMA-IR、糖化血红蛋白 (HbA1c)。 | 药物（尤其胰岛素与磺脲类）需动态调整剂量，严禁自行停药。 |
| **心血管与血脂** | 降低甘油三酯与小而密 LDL，提升 HDL。 | 减少肝内脂肪；提升脂蛋白代谢效率。 | 周期性 FMD 或 5:2，结合抗炎饮食模式。 | 血脂谱、血压、超声评估肝脂肪。 | 比起单纯降脂药物，断食可作为“生活方式药物”的核心组成，但不是替代品。 |
| **脑健康与情绪** | 提升认知清晰度与情绪稳定。 | BDNF 增加；酮体 (Ketone Bodies) 提供清洁燃料；炎症因子下降。 | 14:10/16:8 + 规律睡眠 + 轻度有氧。 | 主观专注度、情绪量表、睡眠质量。 | 躯体脆弱或有精神疾病史者，应在心理/精神科专业人士知情下实施。 |
| **长寿与抗衰老** | 延缓多系统老化；降龄 (Biological Age Reduction)。 | 自噬 (Autophagy)、线粒体更新、干细胞激活、mTOR/AMPK 平衡。 | 每季 1 次 FMD + 日常温和 TRF。 | 炎症指标 (CRP)、空腹胰岛素、肝酶、体能测试。 | 长寿效应多来自“长期一致性的小剂量刺激”，而非一次性极端断食。 |
| **免疫与自身免疫** | 缓解慢性炎症，重塑免疫谱。 | 免疫细胞凋亡与再生、肠道菌群重塑。 | 高风险，需在专科（风湿免疫/功能医学）中个案评估。 | 自身抗体滴度、炎症指标、症状评分。 | 对类风湿等疾病，断食应视为“辅助调节工具”，不能替代规范免疫抑制治疗。 |
